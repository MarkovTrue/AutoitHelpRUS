# Autoit Help RUS

Мне стало любопытно, за какое время смогу организовать систему и перевести справку из нескольких тысяч страниц.
- Основная база и логика была продумана и сделана за 3 дня.
- Ещё пара дней доводка системы и поиск проблем
- И 2 месяца, чтобы это всё оформить и выложить на GitHub
- В остальном шлифовкой можно заниматься бесконечно

В оригинальной справке, особенно для пользовательских функций, не всегда следуют шаблону оформления.
Разработчики пользовательских функций не всегда могут понятно объяснить, как она работает.
Поэтому очень строгий перевод один к одному отпал сам собой.

Русский вариант сделан более системно, по оформлению и по шаблону описания.
Киллер фича, которая снимает с меня всю ответственность - это кнопка, которая позволяет быстро увидеть оригинал страницы.
В любом случае перевод в основном очень хороший, современные нейросети ТОП.

Огромная благодарность автору перевода старой русской справки AZJIO, его работа взята за основу.

В этой репе не будет исходников, здесь хранится только скомпинированная версия справки.
Невозможно сразу проверить несколько тысяч страниц. По возможности будут выкладываться обновления.
