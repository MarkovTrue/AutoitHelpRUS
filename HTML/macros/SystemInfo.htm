Макросы системной информации

Ниже приведён список макросов системной информации.

Полный список макросов – здесь.

 

Макрос Описание
@CPUArch Возвращает "X86" для 32-битного и "X64" для 64-битного процессора.
@KBLayout Возвращает код раскладки клавиатуры. Возможные значения – в Приложении.
@MUILang Возвращает код многоязычного интерфейса, если доступен (Vista поддерживает по умолчанию). Возможные значения – в Приложении.
@OSArch Возвращает одно из значений: "X86", "IA64", "X64" – архитектура работающей операционной системы.
@OSLang Возвращает код языка операционной системы. Возможные значения – в Приложении.
@OSType Возвращает "WIN32_NT" для XP/2003/Vista/2008/Win7/2008R2/Win8/2012/Win8.1/2012R2.
@OSVersion Возвращает одно из значений: "WIN_11", "WIN_10", "WIN_81", "WIN_8", "WIN_7", "WIN_VISTA", "WIN_XP", "WIN_XPe",
    для серверных ОС: "WIN_2019", "WIN_2022", "WIN_2016", "WIN_2012R2", "WIN_2012", "WIN_2008R2", "WIN_2008", "WIN_2003".
@OSBuild Возвращает номер сборки ОС. Например, для Windows 2003 Server возвращает 3790.
@OSServicePack Информация о пакете обновлений в виде "Service Pack 3".
@ComputerName Сетевое имя компьютера.
@UserName ID текущего вошедшего пользователя.
@IPAddress1 IP-адрес первого сетевого адаптера. На некоторых компьютерах часто возвращает 127.0.0.1.
@IPAddress2 IP-адрес второго сетевого адаптера. Возвращает 0.0.0.0, если адаптер отсутствует.
@IPAddress3 IP-адрес третьего сетевого адаптера. Возвращает 0.0.0.0, если адаптер отсутствует.
@IPAddress4 IP-адрес четвёртого сетевого адаптера. Возвращает 0.0.0.0, если адаптер отсутствует.

@DesktopHeight Высота основного дисплея в пикселях (вертикальное разрешение).
@DesktopWidth Ширина основного дисплея в пикселях (горизонтальное разрешение).
@DesktopDepth Глубина цвета основного дисплея в бит/пикс.
@DesktopRefresh Частота обновления основного дисплея в герцах.
