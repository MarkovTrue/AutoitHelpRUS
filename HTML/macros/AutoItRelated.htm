Связанные с AutoIt

Ниже приведён список макросов, связанных с AutoIt.

Полный список макросов – здесь.

 

Макрос Описание
@Compiled Возвращает 1, если скрипт – скомпилированный исполняемый файл или .a3x; возвращает 0, если это .au3.
@error Флаг ошибки. См. функцию SetError().
@exitCode Код завершения, установленный оператором Exit.
@exitMethod Метод завершения скрипта. См. функцию OnAutoItExitRegister().
@extended Расширенный код возврата – используется в ряде функций, например StringReplace().
@NumParams Количество параметров, переданных при вызове пользовательской функции.
@ScriptName Имя файла запущенного скрипта.
@ScriptDir Каталог, содержащий запущенный скрипт. Завершающая обратная косая черта добавляется только если скрипт находится в корне диска.
@ScriptFullPath Эквивалентно @ScriptDir & "\" & @ScriptName.
@ScriptLineNumber Номер выполняемой строки – полезен для отладочных сообщений (например, для отображения места вызова функции). Значим только в неоткомпилированных скриптах; файлы #include возвращают собственную нумерацию строк.
@WorkingDir Текущий (активный) рабочий каталог. Завершающая обратная косая черта добавляется только если скрипт находится в корне диска.
@AutoItExe Полный путь и имя выполняющегося в данный момент исполняемого файла AutoIt. Для скомпилированных скриптов – путь к скомпилированному скрипту; для .a3x и .au3 – путь к интерпретатору, запустившему файл.
@AutoItPID Идентификатор процесса (PID) текущего скрипта.
@AutoItVersion Номер версии AutoIt, например, 3.3.10.2.
@AutoItX64 Возвращает 1, если скрипт выполняется под нативной x64-версией AutoIt.

@COM_EventObj Объект, для которого сработало COM-событие. Действителен только внутри функции-обработчика COM-события.
@GUI_CtrlId Идентификатор последнего нажатого элемента GUI. Действителен только в функции обработки события. См. GUICtrlSetOnEvent().
@GUI_CtrlHandle Дескриптор последнего нажатого элемента GUI. Действителен только в функции обработки события. См. GUICtrlSetOnEvent().
@GUI_DragId Идентификатор источника перетаскивания (Drag). Действителен только в событии Drop. См. GUISetOnEvent().
@GUI_DragFile Имя перетаскиваемого файла. Действителен только в событии Drop. См. GUISetOnEvent().
@GUI_DropId Идентификатор цели перетаскивания (Drop). Действителен только в событии Drop. См. GUISetOnEvent().
@GUI_WinHandle Дескриптор последнего нажатого окна GUI. Действителен только в функции обработки события. См. GUICtrlSetOnEvent().
@HotKeyPressed Последнее нажатое сочетание клавиш. См. функцию HotKeySet().

Для использования с функциями WinSetState(), Run(), RunWait, FileCreateShortcut() и FileGetShortcut():
@SW_DISABLE Отключает окно.
@SW_ENABLE Включает окно.
@SW_HIDE Скрывает окно и активирует другое окно.
@SW_LOCK Блокирует окно, чтобы избежать перерисовки.
@SW_MAXIMIZE Активирует окно и показывает его развёрнутым на весь экран.
@SW_MINIMIZE Сворачивает указанное окно и активирует следующее верхнеуровневое окно в Z-порядке.
@SW_RESTORE Активирует и показывает окно. Если окно свёрнуто или развёрнуто, система восстанавливает его исходный размер и положение. Приложение должно указывать этот флаг при восстановлении свёрнутого окна.
@SW_SHOW Активирует окно и показывает его в текущем размере и положении.
@SW_SHOWDEFAULT Устанавливает состояние отображения на основе SW_-значения, заданного программой, которая запустила приложение.
@SW_SHOWMAXIMIZED Активирует окно и показывает его развёрнутым на весь экран.
@SW_SHOWMINIMIZED Активирует окно и показывает его свёрнутым.
@SW_SHOWMINNOACTIVE Отображает окно свёрнутым. Аналогично @SW_SHOWMINIMIZED, но без активации окна.
@SW_SHOWNA Отображает окно в текущем размере и положении. Аналогично @SW_SHOW, но без активации окна.
@SW_SHOWNOACTIVATE Отображает окно в последнем размере и положении. Аналогично @SW_SHOWNORMAL, но без активации окна.
@SW_SHOWNORMAL Активирует и показывает окно. Если окно свёрнуто или развёрнуто, система восстанавливает его исходный размер и положение. Приложение должно указывать этот флаг при первом отображении окна.
@SW_UNLOCK Разблокирует окно, разрешая перерисовку.

@TRAY_ID Идентификатор последнего нажатого элемента при обработке события TraySetOnEvent() или TrayItemSetOnEvent().
@TrayIconFlashing Возвращает 1, если значок в трее мигает; иначе возвращает 0.
@TrayIconVisible Возвращает 1, если значок в трее видим; иначе возвращает 0.

@CR Возврат каретки, Chr(13); иногда используется для переноса строки.
@LF Перевод строки, Chr(10); иногда используется для переноса строки.
@CRLF @CR & @LF; обычно используется для переноса строки.
@TAB Символ табуляции, Chr(9).
