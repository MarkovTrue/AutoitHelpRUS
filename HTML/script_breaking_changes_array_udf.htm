Array UDF – критические изменения

Array UDF почти полностью переписан для поддержки двумерных массивов, ускорения множественных вставок и удалений, а также для исправления ошибок.
Это означает, что скрипты, написанные для AutoIt до версии 3.3.12.0, могут работать некорректно с более поздними версиями и потребуют изменений.
В таблице ниже перечислены различия между старой и новой версией.

Изменённые функции/параметры
Функция/параметр UDF Примечания
Все изменённые функции Значения @error при ошибках могли измениться – подробности на страницах конкретных функций.
_ArrayAdd() Теперь поддерживает двумерные массивы и может добавлять несколько строк за один вызов.
   $vValue Изменён: добавляемые значения – может быть одним элементом, строкой с разделителями или массивом.
   $iStart Добавлен: для двумерных массивов – столбец, начиная с которого вставляются данные в каждой строке (по умолчанию 0).
   $sDelim_Item
   $sDelim_Row		 Добавлены: если данные переданы строкой с разделителями, определяют количество добавляемых строк/столбцов.
   $iForce Добавлен: изменяет поведение функции – принудительно добавляет данные как один элемент или задаёт тип данных для добавляемых элементов.
_ArrayBinarySearch() Теперь поддерживает двумерные массивы.
   $iColumn Добавлен: индекс столбца для поиска, начиная с 0 (по умолчанию 0).
_ArrayConcatenate() Теперь поддерживает двумерные массивы с одинаковым числом столбцов.
_ArrayDelete() Теперь поддерживает удаление нескольких элементов.
   $vRange Изменён: удаляемые элементы – одиночный индекс, строка диапазона или одномерный массив с количеством в элементе [0].
_ArrayDisplay() Кнопки внизу диалога можно скрыть.
_ArrayFindAll() Теперь поддерживает поиск в двумерных массивах по строкам и по столбцам.
   $fRow Добавлен: если True, $iSubItem задаёт строку, а не столбец в двумерном массиве.
_ArrayInsert() Теперь поддерживает двумерные массивы и может вставлять несколько строк в одной или разных позициях за один вызов.
   $vRange Изменён: элементы, перед которыми выполняется вставка – одиночный индекс, строка диапазона или одномерный массив с количеством в элементе [0].
   $vValue Добавлен: вставляемые значения – может быть одним элементом, строкой с разделителями или массивом.
   $iStart Добавлен: для двумерных массивов – столбец, начиная с которого вставляются данные в каждой строке (по умолчанию 0).
   $sDelim_Item
   $sDelim_Row		 Добавлены: если данные переданы строкой с разделителями, определяют количество добавляемых строк/столбцов.
   $hDataType Добавлен: если $vValue – строка с разделителями, данные конвертируются в строку. Передача функции типа данных (Number, HWnd и т.п.) приводит данные к этому типу.
_ArrayMax() Теперь поддерживает двумерные массивы.
   $iSubItem Добавлен: столбец для поиска, начиная с 0.
_ArrayMaxIndex() Теперь поддерживает двумерные массивы.
   $iSubItem Добавлен: столбец для поиска, начиная с 0.
_ArrayMin() Теперь поддерживает двумерные массивы.
   $iSubItem Добавлен: столбец для поиска, начиная с 0.
_ArrayMinIndex() Теперь поддерживает двумерные массивы.
   $iSubItem Добавлен: столбец для поиска, начиная с 0.
_ArraySearch() Теперь поддерживает поиск в двумерных массивах по строкам и по столбцам.
   $fRow Добавлен: если True, $iSubItem задаёт строку, а не столбец в двумерном массиве.
_ArraySwap() Полностью переписан для перестановки целых или частичных столбцов/строк в двумерных массивах.
   $iIndex_1
   $iIndex_2		 Изменены: индексы переставляемых строк/столбцов.
    $bCol Добавлен: если True, $iIndex_1 и $iIndex_2 задают столбцы, а не строки.
   $iStart
   $iEnd		 Добавлены: элементы внутри переставляемых строк/столбцов.
_ArrayToClip() Теперь является простой обёрткой для _ArrayToString(). Параметры – см. документацию _ArrayToString().
_ArrayToString() Теперь поддерживает двумерные массивы.
   $sDelim_Item
   $sDelim_Row		 Добавлены: определяют разделители столбцов/строк.
   $iStart_Row
   $iEnd_Row		 Добавлены: ограничение диапазона извлекаемых строк.
   $iStart_Col
   $iEnd_Col		 Добавлены: ограничение диапазона извлекаемых столбцов.
_ArrayTranspose() Теперь транспонирует также одномерные массивы.
_ArrayTrim() Теперь поддерживает двумерные массивы.
   $iSubItem Добавлен: столбец для обрезки.
_ArrayUnique() Для единообразия с другими функциями $iColumn теперь начинается с 0, а не с 1.


Новые функции/параметры
Новая функция UDF Примечания
_ArrayColInsert() Вставляет столбцы в одномерные или двумерные массивы.
_ArrayColDelete() Удаляет столбцы из двумерных массивов.
_ArrayExtract() Извлекает подмассив из одномерного или двумерного массива.
_ArrayShuffle() Перемешивает все или часть строк одномерного или двумерного массива алгоритмом Фишера–Йейтса. Для двумерного массива перемешивание можно ограничить одним столбцом.
