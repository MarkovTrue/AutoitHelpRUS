Справочник ключевых слов

Null

Значение ключевого слова для использования при вызове функции.

$var =Null

Примечания

Используйте это ключевое слово для запуска специфических операций функций, которые его принимают как параметр.
Null вычисляется только в Null, хотя в математических операциях это ключевое слово имеет значение 0, и как строка оно не имеет значения (пустая строка).
Вычисление производится только во время математических операций. Для операции сравнения "if $v = Null" успех будет достигнут
при условии $v = Null.

Метод COM получит Null как VT_NULL VARIANT. Типы DllCall's wstr и str будут установлены на NULL (указатели).

См. также

Default, IsKeyword

Пример

; Окно сообщения с названием "Error".

DllCall("user32.dll", "int", "MessageBoxW", "hwnd", 0, "wstr", "Some text", "wstr", Null, "dword", 0)
