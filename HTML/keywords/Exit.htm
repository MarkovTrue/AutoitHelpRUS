Справочник ключевых слов

Exit

Завершает скрипт.

Exit [return code]

Параметры

return code [необязательный] Целое число, устанавливающее код возврата скрипта. Этот код может быть использован Windows или переменной DOS %ERRORLEVEL%. По умолчанию 0. Скрипты обычно устанавливают errorlevel 0, если скрипт выполнен правильно; уровни ошибок 1 и выше обычно указывают, что скрипт выполнен неправильно.

Примечания

Параметр, если включен, может быть заключен в скобки. Таким образом, следующие эквивалентны: Exit, Exit 0 и Exit(0). Однако Exit() недействителен.

Код можно получить в функции OnAutoItExitRegister через @exitCode.

См. также

ExitLoop, OnAutoItExitRegister

Пример

Пример 1

; очень простой скрипт

Exit

Пример 2

; Завершить скрипт, если нет аргументов командной строки
If $CmdLine[0] = 0 Then Exit (1)
