Квалификатор функции.

...
EndFunc

Примечания

Используйте квалификатор Volatile для изменения определённых аспектов выполнения функции.

Следующие правила применяются к функциям, объявленным с этим квалификатором:

Функция обратного вызова При выполнении функции основной насос сообщений AutoIt не блокируется, как это обычно происходит для обратных вызовов, не являющихся Volatile.
Функция события COM Выполнение синхронное (см. пример).

Обратные вызовы SetTimer не безопасны как Volatile, поэтому обязательно избегайте их.
Во всех остальных случаях нет разницы в выполнении функций Volatile по сравнению с обычными функциями.

Func _COM_Example()
    ; Создать объект Internet Explorer
    Local $oIE = ObjCreate("InternetExplorer.Application")
    ; Проверить ошибки
    If @error Then Return
    $oIE.Visible = True ; Установить видимость.
    ; Кастомный sink-объект
    ObjEvent($oIE, "_IEEvent_", "DWebBrowserEvents2")
    ; Перейти куда-нибудь
    $oIE.navigate("http://www.google.com")
    ; Проверить ошибки при загрузке
    If @error Then
        $oIE.Quit()
        Return
    EndIf
    ; Ожидать загрузки страницы.
    While 1
        If $oIE.readyState = "complete" Or $oIE.readyState = 4 Then ExitLoop
        Sleep(10)
    WEnd

    ; Подождать несколько секунд – проверить, появятся ли ещё события.
    Sleep(3000)
    ; Больше ничего делать не нужно. Закрыть IE и вернуться из функции
    $oIE.Quit()
EndFunc   ;==>_COM_Example
