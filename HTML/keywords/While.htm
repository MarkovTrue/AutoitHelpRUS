Справочник ключевых слов

While...WEnd

Цикл с условием.

While <expression>
statements
...
WEnd

Параметры

expression Если выражение истинно, выполняются следующие инструкции до инструкции WEnd. Цикл продолжается до тех пор, пока выражение не станет ложным.

Примечания

Инструкции While...WEnd могут быть вложены.
Выражение проверяется перед выполнением цикла, поэтому цикл будет выполнен нуль или более раз.
Для создания бесконечного цикла вы можете использовать ненулевое число как expression.

См. также

ContinueLoop, ExitLoop

Пример

Пример 1

#include <MsgBoxConstants.au3>

Local $i = 0
While $i <= 10
    MsgBox($MB_SYSTEMMODAL, "", "Value of $i is: " & $i)
    $i = $i + 1
WEnd

Пример 2

; Установить горячую клавишу Escape для завершения скрипта.
HotKeySet("{ESC}", "_Terminate")

Example()

Func Example()
    ; Инициализировать локальную переменную.
    Local $aMgp = 0

    ; Создать бесконечный цикл, 1 всегда будет 1, поэтому True.
    While 1
        ; Присвоить локальной переменной координаты курсора (массив).
        $aMgp = MouseGetPos()

        ; Вывести подсказку рядом с курсором с его координатами.
        ToolTip("x: " & $aMgp[0] & ", y: " & $aMgp[1], $aMgp[0] + 10, $aMgp[1] + 10)

        ; Избежать высокого использования CPU.
        Sleep(50)
    WEnd
EndFunc   ;==>Пример

Func _Terminate()
    Exit
EndFunc   ;==>_Terminate
