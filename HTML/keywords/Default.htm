Справочник ключевых слов

Default

Значение ключевого слова для использования при вызове функции.

$var = Default

Примечания

Это ключевое слово не стоит использовать в общем вычислительном выражении. AutoIt не обнаружит такие ситуации из-за штрафа на производительность.

При использовании в передаче параметров поведение определяется в соответствующей документации функции AutoIt.
Для UDF ответственность скриптера проверить, установлен ли параметр на Default, и выполнить нужное поведение в этой ситуации.
Если используется, переданный параметр будет установлен на ключевое слово Default, а не на необязательное значение параметра, если оно определено.

См. также

IsKeyword, Null

Пример

#include <MsgBoxConstants.au3>

Example(Default, Default)

Func Example($vParam1 = Default, $vParam2 = "Two", $vParam3 = Default)
    If $vParam1 = Default Then $vParam1 = "One" ; Если используется ключевое слово Default, присвоить переменной "One"
    If $vParam3 = Default Then $vParam3 = "Three" ; Если используется ключевое слово Default, присвоить переменной "Three"

    ; Вывести следующие параметры, переданные функции.
    MsgBox($MB_SYSTEMMODAL, "Paramaters", "1 = " & $vParam1 & @CRLF & _
            "2 = " & $vParam2 & @CRLF & _
            "3 = " & $vParam3)
EndFunc   ;==>Example
