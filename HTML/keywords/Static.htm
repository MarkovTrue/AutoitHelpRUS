Справочник ключевых слов

Static

Объявить статическую переменную или создать статический массив.

Static [Scope] $variable [ = initializer ]
Static [Scope] $aArray[subscript 1]...[subscript n] [ = initializer ]

Параметры

Scope Необязательный модификатор, Local или Global, указывающий, где переменная видна.
$variable Имя объявляемой статической переменной.
initializer Значение, которое будет изначально присвоено переменной. Инициализатор может быть вызовом функции или включать математические или строковые операции. Этот инициализатор вычисляется только при первой встречи этого объявления переменной.
subscript Количество элементов измерения массива (нумерация с 0 до n-1).

Примечания

Ключевое слово Static может появляться на строке перед необязательным спецификатором области видимости или после. например Local Static или Static Local оба приемлемы.

Если используется спецификатор области видимости Local, то статическая переменная видна и может использоваться только в функции, в которой она объявлена, и разрешается в среде выполнения (логическая область видимости). Это означает, что условно объявленная переменная видна только когда выполнено условие объявления. Если используется спецификатор области видимости Global, то статическая переменная видна и может использоваться во всех частях скрипта; в этом отношении Global Static имеет очень мало отличий от переменной Global. Если спецификатор области видимости не используется, то статическая переменная будет создана в локальной области видимости; таким образом Static похожа на Dim.

Разница между Local и Static – время жизни переменной. Локальные переменные хранятся только во время вызова функции и видны только внутри функции, в которой они объявлены; когда функция возвращается, все её локальные переменные высвобождаются. Статические переменные также видны только в функции, в которой они объявлены, но они продолжают существовать, сохраняя своё последнее значение после завершения выполнения функции. При поиске переменных сначала проверяется локальная область видимости, затем глобальная.

Ключевое слово Static выполняет функции, аналогичные ключевым словам Global/Local/Dim.

  1. Они все объявляют переменную перед её использованием.

  2. Они все могут создавать массив.


Примечание: Статические переменные должны быть объявлены с помощью ключевого слова Static перед использованием, независимо от того, как устанавливается AutoItSetOption("MustDeclareVars"). Статические переменные не могут быть Const.

Вы также можете объявить несколько статических переменных на одной строке:

Static $a, $b, $c


И инициализировать переменные:

Static $a = 2, $b = 10, $c = 20



При инициализации статической переменной значение инициализации вычисляется и присваивается только первый раз, когда создаётся переменная. При всех последующих проходах инициализатор игнорируется.

См. Local для более подробной информации об использовании массивов, которые имеют всю одинаковую функциональность как в Local, за исключением:
  1. Переинициализация статической переменной не имеет эффекта.

  2. Вы не можете изменить статическую переменную на локальную или глобальную переменную и наоборот.


Если вы хотите изменить размер массива, используйте ReDim.

См. также

AutoItSetOption, Local, ReDim, UBound

Пример

Пример 1

#include <MsgBoxConstants.au3>

; Вызвать функцию Example для инициализации статической переменной в локальной области видимости.
Example()

; Вызвать функцию Example второй раз, чтобы показать, что переменная сохранила данные, которые мы ей последний раз присвоили.
Example()

Func Example()
    Local Static $sString = "This is a line of text which is declared using a Static variable in Local scope." & @CRLF & @CRLF & _
            "The variable $sString will now be visible to this function only and until the script closes."
    MsgBox($MB_SYSTEMMODAL, "", $sString)
    $sString = "If using just Local scope this string wouldn't be visible if this function was called multiple times, but since we're using the Static keyword" & @CRLF & _
            "the variable $sString will retain the data last assigned to it."
EndFunc   ;==>Example

Пример 2

#include <MsgBoxConstants.au3>

Example()

Func Example()
    SomeFunc() ; Это выведет окно сообщения 1, 1.
    SomeFunc() ; Это выведет окно сообщения 1, 2.
    SomeFunc() ; Это выведет окно сообщения 1, 3.
EndFunc   ;==>Example

Func SomeFunc()
    ; Это инициализирует статическую переменную в локальной области видимости. Когда переменная объявляется только в локальной области видимости (внутри функции,)
    ; она уничтожается, когда функция завершается/возвращается. Это не имеет место для статической переменной. Переменная не может быть
    ; доступна с любого другого места в скрипте, кроме функции, в которой она была объявлена.
    Local Static $vVariableThatIsStatic = 0
    Local $vVariableThatIsLocal = 0
    $vVariableThatIsLocal += 1 ; Это всегда будет 1, так как она была уничтожена, когда возвращалась из SomeFunc.
    $vVariableThatIsStatic += 1 ; Это будет увеличиваться на 1.
    MsgBox($MB_SYSTEMMODAL, $vVariableThatIsLocal, $vVariableThatIsStatic)
EndFunc   ;==>SomeFunc
