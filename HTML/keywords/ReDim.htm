Справочник ключевых слов

ReDim

Изменяет размер существующего массива.

ReDim $aArray[subscript 1]...[subscript n]

Параметры

$aArray Имя массива для изменения размера.
subscript Количество элементов измерения массива (нумерация с 0 до n-1).

Примечания

Ключевое слово ReDim похоже на Global/Local, за исключением того, что ReDim сохраняет значения в массиве вместо удаления значений при изменении размера массива.
Количество измерений должно оставаться прежним, иначе старый массив будет забыт во время ReDim.
Массив сохранит область видимости (Global или Local), которую он имел до изменения размера.

См. также

Dim, UBound

Пример

Пример 1

#include <Array.au3>

Example()

Func Example()
    Local $aArray[1] ; Создать одномерный массив с 1 элементом.
    For $i = 0 To 100 ; Перейти через значения от 0 до 100 для доступа к индексу массива
        ReDim $aArray[UBound($aArray) + 1] ; Определить текущий размер массива и увеличить на 1
        $aArray[$i] = $i
    Next
    _ArrayDisplay($aArray) ; Вывести массив.
EndFunc   ;==>Пример

Пример 2

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aArray[1] = [0], $iArrayTotal = 5000, $iCount = 0, $iDimension = 0

    Local $hTimer = TimerInit() ; Запустить таймер.
    For $i = 1 To $iArrayTotal
        ; Возвращает True если ReDimmed или False если не было. Это использует ByRef, поэтому просто передайте Array и ранее объявленную переменную для мониторинга размерности.
        If ReDim1D($aArray, $iDimension) Then
            $iCount += 1 ; Если массив был ReDimmed, то добавить к счётчику ReDim для вывода в конце примера.
        EndIf
        $aArray[0] += 1 ; Увеличить нулевой индекс [0] на значение 1.
        $aArray[$i] = "Row " & $i & ": Col 0" ; Добавить случайные данные.
    Next
    $hTimer = Round(TimerDiff($hTimer)) ; Завершить таймер и найти разницу от начала до конца.

    ReDim $aArray[$aArray[0] + 1] ; Удалить пустые элементы ReDimming в общий счёт плюс элемент для удержания счёта.

    MsgBox($MB_SYSTEMMODAL, "How many times?", "The amount of times a " & $iArrayTotal & " element array was ""ReDimmed"" was " & $iCount & " times." & @CRLF & _
            "It only took " & $hTimer & " milliseconds to complete.")
    ; При использовании метода увеличения размера массива на 1, $iCount вернула бы 5000, а не 29, как показано в этом примере.
EndFunc   ;==>Пример

; ReDim при необходимости путём увеличения общего размера массива как totalsize * 1.3 и округления до следующего наибольшего целого числа.
Func ReDim1D(ByRef $aArray, ByRef $iDimension) ; Где нулевой индекс [0] – счёт элементов.
    Local $bReturn = False
    If ($aArray[0] + 1) >= $iDimension Then
        $bReturn = True
        $iDimension = Ceiling(($aArray[0] + 1) * 1.3)
        ReDim $aArray[$iDimension]
    EndIf
    Return $bReturn
EndFunc   ;==>ReDim1D
