Справочник ключевых слов

#OnAutoItStartRegister

Регистрирует функцию, которая будет вызвана при запуске AutoIt.

#OnAutoItStartRegister "function"

Параметры

function Имя вызываемой пользовательской функции.

Примечания

Так как функция запускается перед выполнением любого кода основного скрипта, функция не может ссылаться на переменные, определённые в #include, и не может ссылаться на переменные, определённые как Global внутри скрипта. Все переменные внутри функции рассматриваются как Local.

См. также

OnAutoItExitRegister

Пример

#include <MsgBoxConstants.au3>

; Зарегистрировать Example() и SomeFunc() для вызова при запуске AutoIt.

#OnAutoItStartRegister "Example"
#OnAutoItStartRegister "SomeFunc"

Sleep(1000)

Func Example()
    MsgBox($MB_SYSTEMMODAL, "", "Function 'Example' is called first.")
EndFunc   ;==>Example

Func SomeFunc()
    MsgBox($MB_SYSTEMMODAL, "", "Function 'SomeFunc' is called second.")
EndFunc   ;==>SomeFunc
