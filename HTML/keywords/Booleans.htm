Справочник ключевых слов

False / True

Булевы значения для использования в логических выражениях.

$var = <strong>False</strong>
$var = <strong>True</strong>

Примечания

Эти ключевые слова стоит использовать только в логических выражениях, так как AutoIt не обнаружит такого "неправильного" использования и результаты будут непредсказуемыми.

См. также

IsBool

Пример

#include <MsgBoxConstants.au3>

Local $bBoolean = False
If Not ($bBoolean = True) Then MsgBox($MB_SYSTEMMODAL, "Bool comparison", "OK") ; Если $bBoolean НЕ равна True, то показать окно сообщения
