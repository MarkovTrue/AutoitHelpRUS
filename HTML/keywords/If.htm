Справочник ключевых слов

If...Then

Условно выполнить одну инструкцию.

<strong>If</strong> <expression> <strong>Then</strong> <em>statement</em>

Параметры

expression Если выражение истинно, инструкция выполняется.

Примечания

Эта версия инструкции If используется для выполнения одной инструкции без затрат на EndIf.
expression может содержать булевы операторы And, Or и Not, а также логические операторы <, <=, >, >=, =, ==, и <>, сгруппированные скобками по мере необходимости.

См. также

If...Else...EndIf, Select...Case...EndSelect, Switch...EndSwitch, Ternary

Пример

#include <MsgBoxConstants.au3>

; Завершает скрипт, если нет аргументов командной строки
If $CmdLine[0] = 0 Then Exit

; Альтернатива:
If $CmdLine[0] = 0 Then
    Exit
EndIf

MsgBox($MB_SYSTEMMODAL, "", "Script has command-line arguments.")
