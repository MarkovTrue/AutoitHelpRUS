Справочник ключевых слов

Switch...Case...EndSwitch

Условное выполнение операторов.

Switch <expression>
Case <value> [To <value>] [,<value> [To <value>] ...]
statement1
...
[Case <value> [To <value>] [,<value> [To <value>] ...]
statement2
...]
[Case Else
statementN
...]
EndSwitch

Параметры

<expression> Выражение, которое возвращает значение. Значение из выражения затем сравнивается со значениями каждого случая до тех пор, пока не будет найдено совпадение. Это выражение всегда вычисляется ровно один раз каждый раз при проходе через структуру.
<value> To <value> Случай выполняется, если выражение находится между двумя значениями.
<value> Случай выполняется, если выражение соответствует значению.

Примечания

Если ни один из случаев не соответствует значению Switch, то выполняется раздел Case Else, если он присутствует. Если ни один случай не совпадает и Case Else не определен, то ни один из кодов внутри структуры Switch, кроме начального выражения, не будет выполнен.

Инструкции Switch могут быть вложены. Инструкции Switch не чувствительны к регистру.

См. также

If...Then, If...Else...EndIf, Select...EndSelect, ContinueCase

Пример

#include <MsgBoxConstants.au3>

Local $sMsg = ""

Switch @HOUR
    Case 6 To 11
        $sMsg = "Good Morning"
    Case 12 To 17
        $sMsg = "Good Afternoon"
    Case 18 To 21
        $sMsg = "Good Evening"
    Case Else
        $sMsg = "What are you still doing up?"
EndSwitch

MsgBox($MB_SYSTEMMODAL, "", $sMsg)
