Справочник операторов

Ternary

Условно выбирает один из двух ответов на основе результата выражения.

(expression) ?(expression1 if expression isTrue) :(expression2 if expression isFalse)

Параметры

expression Если выражение истинно, выполняется выражение1 – если ложно, выполняется выражение2.

Примечания

Этот оператор условия позволяет выполнить бинарный выбор без затрат на структуру If...Else...EndIf.

Хотя не всегда необходимо, настоятельно рекомендуется заключить 3 элемента в скобки. (См. оператор приоритета для уверенности).

См. также

If...Else...EndIf, Select...Case...EndSelect, Switch...EndSwitch

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Значения одинаковые, поэтому выражение True
    MsgBox($MB_SYSTEMMODAL, "Result: 1=1", (1 = 1) ? "True!" : "False!")
    ; Значения не одинаковые, поэтому выражение False
    MsgBox($MB_SYSTEMMODAL, "Result: 1=2", (1 = 2) ? "True!" : "False!")
EndFunc   ;==>Пример
