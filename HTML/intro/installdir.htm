Каталог установки

Установщик AutoIt создаёт структуру каталогов (обычно в \Program Files\AutoIt3), приведённую в таблице ниже. Установщик также создаёт ярлыки в меню "Пуск", но никаких других файлов не добавляется и не изменяется.

Файлы и каталоги Описание
(Файлы верхнего уровня)
AutoIt3.exe Главная программа AutoIt и единственный файл, необходимый для запуска скриптов!
AutoIt3_x64.exe x64-версия AutoIt (если установлена).
Au3Info.exe AutoIt Window Info Tool.
Au3Info_x64.exe x64-версия Au3Info (если установлена).
AU3Check.exe Проверка синтаксиса AutoIt.
AutoIt.chm Этот файл справки.
Uninstall.exe Деинсталлятор AutoIt.
AutoIt v3 Website.url Ярлык на https://www.autoitscript.com/site/autoit/
Aut2Exe

Icons\ Содержит значки, используемые для значка типа файла .au3 в Проводнике.

Aut2Exe.exe Компилятор скриптов.

Aut2Exe_x64.exe x64-версия Aut2Exe (если установлена).

UPX.exe Компрессор UPX (уменьшает размер exe-файлов).
Examples

COM\ Содержит примеры COM на AutoIt.

GUI\ Содержит примеры GUI на AutoIt.

Helpfile\ Содержит скрипты, используемые во многих примерах из файла справки.
Extras

AutoUpdateIt\ Содержит скрипт для удобного получения последней версии AutoIt3.

Editors\ Содержит определения подсветки синтаксиса для нескольких популярных текстовых редакторов.

Geshi\ Содержит файл подсветки синтаксиса для серверного подсветчика Geshi.

Prettify\ Содержит файл подсветки синтаксиса для клиентского JavaScript-подсветчика Google Prettify.
Icons

Содержит значки, используемые для значка типа файла .au3 в Проводнике.
Include

Содержит стандартные include-файлы (готовые пользовательские функции). См. справочник пользовательских функций.
AutoItX

Содержит DLL-версию AutoIt v3, которая предоставляет подмножество возможностей AutoIt через ActiveX/COM- и DLL-интерфейсы.
SciTE

Содержит лёгкую версию SciTE с подсветкой синтаксиса.

Стоит отметить, что для запуска AutoIt-скриптов нужен только файл AutoIt3.exe. Если скомпилировать скрипт в исполняемый файл, пользователю не нужен установленный AutoIt для запуска такого исполняемого файла.

Ключи реестра

Установщик AutoIt создаёт ключи реестра в HKEY_LOCAL_MACHINE\Software\AutoIt v3 (или HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\AutoIt v3\ на 64-битной машине) и в HKEY_CURRENT_USER\Software\AutoIt v3. Ключи НЕ используются и не создаются, когда утилиты AutoIt запускаются на машинах без полной установки – AutoIt "чист" для запуска на серверах и т. д.

В таблице ниже приведены ключи реестра по умолчанию (или типовые). Ключи курсивом не создаются самим установщиком – их создаёт первый запуск соответствующей утилиты:

Version REG_SZ Version Number
