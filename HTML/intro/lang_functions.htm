Функции

Функция – это участок кода, который можно вызвать из скрипта для выполнения определённой задачи. В AutoIt существует два вида функций: встроенные и пользовательские.

Имена функций нечувствительны к регистру: msgbox() читается как MsgBox(), а MyFunc() и MyfunC() – одно и то же.

Пользовательские функции

Пользовательские функции объявляются с помощью операторов Func...EndFunc.

Функции могут принимать параметры и возвращать значения по необходимости.

Имя функции должно начинаться с буквы или подчёркивания, остальная часть имени может содержать любую комбинацию букв, цифр и подчёркиваний. Примеры допустимых имён:

    MyFunc

    Func1

    _My_Func1

Пример использования функции для удвоения числа 10 раз:

#include <MsgBoxConstants.au3>

Local $iNumber = 10
Local $iDoubled = 0

For $i = 1 To 10
    $iDoubled = MyDouble($iNumber)
    MsgBox($MB_SYSTEMMODAL, "", $iNumber & " doubled is " & $iDoubled)
    $iNumber = $iDoubled
Next
Exit

Func MyDouble($iValue)
    $iValue = $iValue * 2
    Return $iValue
EndFunc   ;==>MyDouble
