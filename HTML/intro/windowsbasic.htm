Заголовки и текст окон (основы)

При автоматизации большинство окон можно однозначно идентифицировать по заголовку или сочетанию заголовка и текста. С помощью AutoIt Window Info Tool (или просто на глаз) эту информацию легко получить. Заголовки большинства окон вполне очевидны: например, Безымянный – Блокнот – это заголовок редактора notepad.exe, и часто этого достаточно для автоматизации.

 

Примечание: если для параметров title и text передать пустую строку "", будет использоваться текущее активное окно (в некоторых более продвинутых режимах WinTitleMatchModes это правило не работает)!

 

Заголовки окон и текст чувствительны к регистру. Регистр и пунктуация должны совпадать в точности. Чтобы избежать ошибок, выделите заголовок/текст в Window Info Tool и скопируйте через ctrl-c, затем вставьте прямо в скрипт.
Можно принудительно проверять совпадение в нижнем регистре через расширенные режимы.

 

В большинстве оконных функций AutoIt есть параметры для заголовка и текста, например функция WinWaitActive. Эта функция приостанавливает выполнение скрипта, пока не появится и не станет активным указанное окно.

WinWaitActive ( "title" [, "text" [, timeout]] )

 

Единственный обязательный параметр – title. Параметры text и timeout необязательны. В некоторых функциях параметр text не является необязательным. Если не нужно указывать текст, передайте просто "" (пустую строку). Пустая строка или вообще отсутствие текста в параметре text означает, что любой текст подходит.

 

Использовать функцию выше для любого окна Notepad можно обоими способами:

WinWaitActive("Untitled - Notepad")

и

WinWaitActive("Untitled - Notepad", "")

Обратите внимание, что Window Info Tool "видит" заголовок и текст окна Notepad. Всё, что видит Window Info Tool, видит и AutoIt. Теперь у нас достаточно информации, чтобы однозначно идентифицировать именно это окно, даже если открыто много других окон Notepad. В этом случае используем:

 

WinWaitActive("Untitled - Notepad", "This is some text!")

 

Текст окна

Текст окна – это весь текст, который "видит" AutoIt. Обычно это содержимое контролов edit (как выше с "This is some text!"), но также включает другой текст:

  • Текст кнопок: &Да, &Нет, &Далее (символ & обозначает подчёркнутую букву).
  • Текст диалогов: "Вы уверены, что хотите продолжить?".
  • Текст контролов.
  • Прочий текст – иногда непонятно, откуда он :)

Главное – вы можете использовать текст вместе с заголовком, чтобы однозначно идентифицировать окно.

Параметр text в оконных функциях рассматривается как подстрока. Поэтому в примере выше если передать "is some ", совпадение будет найдено.

 

Описанное – это режим по умолчанию, в котором работает AutoIt. Существует ряд более расширенных режимов для случаев, когда всё не так просто.

 

Примечание: скрытые окна можно сопоставлять только по "title", если "text" пуст ("").
