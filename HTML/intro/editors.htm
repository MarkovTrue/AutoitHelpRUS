Редакторы скриптов

Скрипты AutoIt – это обычные текстовые файлы, которые можно редактировать в любом текстовом редакторе. Однако существует много бесплатных и условно-бесплатных редакторов, гораздо лучше подходящих для написания скриптов; во многих из них есть подсветка синтаксиса, которая выделяет ключевые слова и функции AutoIt – это значительно упрощает написание скриптов и снижает риск ошибок.

На данный момент большинство пользователей AutoIt используют SciTE. Команда разработки AutoIt создала специальную сборку SciTE с полной подсветкой синтаксиса, которая интегрирует разные сторонние инструменты AutoIt (проверка синтаксиса, форматирование скриптов и т. д.). "Лёгкая" версия SciTE поставляется вместе с установочным пакетом AutoIt. Полную AutoIt-версию SciTE со всеми инструментами можно скачать отдельно: http://www.autoitscript.com/autoit3/scite/.

 

Другие рекомендуемые редакторы:

  • TextPad
  • Crimson Editor (бесплатный)
  • Source Edit (бесплатный)
  • UltraEdit
  • Notepad++ (бесплатный)

В поставке AutoIt есть готовые файлы синтаксиса для многих редакторов – их можно найти в каталоге Extra (ссылка есть в меню Пуск / AutoIt v3 / Extra).
