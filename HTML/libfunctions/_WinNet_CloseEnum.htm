Справочник функций

_WinNet_CloseEnum

Завершает перечисление сетевых ресурсов, начатое вызовом _WinNet_OpenEnum().

#include <WinNet.au3>
_WinNet_CloseEnum ( $hEnum )

Параметры

$hEnum Дескриптор, который идентифицирует экземпляр перечисления. Дескриптор возвращается функцией _WinNet_OpenEnum().

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
