_WinAPI_GetFinalPathNameByHandleEx

Получает конечный полный путь указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetFinalPathNameByHandleEx ( $hFile [, $iFlags = 0] )

Параметры

$hFile Дескриптор файла или каталога.
$iFlags [необязательный] Тип возвращаемого результата.
Этот параметр может быть комбинацией одного значения $FILE_NAME_* и одного значения $VOLUME_NAME_*.
$FILE_NAME_NORMALIZED
$FILE_NAME_OPENED

$VOLUME_NAME_DOS
$VOLUME_NAME_GUID
$VOLUME_NAME_NONE
$VOLUME_NAME_NT

Возвращаемое значение

Успех: Возвращает полный путь файла.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

Дополнительно

Искать GetFinalPathNameByHandle в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

Local $hFile = _WinAPI_CreateFile(@ScriptFullPath, 2, 0, 6)

ConsoleWrite(_WinAPI_GetFinalPathNameByHandleEx($hFile) & @CRLF)
ConsoleWrite(_WinAPI_GetFinalPathNameByHandleEx($hFile, $VOLUME_NAME_GUID) & @CRLF)
ConsoleWrite(_WinAPI_GetFinalPathNameByHandleEx($hFile, $VOLUME_NAME_NT) & @CRLF)
ConsoleWrite(_WinAPI_GetFinalPathNameByHandleEx($hFile, $VOLUME_NAME_NONE) & @CRLF)

_WinAPI_CloseHandle($hFile)
