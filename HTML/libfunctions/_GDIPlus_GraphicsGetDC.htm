_GDIPlus_GraphicsGetDC

Возвращает дескриптор контекста устройства объекта Graphics.

#include <GDIPlus.au3>
_GDIPlus_GraphicsGetDC ( $hGraphics )

Параметры

$hGraphics Дескриптор объекта Graphics.

Возвращаемое значение

Успех: Возвращает дескриптор DC объекта Graphics.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

Каждый вызов _GDIPlus_GraphicsGetDC() должен сопровождаться вызовом _GDIPlus_GraphicsReleaseDC().
Между этими вызовами нельзя обращаться к методам объекта Graphics.

См. также
