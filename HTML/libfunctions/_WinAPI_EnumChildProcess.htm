_WinAPI_EnumChildProcess

Перечисляет дочерние процессы указанного процесса.

#include <WinAPIProc.au3>
_WinAPI_EnumChildProcess ( [$iPID = 0] )

Параметры

$iPID [необязательный] PID процесса. По умолчанию (0) – текущий процесс.

Возвращаемое значение

Успех: Возвращает двумерный массив PID и имён процессов.
[0][0] – количество строк массива (n)
[0][1] – не используется
[n][0] – PID
[n][1] – имя процесса
Ошибка: Устанавливает @error ≠ 0.

Пример

#include <Array.au3>
#include <WinAPIProc.au3>

Local $aData = _WinAPI_EnumChildProcess(_WinAPI_GetParentProcess())

_ArrayDisplay($aData, '_WinAPI_EnumChildProcess')
