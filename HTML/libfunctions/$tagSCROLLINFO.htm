Справочник ключевых слов

$tagSCROLLINFO

Содержит параметры полосы прокрутки для сообщения $SBM_SETSCROLLINFO или $SBM_GETSCROLLINFO.

Global Const $tagSCROLLINFO = "uint cbSize;uint fMask;int nMin;int nMax;uint nPage;int nPos;int nTrackPos"

Параметры

cbSize Размер структуры в байтах. Вызывающий код должен установить это значение в DllStructGetSize($tagSCROLLINFO).
fMask Указывает параметры полосы прокрутки для установки или извлечения. Может быть комбинацией следующих значений:
$SIF_ALL – комбинация $SIF_PAGE, $SIF_POS, $SIF_RANGE и $SIF_TRACKPOS.
$SIF_DISABLENOSCROLL – используется только при установке параметров полосы прокрутки. Если новые параметры делают полосу прокрутки ненужной, отключает полосу вместо удаления.
$SIF_PAGE – член nPage содержит размер страницы для пропорциональной полосы прокрутки.
$SIF_POS – член nPos содержит позицию ползунка (не обновляется во время перетаскивания).
$SIF_RANGE – члены nMin и nMax содержат минимальное и максимальное значения диапазона прокрутки.
$SIF_TRACKPOS – член nTrackPos содержит текущую позицию ползунка во время его перетаскивания.
nMin Минимальная позиция прокрутки.
nMax Максимальная позиция прокрутки.
nPage Размер страницы. Полоса прокрутки использует это значение для определения размера пропорционального ползунка.
nPos Позиция ползунка.
nTrackPos Текущая позиция ползунка, которого пользователь перетаскивает.
Приложение может получить это значение при обработке кода запроса $SB_THUMBTRACK.
Приложение не может установить немедленную позицию прокрутки, функция SetScrollInfo игнорирует этот член.

Примечания

$SIF_xxxxx и $SB_XXX для полосы прокрутки требуют #include <ScrollbarsConstants.au3>
