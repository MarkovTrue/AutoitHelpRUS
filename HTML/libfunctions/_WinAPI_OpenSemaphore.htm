_WinAPI_OpenSemaphore

Открывает существующий именованный объект семафора.

#include <WinAPIProc.au3>
_WinAPI_OpenSemaphore ( $sSemaphore [, $iAccess = $SEMAPHORE_ALL_ACCESS [, $bInherit = False]] )

Параметры

$sSemaphore Имя семафора для открытия. Сравнение имён чувствительно к регистру.
$iAccess [необязательный] Доступ к объекту семафора. Функция завершается с ошибкой, если дескриптор безопасности указанного
объекта не позволяет требуемый доступ для вызывающего процесса.
Этот параметр может быть одним из следующих значений:
$SEMAPHORE_ALL_ACCESS (по умолчанию)
$SEMAPHORE_MODIFY_STATE
$bInherit [необязательный] Указывает, может ли дескриптор быть унаследован процессами; допустимые значения:
True - Процессы, созданные этим процессом, унаследуют дескриптор.
False - Процессы не наследуют этот дескриптор (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор объекта семафора.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция успешно выполняется только если какой-либо процесс уже создал семафор с использованием функции _WinAPI_CreateSemaphore().
Вызывающий процесс может использовать возвращённый дескриптор в любой функции, которая требует дескриптор
объекта семафора.
