_GUICtrlComboBoxEx_LimitText

Ограничивает длину текста, который пользователь может вводить в поле ввода.

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_LimitText ( $hWnd [, $iLimit = 0] )

Параметры

$hWnd Дескриптор элемента управления.
$iLimit [необязательный] Ограничение длины текста

Возвращаемое значение

Отсутствует.

Примечания

Если параметр $iLimit равен нулю, длина текста ограничивается 0x7FFFFFFE символами.

Если ComboBox не имеет стиля $CBS_AUTOHSCROLL, установка ограничения длины текста больше размера поля ввода не действует.

Функция _GUICtrlComboBox_LimitText() ограничивает только текст, который может ввести пользователь. Она не влияет на текст, уже находящийся в поле ввода при отправке сообщения, и не влияет на длину текста, скопированного в поле ввода при выборе строки из ListBox.

По умолчанию пользователь может ввести в поле ввода максимум 30 000 символов.

Пример

#include <DirConstants.au3>
#include <GuiComboBoxEx.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hCombo

    ; Создать GUI
    $hGUI = GUICreate("ComboBoxEx Limit Text", 400, 300)
    $hCombo = _GUICtrlComboBoxEx_Create($hGUI, "", 2, 2, 394, 100)
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, "", $DDL_DRIVES, False)
    _GUICtrlComboBoxEx_AddDir($hCombo, "", $DDL_DRIVES)
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBoxEx_EndUpdate($hCombo)
    _GUICtrlComboBoxEx_EndUpdate($hCombo)

    ; Ограничить текст в edit
    _GUICtrlComboBoxEx_LimitText($hCombo, 10)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
