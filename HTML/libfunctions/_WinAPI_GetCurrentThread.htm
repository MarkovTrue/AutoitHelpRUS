_WinAPI_GetCurrentThread

Получает псевдодескриптор вызывающего потока.

#include <WinAPIProc.au3>
_WinAPI_GetCurrentThread ( )

Возвращаемое значение

Возвращает псевдодескриптор текущего потока.

Примечания

Псевдодескриптор – особая константа, которая интерпретируется как дескриптор текущего потока.
Вызывающий поток может использовать его, чтобы указать сам на себя везде, где требуется дескриптор потока.
