Справочник функций

_WinAPI_GetCurrentThread

Получает псевдо-дескриптор текущего потока.

#include <WinAPIProc.au3>
_WinAPI_GetCurrentThread ( )

Возвращаемое значение

Возвращает псевдодескриптор текущего потока.

Примечания

Псевдо-дескриптор – это специальная константа, в настоящее время (DWORD)-2, которая интерпретируется как дескриптор текущего потока во всех API, принимающих дескриптор потока как входные данные. Вызывающий поток может использовать этот дескриптор для указания собственного потока, когда требуется дескриптор потока. Псевдо-дескрипторы не наследуются дочерними процессами.
