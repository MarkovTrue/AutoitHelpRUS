Справочник функций

_EventLog__Notify

Разрешает приложению получать уведомления о событиях журнала.

#include <EventLog.au3>
_EventLog__Notify ( $hEventLog, $hEvent )

Параметры

$hEventLog Дескриптор журнала событий.
$hEvent Дескриптор объекта события с ручным сбросом.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция не работает с удалёнными дескрипторами.
Если $hEventLog – дескриптор журнала событий на удалённом компьютере, функция возвращает ноль, а GetLastError возвращает ERROR_INVALID_HANDLE.
Когда в журнал, заданный $hEventLog, записывается событие, система вызывает PulseEvent для перевода объекта $hEvent в сигнальное состояние.
Если поток не ожидает событие в момент вызова PulseEvent, уведомление не будет доставлено.
Поэтому рекомендуется создавать отдельный поток для ожидания уведомлений.
Система вызывает PulseEvent не чаще одного раза в пять секунд.
Поэтому даже при нескольких изменениях журнала в течение пяти секунд будет получено только одно уведомление.
Система продолжает уведомлять об изменениях до закрытия дескриптора журнала событий.
Для закрытия используйте _EventLog__Close() или _EventLog__DeregisterSource().

См. также

_EventLog__Close, _EventLog__DeregisterSource

Пример

#include "Extras\HelpFileInternals.au3"

#include <EventLog.au3>
#include <FontConstants.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

Example()

Func Example()
    Local $hEventLog, $hEvent, $iResult

    ; Создать GUI
    GUICreate("EventLog", 600, 300)
    _MemoCreate(2, 2, 596, 294, 0)
    GUISetState(@SW_SHOW)

    ; Настраиваем объект события.
    $hEventLog = _EventLog__Open("", "Security")
    $hEvent = _WinAPI_CreateEvent(0, False, False, "")
    _EventLog__Notify($hEventLog, $hEvent)

    ; Ожидаем нового события.
    _MemoWrite("Waiting for new event")
    $iResult = _WinAPI_WaitForSingleObject($hEvent)
    _WinAPI_CloseHandle($hEvent)
    _EventLog__Close($hEventLog)

    ; Выводим результаты.
    If $iResult = -1 Then
        _MemoWrite("Wait failed")
    Else
        _MemoWrite("New event occurred")
    EndIf

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
