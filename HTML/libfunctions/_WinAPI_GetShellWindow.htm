Справочник функций

_WinAPI_GetShellWindow

Получает дескриптор окна рабочего стола оболочки.

#include <WinAPISysWin.au3>
_WinAPI_GetShellWindow ( )

Возвращаемое значение

Успех: Возвращает дескриптор окна рабочего стола оболочки.
Ошибка: Возвращает 0.
