Справочник функций

_WinAPI_UrlCanonicalize

Преобразует строку URL в каноническую форму.

#include <WinAPIShPath.au3>
_WinAPI_UrlCanonicalize ( $sUrl, $iFlags )

Параметры

$sUrl URL.
$iFlags Флаги, которые указывают, как должен быть преобразован URL. Это может быть комбинацией следующих значений.
$URL_DONT_SIMPLIFY
$URL_ESCAPE_PERCENT
$URL_ESCAPE_SPACES_ONLY
$URL_ESCAPE_UNSAFE
$URL_NO_META
$URL_PLUGGABLE_PROTOCOL
$URL_UNESCAPE

Windows 7 или более поздняя версия
$URL_ESCAPE_AS_UTF8

Возвращаемое значение

Успех: Возвращает преобразованный URL.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Если строка URL содержит "/../" или "/./", _WinAPI_UrlCanonicalize() обычно обрабатывает символы так, как будто они
указывают навигацию в иерархии URL. Функция упрощает URL перед их объединением. Например,
"/hello/cruel/../world" упрощается до "/hello/world". Если установлен флаг $URL_DONT_SIMPLIFY, функция
не упрощает URL. В этом случае "/hello/cruel/../world" остаётся как есть.

Дополнительно

Искать UrlCanonicalize в библиотеке MSDN

Пример

#include <WinAPIShPath.au3>

Local $sUrl = 'http://msdn.microsoft.com/en-us/library/ee663300%28VS.85%29.aspx'

ConsoleWrite(_WinAPI_UrlCanonicalize($sUrl, $URL_UNESCAPE) & @CRLF)
