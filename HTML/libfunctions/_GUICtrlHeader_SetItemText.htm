Справочник функций

_GUICtrlHeader_SetItemText

Устанавливает текст элемента.

#include <GuiHeader.au3>
_GUICtrlHeader_SetItemText ( $hWnd, $iIndex, $sText )

Параметры

$hWnd Дескриптор элемента управления.
$iIndex Индекс элемента (нумерация с 0).
$sText Новый текст элемента. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Если нужен обратный вызов уведомления, вы должны указать $sText = -1 (LPSTR_TEXTCALLBACK).

См. также

_GUICtrlHeader_GetItemText

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiHeader.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Header Get/Set Item Text (v" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $hHeader = _GUICtrlHeader_Create($hGUI)
    _MemoCreate(2, 52, 444, 220)

;~ _GUICtrlHeader_SetUnicodeFormat($hHeader, True)

    GUISetState(@SW_SHOW)

    ; Добавить столбцы
    _GUICtrlHeader_AddItem($hHeader, "Column 0", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 1", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 2", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 3", 100)

    ; Установить текст столбца 0
    _GUICtrlHeader_SetItemText($hHeader, 0, "Column X")

    ; Показать текст столбца 0
    _MemoWrite("Column 0 text: " & _GUICtrlHeader_GetItemText($hHeader, 0))

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example

Пример 2

; == Пример 2 : Header получить/установить текст элемента во внешнем процессе

#include "Extras\HelpFileInternals.au3"

#include <GuiHeader.au3>

Example()

Func Example()
    Local $sFromTo
    Local $hWin = _MemoRunAU3OutProcess($sFromTo) ; OK при запуске в другом режиме

    Local $hHeader = _MemoCreateOutProcess($hWin, "SysHeader32", 0, $sFromTo)

    ; Установить текст столбца 1
    _GUICtrlHeader_SetItemText($hHeader, 1, "<<<Column Y")
    _MemoWrite("<<< Column 1 text: " & _GUICtrlHeader_GetItemText($hHeader, 1))

    _MemoMsgBoxStatus("", -1, $hWin) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
