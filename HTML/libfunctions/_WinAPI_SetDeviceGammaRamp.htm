_WinAPI_SetDeviceGammaRamp

Устанавливает гамму на прямых цветовых дисплеях, поддерживающих загружаемые гаммы в оборудовании.

#include <WinAPIGdi.au3>
_WinAPI_SetDeviceGammaRamp ( $hDC, Const ByRef $aRamp )

Параметры

$hDC Дескриптор контекста устройства для доски прямого цветного дисплея в вопросе.
$aRamp 2D массив ([r1, g1, b1], [r2, g2, b2], ... [r256, g256, b256]), содержащий гамму для установки.
Каждый элемент в этом массиве: целочисленное значение в диапазоне от 0 до 65535, которое является отображением между
значениями RGB в буфере кадров и значениями цифро-аналогового преобразователя (DAC). Значения RGB должны быть сохранены
в наиболее значащих битах каждого WORD для повышения независимости DAC.
(подробнее см. MSDN)

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Режимы прямого цветового дисплея не используют таблицы поиска цветов и обычно составляют 16, 24 или 32 бита. Не все доски
прямого цветового видео поддерживают загружаемые гаммы. _WinAPI_SetDeviceGammaRamp() работает только для устройств
с драйверами, поддерживающими загружаемые гаммы в оборудовании.
