Справочник функций

_WinAPI_FindFirstFile

Ищет в каталоге файл или подкаталог с именем, соответствующим заданному шаблону.

#include <WinAPIFiles.au3>
_WinAPI_FindFirstFile ( $sFilePath, $tData )

Параметры

$sFilePath Каталог или путь и имя файла, которое может содержать подстановочные знаки – например, "*" или "?".
Если строка заканчивается подстановочным знаком, точкой "." или именем каталога, у пользователя должны быть права доступа к корню и всем подкаталогам пути.
$tData Структура $tagWIN32_FIND_DATA или указатель на неё, в которую записывается информация о найденном файле или каталоге.

Возвращаемое значение

Успех: Возвращает дескриптор поиска.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Функция открывает дескриптор поиска и возвращает информацию о первом файле, который файловая система находит по указанному шаблону имени.
Этот файл может не совпадать с первым файлом или каталогом, который покажет программа просмотра содержимого каталога для того же шаблона.

После получения дескриптора поиска используйте его для поиска остальных файлов по тому же шаблону через _WinAPI_FindNextFile().

Если подходящих файлов нет, функция завершается с @extended = ERROR_FILE_NOT_FOUND (2).

Когда дескриптор поиска больше не нужен, закройте его через _WinAPI_FindClose().

См. также

_WinAPI_FindClose, _WinAPI_FindNextFile

Пример

#include <Array.au3>
#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <StructureConstants.au3>
#include <WinAPIConv.au3>
#include <WinAPIError.au3>
#include <WinAPIFiles.au3>

Local $aList[101][2] = [[0]]

Local $tData = DllStructCreate($tagWIN32_FIND_DATA)

Local $sFile
Local $hSearch = _WinAPI_FindFirstFile(@ScriptDir & '\*', $tData)
While Not @error
    $sFile = DllStructGetData($tData, 'cFileName')
    Switch $sFile
        Case '.', '..'

        Case Else
            If Not BitAND(DllStructGetData($tData, 'dwFileAttributes'), $FILE_ATTRIBUTE_DIRECTORY) Then
                $aList[0][0] += 1
                If $aList[0][0] > UBound($aList) - 1 Then
                    ReDim $aList[UBound($aList) + 100][2]
                EndIf
                $aList[$aList[0][0]][0] = $sFile
                $aList[$aList[0][0]][1] = _WinAPI_MakeQWord(DllStructGetData($tData, 'nFileSizeLow'), DllStructGetData($tData, 'nFileSizeHigh'))
            EndIf
    EndSwitch
    _WinAPI_FindNextFile($hSearch, $tData)
WEnd

Switch @extended
    Case 18 ; ERROR_NO_MORE_FILES

    Case Else
        Local $iError = @error
        Local $iExtended = @extended
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "Error = " & $iError, _WinAPI_GetErrorMessage($iExtended))
        Exit
EndSwitch

_WinAPI_FindClose($hSearch)

_ArrayDisplay($aList, '_WinAPI_Find...', $aList[0][0])
