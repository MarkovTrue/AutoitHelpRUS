Справочник ключевых слов

$tagGDIPIMAGECODECINFO

Структура сведений о кодеке изображений GDI+.

Global Const $tagGDIPIMAGECODECINFO = "byte CLSID[16];byte FormatID[16];ptr CodecName;ptr DllName;ptr FormatDesc;ptr FileExt;" & _
"ptr MimeType;dword Flags;dword Version;dword SigCount;dword SigSize;ptr SigPattern;ptr SigMask"

Параметры

CLSID Идентификатор кодека (структура GUID).
FormatID Идентификатор формата файла (структура GUID).
CodecName Указатель на строку Unicode с завершением нулевым символом, содержащую имя кодека.
DllName Указатель на строку Unicode с завершением нулевым символом, содержащую имя пути DLL, в которой находится кодек.
Если кодек находится не в DLL, этот указатель равен 0.
FormatDesc Указатель на строку Unicode с завершением нулевым символом, содержащую имя формата файла, используемого кодеком.
FileExt Указатель на строку Unicode с завершением нулевым символом, содержащую все расширения имён файлов, связанные с кодеком.
Расширения разделены точками с запятой.
MimeType Указатель на строку с завершением нулевым символом, содержащую тип MIME кодека.
Flags Комбинация флагов $GDIP_ICF.
Version Указывает версию кодека.
SigCount Указывает количество подписей, используемых форматом файла, связанным с кодеком.
SigSize Указывает количество байтов в каждой подписи.
SigPattern Указатель на массив байтов, содержащий шаблон каждой подписи.
SigMask Указатель на массив байтов, содержащий маску для каждой подписи.
