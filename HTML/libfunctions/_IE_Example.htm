Справочник функций

_IE_Example

Отображает новое окно браузера с предзагруженными документами для использования в примерах IE.au3 или ваших собственных тестах.

#include <IE.au3>
_IE_Example ( [$sModule = "basic"] )

Параметры

$sModule [необязательный] задаёт, какой модуль запустить
"basic" = (по умолчанию) простая HTML-страница с текстом, ссылками и изображениями
"form" = простая HTML-страница с несколькими элементами формы
"frameset" = простая HTML-страница с фреймами
"iframe" = простая HTML-страница с iframe
"table" = простая HTML-страница с таблицами

Возвращаемое значение

Успех: Возвращает объектную переменную, указывающую на объект InternetExplorer.Application.
Ошибка: Устанавливает @error ≠ 0.
@error: 5 ($_IEStatus_InvalidValue) – неверное значение.
@extended: Содержит номер недопустимого параметра.

Примечания

Эта функция отображает окна браузера, заполненные HTML и HTML-элементами, используемыми во всём справочнике и примерах IE.au3.
Их также можно использовать для экспериментов с набором функций IE.au3.

См. также

_IE_Introduction

Пример

; Создание окон браузера с каждой из страниц-примеров.
; Возвращённую объектную переменную можно использовать так же, как
; переменные, возвращаемые функциями _IECreate или _IEAttach

#include <IE.au3>

Local $oIE_basic = _IE_Example("basic")
Local $oIE_form = _IE_Example("form")
Local $oIE_table = _IE_Example("table")
Local $oIE_frameset = _IE_Example("frameset")
Local $oIE_iframe = _IE_Example("iframe")
