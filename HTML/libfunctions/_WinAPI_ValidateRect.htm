_WinAPI_ValidateRect

Удаляет прямоугольник из текущей области обновления указанного окна.

#include <WinAPIGdi.au3>
_WinAPI_ValidateRect ( $hWnd [, $tRECT = 0] )

Параметры

$hWnd Дескриптор окна.
$tRECT [необязательный] Структура $tagRECT, которая содержит клиентские координаты прямоугольника, который должен быть удален из
области обновления. Если этот параметр равен 0 (по умолчанию), вся клиентская область удаляется.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
