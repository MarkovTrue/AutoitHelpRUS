Справочник функций

_WinAPI_SetWindowText

Изменяет текст заголовка указанного окна.

#include <WinAPISysWin.au3>
_WinAPI_SetWindowText ( $hWnd, $sText )

Параметры

$hWnd Дескриптор окна.
$sText Строка, которая станет новым заголовком или текстом элемента управления.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Если целевое окно принадлежит текущему процессу, SetWindowText отправляет сообщение $WM_SETTEXT на указанное окно или элемент управления. Однако если элемент управления – список с стилем $WS_CAPTION, функция устанавливает текст для элемента управления, а не для элементов списка. Для установки текста элемента управления в другом процессе отправьте сообщение $WM_SETTEXT напрямую вместо вызова SetWindowText.
