Справочник функций

_WinAPI_CreateMargins

создаёт структуру $tagMARGINS с указанными границами удержания слева, справа, сверху и снизу.

#include <WinAPIMisc.au3>
_WinAPI_CreateMargins ( $iLeftWidth, $iRightWidth, $iTopHeight, $iBottomHeight )

Параметры

$iLeftWidth Количество / Размер.
$iRightWidth Количество / Размер.
$iTopHeight Количество / Размер.
$iBottomHeight Количество / Размер.

Возвращаемое значение

Возвращает структуру $tagMARGINS, содержащую заданные удерживающие границы.
