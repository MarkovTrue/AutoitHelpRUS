_WinAPI_GetRgnBox

Получает описывающий прямоугольник указанной области.

#include <WinAPIGdi.au3>
_WinAPI_GetRgnBox ( $hRgn, ByRef $tRECT )

Параметры

$hRgn Дескриптор региона.
$tRECT Возвращает структуру $tagRECT, которая создана этой функцией и содержит описывающий прямоугольник в логических единицах.

Возвращаемое значение

Успех: Сложность области. Может принимать одно из значений:
$COMPLEXREGION
$NULLREGION
$SIMPLEREGION
Ошибка: Возвращает 0.
