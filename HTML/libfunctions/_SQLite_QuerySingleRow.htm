Справочник функций

_SQLite_QuerySingleRow

Читает первую строку результата из указанного запроса.

#include <SQLite.au3>
_SQLite_QuerySingleRow ( $hDB, $sSQL, ByRef $aRow )

Параметры

$hDB Открытая база данных; -1 – использовать последнюю открытую базу данных.
$sSQL Выполняемый SQL-запрос.
$aRow Выходной параметр (количество строк данных).

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове _SQLite_Query()
2 – вызов запрещён безопасным режимом
3 – ошибка при вызове _SQLite_FetchData()
4 – ошибка при вызове _SQLite_QueryFinalize()

См. также

_SQLite_Query

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $aRow
_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; открыть :memory: базу данных
_SQLite_Exec(-1, "CREATE TEMP TABLE settings (key,value);")
_SQLite_Exec(-1, "INSERT INTO settings (key,value) VALUES ('1','setting one');")
_SQLite_Exec(-1, "INSERT INTO settings (key,value) VALUES ('2','setting two');")
_SQLite_Exec(-1, "INSERT INTO settings (key,value) VALUES ('3','setting Three');")
_SQLite_Exec(-1, "INSERT INTO settings (key,value) VALUES ('4','setting Four');")
_SQLite_QuerySingleRow(-1, "SELECT value FROM settings WHERE key > '2';", $aRow) ; Выбор одной строки и одного поля!
MsgBox($MB_SYSTEMMODAL, "One of the values for key > 2:", $aRow[0])
_SQLite_Close()
_SQLite_Shutdown()
