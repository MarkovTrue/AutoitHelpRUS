Справочник функций

_WinNet_AddConnection3

Подключает локальное устройство к сетевому ресурсу.

#include <WinNet.au3>
_WinNet_AddConnection3 ( $hWnd, $sLocalName, $sRemoteName [, $sUserName = 0 [, $sPassword = 0 [, $iType = 1 [, $iOptions = 1]]]] )

Параметры

$hWnd Дескриптор окна.
$sLocalName Имя локального устройства для переадресации, например "F:" или "LPT1". Строка обрабатывается независимо от регистра. Если использовано ключевое слово Null или пусто, подключение к сетевому ресурсу происходит без переадресации локального устройства.
$sRemoteName Имя сетевого ресурса для подключения.
$sUsername [необязательный] Имя пользователя для подключения. Если 0, функция использует имя пользователя по умолчанию.
$sPassword [необязательный] Пароль для подключения. Если 0, используется пароль по умолчанию. Если строка пуста, пароль не используется.
$iType [необязательный] Указывает тип сетевого ресурса для подключения:
0 – Любой (только если $sLocalName равен 0)
1 – Диск
2 – Принтер.
$iOptions [необязательный] Параметры подключения. Может быть одним или несколькими из следующих:
1 – Подключение к сетевому ресурсу следует запомнить
2 – Операционная система может взаимодействовать с пользователем в целях аутентификации
4 – Система не использует параметры по умолчанию для имён пользователей или паролей без предоставления пользователю возможности указать альтернативу. Этот флаг игнорируется, если также не установлен бит 2 (интерактивный).
8 – Принудительная переадресация локального устройства при подключении
16 – Операционная система приглашает пользователя на аутентификацию через командную строку вместо графического интерфейса.
    Этот флаг игнорируется, если также не установлен бит 2 (интерактивный).
32 – Если этот бит установлен и операционная система запрашивает учётные данные, они сохраняются диспетчером учётных данных.
    Если диспетчер учётных данных отключен для сеанса входа вызывающей стороны, или если сетевой поставщик не поддерживает сохранение учётных данных, этот флаг игнорируется.
    Этот флаг также игнорируется, если не установлен
бит 5 (командная строка вместо графического интерфейса).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
