Справочник функций

_IEAttach

Подключиться к указанному экземпляру Internet Explorer, где подстрока строки поиска совпадает (на основе выбранного режима).

#include <IE.au3>
_IEAttach ( $sString [, $sMode = "title" [, $iInstance = 1]] )

Параметры

$sString Строка для поиска.
$sMode [необязательный] задаёт режим поиска
"title" = (по умолчанию) подстрока заголовка основного документа
"windowtitle" = подстрока полного заголовка окна (вместо заголовка документа)
"url" = подстрока или URL текущей страницы
"text" = подстрока в тексте из тела текущей страницы
"html" = подстрока в HTML из тела текущей страницы
"hwnd" = hwnd окна браузера
"embedded" = подстрока заголовка или hwnd окна, в котором встроена ячейка управления
"dialogbox" = подстрока заголовка или hwnd модального/немодального диалогового окна
"instance" = $sString игнорируется, возвращается одна ссылка на браузер (по номеру экземпляра) из всех доступных экземпляров браузера
$iInstance [необязательный] индекс (начиная с 1) в группе браузеров или встроенных браузеров, соответствующих $sString и $sMode. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает переменная объекта, указывающая на объект InternetExplorer для всех режимов, кроме Embedded и DialogBox, которые возвращают объект Window.
Ошибка: Устанавливает @error ≠ 0.
@error: 5 ($_IEStatus_InvalidValue) – неверное значение
7 ($_IEStatus_NoMatch) – совпадения не найдены
@extended: Содержит номер недопустимого параметра.

Примечания

_IEAttach() обеспечивает параметр "dialogbox" для подключения к модальным и немодальным диалоговым окнам, созданным браузером.
Важно отметить, что не все диалоговые окна, созданные при взаимодействии с браузером, можно подключать и контролировать таким образом.
Многие из этих диалоговых окон на самом деле являются стандартными окнами и могут быть контролированы через традиционные функции окон AutoIt. Надежный способ отличить эти типы окон – использовать инструмент "AutoIt Window Info" для их проверки – если окно содержит элемент управления под названием "Internet Explorer_Server", то можно подключиться к нему с помощью этой функции, если нет – это стандартное окно и должны использоваться традиционные функции окон AutoIt для его управления.

Окна HyperTextApplication (.hta) могут быть подключены с использованием опции "embedded".

Расширенный синтаксис выбора заголовка окна, доступный для стандартных функций Win*, может быть использован вместо подстроки заголовка для режимов "dialogbox" и "embedded".

Использование $iInstance с режимом "embedded": используется для возврата ссылки на конкретный экземпляр WebBrowser и особенно полезно, когда в одном окне существует несколько.
Если ты передашь заголовок окна в $sString в режиме embedded, будет использовано только первое окно с соответствующим заголовком.
Если элемент управления WebBrowser, который требуется, находится в другом окне, необходимо передать HWnd этого окна вместо заголовка или использовать расширенный синтаксис выбора заголовка окна, доступный для стандартных функций Win*.

Использование $iInstance со всеми режимами, кроме "embedded": используется для возврата ссылки на браузер из группы всех окон, которые соответствуют критериям из $sString и $sMode.
Порядок экземпляров для режима "dialogbox" определяется порядком, возвращаемым WinList() с соответствующим заголовком.
Для всех остальных режимов порядок экземпляров определяется коллекцией Shell.Windows.

Значения $iInstance > 1 устанавливаются равными 1 и выводится предупреждающее сообщение при использовании режима "hwnd" или режима "dialogbox" при передаче HWnd в $sString.

Режимы "dialogbox" и "embedded" можно использовать для подключения к стандартным окнам браузера, но возвращаемый объект – это верхнее окно в браузере, а не объект InternetExplorer.
Объекты Window не предоставляют доступ ко всем атрибутам объекта InternetExplorer (например, текст состояния, адресная строка и т. д.).
Кроме того, если попытаться использовать функцию типа _IENavigate() на таком объекте, могут возникнуть ошибки COM из-за способа реализации вкладок в IE7.
Может быть полезно находить экземпляры браузера таким способом, но рекомендуется немедленно использовать _IEAttach() с одним из других режимов и с использованием информации, полученной из объекта Window, чтобы получить ссылку на связанный объект InternetExplorer.

См. также

_IECreate, _IECreateEmbedded, _IEQuit

Пример

Пример 1

; Подключиться к браузеру с "AutoIt" в заголовке, отобразить URL

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("www.autoitscript.com")
Local $oIE1 = _IECreate("www.google.com")

$oIE = _IEAttach("AutoIt")
MsgBox($MB_SYSTEMMODAL, "The URL", _IEPropertyGet($oIE, "locationurl"))

_IEQuit($oIE)
_IEQuit($oIE1)

Пример 2

; Подключение к браузеру с "простой HTML-страницей"
; в тексте его документа верхнего уровня

#include <IE.au3>

Local $oIE_basic = _IE_Example("basic")

Local $oIE = _IEAttach("simple HTML page", "text")
ConsoleWrite('@@ Debug(' & @ScriptLineNumber & ') : $oIE = ' & $oIE & @CRLF & '>Error code: ' & @error & ' Extended code: 0x' & Hex(@extended) & @CRLF) ;### Отладочная консоль

Пример 3

; Подключиться к элементу управления браузера, встроенному в другое окно

#include <IE.au3>

Local $oIE = _IEAttach("A Window Title", "embedded")
ConsoleWrite('@@ Debug(' & @ScriptLineNumber & ') : $oIE = ' & $oIE & @CRLF & '>Error code: ' & @error & ' Extended code: 0x' & Hex(@extended) & @CRLF) ;### Отладочная консоль

Пример 4

; Подключиться к 3-му элементу управления браузера, встроенному в другое окно
; Используй расширенный синтаксис заголовка окна для 2-го окна
; со строкой 'ICQ' в заголовке

#include <IE.au3>

Local $oIE = _IEAttach("[REGEXPTITLE:ICQ; INSTANCE:2]", "embedded", 3)

Пример 5

; Создать массив ссылок на объекты для всех текущих экземпляров браузера
; Первый элемент массива будет содержать количество найденных экземпляров

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $aIE[1]
$aIE[0] = 0

Local $i = 1, $oIE
While 1
    $oIE = _IEAttach("", "instance", $i)
    If @error = $_IEStatus_NoMatch Then ExitLoop
    ReDim $aIE[$i + 1]
    $aIE[$i] = $oIE
    $aIE[0] = $i
    $i += 1
WEnd

MsgBox($MB_SYSTEMMODAL, "Browsers Found", "Number of browser instances in the array: " & $aIE[0])
