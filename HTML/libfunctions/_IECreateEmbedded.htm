Справочник функций

_IECreateEmbedded

Создаёт объект веб-браузера, пригодный для встраивания в GUI AutoIt через GUICtrlCreateObj().

#include <IE.au3>
_IECreateEmbedded ( )

Возвращаемое значение

Успех: Возвращает a Webбраузер object reference.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) – общая ошибка.

Примечания

До встраивания этого объекта в родительское приложение над ним нельзя выполнять никаких действий (например, нельзя вызвать _IENavigate()).
Из-за этого ограничения браузер не переходит автоматически на 'about:blank', как это делает браузер, созданный _IECreate().
Поэтому нужно использовать _IENavigate(), чтобы перевести браузер на 'about:blank' после встраивания в родительское приложение и до выполнения любых операций, требующих загруженного документа (например, _IEBodyWriteHTML()
).

Несколько свойств, относящихся к объекту InternetExplorer (например, возвращаемому _IECreate()), к данному объекту неприменимы.
К ним относятся текст статуса, адресная строка и другие, существующие для браузера, но отсутствующие для встраиваемого контрола.

Возможны проблемы с видимостью объекта, если не использовать стиль Windows $WS_CLIPCHILDREN в GUICreate().

Функцию _IEQuit() нельзя использовать с этим объектом. Объект будет уничтожен при уничтожении его родительского GUI.

См. также

_IEAttach, _IECreate, _IENavigate

Пример

; Перехватывать COM-ошибки, чтобы "Назад" и "Вперёд"
; за пределами истории не прерывали скрипт
; (COM-ошибки будут уходить в консоль)

#include <GUIConstantsEx.au3>
#include <IE.au3>
#include <WindowsStylesConstants.au3>

Local $oIE = _IECreateEmbedded()
GUICreate("Embedded Web control Test", 640, 580, _
        (@DesktopWidth - 640) / 2, (@DesktopHeight - 580) / 2, _
        $WS_OVERLAPPEDWINDOW + $WS_CLIPSIBLINGS + $WS_CLIPCHILDREN)
GUICtrlCreateObj($oIE, 10, 40, 600, 360)
Local $idButton_Back = GUICtrlCreateButton("Back", 10, 420, 100, 30)
Local $idButton_Forward = GUICtrlCreateButton("Forward", 120, 420, 100, 30)
Local $idButton_Home = GUICtrlCreateButton("Home", 230, 420, 100, 30)
Local $idButton_Stop = GUICtrlCreateButton("Stop", 340, 420, 100, 30)

Global $g_idLbl_Error_Message = GUICtrlCreateLabel("", 100, 500, 500, 30)
GUICtrlSetColor(-1, 0xff0000)

GUISetState(@SW_SHOW) ;Показать GUI

_IENavigate($oIE, "http://www.autoitscript.com")
_IEAction($oIE, "stop")

; Ожидание, пока пользователь закроет окно
While 1
    Local $iMsg = GUIGetMsg()
    Select
        Case $iMsg = $GUI_EVENT_CLOSE
            ExitLoop
        Case $iMsg = $idButton_Home
            _IENavigate($oIE, "http://www.autoitscript.com")
            _IEAction($oIE, "stop")
            _IEAction($oIE, "back")
            CheckError("Home", @error, @extended)
        Case $iMsg = $idButton_Back
            _IEAction($oIE, "back")
            CheckError("Back", @error, @extended)
        Case $iMsg = $idButton_Forward
            _IEAction($oIE, "forward")
            CheckError("Forward", @error, @extended)
        Case $iMsg = $idButton_Stop
            _IEAction($oIE, "stop")
            CheckError("Stop", @error, @extended)
    EndSelect
WEnd

GUIDelete()

Exit

Func CheckError($sMsg, $iError, $iExtended)
    If $iError Then
        $sMsg = "Error using " & $sMsg & " button (" & $iExtended & ")"
    Else
        $sMsg = ""
    EndIf
    GUICtrlSetData($g_idLbl_Error_Message, $sMsg)
EndFunc   ;==>CheckError
