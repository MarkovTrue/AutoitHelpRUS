Справочник функций

_WinAPI_ShellFileOperation

Копирует, перемещает, переименовывает или удаляет объект файловой системы.

#include <WinAPIShellEx.au3>
_WinAPI_ShellFileOperation ( $sFrom, $sTo, $iFunc, $iFlags [, $sTitle = '' [, $hParent = 0]] )

Параметры

$sFrom Одна строка или массив строк, содержащих имя(имена) исходного файла. Эти имена должны быть
полностью определёнными путями для предотвращения неожиданных результатов.
$sTo Одна строка или массив строк, содержащих имя(имена) файла или каталога назначения (если используется).
Эти имена должны быть полностью определёнными путями для предотвращения неожиданных результатов.
$iFunc Значение, указывающее, какую операцию выполнить. Этот параметр может быть одним из следующих значений.
$FO_COPY
$FO_DELETE
$FO_MOVE
$FO_RENAME
$iFlags Флаги, управляющие операцией файла. Этот параметр может быть одним из следующих значений.
$FOF_ALLOWUNDO
$FOF_CONFIRMMOUSE
$FOF_FILESONLY
$FOF_MULTIDESTFILES
$FOF_NOCONFIRMATION
$FOF_NOCONFIRMMKDIR
$FOF_NO_CONNECTED_ELEMENTS
$FOF_NOCOPYSECURITYATTRIBS
$FOF_NOERRORUI
$FOF_NORECURSEREPARSE
$FOF_NORECURSION
$FOF_RENAMEONCOLLISION
$FOF_SILENT
$FOF_SIMPLEPROGRESS
$FOF_WANTMAPPINGHANDLE
$FOF_WANTNUKEWARNING
$FOF_NO_UI
$sTitle [необязательный] Заголовок диалога выполнения. Этот параметр используется только если $iFlags включает флаг $FOF_SIMPLEPROGRESS.
$hParent [необязательный] Дескриптор диалога для отображения информации о статусе операции файла.

Возвращаемое значение

Успех: Возвращает структура $tagSHFILEOPSTRUCT, содержащая дополнительную информацию.
Ошибка: Устанавливает флаг @error, @extended может быть установлен на одно из следующих значений.
113 – исходный и целевой файлы – это один и тот же файл.
114 – несколько путей файлов указаны в буфере источника, но только один путь целевого файла.
115 – операция переименования была указана, но путь назначения является другим каталогом. Используйте операцию перемещения вместо этого.
116 – источник является корневым каталогом, который не может быть перемещен или переименован.
117 – операция была отменена пользователем или молча отменена если были предоставлены соответствующие флаги для _WinAPI_ShellFileOperation().
118 – назначение является поддеревом источника.
120 – параметры безопасности запретили доступ к источнику.
121 – путь источника или назначения превышен или превысит MAX_PATH.
122 – операция включала несколько путей назначения, что может привести к ошибке в случае операции перемещения.
124 – путь в источнике или назначении или оба были недействительны.
125 – источник и назначение имеют одну и ту же родительскую папку.
126 – путь назначения является существующим файлом.
128 – путь назначения является существующей папкой.
129 – имя файла превышает MAX_PATH.
130 – назначение – это только для чтения CD-ROM, возможно не отформатирован.
131 – назначение – это только для чтения DVD, возможно не отформатирован.
132 – назначение – это доступный для записи CD-ROM, возможно не отформатирован.
133 – файл, участвующий в операции, слишком велик для целевого носителя или файловой системы.
134 – источник – это только для чтения CD-ROM, возможно не отформатирован.
135 – источник – это только для чтения DVD, возможно не отформатирован.
136 – источник – это доступный для записи CD-ROM, возможно не отформатирован.
183 – MAX_PATH была превышена во время операции.
026 – произошла неизвестная ошибка. Обычно это происходит из-за неправильного пути в источнике или назначении.
536 – неуточненная ошибка произошла на назначении.
652 – назначение является корневым каталогом и не может быть переименовано.

Примечания

Вы должны использовать полностью определённые имена путей с этой функцией. Использование её с относительными именами путей не является потокобезопасным.

Пример

#include <WinAPIFiles.au3>
#include <WinAPIShellEx.au3>

; Создать временный файл
Local $sTemp = _WinAPI_GetTempFileName(@TempDir)

ShellExecute('::{645FF040-5081-101B-9F08-00AA002F954E}')

; Удалить файл в Корзину
If FileExists($sTemp) Then
    _WinAPI_ShellFileOperation($sTemp, '', $FO_DELETE, BitOR($FOF_ALLOWUNDO, $FOF_NO_UI))
EndIf
