Справочник функций

_IETableWriteToArray

Считывает содержимое таблицы в массив.

#include <IE.au3>
_IETableWriteToArray ( ByRef $oObject [, $bTranspose = False] )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$bTranspose [необязательный] Boolean-значение, задающее, менять ли местами строки и столбцы в выходном массиве

Возвращаемое значение

Успех: Возвращает двумерный массив с содержимым таблицы.
Ошибка: Устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
@extended: Содержит номер недопустимого параметра.

Примечания

Если ячейки таблицы охватывают несколько столбцов или строк, для правильного выравнивания результата добавляются пустые элементы массива.
Данные охватывающих ячеек попадают в самые левые или самые верхние элементы массива.

В HTML-документах таблицы часто бывают вложенными.
Если все данные неожиданно возвращаются в одном элементе массива, возможно, в функцию нужно передать более глубоко вложенную таблицу.

См. также

_IETableGetCollection

Пример

Пример 1

; Открытие браузера с примером таблицы, получение ссылки на вторую таблицу
; на странице (индекс 1) и считать её содержимое в двумерный массив

#include <Array.au3>
#include <IE.au3>

Local $oIE = _IE_Example("table")
Local $oTable = _IETableGetCollection($oIE, 1)
Local $aTableData = _IETableWriteToArray($oTable)

_ArrayDisplay($aTableData)

_IEQuit($oIE)

Пример 2

; То же, что в примере 1, но транспонировать выходной массив и показать
; the results with _ArrayDisplay()

#include <Array.au3>
#include <IE.au3>

Local $oIE = _IE_Example("table")
Local $oTable = _IETableGetCollection($oIE, 1)
Local $aTableData = _IETableWriteToArray($oTable, True)
_ArrayDisplay($aTableData)

_IEQuit($oIE)
