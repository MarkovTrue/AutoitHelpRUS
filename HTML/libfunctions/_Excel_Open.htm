Справочник функций

_Excel_Open

Подключается к существующему экземпляру Excel или создаёт новый.

#include <Excel.au3>
_Excel_Open ( [$bVisible = True [, $bDisplayAlerts = False [, $bScreenUpdating = True [, $bInteractive = True [, $bForceNew = False]]]]] )

Параметры

$bVisible [необязательный] True делает приложение видимым (по умолчанию True).
$bDisplayAlerts [необязательный] False подавляет все запросы и сообщения об ошибках при открытии книги (по умолчанию False).
$bScreenUpdating [необязательный] False отключает обновление экрана для ускорения скрипта (по умолчанию True).
$bInteractive [необязательный] При False Excel блокирует весь ввод с клавиатуры и мыши (кроме диалоговых окон) (по умолчанию True).
$bForceNew [необязательный] True принудительно создаёт новый экземпляр Excel, даже если уже запущен (по умолчанию False).

Возвращаемое значение

Успех: Возвращает объект приложения Excel.
Устанавливает @extended в:
0 – Excel уже был запущен
1 – Excel не был запущен или $bForceNew = True. Создан новый экземпляр Excel.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – ошибка, возвращённая ObjCreate. @extended содержит COM-код ошибки.

Примечания

Если $bDisplayAlerts = False и сообщение требует ответа, Excel выбирает ответ по умолчанию.

Для повышения производительности задайте $bScreenUpdating = False. Экран не будет обновляться, пока вы не установите True ($oExcel.ScreenUpdating = True).

Блокировка ввода ($bInteractive = False) препятствует вмешательству пользователя в скрипт AutoIt.
При установке False не забудьте задать True ($oExcel.Interactive = True)

Пример

Пример 1

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создаём объект приложения или подключаемся к уже запущенному экземпляру Excel.
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Open Example 1", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Open Example 1", "Excel Application has been opened successfully." & @CRLF & @CRLF & "Will _Excel_Close close the application?: " & @extended)

Пример 2

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать новый экземпляр приложения Excel и отображать оповещения
_Excel_Open(Default, Default, Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Open Example 2", "Error creating a new Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $aProcesses = ProcessList("Excel.exe")
If $aProcesses[0][0] = 1 Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Open Example 2", "Excel Application has been opened successfully." & @CRLF & @CRLF & $aProcesses[0][0] & " Excel instance is running.")
Else
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Open Example 2", "Excel Application has been opened successfully." & @CRLF & @CRLF & $aProcesses[0][0] & " Excel instances are running.")
EndIf
