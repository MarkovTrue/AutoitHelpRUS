Справочник функций

_GUICtrlComboBox_SetHorizontalExtent

Устанавливает ширину в пикселях, на которую список может прокручиваться по горизонтали.

#include <GuiComboBox.au3>
_GUICtrlComboBox_SetHorizontalExtent ( $hWnd, $iWidth )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iWidth Задаёт прокручиваемую ширину поля списка в пикселях.

Возвращаемое значение

Отсутствует.

Примечания

Если ширина элемента управления ListBox меньше $iWidth, горизонтальная полоса прокрутки позволяет горизонтально прокручивать элементы в поле списка.

Если ширина элемента управления ListBox больше или равна $iWidth, горизонтальная полоса прокрутки скрывается или, если элемент управления ComboBox имеет стиль $CBS_DISABLENOSCROLL, отключается.

См. также

_GUICtrlComboBox_GetHorizontalExtent

Пример

#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("ComboBox Get/Set HorizontalExtent (v" & @AutoItVersion & ")", 420, 296)
    Local $idCombo = GUICtrlCreateCombo("", 2, 2, 416, 296, BitOR($CBS_SIMPLE, $CBS_DISABLENOSCROLL, $WS_HSCROLL))
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBox_BeginUpdate($idCombo)
    _GUICtrlComboBox_AddDir($idCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($idCombo)

    ; Получить горизонтальный размер
    MsgBox($MB_SYSTEMMODAL, "Information", "Horizontal Extent: " & _GUICtrlComboBox_GetHorizontalExtent($idCombo))

    ; Установить горизонтальный размер
    _GUICtrlComboBox_SetHorizontalExtent($idCombo, 500)

    ; Получить горизонтальный размер
    MsgBox($MB_SYSTEMMODAL, "Information", "Horizontal Extent: " & _GUICtrlComboBox_GetHorizontalExtent($idCombo))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
