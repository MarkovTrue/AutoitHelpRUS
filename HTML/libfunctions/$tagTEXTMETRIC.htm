Справочник ключевых слов

$tagTEXTMETRIC

Содержит основные сведения о физическом шрифте.

Global Const $tagTEXTMETRIC = "long tmHeight;long tmAscent;long tmDescent;long tmInternalLeading;long tmExternalLeading;" & _
"long tmAveCharWidth;long tmMaxCharWidth;long tmWeight;long tmOverhang;long tmDigitizedAspectX;long tmDigitizedAspectY;" & _
"wchar tmFirstChar;wchar tmLastChar;wchar tmDefaultChar;wchar tmBreakChar;byte tmItalic;byte tmUnderlined;byte tmStruckOut;" & _
"byte tmPitchAndFamily;byte tmCharSet"

Параметры

tmHeight Высота символов (подъём + спуск) в пикселах.
tmAscent Подъём символов (расстояние выше базовой линии) в пикселах.
tmDescent Спуск символов (расстояние ниже базовой линии) в пикселах.
tmInternalLeading Внутренний межстрочный интервал (пространство) внутри границ, определённых членом tmHeight.
В этой области могут находиться диакритические знаки и другие символы с диакритикой. Разработчик может установить значение нулём.
tmExternalLeading Дополнительный межстрочный интервал (пространство), добавляемый приложением между строками.
Поскольку эта область находится вне шрифта, она не содержит знаков и не изменяется вызовами вывода текста в режиме OPAQUE или TRANSPARENT.
Разработчик может установить значение нулём.
tmAveCharWidth Средняя ширина символов в шрифте (обычно определяется как ширина буквы x).
Это значение не включает выступ, необходимый для полужирных и курсивных символов.
tmMaxCharWidth Ширина самого широкого символа в шрифте.
tmWeight Толщина (вес) шрифта.
Значение, определённое приложением, связанное с кнопкой.
tmOverhang Дополнительная ширина на строку, которая может быть добавлена к некоторым синтезированным шрифтам.
При синтезировании некоторых атрибутов, таких как полужирный или курсив, графический интерфейс (GDI) или устройство может добавлять ширину к строке как для отдельных символов, так и для строки в целом.
Например, GDI делает строку полужирной, увеличивая промежуток между символами и перехватывая со смещением; курсив реализуется сдвигом строки. В обоих случаях есть выступ за пределы исходной строки.
Для полужирных строк выступ – это расстояние, на которое сдвинут перехват. Для курсивных строк выступ – это величина сдвига верхней части шрифта относительно нижней.
Член tmOverhang позволяет приложению определить, какая часть ширины символа, возвращаемой функцией GetTextExtentPoint32 для отдельного символа, является фактической шириной символа, а какая – дополнительной шириной строки.
Фактическая ширина равна размеру минус выступ.
tmDigitizedAspectX Горизонтальный коэффициент растра устройства, для которого был разработан шрифт.
tmDigitizedAspectY Вертикальный коэффициент растра устройства, для которого был разработан шрифт.
Отношение членов tmDigitizedAspectX и tmDigitizedAspectY – это коэффициент формата устройства, для которого был разработан шрифт.
tmFirstChar Код первого символа, определённого в шрифте.
tmLastChar Код последнего символа, определённого в шрифте.
tmDefaultChar Код символа-заменителя для символов, отсутствующих в шрифте.
tmBreakChar Код символа, используемый для определения границ слов при выравнивании текста.
tmItalic Ненулевое значение указывает курсивный шрифт.
tmUnderlined Ненулевое значение указывает подчёркнутый шрифт.
tmStruckOut Ненулевое значение указывает перечёркнутый шрифт.
tmPitchAndFamily Информация о шаге (пропорциональности), технологии и семействе шрифта.
Четыре младших бита этого члена содержат информацию о шаге и технологии шрифта. Для каждого из четырёх битов определена константа.
$TMPF_FIXED_PITCH Если бит установлен, то шрифт переменного шага. Если бит очищен, то шрифт фиксированного шага. Учтите, что значение противоположно названию константы.
$TMPF_VECTOR Если бит установлен, то шрифт векторный.
$TMPF_TRUETYPE Если бит установлен, то шрифт TrueType.
$TMPF_DEVICE Если бит установлен, то шрифт является шрифтом устройства.
tmCharSet Кодировка символов шрифта. Может быть одним из следующих значений.
ANSI_CHARSET
BALTIC_CHARSET
CHINESEBIG5_CHARSET
DEFAULT_CHARSET
EASTEUROPE_CHARSET
GB2312_CHARSET
GREEK_CHARSET
HANGUL_CHARSET
MAC_CHARSET
OEM_CHARSET
RUSSIAN_CHARSET
SHIFTJIS_CHARSET
SYMBOL_CHARSET
TURKISH_CHARSET
VIETNAMESE_CHARSET

Примечания

Все размеры указаны в логических единицах, то есть зависят от текущего режима отображения контекста дисплея.
