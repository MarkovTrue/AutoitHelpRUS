Справочник ключевых слов

$tagTEXTMETRIC

Содержит основные сведения о физическом шрифте.

Global Const $tagTEXTMETRIC = "long tmHeight;long tmAscent;long tmDescent;long tmInternalLeading;long tmExternalLeading;" & _
"long tmAveCharWidth;long tmMaxCharWidth;long tmWeight;long tmOverhang;long tmDigitizedAspectX;long tmDigitizedAspectY;" & _
"wchar tmFirstChar;wchar tmLastChar;wchar tmDefaultChar;wchar tmBreakChar;byte tmItalic;byte tmUnderlined;byte tmStruckOut;" & _
"byte tmPitchAndFamily;byte tmCharSet"

Параметры

tmHeight Высота символов (надстрочная часть + подстрочная часть).
tmAscent Надстрочная часть символов (единицы над базовой линией).
tmDescent Подстрочная часть символов (единицы под базовой линией).
tmInternalLeading Величина межстрочного интервала (пространства) внутри границ, заданных полем tmHeight.
В этой области могут располагаться диакритические знаки. Разработчик шрифта может задать это поле равным нулю.
tmExternalLeading Величина дополнительного межстрочного интервала (пространства), который приложение добавляет между строками.
Поскольку эта область находится за пределами шрифта, в ней нет знаков, и она не изменяется при выводе текста ни в режиме OPAQUE, ни в TRANSPARENT.
Разработчик шрифта может задать это поле равным нулю.
tmAveCharWidth Средняя ширина символов шрифта (обычно определяется как ширина буквы x).
Это значение не включает выступ, необходимый для полужирных или курсивных символов.
tmMaxCharWidth Ширина самого широкого символа шрифта.
tmWeight Насыщенность (вес) шрифта.
Значение, определяемое приложением и связанное с элементом.
tmOverhang Дополнительная ширина строки, которая может добавляться к некоторым синтезированным шрифтам.
При синтезе некоторых атрибутов, например полужирного или курсива, интерфейс графических устройств (GDI) или устройство может добавлять ширину к строке как для каждого символа, так и для строки в целом.
Например, GDI делает строку полужирной, увеличивая интервал каждого символа и накладывая его со смещением, а курсив создаёт сдвигом строки. В обоих случаях за пределами базовой строки появляется выступ.
Для полужирных строк выступ равен величине смещения наложения. Для курсивных строк выступ равен величине, на которую верх шрифта сдвинут относительно его низа.
Поле tmOverhang позволяет приложению определить, какая часть ширины символа, возвращаемой вызовом GetTextExtentPoint32 для одного символа, является фактической шириной символа, а какая дополнительной шириной строки.
Фактическая ширина равна общей ширине минус выступ.
tmDigitizedAspectX Горизонтальное соотношение устройства, для которого был разработан шрифт.
tmDigitizedAspectY Вертикальное соотношение устройства, для которого был разработан шрифт.
Отношение полей tmDigitizedAspectX и tmDigitizedAspectY задаёт соотношение сторон устройства, для которого был разработан шрифт.
tmFirstChar Код первого символа, определённого в шрифте.
tmLastChar Код последнего символа, определённого в шрифте.
tmDefaultChar Код символа, которым заменяются символы, отсутствующие в шрифте.
tmBreakChar Код символа, по которому определяются разрывы слов при выравнивании текста по ширине.
tmItalic Если значение не ноль, шрифт курсивный.
tmUnderlined Если значение не ноль, шрифт подчёркнутый.
tmStruckOut Если значение не ноль, шрифт зачёркнутый.
tmPitchAndFamily Сведения о ширине (шаге), технологии и семействе физического шрифта.
Четыре младших бита этого поля задают сведения о шаге и технологии шрифта. Для каждого из четырёх битов определена константа.
$TMPF_FIXED_PITCH если этот бит установлен, шрифт с переменной шириной символов. Если бит сброшен, шрифт с фиксированной шириной. Обратите особое внимание: смысл противоположен тому, что подсказывает имя константы.
$TMPF_VECTOR если этот бит установлен, шрифт векторный.
$TMPF_TRUETYPE если этот бит установлен, шрифт TrueType.
$TMPF_DEVICE если этот бит установлен, шрифт устройства.
tmCharSet Набор символов шрифта. Может быть одним из следующих значений.
ANSI_CHARSET
BALTIC_CHARSET
CHINESEBIG5_CHARSET
DEFAULT_CHARSET
EASTEUROPE_CHARSET
GB2312_CHARSET
GREEK_CHARSET
HANGUL_CHARSET
MAC_CHARSET
OEM_CHARSET
RUSSIAN_CHARSET
SHIFTJIS_CHARSET
SYMBOL_CHARSET
TURKISH_CHARSET
VIETNAMESE_CHARSET

Примечания

Все размеры задаются в логических единицах, то есть зависят от текущего режима отображения контекста устройства.
