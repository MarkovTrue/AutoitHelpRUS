_WinAPI_SetWorldTransform

Задаёт двумерное линейное преобразование между мировыми координатами и координатами страницы для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetWorldTransform ( $hDC, $tXFORM )

Параметры

$hDC Дескриптор контекста устройства.
$tXFORM Структура $tagXFORM с данными преобразования.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция _WinAPI_SetWorldTransform() сработает только если для контекста устройства предварительно установлен графический режим
$GM_ADVANCED через _WinAPI_SetGraphicsMode(). Аналогично, нельзя вернуть
режим обратно в $GM_COMPATIBLE по умолчанию, пока мировое преобразование не сброшено в тождественное
повторным вызовом _WinAPI_SetWorldTransform().

Для любых мировых координат (x, y) преобразованные координаты страницы (x', y') считаются по формуле:
    x' = x * eM11 + y * eM21 + eDx
    y' = x * eM12 + y * eM22 + eDy
