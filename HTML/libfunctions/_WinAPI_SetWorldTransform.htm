_WinAPI_SetWorldTransform

Задаёт двумерное линейное преобразование между мировыми координатами и координатами страницы для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetWorldTransform ( $hDC, $tXFORM )

Параметры

$hDC Дескриптор контекста устройства.
$tXFORM Структура $tagXFORM, которая задаёт мировую трансформацию.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция _WinAPI_SetWorldTransform() завершится неудачей, если графический режим для данного контекста устройства не был установлен на $GM_ADVANCED путём предварительного вызова _WinAPI_SetGraphicsMode(). Аналогичным образом, не будет возможности сбросить графический режим контекста устройства в режим по умолчанию $GM_COMPATIBLE, если мировое преобразование предварительно не было сброшено на преобразование идентичности по умолчанию путём вызова _WinAPI_SetWorldTransform().

Для любых координат (x, y) в мировом пространстве преобразованные координаты в пространстве страницы (x', y') можно определить по следующему алгоритму:
    x' = x * eM11 + y * eM21 + eDx
    y' = x * eM12 + y * eM22 + eDy
