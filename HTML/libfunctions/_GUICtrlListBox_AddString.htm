_GUICtrlListBox_AddString

Добавляет строку.

#include <GuiListBox.au3>
_GUICtrlListBox_AddString ( $hWnd, $sText )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$sText Строка для добавления.

Возвращаемое значение

Успех: Возвращает индекс (начиная с 0) добавленной строки.
Ошибка: Возвращает -1.

Примечания

Если для новой строки не хватает памяти, возвращается $LB_ERRSPACE.

Если список не имеет стиля $LBS_SORT, новая строка добавляется в конец. Иначе строка вставляется в позицию, соответствующую сортировке.

См. также

_GUICtrlListBox_AddFile, _GUICtrlListBox_DeleteString, _GUICtrlListBox_InitStorage, _GUICtrlListBox_InsertString

Пример

#include <GUIConstantsEx.au3>
#include <GuiListBox.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $idListBox

    ; Создать GUI
    GUICreate("List Box Add String", 400, 296)
    $idListBox = GUICtrlCreateList("", 2, 2, 396, 296, BitOR($WS_BORDER, $WS_VSCROLL, $LBS_NOTIFY, $LBS_DISABLENOSCROLL, $WS_HSCROLL))

    GUISetState(@SW_SHOW)

    ; Добавить строки
    _GUICtrlListBox_BeginUpdate($idListBox)
    For $iI = 1 To 9
        _GUICtrlListBox_AddString($idListBox, StringFormat("%03d : Random string", Random(1, 100, 1)))
    Next
    _GUICtrlListBox_InsertString($idListBox, "Exact teXt", 3)
    _GUICtrlListBox_InsertString($idListBox, "This is the string that we're looking for", 6)
    _GUICtrlListBox_InsertString($idListBox, _
            "Let's add one really long line of text so that we can set the horizontal scroll bar and " & _
            "show that, unless we dynamically update the scroll bar, it won't show the full line.", 9)
    _GUICtrlListBox_UpdateHScroll($idListBox)
    _GUICtrlListBox_EndUpdate($idListBox)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
