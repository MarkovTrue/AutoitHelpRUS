Справочник функций

_WinAPI_GetBoundsRect

Получает текущий накопленный ограничивающий прямоугольник для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_GetBoundsRect ( $hDC [, $iFlags = 0] )

Параметры

$hDC Дескриптор контекста устройства.
$iFlags [необязательный] Флаги, определяющие поведение функции. Может быть следующим значением:
$DCB_RESET

Возвращаемое значение

Успех: Возвращает структуру $tagRECT, содержащую текущий ограничивающий прямоугольник. Флаг @extended будет содержать значение, определяющее состояние накопленного ограничивающего прямоугольника ($DCB_*).
Ошибка: Устанавливает @error ≠ 0.
