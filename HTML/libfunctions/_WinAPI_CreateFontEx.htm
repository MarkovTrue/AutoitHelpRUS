Справочник функций

_WinAPI_CreateFontEx

создаёт логический шрифт с указанными характеристиками.

#include <WinAPIGdi.au3>
_WinAPI_CreateFontEx ( $iHeight [, $iWidth = 0 [, $iEscapement = 0 [, $iOrientation = 0 [, $iWeight = $FW_NORMAL [, $bItalic = False [, $bUnderline = False [, $bStrikeOut = False [, $iCharSet = $DEFAULT_CHARSET [, $iOutPrecision = $OUT_DEFAULT_PRECIS [, $iClipPrecision = $CLIP_DEFAULT_PRECIS [, $iQuality = $DEFAULT_QUALITY [, $iPitchAndFamily = $DEFAULT_PITCH [, $sFaceName = '' [, $iStyle = 0]]]]]]]]]]]]]] )

Параметры

$iHeight Высота знакоместа или символа шрифта в логических единицах.
$iWidth Необязательный параметр.
$iEscapement Необязательный параметр.
$iOrientation Необязательный параметр.
$iWeight [необязательный] Толщина шрифта в диапазоне от 0 до 1000 или одно из следующих значений.
$FW_DONTCARE
$FW_THIN
$FW_EXTRALIGHT
$FW_ULTRALIGHT
$FW_LIGHT
$FW_NORMAL (по умолчанию)
$FW_REGULAR
$FW_MEDIUM
$FW_SEMIBOLD
$FW_DEMIBOLD
$FW_BOLD
$FW_EXTRABOLD
$FW_ULTRABOLD
$FW_HEAVY
$FW_BLACK
$bItalic Значение.
$bUnderline Значение.
$bStrikeOut Значение.
$iCharSet [необязательный] Набор символов. Может быть одним из следующих значений.
$ANSI_CHARSET
$BALTIC_CHARSET
$CHINESEBIG5_CHARSET
$DEFAULT_CHARSET (по умолчанию)
$EASTEUROPE_CHARSET
$GB2312_CHARSET
$GREEK_CHARSET
$HANGEUL_CHARSET
$MAC_CHARSET
$OEM_CHARSET
$RUSSIAN_CHARSET
$SHIFTJIS_CHARSET
$SYMBOL_CHARSET
$TURKISH_CHARSET
$VIETNAMESE_CHARSET
Корейская языковая версия Windows:
$JOHAB_CHARSET
Ближневосточная языковая версия Windows:
$ARABIC_CHARSET
$HEBREW_CHARSET
Тайская языковая версия Windows:
$THAI_CHARSET
$iOutPrecision [необязательный] Точность вывода. Может быть одним из следующих значений.
$OUT_CHARACTER_PRECIS
$OUT_DEFAULT_PRECIS (по умолчанию)
$OUT_DEVICE_PRECIS
$OUT_OUTLINE_PRECIS
$OUT_PS_ONLY_PRECIS
$OUT_RASTER_PRECIS
$OUT_STRING_PRECIS
$OUT_STROKE_PRECIS
$OUT_TT_ONLY_PRECIS
$OUT_TT_PRECIS
$iClipPrecision [необязательный] Точность отсечения. Может быть одним или несколькими из следующих значений.
$CLIP_CHARACTER_PRECIS
$CLIP_DEFAULT_PRECIS (по умолчанию)
$CLIP_DFA_DISABLE
$CLIP_EMBEDDED
$CLIP_LH_ANGLES
$CLIP_DFA_OVERRIDE
$CLIP_STROKE_PRECIS
$iQuality [необязательный] Качество вывода. Может быть одним из следующих значений.
$ANTIALIASED_QUALITY
$CLEARTYPE_QUALITY
$DEFAULT_QUALITY (по умолчанию)
$DRAFT_QUALITY
$NONANTIALIASED_QUALITY
$PROOF_QUALITY
$iPitchAndFamily [необязательный] Шаг и семейство шрифта. Два младших бита задают шаг шрифта и могут быть одним из следующих значений.
$DEFAULT_PITCH (по умолчанию)
$FIXED_PITCH
$VARIABLE_PITCH
Четыре старших бита задают семейство шрифта и могут быть одним из следующих значений.
$FF_DECORATIVE
$FF_DONTCARE
$FF_MODERN
$FF_ROMAN
$FF_SCRIPT
$FF_SWISS
$sFaceName Координаты.
$iStyle [необязательный] Стиль шрифта. Может быть одним или несколькими из следующих значений.
$FS_REGULAR (по умолчанию)
$FS_BOLD
$FS_ITALIC

Возвращаемое значение

Успех: Возвращает дескриптор на созданный шрифт.
Ошибка: Возвращает 0.

Примечания

Если запрошенный шрифт недоступен в системе, шрифтовый сопоставитель подставит шрифт со сходными атрибутами, но другим именем.

Когда шрифт больше не нужен, удалите его вызовом _WinAPI_DeleteObject().
