_WinAPI_CreateFontEx

создаёт логический шрифт с указанными характеристиками.

#include <WinAPIGdi.au3>
_WinAPI_CreateFontEx ( $iHeight [, $iWidth = 0 [, $iEscapement = 0 [, $iOrientation = 0 [, $iWeight = $FW_NORMAL [, $bItalic = False [, $bUnderline = False [, $bStrikeOut = False [, $iCharSet = $DEFAULT_CHARSET [, $iOutPrecision = $OUT_DEFAULT_PRECIS [, $iClipPrecision = $CLIP_DEFAULT_PRECIS [, $iQuality = $DEFAULT_QUALITY [, $iPitchAndFamily = $DEFAULT_PITCH [, $sFaceName = '' [, $iStyle = 0]]]]]]]]]]]]]] )

Параметры

$iHeight Высота знакоместа или символа шрифта в логических единицах.
$iWidth Необязательный параметр.
$iEscapement Необязательный параметр.
$iOrientation Необязательный параметр.
$iWeight Координаты.
$bItalic Значение.
$bUnderline Значение.
$bStrikeOut Значение.
$iCharSet Имя.
$iOutPrecision Строка.
$iClipPrecision Значение.
$iQuality Значение.
$iPitchAndFamily Координаты.
$sFaceName Координаты.
$iStyle Значение.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный шрифт.
Ошибка: Возвращает 0.

Примечания

Если запрошенный шрифт недоступен в системе, шрифтовый сопоставитель подставит шрифт со сходными атрибутами, но другим именем.

Когда шрифт больше не нужен, удалите его вызовом _WinAPI_DeleteObject().
