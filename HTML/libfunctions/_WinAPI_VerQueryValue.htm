Справочник функций

_WinAPI_VerQueryValue

Получает нефиксированную (строковую) информацию о версии из указанного ресурса информации о версии.

#include <WinAPIRes.au3>
_WinAPI_VerQueryValue ( $pData [, $sValues = ''] )

Параметры

$pData Указатель на буфер, который содержит ресурс информации о версии, возвращённый функцией
_WinAPI_GetFileVersionInfo().
$sValues [необязательный] Строка, содержащая имена полей, для которых вы хотите получить значения. Имена должны быть разделены символом "|".
Например, "name1|name2|...|namei". Если некоторые поля не существуют, соответствующие элементы массива – пустая строка.
Если этот параметр не задан (пустая строка), используются зарезервированные имена (12) в следующем порядке:
$FV_COMMENTS ("Comments")
$FV_COMPANYNAME ("CompanyName")
$FV_FILEDESCRIPTION ("FileDescription")
$FV_FILEVERSION ("FileVersion")
$FV_INTERNALNAME ("InternalName")
$FV_LEGALCOPYRIGHT ("LegalCopyright")
$FV_LEGALTRADEMARKS ("LegalTrademarks")
$FV_ORIGINALFILENAME ("OriginalFilename")
$FV_PRODUCTNAME ("ProductName")
$FV_PRODUCTVERSION ("ProductVersion")
$FV_PRIVATEBUILD ("PrivateBuild")
$FV_SPECIALBUILD ("SpecialBuild")

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает двумерный массив строковых значений, указанных параметром $sValues для каждого языка.
Нулевой элемент массива [0][0] содержит количество языков ресурса.
Размер массива (i) равен количеству указанных параметров + 1.
Нулевой элемент массива [n][0] содержит идентификатор языка (ID).
[0][0] – количество языков (n)
[0][i] – не используется
[n][0] – ID
[n][i] – значение
Ошибка: Устанавливает @error ≠ 0.

Дополнительно

Искать VerQueryValue в библиотеке MSDN

Пример

#include <Array.au3>
#include <WinAPIMem.au3>
#include <WinAPIRes.au3>

Local $pData = 0
If Not _WinAPI_GetFileVersionInfo(@SystemDir & '\shell32.dll', $pData) Then
    Exit
EndIf

Local $aData = _WinAPI_VerQueryValue($pData)

_ArrayDisplay($aData, '_WinAPI_VerQueryValue')

_WinAPI_FreeMemory($pData)
