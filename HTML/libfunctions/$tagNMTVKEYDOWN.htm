Справочник ключевых слов

$tagNMTVKEYDOWN

Содержит сведения о нажатии клавиши в элементе управления TreeView для кода уведомления $TVN_KEYDOWN.

Global Const $tagNMTVKEYDOWN = $tagNMHDR & ";word VKey;uint Flags"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
VKey Код виртуальной клавиши.
Flags Всегда нулём.
