Справочник ключевых слов

$tagTBBUTTON

Содержит сведения о кнопке на панели инструментов.

Global Const $tagTBBUTTON = "int Bitmap;int Command;byte State;byte Style;short Reserved;dword_ptr Param;int_ptr String"

Параметры

Bitmap 0-базовый индекс изображения кнопки. Установите этот член в $I_IMAGECALLBACK, и панель инструментов отправит уведомление $TBN_GETDISPINFO для получения индекса изображения, когда это необходимо.
Установите этот член в $I_IMAGENONE, чтобы указать, что кнопка не имеет изображения. Макет кнопки не будет включать место для растрового изображения, только текст.
Если кнопка является разделителем, Bitmap определяет ширину разделителя в пиксах.
Command Идентификатор команды, связанный с кнопкой. Этот идентификатор используется в сообщении $WM_COMMAND, когда кнопка выбирается.
State Флаги состояния кнопки.
Style Флаги стиля кнопки.
Reserved Зарезервировано 2 байта в режиме x86, 6 в режиме x64.
Param Значение, определённое приложением.
String 0-базовый индекс строки кнопки или указатель на строку, содержащую текст кнопки
