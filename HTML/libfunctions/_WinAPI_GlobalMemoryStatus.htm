Справочник функций

_WinAPI_GlobalMemoryStatus

Получает сведения о текущей доступной памяти.

#include <WinAPIMem.au3>
_WinAPI_GlobalMemoryStatus ( )

Возвращаемое значение

Успех: Возвращает массив следующего формата:
$aMem[0] – процент использованной памяти
$aMem[1] – физическая память: всего
$aMem[2] – физическая память: свободно
$aMem[3] – файл подкачки: всего
$aMem[4] – файл подкачки: свободно
$aMem[5] – пользовательская память: всего
$aMem[6] – пользовательская память: свободно
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

В отличие от функции AutoIt MemGetStats(), данная функция возвращает значения в байтах
