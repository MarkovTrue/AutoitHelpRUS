Справочник функций

_GUICtrlTreeView_AddChild

Добавляет новый элемент.

#include <GuiTreeView.au3>
_GUICtrlTreeView_AddChild ( $hWnd, $hParent, $sText [, $iImage = -1 [, $iSelImage = -1]] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$hParent Родительский элемент.
$sText Текст элемента.
$iImage [необязательный] Индекс значка элемента в списке изображений элемента управления (начиная с нуля)
$iSelImage [необязательный] Индекс значка элемента в списке изображений элемента управления (начиная с нуля)

Возвращаемое значение

Успех: Возвращает дескриптор нового элемента.
Ошибка: Возвращает 0.

Примечания

Элемент добавляется как дочерний элемент $hParent. Он добавляется в конец списка дочерних элементов $hParent.
Если элемент управления отсортирован, функция вставляет элемент в правильную позицию сортировки, а не как последний дочерний элемент $hParent.

См. также

_GUICtrlTreeView_AddChildFirst

Пример

#include <GUIConstantsEx.au3>
#include <GuiImageList.au3>
#include <GuiTreeView.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    GUICreate("TreeView Add Child (v" & @AutoItVersion & ")", 400, 300)

    Local $iStyle = BitOR($TVS_EDITLABELS, $TVS_HASBUTTONS, $TVS_HASLINES, $TVS_LINESATROOT, $TVS_DISABLEDRAGDROP, $TVS_SHOWSELALWAYS, $TVS_CHECKBOXES)
    Local $idTreeView = GUICtrlCreateTreeView(2, 2, 396, 268, $iStyle, $WS_EX_CLIENTEDGE)
    GUISetState(@SW_SHOW)

    Local $hImage = _GUIImageList_Create(16, 16, 5, 3)
    _GUIImageList_AddIcon($hImage, "shell32.dll", 110)
    _GUIImageList_AddIcon($hImage, "shell32.dll", 131)
    _GUIImageList_AddIcon($hImage, "shell32.dll", 165)
    _GUIImageList_AddIcon($hImage, "shell32.dll", 168)
    _GUIImageList_AddIcon($hImage, "shell32.dll", 137)
    _GUIImageList_AddIcon($hImage, "shell32.dll", 146)
    _GUICtrlTreeView_SetNormalImageList($idTreeView, $hImage)

    _GUICtrlTreeView_BeginUpdate($idTreeView)
    Local $hItem, $iImage
    For $x = 0 To 4
        $iImage = Random(0, 5, 1)
        $hItem = _GUICtrlTreeView_Add($idTreeView, 0, StringFormat("[%02d] New Item", $x), $iImage, $iImage)
        For $y = 0 To 3
            $iImage = Random(0, 5, 1)
            _GUICtrlTreeView_AddChild($idTreeView, $hItem, StringFormat("[%02d] New Child", $y), $iImage, $iImage)
        Next
    Next
    _GUICtrlTreeView_EndUpdate($idTreeView)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
