_WinAPI_ShellGetSpecialFolderLocation

Получает указатель на структуру ITEMIDLIST (PIDL) специальной папки.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetSpecialFolderLocation ( $iCSIDL )

Параметры

$iCSIDL Идентификатор CSIDL ($CSIDL_*), который идентифицирует интересующую папку.

Возвращаемое значение

Успех: Возвращает PIDL, указывающий расположение папки относительно корня пространства имён (рабочего стола).
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Чтобы освободить возвращённый PIDL, вызовите функцию _WinAPI_CoTaskMemFree().
