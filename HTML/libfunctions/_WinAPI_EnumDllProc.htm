Справочник функций

_WinAPI_EnumDllProc

Перечисляет экспортируемые функции указанной динамической библиотеки (DLL).

#include <WinAPIDiag.au3>
_WinAPI_EnumDllProc ( $sFilePath [, $sMask = '' [, $iFlags = 0]] )

Параметры

$sFilePath Путь к библиотеке.
Функция ищет файл, если задан относительный путь или имя без пути, но для однозначного результата лучше задавать полный путь.
$sMask [необязательный] Строка-шаблон с именами перечисляемых функций.
Может содержать подстановочные знаки "*" и "?". Если параметр – пустая строка или опущен (по умолчанию), перечисляются все экспортируемые функции.
$iFlags [необязательный] Дополнительные флаги. Может быть сочетанием значений:
$SYMOPT_CASE_INSENSITIVE
$SYMOPT_UNDNAME

Возвращаемое значение

Успех: Возвращает двумерный массив со следующей информацией; @extended получает одну из констант $IMAGE_FILE_MACHINE_*.
$aArray[0][0] – количество строк массива (n)
$aArray[0][1] – не используется
[n][0] – адрес функции относительно базового адреса библиотеки (UINT64).
[n][1] – имя функции.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Функция работает как с 32-, так и с 64-битными DLL независимо от разрядности системы; однако 64-битные исполняемые файлы не могут загружать 32-битные DLL и наоборот.

Для получения полного пути к найденной DLL, загруженной для перечисления функций, вызовите _WinAPI_GetExtended().

Используйте полный путь, чтобы _WinAPI_EnumDllProc() точно загрузил нужную DLL.
Кроме того, функция не использует перенаправление файловой системы.

Требуется dbghelp.dll версии 5.1 или выше.

См. также

_WinAPI_GetExtended

Пример

#include <Array.au3>
#include <WinAPIDiag.au3>
#include <WinAPIMisc.au3>

_Example()

Func _Example()
    Local $aData = _WinAPI_EnumDllProc('ntdll.dll', 'Rtl*', $SYMOPT_CASE_INSENSITIVE)

    If IsArray($aData) Then
        For $i = 1 To $aData[0][0]
            $aData[$i][0] = '0x' & Hex($aData[$i][0])
        Next
    EndIf

    _ArrayDisplay($aData, _WinAPI_GetExtended())
EndFunc   ;==>_Example
