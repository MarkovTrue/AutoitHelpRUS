_WinAPI_GetWindowRgn

Получает копию области окна.

#include <WinAPIGdi.au3>
_WinAPI_GetWindowRgn ( $hWnd, $hRgn )

Параметры

$hWnd Дескриптор окна.
$hRgn Дескриптор области, которая будет изменена для представления области окна.

Возвращаемое значение

Успех: Возвращает указывает тип области, которую функция получает. Может быть одним из следующих значений
$NULLREGION – область пуста.
$SIMPLEREGION – область – это один прямоугольник.
$COMPLEXREGION – область содержит более одного прямоугольника.
Ошибка: 0 Указанное окно не имеет области, или произошла ошибка при попытке вернуть область.

Примечания

Область окна задаётся путём вызова функции SetWindowRgn.
Область окна определяет область в окне, в которой система разрешает рисование.
Система не отображает какую-либо часть окна, которая находится вне области окна
Координаты области окна относительны верхнему левому углу окна, а не клиентской области окна.
Чтобы установить область окна, вызовите функцию SetWindowRgn.
