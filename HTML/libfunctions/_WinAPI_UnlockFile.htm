_WinAPI_UnlockFile

Разблокирует регион в открытом файле.

#include <WinAPIFiles.au3>
_WinAPI_UnlockFile ( $hFile, $iOffset, $iLength )

Параметры

$hFile Дескриптор файла, который содержит регион, заблокированный функцией _WinAPI_LockFile().
$iOffset Начальное смещение байта в файле, где начинается заблокированный регион.
$iLength Длина диапазона байтов для разблокирования.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Разблокирование региона файла освобождает ранее полученную блокировку файла. Регион для разблокирования должен соответствовать
точно существующему заблокированному региону. Два смежных региона файла не могут быть заблокированы отдельно, а затем разблокированы
используя единый регион, который охватывает оба заблокированные региона.
