_WinAPI_FindWindow

Получает дескриптор окна верхнего уровня, имя класса и имя окна которого совпадают с указанными.

#include <WinAPISysWin.au3>
_WinAPI_FindWindow ( $sClassName, $sWindowName )

Параметры

$sClassName Строка с именем класса или атом, идентифицирующий строку имени класса.
Если параметр – атом, он должен быть глобальным атомом, созданным вызовом GlobalAddAtom.
Атом (16-битное значение) должен быть размещён в младшем слове $sClassName, старшее слово должно быть нулевым.
$sWindowName Строка, которая определяет the имя окна. Если этот/эта parameter is blank, all имя окнаs match.

Возвращаемое значение

Успех: Первое слово предложения дескриптор к окно.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
