_WinAPI_MAKELANGID

Строит идентификатор языка из первичного идентификатора языка и идентификатора подъязыка.

#include <WinAPIConv.au3>
_WinAPI_MAKELANGID ( $iLngIDPrimary, $iLngIDSub )

Параметры

$iLngIDPrimary Первичный идентификатор языка.
$iLngIDSub Идентификатор подъязыка.

Возвращаемое значение

Возвращает идентификатор языка

Примечания

Константы для $iLngIDPrimary или $iLngIDSub определены в WinAPILangConstants.au3.
