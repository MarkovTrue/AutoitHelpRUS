Справочник функций

_IEGetObjById

Возвращает объектную переменную по id.

#include <IE.au3>
_IEGetObjById ( ByRef $oObject, $sID )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sID Идентификатор объекта, который нужно найти.

Возвращаемое значение

Успех: Возвращает объектную переменную, ссылающуюся на указанный объект.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных
7 ($_IEStatus_NoMatch) – совпадение не найдено

Примечания

Эта функция возвращает ссылку на любой объект в документе с указанным id или name.
DOM-элементы могут иметь атрибуты Name, ID или оба сразу.
Конкретный ID может быть присвоен только одному элементу, тогда как конкретное Name может быть присвоено нескольким элементам.
В Internet Explorer пространства имён Name и ID являются общими.

См. также

_IEGetObjByName

Пример

; Открыть браузер на базовом примере, получить ссылку на объект
; к элементу DIV с ID "line1". Отображение innerText
; этого элемента в консоль.

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("basic")
Local $oDiv = _IEGetObjById($oIE, "line1")

MsgBox($MB_SYSTEMMODAL, "Line1", $oDiv.innertext)
