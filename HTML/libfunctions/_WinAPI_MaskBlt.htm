Справочник функций

_WinAPI_MaskBlt

Объединяет данные цвета исходного и конечного растровых изображений, используя указанную маску и растровую операцию.

#include <WinAPIGdi.au3>
_WinAPI_MaskBlt ( $hDestDC, $iXDest, $iYDest, $iWidth, $iHeight, $hSrcDC, $iXSrc, $iYSrc, $hMask, $iXMask, $iYMask, $iRop )

Параметры

$hDestDC Дескриптор контекста устройства назначения.
$iXDest Координата X верхнего левого угла целевого прямоугольника в логических единицах.
$iYDest Координата Y верхнего левого угла целевого прямоугольника в логических единицах.
$iWidth Ширина прямоугольника назначения и исходного растрового изображения в логических единицах.
$iHeight Высота прямоугольника назначения и исходного растрового изображения в логических единицах.
$hSrcDC Дескриптор контекста устройства, из которого копируется растровое изображение.
$iXSrc Координата x в логических единицах верхнего левого угла исходного растрового изображения.
$iYSrc Координата y в логических единицах верхнего левого угла исходного растрового изображения.
$hMask Дескриптор монохромного растрового изображения маски, объединённого с цветным растровым изображением в исходном контексте устройства.
$iXMask Горизонтальное смещение в пиксельях для растрового изображения маски, заданного параметром hbmMask.
$iYMask Вертикальное смещение в пиксельях для растрового изображения маски, заданного параметром hbmMask.
$iRop Код растровой операции (как для _WinAPI_BitBlt()).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Значение 1 в маске, заданной $hMask, указывает, что код растровой операции переднего плана, заданный
$iRop, должен применяться в этом месте. Значение 0 в маске указывает, что код растровой операции фона
, заданный $iRop, должен применяться в этом месте.

Если растровое изображение маски не предоставлено, эта функция работает точно так же, как _WinAPI_BitBlt(), используя код растровой операции переднего плана.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPIIcons.au3>
#include <WinAPIRes.au3>
#include <WinAPIShellEx.au3>

; Извлечь значок и получить его растровое изображение маски И
Local $sIco = _Extras_PathFull('Arrow.ico')
Local $hIcon = _WinAPI_ShellExtractIcon($sIco, 0, 64, 64)
Local $aInfo = _WinAPI_GetIconInfo($hIcon)
_WinAPI_DeleteObject($aInfo[5])
_WinAPI_DestroyIcon($hIcon)

; Создать инвертированный bitmask bitmap
$aInfo[5] = _WinAPI_InvertANDBitmap($aInfo[4])

; Загрузить растровое изображение узора
Local $sBmp = _Extras_PathFull('Pattern.bmp')
Local $hPattern = _WinAPI_LoadImage(0, $sBmp, $IMAGE_BITMAP, 0, 0, $LR_LOADFROMFILE)

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 320, 64)
GUICtrlCreateIcon($sIco, 0, 0, 0, 64, 64)
GUICtrlCreatePic($sBmp, 192, 0, 64, 64)
Local $idIcon = GUICtrlCreateIcon('', 0, 256, 0, 64, 64)
Local $aidPic[2], $ahPic[2]
$aidPic[0] = GUICtrlCreatePic('', 64, 0, 64, 64)
$aidPic[1] = GUICtrlCreatePic('', 128, 0, 64, 64)
For $i = 0 To 1
    $ahPic[$i] = GUICtrlGetHandle($aidPic[$i])
Next

; Создать растровое изображение маски XOR
Local $hDC = _WinAPI_GetDC($ahPic)
Local $hSrcDC = _WinAPI_CreateCompatibleDC($hDC)
Local $hSrcSv = _WinAPI_SelectObject($hSrcDC, $hPattern)
Local $hDestDC = _WinAPI_CreateCompatibleDC($hDC)
Local $hXOR = _WinAPI_CreateCompatibleBitmap($hDC, 64, 64)
Local $hDestSv = _WinAPI_SelectObject($hDestDC, $hXOR)
_WinAPI_MaskBlt($hDestDC, 0, 0, 64, 64, $hSrcDC, 0, 0, $aInfo[5], 0, 0, $SRCCOPY)
_WinAPI_ReleaseDC($ahPic, $hDC)
_WinAPI_SelectObject($hDestDC, $hDestSv)
_WinAPI_DeleteDC($hDestDC)
_WinAPI_SelectObject($hSrcDC, $hSrcSv)
_WinAPI_DeleteObject($hPattern)
_WinAPI_DeleteDC($hSrcDC)

; Создать иконку

$hIcon = _WinAPI_CreateIconIndirect($hXOR, $aInfo[4])
_WinAPI_DeleteObject($hXOR)

; Установить оба растровых изображения маски для элементов управления
Local $hObj
For $i = 0 To 1
    _SendMessage($ahPic[$i], $STM_SETIMAGE, 0, $aInfo[$i + 4])
    $hObj = _SendMessage($ahPic[$i], $STM_GETIMAGE)
    If $hObj <> $aInfo[$i + 4] Then
        _WinAPI_DeleteObject($aInfo[$i + 4])
    EndIf
Next

; Установить значок для элемента управления
GUICtrlSendMsg($idIcon, $STM_SETIMAGE, 1, $hIcon)

; Показать GUI
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
