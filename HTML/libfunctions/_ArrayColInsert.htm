Справочник функций

_ArrayColInsert

Вставляет указанный столбец в одномерный 1D или двумерный 2D массив.

#include <Array.au3>
_ArrayColInsert ( ByRef $aArray, $iColumn )

Параметры

$aArray Изменяемый массив.
$iColumn Вставляемый столбец. Если массив 1D, он автоматически преобразуется в 2D.

Возвращаемое значение

Успех: Возвращает количество столбцов в массиве после вставки.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не массив
2 – $aArray не 1D и не 2D массив
3 – $iColumn меньше 0 или больше верхней границы массива плюс 1

Примечания

Функция НЕ обновляет счётчик внутри массива; при успехе возвращается итоговое количество столбцов.

См. также

_ArrayColDelete

Пример

#include <Array.au3>

Local $aArray[4] = [0, 1, 2, 3]
_ArrayDisplay($aArray, "Original")

_ArrayColInsert($aArray, 0)
; Теперь 2D массив.
_ArrayDisplay($aArray, "Col 0 inserted")

_ArrayColInsert($aArray, 1)
_ArrayDisplay($aArray, "Col 1 inserted")

_ArrayColInsert($aArray, 3)
_ArrayDisplay($aArray, "Col 3 inserted")
