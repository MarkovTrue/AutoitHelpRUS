_WinAPI_CreateSolidBitmap

Создаёт точечный рисунок одного цвета.

#include <WinAPIGdi.au3>
_WinAPI_CreateSolidBitmap ( $hWnd, $iColor, $iWidth, $iHeight [, $bRGB = 1] )

Параметры

$hWnd Дескриптор окна.
$iColor Цвет точечного рисунка в формате RGB.
$iWidth Ширина точечного рисунка.
$iHeight Высота точечного рисунка.
$bRGB Цвет (RGB).

Возвращаемое значение

Успех: Возвращает дескриптор на созданное растровое изображение.
Ошибка: Возвращает 0.
