Справочник функций

_GUICtrlHeader_GetItemFlags

Возвращает информацию о флагах элемента.

#include <GuiHeader.au3>
_GUICtrlHeader_GetItemFlags ( $hWnd, $iIndex )

Параметры

$hWnd Дескриптор элемента управления.
$iIndex Индекс элемента (нумерация с 0).

Возвращаемое значение

Возвращает информацию о флагах элемента:
    1 – отображает изображение из списка изображений
    2 – текст читается в противоположном направлении от текста в родительском окне
    4 – рисует стрелку вниз на этом элементе
    8 – рисует стрелку вверх на этом элементе

См. также

_GUICtrlHeader_SetItemFlags

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiHeader.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Header Get ItemFlags (v" & @AutoItVersion & ")", 450, 300, 100, 100)

    Local $hHeader = _GUICtrlHeader_Create($hGUI)
    _MemoCreate(2, 52, 444, 220)

;~ _GUICtrlHeader_SetUnicodeFormat($hHeader, True)

    GUISetState(@SW_SHOW)

    ; Добавить столбцы
    _GUICtrlHeader_AddItem($hHeader, "Column 0", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 1", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 2", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 3", 100)

    ; Установить флаги столбца 1
    _GUICtrlHeader_SetItemFlags($hHeader, 0, 4)

    ; Показать флаги столбца 1
    _MemoWrite("Column 0 flags: " & _GUICtrlHeader_GetItemFlags($hHeader, 0))

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
