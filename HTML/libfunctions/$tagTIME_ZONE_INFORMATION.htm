Справочник ключевых слов

$tagTIME_ZONE_INFORMATION

Задаёт сведения о часовом поясе.

Global Const $tagTIME_ZONE_INFORMATION = "struct; long Bias;wchar StdName[32];word StdDate[8];long StdBias;wchar DayName[32];word DayDate[8];long DayBias; endstruct"

Параметры

Bias Текущее смещение для преобразования локального времени на этом компьютере в минутах.
StdName Описание стандартного времени.
StdDate Структура SYSTEMTIME, содержащая дату и локальное время переходa с летнего времени на стандартное время в этой операционной системе.
StdBias Значение смещения для использования при преобразованиях локального времени, происходящих в стандартное время.
DayName Описание летнего времени.
DayDate Структура SYSTEMTIME, содержащая дату и локальное время перехода со стандартного времени на летнее время в этой операционной системе.
DayBias Значение смещения для использования при преобразованиях локального времени, происходящих в летнее время.
