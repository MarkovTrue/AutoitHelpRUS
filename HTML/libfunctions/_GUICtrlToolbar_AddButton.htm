_GUICtrlToolbar_AddButton

Добавляет кнопку.

#include <GuiToolbar.au3>
_GUICtrlToolbar_AddButton ( $hWnd, $iID, $iImage [, $iString = 0 [, $iStyle = 0 [, $iState = $TBSTATE_ENABLED [, $iParam = 0]]]] )

Параметры

$hWnd Дескриптор элемента управления.
$iID Идентификатор команды.
$iImage 0-based индекс изображения кнопки.
Задайте $I_IMAGECALLBACK, и элемент управления отправит уведомление $TBN_GETDISPINFO для получения индекса изображения, когда оно понадобится.
Задайте $I_IMAGENONE, чтобы указать, что у кнопки нет изображения.
В разметке кнопки будет место только для текста.
Задайте $I_IMAGE (по умолчанию), если у кнопки есть изображение.
Если кнопка является разделителем, это ширина разделителя в пикселах.
$iString Индекс строки.
$iStyle [необязательный] Стиль кнопки. Может быть комбинацией следующих значений:
$BTNS_AUTOSIZE – элемент управления не назначает кнопке стандартную ширину
$BTNS_BUTTON – обычная кнопка (по умолчанию)
$BTNS_CHECK – переключается между нажатым и ненажатым состоянием
$BTNS_CHECKGROUP – кнопка остаётся нажатой, пока не нажата другая кнопка группы
$BTNS_DROPDOWN – создаёт раскрывающуюся кнопку, способную показывать список
$BTNS_GROUP – кнопка остаётся нажатой, пока не нажата другая кнопка группы
$BTNS_NOPREFIX – текст кнопки не имеет префикса ускорителя
$BTNS_SEP – создаёт разделитель
$BTNS_SHOWTEXT – текст кнопки должен отображаться
$BTNS_WHOLEDROPDOWN – у кнопки будет стрелка раскрытия.
$iState [необязательный] Состояние кнопки. Может быть комбинацией следующих значений:
$TBSTATE_CHECKED – кнопка имеет стиль $TBSTYLE_CHECK и нажата
$TBSTATE_PRESSED – кнопка нажата
$TBSTATE_ENABLED – кнопка принимает ввод пользователя (по умолчанию)
$TBSTATE_HIDDEN – кнопка не видна и не может получать ввод пользователя
$TBSTATE_INDETERMINATE – кнопка неактивна (серая)
$TBSTATE_WRAP – за кнопкой следует перенос строки
$TBSTATE_ELLIPSES – текст кнопки обрезается и показывается многоточие
$TBSTATE_MARKED – кнопка помечена
По умолчанию: $TBSTATE_ENABLED.
$iParam [необязательный] Определяемое приложением значение

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_GUICtrlToolbar_DeleteButton, _GUICtrlToolbar_InsertButton

Пример

#include <GUIConstantsEx.au3>
#include <GuiToolbar.au3>
#include <WinAPIConstants.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Toolbar Add Bitmao (v" & @AutoItVersion & ")", 400, 300)
    Local $hToolbar = _GUICtrlToolbar_Create($hGUI)
    GUISetState(@SW_SHOW)

    ; Установить формат ANSI
;~ _GUICtrlToolbar_SetUnicodeFormat($hToolbar, False)

    ; Добавить стандартные системные изображения
    _GUICtrlToolbar_AddBitmap($hToolbar, 1, -1, $IDB_STD_LARGE_COLOR)

    ; Добавить кнопки
    Local Enum $e_idNew = 1000, $e_idOpen, $e_idSave, $e_idHelp
    _GUICtrlToolbar_AddButton($hToolbar, $e_idNew, $STD_FILENEW)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idOpen, $STD_FILEOPEN)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idSave, $STD_FILESAVE)
    _GUICtrlToolbar_AddButtonSep($hToolbar)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idHelp, $STD_HELP)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
