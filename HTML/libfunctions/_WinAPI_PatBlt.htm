_WinAPI_PatBlt

Закрашивает указанный прямоугольник, используя кисть, которая в данный момент выбрана в указанном контексте устройства.

#include <WinAPIGdi.au3>
_WinAPI_PatBlt ( $hDC, $iX, $iY, $iWidth, $iHeight, $iRop )

Параметры

$hDC Дескриптор контекста устройства.
$iX Координата x в логических единицах верхнего левого угла заполняемого прямоугольника.
$iY Координата y в логических единицах верхнего левого угла заполняемого прямоугольника.
$iWidth Ширина прямоугольника в логических единицах.
$iHeight Высота прямоугольника в логических единицах.
$iRop Код растровой операции. Это значение может быть одной из следующих констант.
$BLACKNESS
$DSTINVERT
$PATCOPY
$PATINVERT
$WHITENESS

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
