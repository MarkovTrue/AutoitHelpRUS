_WinAPI_SfcIsFileProtected

Определяет, защищён ли указанный файл.

#include <WinAPIFiles.au3>
_WinAPI_SfcIsFileProtected ( $sFilePath )

Параметры

$sFilePath Имя файла для проверки.

Возвращаемое значение

True: Возвращает файл защищён.
False: файл не защищён.

Дополнительно

Искать SfcIsFileProtected в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

MsgBox($MB_SYSTEMMODAL, "Title", 'File Protection Status = ' & _WinAPI_SfcIsFileProtected(@WindowsDir & "\explorer.exe"))
