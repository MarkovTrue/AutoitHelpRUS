Справочник функций

_IEPropertySet

Устанавливает выбранное свойство браузера или DOM-элемента.

#include <IE.au3>
_IEPropertySet ( ByRef $oObject, $sProperty, $vValue )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$sProperty Имя свойства (см. раздел Примечания).
$vValue Новое значение, которое нужно установить свойству браузера.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – неверное значение
@extended: Содержит номер недопустимого параметра.

Примечания

В таблице ниже приведено описание каждого доступного свойства.
Некоторые свойства относятся к DOM-элементам, другие – к браузеру.

Свойства браузера

Свойство Описание
"addressbar" Задаёт, видна ли адресная строка объекта.
"contenteditable" Задаёт Boolean-значение, указывающее, можно ли редактировать объект мышью и клавиатурой.
"height" Задаёт высоту главного окна браузера.
"innerhtml" Устанавливает отрендеренный HTML элемента без начального и конечного тегов элемента.
"innertext" Устанавливает отрендеренный текст (без тегов) элемента. Обычно идентичен outertext.
"left" Задаёт экранную координату левого края главного окна браузера.
"menubar" Задаёт Boolean-значение, указывающее, видна ли строка меню браузера.
"offline" Задаёт Boolean-значение, указывающее, работает ли браузер в данный момент в автономном режиме.
"outertext" Устанавливает отрендеренный текст (без тегов) элемента. Обычно идентичен innertext.
"outerhtml" Устанавливает отрендеренный HTML элемента, включая начальный и конечный теги элемента.
"resizable" Задаёт Boolean-значение, указывающее, можно ли изменять размер объекта.
"silent" Задаёт Boolean-значение, указывающее, может ли браузер показывать диалоговые окна.
"statusbar" Задаёт значение, указывающее, видна ли строка состояния браузера.
"statustext" Задаёт текст в строке состояния браузера.
"theatermode" Задаёт Boolean-значение, указывающее, находится ли браузер в режиме "театра" (Theater Mode). В режиме театра главное окно браузера занимает весь экран и отображает панель инструментов с минимальным набором навигационных кнопок.
"title" Устанавливает заголовок документа. Заметьте, он отличается от заголовка окна, который обычно начинается с заголовка документа, но дополняется текстом из реестра Windows.
"toolbar" Задаёт Boolean-значение, указывающее, видна ли панель инструментов браузера.
"top" Задаёт экранную координату верхнего края главного окна браузера.
"width" Задаёт ширину главного окна браузера.

См. также

Пример

; Открыть браузер с базовым примером, проверить,
; видна ли адресная строка; если нет – показать. Затем изменить
; текст, отображаемый в строке состояния

#include <IE.au3>

Local $oIE = _IE_Example("basic")
If Not _IEPropertyGet($oIE, "statusbar") Then _IEPropertySet($oIE, "statusbar", True)
_IEPropertySet($oIE, "statustext", "Look What I can Do")
Sleep(2000)
_IEPropertySet($oIE, "statustext", "I can change the status text")
