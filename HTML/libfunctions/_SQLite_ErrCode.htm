Справочник функций

_SQLite_ErrCode

Возвращает последний код ошибки (число).

#include <SQLite.au3>
_SQLite_ErrCode ( [$hDB = -1] )

Параметры

$hDB [необязательный] Открытая база данных; по умолчанию последняя открытая

Возвращаемое значение

Успех: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
Ошибка: Возвращает 0.
@error: 1 – ошибка при вызове SQLite API "sqlite3_errcode"
2 – вызов запрещён безопасным режимом

См. также

_SQLite_ErrMsg

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open()
If $SQLITE_OK <> _SQLite_Exec(-1, "CREATE TABLE test (a', 'b');") Then _ ; отсутствует кавычка
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", "Error Code: " & _SQLite_ErrCode() & @CRLF & "Error Message: " & _SQLite_ErrMsg())
_SQLite_Close()
_SQLite_Shutdown()
