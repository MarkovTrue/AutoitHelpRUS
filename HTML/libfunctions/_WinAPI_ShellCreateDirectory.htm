_WinAPI_ShellCreateDirectory

Создаёт новую папку в файловой системе.

#include <WinAPIShellEx.au3>
_WinAPI_ShellCreateDirectory ( $sFilePath [, $hParent = 0 [, $tSecurity = 0]] )

Параметры

$sFilePath Полный квалифицированный путь каталога.
$hParent [необязательный] Дескриптор родительского окна.
$tSecurity [необязательный] Структура $tagSECURITY_ATTRIBUTES с атрибутом безопасности каталога. Если этот параметр равен 0 (по умолчанию), атрибуты безопасности не устанавливаются.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Если одна или несколько промежуточных папок не существуют, они также создаются. Кроме того, эта функция проверяет, что файлы в каталоге будут видны, если путь является сетевым путём, чтобы иметь дело с сетевыми драйверами, которые могут иметь ограниченную, но неизвестную максимальную длину пути.
