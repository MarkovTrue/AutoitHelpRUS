Справочник функций

_ArrayBinarySearch

Ищет значение в одномерном 1D или двумерном 2D массиве алгоритмом бинарного поиска.

#include <Array.au3>
_ArrayBinarySearch ( Const ByRef $aArray, $vValue [, $iStart = 0 [, $iEnd = 0 [, $iColumn = 0]]] )

Параметры

$aArray Массив для поиска.
$vValue Искомое значение.
$iStart [необязательный] Индекс массива для начала поиска
$iEnd [необязательный] Индекс элемента, на котором заканчивается поиск
$iColumn [необязательный] Столбец массива для поиска

Возвращаемое значение

Успех: Возвращает индекс найденного элемента.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не массив
2 – $vValue вне диапазона значений массива
3 – $vValue не найден в массиве
4 – $iStart больше $iEnd
5 – $aArray не 1D и не 2D массив
6 – $aArray пуст
7 – $iColumn вне границ массива

Примечания

Перед бинарным поиском массив (для 2D массива – заданный столбец) ОБЯЗАН быть отсортирован по ВОЗРАСТАНИЮ, иначе результат не определён.

См. также

_ArrayFindAll, _ArraySearch

Пример

Пример 1

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $iIndex

Local $aArray[5][2]
For $i = 0 To 4
    For $j = 0 To 1
        $aArray[$i][$j] = "#" & $i & $j
    Next
Next
_ArrayDisplay($aArray, "Array")

; Ищем в столбце 0
$iIndex = _ArrayBinarySearch($aArray, "#10", 0, 0, 0)
MsgBox($MB_SYSTEMMODAL, "Index", $iIndex)

; Ищем в столбце 1
$iIndex = _ArrayBinarySearch($aArray, "#31", 0, 0, 1)
MsgBox($MB_SYSTEMMODAL, "Index", $iIndex)

Пример 2

; используем массив, возвращённый StringSplit()

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $avArray = StringSplit("a,b,d,c,e,f,g,h,i", ",")

; сортируем массив для бинарного поиска
_ArraySort($avArray, 0, 1) ; начать с индекса 1, пропустив $avArray[0]

; вывести отсортированный массив
_ArrayDisplay($avArray, "$avArray AFTER _ArraySort()")

; начать с индекса 1, пропустив $avArray[0]
Local $iKeyIndex = _ArrayBinarySearch($avArray, "c", 1)
If Not @error Then
    MsgBox($MB_SYSTEMMODAL, 'Entry found', ' Index: ' & $iKeyIndex)
Else
    MsgBox($MB_SYSTEMMODAL, 'Entry Not found', ' Error: ' & @error)
EndIf
