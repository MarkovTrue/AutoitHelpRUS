Справочник функций

_WinNet_GetConnectionPerformance

Возвращает информацию о производительности ресурса сетевого подключения.

#include <WinNet.au3>
_WinNet_GetConnectionPerformance ( $sLocalName, $sRemoteName )

Параметры

$sLocalName Имя локального устройства, например "F:" или "LPT1", переадресованного на сетевой ресурс для запроса. Если пусто, сетевой ресурс указан в $sRemoteName. Если этот параметр указывает локальное устройство, $sRemoteName игнорируется.
$sRemoteName Имя сетевого ресурса для запроса. Ресурс должен иметь установленное подключение. Например, если ресурс является файлом на файловом сервере, то открытие файла обеспечит подключение.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
$aArray[0] – описание подключения. Может быть одним или несколькими из следующих:
1 – возвращённая информация относится к производительности сетевой карты
2 – подключение не маршрутизируется
4 – подключение использует среду, которая обычно медленная
8 – некоторая возвращённая информация вычисляется динамически
$aArray[1] – скорость носителя к сетевому ресурсу в 100 бит/с
$aArray[2] – задержка в одну сторону при отправке информации
$aArray[3] – размер данных, которые приложение должно использовать при выполнении одного запроса к ресурсу
Ошибка: Устанавливает @error ≠ 0.
