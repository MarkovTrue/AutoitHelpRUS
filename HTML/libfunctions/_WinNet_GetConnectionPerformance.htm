Справочник функций

_WinNet_GetConnectionPerformance

Возвращает информацию о производительности ресурса сетевого подключения.

#include <WinNet.au3>
_WinNet_GetConnectionPerformance ( $sLocalName, $sRemoteName )

Параметры

$sLocalName Имя локального устройства, например "F:" или "LPT1", переадресованного на сетевой ресурс для запроса. Если пусто, сетевой ресурс указан в $sRemoteName. Если этот параметр указывает локальное устройство, $sRemoteName игнорируется.
$sRemoteName Имя сетевого ресурса для запроса. Ресурс должен иметь установленное подключение. Например, если ресурс является файлом на файловом сервере, то открытие файла обеспечит подключение.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:     $aInfo[0] – Описание подключения. Может быть одним или несколькими из следующих:         1 – Возвращённая информация относится к производительности сетевой карты         2 – Подключение не маршрутизируется         4 – Подключение над средой, которая обычно является медленной         8 – Некоторая из возвращённой информации вычисляется динамически     $aInfo[1] – Скорость носителя к сетевому ресурсу в 100 бит/с     $aInfo[2] – Время задержки в одну сторону при отправке информации     $aInfo[3] – Размер данных, которые приложение должно использовать при выполнении одного запроса к ресурсу.
Ошибка: Устанавливает @error ≠ 0.
