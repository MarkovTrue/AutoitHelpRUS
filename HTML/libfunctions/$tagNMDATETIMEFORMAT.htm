Справочник ключевых слов

$tagNMDATETIMEFORMAT

Содержит сведения о части строки формата, определяющей поле обратного вызова в элементе управления "Выбор даты и времени" (DTP).

Global Const $tagNMDATETIMEFORMAT = $tagNMHDR & ";ptr Format;" & $tagSYSTEMTIME & ";ptr pDisplay;wchar Display[64]"

Параметры

$tagNMHDR Содержит информацию о сообщении уведомления.
Format Указатель на строку с нулём в конце, которая определяет поле обратного вызова элемента управления. Строка состоит из одного или нескольких символов "X".
$tagSYSTEMTIME Дата и время, которые будут отформатированы.
pDisplay Указатель на строку с нулём в конце, которая содержит отображаемый текст элемента управления. По умолчанию это адрес элемента Display этой структуры.
Допустимо, чтобы pDisplay указывал на существующую строку. В этом случае вам не нужно присваивать значение Display.
Однако строка, на которую указывает pDisplay, должна оставаться действительной до тех пор, пока не будет отправлено другое уведомление $DTN_FORMAT или пока элемент управления не будет уничтожен.
Display Буфер на 64 символа, который будет содержать строку с нулём в конце, которая будет отображаться элементом управления.
Не требуется заполнять весь буфер.

Примечания

Содержит строку, которая определяет поле обратного вызова, и содержит буфер для получения строки, которая будет отображаться в поле обратного вызова.
Эта структура используется с сообщением уведомления $DTN_FORMAT.
