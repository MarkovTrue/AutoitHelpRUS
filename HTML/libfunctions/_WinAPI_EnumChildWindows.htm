Справочник функций

_WinAPI_EnumChildWindows

Перечисляет дочерние окна указанного родительского окна.

#include <WinAPISysWin.au3>
_WinAPI_EnumChildWindows ( $hWnd [, $bVisible = True] )

Параметры

$hWnd Дескриптор родительского окна, дочерние окна которого перечисляются.
Если параметр равен 0, функция эквивалентна _WinAPI_EnumWindows().
$bVisible [необязательный] Перечислять ли невидимые окна:
True – только видимые окна (по умолчанию).
False – все окна.

Возвращаемое значение

Успех: Возвращает двумерный массив дескрипторов и классов дочерних окон указанного родителя.
$aArray[0][0] – количество строк массива (n)
$aArray[0][1] – не используется
[n][0] – дескриптор окна
[n][1] – имя класса окна
Ошибка: Устанавливает @error ≠ 0.

GUICtrlCreateGroup('Group', 10, 10, 140, 95)
GUICtrlCreateCheckbox('Check1', 22, 26, 120, 23)
GUICtrlCreateCheckbox('Check2', 22, 49, 120, 23)
GUICtrlCreateCheckbox('Check3', 22, 72, 120, 23)
GUICtrlCreateGroup('Group', 160, 10, 140, 95)

GUICtrlCreateRadio('Radio2', 172, 49, 120, 23)
GUICtrlCreateRadio('Radio3', 172, 72, 120, 23)

GUICtrlCreateTab(10, 118, 292, 206)

GUICtrlCreateTabItem('Tab2')

Local $aData = _WinAPI_EnumChildWindows($hForm)

_ArrayDisplay($aData, '_WinAPI_EnumChildWindows')
