_WinAPI_ExtractIcon

Извлекает значок из указанного исполняемого файла, DLL или файла значков.

#include <WinAPIIcons.au3>
_WinAPI_ExtractIcon ( $sIcon, $iIndex [, $bSmall = False] )

Параметры

$sIcon Имя исполняемого файла, DLL или файла значков, из которого извлекаются значки.
$iIndex Индекс извлекаемого значка (нумерация с 0).
Если значение отрицательное, функция извлекает значок, идентификатор ресурса которого равен модулю $iIndex.
$bSmall [необязательный] Извлекать ли маленький значок:
True – маленький значок.
False – большой значок (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор извлечённой иконки.
Ошибка: Возвращает 0.

Примечания

Когда значок больше не нужен, уничтожьте его функцией _WinAPI_DestroyIcon().
