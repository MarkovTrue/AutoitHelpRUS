Справочник ключевых слов

$tagNMLVKEYDOWN

Содержит сведения для обработки уведомляющего сообщения $LVN_KEYDOWN.

Global Const $tagNMLVKEYDOWN = $tagNMHDR & ";word VKey;uint Flags"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
VKey Виртуальный код клавиши.
Flags Должно быть нулём.
