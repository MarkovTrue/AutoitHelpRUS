Справочник функций

_Net_Share_StatisticsGetSvr

Получает статистику операций для сервера.

#include <NetShare.au3>
_Net_Share_StatisticsGetSvr ( [$sServer = ""] )

Параметры

$sServer DNS- или NetBIOS-имя удалённого сервера, на котором выполняется функция.
Если параметр пустой, используется локальный компьютер.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
[ 0] – указывает время, когда началась сбор статистики.
Значение хранится как количество прошедших секунд с 00:00:00, 1 января 1970 г., GMT.
[ 1] – указывает количество раз, когда файл открывается на сервере
[ 2] – указывает количество раз, когда устройство сервера открывается
[ 3] – указывает количество заданий печати сервера в очереди
[ 4] – указывает количество раз, когда сессия сервера начиналась
[ 5] – указывает количество раз, когда сессия сервера отключалась
[ 6] – указывает количество раз, когда сессии сервера завершались с ошибкой
[ 7] – указывает количество нарушений пароля сервера
[ 8] – указывает количество ошибок разрешения доступа сервера
[ 9] – указывает количество системных ошибок сервера
[10] – количество байтов сервера, отправленных в сеть
[11] – количество байтов сервера, полученных из сети
[12] – указывает среднее время ответа сервера (в миллисекундах)
[13] – указывает количество раз, когда серверу требовался буфер запроса, но он не смог его выделить
[14] – указывает количество раз, когда серверу требовался большой буфер, но он не смог его выделить
Ошибка: Устанавливает @error ≠ 0.

Примечания

Специального членства в группе не требуется для получения статистики рабочей станции.
Только члены локальной группы администраторов или операторов серверов могут успешно выполнить эту функцию на удалённом сервере.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <NetShare.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $aInfo

    ; Создать GUI
    GUICreate("NetShare", 400, 300)

    ; Создать элемент memo
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Получить статистику сервера
    $aInfo = _Net_Share_StatisticsGetSvr(@ComputerName)
    _MemoWrite("Statistics started ......: " & $aInfo[0])
    _MemoWrite("Times file opened .......: " & $aInfo[1])
    _MemoWrite("Times device opened .....: " & $aInfo[2])
    _MemoWrite("Print jobs spooled ......: " & $aInfo[3])
    _MemoWrite("Sessions started ........: " & $aInfo[4])
    _MemoWrite("Sessions disconnected ...: " & $aInfo[5])
    _MemoWrite("Session errors ..........: " & $aInfo[6])
    _MemoWrite("Password violations .....: " & $aInfo[7])
    _MemoWrite("Permission errors .......: " & $aInfo[8])
    _MemoWrite("Server system errors ....: " & $aInfo[9])
    _MemoWrite("Network bytes sent ......: " & $aInfo[10])
    _MemoWrite("Network bytes recv ......: " & $aInfo[11])
    _MemoWrite("Average response time ...: " & $aInfo[12])
    _MemoWrite("Req buffer failures .....: " & $aInfo[13])
    _MemoWrite("Big buffer failures .....: " & $aInfo[14])

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
