_GDIPlus_PathAddArc

Добавляет эллиптическую дугу к текущей фигуре пути.

#include <GDIPlus.au3>
_GDIPlus_PathAddArc ( $hPath, $nX, $nY, $nWidth, $nHeight, $fStartAngle, $fSweepAngle )

Параметры

$hPath Указатель на объект GraphicsPath.
$nX Координата X верхнего левого угла прямоугольника, описывающего эллипс с дугой.
$nY Координата Y верхнего левого угла прямоугольника, описывающего эллипс с дугой.
$nWidth Ширина описывающего прямоугольника для эллипса с дугой.
$nHeight Высота описывающего прямоугольника для эллипса с дугой.
$fStartAngle Угол в градусах между осью X и начальной точкой дуги.
$fSweepAngle Угол в градусах между начальной и конечной точками дуги.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Дополнительно

Искать GdipAddPathArc в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic, $hBrush, $hPen, $hPath

    ; Создать GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Создать графический объект из дескриптора окна
    _GDIPlus_GraphicsSetSmoothingMode($hGraphic, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;Задаёт качество отрисовки графики (сглаживание)
    _GDIPlus_GraphicsClear($hGraphic, 0xFFFFFFFF)

    $hBrush = _GDIPlus_BrushCreateSolid(0x7F8800AA)
    $hPen = _GDIPlus_PenCreate(0xFF8800AA, 2)

    $hPath = _GDIPlus_PathCreate() ;Создать новый объект path

    _GDIPlus_PathAddArc($hPath, 50, 50, 100, 100, 180, 180)
    _GDIPlus_PathAddArc($hPath, 50, 150, 100, 100, 0, 180)
    _GDIPlus_PathCloseFigure($hPath) ;соединить последнюю точку arc2 с первой точкой arc1

    _GDIPlus_PathAddArc($hPath, 250, 50, 100, 100, 180, 180)
    _GDIPlus_PathCloseFigure($hPath) ;не соединять дуги
    _GDIPlus_PathAddArc($hPath, 250, 150, 100, 100, 0, 180)
    _GDIPlus_PathCloseFigure($hPath) ;соединить последнюю точку arc2 с первой точкой arc2

    _GDIPlus_GraphicsFillPath($hGraphic, $hPath, $hBrush) ;Залить path в дескрипторе графики (GUI)
    _GDIPlus_GraphicsDrawPath($hGraphic, $hPath, $hPen) ;Нарисовать path в дескрипторе графики (GUI)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_PathDispose($hPath)
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_PenDispose($hPen)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
