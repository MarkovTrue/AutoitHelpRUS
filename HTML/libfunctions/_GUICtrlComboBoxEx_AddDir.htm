_GUICtrlComboBoxEx_AddDir

Добавляет имена каталогов и файлов.

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_AddDir ( $hWnd, $sFilePath [, $iAttributes = 0 [, $bBrackets = True]] )

Параметры

$hWnd Дескриптор элемента управления.
$sFilePath Абсолютный путь, относительный путь или имя файла.
$iAttributes [необязательный] Атрибуты добавляемых файлов или каталогов:
$DDL_READWRITE – включает обычные файлы (для чтения и записи) без дополнительных атрибутов
$DDL_READONLY – включает файлы только для чтения
$DDL_HIDDEN – включает скрытые файлы
$DDL_SYSTEM – включает системные файлы
$DDL_DIRECTORY – включает подкаталоги
$DDL_ARCHIVE – включает архивные файлы
$DDL_DRIVES – добавляет в список все подключённые диски
$DDL_EXCLUSIVE – включает только файлы с указанными атрибутами.
$bBrackets [необязательный] Включать или исключать скобки при использовании $DDL_DRIVES

Возвращаемое значение

Успех: Возвращает индекс последнего добавленного имени (нумерация с 0).
Ошибка: Возвращает -1.

Примечания

Если недостаточно памяти для сохранения новых строк, возвращаемое значение – $CB_ERRSPACE.

Вышеприведённые константы требуют #include <DirConstants.au3>

См. также

_GUICtrlComboBoxEx_InitStorage

Пример

#include <GuiComboBoxEx.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hCombo

    ; Создать GUI
    $hGUI = GUICreate("ComboBoxEx Add Dir", 400, 300, -1, -1, -1)
    $hCombo = _GUICtrlComboBoxEx_Create($hGUI, "", 2, 2, 394, 100)
    GUICtrlCreateInput("Input control", 2, 30, 120)
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBoxEx_EndUpdate($hCombo)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
