_WinAPI_GetApplicationRestartSettings

Получает информацию о перезапуске, зарегистрированную для указанного процесса.

#include <WinAPIDiag.au3>
_WinAPI_GetApplicationRestartSettings ( [$iPID = 0] )

Параметры

$iPID [необязательный] PID процесса. По умолчанию (0) – текущий процесс.

Возвращаемое значение

Успех: Возвращает массив со следующими данными:
$aArray[0] – командная строка перезапуска.
$aArray[1] – флаги.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Эти данные доступны только для текущего процесса; вызвать _WinAPI_GetApplicationRestartSettings() после перезапуска программы и получить её командную строку перезапуска нельзя.

Функция требует Windows Vista или новее.
