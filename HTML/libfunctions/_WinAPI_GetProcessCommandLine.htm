Справочник функций

_WinAPI_GetProcessCommandLine

Получает строку командной строки для указанного процесса.

#include <WinAPIProc.au3>
_WinAPI_GetProcessCommandLine ( [$iPID = 0] )

Параметры

$iPID [необязательный] PID процесса. По умолчанию (0) – текущий процесс.

Возвращаемое значение

Успех: Возвращает строка командной строки.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.

Примечания

Эта функция использует недокументированные API функции и может перестать работать должным образом в будущих версиях Windows.

Использование этой функции для некоторых процессов может потребовать полных прав доступа. Используйте _WinAPI_AdjustTokenPrivileges()
функцию для включения привилегии $SE_DEBUG_NAME перед вызовом этой функции.

См. также

_WinAPI_AdjustTokenPrivileges

Пример

#RequireAdmin

#include <Array.au3>
#include <SecurityConstants.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

Local $aAdjust, $aList = 0

; Включить привилегию SeDebugPrivilege для получения полного доступа к другим процессам
Local $hToken = _WinAPI_OpenProcessToken(BitOR($TOKEN_ADJUST_PRIVILEGES, $TOKEN_QUERY))

_WinAPI_AdjustTokenPrivileges($hToken, $SE_DEBUG_NAME, $SE_PRIVILEGE_ENABLED, $aAdjust)

; Получить аргументы командной строки для всех процессов в системе
If Not (@error Or @extended) Then
    $aList = ProcessList()
    For $i = 1 To $aList[0][0]
        $aList[$i][1] = _WinAPI_GetProcessCommandLine($aList[$i][1])
    Next
EndIf

; Включить привилегию SeDebugPrivilege по умолчанию
_WinAPI_AdjustTokenPrivileges($hToken, $aAdjust, 0, $aAdjust)
_WinAPI_CloseHandle($hToken)

_ArrayDisplay($aList, '_WinAPI_GetProcessCommandLine')
