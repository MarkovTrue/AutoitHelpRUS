_GUICtrlRebar_AddBand

Добавляет новую полосу в элемент Rebar.

#include <GuiReBar.au3>
_GUICtrlRebar_AddBand ( $hWndRebar, $hWndChild [, $iMinWidth = 100 [, $iDefaultWidth = 100 [, $sText = "" [, $iIndex = -1 [, $iStyle = -1]]]]] )

Параметры

$hWndRebar Дескриптор элемента управления Rebar.
$hWndChild Дескриптор элемента управления для добавления.
$iMinWidth [необязательный] Минимальная ширина полосы
$iDefaultWidth [необязательный] Длина полосы, в пиксах
$sText [необязательный] Текст отображаемый на полосе
$iIndex [необязательный] Индекс позиции, в которую будет вставлена полоса (отсчёт от 0).
Если задать значение –1, новая полоса будет добавлена в конец.
$iStyle [необязательный] Флаги стиля полосы. Может быть комбинацией следующих:
$RBBS_BREAK – полоса на новой строке.
$RBBS_CHILDEDGE – у полосы есть рамка в верхней и нижней части дочернего окна.
$RBBS_FIXEDBMP – фоновое изображение не смещается при изменении размера полосы.
$RBBS_FIXEDSIZE – размер полосы неизменяем; рычаг изменения размера не показывается.
$RBBS_GRIPPERALWAYS – версия 4.71. Полоса всегда имеет рычаг изменения размера, даже если это единственная полоса.
$RBBS_HIDDEN – полоса не видна.
$RBBS_NOGRIPPER – версия 4.71. Полоса никогда не имеет рычаг изменения размера.
$RBBS_USECHEVRON – версия 5.80. Показывать кнопку шеврона если полоса меньше cxIdeal.
$RBBS_VARIABLEHEIGHT – версия 4.71. Размер полосы изменяется Rebar; cyIntegral и cyMaxChild влияют на изменение размера.
$RBBS_NOVERT – не показывать при вертикальной ориентации.
$RBBS_USECHEVRON – отображать кнопку раскрывающегося списка.
$RBBS_HIDETITLE – скрывать заголовок полосы.
$RBBS_TOPALIGN – размещать полосу в верхней строке.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_GUICtrlRebar_AddToolBarBand, _GUICtrlRebar_DeleteBand

Пример

#include <AutoItConstants.au3>
#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>
#include <GuiDateTimePicker.au3>
#include <GuiReBar.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hGui, $idBtn_Exit, $hCombo, $hReBar, $hDTP, $idInput

    $hGui = GUICreate("Rebar", 400, 396, -1, -1, BitOR($WS_MINIMIZEBOX, $WS_CAPTION, $WS_POPUP, $WS_SYSMENU, $WS_MAXIMIZEBOX))

    ; создать элемент rebar
    $hReBar = _GUICtrlRebar_Create($hGui, BitOR($CCS_TOP, $WS_BORDER, $RBS_VARHEIGHT, $RBS_AUTOSIZE, $RBS_BANDBORDERS))

    ; создать combobox для размещения в rebar
    $hCombo = _GUICtrlComboBox_Create($hGui, "", 0, 0, 120)

    _GUICtrlComboBox_BeginUpdate($hCombo)
    _GUICtrlComboBox_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($hCombo)

    ; создание средства выбора даты/времени для размещения в Rebar
    $hDTP = _GUICtrlDTP_Create($hGui, 0, 0, 190)

    ; создать поле ввода для размещения в rebar
    $idInput = GUICtrlCreateInput("Input control", 0, 0, 120, 20)

    ; добавить полосу с элементом управления
    _GUICtrlRebar_AddBand($hReBar, $hCombo, 120, 200, "Dir *.exe")

    ; добавить и разместить на второй строке
    _GUICtrlRebar_AddBand($hReBar, $hDTP, 120)
    _GUICtrlRebar_SetBandStyleBreak($hReBar, 1)

    ; добавить в начало Rebar
    _GUICtrlRebar_AddBand($hReBar, GUICtrlGetHandle($idInput), 120, 200, "Name:", 0)

    $idBtn_Exit = GUICtrlCreateButton("Exit", 150, 360, 100, 25)
    GUISetState(@SW_SHOW)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idBtn_Exit
                Exit
        EndSwitch
    WEnd
EndFunc   ;==>Example
