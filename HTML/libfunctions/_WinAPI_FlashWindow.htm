_WinAPI_FlashWindow

Мигает указанным окном один раз.

#include <WinAPISysWin.au3>
_WinAPI_FlashWindow ( $hWnd [, $bInvert = True] )

Параметры

$hWnd Дескриптор окна.
$bInvert [необязательный] Если True, окно переключается между двумя состояниями подсветки.
Если False, окно возвращается в исходное состояние.
Когда приложение свёрнуто и параметр равен True, кнопка окна на панели задач мигает активным/неактивным цветом.
При False кнопка мигает в неактивном виде – цвета не меняются;
кнопка мигает как при перерисовке, но визуального инвертирования пользователю не показывается.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция не меняет активное состояние окна.
Чтобы мигать окном заданное число раз, используйте FlashWindowEx.

См. также

_WinAPI_FlashWindowEx

Дополнительно

Искать FlashWindow в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <WinAPISysWin.au3>

Example()

Func Example()
    Local $hWnd, $iMsg, $idBtn_Flash
    $hWnd = GUICreate("_WinAPI_FlashWindow Example", 200, 200)
    $idBtn_Flash = GUICtrlCreateButton("Flash Window", 50, 85, 100, 30)
    GUISetState(@SW_SHOW)
    While 1
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                Exit
            Case $iMsg = $idBtn_Flash
                GUISetState(@SW_MINIMIZE)
                Sleep(1000)
                _WinAPI_FlashWindow($hWnd)
                Sleep(6000)
        EndSelect
    WEnd
EndFunc   ;==>Example
