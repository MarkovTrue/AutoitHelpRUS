Справочник функций

_GDIPlus_CustomLineCapDispose

Освобождает объект пользовательского наконечника линии.

#include <GDIPlus.au3>
_GDIPlus_CustomLineCapDispose ( $hCap )

Параметры

$hCap Дескриптор объекта пользовательского наконечника линии.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).
