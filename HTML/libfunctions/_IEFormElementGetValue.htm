Справочник функций

_IEFormElementGetValue

Возвращает значение указанного элемента формы.

#include <IE.au3>
_IEFormElementGetValue ( ByRef $oObject )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.

Возвращаемое значение

Успех: Возвращает строковое значение указанного Form Element.
Ошибка: Устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
@extended: Содержит номер недопустимого параметра.

Примечания

См. раздел Примечания к _IEFormElementSetValue().

См. также

_IEFormElementCheckBoxSelect, _IEFormElementGetCollection, _IEFormElementGetObjByName, _IEFormElementOptionSelect, _IEFormElementRadioSelect, _IEFormElementSetValue

Пример

; Открытие браузера с примером формы, установка значения текстового
; элемент формы, получить и отобразить значение элемента

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oText = _IEFormElementGetObjByName($oForm, "textExample")
Local $aIEAu3Version = _IE_VersionInfo()
_IEFormElementSetValue($oText, $aIEAu3Version[5])
MsgBox($MB_SYSTEMMODAL, "Form Element Value", _IEFormElementGetValue($oText))

_IEQuit($oIE)
