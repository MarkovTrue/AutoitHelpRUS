_WinAPI_GetTextExtentPoint32

Вычисляет ширину и высоту указанной строки текста.

#include <WinAPIGdi.au3>
_WinAPI_GetTextExtentPoint32 ( $hDC, $sText )

Параметры

$hDC Определяет контекст устройства.
$sText Строка текста.

Возвращаемое значение

Успех: Структура $tagSIZE, в которой должны быть возвращены размеры строки.
Ошибка: Устанавливает @error ≠ 0.

См. также

$tagSIZE

Дополнительно

Искать GetTextExtentPoint32 в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <SendMessage.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WindowsNotifsConstants.au3>

Example()

Func Example()
    Local $hGUI = GUICreate("String height & width.")
    GUISetState(@SW_SHOW, $hGUI)

    Local $sText = "This is some text" ; Текст, для которого мы хотим найти высоту и ширину.
    Local $aStringDimension = GetStringDimensions($hGUI, $sText) ; Получить одномерный массив с $aArray[0] = ширина и $aArray[1] = высота.
    MsgBox($MB_SYSTEMMODAL, "String Height and Width", 'The height and width of the string "' & $sText & '" is:' & @CRLF & _
            "Width: " & $aStringDimension[0] & @CRLF & _
            "Height: " & $aStringDimension[1] & @CRLF)
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

Func GetStringDimensions($hWnd, $sText)
    Local $hDC = _WinAPI_GetDC($hWnd) ; Получить дескриптор контекста устройства текущего окна.
    Local $hFont = _SendMessage($hWnd, $WM_GETFONT) ; Получить шрифт, которым элемент управления в настоящее время рисует свой текст.
    Local $hSelectObject = _WinAPI_SelectObject($hDC, $hFont) ; Выбрать объект контекста устройства.
    Local $tSIZE = _WinAPI_GetTextExtentPoint32($hDC, $sText) ; Получить высоту и ширину строки.

    _WinAPI_SelectObject($hDC, $hSelectObject)
    _WinAPI_ReleaseDC($hWnd, $hDC) ; Освободить контекст устройства.
    Local $aReturn[2] = [DllStructGetData($tSIZE, 1), DllStructGetData($tSIZE, 2)] ; Установить массив с шириной и высотой строки.
    Return $aReturn
EndFunc   ;==>GetStringDimensions
