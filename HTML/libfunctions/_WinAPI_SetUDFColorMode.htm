Справочник функций

_WinAPI_SetUDFColorMode

Задаёт режим цвета для библиотеки WinAPIEx.

#include <WinAPIGdi.au3>
_WinAPI_SetUDFColorMode ( $iMode )

Параметры

$iMode Режим цвета. Этот параметр может быть одним из следующих значений.
$UDF_BGR
$UDF_RGB

Возвращаемое значение

Отсутствует.

Примечания

Функции этой библиотеки, в которых передаются или возвращаются значения цветов, работают со значениями в соответствии с режимом цвета, установленным этой функцией. Начальный режим цвета RGB.
