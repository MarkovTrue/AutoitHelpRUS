Справочник функций

_ClipBoard_Open

Открывает буфер обмена и запрещает другим приложениям его изменять.

#include <Clipboard.au3>
_ClipBoard_Open ( $hOwner )

Параметры

$hOwner Дескриптор окна, связываемого с открытым буфером обмена. Если параметр равен 0, буфер обмена связывается с текущей задачей.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция завершается ошибкой, если другое окно уже открыло буфер обмена. После каждого успешного вызова этой функции необходимо вызывать _ClipBoard_Close().
Окно, указанное в $hOwner, не становится владельцем буфера обмена, пока не будет вызвана _ClipBoard_Empty(). Если при вызове _ClipBoard_Open() передан hwnd=0, то _ClipBoard_Empty() установит владельца буфера в 0, что приведёт к ошибке _ClipBoard_SetData().

См. также

_ClipBoard_Close, _ClipBoard_Empty, _ClipBoard_EnumFormats, _ClipBoard_GetOpenWindow, _ClipBoard_SetDataEx

Пример

#include "Extras\HelpFileInternals.au3"

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIError.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hGUI

    ; Создать GUI
    $hGUI = GUICreate("Clipboard", 600, 400)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Открыть буфер обмена
    If _ClipBoard_Open($hGUI) Then

        ShowData($hGUI)

        ; Закрыть буфер обмена
        _ClipBoard_Close()
    Else
        _WinAPI_ShowError("_ClipBoard_Open failed")
    EndIf

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Показать статистику буфера обмена
Func ShowData($hGUI)
    _MemoWrite("GUI handle ............: " & $hGUI)
    _MemoWrite("Clipboard owner .......: " & _ClipBoard_GetOwner())
    _MemoWrite("Clipboard open window .: " & _ClipBoard_GetOpenWindow())
    _MemoWrite("Clipboard sequence ....: " & _ClipBoard_GetSequenceNumber())
    _MemoWrite()
EndFunc   ;==>ShowData
