Справочник функций

_Crypt_DeriveKey

Создаёт криптографический ключ на основе алгоритма и пароля.

#include <Crypt.au3>
_Crypt_DeriveKey ( $vPassword, $iAlgID [, $iHashPasswordID = $CALG_MD5] )

Параметры

$vPassword Используемый пароль.
$iAlgID Идентификатор алгоритма шифрования для ключа. См. раздел Примечания.
$iHashPasswordID [необязательный] Идентификатор алгоритма хеширования пароля. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает дескриптор криптографического ключа.
Ошибка: Устанавливает @error ≠ 0.
@error: 10+ – не удалось создать объект хеша
20+ – ошибка хеширования пароля
30+ – ошибка генерации ключа
1000+ – ошибка _Crypt_Startup()

Примечания

Созданный ключ необходимо уничтожить вызовом _Crypt_DestroyKey().

    Идентификатор алгоритма шифрования

$iAlgID Значение Тип Длина ключа
$CALG_AES_128 0x0000660e Шифрование с CBC 128 бит
$CALG_AES_192 0x0000660f Шифрование с CBC 192 бита
$CALG_AES_256 0x00006610 Шифрование с CBC 256 бит
$CALG_DES 0x00006601 Шифрование с CBC 56 бит
$CALG_3DES 0x00006603 Шифрование с CBC 168 бит
$CALG_RC2 0x00006602 Блочный алгоритм шифрования 128 бит
$CALG_RC4 0x00006801 Потоковый алгоритм шифрования 128 бит
$CALG_USERKEY 0

    Идентификатор алгоритма хеширования

$iHashPasswordID Значение Тип Длина хеша
$CALG_MD2 0x00008001 Алгоритм хеширования 128 бит
$CALG_MD4 0x00008002 Алгоритм хеширования 128 бит
$CALG_MD5 0x00008003 Алгоритм хеширования 128 бит
$CALG_SHA1 0x00008004 Алгоритм хеширования 160 бит
$CALG_SHA_256 0x0000800c Алгоритм хеширования 256 бит
$CALG_SHA_384 0x0000800d Алгоритм хеширования 384 бита
$CALG_SHA_512 0x0000800e Алгоритм хеширования 512 бит

Пример

#include <Crypt.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aStringsToEncrypt[6] = ["AutoIt", "SciTE", "Crypt", ".au3", 42, "42"]
    Local $sOutput = ""

    Local $hKey = _Crypt_DeriveKey("CryptPassword", $CALG_RC4) ; Объявить парольную строку и алгоритм для создания криптографического ключа.

    For $vText In $aStringsToEncrypt
        $sOutput &= $vText & @TAB & " = " & _Crypt_EncryptData($vText, $hKey, $CALG_USERKEY) & @CRLF ; Зашифровать текст криптографическим ключом.
    Next

    MsgBox($MB_SYSTEMMODAL, "Encrypted data", $sOutput)

    _Crypt_DestroyKey($hKey) ; Уничтожить криптографический ключ.
EndFunc   ;==>Example
