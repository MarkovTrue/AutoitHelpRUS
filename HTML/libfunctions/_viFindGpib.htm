Справочник функций

_viFindGpib

Отправляет команду или запрос на прибор или устройство через интерфейс VISA (GPIB / TCP).

#include <Visa.au3>
_viFindGpib ( ByRef $aDescriptorList, ByRef $aIDNList [, $iShow_Search_Results = 0] )

Параметры

$aDescriptorList ВОЗВРАЩАЕТ массив дескрипторов ресурсов VISA приборов, найденных на шине GPIB (см. раздел Примечания функции _viExecCommand() для дополнительной информации).
$aIDNList ВОЗВРАЩАЕТ массив IDN (т. е. имён) приборов, найденных на шине GPIB.
$iShow_Search_Results [необязательный] Если 1, то будет показано диалоговое окно с результатами поиска.
По умолчанию 0, что означает, что результаты не показываются.

Возвращаемое значение

Успех: Возвращает количество найденных приборов (0 или более).
Ошибка: отрицательное значение и устанавливает флаг @error в ненулевое значение.

Примечания

Как и для всех функций VISA, библиотеки VISA должны быть установлены (вы можете проверить, находится ли visa32.dll в {WINDOWS}\system32), а также должна быть установлена карта GPIB (такая как NI PCI-GPIB от National Instruments или Agilent 82350B PCI High-Performance GPIB)

* Подробное описание наиболее распространённых ДЕСКРИПТОРОВ VISA см. в примечаниях функции _viExecCommand()

См. также

_viClose, _viExecCommand, _viGpibBusReset, _viGTL, _viOpen

Пример

; Этот пример выполняет поиск на шине GPIB и отображает результаты в MsgBox

#include <Visa.au3>

Local $a_Descriptor_list[1], $a_Idn_list[1]
_viFindGpib($a_Descriptor_list, $a_Idn_list, 1)
