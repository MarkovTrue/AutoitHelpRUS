_WinAPI_DuplicateTokenEx

Создаёт новый первичный или олицетворяющий токен доступа, дублирующий существующий токен.

#include <WinAPIProc.au3>
_WinAPI_DuplicateTokenEx ( $hToken, $iAccess, $iLevel [, $iType = $TOKENPRIMARY [, $tSecurity = 0]] )

Параметры

$hToken Дескриптор токена доступа, открытого с правом $TOKEN_DUPLICATE.
$iAccess Запрашиваемые права доступа для нового токена. Чтобы запросить те же права, что и у исходного токена, передайте 0.
Чтобы запросить все доступные вызывающей стороне права, укажите $TOKEN_ALL_ACCESS.
$iLevel Уровень имперсонации безопасности.
$SECURITYANONYMOUS
$SECURITYIDENTIFICATION
$SECURITYIMPERSONATION
$SECURITYDELEGATION
$iType [необязательный] Первое слово предложения token type.
$TOKENPRIMARY (Default)
$TOKENIMPERSONATION
$tSecurity [необязательный] Структура $tagSECURITY_ATTRIBUTES, задающая дескриптор безопасности для нового токена и определяющая, могут ли дочерние процессы наследовать этот токен.
Если параметр равен 0 (по умолчанию), токен получает дескриптор безопасности по умолчанию, а его дескриптор не может быть унаследован.
Если дескриптор безопасности содержит системный список контроля доступа, токен получает право доступа $ACCESS_SYSTEM_SECURITY, даже если оно не было запрошено.

Возвращаемое значение

Успех: Возвращает первое слово предложения дескриптор к new token.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Когда новый токен больше не нужен, вызовите функцию _WinAPI_CloseHandle() для закрытия дескриптора токена.
