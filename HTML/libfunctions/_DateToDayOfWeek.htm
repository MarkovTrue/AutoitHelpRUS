Справочник функций

_DateToDayOfWeek

Возвращает номер дня недели для указанной даты.

#include <Date.au3>
_DateToDayOfWeek ( $iYear, $iMonth, $iDay )

Параметры

$iYear Корректный год в формате ГГГГ.
$iMonth Корректный месяц в формате ММ.
$iDay Корректный день в формате ДД.

Возвращаемое значение

Успех: Возвращает номер дня недели в диапазоне от 1 до 7, где 1 = воскресенье.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – Недопустимая входная дата.

См. также

_DateAdd, _DateDayOfWeek, _DateDiff, _DateToDayOfWeekISO, _DayValueToDate

Пример

#include <Date.au3>
#include <MsgBoxConstants.au3>

; Номер дня недели для заданной даты.
Local $iWeekday = _DateToDayOfWeek(@YEAR, @MON, @MDAY)
; Должно быть равно @WDAY
MsgBox($MB_SYSTEMMODAL, "", "Todays WeekdayNumber is: " & $iWeekday)
MsgBox($MB_SYSTEMMODAL, "", "Today is a : " & _DateDayOfWeek($iWeekday))
