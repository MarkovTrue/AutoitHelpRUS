Справочник функций

_WinAPI_CreateRectEx

создаёт структуру $tagRECT с координатами указанного прямоугольника.

#include <WinAPIMisc.au3>
_WinAPI_CreateRectEx ( $iX, $iY, $iWidth, $iHeight )

Параметры

$iX Координата X левого верхнего угла прямоугольника.
$iY Координата Y левого верхнего угла прямоугольника.
$iWidth Ширина прямоугольника.
$iHeight Высота прямоугольника.

Возвращаемое значение

Возвращает структуру $tagRECT, содержащую указанный прямоугольник.
