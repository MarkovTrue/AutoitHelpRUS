Справочник функций

_Security__AdjustTokenPrivileges

Включает или отключает права доступа в указанном маркере доступа.

#include <Security.au3>
_Security__AdjustTokenPrivileges ( $hToken, $bDisableAll, $tNewState, $iBufferLen [, $tPrevState = 0 [, $pRequired = 0]] )

Параметры

$hToken Дескриптор маркера доступа, содержащего права доступа, которые должны быть изменены.
$bDisableAll Если True, функция отключает все права доступа и игнорирует параметр NewState.
Если False, функция изменяет права доступа на основе информации, на которую указывает параметр $pNewState.
$tNewState Структура $tagTOKEN_PRIVILEGES или указатель на неё, содержащая право доступа и его атрибуты.
$iBufferLen Размер буфера в байтах, на который указывает $pNewState.
$tPrevState [необязательный] Структура $tagTOKEN_PRIVILEGES или указатель на неё, которая указывает на предыдущее состояние права доступа, изменённое функцией. Может быть 0.
$pRequired [необязательный] Указатель на переменную, которая получает требуемый размер буфера в байтах, на который указывает $tPrevState.
Этот параметр может быть 0, если $tPrevState равен 0.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция не может добавить новые права доступа к маркеру доступа. Она может только включить или отключить существующие права маркера.

См. также

$tagTOKEN_PRIVILEGES

Пример

#RequireAdmin ; чтобы пример имел смысл

#include <MsgBoxConstants.au3>
#include <Security.au3>
#include <WinAPIHObj.au3>

Local $hToken = _Security__OpenProcessToken(_WinAPI_GetCurrentProcess(), $TOKEN_ALL_ACCESS)
If $hToken Then
    ; $hToken – маркер этого процесса с доступом $TOKEN_ALL_ACCESS

    ; Отключить все права доступа для этого маркера
    If _Security__AdjustTokenPrivileges($hToken, True, 0, 0) Then
        ;... Выполнить любые действия с этим токеном...
        MsgBox($MB_SYSTEMMODAL, "TokenPrivileges", "All TokenPrivileges disabled!")
    EndIf

    ; Закрыть дескриптор по завершении
    _WinAPI_CloseHandle($hToken)
EndIf
