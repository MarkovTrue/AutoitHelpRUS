Справочник функций

_GUICtrlToolbar_GetStyle

Возвращает стили, используемые в настоящее время для элемента управления панели инструментов.

#include <GuiToolbar.au3>
_GUICtrlToolbar_GetStyle ( $hWnd )

Параметры

$hWnd Дескриптор элемента управления.

Возвращаемое значение

Возвращает стили элемента – сочетание следующих значений:
    $TBSTYLE_TOOLTIPS – создаёт элемент всплывающих подсказок
    $TBSTYLE_WRAPABLE – панель может иметь несколько строк кнопок
    $TBSTYLE_ALTDRAG – позволяет пользователю менять позицию кнопок перетаскиванием
    $TBSTYLE_FLAT – плоская панель инструментов
    $TBSTYLE_LIST – плоская панель с текстом кнопок справа от изображения
    $TBSTYLE_CUSTOMERASE – отправляет $NM_CUSTOMDRAW при обработке $WM_ERASEBKGND
    $TBSTYLE_REGISTERDROP – отправляет $TBN_GETOBJECT для запроса объектов-приёмников
    $TBSTYLE_TRANSPARENT – прозрачная панель инструментов

См. также

_GUICtrlToolbar_SetStyle

Пример

#include "Extras\HelpFileInternals.au3"

#include <AutoItConstants.au3>
#include <GUIConstantsEx.au3>
#include <GuiToolbar.au3>
#include <WinAPIConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hGUI, $hToolbar
    Local Enum $e_idNew = 1000, $e_idOpen, $e_idSave, $e_idHelp

    ; Создать GUI
    $hGUI = GUICreate("Toolbar", 400, 300)
    $hToolbar = _GUICtrlToolbar_Create($hGUI)
    _MemoCreate(2, 36, 396, 262, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Установить стиль элемента управления
    _GUICtrlToolbar_SetStyle($hToolbar, BitOR($TBSTYLE_LIST, $CCS_TOP))

    ; Добавить стандартные системные изображения
    Switch _GUICtrlToolbar_GetBitmapFlags($hToolbar)
        Case 0
            _GUICtrlToolbar_AddBitmap($hToolbar, 1, -1, $IDB_STD_SMALL_COLOR)
        Case 2
            _GUICtrlToolbar_AddBitmap($hToolbar, 1, -1, $IDB_STD_LARGE_COLOR)
    EndSwitch

    ; Добавить кнопки
    _GUICtrlToolbar_AddButton($hToolbar, $e_idNew, $STD_FILENEW)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idOpen, $STD_FILEOPEN)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idSave, $STD_FILESAVE)
    _GUICtrlToolbar_AddButtonSep($hToolbar)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idHelp, $STD_HELP)

    ; Показать стиль элемента управления
    _MemoWrite("Toolbar style .: 0x" & Hex(_GUICtrlToolbar_GetStyle($hToolbar)))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
