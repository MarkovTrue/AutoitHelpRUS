Справочник функций

_WinAPI_DwmEnableBlurBehindWindow

Включает эффект размытия для указанного окна.

#include <WinAPIGdi.au3>
_WinAPI_DwmEnableBlurBehindWindow ( $hWnd [, $bEnable = True [, $bTransition = False [, $hRgn = 0]]] )

Параметры

$hWnd Дескриптор окна.
$bEnable [необязательный] Регистрация или снятие регистрации дескриптора окна для размытия фона DWM:
True    – зарегистрировать (по умолчанию).
False – снять регистрацию.
$bTransition [необязательный] Подкрашивать ли переход в тон развёрнутых окон:
True    – подкрашивать.
False – не подкрашивать (по умолчанию).
$hRgn [необязательный] Регион клиентской области, к которому применяется размытие фона.
Значение 0 (по умолчанию) применяет размытие ко всей клиентской области.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Альфа-значения в окне сохраняются, и итоговая прорисовка поверх размытия использует эти альфа-значения.
Корректность альфа-значений всех пикселей окна – зона ответственности приложения.
Некоторые операции Windows GDI могут устанавливать альфа в 0; учитывайте это, когда работаете с размытием.

Дополнительно

Искать DwmEnableBlurBehindWindow в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIHObj.au3>
#include <WinAPIMisc.au3>

If Not _WinAPI_DwmIsCompositionEnabled() Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later with enabled Aero theme.')
    Exit
EndIf

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400)

GUISetBkColor(0)

; Создание эффекта "стеклянного листа" для указанной области. Эту функцию необходимо вызывать при каждом переключении композиции DWM.
_WinAPI_DwmGetColorizationColor()
Local $hRgn = _WinAPI_CreateEllipticRgn(_WinAPI_CreateRectEx(50, 50, 300, 300))
_WinAPI_DwmEnableBlurBehindWindow($hForm, 1, 0, $hRgn)
If $hRgn Then
    _WinAPI_DeleteObject($hRgn)
EndIf

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
