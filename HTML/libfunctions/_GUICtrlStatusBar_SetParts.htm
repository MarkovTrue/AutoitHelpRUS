Справочник функций

_GUICtrlStatusBar_SetParts

Устанавливает значение.

#include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetParts ( $hWnd [, $vPartEdge = -1 [, $vPartWidth = 25]] )

Параметры

$hWnd Дескриптор элемента управления.
$vPartEdge [необязательный] Количество частей, либо массив целых чисел с отсчётом от 0 в виде:
$vPartEdge[0] – правая граница части #1
$vPartEdge[n] – правая граница части #n
$vPartWidth [необязательный] Ширина элемента управления.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Если элемент равен -1, правая граница соответствующей части продлевается до края окна.

$vPartWidth игнорируется, если $vPartEdge – массив.
$vPartEdge и $vPartWidth заранее не должны превышать число частей строки состояния.

См. также

_GUICtrlStatusBar_GetParts

Пример

Пример 1

; == Пример 1 : с ширинами по умолчанию

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiStatusBar.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("StatusBar Set Parts (v" & @AutoItVersion & ")", 450, 320, 100, 100)
    Local $hStatus = _GUICtrlStatusBar_Create($hGUI)

    ; Создать элемент memo
    _MemoCreate(2, 8, 444, 259)
    GUISetState(@SW_SHOW)

    ; Установить/получить части
    Local $aParts[3] = [75, 150, -1]
    _GUICtrlStatusBar_SetParts($hStatus, $aParts)

    ;Установить текст / получить ширину
    Local $iParts = _GUICtrlStatusBar_GetCount($hStatus)
    For $iI = 0 To $iParts - 1
        _GUICtrlStatusBar_SetText($hStatus, "Text " & $iI, $iI)
        _MemoWrite("Part " & $iI & " width .: " & _GUICtrlStatusBar_GetWidth($hStatus, $iI))
    Next

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
