Справочник функций

_Security__GetLengthSid

Возвращает длину действительного SID в байтах.

#include <Security.au3>
_Security__GetLengthSid ( $pSID )

Параметры

$pSID Указатель на SID.

Возвращаемое значение

Успех: Возвращает длина SID.
Ошибка: Возвращает 0.

См. также

_Security__IsValidSid

Дополнительно

Искать GetLengthSid в библиотеке MSDN

Пример

#include <Security.au3>

Local $tSID = _Security__GetAccountSid(@UserName)
If Not @error Then
    ; $tSID – структура с SID-данными указанной учётной записи
    ; Проверить его длину:
    Local $iLength = _Security__GetLengthSid($tSID)
    ConsoleWrite("The length of SID is: " & $iLength & " bytes" & @CRLF)

    ; ... Остальная часть скрипта здесь...
EndIf
