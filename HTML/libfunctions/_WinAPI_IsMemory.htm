_WinAPI_IsMemory

Определяет, указывает ли указанный указатель на блок памяти во внутренней куче библиотеки.

#include <WinAPIMem.au3>
_WinAPI_IsMemory ( $pMemory )

Параметры

$pMemory Указатель на блок памяти для проверки.

Возвращаемое значение

True: Возвращает указанный указатель является допустимым указателем памяти.
False: указанный указатель не является допустимым указателем памяти.

Примечания

Функция _WinAPI_IsMemory() принимает только указатели, возвращаемые некоторыми функциями _WinAPI_* (см. описание),
в противном случае всегда возвращает False.
