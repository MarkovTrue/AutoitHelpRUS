_WinNet_GetLastError

Получает самую свежую расширенную ошибку.

#include <WinNet.au3>
_WinNet_GetLastError ( ByRef $iError, ByRef $sError, ByRef $sName )

Параметры

$iError При возврате содержит самый свежий код расширенной ошибки.
$sError При возврате содержит сообщение кода самой свежей расширенной ошибки.
$sName При возврате сетевой поставщик, который вызвал ошибку.

Возвращаемое значение

Успех: Возвращает сообщение самой свежей ошибки.
Ошибка: Устанавливает @error ≠ 0.
