Справочник функций

_WinAPI_ShellQueryUserNotificationState

Проверяет состояние компьютера для текущего пользователя.

#include <WinAPIShellEx.au3>
_WinAPI_ShellQueryUserNotificationState ( )

Возвращаемое значение

Успех: Возвращает текущее состояние компьютера ($QUNS_*).
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
