Справочник функций

_GUICtrlEdit_BeginUpdate

Блокирует обновление элемента управления до вызова функции _GUICtrlEdit_EndUpdate().

#include <GuiEdit.au3>
_GUICtrlEdit_BeginUpdate ( $hWnd )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_GUICtrlEdit_EndUpdate

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <GuiStatusBar.au3>

Example()

Func Example()
    Local $idEdit, $hStatusBar, $hGUI
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\include\_ReadMe_.txt"

    ; Создать GUI
    $hGUI = GUICreate("Edit Begin Update", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268)
    $hStatusBar = _GUICtrlStatusBar_Create($hGUI, -1)
    GUISetState(@SW_SHOW)

    _GUICtrlEdit_BeginUpdate($idEdit)
    _GUICtrlEdit_SetText($idEdit, FileRead($sFile))
    _GUICtrlEdit_EndUpdate($idEdit)

    _GUICtrlStatusBar_SetText($hStatusBar, @TAB & "Lines: " & _GUICtrlEdit_GetLineCount($idEdit))
    _GUICtrlStatusBar_SetIcon($hStatusBar, 0, 97, "shell32.dll")

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
