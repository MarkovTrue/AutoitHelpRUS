Справочник функций

_Security__OpenProcessToken

Возвращает токен доступа, связанный с процессом.

#include <Security.au3>
_Security__OpenProcessToken ( $hProcess, $iAccess )

Параметры

$hProcess Дескриптор процесса, чей токен доступа открывается.
Процесс должен был получить разрешение на доступ $PROCESS_QUERY_INFORMATION.
$iAccess Указывает маску доступа, определяющую запрашиваемые типы доступа к токену доступа.

Возвращаемое значение

Успех: Возвращает дескриптор, идентифицирующий вновь открытый токен доступа при возврате функции.
Ошибка: Возвращает 0.

Примечания

Закройте дескриптор токена доступа, вернув функцию _WinAPI_CloseHandle().

См. также

_Security__OpenThreadToken

Дополнительно

Искать OpenProcessToken в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <Security.au3>
#include <WinAPIHObj.au3>

Local $hToken = _Security__OpenProcessToken(_WinAPI_GetCurrentProcess(), $TOKEN_QUERY)
If $hToken Then
    ; <em>$hToken</em> – токен этого процесса с доступом <em>$TOKEN_QUERY</em>

    ; ... Выполнить любые действия с этим токеном...
    MsgBox($MB_SYSTEMMODAL, "OpenProcessToken", "$hToken = " & $hToken)

    ; Закрыть дескриптор по завершении
    _WinAPI_CloseHandle($hToken)
EndIf
