Справочник ключевых слов

$tagNMREBAR

Содержит сведения для обработки различных уведомляющих сообщений rebar.

Global Const $tagNMREBAR = $tagNMHDR & ";dword dwMask;uint uBand;uint fStyle;uint wID;lparam lParam"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
dwMask Набор флагов, определяющих, какие элементы этой структуры содержат действительную информацию. Это может быть одно или несколько из следующих значений:
$RBNM_ID – элемент wID содержит действительную информацию.
$RBNM_LPARAM – элемент lParam содержит действительную информацию.
$RBNM_STYLE – элемент fStyle содержит действительную информацию.
uBand Индекс с отсчётом от нуля полосы, затронутой уведомлением. Это будет -1, если ни одна полоса не затронута.
fStyle Стиль полосы. Это один или несколько стилей $RBBS_, подробно описанных в элементе fStyle структуры $tagREBARBANDINFO.
Этот элемент действителен только если dwMask содержит $RBNM_STYLE.
wID Определённый приложением идентификатор полосы. Этот элемент действителен только если dwMask содержит $RBNM_ID.
lParam Определённое приложением значение, связанное с полосой. Этот элемент действителен только если dwMask содержит $RBNM_LPARAM.
