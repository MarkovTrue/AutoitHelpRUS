_GDIPlus_BitmapSetResolution

Задаёт разрешение объекта Bitmap.

#include <GDIPlus.au3>
_GDIPlus_BitmapSetResolution ( $hBitmap, $fDpiX, $fDpiY )

Параметры

$hBitmap Указатель на объект Bitmap.
$fDpiX Горизонтальное разрешение в точках на дюйм.
$fDpiY Вертикальное разрешение в точках на дюйм.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_ImageGetHorizontalResolution, _GDIPlus_ImageGetVerticalResolution

Дополнительно

Искать GdipBitmapSetResolution в библиотеке MSDN

#include <GUIConstantsEx.au3>

Func Example($sFileName, $fDPI, $fPageW = 210, $fPageH = 297)
    _GDIPlus_Startup()

    Local $iBMP_W = Ceiling($fPageW / 25.4 * $fDPI) ;Вычислить размер изображения в пикселях.
    Local $iBMP_H = Ceiling($fPageH / 25.4 * $fDPI)
    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0($iBMP_W, $iBMP_H)
    Local $hContext = _GDIPlus_ImageGetGraphicsContext($hBitmap)
    _GDIPlus_GraphicsClear($hContext, 0xFFFFFFFF)

    _GDIPlus_BitmapSetResolution($hBitmap, $fDPI, $fDPI) ;Установить разрешение DPI.

    _GDIPlus_GraphicsDrawString($hContext, StringFormat("%.1f x %.1f mm\n%.1f x %.1f in\n%.1f DPI", $fPageW, $fPageH, $fPageW / 25.4, $fPageH / 25.5, $fDPI), 20, 20, "ARIAL", 48)

    _GDIPlus_ImageSaveToFile($hBitmap, $sFileName)

    _GDIPlus_GraphicsDispose($hContext)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
