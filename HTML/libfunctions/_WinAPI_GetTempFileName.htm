_WinAPI_GetTempFileName

Создаёт имя для временного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetTempFileName ( $sFilePath [, $sPrefix = ''] )

Параметры

$sFilePath Путь каталога для имени файла. Приложения обычно указывают точку (.) для текущего каталога.
$sPrefix [необязательный] строка префикса. Функция использует до первых трёх символов этой строки в качестве префикса
имени файла.

Возвращаемое значение

Успех: Возвращает имя временного файла.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Временные файлы, имена которых созданы этой функцией, не удаляются автоматически.
