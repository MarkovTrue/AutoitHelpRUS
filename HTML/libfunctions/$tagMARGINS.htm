Справочник ключевых слов

$tagMARGINS

Определяет поля окон, к которым применены визуальные стили.

Global Const $tagMARGINS = "int cxLeftWidth;int cxRightWidth;int cyTopHeight;int cyBottomHeight"

Параметры

cxLeftWidth Ширина левой границы, которая сохраняет свой размер.
cxRightWidth Ширина правой границы, которая сохраняет свой размер.
cyTopHeight Высота верхней границы, которая сохраняет свой размер.
cyBottomHeight Высота нижней границы, которая сохраняет свой размер.
