Справочник ключевых слов

$tagNMHDR

Содержит сведения об уведомляющем сообщении.

Global Const $tagNMHDR = "struct; hwnd hWndFrom;uint_ptr IDFrom;INT Code; endstruct"

Параметры

hWndFrom Дескриптор окна элемента управления, отправляющего сообщение.
IDFrom Идентификатор элемента управления, отправляющего сообщение.
Code Код уведомления.
