Справочник функций

_GUICtrlComboBox_SetCurSel

Выбирает строку в списке выпадающего списка.

#include <GuiComboBox.au3>
_GUICtrlComboBox_SetCurSel ( $hWnd [, $iIndex = -1] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iIndex [необязательный] Задаёт индекс базового элемента для поиска (по умолчанию -1)

Возвращаемое значение

Успех: Возвращает индекс выбранного элемента, или $CB_ERR если ошибка.
Ошибка: Возвращает -1.

Примечания

Выбирает элемент в списке выпадающего списка.

См. также

_GUICtrlComboBox_GetCurSel

Пример

#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("ComboBox Get/Set Cur Sel (v" & @AutoItVersion & ")", 400, 296)
    Local $idCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBox_BeginUpdate($idCombo)
    _GUICtrlComboBox_AddDir($idCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($idCombo)

    ; Выбор элемента
    _GUICtrlComboBox_SetCurSel($idCombo, 2)

    ; Получить текущее выделение
    MsgBox($MB_SYSTEMMODAL, "Information", "Cur Sel: " & _GUICtrlComboBox_GetCurSel($idCombo))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
