Справочник функций

_Excel_BookSaveAs

Сохраняет указанную книгу под новым именем и/или в новом формате.

#include <Excel.au3>
_Excel_BookSaveAs ( $oWorkbook, $sFilePath [, $iFormat = $xlWorkbookDefault [, $bOverWrite = False [, $sPassword = Default [, $sWritePassword = Default [, $bReadOnlyRecommended = False]]]]] )

Параметры

$oWorkbook Объект сохраняемой книги.
$sFilePath Путь и имя файла для сохранения.
$iFormat [необязательный] Формат файла Excel. Допустимые значения – константы XlFileFormat.
$bOverWrite [необязательный] True – перезаписать существующий файл (по умолчанию False).
$sPassword [необязательный] Пароль для защиты файла. При значении Default пароль не задаётся (по умолчанию Default).
$sWritePassword [необязательный] Пароль для защиты файла от записи. При значении Default пароль не задаётся (по умолчанию Default).
$bReadOnlyRecommended [необязательный] True – при открытии файла отображать сообщение с рекомендацией открыть его в режиме только для чтения (по умолчанию False).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом
2 – $iFormat не является числом
3 – файл существует, флаг перезаписи не установлен
4 – файл существует, но не может быть удалён
5 – ошибка при сохранении книги.
@extended: Содержит код COM-ошибки метода SaveAs.

Примечания

Начиная с Excel 2007, необходимо указывать как параметр $iFormat, так и соответствующее расширение файла в $sFilePath.
Например, $iFormat = $xlExcel8 с расширением "xlsx" вернёт ошибку.
Возможные значения форматов см. в перечислении XlFileFormat в файле ExcelConstants.au3.

_Excel_BookSave

Пример

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookSaveAs Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookSaveAs Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Сохраняем книгу (xls) в другом формате (html) в другой каталог и
; перезаписываем существующий файл.
Local $sWorkbook = @TempDir & "\_Excel1.html"
_Excel_BookSaveAs($oWorkbook, $sWorkbook, $xlHtml, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookSaveAs Example 1", "Error saving workbook to '" & $sWorkbook & "'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookSaveAs Example 1", "Workbook successfully saved as '" & $sWorkbook & "'.")
ShellExecute($sWorkbook)
