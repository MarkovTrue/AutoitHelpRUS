Справочник функций

_viSetAttribute

Устанавливает любой атрибут VISA. Эта функция, которая вызывается _viSetTimeout(), также может использоваться для установки других специфических атрибутов VISA. Прочитайте документацию VISA для получения дополнительной информации и списка атрибутов VISA и их соответствующих значений.

#include <Visa.au3>
_viSetAttribute ( $hSession, $iAttribute, $iValue )

Параметры

$hSession Дескриптор VISA (строка) или дескриптор сессии VISA (целое число).
Подробности см. раздел Примечания _viExecCommand().
$iAttribute Индекс атрибута, который должен быть изменён.
Атрибуты определены в библиотеке VISA.
Эта библиотека AutoIt определяет только КОНСТАНТУ для атрибута TIMEOUT ($VI_ATTR_TMO_VALUE) и для атрибутов и значений серийного интерфейса (см. раздел Примечания), но вы можете передать любой другой действительный индекс (как целое число), если хотите.
$iValue Значение атрибута. Это должно быть целое число, и возможные значения зависят от типа атрибута и определены в документации VISA.

Возвращаемое значение

Успех: Возвращает 0.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0, если DLL-библиотеку VISA не удаётся открыть
или ненулевое значение, представляющее код ошибки VISA (см. руководство программиста VISA).

Примечания

Это список предопределённых атрибутов и значений. Помните, что вы можете использовать любой другой действительный атрибут/значение, передав соответствующий целочисленный индекс (как определено в руководстве программиста VISA) этой функции.

* Атрибут: $VI_ATTR_TMO_VALUE – установить тайм-аут
* Значения:
    Тайм-аут в МИЛЛИСЕКУНДАХ или
    $VI_TMO_IMMEDIATE (или 0) для "немедленного возврата"
    VI_TMO_INFINITE (или "INF") для "нет тайм-аута"

* Атрибут: $VI_ATTR_ASRL_BAUD
* Значения:
    Любая действительная скорость передачи (9600, 115200 и т. д.)

* Атрибут: $VI_ATTR_ASRL_DATA_BITS
* Значения:
    От 5 до 8

* Атрибут: $VI_ATTR_ASRL_PARITY
* Значения:
    $VI_ASRL_PAR_NONE
    $VI_ASRL_PAR_ODD
    $VI_ASRL_PAR_EVEN
    $VI_ASRL_PAR_MARK
    $VI_ASRL_PAR_SPACE

* Атрибут: $VI_ATTR_ASRL_STOP_BITS
* Значения:
    $VI_ASRL_STOP_ONE
    $VI_ASRL_STOP_ONE5
    $VI_ASRL_STOP_TWO

* Атрибут: $VI_ATTR_ASRL_FLOW_CNTRL
* Значения:
    $VI_ASRL_FLOW_NONE
    $VI_ASRL_FLOW_XON_XOFF
    $VI_ASRL_FLOW_RTS_CTS
    $VI_ASRL_FLOW_DTR_DSR

Как и для всех функций VISA, библиотеки VISA должны быть установлены (вы можете проверить, находится ли visa32.dll в {WINDOWS}\system32).
Карта GPIB (такая как NI PCI-GPIB от National Instruments или Agilent 82350B PCI High-Performance GPIB) требуется только если вы используете функции, связанные с GPIB.

* Подробное описание наиболее распространённых ДЕСКРИПТОРОВ VISA см. в примечаниях функции _viExecCommand()

См. также

_viClose, _viExecCommand, _viOpen, _viSetTimeout

Пример

; - Предполагает, что прибор установлен на адрес GPIB 3.
; Если прибор находится по другому адресу, измените "GPIB::3::0" на
; соответствующий дескриптор. То же самое для вызова _viOpen
; Показывает, как использовать _viSetAttribute. В этом примере мы используем _viSetAttribute
; вместо _viSetTimeout для установки тайм-аута GPIB операции _viExecCommand.

#include <MsgBoxConstants.au3>
#include <Visa.au3>

Local $h_Session = 0

; Запросить ID прибора по адресу GPIB 3
MsgBox($MB_SYSTEMMODAL, "Step 1", "Simple GPIB query with explicit TIMEOUT set")
Local $s_Answer = _viExecCommand("GPIB::3::0", "*IDN?", 10000) ; таймаут 10 секунд
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

; Это то же самое, что сначала использовать функцию _viSetAttribute:
MsgBox($MB_SYSTEMMODAL, "Step 2", "_vOpen + timeout using _viSetAttribute + GPIB query")
Local $h_Instr = _viOpen(3)
; ПРИМЕЧАНИЕ – это то же самое, что: _viSetTimeout($h_Instr, 10000)
_viSetAttribute($h_Instr, $VI_ATTR_TMO_VALUE, 10000) ; 10000 мс = 10 секунд

$s_Answer = _viExecCommand($h_Instr, "*IDN?") ; Сейчас не нужно задавать таймаут
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

MsgBox($MB_SYSTEMMODAL, "Step 3", "Close the Instrument connection using _viClose")
_viClose($h_Instr) ; Закрыть соединение с прибором
