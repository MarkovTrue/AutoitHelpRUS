Справочник функций

_FTP_Command

Отправляет команду FTP-серверу.

#include <FTPEx.au3>
_FTP_Command ( $hFTPSession, $sFTPCommand [, $iFlags = $FTP_TRANSFER_TYPE_ASCII [, $iExpectResponse = 0 [, $iContext = 0]]] )

Параметры

$hFTPSession возвращается функцией _FTP_Connect()
$sFTPCommand Строка команды для отправки FTP-серверу.
$iFlags [необязательный] $FTP_TRANSFER_TYPE_ASCII (1) или $FTP_TRANSFER_TYPE_BINARY (2)
$iExpectResponse [необязательный] Сокет данных для ответа в асинхронном режиме. См. раздел Примечания.
$iContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает идентификатор.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Примеры команд зависят от синтаксиса сервера.
Для двоичной передачи, ASCII-передачи и пассивного режима (используется с межсетевыми экранами) применяются команды 'type I', 'type A', 'pasv'.

Если $iExpectResponse = True, в @extended возвращается указатель на дескриптор, созданный при открытии допустимого сокета данных.

См. также

_FTP_Connect, _FTP_SetStatusCallback
