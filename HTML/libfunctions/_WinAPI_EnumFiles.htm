Справочник функций

_WinAPI_EnumFiles

Перечисляет файлы и подкаталоги указанного каталога, имена которых соответствуют шаблону.

#include <WinAPIFiles.au3>
_WinAPI_EnumFiles ( $sDir [, $iFlag = 0 [, $sTemplate = '' [, $bExclude = False]]] )

Параметры

$sDir Путь к каталогу.
$iFlag [необязательный] Флаг, указывающий что перечислять – файлы, подкаталоги или то и другое.
Допустимые значения:
0 – и файлы, и подкаталоги (по умолчанию).
1 – только файлы.
2 – только подкаталоги.
$sTemplate [необязательный] Шаблон для перечисления файлов и каталогов. Например, "*.jpe;*.jpeg;*.jpg".
$bExclude [необязательный] Использовать ли шаблон для исключения файлов:
True – файлы, подходящие под шаблон, не перечисляются.
False – перечисляются только файлы, подходящие под шаблон (по умолчанию).

Возвращаемое значение

Успех: Возвращает двумерный массив со следующей информацией:
$aArray[0][0] – количество строк массива (n)
[0][i] – не используется
[n][0] – имя файла.
[n][1] – время создания файла (UTC).
[n][2] – время последнего обращения к файлу (UTC).
[n][3] – время последнего изменения файла (UTC).
[n][4] – размер файла в байтах.
[n][5] – размер выделения файла в байтах.
[n][6] – атрибуты файла ($FILE_ATTRIBUTE_*).
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки NTSTATUS.

Примечания

Если целевой каталог пуст или ни один файл не подходит под заданные условия, функция всё равно возвращает массив, у которого нулевой элемент равен 0.

Пример

#include <Array.au3>
#include <WinAPIFiles.au3>

Local $aData = _WinAPI_EnumFiles(@SystemDir, 1, '*.ax;*.cpl;*.dll;*.drv;*.exe;*.ocx;*.scr')

_ArrayDisplay($aData, '_WinAPI_EnumFiles')
