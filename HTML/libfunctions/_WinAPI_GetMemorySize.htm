_WinAPI_GetMemorySize

Получает размер блока памяти, выделенного из внутренней кучи библиотеки.

#include <WinAPIMem.au3>
_WinAPI_GetMemorySize ( $pMemory )

Параметры

$pMemory Указатель на действительный блок памяти, размер которого требуется получить.

Возвращаемое значение

Успех: Возвращает размер указанного блока памяти в байтах.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Функция _WinAPI_GetMemorySize() принимает только указатели, возвращаемые некоторыми функциями _WinAPI_* (см. описание).
