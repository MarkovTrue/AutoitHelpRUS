_WinAPI_RegQueryReflectionKey

Определяет, отключена или включена отражение для указанного ключа.

#include <WinAPIReg.au3>
_WinAPI_RegQueryReflectionKey ( $hKey )

Параметры

$hKey Дескриптор открытого раздела реестра.
Возвращается функциями _WinAPI_RegCreateKey() и _WinAPI_RegOpenKey();
не может указывать на раздел удалённого компьютера.
Если раздел не находится в списке отражений, функция успешно завершится, но эффекта не окажет.

Возвращаемое значение

Успех: 1 – отражение отключено;
0 – включено.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

В WOW64 32-разрядные приложения видят дерево реестра, которое отделено от дерева реестра, которое видят 64-разрядные приложения.
Отражение реестра копирует конкретные ключи и значения реестра между двумя представлениями.

ВНИМАНИЕ: Отражение реестра было удалено начиная с Windows 7 и Windows Server 2008 R2.
