Справочник функций

_WinAPI_GetFontName

Получает уникальное имя шрифта на основе его названия гарнитуры, кодировки символов и стиля.

#include <WinAPIGdi.au3>
_WinAPI_GetFontName ( $sFaceName [, $iStyle = $FS_REGULAR [, $iCharSet = $DEFAULT_CHARSET]] )

Параметры

$sFaceName Название гарнитуры шрифта (без учёта стиля). Например, "Arial", "Tahoma" и т. д.
$iStyle [необязательный] Стиль шрифта.
Может быть одним или несколькими из следующих значений:
$FS_REGULAR (по умолчанию)
$FS_BOLD
$FS_ITALIC
$iCharSet [необязательный] Кодировка символов.
Может быть одним из следующих значений:
$ANSI_CHARSET
$BALTIC_CHARSET
$CHINESEBIG5_CHARSET
$DEFAULT_CHARSET (по умолчанию)
$EASTEUROPE_CHARSET
$GB2312_CHARSET
$GREEK_CHARSET
$HANGEUL_CHARSET
$MAC_CHARSET
$OEM_CHARSET
$RUSSIAN_CHARSET
$SHIFTJIS_CHARSET
$SYMBOL_CHARSET
$TURKISH_CHARSET
$VIETNAMESE_CHARSET
Корейское издание Windows:
$JOHAB_CHARSET
Издание Windows для стран Ближнего Востока:
$ARABIC_CHARSET
$HEBREW_CHARSET
Тайское издание Windows:
$THAI_CHARSET

Возвращаемое значение

Успех: Возвращает уникальное имя шрифта.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.

Примечания

Эта функция может использоваться для получения соответствующего имени шрифта в разных языковых версиях ОС.
Например, в английской версии можно написать "Arial Bold", однако в русской версии этот шрифт должен быть указан как "Arial Полужирный".
Тем не менее большинство шрифтов используют только английские имена и должны быть указаны явно.

Пример

#include <FontConstants.au3>
#include <WinAPIGdi.au3>

Local Const $sFaceName = 'Arial'

ConsoleWrite($sFaceName & ' Regular => ' & _WinAPI_GetFontName($sFaceName, $FS_REGULAR) & @CRLF)
ConsoleWrite($sFaceName & ' Bold => ' & _WinAPI_GetFontName($sFaceName, $FS_BOLD) & @CRLF)
ConsoleWrite($sFaceName & ' Italic => ' & _WinAPI_GetFontName($sFaceName, $FS_ITALIC) & @CRLF)
ConsoleWrite($sFaceName & ' Bold Italic => ' & _WinAPI_GetFontName($sFaceName, BitOR($FS_BOLD, $FS_ITALIC)) & @CRLF)
