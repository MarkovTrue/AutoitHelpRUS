Справочник функций

_WinAPI_NtStatusToDosError

Преобразует указанный код ошибки NTSTATUS в эквивалентный системный код ошибки.

#include <WinAPIDiag.au3>
_WinAPI_NtStatusToDosError ( $iStatus )

Параметры

$iStatus Код ошибки NTSTATUS для преобразования.

Возвращаемое значение

Возвращает системный код ошибки. ERROR_MR_MID_NOT_FOUND (317) возвращается, если указанный код ошибки NTSTATUS
    не имеет соответствующего системного кода ошибки.

Дополнительно

Искать RtlNtStatusToDosError в библиотеке MSDN

Пример

#include <WinAPIDiag.au3>
#include <WinAPIError.au3>

Local $iError = _WinAPI_NtStatusToDosError(0xC0000023) ; STATUS_BUFFER_TOO_SMALL

ConsoleWrite($iError & ' - ' & _WinAPI_GetErrorMessage($iError) & @CRLF)
