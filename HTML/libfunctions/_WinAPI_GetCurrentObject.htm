_WinAPI_GetCurrentObject

Получает дескриптор объекта указанного типа, выбранного в указанный контекст устройства.

#include <WinAPIHObj.au3>
_WinAPI_GetCurrentObject ( $hDC, $iType )

Параметры

$hDC Дескриптор контекста устройства.
$iType Тип объекта для запроса. Этот параметр может быть одним из следующих значений.
$OBJ_BITMAP
$OBJ_BRUSH
$OBJ_COLORSPACE
$OBJ_FONT
$OBJ_PAL
$OBJ_PEN

Возвращаемое значение

Успех: Возвращает дескриптор указанного объекта.
Ошибка: Возвращает 0.
