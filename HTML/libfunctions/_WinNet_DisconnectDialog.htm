_WinNet_DisconnectDialog

Запускает диалоговое окно общего просмотра для отключения от сетевых ресурсов.

#include <WinNet.au3>
_WinNet_DisconnectDialog ( $hWnd )

Параметры

$hWnd Дескриптор окна-владельца диалогового окна.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
