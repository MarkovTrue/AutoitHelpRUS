_WinAPI_DwmQueryThumbnailSourceSize

Возвращает исходный размер миниатюры Диспетчера окон рабочего стола (DWM).

#include <WinAPIGdi.au3>
_WinAPI_DwmQueryThumbnailSourceSize ( $hThumbnail )

Параметры

$hThumbnail Дескриптор миниатюры, у которой нужно получить размер исходного окна.

Возвращаемое значение

Успех: Возвращает структуру $tagSIZE с размером исходной миниатюры.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
