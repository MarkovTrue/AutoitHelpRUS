_WinAPI_DwmUnregisterThumbnail

Удаляет связь миниатюры Desktop Window Manager (DWM).

#include <WinAPIGdi.au3>
_WinAPI_DwmUnregisterThumbnail ( $hThumbnail )

Параметры

$hThumbnail Дескриптор связи миниатюры, которую нужно удалить.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
