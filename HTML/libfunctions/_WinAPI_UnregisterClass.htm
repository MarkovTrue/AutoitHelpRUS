Справочник функций

_WinAPI_UnregisterClass

Отменяет регистрацию класса окна, освобождая память, необходимую для класса.

#include <WinAPISysWin.au3>
_WinAPI_UnregisterClass ( $sClass [, $hInstance = 0] )

Параметры

$sClass Строка, оканчивающаяся нулём, или атом класса. Если $sClass – строка, она указывает имя класса окна.
Это имя класса должно было быть зарегистрировано при предыдущем вызове функции _WinAPI_RegisterClass()
или _WinAPI_RegisterClassEx(). Если этот параметр – атом, он должен быть в младшем
слове $sClass; старший слово должен быть нулевым.
$hInstance [необязательный] Дескриптор экземпляра модуля, который создал класс.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Перед вызовом этой функции приложение должно уничтожить все окна, созданные с указанным классом.
Все классы окон, которые регистрирует приложение, отменяют регистрацию при его завершении.

См. также

_WinAPI_RegisterClass, _WinAPI_RegisterClassEx
