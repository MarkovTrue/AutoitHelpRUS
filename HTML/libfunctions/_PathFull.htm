Справочник функций

_PathFull

Создаёт полный путь на основе относительного пути. Возвращает созданный абсолютный путь.

#include <File.au3>
_PathFull ( $sRelativePath [, $sBasePath = @WorkingDir] )

Параметры

$sRelativePath Относительный путь для преобразования.
$sBasePath [необязательный] Базовый путь для использования. По умолчанию = @WorkingDir

Возвращаемое значение

Возвращает относительный путь будет построен относительно $sBasePath. Чтобы обойти это поведение, используйте абсолютный путь.

Примечания

Поддерживаются пути UNC.
Передайте "\" для получения корневого диска $sBasePath.
Передайте "" или "." для возврата $sBasePath.

См. также

_PathMake, _PathSplit, _WinAPI_GetFullPathName

Пример

#include <File.au3>
#include <MsgBoxConstants.au3>

Local $sTestPath = _PathFull(@ScriptDir & "\..\..\test")
MsgBox($MB_SYSTEMMODAL, "", @ScriptDir & @CRLF & $sTestPath)
