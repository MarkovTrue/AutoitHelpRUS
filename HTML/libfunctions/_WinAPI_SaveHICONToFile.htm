Справочник функций

_WinAPI_SaveHICONToFile

Сохраняет один или несколько значков (HICON) в указанный файл значков (.ico).

#include <WinAPIGdi.au3>
_WinAPI_SaveHICONToFile ( $sFilePath, Const ByRef $vIcon [, $bCompress = 0 [, $iStart = 0 [, $iEnd = -1]]] )

Параметры

$sFilePath Имя файла .ico для сохранения значка.
$vIcon Дескриптор значка или массив дескрипторов значков для сохранения.
$bCompress [необязательный] Указывает, использовать ли сжатие PNG для значков 32 бит/пикс, если их размер превышает или
равен 256x256 пикселей (262144 байта), допустимые значения:
True – значок будет сохранён как изображение PNG.
False – значок будет сохранён напрямую (по умолчанию).
$iStart [необязательный] Индекс массива, с которого начать сохранение.
$iEnd [необязательный] Индекс массива, где закончить сохранение.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Функция работает со значками 16, 24 и 32 бит/пикс. Поскольку дескрипторы значков всегда содержат зависимый от устройства
XOR-растр (DDB), полученное изображение, сохранённое в файле .ico, может отличаться от исходного изображения.

Если указанный значок – это значок 32 бит/пикс и использует маску AND вместо альфа-канала,
он будет преобразован в значок с альфа-каналом (RGB + Alpha).

Отметим, что значки со сжатием PNG поддерживаются только начиная с Windows Vista. Кроме того,
не все приложения могут работать с такими значками.

Пример

#include <WinAPIGdi.au3>
#include <WinAPIIcons.au3>
#include <WinAPIShellEx.au3>

Local $aIcon[3] = [48, 32, 16]

For $i = 0 To UBound($aIcon) - 1
    $aIcon[$i] = _WinAPI_Create32BitHICON(_WinAPI_ShellExtractIcon(@SystemDir & '\shell32.dll', 32, $aIcon[$i], $aIcon[$i]), 1)
Next
_WinAPI_SaveHICONToFile(@TempDir & '\MyIcon.ico', $aIcon)
For $i = 0 To UBound($aIcon) - 1
    _WinAPI_DestroyIcon($aIcon[$i])
Next

ShellExecute('mspaint', '"' & @TempDir & '\MyIcon.ico' & '"')
