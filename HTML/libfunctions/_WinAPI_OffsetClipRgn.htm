_WinAPI_OffsetClipRgn

Перемещает область отсечения контекста устройства на указанные смещения.

#include <WinAPIGdi.au3>
_WinAPI_OffsetClipRgn ( $hDC, $iXOffset, $iYOffset )

Параметры

$hDC Дескриптор контекста устройства.
$iXOffset Количество логических единиц для перемещения по горизонтали.
$iYOffset Количество логических единиц для перемещения по вертикали.

Возвращаемое значение

Успех: Сложность новой области отсечения. Может принимать одно из значений:
$COMPLEXREGION
$NULLREGION
$SIMPLEREGION
Ошибка: Возвращает 0.
