Справочник функций

_SetDate

Устанавливает текущую дату системы.

#include <Date.au3>
_SetDate ( $iDay [, $iMonth = 0 [, $iYear = 0]] )

Параметры

$iDay День.
$iMonth [необязательный] Месяц. Значения: 1-12
$iYear [необязательный] Год. Значения: > 0 (Windows может ограничить это дальше!!)

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – Ошибка.
@extended: _WinAPI_GetLastError()
Код ошибки: MSDN: ms681381.aspx.

Примечания

Если необязательные параметры ($iMonth,$iYear) НЕ определены, функция не изменит текущее значение!

Пример

#include <Date.au3>

_SetDate(20) ; Установить текущую дату на 20-xx-xxxx
_SetDate(20, 10) ; Установить текущую дату на 20-10-xxxx
_SetDate(20, 10, 2004) ; Установить текущую дату на 20-10-2004
