_WinAPI_SetTextJustification

Указывает объём пространства, которое система должна добавить к символам разрыва в строке текста.

#include <WinAPIGdi.au3>
_WinAPI_SetTextJustification ( $hDC, $iBreakExtra, $iBreakCount )

Параметры

$hDC Дескриптор контекста устройства.
$iBreakExtra Общее дополнительное пространство в логических единицах, добавляемое к строке текста.
$iBreakCount Количество символов разрыва в строке.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
