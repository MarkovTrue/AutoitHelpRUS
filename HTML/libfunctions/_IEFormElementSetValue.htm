Справочник функций

_IEFormElementSetValue

Задаёт значение указанного элемента формы.

#include <IE.au3>
_IEFormElementSetValue ( ByRef $oObject, $sNewValue [, $iFireEvent = 1] )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$sNewValue Новое значение, которое нужно установить элементу формы.
$iFireEvent [необязательный] задаёт, генерировать ли событие OnChange после изменения значения
0 = Не генерировать события OnChange или OnClick после установки значения
1 = (по умолчанию) генерировать события OnChange и OnClick после установки значения

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
@extended: Содержит номер недопустимого параметра.

Примечания

Хотя у всех Form Elements есть значение, только текстовые элементы используют атрибут value очевидным образом (типы text, textarea, hidden, password и file).
Значение остальных элементов формы не влияет на то, что отображается в пользовательском интерфейсе; это скорее значение, возвращаемое элементом при его выборе или активации.

Подробнее см. _IEFormElementOptionSelect(), _IEFormElementCheckBoxSelect(), _IEFormElementRadioSelect() и _IEFormImageClick().

Примечание: _IEFormElementSetValue() нельзя использовать для установки значения элемента INPUT TYPE=FILE. Ограничения безопасности браузера запрещают скриптовать такой элемент. Обходное решение см. в примере ниже.

См. также

_IEFormElementCheckBoxSelect, _IEFormElementGetCollection, _IEFormElementGetObjByName, _IEFormElementGetValue, _IEFormElementOptionSelect, _IEFormElementRadioSelect

Пример

Пример 1

; Открытие браузера с примером формы, установка значения текстового элемента формы

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oText = _IEFormElementGetObjByName($oForm, "textExample")
_IEFormElementSetValue($oText, "Hey! This works!")

Пример 2

; Получить ссылку на элемент формы и задать его значение.
; В этом случае отправить запрос в поисковую систему Google

#include <IE.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oForm = _IEFormGetCollection($oIE, 0)
Local $oQuery = _IEFormElementGetCollection($oForm, 4)
_IEFormElementSetValue($oQuery, "AutoIt IE.au3")
_IEFormSubmit($oForm)

Пример 3

; Установка значения элемента INPUT TYPE=TEXT через Send()

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oInputFile = _IEFormElementGetObjByName($oForm, "textExample")

; Передать фокус ввода полю и отправить текстовую строку
_IEAction($oInputFile, "focus")

; Выделить существующее содержимое, чтобы оно было перезаписано.
_IEAction($oInputFile, "selectall")

Send("This works")

Пример 4

; Установка значения элемента INPUT TYPE=TEXT в невидимом
; окно через ControlSend()

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("form")

; Скрытие окна браузера для демонстрации отправки текста в невидимое окно
_IEAction($oIE, "invisible")

Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oInputFile = _IEFormElementGetObjByName($oForm, "textExample")

; Передать фокус ввода полю и отправить текстовую строку
_IEAction($oInputFile, "focus")

; Выделить существующее содержимое, чтобы оно было перезаписано.
_IEAction($oInputFile, "selectall")

; Получить дескриптор окна IE.
Local $hIE = _IEPropertyGet($oIE, "hwnd")
ControlSend($hIE, "", "[CLASS:Internet Explorer_Server; INSTANCE:1]", "This works")

_IEAction($oIE, "visible")
MsgBox($MB_SYSTEMMODAL, "Success", "Value set to 'This works'")
