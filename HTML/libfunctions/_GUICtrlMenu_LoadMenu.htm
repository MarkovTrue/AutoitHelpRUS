Справочник функций

_GUICtrlMenu_LoadMenu

Загружает указанный ресурс меню из исполняемого файла.

#include <GuiMenu.au3>
_GUICtrlMenu_LoadMenu ( $hInst, $sMenuName )

Параметры

$hInst Дескриптор модуля, содержащего ресурс меню.
$sMenuName Строка с именем ресурса меню.
Вместо неё можно передать идентификатор ресурса в виде числа, преобразованного функцией _WinAPI_MakeIntResource().

Возвращаемое значение

Успех: Возвращает дескриптор ресурса меню.
Ошибка: Возвращает 0.
