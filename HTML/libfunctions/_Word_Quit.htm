_Word_Quit

Закрывает все документы, приложение Microsoft Word и удаляет ссылку на объект.

#include <Word.au3>
_Word_Quit ( $oAppl [, $iSaveChanges = $WdDoNotSaveChanges [, $iOriginalFormat = $WdWordDocument [, $bForceClose = False]]] )

Параметры

$oAppl Объект Word, возвращённый предыдущим вызовом _Word_Create().
$iSaveChanges [необязательный] Указывает, следует ли сохранять изменённые документы перед закрытием.
Может быть одним из перечисления WdSaveOptions (по умолчанию = $WdDoNotSaveChanges).
$iOriginalFormat [необязательный] Указывает, как Microsoft Word сохраняет документы, исходный формат которых не был форматом документа Microsoft Word.
Может быть любым из перечисления WdOriginalFormat (по умолчанию = $WdWordDocument).
$bForceClose [необязательный] Если True, приложение Microsoft Word закрывается даже если оно было запущено при вызове _Word_Create() (по умолчанию = False)

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $oAppl не является объектом
2 – Ошибка, возвращённая методом Application.Quit. @extended задан код ошибки COM

Примечания

Если Microsoft Word не был запущен при вызове _Word_Create(), то _Word_Quit() закрывает все документы указанного
экземпляра (даже те, которые были открыты пользователем вручную после _Word_Create()) и приложение Microsoft Word.
Если Microsoft Word был запущен, необходимо установить $bForceClose в True для выполнения того же самого.

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocQuit Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $bWordClose = @extended
; Открыть тестовый документ только для чтения
_Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocQuit Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Если Microsoft Word был запущен при запуске этого скрипта, будет удалена только ссылка на
; объект.
; Если Microsoft Word был запущен этим примером, все документы и Microsoft Word будут закрыты.
Local $iResult
If $bWordClose Then
    $iResult = MsgBox(($MB_OKCANCEL + $MB_SYSTEMMODAL), "Word UDF: _Word_Quit Example", _
            "If you click OK ALL unsaved changes in ALL open documents of this Word instance will be lost and the instance will be closed.")
Else
    $iResult = MsgBox(($MB_OKCANCEL + $MB_SYSTEMMODAL), "Word UDF: _Word_Quit Example", _
            "Word was already running when function _Word_Create was called. Hence only the reference to the object will be removed.")
EndIf
If $iResult = 2 Then Exit
_Word_Quit($oWord)
If @error Then MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_Quit Example", _
        "Error closing the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
