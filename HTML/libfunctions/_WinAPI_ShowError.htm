Справочник функций

_WinAPI_ShowError

Отображает окно сообщения об ошибке с опциональным выходом.

#include <WinAPIError.au3>
_WinAPI_ShowError ( $sText [, $bExit = True] )

Параметры

$sText Текст ошибки для отображения.
$bExit [необязательный] Указывает, выходить ли после отображения:
True - Выход из программы после отображения
False - Нормальный возврат после отображения.

Возвращаемое значение

Отсутствует.

См. также

_WinAPI_MsgBox, _WinAPI_ShowMsg
