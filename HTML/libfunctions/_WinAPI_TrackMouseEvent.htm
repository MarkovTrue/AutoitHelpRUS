_WinAPI_TrackMouseEvent

Отправляет сообщения, когда указатель мыши покидает окно или находится над окном указанное время.

#include <WinAPISys.au3>
_WinAPI_TrackMouseEvent ( $hWnd, $iFlags [, $iTime = -1] )

Параметры

$hWnd Дескриптор окна.
$iFlags Запрашиваемые услуги. Этот параметр может быть сочетанием следующих значений.
$TME_CANCEL
$TME_HOVER
$TME_LEAVE
$TME_NONCLIENT
$TME_QUERY
$iTime [необязательный] Время ожидания при наведении (если $TME_HOVER был задан в $Flags) в миллисекундах. Может быть (-1) (по умолчанию), что
означает использование времени ожидания наведения по умолчанию в системе.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
