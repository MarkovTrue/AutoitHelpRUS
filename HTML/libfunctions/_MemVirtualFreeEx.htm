_MemVirtualFreeEx

Освобождает область страниц в виртуальном адресном пространстве процесса.

#include <Memory.au3>
_MemVirtualFreeEx ( $hProcess, $pAddress, $iSize, $iFreeType )

Параметры

$hProcess Дескриптор процесса.
$pAddress Указатель на начальный адрес области памяти для освобождения.
$iSize Размер области памяти для освобождения в байтах.
$iFreeType Задаёт тип операции освобождения:
$MEM_DECOMMIT – Отменяет выделение указанной области выделённых страниц
$MEM_RELEASE – Освобождает указанную область зарезервированных страниц.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
