_WinAPI_DrawThemeParentBackground

Рисует часть родительского элемента управления, перекрытую частично прозрачным или альфа-смешанным дочерним элементом.

#include <WinAPITheme.au3>
_WinAPI_DrawThemeParentBackground ( $hWnd, $hDC [, $tRECT = 0] )

Параметры

$hWnd Дескриптор дочернего элемента управления.
$hDC дескриптор к child control's DC.
$tRECT [необязательный] Структура $tagRECT, задающая прорисовываемую область в координатах дочернего окна.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
