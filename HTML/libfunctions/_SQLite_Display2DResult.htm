Справочник функций

_SQLite_Display2DResult

Возвращает или выводит в консоль отформатированный вывод двумерного массива.

#include <SQLite.au3>
_SQLite_Display2DResult ( $aResult [, $iCellWidth = 0 [, $bReturn = False [, $sDelim_Col = "" [, $sDelim_Row = @CRLF]]]] )

Параметры

$aResult Массив для отображения.
$iCellWidth [необязательный] указывает размер поля данных
$bReturn [необязательный] если True, отформатированная строка возвращается, не выводится
Если False, отформатированная строка отправляется в StdOut.
$sDelim_Col [необязательный] указывает разделитель столбцов (по умолчанию = "")
$sDelim_Row [необязательный] указывает разделитель строк (по умолчанию = @CRLF)

Возвращаемое значение

Успех: Возвращает ничего или отформатированная строка.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – $aResult не является массивом или имеет неправильную размерность

См. также

_SQLite_GetTable2D

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $aResult, $iRows, $iColumns, $iRval

_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; Открыть :memory: базу данных
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "Can't Load Database!")
    Exit -1
EndIf

; Пример таблицы
; Имя | Возраст
; -----------------------
; Алиса | 43
; Боб | 28
; Синди | 21

If Not _SQLite_Exec(-1, "CREATE TEMP TABLE persons (Name, Age);") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Alice','43');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Bob','28');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Cindy','21');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())

; Запрос
$iRval = _SQLite_GetTable2D(-1, "SELECT * FROM persons;", $aResult, $iRows, $iColumns)
If $iRval = $SQLITE_OK Then
    _SQLite_Display2DResult($aResult)

    ; $aResult выглядит так:
    ; Имя Возраст
    ; Alice 43
    ; Bob 28
    ; Cindy 21
    ; Если размерности будут переключены в _SQLite_GetTable2D, результат будет выглядеть так:
    ; Name Alice Bob Cindy
    ; Возраст 43 28 21

Else
    MsgBox($MB_SYSTEMMODAL, "SQLite Error: " & $iRval, _SQLite_ErrMsg())
EndIf

_SQLite_Close()
_SQLite_Shutdown()
