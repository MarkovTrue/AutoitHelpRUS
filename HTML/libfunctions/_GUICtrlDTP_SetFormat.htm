Справочник функций

_GUICtrlDTP_SetFormat

Задаёт отображение по заданной строке формата.

#include <GuiDateTimePicker.au3>
_GUICtrlDTP_SetFormat ( $hWnd, $sFormat )

Параметры

$hWnd Дескриптор элемента управления.
$sFormat Строка, задающая нужный формат. Пустая строка возвращает элемент к формату по умолчанию для текущего стиля. Доступны следующие подстроки формата:
"d" – день из одной или двух цифр
"dd" – день из двух цифр. Однозначные значения дополняются ведущим нулём
"ddd" – трёхбуквенное сокращение дня недели
"dddd" – полное название дня недели
"h" – час из одной или двух цифр в 12-часовом формате
"hh" – час из двух цифр в 12-часовом формате
"H" – час из одной или двух цифр в 24-часовом формате
"HH" – час из двух цифр в 24-часовом формате
"m" – минуты из одной или двух цифр
"mm" – минуты из двух цифр
"M" – номер месяца из одной или двух цифр
"MM" – номер месяца из двух цифр
"MMM" – трёхбуквенное сокращение месяца
"MMMM" – полное название месяца
"t" – однобуквенное обозначение AM/PM
"tt" – двухбуквенное обозначение AM/PM
"yy" – две последние цифры года
"yyyy" – год полностью

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

В строку формата можно добавлять дополнительные символы для более выразительного отображения.
Однако все неформатные символы должны быть заключены в одинарные кавычки.
Например, строка формата "'Today is:
    'hh':'m':'s ddddMMMdd', 'yyy" даст результат вида "Today is: 04:22:31 Tuesday Mar 23, 1996".

Примечание: при использовании формата по умолчанию элемент DTP отслеживает изменения локали.
Если задать собственный формат, он не будет обновляться при смене локали.

Пример

#include <GUIConstantsEx.au3>
#include <GuiDateTimePicker.au3>

Example()

Func Example()
    Local $hDTP

    ; Создать GUI
    GUICreate("DateTimePick Set Format (v" & @AutoItVersion & ")", 400, 300)
    $hDTP = GUICtrlGetHandle(GUICtrlCreateDate("", 2, 6, 190))

    GUISetState(@SW_SHOW)

    ; Установить формат отображения
    Local $iRet = _GUICtrlDTP_SetFormat($hDTP, "ddd MMM dd, yyyy hh:mm ttt")
    ConsoleWrite('@@ Debug(' & @ScriptLineNumber & ') : $iRet = ' & $iRet & @CRLF & '>Error code: ' & @error & ' Extended code: ' & @extended & ' (0x' & Hex(@extended) & ')' & @CRLF) ;### Отладочная консоль

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
