Справочник функций

_Net_Share_FileEnum

Возвращает информацию об открытых файлах на сервере.

#include <NetShare.au3>
_Net_Share_FileEnum ( [$sServer = "" [, $sBaseName = "" [, $sUserName = ""]]] )

Параметры

$sServer [необязательный] Строка, содержащая имя сервера, на котором должна выполниться функция.
Пустая строка указывает локальный компьютер.
$sBaseName [необязательный] Строка, содержащая уточнение для возвращённой информации.
Если пуста – перечисляются все открытые ресурсы.
Если не пуста – функция перечисляет только ресурсы, имеющие $sBaseName в качестве префикса.
$sUserName [необязательный] Строка, указывающая имя пользователя.
Если не пуста, $sUserName служит уточнением для перечисления.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
$aArray[0][0] – количество записей в массиве (n)
$aArray[1][0] – номер ID, назначенный ресурсу при его открытии
$aArray[1][1] – разрешения доступа, связанные с открывающим приложением:
1 – разрешение на чтение ресурса и выполнение ресурса
2 – разрешение на запись в ресурс.
4 – разрешение на создание ресурса
8 – разрешение на выполнение
16 – разрешение на удаление
32 – разрешение на изменение атрибутов
64 – разрешение на изменение ACL
$aArray[1][2] – содержит количество блокировок файлов на ресурсе
$aArray[1][3] – указывает путь открытого ресурса
$aArray[1][4] – указывает, какой пользователь или какой компьютер открыл ресурс
Ошибка: Устанавливает @error ≠ 0.

Примечания

Эту функцию могут выполнять только члены локальных групп "Администраторы" или "Операторы серверов".

См. также

_Net_Share_ConnectionEnum, _Net_Share_SessionEnum, _Net_Share_ShareEnum

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <NetShare.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $sServer, $aInfo

    ; Создать GUI
    GUICreate("NetShare", 400, 300)

    ; Создать элемент memo
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Получить сведения о сервере и общем ресурсе
    $sServer = InputBox("NetWork Demo", "Enter Server Name:", "\\MyServer", "", 200, 130)
    If @error Then Exit

    ; Перечислить открытые файлы на сервере
    $aInfo = _Net_Share_FileEnum($sServer)
    _MemoWrite("Error ...................: " & @error)
    _MemoWrite("Entries read ............: " & $aInfo[0][0])
    For $iI = 1 To $aInfo[0][0]
        _MemoWrite("Resource ID .............: " & $aInfo[$iI][0])
        _MemoWrite("Resource permissions ....: " & _Net_Share_PermStr($aInfo[$iI][1]))
        _MemoWrite("Resource locks ..........: " & $aInfo[$iI][2])
        _MemoWrite("Resource path ...........: " & $aInfo[$iI][3])
        _MemoWrite("Resource user ...........: " & $aInfo[$iI][4])
        _MemoWrite()
    Next

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
