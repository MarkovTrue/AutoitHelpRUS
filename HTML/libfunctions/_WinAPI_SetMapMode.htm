_WinAPI_SetMapMode

Устанавливает режим сопоставления для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetMapMode ( $hDC, $iMode )

Параметры

$hDC Дескриптор контекста устройства.
$iMode Новый режим сопоставления. Этот параметр может быть одним из следующих значений.
$MM_ANISOTROPIC
$MM_HIENGLISH
$MM_HIMETRIC
$MM_ISOTROPIC
$MM_LOENGLISH
$MM_LOMETRIC
$MM_TEXT
$MM_TWIPS

Возвращаемое значение

Успех: Возвращает предыдущий режим сопоставления.
Ошибка: Возвращает 0.
