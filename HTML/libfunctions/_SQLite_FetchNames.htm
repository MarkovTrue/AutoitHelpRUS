_SQLite_FetchNames

Читает имена столбцов из запроса, основанного на _SQLite_Query().

#include <SQLite.au3>
_SQLite_FetchNames ( $hQuery, ByRef $aNames )

Параметры

$hQuery Дескриптор запроса, генерируемый SQLite_Query().
$aNames Одномерный массив, содержащий имена столбцов.

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове SQLite API "sqlite3_data_count"
2 – ошибка при вызове SQLite API "sqlite3_column_name16"
3 – вызов запрещён безопасным режимом

См. также

_SQLite_Query

Пример

#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $hQuery, $aRow, $aNames
_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; открыть :memory: базу данных
_SQLite_Exec(-1, "CREATE TABLE aTest (A,B,C);")
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('c','2','World');")
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('b','3',' ');")
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('a','1','Hello');")
_SQLite_Query(-1, "SELECT ROWID,* FROM aTest ORDER BY a;", $hQuery)
_SQLite_FetchNames($hQuery, $aNames) ; Чтение имён столбцов
ConsoleWrite(StringFormat(" %-10s %-10s %-10s %-10s ", $aNames[0], $aNames[1], $aNames[2], $aNames[3]) & @CRLF)
While _SQLite_FetchData($hQuery, $aRow) = $SQLITE_OK
    ConsoleWrite(StringFormat(" %-10s %-10s %-10s %-10s ", $aRow[0], $aRow[1], $aRow[2], $aRow[3]) & @CRLF)
WEnd
_SQLite_Exec(-1, "DROP TABLE aTest;")
_SQLite_Close()
_SQLite_Shutdown()

; Вывод:
; rowid A B C
; 3 a 1 Hello
; 2 b 3
; 1 c 2 World
