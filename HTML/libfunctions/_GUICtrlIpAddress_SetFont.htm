Справочник функций

_GUICtrlIpAddress_SetFont

Задаёт шрифт элемента управления.

#include <GuiIPAddress.au3>
_GUICtrlIpAddress_SetFont ( $hWnd [, $sFaceName = "Arial" [, $iFontSize = 12 [, $iFontWeight = 400 [, $bFontItalic = False]]]] )

Параметры

$hWnd Дескриптор элемента управления.
$sFaceName [необязательный] Название шрифта
$iFontSize [необязательный] Размер шрифта
$iFontWeight [необязательный] Толщина шрифта
$bFontItalic [необязательный] Использовать атрибут курсив

Возвращаемое значение

Отсутствует.

Пример

#include <GUIConstantsEx.au3>
#include <GuiIPAddress.au3>

Example()

Func Example()
    Local $hGui, $hIPAddress, $hIPAddress2

    $hGui = GUICreate("IP Address Control Set Font Example", 300, 150)
    $hIPAddress = _GUICtrlIpAddress_Create($hGui, 10, 10, 150, 30)
    $hIPAddress2 = _GUICtrlIpAddress_Create($hGui, 10, 50, 150, 30)
    GUISetState(@SW_SHOW)

    _GUICtrlIpAddress_Set($hIPAddress, "24.168.2.128")
    _GUICtrlIpAddress_SetFont($hIPAddress, "Times New Roman", 14, 800, True)
    _GUICtrlIpAddress_Set($hIPAddress2, "24.168.2.128")
    _GUICtrlIpAddress_SetFont($hIPAddress2, "Arial", 10, 300)

    ; Ждать закрытия GUI пользователем
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
