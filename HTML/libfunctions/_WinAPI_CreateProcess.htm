Справочник функций

_WinAPI_CreateProcess

создаёт новый процесс и его основной поток.

#include <WinAPIProc.au3>
_WinAPI_CreateProcess ( $sAppName, $sCommand, $tSecurity, $tThread, $bInherit, $iFlags, $pEnviron, $sDir, $tStartupInfo, $tProcess )

Параметры

$sAppName Имя.
$sCommand Командная строка для выполнения.
$tSecurity Дескриптор.
$tThread Дескриптор.
$bInherit Дескриптор.
$iFlags Флаги.
$pEnviron Указатель на структуру.
$sDir Полный путь к текущему каталогу процесса.
$tStartupInfo Указатель на структуру.
$tProcess Указатель на структуру.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

См. также

$tagPROCESS_INFORMATION, $tagSECURITY_ATTRIBUTES, $tagSTARTUPINFO

Дополнительно

Искать CreateProcess в библиотеке MSDN

Пример

#include <StructureConstants.au3>
#include <WinAPIError.au3>
#include <WinAPIProc.au3>

Local $tProcess = DllStructCreate($tagPROCESS_INFORMATION)
Local $tStartup = DllStructCreate($tagSTARTUPINFO)
If _WinAPI_CreateProcess('', @SystemDir & '\notepad.exe', 0, 0, 0, $CREATE_NEW_PROCESS_GROUP, 0, 0, $tStartup, $tProcess) Then
    ToolTip("Waiting notepad Closing")
    ProcessWaitClose(DllStructGetData($tProcess, 'ProcessID'))
Else
    _WinAPI_ShowLastError()
EndIf
