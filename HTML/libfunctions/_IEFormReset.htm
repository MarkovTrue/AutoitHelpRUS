Справочник функций

_IEFormReset

Сбрасывает указанную форму, возвращая её значения к загруженным по умолчанию.

#include <IE.au3>
_IEFormReset ( ByRef $oObject )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application или Form.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
@extended: Содержит номер недопустимого параметра.

См. также

_IEFormGetCollection, _IEFormGetObjByName, _IEFormSubmit

Пример

; Открытие браузера с примером формы, заполнение поля формы и
; сбросить форму на значения по умолчанию

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oText = _IEFormElementGetObjByName($oForm, "textExample")
_IEFormElementSetValue($oText, "Hey! It works!")
Sleep(2000)
_IEFormReset($oForm)
