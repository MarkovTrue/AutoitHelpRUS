_WinAPI_SetBrushOrg

Устанавливает начало кисти, которое GDI назначает следующей кисти, выбираемой приложением в указанном контексте устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetBrushOrg ( $hDC, $iX, $iY )

Параметры

$hDC Дескриптор контекста устройства.
$iX X-координата в единицах устройства нового начала кисти.
$iY Y-координата в единицах устройства нового начала кисти.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Вызовите _WinAPI_GetExtended() для получения структуры $tagPOINT, содержащей предыдущее начало кисти.

См. также
