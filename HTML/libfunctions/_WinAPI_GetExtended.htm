Справочник функций

_WinAPI_GetExtended

Получает последнее расширенное значение возврата функции.

#include <WinAPIMisc.au3>
_WinAPI_GetExtended ( )

Возвращаемое значение

Возвращает последнее расширенное значение.

Примечания

Обратите внимание, что не все функции возвращают расширенное значение.
Например, функция _WinAPI_CreateDIB() возвращает указатель на местоположение значений бит DIB.
Функция _WinAPI_GetExtended() должна быть вызвана сразу же после вызова соответствующей функции.
