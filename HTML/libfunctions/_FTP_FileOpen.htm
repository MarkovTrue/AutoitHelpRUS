Справочник функций

_FTP_FileOpen

Открывает доступ к удалённому файлу на FTP-сервере для чтения или записи.

#include <FTPEx.au3>
_FTP_FileOpen ( $hConnect, $sFileName [, $iAccess = $GENERIC_READ [, $iFlags = $INTERNET_FLAG_TRANSFER_BINARY [, $iContext = 0]]] )

Параметры

$hConnect возвращается функцией _FTP_Connect().
$sFileName Имя открываемого файла на FTP.
$iAccess [необязательный] Локальный атрибут доступа (по умолчанию $GENERIC_READ).
$iFlags [необязательный] Задаёт условия передачи. Выберите один тип передачи и любые флаги, управляющие кэшированием файла.

Тип передачи – одно из значений:
$FTP_TRANSFER_TYPE_ASCII (1) – передача файла методом FTP ASCII (тип A). Управляющая информация и форматирование преобразуются в локальные эквиваленты.
$FTP_TRANSFER_TYPE_BINARY (2) – передача файла методом FTP Image (тип I). Файл передаётся как есть, без изменений. Метод передачи по умолчанию.
$FTP_TRANSFER_TYPE_UNKNOWN (0) – соответствует $FTP_TRANSFER_TYPE_BINARY.
$INTERNET_FLAG_TRANSFER_ASCII (1) – передача файла как ASCII.
$INTERNET_FLAG_TRANSFER_BINARY (2) – передача файла как двоичного.

Следующие значения управляют кэшированием файла. Можно использовать одно или несколько:
$INTERNET_FLAG_HYPERLINK (0x00000400) – принудительная перезагрузка, если сервер не вернул время Expires и LastModified при определении необходимости перезагрузки элемента из сети.
$INTERNET_FLAG_NEED_FILE (0x00000010) – создаёт временный файл, если файл нельзя кэшировать.
$INTERNET_FLAG_RELOAD (0x80000000) – принудительная загрузка запрошенного файла, объекта или списка каталога с исходного сервера, а не из кэша.
$INTERNET_FLAG_RESYNCHRONIZE (0x00000800) – перезагружает HTTP-ресурсы, если ресурс изменился с момента последней загрузки. Все ресурсы FTP и Gopher перезагружаются.
$iContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает дескриптор FTP-файла для чтения/записи через _FTP_FileRead().
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
