Справочник ключевых слов

$tagGDIPBITMAPDATA

Данные точечного рисунка.

Global Const $tagGDIPBITMAPDATA = "uint Width;uint Height;int Stride;uint Format;ptr Scan0;uint_ptr Reserved"

Параметры

Width Количество пиксельв одной строке растра растрового изображения.
Height Количество строк растра в растровом изображении.
Stride Смещение в байтах между последовательными строками растра растрового изображения.
Если страйд положительный, растровое изображение имеет ориентацию сверху вниз.
Если страйд отрицательный, растровое изображение имеет ориентацию снизу вверх.
Format Указывает формат пиксельв растрового изображения.
Scan0 Указатель на первую (индекс 0) строку растра растрового изображения.
Reserved Зарезервировано для будущего использования.
