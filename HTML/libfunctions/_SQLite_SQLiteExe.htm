Справочник функций

_SQLite_SQLiteExe

Выполняет команды в sqlite3.exe.

#include <SQLite.au3>
_SQLite_SQLiteExe ( $sDatabaseFile, $sInput, ByRef $sOutput [, $sSQLiteExeFilename = "sqlite3.exe" [, $bDebug = False]] )

Параметры

$sDatabaseFile Имя файла базы данных.
$sInput Команды для sqlite3.exe.
$sOutput Необработанный вывод из sqlite3.exe.
$sSQLiteExeFilename [необязательный] Путь к sqlite3.exe
$bDebug [необязательный] Запись кода выхода sqlite3.exe через функцию обратного вызова, указанную в _SQLite_Startup() (по умолчанию не выводить)

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: 1 – не удалось создать новую БД
2 – sqlite3.exe не найден
3 – ошибка SQL / неполный SQL
4 – не удалось открыть входной файл

Примечания

Запустите sqlite3.exe, нажмите Enter и введите .Help для получения дополнительной информации о командах SQLite3.exe.

Если включён #include <SQLite.dll.au3>, используется версия SQLite.
Другие версии можно найти на веб-сайте SQLite.

Пример

#include <File.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>
#include <StringConstants.au3>

; Имена файлов
Local $sTsvFile = FileGetShortName(_TempFile(@TempDir, "~", ".tsv"))
Local $sDbFile = FileGetShortName(_TempFile(@TempDir, "~", ".db"))

; Создание файла TSV
FileWriteLine($sTsvFile, "a" & @TAB & "b" & @TAB & "c")
FileWriteLine($sTsvFile, "a1" & @TAB & "b1" & @TAB & "c1")
FileWriteLine($sTsvFile, "a2" & @TAB & "b2" & @TAB & "c2")

; импорт (используя SQLite3.exe)
Local $sIn, $sOut, $i, $sCreate = "CREATE TABLE TblImport (";
For $i = 1 To _StringCountOccurance(FileReadLine($sTsvFile, 1), @TAB) + 1
    $sCreate &= "Column_" & $i & ","
Next
$sCreate = StringTrimRight($sCreate, 1) & ");"
$sIn = $sCreate & @CRLF ; Создать таблицу
$sIn &= ".separator \t" & @CRLF ; Выбор @TAB в качестве разделителя
$sIn &= ".import '" & $sTsvFile & "' TblImport" & @CRLF
_SQLite_SQLiteExe($sDbFile, $sIn, $sOut, -1, True)

If @error = 0 Then
    ; Отображение таблицы (используя SQLite3.dll)
    Local $iRows, $iColumns, $aRes
    _SQLite_Startup()
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
        Exit -1
    EndIf
    ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
    _SQLite_Open($sDbFile)
    _SQLite_GetTable2D(-1, "SELECT ROWID,* FROM TblImport;", $aRes, $iRows, $iColumns)
    _SQLite_Display2DResult($aRes) ; Вывод в консоль
    _SQLite_Close()
    _SQLite_Shutdown()
Else
    If @error = 2 Then
        ConsoleWrite("ERROR: Sqlite3.exe file not found" & @CRLF)
    Else
        ConsoleWrite("ERROR: @error=" & @error & " when calling _SQLite_SQLiteExe" & @CRLF)
    EndIf
EndIf

; Удаление временных файлов
FileDelete($sTsvFile)
FileDelete($sDbFile)

; Вывод:
; rowid Column_1 Column_2 Column_3
; 1 a b c
; 2 a1 b1 c1
; 3 a2 b2 c2

Func _StringCountOccurance($sSearchString, $sSubString, $iCaseSense = $STR_NOCASESENSE) ; Возврат количества $sSubString в $sSearchString
    Local $iOccCnt = 1
    Do
        If StringInStr($sSearchString, $sSubString, $iCaseSense, $iOccCnt) > 0 Then
            $iOccCnt += 1
        Else
            ExitLoop
        EndIf
    Until 0
    Return $iOccCnt - 1
EndFunc   ;==>_StringCountOccurance
