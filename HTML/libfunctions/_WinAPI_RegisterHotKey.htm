Справочник функций

_WinAPI_RegisterHotKey

Определяет системную горячую клавишу.

#include <WinAPISys.au3>
_WinAPI_RegisterHotKey ( $hWnd, $iID, $iModifiers, $vKey )

Параметры

$hWnd Дескриптор окна.
$iID Идентификатор горячей клавиши. Приложение должно указать значение id в диапазоне
0x0000 по 0xBFFF.
$iModifiers Клавиши, которые должны нажиматься в сочетании с клавишей, указанной параметром $vKey
для создания сообщения WM_HOTKEY.
Параметр $iModifiers может быть комбинацией следующих значений.
$MOD_ALT
$MOD_CONTROL
$MOD_SHIFT
$MOD_WIN

Windows 7 или более поздняя версия
$MOD_NOREPEAT
$vKey Код виртуальной клавиши горячей клавиши ($VK_*).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Когда нажимается клавиша, система ищет совпадение со всеми горячими клавишами. Найдя совпадение, система отправляет
сообщение WM_HOTKEY в очередь сообщений окна, с которым связана горячая клавиша. Если горячая клавиша не
связана с окном, то сообщение WM_HOTKEY отправляется потоку, связанному с горячей клавишей.

_WinAPI_RegisterHotKey() завершается ошибкой, если последовательность клавиш, указанная для горячей клавиши, уже зарегистрирована
другой горячей клавишей.

В Windows XP, если горячая клавиша уже существует с теми же параметрами $hWnd и $iID,
она заменяется новой горячей клавишей.

В Windows Vista и последующих версиях Windows, если горячая клавиша уже существует с теми же параметрами $hWnd и $iID,
она сохраняется вместе с новой горячей клавишей. В этих версиях Windows приложение должно
явно вызвать _WinAPI_UnregisterHotKey(), чтобы отменить регистрацию старой горячей клавиши.

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIConv.au3>
#include <WinAPISys.au3>
#include <WindowsNotifsConstants.au3>

Opt('TrayAutoPause', 0)

OnAutoItExitRegister('OnAutoItExit')

Local $hWnd = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'))
GUIRegisterMsg($WM_HOTKEY, 'WM_HOTKEY')

; Установить ALT-D
_WinAPI_RegisterHotKey($hWnd, 0x0144, $MOD_ALT, 0x44)
; Установить ESC
_WinAPI_RegisterHotKey($hWnd, 0x011B, 0, 0x1B)

While 1
    Sleep(1000)
WEnd

Func WM_HOTKEY($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg, $wParam

    Switch _WinAPI_HiWord($lParam)
        Case 0x44
            MsgBox($MB_SYSTEMMODAL, '', 'You pressed ALT-D')
        Case 0x1B
            MsgBox($MB_SYSTEMMODAL, '', 'You pressed ESC')
            Exit
    EndSwitch
EndFunc   ;==>WM_HOTKEY

Func OnAutoItExit()
    _WinAPI_UnregisterHotKey($hWnd, 0x0144)
    _WinAPI_UnregisterHotKey($hWnd, 0x011B)
EndFunc   ;==>OnAutoItExit
