Справочник функций

_WinAPI_GetErrorMode

Получает режим ошибок текущего процесса.

#include <WinAPIDiag.au3>
_WinAPI_GetErrorMode ( )

Возвращаемое значение

Успех: Возвращает режим ошибок процесса ($SEM_*).
Ошибка: Возвращает 0.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
