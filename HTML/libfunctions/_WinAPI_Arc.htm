Справочник функций

_WinAPI_Arc

Рисует эллиптическую дугу.

#include <WinAPIGdi.au3>
_WinAPI_Arc ( $hDC, $tRECT, $iXStartArc, $iYStartArc, $iXEndArc, $iYEndArc )

Параметры

$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT, содержащая логические координаты ограничивающего прямоугольника.
$iXStartArc Координата x (в логических единицах) конечной точки радиальной линии, определяющей начальную точку дуги.
$iYStartArc Координата y (в логических единицах) конечной точки радиальной линии, определяющей начальную точку дуги.
$iXEndArc Координата x (в логических единицах) конечной точки радиальной линии, определяющей конечную точку дуги.
$iYEndArc Координата y (в логических единицах) конечной точки радиальной линии, определяющей конечную точку дуги.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Дуга рисуется с использованием текущего пера; она не заполняется. Текущая позиция ни используется, ни обновляется функцией _WinAPI_Arc().

Используйте функции _WinAPI_GetArcDirection() и _WinAPI_SetArcDirection() для получения и установки текущего направления рисования для контекста устройства.
По умолчанию направление рисования против часовой стрелки.
