Справочник функций

_Word_DocSaveAs

Сохраняет указанный документ Microsoft Word с новым именем или форматом.

#include <Word.au3>
_Word_DocSaveAs ( $oDoc [, $sFileName = Default [, $iFileFormat = $WdFormatDocument [, $bReadOnlyRecommended = False [, $bAddToRecentFiles = True [, $sPassword = "" [, $sWritePassword = ""]]]]]] )

Параметры

$oDoc Объект документа Word.
$sFileName [необязательный] Полное имя пути для документа (по умолчанию = текущая папка и текущее имя файла, если документ был сохранён ранее). Если документ никогда не был сохранён ранее, используется имя по умолчанию и документ сохраняется в папку MyDocuments, например "@MyDocumentsDir\Doc1.doc".
$iFileFormat [необязательный] Формат, в котором сохраняется документ. Может быть любой константой WdSaveFormat (по умолчанию = $WdFormatDocument)
$bReadOnlyRecommended [необязательный] True для предложения в Microsoft Word статуса только для чтения при каждом открытии документа (по умолчанию = False)
$bAddToRecentFiles [необязательный] True добавить документ в список недавно использованных файлов (по умолчанию = True)
$sPassword [необязательный] Строка пароля для открытия документа (по умолчанию = "")
$sWritePassword [необязательный] Строка пароля для сохранения изменений в документе (по умолчанию = "")

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $oDoc не является объектом 2 – При сохранении указанного документа произошла ошибка. @extended задан код ошибки COM

Примечания

Если документ с указанным именем файла уже существует, документ перезаписывается без предварительного запроса пользователю.

Пример

#include "Extras\HelpFileInternals.au3"

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <Word.au3>

; Скопировать \Extras\Test.doc в @TempDir
If FileCopy(_Extras_PathFull("Test.doc"), @TempDir & "\_Word_Test.doc", $FC_OVERWRITE) = 0 Then Exit MsgBox($MB_SYSTEMMODAL, _
        "Word UDF: _Word_DocSaveAs Example", "Couldn't copy '.\Extras\Test.doc' as '_Word_Test.doc' to the @TempDir directory.")
; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocSaveAs Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ
Local $oDoc = _Word_DocOpen($oWord, @TempDir & "\_Word_Test.doc")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocSaveAs Example", _
        "Error opening '_Word_Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Сохранить документ как _Word_Test2.doc
; Вставить текст в начале
Local $oRange = _Word_DocRangeSet($oDoc, -1)
$oRange.Text = "Bold text at the beginning. "
$oRange.Bold = True
; Сохранить документ
_Word_DocSaveAs($oDoc, @TempDir & "\_Word_Test2.doc")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocSaveAs Example", _
        "Error saving the Word document." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocSaveAs Example", "Document successfully saved as '" & _
        @TempDir & "\_Word_Test2.doc'.")
