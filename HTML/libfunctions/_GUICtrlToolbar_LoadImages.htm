Справочник функций

_GUICtrlToolbar_LoadImages

Загружает системные значки кнопок в список изображений элемента управления панели инструментов.

#include <GuiToolbar.au3>
_GUICtrlToolbar_LoadImages ( $hWnd, $iBitMapID )

Параметры

$hWnd Дескриптор элемента управления.
$iBitMapID Идентификатор системного списка изображений кнопок. Может быть одним из следующих значений:
$IDB_HIST_LARGE_COLOR
$IDB_HIST_SMALL_COLOR
$IDB_STD_LARGE_COLOR
$IDB_STD_SMALL_COLOR
$IDB_VIEW_LARGE_COLOR
$IDB_VIEW_SMALL_COLOR
$IDB_HIST_NORMAL
$IDB_HIST_HOT
$IDB_HIST_DISABLED
$IDB_HIST_PRESSED

Возвращаемое значение

Успех: Возвращает количество изображений в списке без учёта только что добавленных.
Ошибка: Возвращает 0.

Пример

#include <GUIConstantsEx.au3>
#include <GuiToolbar.au3>
#include <WinAPIConstants.au3>

Example()

Func Example()
    Local $hGUI, $hToolbar, $aStrings[4]
    Local Enum $e_idNew = 1000, $e_idOpen, $e_idSave, $e_idHelp

    ; Создать GUI
    $hGUI = GUICreate("Toolbar", 400, 300)
    $hToolbar = _GUICtrlToolbar_Create($hGUI)
    GUISetState(@SW_SHOW)

    ; Добавить стандартные системные изображения
    _GUICtrlToolbar_SetBitmapSize($hToolbar, 24, 24)
    _GUICtrlToolbar_LoadImages($hToolbar, $IDB_STD_LARGE_COLOR)

    ; Добавить строки
    $aStrings[0] = _GUICtrlToolbar_AddString($hToolbar, "&New Button")
    $aStrings[1] = _GUICtrlToolbar_AddString($hToolbar, "&Open Button")
    $aStrings[2] = _GUICtrlToolbar_AddString($hToolbar, "&Save Button")
    $aStrings[3] = _GUICtrlToolbar_AddString($hToolbar, "&Help Button")

    ; Добавить кнопки
    _GUICtrlToolbar_AddButton($hToolbar, $e_idNew, $STD_FILENEW, $aStrings[0])
    _GUICtrlToolbar_AddButton($hToolbar, $e_idOpen, $STD_FILEOPEN, $aStrings[1])
    _GUICtrlToolbar_AddButton($hToolbar, $e_idSave, $STD_FILESAVE, $aStrings[2])
    _GUICtrlToolbar_AddButtonSep($hToolbar)
    _GUICtrlToolbar_AddButton($hToolbar, $e_idHelp, $STD_HELP, $aStrings[3])

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
