Справочник функций

_DebugCOMError

Устанавливает, сбрасывает или запрашивает уровень отладки COM-ошибок.

#include <Debug.au3>
_DebugCOMError ( [$iComDebug = 1 [, $bExit = False]] )

Параметры

$iComDebug [необязательный] Уровень отладки. Возможные значения:
-1 – вернуть текущие настройки
0 – отключить отладку
1 – включить отладку; вывести отладочную информацию (по умолчанию)
$bExit [необязательный] True – скрипт должен быть завершён

Возвращаемое значение

Успех: Возвращает 1, если $iComDebug = 0 или 1 и устанавливает @extended:
0 – обработчик COM-ошибок не инициализирован
1 – обработчик COM-ошибок инициализирован
2 – обработчик COM-ошибок уже был активен
3 – обработчик COM-ошибок уже отключён
Возвращает 1, если $iComDebug = -1 и устанавливает @error и @extended с текущими параметрами:
@error = текущий уровень отладки
@extended = текущий флаг завершения скрипта.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $iComDebug не является целым числом или выходит за пределы [-1, 1]
2 – обработчик COM-ошибок уже назначен другой функции (не этой)
3 – _DebugSetup() не была корректно выполнена; убедитесь, что _DebugSetup() запущена перед вызовом этой функции
4 – установка пользовательского обработчика ошибок не удалась.
@extended: Содержит код ошибки, возвращённый ObjEvent().

Примечания

Перед вызовом этой функции необходимо вызвать _DebugSetup() для создания сеанса отладки.

См. также

_DebugSetup

Пример

#include <Debug.au3>
#include <MsgBoxConstants.au3>

_DebugSetup(Default, True, 1, Default, True) ; 1 = Записывать сообщения с временными метками в окно журнала отчётов.

Example()

; Пример
; Вызываем COM-ошибку и позволяем _DebugCOMError() отобразить сообщения об ошибке в окне журнала.
Func Example()
    ; Создаём объект приложения Microsoft Internet Explorer.
    Local $oIE = ObjCreate("InternetExplorer.Application")
    If @error <> 0 Then Exit MsgBox($MB_SYSTEMMODAL, "_DebugCOMError UDF", "Error creating a new IE application object." & _
            @CRLF & "@error = " & @error & ", @extended = " & @extended)
    ; Настраиваем обработчик COM-ошибок для записи сообщений об ошибке в окно журнала.
    _DebugCOMError(1)
    ; Вызываем COM-ошибку. Функция неизвестна.
    $oIE.xyz()
    If @error <> 0 Then Return MsgBox($MB_SYSTEMMODAL, "_DebugCOMError UDF", "Error occurred and handled." & _
            @CRLF & "@error = " & @error)
    ; Закрываем Internet Explorer.
    $oIE.Quit()
    $oIE = 0
EndFunc   ;==>Example
