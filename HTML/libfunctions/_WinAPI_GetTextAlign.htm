_WinAPI_GetTextAlign

Получает параметр выравнивания текста для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_GetTextAlign ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Состояние флагов выравнивания текста. Возвращаемое значение является комбинацией констант $TA_* и $VTA_*.
Ошибка: (-1).
