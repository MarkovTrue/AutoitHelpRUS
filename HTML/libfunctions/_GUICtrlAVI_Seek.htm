Справочник функций

_GUICtrlAVI_Seek

Указывает элементу AVI отобразить определённый кадр AVI-клипа.

#include <GuiAVI.au3>
_GUICtrlAVI_Seek ( $hWnd, $iFrame )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iFrame Индекс кадра для отображения (отсчёт от 0).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_GUICtrlAVI_Play

Пример

Пример 1

; используя UDF

#include <GuiAVI.au3>
#include <GUIConstantsEx.au3>

_Example_Internal()

Func _Example_Internal()
    ; Создать GUI
    GUICreate("(Internal) AVI Seek", 300, 100)
    Local $idAVI = GUICtrlCreateAvi(@SystemDir & "\shell32.dll", 160, 10, 10)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    Do
        Sleep(100)
        ; Перейти к случайному кадру в AVI-ролике
        _GUICtrlAVI_Seek($idAVI, Random(1, 30, 1))
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Закрыть AVI-клип
    _GUICtrlAVI_Close($idAVI)

    GUIDelete()
EndFunc   ;==>_Example_Internal
