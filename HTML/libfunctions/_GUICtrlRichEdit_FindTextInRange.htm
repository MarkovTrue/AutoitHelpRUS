_GUICtrlRichEdit_FindTextInRange

Ищет текст в диапазоне позиций между символами.

#include <GuiRichEdit.au3>
_GUICtrlRichEdit_FindTextInRange ( $hWnd, $sText [, $iStart = 0 [, $iEnd = -1 [, $bMatchCase = False [, $bWholeWord = False [, $iBehavior = 0]]]]] )

Параметры

$hWnd Дескриптор элемента управления.
$sText Текст для поиска.
$iStart [необязательный] Начальная межсимвольная позиция поиска.
По умолчанию – начало элемента.
$iEnd [необязательный] Конечная межсимвольная позиция поиска.
По умолчанию – конец элемента.
$bMatchCase [необязательный] Поиск с учётом регистра.
По умолчанию – без учёта регистра.
$bWholeWord [необязательный] Искать только как целое слово.
По умолчанию – допускается частичное совпадение.
$iBehavior [необязательный] Любое сочетание (BitOR) $FR_MATCHALEFHAMZA, $FR_MATCHDIAC и $FR_MATCHKASHIDA.
По умолчанию: 0.

Возвращаемое значение

Успех: Возвращает True, если текст найден.
Ошибка: Устанавливает @error ≠ 0.
@error: 101 – $hWnd не является дескриптором
102 – $sText = ""
103 – $iStart не является положительным числом, нулём или -1
104 – $iEnd не является положительным числом, нулём или -1
105 – $bMatchCase должен быть True или False
106 – $bWholeWord должен быть True или False
1071 – $iBehavior не является числом
1072 – значение $iBehavior недопустимо

Примечания

Позиция между символами в начале элемента управления – 0.
Диапазон символов по умолчанию (0 до -1) осуществляет поиск во всём тексте вниз.
Если установить $iEnd в -1, поиск выполняется вниз до конца элемента.
Если установить $iStart в -1, поиск выполняется вверх до начала элемента.

Пример

#include <GUIConstantsEx.au3>
#include <GuiRichEdit.au3>
#include <WindowsStylesConstants.au3>

Global $g_idLbl_Msg

Example()

Func Example()
    Local $hGui, $iMsg, $aiPos, $hRichEdit
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName, StringLen(".exe")) & ")", 320, 350, -1, -1)
    $hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 300, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    $g_idLbl_Msg = GUICtrlCreateLabel("", 10, 235, 300, 60)
    GUISetState(@SW_SHOW)

    _GUICtrlRichEdit_AppendText($hRichEdit, @CRLF & "This is appended text.")

    $aiPos = _GUICtrlRichEdit_FindTextInRange($hRichEdit, "test", 20, 6)
    Report('"test" found between inter-character positions ' & $aiPos[0] & " and " & $aiPos[1])

    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                _GUICtrlRichEdit_Destroy($hRichEdit) ; нужно, если только скрипт не упадёт
                ; GUIDelete() ; так тоже работает
                Exit
        EndSelect
    WEnd
EndFunc   ;==>Example

Func Report($sMsg)
    GUICtrlSetData($g_idLbl_Msg, $sMsg)
EndFunc   ;==>Report
