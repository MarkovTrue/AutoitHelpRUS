Справочник функций

_GUICtrlIpAddress_IsBlank

Определяет, пусты ли все поля в элементе управления IP-адрес.

#include <GuiIPAddress.au3>
_GUICtrlIpAddress_IsBlank ( $hWnd )

Параметры

$hWnd Дескриптор элемента управления.

Возвращаемое значение

True: Возвращает IP-адрес пустой.
False: IP-адрес не пустой.

См. также

_GUICtrlIpAddress_ClearAddress

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiIPAddress.au3>

Example()

Func Example()
    Local $hGui, $hIPAddress

    $hGui = GUICreate("IP Address Control IsBlank Example", 400, 300)
    $hIPAddress = _GUICtrlIpAddress_Create($hGui, 2, 4)
    _MemoCreate(2, 28, 396, 270, 0)
    GUISetState(@SW_SHOW)

    ; Проверить, пустой ли IP-адрес
    _MemoWrite("Blank: " & _GUICtrlIpAddress_IsBlank($hIPAddress))

    Sleep(1000)

    _GUICtrlIpAddress_Set($hIPAddress, "24.168.2.128")

    ; Проверить, пустой ли IP-адрес
    _MemoWrite("Blank: " & _GUICtrlIpAddress_IsBlank($hIPAddress))

    ; Ждать закрытия GUI пользователем
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
