Справочник функций

_WinAPI_WaitForMultipleObjects

Ожидает, пока один или все указанные объекты находятся в сигнальном состоянии.

#include <WinAPIProc.au3>
_WinAPI_WaitForMultipleObjects ( $iCount, $paHandles [, $bWaitAll = False [, $iTimeout = -1]] )

Параметры

$iCount Количество дескрипторов объектов в массиве, на который указывает $paHandles.
$paHandles Указатель на массив дескрипторов объектов.
$bWaitAll [необязательный] Если True, функция возвращается, когда состояние всех объектов в массиве $paHandles имеет сигнальное значение.
Если False, функция возвращается, когда состояние любого одного объекта имеет сигнальное значение.
В последнем случае возвращаемое значение указывает объект, состояние которого вызвало возврат функции.
$iTimeout [необязательный] Интервал времени ожидания в миллисекундах.
Функция возвращается, если истекает интервал, даже если условия, указанные параметром $bWaitAll, не выполняются.
Если 0, функция проверяет состояния указанных объектов и немедленно возвращается.
Если -1, интервал времени ожидания функции никогда не истекает.

Возвращаемое значение

Успех: Событие, которое вызвало возврат функции.
Ошибка: (-1) WAIT_FAILED; вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке.

Пример

#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

Global $g_tEvents = DllStructCreate("handle Event[3];")
$g_tEvents.Event(1) = _WinAPI_CreateEvent(0, True, False)
$g_tEvents.Event(2) = _WinAPI_CreateEvent(0, True, False)
$g_tEvents.Event(3) = _WinAPI_CreateEvent(0, True, False)

HotKeySet("{ESC}", "_Exit")
AdlibRegister("_FireEvent_1", 500)
AdlibRegister("_FireEvent_2", 800)

Local $iEvent
While 1
    $iEvent = _WinAPI_WaitForMultipleObjects(3, $g_tEvents, False, 100)
    Switch $iEvent
        Case 0
            ConsoleWrite("+ First Event" & @CRLF)
            _WinAPI_ResetEvent($g_tEvents.Event(1))
        Case 1
            ConsoleWrite("> Second Event" & @CRLF)
            _WinAPI_ResetEvent($g_tEvents.Event(2))
        Case 2
            ConsoleWrite("! Exit Event" & @CRLF)
            AdlibUnRegister("_FireEvent_1")
            AdlibUnRegister("_FireEvent_2")
            _WinAPI_CloseHandle($g_tEvents.Event(1))
            _WinAPI_CloseHandle($g_tEvents.Event(2))
            _WinAPI_CloseHandle($g_tEvents.Event(3))
            ExitLoop
        Case -1 ;Ошибка
            ExitLoop
    EndSwitch
    Sleep(10)
WEnd

Func _Exit()
    _WinAPI_SetEvent($g_tEvents.Event(3))
EndFunc   ;==>_Exit

Func _FireEvent_1()
    _WinAPI_SetEvent($g_tEvents.Event(1))
EndFunc   ;==>_FireEvent_1

Func _FireEvent_2()
    _WinAPI_SetEvent($g_tEvents.Event(2))
EndFunc   ;==>_FireEvent_2
