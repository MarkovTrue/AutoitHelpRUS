Справочник функций

_WinAPI_DrawThemeEdge

Рисует одно или несколько рёбер прямоугольника, заданных визуальным стилем.

#include <WinAPITheme.au3>
_WinAPI_DrawThemeEdge ( $hTheme, $iPartID, $iStateID, $hDC, $tRECT, $iEdge, $iFlags [, $tAREA = 0] )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, в которой расположен прямоугольник.
$iStateID Состояние части.
$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT с прямоугольником.
$iEdge Тип внутреннего и внешнего рёбер.
Параметр – комбинация одного флага внутренней рамки и одного флага внешней ($BDR_*) либо одна из объединённых констант ($EDGE_*).
$iFlags Тип границы. Может быть комбинацией констант $BF_*.
$tAREA [необязательный] Структура $tagRECT, в которую записывается внутренний прямоугольник, если задан флаг $BF_ADJUST.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Константы $BDR_*, $EDGE_*, $BF_* требуют #include <BorderConstants.au3>.
