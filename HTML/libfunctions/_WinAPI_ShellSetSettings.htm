Справочник функций

_WinAPI_ShellSetSettings

Задаёт параметры состояния оболочки.

#include <WinAPIShellEx.au3>
_WinAPI_ShellSetSettings ( $iFlags, $bSet )

Параметры

$iFlags Флаги, которые указывают, какие параметры должны быть установлены. Этот параметр может быть одним или несколькими из следующих значений (используйте ТОЛЬКО эти флаги). $SSF_SHOWALLOBJECTS $SSF_SHOWEXTENSIONS $SSF_SHOWCOMPCOLOR $SSF_SHOWSYSFILES $SSF_DOUBLECLICKINWEBVIEW $SSF_DESKTOPHTML $SSF_WIN95CLASSIC $SSF_DONTPRETTYPATH $SSF_MAPNETDRVBUTTON $SSF_SHOWINFOTIP $SSF_HIDEICONS $SSF_NOCONFIRMRECYCLE $SSF_WEBVIEW $SSF_SHOWSUPERHIDDEN $SSF_SEPPROCESS $SSF_NONETCRAWLING $SSF_STARTPANELON
Windows Vista и более поздние версии $SSF_AUTOCHECKSELECT $SSF_ICONSONLY $SSF_SHOWTYPEOVERLAY.
$bSet Указывает, должен ли параметр ($SSF_*) быть включён или отключён, допустимые значения: True – включить. False – отключить.

Возвращаемое значение

Отсутствует.

Дополнительно

Искать SHGetSetSettings в библиотеке MSDN

Пример

#include <WinAPIShellEx.au3>

_WinAPI_ShellSetSettings($SSF_SHOWEXTENSIONS, Not _WinAPI_ShellGetSettings($SSF_SHOWEXTENSIONS))

ConsoleWrite('Hide extensions for known file types: ' & (Not _WinAPI_ShellGetSettings($SSF_SHOWEXTENSIONS)) & @CRLF)
