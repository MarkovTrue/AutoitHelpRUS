Справочник функций

_WinAPI_GetOpenFileName

Создаёт диалоговое окно "Открыть", которое позволяет пользователю указать диск, каталог и имя файла или набора файлов для открытия.

#include <WinAPIDlg.au3>
_WinAPI_GetOpenFileName ( [$sTitle = "" [, $sFilter = "All files (*.*)" [, $sInitalDir = "." [, $sDefaultFile = "" [, $sDefaultExt = "" [, $iFilterIndex = 1 [, $iFlags = 0 [, $iFlagsEx = 0 [, $hWndOwner = 0]]]]]]]]] )

Параметры

$sTitle [необязательный] Строка для размещения в строке заголовка диалогового окна
$sFilter [необязательный] Пары строк фильтра (например "Текстовые файлы (*.txt)|Все файлы (*.*)")
Первая строка в каждой паре – отображаемая строка, описывающая фильтр (например, "Текстовые файлы")
Вторая строка задаёт шаблон фильтра (например, "*.TXT")
Для задания нескольких шаблонов фильтра для одной отображаемой строки используйте точку с запятой для разделения шаблонов (например, "*.TXT;*.DOC;*.BAK")
Строка шаблона может быть комбинацией действительных символов имён файлов и подстановочного символа астериск (*)
Не включайте пробелы в строку шаблона.
$sInitalDir [необязательный] Строка, которая может задать исходный каталог
$sDefaultFile [необязательный] Имя файла, используемое для инициализации элемента управления редактирования имена файла
$sDefaultExt [необязательный] Строка, содержащая расширение по умолчанию
$iFilterIndex [необязательный] Задаёт индекс выбранного в данный момент фильтра в элементе управления "Тип файлов"
$iFlags [необязательный] See Flags в информации $tagOPENFILENAME
$iFlagsEx [необязательный] See FlagEx в информации $tagOPENFILENAME
$hWndOwner [необязательный] Дескриптор окна, которому принадлежит диалоговое окно. Это может быть любой действительный дескриптор окна или 0, если диалоговое окно не имеет хозяина

Возвращаемое значение

Успех: Возвращает массив в следующем формате:
$aArray[0] – содержит число строк
$aArray[1] – содержит выбранный путь
$aArray[2] – содержит выбранный файл
[n] – содержит выбранный файл
Ошибка: Устанавливает флаг @error в ненулевое значение, вызовите _WinAPI_CommDlgExtendedError() для получения расширенной информации об ошибке.

См. также

$tagOPENFILENAME, _WinAPI_CommDlgExtendedError, _WinAPI_GetSaveFileName

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <WinAPIConstants.au3>
#include <WinAPIDlg.au3>
#include <WindowsStylesConstants.au3>

_Example_Defaults()
_Example_ExplorerStyleMultiSelect()
_Example_OldStyle()
_Example_ExplorerStyleSingleSelect()
_Example_ExplorerStyle_NoPlaceBar()

Func _Example_Defaults()
    Local $hGui, $idBtn_Dialog, $aFile, $sError

    ; Создать GUI
    $hGui = GUICreate("GetOpenFileName use defaults", 400, 296)
    _MemoCreate(2, 32, 396, 226, $WS_HSCROLL)
    $idBtn_Dialog = GUICtrlCreateButton("Open Dialog", 155, 270, 90, 20)
    GUISetState(@SW_SHOW)

    While 1
        Switch GUIGetMsg()
            Case $idBtn_Dialog
                $aFile = _WinAPI_GetOpenFileName() ; используйте значения по умолчанию
                If @error Then
                    $sError = _WinAPI_CommDlgExtendedError()
                    _MemoWrite("CommDlgExtendedError (" & @error & "): " & $sError)
                Else
                    For $x = 1 To $aFile[0]
                        _MemoWrite($aFile[$x])
                    Next
                EndIf
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
    GUIDelete($hGui)
EndFunc   ;==>_Example_Defaults

Func _Example_ExplorerStyleMultiSelect()

    ; Создать GUI

    _MemoCreate(2, 32, 396, 226, $WS_HSCROLL)

    While 1
        Switch GUIGetMsg()
            Case $idBtn_Dialog
                $aFile = _WinAPI_GetOpenFileName("My Open File Dialog", _
                        "Text File (*.txt;*.au3)", ".", @ScriptName, "", 1, _
                        BitOR($OFN_ALLOWMULTISELECT, $OFN_EXPLORER), 0, $hGui)

                    Next
                EndIf
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
    GUIDelete($hGui)
EndFunc   ;==>_Example_ExplorerStyleMultiSelect
