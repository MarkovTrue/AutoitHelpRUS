Справочник функций

_viOpen

Открывает VISA-соединение с прибором или устройством.

#include <Visa.au3>
_viOpen ( $sVisa_Address [, $sVisa_Secondary_Address = 0] )

Параметры

$sVisa_Address Строка дескриптора ресурса VISA (см. ПРИМЕЧАНИЯ функции _viExecCommand() выше для дополнительной информации)
В качестве сокращения вы также можете напрямую передать адрес GPIB в виде целого числа.
$sVisa_Secondary_Address [необязательный] "Вторичный адрес GPIB". Используется только если основной адрес передан как целое число.
Только некоторые приборы GPIB имеют вторичные адреса. В этих случаях этот необязательный параметр может использоваться для его указания.
По умолчанию этот параметр равен НУЛЮ, что означает отсутствие вторичного адреса.

Возвращаемое значение

Успех: (ПОЛОЖИТЕЛЬНЫЙ) дескриптор прибора VISA.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.

Примечания

Как и для всех функций VISA, библиотеки VISA должны быть установлены (вы можете проверить, находится ли visa32.dll в {WINDOWS}\system32), а также должна быть установлена карта GPIB (такая как NI PCI-GPIB от National Instruments или Agilent 82350B PCI High-Performance GPIB)

* Подробное описание наиболее распространённых ДЕСКРИПТОРОВ VISA см. в примечаниях функции _viExecCommand()

См. также

_viClose, _viExecCommand, _viFindGpib, _viGpibBusReset, _viGTL, _viSetAttribute, _viSetTimeout

Пример

; - Предполагает, что приборубставлен на адрес GPIB 1
; Демонстрирует использование _viExecCommand в автономном режиме и в сочетании
; с _viOpen и _viClose.
; Также демонстрирует функцию _viGTL

#include <MsgBoxConstants.au3>
#include <Visa.au3>

Local $h_Session = 0

; Запросить ID прибора по адресу GPIB 3
MsgBox($MB_SYSTEMMODAL, "Step 1", "Open the instrument connection with _viOpen")
Local $h_Instr = _viOpen("GPIB::3::0")
MsgBox($MB_SYSTEMMODAL, "Instrument Handle obtained", "$h_Instr = " & $h_Instr) ; Показать дескриптор сессии
; Запросить прибор

MsgBox($MB_SYSTEMMODAL, "Step 2", "Query the instrument using the VISA instrument handle")
Local $s_Answer = _viExecCommand($h_Instr, "*IDN?") ; $h_Instr теперь НЕ СТРОКА!
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ
; Запросить снова. Нет необходимости открывать соединение заново

MsgBox($MB_SYSTEMMODAL, "Step 3", "Query again. There is no need to OPEN the link again")
$s_Answer = _viExecCommand($h_Instr, "*IDN?")
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

MsgBox($MB_SYSTEMMODAL, "Step 4", "Close the instrument connection using _viClose")
_viClose($h_Instr) ; Закрыть соединение с прибором

MsgBox($MB_SYSTEMMODAL, "Step 5", "Open the Instrument connection using only the address number")
$h_Instr = _viOpen(3)
MsgBox($MB_SYSTEMMODAL, "Instrument Handle obtained", "$h_Instr = " & $h_Instr) ; Показать дескриптор сессии
; Запросить прибор

MsgBox($MB_SYSTEMMODAL, "Step 6", "Query the instrument using the VISA instrument handle")
$s_Answer = _viExecCommand($h_Instr, "*IDN?") ; $h_Instr теперь НЕ СТРОКА!
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ
; Запросить снова. Нет необходимости открывать соединение заново

MsgBox($MB_SYSTEMMODAL, "Step 7", "Query again. There is no need to OPEN the link again")
$s_Answer = _viExecCommand($h_Instr, "*IDN?")
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

MsgBox($MB_SYSTEMMODAL, "Step 8", "Close the instrument connection using _viClose")
_viClose($h_Instr) ; Закрыть соединение с прибором
