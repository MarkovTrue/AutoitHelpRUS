Справочник функций

_IELinkClickByIndex

Имитирует клик мыши по ссылке с заданным индексом (отсчёт от 0, в порядке исходного кода).

#include <IE.au3>
_IELinkClickByIndex ( ByRef $oObject, $iIndex [, $iWait = 1] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$iIndex Индекс ссылки (отсчёт от 0), которую нужно найти.
$iWait [необязательный] Ждать ли загрузку страницы перед возвратом:
0 – вернуться сразу, не ожидая загрузки
1 – (по умолчанию) дождаться полной загрузки и вернуться

Возвращаемое значение

Успех: Нет.
Ошибка: Возвращает 0 или -1 и устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) – общая ошибка
2 ($_IEStatus_COMError) – ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
6 ($_IEStatus_LoadWaitTimeout) – истекло ожидание загрузки
7 ($_IEStatus_NoMatch) – нет совпадения
8 ($_IEStatus_AccessIsDenied) – доступ запрещён
9 ($_IEStatus_ClientDisconnected) – клиент отключён
@extended: Содержит номер недопустимого параметра.

Примечания

Не все элементы, выглядящие как ссылки, ими являются.
Частая практика – привязка JavaScript-событий onClick к другим DOM-элементам для имитации поведения ссылок.
Для активации таких элементов используйте "click" через _IEAction().

См. также

_IELinkClickByText, _IELoadWait

Пример

; Открытие браузера с базовым примером, щелчок по 3-й ссылке
; on the page (note: the first link is index 0)

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IELinkClickByIndex($oIE, 2)
