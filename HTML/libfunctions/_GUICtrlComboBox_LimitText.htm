_GUICtrlComboBox_LimitText

Ограничивает длину текста, который пользователь может вводить в поле ввода.

#include <GuiComboBox.au3>
_GUICtrlComboBox_LimitText ( $hWnd [, $iLimit = 0] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iLimit [необязательный] Ограничить длину текста (по умолчанию без ограничений)

Возвращаемое значение

Отсутствует.

Примечания

Если параметр $iLimit равен нулю, длина текста ограничена 0x7FFFFFFE символами.

Если ComboBox не имеет стиля $CBS_AUTOHSCROLL, установка предела текста больше размера поля редактирования не влияет.

Функция _GUICtrlComboBox_LimitText() ограничивает только текст, который может ввести пользователь. Она не влияет на текст, уже находящийся в поле редактирования при отправке сообщения, и не влияет на длину текста, скопированного в поле редактирования при выборе строки из ListBox.

Предел по умолчанию для текста, который пользователь может ввести в поле редактирования, составляет 30 000 символов.

Пример

#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $idCombo

    ; Создать GUI
    GUICreate("ComboBox Limit Text", 400, 296)
    $idCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    GUISetState(@SW_SHOW)

    ; Ограничить длину текста в поле ввода
    _GUICtrlComboBox_LimitText($idCombo, 10)

    ; Добавить файлы
    _GUICtrlComboBox_BeginUpdate($idCombo)
    _GUICtrlComboBox_AddDir($idCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($idCombo)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
