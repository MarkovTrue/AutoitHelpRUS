Справочник функций

_WinAPI_DrawText

Рисует форматированный текст в указанном прямоугольнике.

#include <WinAPIGdiDC.au3>
_WinAPI_DrawText ( $hDC, $sText, ByRef $tRECT, $iFlags )

Параметры

$hDC Идентифицирует контекст устройства.
$sText Строка для рисования.
$tRECT Структура $tagRECT, которая содержит прямоугольник для текста.
$iFlags Способ форматирования текста:
$DT_BOTTOM - Выравнивает текст к нижней части прямоугольника
$DT_CALCRECT - Определяет ширину и высоту прямоугольника
$DT_CENTER - Центрирует текст горизонтально в прямоугольнике
$DT_EDITCONTROL - Дублирует характеристики отображения текста многострочного элемента редактирования
$DT_END_ELLIPSIS - Заменяет часть строки многоточием, если необходимо
$DT_EXPANDTABS - Раскрывает символы табуляции
$DT_EXTERNALLEADING - Включает внешний интерлиньяж шрифта в высоту строки
$DT_HIDEPREFIX - Игнорирует символ амперсанда (&) в префиксе текста.
Буква после префикса не будет подчёркнута, но другие символы мнемонического префикса обрабатываются.
$DT_INTERNAL - Использует системный шрифт для расчёта метрик текста
$DT_LEFT - Выравнивает текст влево
$DT_MODIFYSTRING - Изменяет строку для соответствия отображаемому тексту
$DT_NOCLIP - Рисует без обрезки
$DT_NOFULLWIDTHCHARBREAK - Предотвращает разрыв строки в DBCS (двойной ширины), чтобы правило разрыва было эквивалентно SBCS.
Например, это может использоваться в корейских окнах для лучшей читаемости меток значков.
Это значение не имеет эффекта, если не указан $DT_WORDBREAK
$DT_NOPREFIX - Отключает обработку символов префикса
$DT_PATH_ELLIPSIS - Заменяет символы в середине строки многоточием, чтобы результат поместился в указанный прямоугольник.
Если строка содержит символы обратной косой черты (\), $DT_PATH_ELLIPSIS сохраняет максимальный объём текста после последней обратной косой черты.
Строка не изменяется, если не указан флаг $DT_MODIFYSTRING
$DT_PREFIXONLY - Рисует только подчёркивание под буквой после символа амперсанда (&).
Не рисует другие символы в строке
$DT_RIGHT - Выравнивает текст вправо
$DT_RTLREADING - Расположение в порядке чтения справа налево для двунаправленного текста
$DT_SINGLELINE - Отображает текст на одной строке только
$DT_TABSTOP - Устанавливает позиции табуляции. Биты 15-8 параметра $iFlags указывают количество символов для каждой табуляции
$DT_TOP - Выравнивает текст к верхней части (одна строка только)
$DT_VCENTER - Центрирует текст вертикально (одна строка только)
$DT_WORDBREAK - Разрывает слова
$DT_WORD_ELLIPSIS - Обрезает слово, которое не умещается в прямоугольнике, и добавляет многоточие.

Возвращаемое значение

Успех: Возвращает высота текста.
Ошибка: Возвращает 0.

Примечания

Функция DrawText использует выбранный шрифт контекста устройства, цвет текста и цвет фона для рисования текста.
Если не используется формат $DT_NOCLIP, DrawText обрезает текст так, чтобы он не отображался вне указанного прямоугольника.
Всё форматирование предполагает несколько строк, если не указан формат $DT_SINGLELINE.
Если выбранный шрифт слишком большой, DrawText не пытается подставить меньший шрифт.

Вышеуказанные константы требуют #include <APIGdiConstants.au3>

См. также

$tagRECT, _WinAPI_GetBkMode, _WinAPI_SetBkMode

Дополнительно

Искать DrawText в библиотеке MSDN

Пример

#include <FontConstants.au3>
#include <StructureConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPISysWin.au3>

Global $g_tRECT, $g_hFont, $g_hOldFont, $g_hDC

HotKeySet("{ESC}", "_Exit")

$g_tRECT = DllStructCreate($tagRect)
DllStructSetData($g_tRECT, "Left", 5)
DllStructSetData($g_tRECT, "Top", 5)
DllStructSetData($g_tRECT, "Right", 250)
DllStructSetData($g_tRECT, "Bottom", 50)

$g_hDC = _WinAPI_GetDC(0)
$g_hFont = _WinAPI_CreateFont(50, 0, 0, 0, 400, False, False, False, $DEFAULT_CHARSET, _
        $OUT_DEFAULT_PRECIS, $CLIP_DEFAULT_PRECIS, $DEFAULT_QUALITY, 0, 'Arial')
$g_hOldFont = _WinAPI_SelectObject($g_hDC, $g_hFont)

_WinAPI_SetTextColor($g_hDC, 0x0000FF)
_WinAPI_SetBkColor($g_hDC, 0x000000)
; закомментируйте следующую строку для чёрного фона вместо прозрачного
_WinAPI_SetBkMode($g_hDC, $TRANSPARENT)

While 1
    _WinAPI_DrawText($g_hDC, "Hello world!", $g_tRECT, $DT_CENTER)
    Sleep(100)
WEnd

Func _Exit()
    _WinAPI_SelectObject($g_hDC, $g_hOldFont)
    _WinAPI_DeleteObject($g_hFont)
    _WinAPI_ReleaseDC(0, $g_hDC)
    _WinAPI_InvalidateRect(0, 0)
    $g_tRECT = 0
    Exit
EndFunc   ;==>_Exit
