Справочник функций

_WinAPI_RegCopyTreeEx

Копирует указанный ключ реестра со всеми его значениями и подключами в указанный ключ назначения.

#include <WinAPIReg.au3>
_WinAPI_RegCopyTreeEx ( $hSrcKey, $sSrcSubKey, $hDestKey )

Параметры

$hSrcKey Дескриптор открытого ключа реестра. Ключ должен быть открыт с правом доступа $KEY_READ.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey(),
илиможет быть одним из предопределённых ключей реестра ($HKEY_*).
$sSrcSubKey Имя ключа. Этот ключ должен быть подключом ключа, определённого параметром $hSrcKey.
$hDestKey Дескриптор ключа назначения. Вызывающий процесс должен иметь доступ $KEY_CREATE_SUB_KEY к ключу.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey(),
или может быть одним из предопределённых ключей реестра ($HKEY_*).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Эта функция также копирует дескриптор безопасности ключа.

Эта функция требует Windows Vista или позднее.
