Справочник ключевых слов

$tagNMTVDISPINFO

Содержит и принимает сведения для отображения элемента TreeView.

Global Const $tagNMTVDISPINFO = $tagNMHDR & ";int Mask;int hItem;int State;int StateMask;" & _
"ptr Text;int TextMax;int Image;int SelectedImage;int Children;int Param"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Mask Определяет, какая информация устанавливается или получается. Это может быть одно или несколько из следующих значений:
$TVIF_CHILDREN – член Children действителен
$TVIF_IMAGE – член Image действителен
$TVIF_SELECTEDIMAGE – член SelectedImage действителен
$TVIF_TEXT – члены Text и TextMax действительны.
hItem Элемент, к которому относится эта структура.
State Набор битовых флагов и индексов списка изображений, указывающих на состояние элемента. При установке состояния элемента член StateMask указывает, какие биты этого члена действительны. При получении состояния элемента этот член возвращает текущее состояние для битов, указанных в члене StateMask.
Биты 0–7 этого члена содержат флаги состояния элемента.
Биты 8–11 этого члена указывают индекс изображения наложения с отсчётом от единицы.
StateMask Биты члена state, которые действительны. При получении состояния элемента установите биты члена stateMask, чтобы указать биты, возвращаемые в члене state.
При установке состояния элемента установите биты члена stateMask, чтобы указать биты члена state, которые вы хотите установить.
Text Указатель на строку с нулём в конце, содержащую текст элемента.
TextMax Размер буфера, на который указывает член Text, в символах.
Image Индекс в списке изображений иконки, используемой, когда элемент находится в невыбранном состоянии.
SelectedImage Индекс в списке изображений иконки, используемой, когда элемент находится в выбранном состоянии.
Children Флаг, указывающий, имеет ли элемент связанные дочерние элементы. Этот член может быть одним из следующих значений:
0 – элемент не имеет дочерних элементов
1 – элемент имеет один или несколько дочерних элементов
Param Значение для связи с элементом.
