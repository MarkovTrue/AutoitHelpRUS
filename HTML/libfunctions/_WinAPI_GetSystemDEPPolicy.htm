_WinAPI_GetSystemDEPPolicy

Получает параметр политики предотвращения выполнения данных (DEP) системы.

#include <WinAPISys.au3>
_WinAPI_GetSystemDEPPolicy ( )

Возвращаемое значение

Возвращает параметр политики DEP, который может быть одним из следующих значений.

    0 – AlwaysOff
    DEP отключена для всех частей системы, независимо от поддержки аппаратурой DEP. Процессор
    работает в режиме PAE с 32-разрядными версиями Windows, если PAE не отключена в данных конфигурации загрузки.

    1 – AlwaysOn
    DEP включена для всех частей системы. Все процессы всегда работают с включённой DEP. DEP не может быть
    явно отключена для выбранных приложений. Исправления совместимости системы игнорируются.

    2 – OptIn
    В системах с процессорами, способными к аппаратному обеспечению DEP, DEP автоматически включается
    только для компонентов операционной системы. Это параметр по умолчанию для клиентских версий Windows.
    DEP может быть явно включена для выбранных приложений или текущего процесса.

    3 – OptOut
    DEP автоматически включена для компонентов операционной системы и всех процессов. Это параметр по умолчанию
    для версий Windows Server. DEP может быть явно отключена для выбранных приложений или
    текущего процесса. Исправления совместимости системы для DEP вступают в силу.

Примечания

Эта функция требует Windows XP с SP3, Windows Vista с SP1 или более позднюю версию.
