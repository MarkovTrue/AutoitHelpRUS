Справочник функций

_ArrayConcatenate

Объединяет два массива – одномерных 1D или двумерных 2D с одинаковым числом столбцов.

#include <Array.au3>
_ArrayConcatenate ( ByRef $aArrayTarget, Const ByRef $aArraySource [, $iStart = 0] )

Параметры

$aArrayTarget Массив, к которому присоединяется исходный массив.
$aArraySource Присоединяемый массив.
$iStart [необязательный] Индекс первого элемента исходного массива (по умолчанию 0)

Возвращаемое значение

Успех: Возвращает количество элементов/строк в итоговом целевом массиве.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArrayTarget не массив
2 – $aArraySource не массив
3 – $aArrayTarget не 1D и не 2D массив
4 – несоответствие размерностей 1D/2D у $aArrayTarget и $aArraySource
5 – $aArrayTarget и $aArraySource имеют разное число столбцов (только для 2D массива)
6 – $iStart вне границ массива

Примечания

Параметр $iStart удобен при объединении массивов со счётчиком в элементе [0].

Функция НЕ обновляет счётчик целевого массива; при успехе возвращается итоговое количество строк в объединённом массиве.

См. также

_ArrayAdd

Пример

#include <Array.au3>

Local $aArrayTarget[5] = ["0", "1", "2", "3", "4"]
Local $aArraySource[5] = ["5", "6", "7", "8", "9"]

_ArrayDisplay($aArrayTarget, "1D Target")
_ArrayDisplay($aArraySource, "1D Source")
_ArrayConcatenate($aArrayTarget, $aArraySource)
_ArrayDisplay($aArrayTarget, "1D Target and Source concatenated")

Local $aArrayTarget[4][3]
For $i = 0 To 3
    For $j = 0 To 2
        $aArrayTarget[$i][$j] = $i & $j
    Next
Next

Local $aArraySource[4][3]
For $i = 0 To 3
    For $j = 0 To 2
        $aArraySource[$i][$j] = (4 + $i) & $j
    Next
Next

_ArrayDisplay($aArrayTarget, "2D Target")
_ArrayDisplay($aArraySource, "2D Source")
_ArrayConcatenate($aArrayTarget, $aArraySource)
_ArrayDisplay($aArrayTarget, "2D Target and Source concatenated")
