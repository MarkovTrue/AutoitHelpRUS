Справочник функций

_WinAPI_LoadIcon

Загружает указанный ресурс значка из исполняемого файла (.exe), связанного с экземпляром приложения.

#include <WinAPIIcons.au3>
_WinAPI_LoadIcon ( $hInstance, $sName )

Параметры

$hInstance Дескриптор экземпляра модуля, исполняемый файл которого содержит загружаемый значок.
$sName Имя ресурса значка для загрузки. Кроме того, если $hInstance равен 0, может быть
одно из следующих предопределённых значений.
$IDI_APPLICATION
$IDI_HAND
$IDI_QUESTION
$IDI_EXCLAMATION
$IDI_ASTERISK
$IDI_WINLOGO
$IDI_SHIELD
$IDI_ERROR
$IDI_INFORMATION
$IDI_WARNING

Возвращаемое значение

Успех: Возвращает дескриптор значка.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

_WinAPI_LoadIcon() загружает ресурс значка только если он ещё не был загружен; в противном случае возвращает
дескриптор существующего ресурса.

_WinAPI_LoadIcon() может загружать только значки размер которых соответствует системным метрикам $SM_CXICON и $SM_CYICON.
Для загрузки значков других размеров используйте функцию _WinAPI_LoadImage().

Когда закончите использовать значок, удалите его с помощью функции _WinAPI_DestroyIcon().
