_NamedPipes_SetNamedPipeHandleState

Устанавливает режим чтения и режим блокировки указанного именованного канала.

#include <NamedPipes.au3>
_NamedPipes_SetNamedPipeHandleState ( $hNamedPipe, $iRead, $iWait [, $iBytes = 0 [, $iTimeOut = 0]] )

Параметры

$hNamedPipe Дескриптор экземпляра именованного канала.
$iRead Режим чтения канала. Должен быть одним из следующих:
0 – данные читаются из канала как поток байтов
1 – данные читаются из канала как поток сообщений
$iWait Режим ожидания канала. Должен быть одним из следующих:
0 – режим блокировки включен
1 – режим без блокировки включен
$iBytes [необязательный] Максимальное количество байтов для сбора на компьютере клиента перед передачей на сервер
$iTimeout [необязательный] Максимальное время в миллисекундах, которое может пройти перед передачей информации удалённым именованным каналом

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
