_WinAPI_GetKeyNameText

Получает строку, представляющую имя клавиши.

#include <WinAPISys.au3>
_WinAPI_GetKeyNameText ( $lParam )

Параметры

$lParam Задаёт второй параметр сообщения клавиатуры (например WM_KEYDOWN) для обработки.

Возвращаемое значение

Успех: Возвращает строка, содержащая имя клавиши.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Формат строки имени клавиши зависит от текущей раскладки клавиатуры. Драйвер клавиатуры ведёт список
имён в виде строк символов для клавиш с именами, превышающими один символ. Имя клавиши
переводится в соответствии с расположением установленной клавиатуры, поэтому функция может давать различные
результаты для разных локалей ввода. Имя символьной клавиши – это сам символ. Имена безды
клавиш пишутся полностью.

Пример

#include <GUIConstantsEx.au3>
#include <WinAPISys.au3>
#include <WindowsNotifsConstants.au3>

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'))
GUIRegisterMsg($WM_KEYDOWN, 'WM_KEYDOWN')
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE

Func WM_KEYDOWN($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg, $wParam

    ConsoleWrite(_WinAPI_GetKeyNameText($lParam) & @CRLF)
    Return $GUI_RUNDEFMSG
EndFunc   ;==>WM_KEYDOWN
