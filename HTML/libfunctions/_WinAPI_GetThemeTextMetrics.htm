_WinAPI_GetThemeTextMetrics

Получает информацию о шрифте, указанном визуальным стилем для конкретной части.

#include <WinAPITheme.au3>
_WinAPI_GetThemeTextMetrics ( $hTheme, $iPartID, $iStateID [, $hDC = 0] )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть для получения информации о шрифте.
$iStateID Состояние части.
$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает структура $tagTEXTMETRIC, которая содержит информацию о шрифте.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
