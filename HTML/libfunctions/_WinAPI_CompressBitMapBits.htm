Справочник функций

_WinAPI_CompressBitmapBits

Создаёт сжатый блок данных из указанного растра.

#include <WinAPIGdi.au3>
_WinAPI_CompressBitmapBits ( $hBitmap, ByRef $pBuffer [, $iCompression = $COMPRESSION_BITMAP_PNG [, $iQuality = 100]] )

Параметры

$hBitmap Дескриптор растра, который должен быть сжат.
$pBuffer Указатель на блок памяти (буфер), который получает сжатые данные. При необходимости вы можете установить этот параметр на 0 перед вызовом функции, тогда функция сама выделит требуемый блок памяти.
В противном случае это должен быть действительный указатель на память, возвращённый функцией _WinAPI_CreateBuffer() или предыдущим вызовом этой функции.
$iCompression [необязательный] Метод сжатия. Этот параметр может быть одним из следующих значений.
$COMPRESSION_BITMAP_PNG (по умолчанию)
$COMPRESSION_BITMAP_JPEG
$iQuality [необязательный] Качество изображения JPEG в процентах. Это значение игнорируется при сжатии, не являющемся JPEG. По умолчанию 100.

Возвращаемое значение

Успех: Возвращает количество байт, скопированных в буфер.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Фактически данные, возвращаемые этой функцией, представляют файл изображения .jpeg или .png в двоичном виде. Вы можете использовать функцию _WinAPI_WriteFile() для создания соответствующего файла изображения непосредственно из этих данных.

Возвращаемые данные всегда представляют изображение .jpeg с глубиной цвета 24 бита или изображение .png с глубиной цвета 32 бита (с альфа-каналом или без него), независимо от глубины цвета исходного растра.

Когда буфер, выделенный этой функцией, больше не требуется, вы должны вызвать функцию _WinAPI_FreeMemory() (не используйте никакие другие подпрограммы управления памятью) для освобождения занимаемой памяти.

Эта функция внутренне использует библиотеку GDI+ DLL.

См. также

_WinAPI_FreeMemory, _WinAPI_WriteFile

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>
#include <WinAPIGdi.au3>
#include <WinAPIHObj.au3>
#include <WinAPIMem.au3>
#include <WinAPIRes.au3>

; Загрузить изображение
Local $sBmp = _Extras_PathFull('AutoIt.bmp')
Local $hSource = _WinAPI_LoadImage(0, $sBmp, $IMAGE_BITMAP, 0, 0, BitOR($LR_LOADFROMFILE, $LR_CREATEDIBSECTION))

;Изменить размер растра на 256x256 пиксели
Local $hBitmap = _WinAPI_AdjustBitmap($hSource, 256, 256, $HALFTONE)

;Создать сжатые данные PNG
Local $pData = 0
Local $iLength = _WinAPI_CompressBitmapBits($hBitmap, $pData, 2) ;$COMPRESSION_BITMAP_PNG)

;Создать файл .ico
If Not @error Then
    Local $tICO = DllStructCreate('align 1;ushort Reserved;ushort Type;ushort Count;byte Header[20]')
    Local $tHDR = DllStructCreate('byte Width;byte Height;byte ColorCount;byte Reserved;ushort Planes;ushort BitCount;long Size;long Offset', DllStructGetPtr($tICO, 'Header'))
    DllStructSetData($tICO, 'Reserved', 0)
    DllStructSetData($tICO, 'Type', 1)
    DllStructSetData($tICO, 'Count', 1)
    DllStructSetData($tHDR, 'Width', 0)
    DllStructSetData($tHDR, 'Height', 0)
    DllStructSetData($tHDR, 'ColorCount', 0)
    DllStructSetData($tHDR, 'Reserved', 0)
    DllStructSetData($tHDR, 'Planes', 1)
    DllStructSetData($tHDR, 'BitCount', 32)
    DllStructSetData($tHDR, 'Size', $iLength)
    DllStructSetData($tHDR, 'Offset', DllStructGetSize($tICO))
    Local $hFile = _WinAPI_CreateFile(@TempDir & '\MyIcon.ico', 1, 4)
    Local $iBytes
    _WinAPI_WriteFile($hFile, $tICO, DllStructGetSize($tICO), $iBytes)
    _WinAPI_WriteFile($hFile, $pData, $iLength, $iBytes)
    _WinAPI_CloseHandle($hFile)
    ShellExecute("MsPaint", '"' & @TempDir & '\MyIcon.ico' & '"')
Else
    MsgBox($MB_SYSTEMMODAL, "Error", "_WinAPI_CompressBitmapBits() failed (@error = " & @error & ")")
EndIf

;Удалить ненужные растры
_WinAPI_DeleteObject($hSource)
_WinAPI_DeleteObject($hBitmap)

; Освободить память
_WinAPI_FreeMemory($pData)
