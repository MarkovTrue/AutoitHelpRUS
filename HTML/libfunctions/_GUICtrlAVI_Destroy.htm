Справочник функций

_GUICtrlAVI_Destroy

Удаляет элемент управления.

#include <GuiAVI.au3>
_GUICtrlAVI_Destroy ( ByRef $hWnd )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.

Возвращаемое значение

Успех: Возвращает True, $hWnd устанавливается в 0.
Ошибка: Возвращает False.

Примечания

Применяется только к элементам AVI, созданным функцией _GUICtrlAVI_Create().

См. также

_GUICtrlAVI_Create

Пример

#include <GuiAVI.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"
    Local $hGUI, $hAVI, $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\Examples\GUI\SampleAVI.avi"

    ; Создать GUI
    $hGUI = GUICreate("(External) AVI Destroy", 300, 100)
    $hAVI = _GUICtrlAVI_Create($hGUI, "", -1, 10, 10)
    GUISetState(@SW_SHOW)

    ; Воспроизвести пример AVI AutoIt
    _GUICtrlAVI_Open($hAVI, $sFile)

    ; Воспроизвести пример AVI AutoIt
    _GUICtrlAVI_Play($hAVI)

    Sleep(3000)

    ; Остановить AVI-клип через 3 секунды
    _GUICtrlAVI_Stop($hAVI)

    ; Закрыть AVI-клип
    _GUICtrlAVI_Close($hAVI)

    MsgBox($MB_SYSTEMMODAL, "Information", "Destroy AVI Control")
    _GUICtrlAVI_Destroy($hAVI)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    GUIDelete()
EndFunc   ;==>Example
