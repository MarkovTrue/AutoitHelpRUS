_GDIPlus_ImageSaveToStream

Сохраняет объект Image в поток.

#include <GDIPlus.au3>
_GDIPlus_ImageSaveToStream ( $hImage, $pStream, $tEncoder [, $tParams = 0] )

Параметры

$hImage Указатель на объект Image.
$pStream Указатель на интерфейс IStream.
$tEncoder Структура $tagGUID, задающая GUID кодировщика изображения.
$tParams [необязательный] Структура $tagGDIPENCODERPARAMS или указатель на неё.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

Используйте эту функцию для сохранения выбранных кадров многокадрового изображения в другое многокадровое изображение.

См. также

$tagGDIPENCODERPARAMS, _GDIPlus_EncodersGetCLSID, _GDIPlus_ImageLoadFromStream, _GDIPlus_ImageSaveToFile

Дополнительно

Искать GdipSaveImageToStream в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPICom.au3>
#include <WinAPIConv.au3>
#include <WinAPIHObj.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = @DesktopWidth / 2, $iH = @DesktopHeight / 2
    Local $hGUI = GUICreate("GDI+ test", $iW, $iH, -1, -1)
    GUISetState(@SW_SHOW)

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, @DesktopWidth / 2, @DesktopHeight / 2) ;создать GDI bitmap захватом 1/4 рабочего стола
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;преобразовать GDI bitmap в GDI+ bitmap
    _WinAPI_DeleteObject($hHBmp) ;освободить ресурс GDI bitmap – больше не нужен

    Local $sImgCLSID = _GDIPlus_EncodersGetCLSID("jpg") ;создать CLSID для типа JPG
    Local $tGUID = _WinAPI_GUIDFromString($sImgCLSID) ;преобразовать CLSID GUID в бинарный вид, вернуть структуру $tagGUID
    Local $tParams = _GDIPlus_ParamInit(1) ;инициализировать список параметров кодера, вернуть структуру $tagGDIPENCODERPARAMS
    Local $tData = DllStructCreate("int Quality") ;создать структуру для настройки качества JPG
    DllStructSetData($tData, "Quality", 10) ;качество 0-100 (0: минимум, 100: максимум)
    Local $pData = DllStructGetPtr($tData) ;получить указатель на quality-структуру
    _GDIPlus_ParamAdd($tParams, $GDIP_EPGQUALITY, 1, $GDIP_EPTLONG, $pData) ;добавить значение в список параметров кодера
    Local $pStream = _WinAPI_CreateStreamOnHGlobal() ;создать поток
    _GDIPlus_ImageSaveToStream($hBitmap, $pStream, $tGUID, $tParams) ;сохранить bitmap в формате JPG в память
    Local $hBitmapFromStream = _GDIPlus_BitmapCreateFromStream($pStream) ;создать bitmap из потока (здесь – из JPG в памяти)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать графический объект из дескриптора окна
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmapFromStream, 0, 0) ;показать потоковое изображение

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;освободить ресурсы
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_BitmapDispose($hBitmapFromStream)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
