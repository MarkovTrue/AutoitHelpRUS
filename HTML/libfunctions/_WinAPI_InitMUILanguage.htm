Справочник функций

_WinAPI_InitMUILanguage

Позволяет приложению указать язык для использования со стандартными элементами управления, отличающийся от языка системы.

#include <WinAPISys.au3>
_WinAPI_InitMUILanguage ( $iLanguage )

Параметры

$iLanguage Идентификатор языка, используемый стандартными элементами управления.

Возвращаемое значение

Отсутствует.

Примечания

Эта функция позволяет приложению переопределить параметр языка системы и указать другой язык
для стандартных элементов управления. Выбранный язык применяется только к текущему процессу.
