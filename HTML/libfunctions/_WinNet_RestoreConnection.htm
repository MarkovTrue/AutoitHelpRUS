Справочник функций

_WinNet_RestoreConnection

Восстанавливает подключение к сетевому ресурсу.

#include <WinNet.au3>
_WinNet_RestoreConnection ( [$sDevice = "" [, $hWnd = 0 [, $bUseUI = True]]] )

Параметры

$sDevice [необязательный] Локальное имя диска для подключения, например "Z:". Если пусто, функция переподключает все постоянные диски, сохранённые в реестре для текущего пользователя.
$hWnd [необязательный] Дескриптор окна-родителя, который функция использует для отображения пользовательского интерфейса, который предлагает пользователю имя и пароль при создании сетевого подключения. Если 0, окна-владельца нет.
$bUseUI [необязательный] Если True, отобразить приглашение имени пользователя и пароля для вызывающей стороны

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
