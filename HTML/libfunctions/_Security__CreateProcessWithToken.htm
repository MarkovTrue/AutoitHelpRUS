Справочник функций

_Security__CreateProcessWithToken

Создаёт новый процесс и его основной поток, выполняющиеся в контексте безопасности указанного токена.

#include <Security.au3>
_Security__CreateProcessWithToken ( $hToken, $iLogonFlags, $sCommandLine, $iCreationFlags, $sCurDir, $tSTARTUPINFO, $tPROCESS_INFORMATION )

Параметры

$hToken Дескриптор основного токена, представляющего пользователя.
$iLogonFlags Параметр входа.
$sCommandLine Командная строка для выполнения.
$iCreationFlags Флаги, управляющие тем, как создаётся процесс.
$sCurDir Полный путь к текущему каталогу для процесса.
$tSTARTUPINFO Структура STARTUPINFO (или указатель на неё).
$tPROCESS_INFORMATION Структура PROCESS_INFORMATION (или указатель на неё), получающая сведения об идентификации нового процесса.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Имя модуля должно быть первым токеном, разделённым пробелом, в параметре $sCommandLine.

См. также

_Security__DuplicateTokenEx

Дополнительно

Искать CreateProcessWithTokenW в библиотеке MSDN

Пример

#RequireAdmin ; чтобы пример имел смысл

#include <ProcessConstants.au3>
#include <Security.au3>
#include <StructureConstants.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

Example_ProcessWithTok()

Func Example_ProcessWithTok()
    ; Запустить AutoIt без повышения прав, независимо от наличия полных прав администратора, полученных через #RequireAdmin или другие способы
    _RunNonElevated('"' & @AutoItExe & '" /AutoIt3ExecuteLine "MsgBox(4096, ''RunNonElevated'', ''IsAdmin() = '' & "IsAdmin()" & '', PID = '' & "@AutoItPID")"')
EndFunc   ;==>Example_ProcessWithTok

Func _RunNonElevated($sCommandLine = "")
    If Not IsAdmin() Then Return Run($sCommandLine) ; если текущий процесс запущен без повышения прав, то просто запустить новый

    ; Структуры, необходимые для создания процесса
    Local $tSTARTUPINFO = DllStructCreate($tagSTARTUPINFO)
    Local $tPROCESS_INFORMATION = DllStructCreate($tagPROCESS_INFORMATION)

    ; Дескриптор процесса, запущенного без повышения прав. Например "Explorer"
    Local $hProcess = _WinAPI_OpenProcess($PROCESS_ALL_ACCESS, 0, ProcessExists("explorer.exe"))

    ; В случае успеха
    If $hProcess Then
        ; Токен...
        Local $hTokOriginal = _Security__OpenProcessToken($hProcess, $TOKEN_ALL_ACCESS)
        ; Дескриптор процесса больше не нужен – закрыть
        _WinAPI_CloseHandle($hProcess)
        ; В случае успеха
        If $hTokOriginal Then
            ; Создать дубликат исходного токена
            Local $hTokDuplicate = _Security__DuplicateTokenEx($hTokOriginal, $TOKEN_ALL_ACCESS, $SECURITYIMPERSONATION, $TOKENPRIMARY)
            ; Закрыть исходный токен
            _WinAPI_CloseHandle($hTokOriginal)
            ; В случае успеха
            If $hTokDuplicate Then
                ; Создать процесс с этим новым токеном
                _Security__CreateProcessWithToken($hTokDuplicate, 0, $sCommandLine, 0, @ScriptDir, $tSTARTUPINFO, $tPROCESS_INFORMATION)

                ; Закрыть этот токен
                _WinAPI_CloseHandle($hTokDuplicate)
                ; Закрыть дескрипторы
                _WinAPI_CloseHandle(DllStructGetData($tPROCESS_INFORMATION, "hProcess"))
                _WinAPI_CloseHandle(DllStructGetData($tPROCESS_INFORMATION, "hThread"))
                ; Вернуть PID новосозданного процесса
                Return DllStructGetData($tPROCESS_INFORMATION, "ProcessID")
            EndIf
        EndIf
    EndIf
EndFunc   ;==>_RunNonElevated
