Справочник функций

_WinAPI_GetWindow

Получает дескриптор окна, которое имеет указанное отношение к заданному окну.

#include <WinAPISysWin.au3>
_WinAPI_GetWindow ( $hWnd, $iCmd )

Параметры

$hWnd Дескриптор окна.
$iCmd Указывает отношение между указанным окном и окном, дескриптор которого нужно получить.
Этот параметр может быть одним из следующих значений:
$GW_CHILD – полученный дескриптор идентифицирует дочернее окно в верхней части порядка Z, если указанное
    окно является родительским окном; в противном случае полученный дескриптор равен 0.
    Функция проверяет только дочерние окна указанного окна. Она не проверяет потомков окна.
$GW_HWNDFIRST – полученный дескриптор идентифицирует окно того же типа, находящееся выше всех в порядке Z.
    Если указанное окно находится в верхней части экрана, дескриптор идентифицирует верхнее окно, находящееся выше всех в порядке Z.
    Если указанное окно является окном верхнего уровня, дескриптор идентифицирует окно верхнего уровня, находящееся выше всех в порядке Z.
    Если указанное окно является дочерним окном, дескриптор идентифицирует соседнее окно, находящееся выше всех в порядке Z.
$GW_HWNDLAST – полученный дескриптор идентифицирует окно того же типа, находящееся ниже всех в порядке Z.
    Если указанное окно находится в верхней части экрана, дескриптор идентифицирует верхнее окно, находящееся ниже всех в порядке Z.
    Если указанное окно является окном верхнего уровня, дескриптор идентифицирует окно верхнего уровня, находящееся ниже всех в порядке Z.
    Если указанное окно является дочерним окном, дескриптор идентифицирует соседнее окно, находящееся ниже всех в порядке Z.
$GW_HWNDNEXT – полученный дескриптор идентифицирует окно ниже указанного окна в порядке Z.
    Если указанное окно находится в верхней части экрана, дескриптор идентифицирует верхнее окно ниже указанного окна.
    Если указанное окно является окном верхнего уровня, дескриптор идентифицирует окно верхнего уровня ниже указанного окна.
    Если указанное окно является дочерним окном, дескриптор идентифицирует соседнее окно ниже указанного окна.
$GW_HWNDPREV – полученный дескриптор идентифицирует окно выше указанного окна в порядке Z.
    Если указанное окно находится в верхней части экрана, дескриптор идентифицирует верхнее окно выше указанного окна.
    Если указанное окно является окном верхнего уровня, дескриптор идентифицирует окно верхнего уровня выше указанного окна.
    Если указанное окно является дочерним окном, дескриптор идентифицирует соседнее окно выше указанного окна.
$GW_OWNER – полученный дескриптор идентифицирует окно-владельца указанного окна, если таковое имеется.

Возвращаемое значение

Успех: Возвращает дескриптор окна.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Для приведённых констант требуется #include <WinAPIConstants.au3>
