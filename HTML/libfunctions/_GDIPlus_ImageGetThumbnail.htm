_GDIPlus_ImageGetThumbnail

Возвращает уменьшенную копию (thumbnail) из объекта Image.

#include <GDIPlus.au3>
_GDIPlus_ImageGetThumbnail ( $hImage [, $iWidth = 0 [, $iHeight = 0 [, $bKeepRatio = True [, $hCallback = Null [, $hCallbackData = Null]]]]] )

Параметры

$hImage Дескриптор объекта Image.
$iWidth [необязательный] Ширина запрашиваемого thumbnail в пикселях. По умолчанию 0.
$iHeight [необязательный] Высота запрашиваемого thumbnail в пикселях. По умолчанию 0.
$bKeepRatio [необязательный] Если True, ширина и высота итогового изображения пересчитываются для сохранения соотношения сторон. По умолчанию True.
$hCallback [необязательный] Функция обратного вызова. В процессе создания или получения thumbnail GDI+ вызывает её, давая возможность прервать процесс. По умолчанию NULL.
$hCallbackData [необязательный] Указатель на блок памяти с данными для функции обратного вызова. По умолчанию NULL.

Возвращаемое значение

Успех: Возвращает дескриптор объекта Image, содержащего уменьшенную копию.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

Уменьшенная копия (thumbnail) – это маленькая версия изображения. Некоторые файлы содержат встроенную уменьшенную копию; в этом случае метод возвращает её. Если встроенной копии нет, метод создаёт её путём масштабирования основного изображения до размеров, заданных параметрами $iWidth и $iHeight. Если оба параметра равны 0, используется размер, определённый системой.
Не забудьте освободить объект с помощью _GDIPlus_ImageDispose().

См. также

_GDIPlus_GraphicsCreateFromHWND, _GDIPlus_ImageLoadFromFile, _GDIPlus_ImageDispose

Пример

#include <FileConstants.au3>
#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hImage, $sFile, $hGUI, $hGraphic, $hThumbnail, $iW_new, $iH_new

    $sFile = FileOpenDialog("Please select an image", "", "Image (*.jpg;*.png;*.bmp;*.gif;*.tif)", BitOR($FD_PATHMUSTEXIST, $FD_FILEMUSTEXIST))
    If @error Then Exit MsgBox(($MB_TOPMOST + $MB_ICONERROR), "Error", "No image file has been selected", 30)

    _GDIPlus_Startup()

    $hImage = _GDIPlus_ImageLoadFromFile($sFile)
    If @error Or Not $hImage Then
        MsgBox(($MB_TOPMOST + $MB_ICONERROR), "Error", "This file isn't supported by GDIPlus!")
    Else
        $hGUI = GUICreate("GDI+ _GDIPlus_ImageGetThumbnail Demo", 320, 200)
        GUISetBkColor(0x202020)
        GUISetState()

        $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
        $hThumbnail = _GDIPlus_ImageGetThumbnail($hImage, 96, 96)
        $iW_new = _GDIPlus_ImageGetWidth($hThumbnail)
        $iH_new = _GDIPlus_ImageGetHeight($hThumbnail)
        _GDIPlus_GraphicsDrawImageRect($hGraphic, $hThumbnail, (320 - $iW_new) / 2, (200 - $iH_new) / 2, $iW_new, $iH_new) ;центрировать изображение в GUI

        Do
        Until GUIGetMsg() = $GUI_EVENT_CLOSE

        _GDIPlus_ImageDispose($hImage)
        _GDIPlus_ImageDispose($hThumbnail)
    EndIf

    _GDIPlus_Shutdown()
EndFunc   ;==>Example
