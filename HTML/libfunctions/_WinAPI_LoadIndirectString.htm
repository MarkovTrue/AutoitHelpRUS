Справочник функций

_WinAPI_LoadIndirectString

Извлекает строку из указанного ресурса по непрямой строке.

#include <WinAPIRes.au3>
_WinAPI_LoadIndirectString ( $sStrIn )

Параметры

$sStrIn Входная непрямая строка.

Возвращаемое значение

Успех: Возвращает выходная строка.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Если строка является непрямой строкой (строка, начинающаяся с символа "@"), она имеет следующую форму.

    @имя_файла,ресурс

Строка извлекается из указанного файла, используя значение ресурса в качестве локатора. Если значение ресурса
оказывается нулевым или положительным, число становится индексом строки в двоичном файле. Если число отрицательное,
оно становится идентификатором ресурса. К непрямой строке можно добавить модификатор версии, как показано здесь:

    @имя_файла,ресурс;v2

Эта форма может использоваться, когда строка изменена, но все ещё использует тот же индекс или идентификатор ресурса, что и старая строка.
Без модификатора версии кэш многоязычного пользовательского интерфейса (MUI) не распознает, что строка изменилась,
и продолжит использовать старое значение. Добавляя модификатор версии, значение воспринимается как
новый ресурс и добавляется в кэш. Обратите внимание, что рекомендуется использовать новый идентификатор ресурса или индекс
для новой строки и использовать модификатор версии только когда это невозможно.

Если входная строка не является непрямой строкой, функция возвращает входную строку без изменений.
