Справочник функций

_IEGetObjByName

Возвращает объектную переменную по имени.

#include <IE.au3>
_IEGetObjByName ( ByRef $oObject, $sName [, $iIndex = 0] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sName Имя объекта, который нужно найти.
$iIndex [необязательный] Если name встречается больше одного раза, задаёт экземпляр по индексу (с нуля)
0 (по умолчанию) или положительное целое – возвращает экземпляр по индексу
-1 – возвращает коллекцию указанных объектов

Возвращаемое значение

Успех: Возвращает объектную переменную, ссылающуюся на указанный объект; @extended – количество найденных объектов.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных
7 ($_IEStatus_NoMatch) – совпадение не найдено
@extended: Содержит номер недопустимого параметра.

Примечания

Эта функция возвращает ссылку на любой объект в документе с указанным name или id.
DOM-элементы могут иметь атрибуты Name, ID или оба сразу.
Конкретный ID может быть присвоен только одному элементу, тогда как конкретное Name может быть присвоено нескольким элементам.
В Internet Explorer пространства имён Name и ID являются общими.

См. также

_IEGetObjById

Пример

; Открыть браузер на примере формы, получить ссылку на объект
; к элементу с именем "ExampleForm". В данном случае
; result is identical to using $oForm = _IEFormGetObjByName($oIE, "ExampleForm")

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEGetObjByName($oIE, "ExampleForm")
MsgBox($MB_SYSTEMMODAL, "ExampleForm", _IEPropertyGet($oForm, "innertext") & @CRLF)
