_WinAPI_PathCompactPath

Усекает путь файла, чтобы он поместился в заданную ширину пикселей путём замены компонентов пути на многоточия.

#include <WinAPIShPath.au3>
_WinAPI_PathCompactPath ( $hWnd, $sFilePath [, $iWidth = 0] )

Параметры

$hWnd Дескриптор окна.
$sFilePath Путь для изменения.
$iWidth [необязательный] Ширина в пикселях, в которую должна поместиться строка. Если этот параметр равен 0 (по умолчанию), ширина будет равна
ширине клиентской области окна. Если этот параметр отрицательное число, ширина будет
уменьшена до его абсолютного значения.

Возвращаемое значение

Успех: Возвращает изменённый путь.
Ошибка: Исходный параметр $sFilePath и устанавливает @error ≠ 0.

Примечания

Эта функция не будет уплотнять путь сверх базового имени файла, перед которым идут многоточия.

Дополнительно

Искать PathCompactPath в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <WinAPIShPath.au3>

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 260, 60)
Local $idLabel = GUICtrlCreateLabel('', 10, 22, 240, 16)
Local $hLabel = GUICtrlGetHandle($idLabel)
GUICtrlSetData(-1, _WinAPI_PathCompactPath($hLabel, @ScriptFullPath))
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
