_GDIPlus_GraphicsSetClipRect

Обновляет область отсечения объекта Graphics, объединяя её с прямоугольником.

#include <GDIPlus.au3>
_GDIPlus_GraphicsSetClipRect ( $hGraphics, $nX, $nY, $nWidth, $nHeight [, $iCombineMode = 0] )

Параметры

$hGraphics Указатель на объект Graphics.
$nX Координата X левого верхнего угла прямоугольника.
$nY Координата Y левого верхнего угла прямоугольника.
$nWidth Ширина прямоугольника.
$nHeight Высота прямоугольника.
$iCombineMode [необязательный] Режим объединения областей:
0 – существующая область заменяется новой
1 – существующая область заменяется пересечением с новой
2 – существующая область заменяется объединением с новой
3 – существующая область заменяется результатом XOR двух областей
4 – существующая область заменяется той её частью, которая находится вне новой области
5 – существующая область заменяется той частью новой области, которая находится вне существующей

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI = GUICreate("GDI+", 320, 240)
    GUICtrlCreateButton("Button", 20, 20, 100, 25)
    GUICtrlCreateSlider(100, 100, 100, 30)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Создать графический объект из дескриптора окна

    _GDIPlus_GraphicsSetClipRect($hGraphics, 20, 20, 100, 25, 4) ;вычесть область кнопки из области отсечения
    _GDIPlus_GraphicsSetClipRect($hGraphics, 100, 100, 100, 30, 4) ;вычесть область слайдера из области отсечения

    Local $iTimer = TimerInit()
    ; Цикл до выхода пользователя.
    Do
        If TimerDiff($iTimer) > 100 Then
            _GDIPlus_GraphicsClear($hGraphics, BitOR(0xFF000000, Random(0, 0xFFFFFF, 1)))
            $iTimer = TimerInit()
        EndIf
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
