Справочник функций

_WinAPI_CreateSemaphore

создаёт или открывает именованный или безымянный объект семафора.

#include <WinAPIProc.au3>
_WinAPI_CreateSemaphore ( $sSemaphore, $iInitial, $iMaximum [, $tSecurity = 0] )

Параметры

$sSemaphore Имя.
$iInitial Количество / Размер.
$iMaximum Количество / Размер.
$tSecurity Необязательный параметр.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Если объект-семафор с таким именем существовал до вызова функции, функция возвращает дескриптор существующего объекта.

Любой процесс может передать дескриптор семафора в вызов функций _WinAPI_WaitFor.... Функции ожидания одного объекта возвращают управление, когда состояние указанного объекта становится сигнальным. Функции ожидания нескольких объектов можно инструктировать возвращать управление либо когда любой из, либо когда все из указанных объектов становятся сигнальными. Когда функция ожидания возвращает управление, ждущий процесс освобождается для продолжения своего выполнения.

Состояние объекта семафора сигнальное, когда его счётчик больше нуля, и несигнальное, когда его счётчик равен нулю. Параметр $iInitial задаёт начальный счётчик. Каждый раз, когда ждущий процесс освобождается из-за сигнального состояния семафора, счётчик семафора уменьшается на один. Используйте функцию _WinAPI_ReleaseSemaphore() для увеличения счётчика семафора на указанное значение. Счётчик никогда не может быть меньше нуля или больше значения, указанного в параметре $iMaximum.

Используйте функцию _WinAPI_CloseHandle() для закрытия дескриптора. Система автоматически закрывает дескриптор при завершении процесса. Объект семафора уничтожается, когда его последний дескриптор был закрыт.

См. также

_WinAPI_CloseHandle, _WinAPI_ReleaseSemaphore

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

If Not @Compiled Then
    MsgBox($MB_SYSTEMMODAL, '', 'To run this script, you must first compile it and then run the (.exe) file.')
    Exit
EndIf

Local $hSemaphore = _WinAPI_CreateSemaphore('MySemaphore', 2, 2)

_WinAPI_WaitForSingleObject($hSemaphore)
_MyGUI()
_WinAPI_ReleaseSemaphore($hSemaphore)
_WinAPI_CloseHandle($hSemaphore)

Func _MyGUI()
    GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'))
    GUISetState(@SW_SHOW)
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>_MyGUI
