Справочник функций

_ArrayDelete

Удаляет указанный элемент или элементы из одномерного или двумерного массива.

#include <Array.au3>
_ArrayDelete ( ByRef $aArray, $vRange )

Параметры

$aArray Изменяемый массив.
$vRange Удаляемый элемент или элементы – одиночный индекс, строка диапазона или одномерный массив со счётчиком в элементе [0] (см. пример).

Возвращаемое значение

Успех: Возвращает новый размер массива. См. раздел Примечания.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным или двумерным массивом
3 – $vRange содержит недопустимую строку диапазона
4 – $vRange не является одномерным массивом или содержит только 1 элемент
5 – содержимое $vRange выходит за границы массива

Примечания

$vRange может быть строкой, задающей удаляемые строки. Это одно число или диапазон "первая-последняя" через дефис (-), несколько диапазонов разделяются точкой с запятой (;).

$vRange может также быть одномерным массивом, перечисляющим удаляемые строки, со счётчиком в элементе [0].

В обоих случаях строки не обязаны быть упорядочены по возрастанию и могут повторяться.

Функция НЕ обновляет счётный элемент массива, однако при успехе возвращает новый наибольший индекс строки массива.

См. также

_ArrayAdd, _ArrayInsert

Пример

#include <Array.au3>

Local $aArray[5] = [0, 1, 2, 3, 4]

_ArrayDisplay($aArray, "Original")
_ArrayDelete($aArray, 2)
_ArrayDisplay($aArray, "Element 2 deleted")

Local $aArray_Base[25][4]
For $i = 0 To 24
    For $j = 0 To 3
        $aArray_Base[$i][$j] = $i & "-" & $j
    Next
Next

; Одна строка
$aArray = $aArray_Base
_ArrayDisplay($aArray, "BEFORE deletion")
_ArrayDelete($aArray, 7)
_ArrayDisplay($aArray, "SINGLE ROW deleted")

; Строка диапазона
$aArray = $aArray_Base
Local $sRange = "0;11-15;24"
_ArrayDisplay($aArray, "BEFORE deletion")
_ArrayDelete($aArray, $sRange)
ConsoleWrite(" " & @error & @CRLF)
_ArrayDisplay($aArray, "RANGE STRING deleted")

; Удаляем по строке диапазона.
$aArray = $aArray_Base
Local $aDel[4] = [3, 5, 11, 13]
_ArrayDisplay($aArray, "BEFORE deletion")
_ArrayDelete($aArray, $aDel)
_ArrayDisplay($aArray, "RANGE ARRAY deleted")
