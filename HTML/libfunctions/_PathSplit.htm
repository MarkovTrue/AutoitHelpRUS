Справочник функций

_PathSplit

Разделяет путь на компоненты: диск, каталог, имя файла и расширение. Пустая строка устанавливается, если компонент отсутствует.

#include <File.au3>
_PathSplit ( $sFilePath, ByRef $sDrive, ByRef $sDir, ByRef $sFileName, ByRef $sExtension )

Параметры

$sFilePath Путь для разделения (может содержать сервер UNC или букву диска).
$sDrive Строка для хранения диска.
$sDir Строка для хранения каталога.
$sFileName Строка для хранения имени файла.
$sExtension Строка для хранения расширения файла.

Возвращаемое значение

Возвращает массив с 5 элементами:
    $aArray[$PATH_ORIGINAL] = исходный путь
    $aArray[$PATH_DRIVE] = диск
    $aArray[$PATH_DIRECTORY] = каталог
    $aArray[$PATH_FILENAME] = имя файла
    $aArray[$PATH_EXTENSION] = расширение

Примечания

Функция не принимает строку командной строки. Она работает с путями, а не с путями и аргументами.

$PATH_ORIGINAL 0
$PATH_DRIVE 1
$PATH_DIRECTORY 2
$PATH_FILENAME 3
$PATH_EXTENSION 4

Константы определены в FileConstants.au3.

См. также

_PathFull, _PathMake

Пример

#include <Array.au3>
#include <File.au3>

Local $sDrive = "", $sDir = "", $sFileName = "", $sExtension = ""
Local $aPathSplit = _PathSplit(@ScriptFullPath, $sDrive, $sDir, $sFileName, $sExtension)
_ArrayDisplay($aPathSplit, "_PathSplit of " & @ScriptFullPath)
