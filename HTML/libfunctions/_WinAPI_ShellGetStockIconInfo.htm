_WinAPI_ShellGetStockIconInfo

Получает информацию о значках оболочки, определённых системой.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetStockIconInfo ( $iSIID, $iFlags )

Параметры

$iSIID Одна из констант $SIID_*, которая указывает, какой значок должен быть получен. Эти константы определены в APIShellExConstants.au3.
$iFlags Флаги, которые указывают, какая информация запрашивается. Этот параметр может быть комбинацией сле dующих значений. $SHGSI_ICONLOCATION $SHGSI_ICON $SHGSI_SYSICONINDEX $SHGSI_LINKOVERLAY $SHGSI_SELECTED $SHGSI_LARGEICON $SHGSI_SMALLICON $SHGSI_SHELLICONSIZE.

Возвращаемое значение

Успех: Возвращает структура $tagSHSTOCKICONINFO, содержащая запрошенную информацию.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Если эта функция возвращает дескриптор значка в члене "hIcon" структуры $tagSHSTOCKICONINFO, вы несёте ответственность за освобождение значка с помощью _WinAPI_DestroyIcon(), когда он больше не нужен.
Эта функция требует Windows Vista или более поздней версии.

См. также

_WinAPI_DestroyIcon

Дополнительно

Искать SHGetStockIconInfo в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIIcons.au3>
#include <WinAPIShellEx.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 200, 236)
GUICtrlCreateIcon('', 0, 36, 36, 128, 128)
Local $h_Icon = GUICtrlGetHandle(-1)
GUICtrlSetState(-1, $GUI_DISABLE)
Local $idLabel = GUICtrlCreateLabel('', 70, 174, 60, 14, $SS_CENTER)
Local $idBtn_Prev = GUICtrlCreateButton('<', 32, 200, 60, 24)
Local $idBtn_Next = GUICtrlCreateButton('>', 108, 200, 60, 24)
GUISetState(@SW_SHOW)

Local $tSHSTOCKICONINFO, $hIcon, $hOld, $iCount = 0, $bUpdate = True
While 1
    If $bUpdate Then
        GUICtrlSetData($idLabel, 'SIID: ' & $iCount)
        $tSHSTOCKICONINFO = _WinAPI_ShellGetStockIconInfo($iCount, $SHGSI_ICONLOCATION)
        $hIcon = _WinAPI_ShellExtractIcon(DllStructGetData($tSHSTOCKICONINFO, 'Path'), DllStructGetData($tSHSTOCKICONINFO, 'iIcon'), 128, 128)
        $hOld = _SendMessage($h_Icon, $STM_SETIMAGE, 1, $hIcon)
        If $hOld Then
            _WinAPI_DestroyIcon($hOld)
        EndIf
        $bUpdate = 0
    EndIf
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE
            ExitLoop
        Case $idBtn_Prev
            $iCount -= 1
            If $iCount < 0 Then
                $iCount = $SIID_MAX_ICONS - 1
            EndIf
            $bUpdate = 1
        Case $idBtn_Next
            $iCount += 1
            If $iCount > $SIID_MAX_ICONS - 1 Then
                $iCount = 0
            EndIf
            $bUpdate = 1
    EndSwitch
WEnd
