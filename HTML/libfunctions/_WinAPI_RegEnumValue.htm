_WinAPI_RegEnumValue

Перечисляет значения указанного открытого ключа реестра.

#include <WinAPIReg.au3>
_WinAPI_RegEnumValue ( $hKey, $iIndex )

Параметры

$hKey Дескриптор открытого раздела реестра. Раздел должен быть открыт с правом доступа $KEY_QUERY_VALUE.
Дескриптор возвращается функциями _WinAPI_RegCreateKey() и _WinAPI_RegOpenKey().
Также допустимы предопределённые ключи:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_PERFORMANCE_DATA
$HKEY_USERS
$iIndex Индекс получаемого значения.
Этот параметр должен быть нулевым для первого вызова функции _WinAPI_RegEnumValue() и затем увеличиваться для последующих вызовов.

Возвращаемое значение

Успех: Возвращает строка, содержащая имя значения, флаг @extended будет содержать код, указывающий тип данных ($REG_*), хранящихся в указанном значении.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Для перечисления значений приложение должно первоначально вызвать функцию _WinAPI_RegEnumValue() с параметром $iIndex, установленным в нуль.
Приложение затем должно увеличить $iIndex и вызвать функцию _WinAPI_RegEnumValue(), пока нет больше значений (пока флаг @extended не установлен в ERROR_NO_MORE_ITEMS (259)).

Приложение также может установить $iIndex на индекс последнего значения при первом вызове функции и уменьшить индекс до перечисления значения с индексом 0.
Для получения индекса последнего значения используйте функцию _WinAPI_RegQueryInfoKey().

Од время использования _WinAPI_RegEnumValue() приложение не должно вызывать какие-либо функции реестра, которые могут изменить запрашиваемый ключ.

См. также

_WinAPI_RegCreateKey, _WinAPI_RegOpenKey, _WinAPI_RegQueryInfoKey

Пример

#include <Debug.au3>
#include <WinAPIError.au3>
#include <WinAPIReg.au3>

Example()

Func Example()
    Local $hKey = _WinAPI_RegOpenKey($HKEY_CURRENT_USER, 'Software\Microsoft\Windows\CurrentVersion\Run', $KEY_READ)
    If @error Then
        _DebugSetup(Default, True)
        _DebugReport("! RegOpenKey @error =" & @error & @TAB & _WinAPI_GetErrorMessage(@extended) & @CRLF)
        Exit
    EndIf
    Local $iCount = _WinAPI_RegQueryInfoKey($hKey)
    Local $aKey[$iCount[2]]
    For $i = 0 To UBound($aKey) - 1
        $aKey[$i] = _WinAPI_RegEnumValue($hKey, $i)
    Next

    _WinAPI_RegCloseKey($hKey)

    _DebugArrayDisplay($aKey, '_WinAPI_RegEnumValue')

EndFunc   ;==>Example
