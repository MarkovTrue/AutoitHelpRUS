Справочник функций

_WinAPI_GetWindowFileName

Получает полностью определённый путь модуля, связанного с указанным дескриптором окна.

#include <WinAPIProc.au3>
_WinAPI_GetWindowFileName ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

Успех: Возвращает полный путь и имя файла.
Ошибка: Устанавливает @error ≠ 0.
