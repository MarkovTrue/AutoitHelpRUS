Справочник функций

_GDIPlus_TextureCreate2

Создаёт объект TextureBrush на основе изображения, режима повтора и ограничивающего прямоугольника.

#include <GDIPlus.au3>
_GDIPlus_TextureCreate2 ( $hImage, $nX, $nY, $nWidth, $nHeight [, $iWrapMode = 0] )

Параметры

$hImage Указатель на объект Image.
$nX Координата X левого края фрагмента изображения, используемого кистью.
$nY Координата Y верхнего края фрагмента изображения, используемого кистью.
$nWidth Ширина кисти и фрагмента изображения, используемого кистью.
$nHeight Высота кисти и фрагмента изображения, используемого кистью.
$iWrapMode [необязательный] Режим повтора, определяющий, как копии изображения заполняют область мозаикой:
0 – без отражения
1 – горизонтальное отражение при переходе к следующей плитке в строке
2 – вертикальное отражение при переходе к следующей плитке в столбце
3 – горизонтальное отражение по строкам и вертикальное по столбцам
4 – мозаика не используется

Возвращаемое значение

Успех: Возвращает указатель на новый объект TextureBrush.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

После завершения работы с объектом вызовите _GDIPlus_BrushDispose() для освобождения ресурсов.

См. также

_GDIPlus_BrushDispose

Дополнительно

Искать GdipCreateTexture2 в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Поддержка работы в X64
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ;получить каталог установки AutoIt
    Local $sRegPath = "HKLM\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt"

    Local $sFile = RegRead($sRegPath, "InstallDir") & "\Examples\GUI\logo4.gif"
    If Not FileExists($sFile) Then
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", $sFile & " not found!", 30)
        Return False
    EndIf

    _GDIPlus_Startup()
    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile) ;создать объект изображения на основе файла
    If @error Then
        _GDIPlus_Shutdown()
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "An error has occured - unable to load image!", 30)
        Return False
    EndIf

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", 320, 200)
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать объект Graphics из дескриптора окна
    _GDIPlus_GraphicsClear($hGraphics, 0xFF404040) ;очистить графический дескриптор тёмно-серым цветом (фон)
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;задаёт качество отрисовки графики (сглаживание)
    Local $hTexture = _GDIPlus_TextureCreate2($hImage, 5, 4, 59, 59) ;создать текстурную кисть только из заданного прямоугольника
    _GDIPlus_GraphicsFillEllipse($hGraphics, 10, 40, 300, 120, $hTexture) ;нарисовать эллипс, используя текстуру в качестве кисти

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;освободить ресурсы
    _GDIPlus_BrushDispose($hTexture)
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
