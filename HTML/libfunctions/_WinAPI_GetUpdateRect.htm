Справочник функций

_WinAPI_GetUpdateRect

Получает координаты прямоугольника, который полностью охватывает прямоугольник обновления указанного окна.

#include <WinAPIGdi.au3>
_WinAPI_GetUpdateRect ( $hWnd [, $bErase = True] )

Параметры

$hWnd Дескриптор окна.
$bErase [необязательный] Указывает, должен ли фон в прямоугольнике обновления быть стёрт, допустимые значения:
True – фон стирается (по умолчанию).
False – фон остаётся неизменным.

Возвращаемое значение

Успех: Возвращает структура $tagRECT, содержащая координаты охватывающего прямоугольника в единицах устройства.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Прямоугольник обновления, полученный функцией _WinAPI_BeginPaint(), идентичен тому, который получает
_WinAPI_GetUpdateRect()

_WinAPI_BeginPaint() автоматически проверяет прямоугольник обновления, поэтому любой вызов _WinAPI_GetUpdateRect(), выполненный
    сразу после вызова _WinAPI_BeginPaint(), получает пустой прямоугольник обновления.
