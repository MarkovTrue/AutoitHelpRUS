Справочник функций

_WinAPI_RegSetValue

Задаёт данные и тип указанного значения под ключом реестра.

#include <WinAPIReg.au3>
_WinAPI_RegSetValue ( $hKey, $sValueName, $iType, $tValueData, $iBytes )

Параметры

$hKey Дескриптор открытого ключа реестра. Ключ должен быть открыт с правом доступа $KEY_SET_VALUE.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey().
Это также может быть один из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_PERFORMANCE_DATA
$HKEY_USERS
$sValueName Имя значения для установки.
Если значение с этим именем уже не присутствует в ключе, функция добавляет его в ключ.
Если $sValueName – пустая строка, функция задаёт тип и данные для безымянного или значения по умолчанию ключа.
$iType Тип данных. Этот параметр может быть одним из следующих значений:
$REG_BINARY
$REG_DWORD
$REG_DWORD_BIG_ENDIAN
$REG_DWORD_LITTLE_ENDIAN
$REG_EXPAND_SZ
$REG_LINK
$REG_MULTI_SZ
$REG_NONE
$REG_QWORD
$REG_QWORD_LITTLE_ENDIAN
$REG_SZ
$tValueData Структура (буфер), содержащая данные для сохранения.
Для типов данных на основе строк, таких как REG_SZ, строка должна быть завершена нулевым символом.
Для типа данных REG_MULTI_SZ строка должна быть завершена двумя нулевыми символами.
Обратная косая черта должна предваряться другой обратной косой чертой как символ экранирования.
Например, укажите "C:\\mydir\\myfile" для сохранения строки "C:\mydir\myfile".
$iBytes Размер данных в байтах.
Если данные имеют тип REG_SZ, REG_MULTI_SZ или REG_EXPAND_SZ, этот размер включает все завершающие нулевые символы, если данные были сохранены без них.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.
