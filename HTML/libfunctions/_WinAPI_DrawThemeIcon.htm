Справочник функций

_WinAPI_DrawThemeIcon

Рисует изображение из списка изображений с эффектом значка, заданным визуальным стилем.

#include <WinAPITheme.au3>
_WinAPI_DrawThemeIcon ( $hTheme, $iPartID, $iStateID, $hDC, $tRECT, $hIL, $iIndex )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, в которой рисуется изображение.
$iStateID Состояние части.
$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT с прямоугольником, в котором рисуется изображение.
$hIL Дескриптор списка изображений, содержащего рисуемое изображение.
$iIndex Индекс рисуемого изображения.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
