Справочник функций

_WinAPI_CreateSymbolicLink

создаёт символическую ссылку.

#include <WinAPIFiles.au3>
_WinAPI_CreateSymbolicLink ( $sSymlink, $sTarget [, $bDirectory = False] )

Параметры

$sSymlink Имя нового файла.
$sTarget Имя.
$bDirectory Необязательный параметр.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Чтобы удалить символическую ссылку, удалите файл (через _WinAPI_DeleteFile() или аналогичные API) или удалите каталог
(через _WinAPI_RemoveDirectory() или аналогичные API) в зависимости от типа символической ссылки.

Вызывающий процесс должен иметь привилегию $SE_CREATE_SYMBOLIC_LINK_NAME, иначе функция завершится с ошибкой и
_WinAPI_GetLastError() вернёт ERROR_PRIVILEGE_NOT_HELD (1314).

Данная функция требует Windows Vista или новее.

См. также

_WinAPI_DeleteFile, _WinAPI_RemoveDirectory

Дополнительно

Искать CreateSymbolicLink в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <SecurityConstants.au3>
#include <WinAPIError.au3>
#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

; Включение привилегии "SeCreateSymbolicLinkPrivilege" для создания символических ссылок
Local $hToken = _WinAPI_OpenProcessToken(BitOR($TOKEN_ADJUST_PRIVILEGES, $TOKEN_QUERY))
Local $aAdjust
_WinAPI_AdjustTokenPrivileges($hToken, $SE_CREATE_SYMBOLIC_LINK_NAME, $SE_PRIVILEGE_ENABLED, $aAdjust)
If @error Or @extended Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'You do not have the required privileges.')
    Exit
EndIf

; Создание файла
If Not _WinAPI_CreateSymbolicLink(@DesktopDir & '\' & StringRegExpReplace(@ScriptDir, '^.*\\', '@'), @ScriptDir, 1) Then
    _WinAPI_ShowLastError()
EndIf

; Восстановить привилегию "SeCreateSymbolicLinkPrivilege" по умолчанию
_WinAPI_AdjustTokenPrivileges($hToken, $aAdjust, 0, $aAdjust)
_WinAPI_CloseHandle($hToken)
