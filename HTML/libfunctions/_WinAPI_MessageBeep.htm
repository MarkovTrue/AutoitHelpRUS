Справочник функций

_WinAPI_MessageBeep

Воспроизводит звуковую волну.

#include <WinAPIError.au3>
_WinAPI_MessageBeep ( [$iType = 1] )

Параметры

$iType [необязательный] Тип звука, определённый записью в реестре. Это может быть одно из следующих
значений:
0 – простой звуковой сигнал. Если звуковая карта недоступна, используется динамик.
1 – OK
2 – Hand
3 – Question
4 – Exclamation
5 – Asterisk.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

После ставления звука в очередь функция MessageBeep возвращает управление вызывающей функции и воспроизводит звук
asynchronously. Если функция не может воспроизвести указанный звук оповещения, MessageBeep пытается воспроизвести звук по умолчанию системы.
Если функция не может воспроизвести звук по умолчанию системы, функция выдаёт стандартный звуковой сигнал через
динамик компьютера.
