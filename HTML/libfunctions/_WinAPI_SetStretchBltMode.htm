Справочник функций

_WinAPI_SetStretchBltMode

Задаёт режим растяжения растрового изображения в указанном контексте устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetStretchBltMode ( $hDC, $iMode )

Параметры

$hDC Дескриптор контекста устройства.
$iMode Режим растяжения. Этот параметр может быть одним из следующих значений.
$BLACKONWHITE
$COLORONCOLOR
$HALFTONE
$WHITEONBLACK
$STRETCH_ANDSCANS
$STRETCH_DELETESCANS
$STRETCH_HALFTONE
$STRETCH_ORSCANS

Возвращаемое значение

Успех: Возвращает предыдущий режим растяжения.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0, @extended может содержать ERROR_INVALID_PARAMETER.

Примечания

Режимы $BLACKONWHITE ($STRETCH_ANDSCANS) и $WHITEONBLACK ($STRETCH_ORSCANS) обычно используются для
сохранения пикселей переднего плана в монохромных растровых изображениях. Режим $COLORONCOLOR ($STRETCH_DELETESCANS) обычно
используется для сохранения цвета в цветных растровых изображениях. Режим $HALFTONE ($STRETCH_HALFTONE) медленнее и требует больше
обработки исходного изображения, чем три других режима; но дает изображения более высокого качества.
