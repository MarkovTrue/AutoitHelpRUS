Справочник функций

_WinAPI_SetParent

Изменяет родительское окно указанного дочернего окна.

#include <WinAPISysWin.au3>
_WinAPI_SetParent ( $hWndChild, $hWndParent )

Параметры

$hWndChild Дескриптор окна дочернего окна.
$hWndParent Дескриптор нового родительского окна. Если 0, окно рабочего стола становится новым родительским окном.

Возвращаемое значение

Успех: Возвращает дескриптор предыдущего родительского окна.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Приложение может использовать функцию SetParent для установки родительского окна всплывающего, перекрывающегося или дочернего
окна. Новое родительское окно и дочернее окно должны принадлежать одному приложению.

Дополнительно

Искать SetParent в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <WinAPISysWin.au3>
#include <WindowsStylesConstants.au3>

Local $hParent = WinGetHandle('[CLASS:Progman;TITLE:Program Manager]')

Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400, 100, 100, -1, $WS_EX_TOOLWINDOW)

; Attach window to the desktop (always on bottom)
_WinAPI_SetParent($hForm, $hParent)

GUISetState(@SW_SHOWNOACTIVATE)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
