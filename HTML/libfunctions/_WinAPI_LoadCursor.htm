Справочник функций

_WinAPI_LoadCursor

Загружает указанный ресурс курсора из исполняемого файла (.exe).

#include <WinAPIRes.au3>
_WinAPI_LoadCursor ( $hInstance, $sName )

Параметры

$hInstance Дескриптор экземпляра модуля, исполняемый файл которого содержит загружаемый курсор.
$sName Имя ресурса курсора или идентификатор ресурса для загрузки. Для использования одного из предопределённых
курсоров приложение должно установить параметр $hInstance в 0
и параметр $sName в одно из следующих значений.
$OCR_NORMAL ; IDC_ARROW в MSDN
$OCR_IBEAM
$OCR_WAIT
$OCR_CROSS
$OCR_UP    ; IDC_UPARROW в MSDN
$OCR_SIZE
$OCR_ICON
$OCR_SIZENWSE
$OCR_SIZENESW
$OCR_SIZEWE
$OCR_SIZENS
$OCR_SIZEALL
$OCR_NO
$OCR_HAND
$OCR_APPSTARTING
$OCR_HELP

Поскольку $IDC_* уже определены для MouseGetCursor(), следует использовать $OCR_*, несмотря на именование в документации MSDN.

Возвращаемое значение

Успех: Возвращает дескриптор загруженного курсора.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Эта функция загружает ресурс курсора только если он ещё не был загружен; в противном случае возвращает дескриптор
существующего ресурса.
