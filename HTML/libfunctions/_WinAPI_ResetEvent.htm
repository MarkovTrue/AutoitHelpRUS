Справочник функций

_WinAPI_ResetEvent

Устанавливает указанный объект события в несигнальное состояние.

#include <WinAPIProc.au3>
_WinAPI_ResetEvent ( $hEvent )

Параметры

$hEvent Дескриптор объекта события. Функция _WinAPI_CreateEvent() возвращает этот дескриптор.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Состояние объекта события остаётся несигнальным, пока оно не будет явно установлено в сигнальное состояние функцией _WinAPI_SetEvent().
Это несигнальное состояние блокирует выполнение любых потоков, которые указали объект события при вызове одной из функций _WinAPI_Wait....

Функция _WinAPI_ResetEvent() используется в основном для объектов события с ручным сбросом, которые должны быть явно установлены в несигнальное состояние.
Объекты события с автоматическим сбросом автоматически переходят из сигнального в несигнальное состояние после освобождения одного ожидающего потока.

См. также

_WinAPI_CreateEvent, _WinAPI_SetEvent

Искать ResetEvent в библиотеке MSDN

Пример

#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

Global $g_tEvents = DllStructCreate("handle Event[3];")
$g_tEvents.Event(1) = _WinAPI_CreateEvent(0, True, False)
$g_tEvents.Event(2) = _WinAPI_CreateEvent(0, True, False)
$g_tEvents.Event(3) = _WinAPI_CreateEvent(0, True, False)

HotKeySet("{ESC}", "_Exit")
AdlibRegister("_FireEvent_1", 500)
AdlibRegister("_FireEvent_2", 800)

Local $iEvent
While 1
    $iEvent = _WinAPI_WaitForMultipleObjects(3, $g_tEvents, False, 100)
    Switch $iEvent
        Case 0
            ConsoleWrite("+ First Event" & @CRLF)
            _WinAPI_ResetEvent($g_tEvents.Event(1))
        Case 1
            ConsoleWrite("> Second Event" & @CRLF)
            _WinAPI_ResetEvent($g_tEvents.Event(2))
        Case 2
            ConsoleWrite("! Exit Event" & @CRLF)
            AdlibUnRegister("_FireEvent_1")
            AdlibUnRegister("_FireEvent_2")
            _WinAPI_CloseHandle($g_tEvents.Event(1))
            _WinAPI_CloseHandle($g_tEvents.Event(2))
            _WinAPI_CloseHandle($g_tEvents.Event(3))
            ExitLoop
        Case -1 ;Ошибка
            ExitLoop
    EndSwitch
    Sleep(10)
WEnd

Func _Exit()
    _WinAPI_SetEvent($g_tEvents.Event(3))
EndFunc   ;==>_Exit

Func _FireEvent_1()
    _WinAPI_SetEvent($g_tEvents.Event(1))
EndFunc   ;==>_FireEvent_1

Func _FireEvent_2()
    _WinAPI_SetEvent($g_tEvents.Event(2))
EndFunc   ;==>_FireEvent_2
