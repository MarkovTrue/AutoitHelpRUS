Справочник функций

_WinAPI_ActivateKeyboardLayout

Устанавливает идентификатор локали ввода для вызывающего потока или текущего процесса.

#include <WinAPISys.au3>
_WinAPI_ActivateKeyboardLayout ( $hLocale [, $iFlag = 0] )

Параметры

$hLocale Идентификатор локали ввода, который будет активирован.
Этот параметр должен быть либо дескриптором раскладки клавиатуры, либо одним из следующих значений:
$HKL_NEXT
$HKL_PREV
$iFlag [необязательный] Флаг, который определяет способ активации идентификатора локали ввода.
Этот параметр может быть одним из следующих значений:
$KLF_REORDER
$KLF_RESET
$KLF_SETFORPROCESS
$KLF_SHIFTLOCK

Возвращаемое значение

Успех: Возвращает предыдущий идентификатор локали ввода.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Пример

#include <StaticConstants.au3>
#include <WinAPILocale.au3>
#include <WinAPISys.au3>

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 300, 200)
Local $idLabel = GUICtrlCreateLabel('', 10, 66, 280, 40, $SS_CENTER)
GUICtrlSetFont(-1, 12, 800, 0, 'Tahoma')
Local $idBtn_Prev = GUICtrlCreateButton('<', 82, 164, 60, 24)
Local $idBtn_Next = GUICtrlCreateButton('>', 158, 164, 60, 24)
GUISetState(@SW_SHOW)

Local $iUpdate = 1
While 1
    If $iUpdate Then
        GUICtrlSetData($idLabel, '0x' & StringRight(@KBLayout, 4) & @CRLF & _WinAPI_GetLocaleInfo(Number('0x' & @KBLayout), $LOCALE_SLANGUAGE))
        $iUpdate = 0
    EndIf

    Switch GUIGetMsg()
        Case -3
            ExitLoop
        Case $idBtn_Next
            _WinAPI_ActivateKeyboardLayout($HKL_NEXT)
            $iUpdate = 1
        Case $idBtn_Prev
            _WinAPI_ActivateKeyboardLayout($HKL_PREV)
            $iUpdate = 1
    EndSwitch
WEnd
