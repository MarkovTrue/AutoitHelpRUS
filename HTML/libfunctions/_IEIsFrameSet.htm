Справочник функций

_IEIsFrameSet

Проверяет, содержит ли указанное окно FrameSet.

#include <IE.au3>
_IEIsFrameSet ( ByRef $oObject )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.

Возвращаемое значение

Успех: Возвращает 1, если объект ссылается на страницу FrameSet.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) – неверный тип данных
@extended: Содержит номер недопустимого параметра.

Примечания

_IEFrameGetCollection() не различает, являются ли найденные фреймы частью FrameSet или это iFrame.
Если на странице есть фреймы, а эта функция возвращает False, значит, это iFrame.

См. также

_IEFrameGetCollection

Пример

; Открыть пример с frameset, получить коллекцию фреймов,
; проверить количество фреймов, отобразить число фреймов или iFrame, присутствующих на странице

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("frameset")
Local $oFrames = _IEFrameGetCollection($oIE)
Local $iNumFrames = @extended
If $iNumFrames > 0 Then
    If _IEIsFrameSet($oIE) Then
        MsgBox($MB_SYSTEMMODAL, "Frame Info", "Page contains " & $iNumFrames & " frames in a FrameSet")
    Else
        MsgBox($MB_SYSTEMMODAL, "Frame Info", "Page contains " & $iNumFrames & " iFrames")
    EndIf
Else
    MsgBox($MB_SYSTEMMODAL, "Frame Info", "Page contains no frames")
EndIf

_IEQuit($oIE)
