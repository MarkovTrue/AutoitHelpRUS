Справочник функций

_SQLite_Query

Подготавливает запрос SQLite.

#include <SQLite.au3>
_SQLite_Query ( $hDB, $sSQL, ByRef $hQuery )

Параметры

$hDB Открытая база данных; -1 – использовать последнюю открытую базу данных.
$sSQL Выполняемый SQL-запрос.
$hQuery Выходной параметр, дескриптор запроса.

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове SQLite API "sqlite3_prepare16_v2"
2 – вызов запрещён безопасным режимом

См. также

_SQLite_Exec, _SQLite_FetchData, _SQLite_FetchNames, _SQLite_GetTable, _SQLite_GetTable2D, _SQLite_QueryFinalize, _SQLite_QueryReset, _SQLite_QuerySingleRow, _SQLite_SetTimeout

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $hQuery, $aRow, $sMsg
_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; открыть :memory: базу данных
_SQLite_Exec(-1, "CREATE TABLE aTest (a,b,c);") ; Создание таблицы
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('c','2','World');") ; Вставить данные
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('b','3',' ');") ; Вставить данные
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('a','1','Hello');") ; Вставить данные
_SQLite_Query(-1, "SELECT c FROM aTest ORDER BY a;", $hQuery) ; запрос
While _SQLite_FetchData($hQuery, $aRow) = $SQLITE_OK
    $sMsg &= $aRow[0]
WEnd
_SQLite_Exec(-1, "DROP TABLE aTest;") ; Удаление таблицы
MsgBox($MB_SYSTEMMODAL, "SQLite", "Get Data using a Query : " & $sMsg)
_SQLite_Close()
_SQLite_Shutdown()

; Вывод:
; Hello World
