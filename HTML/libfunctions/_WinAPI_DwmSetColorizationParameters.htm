_WinAPI_DwmSetColorizationParameters

Задаёт параметры окраски для Диспетчера окон рабочего стола (DWM).

#include <WinAPIGdi.au3>
_WinAPI_DwmSetColorizationParameters ( $tDWMCP )

Параметры

$tDWMCP Структура $tagDWM_COLORIZATION_PARAMETERS, содержащая задаваемые параметры окраски.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
