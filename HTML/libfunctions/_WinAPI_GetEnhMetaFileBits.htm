_WinAPI_GetEnhMetaFileBits

Получает содержимое указанного улучшенного метафайла (EMF).

#include <WinAPIGdi.au3>
_WinAPI_GetEnhMetaFileBits ( $hEmf, ByRef $pBuffer )

Параметры

$hEmf Дескриптор расширенного метафайла.
$pBuffer Возвращает указатель на блок памяти (буфер), получающий данные метафайла.
Вы можете установить этот параметр в 0 перед вызовом функции, и она выделит необходимый блок памяти самостоятельно.
В противном случае это должен быть действительный указатель на память, возвращённый функцией _WinAPI_CreateBuffer(), или полученный предыдущим вызовом этой функции.

Возвращаемое значение

Успех: Возвращает количество байт, скопированных в буфер.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

После получения бит метафайла их можно использовать для создания метафайла, основанного на памяти, вызвав функцию _WinAPI_SetEnhMetaFileBits().

Когда буфер, выделенный этой функцией, больше не нужен, следует вызвать функцию _WinAPI_FreeMemory() (не используйте другие подпрограммы работы с памятью) для освобождения занимаемой памяти.

См. также

_WinAPI_CreateBuffer, _WinAPI_FreeMemory, _WinAPI_SetEnhMetaFileBits
