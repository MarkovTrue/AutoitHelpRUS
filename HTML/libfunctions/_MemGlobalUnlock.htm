_MemGlobalUnlock

Уменьшает счётчик блокировки объекта памяти, выделенного с помощью GMEM_MOVEABLE.

#include <Memory.au3>
_MemGlobalUnlock ( $hMemory )

Параметры

$hMemory Дескриптор объекта глобальной памяти.

Возвращаемое значение

Отсутствует.
