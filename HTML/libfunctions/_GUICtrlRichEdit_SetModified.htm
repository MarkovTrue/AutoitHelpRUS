Справочник функций

_GUICtrlRichEdit_SetModified

Устанавливает или очищает флаг изменения.

#include <GuiRichEdit.au3>
_GUICtrlRichEdit_SetModified ( $hWnd [, $bState = True] )

Параметры

$hWnd Дескриптор элемента управления.
$bState [необязательный] Новое значение флага изменения:
True – текст изменён (по умолчанию)
False – текст не изменён.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@error: 101 – $hWnd не является дескриптором
102 – $bState должен быть True или False

Примечания

Система автоматически устанавливает флаг изменения в False при создании элемента управления.

См. также

_GUICtrlRichEdit_IsModified

Дополнительно

Искать EM_SETMODIFY в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <GuiRichEdit.au3>
#include <WindowsStylesConstants.au3>

Global $g_idLbl_Msg, $g_hRichEdit

Example()

Func Example()
    Local $hGui, $iMsg, $idBtn_Next, $iStep = 0
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName, StringLen(".exe")) & ")", 320, 350, -1, -1)
    $g_hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 300, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    $g_idLbl_Msg = GUICtrlCreateLabel("", 10, 235, 300, 60)
    $idBtn_Next = GUICtrlCreateButton("Next", 270, 310, 40, 30)
    GUISetState(@SW_SHOW)

    Report("Initial state after creation")

    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                _GUICtrlRichEdit_Destroy($g_hRichEdit) ; нужно, если только скрипт не упадёт
                ; GUIDelete() ; так тоже работает
                Exit
            Case $iMsg = $idBtn_Next
                $iStep += 1
                Switch $iStep
                    Case 1
                        _GUICtrlRichEdit_AppendText($g_hRichEdit, @CRLF & "A paragraph")
                        Report("Added some text")
                    Case 2
                        _GUICtrlRichEdit_SetModified($g_hRichEdit, False)
                        Report("After clearing the modification flag")
                    Case 3
                        GUICtrlSetData($g_idLbl_Msg, "Type some text. Then click on Next")
                    Case 4
                        Report("After typing")
                        GUICtrlSetState($idBtn_Next, $GUI_DISABLE)
                EndSwitch
        EndSelect
    WEnd
EndFunc   ;==>Example

Func Report($sMsg)
    $sMsg = $sMsg & @CRLF & "State is " & _GUICtrlRichEdit_IsModified($g_hRichEdit)
    GUICtrlSetData($g_idLbl_Msg, $sMsg)
EndFunc   ;==>Report
