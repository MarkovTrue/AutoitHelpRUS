_GDIPlus_ImageRotateFlip

Поворачивает и отражает изображение.

#include <GDIPlus.au3>
_GDIPlus_ImageRotateFlip ( $hImage, $iRotateFlipType )

Параметры

$hImage Указатель на объект Image.
$iRotateFlipType Тип поворота и отражения:
0 – без поворота и отражения (эквивалентно: поворот на 180°, горизонтальное + вертикальное отражение)
1 – поворот на 90° без отражения (эквивалентно: поворот на 270°, горизонтальное + вертикальное отражение)
2 – поворот на 180° без отражения (эквивалентно: без поворота, горизонтальное + вертикальное отражение)
3 – поворот на 270° без отражения (эквивалентно: поворот на 90°, горизонтальное + вертикальное отражение)
4 – без поворота, горизонтальное отражение (эквивалентно: поворот на 180° + вертикальное отражение)
5 – поворот на 90° + горизонтальное отражение (эквивалентно: поворот на 270° + вертикальное отражение)
6 – поворот на 180° + горизонтальное отражение (эквивалентно: без поворота + вертикальное отражение)
7 – поворот на 270° + горизонтальное отражение (эквивалентно: поворот на 90° + вертикальное отражение)

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPIHObj.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = @DesktopWidth / 2, $iH = @DesktopHeight / 2

    Local $hHBmp = _ScreenCapture_Capture("", 0, 0, @DesktopWidth / 2, @DesktopHeight / 2) ;создать GDI bitmap захватом 1/4 рабочего стола
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;преобразовать GDI bitmap в GDI+ bitmap
    _WinAPI_DeleteObject($hHBmp) ;освободить ресурс GDI bitmap – больше не нужен
    _GDIPlus_ImageRotateFlip($hBitmap, 1) ;повернуть изображение на 90° без отражения

    Local $hGUI = GUICreate("GDI+ test", $iH, $iW, -1, -1) ;создать тестовый GUI для отображения повёрнутого изображения
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать графический объект из дескриптора окна
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap, 0, 0) ;отобразить повёрнутое изображение

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;освободить ресурсы
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
