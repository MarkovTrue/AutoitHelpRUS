Справочник функций

_IsPressed

Проверяет, была ли нажата клавиша.

#include <Misc.au3>
_IsPressed ( [$vKey = Default [, $vDLL = Default [, $bCheckModifierKey = False]]] )

Параметры

$vKey [необязательный] Клавиша для проверки (строка), $VK_* как определено в WinAPIvkeysConstants.au3 или массив клавиш для проверки. Если Default, то внутренний дескриптор DLL будет закрыт. Не забудьте закрыть его с помощью _IsPressed() когда он больше не нужен.
$vDLL [необязательный] Дескриптор DLL или по умолчанию user32.dll
$bCheckModifierKey [необязательный] Клавиши модификатора SHIFT, CTRL, ALT или WIN будут проверены, чтобы избежать возврата "клавиша модификатора" + $vKey при нажатии только $vKey (см. раздел Примечания) (По умолчанию)

Возвращаемое значение

>0: Возвращает 1, если клавиша нажата, или порядковый номер нажатой клавиши в массиве (см. раздел Примечания).
0: Возвращает 0, если клавиша не нажата или нет активного _IsPressed(...).

Примечания

При повторяющихся вызовах функции следует открыть "user32.dll" и передать дескриптор.
Обязательно закройте дескриптор в конце скрипта с помощью DllClose().
Если вы используете $vDLL = Default и не завершаете скрипт AutoIt, а затем перестаёте использовать _IsPressed(...),
тогда вызовите _IsPressed() для закрытия внутреннего дескриптора.

Если $vKey – строка или $VK_*, как определено в WinAPIvkeysConstants.au3, то
    _IsPressed() будет возвращать 1, пока клавиша нажата.
    Даже кратковременное нажатие может привести к нескольким возвращаемым значениям в цикле.
    Если вызванный код не содержит блокирующую функцию (такую как MsgBox) и пользователю не требуются несколько возвращений,
    скрипт должен дождаться, пока _IsPressed() вернёт 0, прежде чем продолжить (см. пример 1).

Если $vKey – массив, функция возвращает значение при нажатии клавиши,
    возвращаемое значение – 1, если нажата $vKey[0], 2, если нажата $vKey[1], ... (см. пример 2).

Если $bCheckModifierKey не равна True, то комбинация "клавиши модификатора" + клавиша вернёт 0,
    и @extended устанавливается при нажатии клавиши модификатора: 1 = SHIFT, 2 = CTRL, 3 = ALT, 4 = WIN, 5 = правая WIN (см. пример 2).

Если $bCheckModifierKey = True, возврат происходит при отпускании "клавиши модификатора".
Если $bCheckModifierKey = -1, возврат происходит при нажатии "клавиши модификатора".

Пример

Пример 1

; == Пример 1 : _IsPressed с использованием $vkey в качестве строки

#include <Misc.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIvkeysConstants.au3>

Local $hDLL = DllOpen("user32.dll")

While 1
    If _IsPressed("10", $hDLL) Then
        ConsoleWrite("_IsPressed - Shift Key was pressed. @extended = " & @extended & @CRLF)
        ; Ждать отпускания клавиши.
        While _IsPressed("10", $hDLL)
            Sleep(100)
        WEnd
        ConsoleWrite("_IsPressed - Shift Key was released. @extended = " & @extended & @CRLF)
    ElseIf _IsPressed("1B", $hDLL) Then
        ConsoleWrite("_IsPressed - Esc Key was pressed. @extended = " & @extended & @CRLF)
        MsgBox($MB_SYSTEMMODAL, "_IsPressed", "The Esc Key was pressed, therefore we will close the application.", 3)
        ExitLoop
    EndIf
    Sleep(100)
WEnd

DllClose($hDLL)

Пример 2

; == Пример 2 : _IsPressed с использованием $vkey как массива с проверкой клавиш-модификаторов

#include <Misc.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIvkeysConstants.au3>

Local $asModifierKeys[6] = [0, "VK_SHIFT", "VK_CONTROL", "VK_MENU", "VK_LWIN", "VK_RWIN"]

Local $aKeys[3] = [$VK_ESCAPE, $VK_LBUTTON, $VK_TAB]
While 1
    Local $iRet = _IsPressed($aKeys, Default, True) ; Проверить модификатор
    If Not $iRet And @extended Then ConsoleWrite("The modifier key " & $asModifierKeys[@extended] & " has been pressed. @extended = " & @extended & @CRLF)

    Local $sKey
    Switch $iRet
        Case 1 ; Клавиатура ESC
            $sKey = "{ESCAPE}"
            ExitLoop

        Case 2 ; MouseClick Left
            $sKey = "{LBUTTON}"
            ExitLoop

        Case 3 ; Клавиатура Tab
            $sKey = "{TAB}"
            ExitLoop

    EndSwitch

    Sleep(100)
WEnd

_IsPressed() ; вызвать DLLClose для Dll по умолчанию, созданной предыдущим _IsPressed_(...)

Send("^z") ; чтобы избежать изменения в окне Scite

ConsoleWrite("The key " & $sKey & " has been pressed." & @CRLF)
MsgBox($MB_SYSTEMMODAL, "Result", "The key " & $sKey & " has been pressed.", 2)

Пример 3

; == Пример 3 : _IsPressed с использованием $vkey как целого числа с проверкой клавиш-модификаторов

#include <Misc.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIvkeysConstants.au3>

Example()
Func Example()
    While 1
        If _IsPressed($VK_F2, Default, True) Then
            ConsoleWrite("_IsPressed - F2 (without any modifier keys pressed) was pressed." & @CRLF)
            ; Ждать отпускания клавиши.
            While _IsPressed($VK_F2, Default, True)
                Sleep(100)
            WEnd
            ConsoleWrite("_IsPressed - F2 (without any modifier keys pressed) was released." & @CRLF)
        ElseIf _IsPressed($VK_ESCAPE) Then
            MsgBox($MB_SYSTEMMODAL, "_IsPressed", "The Esc Key was pressed, therefore we will close the application.", 3)
            ExitLoop
        EndIf
        Sleep(100)
    WEnd

    _IsPressed() ; чтобы освободить внутренний дескриптор, созданный предыдущим вызовом _IsPressed(...)

EndFunc   ;==>Example
