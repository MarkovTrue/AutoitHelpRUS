Справочник функций

_WinAPI_FillRect

Заливает прямоугольник указанной кистью.

#include <WinAPIGdiDC.au3>
_WinAPI_FillRect ( $hDC, $tRECT, $hBrush )

Параметры

$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT или указатель на неё с логическими координатами заливаемого прямоугольника.
$hBrush Дескриптор кисти, используемой для заливки прямоугольника.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Параметр $hBrush может быть дескриптором логической кисти или значением цвета.
Для логической кисти используйте _WinAPI_CreateSolidBrush().
Также дескриптор стандартной кисти можно получить через _WinAPI_GetStockObject().
Если $hBrush задаётся как значение цвета, оно должно быть одним из стандартных системных цветов (к выбранному цвету нужно прибавить 1).
