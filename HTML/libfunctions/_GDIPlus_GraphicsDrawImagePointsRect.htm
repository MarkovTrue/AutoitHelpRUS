_GDIPlus_GraphicsDrawImagePointsRect

Рисует изображение.

#include <GDIPlus.au3>
_GDIPlus_GraphicsDrawImagePointsRect ( $hGraphics, $hImage, $nULX, $nULY, $nURX, $nURY, $nLLX, $nLLY, $nSrcX, $nSrcY, $nSrcWidth, $nSrcHeight [, $hImageAttributes = 0 [, $iUnit = 2]] )

Параметры

$hGraphics Указатель на объект Graphics.
$hImage Указатель на объект Image.
$nULX Координата X верхнего левого угла исходного изображения.
$nULY Координата Y верхнего левого угла исходного изображения.
$nURX Координата X верхнего правого угла исходного изображения.
$nURY Координата Y верхнего правого угла исходного изображения.
$nLLX Координата X нижнего левого угла исходного изображения.
$nLLY Координата Y нижнего левого угла исходного изображения.
$nSrcX Координата X верхнего левого угла области исходного изображения для отрисовки.
$nSrcY Координата Y верхнего левого угла области исходного изображения для отрисовки.
$nSrcWidth Ширина области исходного изображения для отрисовки.
$nSrcHeight Высота области исходного изображения для отрисовки.
$hImageAttributes [необязательный] Указатель на объект ImageAttributes, задающий атрибуты цвета и размера отображаемого изображения.
$iUnit [необязательный] Единица измерения:
0 – мировые координаты (нефизическая единица)
1 – единицы отображения
2 – 1 пиксель
3 – 1 пункт (1/72 дюйма)
4 – 1 дюйм
5 – 1/300 дюйма
6 – 1 миллиметр

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Поддержка работы в X64
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ;получить каталог установки AutoIt
    Local $sRegPath = "HKLM\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt"

    Local $sFile = RegRead($sRegPath, "InstallDir") & "\Examples\GUI\logo4.gif"
    If Not FileExists($sFile) Then
        MsgBox(($MB_SYSTEMMODAL + $MB_ICONHAND), "", $sFile & " not found!", 30)
        Return False
    EndIf

    _GDIPlus_Startup()
    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile) ;создать объект изображения на основе файла
    If @error Then
        _GDIPlus_Shutdown()
        MsgBox(($MB_SYSTEMMODAL + $MB_ICONHAND), "", "An error has occured - unable to load image!", 30)
        Return False
    EndIf

    Local $hGUI, $hGraphic, $iImgW, $iImgH

    ; Создать GUI
    $hGUI = GUICreate("GDI+", 800, 400)
    GUISetState(@SW_SHOW)
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Создать графический объект из дескриптора окна

    $iImgW = _GDIPlus_ImageGetWidth($hImage)
    $iImgH = _GDIPlus_ImageGetHeight($hImage)

    _GDIPlus_GraphicsDrawImagePointsRect($hGraphic, $hImage, 100, 10, 300, 60, 90, 160, 0, 0, $iImgW * 0.45, $iImgH)
    _GDIPlus_GraphicsDrawImagePointsRect($hGraphic, $hImage, 300, 60, 360, 30, 290, 210, $iImgW * 0.45, 0, $iImgW * 0.2, $iImgH)
    _GDIPlus_GraphicsDrawImagePointsRect($hGraphic, $hImage, 360, 30, 560, 80, 350, 180, $iImgW * 0.65, 0, $iImgW * 0.35, $iImgH)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;освободить ресурсы
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
