_NamedPipes_GetNamedPipeHandleState

Получает информацию об указанном именованном канале.

#include <NamedPipes.au3>
_NamedPipes_GetNamedPipeHandleState ( $hNamedPipe )

Параметры

$hNamedPipe Дескриптор конца сервера экземпляра именованного канала.

Возвращаемое значение

Успех: Возвращает массив в следующем формате:
$aState[0] – True, если дескриптор канала в режиме без блокировки, иначе режим блокировки
$aState[1] – True, если дескриптор канала в режиме чтения сообщений, иначе режим чтения байтов
$aState[2] – количество текущих экземпляров канала
$aState[3] – максимальное количество байтов для сбора на компьютере клиента перед передачей
$aState[4] – максимальное время в миллисекундах, которое может пройти перед передачей данных удалённым именованным каналом по сети
$aState[5] – строка с именем пользователя, связанная с клиентским приложением. Сервер может получить эту информацию только если клиент открыл канал с привилегией доступа SECURITY_IMPERSONATION
Ошибка: Устанавливает @error ≠ 0.
