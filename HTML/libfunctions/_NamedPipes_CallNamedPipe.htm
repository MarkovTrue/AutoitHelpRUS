_NamedPipes_CallNamedPipe

Выполняет операцию чтения/записи на именованном канале.

#include <NamedPipes.au3>
_NamedPipes_CallNamedPipe ( $sPipeName, $pInpBuf, $iInpSize, $pOutBuf, $iOutSize, ByRef $iRead [, $iTimeOut = 0] )

Параметры

$sPipeName Имя канала.
$pInpBuf Указатель на буфер, содержащий данные, записываемые в канал.
$iInpSize Размер буфера записи в байтах.
$pOutBuf Указатель на буфер, который получает данные, прочитанные из канала.
$iOutSize Размер буфера чтения в байтах.
$iRead При возврате содержит количество байтов, прочитанных из канала.
$iTimeOut [необязательный] Количество миллисекунд для ожидания доступности именованного канала.
Помимо числовых значений можно указать следующие специальные значения:
-1 – ждать бесконечно
0 – использует тайм-аут по умолчанию, указанный при вызове _NamedPipes_CreateNamedPipe()
1 – не ждать. Если канал недоступен – вернуть ошибку

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Вызов _NamedPipes_CallNamedPipe() эквивалентен вызову функций _WinAPI_CreateFile() (или _NamedPipes_WaitNamedPipe(), если _WinAPI_CreateFile() не может открыть канал немедленно), TransactNamedPipe и CloseHandle.
_WinAPI_CreateFile() вызывается с флагом доступа GENERIC_READ | GENERIC_WRITE и флагом наследования дескриптора False.
_NamedPipes_CallNamedPipe() не удаётся выполнить, если канал является каналом байтового типа.

См. также

_NamedPipes_TransactNamedPipe, _NamedPipes_WaitNamedPipe
