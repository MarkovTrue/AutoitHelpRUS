Справочник функций

_Word_DocOpen

Открывает документ Microsoft Word.

#include <Word.au3>
_Word_DocOpen ( $oAppl, $sFilePath [, $bConfirmConversions = False [, $iFormat = $WdOpenFormatAuto [, $bReadOnly = False [, $bRevert = False [, $bAddToRecentFiles = False [, $sOpenPassword = "" [, $sWritePassword = ""]]]]]]] )

Параметры

$oAppl Объект Word, возвращённый предыдущим вызовом _Word_Create().
$sFilePath Имя или полный путь документа для открытия.
$bConfirmConversions [необязательный] True отображает диалоговое окно Convert File, если файл не в формате Microsoft Word (по умолчанию = False)
$iFormat [необязательный] Указывает формат для использования при открытии документа. Может быть любым из перечисления WdOpenFormat (по умолчанию = $WdOpenFormatAuto = существующий формат)
$bReadOnly [необязательный] True открыть документ только для чтения (по умолчанию = False). Примечание: этот аргумент не переопределяет параметр, рекомендуемый только для чтения в сохранённом документе.
$bRevert [необязательный] Управляет тем, что происходит, если $sFilePath является именем открытого документа.     True отбросить любые несохранённые изменения в открытом документе и переоткрыть файл.     False активировать открытый документ (по умолчанию = False)
$bAddToRecentFiles [необязательный] True добавляет имя файла в список недавно использованных файлов в нижней части меню File (по умолчанию = False)
$sOpenPassword [необязательный] Пароль для открытия документа
$sWritePassword [необязательный] Пароль для сохранения изменений в документе

Возвращаемое значение

Успех: Возвращает переменная, указывающая на объект документа. @extended задан 1, если $bReadOnly = False, но доступ для чтения и записи не удалось предоставить. Для получения подробной информации см. раздел Примечания.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – $oAppl не является объектом 2 – Указанный документ не существует 3 – При открытии указанного документа произошла ошибка. @extended задан код ошибки COM

Примечания

Если установлено $bReadOnly = False, но документ не может быть открыт для чтения и записи, @extended задан 1. Документ был открыт только для чтения, так как он уже был открыт другим пользователем/задачей или файл установлен только для чтения файловой системой. Если вы изменяете документ, вам необходимо использовать _Word_DocSaveAs() для сохранения его в другое место или с другим именем.

См. также

_Word_DocClose, _Word_DocSave, _Word_DocSaveAs

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocOpen Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Открыть документ только для чтения
Local $sDocument = _Extras_PathFull("Test.doc")
_Word_DocOpen($oWord, $sDocument, Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocOpen Example 1", "Error opening '.\Extras\Test.doc'." & _
        @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocOpen Example 1", "Document '" & $sDocument & "' has been opened successfully.")
