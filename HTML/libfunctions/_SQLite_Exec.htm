Справочник функций

_SQLite_Exec

Выполняет запрос SQLite, не обрабатывает результаты.

#include <SQLite.au3>
_SQLite_Exec ( $hDB, $sSQL [, $sCallBack = ""] )

Параметры

$hDB Открытая база данных; -1 – использовать последнюю открытую базу данных.
$sSQL Выполняемый SQL-запрос.
$sCallback [необязательный] если указана, функция будет вызвана для каждой строки

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове SQLite API "sqlite3_exec"
2 – вызов запрещён безопасным режимом
3 – ошибка при обработке обратного вызова из _SQLite_GetTable2D()
4 – ошибка при преобразовании SQL-инструкции в UTF-8

Примечания

Функция обратного вызова должна принять 1 параметр и может вернуть $SQLITE_ABORT для остановки обработки (см. пример).
Первая строка в последовательности обратного вызова будет содержать имена столбцов.

См. также

_SQLite_GetTable, _SQLite_GetTable2D, _SQLite_Query

Пример

#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $hQuery
_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open()

; Без $sCallback это инструкция без результата
_SQLite_Exec(-1, "Create table tblTest (a,b int,c single not null);" & _
        "Insert into tblTest values ('1',2,3);" & _
        "Insert into tblTest values (Null,5,6);")

Local $d = _SQLite_Exec(-1, "Select rowid,* From tblTest", "_cb") ; _cb будет вызвана для каждой строки

_SQLite_Close()
_SQLite_Shutdown()

; Вывод:
;rowid a b c
; 1 1 2 3
; 2 5 6

Func _cb($aResult)
    For $s In $aResult
        ConsoleWrite($s & @TAB)
    Next
    ConsoleWrite(@CRLF)
    ; Вернуть $SQLITE_ABORT; приведёт к прерыванию процесса и вызовет @error в _SQLite_Exec()
EndFunc   ;==>_cb
