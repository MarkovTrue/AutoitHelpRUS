Справочник функций

_ArrayExtract

Извлекает массив из указанных элементов одномерного 1D или двумерного 2D массива.

#include <Array.au3>
_ArrayExtract ( Const ByRef $aArray [, $iStart_Row = -1 [, $iEnd_Row = -1 [, $iStart_Col = -1 [, $iEnd_Col = -1]]]] )

Параметры

$aArray Исходный массив для извлечения.
$iStart_Row [необязательный] Первая строка извлекаемого массива
$iEnd_Row [необязательный] Последняя строка извлекаемого массива
$iStart_Col [необязательный] Первый столбец извлекаемого массива (только для 2D массива)
$iEnd_Col [необязательный] Последний столбец извлекаемого массива (только для 2D массива)

Возвращаемое значение

Успех: Возвращает извлечённый массив.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не массив
2 – $aArray не 1D и не 2D массив
3 – $iStart_Row или $iEnd_Row вне границ массива
4 – $iStart_Row больше $iEnd_Row
5 – $iStart_Col или $iEnd_Col вне границ массива
6 – $iStart_Col больше $iEnd_Col

См. также

_ArrayAdd, _ArrayInsert

Пример

#include <Array.au3>

Local $aArray[4][4]
For $i = 0 To 3
    For $j = 0 To 3
        $aArray[$i][$j] = $i & $j
    Next
Next
_ArrayDisplay($aArray, "Original")
Local $aExtract = _ArrayExtract($aArray, 1, 2, 2, 3)
_ArrayDisplay($aExtract, "Row 1-2 cols 2-3")
