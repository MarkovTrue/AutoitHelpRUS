Справочник функций

_WinAPI_GetFileVersionInfo

Получает информацию о версии для указанного файла.

#include <WinAPIRes.au3>
_WinAPI_GetFileVersionInfo ( $sFilePath, ByRef $pBuffer [, $iFlags = 0] )

Параметры

$sFilePath Имя файла.
$pBuffer Возвращает указатель на блок памяти (буфер), получающий информацию о версии файла.
Вы можете установить этот параметр в 0 перед вызовом функции, и она выделит необходимый блок памяти самостоятельно.
В противном случае это должен быть действительный указатель на память, возвращённый функцией _WinAPI_CreateBuffer(), или полученный предыдущим вызовом этой функции.
$iFlags [необязательный] Флаги, управляющие тем, из каких DLL-файлов MUI (если они есть) извлекается ресурс версии.
Может быть 0 или более следующих значений (действительно только для Windows Vista или более поздней версии).
$FILE_VER_GET_LOCALISED
$FILE_VER_GET_NEUTRAL
$FILE_VER_GET_PREFETCHED

Возвращаемое значение

Успех: Возвращает количество байт, скопированных в буфер.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Когда буфер, выделенный этой функцией, больше не нужен, следует вызвать функцию _WinAPI_FreeMemory() (не используйте другие подпрограммы работы с памятью) для освобождения занимаемой памяти.

См. также

_WinAPI_CreateBuffer, _WinAPI_FreeMemory
