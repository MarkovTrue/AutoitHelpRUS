Справочник функций

_FTP_GetLastResponseInfo

Возвращает последнее описание ошибки или ответ сервера для вызывающего потока.

#include <FTPEx.au3>
_FTP_GetLastResponseInfo ( ByRef $iError, ByRef $sMessage )

Параметры

$iError Возвращает сообщение об ошибке, связанной с неудавшейся операцией.
$sMessage Возвращает текст ошибки.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Дополнительно

Искать InternetGetLastResponseInfo в библиотеке MSDN

Пример

#include <FTPEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
;~ Local $sServer = 'ftp.cs.brown.edu' ; Brown Computer Science
    Local $sServer = 'speedtest.tele2.net' ; Сервис Tele2 Speedtest
    Local $sUsername = ''
    Local $sPass = ''

    Local $hOpen = _FTP_Open('MyFTP Control')
    Local $hConn = _FTP_Connect($hOpen, $sServer, $sUsername, $sPass)
    If @error Then
        MsgBox($MB_SYSTEMMODAL, '_FTP_Connect', 'ERROR=' & @error)
    Else
        Local $iErr = @error, $sFTP_Message
        _FTP_GetLastResponseInfo($iErr, $sFTP_Message)
        ConsoleWrite('$iErr=' & $iErr & ' $sFTP_Message:' & @CRLF & $sFTP_Message & @CRLF)
        ; выполнить нужные действия...
    EndIf
    _FTP_Close($hConn)
    _FTP_Close($hOpen)
EndFunc   ;==>_Example
