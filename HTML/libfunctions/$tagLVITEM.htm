Справочник ключевых слов

$tagLVITEM

Задаёт или принимает атрибуты элемента ListView.

Global Const $tagLVITEM = "struct; uint Mask;int Item;int SubItem;uint State;uint StateMask;ptr Text;int TextMax;int Image;lparam Param;" & _
"int Indent;int GroupID;uint Columns;ptr pColumns;ptr piColFmt;int iGroup; endstruct"

Параметры

Mask Набор флагов, определяющих, какие члены этой структуры содержат данные для установки или какие члены запрашиваются. Этот член может иметь один или несколько из следующих флагов:
$LVIF_COLFMT – член piColFmt действителен или должен быть установлен. Если используется этот флаг, то член cColumns действителен или должен быть установлен Windows Vista и позже.
$LVIF_COLUMNS – член Columns действителен
$LVIF_DI_SETITEM – операционная система должна сохранить запрашиваемую информацию элемента списка
$LVIF_GROUPID – член GroupID действителен
$LVIF_IMAGE – член Image действителен
$LVIF_INDENT – член Indent действителен
$LVIF_NORECOMPUTE – элемент управления не будет генерировать LVN_GETDISPINFO для получения текстовой информации
$LVIF_PARAM – член Param действителен
$LVIF_STATE – член State действителен
$LVIF_TEXT – член Text действителен.
Item Индекс элемента (с отсчётом от нуля), к которому относится эта структура.
SubItem Индекс подэлемента (с отсчётом от единицы), к которому относится эта структура.
State Состояние элемента, изображение состояния и наложенное изображение.
StateMask Значение, определяющее, какие биты члена state будут получены или изменены.
Text Указатель на строку с текстом элемента.
TextMax Количество байтов в буфере, на который указывает Text, включая завершающий нулевой символ.
Image Индекс значка элемента в списке изображений элемента управления.
Param Значение, специфичное для элемента.
Indent Количество ширин изображений для отступа элемента.
GroupID Идентификатор группы плиточного представления, которая получает элемент.
Columns Количество столбцов плиточного представления для отображения этого элемента.
pColumns Указатель на массив индексов столбцов.
piColFmt Указатель на массив следующих флагов (по отдельности или в комбинации), определяющих формат каждого подэлемента в расширенном плиточном представлении (Windows 7 и позже).
Значение Смысл:
$LVCFMT_LINE_BREAK – заставляет столбец переноситься на верхнюю часть следующего списка столбцов.
$LVCFMT_FILL – заполняет остаток площади плитки. Может иметь заголовок.
$LVCFMT_WRAP – позволяет столбцу переноситься в оставшееся место в его списке столбцов.
$LVCFMT_NO_TITLE – удаляет заголовок из подэлемента.
$LVCFMT_TILE_PLACEMENTMASK – эквивалентна комбинации $LVCFMT_LINE_BREAK и $LVCFMT_FILL.
iGroup Индекс группы элемента. Действителен только для данных владельца/обратного вызова (один элемент в нескольких группах) (Windows Vista и позже).
