_WinAPI_ShellRestricted

Определяет, действует ли указанная политика администратора.

#include <WinAPIShellEx.au3>
_WinAPI_ShellRestricted ( $iRestriction )

Параметры

$iRestriction Ограничение. Этот параметр может быть одной из констант $REST_*, определённых в APIShellExConstants.au3.

Возвращаемое значение

Успех: Возвращает True – указанное ограничение действует.
Ошибка: Возвращает False.
