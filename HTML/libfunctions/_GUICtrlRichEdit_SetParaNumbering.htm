Справочник функций

_GUICtrlRichEdit_SetParaNumbering

Устанавливает нумерацию абзаца(ев) в текущем выделении или, если выделения нет, абзаца(ев), вставленного в точку вставки.

#include <GuiRichEdit.au3>
_GUICtrlRichEdit_SetParaNumbering ( $hWnd, $sStyle [, $iTextToNbrSpace = Default [, $bForceRoman = False]] )

Параметры

$hWnd Дескриптор элемента управления.
$sStyle $sStyle – строка, задающая стиль и начальный "номер": например "." (маркер)
$iTextToNbrSpace [необязательный] Расстояние между номером/маркером и абзацем (в единицах пробела).
По умолчанию – количество завершающих пробелов, умноженное на размер в пунктах.
$bForceRoman [необязательный] False – i, v, x ... в $sStyle трактуются как буквы i, v, x ... (по умолчанию)

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@error: 101 – $hWnd не является дескриптором
103 – $iTextToNbrSpace не является положительным числом
104 – $bForceRoman должен быть True или False

Примечания

Чтобы установить "единицы пространства", вызовите _GUICtrlRichEdit_SetSpaceUnit().

См. также

_GUICtrlRichEdit_GetParaNumbering, _GUICtrlRichEdit_SetSpaceUnit

Дополнительно

Искать EM_SETPARAFORMAT в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <GuiRichEdit.au3>
#include <WindowsStylesConstants.au3>

Global $g_idLbl_Msg, $g_hRichEdit

Example()

Func Example()
    Local $hGui, $iMsg, $idBtn_Next, $iStep = 0
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName, StringLen(".exe")) & ")", 320, 350, -1, -1)
    $g_hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 300, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    $g_idLbl_Msg = GUICtrlCreateLabel("", 10, 235, 300, 60)
    $idBtn_Next = GUICtrlCreateButton("Next", 270, 310, 40, 30)
    GUISetState(@SW_SHOW)

    _GUICtrlRichEdit_SetText($g_hRichEdit, "First paragraph")
    _GUICtrlRichEdit_SetSel($g_hRichEdit, 0, 1)
    Report("Para with default indent settings")

    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                _GUICtrlRichEdit_Destroy($g_hRichEdit) ; нужно, если только скрипт не упадёт
                ; GUIDelete() ; так тоже работает
                Exit
            Case $iMsg = $idBtn_Next
                $iStep += 1
                Switch $iStep
                    Case 1
                        _GUICtrlRichEdit_AppendText($g_hRichEdit, @CRLF & "Second paragraph")
                        _GUICtrlRichEdit_SetParaNumbering($g_hRichEdit, ".")
                        Report("Added second paragraph")
                    Case 2
                        _GUICtrlRichEdit_SetSel($g_hRichEdit, 0, 2)
                        Report("Style of first paragraph in the selection")
                    Case 3
                        _GUICtrlRichEdit_SetSel($g_hRichEdit, 10, 26)
                        _GUICtrlRichEdit_SetParaNumbering($g_hRichEdit, "v) ", Default, True)
                        Report("Change settings of both paragraphs")
                    Case 4
                        _GUICtrlRichEdit_SetSel($g_hRichEdit, 10, 26)
                        _GUICtrlRichEdit_SetParaNumbering($g_hRichEdit, "2. ", Default, True)
                        Report("Changed settings for both paragraphs")
                    Case 5
                        ; Передать весь текст на Рабочий стол – можно посмотреть настройки в Word
                        _GUICtrlRichEdit_Deselect($g_hRichEdit)
                        _GUICtrlRichEdit_StreamToFile($g_hRichEdit, @DesktopDir & "\gcre.rtf")
                        GUICtrlSetState($idBtn_Next, $GUI_DISABLE)
                EndSwitch
        EndSelect
    WEnd
EndFunc   ;==>Example

Func Report($sMsg)
    $sMsg = $sMsg & @CRLF & @CRLF & "Get function returns " & @CRLF & _GUICtrlRichEdit_GetParaNumbering($g_hRichEdit)
    GUICtrlSetData($g_idLbl_Msg, $sMsg)
EndFunc   ;==>Report
