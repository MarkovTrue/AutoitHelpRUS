Справочник функций

_GUICtrlListView_GetNextItem

Ищет элемент с указанными свойствами.

#include <GuiListView.au3>
_GUICtrlListView_GetNextItem ( $hWnd [, $iStart = -1 [, $iSearch = 0 [, $iState = 8]]] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iStart [необязательный] Индекс элемента, с которого начинается поиск; -1 – найти первый элемент, удовлетворяющий заданным флагам.
Сам указанный элемент в поиске не участвует.
$iSearch [необязательный] Положение искомого элемента относительно начального:
0 – следующий элемент по индексу
1 – элемент выше указанного
2 – элемент ниже указанного
3 – элемент слева от указанного
4 – элемент справа от указанного
$iState [необязательный] Состояние искомого элемента. Может быть сочетанием значений:
1 – вырезан
2 – подсвечен
4 – в фокусе
8 – выбран

Возвращаемое значение

Успех: Возвращает индекс на основе 0.
Ошибка: Возвращает -1.

Пример

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $idListview

    GUICreate("ListView Get Next Item", 400, 300)
    $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Добавить столбцы
    _GUICtrlListView_AddColumn($idListview, "Items", 100)

    ; Добавить элементы
    _GUICtrlListView_AddItem($idListview, "Item 1")
    _GUICtrlListView_AddItem($idListview, "Item 2")
    _GUICtrlListView_AddItem($idListview, "Item 3")

    Выделяет
    _GUICtrlListView_SetItemSelected($idListview, 1)

    Найти
    MsgBox($MB_SYSTEMMODAL, "Information", "Selected Item: " & _GUICtrlListView_GetNextItem($idListview))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
