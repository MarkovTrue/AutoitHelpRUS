Справочник функций

_WinAPI_RemoveDirectory

Удаляет существующую пустую папку.

#include <WinAPIFiles.au3>
_WinAPI_RemoveDirectory ( $sDirPath )

Параметры

$sDirPath Путь пустой папки для удаления.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция _WinAPI_RemoveDirectory() отмечает папку для удаления при закрытии. Поэтому папка
не удаляется, пока не будет закрыт последний дескриптор папки.
