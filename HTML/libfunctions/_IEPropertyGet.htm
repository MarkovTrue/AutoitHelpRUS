Справочник функций

_IEPropertyGet

Возвращает выбранное свойство браузера или DOM-элемента.

#include <IE.au3>
_IEPropertyGet ( ByRef $oObject, $sProperty )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$sProperty Имя свойства (см. раздел Примечания).

Возвращаемое значение

Успех: Возвращает значение выбранного свойства.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – неверное значение
@extended: Содержит номер недопустимого параметра.

Примечания

В таблицах ниже приведено описание каждого доступного свойства.
Некоторые свойства относятся к DOM-элементам, другие – к браузеру.

Свойства браузера

Свойство Описание
"addressbar" Возвращает Boolean-значение, указывающее, видна ли адресная строка браузера.
"browserx" Возвращает x-координату левого верхнего угла элемента документа относительно браузера.
"browsery" Возвращает y-координату левого верхнего угла элемента документа относительно браузера.
"busy" Возвращает Boolean-значение, указывающее, занят ли объект операцией навигации или загрузки.
"contenteditable" Возвращает Boolean-значение, указывающее, можно ли редактировать объект мышью и клавиатурой.
"fullscreen" Возвращает Boolean-значение, указывающее, находится ли браузер в полноэкранном или обычном оконном режиме.
"height" Возвращает высоту главного окна браузера или элемента документа.
"hwnd" Возвращает дескриптор главного окна Internet Explorer. Может использоваться в большинстве функций AutoIt Win*.
"innerhtml" Возвращает отрендеренный HTML элемента без начального и конечного тегов элемента.
"innertext" Возвращает отрендеренный текст (без тегов) элемента. Обычно идентичен outertext.
"isdisabled" Возвращает значение, указывающее, может ли пользователь взаимодействовать с объектом.
"left" Возвращает экранную координату левого края главного окна объекта.
"locationname" Возвращает имя ресурса, который в данный момент отображает Internet Explorer.
"locationurl" Возвращает URL ресурса, который в данный момент отображает Internet Explorer.
"menubar" Возвращает Boolean-значение, указывающее, видна ли строка меню браузера.
"offline" Возвращает Boolean-значение, указывающее, работает ли браузер в данный момент в автономном режиме.
"outertext" Возвращает отрендеренный текст (без тегов) элемента. Обычно идентичен innertext.
"outerhtml" Возвращает отрендеренный HTML элемента, включая начальный и конечный теги элемента.
"readystate" Возвращает состояние готовности (ready state) объекта.
"referrer" Возвращает строку URL страницы, с которой был осуществлён переход на текущую (только если переход был по ссылке на той странице, иначе строка пустая).
"resizable" Возвращает значение, указывающее, можно ли изменять размер объекта.
"screenx" Возвращает x-координату левого верхнего угла объекта документа или браузера относительно экрана.
"screeny" Возвращает y-координату левого верхнего угла объекта документа или браузера относительно экрана.
"silent" Возвращает значение, указывающее, может ли браузер показывать диалоговые окна.
"statusbar" Возвращает значение, указывающее, видна ли строка состояния объекта.
"statustext" Возвращает текст строки состояния объекта.
"theatermode" Возвращает Boolean-значение, указывающее, находится ли браузер в режиме "театра" (Theater Mode). В режиме театра главное окно браузера занимает весь экран и отображает панель инструментов с минимальным набором навигационных кнопок.
"title" Возвращает заголовок документа. Заметьте, он отличается от заголовка окна, который обычно начинается с заголовка документа, но дополняется текстом из реестра Windows.
"toolbar" Возвращает Boolean-значение, указывающее, видна ли панель инструментов браузера.
"top" Возвращает экранную координату верхнего края главного окна объекта.
"visible" Возвращает значение, указывающее, виден ли объект.
"width" Возвращает ширину главного окна браузера или элемента документа.
"uniqueid" Возвращает ID, присвоенный DOM, уникальный среди всех элементов. Может использоваться в любом контексте, требующем ID элемента.

Свойства ClientInfo
Свойство Описание
"appcodename" Возвращает кодовое имя браузера (значение по умолчанию – Mozilla).
"appminorversion" Возвращает значение минорной версии приложения.

"javaenabled" Возвращает, включён ли Java.
"online" Возвращает значение, указывающее, находится ли система в глобальном автономном режиме.
"platform" Возвращает имя операционной системы пользователя (возвращаемые значения см. здесь).
"systemlanguage" Возвращает язык, используемый операционной системой по умолчанию (значение – один из кодов языков).
"useragent" Возвращает строку, эквивалентную HTTP-заголовку запроса user-agent.
"userlanguage" Возвращает настройку естественного языка операционной системы (значение – один из кодов языков).

Пример

Пример 1

; Открыть браузер с базовым примером, проверить,
; видна ли адресная строка; если да – скрыть, если нет – показать

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("basic")
If _IEPropertyGet($oIE, "addressbar") Then
    MsgBox($MB_SYSTEMMODAL, "AddressBar Status", "AddressBar Visible, turning it off")
    _IEPropertySet($oIE, "addressbar", False)
Else
    MsgBox($MB_SYSTEMMODAL, "AddressBar Status", "AddressBar Invisible, turning it on")
    _IEPropertySet($oIE, "addressbar", True)
EndIf

Пример 2

; Открытие браузера с примером формы и получение ссылки на форму
; элемент textarea. Получение координат и размеров области текста,
; обвести её мышью и остановиться в центре

#include <IE.au3>

Local $oIE = _IE_Example("form")

Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oTextArea = _IEFormElementGetObjByName($oForm, "textareaExample")

; Получить координаты и размеры textarea
Local $iScreenX = _IEPropertyGet($oTextArea, "screenx")
Local $iScreenY = _IEPropertyGet($oTextArea, "screeny")
Local $iWidth = _IEPropertyGet($oTextArea, "width")
Local $iHeight = _IEPropertyGet($oTextArea, "height")

; Обвести textarea мышью, остановиться в центре
Local $iMousespeed = 50
MouseMove($iScreenX, $iScreenY, $iMousespeed)
MouseMove($iScreenX + $iWidth, $iScreenY, $iMousespeed)
MouseMove($iScreenX + $iWidth, $iScreenY + $iHeight, $iMousespeed)
MouseMove($iScreenX, $iScreenY + $iHeight, $iMousespeed)
MouseMove($iScreenX, $iScreenY, $iMousespeed)
MouseMove($iScreenX + $iWidth / 2, $iScreenY + $iHeight / 2, $iMousespeed)
