Справочник функций

_WinAPI_SetFilePointerEx

Перемещает указатель файла указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_SetFilePointerEx ( $hFile, $iPos [, $iMethod = $FILE_BEGIN] )

Параметры

$hFile Дескриптор файла.
$iPos Количество байтов для перемещения указателя файла. Положительное значение перемещает указатель вперёд в
файле, отрицательное значение перемещает указатель файла назад.
$iMethod [необязательный] Начальная точка для перемещения указателя файла. Этот параметр может быть одним из следующих значений.
$FILE_BEGIN (по умолчанию)
$FILE_CURRENT
$FILE_END

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
