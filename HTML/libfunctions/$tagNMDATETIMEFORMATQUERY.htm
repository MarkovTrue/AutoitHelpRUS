Справочник ключевых слов

$tagNMDATETIMEFORMATQUERY

Содержит сведения о поле обратного вызова элемента управления.

Global Const $tagNMDATETIMEFORMATQUERY = $tagNMHDR & ";ptr Format;struct ;long SizeX;long SizeY; endstruct"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Format Указатель на строку с нулём в конце, определяющую поле обратного вызова элемента управления.
Строка состоит из одного или нескольких символов «X».
SizeX Должно быть заполнено максимальной шириной текста, который будет отображаться в поле обратного вызова.
SizeY Должно быть заполнено максимальной высотой текста, который будет отображаться в поле обратного вызова.

Примечания

Содержит строку (взятую из строки формата элемента управления), определяющую поле обратного вызова.
Структура получает максимально допустимый размер текста, который будет отображаться в поле обратного вызова.
Эта структура используется с сообщением уведомления $DTN_FORMATQUERY.
