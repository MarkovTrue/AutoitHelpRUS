Справочник функций

_WinAPI_SetWindowLong

Задаёт информацию об указанном окне.

#include <WinAPISysWin.au3>
_WinAPI_SetWindowLong ( $hWnd, $iIndex, $iValue )

Параметры

$hWnd Дескриптор окна.
$iIndex Задаёт смещение на основе 0 значения, которое будет установлено.
Допустимые значения находятся в диапазоне от нуля до количества байтов дополнительной памяти окна, минус четыре;
например, если вы задали 12 или более байтов дополнительной памяти, значение 8 будет индексом третьего 32-разрядного целого числа.
Чтобы получить любое другое значение, укажите одно из следующих значений:
$GWL_EXSTYLE – задаёт расширенные стили окна
$GWL_STYLE – задаёт стили окна
$GWL_WNDPROC – задаёт адрес процедуры окна
$GWL_HINSTANCE – задаёт дескриптор экземпляра приложения
$GWL_HWNDPARENT – задаёт дескриптор родительского окна (если есть)
$GWL_ID – задаёт идентификатор окна
$GWL_USERDATA – задаёт 32-разрядное значение, связанное с окном.
$iValue Задаёт заменяющее значение.

Возвращаемое значение

Успех: Возвращает предыдущее значение.
Ошибка: Возвращает 0.

Примечания

Для приведённых констант требуется #include <WinAPIConstants.au3>
