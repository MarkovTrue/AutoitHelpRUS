Справочник функций

_WinAPI_GetBufferedPaintTargetRect

Получает целевой прямоугольник.

#include <WinAPITheme.au3>
_WinAPI_GetBufferedPaintTargetRect ( $hBP )

Параметры

$hBP Дескриптор контекста буферизованного рисования.

Возвращаемое значение

Успех: Возвращает структуру $tagRECT, содержащую прямоугольник, указанный в функции _WinAPI_BeginBufferedPaint().
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

См. также

_WinAPI_BeginBufferedPaint
