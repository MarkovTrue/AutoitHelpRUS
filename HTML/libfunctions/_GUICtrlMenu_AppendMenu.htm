_GUICtrlMenu_AppendMenu

Добавляет новый пункт в конец строки меню, выпадающего меню, подменю или контекстного меню.

#include <GuiMenu.au3>
_GUICtrlMenu_AppendMenu ( $hMenu, $iFlags, $iNewItem, $vNewItem )

Параметры

$hMenu Дескриптор меню.
$iFlags Флаги, управляющие внешним видом и поведением нового пункта меню:
$MF_BITMAP – использовать растровое изображение
$MF_CHECKED – установить флажок рядом с пунктом
$MF_DISABLED – отключить пункт
$MF_ENABLED – включить пункт
$MF_GRAYED – отключить и выделить серым
$MF_MENUBARBREAK – разделить столбцы разделителем
$MF_MENUBREAK – разместить в новой строке/столбце
$MF_OWNERDRAW – пункт отрисовывает владелец
$MF_POPUP – задаёт выпадающее меню или подменю
$MF_SEPARATOR – добавить горизонтальный разделитель
$MF_STRING – текстовый пункт меню
$MF_UNCHECKED – убрать флажок рядом с пунктом.
$iNewItem Идентификатор нового пункта или, если в параметре $iFlags установлен флаг всплывающего окна, дескриптор выпадающего меню или подменю.
$vNewItem Содержимое нового пункта меню. Интерпретация зависит от значения параметра $iFlags.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Пример

#include <GUIConstantsEx.au3>
#include <GuiMenu.au3>

Example()

Func Example()
    Local $hGUI, $hFile, $hEdit, $hHelp, $hMain
    Local Enum $e_idNew = 1000, $e_idOpen, $e_idSave, $e_idExit, $e_idCut, $e_idCopy, $e_idPaste, $e_idAbout

    ; Создать GUI
    $hGUI = GUICreate("Menu", 400, 300)

    ; Создать меню Файл
    $hFile = _GUICtrlMenu_CreateMenu()
    _GUICtrlMenu_InsertMenuItem($hFile, 0, "&New", $e_idNew)
    _GUICtrlMenu_InsertMenuItem($hFile, 1, "&Open", $e_idOpen)
    _GUICtrlMenu_InsertMenuItem($hFile, 2, "&Save", $e_idSave)
    _GUICtrlMenu_InsertMenuItem($hFile, 3, "", 0)
    _GUICtrlMenu_InsertMenuItem($hFile, 4, "E&xit", $e_idExit)

    ; Создать меню Правка
    $hEdit = _GUICtrlMenu_CreateMenu()
    _GUICtrlMenu_InsertMenuItem($hEdit, 0, "&Cut", $e_idCut)
    _GUICtrlMenu_InsertMenuItem($hEdit, 1, "C&opy", $e_idCopy)
    _GUICtrlMenu_InsertMenuItem($hEdit, 2, "&Paste", $e_idPaste)

    ; Создать меню Справка
    $hHelp = _GUICtrlMenu_CreateMenu()

    ; Создать главное меню
    $hMain = _GUICtrlMenu_CreateMenu()
    _GUICtrlMenu_InsertMenuItem($hMain, 0, "&File", 0, $hFile)
    _GUICtrlMenu_InsertMenuItem($hMain, 1, "&Edit", 0, $hEdit)
    _GUICtrlMenu_InsertMenuItem($hMain, 2, "&Help", 0, $hHelp)

    ; Установить меню окна
    _GUICtrlMenu_SetMenu($hGUI, $hMain)
    GUISetState(@SW_SHOW)

    ; Добавить пункт "О программе"
    _GUICtrlMenu_AppendMenu($hHelp, $MF_STRING, $e_idAbout, "&About")

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
