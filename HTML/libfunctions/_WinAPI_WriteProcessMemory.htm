_WinAPI_WriteProcessMemory

Записывает память в указанный процесс.

#include <WinAPIMem.au3>
_WinAPI_WriteProcessMemory ( $hProcess, $pBaseAddress, $pBuffer, $iSize, ByRef $iWritten [, $sBufferType = "ptr"] )

Параметры

$hProcess Определяет открытый дескриптор процесса, память которого должна быть записана.
$pBaseAddress Указывает на базовый адрес в указанном процессе для записи.
$pBuffer Указывает на буфер, который содержит данные для записи в адресное пространство.
$iSize Указывает количество байтов для записи в указанный процесс.
$iWritten Фактическое количество байтов, переданных в указанный процесс.
$sBufferType [необязательный] Содержит тип данных, которые представляет $pBuffer

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
