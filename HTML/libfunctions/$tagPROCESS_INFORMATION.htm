Справочник ключевых слов

$tagPROCESS_INFORMATION

Содержит сведения о только что созданном процессе и его основном потоке.

Global Const $tagPROCESS_INFORMATION = "hwnd hProcess;hwnd hThread;int ProcessID;int ThreadID"

Параметры

hProcess Дескриптор вновь созданного процесса.
hThread Дескриптор основного потока вновь созданного процесса.
ProcessID Значение, которое может быть использовано для определения процесса.
ThreadID Значение, которое может быть использовано для определения потока.
