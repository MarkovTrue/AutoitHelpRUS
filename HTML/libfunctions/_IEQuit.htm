Справочник функций

_IEQuit

Закрывает браузер и удаляет объектную ссылку на него.

#include <IE.au3>
_IEQuit ( ByRef $oObject )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
@extended: Содержит номер недопустимого параметра.

Примечания

_IEQuit() особенно важна при создании невидимых экземпляров Internet Explorer или работе с ними.
Когда AutoIt завершается при работе с видимыми экземплярами браузера, браузер продолжает работать, и видимое окно остаётся пользователю.
Однако если экземпляр браузера невидим, после выхода AutoIt процесс браузера продолжает работать без пользовательского интерфейса, и в системе остаётся "осиротевший" процесс Iexplore.exe.
Поэтому хорошей практикой является вызов _IEQuit() всякий раз, когда процесс браузера больше не нужен, или непосредственно перед выходом из скрипта.

_IEQuit() нельзя использовать со встроенными объектами браузера, созданными через _IECreateEmbedded().
Процессы браузера, связанные с такими объектами, будут закрыты при уничтожении их родительского GUI-окна.

См. также

_IEAttach, _IECreate

Пример

; Создание невидимого окна браузера, переход к
; сайт, получить какие-то данные и выйти

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("http://sourceforge.net")
; Отобразить innerText элемента на странице с именем "sfmarquee"
Local $oMarquee = _IEGetObjByName($oIE, "sfmarquee")
If IsObj($oMarquee) Then
    MsgBox($MB_SYSTEMMODAL, "SourceForge Information", $oMarquee.innerText)
Else
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "SourceForge Information", "NO sfmarquee FOUND !!!")
EndIf
_IEQuit($oIE)
