Справочник функций

_WinAPI_ShellAddToRecentDocs

Добавляет файл в список недавно и часто используемых элементов.

#include <WinAPIShellEx.au3>
_WinAPI_ShellAddToRecentDocs ( $sFilePath )

Параметры

$sFilePath Имя файла, который будет добавлен. Установите этот параметр на пустую строку, чтобы очистить все данные об использовании всех элементов.

Возвращаемое значение

Отсутствует.

Дополнительно

Искать SHAddToRecentDocs в библиотеке MSDN

Пример

#include <WinAPIShellEx.au3>

_WinAPI_ShellAddToRecentDocs(@ScriptFullPath)
