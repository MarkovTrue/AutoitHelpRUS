_GDIPlus_GraphicsReleaseDC

Освобождает контекст устройства объекта Graphics.

#include <GDIPlus.au3>
_GDIPlus_GraphicsReleaseDC ( $hGraphics, $hDC )

Параметры

$hGraphics Дескриптор объекта Graphics.
$hDC Дескриптор контекста устройства Graphics.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).
