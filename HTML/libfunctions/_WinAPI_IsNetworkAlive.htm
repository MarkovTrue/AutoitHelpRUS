Справочник функций

_WinAPI_IsNetworkAlive

Определяет, подключена ли локальная система к сети, и определяет тип сетевого подключения.

#include <WinAPIDiag.au3>
_WinAPI_IsNetworkAlive ( )

Возвращаемое значение

Успех: Возвращает тип сетевого подключения ($NETWORK_ALIVE_*), если локальная система подключена к сети.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0, флаг @extended содержит код последней ошибки.

Примечания

Всегда проверяйте флаг @error перед проверкой возвращаемого значения этой функции. Если @error не равен 0,
функция не выполнена и следующие значения не применяются.

Эта функция доступна только для соединений TCP/IP.

Дополнительно

Искать IsNetworkAlive в библиотеке MSDN

Пример

#include <WinAPIDiag.au3>

ConsoleWrite('Internet connected: ' & (_WinAPI_IsNetworkAlive() <> 0) & @CRLF)
