Справочник функций

_GUIToolTip_Update

Заставляет текущий инструмент перерисовываться при изменении.

#include <GuiToolTip.au3>
_GUIToolTip_Update ( $hTool )

Параметры

$hTool Дескриптор элемента управления ToolTip (возвращается функцией _GUIToolTip_Create).

Возвращаемое значение

Отсутствует.

Примечания

Это перерисует отображаемую в данный момент подсказку; она не будет влиять на инструмент если он не отображается при вызове этой функции.
