Справочник ключевых слов

$tagNMREBARCHILDSIZE

Содержит сведения для обработки уведомляющего сообщения RBN_CHILDSIZE.

Global Const $tagNMREBARCHILDSIZE = $tagNMHDR & ";uint uBand;uint wID;" & _
"struct; long CLeft;long CTop;long CRight;long CBottom; endstruct;" & _
"struct; long BLeft;long BTop;long BRight;long BBottom; endstruct"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
uBand Индекс полосы, отправляющей уведомление (с отсчётом от нуля).
wID Идентификатор полосы, определённый приложением.
CLeft X-координата верхнего левого угла прямоугольника нового размера дочернего окна.
Этот член может быть изменён во время уведомления для изменения положения и размера дочернего окна.
CTop Y-координата верхнего левого угла прямоугольника нового размера дочернего окна.
Этот член может быть изменён во время уведомления для изменения положения и размера дочернего окна.
CRight X-координата нижнего правого угла прямоугольника нового размера дочернего окна.
Этот член может быть изменён во время уведомления для изменения положения и размера дочернего окна.
CBottom Y-координата нижнего правого угла прямоугольника нового размера дочернего окна.
Этот член может быть изменён во время уведомления для изменения положения и размера дочернего окна.
BLeft X-координата верхнего левого угла прямоугольника нового размера полосы.
BTop Y-координата верхнего левого угла прямоугольника нового размера полосы.
BRight X-координата нижнего правого угла прямоугольника нового размера полосы.
BBottom Y-координата нижнего правого угла прямоугольника нового размера полосы.
