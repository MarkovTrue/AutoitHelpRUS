_WinAPI_SetFileValidData

Устанавливает допустимую длину данных указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_SetFileValidData ( $hFile, $iLength )

Параметры

$hFile Дескриптор файла. Этот файл должен был открыт с правом доступа $GENERIC_WRITE и
привилегией $SE_MANAGE_VOLUME_NAME включена. Файл не может быть сетевым файлом, сжатым,
разреженным или транзакционным.
$iLength Новая допустимая длина данных. Этот параметр должен быть положительным значением, которое больше текущей
допустимой длины данных, но меньше текущего размера файла.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция устанавливает логический конец файла. Чтобы установить размер файла, используйте функцию
_WinAPI_SetEndOfFile(). Физический размер файла также называется концом файла.

Обычно функция _WinAPI_SetFileValidData() используется приложениями системного уровня на их собственных
частных данных. Не все файловые системы используют допустимую длину данных. Некоторые файловые системы могут отслеживать несколько допустимых диапазонов данных.
В целом, большинству приложений никогда не потребуется вызывать эту функцию.
