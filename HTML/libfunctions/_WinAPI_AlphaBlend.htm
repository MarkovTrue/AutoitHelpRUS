Справочник функций

_WinAPI_AlphaBlend

Отображает растровые изображения, которые имеют прозрачные или полупрозрачные пиксели.

#include <WinAPIGdi.au3>
_WinAPI_AlphaBlend ( $hDestDC, $iXDest, $iYDest, $iWidthDest, $iHeightDest, $hSrcDC, $iXSrc, $iYSrc, $iWidthSrc, $iHeightSrc, $iAlpha [, $bAlpha = False] )

Параметры

$hDestDC Дескриптор контекста устройства назначения.
$iXDest Координата X верхнего левого угла целевого прямоугольника в логических единицах.
$iYDest Координата Y верхнего левого угла целевого прямоугольника в логических единицах.
$iWidthDest Ширина целевого прямоугольника в логических единицах.
$iHeightDest Высота целевого прямоугольника в логических единицах.
$hSrcDC Дескриптор исходного контекста устройства.
$iXSrc Координата X верхнего левого угла исходного прямоугольника в логических единицах.
$iYSrc Координата Y верхнего левого угла исходного прямоугольника в логических единицах.
$iWidthSrc Ширина исходного прямоугольника в логических единицах.
$iHeightSrc Высота исходного прямоугольника в логических единицах.
$iAlpha Значение альфа-прозрачности, которое будет использоваться для всего исходного растрового изображения.
Это значение объединяется с любыми значениями альфа на пиксель в исходном растровом изображении.
Если вы установите $iAlpha на 0, предполагается, что ваше изображение прозрачно.
Установите значение $iAlpha на 255 (непрозрачный), когда вы хотите использовать только значения альфа на пиксель.
$bAlpha [необязательный] Определяет, использовать ли альфа-канал из исходного растрового изображения, допустимые значения:
True – использовать альфа-канал (то есть, альфа на пиксель).
Учтите, что API используют предварительно умноженную альфа, что означает, что значения каналов красного, зелёного и синего в растровом изображении должны быть предварительно умножены на значение альфа-канала.
Например, если значение альфа-канала равно x, каналы красного, зелёного и синего должны быть умножены на x и разделены на 255 до вызова.
False – не использовать альфа-канал (по умолчанию).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Если исходный прямоугольник и прямоугольник назначения имеют разные размеры, исходное растровое изображение растягивается в соответствии с прямоугольником назначения.
Если используется функция _WinAPI_SetStretchBltMode(), значение режима растяжения автоматически преобразуется в $COLORONCOLOR для этой функции (то есть, $BLACKONWHITE, $WHITEONBLACK и $HALFTONE преобразуются в $COLORONCOLOR).

Если растровые изображения назначения и источника не имеют одного и того же формата цвета, функция _WinAPI_AlphaBlend() преобразует исходное растровое изображение в соответствие с растровым изображением назначения.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <SliderConstants.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPIRes.au3>
#include <WinAPISysWin.au3>
#include <WindowsNotifsConstants.au3>

Opt('TrayAutoPause', 0)

; Загрузить изображение
Local $sBmp = _Extras_PathFull('AutoIt.bmp')
Global $g_hBitmap = _WinAPI_LoadImage(0, $sBmp, $IMAGE_BITMAP, 0, 0, $LR_LOADFROMFILE)
Local $tSIZE = _WinAPI_GetBitmapDimension($g_hBitmap)
Local $W = DllStructGetData($tSIZE, 'X')
Local $H = DllStructGetData($tSIZE, 'Y')

; Создать GUI
Global $g_hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), $W, $H + 26)
Global $g_idPic = GUICtrlCreatePic('', 0, 0, $W, $H)
GUICtrlCreateGraphic(0, $H, $W, 1)
GUICtrlSetBkColor(-1, 0xDFDFDF)
Global $g_idSlider = GUICtrlCreateSlider(0, $H + 1, $W, 25, BitOR($TBS_BOTH, $TBS_NOTICKS))
Global $g_hSlider = GUICtrlGetHandle(-1)
GUICtrlSetLimit(-1, 255, 0)
GUICtrlSetData(-1, 255)

; Устанавливает растровое изображение для управления с альфой
_SetBitmapAlpha($g_idPic, $g_hBitmap, 255)

; Зарегистрировать WM_HSCROLL для живой прокрутки и показать GUI
GUIRegisterMsg($WM_HSCROLL, 'WM_HSCROLL')
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE

Func _SetBitmapAlpha($hWnd, $hBitmap, $iAlpha)
    If Not IsHWnd($hWnd) Then
        $hWnd = GUICtrlGetHandle($hWnd)
        If Not $hWnd Then
            Return 0
        EndIf
    EndIf

    Local $aW[2], $aH[2]
    Local $tRECT = _WinAPI_GetClientRect($hWnd)
    $aW[0] = DllStructGetData($tRECT, 3) - DllStructGetData($tRECT, 1)
    $aH[0] = DllStructGetData($tRECT, 4) - DllStructGetData($tRECT, 2)
    Local $tSIZE = _WinAPI_GetBitmapDimension($hBitmap)
    $aW[1] = DllStructGetData($tSIZE, 1)
    $aH[1] = DllStructGetData($tSIZE, 2)
    Local $hDC = _WinAPI_GetDC($hWnd)
    Local $hDestDC = _WinAPI_CreateCompatibleDC($hDC)
    Local $hBmp = _WinAPI_CreateCompatibleBitmapEx($hDC, $aW[0], $aH[0], 0xFFFFFF)
    Local $hDestSv = _WinAPI_SelectObject($hDestDC, $hBmp)
    Local $hSrcDC = _WinAPI_CreateCompatibleDC($hDC)
    Local $hSrcSv = _WinAPI_SelectObject($hSrcDC, $hBitmap)
    _WinAPI_AlphaBlend($hDestDC, 0, 0, $aW[0], $aH[0], $hSrcDC, 0, 0, $aW[1], $aH[1], $iAlpha, 0)
    _WinAPI_SelectObject($hDestDC, $hDestSv)
    _WinAPI_DeleteDC($hDestDC)
    _WinAPI_SelectObject($hSrcDC, $hSrcSv)
    _WinAPI_DeleteDC($hSrcDC)
    _WinAPI_ReleaseDC($hWnd, $hDC)
    Local $hObj = _SendMessage($hWnd, $STM_SETIMAGE, $IMAGE_BITMAP, $hBmp)
    If $hObj Then
        _WinAPI_DeleteObject($hObj)
    EndIf
    $hObj = _SendMessage($hWnd, $STM_GETIMAGE)
    If $hObj <> $hBmp Then
        _WinAPI_DeleteObject($hBmp)
    EndIf
    Return 1
EndFunc   ;==>_SetBitmapAlpha

Func WM_HSCROLL($hWnd, $iMsg, $wParam, $lParam)
    #forceref $iMsg, $wParam

    Switch $hWnd
        Case $g_hForm
            Switch $lParam
                Case $g_hSlider
                    _SetBitmapAlpha($g_idPic, $g_hBitmap, GUICtrlRead($g_idSlider))
            EndSwitch
    EndSwitch
    Return $GUI_RUNDEFMSG
EndFunc   ;==>WM_HSCROLL
