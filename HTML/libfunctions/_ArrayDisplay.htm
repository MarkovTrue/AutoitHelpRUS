Справочник функций

_ArrayDisplay

Отображает одномерный 1D или двумерный 2D массив в виде ListView.

#include <Array.au3>
_ArrayDisplay ( Const ByRef $aArray [, $sTitle = "ArrayDisplay" [, $sArrayRange = "" [, $iFlags = 0 [, $vUser_Separator = Default [, $sHeader = Default [, $iDesired_Colwidth = Default]]]]]] )

Параметры

$aArray Отображаемый массив.
$sTitle [необязательный] Заголовок диалога. По умолчанию "ArrayDisplay".
$sArrayRange [необязательный] Диапазон отображаемых строк/столбцов. По умолчанию ("") – весь массив (подробности ниже).
$iFlags [необязательный] Параметры UDF. Складывайте нужные значения:
$ARRAYDISPLAY_COLALIGNLEFT (0) = (по умолчанию) выравнивание текста в столбцах по левому краю
$ARRAYDISPLAY_TRANSPOSE (1) = транспонировать массив (только для 2D массива)
$ARRAYDISPLAY_COLALIGNRIGHT (2) = выравнивание по правому краю
$ARRAYDISPLAY_COLALIGNCENTER (4) = выравнивание по центру
$ARRAYDISPLAY_VERBOSE (8) = подробный режим – показывать MsgBox при ошибке и заставку при обработке больших массивов
16 = устарело
32 = устарело
$ARRAYDISPLAY_NOROW (64) = не отображать столбец "Row"
$ARRAYDISPLAY_CHECKERROR (128) = возвращать, если @error вызывающего <> 0.
$vUser_Separator [необязательный] Устарело. Всегда используйте Opt("GUIDataSeparatorChar").
$sHeader [необязательный] Имена столбцов заголовка – строка имён, разделённых текущим символом-разделителем (обычно "|"). Значение по умолчанию – см. раздел Примечания.
$iDesired_Colwidth [необязательный] Положительное значение – максимальная ширина столбца ListView для отображения содержимого. По умолчанию 350 пикселей.
Отрицательное значение – все столбцы, кроме первого (55), устанавливаются равными Abs($iDesired_Colwidth).
Учтите: ширина столбца всегда адаптируется под длину отображаемого элемента.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает флаг @error следующим образом:.
@error: 1 – $aArray не массив
2 – у $aArray слишком много измерений (поддерживаются только 1D и 2D)
3 – @error был установлен в момент вызова функции

Примечания

Если функции передана не массив или массив с числом измерений больше 2, функция возвращает ошибку, а скрипт продолжает работу. Если в $iFlags задан флаг "verbose", появляется MsgBox с предложением немедленно завершить скрипт или продолжить с обычным возвратом ошибки.

Ограничений на размер отображаемого массива нет, но из-за ограничений Windows-контрола заголовки и столбцы ListView не выравниваются, если столбцов больше ~600.

Синтаксис параметра $sArrayRange:

"7" Строки 0–7, все столбцы
"7:" Строки 7 – конец, все столбцы
"|7" Все строки, столбцы 0–7
"|7:" Все строки, столбцы 7 – конец
"7|7" Строки 0–7, столбцы 0–7
"5:7" Строки 5–7, все столбцы
"|5:7" Все строки, столбцы 5–7
"7|5:7" Строки 0–7, столбцы 5–7
"5:7|7" Строки 5–7, столбцы 0–7
"5:7|5:7" Строки 5–7, столбцы 5–7

Для 1D массивов значения столбцов игнорируются.

Имена $sHeader (разделённые текущим символом-разделителем) используются для соответствующих столбцов. Если пользовательские имена не указаны или указаны не для всех столбцов, применяется заголовок по умолчанию: "Row|Col0" для 1D массива или "Row|Col0|Col1|...|Col n" для 2D. При транспонировании массива заголовок игнорируется.
Переменная $ARRAYDISPLAY_ROWPREFIX задаёт префикс нумерации строк (по умолчанию: #).
Чтобы принудительно использовать числовое сравнение для конкретного столбца, завершите соответствующее имя символом $ARRAYDISPLAY_NUMERICSORT (по умолчанию: *).

Размеры массива выводятся под ним. Текст красный, если массив транспонирован или отображается только диапазон – во всплывающей подсказке указываются конкретные случаи.

Если в $iFlags задан флаг "verbose", при начальной обработке массива с более чем 10000 элементов показывается заставка.

Элемент массива, содержащий другой массив или Map, отображается как {Array} или {Map}.

При использовании $ARRAYDISPLAY_TRANSPOSE или $sArrayRange создаётся копия $aArray – это увеличивает потребление памяти.

Щелчок по заголовку столбца сортирует его. При большом числе строк сортировка может занять несколько секунд – об этом предупреждает всплывающая подсказка.

Пример

#include <Array.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()

    ; Создать одномерный массив для отображения
    Local $aArray_1D[5] = ["Item 0", "Item 1", "A longer Item 2 to show column expansion", "Item 3", "Item 4"]

    _ArrayDisplay($aArray_1D, "1D display")

    ; Создать двумерный массив для отображения
    Local $aArray_2D[20][15]
    For $i = 0 To UBound($aArray_2D) - 1
        For $j = 0 To UBound($aArray_2D, 2) - 1
            $aArray_2D[$i][$j] = "Item " & StringFormat("%02i", $i) & StringFormat("%02i", $j)
        Next
    Next

    _ArrayDisplay($aArray_2D, "2D display")
    _ArrayDisplay($aArray_2D, "2D display transposed", Default, 1)

    ReDim $aArray_2D[20][10]
    $aArray_2D[5][5] = "A longer item to show column expansion"
    _ArrayDisplay($aArray_2D, "Expanded column - custom titles - no row/column", Default, 64, Default, "AA|BB|CC|DD|EE|FF|GG|HH|II|JJ")

    $aArray_2D[5][5] = "Column alignment set to right"
    _ArrayDisplay($aArray_2D, "Range set - right align", "3:7|4:9", 2, Default, "AA|BB|CC|DD|EE|FF")

    $aArray_2D[5][5] = "Column alignment set to left"
    Opt("GUIDataSeparatorChar", "!")
    _ArrayDisplay($aArray_2D, "! Header separator", "3:7|4:9", Default, Default, "AA!BB!CC!DD!EE!FF")

    ; Создать не-массивную переменную, чтобы вызвать ошибку – MsgBox показывается, так как $iFlags задан
    Local $vVar = 0, $iRet, $iError
    $iRet = _ArrayDisplay($vVar, "No MsgBox on Error")
    $iError = @error
    MsgBox(0, "_ArrayDisplay() Error", "return without internal Msgbox $iret =" & $iRet & " @error=" & $iError)

    $iRet = _ArrayDisplay($vVar, "MsgBox on Error", Default, 8)
    $iError = @error
    MsgBox(0, "_ArrayDisplay() Error", "return internal Msgbox with no force Exit $iret =" & $iRet & " @error=" & $iError)

EndFunc   ;==>Example
