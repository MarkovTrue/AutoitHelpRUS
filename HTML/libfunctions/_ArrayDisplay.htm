Справочник функций

_ArrayDisplay

Отображает одномерный или двумерный массив в виде ListView.

#include <Array.au3>
_ArrayDisplay ( Const ByRef $aArray [, $sTitle = "ArrayDisplay" [, $sArrayRange = "" [, $iFlags = 0 [, $vUser_Separator = Default [, $sHeader = Default [, $iDesired_Colwidth = Default]]]]]] )

Параметры

$aArray Отображаемый массив.
$sTitle [необязательный] Заголовок диалога. По умолчанию "ArrayDisplay".
$sArrayRange [необязательный] Диапазон отображаемых строк или столбцов. По умолчанию ("") отображается весь массив (подробности ниже).
$iFlags [необязательный] Параметры UDF. Складывайте нужные значения:
$ARRAYDISPLAY_COLALIGNLEFT (0) = (по умолчанию) выравнивание текста столбцов по левому краю
$ARRAYDISPLAY_TRANSPOSE (1) = транспонировать массив (только для двумерного массива)
$ARRAYDISPLAY_COLALIGNRIGHT (2) = выравнивание текста столбцов по правому краю
$ARRAYDISPLAY_COLALIGNCENTER (4) = выравнивание текста столбцов по центру
$ARRAYDISPLAY_VERBOSE (8) = подробный режим – показывать MsgBox при ошибке и заставку при обработке больших массивов
16 = устарело
32 = устарело
$ARRAYDISPLAY_NOROW (64) = столбец "Row" не отображается
$ARRAYDISPLAY_CHECKERROR (128) = вернуть управление, если @error вызывающего ≠ 0.
$vUser_Separator [необязательный] Устарело. Всегда используйте Opt("GUIDataSeparatorChar").
$sHeader [необязательный] Имена столбцов в заголовке (строка имён, разделённых текущим символом-разделителем, обычно "|"). По умолчанию см. раздел Примечания.
$iDesired_Colwidth [необязательный] При положительном значении максимальная ширина, до которой столбец ListView расширяется для показа содержимого. По умолчанию 350 пикселей.
При отрицательном значении все столбцы, кроме первого (он равен 55), устанавливаются в Abs($iDesired_Colwidth).
Помните, ширина столбца всегда подстраивается под длину отображаемого элемента.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0:
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным или двумерным массивом
3 – @error был установлен на момент вызова функции

Примечания

Если функции передана переменная, не являющаяся массивом, или массив с числом измерений больше 2, функция возвращает ошибку, и скрипт продолжает работу. Если в $iFlags установлен флаг "verbose", появляется MsgBox с возможностью немедленно завершить скрипт или продолжить с обычным возвратом ошибки.

На размер отображаемого массива ограничений нет, однако из-за ограничения Windows-контрола заголовки и столбцы ListView перестают выравниваться, если их больше примерно 600.

Синтаксис параметра $sArrayRange:

"7" Строки с 0 по 7, все столбцы
"7:" Строки с 7 до конца, все столбцы
"|7" Все строки, столбцы с 0 по 7
"|7:" Все строки, столбцы с 7 до конца
"7|7" Строки с 0 по 7, столбцы с 0 по 7
"5:7" Строки с 5 по 7, все столбцы
"|5:7" Все строки, столбцы с 5 по 7
"7|5:7" Строки с 0 по 7, столбцы с 5 по 7
"5:7|7" Строки с 5 по 7, столбцы с 0 по 7
"5:7|5:7" Строки с 5 по 7, столбцы с 5 по 7

Для одномерных массивов значения столбцов игнорируются.

Имена из $sHeader (разделённые текущим символом-разделителем) применяются к стольким столбцам, сколько задано имён. Если пользовательские имена не заданы или их не хватает, подставляется заголовок по умолчанию "Row|Col0" для одномерного массива или "Row|Col0|Col1|...|Col n" для двумерного. При отображении массива транспонированным заголовок игнорируется.
Переменная $ARRAYDISPLAY_ROWPREFIX задаёт префикс нумерации строк (по умолчанию: #).
Чтобы для конкретного столбца использовать числовое сравнение, завершите соответствующее имя символом $ARRAYDISPLAY_NUMERICSORT (по умолчанию: *).

Размеры массива показываются под ним. Текст красный, если массив транспонирован или показан только диапазон элементов – конкретные случаи указываются во всплывающей подсказке.

Если в $iFlags установлен флаг "verbose", при начальной обработке массива с более чем 10000 элементами показывается заставка.

Элемент массива, содержащий другой массив или Map, отображается как {Array} или {Map}.

При использовании $ARRAYDISPLAY_TRANSPOSE или $sArrayRange создаётся копия $aArray, поэтому расходуется дополнительная память.

Щёлкните по заголовку столбца, чтобы отсортировать его. При большом числе строк это может занять несколько секунд. Во время загрузки появляется всплывающая подсказка с предупреждением, что сортировка может занять несколько секунд.

Пример

#include <Array.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()

    ; Создать одномерный массив для отображения
    Local $aArray_1D[5] = ["Item 0", "Item 1", "A longer Item 2 to show column expansion", "Item 3", "Item 4"]

    _ArrayDisplay($aArray_1D, "1D display")

    ; Создать двумерный массив для отображения
    Local $aArray_2D[20][15]
    For $i = 0 To UBound($aArray_2D) - 1
        For $j = 0 To UBound($aArray_2D, 2) - 1
            $aArray_2D[$i][$j] = "Item " & StringFormat("%02i", $i) & StringFormat("%02i", $j)
        Next
    Next

    _ArrayDisplay($aArray_2D, "2D display")
    _ArrayDisplay($aArray_2D, "2D display transposed", Default, 1)

    ReDim $aArray_2D[20][10]
    $aArray_2D[5][5] = "A longer item to show column expansion"
    _ArrayDisplay($aArray_2D, "Expanded column - custom titles - no row/column", Default, 64, Default, "AA|BB|CC|DD|EE|FF|GG|HH|II|JJ")

    $aArray_2D[5][5] = "Column alignment set to right"
    _ArrayDisplay($aArray_2D, "Range set - right align", "3:7|4:9", 2, Default, "AA|BB|CC|DD|EE|FF")

    $aArray_2D[5][5] = "Column alignment set to left"
    Opt("GUIDataSeparatorChar", "!")
    _ArrayDisplay($aArray_2D, "! Header separator", "3:7|4:9", Default, Default, "AA!BB!CC!DD!EE!FF")

    ; Создать переменную, не являющуюся массивом, чтобы вызвать ошибку – MsgBox показывается, так как задан $iFlags
    Local $vVar = 0, $iRet, $iError
    $iRet = _ArrayDisplay($vVar, "No MsgBox on Error")
    $iError = @error
    MsgBox(0, "_ArrayDisplay() Error", "return without internal Msgbox $iret =" & $iRet & " @error=" & $iError)

    $iRet = _ArrayDisplay($vVar, "MsgBox on Error", Default, 8)
    $iError = @error
    MsgBox(0, "_ArrayDisplay() Error", "return internal Msgbox with no force Exit $iret =" & $iRet & " @error=" & $iError)

EndFunc   ;==>Example
