Справочник функций

_WinAPI_GetClassInfoEx

Получает информацию о классе окна.

#include <WinAPISysWin.au3>
_WinAPI_GetClassInfoEx ( $sClass [, $hInstance = 0] )

Параметры

$sClass Имя класса. Имя должно быть именем предварительно зарегистрированного класса или класса, зарегистрированного предыдущим вызовом функции _WinAPI_RegisterClass() или _WinAPI_RegisterClassEx().
В качестве альтернативы этот параметр может быть атомом класса.
Атом должен находиться в младшем слове $sClass; старший слово должен быть нулевым.
$hInstance [необязательный] Дескриптор экземпляра приложения, создавшего класс.
Для получения информации о классах, определённых системой (например, кнопки или списки), установите этот параметр в 0 (по умолчанию).

Возвращаемое значение

Успех: Возвращает структуру $tagWNDCLASSEX, содержащую информацию о классе.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

См. также

_WinAPI_RegisterClass, _WinAPI_RegisterClassEx
