Справочник функций

_Security__IsValidSid

Проверяет корректность SID.

#include <Security.au3>
_Security__IsValidSid ( $pSID )

Параметры

$pSID Указатель на SID.

Возвращаемое значение

True: Возвращает SID является действительным.
False: SID не является действительным.

См. также

_Security__GetLengthSid

Дополнительно

Искать IsValidSid в библиотеке MSDN

Пример

#include <Security.au3>

Local $tSID = _Security__GetAccountSid(@ComputerName)
If Not @error Then
    ; $tSID – структура с SID-данными указанной учётной записи
    ; Проверить этот SID на корректность:
    Local $bValid = _Security__IsValidSid($tSID)
    If $bValid Then
        ConsoleWrite("The SID is valid." & @CRLF)
        ; ... Остальная часть скрипта здесь...
    Else
        ConsoleWrite("The SID is NOT valid." & @CRLF)
        Exit
    EndIf
EndIf
