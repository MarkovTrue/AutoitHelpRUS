_GDIPlus_LineBrushSetColors

Задаёт начальный и конечный цвета кисти линейного градиента.

#include <GDIPlus.au3>
_GDIPlus_LineBrushSetColors ( $hLineGradientBrush, $iARGBStart, $iARGBEnd )

Параметры

$hLineGradientBrush Указатель на объект LinearGradientBrush.
$iARGBStart Компоненты Alpha, Red, Green, Blue начального цвета.
$iARGBEnd Компоненты Alpha, Red, Green, Blue конечного цвета.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_LineBrushGetColors

Дополнительно

Искать GdipSetLineColors в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;инициализировать GDI+
    Local Const $iWidth = 600, $iHeight = 600, $iBgColor = 0x303030 ;$iBGColor в формате RRGGBB

    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;создать тестовый GUI
    GUISetBkColor($iBgColor, $hGUI) ;установить цвет фона GUI
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать графический объект из дескриптора окна
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;задаёт качество отрисовки графики (сглаживание)
    Local $hBrush = _GDIPlus_LineBrushCreate(0, 0, 280, 570, 0xFFFFFF00, 0xFF4020FF, 1) ;создать линейную градиентную кисть с отражением

    Local $aBlends[5][2]
    $aBlends[0][0] = 4 ;использовать 4 коэффициента и позиции
    $aBlends[1][0] = 0 ; Фактор
    $aBlends[1][1] = 0 ; Позиция
    $aBlends[2][0] = 1 ;смешивание 100% (переход от красного к зелёному завершается в первых 20%)
    $aBlends[2][1] = 0.2 ;расстояние от левой границы кисти – 20%
    $aBlends[3][0] = 0 ;смешивание 0% (переход от зелёного к красному с 20% до 70%)
    $aBlends[3][1] = 0.7 ;расстояние от левой границы кисти – 70%
    $aBlends[4][0] = 1 ;смешивание 100% (переход с 70% до 100%)
    $aBlends[4][1] = 1 ;расстояние от левой границы кисти – 100%

    _GDIPlus_LineBrushSetBlend($hBrush, $aBlends) ;задать позиции и коэффициенты кисти

    _GDIPlus_GraphicsFillEllipse($hGraphics, 100, 50, 380, 500, $hBrush) ;нарисовать яйцо

    Sleep(2000)

    _GDIPlus_GraphicsClear($hGraphics, 0xFF000000 + $iBgColor) ;очистить битмап
    _GDIPlus_LineBrushSetColors($hBrush, 0xFFFFFFFF, 0xFF000000) ;перевести цвета в оттенки серого

    _GDIPlus_GraphicsFillEllipse($hGraphics, 100, 50, 380, 500, $hBrush) ;нарисовать яйцо снова

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;освободить ресурсы GDI+
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
