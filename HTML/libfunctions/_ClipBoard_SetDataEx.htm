Справочник функций

_ClipBoard_SetDataEx

Помещает данные в буфер обмена в указанном формате.

#include <Clipboard.au3>
_ClipBoard_SetDataEx ( ByRef $hMemory [, $iFormat = $CF_TEXT] )

Параметры

$hMemory Дескриптор данных в указанном формате. Может быть NULL, и тогда окно предоставляет данные в этом формате по запросу.
Если окно откладывает предоставление данных, оно должно обрабатывать сообщения $WM_RENDERFORMAT и $WM_RENDERALLFORMATS.
При успешном выполнении функции объект, заданный параметром $hMemory, переходит во владение системы.
После передачи владения системе приложению нельзя записывать или освобождать данные, но можно блокировать и читать их до вызова _ClipBoard_Close().
Память нужно разблокировать до закрытия буфера обмена.
Если $hMemory указывает на объект памяти, он должен быть выделен с флагом $GMEM_MOVEABLE.
$iFormat [необязательный] Формат буфера обмена:
$CF_TEXT – текстовый формат (по умолчанию)
$CF_BITMAP – дескриптор растрового изображения (HBITMAP)
$CF_METAFILEPICT – дескриптор рисунка в виде метафайла (METAFILEPICT)
$CF_SYLK – формат Microsoft Symbolic Link (SYLK)
$CF_DIF – формат обмена данными Software Arts' Data Interchange Format
$CF_TIFF – формат TIFF
$CF_OEMTEXT – текст в кодировке OEM
$CF_DIB – структура BITMAPINFO с битами растра
$CF_PALETTE – дескриптор цветовой палитры
$CF_PENDATA – данные для расширений пера (Pen Computing)
$CF_RIFF – аудиоданные в формате RIFF
$CF_WAVE – аудиоданные в формате WAVE
$CF_UNICODETEXT – текст в формате Unicode
$CF_ENHMETAFILE – дескриптор расширенного метафайла (HENHMETAFILE)
$CF_HDROP – дескриптор HDROP со списком файлов
$CF_LOCALE – дескриптор идентификатора языкового стандарта текста в буфере обмена
$CF_DIBV5 – структура BITMAPV5HEADER с цветами и битами растра
$CF_OWNERDISPLAY – формат отображения владельца
$CF_DSPTEXT – текстовый формат отображения для частного формата
$CF_DSPBITMAP – растровый формат отображения для частного формата
$CF_DSPMETAFILEPICT – формат отображения рисунка метафайла для частного формата
$CF_DSPENHMETAFILE – формат отображения расширенного метафайла для частного формата.

Возвращаемое значение

Успех: Возвращает дескриптор данных.
Ошибка: Возвращает 0.

Примечания

Параметр $iFormat может задавать зарегистрированный или стандартный формат буфера обмена.
Если приложение вызывает эту функцию в ответ на $WM_RENDERFORMAT или $WM_RENDERALLFORMATS, то после её вызова дескриптор использовать нельзя.
Если приложение вызывает _ClipBoard_Open() с дескриптором NULL, то _ClipBoard_Empty() устанавливает владельца буфера обмена в NULL; из-за этого функция завершается с ошибкой.

См. также

_ClipBoard_Empty, _ClipBoard_GetData, _ClipBoard_Open, _ClipBoard_SetData

Пример

Пример 1

; == Example 1 use _ClipBoard_SetData()

#include "Extras\HelpFileInternals.au3"

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIError.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Clipboard Get/Set DataEx (v" & @AutoItVersion & ")", 600, 450)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    Local $idBtn_SetData = GUICtrlCreateButton("Set ClipBoard Data", 150, 410, 120, 30)
    Local $idBtn_GetData = GUICtrlCreateButton("Get ClipBoard Data", 300, 410, 120, 30)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    Local $hMemory, $tData
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtn_SetData
                _ClipBoard_SetData("ClipBoard Library")
            Case $idBtn_GetData
                ; Открыть буфер обмена
                If _ClipBoard_Open($hGUI) Then

                    ; Прочитать текст из буфера обмена
                    $hMemory = _ClipBoard_GetDataEx($CF_TEXT)
                    If $hMemory = 0 Then _WinAPI_ShowError("_ClipBoard_GetDataEx failed")
                    $tData = DllStructCreate("char Text[8192]", $hMemory)
                    _MemoWrite(DllStructGetData($tData, "Text"))

                    ; Закрыть буфер обмена
                    _ClipBoard_Close()
                Else
                    _WinAPI_ShowError("_ClipBoard_Open failed")
                EndIf
        EndSwitch
    WEnd
EndFunc   ;==>Example

Пример 2

; == Example 2 use _ClipBoard_GetData()

#include "Extras\HelpFileInternals.au3"

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <Memory.au3>
#include <WinAPIError.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("Clipboard Get/Set DataEx (v" & @AutoItVersion & ")", 600, 450)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    Local $idBtn_SetData = GUICtrlCreateButton("Set ClipBoard Data", 150, 410, 120, 30)
    Local $idBtn_GetData = GUICtrlCreateButton("Get ClipBoard Data", 300, 410, 120, 30)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    Local $hMemory, $hLock, $tData, $sData, $iSize
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtn_SetData
                ; Открыть буфер обмена
                If _ClipBoard_Open(0) Then

                    ; Очистить буфер обмена
                    If _ClipBoard_Empty() Then

                        ; Создать глобальный буфер памяти (показывает, почему _ClipBoard_SetData НАМНОГО проще!)
                        $sData = "Hello from AutoIt"
                        $iSize = StringLen($sData) + 1
                        $hMemory = _MemGlobalAlloc($iSize, $GHND)
                        If $hMemory <> 0 Then
                            $hLock = _MemGlobalLock($hMemory)
                            If $hLock = 0 Then _WinAPI_ShowError("_Mem_GlobalLock failed")
                            $tData = DllStructCreate("char Text[" & $iSize & "]", $hLock)
                            DllStructSetData($tData, "Text", $sData)
                            _MemGlobalUnlock($hMemory)

                            ; Записать текст в буфер обмена
                            If Not _ClipBoard_SetDataEx($hMemory, $CF_TEXT) Then _WinAPI_ShowError("_ClipBoard_SetDataEx failed")
                        Else
                            _WinAPI_ShowError("_Mem_GlobalAlloc failed")
                        EndIf

                        ; Закрыть буфер обмена
                        _ClipBoard_Close()
                    Else
                        ; Закрыть буфер обмена
                        _ClipBoard_Close()
                        _WinAPI_ShowError("_ClipBoard_Empty failed")
                    EndIf
                Else
                    _WinAPI_ShowError("_ClipBoard_Open failed")
                EndIf

            Case $idBtn_GetData
                _MemoWrite(_ClipBoard_GetData())
        EndSwitch
    WEnd
EndFunc   ;==>Example
