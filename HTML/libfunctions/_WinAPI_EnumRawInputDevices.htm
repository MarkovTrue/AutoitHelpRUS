_WinAPI_EnumRawInputDevices

Перечисляет устройства raw input, подключённые к системе.

#include <WinAPISys.au3>
_WinAPI_EnumRawInputDevices ( )

Возвращаемое значение

Успех: Возвращает двумерный массив дескрипторов и типов устройств.
$aArray[0][0] – количество строк массива (n)
$aArray[0][1] – не используется
[n][0] – дескриптор устройства raw input.
[n][1] – тип устройства ($RIM_*).
Ошибка: Устанавливает @error ≠ 0.

Дополнительно

Искать GetRawInputDeviceList в библиотеке MSDN

Пример

#include <Array.au3>
#include <WinAPISys.au3>

Local $tText, $aData = _WinAPI_EnumRawInputDevices()

If IsArray($aData) Then
    ReDim $aData[$aData[0][0] + 1][3]
    $tText = DllStructCreate('wchar[256]')
    For $i = 1 To $aData[0][0]
        If _WinAPI_GetRawInputDeviceInfo($aData[$i][0], $tText, 256, $RIDI_DEVICENAME) Then
            $aData[$i][2] = DllStructGetData($tText, 1)
        Else
            $aData[$i][2] = ''
        EndIf
    Next

EndIf

_ArrayDisplay($aData, '_WinAPI_EnumRawInputDevices')
