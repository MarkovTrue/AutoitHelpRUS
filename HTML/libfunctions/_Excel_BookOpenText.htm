Справочник функций

_Excel_BookOpenText

Открывает текстовый файл и разбирает его содержимое в новую книгу с одним листом.

#include <Excel.au3>
_Excel_BookOpenText ( $oExcel, $sFilePath [, $iStartRow = 1 [, $iDataType = Default [, $sTextQualifier = $xlTextQualifierDoubleQuote [, $bConsecutiveDelimiter = False [, $sDelimiter = "," [, $aFieldInfo = "" [, $sDecimalSeparator = Default [, $sThousandsSeparator = Default [, $bTrailingMinusNumbers = True [, $iOrigin = Default]]]]]]]]]] )

Параметры

$oExcel Объект приложения Excel.
$sFilePath Путь и имя открываемого файла.
$iStartRow [необязательный] Строка, с которой начинается разбор файла (по умолчанию 1).
$iDataType [необязательный] Задаёт формат столбцов данных в файле. Может быть любым значением из перечисления XlTextParsingType.
При значении Default Excel пытается определить формат столбцов (по умолчанию Default).
$sTextQualifier [необязательный] Задаёт ограничитель текста (по умолчанию $xlTextQualifierDoubleQuote).
$bConsecutiveDelimiter [необязательный] True – несколько последовательных разделителей считаются одним разделителем (по умолчанию False).
$sDelimiter [необязательный] Один или несколько символов, используемых в качестве разделителя (по умолчанию ",").
$aFieldInfo [необязательный] Массив с информацией о разборе отдельных столбцов.
Интерпретация зависит от значения DataType.
При разборе разделённых данных это массив двухэлементных массивов, каждый из которых
задаёт параметры преобразования для конкретного столбца.
Первый элемент – номер столбца (начиная с 1), второй элемент – одна из констант XlColumnDataType,
определяющая способ разбора столбца (по умолчанию Default).
$sDecimalSeparator [необязательный] Десятичный разделитель, используемый Excel при распознавании чисел.
По умолчанию используется системная настройка (Default).
$sThousandsSeparator [необязательный] Разделитель тысяч, используемый Excel при распознавании чисел.
По умолчанию используется системная настройка (Default).
$bTrailingMinusNumbers [необязательный] True – числа с минусом в конце считаются отрицательными.
False – такие числа считаются текстом (по умолчанию True).
$iOrigin [необязательный] Происхождение текстового файла. Может быть одной из констант XlPlatform.
Также можно указать целое число – номер кодовой страницы.
Например, "1256" означает арабскую кодировку (Windows).
Если параметр опущен, используется текущая настройка параметра "Источник файла" в мастере импорта текста.

Возвращаемое значение

Успех: Возвращает объект рабочей книги.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oExcel не является объектом или не является объектом приложения
2 – указанный файл $sFilePath не существует
3 – не удалось открыть или разобрать $sFilePath. @extended содержит COM-код ошибки метода OpenText

Примечания

Параметр $aFieldInfo должен быть массивом, содержащим массивы, а не двумерным массивом. См. пример 1.

См. также

_Excel_BookAttach, _Excel_BookClose, _Excel_BookNew, _Excel_BookOpen

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpenText Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Открываем текстовый файл с разделителем |, передаём fieldinfo и задаём
; Десятичный разделитель и разделитель тысяч.
Local $sTextFile = _Extras_PathFull("_Excel1.txt")
Local $aField1[2] = [1, $xlTextFormat]
Local $aField2[2] = [2, $xlTextFormat]
Local $aField3[2] = [3, $xlGeneralFormat]
Local $aField4[2] = [4, $xlDMYFormat]
Local $aField5[2] = [5, $xlTextFormat]
Local $aFieldInfo[5] = [$aField1, $aField2, $aField3, $aField4, $aField5]
_Excel_BookOpenText($oExcel, $sTextFile, Default, $xlDelimited, Default, True, "|", $aFieldInfo, ",", ".")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpenText Example 1", "Error opening '" & $sTextFile & "'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpenText Example 1", "Workbook '" & $sTextFile & "' has been opened successfully.")

Пример 2

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox(16, "Excel UDF: _Excel_BookOpenText Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Открываем текстовый файл с фиксированной шириной столбцов; остальные параметры Excel определит сам.
Local $sTextFile = _Extras_PathFull("_Excel2.txt")
_Excel_BookOpenText($oExcel, $sTextFile, Default, $xlFixedWidth)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpenText Example 2", "Error opening '" & $sTextFile & "'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpenText Example 2", "Workbook '" & $sTextFile & "' has been opened successfully.")
