Справочник функций

_WinAPI_CreateBuffer

выделяет блок памяти из внутреннего кучи библиотеки.

#include <WinAPIMem.au3>
_WinAPI_CreateBuffer ( $iLength [, $pBuffer = 0 [, $bAbort = True]] )

Параметры

$iLength Требуемая длина буфера в байтах.
$pBuffer [необязательный] Указатель на существующий буфер, который нужно заменить новым.
Если это допустимый указатель на буфер, память будет перевыделена под новый буфер.
Однако новая память выделяется в другом месте. Поэтому всегда используйте указатель, который возвращает эта функция.
Если параметр равен 0 (по умолчанию) или указатель недопустим, функция просто выделяет новую память.
$bAbort Координаты.

Возвращаемое значение

Успех: Возвращает указатель на new буфер.
Ошибка: Ошибка.

Примечания

Когда буфер больше не нужен, его нужно уничтожить вызовом _WinAPI_FreeMemory().
