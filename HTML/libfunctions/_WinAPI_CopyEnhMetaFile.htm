_WinAPI_CopyEnhMetaFile

Копирует содержимое метафайла расширенного формата в указанный файл.

#include <WinAPIGdi.au3>
_WinAPI_CopyEnhMetaFile ( $hEmf [, $sFilePath = ''] )

Параметры

$hEmf Дескриптор копируемого расширенного метафайла.
$sFilePath [необязательный] Имя файла назначения (.emf). Если этот параметр равен пустой строке (по умолчанию), исходный метафайл копируется в память.

Возвращаемое значение

Успех: Возвращает дескриптор копии расширенного метафайла.
Ошибка: Возвращает 0.

Примечания

Когда приложению больше не нужен дескриптор расширенного метафайла, его следует удалить вызовом
_WinAPI_DeleteEnhMetaFile().

См. также

_WinAPI_DeleteEnhMetaFile
