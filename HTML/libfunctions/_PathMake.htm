Справочник функций

_PathMake

Собирает путь из компонентов диска, каталога, имени файла и расширения.

#include <File.au3>
_PathMake ( $sDrive, $sDir, $sFileName, $sExtension )

Параметры

$sDrive Диск (может быть UNC). Если это буква диска, двоеточие добавляется автоматически.
$sDir Каталог. Косая черта в конце и в начале добавляется, если её нет.
$sFileName Имя файла.
$sExtension Расширение файла. Точка добавляется, если её нет в расширении.

Возвращаемое значение

Возвращает строка, содержащая полный путь.

Примечания

Путь всё равно будет построен из переданных значений. Это не проверяет допустимость созданного пути, он может содержать недопустимые для файловой системы символы.
Вы можете передать "" (пустую строку) для любой части пути, которая не требуется в окончательном результате.

См. также

_PathFull, _PathSplit

Пример

#include <File.au3>
#include <MsgBoxConstants.au3>

; Создать путь.
Local $sFilePath = _PathMake("C:", "\Temp", "Boot", "ini")
MsgBox($MB_SYSTEMMODAL, "", $sFilePath)
