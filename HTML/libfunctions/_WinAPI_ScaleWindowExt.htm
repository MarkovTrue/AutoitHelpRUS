_WinAPI_ScaleWindowExt

Изменяет окно для контекста устройства, используя пропорции, образованные указанными множителями и делителями.

#include <WinAPIGdi.au3>
_WinAPI_ScaleWindowExt ( $hDC, $iXNum, $iXDenom, $iYNum, $iYDenom )

Параметры

$hDC Дескриптор контекста устройства.
$iXNum Значение, на которое следует умножить текущий горизонтальный размер.
$iXDenom Значение, на которое следует разделить текущий горизонтальный размер.
$iYNum Значение, на которое следует умножить текущий вертикальный размер.
$iYDenom Значение, на которое следует разделить текущий вертикальный размер.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Вызовите _WinAPI_GetExtended() для получения структуры $tagSIZE, содержащей предыдущие размеры окна.
