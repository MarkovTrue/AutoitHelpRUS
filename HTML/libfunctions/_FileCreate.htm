Справочник функций

_FileCreate

Создаёт указанный файл или обнуляет его содержимое.

#include <File.au3>
_FileCreate ( $sFilePath )

Параметры

$sFilePath Путь и имя создаваемого файла.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – ошибка при открытии указанного файла
2 – невозможно записать в файл

Пример

#include <File.au3>
#include <MsgBoxConstants.au3>

If Not _FileCreate("error.log") Then
    MsgBox($MB_SYSTEMMODAL, "Error", " Error Creating/Resetting log. error:" & @error)
EndIf
