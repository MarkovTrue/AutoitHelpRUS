Справочник функций

_GUICtrlHeader_GetItem

Получает информацию о элементе заголовка.

#include <GuiHeader.au3>
_GUICtrlHeader_GetItem ( $hWnd, $iIndex, ByRef $tItem )

Параметры

$hWnd Дескриптор элемента управления.
$iIndex Индекс элемента (нумерация с 0).
$tItem Структура $tagHDITEM.

Возвращаемое значение

Успех: Возвращает True, переменная @extended может быть установлена (см. раздел Примечания).
Ошибка: Возвращает False.

Примечания

Если элемент управления, на который ссылается $hWnd, находится в другом процессе и оба процесса работают в разных режимах AutoIt (@AutoItVersion),
переменная @extended будет установлена так, чтобы $tItem нужно было адаптировать для размера указателя.

Когда сообщение отправляется, элемент mask указывает тип запрашиваемой информации.
Когда сообщение возвращается, другие элементы получают запрашиваемую информацию.
Если элемент mask имеет значение ноль, сообщение возвращает True, но не копирует информацию в структуру.

См. также

$tagHDITEM, _GUICtrlHeader_SetItem

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiHeader.au3>
#include <GuiImageList.au3>
#include <StructureConstants.au3>
#include <WinAPIGdi.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Header Get/Set Item (v" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $hHeader = _GUICtrlHeader_Create($hGUI)
    _GUICtrlHeader_SetUnicodeFormat($hHeader, True)
    _MemoCreate(2, 52, 444, 220)

    GUISetState(@SW_SHOW)

    ; Создать image list с изображениями
    Local $hImage = _GUIImageList_Create(11, 11)
    _GUIImageList_Add($hImage, _WinAPI_CreateSolidBitmap($hGUI, 0xFF0000, 11, 11))
    _GUIImageList_Add($hImage, _WinAPI_CreateSolidBitmap($hGUI, 0x00FF00, 11, 11))
    _GUIImageList_Add($hImage, _WinAPI_CreateSolidBitmap($hGUI, 0x0000FF, 11, 11))
    _GUICtrlHeader_SetImageList($hHeader, $hImage)

    ; Добавить столбцы
    _GUICtrlHeader_AddItem($hHeader, "Column 0", 100, 0, 0)
    _GUICtrlHeader_AddItem($hHeader, "Column 1", 100, 0, 1)
    _GUICtrlHeader_AddItem($hHeader, "Column 2", 100, 0, 2)
    _GUICtrlHeader_AddItem($hHeader, "Column 3", 100)

    ; Установить индекс изображения столбца 2
    Local $tItem = DllStructCreate($tagHDITEM)
    DllStructSetData($tItem, "Mask", $HDI_IMAGE)
    DllStructSetData($tItem, "Image", 0)
    _GUICtrlHeader_SetItem($hHeader, 2, $tItem)

    ; Показать индекс изображения столбца 2
    $tItem = DllStructCreate($tagHDITEM)
    DllStructSetData($tItem, "Mask", $HDI_IMAGE)
    _GUICtrlHeader_GetItem($hHeader, 2, $tItem)
    _MemoWrite("Column 2 image index: " & DllStructGetData($tItem, "Image"))

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
