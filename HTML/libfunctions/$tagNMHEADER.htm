Справочник ключевых слов

$tagNMHEADER

Содержит сведения об уведомляющих сообщениях элемента управления.

Global Const $tagNMHEADER = $tagNMHDR & ";int Item;int Button;ptr pItem"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Item Индекс с отсчётом от нуля элемента управления, к которому относится эта структура.
Button Индекс кнопки мыши, используемой для создания сообщения уведомления. Этот элемент может быть одним из следующих значений:
0 – левая кнопка
1 – правая кнопка
2 – средняя кнопка
pItem Необязательный указатель на структуру tagHDITEM, содержащую информацию об элементе, указанном в Item.
Элемент mask структуры tagHDITEM указывает, какие из его элементов действительны.
