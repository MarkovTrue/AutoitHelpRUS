_WinAPI_SetDIBColorTable

Устанавливает таблицу цветов RGB в DIB-section растр.

#include <WinAPIGdi.au3>
_WinAPI_SetDIBColorTable ( $hBitmap, $tColorTable, $iColorCount )

Параметры

$hBitmap DIB-section растр, в котором требуется установить таблицу цветов.
$tColorTable Структура "dword[n]", представляющая таблицу цветов DIB для установки.
$iColorCount Количество записей таблицы цветов для установки.

Возвращаемое значение

Успех: Возвращает количество записей таблицы цветов, которые устанавливает функция.
Ошибка: Возвращает 0.

Примечания

Функция _WinAPI_GetDIBColorTable() устанавливает таблицу цветов для DIB-section растров, использующих 1, 4 или 8 бит/пикс.
DIB-section растры, использующие значение бит/пикс больше восьми, не имеют таблицы цветов.
