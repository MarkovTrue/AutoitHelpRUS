_WinAPI_GetDIBits

Извлекает данные растрового изображения и копирует их в буфер в формате DIB.

#include <WinAPIGdi.au3>
_WinAPI_GetDIBits ( $hDC, $hBitmap, $iStartScan, $iScanLines, $pBits, $tBI, $iUsage )

Параметры

$hDC Дескриптор контекста устройства.
$hBitmap Дескриптор растра. Должен быть device-dependent bitmap (DDB).
$iStartScan Номер первой строки развёртки, которую нужно извлечь.
$iScanLines Сколько строк развёртки извлечь.
$pBits Указатель на буфер для получения данных битмапа.
Если параметр равен 0, функция передаёт размеры и формат битмапа в структуру $tagBITMAPINFO, на которую указывает параметр $tBI.
$tBI Структура $tagBITMAPINFO или указатель на неё, задающая желаемый формат данных DIB.
$iUsage Формат поля bmiColors структуры $tagBITMAPINFO.
Должно быть одним из:
$DIB_PAL_COLORS – таблица цветов содержит массив 16-битных индексов в текущей палитре
$DIB_RGB_COLORS – таблица цветов содержит явные значения red, green, blue.

Возвращаемое значение

Успех: Если $pBits не 0 и функция успешна, возвращается количество скопированных из растра строк развёртки.
Если $pBits = 0 и GetDIBits успешно заполнил структуру, возвращается True.
Ошибка: Возвращает False.
