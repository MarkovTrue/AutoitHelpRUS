_WinAPI_SetTextAlign

Устанавливает флаги выравнивания текста для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetTextAlign ( $hDC [, $iMode = 0] )

Параметры

$hDC Дескриптор контекста устройства.
$iMode [необязательный] Выравнивание текста с использованием маски значений в следующем списке. Можно выбрать только один флаг
из тех, которые влияют на горизонтальное и вертикальное выравнивание. Кроме того, можно выбрать только один из двух флагов, которые
изменяют текущую позицию.
$TA_BASELINE
$TA_BOTTOM
$TA_TOP (по умолчанию)
$TA_CENTER
$TA_LEFT (по умолчанию)
$TA_RIGHT
$TA_NOUPDATECP (по умолчанию)
$TA_RTLREADING
$TA_UPDATECP

Когда текущий шрифт имеет вертикальную базовую линию по умолчанию, как с Kanji, следующие значения должны быть
использованы вместо $TA_BASELINE и $TA_CENTER.
$VTA_BASELINE
$VTA_CENTER

Значения по умолчанию $TA_LEFT, $TA_TOP и $TA_NOUPDATECP.

Возвращаемое значение

Успех: Возвращает предыдущая настройка выравнивания текста ($TA_* и $VTA_*).
Ошибка: (-1) GDI_ERROR.
