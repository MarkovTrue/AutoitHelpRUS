Справочник функций

_SQLite_SafeMode

Отключает или включает безопасный режим.

#include <SQLite.au3>
_SQLite_SafeMode ( $bSafeModeState )

Параметры

$bSafeModeState True или False для включения или отключения безопасного режима.

Возвращаемое значение

Возвращает $SQLITE_OK.

Примечания

Безопасный режим предотвращает краши SQLite, которые могут быть вызваны передачей неправильных дескрипторов запроса или базы данных.
Он включён по умолчанию.

Пример

#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_SafeMode(False)
_SQLite_Exec(-1, "CREATE tblTest (a,b,c);"); Нет открытой БД, произойдёт ошибка из-за SafeMode = false
_SQLite_Shutdown()
