Справочник функций

_GUICtrlAVI_Show

Показывает или скрывает AVI-элемент управления.

#include <GuiAVI.au3>
_GUICtrlAVI_Show ( $hWnd, $iState )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iState Состояние AVI, одно из значений:

Возвращаемое значение

Успех: True – элемент управления был виден ранее.
False – элемент управления был скрыт ранее.
Ошибка: Устанавливает @error ≠ 0.

Пример

Пример 1

; используя функцию AutoIt

#include <GuiAVI.au3>
#include <GUIConstantsEx.au3>

_Example_Internal()

Func _Example_Internal()
    Local $idAvi_AVI, $idBtn_Start, $idBtn_Stop, $idBtn_Show

    ; Создать GUI
    GUICreate("(Internal) AVI Play", 350, 200)
    $idAvi_AVI = GUICtrlCreateAvi(@SystemDir & "\shell32.dll", 160, 10, 10)
    $idBtn_Start = GUICtrlCreateButton("start", 25, 150, 70, 22)
    $idBtn_Stop = GUICtrlCreateButton("stop", 125, 150, 70, 22)
    $idBtn_Show = GUICtrlCreateButton("hide", 225, 150, 70, 22)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtn_Start
                ; Воспроизвести часть AVI-клипа
                _GUICtrlAVI_Play($idAvi_AVI)
            Case $idBtn_Stop
                ; Остановить AVI-клип
                _GUICtrlAVI_Stop($idAvi_AVI)
            Case $idBtn_Show
                If GUICtrlRead($idBtn_Show) = "hide" Then
                    _GUICtrlAVI_Show($idAvi_AVI, @SW_HIDE)
                    GUICtrlSetData($idBtn_Show, "show")
                Else
                    _GUICtrlAVI_Show($idAvi_AVI, @SW_SHOW)
                    GUICtrlSetData($idBtn_Show, "hide")
                EndIf
        EndSwitch
    WEnd

    ; Закрыть AVI-клип
    _GUICtrlAVI_Close($idAvi_AVI)

    GUIDelete()
EndFunc   ;==>_Example_Internal
