Справочник функций

_WinAPI_StringLenA

Вычисляет размер строки ANSI.

#include <WinAPIMisc.au3>
_WinAPI_StringLenA ( Const ByRef $tString )

Параметры

$tString Структура строки для обработки.

Возвращаемое значение

Успех: Возвращает длина строки в символах.
Ошибка: Возвращает 0.

Примечания

Возвращаемая длина не включает завершающий нулевой символ.

См. также

_WinAPI_StringLenW

Дополнительно

Искать lstrlenA в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIMisc.au3>

_Example()

Func _Example()
    ; Создайте буфер строки. Это структура типа "char". Выбор размера 64 символа.
    Local $tStringBuffer = DllStructCreate("char Data[64]")
    ; Заполнить какими-нибудь данными
    DllStructSetData($tStringBuffer, "Data", "Callipygian")

    MsgBox($MB_SYSTEMMODAL, "_WinAPI_StringLenA", "Length of a string inside the buffer is " & _WinAPI_StringLenA($tStringBuffer) & " characters.")
EndFunc   ;==>_Example
