Справочник ключевых слов

$tagGDIP_EFFECTPARAMS_Levels

Структура параметров эффекта уровней.

Global Const $tagGDIP_EFFECTPARAMS_Levels = "int Highlight; int Midtone; int Shadow"

Параметры

Highlight Целое число от 0 до 100, указывающее, какие пикселы должны быть осветлены.
Midtone Целое число от -100 до 100, указывающее, насколько осветлить или затемнить изображение.
Shadow Целое число от 0 до 100, указывающее, какие пикселы должны быть затемнены.
