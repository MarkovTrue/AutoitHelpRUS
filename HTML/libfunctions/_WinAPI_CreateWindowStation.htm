Справочник функций

_WinAPI_CreateWindowStation

создаёт объект оконной станции, связывает его с вызывающим процессом и назначает текущей сессии.

#include <WinAPISys.au3>
_WinAPI_CreateWindowStation ( [$sName = '' [, $iAccess = 0 [, $iFlags = 0 [, $tSecurity = 0]]]] )

Параметры

$sName Имя.
$iAccess [необязательный] Тип доступа возвращаемого дескриптора к оконной станции. Может быть одним или несколькими из следующих значений:
$WINSTA_ALL_ACCESS
$WINSTA_ACCESSCLIPBOARD
$WINSTA_ACCESSGLOBALATOMS
$WINSTA_CREATEDESKTOP
$WINSTA_ENUMDESKTOPS
$WINSTA_ENUMERATE
$WINSTA_EXITWINDOWS
$WINSTA_READATTRIBUTES
$WINSTA_READSCREEN
$WINSTA_WRITEATTRIBUTES
$iFlags Значение.
$tSecurity Дескриптор.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Когда дескриптор больше не нужен, освободите его вызовом _WinAPI_CloseWindowStation().

См. также

_WinAPI_CloseWindowStation
