Справочник функций

_DebugReport

Записывает данные в сеанс отладки.

#include <Debug.au3>
_DebugReport ( $sData [, $bLastError = False [, $bExit = False]] )

Параметры

$sData Данные для записи в журнал.
$bLastError [необязательный] True – добавить к выводимым данным результат _WinAPI_GetLastErrorMessage().
$bExit [необязательный] True – скрипт должен быть завершён

Возвращаемое значение

Успех: Возвращает 1 (см. раздел Примечания).
Ошибка: Возвращает 0.

Примечания

Если _DebugSetup() не была вызвана, функция всегда возвращает управление.
Значение @error вызывающего кода сохраняется.
@extended может быть установлен равным значению _WinAPI_GetLastError(), если $bLastError = True;
в противном случае @extended сохраняется.

Каждый вызов _DebugReport() выводит данные на отдельной строке, если $bLastError = True.

См. также

_DebugSetup

Пример

#include <Debug.au3>

_DebugSetup()

_DebugReport("message1")
_DebugReport("message2", True) ; с добавленным последним сообщением об ошибке
