_WinAPI_GetObjectType

Получает тип указанного объекта.

#include <WinAPIHObj.au3>
_WinAPI_GetObjectType ( $hObject )

Параметры

$hObject Дескриптор графического объекта.

Возвращаемое значение

Успех: Возвращает значение определяет объект ($OBJ_*).
Ошибка: Возвращает 0.
