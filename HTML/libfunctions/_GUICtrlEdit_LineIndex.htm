_GUICtrlEdit_LineIndex

Получает индекс символа первого символа указанной строки.

#include <GuiEdit.au3>
_GUICtrlEdit_LineIndex ( $hWnd [, $iIndex = -1] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iIndex [необязательный] указывает номер строки (начиная с 0)

Возвращаемое значение

Успех: Возвращает индекс символа строки, указанной в параметре $iIndex.
Ошибка: Возвращает -1, если указанный номер строки больше, чем количество строк в элементе управления.

Примечания

$iIndex = -1 указывает на текущий номер строки (строку, содержащую каретку).

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <GuiStatusBar.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hStatusBar, $idEdit, $hGUI
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\include\_ReadMe_.txt"
    Local $aPartRightSide[2] = [378, -1], $iRandom

    ; Создать GUI
    $hGUI = GUICreate("Edit Line Index", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268, BitOR($ES_WANTRETURN, $WS_VSCROLL))
    $hStatusBar = _GUICtrlStatusBar_Create($hGUI, $aPartRightSide)
    _GUICtrlStatusBar_SetIcon($hStatusBar, 1, 97, "shell32.dll")
    GUISetState(@SW_SHOW)

    ; Задать отступы
    _GUICtrlEdit_SetMargins($idEdit, BitOR($EC_LEFTMARGIN, $EC_RIGHTMARGIN), 10, 10)

    ; Установить текст
    _GUICtrlEdit_SetText($idEdit, FileRead($sFile))

    ; Индекс строки
    $iRandom = Random(0, _GUICtrlEdit_GetLineCount($idEdit) - 1, 1)
    _GUICtrlStatusBar_SetText($hStatusBar, "Char Index: " & _GUICtrlEdit_LineIndex($idEdit, $iRandom) & " from Line: " & $iRandom)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
