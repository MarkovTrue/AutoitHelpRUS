Справочник функций

_IEFormElementRadioSelect

Задаёт значение указанного элемента формы.

#include <IE.au3>
_IEFormElementRadioSelect ( ByRef $oObject, $sString, $sName [, $iSelect = 1 [, $sMode = "byValue" [, $iFireEvent = 1]]] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application или Form.
$sString Значение для сравнения с элементом – обработка зависит от $sMode.
$sName Имя или Id группы радиокнопок.
$iSelect [необязательный] задаёт, должен ли элемент быть выделен или снят выделение
-1 = Вернуть состояние выделения
0 = Снять выделение с элемента
1 = (по умолчанию) Выделить элемент
$sMode [необязательный] задаёт режим поиска
"byValue" = (по умолчанию) значение радио-кнопки, которую требуется выбрать
"byIndex" = индекс радио-кнопки (с нуля), которую требуется выбрать
$iFireEvent [необязательный] Запускать ли события OnChange и OnClick после изменения значения:
0 – не запускать события OnChange/OnClick после установки значения
1 – (по умолчанию) запускать события OnChange и OnClick после установки значения

Возвращаемое значение

Успех: текущее состояние выбора, если $iSelect = -1; иначе возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадений не найдено
@extended: Содержит номер недопустимого параметра.

Примечания

Параметр $iFireEvent имеет значение только если с элементом формы связано событие onChange.

Параметр $sName для этой функции обязателен.
Радио-кнопки обрабатываются группами, имеющими одинаковое имя.
В любой момент в группе может быть выбрано не более одного элемента – при выборе одного остальные снимают выделение.

См. также

_IEFormElementCheckBoxSelect, _IEFormElementGetValue, _IEFormElementOptionSelect, _IEFormElementSetValue

Пример

Пример 1

; Открыть браузер с примером формы, получить ссылку на форму, выделить
; каждую радиокнопку по значению, затем снять выбор с последней, не оставив ни одной выбранной.

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
_IEAction($oForm, "focus")
For $i = 1 To 3
    _IEFormElementRadioSelect($oForm, "vehicleAirplane", "radioExample", 1, "byValue")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, "vehicleTrain", "radioExample", 1, "byValue")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, "vehicleBoat", "radioExample", 1, "byValue")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, "vehicleCar", "radioExample", 1, "byValue")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, "vehicleCar", "radioExample", 0, "byValue")
    Sleep(1000)
Next

_IEQuit($oIE)

Пример 2

; Открыть браузер с примером формы, получить ссылку на форму, выделить
; каждую радиокнопку по индексу, затем снять выбор с последней, не оставив ни одной выбранной.
; Примечание: вероятно, потребуется прокрутить страницу, чтобы увидеть изменения

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
For $i = 1 To 3
    _IEFormElementRadioSelect($oForm, 3, "radioExample", 1, "byIndex")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, 2, "radioExample", 1, "byIndex")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, 1, "radioExample", 1, "byIndex")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, 0, "radioExample", 1, "byIndex")
    Sleep(1000)
    _IEFormElementRadioSelect($oForm, 0, "radioExample", 0, "byIndex")
    Sleep(1000)
Next

_IEQuit($oIE)
