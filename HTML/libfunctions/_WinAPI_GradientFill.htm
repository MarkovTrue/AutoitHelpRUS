Справочник функций

_WinAPI_GradientFill

Заливает прямоугольник или треугольник с градиентом.

#include <WinAPIGdi.au3>
_WinAPI_GradientFill ( $hDC, Const ByRef $aVertex [, $iStart = 0 [, $iEnd = -1 [, $bRotate = False]]] )

Параметры

$hDC Дескриптор контекста устройства.
$aVertex Двумерный массив ([x1, y1, $iRGB1], [x2, y2, $iRGB2], ... [xN, yN, $iRGBN]), содержащий необходимые вершины градиента. Каждая вершина в этом массиве содержит следующие параметры. x – координата x в логических единицах. y – координата y в логических единицах. rgb – информация о цвете в точке x, y.
$iStart [необязательный] Индекс массива, с которого начинать заливку.
$iEnd [необязательный] Индекс массива, на котором остановить заливку.
$bRotate [необязательный] Указывает, заливает ли прямоугольник слева направо (горизонтальный градиент).     $bRotate используется только для прямоугольных градиентов; для треугольных градиентов этот параметр будет проигнорирован,     допустимые значения:         True – заливка слева направо.         False – заливка сверху вниз (по умолчанию).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Если количество вершин, определённых параметрами $iStart и $iEnd, равно 2, функция _WinAPI_GradientFill() заливает прямоугольник. Если количество вершин равно 3, заливает треугольник. Для прямоугольника вершины должны указывать его верхний левый и нижний правый углы. Обратите внимание, что массив $aVertex может содержать любое количество вершин градиента, но одновременно можно использовать только 2 или 3 вершины из этого массива. В противном случае функция не выполнится.
Функция _WinAPI_GradientFill() может заливать прямоугольник или треугольник только за один вызов. Используйте несколько вызовов этой функции для заливки сложных градиентов.

Пример

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>

Local $aVertex[6][3] = [[0, 0, 0xFF0000], [400, 400, 0x00FF00], [0, 400, 0x0000FF], [0, 0, 0xFF0000], [400, 0, 0xFFFF00], [400, 400, 0x00FF00]]

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400)
Local $idPic = GUICtrlCreatePic('', 0, 0, 400, 400)
Local $hPic = GUICtrlGetHandle($idPic)

; Создать градиент
Local $hDC = _WinAPI_GetDC($hPic)
Local $hDestDC = _WinAPI_CreateCompatibleDC($hDC)
Local $hBitmap = _WinAPI_CreateCompatibleBitmap($hDC, 400, 400)
Local $hDestSv = _WinAPI_SelectObject($hDestDC, $hBitmap)
_WinAPI_GradientFill($hDestDC, $aVertex, 0, 2)
_WinAPI_GradientFill($hDestDC, $aVertex, 3, 5)

_WinAPI_ReleaseDC($hPic, $hDC)
_WinAPI_SelectObject($hDestDC, $hDestSv)
_WinAPI_DeleteDC($hDestDC)

; Установить градиент для элемента управления
_SendMessage($hPic, $STM_SETIMAGE, 0, $hBitmap)
Local $hObj = _SendMessage($hPic, $STM_GETIMAGE)
If $hObj <> $hBitmap Then
    _WinAPI_DeleteObject($hBitmap)
EndIf

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
