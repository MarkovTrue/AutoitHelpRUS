_WinAPI_GetWindowDisplayAffinity

Получает текущий параметр политики отображения, из любого процесса, для заданного окна.

#include <WinAPISysWin.au3>
_WinAPI_GetWindowDisplayAffinity ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

Успех: Возвращает параметр политики отображения текущего окна ($WDA_*).
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Эта функция работает только когда окно слоистое и диспетчер рабочего стола Windows (DWM) составляет рабочий стол.
Для получения дополнительной информации см. _WinAPI_SetWindowDisplayAffinity().

Эта функция требует Windows 7 или более позднюю версию.
