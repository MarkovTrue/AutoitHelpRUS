_GDIPlus_LineBrushCreateFromRectWithAngle

Создаёт объект LinearGradientBrush по прямоугольнику, граничным цветам и углу направления.

#include <GDIPlus.au3>
_GDIPlus_LineBrushCreateFromRectWithAngle ( $tRECTF, $iARGBClr1, $iARGBClr2, $fAngle [, $bIsAngleScalable = True [, $iWrapMode = 0]] )

Параметры

$tRECTF Структура , задающая начальную и конечную точки градиента.
$iARGBClr1 Компоненты альфа, красный, зелёный и синий начального цвета линии.
$iARGBClr2 Компоненты альфа, красный, зелёный и синий конечного цвета линии.
$fAngle В зависимости от значения – базовый угол или угол направления в градусах (см. раздел Примечания).
$bIsAngleScalable [необязательный] Если True – угол направления является масштабируемым; иначе – фиксированным.
$iWrapMode [необязательный] Режим мозаичного заполнения областей кистью:
0 - Без отражения
1 - Плитки отражаются горизонтально при переходе к следующей плитке в строке
2 - Плитки отражаются вертикально при переходе к следующей плитке в столбце
3 - Плитки отражаются горизонтально по строке и вертикально по столбцу
4 - Без мозаики.

Возвращаемое значение

Успех: Возвращает указатель на новый объект LinearGradientBrush.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

Если равен True, параметр задаёт базовый угол, из которого вычисляется угол направления. Иначе задаёт угол направления напрямую. Угол измеряется от верхнего края прямоугольника в градусах. Градиент следует вдоль направляющей линии.
По окончании работы вызовите _GDIPlus_BrushDispose() для освобождения ресурсов объекта.

См. также

_GDIPlus_BrushDispose

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic, $hBrush, $tRECTF, $hPen, $fAngle

    $hGUI = GUICreate("GDI+", 830, 340)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsClear($hGraphic, 0xFFFFFFFF)

    $fAngle = 45

    $tRECTF = _GDIPlus_RectFCreate(10, 10, 400, 300)
    $hBrush = _GDIPlus_LineBrushCreateFromRectWithAngle($tRECTF, 0xFF000000, 0xFFFFFFFF, $fAngle, False)
    _GDIPlus_LineBrushSetSigmaBlend($hBrush, 1)
    _GDIPlus_GraphicsFillRect($hGraphic, 10, 10, 400, 300, $hBrush)

    _GDIPlus_BrushDispose($hBrush)

    $tRECTF = _GDIPlus_RectFCreate(420, 10, 400, 300)
    $hBrush = _GDIPlus_LineBrushCreateFromRectWithAngle($tRECTF, 0xFF000000, 0xFFFFFFFF, $fAngle, True)
    _GDIPlus_LineBrushSetSigmaBlend($hBrush, 1)
    _GDIPlus_GraphicsFillRect($hGraphic, 420, 10, 400, 300, $hBrush)

    $hPen = _GDIPlus_PenCreate(0x6F00007F, 2)
    $fAngle *= 0.01745
    _GDIPlus_GraphicsDrawLine($hGraphic, 10, 10, 10 + Cos($fAngle) * 300, 10 + Sin($fAngle) * 300, $hPen)
    _GDIPlus_GraphicsDrawString($hGraphic, "$bIsAngleScalable = False", 10, 320)

    $fAngle = (400 / 300) * ATan($fAngle)
    _GDIPlus_GraphicsDrawLine($hGraphic, 420, 10, 420 + Cos($fAngle) * 300, 10 + Sin($fAngle) * 300, $hPen)
    _GDIPlus_GraphicsDrawString($hGraphic, "IsAngleScalable = True", 420, 320)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_PenDispose($hPen)
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
