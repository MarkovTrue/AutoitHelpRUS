Справочник ключевых слов

$tagNMDATETIMEKEYDOWN

Содержит сведения для обработки уведомляющего сообщения $DTN_WMKEYDOWN.

Global Const $tagNMDATETIMEKEYDOWN = $tagNMHDR & ";int VirtKey;ptr Format;" & $tagSYSTEMTIME

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
VirtKey Код виртуальной клавиши, представляющий клавишу, нажатую пользователем.
Format Указатель на строку с нулём в конце, которая определяет поле обратного вызова элемента управления.
Строка состоит из одного или нескольких символов "X".
$tagSYSTEMTIME Содержит информацию о текущей системной дате и времени из элемента управления DTP.
Владелец элемента управления должен изменить информацию о времени в зависимости от нажатия клавиши пользователем.
