Справочник функций

_ArrayFindAll

Находит индексы всех вхождений искомого значения в одномерном 1D или двумерном 2D массиве через _ArraySearch().

#include <Array.au3>
_ArrayFindAll ( Const ByRef $aArray, $vValue [, $iStart = 0 [, $iEnd = 0 [, $iCase = 0 [, $iCompare = 0 [, $iSubItem = 0 [, $bRow = False]]]]]] )

Параметры

$aArray Массив для поиска.
$vValue Искомое значение в $aArray.
$iStart [необязательный] Индекс массива для начала поиска
$iEnd [необязательный] Индекс массива для окончания поиска
$iCase [необязательный] Если 1, поиск с учётом регистра
$iCompare [необязательный]
0 – приведение переменных к одному типу (по умолчанию): "string" = 0, "" = 0 или "0" = 0 совпадают (если $iCase = 0)
1 – частичный поиск
2 – совпадение при одинаковом типе и значении переменных
3 – сравнение по регулярному выражению, переданному в $vValue
$iSubItem [необязательный] Подиндекс для поиска в 2D массивах
$bRow [необязательный] Если True, $iSubItem задаёт строку для поиска; False (по умолчанию) – поиск по столбцам

Возвращаемое значение

Успех: Возвращает массив всех индексов элементов, содержащих $vValue.
Ошибка: Устанавливает @error ≠ 0 (коды ошибок – в описании _ArraySearch()).

Примечания

Значения $iCompare нельзя комбинировать.

См. также

_ArrayBinarySearch, _ArraySearch

Пример

#include <Array.au3>

Local $aArray[5] = [0, 1, 2, 1, 0]
_ArrayDisplay($aArray, "1D array")
Local $aiResult = _ArrayFindAll($aArray, 0)
_ArrayDisplay($aiResult, "Found")

Local $aArray[5][5] = [[0, 1, 2, 1, 0], _
        [4, 5, 5, 4, 2], _
        [4, 1, 3, 1, 3], _
        [0, 3, 2, 1, 0], _
        [1, 5, 5, 4, 1]]
_ArrayDisplay($aArray, "2D array")

Local $aResult = _ArrayFindAll($aArray, 0, Default, Default, Default, Default, 4)
_ArrayDisplay($aResult, "Found in Column 4")

$aResult = _ArrayFindAll($aArray, 1, Default, Default, Default, Default, 2, True)
_ArrayDisplay($aResult, "Found in Row 2")
