Справочник функций

_WinAPI_FindNextFileName

Продолжает перечисление жёстких ссылок.

#include <WinAPIFiles.au3>
_WinAPI_FindNextFileName ( $hSearch, ByRef $sLink )

Параметры

$hSearch Дескриптор перечисления, возвращённый успешным вызовом _WinAPI_FindFirstFileName().
$sLink Возвращает имя следующей найденной ссылки.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Если подходящих файлов нет, функция завершается с @extended = ERROR_HANDLE_EOF (38).

Требуется Windows Vista или новее.

См. также

_WinAPI_FindFirstFileName

Дополнительно

Искать FindNextFileNameW в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>
#include <WinAPIFiles.au3>
#include <WinAPIShPath.au3>

Local $sFile = @DesktopDir & '\@' & StringRegExpReplace(_WinAPI_PathFindFileName(@ScriptName), '\A_+', '')

; Создать жёсткую ссылку на текущий файл с префиксом "@" на рабочем столе
If Not _WinAPI_CreateHardLink($sFile, @ScriptFullPath) Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to create hard link.')
    Exit
EndIf

; Перечислить все жёсткие ссылки на файл

Local $sLink
Local $hSearch = _WinAPI_FindFirstFileName($sFile, $sLink)
While Not @error
    ConsoleWrite(_WinAPI_PathAppend(_WinAPI_PathStripToRoot($sFile), $sLink) & @CRLF)
    _WinAPI_FindNextFileName($hSearch, $sLink)
WEnd

Switch @extended
    Case 38 ; ERROR_HANDLE_EOF

    Case Else
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), @extended, _WinAPI_GetErrorMessage(@extended))
EndSwitch

FileDelete($sFile)
