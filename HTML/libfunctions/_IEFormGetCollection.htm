Справочник функций

_IEFormGetCollection

Возвращает объект коллекции, представляющий формы в документе или отдельную форму по индексу.

#include <IE.au3>
_IEFormGetCollection ( ByRef $oObject [, $iIndex = -1] )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$iIndex [необязательный] Что возвращать – коллекцию или элемент по индексу:
0 или положительное число – элемент с этим индексом
-1 – (по умолчанию) коллекцию

Возвращаемое значение

Успех: Возвращает переменная объекта с коллекцией всех форм в документе, @extended = количество форм.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадение не найдено
@extended: Содержит номер недопустимого параметра.

См. также

_IEFormGetObjByName, _IEFormReset, _IEFormSubmit

Пример

Пример 1

; Получить ссылку на определённую форму по индексу (начиная с 0),
; в этом случае первую форму на странице

#include <IE.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oForm = _IEFormGetCollection($oIE, 0)
Local $oQuery = _IEFormElementGetCollection($oForm, 4)
_IEFormElementSetValue($oQuery, "AutoIt IE.au3")
_IEFormSubmit($oForm)

Пример 2

; Получить ссылку на коллекцию форм страницы,
; и пройти по ним в цикле, показывая сведения о каждом

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("http://www.autoitscript.com")
Local $oForms = _IEFormGetCollection($oIE)
MsgBox($MB_SYSTEMMODAL, "Forms Info", "There are " & @extended & " form(s) on this page")
For $oForm In $oForms
    MsgBox($MB_SYSTEMMODAL, "Form Info", $oForm.name)
Next

Пример 3

; Получить ссылку на коллекцию форм страницы,
; и пройти по ним в цикле, показывая сведения о каждом
; демонстрация использования индекса формы

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("http://www.autoitscript.com")
Local $oForms = _IEFormGetCollection($oIE)
Local $iNumForms = @extended
MsgBox($MB_SYSTEMMODAL, "Forms Info", "There are " & $iNumForms & " forms on this page")
Local $oForm
For $i = 0 To $iNumForms - 1
    $oForm = _IEFormGetCollection($oIE, $i)
    MsgBox($MB_SYSTEMMODAL, "Form Info", $oForm.name)
Next
