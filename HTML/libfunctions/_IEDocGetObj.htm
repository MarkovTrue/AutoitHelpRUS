Справочник функций

_IEDocGetObj

По любому DOM-объекту возвращает ссылку на связанный с ним объект документа.

#include <IE.au3>
_IEDocGetObj ( ByRef $oObject )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.

Возвращаемое значение

Успех: Возвращает объектную переменную, ссылающуюся на объект Document.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных.
@extended: Содержит номер недопустимого параметра.

Примечания

По любому DOM-объекту (например, Browser, Window, Frame, iFrame, Document или элемент документа) функция возвращает ссылку на связанный с ним объект документа.
Особенно полезно при написании обобщённых функций, когда тип передаваемого объекта заранее неизвестен.

Пример

; Открыть браузер на AutoIt Homepage, получить ссылку
; К объекту документа и отобразить атрибут документа

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oDoc = _IEDocGetObj($oIE)
MsgBox($MB_SYSTEMMODAL, "Document Created Date", $oDoc.fileCreatedDate)
