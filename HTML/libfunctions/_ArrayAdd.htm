Справочник функций

_ArrayAdd

Добавляет указанное значение в конец существующего одномерного 1D или двумерного 2D массива.

#include <Array.au3>
_ArrayAdd ( ByRef $aArray, $vValue [, $iStart = 0 [, $sDelim_Item = "|" [, $sDelim_Row = @CRLF [, $iForce = $ARRAYFILL_FORCE_DEFAULT]]]] )

Параметры

$aArray Изменяемый массив.
$vValue Добавляемое значение или значения: один элемент, строка с разделителями или массив.
$iStart [необязательный] Столбец, с которого начинается добавление (только для 2D массива)
$sDelim_Item [необязательный] Разделитель, по которому строка разбивается на элементы
$sDelim_Row [необязательный] Разделитель, по которому строка разбивается на строки массива (только для 2D массива)
$iForce [необязательный] Сохраняет поведение по умолчанию:
$ARRAYFILL_FORCE_DEFAULT (0)
Принудительно добавляет $vValue как один элемент:
$ARRAYFILL_FORCE_SINGLEITEM (1)
Или принудительно задаёт тип данных для всех добавляемых элементов:
$ARRAYFILL_FORCE_INT (2)
$ARRAYFILL_FORCE_NUMBER (3)
$ARRAYFILL_FORCE_PTR (4)
$ARRAYFILL_FORCE_HWND (5)
$ARRAYFILL_FORCE_STRING (6)
$ARRAYFILL_FORCE_BOOLEAN (7)
См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает индекс последнего добавленного элемента.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным 1D или двумерным 2D массивом
3 – у $vValue слишком много столбцов для размещения в $aArray
4 – $iStart выходит за границы массива (только для 2D)
5 – число измерений $avArray и $vValue не совпадает

Примечания

Режим добавления зависит от типа $vValue, однако параметр $iForce может изменить поведение. По умолчанию:

1D-массивы:
    Одиночный элемент: добавляет 1 элемент
    Строка с разделителем $sDelim_Item: добавляет столько элементов, сколько частей
    1D-массив с нулевым индексом: добавляет столько элементов, сколько в массиве

2D-массивы:
    Строка с $sDelim_Item: добавляет 1 строку, столбцы заполняются при наличии достаточного числа элементов
    Строка с $sDelim_Row: добавляет столько строк, сколько частей, заполняется только 1 столбец
    Строка с $sDelim_Item и $sDelim_Row: добавляет строки и элементы в зависимости от точек разбивки
    2D-массив с нулевым индексом: добавляет столько строк/столбцов, сколько в массиве

Если в $vValue недостаточно элементов, добавляется пустая строка (""). Лишние элементы игнорируются.

Параметр $iForce может изменить режим вставки или тип данных добавляемых элементов. Может быть установлен следующим образом:
    – $ARRAYFILL_FORCE_DEFAULT (по умолчанию): элементы разбиваются как описано выше; одиночные элементы и элементы массива сохраняют свои типы данных, строки с разделителями добавляются как строки.
    – $ARRAYFILL_FORCE_SINGLEITEM: если $aArray является 1D, $vValue добавляется как один элемент; если $aArray является 2D, параметр игнорируется.
    – $ARRAYFILL_FORCE_INT, $ARRAYFILL_FORCE_NUMBER, $ARRAYFILL_FORCE_PTR, $ARRAYFILL_FORCE_HWND, $ARRAYFILL_FORCE_STRING, $ARRAYFILL_FORCE_BOOLEAN: если $vValue представляет собой одиночный элемент, он приводится к указанному типу данных (по умолчанию сохраняет существующий тип); если $vValue представляет собой строку с разделителями, все элементы приводятся к указанному типу (по умолчанию добавляются как строки; при необходимости разных типов элементы должны передаваться массивом); если $vValue является массивом, параметр игнорируется и элементы добавляются с сохранением существующих типов.

Любое иное значение $iForce игнорируется.

Для $ARRAYFILL_FORCE_BOOLEAN строки интерпретируются следующим образом: "True" и "1" как True, "False", "0" и "" (пустая строка) как False.

Функция НЕ обновляет счётный элемент массива, однако при успехе возвращает новый наибольший индекс строки массива.

См. также

_ArrayConcatenate, _ArrayDelete, _ArrayInsert, _ArrayPop, _ArrayPush

Пример

Пример 1

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray_Base[2] = ["Org Item 0", "Org item 1"]
_ArrayDisplay($aArray_Base, "1D - Base array")

; Добавить один элемент
Local $aArray = $aArray_Base
Local $sSingleFill = "New Item 2"
_ArrayAdd($aArray, $sSingleFill)
_ArrayDisplay($aArray, "1D - Single")

; Добавить строку с разделителями – каждая часть добавляет новый элемент
$aArray = $aArray_Base
Local $sFill = ""
For $i = 1 To 5
    $sFill &= "New Item " & $i + 1 & "|"
Next
$sFill = StringTrimRight($sFill, 1)
MsgBox($MB_SYSTEMMODAL, "Delimited string to add", $sFill)
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "1D - Delim string")

; Добавить 1D массив – каждый элемент добавляет новый элемент
$aArray = $aArray_Base
Local $aFill[5]
For $i = 0 To 4
    $aFill[$i] = "New Item " & $i + 2
Next
_ArrayDisplay($aFill, "Array to add")
_ArrayAdd($aArray, $aFill)
_ArrayDisplay($aArray, "1D - 1D array")

Пример 2

#include <Array.au3>

Local $aArray, $sFill

Local $aArray_Base[2][2] = [["Item 0 - 0", "Item 0 - 1"], ["Item 1 - 0", "Item 1 - 1"]]
_ArrayDisplay($aArray_Base, "2D - Base array")

; Добавить строку с разделителем элементов
$aArray = $aArray_Base
$sFill = "New Item 2 - 0|New Item 2 - 1"
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "2D - Item delimited")

; Добавить строку с разделителем строк – загрузка в столбец 1
$aArray = $aArray_Base
$sFill = "New Item 2 - 1" & @CRLF & "New Item 3 - 1"
_ArrayAdd($aArray, $sFill, 1)
_ArrayDisplay($aArray, "2D - Row Delimited")

; Добавить строку с разделителями элементов и строк
$aArray = $aArray_Base
$sFill = "New Item 2 - 0|New Item 2 - 1" & @CRLF & "New Item 3 - 0|New Item 3 - 1"
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "2D - Item & row Delimited")

; Добавить 2D массив
$aArray = $aArray_Base
Local $aFill[2][2] = [["New Item 2 - 1", "New Item 2 - 2"], ["New Item 3 - 1", "New Item 3 - 2"]]
_ArrayAdd($aArray, $aFill)
_ArrayDisplay($aArray, "2D - 2D Array")

; Добавить 2D массив – один элемент/столбец – загрузка в столбец 1
$aArray = $aArray_Base
Local $aFill[2][1] = [["New Item 2 - 1"], ["New Item 3 - 1"]]
_ArrayAdd($aArray, $aFill, 1)
_ArrayDisplay($aArray, "2D - 2D Array")

Пример 3

#Include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray_Base[2] = ["Org Item 0", "Org item 1"]
Local $aArray
Local $sFill = ""
For $i = 2 To 6
    $sFill &= $i & "|" ; Здесь переменные добавляются с типом Number
Next
$sFill = StringTrimRight($sFill, 1)
MsgBox($MB_SYSTEMMODAL, "Delimited string to add", $sFill)

; Добавить элементы через строку с разделителями
$aArray = $aArray_Base
_ArrayAdd($aArray, $sFill)
_ArrayDisplay($aArray, "Converted to string")
; Но при вставке преобразуются в тип String
MsgBox($MB_SYSTEMMODAL, "Result", "Data:" & @TAB & $aArray[6] & @CRLF & "Datatype:" & @TAB & VarGetType($aArray[6]))

; Добавить элементы через строку с разделителями
$aArray = $aArray_Base
; Теперь принудительно тип Number
_ArrayAdd($aArray, $sFill, Default, Default, Default, $ARRAYFILL_FORCE_NUMBER)
_ArrayDisplay($aArray, "Forced to number")
; Тип данных приводится к нужному
MsgBox($MB_SYSTEMMODAL, "Result", "Data:" & @TAB & $aArray[6] & @CRLF & "Datatype:" & @TAB & VarGetType($aArray[6]))
