Справочник ключевых слов

$tagNMTVGETINFOTIP

Содержит и принимает сведения об элементе TreeView для отображения подсказки.

Global Const $tagNMTVGETINFOTIP = $tagNMHDR & ";ptr Text;int TextMax;hwnd hItem;int lParam"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Text Адрес буфера символов, содержащего текст для отображения.
TextMax Размер буфера в Text в символах. Хотя вы никогда не должны предполагать, что этот буфер будет определённого размера, значение $INFOTIPSIZE можно использовать в целях проектирования.
hItem Дескриптор дерева элемента, для которого отображается подсказка.
lParam Данные, определённые приложением, связанные с элементом, для которого отображается подсказка.
