Справочник функций

_WinAPI_ShellGetSetFolderCustomSettings

Задаёт или получает пользовательские параметры папки.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetSetFolderCustomSettings ( $sFilePath, $iFlag, ByRef $tSHFCS )

Параметры

$sFilePath Путь к папке.
$iFlag Флаг, управляющий действием функции. Это может быть одно из следующих значений.
$FCS_READ
$FCS_FORCEWRITE
$FCS_WRITE
$tSHFCS Структура $tagSHFOLDERCUSTOMSETTINGS, которая предоставляет или получает пользовательские параметры папки. Эта структура
должна быть создана перед вызовом функции.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Эта функция читает из и пишет в Desktop.ini.

Дополнительно

Искать SHGetSetFolderCustomSettings в библиотеке MSDN

Пример

#include <WinAPIDlg.au3>
#include <WinAPIShellEx.au3>

; Выбрать папку
Local $sPath = FileSelectFolder('Select Folder', '', 0, @ScriptDir)
If Not $sPath Then
    Exit
EndIf

; Выбрать значок
Local $aIcon = _WinAPI_PickIconDlg()
If @error Then Exit

; Установить значок выбранной папки
Local $tSHFCS = DllStructCreate($tagSHFOLDERCUSTOMSETTINGS & ';wchar szIconFile[' & (StringLen($aIcon[0]) + 1) & ']')
DllStructSetData($tSHFCS, 'Size', DllStructGetPtr($tSHFCS, 'szIconFile') - DllStructGetPtr($tSHFCS))
DllStructSetData($tSHFCS, 'Mask', $FCSM_ICONFILE)
DllStructSetData($tSHFCS, 'IconFile', DllStructGetPtr($tSHFCS, 'szIconFile'))
DllStructSetData($tSHFCS, 'SizeIF', 260)
DllStructSetData($tSHFCS, 'IconIndex', $aIcon[1])
DllStructSetData($tSHFCS, 'szIconFile', $aIcon[0])

_WinAPI_ShellGetSetFolderCustomSettings($sPath, $FCS_FORCEWRITE, $tSHFCS)
