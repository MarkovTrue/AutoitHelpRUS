_WinAPI_GetThemeFilename

Получает значение свойства имени файла.

#include <WinAPITheme.au3>
_WinAPI_GetThemeFilename ( $hTheme, $iPartID, $iStateID, $iPropID )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит свойство имени файла.
$iStateID Состояние части.
$iPropID Получаемое свойство ($TMT_*).

Возвращаемое значение

Успех: Возвращает полученное имя файла.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
