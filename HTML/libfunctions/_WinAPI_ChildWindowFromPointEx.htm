_WinAPI_ChildWindowFromPointEx

Определяет, какое из дочерних окон указанного родительского окна содержит указанную точку.

#include <WinAPISysWin.au3>
_WinAPI_ChildWindowFromPointEx ( $hWnd, $tPOINT [, $iFlags = $CWP_ALL] )

Параметры

$hWnd Дескриптор родительского окна.
$tPOINT Структура $tagPOINT, которая определяет координаты клиента (относительно родительского окна) проверяемой точки.
$iFlags [необязательный] Флаги, которые указывают, какие дочерние окна пропустить. Этот параметр может быть одним или несколькими из следующих значений:
$CWP_ALL (по умолчанию)
$CWP_SKIPINVISIBLE
$CWP_SKIPDISABLED
$CWP_SKIPTRANSPARENT

Возвращаемое значение

Успех: Возвращает дескриптор первого дочернего окна. Если точка находится в родительском окне, но не в каком-либо дочернем окне, которое соответствует критериям, возвращаемое значение – это дескриптор родительского окна. Если точка находится вне родительского окна, возвращаемое значение 0.
Ошибка: Возвращает 0.

Примечания

Система ведёт внутренний список, содержащий дескрипторы дочерних окон, связанных с родительским окном.
Порядок дескрипторов в списке зависит от порядка Z дочерних окон.
Если несколько дочерних окон содержат указанную точку, система возвращает дескриптор первого окна в списке, которое содержит точку и соответствует критериям, указанным в $iFlags.
