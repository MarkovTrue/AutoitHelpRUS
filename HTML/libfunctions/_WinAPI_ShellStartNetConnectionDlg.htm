Справочник функций

_WinAPI_ShellStartNetConnectionDlg

Отображает диалоговое окно общего просмотра для подключения сетевого ресурса.

#include <WinAPIDlg.au3>
_WinAPI_ShellStartNetConnectionDlg ( [$sRemote = '' [, $iFlags = 0 [, $hParent = 0]]] )

Параметры

$sRemote [необязательный] Имя удалённой сети.
$iFlags [необязательный] Флаги, которые определяют тип ресурса, который диалоговое окно установлено для поиска. Этот параметр может быть одним или несколькими из следующих значений:     $RESOURCETYPE_ANY = 0x00     $RESOURCETYPE_DISK = 0x01     $RESOURCETYPE_PRINT = 0x02 Эти константы определены в "WinNet.au3".
$hParent [необязательный] Дескриптор родительского окна.

Возвращаемое значение

Отсутствует.
