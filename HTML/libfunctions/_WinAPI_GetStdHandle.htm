_WinAPI_GetStdHandle

Получает дескриптор для стандартного устройства ввода, стандартного вывода или стандартной ошибки.

#include <WinAPIHObj.au3>
_WinAPI_GetStdHandle ( $iStdHandle )

Параметры

$iStdHandle Стандартное устройство, для которого должен быть возвращен дескриптор. Может быть одним из следующих значений:
0 – дескриптор стандартного устройства ввода
1 – дескриптор стандартного устройства вывода
2 – дескриптор стандартного устройства ошибок

Возвращаемое значение

Успех: Возвращает дескриптор указанного устройства.
Ошибка: (-1), вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке

Примечания

Дескриптор имеет права доступа GENERIC_READ и GENERIC_WRITE, если приложение не использовало SetStdHandle для установки стандартного дескриптора с меньшими правами доступа.
Если приложение не имеет связанных стандартных дескрипторов, возвращаемое значение равно 0.
