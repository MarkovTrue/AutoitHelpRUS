_GDIPlus_BrushGetSolidColor

Возвращает цвет объекта сплошной кисти.

#include <GDIPlus.au3>
_GDIPlus_BrushGetSolidColor ( $hBrush )

Параметры

$hBrush Дескриптор объекта Brush.

Возвращаемое значение

Успех: Возвращает цвет кисти.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Дополнительно

Искать GdipGetSolidFillColor в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $idLabel1, $idLabel2, $hGraphic, $hBrush1, $iClr1, $iClr2

    ; Создать GUI
    $hGUI = GUICreate("GDI+", 345, 150)
    $idLabel1 = GUICtrlCreateLabel("", 2, 2, 150, 20)
    $idLabel2 = GUICtrlCreateLabel("", 202, 2, 150, 20)
    GUISetState(@SW_SHOW)
    Sleep(100)

    ; Запустить GDIPlus.
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)

    ; Создать сплошную кисть
    $hBrush1 = _GDIPlus_BrushCreateSolid()

    ; Получить цвет сплошной кисти.
    $iClr1 = _GDIPlus_BrushGetSolidColor($hBrush1)

    ; Нарисовать что-то исходным цветом кисти
    _GDIPlus_GraphicsFillEllipse($hGraphic, 25, 25, 100, 100, $hBrush1)

    ; Установить новый цвет кисти (0xFFFF0000 = красный)
    _GDIPlus_BrushSetSolidColor($hBrush1, 0xFFFF0000)

    ; Получить новый цвет сплошной кисти.
    $iClr2 = _GDIPlus_BrushGetSolidColor($hBrush1)

    ; Нарисовать что-то новым цветом кисти
    _GDIPlus_GraphicsFillRect($hGraphic, 220, 25, 100, 100, $hBrush1)

    ; Записать исходный цвет кисти в Label1.
    GUICtrlSetData($idLabel1, "Brush orignal color: " & Hex($iClr1))

    ; Записать новый цвет кисти в Label2.
    GUICtrlSetData($idLabel2, "Brush new color: " & Hex($iClr2))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_BrushDispose($hBrush1)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
