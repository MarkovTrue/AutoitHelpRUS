_WinAPI_IsProcessInJob

Определяет, работает ли процесс в указанной задаче.

#include <WinAPIProc.au3>
_WinAPI_IsProcessInJob ( $hProcess [, $hJob = 0] )

Параметры

$hProcess Дескриптор процесса для проверки. Дескриптор должен иметь право доступа $PROCESS_QUERY_INFORMATION или
$PROCESS_QUERY_LIMITED_INFORMATION.
$hJob [необязательный] Дескриптор задачи. Если этот параметр равен 0 (по умолчанию), функция проверяет, работает ли процесс в какой-либо задаче.

Возвращаемое значение

True: Возвращает процесс работает в задаче.
False: процесс не работает в задаче, вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке.

Примечания

Приложение не может получить дескриптор объекта задачи, в которой оно работает, если только оно не знает имя объекта задачи.
Однако приложение может вызвать функцию _WinAPI_QueryInformationJobObject() с 0 для получения
информации об объекте задачи.
