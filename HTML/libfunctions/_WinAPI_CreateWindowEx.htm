_WinAPI_CreateWindowEx

создаёт наложенное, всплывающее или дочернее окно.

#include <WinAPISysWin.au3>
_WinAPI_CreateWindowEx ( $iExStyle, $sClass, $sName, $iStyle, $iX, $iY, $iWidth, $iHeight, $hParent [, $hMenu = 0 [, $hInstance = 0 [, $pParam = 0]]] )

Параметры

$iExStyle Координаты.
$sClass Имя.
$sName Имя.
$iStyle окно style
$iX Горизонтальная позиция окна.
$iY Вертикальная позиция окна.
$iWidth окно width
$iHeight окно height
$hParent Дескриптор родительского или владеющего окна.
$hMenu Дескриптор окна.
$hInstance Дескриптор.
$pParam Указатель на структуру.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
