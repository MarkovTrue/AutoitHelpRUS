Справочник функций

_GUICtrlIpAddress_SetFocus

Задаёт фокус ввода в указанное поле элемента ввода IP-адреса.

#include <GuiIPAddress.au3>
_GUICtrlIpAddress_SetFocus ( $hWnd, $iIndex )

Параметры

$hWnd Дескриптор элемента управления.
$iIndex Индекс октета, на который нужно установить фокус (отсчёт от 0).
Если значение больше количества октетов, фокус устанавливается на первый пустой октет.

Возвращаемое значение

Отсутствует.

Пример

#include <GUIConstantsEx.au3>
#include <GuiIPAddress.au3>

Example()

Func Example()
    Local $hGui, $hIPAddress

    $hGui = GUICreate("IP Address Control Set Focus Example", 400, 300)
    $hIPAddress = _GUICtrlIpAddress_Create($hGui, 2, 4)
    GUISetState(@SW_SHOW)

    _GUICtrlIpAddress_Set($hIPAddress, "24.168.2.128")

    _GUICtrlIpAddress_SetFocus($hIPAddress, 1)
    ; Ждать закрытия GUI пользователем
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
