Справочник функций

_IENavigate

Переводит существующее окно браузера на указанный URL.

#include <IE.au3>
_IENavigate ( ByRef $oObject, $sUrl [, $iWait = 1] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sUrl Указывает URL, на который переходить.
$iWait [необязательный] Ждать ли загрузку страницы перед возвратом:
0 – вернуться сразу, не ожидая загрузки
1 – (по умолчанию) дождаться полной загрузки и вернуться

Возвращаемое значение

Успех: Нет.
Ошибка: Возвращает 0 или -1 и устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) – общая ошибка
2 ($_IEStatus_COMError) – ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
6 ($_IEStatus_LoadWaitTimeout) – таймаут ожидания загрузки
8 ($_IEStatus_AccessIsDenied) – доступ запрещён
9 ($_IEStatus_ClientDisconnected) – клиент отключён
@extended: Содержит номер недопустимого параметра.

Примечания

Эта функция всегда возвращает -1.
Дело в том, что метод navigate не имеет полезного возвращаемого значения, поэтому из него ничего нельзя вывести.
Для определения успешности перехода нужно использовать другие способы.

Новая система безопасности Windows Vista приводит к созданию нового окна браузера, когда браузеру дают команду перейти на URL в другой зоне безопасности.
То же происходит при первичном создании и переходе через _IECreate(). Новое окно – это новый экземпляр браузера, и прежняя объектная переменная браузера на него уже не указывает.
Обходные пути:
    1) добавить #RequireAdmin в скрипт (нужно даже если учётная запись входит в группу администраторов; при необходимости запросит учётные данные);
    2) использовать _IEAttach() для подключения к новому окну браузера;
    3) добавить целевой сайт в зону "Надёжные узлы" в IE;
    4) выключить "Защищённый режим" в IE;
    5) отключить UAC. Отключение функций безопасности IE при работе с недоверенными сайтами требует понимания возможных последствий.

См. также

_IECreate, _IECreateEmbedded, _IELoadWait

Пример

Пример 1

; Создать окно браузера и перейти на сайт,
; подождать 5 секунд и перейти к другому
; подождать 5 секунд и перейти к другому

#include <IE.au3>

Local $oIE = _IECreate("www.autoitscript.com")
Sleep(5000)
_IENavigate($oIE, "http://www.autoitscript.com/forum/index.php?")
Sleep(5000)
_IENavigate($oIE, "http://www.autoitscript.com/forum/index.php?showforum=9")

Пример 2

; Создать окно браузера и перейти на сайт,
; не ждать полной загрузки страницы перед переходом на следующую строку

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("www.autoitscript.com", 0)
MsgBox($MB_SYSTEMMODAL, "_IENavigate()", "This code executes immediately")
