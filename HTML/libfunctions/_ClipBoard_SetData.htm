Справочник функций

_ClipBoard_SetData

Помещает данные в буфер обмена в указанном формате.

#include <Clipboard.au3>
_ClipBoard_SetData ( $vData [, $iFormat = $CF_TEXT] )

Параметры

$vData Данные в двоичном или строковом формате либо NULL (0) (тогда данные предоставляет владелец, см. ниже).
ВАЖНО: если передана строка не типа $CF_TEXT, $CF_OEMTEXT или $CF_UNICODETEXT, она будет воспринята как ANSI-строка.
Чтобы задать строки Unicode для остальных типов, передавайте данные в двоичном формате.
Также НЕ передавайте $CF_UNICODETEXT в двоичном формате (данные исказятся).
Если $vData равно NULL, окно предоставляет данные в указанном формате по запросу.
Если окно откладывает предоставление данных, оно должно обрабатывать сообщения $WM_RENDERFORMAT и $WM_RENDERALLFORMATS.
При успешном выполнении функции объект, заданный параметром $hMemory, переходит во владение системы.
После передачи владения системе приложению нельзя записывать или освобождать данные, но можно блокировать и читать их до вызова _ClipBoard_Close().
Память нужно разблокировать до закрытия буфера обмена.
Если $hMemory указывает на объект памяти, он должен быть выделен с флагом $GMEM_MOVEABLE.
$iFormat [необязательный] Формат буфера обмена:
$CF_TEXT – текстовый формат (по умолчанию)
$CF_BITMAP – дескриптор растрового изображения (HBITMAP)
$CF_METAFILEPICT – дескриптор рисунка в виде метафайла (METAFILEPICT)
$CF_SYLK – формат Microsoft Symbolic Link (SYLK)
$CF_DIF – формат обмена данными Software Arts' Data Interchange Format
$CF_TIFF – формат TIFF
$CF_OEMTEXT – текст в кодировке OEM
$CF_DIB – структура BITMAPINFO с битами растра
$CF_PALETTE – дескриптор цветовой палитры
$CF_PENDATA – данные для расширений пера (Pen Computing)
$CF_RIFF – аудиоданные в формате RIFF
$CF_WAVE – аудиоданные в формате WAVE
$CF_UNICODETEXT – текст в формате Unicode
$CF_ENHMETAFILE – дескриптор расширенного метафайла (HENHMETAFILE)
$CF_HDROP – дескриптор HDROP со списком файлов
$CF_LOCALE – дескриптор идентификатора языкового стандарта текста в буфере обмена
$CF_DIBV5 – структура BITMAPV5HEADER с цветами и битами растра
$CF_OWNERDISPLAY – формат отображения владельца
$CF_DSPTEXT – текстовый формат отображения для частного формата
$CF_DSPBITMAP – растровый формат отображения для частного формата
$CF_DSPMETAFILEPICT – формат отображения рисунка метафайла для частного формата
$CF_DSPENHMETAFILE – формат отображения расширенного метафайла для частного формата.

Возвращаемое значение

Успех: Возвращает дескриптор данных.
Ошибка: Возвращает 0.

Примечания

Функция делает всё необходимое, чтобы поместить данные в буфер обмена: выделяет глобальный объект памяти, открывает буфер, кладёт в него данные и закрывает. Для более точного управления записью данных используйте _ClipBoard_SetDataEx().

См. также

_ClipBoard_Empty, _ClipBoard_GetData, _ClipBoard_GetDataEx, _ClipBoard_SetDataEx

Пример

#include "Extras\HelpFileInternals.au3"

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $idBtn_SetData, $idBtn_GetData

    ; Создать GUI
    GUICreate("Clipboard", 600, 450)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    $idBtn_SetData = GUICtrlCreateButton("Set ClipBoard Data", 150, 410, 120, 30)
    $idBtn_GetData = GUICtrlCreateButton("Get ClipBoard Data", 300, 410, 120, 30)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtn_SetData
                _ClipBoard_SetData("ClipBoard Library")
            Case $idBtn_GetData
                _MemoWrite(_ClipBoard_GetData())
        EndSwitch
    WEnd
EndFunc   ;==>Example
