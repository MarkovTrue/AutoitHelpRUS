_GDIPlus_ImageGetRawFormat

Возвращает GUID формата файла и название формата изображения.

#include <GDIPlus.au3>
_GDIPlus_ImageGetRawFormat ( $hImage )

Параметры

$hImage Дескриптор объекта Image.

Возвращаемое значение

Успех: Возвращает массив следующего формата:
[0] – строка GUID формата файла
[1] – строка с названием формата изображения
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).
@error: 10 – недопустимый формат.
11 – недопустимый дескриптор изображения.
12 – недопустимый GUID формата файла.
13 – неизвестный GUID формата файла.

См. также

_GDIPlus_ImageGetType

Дополнительно

Искать GdipGetImageRawFormat в библиотеке MSDN

Пример

#include "Extras\HelpFileInternals.au3"

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPIHObj.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hBitmap, $hImage, $aRet

    ; Создать GUI
    GUICreate("GDI+", 600, 400)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Инициализировать библиотеку GDI+
    _GDIPlus_Startup()

    ; Захватить 32-битный bitmap
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; вывести формат изображения для захвата экрана
    $aRet = _GDIPlus_ImageGetRawFormat($hImage)
    _MemoWrite("Image format of screen capture: " & $aRet[1]) ;
    _MemoWrite("GUID: " & $aRet[0]) ;
    _MemoWrite() ;

    ; Сохранить снимок экрана в файл
    _GDIPlus_ImageSaveToFile($hImage, @MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Освободить ресурсы
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Загрузить снимок экрана из файла
    $hImage = _GDIPlus_ImageLoadFromFile(@MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; вывести формат изображения для сохранённого файла
    $aRet = _GDIPlus_ImageGetRawFormat($hImage)
    _MemoWrite("Image format of saved file: " & $aRet[1]) ;
    _MemoWrite("GUID: " & $aRet[0]) ;

    ; Освободить ресурсы
    _GDIPlus_ImageDispose($hImage)

    ; Завершить работу с библиотекой GDI+
    _GDIPlus_Shutdown()

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
