_WinAPI_RegConnectRegistry

Устанавливает подключение к предопределённому ключу реестра на другом компьютере.

#include <WinAPIReg.au3>
_WinAPI_RegConnectRegistry ( $sComputer, $hKey )

Параметры

$sComputer Имя удалённого компьютера. Строка имеет вид "\\computername".
Вызывающий должен иметь доступ к удалённому компьютеру, иначе функция завершится с ошибкой.
Если этот параметр равен 0, используется имя локального компьютера.
$hKey Предопределённый дескриптор реестра. Этот параметр может быть одним из следующих предопределённых ключей на удалённом компьютере:
$HKEY_LOCAL_MACHINE
$HKEY_PERFORMANCE_DATA
$HKEY_USERS

Возвращаемое значение

Успех: Возвращает дескриптор ключа на удалённом компьютере.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Эта функция требует, чтобы на удалённом компьютере работал сервис Remote Registry.

Если текущий пользователь не имеет надлежащего доступа к удалённому компьютеру, вызов функции _WinAPI_RegConnectRegistry() завершится с ошибкой.
Если компьютер присоединён к рабочей группе и политика "Force network logons using local accounts to authenticate as Guest" включена, функция завершится с ошибкой.
Отметим, что эта политика включена по умолчанию, если компьютер присоединён к рабочей группе.

Когда дескриптор, возвращённый функцией _WinAPI_RegConnectRegistry(), больше не требуется,
его следует закрыть вызовом функции _WinAPI_RegCloseKey().
