_WinAPI_BufferedPaintSetAlpha

Устанавливает альфа на указанное значение в заданном прямоугольнике.

#include <WinAPITheme.au3>
_WinAPI_BufferedPaintSetAlpha ( $hBP [, $iAlpha = 255 [, $tRECT = 0]] )

Параметры

$hBP Дескриптор контекста буферизованного рисования.
$iAlpha [необязательный] Значение альфа для установки. Значение альфа может быть в диапазоне от нуля (полностью прозрачный) до 255 (по умолчанию)(полностью непрозрачный).
$tRECT [необязательный] Структура $tagRECT, определяющая прямоугольник, в котором нужно установить альфа.
Если этот параметр равен 0 (по умолчанию), используется весь буфер.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
