Справочник функций

_WinAPI_CreateFont

создаёт логический шрифт с указанными характеристиками.

#include <WinAPIGdi.au3>
_WinAPI_CreateFont ( $iHeight, $iWidth [, $iEscape = 0 [, $iOrientn = 0 [, $iWeight = $FW_NORMAL [, $bItalic = False [, $bUnderline = False [, $bStrikeout = False [, $iCharset = $DEFAULT_CHARSET [, $iOutputPrec = $OUT_DEFAULT_PRECIS [, $iClipPrec = $CLIP_DEFAULT_PRECIS [, $iQuality = $DEFAULT_QUALITY [, $iPitch = $FF_DONTCARE [, $sFace = 'Arial']]]]]]]]]]]] )

Параметры

$iHeight высота шрифта
$iWidth средняя ширина символа
$iEscape Необязательный параметр.
$iOrientn Необязательный параметр.
$iWeight [необязательный] Толщина шрифта. Для удобства определены следующие значения:
$FW_DONTCARE – 0
$FW_THIN – 100
$FW_EXTRALIGHT – 200
$FW_LIGHT – 300
$FW_NORMAL – 400 (по умолчанию)
$FW_MEDIUM – 500
$FW_SEMIBOLD – 600
$FW_BOLD – 700
$FW_EXTRABOLD – 800
$FW_HEAVY – 900
$bItalic Необязательный параметр.
$bUnderline Необязательный параметр.
$bStrikeout Необязательный параметр.
$iCharset [необязательный] Задаёт набор символов. Предопределены следующие значения:
$ANSI_CHARSET – 0
$BALTIC_CHARSET – 186
$CHINESEBIG5_CHARSET – 136
$DEFAULT_CHARSET – 1 (по умолчанию)
$EASTEUROPE_CHARSET – 238
$GB2312_CHARSET – 134
$GREEK_CHARSET – 161
$HANGEUL_CHARSET – 129
$MAC_CHARSET – 77
$OEM_CHARSET – 255
$RUSSIAN_CHARSET – 204
$SHIFTJIS_CHARSET – 128
$SYMBOL_CHARSET – 2
$TURKISH_CHARSET – 162
$VIETNAMESE_CHARSET – 163
$iOutputPrec [необязательный] Задаёт точность вывода. Может быть одним из следующих значений:
$OUT_CHARACTER_PRECIS – не используется
$OUT_DEFAULT_PRECIS – задаёт поведение сопоставителя шрифтов по умолчанию (по умолчанию)
$OUT_DEVICE_PRECIS – указывает сопоставителю выбрать устройственный шрифт, когда в системе несколько шрифтов с одинаковым именем
$OUT_OUTLINE_PRECIS – указывает сопоставителю выбирать из TrueType и других контурных шрифтов
$OUT_PS_ONLY_PRECIS – указывает сопоставителю выбирать только из PostScript-шрифтов.
Если PostScript-шрифтов в системе нет, сопоставитель возвращается к поведению по умолчанию
$OUT_RASTER_PRECIS – указывает сопоставителю выбрать растровый шрифт, когда в системе несколько шрифтов с одинаковым именем
$OUT_STRING_PRECIS – сопоставителем не используется, но возвращается при перечислении растровых шрифтов
$OUT_STROKE_PRECIS – сопоставителем не используется, но возвращается при перечислении TrueType, других контурных и векторных шрифтов
$OUT_TT_ONLY_PRECIS – указывает сопоставителю выбирать только из TrueType-шрифтов. Если их нет в системе, сопоставитель возвращается к поведению по умолчанию
$OUT_TT_PRECIS – указывает сопоставителю выбрать TrueType-шрифт, когда в системе несколько шрифтов с одинаковым именем.
$iClipPrec [необязательный] Задаёт точность отсечения. Может быть одним или несколькими из следующих значений:
$CLIP_CHARACTER_PRECIS – не используется
$CLIP_DEFAULT_PRECIS – задаёт поведение отсечения по умолчанию (по умолчанию)
$CLIP_EMBEDDED – нужно указать этот флаг для использования встроенного шрифта только для чтения
$CLIP_LH_ANGLES – при этом значении поворот всех шрифтов зависит от того, левосторонняя или правосторонняя ориентация системы координат.
Если не используется, устройственные шрифты всегда поворачиваются против часовой стрелки, а поворот остальных зависит от ориентации системы координат.
$CLIP_MASK – не используется
$CLIP_STROKE_PRECIS – сопоставителем не используется, но возвращается при перечислении растровых, векторных или TrueType-шрифтов
Для совместимости это значение всегда возвращается при перечислении шрифтов
$CLIP_TT_ALWAYS – не используется.
$iQuality [необязательный] Задаёт качество вывода. Может быть одним из следующих значений:
$ANTIALIASED_QUALITY – шрифт сглаживается, если он это поддерживает и его размер не слишком мал или велик.
Кроме того, перед использованием в DIBSection нужно выбрать TrueType-шрифт в экранный DC, иначе сглаживание не происходит
$DEFAULT_QUALITY – внешний вид шрифта не важен (по умолчанию)
$DRAFT_QUALITY – внешний вид шрифта менее важен, чем при значении PROOF_QUALITY.
Для растровых шрифтов GDI включается масштабирование, то есть доступно больше размеров, но качество может быть ниже.
Полужирные, курсивные, подчёркнутые и зачёркнутые шрифты при необходимости синтезируются
$NONANTIALIASED_QUALITY – шрифт никогда не сглаживается
$PROOF_QUALITY – качество символов шрифта важнее точного соответствия атрибутам логического шрифта.
Для растровых шрифтов GDI масштабирование отключается и выбирается ближайший по размеру шрифт.
Хотя выбранный размер при PROOF_QUALITY может не соответствовать точно, качество шрифта высокое и нет искажений вида.
Полужирные, курсивные, подчёркнутые и зачёркнутые шрифты при необходимости синтезируются.
$iPitch [необязательный] Задаёт шаг и семейство шрифта. Два младших бита задают шаг шрифта и могут быть одним из значений:
$DEFAULT_PITCH, $FIXED_PITCH, $VARIABLE_PITCH
Четыре старших бита задают семейство шрифта и могут быть одним из значений:
$FF_DECORATIVE – декоративные шрифты. Пример – Old English
$FF_DONTCARE – использовать шрифт по умолчанию (по умолчанию)
$FF_MODERN – шрифты с постоянной шириной штриха, с засечками или без. Примеры – Pica, Elite, Courier New
$FF_ROMAN – шрифты с переменной шириной штриха и с засечками. Пример – MS Serif
$FF_SCRIPT – шрифты, имитирующие рукописный текст. Примеры – Script, Cursive
$FF_SWISS – шрифты с переменной шириной штриха и без засечек. Пример – MS Sans Serif.
$sFace Имя.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный шрифт.
Ошибка: Возвращает 0.

Примечания

Когда шрифт больше не нужен, удалите его вызовом _WinAPI_DeleteObject().

Перечисленные константы требуют #include <FontConstants.au3>

Дополнительно

Искать CreateFont в библиотеке MSDN

Пример

#include <FontConstants.au3>
#include <StructureConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPISysWin.au3>

Global $g_tRECT, $g_hFont, $g_hOldFont, $g_hDC

HotKeySet("{ESC}", "_Exit")

$g_tRECT = DllStructCreate($tagRect)
DllStructSetData($g_tRECT, "Left", 5)
DllStructSetData($g_tRECT, "Top", 5)
DllStructSetData($g_tRECT, "Right", 250)
DllStructSetData($g_tRECT, "Bottom", 50)

$g_hDC = _WinAPI_GetDC(0)
$g_hFont = _WinAPI_CreateFont(50, 0, 0, 0, 400, False, False, False, $DEFAULT_CHARSET, _
        $OUT_DEFAULT_PRECIS, $CLIP_DEFAULT_PRECIS, $DEFAULT_QUALITY, 0, 'Arial')
$g_hOldFont = _WinAPI_SelectObject($g_hDC, $g_hFont)

_WinAPI_SetTextColor($g_hDC, 0x0000FF)
_WinAPI_SetBkColor($g_hDC, 0x000000)
; закомментируйте следующую строку для чёрного фона вместо прозрачного
_WinAPI_SetBkMode($g_hDC, $TRANSPARENT)

While 1
    _WinAPI_DrawText($g_hDC, "Hello world!", $g_tRECT, $DT_CENTER)
    Sleep(100)
WEnd

Func _Exit()
    _WinAPI_SelectObject($g_hDC, $g_hOldFont)
    _WinAPI_DeleteObject($g_hFont)
    _WinAPI_ReleaseDC(0, $g_hDC)
    _WinAPI_InvalidateRect(0, 0)
    $g_tRECT = 0
    Exit
EndFunc   ;==>_Exit
