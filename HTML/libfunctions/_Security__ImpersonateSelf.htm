Справочник функций

_Security__ImpersonateSelf

Получает токен доступа, который имперсонирует контекст безопасности вызывающего процесса.

#include <Security.au3>
_Security__ImpersonateSelf ( [$iLevel = $SECURITYIMPERSONATION] )

Параметры

$iLevel [необязательный] Уровень имперсонации нового токена:
$SECURITYANONYMOUS. Серверный процесс не может получить информацию об идентификации клиента и не может его имперсонировать.
$SECURITYIDENTIFICATION. Серверный процесс может получить информацию о клиенте, такую как идентификаторы безопасности и привилегии, но не может его имперсонировать.
$SECURITYIMPERSONATION. Серверный процесс может имперсонировать контекст безопасности клиента на своей локальной системе. Сервер не может имперсонировать клиента в удалённых системах.
$SECURITYDELEGATION. Серверный процесс может имперсонировать контекст безопасности клиента в удалённых системах.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_Security__OpenThreadTokenEx

Дополнительно

Искать ImpersonateSelf в библиотеке MSDN

Пример

#include <Security.au3>
#include <WinAPIError.au3>
#include <WinAPIHObj.au3>

Local $hToken = _Security__OpenThreadToken($TOKEN_ADJUST_PRIVILEGES)
If $hToken Then
    _WinAPI_CloseHandle($hToken)
Else
    ConsoleWrite("_Security__OpenThreadToken failed with error description: " & _WinAPI_GetLastErrorMessage() & @CRLF)
    ConsoleWrite("New attempt..." & @CRLF)

    ; Прочитать примечания к функции _Security__OpenThreadToken
    _Security__ImpersonateSelf()
    $hToken = _Security__OpenThreadToken($TOKEN_ADJUST_PRIVILEGES)
    If $hToken Then
        ConsoleWrite(">>> SUCCESS, $hToken = " & $hToken & @CRLF)
        _WinAPI_CloseHandle($hToken)
    Else
        ConsoleWrite("!FAILED" & @CRLF)
    EndIf
EndIf
