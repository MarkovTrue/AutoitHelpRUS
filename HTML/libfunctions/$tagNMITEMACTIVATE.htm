Справочник ключевых слов

$tagNMITEMACTIVATE

Отправляется элементом управления ListView при активации элемента пользователем.

Global Const $tagNMITEMACTIVATE = $tagNMHDR & ";int Index;int SubItem;int NewState;int OldState;" & _
"int Changed;int X;int Y;int lParam;int KeyFlags"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Index Индекс элемента представления списка. Если индекс элемента не используется для уведомления, этот член будет содержать -1.
SubItem Индекс подэлемента с отсчётом от единицы. Если индекс подэлемента не используется для уведомления или уведомление не применяется к подэлементу, этот член будет содержать нуль.
NewState Новое состояние элемента. Этот член равен нулю для уведомлений, которые его не используют.
OldState Старое состояние элемента. Этот член равен нулю для уведомлений, которые его не используют.
Changed Набор флагов, указывающих изменённые атрибуты элемента.
Этот член равен нулю для уведомлений, которые его не используют.
Этот член может иметь один или несколько из следующих флагов:
$LVIF_COLUMNS – член Columns действителен
$LVIF_DI_SETITEM – операционная система должна сохранить запрашиваемую информацию о элементе списка
$LVIF_GROUPID – член GroupID действителен
$LVIF_IMAGE – член Image действителен
$LVIF_INDENT – член Indent действителен
$LVIF_NORECOMPUTE – элемент управления не будет создавать LVN_GETDISPINFO для получения текстовой информации
$LVIF_PARAM – член Param действителен
$LVIF_STATE – член State действителен
$LVIF_TEXT – член Text действителен.
X Координата X точки.
Y Координата Y точки.
lParam Определяемое приложением значение элемента. Этот член не определён для уведомлений, которые его не используют.
KeyFlags Клавиши-модификаторы, которые были нажаты в момент активации.
Этот член содержит нуль или комбинацию следующих флагов:
$LVKF_ALT (1) – клавиша нажата.
$LVKF_CONTROL (2) – клавиша нажата.
$LVKF_SHIFT (4) – клавиша нажата.
