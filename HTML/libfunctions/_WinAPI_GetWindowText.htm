_WinAPI_GetWindowText

Получает текст строки заголовка указанного окна.

#include <WinAPISysWin.au3>
_WinAPI_GetWindowText ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

Успех: Возвращает текст строки заголовка окна.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Дополнительно

Искать GetWindowText в библиотеке MSDN

Пример

#include <WinAPISysWin.au3>

_Example()
Func _Example()
    Run("notepad.exe")
    Local $hWnd = WinWait("[CLASS:Notepad]", "")
    ConsoleWrite("! " & _WinAPI_GetWindowText($hWnd) & @CRLF)
EndFunc   ;==>_Example
