_WinAPI_RegisterApplicationRestart

Регистрирует активный экземпляр приложения для перезапуска.

#include <WinAPIDiag.au3>
_WinAPI_RegisterApplicationRestart ( [$iFlags = 0 [, $sCmd = '']] )

Параметры

$iFlags [необязательный] Флаги, определяющие события, при которых приложение не будет перезапущено. Этот параметр может быть
0 или одно или несколько следующих значений.
$RESTART_NO_CRASH
$RESTART_NO_HANG
$RESTART_NO_PATCH
$RESTART_NO_REBOOT
$sCmd [необязательный] Аргументы командной строки для приложения при его перезапуске. Максимальный размер командной
строки, которую можно указать, составляет 2048 символов. Если этот параметр – пустая строка (по умолчанию), то ранее
зарегистрированная командная строка удаляется.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Первоначальная регистрация для перезапуска должна происходить до того, как приложение столкнётся с необработанным исключением
или станет неответчивым. Затем вы можете вызвать эту функцию из функции обратного вызова восстановления, чтобы обновить
командную строку. Чтобы предотвратить циклические перезапуски, система перезапустит приложение только если оно работает
минимум 60 секунд.

Если вы зарегистрировали приложение на перезапуск и оно столкнулось с необработанным исключением или не отвечает,
пользователю предоставляется возможность перезапустить приложение; приложение не перезапускается автоматически
без согласия пользователя.

Эта функция требует Windows Vista или более позднюю версию.

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <StaticConstants.au3>
#include <WinAPIDiag.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

Global $g_iCount = 10

Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 300, 100)
Local $idLabel = GUICtrlCreateLabel('The application will be crashes after ' & $g_iCount & ' seconds.', 10, 43, 280, 14, $SS_CENTER)
GUISetState(@SW_SHOW)

If $CmdLine[0] And ($CmdLine[1] = '/crash') Then
    MsgBox(($MB_ICONWARNING + $MB_SYSTEMMODAL), 'Attention', 'The application has been restarted after an abnormal exit.', 0, $hForm)
EndIf

If Not @Compiled Then
    _WinAPI_RegisterApplicationRestart(BitOR($RESTART_NO_PATCH, $RESTART_NO_REBOOT), '"' & @ScriptFullPath & '" /crash')
Else
    _WinAPI_RegisterApplicationRestart(BitOR($RESTART_NO_PATCH, $RESTART_NO_REBOOT), '/crash')
EndIf

AdlibRegister('_Countdown', 1000)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE

Func _Countdown()
    Local $iData
    #forceref $iData

    $g_iCount -= 1
    If $g_iCount Then
        GUICtrlSetData($idLabel, 'The application will be crashes after ' & $g_iCount & ' seconds.')
    Else
        Local $tData
        ; Принудительный сбой скрипта из-за нарушения доступа к памяти
        $tData = DllStructCreate('int', 0x12345678)
        $iData = DllStructGetData($tData, 1)
    EndIf
EndFunc   ;==>_Countdown
