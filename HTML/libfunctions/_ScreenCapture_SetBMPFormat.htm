_ScreenCapture_SetBMPFormat

Задаёт битовую глубину для снимков экрана в формате BMP.

#include <ScreenCapture.au3>
_ScreenCapture_SetBMPFormat ( $iFormat )

Параметры

$iFormat Глубина цвета изображения (бит/пикс, bpp):
0 = 16 bpp; 5 бит на каждую RGB-компоненту
1 = 24 bpp; 8 бит на каждую RGB-компоненту
2 = 32 bpp; 8 бит на каждую RGB-компоненту

Возвращаемое значение

Отсутствует.

Примечания

Если значение не задано, по умолчанию используется 24 bpp.

Пример

#include <ScreenCapture.au3>

Example()

Func Example()
    ; Захватить весь экран
    _ScreenCapture_SetBMPFormat(0)
    _ScreenCapture_Capture(@MyDocumentsDir & "\GDIPlus_Image.bmp")

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.bmp")
EndFunc   ;==>Example
