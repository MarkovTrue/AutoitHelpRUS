_WinAPI_MirrorIcon

Переворачивает (отражает) значки так, чтобы они правильно отображались в отражённом контексте устройства.

#include <WinAPIIcons.au3>
_WinAPI_MirrorIcon ( $hIcon [, $bDelete = False] )

Параметры

$hIcon Дескриптор значка.
$bDelete [необязательный] Указывает, нужно ли удалить значок после успешного выполнения функции, допустимые значения:
True – значок будет удалён если функция успешна.
False – не удалять, вы должны освободить значок когда закончите его использовать (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор созданного значка.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Когда значок больше не нужен, уничтожьте его функцией _WinAPI_DestroyIcon().

См. также

_WinAPI_DestroyIcon

Дополнительно

Искать MirrorIcon в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <StaticConstants.au3>
#include <WinAPIIcons.au3>
#include <WinAPIShellEx.au3>

Local $hIcon = _WinAPI_ShellExtractIcon(@SystemDir & '\shell32.dll', 4, 32, 32)

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 156, 108)
GUICtrlCreateIcon('', 0, 30, 38, 32, 32)
GUICtrlSendMsg(-1, $STM_SETIMAGE, 1, $hIcon)
GUICtrlCreateIcon('', 0, 88, 38, 32, 32)
GUICtrlSendMsg(-1, $STM_SETIMAGE, 1, _WinAPI_MirrorIcon($hIcon))
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
