_WinAPI_PathGetCharType

Определяет тип символа в отношении пути.

#include <WinAPIShPath.au3>
_WinAPI_PathGetCharType ( $sChar )

Параметры

$sChar Символ, для которого необходимо определить тип.

Возвращаемое значение

Возвращает одно или несколько следующих значений, которые определяют тип символа.
    $GCT_INVALID
    $GCT_LFNCHAR
    $GCT_SEPARATOR
    $GCT_SHORTCHAR
    $GCT_WILD
