_WinAPI_GetWorldTransform

Получает текущее преобразование мирового пространства в пространство страницы.

#include <WinAPIGdi.au3>
_WinAPI_GetWorldTransform ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает структура $tagXFORM, содержащая текущее преобразование мировых координат в координаты страницы.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Точность преобразования может быть изменена, если приложение вызывает _WinAPI_ModifyWorldTransform()
перед вызовом _WinAPI_GetWorldTransform(). Это потому, что внутренний формат для хранения значений преобразования
использует более высокую точность, чем значение FLOAT.
