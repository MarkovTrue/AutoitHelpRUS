_GUICtrlMenu_GetItemHighlighted

Получает состояние выделения пункта меню.

#include <GuiMenu.au3>
_GUICtrlMenu_GetItemHighlighted ( $hMenu, $iItem [, $bByPos = True] )

Параметры

$hMenu Дескриптор меню.
$iItem Идентификатор или позиция пункта меню.
$bByPos [необязательный] Флаг идентификатора меню:
True – $iItem задаёт позицию элемента (начиная с нуля)
False – $iItem задаёт идентификатор элемента меню.

Возвращаемое значение

True: $iItem выделен.
False: $iItem не выделен.

См. также

_GUICtrlMenu_SetItemHighlighted

Пример

#include "Extras\HelpFileInternals.au3"

#include <GuiMenu.au3>

Global $g_hWnd

Example()

Func Example()
    Local $hMain, $hFile

    ; Открыть Блокнот
    Run("notepad.exe")
    $g_hWnd = WinWaitActive("[CLASS:Notepad]")
    $hMain = _GUICtrlMenu_GetMenu($g_hWnd)
    $hFile = _GUICtrlMenu_GetItemSubMenu($hMain, 0)

    ; Получить/установить подсветку Open
    Writeln("Open is highlighted: " & _GUICtrlMenu_GetItemHighlighted($hFile, 1))
    _GUICtrlMenu_SetItemHighlighted($hFile, 1)
    Writeln("Open is highlighted: " & _GUICtrlMenu_GetItemHighlighted($hFile, 1))

    _NotepadForceClose($g_hWnd)
EndFunc   ;==>Example

; Записать строку текста в Блокнот
Func Writeln($sText, $hWnd = $g_hWnd)
    ControlSend($hWnd, "", ControlGetFocus($hWnd), $sText & @CRLF)
EndFunc   ;==>Writeln
