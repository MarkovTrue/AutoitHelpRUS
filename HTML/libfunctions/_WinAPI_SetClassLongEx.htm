Справочник функций

_WinAPI_SetClassLongEx

Заменяет указанное значение в указанном окне.

#include <WinAPISysWin.au3>
_WinAPI_SetClassLongEx ( $hWnd, $iIndex, $iNewLong )

Параметры

$hWnd Дескриптор окна.
$iIndex Значение для замены. Этот параметр может быть одним из следующих значений.
$GCL_CBCLSEXTRA
$GCL_CBWNDEXTRA
$GCL_HBRBACKGROUND
$GCL_HCURSOR
$GCL_HICON
$GCL_HICONSM
$GCL_HMODULE
$GCL_MENUNAME
$GCL_STYLE
$GCL_WNDPROC
$iNewLong Значение для замены.

Возвращаемое значение

Успех: Возвращает предыдущее значение.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Если это не было установлено ранее, возвращаемое значение это ноль.

Дополнительно

Искать SetClassLong в библиотеке MSDN

Пример

#NoTrayIcon

#include <GUIConstantsEx.au3>
#include <WinAPISysWin.au3>
#include <WindowsStylesConstants.au3>

Local $hParent = GUICreate('', 0, 0, 0, 0, 0, $WS_EX_TOOLWINDOW)
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400, -1, -1, BitOR($WS_CAPTION, $WS_POPUP, $WS_SYSMENU), BitOR($WS_EX_DLGMODALFRAME, $WS_EX_TOPMOST), $hParent)

; Удалить значок окна
_WinAPI_SetClassLongEx($hForm, $GCL_HICONSM, 0)
_WinAPI_SetClassLongEx($hForm, $GCL_HICON, 0)

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
