_WinAPI_GetTextMetrics

Получает основную информацию о текущем выбранном шрифте.

#include <WinAPIGdi.au3>
_WinAPI_GetTextMetrics ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает структура $tagTEXTMETRIC, которая содержит информацию о текущем выбранном шрифте.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

См. также
