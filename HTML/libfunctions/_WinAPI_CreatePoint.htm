Справочник функций

_WinAPI_CreatePoint

создаёт структуру $tagPOINT с координатами x и y указанной точки.

#include <WinAPIMisc.au3>
_WinAPI_CreatePoint ( $iX, $iY )

Параметры

$iX X-координата точки.
$iY Y-координата точки.

Возвращаемое значение

Возвращает структуру $tagPOINT с указанной точкой.
