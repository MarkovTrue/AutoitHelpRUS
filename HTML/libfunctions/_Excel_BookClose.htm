Справочник функций

_Excel_BookClose

Закрывает указанную книгу.

#include <Excel.au3>
_Excel_BookClose ( $oWorkbook [, $bSave = True] )

Параметры

$oWorkbook Объект книги.
$bSave [необязательный] True – сохранить книгу перед закрытием (по умолчанию True).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом
2 – $bSave не является булевым значением
3 – Excel вернул ошибку.
@extended: Содержит код COM-ошибки.

См. также

_Excel_BookAttach, _Excel_BookNew, _Excel_BookOpen, _Excel_BookOpenText

Пример

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookClose Example 1", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Сохраняем книгу перед закрытием.
; Создать новую рабочую книгу
Local $oWorkbook = _Excel_BookNew($oExcel)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookClose Example 1", "Error creating new workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Закрываем книгу и сохраняем, если были изменения.
_Excel_RangeWrite($oWorkbook, Default, "Test", "A1")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookClose Example 1", "Error writing to cell 'A1'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox(64, "Excel UDF: _Excel_BookClose Example 1", "Click OK to close the workbook without saving.")
; Закрываем книгу (по умолчанию сохраняется).
_Excel_BookClose($oWorkbook, False)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookClose Example 1", "Error closing workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookClose Example 1", "Workbook has been successfully closed.")
