Справочник функций

_IELoadWaitTimeout

Возвращает или задаёт текущий таймаут (в миллисекундах), который _IELoadWait() будет ждать перед прерыванием.

#include <IE.au3>
_IELoadWaitTimeout ( [$iTimeout = -1] )

Параметры

$iTimeout [необязательный] получить или задать число миллисекунд
0 или положительное целое – устанавливает таймаут в это значение
-1 = (по умолчанию) возвращает текущее значение таймаута (хранится в глобальной переменной $__IELoadWaitTimeout)

Возвращаемое значение

Возвращает текущее значение таймаута, если $iTimeout = -1; иначе возвращает 1.

Примечания

Полностью отключить таймаут нельзя.
Функционально это можно сделать, задав очень большое положительное число.
Исходное значение таймаута – 30000 мс (5 минут).

См. также

_IELoadWait

Пример

Пример 1

; Получить текущее значение таймаута

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $iCurrentTimeout = _IELoadWaitTimeout()
MsgBox($MB_SYSTEMMODAL, "_IELoadWaitTimeout", $iCurrentTimeout / 1000 & " sec")

Пример 2

; Set timeout to 1 minute (60000 milliseconds)

#include <IE.au3>
#include <MsgBoxConstants.au3>

_IELoadWaitTimeout(60000)

Local $iCurrentTimeout = _IELoadWaitTimeout()
MsgBox($MB_SYSTEMMODAL, "_IELoadWaitTimeout", $iCurrentTimeout / 1000 & " sec")
