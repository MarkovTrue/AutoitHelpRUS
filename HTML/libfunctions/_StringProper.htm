Справочник функций

_StringProper

Преобразует строку в формат правильного регистра, как функция =Proper в Excel.

#include <String.au3>
_StringProper ( $sString )

Параметры

$sString Входная строка.

Возвращаемое значение

Успех: Возвращает строка в формате правильного регистра.
Ошибка: Возвращает пустую строку.

Примечания

Функция переводит в верхний регистр каждый символ, следующий за не буквенным символом.

См. также

StringLower, StringUpper

Пример

#include <MsgBoxConstants.au3>
#include <String.au3>

; Вернёт: Somebody Lastnames
MsgBox($MB_SYSTEMMODAL, '', _StringProper("somebody lastnames"))
; Вернёт: Somebody Lastnames
MsgBox($MB_SYSTEMMODAL, '', _StringProper("SOME.BODY LAST(NAME)"))
Exit
