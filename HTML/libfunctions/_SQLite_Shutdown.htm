Справочник функций

_SQLite_Shutdown

Выгружает SQLite.dll.

#include <SQLite.au3>
_SQLite_Shutdown ( )

Возвращаемое значение

Отсутствует.

Примечания

Если _SQLite_Startup() завершилась ошибкой или не выполнена, будет выброшена ошибка выполнения AutoIt и скрипт завершится.
AutoIt автоматически закрывает все загруженные DLL, но вызов _SQLite_Shutdown() по-прежнему хорошая идея.

См. также

_SQLite_Startup

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Shutdown()
