Справочник функций

_StringInsert

Вставляет строку в другую строку.

#include <String.au3>
_StringInsert ( $sString, $sInsertion, $iPosition )

Параметры

$sString Исходная строка.
$sInsertion Строка для вставки.
$iPosition Позиция для вставки строки (отрицательные значения вставляют справа).

Возвращаемое значение

Успех: Возвращает новая изменённая строка.
Ошибка: Исходная строка и устанавливает флаг @error в ненулевое значение.
@error: 1 – Неверная позиция.

Примечания

Используйте отрицательные значения позиции для вставки строки справа.

Пример

#include <MsgBoxConstants.au3>
#include <String.au3>

Example()

Func Example()
    ; Переменная для хранения вывода
    Local $sOutput = ""

    ; Вставляет три "движущихся" подчёркивания и выводит их в консоль.
    For $i = -20 To 20
        $sOutput &= $i & @TAB & _StringInsert("Supercalifragilistic", "___", $i) & @CRLF
    Next

    ; Вывести результат
    MsgBox($MB_SYSTEMMODAL, "", $sOutput)
EndFunc   ;==>Example
