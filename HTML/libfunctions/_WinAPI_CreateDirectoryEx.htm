Справочник функций

_WinAPI_CreateDirectoryEx

создаёт новый каталог с атрибутами шаблонного каталога.

#include <WinAPIFiles.au3>
_WinAPI_CreateDirectoryEx ( $sNewDir, $sTemplateDir [, $tSecurity = 0] )

Параметры

$sNewDir Путь к создаваемому каталогу.
$sTemplateDir Путь к каталогу, используемому как шаблон при создании нового каталога.
$tSecurity Необязательный параметр.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

_WinAPI_CreateDirectoryEx() позволяет создавать каталоги, наследующие потоковую информацию от других каталогов.
Это полезно, например, при работе с каталогами Macintosh, у которых есть потоки ресурсов, необходимые для распознавания файлов как ресурсных.
