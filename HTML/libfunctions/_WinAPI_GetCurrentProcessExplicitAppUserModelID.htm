Справочник функций

_WinAPI_GetCurrentProcessExplicitAppUserModelID

Возвращает явный AppUserModelID, заданный приложением для текущего процесса.

#include <WinAPIProc.au3>
_WinAPI_GetCurrentProcessExplicitAppUserModelID ( )

Возвращаемое значение

Успех: Возвращает строка с Application User Model ID (AppUserModelID).
Ошибка: Возвращает пустую строку и устанавливает @error в ненулевое значение.
@extended: Может содержать системный код ошибки.

Примечания

Можно получить только тот AppUserModelID, который был явно задан через _WinAPI_SetCurrentProcessExplicitAppUserModelID().
AppUserModelID, назначенные системой по умолчанию, прочитать нельзя.

Требуется Windows 7 и выше.
