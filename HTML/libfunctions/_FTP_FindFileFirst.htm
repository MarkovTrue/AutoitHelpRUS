Справочник функций

_FTP_FindFileFirst

Находит первый файл на FTP-сервере.

#include <FTPEx.au3>
_FTP_FindFileFirst ( $hFTPSession, $sRemotePath, ByRef $hFTPFind [, $iFlags = 0 [, $iContext = 0]] )

Параметры

$hFTPSession возвращается функцией _FTP_Connect().
$sRemotePath Путь, используемый при поиске файла.
$hFTPFind Возвращает дескриптор для использования в _FTP_FindFileNext() или _FTP_FindFileClose().
$iFlags [необязательный] $iFlags может быть комбинацией:
$INTERNET_FLAG_HYPERLINK (0x00000400)
$INTERNET_FLAG_NEED_FILE (0x00000010)
$INTERNET_FLAG_NO_CACHE_WRITE (0x04000000)
$INTERNET_FLAG_RELOAD (0x80000000)
$INTERNET_FLAG_RESYNCHRONIZE (0x00000800).
$iContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает массив:
$aArray[0] – количество элементов
$aArray[1] – атрибуты файла
$aArray[2] – младшая часть времени создания
$aArray[3] – старшая часть времени создания
$aArray[4] – младшая часть времени доступа
$aArray[5] – старшая часть времени доступа
$aArray[6] – младшая часть времени последней записи
$aArray[7] – старшая часть времени последней записи
$aArray[8] – старшая часть размера файла
$aArray[9] – младшая часть размера файла
$aArray[10] – имя файла
$aArray[11] – альтернативное имя
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

См. также

Пример

#include <FTPEx.au3>

;~ Local $sServer = 'ftp.cs.brown.edu' ; Brown Computer Science
Local $sServer = 'speedtest.tele2.net' ; Сервис Tele2 Speedtest
Local $sUsername = ''
Local $sPass = ''

Local $hOpen = _FTP_Open('MyFTP Control')
Local $hConn = _FTP_Connect($hOpen, $sServer, $sUsername, $sPass)

Local $h_Handle
;~ Local $aFile = _FTP_FindFileFirst($hConn, "/pub/papers/graphics/research/", $h_Handle)
Local $aFile = _FTP_FindFileFirst($hConn, "/", $h_Handle)
ConsoleWrite('$sFileName = ' & $aFile[10] & ' attribute = ' & $aFile[1] & ' -> Error code: ' & @error & @CRLF)

Local $iFindClose = _FTP_FindFileClose($h_Handle)

Local $iFtpc = _FTP_Close($hConn)
Local $iFtpo = _FTP_Close($hOpen)
