Справочник функций

_WinAPI_ShellUserAuthenticationDlgEx

Создаёт и отображает настраиваемое диалоговое окно, которое принимает информацию учётных данных от пользователя.

#include <WinAPIDlg.au3>
_WinAPI_ShellUserAuthenticationDlgEx ( $sCaption, $sMessage, $sUser, $sPassword [, $iFlags = 0 [, $iAuthError = 0 [, $bSave = False [, $iPackage = 0 [, $hParent = 0]]]]] )

Параметры

$sCaption Заголовок диалогового окна.
$sMessage Краткое сообщение для отображения в диалоговом окне.
$sUser Имя пользователя для заполнения полей учётных данных в диалоговом окне. Для пользователей домена строка должна быть в следующем формате:
DomainName\UserName.
$sPassword Начальный пароль.
$iFlags [необязательный] Флаги, которые определяют поведение этой функции. Это может быть комбинация следующих значений. $CREDUIWIN_AUTHPACKAGE_ONLY $CREDUIWIN_CHECKBOX $CREDUIWIN_ENUMERATE_ADMINS $CREDUIWIN_ENUMERATE_CURRENT_USER $CREDUIWIN_GENERIC $CREDUIWIN_IN_CRED_ONLY $CREDUIWIN_SECURE_PROMPT $CREDUIWIN_PACK_32_WOW $CREDUIWIN_PREPROMPTING
$iAuthError [необязательный] Код ошибки системы, который отображается в диалоговом окне.
$bSave [необязательный] Указывает, установлен ли флажок "Сохранить" в диалоговом окне (имеет смысл только если используется флаг $CREDUIWIN_CHECKBOX), допустимые значения: True – отмечено. False – не отмечено (по умолчанию).
$iPackage [необязательный] Пакет аутентификации, для которого сериализуются учётные данные.
$hParent [необязательный] Диалоговое окно является модальным по отношению к родительскому окну. Если этот параметр равен 0 (по умолчанию), рабочий стол является родительским окном диалогового окна.

Возвращаемое значение

Успех: Возвращает массив, содержащий следующую информацию:
$aArray[0] – имя пользователя, включая имя домена (если необходимо).
$aArray[1] – пароль.
$aArray[2] – состояние флажка "Сохранить".
$aArray[3] – пакет аутентификации.
Ошибка: Устанавливает @error ≠ 0. Если функция отменена пользователем, флаг @extended будет содержать код ошибки системы ERROR_CANCELLED (1223). Любое другое значение указывает, что функция не смогла загрузиться.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

Дополнительно

Искать CredUIPromptForWindowsCredentials в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIDlg.au3>
#include <WinAPIShPath.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

Local $aData = _WinAPI_ShellUserAuthenticationDlgEx('Authentication', 'To continue, type a login and password, and then click OK.', '', '', BitOR($CREDUIWIN_ENUMERATE_CURRENT_USER, $CREDUIWIN_CHECKBOX))
If @error Then Exit

Local $aUser = _WinAPI_ParseUserName($aData[0])
If @error Then
    Exit
EndIf

ConsoleWrite('Domain: ' & $aUser[0] & @CRLF)
ConsoleWrite('User: ' & $aUser[1] & @CRLF)
ConsoleWrite('Password: ' & $aData[1] & @CRLF)
ConsoleWrite('Save: ' & $aData[2] & @CRLF)
ConsoleWrite('Package: ' & $aData[3] & @CRLF)
