Справочник функций

_GUICtrlIpAddress_ShowHide

Показывает или скрывает элемент управления IP-адрес.

#include <GuiIPAddress.au3>
_GUICtrlIpAddress_ShowHide ( $hWnd, $iState )

Параметры

$hWnd Дескриптор элемента управления.
$iState Состояние элемента управления IP-адрес, может быть одним из следующих значений: @SW_HIDE или @SW_SHOW.

Возвращаемое значение

True: Возвращает ненулевое значение, если элемент управления был виден до вызова.
False: элемент управления был ранее скрыт.

Пример

#include <GUIConstantsEx.au3>
#include <GuiIPAddress.au3>

Example()

Func Example()
    Local $hGui, $hIPAddress

    $hGui = GUICreate("IP Address Control Show/Hide Example", 300, 150)
    $hIPAddress = _GUICtrlIpAddress_Create($hGui, 10, 10)
    GUISetState(@SW_SHOW)

    _GUICtrlIpAddress_Set($hIPAddress, "24.168.2.128")

    Sleep(1000)
    _GUICtrlIpAddress_ShowHide($hIPAddress, @SW_HIDE)
    Sleep(1000)
    _GUICtrlIpAddress_ShowHide($hIPAddress, @SW_SHOW)

    ; Ждать закрытия GUI пользователем
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
