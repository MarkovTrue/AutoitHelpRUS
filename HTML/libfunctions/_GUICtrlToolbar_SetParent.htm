_GUICtrlToolbar_SetParent

Задаёт окно, в которое элемент отправляет уведомления.

#include <GuiToolbar.au3>
_GUICtrlToolbar_SetParent ( $hWnd, $hParent )

Параметры

$hWnd Дескриптор элемента управления.
$hParent Дескриптор родительского окна.

Возвращаемое значение

Успех: Возвращает дескриптор предыдущего окна-приёмника уведомлений.
Ошибка: Возвращает 0.

Примечания

Сообщение $TB_SETPARENT не меняет родительское окно, заданное при создании элемента.
Вызов _WinAPI_GetParent() вернёт реальное родительское окно, а не указанное в $TB_SETPARENT.
Чтобы сменить родителя элемента, используйте _WinAPI_SetParent().
