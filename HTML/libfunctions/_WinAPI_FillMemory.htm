Справочник функций

_WinAPI_FillMemory

Заполняет блок памяти заданным значением.

#include <WinAPIMem.au3>
_WinAPI_FillMemory ( $pMemory, $iLength [, $iValue = 0] )

Параметры

$pMemory Указатель на начальный адрес заполняемой памяти.
$iLength Задаёт количество байт для заполнения.
$iValue [необязательный] Значение для заполнения памяти. По умолчанию: 0.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Дополнительно

Искать RtlFillMemory в библиотеке MSDN

Пример

#include <WinAPIMem.au3>

Local $tStruct = DllStructCreate('byte[8]')
ConsoleWrite(DllStructGetData($tStruct, 1) & @CRLF)

_WinAPI_FillMemory($tStruct, 8, 0xAB)
ConsoleWrite(DllStructGetData($tStruct, 1) & @CRLF)

_WinAPI_ZeroMemory($tStruct, 8)
ConsoleWrite(DllStructGetData($tStruct, 1) & @CRLF)
