Справочник функций

_WinAPI_DwmGetColorizationParameters

Получает параметры окраски, используемые Диспетчером окон рабочего стола (DWM).

#include <WinAPIGdi.au3>
_WinAPI_DwmGetColorizationParameters ( )

Возвращаемое значение

Успех: Возвращает структуру $tagDWM_COLORIZATION_PARAMETERS с параметрами окрашивания.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
