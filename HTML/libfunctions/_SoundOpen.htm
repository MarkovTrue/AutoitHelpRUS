Справочник функций

_SoundOpen

Открывает звуковой файл для использования с другими функциями _Sound.

#include <Sound.au3>
_SoundOpen ( $sFilePath )

Параметры

$sFilePath Путь к звуковому файлу.

Возвращаемое значение

Успех: Возвращает массив из 4 элементов, содержащий следующую информацию:
$aArray[0] = Строка идентификатора звукового файла
$aArray[1] = Коэффициент коррекции длины VBR
$aArray[2] = Коэффициент коррекции времени VBR
$aArray[3] = Маркер идентификатора звука
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – Ошибка открытия MCI
2 – Файл не существует
@extended: Целое число, отличное от 0 = Ошибка MCI.

Примечания

Хотя многие функции _Sound...() принимают имя файла в качестве параметра, если возможно, что файл закодирован с переменной скоростью передачи (VBR), настоятельно рекомендуется сначала использовать _SoundOpen(), а затем использовать возвращённый массив как параметр идентификатора в других функциях _Sound...().Это связано с тем, что DLL MCI предполагает, что все файлы закодированы с постоянной скоростью передачи (CBR) и дают неправильные результаты из функций _Sound...(), имеющих дело с длительностью и позицией, когда звуковой файл был закодирован с переменной скоростью передачи (VBR). При использовании массива как параметра ID с функциями _Sound...(), которые изменяют коэффициенты коррекции VBR (_SoundStop()), массив ID будет обновлён при необходимости.
Отдельные элементы возвращённого массива не имеют служебного использования вне функций _Sound, и их внутреннее использование прозрачно для пользователя.

См. также

_SoundClose, _SoundLength, _SoundPause, _SoundPlay, _SoundPos, _SoundResume, _SoundStatus, _SoundStop

Пример

#include <MsgBoxConstants.au3>
#include <Sound.au3>

Local $aSound = _SoundOpen(@WindowsDir & "\media\tada.wav")
If @error = 2 Then
    MsgBox($MB_SYSTEMMODAL, "Error", "The file does not exist")
    Exit
ElseIf @extended <> 0 Then
    Local $iExtended = @extended ; Присвоить, так как @extended устанавливается после DllStructCreate().
    Local $tText = DllStructCreate("char[128]")
    DllCall("winmm.dll", "short", "mciGetErrorStringA", "str", $iExtended, "struct*", $tText, "int", 128)
    MsgBox($MB_SYSTEMMODAL, "Error", "The open failed." & @CRLF & "Error Number: " & $iExtended & @CRLF & "Error Description: " & DllStructGetData($tText, 1) & @CRLF & "Please Note: The sound may still play correctly.")
Else
    MsgBox($MB_SYSTEMMODAL, "Success", "The file opened successfully")
EndIf

_SoundClose($aSound)
