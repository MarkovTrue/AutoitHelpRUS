Справочник функций

_FileReadToArray

Читает указанный файл в одномерный или двумерный массив.

#include <File.au3>
_FileReadToArray ( $sFilePath, ByRef $vReturn [, $iFlags = $FRTA_COUNT [, $sDelimiter = ""]] )

Параметры

$sFilePath Путь и имя читаемого файла.
$vReturn Переменная для возвращаемых данных, массивом быть не обязана.
$iFlags [необязательный] Складывайте нужные значения
$FRTA_NOCOUNT (0) – массив с нулевым индексом, размер через UBound()
$FRTA_COUNT (1) – количество элементов массива в первом элементе (по умолчанию)
$FRTA_INTARRAYS (2) – создать "массив массивов" – см. раздел Примечания
$FRTA_ENTIRESPLIT (4) – использовать всю строку-разделитель как точку разбиения (по умолчанию каждый символ задаёт точку разбиения).
$sDelimiter [необязательный]
Дополнительно разбивает каждую строку файла – например, для чтения CSV-файлов в двумерный массив.

Возвращаемое значение

Успех: Возвращает 1 и $vReturn содержит массив.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0, $vReturn = 0.
@error: 1 – ошибка открытия указанного файла
2 – не удалось разбить файл
3 – строки файла содержат разное число полей (только если флаг $FRTA_INTARRAYS не задан)
4 – разделители не найдены (только если флаг $FRTA_INTARRAYS не задан)

Примечания

Если разделитель не задан, функция возвращает одномерный массив, где каждый элемент содержит строку файла – окончания строк могут быть любым сочетанием @CR, @LF и @CRLF.

Если разделитель задан, функция пытается дополнительно разбить каждую строку файла – способ зависит от флага $FRTA_INTARRAYS. Если флаг не задан и в каждой строке одинаковое число полей, разделённых разделителем, создаётся двумерный массив, но если это не так, @error устанавливается в 3 и массив не возвращается. Если флаг $FRTA_INTARRAYS задан, функция создаёт одномерный массив, где каждый элемент – вложенный массив с полями строки, разбитыми по разделителю, при этом строки могут иметь разное число полей. См. пример ниже.

Если разделитель длиннее одного символа, способ разбиения определяется флагом $FRTA_ENTIRESPLIT.

См. также

_FileWriteFromArray

Пример

Пример 1

#include <Array.au3>
#include <File.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aRetArray, $sFilePath = @TempDir & "\Test.txt"

    ; Создать одномерный массив.
    Local $aArray[] = ["0", "1", "2", "3", "4"]
    ; Записать в файл
    _FileWriteFromArray($sFilePath, $aArray, Default, Default, @CRLF)
    Sleep(1000)

    ; Перечитать, с подсчётом
    _FileReadToArray($sFilePath, $aRetArray)
    _ArrayDisplay($aRetArray, "1D array - count", Default, 8)

    ; Перечитать, без подсчёта
    _FileReadToArray($sFilePath, $aRetArray, $FRTA_NOCOUNT)
    _ArrayDisplay($aRetArray, "1D array - no count", Default, 8)

    ; ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    ; Создать "квадратный" двумерный массив.
    Local $aArray[][] = [ _
            ["00", "01", "02", "03"], _
            ["10", "11", "12", "13"], _
            ["20", "21", "22", "23"], _
            ["30", "31", "32", "33"]]
    _ArrayDisplay($aArray, "Original", Default, 8)
    ; Записать в файл
    _FileWriteFromArray($sFilePath, $aArray, Default, Default, ",")
    Sleep(1000)

    ; Перечитать, с подсчётом
    _FileReadToArray($sFilePath, $aRetArray, Default, ",")
    _ArrayDisplay($aRetArray, "2D array - count", Default, 8)

    ; Перечитать, без подсчёта
    _FileReadToArray($sFilePath, $aRetArray, $FRTA_NOCOUNT, ",")
    _ArrayDisplay($aRetArray, "2D array - no count", Default, 8)

    ; Прочитать в "массив массивов" со счётчиком.
    _FileReadToArray($sFilePath, $aRetArray, $FRTA_COUNT + $FRTA_INTARRAYS, ",")
    _ArrayDisplay($aRetArray, "Array of arrays - count", Default, 8)
    ; Заглянуть внутрь вложенных массивов.
    _ArrayDisplay($aRetArray[1], "Array 1 inside RetArray - count", Default, 8)

    ; ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    ; Перезаписать двумерный массив с несколькими разделителями.
    _FileWriteFromArray($sFilePath, $aArray, Default, Default, ":|")
    Sleep(1000)

    ; Перечитать – каждый символ разделителя является точкой разбиения.
    _FileReadToArray($sFilePath, $aRetArray, $FRTA_NOCOUNT, ":|")
    _ArrayDisplay($aRetArray, "Split on each character", Default, 8)

    ; Перечитать – строка разделителя используется целиком.
    _FileReadToArray($sFilePath, $aRetArray, $FRTA_NOCOUNT + $FRTA_ENTIRESPLIT, ":|")
    _ArrayDisplay($aRetArray, "Split on full delimiter", Default, 8)

    FileDelete($sFilePath)
EndFunc   ;==>Example

Пример 2

#include <Array.au3>
#include <File.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Объявить переменную для передачи в _FileReadToArray.
    Local $aArray = 0

    ; Прочитать текущий файл скрипта в массив, используя ранее объявленную переменную.
    ; $iFlag = 0: счётчик в массиве не сохраняется. Для определения размера используйте UBound().
    If Not _FileReadToArray(@ScriptFullPath, $aArray, 0) Then
        MsgBox($MB_SYSTEMMODAL, "", "There was an error reading the file. @error: " & @error) ; Произошла ошибка при чтении текущего файла скрипта.
    EndIf

    ; Отобразить массив через _ArrayDisplay.
    _ArrayDisplay($aArray)
EndFunc   ;==>Example
