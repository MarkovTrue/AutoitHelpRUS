Справочник функций

_WinAPI_GetStartupInfo

Получает содержимое структуры STARTUPINFO, которая была указана при создании вызывающего процесса.

#include <WinAPISys.au3>
_WinAPI_GetStartupInfo ( )

Возвращаемое значение

Возвращает структуру $tagSTARTUPINFO, которая содержит сведения о запуске.
