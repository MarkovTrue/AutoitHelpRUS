_WinAPI_GetPEType

Получает тип машины для указанного переносимого исполняемого файла (PE).

#include <WinAPIFiles.au3>
_WinAPI_GetPEType ( $sFilePath )

Параметры

$sFilePath Полный путь к PE, тип машины которого требуется получить.

Возвращаемое значение

Успех: Возвращает тип машины ($IMAGE_FILE_MACHINE_*).
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Константы $IMAGE_FILE_MACHINE_* требуют #include <APIFilesConstants.au3>
