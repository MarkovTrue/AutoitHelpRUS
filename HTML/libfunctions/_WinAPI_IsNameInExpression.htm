_WinAPI_IsNameInExpression

Определяет, соответствует ли строка указанному шаблону.

#include <WinAPIShPath.au3>
_WinAPI_IsNameInExpression ( $sString, $sPattern [, $bCaseSensitive = False] )

Параметры

$sString Строка для сравнения с шаблоном. Эта строка не может содержать подстановочные символы.
$sPattern Строка шаблона. Эта строка может содержать подстановочные символы.
$bCaseSensitive [необязательный] Определяет, следует ли относиться к строке как к чувствительной к регистру при сравнении, допустимые значения:
True – сравнение с учётом регистра.
False – сравнение без учёта регистра (по умолчанию).

Возвращаемое значение

True: Возвращает строка соответствует указанному шаблону.
False: строка не соответствует указанному шаблону.

Примечания

Если только один из параметров является пустой строкой, функция возвращает 0 (False). Это означает, что "*" не
соответствует пустой строке. Если оба параметра являются пустыми строками, функция возвращает 1 (True).

Эта функция требует Windows 7 или более поздней версии.
