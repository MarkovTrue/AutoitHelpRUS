Справочник функций

_WinAPI_EnumFontFamilies

Перечисляет все шрифты системы с уникальными именами, удовлетворяющие заданным характеристикам.

#include <WinAPIGdi.au3>
_WinAPI_EnumFontFamilies ( [$hDC = 0 [, $sFaceName = '' [, $iCharSet = $DEFAULT_CHARSET [, $iFontType = 0x07 [, $sPattern = '' [, $bExclude = False]]]]]] )

Параметры

$hDC [необязательный] Дескриптор контекста устройства, из которого перечисляются шрифты.
Если параметр равен 0, используется DC текущего экрана приложения.
$sFaceName [необязательный] Имя гарнитуры шрифта.
Если параметр – пустая строка (по умолчанию), функция перечисляет по одному шрифту для каждой доступной гарнитуры.
Если задано допустимое имя гарнитуры, функция перечисляет все шрифты с этим именем.
$iCharSet [необязательный] Набор символов. Может быть одним из предопределённых значений.
При $DEFAULT_CHARSET (по умолчанию) функция перечисляет все уникальные по имени шрифты во всех наборах символов.
Если есть два шрифта с одинаковым именем, перечисляется только один.
При указании конкретного набора символов перечисляются только шрифты этого набора.
$ANSI_CHARSET
$BALTIC_CHARSET
$CHINESEBIG5_CHARSET
$DEFAULT_CHARSET (по умолчанию)
$EASTEUROPE_CHARSET
$GB2312_CHARSET
$GREEK_CHARSET
$HANGEUL_CHARSET
$MAC_CHARSET
$OEM_CHARSET
$RUSSIAN_CHARSET
$SHIFTJIS_CHARSET
$SYMBOL_CHARSET
$TURKISH_CHARSET
$VIETNAMESE_CHARSET
Корейская редакция Windows:
$JOHAB_CHARSET
Ближневосточная редакция Windows:
$ARABIC_CHARSET
$HEBREW_CHARSET
Тайская редакция Windows:
$THAI_CHARSET
$iFontType [необязательный] Тип перечисляемых шрифтов. Может быть 0 (векторные шрифты), -1 (все шрифты) или любым сочетанием значений:
$DEVICE_FONTTYPE (по умолчанию)
$RASTER_FONTTYPE (по умолчанию)
$TRUETYPE_FONTTYPE (по умолчанию).
$sPattern [необязательный] Шаблон для включения (или исключения) шрифтов в перечислении.
Имеет смысл только если имя гарнитуры не задано. Может содержать подстановочные знаки.
Если параметр опущен или пустая строка, шаблон игнорируется.
$bExclude [необязательный] Использовать ли шаблон для исключения шрифтов:
True – исключать подходящие под шаблон шрифты.
False – включать подходящие под шаблон шрифты (по умолчанию).

Возвращаемое значение

Успех: Возвращает двумерный массив со следующей информацией:
$aArray[0][0] – количество строк массива (n)
$aArray[0][1] – не используется
[n][0] – имя гарнитуры шрифта.
[n][1] – стиль шрифта.
[n][2] – уникальное имя шрифта.
[n][3] – скрипт, то есть набор символов шрифта.
[n][4] – тип шрифта ($*_FONTTYPE).
[n][5] – шаг, технология и семейство шрифта (как в структуре TEXTMETRIC).
[n][6] – набор символов шрифта ($*_CHARSET).
[n][7] – флаги шрифта ($NTM_*).
Ошибка: Устанавливает @error ≠ 0.

Примечания

У шрифтов многих восточноазиатских языков два имени гарнитуры: английское и локализованное.
Функция возвращает английское имя, если системная локаль не совпадает с языком шрифта.

Если в контексте устройства задан графический режим $GM_ADVANCED и указан флаг $DEVICE_FONTTYPE,
_WinAPI_EnumFontFamilies() возвращает список шрифтов Type 1 и OpenType.
Если графический режим не $GM_ADVANCED – функция возвращает список Type 1, OpenType и TrueType.

Пример

#include <Array.au3>
#include <FontConstants.au3>
#include <WinAPIGdi.au3>

; Перечислить все уникальные по имени шрифты в наборе ANSI, исключая шрифты "@" (предназначенные для вертикального текста)
Local $aData = _WinAPI_EnumFontFamilies(0, '', $ANSI_CHARSET, BitOR($DEVICE_FONTTYPE, $TRUETYPE_FONTTYPE), '@*', 1)
If Not @error Then
    _ArrayDisplay($aData, '_WinAPI_EnumFontFamilies')
EndIf

; Перечислить все варианты начертания шрифта "Arial" в наборе символов ANSI
$aData = _WinAPI_EnumFontFamilies(0, 'Arial', $ANSI_CHARSET)
If Not @error Then
    _ArrayDisplay($aData, '_WinAPI_EnumFontFamilies')
EndIf
