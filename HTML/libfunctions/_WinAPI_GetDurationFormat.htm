_WinAPI_GetDurationFormat

Форматирует длительность как строку времени по правилам указанной локали.

#include <WinAPILocale.au3>
_WinAPI_GetDurationFormat ( $iLCID, $iDuration [, $sFormat = ''] )

Параметры

$iLCID Идентификатор локали (LCID), по которой форматируется длительность, либо одна из готовых констант:
$LOCALE_INVARIANT
$LOCALE_SYSTEM_DEFAULT
$LOCALE_USER_DEFAULT
Windows Vista и выше
$LOCALE_CUSTOM_DEFAULT
$LOCALE_CUSTOM_UI_DEFAULT
$LOCALE_CUSTOM_UNSPECIFIED
$iDuration Длительность в тиках FILETIME (по 100 нс).
Вместо числа можно передать структуру $tagSYSTEMTIME с разбивкой длительности по полям часов, минут, секунд и т.д.
$sFormat [необязательный] Шаблон вывода длительности, например "hh:mm:ss.ff".
Если параметр опущен или пустая строка (по умолчанию), используется формат длительности из настроек локали.

Возвращаемое значение

Успех: Возвращает отформатированную строку длительности.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

Дополнительно

Искать GetDurationFormat в библиотеке MSDN

Пример

#include <WinAPILocale.au3>

Local $iDuration = (90 * 60 + 14) * 1000 * 1000 * 10

ConsoleWrite(_WinAPI_GetDurationFormat(0, $iDuration, 'hh:mm:ss') & @CRLF)
ConsoleWrite(_WinAPI_GetDurationFormat(0, $iDuration, 'mm:ss') & @CRLF)
ConsoleWrite(_WinAPI_GetDurationFormat(0, $iDuration, 'ss') & @CRLF)
