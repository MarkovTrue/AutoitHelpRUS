Справочник функций

_ArraySearch

Находит элемент в одномерном или двумерном массиве. Аналогична _ArrayBinarySearch(), но не требует предварительной сортировки массива.

#include <Array.au3>
_ArraySearch ( Const ByRef $aArray, $vValue [, $iStart = 0 [, $iEnd = 0 [, $iCase = 0 [, $iCompare = 0 [, $iForward = 1 [, $iSubItem = -1 [, $bRow = False]]]]]]] )

Параметры

$aArray Массив для поиска.
$vValue Искомое значение в $aArray.
$iStart [необязательный] Индекс массива для начала поиска.
$iEnd [необязательный] Индекс элемента, на котором заканчивается поиск.
$iCase [необязательный] Если равен 1, поиск чувствителен к регистру.
$iCompare [необязательный]
0 – приведение переменных к одному типу (по умолчанию), "string" = 0, "" = 0 или "0" = 0 совпадают (если $iCase = 0)
1 – частичный поиск
2 – совпадение при одинаковом типе и значении переменных
3 – сравнение по регулярному выражению, переданному в $vValue
$iForward [необязательный] Если равен 0, поиск ведётся с конца массива к началу (вместо начала к концу).
$iSubItem [необязательный] Подиндекс для поиска в двумерных массивах.
$bRow [необязательный] Если True, $iSubItem задаёт строку для поиска. Если False (по умолчанию), то поиск по столбцам.

Возвращаемое значение

Успех: Возвращает индекс, по которому найдено $vValue.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным или двумерным массивом
3 – $aArray пуст
4 – $iStart больше $iEnd
5 – массив не является двумерным, а $bRow установлен в True
6 – $vValue не найден в массиве

Примечания

Эта функция, как правило, медленнее _ArrayBinarySearch(), но удобна, когда массив нельзя отсортировать перед поиском.

Значения $iCompare нельзя комбинировать.

См. также

_ArrayBinarySearch, _ArrayFindAll

Пример

Пример 1

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray[6][4]
For $i = 0 To 5
    For $j = 0 To 3
        $aArray[$i][$j] = "#" & $i & $j
    Next
Next
_ArrayDisplay($aArray, "Looking for '#32'", Default, 8)

; Поиск по столбцу (в столбце 2)
Local $iIndex = _ArraySearch($aArray, "#32", 0, 0, 0, 0, 1, 2)
MsgBox($MB_SYSTEMMODAL, "Found '#32'", "Column 2 on Row " & $iIndex)

; Поиск по строке (в строке 3)
$iIndex = _ArraySearch($aArray, "#32", 0, 0, 0, 0, 1, 3, True)
MsgBox($MB_SYSTEMMODAL, "Found '#32'", "Row 3 in Col " & $iIndex)

Пример 2

; использование двумерного массива

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $avArray[6][2] = [ _
        ["String0", "SubString0"], _
        ["String1", "SubString1"], _
        ["String2", "SubString2"], _
        ["String3", "SubString3"], _
        ["String4", "SubString4"], _
        ["String5", "SubString5"]]

_ArrayDisplay($avArray, "$avArray")

Local $sSearch = InputBox("_ArraySearch() demo", "String to find?")
If @error Then Exit

Local $sColumn = InputBox("_ArraySearch() demo", "Column to search?")
If @error Then Exit
$sColumn = Int($sColumn)

Local $iIndex = _ArraySearch($avArray, $sSearch, 0, 0, 0, 1, 1, $sColumn)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Not Found", '"' & $sSearch & '" was not found on column ' & $sColumn & '.')
Else
    MsgBox($MB_SYSTEMMODAL, "Found", '"' & $sSearch & '" was found in the array at position ' & $iIndex & ' on column ' & $sColumn & '.')
EndIf
