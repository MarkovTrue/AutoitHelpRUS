Справочник функций

_IEBodyWriteHTML

Заменяет HTML внутри тега <body> документа.

#include <IE.au3>
_IEBodyWriteHTML ( ByRef $oObject, $sHTML )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sHTML Строка HTML для записи в документ.

Возвращаемое значение

Успех: Возвращает нет.
Ошибка: Возвращает 0 или -1 и устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) – общая ошибка
2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
6 ($_IEStatus_LoadWaitTimeout) – тайм-аут ожидания загрузки
8 ($_IEStatus_AccessIsDenied) – доступ запрещен
9 ($_IEStatus_ClientDisconnected) – клиент отключен
@extended: Содержит номер недопустимого параметра.

Примечания

Ты можешь использовать _IEBodyReadHTML() для чтения текущего содержимого HTML тела документа, изменения этой строки и её повторной записи с помощью _IEBodyWriteHTML().
Ты также можешь использовать _IEDocInsertHTML(), _IEDocInsertText() и функции innerHTML, innerText, outerHTML и outerText из _IEPropertyGet() и _IEPropertySet() для управления отдельными элементами без переписывания больших блоков текста.

См. также

_IEBodyReadHTML, _IEBodyReadText, _IEDocInsertHTML, _IEDocInsertText, _IELoadWait, _IEPropertyGet, _IEPropertySet

Пример

; Открыть браузер с примером iFrame, получить ссылку
; К iFrame с именем "iFrameTwo" и заменить его HTML-тело

#include <IE.au3>

Local $oIE = _IE_Example("iframe")
Local $oFrame = _IEFrameGetObjByName($oIE, "iFrameTwo")
_IEBodyWriteHTML($oFrame, "Hello <b><font color=red>iFrame!</font></b>")
