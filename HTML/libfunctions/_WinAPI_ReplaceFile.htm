_WinAPI_ReplaceFile

Заменяет один файл другим файлом и создаёт резервную копию исходного файла.

#include <WinAPIFiles.au3>
_WinAPI_ReplaceFile ( $sReplacedFile, $sReplacementFile [, $sBackupFile = '' [, $iFlags = 0]] )

Параметры

$sReplacedFile Имя файла, который должен быть заменён.
$sReplacementFile Имя файла, который заменит файл $sReplacedFile.
$sBackupFile [необязательный] Имя файла, который будет служить резервной копией файла $sReplacedFile. Если этот
параметр – пустая строка, резервный файл не создаётся.
$iFlags [необязательный] Параметры замены. Этот параметр может быть одним или несколькими из следующих значений.
$REPLACEFILE_WRITE_THROUGH
$REPLACEFILE_IGNORE_MERGE_ERRORS
$REPLACEFILE_IGNORE_ACL_ERRORS

Возвращаемое значение

Успех: Возвращает True.
Ошибка: False (см. раздел Примечания).

Примечания

Если эта функция завершается с ошибкой, вызовите функцию _WinAPI_GetLastError(), чтобы получить расширенную информацию об ошибке. Ниже приведены
возможные коды ошибок для этой функции.

ERROR_UNABLE_TO_MOVE_REPLACEMENT (1176)
Вспомогательный файл не удалось переименовать. Если был указан $sBackupFile, замененный и замещающий файлы
сохраняют свои исходные имена файлов. В противном случае замены файл больше не существует, и замещающий файл существует
под своим исходным именем.

ERROR_UNABLE_TO_MOVE_REPLACEMENT_2 (1177)
Вспомогательный файл не удалось переместить. Вспомогательный файл всё ещё существует под своим исходным именем; однако
он унаследовал потоки файлов и атрибуты из файла, который он заменяет. Файл, который должен быть заменён, всё ещё
существует с именем, указанным в параметре $sReplacedFile.

ERROR_UNABLE_TO_REMOVE_REPLACED (1175)
Замененный файл не удалось удалить. Замененный и замещающий файлы сохраняют свои исходные имена файлов.

Если возвращается какая-либо другая ошибка, например ERROR_INVALID_PARAMETER (87), замененный и замещающий файлы
сохранят свои исходные имена файлов. В этом случае резервный файл не существует, и не гарантируется,
что замещающий файл унаследует все атрибуты и потоки замененного файла.

Пример

#include <WinAPIError.au3>
#include <WinAPIFiles.au3>

Opt('TrayAutoPause', 0)

FileDelete(@TempDir & '\Test*.tmp')

Local $sFile = @TempDir & '\Test.tmp'
Local $hFile = FileOpen($sFile, 2)
For $i = 1 To 10000
    FileWriteLine($hFile, " ")
Next
FileClose($hFile)

Local $sNewFile = @TempDir & '\TestNew.tmp'
FileCopy($sFile, $sNewFile)
If Not _WinAPI_ReplaceFile($sNewFile, $sFile) Then
    _WinAPI_ShowLastError('Error replacing no backup : ' & $sFile)
EndIf

FileCopy($sNewFile, $sFile, 1)
If Not _WinAPI_ReplaceFile($sNewFile, $sFile, @TempDir & '\TestBackup.tmp') Then
    _WinAPI_ShowLastError('Error replacing : ' & $sFile)
EndIf

FileDelete(@TempDir & '\Test*.tmp')
