Справочник функций

_WinAPI_ShellChangeNotify

Уведомляет систему о событии, выполненном приложением.

#include <WinAPIShellEx.au3>
_WinAPI_ShellChangeNotify ( $iEvent, $iFlags [, $iItem1 = 0 [, $iItem2 = 0]] )

Параметры

$iEvent Описывает произошедшее событие. Обычно одновременно указывается только одно событие. Если указано более
одного события, значения параметров $iItem1 и $iItem2 должны быть одинаковы
соответственно для всех указанных событий. Этот параметр может быть одним или несколькими из следующих значений.
$SHCNE_ALLEVENTS
$SHCNE_ASSOCCHANGED
$SHCNE_ATTRIBUTES
$SHCNE_CREATE
$SHCNE_DELETE
$SHCNE_DRIVEADD
$SHCNE_DRIVEADDGUI
$SHCNE_DRIVEREMOVED
$SHCNE_EXTENDED_EVENT
$SHCNE_FREESPACE
$SHCNE_MEDIAINSERTED
$SHCNE_MEDIAREMOVED
$SHCNE_MKDIR
$SHCNE_NETSHARE
$SHCNE_NETUNSHARE
$SHCNE_RENAMEFOLDER
$SHCNE_RENAMEITEM
$SHCNE_RMDIR
$SHCNE_SERVERDISCONNECT
$SHCNE_UPDATEDIR
$SHCNE_UPDATEIMAGE
$SHCNE_UPDATEITEM
$SHCNE_DISKEVENTS
$SHCNE_GLOBALEVENTS
$SHCNE_INTERRUPT
(см. MSDN для дополнительной информации)
$iFlags Флаги, указывающие значение параметров $iItem1 и $iItem2. Этот параметр должен быть одним
из следующих значений.
$SHCNF_DWORD
$SHCNF_IDLIST
$SHCNF_PATH
$SHCNF_PRINTER
$SHCNF_FLUSH
$SHCNF_FLUSHNOWAIT
$SHCNF_NOTIFYRECURSIVE
(см. MSDN для дополнительной информации)
$iItem1 [необязательный] Первое зависящее от события значение. По умолчанию 0.
$iItem2 [необязательный] Второе зависящее от события значение. По умолчанию 0.

Возвращаемое значение

Отсутствует.

Дополнительно

Искать SHChangeNotify в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIShellEx.au3>

Local $sIcon = RegRead('HKCR\AutoIt3Script\DefaultIcon', '')

If Not @error Then
    RegWrite('HKCR\AutoIt3Script\DefaultIcon', '', 'REG_SZ', @SystemDir & '\shell32.dll,-152')
    _WinAPI_ShellChangeNotify($SHCNE_ASSOCCHANGED, $SHCNF_FLUSH)
    MsgBox($MB_SYSTEMMODAL, '', 'The icon for .au3 files has been changed. Press OK to restore it.')
    RegWrite('HKCR\AutoIt3Script\DefaultIcon', '', 'REG_SZ', $sIcon)
    _WinAPI_ShellChangeNotify($SHCNE_ASSOCCHANGED, $SHCNF_FLUSH)
EndIf
