_WinAPI_ShellObjectProperties

Вызывает команду контекстного меню "Свойства" для объекта оболочки.

#include <WinAPIShellEx.au3>
_WinAPI_ShellObjectProperties ( $sFilePath [, $iType = $SHOP_FILEPATH [, $sProperty = '' [, $hParent = 0]]] )

Параметры

$sFilePath Имя объекта.
$iType [необязательный] Значение, которое указывает тип объекта. $SHOP_PRINTERNAME $SHOP_FILEPATH (по умолчанию) $SHOP_VOLUMEGUID
$sProperty [необязательный] Имя страницы листа свойств, которая должна быть открыта изначально.
$hParent [необязательный] Дескриптор родительского окна диалогового окна.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Дополнительно

Искать SHObjectProperties в библиотеке MSDN

Пример

#include <WinAPIShellEx.au3>

Opt('WinTitleMatchMode', 4)

If Not _WinAPI_ShellObjectProperties(@ScriptFullPath) Then Exit

Local $hWnd = WinWaitActive(@ScriptName, '', 3)
If Not $hWnd Then
    Exit
EndIf

While WinExists($hWnd)
    Sleep(100)
WEnd
