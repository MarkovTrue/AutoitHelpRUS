_WinNet_GetProviderName

Получает имя поставщика для определённого типа сети.

#include <WinNet.au3>
_WinNet_GetProviderName ( $iType )

Параметры

$iType Тип сети, уникальный для сети.

Возвращаемое значение

Успех: Возвращает сетевой поставщик.
Ошибка: Устанавливает @error ≠ 0.
