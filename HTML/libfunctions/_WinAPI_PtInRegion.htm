_WinAPI_PtInRegion

Определяет, находится ли указанная точка внутри указанного региона.

#include <WinAPIGdi.au3>
_WinAPI_PtInRegion ( $hRgn, $iX, $iY )

Параметры

$hRgn Дескриптор региона, который будет проверяться.
$iX Координата x точки в логических единицах.
$iY Координата y точки в логических единицах.

Возвращаемое значение

Успех: Возвращает истина – указанная точка находится в регионе.
Ошибка: Возвращает False.
