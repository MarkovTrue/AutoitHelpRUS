Справочник функций

_WinAPI_DestroyWindow

Уничтожает указанное окно.

#include <WinAPISysWin.au3>
_WinAPI_DestroyWindow ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Нельзя использовать _WinAPI_DestroyWindow() для уничтожения окна, созданного другим потоком.

См. также

_WinAPI_CreateWindowEx
