Справочник функций

_WinAPI_ShellGetPathFromIDList

Преобразует список идентификаторов элементов в путь файловой системы.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetPathFromIDList ( $pPIDL )

Параметры

$pPIDL Адрес списка идентификаторов элементов, который определяет расположение файла или каталога относительно корня пространства имён (рабочий стол).

Возвращаемое значение

Успех: Возвращает путь файловой системы.
Ошибка: Возвращает пустую строку.
