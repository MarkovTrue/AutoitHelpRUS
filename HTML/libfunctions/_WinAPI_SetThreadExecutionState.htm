Справочник функций

_WinAPI_SetThreadExecutionState

Предотвращает переход системы в режим сна или отключение дисплея во время работы текущего приложения.

#include <WinAPIProc.au3>
_WinAPI_SetThreadExecutionState ( $iFlags )

Параметры

$iFlags Требования к состоянию выполнения потока. Этот параметр может быть одним или несколькими из следующих значений. $ES_AWAYMODE_REQUIRED $ES_CONTINUOUS $ES_DISPLAY_REQUIRED $ES_SYSTEM_REQUIRED $ES_USER_PRESENT.

Возвращаемое значение

Успех: Возвращает значение является предыдущим состоянием выполнения потока.
Ошибка: Возвращает 0.

Примечания

Функция _WinAPI_SetThreadExecutionState() не может использоваться для предотвращения переноса компьютера в режим сна пользователем. Приложения должны учитывать, что пользователь ожидает определённого поведения при закрытии крышки ноутбука или нажатии кнопки питания.
Эта функция не предотвращает запуск экранной заставки. Согласно Microsoft Docs $ES_USER_PRESENT не должна использоваться, и $ES_AWAYMODE_REQUIRED следует использовать только в крайнем случае. Константы определены в APIProcConstants.au3

Дополнительно

Искать SetThreadExecutionState в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIProc.au3>

_Example()

Func _Example()
    ; отключить заставку и другие функции сна Windows
    _WinAPI_SetThreadExecutionState(BitOR($ES_DISPLAY_REQUIRED, $ES_CONTINUOUS, $ES_SYSTEM_REQUIRED, $ES_AWAYMODE_REQUIRED))
    MsgBox($MB_OK, 'Testing', 'Close this tester window when you be sure that your ScreeSaver not runing.')

    ; вернуться к нормальному поведению Windows
    _WinAPI_SetThreadExecutionState($ES_CONTINUOUS)
EndFunc   ;==>_Example
