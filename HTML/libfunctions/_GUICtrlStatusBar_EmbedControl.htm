_GUICtrlStatusBar_EmbedControl

Встраивает элемент управления.

#include <GuiStatusBar.au3>
_GUICtrlStatusBar_EmbedControl ( $hWnd, $iPart, $hControl [, $iFit = 4] )

Параметры

$hWnd Дескриптор элемента управления.
$iPart Индекс части (нумерация с 0).
$hControl Дескриптор элемента управления, встраиваемого в панель.
$iFit [необязательный] Определяет, как разместить элемент управления. Можно комбинировать:
1 – центрировать элемент управления по горизонтали
2 – центрировать элемент управления по вертикали
4 – растянуть элемент управления по горизонтали
8 – растянуть элемент управления по вертикали

Возвращаемое значение

Отсутствует.

Примечания

В строку состояния можно встроить ЛЮБОЙ элемент управления, а не только перечисленные.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiStatusBar.au3>
#include <ProgressConstants.au3>
#include <SendMessage.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("StatusBar Embed Control (v" & @AutoItVersion & ")", 450, 320, 100, 100)

    ; по умолчанию 1 часть, без текста
    Local $hStatus = _GUICtrlStatusBar_Create($hGUI)

    ; Создать элемент memo
    _MemoCreate(2, 8, 444, 259)

    _GUICtrlStatusBar_SetMinHeight($hStatus, 20)

    GUISetState(@SW_SHOW)

    ; Инициализировать части
    Local $aParts[4] = [80, 160, 300, -1]
    _GUICtrlStatusBar_SetParts($hStatus, $aParts)
    _GUICtrlStatusBar_SetText($hStatus, "Part 0")
    _GUICtrlStatusBar_SetText($hStatus, "Part 1", 1)

    ; Встроить индикатор выполнения
    Local $hProgress, $idProgress
    If @OSType = "WIN32_WINDOWS" Then
        $idProgress = GUICtrlCreateProgress(0, 0, -1, -1, $PBS_SMOOTH)
        $hProgress = GUICtrlGetHandle($idProgress)
        _GUICtrlStatusBar_EmbedControl($hStatus, 2, $hProgress)
    Else
        $idProgress = GUICtrlCreateProgress(0, 0, -1, -1, $PBS_MARQUEE) ; marquee работает на Win XP и выше
        $hProgress = GUICtrlGetHandle($idProgress)
        _GUICtrlStatusBar_EmbedControl($hStatus, 2, $hProgress)
        _SendMessage($hProgress, $PBM_SETMARQUEE, True, 200) ; marquee работает на Win XP и выше
    EndIf

    Local $idInput = GUICtrlCreateInput("This is Embedded", 0, 0)
    Local $hInput = GUICtrlGetHandle($idInput)
    _GUICtrlStatusBar_EmbedControl($hStatus, 3, $hInput, 3)

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
