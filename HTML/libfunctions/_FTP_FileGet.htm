Справочник функций

_FTP_FileGet

Загружает файл с FTP-сервера.

#include <FTPEx.au3>
_FTP_FileGet ( $hFTPSession, $sRemoteFile, $sLocalFile [, $bFailIfExists = False [, $iFlagsAndAttributes = 0 [, $iFlags = $FTP_TRANSFER_TYPE_UNKNOWN [, $iContext = 0]]]] )

Параметры

$hFTPSession возвращается функцией _FTP_Connect()
$sRemoteFile Удалённое расположение файла.
$sLocalFile Локальный файл.
$bFailIfExists [необязательный] True – не перезаписывать существующий файл (по умолчанию False).
$iFlagsAndAttributes [необязательный] Атрибуты создаваемого файла.
$iFlags [необязательный] Аналогично параметру в _FTP_FileOpen().
$iContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

См. также

_FTP_Connect, _FTP_FileOpen, _FTP_SetStatusCallback
