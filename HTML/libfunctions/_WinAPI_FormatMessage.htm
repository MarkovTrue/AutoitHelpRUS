Справочник функций

_WinAPI_FormatMessage

Форматирует строку сообщения.

#include <WinAPIError.au3>
_WinAPI_FormatMessage ( $iFlags, $pSource, $iMessageID, $iLanguageID, ByRef $pBuffer, $iSize, $vArguments )

Параметры

$iFlags Набор битовых флагов, задающих аспекты процесса форматирования и интерпретации параметра $pSource.
Младший байт $iFlags задаёт способ обработки переносов строк в выходном буфере.
Также младший байт может задавать максимальную ширину строки в выходе.
Полное описание – см. MSDN.
$pSource Указатель на источник сообщения.
$iMessageID Requested сообщение identifier.
$iLanguageID Идентификатор языка запрашиваемого сообщения.
$pBuffer Указатель на буфер или строковую переменную для приёма сообщения.
$iSize Максимальный размер буфера сообщения.
$vArguments Адрес массива вставок сообщения.

Возвращаемое значение

Успех: Возвращает количество байт, записанных в буфер сообщения.
Ошибка: Устанавливает @error в ненулевое значение.
@extended: = _WinAPI_GetLastError().
Вызовите @extended.
