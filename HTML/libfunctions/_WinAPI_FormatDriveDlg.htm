Справочник функций

_WinAPI_FormatDriveDlg

Открывает системный диалог форматирования.

#include <WinAPIDlg.au3>
_WinAPI_FormatDriveDlg ( $sDrive [, $iOption = $SHFMT_OPT_FULL [, $hParent = 0]] )

Параметры

$sDrive Задаёт диск для форматирования, например D:, E: и т. д.
$iOption [необязательный] Параметр должен быть 0 или одним из следующих значений, изменяющих параметры форматирования диалога по умолчанию:
$SHFMT_OPT_FULL (по умолчанию)
$SHFMT_OPT_QUICKFORMAT
$SHFMT_OPT_SYSONLY
$hParent [необязательный] Дескриптор родительского окна диалога.

Возвращаемое значение

Успех: Возвращает ID последнего успешного формата. LOWORD этого значения можно передать в последующих вызовах как fmtID, чтобы повторить последний формат.
Ошибка: 0 и устанавливает @error в одну из следующих констант:
$SHFMT_ERROR
$SHFMT_CANCEL
$SHFMT_NOFORMAT

Примечания

Управление форматированием идёт через интерфейс диалога:
для начала форматирования пользователь должен нажать OK – программно запустить процесс нельзя.

Пример

#include <Array.au3>
#include <ComboConstants.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIDlg.au3>

Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 200, 100)
Local $vDrive = DriveGetDrive('ALL')
$vDrive = StringUpper(_ArrayToString($vDrive, '|', 1))
GUICtrlCreateLabel('Select drive:', 15, 29, 62, 14)
Local $idCombo = GUICtrlCreateCombo('', 79, 25, 40, 21, $CBS_DROPDOWNLIST)
GUICtrlSetData(-1, $vDrive, StringLeft($vDrive, 2))
Local $idButton = GUICtrlCreateButton('Format...', 65, 70, 70, 23)
GUISetState(@SW_SHOW)

While 1
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE
            ExitLoop
        Case $idButton
            _WinAPI_FormatDriveDlg(GUICtrlRead($idCombo), 0, $hForm)
    EndSwitch
WEnd
