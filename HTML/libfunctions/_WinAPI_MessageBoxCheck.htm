Справочник функций

_WinAPI_MessageBoxCheck

Отображает окно сообщения, которое даёт пользователю возможность подавлять дальнейшие вхождения.

#include <WinAPIDlg.au3>
_WinAPI_MessageBoxCheck ( $iType, $sTitle, $sText, $sRegVal [, $iDefault = -1 [, $hParent = 0]] )

Параметры

$iType Флаги, указывающие содержимое и поведение окна сообщения.

Вы должны указать отображаемые кнопки, установив один и только один из следующих флагов.
$MB_OK
$MB_OKCANCEL
$MB_YESNO

Вы можете отобразить дополнительный значок, установив один и только один из следующих флагов.
$MB_ICONEXCLAMATION
$MB_ICONHAND
$MB_ICONINFORMATION
$MB_ICONQUESTION
$sTitle Строка, содержащая заголовок окна сообщения.
$sText Строка, содержащая отображаемое сообщение.
$sRegVal Строка, содержащая уникальное строковое значение для связи с этим сообщением.
$iDefault [необязательный] Значение, которое функция возвращает, когда пользователь выбрал отсутствие отображения окна сообщения
снова. Если пользователь не выбрал подавление окна сообщения, окно сообщения отображается и
функция игнорирует $iDefault.
$hParent [необязательный] Дескриптор окна владельца окна сообщения.

Возвращаемое значение

Успех: Если пользователь уже выбрал подавление окна сообщения, функция немедленно возвращает
значение , присваиваемое $iDefault, в противном случае возвращает одну из констант $ID_*.
Ошибка: (-1).

Примечания

Эта функция записывает окна сообщений, которые пользователь выбрал для подавления в следующем ключе реестра.

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\LowRegistry\DontShowMeThisDialogAgain

Дополнительно

Искать SHMessageBoxCheck в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIDlg.au3>

Local $iResult = _WinAPI_MessageBoxCheck($MB_ICONINFORMATION, 'MyProg', '_WinAPI_MessageBoxCheck()', 'MyProg')

ConsoleWrite('Return: ' & $iResult & @CRLF)
