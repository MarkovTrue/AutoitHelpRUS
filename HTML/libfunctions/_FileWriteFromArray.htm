Справочник функций

_FileWriteFromArray

Записывает массив в указанный файл.

#include <File.au3>
_FileWriteFromArray ( $sFilePath, Const ByRef $aArray [, $iBase = Default [, $iUBound = Default [, $sDelimiter = "|"]]] )

Параметры

$sFilePath Путь к файлу для записи или дескриптор файла, возвращённый функцией FileOpen().
$aArray Массив, который нужно записать в файл.
$iBase [необязательный] Начальный индекс массива для чтения, обычно 0 или 1. По умолчанию 0.
$iUbound [необязательный] Индекс последнего записываемого элемента. По умолчанию – весь массив.
$sDelimiter [необязательный] Символ(ы)-разделитель для двумерных массивов. По умолчанию "|".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – ошибка при открытии указанного файла
2 – $aArray не является массивом
3 – ошибка записи в файл
4 – $aArray не является одномерным или двумерным массивом
5 – начальный индекс больше параметра $iUbound

Примечания

Если передан строковый путь, файл будет перезаписан и закрыт.
Для других режимов записи – например, дозаписи или Unicode-форматов – откройте файл через FileOpen() и передайте дескриптор.
Если передан дескриптор файла, после записи файл остаётся открытым.

См. также

_FileReadToArray

Пример

#include <File.au3>

; Получить список всех файлов в папке текущего скрипта.
Local $aScriptDir = _FileListToArray(@ScriptDir)

; Создать файл в каталоге %TEMP% пользователя.
Local $sFilePath = @TempDir & "\Examples.txt"

; Записать массив в файл, передав имя файла.
_FileWriteFromArray($sFilePath, $aScriptDir, 1)

; Отобразить файл.
ShellExecute($sFilePath)
