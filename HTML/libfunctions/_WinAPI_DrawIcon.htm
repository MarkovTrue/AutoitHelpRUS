_WinAPI_DrawIcon

Рисует значок или курсор в указанном контексте устройства.

#include <WinAPIGdiDC.au3>
_WinAPI_DrawIcon ( $hDC, $iX, $iY, $hIcon )

Параметры

$hDC Дескриптор контекста устройства.
$iX X-координата левого верхнего угла значка.
$iY Y-координата левого верхнего угла значка.
$hIcon Дескриптор значка, который нужно нарисовать.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
