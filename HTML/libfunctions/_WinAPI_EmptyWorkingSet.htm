Справочник функций

_WinAPI_EmptyWorkingSet

Освобождает максимально возможное число страниц из рабочего набора указанного процесса.

#include <WinAPIProc.au3>
_WinAPI_EmptyWorkingSet ( [$iPID = 0] )

Параметры

$iPID [необязательный] PID процесса. По умолчанию (0) – текущий процесс.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Дополнительно

Искать EmptyWorkingSet в библиотеке MSDN
