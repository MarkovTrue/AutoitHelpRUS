_WinAPI_CoTaskMemAlloc

Выделяет блок памяти задачи.

#include <WinAPICom.au3>
_WinAPI_CoTaskMemAlloc ( $iSize )

Параметры

$iSize Размер выделяемого блока памяти в байтах.

Возвращаемое значение

Успех: Возвращает выделенный блок памяти.
Ошибка: Возвращает 0.

Примечания

Если $iSize равен 0, _WinAPI_CoTaskMemAlloc() выделяет элемент нулевой длины и возвращает действительный указатель на этот элемент.
Выделенный блок может быть больше, чем $iSize байт, из-за пространства, требуемого для выравнивания и для информации о техническом обслуживании.
