Справочник функций

_WinAPI_SetTextCharacterExtra

Задаёт дополнительное расстояние между символами для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetTextCharacterExtra ( $hDC, $iCharExtra )

Параметры

$hDC Дескриптор контекста устройства.
$iCharExtra Величина дополнительного расстояния в логических единицах, добавляемого к каждому символу.

Возвращаемое значение

Успех: Возвращает предыдущее дополнительное расстояние между символами.
Ошибка: (-1).

Примечания

Эта функция поддерживается в основном для совместимости с существующими приложениями.

Рекомендуемый подход – вместо вызова этой функции и последующего TextOut приложения должны вызывать ExtTextOut и использовать его параметр lpDx для указания ширин.
