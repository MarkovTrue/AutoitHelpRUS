_GUICtrlListView_Create

Создаёт элемент управления ListView.

#include <GuiListView.au3>
_GUICtrlListView_Create ( $hWnd, $sHeaderText, $iX, $iY [, $iWidth = 150 [, $iHeight = 150 [, $iStyle = 0x0000000D [, $iExStyle = 0x00000000 [, $bCoInit = False]]]]] )

Параметры

$hWnd Дескриптор родительского или владеющего окна.
$sHeaderText Текст для отображения в заголовке. Разделённый символом "|".
$iX Горизонтальная позиция элемента управления.
$iY Вертикальная позиция элемента управления.
$iWidth [необязательный] Ширина элемента управления
$iHeight [необязательный] Высота элемента управления
$iStyle [необязательный] Стили элемента управления:
$LVS_ALIGNLEFT – элементы выровнены по левому краю в представлении значков и малых значков
$LVS_ALIGNTOP – элементы выровнены по верхнему краю элемента управления в представлении значков и малых значков
$LVS_AUTOARRANGE – значки автоматически организованы в представлении значков и малых значков
$LVS_EDITLABELS – текст элемента можно редактировать на месте
$LVS_ICON – этот стиль указывает представление значков
$LVS_LIST – этот стиль указывает представление списка
$LVS_NOCOLUMNHEADER – заголовки столбцов не отображаются в представлении отчёта
$LVS_NOLABELWRAP – текст элемента отображается в одной строке в представлении значков
$LVS_NOSCROLL – прокрутка отключена
$LVS_NOSORTHEADER – заголовки столбцов не работают как кнопки
$LVS_OWNERDATA – этот стиль указывает виртуальный элемент управления
$LVS_OWNERDRAWFIXED – окно-владелец может рисовать элементы в представлении отчёта
$LVS_REPORT – этот стиль указывает представление отчёта
$LVS_SHAREIMAGELISTS – список изображений не будет удалён
$LVS_SHOWSELALWAYS – выделение, если оно есть, всегда отображается
$LVS_SINGLESEL – в один момент можно выбрать только один элемент
$LVS_SMALLICON – этот стиль указывает представление малых значков
$LVS_SORTASCENDING – индексы элементов сортируются в порядке возрастания
$LVS_SORTDESCENDING – индексы элементов сортируются в порядке убывания

По умолчанию: $LVS_REPORT, $LVS_SINGLESEL, $LVS_SHOWSELALWAYS
Принудительно: $WS_CHILD, $WS_VISIBLE.
$iExStyle [необязательный] Расширенные стили элемента управления. Может быть комбинацией следующих:
$LVS_EX_BORDERSELECT – при выборе элемента цвет границы элемента изменяется
$LVS_EX_CHECKBOXES – включает флажки для элементов в элементе управления ListView
$LVS_EX_DOUBLEBUFFER – рисует с двойной буферизацией, что уменьшает мерцание
$LVS_EX_FLATSB – включает плоские полосы прокрутки в представлении списка
$LVS_EX_FULLROWSELECT – при выборе элемента этот элемент и все его подэлементы выделяются
$LVS_EX_GRIDLINES – отображает сетку вокруг элементов и подэлементов
$LVS_EX_HEADERDRAGDROP – включает перетаскивание для переупорядочения столбцов
$LVS_EX_INFOTIP – сообщение уведомления $LVN_GETINFOTIP отправляется перед отображением подсказки
$LVS_EX_LABELTIP – если не установлено, этикетки развёртываются частично скрытыми только для режима больших значков
$LVS_EX_MULTIWORKAREAS – элемент управления не будет автоматически организовывать свои значки до тех пор, пока не будут определены одна или несколько рабочих областей
$LVS_EX_ONECLICKACTIVATE – элемент управления отправляет сообщение $LVN_ITEMACTIVATE при щелчке пользователя на элемент
$LVS_EX_REGIONAL – устанавливает область элемента управления так, чтобы она включала только значки и текст элемента
$LVS_EX_SIMPLESELECT – в представлении значков перемещает изображение состояния элемента управления в верхний правый угол
$LVS_EX_SUBITEMIMAGES – позволяет отображать изображения для подэлементов
$LVS_EX_TRACKSELECT – включает отслеживание горячей выборки в элементе управления
$LVS_EX_TWOCLICKACTIVATE – элемент управления отправляет сообщение $LVN_ITEMACTIVATE при двойном щелчке пользователя на элемент
$LVS_EX_UNDERLINECOLD – приводит к отображению объектов, которые могут быть активированы, с подчёркнутым текстом
$LVS_EX_UNDERLINEHOT – приводит к отображению горячих элементов, которые могут быть активированы, с подчёркнутым текстом.
$bCoInit [необязательный] Инициализирует библиотеку COM для использования вызывающим потоком.

Возвращаемое значение

Успех: Возвращает дескриптор элемента управления ListView.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – неверный дескриптор окна
2 – текст заголовка не является строкой

Примечания

Данная функция предназначена для опытных пользователей и для изучения принципов работы элемента управления.

См. также

_GUICtrlListView_Destroy, _GUICtrlListView_SetBkImage

Пример

#include "Extras\WM_NOTIFY.au3"

#include <GUIConstantsEx.au3>
#include <GuiImageList.au3>
#include <GuiListView.au3>
#include <StructureConstants.au3>
#include <WindowsNotifsConstants.au3>

Global $g_hListView

Example()

Func Example()
    Local $hGUI, $hImage
    $hGUI = GUICreate("ListView Create (v" & @AutoItVersion & ")", 400, 300)

    $g_hListView = _GUICtrlListView_Create($hGUI, "", 2, 2, 394, 268)
    _GUICtrlListView_SetExtendedListViewStyle($g_hListView, BitOR($LVS_EX_GRIDLINES, $LVS_EX_FULLROWSELECT, $LVS_EX_SUBITEMIMAGES))
    GUISetState(@SW_SHOW)

    _WM_NOTIFY_Register()

    ; Загрузить изображения
    $hImage = _GUIImageList_Create()
    _GUIImageList_Add($hImage, _GUICtrlListView_CreateSolidBitMap($g_hListView, 0xFF0000, 16, 16))
    _GUIImageList_Add($hImage, _GUICtrlListView_CreateSolidBitMap($g_hListView, 0x00FF00, 16, 16))
    _GUIImageList_Add($hImage, _GUICtrlListView_CreateSolidBitMap($g_hListView, 0x0000FF, 16, 16))
    _GUICtrlListView_SetImageList($g_hListView, $hImage, 1)

    ; Добавить столбцы
    _GUICtrlListView_InsertColumn($g_hListView, 0, "Column 1", 100)
    _GUICtrlListView_InsertColumn($g_hListView, 1, "Column 2", 100)
    _GUICtrlListView_InsertColumn($g_hListView, 2, "Column 3", 100)

    ; Добавить элементы
    _GUICtrlListView_AddItem($g_hListView, "Row 1: Col 1", 0)
    _GUICtrlListView_AddSubItem($g_hListView, 0, "Row 1: Col 2", 1)
    _GUICtrlListView_AddSubItem($g_hListView, 0, "Row 1: Col 3", 2)
    _GUICtrlListView_AddItem($g_hListView, "Row 2: Col 1", 1)
    _GUICtrlListView_AddSubItem($g_hListView, 1, "Row 2: Col 2", 1)
    _GUICtrlListView_AddItem($g_hListView, "Row 3: Col 1", 2)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
