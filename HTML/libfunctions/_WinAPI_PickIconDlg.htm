Справочник функций

_WinAPI_PickIconDlg

Отображает диалоговое окно, которое позволяет пользователю выбрать значок.

#include <WinAPIDlg.au3>
_WinAPI_PickIconDlg ( [$sIcon = '' [, $iIndex = 0 [, $hParent = 0]]] )

Параметры

$sIcon [необязательный] Полный путь к файлу, который содержит начальный значок.
$iIndex [необязательный] Индекс начального значка. По умолчанию 0.
$hParent [необязательный] Дескриптор родительского окна.

Возвращаемое значение

Успех: Возвращает массив, содержащий следующую информацию:
[0] - Путь к файлу, который содержит выбранный значок.
[1] - Индекс выбранного значка.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Эта функция также устанавливает флаг @error в 1, если значок не был выбран.

Дополнительно

Искать PickIconDlg в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <WinAPIDlg.au3>

Local $aLast[2] = [@SystemDir & '\shell32.dll', 3]

Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 160, 160)
Local $idButton = GUICtrlCreateButton('Change Icon...', 25, 130, 110, 23)
Local $idIcon = GUICtrlCreateIcon($aLast[0], -(1 + $aLast[1]), 64, 54, 32, 32)
GUISetState(@SW_SHOW)

Local $aData
While 1
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE
            ExitLoop
        Case $idButton
            $aData = _WinAPI_PickIconDlg($aLast[0], $aLast[1], $hForm)
            If IsArray($aData) Then
                GUICtrlSetImage($idIcon, $aData[0], -(1 + $aData[1]))
                $aLast = $aData
            EndIf
    EndSwitch
WEnd
