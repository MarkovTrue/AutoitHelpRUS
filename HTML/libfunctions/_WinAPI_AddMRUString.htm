Справочник функций

_WinAPI_AddMRUString

Добавляет строку в верхнюю часть списка недавно использованных (MRU) элементов.

#include <WinAPIReg.au3>
_WinAPI_AddMRUString ( $hMRU, $sStr )

Параметры

$hMRU Дескриптор списка MRU.
$sStr Строка, которая будет добавлена.

Возвращаемое значение

Успех: Возвращает неотрицательное значение.
Ошибка: (-1).

См. также

_WinAPI_CreateMRUList, _WinAPI_FreeMRUList

Дополнительно

Искать AddMRUStringW в библиотеке MSDN

Пример

#include <ComboConstants.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIReg.au3>

Local $hMRU = _WinAPI_CreateMRUList($HKEY_CURRENT_USER, 'Software\MyProg\MRU', 5)

; Создать MRU-список в реестре (только при первом запуске примера)
RegRead('HKCU\Software\MyProg\MRU', 'MRUList')
If @error Then
    For $i = 5 To 1 Step -1
        _WinAPI_AddMRUString($hMRU, 'String' & $i)
    Next
EndIf

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 320, 92)
Local $idCombo = GUICtrlCreateCombo(_WinAPI_EnumMRUList($hMRU, 0), 10, 20, 300, 21, $CBS_DROPDOWNLIST)
Local $iCount = _WinAPI_EnumMRUList($hMRU, -1)
Local $sData = ''
For $i = 1 To $iCount - 1
    $sData &= _WinAPI_EnumMRUList($hMRU, $i) & '|'
Next
GUICtrlSetData(-1, $sData)
Local $idButton = GUICtrlCreateButton('OK', 125, 58, 70, 23)
GUISetState(@SW_SHOW)

While 1
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE
            ExitLoop
        Case $idButton
            _WinAPI_AddMRUString($hMRU, GUICtrlRead($idCombo))
            ExitLoop
    EndSwitch
WEnd

_WinAPI_FreeMRUList($hMRU)
