_WinAPI_BufferedPaintClear

Очищает указанный прямоугольник в буфере до ARGB = {0,0,0,0}.

#include <WinAPITheme.au3>
_WinAPI_BufferedPaintClear ( $hBP [, $tRECT = 0] )

Параметры

$hBP Дескриптор контекста буферизованного рисования.
$tRECT [необязательный] Структура $tagRECT, определяющая прямоугольник для очистки. Если этот параметр равен 0 (по умолчанию), используется весь буфер.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
