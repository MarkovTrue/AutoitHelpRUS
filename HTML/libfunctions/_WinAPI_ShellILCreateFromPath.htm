_WinAPI_ShellILCreateFromPath

Создаёт указатель на список идентификаторов элементов (PIDL) из пути.

#include <WinAPIShellEx.au3>
_WinAPI_ShellILCreateFromPath ( $sFilePath )

Параметры

$sFilePath Путь для преобразования.

Возвращаемое значение

Успех: Возвращает PIDL пути в $sFilePath.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Чтобы освободить возвращённый PIDL, вызовите функцию _WinAPI_CoTaskMemFree().

См. также

_WinAPI_CoTaskMemFree
