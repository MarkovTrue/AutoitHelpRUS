_WinAPI_PointFromRect

Возвращает верхние/левые координаты $tagRECT в виде структуры $tagPOINT.

#include <WinAPIConv.au3>
_WinAPI_PointFromRect ( ByRef $tRECT [, $bCenter = True] )

Параметры

$tRECT структура $tagRECT
$bCenter [необязательный] Если истина, возвращаемое значение будет точкой в центре прямоугольника, иначе вернутся координаты слева/сверху
координаты.

Возвращаемое значение

Возвращает структуру $tagPOINT

Примечания

Эта функция используется для получения позиции клика для многих функций клика
