Справочник ключевых слов

$tagBITMAPINFO

Определяет размеры и цветовые сведения аппаратно-независимого точечного рисунка (DIB) на основе Windows.

Global Const $tagBITMAPINFO = $tagBITMAPINFOHEADER & ";dword biRGBQuad[1]"

Параметры

$tagBITMAPINFOHEADER Содержит информацию о размерах растрового изображения.
biRGBQuad Массив структур tagRGBQUAD. Элементы массива, составляющие таблицу цветов.

Примечания

Эта структура не должна использоваться для объявления, так как поле biRGBQuad не имеет правильного размера.
Предоставляется только для документации.

Должна использоваться конкатенация подходящего по размеру массива с $tagBITMAPINFOHEADER.
Убедитесь в правильной инициализации поля 'biSize'.
