_ArrayFromString

Восстанавливает массив из строки, созданной _ArrayToString() или _SQLite_Display2DResult().

#include <Array.au3>
_ArrayFromString ( $sArrayStr [, $sDelim_Col = "|" [, $sDelim_Row = @CRLF [, $bForce2D = False [, $iStripWS = $STR_STRIPLEADING + $STR_STRIPTRAILING]]]] )

Параметры

$sArrayStr Строка, сформированная _ArrayToString() или _SQLite_Display2DResult()**.
$sDelim_Col [необязательный] По умолчанию "|"
$sDelim_Row [необязательный] По умолчанию @CRLF
$bForce2D [необязательный] По умолчанию False. True принудительно создаёт 2D массив, даже если данные одномерные.
$iStripWS [необязательный] По умолчанию 3. Флаг для StringStripWS(). Установите 0, чтобы отключить.

Возвращаемое значение

Успех: Возвращает массив.
Ошибка: Устанавливает @error ≠ 0.

Примечания

** Для _SQLite_Display2DResult() параметр $sDelim_Col обязателен.

_ArrayToString() возвращает пустую строку для пустых массивов форматов [0], [1], [0][0], [1][0], [0][1], [1][1]. Эта функция не различает такие случаи и по умолчанию возвращает одноэлементный пустой массив: [1] или [1][1] в зависимости от флага $bForce2D.

См. также

_ArrayToString, _SQLite_Display2DResult

Пример

Пример 1

#include <Array.au3>
#include <Debug.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aArrayFromText = _ArrayFromString(" 1|2 ") ; По умолчанию начальные и конечные
    _DebugArrayDisplay($aArrayFromText, UBound($aArrayFromText, $UBOUND_DIMENSIONS) & "D") ; пробелы удаляются.

    $aArrayFromText = _ArrayFromString("1|2", Default, Default, True) ; Принудительно создаём 2D массив
    _DebugArrayDisplay($aArrayFromText, UBound($aArrayFromText, $UBOUND_DIMENSIONS) & "D")

    $aArrayFromText = _ArrayFromString("1|2" & @CRLF & "3|4")
    _DebugArrayDisplay($aArrayFromText, UBound($aArrayFromText, $UBOUND_DIMENSIONS) & "D")

EndFunc   ;==>Example

Пример 2

#include <Array.au3>
#include <Debug.au3>
#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Example()

Func Example()
    Local $aResult, $iRows, $iColumns, $iRval, $sText, $aArrayFromText

    _SQLite_Startup()
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite.dll Can't be Loaded!")
        Exit -1
    EndIf
    ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)

    _SQLite_Open() ; Открыть :memory: базу данных
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", "Can't Load Database!")
        Exit -1
    EndIf

    ; Пример таблицы
    ; Имя | Возраст
    ; -----------------------
    ; Алиса | 43
    ; Боб | 28
    ; Синди | 21

    If Not _SQLite_Exec(-1, "CREATE TEMP TABLE persons (Name, Age);" & _
            "INSERT INTO persons VALUES ('Alice','43');" & _
            "INSERT INTO persons VALUES ('Bob','28');" & _
            "INSERT INTO persons VALUES ('Cindy','21');") = $SQLITE_OK Then _
            MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())

    ; Запрос
    $iRval = _SQLite_GetTable2D(-1, "SELECT * FROM persons;", $aResult, $iRows, $iColumns)
    If $iRval = $SQLITE_OK Then

        ConsoleWrite('--- --- --- ---' & @CRLF)
        _SQLite_Display2DResult($aResult)
        ConsoleWrite('--- --- --- ---' & @CRLF)

        ; Пример с _SQLite_Display2DResult()
        $sText = _SQLite_Display2DResult($aResult, Default, True, @TAB)
        ConsoleWrite($sText & @CRLF) ; Допустим, вы сохранили это в лог-файл...
        $aArrayFromText = _ArrayFromString($sText, @TAB) ; ...затем восстанавливаем массив
        _DebugArrayDisplay($aArrayFromText, "from _SQLite_Display2DResult()")
        ConsoleWrite('--- --- --- ---' & @CRLF)

        ; Пример с _ArrayToString()
        $sText = _ArrayToString($aResult)
        ConsoleWrite($sText & @CRLF)
        $aArrayFromText = _ArrayFromString($sText)
        _DebugArrayDisplay($aArrayFromText, "from _ArrayToString()")
        ConsoleWrite('--- --- --- ---' & @CRLF)

    Else
        MsgBox($MB_SYSTEMMODAL, "SQLite Error: " & $iRval, _SQLite_ErrMsg())
    EndIf

    _SQLite_Close()
    _SQLite_Shutdown()
EndFunc   ;==>Example
