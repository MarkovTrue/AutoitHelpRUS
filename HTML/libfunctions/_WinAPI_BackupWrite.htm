Справочник функций

_WinAPI_BackupWrite

Восстанавливает файл или папку, которые были скопированы с использованием _WinAPI_BackupRead().

#include <WinAPIFiles.au3>
_WinAPI_BackupWrite ( $hFile, $pBuffer, $iLength, ByRef $iBytes, ByRef $pContext [, $bSecurity = False] )

Параметры

$hFile Дескриптор файла или папки для восстановления. Для получения дескриптора вызовите функцию _WinAPI_CreateFileEx().
Списки управления доступом (SACL) не восстанавливаются, если дескриптор файла не был создан с правом доступа $ACCESS_SYSTEM_SECURITY.
Чтобы убедиться, что записи управления доступом целостности восстановлены правильно, дескриптор файла также должен быть создан с правом доступа $WRITE_OWNER.
$pBuffer Указатель на буфер, из которого функция записывает данные.
$iLength Размер буфера в байтах.
$iBytes Количество записанных байтов.
$pContext Указатель на внутреннюю структуру данных, используемую функцией для сохранения контекстной информации во время операции восстановления.
Вы должны установить этой переменной значение 0 перед первым вызовом _WinAPI_BackupWrite() для указанного файла или папки.
Функция выделяет память для структуры данных и затем устанавливает переменную, чтобы она указывала на эту структуру.
Вы не должны изменять эту переменную или переменную, на которую она указывает, между вызовами _WinAPI_BackupWrite().
$bSecurity [необязательный] Указывает, будет ли функция восстанавливать данные списка управления доступом (ACL), допустимые значения:
True – данные ACL будут восстановлены.
Кроме того, вам нужно указать доступ $WRITE_OWNER и $WRITE_DAC при открытии дескриптора файла или папки.
Если дескриптор не имеет этих прав доступа, операционная система запрещает доступ к данным ACL, и восстановление данных ACL не будет происходить.
False – данные ACL будут пропущены (по умолчанию).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция _WinAPI_BackupWrite() не предназначена для восстановления файлов, зашифрованных в файловой системе EFS.

Когда вы закончите использовать _WinAPI_BackupWrite(), вы должны вызвать функцию _WinAPI_BackupWriteAbort() с соответствующим параметром, чтобы освободить память, используемую внутренней структурой данных.

См. также

_WinAPI_BackupWriteAbort

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <SecurityConstants.au3>
#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>
#include <WinAPIMem.au3>
#include <WinAPIProc.au3>
#include <WinAPIShPath.au3>

Local Const $sFile = @ScriptFullPath

; Включить привилегии "SeBackupPrivilege" и "SeRestorePrivilege" для операций резервного копирования и восстановления
Local $hToken = _WinAPI_OpenProcessToken(BitOR($TOKEN_ADJUST_PRIVILEGES, $TOKEN_QUERY))
Local $aAdjust, $aPrivileges[2] = [$SE_BACKUP_NAME, $SE_RESTORE_NAME]
_WinAPI_AdjustTokenPrivileges($hToken, $aPrivileges, $SE_PRIVILEGE_ENABLED, $aAdjust)
If @error Or @extended Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'You do not have the required privileges.')
    Exit
EndIf

; Создать буфер памяти для хранения данных резервной копии
Local $iBytes = 4096 + FileGetSize($sFile)
Local $pBuffer = _WinAPI_CreateBuffer($iBytes)

; Резервная копия файла, включая сведения о безопасности
Local $pContext = 0
Local $hFile = _WinAPI_CreateFileEx($sFile, $OPEN_EXISTING, $GENERIC_READ)
If Not _WinAPI_BackupRead($hFile, $pBuffer, $iBytes, $iBytes, $pContext, 1) Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to back up a file.')
    Exit
EndIf
_WinAPI_BackupReadAbort($pContext)
_WinAPI_CloseHandle($hFile)

; Восстановить файл (.bak) и ACL-данные
$pContext = 0
$hFile = _WinAPI_CreateFileEx(_WinAPI_PathRenameExtension($sFile, '.bak'), $CREATE_ALWAYS, BitOR($GENERIC_WRITE, $WRITE_DAC, $WRITE_OWNER))
If Not _WinAPI_BackupWrite($hFile, $pBuffer, $iBytes, $iBytes, $pContext, 1) Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to restore a file.')
    Exit
EndIf
_WinAPI_BackupWriteAbort($pContext)
_WinAPI_CloseHandle($hFile)

; Освободить использованную память
_WinAPI_FreeMemory($pBuffer)

; Восстановить привилегии "SeBackupPrivilege" и "SeRestorePrivilege" в значения по умолчанию
_WinAPI_AdjustTokenPrivileges($hToken, $aAdjust, 0, $aAdjust)
_WinAPI_CloseHandle($hToken)
