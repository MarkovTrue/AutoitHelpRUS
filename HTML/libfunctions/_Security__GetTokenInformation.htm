Справочник функций

_Security__GetTokenInformation

Получает указанный тип сведений о токене доступа.

#include <Security.au3>
_Security__GetTokenInformation ( $hToken, $iClass )

Параметры

$hToken Дескриптор токена доступа, из которого извлекаются сведения.
Если $iClass указывает $sTokenSource, дескриптор должен иметь доступ $TOKEN_QUERY_SOURCE.
Для всех остальных значений $iClass дескриптор должен иметь доступ $TOKEN_QUERY.
$iClass Значение для определения типа информации, которую извлекает функция.

Возвращаемое значение

Успех: Возвращает структура байтов, заполненная запрашиваемой информацией.
Ошибка: Возвращает 0.

См. также

_Security__OpenProcessToken, _Security__OpenThreadToken, _Security__OpenThreadTokenEx

Дополнительно

Искать GetTokenInformation в библиотеке MSDN

Func Example_GetTokInfo()
    Local $hProcess = _WinAPI_GetCurrentProcess()
    If @error Then Return ; проверить возможные ошибки

    Local $hToken = _Security__OpenProcessToken($hProcess, $TOKEN_ALL_ACCESS)
    ; Если токен – get...
    If $hToken Then
        ; Получить сведения о типе этого токена:
        Local $tInfo = _Security__GetTokenInformation($hToken, $TOKENTYPE)
        ; Результат будет сырыми двоичными данными. Для $TOKENTYPE это значение TOKEN_TYPE (значение перечисления). Переинтерпретируем как тип "int":
        Local $iTokenType = DllStructGetData(DllStructCreate("int", DllStructGetPtr($tInfo)), 1)

        MsgBox($MB_SYSTEMMODAL, "GetTokenInformation", "Token type is " & $iTokenType) ; Может быть либо $TOKENPRIMARY = 1, либо $TOKENIMPERSONATION = 2

        ; Закрыть дескриптор токена
        _WinAPI_CloseHandle($hToken)
    EndIf
EndFunc   ;==>Example_GetTokInfo

Func Example_SetTokInfo()
    Local $hProcess = _WinAPI_GetCurrentProcess()
    If @error Then Return ; проверить возможные ошибки

    Local $hToken = _Security__OpenProcessToken($hProcess, $TOKEN_ALL_ACCESS)

    ; Если токен – get...
    If $hToken Then
        ; Установить Medium Integrity Level для этого токена.
        Local $tSID = _Security__StringSidToSid($SID_MEDIUM_MANDATORY_LEVEL)
        ; Определить структуру TOKEN_MANDATORY_LABEL
        Local Const $tTOKEN_MANDATORY_LABEL = DllStructCreate("ptr Sid; dword Attributes")
        ; Заполнить нужными данными
        DllStructSetData($tTOKEN_MANDATORY_LABEL, "Sid", DllStructGetPtr($tSID, 1))
        DllStructSetData($tTOKEN_MANDATORY_LABEL, "Attributes", $SE_GROUP_INTEGRITY)

        If _Security__SetTokenInformation($hToken, $TOKENINTEGRITYLEVEL, $tTOKEN_MANDATORY_LABEL, DllStructGetSize($tTOKEN_MANDATORY_LABEL)) Then

            ; По умолчанию IL = $SID_HIGH_MANDATORY_LEVEL, однако...
            MsgBox($MB_SYSTEMMODAL, "SetTokenInformation", "$hToken = " & $hToken & @CRLF & "This token have non-default Medium Integrity Level")

            ; ... Сделать что-то с токеном здесь ...

        EndIf
        ; Закрыть дескриптор токена, когда он больше не нужен
        _WinAPI_CloseHandle($hToken)
    EndIf
EndFunc   ;==>Example_SetTokInfo
