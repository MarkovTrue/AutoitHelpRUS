_WinAPI_SelectClipPath

Выбирает текущий путь как область отсечения, объединяя новую область с любой существующей областью отсечения.

#include <WinAPIGdi.au3>
_WinAPI_SelectClipPath ( $hDC [, $iMode = $RGN_COPY] )

Параметры

$hDC Дескриптор контекста устройства.
$iMode [необязательный] Способ использования пути. Этот параметр может быть одним из следующих значений.
$RGN_AND
$RGN_COPY (по умолчанию)
$RGN_DIFF
$RGN_OR
$RGN_XOR

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
