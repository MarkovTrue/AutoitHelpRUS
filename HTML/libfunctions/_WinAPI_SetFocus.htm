Справочник функций

_WinAPI_SetFocus

Устанавливает фокус клавиатуры на указанное окно.

#include <WinAPISysWin.au3>
_WinAPI_SetFocus ( $hWnd )

Параметры

$hWnd Идентифицирует окно, которое будет получать ввод с клавиатуры. Если этот параметр равен 0, нажатия клавиш
игнорируются.

Возвращаемое значение

Успех: Возвращает дескриптор окна, которое имело фокус клавиатуры.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
