_GUICtrlMonthCal_GetMinReqRectArray

Возвращает минимальный размер, необходимый для отображения целого месяца в месячном календаре.

#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMinReqRectArray ( $hWnd )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.

Возвращаемое значение

Success: Возвращает массив следующего формата:
[0] – количество элементов
[1] – X-координата левого верхнего угла прямоугольника
[2] – Y-координата левого верхнего угла прямоугольника
[3] – X-координата правого нижнего угла прямоугольника
[4] – Y-координата правого нижнего угла прямоугольника
Ошибка: Устанавливает @error ≠ 0.

Примечания

элемент управления

Пример

#include "Extras\HelpFileInternals.au3"

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <GuiMonthCal.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $idMonthCal

    ; Создать GUI
    GUICreate("Month Calendar Get Min Req Rect Array", 400, 300)
    $idMonthCal = GUICtrlCreateMonthCal("", 4, 4, -1, -1, BitOR($WS_BORDER, $MCS_MULTISELECT), 0x00000000)

    ; Создать элемент memo
    _MemoCreate(4, 168, 392, 128, BitOR($WS_VSCROLL, $ES_MULTILINE))
    GUICtrlSendMsg($_g_idLst_Memo, $EM_SETREADONLY, True, 0)
    GUICtrlSetBkColor($_g_idLst_Memo, 0xFFFFFF)
    GUISetState(@SW_SHOW)

    ; Получить минимально необходимые высоту/ширину
    _MemoWrite(_FormatOutPut(_GUICtrlMonthCal_GetMinReqRectArray($idMonthCal)))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example

Func _FormatOutPut($aRect)
    Return "Minimum required Width: " & @TAB & $aRect[3] & @CRLF & "Minimum required Height:" & @TAB & $aRect[4]
EndFunc   ;==>_FormatOutPut
