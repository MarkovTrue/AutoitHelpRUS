_WinAPI_SaveHBITMAPToFile

Сохраняет указанный растр в указанный файл растра (.bmp).

#include <WinAPIGdi.au3>
_WinAPI_SaveHBITMAPToFile ( $sFilePath, $hBitmap [, $iXPelsPerMeter = Default [, $iYPelsPerMeter = Default]] )

Параметры

$sFilePath Имя файла .bmp для сохранения растра.
$hBitmap Дескриптор растра для сохранения.
$iXPelsPerMeter [необязательный] Горизонтальное разрешение в пикселях на метр.
$iYPelsPerMeter [необязательный] Вертикальное разрешение в пикселях на метр.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Функция сохраняет указанный растр без изменений, кроме случаев, когда растр использует глубину цвета 32 бит/пикс.
В этом случае, если растр содержит альфа-канал, он будет сохранён как изображение 32 бит/пикс, и
как 24 бит/пикс в противном случае.

Горизонтальное и вертикальное разрешение, на которые указывают параметры $iXPelsPerMeter и $iYPelsPerMeter, не
влияют на качество изображения, но приложение может использовать это значение для выбора растра из группы ресурсов, которая лучше всего
соответствует характеристикам текущего устройства. Если разрешение не указано, функция использует
текущее разрешение экрана, обычно 2834 пиксела на метр, что соответствует 72 пикселям на дюйм (dpi).

Если исходный растр сжат или использует маски цветов, функция завершится неудачей.

Пример

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPIMem.au3>
#include <WinAPIMisc.au3>

; Создать массив цветов из 256 элементов, нужный для 8-битного bitmap
Local $aColorTable[256]
For $i = 0 To 255
    $aColorTable[$i] = _WinAPI_RGB(0, $i, 255 - $i)
Next

; Создать таблицу цветов из массива цветов
Local $tColorTable = _WinAPI_CreateDIBColorTable($aColorTable)

; Создать 8-битный/пикс device-independent bitmap (DIB) и получить указатель на его биты
Local $hBitmap = _WinAPI_CreateDIB(256, 256, 8, $tColorTable, 256)
Local $pBits = _WinAPI_GetExtended()

; Заполнить индексы цветов bitmap
For $i = 0 To 255
    _WinAPI_FillMemory($pBits + 256 * $i, 256, $i)
Next

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 256, 256)
Local $idPic = GUICtrlCreatePic('', 0, 0, 256, 256)
Local $hPic = GUICtrlGetHandle($idPic)

; Создать DDB из DIB для корректного отображения в элементе управления
Local $hDC = _WinAPI_GetDC($hPic)
Local $hDev = _WinAPI_CreateCompatibleBitmap($hDC, 256, 256)
Local $hMemDC = _WinAPI_CreateCompatibleDC($hDC)
Local $hMemSv = _WinAPI_SelectObject($hMemDC, $hDev)
_WinAPI_DrawBitmap($hMemDC, 0, 0, $hBitmap)
_WinAPI_ReleaseDC($hPic, $hDC)
_WinAPI_SelectObject($hMemDC, $hMemSv)
_WinAPI_DeleteDC($hMemDC)

; Установить bitmap для элемента управления
_SendMessage($hPic, $STM_SETIMAGE, 0, $hDev)
Local $hObj = _SendMessage($hPic, $STM_GETIMAGE)
If $hObj <> $hDev Then
    _WinAPI_DeleteObject($hDev)
EndIf

; Показать GUI
GUISetState(@SW_SHOW)

; Сохранить 8-битный bitmap в .bmp файл
Local $sPath = FileSaveDialog('Save Image', @TempDir, 'Bitmap Image Files (*.bmp)', 2 + 16, 'MyImage.bmp', $hForm)
If $sPath Then
    _WinAPI_SaveHBITMAPToFile($sPath, $hBitmap, 2834, 2834)
EndIf

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
