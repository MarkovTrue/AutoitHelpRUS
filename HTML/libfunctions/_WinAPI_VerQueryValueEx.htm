Справочник функций

_WinAPI_VerQueryValueEx

Получает текстовую информацию из ресурса информации о версии указанного двоичного модуля.

#include <WinAPIRes.au3>
_WinAPI_VerQueryValueEx ( $hModule [, $sValues = '' [, $iLanguage = $LOCALE_USER_DEFAULT]] )

Параметры

$hModule Дескриптор модуля для получения информации. Также этот параметр может указывать имя
модуля для загрузки; он должен быть полным или относительным путём. Если этот параметр равен 0 или пустой строке, это
эквивалентно передаче дескриптора модуля, используемого для создания текущего процесса.
$sValues [необязательный] Строка, содержащая имена полей, для которых вы хотите получить значения. Имена должны быть разделены символом "|".
Например, "name1|name2|...|namei". Если некоторые поля не существуют, соответствующие элементы массива – пустая строка.
Если этот параметр не задан (пустая строка), используются зарезервированные имена (12) в следующем порядке:
$FV_COMMENTS ("Comments")
$FV_COMPANYNAME ("CompanyName")
$FV_FILEDESCRIPTION ("FileDescription")
$FV_FILEVERSION ("FileVersion")
$FV_INTERNALNAME ("InternalName")
$FV_LEGALCOPYRIGHT ("LegalCopyright")
$FV_LEGALTRADEMARKS ("LegalTrademarks")
$FV_ORIGINALFILENAME ("OriginalFilename")
$FV_PRODUCTNAME ("ProductName")
$FV_PRODUCTVERSION ("ProductVersion")
$FV_PRIVATEBUILD ("PrivateBuild")
$FV_SPECIALBUILD ("SpecialBuild")

Константы определены в FileConstants.au3.
$iLanguage [необязательный] Идентификатор языка ресурса информации о версии. Чтобы получить информацию для
языка пользователя по умолчанию, установите этот параметр на $LOCALE_USER_DEFAULT (по умолчанию). Чтобы получить информацию для всех
языков, расположенных в ресурсе, установите этот параметр на (-1).

Возвращаемое значение

Успех: Возвращает двумерный массив строковых значений, указанных параметром $sValues для каждого языка. Нулевой
элемент массива [0][0] содержит количество языков ресурса. Размер массива (i) равен
количеству указанных параметров + 1. Нулевой элемент массива [n][0] содержит идентификатор языка
(ID).
$aArray[0][0] – количество языков (n)
[0][i] – не используется
[n][0] – ID
[n][i] – значение
Ошибка: Устанавливает @error ≠ 0.

Примечания

Идентификатор (ID) ресурса информации о версии должен быть равен 1, в противном случае функция завершится с ошибкой.

Пример

#include "Extras\HelpFileInternals.au3"

#include <Array.au3>
#include <WinAPILocale.au3>
#include <WinAPIRes.au3>

Local $sDll = _Extras_PathFull('Resources.dll')
Local $aData = _WinAPI_VerQueryValueEx($sDll, 'FileDescription|FileVersion|OriginalFilename', -1)

If Not @error Then
    For $i = 1 To $aData[0][0]
        $aData[$i][0] = _WinAPI_GetLocaleInfo($aData[$i][0], $LOCALE_SLANGUAGE)
    Next
EndIf

_ArrayDisplay($aData, '_WinAPI_VerQueryValueEx')
