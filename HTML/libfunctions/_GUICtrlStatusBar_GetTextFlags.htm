Справочник функций

_GUICtrlStatusBar_GetTextFlags

Получает текст.

#include <GuiStatusBar.au3>
_GUICtrlStatusBar_GetTextFlags ( $hWnd, $iPart )

Параметры

$hWnd Дескриптор элемента управления.
$iPart Индекс части (нумерация с 0).

Возвращаемое значение

Возвращает значение, в котором младшее слово определяет длину текста в символах, старшее слово определяет тип операции, используемой для отрисовки текста. Тип может быть одним из следующих значений:
    0 – текст отрисовывается с рамкой, чтобы казаться ниже плоскости окна
    $SBT_NOBORDERS – текст отрисовывается без рамки
    $SBT_OWNERDRAW – текст отрисовывается родительским окном
    $SBT_POPOUT – текст отрисовывается с рамкой, чтобы казаться выше плоскости окна
    $SBT_RTLREADING – текст отображается в направлении, противоположном тексту в родительском окне

См. также

_GUICtrlStatusBar_GetTextLength, _GUICtrlStatusBar_GetTextLengthEx

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiStatusBar.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("StatusBar Get Text Flags (v" & @AutoItVersion & ")", 450, 320, 100, 100)
    Local $hStatus = _GUICtrlStatusBar_Create($hGUI)

    ; Создать элемент memo
    _MemoCreate(2, 8, 444, 259)
    GUISetState(@SW_SHOW)

    ; Задать части
    Local $aParts[3] = [75, 150, -1]
    _GUICtrlStatusBar_SetParts($hStatus, $aParts)
    _GUICtrlStatusBar_SetText($hStatus, "Part 0", 0, $SBT_POPOUT)
    _GUICtrlStatusBar_SetText($hStatus, "Part 1", 1, $SBT_NOBORDERS)
    _GUICtrlStatusBar_SetText($hStatus, "Part 2", 2, $SBT_NOTABPARSING)

    ; Получить сведения о тексте
    For $iI = 0 To _GUICtrlStatusBar_GetCount($hStatus) - 1
        _MemoWrite("Part " & $iI & " text flags ..: 0x" & Hex(_GUICtrlStatusBar_GetTextFlags($hStatus, $iI)))
        _MemoWrite("Part " & $iI & " text length .: " & _GUICtrlStatusBar_GetTextLength($hStatus, $iI))
        _MemoWrite("Part " & $iI & " drawing type : 0x" & Hex(_GUICtrlStatusBar_GetTextLengthEx($hStatus, $iI)))
        _MemoWrite()
    Next

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
