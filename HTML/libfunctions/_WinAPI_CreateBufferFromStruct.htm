Справочник функций

_WinAPI_CreateBufferFromStruct

выделяет блок памяти из внутреннего кучи библиотеки и инициализирует его данными структуры.

#include <WinAPIMem.au3>
_WinAPI_CreateBufferFromStruct ( $tStruct [, $pBuffer = 0 [, $bAbort = True]] )

Параметры

$tStruct Структура, содержащая данные для инициализации буфера.
$pBuffer Указатель на структуру.
$bAbort Координаты.

Возвращаемое значение

Успех: Возвращает указатель на new буфер.
Ошибка: Ошибка.

Примечания

Эта функция создаёт буфер размером с указанную структуру и копирует в него все её данные.

Когда буфер больше не нужен, его нужно уничтожить вызовом _WinAPI_FreeMemory().
