_WinAPI_SetKeyboardLayout

Задаёт идентификатор входной локали для указанного окна.

#include <WinAPISys.au3>
_WinAPI_SetKeyboardLayout ( $hWnd, $iLanguage [, $iFlags = 0] )

Параметры

$hWnd Дескриптор окна.
$iLanguage Идентификатор входной локали.
$iFlags [необязательный] Новая входная локаль. Этот параметр может быть одним или несколькими из следующих значений.
$INPUTLANGCHANGE_BACKWARD
$INPUTLANGCHANGE_FORWARD
$INPUTLANGCHANGE_SYSCHARSET

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
