Справочник ключевых слов

$tagCOLORSCHEME

Содержит сведения о рисовании кнопок на панели инструментов или rebar.

Global Const $tagCOLORSCHEME = "dword Size;dword BtnHighlight;dword BtnShadow"

Параметры

Size Размер этой структуры в байтах.
BtnHighlight Значение COLORREF, представляющее цвет выделения кнопок. Используйте $CLR_DEFAULT для цвета выделения по умолчанию.
BtnShadow Значение COLORREF, представляющее цвет тени кнопок. Используйте $CLR_DEFAULT для цвета тени по умолчанию.
