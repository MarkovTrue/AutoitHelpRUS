Справочник функций

_WinAPI_OpenProcess

Возвращает дескриптор существующего объекта процесса.

#include <WinAPIProc.au3>
_WinAPI_OpenProcess ( $iAccess, $bInherit, $iPID [, $bDebugPriv = False] )

Параметры

$iAccess Задаёт доступ к объекту процесса.
$bInherit Может ли возвращённый дескриптор быть наследован.
$iPID Задаёт идентификатор процесса, который нужно открыть.
$bDebugPriv [необязательный] Некоторые системные процессы невозможно открыть без привилегии отладки.
Если True, функция попытается открыть процесс с привилегиями отладки, если открыть его со стандартными привилегиями доступа не удаётся.

Возвращаемое значение

Успех: Возвращает дескриптор процесса объекта.
Ошибка: Устанавливает @error ≠ 0.

См. также

_WinAPI_CloseHandle, _WinAPI_ReadProcessMemory, _WinAPI_WriteProcessMemory

Дополнительно

Искать OpenProcess в библиотеке MSDN

Пример

#include <ProcessConstants.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>
#include <WinAPISys.au3>

; _WinAPI_CreateProcess() будет лучшим решением
Local $iPID = Run('cmd.exe /k')
If Not $iPID Then
    Exit
EndIf

; Внимание: процесс открывается сразу
Local $hProcess
If Number(_WinAPI_GetVersion()) >= 6.0 Then
    $hProcess = _WinAPI_OpenProcess($PROCESS_QUERY_LIMITED_INFORMATION, 0, $iPID)
Else
    $hProcess = _WinAPI_OpenProcess($PROCESS_QUERY_INFORMATION, 0, $iPID)
EndIf
If Not $hProcess Then
    Exit
EndIf

; Ожидать, пока процесс существует, попробуйте ввести "exit 6"
While ProcessExists($iPID)
    Sleep(100)
WEnd

ConsoleWrite('Exit code: ' & _WinAPI_GetExitCodeProcess($hProcess) & @CRLF)

_WinAPI_CloseHandle($hProcess)
