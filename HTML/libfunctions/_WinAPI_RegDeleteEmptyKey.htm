_WinAPI_RegDeleteEmptyKey

Удаляет пустой ключ.

#include <WinAPIReg.au3>
_WinAPI_RegDeleteEmptyKey ( $hKey [, $sSubKey = ''] )

Параметры

$hKey Дескриптор открытого ключа реестра или один из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sSubKey [необязательный] Имя удаляемого ключа.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Функция _WinAPI_RegDeleteEmptyKey() не удаляет ключ, если он содержит подключи или значения. Вместо этого используйте _WinAPI_RegDeleteTree().
