Справочник функций

_IE_VersionInfo

Возвращает массив со сведениями о версии IE.au3.

#include <IE.au3>
_IE_VersionInfo ( )

Возвращаемое значение

Возвращает массив $__gaIEAU3VersionInfo:
    $__gaIEAU3VersionInfo[0] = тип выпуска (T – тестовый, V – промышленный)
    $__gaIEAU3VersionInfo[1] = старшая версия
    $__gaIEAU3VersionInfo[2] = младшая версия
    $__gaIEAU3VersionInfo[3] = подверсия
    $__gaIEAU3VersionInfo[4] = дата выпуска (YYYYMMDD)
    $__gaIEAU3VersionInfo[5] = отображаемая версия (например, V3.1-0)

Примечания

Содержимое массива позволяет легко проверить, выше или ниже версия UDF требуемой.

Пример

; Получить и отобразить информацию о версии IE.au3

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $aVersion = _IE_VersionInfo()
MsgBox($MB_SYSTEMMODAL, "IE.au3 Version", $aVersion[5] & " released " & $aVersion[4])
