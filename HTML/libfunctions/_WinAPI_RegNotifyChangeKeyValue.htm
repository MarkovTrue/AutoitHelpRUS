Справочник функций

_WinAPI_RegNotifyChangeKeyValue

Уведомляет вызывающего об изменениях атрибутов или содержимого указанного ключа реестра.

#include <WinAPIReg.au3>
_WinAPI_RegNotifyChangeKeyValue ( $hKey, $iFilter [, $bSubtree = False [, $bAsync = False [, $hEvent = 0]]] )

Параметры

$hKey Дескриптор открытого ключа реестра. Ключ должен быть открыт с правом доступа KEY_NOTIFY.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey(). Это также может быть
один из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$iFilter Отчёты об изменениях, которые должны быть выполнены.
Этот параметр может быть одним или несколькими из следующих значений:
$REG_NOTIFY_CHANGE_NAME
$REG_NOTIFY_CHANGE_ATTRIBUTES
$REG_NOTIFY_CHANGE_LAST_SET
$REG_NOTIFY_CHANGE_SECURITY
$bSubtree [необязательный] Указывает, отчитываться ли об изменениях в подключах указанного ключа, допустимые значения:
True – функция отчитывается об изменениях в указанном ключе и всех его подключах.
False – функция отчитывается об изменениях только в указанном ключе (по умолчанию).
$bAsync [необязательный] Указывает, возвращаться ли немедленно, допустимые значения:
True – функция возвращается немедленно и отчитывается об изменениях по сигналу указанного события.
False – функция не возвращается до возникновения изменения (по умолчанию).
$hEvent [необязательный] Дескриптор события.
Если параметр $fAsync равен True, функция возвращается немедленно и изменения отчитываются по сигналу этого события, иначе этот параметр игнорируется (по умолчанию).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Если указанный ключ закрыт, событие сигнализируется.
Это означает, что приложение не должно зависеть от того, что ключ открыт после возврата из операции ожидания события.

См. также

_WinAPI_RegCreateKey, _WinAPI_RegOpenKey

Пример

#include <Debug.au3>
#include <WinAPIError.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>
#include <WinAPIReg.au3>

Opt('TrayAutoPause', 0)

Example()

Func Example()
    Local $sKey = 'Software\AutoIt v3'
    Local $hKey = _WinAPI_RegOpenKey($HKEY_CURRENT_USER, $sKey, $KEY_NOTIFY)
    If @error Then
        _DebugSetup(Default, True)
        _DebugReport("! RegOpenKey @error =" & @error & @CRLF & @TAB & _WinAPI_GetErrorMessage(@extended))
        Exit
    EndIf
    Local $hEvent = _WinAPI_CreateEvent()
    If Not _WinAPI_RegNotifyChangeKeyValue($hKey, $REG_NOTIFY_CHANGE_LAST_SET, 0, 1, $hEvent) Then
        Exit
    EndIf

    RegWrite("HKCU\" & $sKey, "text", "REG_DWORD", 1) ; для имитации изменения

    While 1
        If Not _WinAPI_WaitForSingleObject($hEvent, 0) Then
            Run(@AutoItExe & ' /AutoIt3ExecuteLine "MsgBox(4096, ''Registry'', ''The registry hive has been modified.'' & @CRLF & @CRLF & ''HKEY_CURRENT_USER\Software\AutoIt v3'', 5)"')
            ExitLoop
        EndIf
        Sleep(100)
    WEnd

    RegDelete("HKCU\" & $sKey, "text")
    _WinAPI_CloseHandle($hEvent)
    _WinAPI_RegCloseKey($hKey)

EndFunc   ;==>Example
