_WinAPI_OemToChar

Преобразует строку из набора символов OEM в строку ANSI.

#include <WinAPIConv.au3>
_WinAPI_OemToChar ( $sStr )

Параметры

$sStr Строка символов из набора символов OEM.

Возвращаемое значение

Успех: Возвращает преобразованную строку.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
