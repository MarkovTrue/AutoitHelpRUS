_ScreenCapture_SaveImage

Сохраняет изображение в файл.

#include <ScreenCapture.au3>
_ScreenCapture_SaveImage ( $sFileName, $hBitmap [, $bFreeBmp = True] )

Параметры

$sFileName Полный путь и расширение растрового файла для сохранения.
$hBitmap Дескриптор HBITMAP.
$bFreeBmp [необязательный] Если True, $hBitmap освобождается при успешном сохранении (по умолчанию)

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция сохраняет растр в файл, преобразуя его в формат, определённый по расширению.
Поддерживаемые форматы: BMP, JPG, PNG, GIF, TIF.

См. также

_ScreenCapture_Capture

Пример

#include <ScreenCapture.au3>

Example()

Func Example()
    Local $hBmp

    ; Захватить весь экран
    $hBmp = _ScreenCapture_Capture("")

    ; Сохранить bitmap в файл
    _ScreenCapture_SaveImage(@MyDocumentsDir & "\GDIPlus_Image.jpg", $hBmp)

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.jpg")
EndFunc   ;==>Example
