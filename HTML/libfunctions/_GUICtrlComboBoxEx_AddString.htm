Справочник функций

_GUICtrlComboBoxEx_AddString

Добавляет строку.

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_AddString ( $hWnd, $sText [, $iImage = -1 [, $iSelectedImage = -1 [, $iOverlayImage = -1 [, $iIndent = -1 [, $iParam = -1]]]]] )

Параметры

$hWnd Дескриптор элемента управления.
$sText Текст элемента. Если задано -1, элемент задаётся через уведомление $CBEN_GETDISPINFO.
$iImage [необязательный] Индекс значка элемента в списке изображений элемента управления (начиная с нуля)
$iSelectedImage [необязательный] Индекс значка элемента в списке изображений элемента управления (начиная с нуля)
$iOverlayImage [необязательный] Индекс значка элемента в списке изображений элемента управления (начиная с нуля)
$iIndent [необязательный] Количество ширин изображения для отступа элемента. Один уровень отступа равен ширине изображения.
$iParam [необязательный] Значение, связанное с элементом

Возвращаемое значение

Успех: Возвращает индекс нового элемента.
Ошибка: Возвращает -1.

См. также

_GUICtrlComboBoxEx_DeleteString, _GUICtrlComboBoxEx_InitStorage

Пример

#include <GuiComboBoxEx.au3>
#include <GUIConstantsEx.au3>
#include <GuiImageList.au3>

Example()

Func Example()
    Local $hGUI, $hImage, $hCombo

    ; Создать GUI
    $hGUI = GUICreate("ComboBoxEx Add String", 400, 300)
    $hCombo = _GUICtrlComboBoxEx_Create($hGUI, "", 2, 2, 394, 100)
    GUISetState(@SW_SHOW)

    $hImage = _GUIImageList_Create(16, 16, 5, 3)
    For $x = 0 To 146
        _GUIImageList_AddIcon($hImage, @SystemDir & "\shell32.dll", $x)
    Next
    _GUIImageList_Add($hImage, _GUICtrlComboBoxEx_CreateSolidBitMap($hCombo, 0xFF0000, 16, 16))
    _GUIImageList_Add($hImage, _GUICtrlComboBoxEx_CreateSolidBitMap($hCombo, 0x00FF00, 16, 16))
    _GUIImageList_Add($hImage, _GUICtrlComboBoxEx_CreateSolidBitMap($hCombo, 0x0000FF, 16, 16))
    _GUICtrlComboBoxEx_SetImageList($hCombo, $hImage)

    _GUICtrlComboBoxEx_InitStorage($hCombo, 150, 300)

    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    For $x = 0 To 149
        _GUICtrlComboBoxEx_AddString($hCombo, StringFormat("%03d : Random string", Random(1, 200, 1)), $x, $x)
    Next
    _GUICtrlComboBoxEx_EndUpdate($hCombo)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
