Справочник функций

_Excel_RangeRead

Читает значение, формулу или отображаемый текст из ячейки или диапазона ячеек указанной книги и листа.

#include <Excel.au3>
_Excel_RangeRead ( $oWorkbook [, $vWorksheet = Default [, $vRange = Default [, $iReturn = 1 [, $bForceFunc = False]]]] )

Параметры

$oWorkbook Объект рабочей книги Excel.
$vWorksheet [необязательный] Имя, индекс или объект читаемого листа. При значении Default используется активный лист (по умолчанию Default).
$vRange [необязательный] Объект Range или диапазон A1. При значении Default обрабатываются все заполненные ячейки (по умолчанию Default).
$iReturn [необязательный] Что возвращать из ячейки:
1 – значение (по умолчанию)
2 – формула
3 – отображаемый текст
4 – Value2 (отличается от Value тем, что не использует типы Currency и Date).
$bForceFunc [необязательный] True – принудительно использовать функцию _ArrayTranspose() вместо метода транспонирования Excel (по умолчанию False).
Подробности см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает данные из указанных ячеек: строку для одной ячейки, одномерный или двумерный массив с отсчётом от нуля для диапазона (см. раздел Примечания).
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом или объектом книги
2 – имя или индекс $vWorksheet недопустимы либо $vWorksheet не является объектом листа; @extended содержит код COM-ошибки
3 – $vRange недопустим; @extended содержит код COM-ошибки
4 – недопустимый параметр $iReturn: должен быть >= 1 и <= 4
5 – ошибка при чтении данных методом транспонирования.
@extended: Содержит код COM-ошибки
6 – превышен максимальный размер массива AutoIt (2^24 = 16 777 216 элементов)
7 – ошибка при чтении данных функцией _ArrayTranspose().
@extended: Содержит код COM-ошибки
8 – $iReturn = 3 возвращает только одну ячейку; $vRange содержит более одной ячейки

Примечания

Параметр 2 игнорируется, если параметр 3 задаёт диапазон на листе, отличном от указанного в параметре 2.
Метод транспонирования Excel имеет недокументированное ограничение на количество ячеек (зависит от версии Excel).
При транспонировании более 65 535 ячеек параметр $bForceFunc принудительно устанавливается в True.
Метод транспонирования Excel не поддерживает ячейки с более чем 255 символами; установите $bForceFunc = True для обхода этого ограничения.
$iReturn = 3 позволяет получить отображаемый текст только одной ячейки – это ограничение Excel.
Если функция возвращает массив, при диапазоне из одного столбца возвращается одномерный массив, в остальных случаях – двумерный.

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Читаем данные из одной ячейки на активном листе указанной книги.
Local $sResult = _Excel_RangeRead($oWorkbook, Default, "A1")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example 1", "Error reading from workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example 1", "Data successfully read." & @CRLF & "Value of cell A1: " & $sResult)

Пример 2

#include "Extras\HelpFileInternals.au3"

#include <Array.au3>
#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Читаем формулы из диапазона ячеек на листе 2 указанной книги.
Local $aResult = _Excel_RangeRead($oWorkbook, 2, "A1:C1", 2)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example 2", "Error reading from workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example 2", "Data successfully read." & @CRLF & "Please click 'OK' to display the formulas of cells A1:C1 of sheet 2.")
_ArrayDisplay($aResult, "Excel UDF: _Excel_RangeRead Example 2 - Cells A1:C1 of sheet 2")

#include "Extras\HelpFileInternals.au3"

#include <Array.au3>
#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then

    _Excel_Close($oExcel)
    Exit
EndIf

; Читаем формулы из диапазона ячеек (все заполненные ячейки столбца A).

MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeRead Example 3", "Data successfully read." & @CRLF & "Please click 'OK' to display all formulas in column A.")
_ArrayDisplay($aResult, "Excel UDF: _Excel_RangeRead Example 3 - Formulas in column A")
