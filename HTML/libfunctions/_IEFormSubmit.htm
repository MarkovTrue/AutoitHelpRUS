Справочник функций

_IEFormSubmit

Отправляет указанную форму.

#include <IE.au3>
_IEFormSubmit ( ByRef $oObject [, $iWait = 1] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application или Form.
$iWait [необязательный] Ждать ли загрузку страницы перед возвратом:
0 – вернуться сразу, не ожидая загрузки
1 – (по умолчанию) дождаться полной загрузки и вернуться

Возвращаемое значение

Успех: Нет.
Ошибка: Возвращает 0 или -1 и устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) - общая ошибка
2 ($_IEStatus_COMError) - ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) - неверный тип данных
4 ($_IEStatus_InvalidObjectType) - неверный тип объекта
6 ($_IEStatus_LoadWaitTimeout) - тайм-аут ожидания загрузки
8 ($_IEStatus_AccessIsDenied) - доступ запрещён
9 ($_IEStatus_ClientDisconnected) - клиент отключился
@extended: Содержит номер недопустимого параметра.

Примечания

Для многих форм HTML недостаточно использовать _IEFormSubmit(), так как часто к кнопке отправки привязан пользовательский JavaScript.
В этих случаях потребуется имитировать щёлчок кнопки отправки вместо использования _IEFormSubmit(). См. пример для действия "click" в _IEAction().
Кроме того, обработка некоторых форм зависит от "значения" кнопки отправки, передаваемого вместе с остальными данными формы (часто когда в форме более одной кнопки отправки, предназначенной для различных результатов).
Эта функция не приведёт к передаче значения кнопки отправки. Решение – использовать действие "click" в _IEAction(), как описано выше.
Если возникли проблемы с автоматическим вызовом _IELoadWait(), установите параметр $iWait в 0 и вызовите _IELoadWait() из вашего скрипта, передав ему объект InternetExplorer.

См. также

_IEFormElementGetCollection, _IEFormElementGetObjByName, _IEFormGetCollection, _IEFormGetObjByName, _IEFormReset, _IELoadWait

Пример

Пример 1

; Открыть браузер с примером формы, заполнить поле формы и отправить форму

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oText = _IEFormElementGetObjByName($oForm, "textExample")
_IEFormElementSetValue($oText, "Hey! It works!")
Sleep(2000)
_IEFormSubmit($oForm)

Пример 2

; Получить ссылку на элемент формы и задать его значение.
; В этом случае отправить запрос в поисковую систему Google

#include <IE.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oForm = _IEFormGetCollection($oIE, 0)
Local $oQuery = _IEFormElementGetCollection($oForm, 4)
_IEFormElementSetValue($oQuery, "AutoIt IE.au3")
Sleep(2000)
_IEFormSubmit($oForm)

Пример 3

; Получить ссылку на элемент формы и задать его значение.
; Вызвать _IELoadWait вручную, если стандартный _IELoadWait имеет проблемы

#include <IE.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oForm = _IEFormGetCollection($oIE, 0)
Local $oQuery = _IEFormElementGetCollection($oForm, 4)
_IEFormElementSetValue($oQuery, "AutoIt IE.au3")
_IEFormSubmit($oForm, 0)
_IELoadWait($oIE)
