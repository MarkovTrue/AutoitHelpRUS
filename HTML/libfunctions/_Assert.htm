Справочник функций

_Assert

Отображает сообщение при невыполнении утверждения (assertion).

#include <Debug.au3>
_Assert ( $sCondition [, $bExit = True [, $iCode = 0x7FFFFFFF [, $sLine = @ScriptLineNumber]]] )

Параметры

$sCondition Условие (выражение в кавычках), которое должно быть истинным.
$bExit [необязательный] Если True (по умолчанию), скрипт завершается.
$iCode [необязательный] Код завершения при прерывании скрипта.
$sLine [необязательный] Отображает номер строки, в которой утверждение не выполнилось. Если значение не изменено, по умолчанию показывается правильная строка.

Возвращаемое значение

Возвращает результат вычисления условия (только если скрипт не завершается).

Примечания

@error и @extended не сбрасываются при возврате.
Если была вызвана _DebugSetup(), вместо MsgBox() используется _DebugReport(),
а $bExit принудительно устанавливается в False, чтобы избежать завершения скрипта.

См. также

_DebugReport, _DebugSetup

Пример

#include <Debug.au3>

_DebugSetup(Default, True)

Example()

Func Example()
    Local $sABC = ""
    #forceref $sABC

    _Assert('$sABC > ""')
    _Assert('$sABC > ""')
EndFunc   ;==>Example
