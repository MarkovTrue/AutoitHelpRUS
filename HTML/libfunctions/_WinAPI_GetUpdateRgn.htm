_WinAPI_GetUpdateRgn

Получает область обновления окна, копируя её в указанную область.

#include <WinAPIGdi.au3>
_WinAPI_GetUpdateRgn ( $hWnd, $hRgn [, $bErase = True] )

Параметры

$hWnd Дескриптор окна.
$hRgn Дескриптор области для получения области обновления.
$bErase [необязательный] Указывает, должен ли фон в области обновления быть стёрт, допустимые значения:
True – фон стирается (по умолчанию).
False – фон остаётся неизменным.

Возвращаемое значение

Успех: Возвращает значение, указывающее на сложность полученной области; может быть одним из следующих значений.
$COMPLEXREGION
$NULLREGION
$SIMPLEREGION
Ошибка: Возвращает 0.

Примечания

_WinAPI_BeginPaint() автоматически проверяет область обновления, поэтому любой вызов _WinAPI_GetUpdateRgn(), выполненный
сразу после вызова _WinAPI_BeginPaint(), получает пустую область обновления.
