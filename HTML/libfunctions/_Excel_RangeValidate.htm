Справочник функций

_Excel_RangeValidate

Добавляет проверку данных для указанного диапазона.

#include <Excel.au3>
_Excel_RangeValidate ( $oWorkbook, $vWorksheet, $vRange, $iType, $sFormula1 [, $iOperator = Default [, $sFormula2 = Default [, $bIgnoreBlank = True [, $iAlertStyle = $xlValidAlertStop [, $sErrorMessage = Default [, $sInputMessage = Default]]]]]] )

Параметры

$oWorkbook Объект рабочей книги Excel.
$vWorksheet Имя, индекс или объект листа. При значении Default используется активный лист.
$vRange Объект Range или диапазон A1.
$iType Тип проверки данных. Константа XlDVType.
$sFormula1 Значение первой части условия проверки.
$iOperator [необязательный] Оператор условия проверки. Константа XlFormatConditionOperator (по умолчанию Default).
$sFormula2 [необязательный] Вторая часть условия проверки при $iOperator = xlBetween или xlNotBetween (по умолчанию Default).
$bIgnoreBlank [необязательный] True – пустые ячейки не вызывают ошибку (по умолчанию True).
$iAlertStyle [необязательный] Значок в сообщении об ошибке проверки. Константа XlDVAlertStyle (по умолчанию $xlValidAlertStop).
$sErrorMessage [необязательный] Текст сообщения об ошибке (по умолчанию Default = без сообщения).
$sInputMessage [необязательный] Текст подсказки ввода (по умолчанию Default = без подсказки).

Возвращаемое значение

Успех: Возвращает объект диапазона.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом или объектом книги
2 – имя или индекс $vWorksheet недопустимы либо $vWorksheet не является объектом листа; @extended содержит код COM-ошибки
3 – $vRange недопустим.
@extended: Содержит код COM-ошибки
4 – ошибка метода проверки данных.
@extended: Содержит код COM-ошибки.

Примечания

Для отображения сообщения об ошибке $Excel_DisplayAlerts должен быть установлен в True.
Для проверки списком ($xlValidateList) $sFormula1 должна начинаться с "=".
Существующая проверка данных удаляется перед добавлением новой.

Пример

Пример 1

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open(True, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookNew($oExcel)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example", "Error creating new workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Добавляем проверку данных к диапазону. Допустимы только значения от 1 до 10.
_Excel_RangeValidate($oWorkbook, Default, "A1:A10", $xlValidateList, "Yes;No;Don't know")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example 1", "Error setting range validation." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example 1", "Validation against a list of values successfully set for range 'A1:A10'.")

    _Excel_Close($oExcel)
    Exit
EndIf

; Добавляем проверку списком для ячейки. Допустимы только элементы из списка.
; Добавляем проверку списком для ячейки. Допустимы только значения из указанного диапазона.
Local $aValidation[] = ["10", "20", "30", "40", "50"]
_Excel_RangeWrite($oWorkbook, Default, $aValidation, "D1")
; Добавляем проверку списком для ячейки с сообщением об ошибке.
_Excel_RangeValidate($oWorkbook, Default, "C:C", $xlValidateList, "=D:D")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example 2", "Error setting range validation." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example 2", "Validation against a list of values in cell range '=D:D' successfully set for colum 'C'.")

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open(True, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

EndIf

; Показываем оповещение об ошибке.

If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example 3", "Error setting range validation." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeValidate Example 3", "Only numeric values are valid in column 'B'.")
