Справочник функций

_WinAPI_CreateObjectID

создаёт или получает идентификатор объекта для указанного файла или каталога.

#include <WinAPIFiles.au3>
_WinAPI_CreateObjectID ( $sFilePath )

Параметры

$sFilePath Путь к файлу или каталогу, для которого нужно создать или получить идентификатор объекта.

Возвращаемое значение

Успех: Возвращает структуру $tagGUID с идентификатором объекта файла или каталога в пределах того тома, на котором он находится.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Если у файла или каталога ещё нет идентификатора объекта, _WinAPI_CreateObjectID() создаёт его.
Если идентификатор уже есть, функция просто возвращает его.

Дополнительно

Искать FSCTL_CREATE_OR_GET_OBJECT_ID в библиотеке MSDN

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIConv.au3>
#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>
#include <WinAPIShPath.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

; Создать временный файл
Local $sFile = _WinAPI_GetTempFileName(@TempDir)

; Создать уникальный object ID для файла
Local $tGUID = _WinAPI_CreateObjectID($sFile)

ConsoleWrite('GUID: ' & _WinAPI_StringFromGUID($tGUID) & @CRLF)

; Открыть файл по object ID и получить полный путь
Local $hFile = _WinAPI_OpenFileById(_WinAPI_PathStripToRoot(@TempDir), $tGUID, 0, BitOR($FILE_SHARE_DELETE, $FILE_SHARE_READ, $FILE_SHARE_WRITE))
$sFile = _WinAPI_GetFinalPathNameByHandleEx($hFile)
_WinAPI_CloseHandle($hFile)

ConsoleWrite('Path: ' & StringRegExpReplace($sFile, '\\+.\\', '') & @CRLF)

; Удалить файл
FileDelete($sFile)
