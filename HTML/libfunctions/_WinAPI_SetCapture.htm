_WinAPI_SetCapture

Устанавливает захват мыши для указанного окна, принадлежащего текущему потоку.

#include <WinAPISys.au3>
_WinAPI_SetCapture ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

Успех: Возвращает дескриптор окна, которое ранее захватило мышь.
Ошибка: Возвращает 0.
