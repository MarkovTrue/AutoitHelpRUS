Справочник ключевых слов

$tagHDITEM

Содержит сведения об элементе заголовочного элемента управления.

Global Const $tagHDITEM = "int Mask;int XY;ptr Text;hwnd hBMP;int TextMax;int Fmt;int Param;int Image;int Order;int Type;ptr pFilter;int State"

Параметры

Mask Флаги, указывающие, какие другие члены структуры содержат действительные данные или должны быть заполнены.
XY Ширина или высота элемента.
Text Адрес строки элемента.
hBMP Дескриптор точечного рисунка элемента.
TextMax Длина строки элемента.
Fmt Флаги, определяющие формат элемента.
Param Данные элемента, определённые приложением.
Image Индекс изображения в списке изображений (с отсчётом от нуля).
Order Порядок расположения элемента в элементе управления заголовком слева направо.
Type Тип фильтра, определённый элементом pFilter.
pFilter Адрес элемента данных, определённого приложением.
State Состояние элемента.
