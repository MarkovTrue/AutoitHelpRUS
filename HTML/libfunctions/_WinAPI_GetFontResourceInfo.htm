_WinAPI_GetFontResourceInfo

Получает имя шрифта из указанного файла ресурса шрифта.

#include <WinAPIGdi.au3>
_WinAPI_GetFontResourceInfo ( $sFont [, $bForce = False [, $iFlag = Default]] )

Параметры

$sFont Строка, которая называет файл ресурса шрифта.
Для получения имени шрифта, чья информация поступает из нескольких файлов ресурсов, они должны быть разделены на "|".
Например, abcxxxxx.pfm | abcxxxxx.pfb.
$bForce [необязательный] Указывает, нужно ли добавлять файл в таблицу шрифтов. Допустимые значения:
True – добавить указанный файл в таблицу системных шрифтов и удалить его после получения имени шрифта.
False – не добавлять и не удалять (по умолчанию).
$iFlag [необязательный] Целое число. См. _WinAPI_GetFontMemoryResourceInfo() для определения значения.

Возвращаемое значение

Успех: Возвращает имя шрифта или информация о шрифте в соответствии с $iFlag.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

не задокументировано в MSDN

Пример

Пример 1

#include <Array.au3>
#include <File.au3>
#include <WinAPIGdi.au3>
#include <WinAPIShellEx.au3>

Example()

Func Example()
    Local $aFileList = _FileListToArray(_WinAPI_ShellGetSpecialFolderPath($CSIDL_FONTS), '*.ttf', 1)
    Local $aFontList[UBound($aFileList) - 1][2]

    For $i = 1 To $aFileList[0]
        $aFontList[$i - 1][0] = $aFileList[$i]
        $aFontList[$i - 1][1] = _WinAPI_GetFontResourceInfo($aFileList[$i], 1)
    Next

    _ArrayDisplay($aFontList, '_WinAPI_GetFontResourceInfo')

EndFunc   ;==>Example

Func FontGetInfoFromFile($sFile, $n, $sElement)
    Local $s = _WinAPI_GetFontResourceInfo($sFile, Default, $n)
    If Not @error And $s Then ConsoleWrite($sElement & " = " & $s & @CRLF)
EndFunc   ;==>FontGetInfoFromFile
