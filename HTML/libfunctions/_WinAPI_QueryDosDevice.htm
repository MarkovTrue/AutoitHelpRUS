_WinAPI_QueryDosDevice

Получает текущее отображение для определённого имени устройства MS-DOS.

#include <WinAPIFiles.au3>
_WinAPI_QueryDosDevice ( $sDevice )

Параметры

$sDevice Имя устройства MS-DOS.

Возвращаемое значение

Успех: Возвращает текущее отображение для указанного устройства. Если параметр $sDevice является пустой строкой, возвращает массив
всех существующих имён устройств MS-DOS (например, "\Device\HarddiskVolume1" или "\Device\Floppy0").
Нулевой элемент массива содержит количество имён.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Искать QueryDosDevice в библиотеке MSDN

Пример

#include <WinAPIFiles.au3>

Local $aDrive = DriveGetDrive('ALL')

If IsArray($aDrive) Then
    For $i = 1 To $aDrive[0]
        ConsoleWrite(StringUpper($aDrive[$i]) & ' => ' & _WinAPI_QueryDosDevice($aDrive[$i]) & @CRLF)
    Next
EndIf
