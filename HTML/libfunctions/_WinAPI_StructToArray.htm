_WinAPI_StructToArray

Преобразует структуру в массив строк.

#include <WinAPIMisc.au3>
_WinAPI_StructToArray ( $tStruct [, $iItems = 0] )

Параметры

$tStruct Структура для преобразования. Эта структура должна быть такой же, как для _WinAPI_ArrayToStruct().
$iItems [необязательный] Количество строк, содержащихся в структуре. Если этот параметр равен 0 (По умолчанию), конец структуры
определяется двойным нулевым завершающим символом ("... ;{0};{0}").

Возвращаемое значение

Успех: Возвращает массив строк. Нулевой элемент массива содержит количество строк.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Эта функция работает только со строками Unicode.
