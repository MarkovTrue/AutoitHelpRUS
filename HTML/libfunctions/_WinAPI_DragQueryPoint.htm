_WinAPI_DragQueryPoint

Получает координаты позиции мыши на экране, когда был отпущен файл в операции drag-and-drop.

#include <WinAPISysWin.au3>
_WinAPI_DragQueryPoint ( $hDrop )

Параметры

$hDrop Дескриптор структуры drop, описывающей перетащенный файл. Параметр передаётся в сообщение WM_DROPFILES через WPARAM.

Возвращаемое значение

Успех: Возвращает структуру $tagPOINT с координатами курсора мыши на момент переноса файла.
Ошибка: Устанавливает @error ≠ 0.
