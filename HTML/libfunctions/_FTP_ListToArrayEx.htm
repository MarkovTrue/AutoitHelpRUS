Справочник функций

_FTP_ListToArrayEx

Возвращает имена, размеры, атрибуты и временные метки файлов/папок текущей удалённой папки.

#include <FTPEx.au3>
_FTP_ListToArrayEx ( $hFTPSession [, $iReturnType = 0 [, $iFlags = $INTERNET_FLAG_NO_CACHE_WRITE [, $iFmt = 1 [, $iContext = 0]]]] )

Параметры

$hFTPSession возвращается функцией _FTP_Connect().
$iReturntype [необязательный] 0 – файлы и папки, 1 – только папки, 2 – только файлы.
$iFlags [необязательный] по умолчанию = $INTERNET_FLAG_NO_CACHE_WRITE (0x04000000). См. _FTP_FindFileFirst().
$iFmt [необязательный] Формат строк дат:
1 = yyyy/mm/dd
0 = mm/dd/yyyy
$iContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает двумерный массив:
$Array[0][0] = количество найденных элементов
...
$Array[n][0] = имя файла
$Array[n][1] = размер файла
$Array[n][2] = атрибуты файла
$Array[n][3] = дата и время изменения
$Array[n][4] = дата и время создания
$Array[n][5] = дата и время доступа.
Ошибка: $Array[0][0] = 0.

См. также

_FTP_Connect, _FTP_SetStatusCallback, _FTP_ListToArray, _FTP_ListToArray2D

Пример

#include <Array.au3>
#include <FTPEx.au3>
#include <MsgBoxConstants.au3>

;~ Local $sServer = 'ftp.cs.brown.edu' ; Brown Computer Science
Local $sServer = 'speedtest.tele2.net' ; Сервис Tele2 Speedtest
Local $sUsername = ''
Local $sPass = ''

Local $hOpen = _FTP_Open('MyFTP Control', 0)
If Not @error Then
    ; passive разрешает ответ большинству защищённых FTP
    Local $hConn = _FTP_Connect($hOpen, $sServer, $sUsername, $sPass)
    If Not @error Then
        Local $aFile = _FTP_ListToArrayEx($hConn, 0)
        If Not @error Then
            _ArrayDisplay($aFile)
        Else
            MsgBox($MB_SYSTEMMODAL, "Error", '_FTP_ListToArrayEx($Conn, 0)' & @CRLF & _
                    '@error = ' & @error & ' @extended = ' & @extended)
        EndIf
        Local $iFtpc = _FTP_Close($hConn)
    Else
        MsgBox($MB_SYSTEMMODAL, "Error", '_FTP_Connect($Open, ' & $sServer & ', ' & $sUsername & ', ' & $sPass & ')' & @CRLF & _
                '@error = ' & @error & ' @extended = ' & @extended)
    EndIf

    Local $iFtpo = _FTP_Close($hOpen)
Else
    MsgBox($MB_SYSTEMMODAL, "Error", "_FTP_Open('MyFTP Control')" & @CRLF & _
            '@error = ' & @error & ' @extended = ' & @extended)
EndIf
