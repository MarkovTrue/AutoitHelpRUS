Справочник функций

_WinAPI_DllInstall

Регистрирует OLE-элементы управления: файлы DLL или ActiveX Controls (OCX).

#include <WinAPIReg.au3>
_WinAPI_DllInstall ( $sFilePath )

Параметры

$sFilePath Первое слово предложения path к DLL файл который/которая will be registered.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
