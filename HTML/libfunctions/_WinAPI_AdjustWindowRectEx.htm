_WinAPI_AdjustWindowRectEx

Вычисляет требуемый размер прямоугольника окна на основе требуемого размера прямоугольника клиентской области.

#include <WinAPISysWin.au3>
_WinAPI_AdjustWindowRectEx ( ByRef $tRECT, $iStyle [, $iExStyle = 0 [, $bMenu = False]] )

Параметры

$tRECT Структура $tagRECT, содержащая координаты требуемой клиентской области. Эта структура должна быть создана перед вызовом функции.
$iStyle Стиль окна, для которого должен быть вычислен требуемый размер. Учтите, что вы не можете указать стиль $WS_OVERLAPPED.
$iExStyle [необязательный] Расширенный стиль окна, для которого должен быть вычислен требуемый размер.
$bMenu [необязательный] Определяет, имеет ли окно меню, допустимые значения:
True – окно имеет меню.
False – окно не имеет меню (по умолчанию).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
