Справочник функций

_WinAPI_SetCursor

Устанавливает форму курсора.

#include <WinAPIRes.au3>
_WinAPI_SetCursor ( $hCursor )

Параметры

$hCursor Идентифицирует курсор.

Возвращаемое значение

Успех: Возвращает дескриптор предыдущего курсора, если он был.
Ошибка: Возвращает 0, если предыдущего курсора не было.
