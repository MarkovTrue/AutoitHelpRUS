_WinAPI_GetBitmapBits

Копирует биты растрового изображения указанного устройства-зависимого растра в буфер.

#include <WinAPIGdi.au3>
_WinAPI_GetBitmapBits ( $hBitmap, $iSize, $pBits )

Параметры

$hBitmap Дескриптор растрового изображения, зависимого от устройства.
$iSize Количество байтов для копирования из растра в буфер.
$pBits Указатель на буфер для получения бит указанного bitmap.

Возвращаемое значение

Успех: Возвращает количество байт, скопированных в буфер.
Ошибка: Возвращает 0.
