_WinAPI_RadialGradientFill

Заполняет радиальный градиент.

#include <WinAPIGdi.au3>
_WinAPI_RadialGradientFill ( $hDC, $iX, $iY, $iRadius, $iRGB1, $iRGB2 [, $fAngleStart = 0 [, $fAngleEnd = 360 [, $fStep = 5]]] )

Параметры

$hDC Дескриптор контекста устройства.
$iX Координата X центральной точки в логических единицах.
$iY Координата Y центральной точки в логических единицах.
$iRadius Радиус круга для заполнения градиентом.
$iRGB1 Информация о цвете в центральной точке.
$iRGB2 Информация о цвете на краях круга.
$fAngleStart [необязательный] Угол для начала заполнения в градусах. По умолчанию 0.
$fAngleEnd [необязательный] Угол для окончания заполнения в градусах. По умолчанию 360.
$fStep [необязательный] Шаг заполнения градиента в градусах. Чем больше значение этого параметра, тем лучше градиент,
но требуется больше времени, и наоборот. По умолчанию 5.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Эта функция не заполняет градиент с прозрачностью и не использует сглаживание.

Пример

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400)
Local $idPic = GUICtrlCreatePic('', 0, 0, 400, 400)
Local $hPic = GUICtrlGetHandle($idPic)

; Создать градиент
Local $hDC = _WinAPI_GetDC($hPic)
Local $hDestDC = _WinAPI_CreateCompatibleDC($hDC)
Local $hBitmap = _WinAPI_CreateCompatibleBitmapEx($hDC, 400, 400, 0xFFFFFF)
Local $hDestSv = _WinAPI_SelectObject($hDestDC, $hBitmap)
For $i = 0 To 315 Step 45
    _WinAPI_RadialGradientFill($hDestDC, 200, 200, 180, Random(0, 0xFFFFFF, 1), 0xFFFFFF, $i, $i + 45)
Next
_WinAPI_ReleaseDC($hPic, $hDC)
_WinAPI_SelectObject($hDestDC, $hDestSv)
_WinAPI_DeleteDC($hDestDC)

; Установить градиент для элемента управления
_SendMessage($hPic, $STM_SETIMAGE, 0, $hBitmap)
Local $hObj = _SendMessage($hPic, $STM_GETIMAGE)
If $hObj <> $hBitmap Then
    _WinAPI_DeleteObject($hBitmap)
EndIf

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
