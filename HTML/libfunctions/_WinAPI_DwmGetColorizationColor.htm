Справочник функций

_WinAPI_DwmGetColorizationColor

Получает текущий цвет, используемый для стеклянной композиции Диспетчера окон рабочего стола (DWM).

#include <WinAPIGdi.au3>
_WinAPI_DwmGetColorizationColor ( )

Возвращаемое значение

Успех: Возвращает текущий цвет (0xAARRGGBB), используемый для glass-композиции; @extended указывает, является ли цвет непрозрачным смешением:
1 – непрозрачное смешение.
0 – иначе.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

Дополнительно

Искать DwmGetColorizationColor в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIGdi.au3>

If Not _WinAPI_DwmIsCompositionEnabled() Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later with enabled Aero theme.')
    Exit
EndIf

Local $iColor = _WinAPI_DwmGetColorizationColor()
Local $iBlend = @extended

ConsoleWrite('Color for glass composition: 0x' & Hex($iColor) & @CRLF)
ConsoleWrite('Transparency: ' & (Not $iBlend) & @CRLF)
