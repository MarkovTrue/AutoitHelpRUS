Справочник функций

_Array1DToHistogram

Преобразует одномерный 1D массив в гистограмму.

#include <Array.au3>
_Array1DToHistogram ( $aArray [, $iSizing = 100] )

Параметры

$aArray Массив для преобразования.
$iSizing [необязательный] Масштаб гистограммы. По умолчанию 100 процентов.

Возвращаемое значение

Успех: Возвращает преобразованный массив.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – $aArray не является одномерным массивом.

Примечания

Выравнивание отрицательных значений по правому краю может быть неточным из-за использования пропорционального шрифта.

Пример

Пример 1

#include <Array.au3>

Local $aNEW[] = [0, -80, -49, -44, 80, 100, 8, 7, 6, 5, 4, 3, 2, 1]
Local $aArray = _Array1DToHistogram($aNEW, 125)
_ArrayDisplay($aArray, "_Array1DToHistogram")

Пример 2

#include <Array.au3>

Local $iLimit = 500
Local $aValues[51]
Local $t, $n

For $i = 1 To $iLimit
    $t = TimerInit()
    Sleep(10)
    $n = Round(TimerDiff($t))
    $aValues[$n] += 1
Next
Local $aArray = _Array1DToHistogram($aValues)
_ArrayDisplay($aArray, "_Array1DToHistogram", "10:20")
