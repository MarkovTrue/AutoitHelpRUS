Справочник функций

_SQLite_GetTable2D

Возвращает двумерный массив, содержащий имена столбцов и данные выполненного запроса.

#include <SQLite.au3>
_SQLite_GetTable2D ( $hDB, $sSQL, ByRef $aResult, ByRef $iRows, ByRef $iColumns [, $iCharSize = -1 [, $bSwichDimensions = False]] )

Параметры

$hDB Открытая база данных, используйте -1 для последней открытой базы данных.
$sSQL Выполняемый SQL-запрос.
$aResult Выходной параметр – результат (см. раздел Примечания для получения только строк данных без заголовка).
$iRows Возвращает количество строк данных.
$iColumns Выходной параметр – количество столбцов.
$iCharSize [необязательный] Указывает максимальный размер поля данных
$bSwichDimensions [необязательный] Переключает размерности $aResult

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове _SQLite_Query()
2 – ошибка при вызове SQLite API "sqlite3_free_table"
3 – вызов запрещён безопасным режимом
4 – прерывание или @error, установленный обратным вызовом (@extended установлена ошибка SQLite)

Примечания

Количество значений, вставляемых в $aResult, будет (($iRows) + 1) * ($iColumns).
NULL будет возвращён как число 0.
Эта функция использует больше памяти, чем _SQLite_Query() / _SQLite_Fetch*()... но она быстрее.
Если вам не нужен результат (или результата не будет), рассмотрите использование SQLite_Exec().

Если вы не хотите получить заголовок, просто используйте _SQLite_GetTable().

См. также

_SQLite_Display2DResult, _SQLite_Exec, _SQLite_GetTable, _SQLite_GetTableData2D, _SQLite_Query

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $aResult, $iRows, $iColumns, $iRval

_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; Открыть :memory: базу данных
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "Can't Load Database!")
    Exit -1
EndIf

; Пример таблицы
; Имя | Возраст
; -----------------------
; Алиса | 43
; Боб | 28
; Синди | 21

If Not _SQLite_Exec(-1, "CREATE TEMP TABLE persons (Name, Age);") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Alice','43');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Bob','28');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Cindy','21');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())

; Запрос
$iRval = _SQLite_GetTable2D(-1, "SELECT * FROM persons;", $aResult, $iRows, $iColumns)
If $iRval = $SQLITE_OK Then
    _SQLite_Display2DResult($aResult)

    ; $aResult выглядит так:
    ; Имя Возраст
    ; Alice 43
    ; Bob 28
    ; Cindy 21
    ; Если размерности будут переключены в _SQLite_GetTable2D, результат будет выглядеть так:
    ; Name Alice Bob Cindy
    ; Возраст 43 28 21

Else
    MsgBox($MB_SYSTEMMODAL, "SQLite Error: " & $iRval, _SQLite_ErrMsg())
EndIf

_SQLite_Close()
_SQLite_Shutdown()
