Справочник функций

_IE_Introduction

Отображает вводную информацию об IE.au3 в новом окне браузера.

#include <IE.au3>
_IE_Introduction ( [$sModule = "basic"] )

Параметры

$sModule [необязательный] задаёт, какой модуль запустить
"basic" = (по умолчанию) базовое введение

Возвращаемое значение

Успех: Возвращает объектную переменную, указывающую на объект InternetExplorer.Application.
Ошибка: Устанавливает @error ≠ 0.
@error: 5 ($_IEStatus_InvalidValue) – неверное значение.
@extended: Содержит номер недопустимого параметра.

Примечания

Справочник AutoIt в настоящее время не позволяет UDF иметь общую документацию, охватывающую серию функций; вместо этого документация UDF сосредоточена на отдельных функциях.
Данная функция предоставляет такую общую документацию. Со временем она будет расширяться.
