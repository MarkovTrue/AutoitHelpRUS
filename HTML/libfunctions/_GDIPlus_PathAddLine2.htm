_GDIPlus_PathAddLine2

Добавляет последовательность линий к текущей фигуре пути.

#include <GDIPlus.au3>
_GDIPlus_PathAddLine2 ( $hPath, $aPoints )

Параметры

$hPath Указатель на объект GraphicsPath.
$aPoints Массив точек, задающих линии:
[0][0] – количество точек
[1][0] – X-координата точки 1
[1][1] – Y-координата точки 1
[2][0] – X-координата точки 2
[2][1] – Y-координата точки 2
[n][0] – X-координата точки n
[n][1] – Y-координата точки n

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_PathAddLine

Дополнительно

Искать GdipAddPathLine2 в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphics, $hPath, $hCustomLineCap, $hClonedLineCap, $hPen
    Local $avPoints[4][2] = [[3], [-15, -15], [0, 0], [15, -15]]

    ; Инициализировать GDI+
    _GDIPlus_Startup()

    ;создать объект Graphics из дескриптора окна
    $hGUI = GUICreate("_GDIPlus_CustomLineCapCreate Example", 400, 200)
    GUISetState(@SW_SHOW)

    $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI)

    ;задаёт качество отрисовки графики (сглаживание)
    _GDIPlus_GraphicsSetSmoothingMode($hGraphics, $GDIP_SMOOTHINGMODE_HIGHQUALITY)

    ; Создать GraphicsPath и добавить в него две линии.
    $hPath = _GDIPlus_PathCreate()
    _GDIPlus_PathAddLine2($hPath, $avPoints)

    ; Создать объект CustomLineCap.
    $hCustomLineCap = _GDIPlus_CustomLineCapCreate(0, $hPath)

    ; Создать клон объекта CustomLineCap.
    $hClonedLineCap = _GDIPlus_CustomLineCapClone($hCustomLineCap)

    ; Создать объект Pen, назначить клонированный cap как кастомный end cap и нарисовать линию.
    $hPen = _GDIPlus_PenCreate(0xFFFF0000)
    _GDIPlus_PenSetCustomEndCap($hPen, $hClonedLineCap)

    _GDIPlus_GraphicsDrawLine($hGraphics, 50, 50, 350, 150, $hPen)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Очистка
    _GDIPlus_PenDispose($hPen)
    _GDIPlus_CustomLineCapDispose($hClonedLineCap)
    _GDIPlus_CustomLineCapDispose($hCustomLineCap)
    _GDIPlus_PathDispose($hPath)
    _GDIPlus_GraphicsDispose($hGraphics)

    ; Завершить работу GDI+
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
