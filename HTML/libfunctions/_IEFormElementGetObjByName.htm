Справочник функций

_IEFormElementGetObjByName

Возвращает ссылку на элемент формы по имени.

#include <IE.au3>
_IEFormElementGetObjByName ( ByRef $oObject, $sName [, $iIndex = 0] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application или Form.
$sName Имя элемента формы, который нужно найти.
$iIndex [необязательный] Если имя Form Element встречается больше одного раза, задаёт экземпляр по индексу (с нуля)
0 (по умолчанию) или положительное целое – возвращает экземпляр по индексу
-1 – возвращает коллекцию указанных Form Elements

Возвращаемое значение

Успех: Возвращает объектную переменную, ссылающуюся на элемент формы; @extended – количество форм.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
7 ($_IEStatus_NoMatch) – нет совпадения
@extended: Содержит номер недопустимого параметра.

Примечания

Если у нужного элемента формы нет имени или ID, используйте _IEFormElementGetCollection(), чтобы получить ссылку по индексу.

См. также

_IEFormElementGetCollection, _IEFormElementGetValue, _IEFormElementSetValue, _IEFormSubmit

Пример

; Получить ссылку на конкретный элемент формы по имени.
; В этом случае отправить запрос в поисковую систему Google

#include <IE.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oForm = _IEFormGetObjByName($oIE, "gbqf")
Local $oQuery = _IEFormElementGetObjByName($oForm, "q")
_IEFormElementSetValue($oQuery, "AutoIt IE.au3")
_IEFormSubmit($oForm)
