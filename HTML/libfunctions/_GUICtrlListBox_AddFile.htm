_GUICtrlListBox_AddFile

Добавляет указанный файл, содержащий список каталога.

#include <GuiListBox.au3>
_GUICtrlListBox_AddFile ( $hWnd, $sFilePath )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$sFilePath Имя файла для добавления.

Возвращаемое значение

Успех: Возвращает индекс (начиная с 0) добавленного файла.
Ошибка: Возвращает -1.

См. также

_GUICtrlListBox_AddString, _GUICtrlListBox_DeleteString, _GUICtrlListBox_InitStorage

Пример

#include <GUIConstantsEx.au3>
#include <GuiListBox.au3>

Example()

Func Example()
    Local $idListBox

    ; Создать GUI
    GUICreate("List Box Add File", 400, 296)
    $idListBox = GUICtrlCreateList("", 2, 2, 396, 296)
    GUISetState(@SW_SHOW)

    ; Добавить файл
    _GUICtrlListBox_AddFile($idListBox, @WindowsDir & "\notepad.exe")

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
