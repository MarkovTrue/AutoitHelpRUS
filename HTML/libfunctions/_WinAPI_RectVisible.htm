_WinAPI_RectVisible

Определяет, находится ли какая-либо часть указанного прямоугольника в области отсечения.

#include <WinAPIGdi.au3>
_WinAPI_RectVisible ( $hDC, $tRECT )

Параметры

$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT, содержащая логические координаты указанного прямоугольника.

Возвращаемое значение

Успех: Возвращает 1, 2 – любая часть заданного прямоугольника находится в области отсечения.
0 – в противном случае.
Ошибка: Устанавливает @error ≠ 0, флаг @extended может содержать код ошибки.
