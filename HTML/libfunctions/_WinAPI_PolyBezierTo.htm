_WinAPI_PolyBezierTo

Рисует одну или несколько кривых Безье.

#include <WinAPIGdi.au3>
_WinAPI_PolyBezierTo ( $hDC, Const ByRef $aPoint [, $iStart = 0 [, $iEnd = -1]] )

Параметры

$hDC Дескриптор контекста устройства.
$aPoint Двумерный массив ([x1, y1], [x2, y2], ... [xN, yN]), который содержит конечные точки и контрольные точки
кривой(ых), в логических единицах. Количество точек должно быть в три раза больше количества кривых для рисования,
потому что каждая кривая Безье требует две контрольные точки и конечную точку.
$iStart [необязательный] Индекс массива для начала рисования.
$iEnd [необязательный] Индекс массива для остановки рисования.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция рисует кубические кривые Безье, используя контрольные точки, указанные параметром $aPoint.
Первая кривая рисуется от текущей позиции к третьей точке, используя первые две точки как контрольные точки.
Для каждой последующей кривой функция нуждается ровно в трёх дополнительных точках и использует конечную точку предыдущей
кривой как начальную точку для следующей.

Текущая позиция перемещается на конечную точку последней кривой Безье. Фигура не заполняется.

Эта функция рисует линии, используя текущее перо.
