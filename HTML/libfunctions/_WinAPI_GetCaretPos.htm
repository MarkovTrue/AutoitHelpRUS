_WinAPI_GetCaretPos

Получает положение каретки.

#include <WinAPIRes.au3>
_WinAPI_GetCaretPos ( )

Возвращаемое значение

Успех: Возвращает массив, содержащий клиентские координаты каретки.
$aArray[0] – координата x каретки.
$aArray[1] – координата y каретки.
Ошибка: Устанавливает @error ≠ 0.

Дополнительно

Искать GetCaretPos в библиотеке MSDN

Пример

#include <Debug.au3>
#include <WinAPIRes.au3>

_DebugSetup('Test', True)

GUICreate("test caret")
GUICtrlCreateInput("", 10, 10)
GUISetState()

Local $aCaret = _WinAPI_GetCaretPos()

_DebugReportVar('aCaret', $aCaret, True)
