_NamedPipes_CreatePipe

Создаёт анонимный канал.

#include <NamedPipes.au3>
_NamedPipes_CreatePipe ( ByRef $hReadPipe, ByRef $hWritePipe [, $tSecurity = 0 [, $iSize = 0]] )

Параметры

$hReadPipe Переменная, которая получает дескриптор чтения для канала.
$hWritePipe Переменная, которая получает дескриптор записи для канала.
$tSecurity [необязательный] структура $tagSECURITY_ATTRIBUTES, которая определяет, может ли возвращённый дескриптор быть унаследован дочерними процессами.
Если 0 – дескрипторы не могут быть унаследованы.
$iSize [необязательный] Размер буфера для канала в байтах.
Если 0 – система использует размер буфера по умолчанию.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
