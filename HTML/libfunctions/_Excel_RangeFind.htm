_Excel_RangeFind

Находит ячейки, соответствующие критерию поиска, в диапазоне или книге и возвращает массив с информацией о найденных ячейках.

#include <Excel.au3>
_Excel_RangeFind ( $oWorkbook, $sSearch [, $vRange = Default [, $iLookIn = $xlValues [, $iLookAt = $xlPart [, $bMatchcase = False]]]] )

Параметры

$oWorkbook Объект книги.
$sSearch Строка поиска. Может быть строкой (допустимы символы подстановки *?~) или любым типом данных Excel. См. раздел Примечания.
$vRange [необязательный] Объект Range, диапазон A1 (строка) или Default для поиска по всем листам книги (по умолчанию Default).
$iLookIn [необязательный] Задаёт область поиска. Константа XLFindLookIn (по умолчанию $xlValues).
$iLookAt [необязательный] Задаёт режим сопоставления: целиком или частично. Константа XLLookAt (по умолчанию $xlPart).
$bMatchcase [необязательный] True – с учётом регистра, False – без учёта регистра (по умолчанию False).

Возвращаемое значение

Успех: Возвращает двумерный массив с отсчётом от нуля, содержащий следующие данные:
0 – имя листа
1 – имя ячейки
2 – адрес ячейки
3 – значение ячейки
4 – формула ячейки
5 – комментарий ячейки
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом или объектом книги
2 – $sSearch пустой
3 – $vRange недопустим.
@extended: Содержит код COM-ошибки
4 – ошибка метода Find.
@extended: Содержит код COM-ошибки.

Примечания

Функция воспроизводит поведение Ctrl+F в Excel, дополнительно включая в результат комментарий ячейки.
Excel поддерживает следующие символы подстановки:
? (знак вопроса) – любой одиночный символ
* (звёздочка) – любое количество символов
~ (тильда) перед ?, * или ~ – буквальный знак вопроса, звёздочка или тильда

Если $vRange задан строкой (диапазон A1), поиск выполняется на активном листе книги $oWorkbook.

См. также

_Excel_RangeReplace

Дополнительно

See http://office.microsoft.com/en-us/excel-help/wildcard-characters-HP005203612.aspx.

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <Array.au3>
#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeFind Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeFind Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Находим все вхождения значения "37000" (частичное совпадение).
Local $aResult = _Excel_RangeFind($oWorkbook, "37000")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeFind Example 1", "Error searching the range." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeFind Example 1", "Find all occurrences of value '37000' (partial match)." & @CRLF & "Data successfully searched.")
_ArrayDisplay($aResult, "Excel UDF: _Excel_RangeFind Example 1", "", 0, "|", "Sheet|Name|Cell|Value|Formula|Comment")
