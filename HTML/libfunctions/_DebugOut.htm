Справочник функций

_DebugOut

Выводит данные в сеанс отладки, запущенный функцией _DebugSetup().

#include <Debug.au3>
_DebugOut ( $sOutput )

Параметры

$sOutput Строка (или другое выводимое значение) для вывода в сеанс отладки.

Возвращаемое значение

Успех: Возвращает 1 (см. раздел Примечания).
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $sOutput имеет несовместимый тип
3 – _DebugSetup() не была корректно выполнена.

Примечания

Значения @error и @extended вызывающего кода сохраняются.

Каждый вызов _DebugOut() выводит данные на отдельной строке.

См. также

_DebugSetup

Пример

#cs ----------------------------------------------------------------------------

AutoIt Version: 3.2.8.1
Author: David Nuttall

Script Function:
Базовый скрипт для демонстрации функций отладки.

#ce ----------------------------------------------------------------------------

#include <Debug.au3>

_DebugSetup("Check Excel")
For $i = 1 To 4
    WinActivate("Microsoft Excel")
    ; взаимодействие с Excel
    Send("{Down}")
    _DebugOut("Moved Mouse Down")
Next
