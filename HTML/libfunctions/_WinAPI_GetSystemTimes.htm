Справочник функций

_WinAPI_GetSystemTimes

Получает информацию о синхронизации системы.

#include <WinAPISys.au3>
_WinAPI_GetSystemTimes ( )

Возвращаемое значение

Успех: Возвращает массив, содержащий следующую информацию:
$aArray[0] – время простоя системы.
$aArray[1] – время, которое система потратила на выполнение в режиме ядра, в единицах времени 100 наносекунд.
$aArray[2] – время, которое система потратила на выполнение в пользовательском режиме, в единицах времени 100 наносекунд.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

В многопроцессорной системе функция _WinAPI_GetSystemTimes() возвращает значения, которые являются суммой обозначенных времён
по всем процессорам.
