_MemVirtualFree

Освобождает область страниц в виртуальном адресном пространстве процесса.

#include <Memory.au3>
_MemVirtualFree ( $pAddress, $iSize, $iFreeType )

Параметры

$pAddress Указывает на базовый адрес области страниц, которые должны быть освобождены.
$iSize Задаёт размер области в байтах для освобождения.
$iFreeType Задаёт тип операции освобождения:
$MEM_DECOMMIT – Отменяет выделение указанной области выделённых страниц
$MEM_RELEASE – Освобождает указанную область зарезервированных страниц.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
