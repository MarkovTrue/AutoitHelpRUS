Справочник функций

_IEFormElementOptionSelect

Задаёт значение указанного элемента формы.

#include <IE.au3>
_IEFormElementOptionSelect ( ByRef $oObject, $sString [, $iSelect = 1 [, $sMode = "byValue" [, $iFireEvent = 1]]] )

Параметры

$oObject Объект элемента формы типа "Select Option".
$sString Значение для сравнения с элементом – обработка зависит от $sMode.
$iSelect [необязательный] задаёт, должен ли элемент быть выделен или снят выделение
-1 = Вернуть состояние выделения
0 = Снять выделение с элемента
1 = (по умолчанию) Выделить элемент
$sMode [необязательный] задаёт режим поиска
"byValue" = (по умолчанию) значение опции, которую требуется выбрать
"byText" = текст опции, которую требуется выбрать
"byIndex" = индекс опции (с нуля), которую требуется выбрать
$iFireEvent [необязательный] Запускать ли события OnChange и OnClick после изменения значения:
0 – не запускать события OnChange/OnClick после установки значения
1 – (по умолчанию) запускать события OnChange и OnClick после установки значения

Возвращаемое значение

Успех: текущее состояние выбора, если $iSelect = -1; иначе возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадений не найдено
@extended: Содержит номер недопустимого параметра.

Примечания

Параметр $iFireEvent имеет значение только если с элементом формы связано событие onChange.

См. также

_IEFormElementCheckBoxSelect, _IEFormElementGetValue, _IEFormElementRadioSelect, _IEFormElementSetValue

Пример

Пример 1

; Открыть браузер с примером формы, получить ссылку на форму, получить ссылку
; для выбора элемента, цикл из 10 итераций с выбором опций byValue, byText и byIndex

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oSelect = _IEFormElementGetObjByName($oForm, "selectExample")
_IEAction($oSelect, "focus")
For $i = 1 To 10
    _IEFormElementOptionSelect($oSelect, "Freepage", 1, "byText")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, "midipage.html", 1, "byValue")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, 0, 1, "byIndex")
    Sleep(1000)
Next

_IEQuit($oIE)

Пример 2

; Открыть браузер с примером формы, получить ссылку на форму, получить ссылку
; для выбора нескольких элементов, цикл из 5 итераций с выбором и снятием выделения
; варианты по значению, по тексту и по индексу.

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oSelect = _IEFormElementGetObjByName($oForm, "multipleSelectExample")
_IEAction($oSelect, "focus")
For $i = 1 To 3
    _IEFormElementOptionSelect($oSelect, "Carlos", 1, "byText")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, "Name2", 1, "byValue")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, 5, 1, "byIndex")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, "Carlos", 0, "byText")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, "Name2", 0, "byValue")
    Sleep(1000)
    _IEFormElementOptionSelect($oSelect, 5, 0, "byIndex")
    Sleep(1000)
Next

_IEQuit($oIE)

Пример 3

; Открыть браузер с примером формы, получить ссылку на форму, получить ссылку
; для выбора элемента, проверка, выбрана ли опция "Freepage", и
; вывести результат. Повторить для варианта с индексом 0 и для варианта
; со значением "midipage.html"

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
Local $oSelect = _IEFormElementGetObjByName($oForm, "selectExample")
If _IEFormElementOptionSelect($oSelect, "Freepage", -1, "byText") Then
    MsgBox($MB_SYSTEMMODAL, "Option Selected", "Option Freepage is selected")
Else
    MsgBox($MB_SYSTEMMODAL, "Option Selected", "Option Freepage is Not selected")
EndIf
If _IEFormElementOptionSelect($oSelect, 0, -1, "byIndex") Then
    MsgBox($MB_SYSTEMMODAL, "Option Selected", "The First (index 0) option is selected")
Else
    MsgBox($MB_SYSTEMMODAL, "Option Selected", "The First (index 0) option is Not selected")
EndIf
If _IEFormElementOptionSelect($oSelect, "midipage.html", -1, "byValue") Then
    MsgBox($MB_SYSTEMMODAL, "Option Selected", "The option with value 'midipage.html' is selected")
Else
    MsgBox($MB_SYSTEMMODAL, "Option Selected", "The option with value 'midipage.html' is NOT selected")
EndIf

_IEQuit($oIE)
