Справочник функций

_WinAPI_RemoveFontResourceEx

Удаляет шрифты в указанном файле из таблицы системных шрифтов.

#include <WinAPIGdi.au3>
_WinAPI_RemoveFontResourceEx ( $sFont [, $iFlag = 0 [, $bNotify = False]] )

Параметры

$sFont Строка, которая называет файл ресурса шрифта. Чтобы удалить шрифт, информация о котором получена из нескольких файлов ресурсов,
они должны быть разделены символом "|". Например, abcxxxxx.pfm|abcxxxxx.pfb.
$iFlag [необязательный] Характеристики шрифта, который должен быть удален из системы. Чтобы шрифт был удален, используемые флаги
должны быть теми же, что и при добавлении шрифта с помощью функции _WinAPI_AddFontResourceEx().
$bNotify [необязательный] Указывает, отправляется ли сообщение WM_FONTCHANGE, допустимые значения:
True – отправить сообщение WM_FONTCHANGE всем окнам верхнего уровня после изменения пула ресурсов шрифтов.
False – не отправлять (по умолчанию).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
