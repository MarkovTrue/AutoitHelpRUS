Справочник функций

_WinAPI_EnumWindowStations

Перечисляет все оконные станции текущего сеанса.

#include <WinAPISys.au3>
_WinAPI_EnumWindowStations ( )

Возвращаемое значение

Успех: Возвращает массив имён оконных станций. Нулевой элемент содержит количество оконных станций.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Функция _WinAPI_EnumWindowStations() перечисляет только те оконные станции, к которым вызывающий процесс имеет
право доступа $WINSTA_ENUMERATE.

Дополнительно

Искать EnumWindowStations в библиотеке MSDN

Пример

#include <Array.au3>
#include <WinAPISys.au3>

Local $aData = _WinAPI_EnumWindowStations()

_ArrayDisplay($aData, '_WinAPI_EnumWindowStations')
