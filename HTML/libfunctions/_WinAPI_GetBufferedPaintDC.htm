_WinAPI_GetBufferedPaintDC

Получает контекст устройства отрисовки (DC).

#include <WinAPITheme.au3>
_WinAPI_GetBufferedPaintDC ( $hBP )

Параметры

$hBP Дескриптор контекста буферизованного рисования.

Возвращаемое значение

Успех: Возвращает дескриптор запрошенного DC. Это тот же DC, который возвращает функция _WinAPI_BeginBufferedPaint().
Ошибка: Возвращает 0.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

См. также

_WinAPI_BeginBufferedPaint
