Справочник ключевых слов

$tagNMOBJECTNOTIFY

Содержит сведения для уведомляющих сообщений $TBN_GETOBJECT, $TCN_GETOBJECT, $RBN_GETOBJECT и $PSN_GETOBJECT.

Global Const $tagNMOBJECTNOTIFY = $tagNMHDR & ";int Item;ptr piid;ptr pObject;long Result;dword dwFlags"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Item Идентификатор элемента, специфичный для элемента управления.
piid Указатель на идентификатор интерфейса запрошенного объекта.
pObject Указатель на объект, предоставленный окном, обрабатывающим сообщение уведомления.
Приложение, обрабатывающее сообщение уведомления, устанавливает этот член.
Result Флаги успеха или ошибки COM. Приложение, обрабатывающее сообщение уведомления, устанавливает этот член.
dwFlags флаги, специфичные для элемента управления (подсказки о том, где в iItem произошло попадание)
