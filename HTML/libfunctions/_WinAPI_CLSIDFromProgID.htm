_WinAPI_CLSIDFromProgID

Ищет CLSID в реестре по ProgID.

#include <WinAPICom.au3>
_WinAPI_CLSIDFromProgID ( $sProgID )

Параметры

$sProgID Строка, содержащая ProgID, для которого требуется получить CLSID.

Возвращаемое значение

Успех: Возвращает строковое представление CLSID.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Дополнительно

Искать CLSIDFromProgID в библиотеке MSDN

Пример

#include <WinAPICom.au3>

ConsoleWrite(_WinAPI_CLSIDFromProgID('AutoItX3.Control.1') & @CRLF)
