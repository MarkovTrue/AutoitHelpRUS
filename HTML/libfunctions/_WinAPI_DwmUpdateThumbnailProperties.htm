_WinAPI_DwmUpdateThumbnailProperties

Задаёт свойства миниатюры Диспетчера окон рабочего стола (DWM).

#include <WinAPIGdi.au3>
_WinAPI_DwmUpdateThumbnailProperties ( $hThumbnail [, $bVisible = True [, $bClientAreaOnly = False [, $iOpacity = 255 [, $tRectDest = 0 [, $tRectSrc = 0]]]]] )

Параметры

$hThumbnail Дескриптор миниатюры, у которой нужно получить размер исходного окна.
$bVisible [необязательный] Видима ли миниатюра:
True    – видима (по умолчанию).
False    – скрыта.
$bClientAreaOnly [необязательный] Использовать только клиентскую область окна-источника или всё окно:
True    – только клиентская область источника.
False    – всё окно (по умолчанию).
$iOpacity [необязательный] Непрозрачность отрисовки миниатюры. 0 – полностью прозрачная, 255 (по умолчанию) – полностью непрозрачная.
$tRectDest [необязательный] Структура $tagRECT с прямоугольником в целевом окне, в котором будет отрисована миниатюра.
По умолчанию размер этого прямоугольника равен исходному размеру DWM-миниатюры, который возвращает _WinAPI_DwmQueryThumbnailSourceSize().
$tRectSrc [необязательный] Структура $tagRECT с прямоугольником, задающим область окна-источника, используемую как миниатюра.
По умолчанию миниатюрой служит всё окно.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Связи миниатюр, созданные функцией _WinAPI_DwmRegisterThumbnail(), не будут отрисованы в целевом окне до вызова данной функции.
Последующие вызовы обновляют миниатюру в соответствии с заданными свойствами.

Данная функция требует Windows Vista или новее.

См. также

_WinAPI_DwmQueryThumbnailSourceSize, _WinAPI_DwmRegisterThumbnail

Дополнительно

Искать DwmUpdateThumbnailProperties в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIMisc.au3>

If Not _WinAPI_DwmIsCompositionEnabled() Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later with enabled Aero theme.')
    Exit
EndIf

Run(@SystemDir & '\calc.exe')
Local $hWnd = WinWaitActive("[CLASS:ApplicationFrameWindow]", '', 3)
If Not $hWnd Then
    Exit
EndIf

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400)

GUISetBkColor(0)

; Создать эффект "стеклянного листа" для всего окна. Вызывайте эту функцию при каждом переключении DWM composition.
_WinAPI_DwmExtendFrameIntoClientArea($hForm)

; Создать DWM-связь миниатюр (2:1)
Local $hThumbnail = _WinAPI_DwmRegisterThumbnail($hForm, $hWnd)
Local $tSIZE = _WinAPI_DwmQueryThumbnailSourceSize($hThumbnail)
Local $iWidth = DllStructGetData($tSIZE, 1)
Local $iHeight = DllStructGetData($tSIZE, 2)
Local $tDestRect = _WinAPI_CreateRectEx((400 - $iWidth / 2) / 2, (400 - $iHeight / 2) / 2, $iWidth / 2, $iHeight / 2)
Local $tSrcRect = _WinAPI_CreateRectEx(-20, -20, $iWidth + 40, $iHeight + 40)
_WinAPI_DwmUpdateThumbnailProperties($hThumbnail, 1, 0, 255, $tDestRect, $tSrcRect)

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE

WinClose("[CLASS:ApplicationFrameWindow]", "")
