Справочник функций

_MemGlobalReAlloc

Перевыделяет указанное количество байт из кучи.

#include <Memory.au3>
_MemGlobalReAlloc ( $hMemory, $iBytes [, $iFlags = 0] )

Параметры

$hMemory Дескриптор объекта глобальной памяти для перевыделения. Этот дескриптор возвращается функцией _MemGlobalAlloc или _MemGlobalReAlloc.
$iBytes Новый размер блока памяти в байтах. Если $iFlags задаёт $GMEM_MODIFY, этот параметр игнорируется.
$iFlags [необязательный] Атрибуты выделения памяти:
$GMEM_MOVEABLE – опционально комбинируйте с $GMEM_MODIFY.
Если память является заблокированным блоком памяти $GMEM_MOVEABLE или блоком памяти $GMEM_FIXED и этот флаг не задан, память может перевыделяться только на месте
$GMEM_ZEROINIT – Если этот параметр не задаёт $GMEM_MODIFY, инициализирует дополнительное содержимое памяти нулями, если объект памяти растёт по размеру.

Возвращаемое значение

Успех: Возвращает дескриптор вновь перевыделенного объекта памяти.
Ошибка: Возвращает 0.

Примечания

Если _MemGlobalReAlloc перевыделяет перемещаемый объект, возвращаемое значение представляет собой дескриптор объекта памяти. Для преобразования дескриптора в указатель используйте функцию _MemGlobalLock.

Если _MemGlobalReAlloc перевыделяет неподвижный объект, значение возвращаемого дескриптора является адресом первого байта блока памяти. Для доступа к памяти процесс может просто привести возвращаемое значение к указателю.

Если _MemGlobalReAlloc завершается ошибкой, исходная память не освобождается и исходный дескриптор с указателем остаются действительными.
