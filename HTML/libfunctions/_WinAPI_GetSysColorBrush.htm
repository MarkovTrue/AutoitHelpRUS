_WinAPI_GetSysColorBrush

Получает дескриптор, определяющий логическую кисть, соответствующую указанному индексу цвета.

#include <WinAPIGdi.au3>
_WinAPI_GetSysColorBrush ( $iIndex )

Параметры

$iIndex Элемент отображения, цвет которого должен быть получен.

Возвращаемое значение

Успех: Возвращает логическая кисть, если $iIndex поддерживается текущей платформой.
Ошибка: Возвращает 0.
