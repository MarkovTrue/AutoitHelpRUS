Справочник функций

_WinAPI_SaveFileDlg

Создаёт диалоговое окно, позволяющее пользователю указать диск, каталог и имя файла для сохранения.

#include <WinAPIDlg.au3>
_WinAPI_SaveFileDlg ( [$sTitle = "" [, $sInitDir = "" [, $sFilters = "" [, $iDefaultFilter = 0 [, $sDefaultFilePath = "" [, $sDefaultExt = "" [, $iFlags = 0 [, $iFlagsEx = 0 [, $pOFNProc = 0 [, $pData = 0 [, $hParent = 0]]]]]]]]]]] )

Параметры

$sTitle [необязательный] Строка для размещения в строке заголовка диалогового окна.
Если этот параметр пуст (по умолчанию), система использует стандартный заголовок (то есть "Сохранить как").
$sInitDir [необязательный] Начальный каталог.
$sFilters [необязательный] Пары строк фильтров (например, "Текстовые файлы (*.txt)").
Чтобы указать несколько шаблонов фильтра для одной строки отображения, используйте точку с запятой для их разделения (например, "*.txt;*.doc;*.bak").
Не включайте пробелы в шаблон. Чтобы указать несколько групп фильтров, используйте символ "|" как разделитель (например, "Текстовые файлы (*.txt)|Все файлы (*.**)").
Если этот параметр опущен или содержит пустую строку (по умолчанию), диалоговое окно не отображает никакие фильтры.
$iDefaultFilter [необязательный] Один сном (1-based) индекс текущего выбранного фильтра для инициализации элемента управления раскрывающегося списка.
$sDefaultFilePath [необязательный] Имя файла для инициализации элемента управления редактирования.
$sDefaultExt [необязательный] Расширение по умолчанию, которое добавляется к имени файла, если пользователь не указал расширение.
Эта строка может быть любой длины, но добавляются только первые три символа. Строка не должна содержать точку (.).
Если этот параметр пуст (по умолчанию), расширение не добавляется.
$iFlags [необязательный] Набор флагов, которые можно использовать для инициализации диалогового окна.
Этот параметр может быть 0 или комбинацией следующих значений:
$OFN_ALLOWMULTISELECT
$OFN_CREATEPROMPT
$OFN_DONTADDTORECENT
$OFN_ENABLEHOOK
$OFN_ENABLEINCLUDENOTIFY
$OFN_ENABLESIZING
$OFN_EXPLORER
$OFN_FORCESHOWHIDDEN
$OFN_HIDEREADONLY
$OFN_LONGNAMES
$OFN_NOCHANGEDIR
$OFN_NODEREFERENCELINKS
$OFN_NOLONGNAMES
$OFN_NONETWORKBUTTON
$OFN_NOREADONLYRETURN
$OFN_NOTESTFILECREATE
$OFN_NOVALIDATE
$OFN_OVERWRITEPROMPT
$OFN_PATHMUSTEXIST
$OFN_READONLY
$OFN_SHAREAWARE
$OFN_SHOWHELP
$iFlagsEx [необязательный] Набор флагов для инициализации диалогового окна. Может быть 0 или следующее значение:
$OFN_EX_NOPLACESBAR
$pOFNProc [необязательный] Указатель на процедуру-ловушку. Этот параметр игнорируется, если не установлен флаг $OFN_ENABLEHOOK.
$pData [необязательный] Определённый приложением указатель на данные, которые система передаёт процедуре-ловушке.
$hParent [необязательный] Дескриптор родительского окна для диалогового окна.

Возвращаемое значение

Успех: Если флаг $OFN_ALLOWMULTISELECT не установлен, функция возвращает полный путь к файлу; в противном случае возвращает массив путей.
Нулевой элемент массива содержит количество файлов.
Ошибка: Устанавливает флаг @error в ненулевое значение; флаг @extended может содержать код ошибки диалогового окна ($CDERR_* или $FNERR_*).

Примечания

Константы определены в WinAPIConstants.au3.

Вызовите _WinAPI_GetExtended() для получения внутренней структуры $tagOPENFILENAME, содержащей информацию о выборе пользователя после закрытия диалогового окна.

Если используется ловушка, сохранить внешний вид диалогового окна как в Windows Vista и более поздних версиях невозможно.

Пример

#include <WinAPIConstants.au3>
#include <WinAPIDlg.au3>

Local $sFile = _WinAPI_SaveFileDlg('', @WorkingDir, 'AutoIt v3 Scripts (*.au3)|All Files (*.*)', 1, 'MyScript.au3', '.', BitOR($OFN_PATHMUSTEXIST, $OFN_OVERWRITEPROMPT))

If Not @error Then
    ConsoleWrite($sFile & @CRLF)
EndIf
