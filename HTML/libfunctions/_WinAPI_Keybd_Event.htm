_WinAPI_Keybd_Event

Синтезирует нажатие клавиши.

#include <WinAPISys.au3>
_WinAPI_Keybd_Event ( $vKey, $iFlags [, $iScanCode = 0 [, $iExtraInfo = 0]] )

Параметры

$vKey Код виртуальной клавиши ($VK_*). Должен быть значением в диапазоне от 1 до 254.

0x01 - Левая кнопка мыши
0x02 - Правая кнопка мыши
0x03 - Обработка Ctrl+Break
0x04 - Средняя кнопка мыши (трёхкнопочная мышь)
0x05 - Кнопка мыши X1
0x06 - Кнопка мыши X2

0x08 - Клавиша BACKSPACE
0x09 - Клавиша TAB

0x0C - Клавиша CLEAR
0x0D - Клавиша ENTER

0x10 - Клавиша SHIFT
0x11 - Клавиша CTRL
0x12 - Клавиша ALT
0x13 - Клавиша PAUSE
0x14 - Клавиша CAPS LOCK

0x1B - Клавиша ESC

0x20 - Клавиша SPACEBAR (пробел)
0x21 - Клавиша PAGE UP
0x22 - Клавиша PAGE DOWN
0x23 - Клавиша END
0x24 - Клавиша HOME
0x25 - Клавиша LEFT ARROW (стрелка влево)
0x26 - Клавиша UP ARROW (стрелка вверх)
0x27 - Клавиша RIGHT ARROW (стрелка вправо)
0x28 - Клавиша DOWN ARROW (стрелка вниз)
0x29 - Клавиша SELECT
0x2A - Клавиша PRINT
0x2B - Клавиша EXECUTE
0x2C - Клавиша PRINT SCREEN
0x2D - Клавиша INS
0x2E - Клавиша DEL
0x2F - Клавиша HELP
0x30 - 0x39 - Клавиши (0 - 9)

0x41 - 0x5A - Клавиши (A - Z)
0x5B - Левая клавиша Windows
0x5C - Правая клавиша Windows
0x5D - Клавиша Applications

0x5F - Клавиша Computer Sleep
0x60 - 0x69 - Клавиши цифровой клавиатуры (0 - 9)
0x6A - Клавиша Multiply (умножение)
0x6B - Клавиша Add (сложение)
0x6C - Клавиша Separator (разделитель)
0x6D - Клавиша Subtract (вычитание)
0x6E - Клавиша Decimal (десятичная точка)
0x6F - Клавиша Divide (деление)
0x70 - 0x87 - Клавиши (F1 - F24)

0x90 - Клавиша NUM LOCK
0x91 - Клавиша SCROLL LOCK

0xA0 - Левая клавиша SHIFT
0xA1 - Правая клавиша SHIFT
0xA2 - Левая клавиша CONTROL
0xA3 - Правая клавиша CONTROL
0xA4 - Левая клавиша MENU
0xA5 - Правая клавиша MENU
0xA6 - Browser Back (назад в браузере)
0xA7 - Browser Forward (вперёд в браузере)
0xA8 - Browser Refresh (обновить в браузере)
0xA9 - Browser Stop (остановить в браузере)
0xAA - Browser Search (поиск в браузере)
0xAB - Browser Favorites (избранное в браузере)
0xAC - Browser Start and Home (домашняя страница браузера)
0xAD - Volume Mute (выключить звук)
0xAE - Volume Down (тише)
0xAF - Volume Up (громче)
0xB0 - Next Track (следующая дорожка)
0xB1 - Previous Track (предыдущая дорожка)
0xB2 - Stop Media (остановить воспроизведение)
0xB3 - Play/Pause Media (воспроизведение/пауза)
0xB4 - Start Mail (запустить почту)
0xB5 - Select Media (выбрать медиа)
0xB6 - Start Application 1 (запустить приложение 1)
0xB7 - Start Application 2 (запустить приложение 2)

0xBA - Клавиша ';:'
0xBB - Клавиша '+'
0xBC - Клавиша ','
0xBD - Клавиша '-'
0xBE - Клавиша '.'
0xBF - Клавиша '/?'
0xC0 - Клавиша '`~'

0xDB - Клавиша '[{'
0xDC - Клавиша '\|'
0xDD - Клавиша ']}'
0xDE - Клавиша одинарной/двойной кавычки

0xE2 - Клавиша угловых скобок или обратной косой черты на RT 102-клавишной клавиатуре

0xE7 - Используется для передачи символов Unicode как нажатий клавиш

0xF6 - Клавиша Attn
0xF7 - Клавиша CrSel
0xF8 - Клавиша ExSel
0xF9 - Клавиша Erase EOF
0xFA - Клавиша Play
0xFB - Клавиша Zoom

0xFD - Клавиша PA1
0xFE - Клавиша Clear
$iFlags Этот параметр может быть одним или несколькими из следующих значений.
$KEYEVENTF_EXTENDEDKEY
$KEYEVENTF_KEYUP
$iScanCode [необязательный] Код аппаратного сканирования клавиши. По умолчанию 0.
$iExtraInfo [необязательный] Дополнительное значение, связанное с нажатием клавиши. По умолчанию 0.

Возвращаемое значение

Отсутствует.
