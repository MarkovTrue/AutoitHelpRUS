Справочник функций

_GUIToolTip_AddTool

Регистрирует инструмент (область) в элементе управления ToolTip.

#include <GuiToolTip.au3>
_GUIToolTip_AddTool ( $hTool, $hWnd, $sText [, $iID = 0 [, $iLeft = 0 [, $iTop = 0 [, $iRight = 0 [, $iBottom = 0 [, $iFlags = Default [, $iParam = 0 [, $hInst = 0]]]]]]]] )

Параметры

$hTool Дескриптор элемента управления ToolTip (возвращается функцией _GUIToolTip_Create).
$hWnd Дескриптор окна.
$sText Текст, отображаемый во всплывающей подсказке. См. раздел Примечания.
$iID [необязательный] Идентификатор инструмента или дескриптор окна элемента управления, для которого создаётся инструмент.
$iLeft [необязательный] Координата X левого верхнего угла прямоугольника
$iTop [необязательный] Координата Y левого верхнего угла прямоугольника
$iRight [необязательный] Координата X нижнего правого угла прямоугольника.
$iBottom [необязательный] Координата Y правого нижнего угла прямоугольника
$iFlags [необязательный] Флаги, контролирующие отображение подсказки:
$TTF_IDISHWND – параметр $iID содержит дескриптор окна или элемента управления вместо идентификатора инструмента.
$TTF_CENTERTIP – подсказка центрируется под элементом управления, указанным в $iID.
$TTF_RTLREADING – текст отображается в противоположном направлении относительно окна-родителя (см. раздел Примечания).
$TTF_SUBCLASS – элемент управления выполняет подклассирование окна инструмента.
$TTF_TRACK – подсказка позиционируется рядом с инструментом, к которому она относится.
$TTF_ABSOLUTE – подсказка позиционируется с координатами, переданными в TTM_TRACKPOSITION (см. раздел Примечания).
$TTF_TRANSPARENT – элемент управления передаёт сообщения мыши окну-родителю.
$TTF_PARSELINKS – ссылки в тексте подсказки отображаются как ссылки.
По умолчанию: BitOr($TTF_SUBCLASS, $TTF_IDISHWND).
$iParam [необязательный] Определяемое приложением значение
$hInst [необязательный] Дескриптор экземпляра приложения, связанный с инструментом.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; @error может быть установлен (см. раздел Примечания).

Примечания

Если элемент управления, на который ссылается $hWnd, находится в другом процессе и оба процесса работают в разных режимах AutoIt (@AutoItVersion), переменная @error будет установлена на 6.

Если требуется обратный вызов уведомления, установите $sText = -1 (LPSTR_TEXTCALLBACK).

Если используется флаг $TTF_IDISHWND, координаты в $iLeft, $iTop, $iRight и $iBottom игнорируются.

Если используется флаг $TTF_ABSOLUTE, он должен использоваться вместе с флагом $TTF_TRACK.
Обычно окна отображают текст слева направо (LTR). Окна можно зеркалировать для отображения текстов языков, которые читаются справа налево (RTL). Обычно текст подсказки отображается в том же направлении, что и текст в окне-родителе. Если установлен флаг $TTF_RTLREADING, текст подсказки будет читаться в противоположном направлении от текста окна-родителя.

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GUIToolTip.au3>

Example()

Func Example()
    Local $hGUI = GUICreate("ToolTip Add Tool v(" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $iGUI = 1

    ; создать элемент tooltip с настройками по умолчанию
    Local $hToolTip = _GUIToolTip_Create(0)
    ConsoleWrite('@@ Debug(' & @ScriptLineNumber & ') : $hToolTip = ' & $hToolTip & @CRLF & '>Error code: ' & @error & ' Extended code: ' & @extended & ' (0x' & Hex(@extended) & ')' & @CRLF) ;### Отладочная консоль
    _MemoSetHandleInProcess($hToolTip)

    Local $idButton = GUICtrlCreateButton("Button", 30, 32, 130, 28)
    Local $hButton = GUICtrlGetHandle($idButton)

;~ $iGUI = 0 ; is OK
    ; добавить tool в элемент tooltip
    _GUIToolTip_AddTool($hToolTip, $iGUI * $hGUI, "This is a ToolTip", $hButton)
    _GUIToolTip_AddTool($hToolTip, $iGUI * $hGUI, "This is a ToolTip for the non another ctrl's", $hGUI)

    GUISetState(@SW_SHOW)

    ; Показать tooltip, связанный с кнопкой
    Opt("MouseCoordMode", 2)
    MouseMove(50, 42, 0)
    Sleep(250)

    While 1
        If GUIGetMsg() = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd

    ; Уничтожить элемент tooltip
    _MemoResetHandleInProcess($hToolTip)
    _GUIToolTip_Destroy($hToolTip)
    GUIDelete($hGUI)
EndFunc   ;==>Example

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GUIToolTip.au3>
#include <StaticConstants.au3>

; Этот пример демонстрирует как добавить инструмент; инструмент не назначается какому-либо элементу управления, а просто области GUI
Example()

Func Example()
    Local $hGUI = GUICreate("ToolTip AddTool - v(" & @AutoItVersion & ")", 350, 200, 100, 100)
    ; создать рамки, указывающие где будет отображаться подсказка при движении мыши
    ; в нужное место GUI; эти рамки не имеют назначенной подсказки
    ; эти рамки только для визуального обозначения того, где будет отображаться подсказка
    ; они не требуются для отображения подсказки.
    GUICtrlCreateLabel("", 10, 10, 160, 75, $SS_ETCHEDFRAME)
    ; этот label должен быть отключён, иначе tooltip не отобразится при наведении
    ; мыши на него.
    GUICtrlSetState(-1, $GUI_DISABLE)
    GUICtrlCreateLabel("", 10, 84, 160, 75, $SS_ETCHEDFRAME)
    ; этот label должен быть отключён, иначе tooltip не отобразится при наведении
    ; мыши на него.
    GUICtrlSetState(-1, $GUI_DISABLE)
    GUICtrlCreateLabel("", 169, 10, 160, 75, $SS_ETCHEDFRAME)
    ; этот label должен быть отключён, иначе tooltip не отобразится при наведении
    ; мыши на него.
    GUICtrlSetState(-1, $GUI_DISABLE)
    GUICtrlCreateLabel("", 169, 84, 160, 75, $SS_ETCHEDFRAME)
    ; этот label должен быть отключён, иначе tooltip не отобразится при наведении
    ; мыши на него.
    GUICtrlSetState(-1, $GUI_DISABLE)
    Local $idButton = GUICtrlCreateButton("This is a button", 30, 32, 130, 28)
    Local $hButton = GUICtrlGetHandle($idButton)
    ; создать элемент tooltip с настройками по умолчанию
    Local $hToolTip = _GUIToolTip_Create(0)

    ; добавить 4 инструмента к элементу управления ToolTip; инструменты создаются с использованием расположения на GUI
    ; а не назначением их конкретному элементу управления.
    _GUIToolTip_AddTool($hToolTip, $hGUI, "Upper Left corner", 0, 10, 10, 168, 85, $TTF_SUBCLASS)
    _GUIToolTip_AddTool($hToolTip, $hGUI, "Upper Right corner", 0, 168, 10, 328, 85, $TTF_SUBCLASS)
    _GUIToolTip_AddTool($hToolTip, $hGUI, "Lower Left corner", 0, 10, 85, 168, 160, $TTF_SUBCLASS)
    _GUIToolTip_AddTool($hToolTip, $hGUI, "Lower Right corner", 0, 168, 85, 328, 160, $TTF_SUBCLASS)
    ; добавить инструмент к элементу управления ToolTip, назначенный кнопке
    _GUIToolTip_AddTool($hToolTip, $hGUI, "This tooltip belongs to the button", $hButton)
    GUISetState(@SW_SHOW)

    While 1
        If GUIGetMsg() = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd
    ; Уничтожить элемент tooltip
    _GUIToolTip_Destroy($hToolTip)
    GUIDelete($hGUI)
EndFunc   ;==>Example

Example (OutProcess) : ToolTip Add Tool to an External process

; == Example (OutProcess) : ToolTip Add Tool to an External process

#include "Extras\HelpFileInternals.au3"

#include <GUIToolTip.au3>

Example()

Func Example()
    Local $sFromTo
    Local $hWin = _MemoRunAU3OutProcess($sFromTo, True) ; OK если одинаковый режим, KO если разный

    Local $hToolTip = _MemoGetHandleInProcess()
    Local $hButton = _MemoCreateOutProcess($hWin, "Button", 0, $sFromTo)

    ; добавить tool в элемент tooltip
    Local $iRet = _GUIToolTip_AddTool($hToolTip, $hWin, "<<<This is a ToolTip", $hButton)
    If @error Then
        _MemoMsgBox($MB_ICONERROR, "Info" & $sFromTo, "_GUIToolTip_AddTool()" & " @error = " & @error & @CRLF & _
                @TAB & "cannot be added to an external process" & @CRLF & _
                @TAB & "running in different mode")
    Else
        ; работает только если оба процесса запущены в одном режиме
        _MemoWrite("Info" & $sFromTo & " _GUIToolTip_AddTool() $iRet= " & $iRet)
        _Memowrite(_GUIToolTip_GetText($hWin, $hToolTip, $hButton))
    EndIf

    _MemoMsgBoxStatus("", Default, $hWin) ; больше действий нет, ждём закрытия GUI, закрываем и внешний GUI

EndFunc   ;==>Example
