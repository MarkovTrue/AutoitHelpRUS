Справочник функций

_WinAPI_SetSearchPathMode

Задаёт режим для каждого процесса, который функция _WinAPI_SearchPath() использует при поиске файлов.

#include <WinAPIFiles.au3>
_WinAPI_SetSearchPathMode ( $iFlags )

Параметры

$iFlags Режим поиска для использования. Этот параметр может быть одним из следующих значений.
$BASE_SEARCH_PATH_ENABLE_SAFE_SEARCHMODE
$BASE_SEARCH_PATH_DISABLE_SAFE_SEARCHMODE

По желанию, $BASE_SEARCH_PATH_ENABLE_SAFE_SEARCHMODE можно объединить со следующим флагом, чтобы сделать этот
режим постоянным для текущего процесса.
$BASE_SEARCH_PATH_DISABLE_SAFE_SEARCHMODE не может быть объединён с этим флагом.
$BASE_SEARCH_PATH_PERMANENT

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

После успешного вызова функции _WinAPI_SetSearchPathMode() для текущего процесса параметр
в системном реестре игнорируется в пользу режима, наиболее успешно установленного совсем недавно.

Если функция была успешно вызвана для текущего процесса с установленным флагом $BASE_SEARCH_PATH_PERMANENT,
безопасный режим устанавливается постоянно для вызывающего процесса. Любые последующие вызовы _WinAPI_SetSearchPathMode()
из этого процесса, которые пытаются изменить режим поиска, завершатся с ошибкой ERROR_ACCESS_DENIED (5).

Функция _WinAPI_SetSearchPathMode() не изменяет системный реестр.

Эта функция требует Windows 7 или позже.
