Справочник функций

_ArrayToString

Объединяет элементы одномерного или двумерного массива в одну строку, разделяя их заданными разделителями.

#include <Array.au3>
_ArrayToString ( Const ByRef $aArray [, $sDelim_Col = "|" [, $iStart_Row = Default [, $iEnd_Row = Default [, $sDelim_Row = @CRLF [, $iStart_Col = Default [, $iEnd_Col = Default]]]]]] )

Параметры

$aArray Массив для преобразования в строку.
$sDelim_Col [необязательный] Разделитель элементов 1D-массива или столбцов 2D-массива.
$iStart_Row [необязательный] Индекс строки массива, с которой начинается копирование (по умолчанию – 0).
$iEnd_Row [необязательный] Индекс строки массива, на которой заканчивается копирование (по умолчанию – количество строк).
$sDelim_Row [необязательный] Разделитель строк 2D-массива (только для 2D).
$iStart_Col [необязательный] Индекс столбца массива, с которого начинается копирование (только для 2D; по умолчанию – 0).
$iEnd_Col [необязательный] Индекс столбца массива, на котором заканчивается копирование (только для 2D; по умолчанию – количество столбцов).

Возвращаемое значение

Успех: Возвращает строку из выбранных элементов, разделённых заданными разделителями.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным или двумерным массивом
3 – $iStart_Row или $iEnd_Row выходят за границы массива
4 – $iStart_Row больше $iEnd_Row
5 – $iStart_Col или $iEnd_Col выходят за границы массива
6 – $iStart_Col больше $iEnd_Col

Примечания

Если переданный массив имеет один из следующих форматов и не содержит данных – [0], [1], [0][0], [1][0], [0][1], [1][1] – функция возвращает пустую строку. При последующей передаче этой строки в _ArrayFromString() та не сможет различить эти случаи и вернёт по умолчанию одноэлементный пустой массив: [1] или [1][1] в зависимости от флага $bForce2D.

См. также

_ArrayFromString, _ArrayToClip

Пример

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray[20]
For $i = 0 To 19
    $aArray[$i] = $i
Next
_ArrayDisplay($aArray, "1D Array")
MsgBox($MB_SYSTEMMODAL, "Items 1-7", _ArrayToString($aArray, @TAB, 1, 7))

Local $aArray[10][10]
For $i = 0 To 9
    For $j = 0 To 9
        $aArray[$i][$j] = $i & "-" & $j
    Next
Next
_ArrayDisplay($aArray, "2D Array")
MsgBox($MB_SYSTEMMODAL, "Rows 4-7, cols 2-5", _ArrayToString($aArray, " :: ", 4, 7, @CRLF, 2, 5))
