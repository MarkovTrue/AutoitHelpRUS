_WinAPI_GetThemeSysFont

Получает информацию о шрифте системного шрифта.

#include <WinAPITheme.au3>
_WinAPI_GetThemeSysFont ( $hTheme, $iFontID )

Параметры

$hTheme Дескриптор данных темы.
$iFontID Значение, которое задаёт системный шрифт. Может быть одним из следующих значений.
$TMT_CAPTIONFONT
$TMT_SMALLCAPTIONFONT
$TMT_MENUFONT
$TMT_STATUSFONT
$TMT_MSGBOXFONT
$TMT_ICONTITLEFONT

Возвращаемое значение

Успех: Возвращает структура $tagLOGFONT, которая содержит информацию о шрифте.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Если дескриптор данных темы не равен 0, функция возвращает информацию о шрифте из раздела SysMetrics визуального стиля,
в противном случае функция возвращает значение глобальной системной метрики.
