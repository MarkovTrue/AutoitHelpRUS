Справочник функций

_WinAPI_LoadLibrary

Загружает указанный исполняемый модуль в адресное пространство вызывающего процесса.

#include <WinAPIRes.au3>
_WinAPI_LoadLibrary ( $sFileName )

Параметры

$sFileName Указывает исполняемый модуль Win32 (либо файл .dll, либо файл .exe). Указанное имя это
имя файла исполняемого модуля.

Возвращаемое значение

Успех: Возвращает дескриптор исполняемого модуля.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

См. также

_WinAPI_FreeLibrary, _WinAPI_LoadLibraryEx
