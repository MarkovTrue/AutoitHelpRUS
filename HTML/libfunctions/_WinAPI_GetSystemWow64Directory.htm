Справочник функций

_WinAPI_GetSystemWow64Directory

Получает путь системного каталога, используемого WOW64.

#include <WinAPISys.au3>
_WinAPI_GetSystemWow64Directory ( )

Возвращаемое значение

Успех: Возвращает путь системного каталога. Этот путь не заканчивается обратной косой чертой.
Ошибка: Устанавливает @error ≠ 0, флаг @extended может содержать код последней ошибки.

Примечания

функция всегда не работает на 32-разрядной Windows, @extended устанавливается в ERROR_CALL_NOT_IMPLEMENTED.
