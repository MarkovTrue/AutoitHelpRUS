Справочник ключевых слов

$tagBITMAPINFOHEADER

Определяет размеры аппаратно-независимого точечного рисунка (DIB) на основе Windows.

Global Const $tagBITMAPINFOHEADER = "struct; dword biSize;long biWidth;long biHeight;word biPlanes;word biBitCount;" & _
"dword biCompression;dword biSizeImage;long biXPelsPerMeter;long biYPelsPerMeter;" & _
"dword biClrUsed;dword biClrImportant; endstruct"

Параметры

biSize Количество байтов, необходимых для структуры, минус размер данных RGBQuad.
biWidth Ширина растрового изображения в пикселах.
biHeight Высота растрового изображения в пикселах.
biPlanes Количество плоскостей для целевого устройства. Должно быть установлено в 1.
biBitCount Количество бит на пиксел.
biCompression Тип сжатия для сжатого растрового изображения, ориентированного вверх.
biSizeImage Размер изображения в байтах.
biXPelsPerMeter Горизонтальное разрешение целевого устройства для растрового изображения в пикселах на метр.
biYPelsPerMeter Вертикальное разрешение целевого устройства для растрового изображения в пикселах на метр.
biClrUsed Количество индексов цвета в таблице цветов, которые фактически используются растровым изображением.
biClrImportant Количество индексов цвета, необходимых для отображения растрового изображения.
