Справочник функций

_MouseTrap

Ограничивает движение мыши указанной областью.

#include <Misc.au3>
_MouseTrap ( [$iLeft = 0 [, $iTop = 0 [, $iRight = 0 [, $iBottom = 0]]]] )

Параметры

$iLeft [необязательный] Координата слева
$iTop [необязательный] Координата сверху
$iRight [необязательный] Координата справа
$iBottom [необязательный] Координата снизу

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Используйте _MouseTrap() без параметров для освобождения мыши.

Пример

#include <GUIConstantsEx.au3>
#include <Misc.au3>

Example()

Func Example()
    ; Создать GUI с различными элементами управления.
    Local $hGUI = GUICreate("Mouse Trap Example")

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Удержать положение окна.
    Local $aCoords = 0

    ; Цикл до выхода пользователя.
    While 1
        $aCoords = WinGetPos($hGUI)
        If Not @error Then
            ; Если ошибок не возникает, ограничить движение мыши клиентской областью окна.
            _MouseTrap($aCoords[0], $aCoords[1], $aCoords[0] + $aCoords[2], $aCoords[1] + $aCoords[3])
        EndIf
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    ; Освободить мышь
    _MouseTrap()
EndFunc   ;==>Example
