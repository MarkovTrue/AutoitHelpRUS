Справочник функций

_WinAPI_CreateProcessWithToken

создаёт новый процесс и его основной поток в контексте безопасности указанного токена.

#include <WinAPIProc.au3>
_WinAPI_CreateProcessWithToken ( $sApp, $sCmd, $iFlags, $tStartupInfo, $tProcessInfo, $hToken [, $iLogon = 0 [, $pEnvironment = 0 [, $sDir = '']]] )

Параметры

$sApp Координаты.
$sCmd Координаты.
$iFlags Флаги, управляющие созданием процесса.
Флаги $CREATE_DEFAULT_ERROR_MODE, $CREATE_NEW_CONSOLE и $CREATE_NEW_PROCESS_GROUP включены по умолчанию.
Можно указать дополнительные флаги:
$CREATE_DEFAULT_ERROR_MODE
$CREATE_NEW_CONSOLE
$CREATE_NEW_PROCESS_GROUP
$CREATE_SEPARATE_WOW_VDM
$CREATE_SUSPENDED
$CREATE_UNICODE_ENVIRONMENT
$tStartupInfo Указатель на структуру.
$tProcessInfo Дескриптор.
$hToken Дескриптор.
$iLogon Значение.
$pEnvironment Указатель на структуру.
$sDir Строка.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Чтобы получить токен безопасности, передайте дескриптор процесса из структуры $tagPROCESS_INFORMATION в функцию _WinAPI_OpenProcessToken().

Функция требует Windows Vista или новее.

См. также

_WinAPI_OpenProcessToken
