_NamedPipes_PeekNamedPipe

Копирует данные из канала в буфер без удаления их из канала.

#include <NamedPipes.au3>
_NamedPipes_PeekNamedPipe ( $hNamedPipe )

Параметры

$hNamedPipe Дескриптор канала.

Возвращаемое значение

Успех: Возвращает массив в следующем формате:
$aInfo[0] – данные, прочитанные из канала
$aInfo[1] – количество байтов, прочитанных из канала
$aInfo[2] – всего байтов доступно для чтения
$aInfo[3] – количество байтов, оставшихся для чтения в этом сообщении
Ошибка: Устанавливает @error ≠ 0.
