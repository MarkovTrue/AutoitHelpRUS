Справочник функций

_TCPIpToName

Преобразует IP-адрес в имя хоста.

#include <Inet.au3>
_TCPIpToName ( $sIp [, $iOption = 0 [, $hDll = "Ws2_32.dll"]] )

Параметры

$sIp IP-адрес в формате с точками (v4).
$iOption [необязательный] По умолчанию = 0
0 = Возвращает строку с именем хоста
1 = Возвращает массив (см. раздел Примечания).
$hDll [необязательный] Дескриптор Ws2_32.dll

Возвращаемое значение

Успех: Возвращает имя хоста или массив (см. раздел Примечания).
Ошибка: ""
@error: 1 - Ошибка вызова DllCall inet_addr
2 - Ошибка inet_addr
3 - Ошибка вызова DllCall gethostbyaddr
4 - Ошибка gethostbyaddr, WSAGetLastError = @extended
5 - Ошибка gethostbyaddr, ошибка WSAGetLastError
6 - Ошибка strlen/sZStringRead
7 - Ошибка чтения массива Aliases

Примечания

Требуется успешный вызов WSAStartup (выполняется TCPStartup()).
Если $iOption = 1, возвращённый массив имеет такой вид:
    $aResult[0] = Количество элементов
    $aResult[1] = "Имя хоста"
    $aResult[2] = "Алиас 1"
    $aResult[3] = "Алиас 2"
    ...

Пример

Пример 1

#include <Inet.au3>
#include <MsgBoxConstants.au3>

Local $aResult, $sResult, $sIp

TCPStartup()
$sIp = TCPNameToIP("autoitscript.com")
$sResult = _TCPIpToName($sIp)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "_TCPIpToName()", "@error = " & @error & @CRLF & "@extended = " & @extended)
Else
    MsgBox($MB_SYSTEMMODAL, "autoitscript.com really is: ", $sResult)
EndIf

Пример 2

#include <Array.au3>
#include <Inet.au3>
#include <MsgBoxConstants.au3>

Local $aResult, $sResult, $sIp

TCPStartup()
$sIp = _GetIP()
$aResult = _TCPIpToName($sIp, 1)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "_TCPIpToName()", "@error = " & @error & @CRLF & "@extended = " & @extended)
Else
    _ArrayDisplay($aResult, "Local Hostname(s)")
EndIf
