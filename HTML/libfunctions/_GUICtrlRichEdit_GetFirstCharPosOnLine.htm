_GUICtrlRichEdit_GetFirstCharPosOnLine

Получает позицию между символами перед первым символом строки.

#include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetFirstCharPosOnLine ( $hWnd [, $iLine = -1] )

Параметры

$hWnd Дескриптор элемента управления.
$iLine [необязательный] Номер строки.
По умолчанию – текущая строка.

Возвращаемое значение

Успех: Возвращает позиция символа первого символа строки.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 101 – $hWnd не является дескриптором
1021 – $iLine не положителен и не равен -1
1022 – $iLine превышает количество строк текста

Примечания

Элемент управления без текста содержит одну строку.

Первая строка нумеруется 1. Первая позиция символа в клиентской области – 0.

Дополнительно

Искать EM_GETLINEINDEX в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <GuiRichEdit.au3>
#include <WindowsStylesConstants.au3>

Global $g_idLbl_Msg
Example()

Func Example()
    Local $hGui, $hRichEdit, $iLastLineNumber, $iCharPosition, $iMsg
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName, StringLen(".exe")) & ")", 520, 350, -1, -1)
    $hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 500, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    $g_idLbl_Msg = GUICtrlCreateLabel("", 10, 235, 300, 60)
    GUISetState(@SW_SHOW)

    _GUICtrlRichEdit_AutoDetectURL($hRichEdit, True)
    _GUICtrlRichEdit_AppendText($hRichEdit, @CRLF & "http://www.autoitscript.com")

    $iLastLineNumber = _GUICtrlRichEdit_GetLineCount($hRichEdit)

    ; получить позицию первого символа последней строки
    $iCharPosition = _GUICtrlRichEdit_GetFirstCharPosOnLine($hRichEdit, $iLastLineNumber)

    ; выделить 4 символа – должно получиться слово "http"
    _GUICtrlRichEdit_SetSel($hRichEdit, $iCharPosition, $iCharPosition + 4)
    Report("Character attributes at start of line " & $iLastLineNumber & " are " & _
            _GUICtrlRichEdit_GetCharAttributes($hRichEdit))

    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                _GUICtrlRichEdit_Destroy($hRichEdit) ; нужно, если только скрипт не упадёт
                ; GUIDelete() ; так тоже работает
                Exit
        EndSelect
    WEnd
EndFunc   ;==>Example

Func Report($sMsg)
    GUICtrlSetData($g_idLbl_Msg, $sMsg)
EndFunc   ;==>Report
