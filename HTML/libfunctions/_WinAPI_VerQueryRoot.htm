Справочник функций

_WinAPI_VerQueryRoot

Получает фиксированную информацию о версии из указанного ресурса информации о версии.

#include <WinAPIRes.au3>
_WinAPI_VerQueryRoot ( $pData )

Параметры

$pData Указатель на буфер, который содержит ресурс информации о версии, возвращённый функцией
_WinAPI_GetFileVersionInfo().

Возвращаемое значение

Успех: Возвращает структура $tagVS_FIXEDFILEINFO, которая содержит информацию о версии.
Ошибка: Устанавливает @error ≠ 0.
