Справочник функций

_FileWriteToLine

Записывает текст в указанную строку файла.

#include <File.au3>
_FileWriteToLine ( $sFilePath, $iLine, $sText [, $bOverWrite = False [, $bFill = False]] )

Параметры

$sFilePath Файл, в строку которого выполняется запись.
$iLine Номер строки, в которую записывается текст.
$sText Записываемый текст.
$bOverWrite [необязательный]
True – текст заменяет указанную строку
False – (по умолчанию) текст вставляется перед указанной строкой; все следующие строки сдвигаются вниз.
$bFill [необязательный]
True – при необходимости добавляет пустые строки, чтобы нужный номер строки существовал
False – (по умолчанию) пустые строки не добавляются; см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – файл содержит меньше строк, чем $iLine
2 – файл не существует
3 – ошибка при открытии файла
4 – недопустимый параметр $iLine
5 – недопустимый параметр $bOverWrite
6 – недопустимый параметр $sText
7 – недопустимый параметр $bFill

Примечания

Файл может не открыться из-за прав доступа или атрибутов.

Если _FileWriteToLine() вызывается с $bOverWrite = 1 и $sText = "", содержимое строки удаляется и она становится пустой, но сама строка в файле остаётся.

По умолчанию функция возвращает ошибку, если строка с указанным номером не существует. Установка параметра $bFill в True принудит функцию добавить пустые строки до нужного номера.

Пример

#include <File.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Подготовить данные для записи в файл.
    Local $sData = "Line 1: This is an example of using _FileWriteToLine()" & @CRLF & _
            "Line 2: This is an example of using _FileWriteToLine()" & @CRLF & _
            "Line 3: This is an example of using _FileWriteToLine()" & @CRLF & _
            "Line 4: This is an example of using _FileWriteToLine()" & @CRLF & _
            "Line 5: This is an example of using _FileWriteToLine()" & @CRLF

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, $sData) Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Записать в строку 3 с перезаписью.
    _FileWriteToLine($sFilePath, 3, "Line 3: THIS HAS BEEN REPLACED", True)

    ; Прочитать содержимое файла по пути.
    Local $sFileRead = FileRead($sFilePath)

    ; Показать содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & $sFileRead)

    ; Записать в строку 3 без перезаписи (вставка).
    _FileWriteToLine($sFilePath, 3, "Line 3: THIS HAS BEEN INSERTED", False)

    ; Прочитать содержимое файла по пути.
    $sFileRead = FileRead($sFilePath)

    ; Показать содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & $sFileRead)

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
