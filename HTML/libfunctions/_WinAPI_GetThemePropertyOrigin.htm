_WinAPI_GetThemePropertyOrigin

Получает расположение определения свойства темы для свойства.

#include <WinAPITheme.au3>
_WinAPI_GetThemePropertyOrigin ( $hTheme, $iPartID, $iStateID, $iPropID )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит тему.
$iStateID Состояние части.
$iPropID Получаемое свойство ($TMT_*).

Возвращаемое значение

Успех: Возвращает значение, которое указывает, где было найдено свойство. Может быть одним из следующих значений.
0 – свойство найдено в разделе состояния.
1 – свойство найдено в разделе части.
2 – свойство найдено в разделе класса.
3 – свойство найдено в списке глобальных переменных.
4 – свойство не найдено.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
