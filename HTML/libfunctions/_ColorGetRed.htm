Справочник функций

_ColorGetRed

Возвращает красную составляющую заданного цвета.

#include <Color.au3>
_ColorGetRed ( $iColor )

Параметры

$iColor Цвет RGB для работы (0x00RRGGBB).

Возвращаемое значение

Возвращает компонент цвета в диапазоне 0-255.

См. также

_ColorGetBlue, _ColorGetGreen, _ColorGetRGB

Пример

#include <Color.au3>
#include <MsgBoxConstants.au3>

Local $iRedColor = _ColorGetRed(0x8080ff)
MsgBox($MB_SYSTEMMODAL, "AutoIt", $iRedColor)
