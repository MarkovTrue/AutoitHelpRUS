Справочник ключевых слов

$tagGDIP_EFFECTPARAMS_ColorLUT

Структура параметров эффекта цветовой таблицы поиска (LUT).

Global Const $tagGDIP_EFFECTPARAMS_ColorLUT = "byte LutB[256]; byte LutG[256]; byte LutR[256]; byte LutA[256]"

Параметры

LutB Массив из 256 элементов для синего канала.
LutG Массив из 256 элементов для зелёного канала.
LutR Массив из 256 элементов для красного канала.
LutA Массив из 256 элементов для альфа-канала.

Примечания

Таблицы преобразования для каждого цветового канала.
