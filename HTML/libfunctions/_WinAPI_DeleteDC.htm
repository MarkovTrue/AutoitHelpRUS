Справочник функций

_WinAPI_DeleteDC

Удаляет указанный контекст устройства.

#include <WinAPIGdiDC.au3>
_WinAPI_DeleteDC ( $hDC )

Параметры

$hDC Идентифицирует контекст устройства.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Приложение не должно удалять DC, дескриптор которого получен вызовом _WinAPI_GetDC().
Вместо этого освобождать такой DC нужно вызовом _WinAPI_ReleaseDC().

См. также

_WinAPI_CreateCompatibleDC, _WinAPI_GetDC, _WinAPI_ReleaseDC
