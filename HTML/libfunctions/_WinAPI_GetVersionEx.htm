Справочник функций

_WinAPI_GetVersionEx

Получает сведения о текущей операционной системе.

#include <WinAPISys.au3>
_WinAPI_GetVersionEx ( )

Возвращаемое значение

Успех: Возвращает структура $tagOSVERSIONINFOEX, содержащая сведения о текущей операционной системе.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

При использовании функции _WinAPI_GetVersionEx() для определения того, работает ли приложение на конкретной
версии операционной системы, проверьте версии, которые больше или равны нужным
номерам версий. Это гарантирует, что тест пройдёт успешно для более новых версий операционной системы.
