Справочник ключевых слов

$tagGDIPSTARTUPINPUT

Входные параметры инициализации GDI+.

Global Const $tagGDIPSTARTUPINPUT = "int Version;ptr Callback;int NoThread;int NoCodecs"

Параметры

Version Указывает версию Microsoft Windows GDI+.
Callback Указатель на функцию обратного вызова, которую GDI+ может вызывать в отладочных сборках для утверждений и предупреждений.
NoThread Логическое значение, определяющее, нужно ли подавлять фоновый поток GDI+.
Если установить этот член в True, GdiplusStartup возвращает указатель на функцию перехвата и указатель на функцию отмены перехвата.
Вы должны вызвать эти функции соответствующим образом, чтобы заменить фоновый поток.
Если вы не хотите отвечать за вызов функций перехвата и отмены перехвата, установите этот член в False.
NoCodecs Логическое значение, определяющее, нужно ли GDI+ подавлять внешние кодеки изображений.
VERSION GDI+ версия 1.0 не поддерживает внешние кодеки изображений, поэтому этот параметр игнорируется.
