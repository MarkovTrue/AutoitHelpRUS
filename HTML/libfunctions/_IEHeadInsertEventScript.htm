Справочник функций

_IEHeadInsertEventScript

Вставляет JavaScript в Head документа.

#include <IE.au3>
_IEHeadInsertEventScript ( ByRef $oObject, $sHTMLFor, $sEvent, $sScript )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sHTMLFor HTML-элемент для мониторинга событий (например, "document", "window" или ID элемента).
$sEvent Отслеживаемое событие (например, "onclick" или "oncontextmenu").
$sScript JavaScript-код, который будет выполнен.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) – неверный тип данных
@extended: Содержит номер недопустимого параметра.

Примечания

С помощью ObjEvent() AutoIt может получать уведомления о событиях через COM, но обрабатывает их асинхронно (а не синхронно, как они обрабатываются в контексте браузера).
Данная функция позволяет внедрить код, выполняющийся внутри контекста браузера.

Элементы без присвоенного ID всё равно можно использовать, получив их свойство "uniqueID" через _IEPropertyGet().

См. также

_IEDocInsertHTML, _IEDocInsertText, _IEPropertyGet

Пример

Пример 1

; Открыть в браузере простую страницу примера, вставить
; обработчик в head документа, который создаёт
; JavaScript-уведомление при щелчке по документу

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEHeadInsertEventScript($oIE, "document", "onclick", "alert('Someone clicked the document!');")

Пример 2

; Открыть в браузере простую страницу примера, вставить
; обработчик в head документа, который создаёт
; JavaScript-предупреждение, когда кто-то пытается вызвать правый клик на
; документ, после чего скрипт-обработчик возвращает "false" для предотвращения
; контекстное меню правого клика не появлялось

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEHeadInsertEventScript($oIE, "document", "oncontextmenu", "alert('No Context Menu');return false")

Пример 3

; Открыть в браузере простую страницу примера, вставить
; скрипт-обработчик в head документа, создающий
; JavaScript-предупреждение перед уходом со
; страницы с возможностью отменить переход.

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEHeadInsertEventScript($oIE, "window", "onbeforeunload", _
        "alert('Example warning follows...');return 'Pending changes may be lost';")
_IENavigate($oIE, "www.autoitscript.com")

Пример 4

; Открыть в браузере простую страницу примера, вставить
; обработчик в head документа, который предотвращает
; выделение текста в документе

#include <IE.au3>

Local $oIE = _IE_Example()
_IEHeadInsertEventScript($oIE, "document", "ondrag", "return false;")
_IEHeadInsertEventScript($oIE, "document", "onselectstart", "return false;")

Пример 5

; Открытие браузера с домашней страницей AutoIt, вставка
; обработчик в head документа, который предотвращает
; переход при любом клике по ссылке и записывать URL
; нажатой ссылки в консоль

#include <IE.au3>

Local $oIE = _IECreate("http://www.autoitscript.com")

Local $oLinks = _IELinkGetCollection($oIE)
For $oLink In $oLinks
    Local $sLinkId = _IEPropertyGet($oLink, "uniqueid")
    If @error Then Exit
    _IEHeadInsertEventScript($oIE, $sLinkId, "onclick", "return false;")
    If @error Then Exit
    ObjEvent($oLink, "_Evt_")
Next

; ожидание, пока окно браузера существует
While WinExists(_IEPropertyGet($oIE, "hwnd"))
    Sleep(100)
WEnd

Func _Evt_onClick()
    Local $o_Link = @COM_EventObj
    ConsoleWrite($o_Link.href & @CRLF)
EndFunc   ;==>_Evt_onClick
