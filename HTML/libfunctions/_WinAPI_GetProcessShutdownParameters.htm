Справочник функций

_WinAPI_GetProcessShutdownParameters

Получает параметры завершения для текущего вызываемого процесса.

#include <WinAPISys.au3>
_WinAPI_GetProcessShutdownParameters ( )

Возвращаемое значение

Успех: Возвращает уровень приоритета завершения (см. _WinAPI_SetProcessShutdownParameters()). Если установлен флаг @extended,
будет отображено диалоговое окно повторной попытки для пользователя.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
