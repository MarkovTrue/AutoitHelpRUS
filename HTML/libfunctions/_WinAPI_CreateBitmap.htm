_WinAPI_CreateBitmap

создаёт растровое изображение с указанными шириной, высотой и форматом цвета.

#include <WinAPIGdi.au3>
_WinAPI_CreateBitmap ( $iWidth, $iHeight [, $iPlanes = 1 [, $iBitsPerPel = 1 [, $pBits = 0]]] )

Параметры

$iWidth указывает растровое изображение width в pixels
$iHeight указывает растровое изображение height в pixels
$iPlanes Цвет (RGB).
$iBitsPerPel Цвет (RGB).
$pBits Указатель на структуру.

Возвращаемое значение

Успех: Возвращает дескриптор на созданное растровое изображение.
Ошибка: Возвращает 0.
