Справочник функций

_WinAPI_TransparentBlt

Выполняет побитовую передачу данных цвета, соответствующих прямоугольнику пикселей.

#include <WinAPIGdi.au3>
_WinAPI_TransparentBlt ( $hDestDC, $iXDest, $iYDest, $iWidthDest, $iHeightDest, $hSrcDC, $iXSrc, $iYSrc, $iWidthSrc, $iHeightSrc, $iRGB )

Параметры

$hDestDC Дескриптор контекста устройства назначения.
$iXDest Координата X верхнего левого угла целевого прямоугольника в логических единицах.
$iYDest Координата Y верхнего левого угла целевого прямоугольника в логических единицах.
$iWidthDest Ширина целевого прямоугольника в логических единицах.
$iHeightDest Высота целевого прямоугольника в логических единицах.
$hSrcDC Дескриптор исходного контекста устройства.
$iXSrc Координата X верхнего левого угла исходного прямоугольника в логических единицах.
$iYSrc Координата Y верхнего левого угла исходного прямоугольника в логических единицах.
$iWidthSrc Ширина исходного прямоугольника в логических единицах.
$iHeightSrc Высота исходного прямоугольника в логических единицах.
$iRGB Цвет RGB в исходном растре, который должен быть обработан как прозрачный.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Если исходный и целевой прямоугольники имеют разные размеры, исходный растр растягивается в соответствии с
целевым прямоугольником. Когда используется функция _WinAPI_SetStretchBltMode(), режимы растяжения
$BLACKONWHITE и $WHITEONBLACK преобразуются в $COLORONCOLOR для функции _WinAPI_TransparentBlt().

Эта функция поддерживает все форматы исходных растров. Однако для растров с 32 бит/пикс она просто копирует значение альфа.
Используйте функцию _WinAPI_AlphaBlend() для задания растров с 32 битами на пиксель с прозрачностью.

См. также

_WinAPI_AlphaBlend, _WinAPI_SetStretchBltMode

Дополнительно

Искать GdiTransparentBlt в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPISysWin.au3>
#include <WindowsSysColorConstants.au3>

Local $a_idPic[2], $a_hPic[2], $a_hBitmap[2]

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 260, 140)
$a_idPic[0] = GUICtrlCreatePic('', 20, 20, 100, 100)
$a_idPic[1] = GUICtrlCreatePic('', 140, 20, 100, 100)
For $i = 0 To 1
    $a_hPic[$i] = GUICtrlGetHandle($a_idPic[$i])
Next

; Создать растр1
Local $hDC = _WinAPI_GetDC($a_hPic[0])
Local $hDestDC = _WinAPI_CreateCompatibleDC($hDC)
$a_hBitmap[0] = _WinAPI_CreateCompatibleBitmapEx($hDC, 100, 100, 0xFF00FF)
Local $hDestSv = _WinAPI_SelectObject($hDestDC, $a_hBitmap[0])
Local $hObj = _WinAPI_CreateCompatibleBitmapEx($hDC, 70, 70, 0x00A060)
_WinAPI_DrawBitmap($hDestDC, 15, 15, $hObj)
_WinAPI_DeleteObject($hObj)
$hObj = _WinAPI_CreateCompatibleBitmapEx($hDC, 40, 40, 0xFF00FF)
_WinAPI_DrawBitmap($hDestDC, 30, 30, $hObj)
_WinAPI_DeleteObject($hObj)

_WinAPI_ReleaseDC($a_hPic[0], $hDC)
_WinAPI_SelectObject($hDestDC, $hDestSv)
_WinAPI_DeleteDC($hDestDC)

; Создать растр2
$hDC = _WinAPI_GetDC($a_hPic[1])
$hDestDC = _WinAPI_CreateCompatibleDC($hDC)
$a_hBitmap[1] = _WinAPI_CreateCompatibleBitmapEx($hDC, 100, 100, _WinAPI_SwitchColor(_WinAPI_GetSysColor($COLOR_3DFACE)))
$hDestSv = _WinAPI_SelectObject($hDestDC, $a_hBitmap[1])
Local $hSrcDC = _WinAPI_CreateCompatibleDC($hDC)
Local $hSrcSv = _WinAPI_SelectObject($hSrcDC, $a_hBitmap[0])
_WinAPI_TransparentBlt($hDestDC, 0, 0, 100, 100, $hSrcDC, 0, 0, 100, 100, 0xFF00FF)

_WinAPI_ReleaseDC($a_hPic[1], $hDC)
_WinAPI_SelectObject($hDestDC, $hDestSv)
_WinAPI_SelectObject($hSrcDC, $hSrcSv)
_WinAPI_DeleteDC($hDestDC)
_WinAPI_DeleteDC($hSrcDC)

; Задать оба bitmap для элементов управления
For $i = 0 To 1
    _SendMessage($a_hPic[$i], $STM_SETIMAGE, 0, $a_hBitmap[$i])
    $hObj = _SendMessage($a_hPic[$i], $STM_GETIMAGE)
    If $hObj <> $a_hBitmap[$i] Then
        _WinAPI_DeleteObject($a_hBitmap[$i])
    EndIf
Next

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
