_WinAPI_RegLoadMUIString

Загружает указанную строку из указанного ключа и подключа.

#include <WinAPIReg.au3>
_WinAPI_RegLoadMUIString ( $hKey, $sValueName [, $sDirectory = ''] )

Параметры

$hKey Дескриптор открытого ключа реестра. Ключ должен быть открыт с правом доступа $KEY_QUERY_VALUE.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey(). Это также может быть
один из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sValueName Имя значения реестра.
$sDirectory [необязательный] Путь каталога.

Возвращаемое значение

Успех: Возвращает загруженная строка.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Строки следующей формы получают специальную обработку:

@[path]\dllname,-strID

Строка с идентификатором strID загружается из dllname; путь необязателен.
Если параметр $sDirectory – пустая строка, каталог добавляется в начало пути, указанного в данных реестра.
Отметим, что dllname может содержать переменные окружения, которые должны быть развёрнуты.

Эта функция требует Windows Vista или позднее.
