Справочник функций

_WinAPI_GetDateFormat

Форматирует дату как строку даты для локали, указанной идентификатором локали.

#include <WinAPILocale.au3>
_WinAPI_GetDateFormat ( [$iLCID = 0 [, $tSYSTEMTIME = 0 [, $iFlags = $DATE_SHORTDATE [, $sFormat = '']]]] )

Параметры

$iLCID [необязательный] Идентификатор локали (LCID), указывающий локаль, или одно из следующих предопределённых значений.
$LOCALE_INVARIANT
$LOCALE_SYSTEM_DEFAULT
$LOCALE_USER_DEFAULT (по умолчанию)
Windows Vista или выше
$LOCALE_CUSTOM_DEFAULT
$LOCALE_CUSTOM_UI_DEFAULT
$LOCALE_CUSTOM_UNSPECIFIED
$tSYSTEMTIME Структура $tagSYSTEMTIME.
$iFlags [необязательный] Флаги, определяющие параметры формата даты.
Этот параметр может быть одним или несколькими из следующих значений:
$DATE_LONGDATE
$DATE_SHORTDATE (по умолчанию)
$DATE_USE_ALT_CALENDAR
Windows Vista или выше
$DATE_LTRREADING
$DATE_RTLREADING
$DATE_YEARMONTH
Windows 7 или выше
$DATE_AUTOLAYOUT
$sFormat [необязательный] Строка, используемая для формирования даты. Например, "dddd MMMM, yyyy".
Если этот параметр опущен или является пустой строкой (по умолчанию), функция возвращает строку в соответствии с форматом даты для указанной локали.

Возвращаемое значение

Успех: Возвращает отформатированную строку даты.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Дополнительно

Искать GetDateFormat в библиотеке MSDN

Пример

#include <WinAPILocale.au3>

ConsoleWrite(_WinAPI_GetDateFormat() & @CRLF)
ConsoleWrite(_WinAPI_GetDateFormat(0, 0, $DATE_LONGDATE) & @CRLF)
ConsoleWrite(_WinAPI_GetDateFormat(0, 0, 0, 'dddd dd, yyyy') & @CRLF)
