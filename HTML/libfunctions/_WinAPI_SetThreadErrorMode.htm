_WinAPI_SetThreadErrorMode

Управляет тем, будет ли система обрабатывать указанные типы серьёзных ошибок или это будет делать вызывающий поток.

#include <WinAPIProc.au3>
_WinAPI_SetThreadErrorMode ( $iMode )

Параметры

$iMode Режим ошибок потока. Этот параметр может быть одним или несколькими из следующих значений.     $SEM_FAILCRITICALERRORS     $SEM_NOGPFAULTERRORBOX     $SEM_NOOPENFILEERRORBOX.

Возвращаемое значение

Успех: Возвращает предыдущий режим ошибок потока ($SEM_*).
Ошибка: Устанавливает @error ≠ 0.

Примечания

Данная функция требует Windows 7 или более поздней версии.
