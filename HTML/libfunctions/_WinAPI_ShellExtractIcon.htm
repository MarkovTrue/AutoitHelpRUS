Справочник функций

_WinAPI_ShellExtractIcon

Извлекает значок с указанным размером из указанного файла.

#include <WinAPIShellEx.au3>
_WinAPI_ShellExtractIcon ( $sIcon, $iIndex, $iWidth, $iHeight )

Параметры

$sIcon Путь и имя файла, из которого должны быть извлечены значки.
$iIndex Индекс на основе 0 значка для извлечения. Если это значение отрицательное число, функция извлекает
значок, идентификатор ресурса которого равен абсолютному значению $iIndex.
$iWidth Требуемый горизонтальный размер значка.
$iHeight Требуемый вертикальный размер значка.

Возвращаемое значение

Успех: Возвращает дескриптор извлеченного значка.
Ошибка: Возвращает 0.

Примечания

Если значок с указанным размером не найден в файле, функция выберет ближайший подходящий значок
и изменит его на указанный размер.

Когда вы закончите использовать значок, удалите его с помощью функции _WinAPI_DestroyIcon().

См. также

_WinAPI_DestroyIcon

Дополнительно

Искать SHExtractIcons в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIIcons.au3>
#include <WinAPIShellEx.au3>

Local $iIndex = 0, $iTotal = _WinAPI_ExtractIconEx(@SystemDir & '\shell32.dll', -1, 0, 0, 0)

Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 160, 160)
Local $idButton = GUICtrlCreateButton('Next', 50, 130, 70, 23)
Local $idIcon = GUICtrlCreateIcon(@SystemDir & '\shell32.dll', 0, 69, 54, 32, 32)
Local $hIcon = GUICtrlGetHandle(-1)
GUISetState(@SW_SHOW)

While 1
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE
            ExitLoop
        Case $idButton
            $iIndex += 1
            If $iIndex > $iTotal - 1 Then
                $iIndex = 0
            EndIf
            _WinAPI_DestroyIcon(_SendMessage($hIcon, $STM_SETIMAGE, 1, _WinAPI_ShellExtractIcon(@SystemDir & '\shell32.dll', $iIndex, 32, 32)))
    EndSwitch
WEnd
