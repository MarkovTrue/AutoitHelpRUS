_WinAPI_DisplayStruct

Отображает данные из указанной структуры или по адресу памяти в виде списка.

#include <WinAPIDiag.au3>
_WinAPI_DisplayStruct ( $tStruct [, $sStruct = '' [, $sTitle = '' [, $iItem = 0 [, $iSubItem = 0 [, $iFlags = 0 [, $bTop = True [, $hParent = 0]]]]]]] )

Параметры

$tStruct Структура, созданная вызовом DllStructCreate(), либо адрес памяти, чьи данные нужно отобразить.
$sStruct [необязательный] Строка, описывающая структуру.
Если $tStruct – структура, параметр можно опустить или передать пустую строку. Тогда структура будет показана как "byte[n]".
Если $tStruct – адрес памяти, $sStruct должна быть строкой, описывающей структуру по этому адресу.
$sTitle [необязательный] Заголовок окна; по умолчанию "Structure: ListView Display".
$iItem [необязательный] Индекс (отсчёт от 1) или имя члена структуры, который должен быть выделен в списке.
Если параметр равен 0 (по умолчанию), либо индекс/имя некорректны, выделяется первый элемент структуры.
$iSubItem [необязательный] Индекс (отсчёт от 1) элемента массива внутри члена структуры, на который указывает $iItem.
Если $iItem в $sStruct не объявлен как массив либо индекс некорректен, выделяется сам элемент, на который указывает $iItem.
$iFlags [необязательный] Набор битовых флагов, задающих дополнительные опции отображения.
Принимает 0 либо комбинацию следующих значений:
1 – не показывать поля "<struct>" и "<endstruct>" в начале и конце списка
2 – показать только данные структуры, скрыв заголовки столбцов
4 – не позволять закрыть окно, кроме как через системное меню
$bTop [необязательный] Задаёт, создавать ли окно с атрибутом "Поверх всех окон". Допустимые значения:
True - Окно создаётся с расширенным стилем $WS_EX_TOPMOST (по умолчанию).
False - Флаг "TOPMOST" не устанавливается.
$hParent [необязательный] дескриптор к parent окно.

Возвращаемое значение

Успех: Возвращает 1; @extended может быть установлен (см. раздел Примечания).
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: Возвращает 2, если в переданной строке встретился неизвестный тип данных.
3 – не удалось выделить память под структуру либо передан некорректный указатель.
4 – ошибка выделения памяти под переданную строку.
10 – параметр $tStruct не является корректной DllStruct.
15 – ошибка доступа к памяти.

Примечания

Функция предназначена прежде всего для отладки и не должна использоваться в финальном скрипте, поскольку не несёт полезной информации для конечного пользователя.

Если значение элемента содержит "|", в выводе он заменяется на "¦", так как "|" используется внутренне при формировании результата.

Пример

#include <WinAPIDiag.au3>

Local $sStruct = 'dword Length;short State;uint Flags;handle hBitmap;hwnd hDC;long Rect[4];byte[14];int Reserved[10];wchar Text[80];wchar Text[80]'
Local $tStruct = DllStructCreate($sStruct)

DllStructSetData($tStruct, 1, 80)
DllStructSetData($tStruct, 2, 6)
DllStructSetData($tStruct, 3, 1)
DllStructSetData($tStruct, 4, 0x00010014)
DllStructSetData($tStruct, 5, 0x01010057)
DllStructSetData($tStruct, 6, 20, 1)
DllStructSetData($tStruct, 6, 20, 2)
DllStructSetData($tStruct, 6, 60, 3)
DllStructSetData($tStruct, 6, 80, 4)
DllStructSetData($tStruct, 7, Binary('0x656A6D633835206C6A6764206200'))
DllStructSetData($tStruct, 8, 0)
DllStructSetData($tStruct, 9, 'Simple Text')
DllStructSetData($tStruct, 10, 'Text with | ...')

_WinAPI_DisplayStruct($tStruct, $sStruct)
