Справочник функций

_WinAPI_CreateDesktop

создаёт новый рабочий стол и связывает его с текущей оконной станцией вызывающего процесса.

#include <WinAPISys.au3>
_WinAPI_CreateDesktop ( $sName [, $iAccess = $DESKTOP_CREATEWINDOW [, $iFlags = 0 [, $iHeap = 0 [, $tSecurity = 0]]]] )

Параметры

$sName Имя.
$iAccess [необязательный] Запрашиваемый доступ к рабочему столу. Может быть одним или несколькими из следующих значений:
$DESKTOP_ALL_ACCESS
$DESKTOP_CREATEMENU
$DESKTOP_CREATEWINDOW (по умолчанию)
$DESKTOP_ENUMERATE
$DESKTOP_HOOKCONTROL
$DESKTOP_JOURNALPLAYBACK
$DESKTOP_JOURNALRECORD
$DESKTOP_READOBJECTS
$DESKTOP_SWITCHDESKTOP
$DESKTOP_WRITEOBJECTS
$iFlags Количество / Размер.
$iHeap Количество / Размер.
$tSecurity Дескриптор.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Количество создаваемых рабочих столов ограничено размером системной кучи рабочих столов.
Увеличить число создаваемых рабочих столов можно либо увеличив размер кучи рабочих столов, либо уменьшив объём кучи, резервируемой по умолчанию для каждого рабочего стола в интерактивной оконной станции.
Размер кучи рабочих столов по умолчанию зависит от ряда факторов, например, архитектуры оборудования.
Чтобы получить размер кучи, вызовите функцию _WinAPI_GetUserObjectInformation() с параметром $UOI_HEAPSIZE.

Когда рабочий стол больше не нужен, закройте его функцией _WinAPI_CloseDesktop().

См. также

_WinAPI_CloseDesktop, _WinAPI_GetUserObjectInformation

Дополнительно

Искать CreateDesktopEx в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <StructureConstants.au3>
#include <WinAPIMem.au3>
#include <WinAPIProc.au3>
#include <WinAPISys.au3>

; Получить дескриптор текущего рабочего стола и создать новый "MyDesktop"
Local $hPrev = _WinAPI_GetThreadDesktop(_WinAPI_GetCurrentThreadId())
Local $hDesktop = _WinAPI_CreateDesktop('MyDesktop', BitOR($DESKTOP_CREATEWINDOW, $DESKTOP_SWITCHDESKTOP))
If Not $hDesktop Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to create desktop.')
    Exit
EndIf

; Переключение на только что созданный рабочий стол
_WinAPI_SwitchDesktop($hDesktop)

; Запустить "notepad.exe" на "MyDesktop" и подождать, пока пользователь не закроет процесс
Local $pText = _WinAPI_CreateString('MyDesktop')
Local $tProcess = DllStructCreate($tagPROCESS_INFORMATION)
Local $tStartup = DllStructCreate($tagSTARTUPINFO)
DllStructSetData($tStartup, 'Size', DllStructGetSize($tStartup))
DllStructSetData($tStartup, 'Desktop', $pText)
If _WinAPI_CreateProcess('', @SystemDir & '\notepad.exe', 0, 0, 0, $CREATE_NEW_PROCESS_GROUP, 0, 0, $tStartup, $tProcess) Then
    ProcessWaitClose(DllStructGetData($tProcess, 'ProcessID'))
EndIf

; Переключение на предыдущий рабочий стол и закрытие "MyDesktop"
_WinAPI_SwitchDesktop($hPrev)
_WinAPI_CloseDesktop($hDesktop)

; Освободить память, выделенную для строки
_WinAPI_FreeMemory($pText)
