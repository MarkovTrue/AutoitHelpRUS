_WinAPI_IsThemeActive

Проверяет, активен ли визуальный стиль для текущего приложения.

#include <WinAPITheme.au3>
_WinAPI_IsThemeActive ( )

Возвращаемое значение

True: Возвращает визуальный стиль включён.
False: визуальный стиль отключён.
