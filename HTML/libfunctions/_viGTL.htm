Справочник функций

_viGTL

Переводит в локальный режим: приборы, принимающие эту команду, выходят из режима "удалённого управления" и переходят в режим "локального управления". Если прибор уже находится в режиме "локального управления", эта команда просто игнорируется. Обычно, если прибор не поддерживает эту команду, он просто остаётся в режиме "удалённого управления".

#include <Visa.au3>
_viGTL ( $hSession )

Параметры

$hSession Дескриптор VISA (строка) или дескриптор сессии VISA (целое число).
Подробности см. раздел Примечания _viExecCommand().

Возвращаемое значение

Успех: Возвращает 0.
Ошибка: Возвращает -1, если DLL-библиотеку VISA не удаётся открыть
или ненулевое значение, представляющее код ошибки VISA (см. руководство программиста VISA).

Примечания

Как и для всех функций VISA, библиотеки VISA должны быть установлены (вы можете проверить, находится ли visa32.dll в {WINDOWS}\system32), а также должна быть установлена карта GPIB (такая как NI PCI-GPIB от National Instruments или Agilent 82350B PCI High-Performance GPIB)

* Подробное описание наиболее распространённых ДЕСКРИПТОРОВ VISA см. в примечаниях функции _viExecCommand()

См. также

_viClose, _viExecCommand, _viFindGpib, _viGpibBusReset, _viOpen

Пример

; - Предполагает, что прибор установлен на адрес GPIB 3
; Если прибор находится по другому адресу, измените "GPIB::3::0" на
; соответствующий дескриптор. То же самое для вызова _viOpen
; Показывает, как использовать функцию _viGTL с дескриптором VISA и с
; Обработчиком устройства VISA. Сначала мы используем _viExecCommand, чтобы заставить прибор перейти
; в режим "удалённого управления"

#include <MsgBoxConstants.au3>
#include <Visa.au3>

Local $h_Session = 0

; Запросить ID прибора по адресу GPIB 3
MsgBox($MB_SYSTEMMODAL, "Step 1", "Simple GPIB query using a VISA Descriptor")
Local $s_Answer = _viExecCommand("GPIB::3::0", "*IDN?", 10)
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

MsgBox($MB_SYSTEMMODAL, "Step 2", "Go to LOCAL using VISA Descriptor")
_viGTL("GPIB::1::0") ; Перейти в локальный режим (выйти из удалённого управления)

MsgBox($MB_SYSTEMMODAL, "Step 4", "Open the instrument connection with _viOpen")
Local $h_Instr = _viOpen(3)
MsgBox($MB_SYSTEMMODAL, "Instrument Handle obtained", "$h_Instr = " & $h_Instr) ; Показать дескриптор сессии
; Запросить прибор

MsgBox($MB_SYSTEMMODAL, "Step 5", "Query the instrument using the VISA instrument handle")
$s_Answer = _viExecCommand($h_Instr, "*IDN?") ; $h_Instr теперь НЕ СТРОКА!
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ
; Запросить снова. Нет необходимости открывать соединение заново

MsgBox($MB_SYSTEMMODAL, "Step 6", "Query again. There is no need to OPEN the link again")
$s_Answer = _viExecCommand($h_Instr, "*IDN?")
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

MsgBox($MB_SYSTEMMODAL, "Step 7", "Go to LOCAL using VISA instrument handle")
_viGTL($h_Instr) ; Перейти в локальный режим (опционально)

MsgBox($MB_SYSTEMMODAL, "Step 8", "Close the Instrument connection using _viClose")
_viClose($h_Instr) ; Закрыть соединение с прибором
