_WinAPI_GetCaretBlinkTime

Возвращает время, необходимое для инвертирования пикселей каретки.

#include <WinAPIRes.au3>
_WinAPI_GetCaretBlinkTime ( )

Возвращаемое значение

Успех: Возвращает время мигания в миллисекундах.
Ошибка: Возвращает 0. Для получения расширенной информации об ошибке вызовите _WinAPI_GetLastError().

Примечания

Возвращаемое значение INFINITE указывает на то, что каретка не мигает.
