Справочник функций

_GUICtrlListView_FindItem

Ищет элемент с заданными характеристиками.

#include <GuiListView.au3>
_GUICtrlListView_FindItem ( $hWnd, $iStart, ByRef $tFindInfo [, $sText = ""] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iStart Индекс.
$tFindInfo Структура $tagLVFINDINFO с параметрами поиска.
$sText [необязательный] Строка для сравнения с текстом элемента.
Действует, если в поле Flags установлен $LVFI_STRING или $LVFI_PARTIAL.

Возвращаемое значение

Успех: Возвращает индекс элемента (нумерация с 0).
Ошибка: Возвращает -1.

См. также

$tagLVFINDINFO, _GUICtrlListView_FindNearest, _GUICtrlListView_FindParam

Пример

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>
#include <MsgBoxConstants.au3>
#include <StructureConstants.au3>

Example()

Func Example()
    GUICreate("ListView Find Item (v" & @AutoItVersion & ")", 400, 300)
    Local $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Установить формат ANSI
;~ _GUICtrlListView_SetUnicodeFormat($idListview, False)

    ; Добавить столбцы
    _GUICtrlListView_AddColumn($idListview, "Items", 100)

    ; Добавить элементы
    _GUICtrlListView_BeginUpdate($idListview)
    Local $iI
    For $iI = 1 To 100
        _GUICtrlListView_AddItem($idListview, "Item " & $iI)
    Next
    _GUICtrlListView_EndUpdate($idListview)

    ; Установить параметр элемента 50
    _GUICtrlListView_SetItemParam($idListview, 49, 1234)

    ; Найти целевой элемент
    Local $tInfo = DllStructCreate($tagLVFINDINFO)
    DllStructSetData($tInfo, "Flags", $LVFI_PARAM)
    DllStructSetData($tInfo, "Param", 1234)
    $iI = _GUICtrlListView_FindItem($idListview, -1, $tInfo)
    MsgBox($MB_SYSTEMMODAL, "Information", "Target Item Index: " & $iI)
    _GUICtrlListView_EnsureVisible($idListview, $iI)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
