_WinAPI_RegDeleteValue

Удаляет именованное значение из указанного ключа реестра.

#include <WinAPIReg.au3>
_WinAPI_RegDeleteValue ( $hKey, $sValueName )

Параметры

$hKey Дескриптор открытого ключа реестра. Ключ должен быть открыт с правом доступа $KEY_SET_VALUE.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey(),
или может быть одним из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sValueName Удаляемое значение реестра. Если этот параметр – пустая строка, удаляется безымянное или стандартное значение ключа.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

См. также

_WinAPI_RegCreateKey, _WinAPI_RegOpenKey
