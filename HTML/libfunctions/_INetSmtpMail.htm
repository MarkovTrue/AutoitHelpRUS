Справочник функций

_INetSmtpMail

Отправляет электронное письмо без использования внешней почтовой программы.

#include <Inet.au3>
_INetSmtpMail ( $sSMTPServer, $sFromName, $sFromAddress, $sToAddress [, $sSubject = "" [, $aBody = "" [, $sEHLO = "" [, $sFirst = " " [, $bTrace = 0]]]]] )

Параметры

$sSMTPServer SMTP-сервер, через который будет отправлено письмо. Может быть либо имя сервера, либо численный IP-адрес. Для борьбы со спамом многие провайдеры требуют использовать их серверы.
например "smtp.ispdomain.com", "mail.ispdomain.com" или "192.168.1.1"
$sFromName Имя отправителя, которое будет показано в письме.
например "Bob Smith"
$sFromAddress Адрес электронной почты, который будет показан как адрес отправителя.
например "bob.smith@mydomain.com".
$sToAddress Адрес электронной почты получателя.
например "jane.brown@yourdomain.com"
$sSubject [необязательный] Тема письма.
$aBody [необязательный] Текст письма в виде одномерного массива строк. Каждая строка в письме будет завершена символом @CRLF.
$sEHLO [необязательный] Идентификатор для подключения к SMTP-серверу (по умолчанию @ComputerName). Если SMTP-сервер требует строку "EHLO", установите значение "EHLO " & @ComputerName.
$sFirst [необязательный] Строка, отправляемая перед HELO для подключения к SMTP-серверу (по умолчанию пробел). Чтобы не отправлять символы, установите значение -1; некоторые SMTP-серверы это требуют.
$bTrace [необязательный] Отобразить диалог в окне сообщения

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1
2 – Не удалось запустить TCP
3 – Не удалось разрешить IP
4 – Не удалось создать сокет
5 – Не удалось получить информацию о часовом поясе
5x – Не удалось открыть сеанс SMTP. x – номер индекса последней команды, отправленной на SMTP-сервер.
50x – Не удалось отправить текст письма. x – номер строки массива $aBody (первая строка имеет номер 0).
5000 – Не удалось закрыть сеанс SMTP

Примечания

Функция отправляет письмо напрямую через SMTP-сервер без использования сторонних почтовых клиентов.

См. также

_INetMail

Пример

#include <Inet.au3>
#include <MsgBoxConstants.au3>

Local $s_SmtpServer = "mysmtpserver.com.au"
Local $s_FromName = "My Name"
Local $s_FromAddress = "From eMail Address"
Local $s_ToAddress = "To eMail Address"
Local $s_Subject = "My Test UDF"
Local $as_Body[2]
$as_Body[0] = "Testing the new email udf"
$as_Body[1] = "Second Line"
Local $iResponse = _INetSmtpMail($s_SmtpServer, $s_FromName, $s_FromAddress, $s_ToAddress, $s_Subject, $as_Body)
Local $iErr = @error
If $iResponse = 1 Then
    MsgBox($MB_SYSTEMMODAL, "Success!", "Mail sent")
Else
    MsgBox($MB_SYSTEMMODAL, "Error!", "Mail failed with error code " & $iErr)
EndIf
