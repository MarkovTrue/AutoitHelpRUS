Справочник функций

_WinAPI_UrlApplyScheme

Определяет схему для указанной строки URL и возвращает строку с подходящим префиксом.

#include <WinAPIShPath.au3>
_WinAPI_UrlApplyScheme ( $sUrl [, $iFlags = $URL_APPLY_DEFAULT] )

Параметры

$sUrl URL.
$iFlags [необязательный] Флаги, которые указывают, как определить схему. Это может быть комбинацией следующих значений.
$URL_APPLY_DEFAULT (по умолчанию)
$URL_APPLY_GUESSSCHEME
$URL_APPLY_GUESSFILE
$URL_APPLY_FORCEAPPLY

Возвращаемое значение

Успех: Возвращает URL, преобразованный в схему "//URL_string format".
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Дополнительно

Искать UrlApplyScheme в библиотеке MSDN

Пример

#include <WinAPIShPath.au3>

Local $sUrl = 'www.microsoft.com'

ConsoleWrite(_WinAPI_UrlApplyScheme($sUrl) & @CRLF)

ConsoleWrite(_WinAPI_UrlApplyScheme('') & @CRLF)
