_ArrayPermute

Возвращает массив всевозможных перестановок элементов одномерного массива.

#include <Array.au3>
_ArrayPermute ( ByRef $aArray [, $sDelimiter = ""] )

Параметры

$aArray Исходный массив для получения перестановок.
$sDelimiter [необязательный] Разделитель элементов в строке результата. По умолчанию "" (без разделителя).

Возвращаемое значение

Успех: Возвращает массив перестановок.
$aArray[0] содержит количество возвращённых строк.
Остальные элементы ($aArray[1], $aArray[2] ... $aArray[n]) содержат сами перестановки.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным массивом

Примечания

Входной массив должен быть с нулевым индексом, то есть без счётчика в $aArray[0].
Основан на алгоритме Александра Богомольного (http://www.bearcave.com/random_hacks/permute.html).

Пример

; Объявляем одномерный массив и получаем массив перестановок

#include <Array.au3>

Local $aArray[4] = [1, 2, 3, 4]
Local $aNewArray = _ArrayPermute($aArray, ",") ; Использование параметров по умолчанию.
_ArrayDisplay($aNewArray, "Array Permuted")
