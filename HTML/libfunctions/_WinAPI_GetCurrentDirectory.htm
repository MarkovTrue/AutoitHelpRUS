Справочник функций

_WinAPI_GetCurrentDirectory

Получает текущий каталог для текущего процесса.

#include <WinAPIFiles.au3>
_WinAPI_GetCurrentDirectory ( )

Возвращаемое значение

Успех: Возвращает абсолютный путь к текущему каталогу.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
