Справочник функций

_WinNet_ConnectionDialog1

Запускает диалоговое окно общего просмотра для подключения к сетевым ресурсам.

#include <WinNet.au3>
_WinNet_ConnectionDialog1 ( $hWnd [, $sRemoteName = "" [, $iFlags = 2]] )

Параметры

$hWnd Дескриптор окна-владельца диалогового окна.
$sRemoteName Имя сетевого ресурса для подключения.
$iFlags [необязательный] Параметры диалогового окна. Может быть одним или несколькими из следующих:
1 – Отображать путь только для чтения вместо возможности ввода пути пользователем
2 – Введите недавно использованные пути в поле со списком
4 – Не показывать флажок восстановления подключения при входе
8 – Восстановить подключение при входе
16 – Не восстанавливать подключение при входе.

Возвращаемое значение

Успех: Возвращает номер подключённого устройства. Значение 1 для A:, 2 для B:, 3 для C: и так далее. Если пользователь создал подключение без устройства, значение равно –1.
Ошибка: Устанавливает @error ≠ 0.
