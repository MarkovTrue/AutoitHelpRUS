_WinAPI_EqualRect

Определяет, равны ли два указанных прямоугольника.

#include <WinAPIGdi.au3>
_WinAPI_EqualRect ( $tRECT1, $tRECT2 )

Параметры

$tRECT1 Структура $tagRECT с логическими координатами первого прямоугольника.
$tRECT2 Структура $tagRECT с логическими координатами второго прямоугольника.

Возвращаемое значение

Успех: Возвращает True – прямоугольники идентичны.
Ошибка: Возвращает False.

Примечания

Функция не считает пустые прямоугольники с разными координатами равными.
