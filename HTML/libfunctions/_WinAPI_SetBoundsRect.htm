Справочник функций

_WinAPI_SetBoundsRect

Управляет накоплением информации об ограничивающем прямоугольнике для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetBoundsRect ( $hDC, $iFlags [, $tRECT = 0] )

Параметры

$hDC Дескриптор контекста устройства.
$iFlags Флаги, определяющие, как новый прямоугольник будет объединён с накопленным прямоугольником.
Этот параметр может быть одним или несколькими из следующих значений.
$DCB_ACCUMULATE
$DCB_DISABLE
$DCB_ENABLE
$DCB_RESET
$tRECT [необязательный] Структура $tagRECT для установки ограничивающего прямоугольника в логических координатах.

Возвращаемое значение

Успех: Возвращает значение, определяющее предыдущее состояние ограничивающего прямоугольника ($DCB_*).
Ошибка: Возвращает 0.
