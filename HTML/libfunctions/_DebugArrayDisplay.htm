Справочник функций

_DebugArrayDisplay

Отображает одномерный или двумерный массив в элементе ListView для помощи при отладке.

#include <Debug.au3>
_DebugArrayDisplay ( Const ByRef $aArray [, $sTitle = "DebugArray" [, $sArrayRange = "" [, $iFlags = 0 [, $vUser_Separator = Default [, $sHeader = Default [, $iDesired_Colwidth = Default [, $hUser_Function = ""]]]]]]] )

Параметры

$aArray Массив для отображения.
$sTitle [необязательный] Заголовок диалогового окна. По умолчанию – "DebugArray".
$sArrayRange [необязательный] Диапазон строк/столбцов для отображения. По умолчанию ("") – весь массив. (см. раздел Примечания.)
$iFlags [необязательный] Параметры UDF. Суммируйте нужные значения.
$ARRAYDISPLAY_COLALIGNLEFT (0) = (по умолчанию) выравнивание текста в столбцах – по левому краю
$ARRAYDISPLAY_TRANSPOSE (1) = транспонировать массив (только для 2D)
$ARRAYDISPLAY_COLALIGNRIGHT (2) = выравнивание текста в столбцах – по правому краю
$ARRAYDISPLAY_COLALIGNCENTER (4) = выравнивание текста в столбцах – по центру
$ARRAYDISPLAY_VERBOSE (8) = подробный режим – отображать MsgBox при ошибке и заставки при обработке больших массивов
$ARRAYDISPLAY_NOROW (64) = не показывать столбец "Row"
$ARRAYDISPLAY_CHECKERROR (128) = CheckError – не отображать, если при вызове установлен @error.
$vUser_Separator [необязательный] Параметр отображения столбцов при копировании данных в буфер обмена.
Символ = разделитель между столбцами.
Число = фиксированная ширина столбца; более длинные значения обрезаются.
По умолчанию – текущий символ-разделитель (обычно "|").
$sHeader [необязательный] Имена столбцов в заголовке (строка имён, разделённых текущим символом-разделителем – обычно "|"). Значение по умолчанию – см. раздел Примечания.
$iDesired_Colwidth [необязательный] Если положительное – максимальная ширина, до которой столбец ListView расширяется для отображения содержимого. По умолчанию – 350 пикселей.
Если отрицательное – все столбцы, кроме первого (55 пикселей), устанавливаются равными Abs($iDesired_Colwidth).
Ширина столбца всегда подстраивается под длину отображаемого элемента.
$hUser_Function [необязательный] Переменная, содержащая пользовательскую функцию для вызова. По умолчанию – не задана. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0; флаг @error устанавливается следующим образом:
@error: 1 – $aArray не является массивом
2 – $aArray имеет слишком много измерений (поддерживаются только 1D и 2D)
3 – при вызове функции установлен @error; вызывается _DebugReport()

Примечания

Если функции передана не-массивная переменная или массив с более чем двумя измерениями, функция возвращает ошибку и скрипт продолжает выполнение. Если в $iFlags задан флаг "verbose", отображается MsgBox с предложением немедленно завершить скрипт или продолжить с обычным возвратом ошибки.

Размер отображаемого массива не ограничен, однако из-за ограничений Windows-элемента управления заголовки и столбцы ListView перестают выравниваться при количестве столбцов более ~600.

Синтаксис параметра $sArrayRange:

"7" Строки 0–7, все столбцы
"7:" Строки 7–конец, все столбцы
"|7" Все строки, столбцы 0–7
"|7:" Все строки, столбцы 7–конец
"7|7" Строки 0–7, столбцы 0–7
"5:7" Строки 5–7, все столбцы
"|5:7" Все строки, столбцы 5–7
"7|5:7" Строки 0–7, столбцы 5–7
"5:7|7" Строки 5–7, столбцы 0–7
"5:7|5:7" Строки 5–7, столбцы 5–7

Для одномерных массивов значения столбцов игнорируются.

Имена из $sHeader (разделённые текущим символом-разделителем) используются для соответствующего числа столбцов. Если пользовательских имён нет или их недостаточно, применяется заголовок по умолчанию: "Row|Col0" для 1D или "Row|Col0|Col1|...|Col n" для 2D. При транспонировании массива заголовок игнорируется.
Переменная $ARRAYDISPLAY_ROWPREFIX задаёт префикс нумерации строк (по умолчанию: #).
Чтобы принудительно задать числовое сравнение по конкретному столбцу, добавьте $ARRAYDISPLAY_NUMERICSORT (по умолчанию: *) в конец имени этого столбца.

Четыре кнопки в нижней части диалога:
Copy Data & Hdr/Row Копировать весь массив или выделенные строки в буфер обмена с заголовком и идентификатором строки.
Copy Data Only Копировать весь массив или выделенные строки без заголовка и идентификатора строки.
Run User Func Запустить пользовательскую функцию, переданную в $hUser_Function. Функция полностью независима от UDF и должна быть создана пользователем (см. пример). Кнопка не отображается, если функция не задана.
Exit script Немедленно завершить скрипт.

Размерность массива отображается в левом нижнем углу. Красным цветом – если массив транспонирован или отображается только диапазон элементов; подсказка показывает конкретное условие.

Если в $iFlags задан флаг "verbose" и массив содержит более 10000 элементов, при начальной обработке отображается заставка. Аналогичная заставка появляется при нажатии кнопок "Copy Data" для отображения с более чем 10000 элементами – в этом случае рекомендуется использовать пользовательскую функцию для сохранения массива.

Элемент массива, содержащий другой массив или Map, отображается как {Array} или {Map}.

При использовании $ARRAYDISPLAY_TRANSPOSE или $sArrayRange создаётся копия $aArray, что увеличивает потребление памяти.

Щёлчок по заголовку столбца выполняет сортировку; при большом числе строк это может занять несколько секунд – всплывающая подсказка предупредит об этом.

$hUser_Function должна принимать ровно 2 параметра (см. пример):
    Param1 – отображаемый массив $aArray
    Param2 – одномерный массив индексов выделенных строк; [0] содержит их количество
Эти параметры используются внутри пользовательской функции по необходимости.

Пример

#include <Debug.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()

    ; Создать одномерный массив для отображения
    Local $aArray_1D[5] = ["Item 0", "Item 1", "A longer Item 2 to show column expansion", "Item 3", "Item 4"]

    _DebugArrayDisplay($aArray_1D, "1D display")

    ; Создать двумерный массив для отображения
    Local $aArray_2D[25][15]
    For $i = 0 To UBound($aArray_2D) - 1
        For $j = 0 To UBound($aArray_2D, 2) - 1
            $aArray_2D[$i][$j] = "Item " & StringFormat("%02i", $i) & StringFormat("%02i", $j)
        Next
    Next

    _DebugArrayDisplay($aArray_2D, "2D display")

    $aArray_2D[5][5] = "A longer item to show column expansion"
    _DebugArrayDisplay($aArray_2D, "Expanded column - custom titles - no 'Row' column", Default, $ARRAYDISPLAY_NOROW, Default, "AA|BB|CC|DD|EE|FF|GG|HH|II|JJ")

    ; Присваиваем пользовательскую функцию переменной для передачи в качестве параметра.
    Local $hUserFunction = _UserFunc

    $aArray_2D[5][5] = "Column alignment set to right"
    _DebugArrayDisplay($aArray_2D, "Range set - right align - copy column width - user function", "3:7|4:9", $ARRAYDISPLAY_COLALIGNRIGHT, 15, "AA|BB|CC|DD|EE|FF", Default, $hUserFunction)
    _DebugArrayDisplay($aArray_2D, "Range set - transposed", "3:7|4:9", $ARRAYDISPLAY_TRANSPOSE, Default, "AA|BB|CC|DD|EE|FF") ; Имена столбцов не указываются, так как массив транспонирован.

    $aArray_2D[5][5] = "Column alignment set to left"
    Opt("GUIDataSeparatorChar", "!")
     _DebugArrayDisplay($aArray_2D, "Header separator = !", "3:7|4:9", Default, Default, "AA!BB!CC!DD!EE!FF")

    Local $iRet, $iError = 2
    SetError($iError)
    $iRet = _DebugArrayDisplay($aArray_2D, "Check error no display", "3:7|4:9", $ARRAYDISPLAY_CHECKERROR)
    MsgBox(0, "_DebugArrayDisplay() Error", "return without internal Msgbox $iret =" & $iRet & " @error=" & $iError)

    ; Создать не-массивную переменную, чтобы вызвать ошибку – MsgBox показывается, так как $iFlags задан
    Local $vVar = 0
    $iRet = _DebugArrayDisplay($vVar, "No MsgBox on Error")
    $iError = @error
    MsgBox(0, "_DebugArrayDisplay() Error", "return without internal Msgbox $iret =" & $iRet & " @error=" & $iError)

    $iRet = _DebugArrayDisplay($vVar, "MsgBox on Error", Default, $ARRAYDISPLAY_VERBOSE)
    $iError = @error
    MsgBox(0, "_DebugArrayDisplay() Error", "return internal Msgbox with no force Exit $iret =" & $iRet & " @error=" & $iError)

EndFunc    ;==>Example

; Пользовательская функция ДОЛЖНА иметь ДВА параметра, даже если оба не используются.
Func _UserFunc($aArray_2D, $aSelected)

    ; Если параметр не используется – добавьте заглушку, чтобы избежать предупреждения Au3Check.
    #forceref $aArray_2D

    _DebugArrayDisplay($aSelected, "Selected cols")

EndFunc    ;==>_UserFunc
