Справочник функций

_DebugArrayDisplay

Отображает одномерный или двумерный массив в виде ListView для упрощения отладки.

#include <Debug.au3>
_DebugArrayDisplay ( Const ByRef $aArray [, $sTitle = "DebugArray" [, $sArrayRange = "" [, $iFlags = 0 [, $vUser_Separator = Default [, $sHeader = Default [, $iDesired_Colwidth = Default [, $hUser_Function = ""]]]]]]] )

Параметры

$aArray Отображаемый массив.
$sTitle [необязательный] Заголовок диалогового окна. По умолчанию "DebugArray".
$sArrayRange [необязательный] Диапазон отображаемых строк или столбцов. По умолчанию ("") отображается весь массив (подробности ниже).
$iFlags [необязательный] Параметры UDF. Складывайте нужные значения:
$ARRAYDISPLAY_COLALIGNLEFT (0) = (по умолчанию) выравнивание текста столбцов по левому краю
$ARRAYDISPLAY_TRANSPOSE (1) = транспонировать массив (только для двумерного массива)
$ARRAYDISPLAY_COLALIGNRIGHT (2) = выравнивание текста столбцов по правому краю
$ARRAYDISPLAY_COLALIGNCENTER (4) = выравнивание текста столбцов по центру
$ARRAYDISPLAY_VERBOSE (8) = подробный режим – показывать MsgBox при ошибке и заставки при обработке больших массивов
$ARRAYDISPLAY_NOROW (64) = столбец "Row" не отображается
$ARRAYDISPLAY_CHECKERROR (128) = CheckError – ничего не показывать, если при вызове установлен @error.
$vUser_Separator [необязательный] Задаёт способ отображения столбцов при копировании данных в буфер обмена.
Символ = разделитель между столбцами.
Число = фиксированная ширина столбца – более длинные значения обрезаются.
По умолчанию = текущий символ-разделитель (обычно "|").
$sHeader [необязательный] Имена столбцов в заголовке (строка имён, разделённых текущим символом-разделителем, обычно "|"). По умолчанию см. раздел Примечания.
$iDesired_Colwidth [необязательный] Если положительное, максимальная ширина, до которой столбец ListView расширяется, чтобы показать содержимое. По умолчанию 350 пикселей.
Если отрицательное, все столбцы, кроме первого (для него задано 55), устанавливаются равными Abs($iDesired_Colwidth).
Но помните, что размер столбца всегда подстраивается под длину отображаемого элемента, чтобы показать его данные.
$hUser_Function [необязательный] Переменная, которой присвоена вызываемая пользовательская функция. По умолчанию не задана. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, а флаг @error устанавливается следующим образом:
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным или двумерным массивом
3 – на момент вызова функции был установлен @error, выполняется _DebugReport()

Примечания

Если функции передана переменная, не являющаяся массивом, или массив с числом измерений больше 2, функция возвращает ошибку, и скрипт продолжает работу. Если в $iFlags установлен флаг "verbose", появляется MsgBox с возможностью немедленно завершить скрипт или продолжить с обычным возвратом ошибки.

На размер отображаемого массива ограничений нет, однако из-за ограничения Windows-контрола заголовки и столбцы ListView перестают выравниваться, если их больше примерно 600.

Синтаксис параметра $sArrayRange:

"7" Строки 0-7, все столбцы
"7:" Строки 7-конец, все столбцы
"|7" Все строки, столбцы 0-7
"|7:" Все строки, столбцы 7-конец
"7|7" Строки 0-7, столбцы 0-7
"5:7" Строки 5-7, все столбцы
"|5:7" Все строки, столбцы 5-7
"7|5:7" Строки 0-7, столбцы 5-7
"5:7|7" Строки 5-7, столбцы 0-7
"5:7|5:7" Строки 5-7, столбцы 5-7

Для одномерных массивов значения столбцов игнорируются.

Имена из $sHeader (разделённые текущим символом-разделителем) применяются к стольким столбцам, сколько задано имён. Если пользовательские имена не заданы или их не хватает, подставляется заголовок по умолчанию "Row|Col0" для одномерного массива или "Row|Col0|Col1|...|Col n" для двумерного. При отображении массива транспонированным заголовок игнорируется.
Переменная $ARRAYDISPLAY_ROWPREFIX задаёт префикс нумерации строк (по умолчанию: #).
Чтобы для конкретного столбца использовать числовое сравнение, завершите соответствующее имя символом $ARRAYDISPLAY_NUMERICSORT (по умолчанию: *).

Четыре кнопки внизу диалога выполняют следующие функции:
Copy Data & Hdr/Row Копировать массив или выделенные строки в буфер обмена, добавляя полный заголовок и идентификацию строк.
Copy Data Only Копировать массив или выделенные строки в буфер обмена без заголовка и идентификации строк.
Run User Func Запустить пользовательскую функцию, переданную в $hUser_Function. Эта функция полностью отделена от UDF и должна быть написана пользователем (см. ниже). Кнопка не отображается, если функция не задана.
Exit script Немедленно завершить скрипт.

Размеры массива показываются в левом нижнем углу. Текст красный, если массив транспонирован или показан только диапазон элементов – конкретный случай указывается во всплывающей подсказке.

Если в $iFlags установлен флаг "verbose", при начальной обработке массива с более чем 10000 элементами показывается заставка. Похожее окно появляется при использовании кнопок "Copy Data" на массиве из 10000+ элементов – если такое вероятно, стоит сохранить массив через пользовательскую функцию для дальнейшей обработки.

Элемент массива, содержащий другой массив или Map, отображается как {Array} или {Map}.

При использовании $ARRAYDISPLAY_TRANSPOSE или $sArrayRange создаётся копия $aArray, поэтому расходуется дополнительная память.

Щёлкните по заголовку столбца, чтобы отсортировать его. При большом числе строк это может занять несколько секунд. Во время загрузки появляется всплывающая подсказка с предупреждением, что сортировка может занять несколько секунд.

$hUser_Function должна принимать ровно 2 параметра (см. пример):
    Param1 – отображаемый массив $aArray
    Param2 – одномерный массив индексов выделенных строк, в элементе [0] хранится их количество
Эти параметры можно использовать внутри пользовательской функции по необходимости.

Пример

#include <Debug.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()

    ; Создать одномерный массив для отображения
    Local $aArray_1D[5] = ["Item 0", "Item 1", "A longer Item 2 to show column expansion", "Item 3", "Item 4"]

    _DebugArrayDisplay($aArray_1D, "1D display")

    ; Создать двумерный массив для отображения
    Local $aArray_2D[25][15]
    For $i = 0 To UBound($aArray_2D) - 1
        For $j = 0 To UBound($aArray_2D, 2) - 1
            $aArray_2D[$i][$j] = "Item " & StringFormat("%02i", $i) & StringFormat("%02i", $j)
        Next
    Next

    _DebugArrayDisplay($aArray_2D, "2D display")

    $aArray_2D[5][5] = "A longer item to show column expansion"
    _DebugArrayDisplay($aArray_2D, "Expanded column - custom titles - no 'Row' column", Default, $ARRAYDISPLAY_NOROW, Default, "AA|BB|CC|DD|EE|FF|GG|HH|II|JJ")

    ; Присваиваем пользовательскую функцию переменной для передачи в качестве параметра.
    Local $hUserFunction = _UserFunc

    $aArray_2D[5][5] = "Column alignment set to right"
    _DebugArrayDisplay($aArray_2D, "Range set - right align - copy column width - user function", "3:7|4:9", $ARRAYDISPLAY_COLALIGNRIGHT, 15, "AA|BB|CC|DD|EE|FF", Default, $hUserFunction)
    _DebugArrayDisplay($aArray_2D, "Range set - transposed", "3:7|4:9", $ARRAYDISPLAY_TRANSPOSE, Default, "AA|BB|CC|DD|EE|FF") ; Имена столбцов не указываются, так как массив транспонирован.

    $aArray_2D[5][5] = "Column alignment set to left"
    Opt("GUIDataSeparatorChar", "!")
     _DebugArrayDisplay($aArray_2D, "Header separator = !", "3:7|4:9", Default, Default, "AA!BB!CC!DD!EE!FF")

    Local $iRet, $iError = 2
    SetError($iError)
    $iRet = _DebugArrayDisplay($aArray_2D, "Check error no display", "3:7|4:9", $ARRAYDISPLAY_CHECKERROR)
    MsgBox(0, "_DebugArrayDisplay() Error", "return without internal Msgbox $iret =" & $iRet & " @error=" & $iError)

    ; Создать переменную, не являющуюся массивом, чтобы вызвать ошибку – MsgBox показывается, так как задан $iFlags
    Local $vVar = 0
    $iRet = _DebugArrayDisplay($vVar, "No MsgBox on Error")
    $iError = @error
    MsgBox(0, "_DebugArrayDisplay() Error", "return without internal Msgbox $iret =" & $iRet & " @error=" & $iError)

    $iRet = _DebugArrayDisplay($vVar, "MsgBox on Error", Default, $ARRAYDISPLAY_VERBOSE)
    $iError = @error
    MsgBox(0, "_DebugArrayDisplay() Error", "return internal Msgbox with no force Exit $iret =" & $iRet & " @error=" & $iError)

EndFunc    ;==>Example

; Пользовательская функция ДОЛЖНА иметь ДВА параметра, даже если оба не используются.
Func _UserFunc($aArray_2D, $aSelected)

    ; Если параметр не используется – добавьте заглушку, чтобы избежать предупреждения Au3Check.
    #forceref $aArray_2D

    _DebugArrayDisplay($aSelected, "Selected cols")

EndFunc    ;==>_UserFunc
