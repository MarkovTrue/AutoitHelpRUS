_WinAPI_ClipCursor

Ограничивает курсор прямоугольной областью на экране.

#include <WinAPIRes.au3>
_WinAPI_ClipCursor ( $tRECT )

Параметры

$tRECT Структура $tagRECT, которая содержит координаты экрана ограничивающего прямоугольника.
Если этот параметр 0, курсор может свободно перемещаться по всему экрану.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Курсор – это общий ресурс.
Если приложение ограничивает курсор, оно должно освободить курсор, используя _WinAPI_ClipCursor(), перед передачей управления другому приложению.

Дополнительно

Искать ClipCursor в библиотеке MSDN

Пример

#include <WinAPIMisc.au3>
#include <WinAPIRes.au3>

_Example()

Func _Example()
    _WinAPI_ClipCursor(_WinAPI_CreateRectEx(0, 0, 400, 400))
    Sleep(5000)
    _WinAPI_ClipCursor(0)
EndFunc   ;==>_Example
