Справочник функций

_ArrayColDelete

Удаляет указанный столбец из двумерного 2D массива.

#include <Array.au3>
_ArrayColDelete ( ByRef $aArray, $iColumn [, $bConvert = False] )

Параметры

$aArray Изменяемый массив.
$iColumn Удаляемый столбец.
$bConvert [необязательный] Если True и остался один столбец, массив преобразуется в одномерный 1D.

Возвращаемое значение

Успех: Возвращает количество оставшихся столбцов.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является двумерным 2D массивом
3 – $iColumn выходит за границы массива

Примечания

Функция НЕ обновляет счётный элемент массива, однако при успехе возвращает итоговое количество столбцов.

См. также

_ArrayColInsert

Пример

#include <Array.au3>

Local $aArray[4][3]
For $i = 0 To 3
    For $j = 0 To 2
        $aArray[$i][$j] = $i & $j
    Next
Next
_ArrayDisplay($aArray, "Original")

_ArrayColDelete($aArray, 2)
_ArrayDisplay($aArray, "Col 2 Deleted")

; Копируем массив с двумя столбцами
Local $aArray_2Col = $aArray

; По умолчанию оставшийся столбец остаётся 2D массивом
_ArrayColDelete($aArray, 0)
_ArrayDisplay($aArray, "Col 0 del - 2D array")

; Преобразовать 1D массив из одного столбца
_ArrayColDelete($aArray_2Col, 0, True)
_ArrayDisplay($aArray_2Col, "Col 0 del - 1D array")
