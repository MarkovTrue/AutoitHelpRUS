Справочник функций

_WinAPI_LoadKeyboardLayout

Загружает новый идентификатор входной локали в систему.

#include <WinAPISys.au3>
_WinAPI_LoadKeyboardLayout ( $iLanguage [, $iFlag = 0] )

Параметры

$iLanguage Идентификатор входной локали для загрузки.
$iFlag [необязательный] Флаг, определяющий способ загрузки идентификатора входной локали. Этот параметр может быть
одним из следующих значений.
$KLF_ACTIVATE
$KLF_NOTELLSHELL
$KLF_REORDER
$KLF_REPLACELANG
$KLF_SUBSTITUTE_OK
$KLF_SETFORPROCESS

Возвращаемое значение

Успех: Возвращает идентификатор входной локали, соответствующий запрошенному имени.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
