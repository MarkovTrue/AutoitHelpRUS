Справочник функций

_WinAPI_Create32BitHBITMAP

создаёт 32-битное растровое изображение из указанного значка.

#include <WinAPIGdi.au3>
_WinAPI_Create32BitHBITMAP ( $hIcon [, $bDib = False [, $bDelete = False]] )

Параметры

$hIcon Дескриптор исходного значка.
$bDib [необязательный] Задаёт, создать ли независимый от устройства (DIB) или зависимый от устройства (DDB) растр, допустимые значения:
True – создаёт DIB.
False – создаёт DDB (по умолчанию).
$bDelete [необязательный] Задаёт, удалить ли значок после успешного выполнения функции, допустимые значения:
True – значок будет удалён в случае успеха функции.
False – не удалять, вы должны освободить значок при завершении его использования (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор созданного растра.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Функция _WinAPI_Create32BitHBITMAP() создаёт растр 32 бит/пикс с альфа-каналом (если возможно) из значков с любой глубиной цвета.
Функция внутренне преобразует указанный значок в значок 32 бит/пикс.

По завершении работы с растром удалите его, используя функцию _WinAPI_DeleteObject().

См. также

_WinAPI_DeleteObject

Пример

#NoTrayIcon

#include <GUIMenu.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIShellEx.au3>
#include <WinAPISys.au3>

Opt('TrayMenuMode', 3)

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

Local $hMenu = TrayItemGetHandle(0)
Local $iOptions = TrayCreateItem('Options')
TrayCreateItem('')
Local $idExit = TrayCreateItem('Exit')

_GUICtrlMenu_SetItemBmp($hMenu, 0, _WinAPI_Create32BitHBITMAP(_WinAPI_ShellExtractIcon(@SystemDir & '\shell32.dll', 207, 16, 16), 1, 1))
_GUICtrlMenu_SetItemBmp($hMenu, 2, _WinAPI_Create32BitHBITMAP(_WinAPI_ShellExtractIcon(@SystemDir & '\shell32.dll', 131, 16, 16), 1, 1))

TraySetState()

Do
Until TrayGetMsg() = $idExit
