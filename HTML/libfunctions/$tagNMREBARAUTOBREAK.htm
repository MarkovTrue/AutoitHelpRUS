Справочник ключевых слов

$tagNMREBARAUTOBREAK

Содержит сведения для уведомления $RBN_AUTOBREAK.

Global Const $tagNMREBARAUTOBREAK = $tagNMHDR & ";uint uBand;uint wID;int lParam;uint uMsg;uint fStyleCurrent;int fAutoBreak"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
uBand Индекс полосы, затронутой уведомлением (с отсчётом от нуля). Значение -1, если полоса не затронута.
wID ID полосы, определённый приложением.
lParam Значение, определённое приложением, из члена lParam структуры $tagREBARBANDINFO, определяющей полосу rebar.
uMsg ID сообщения.
fStyleCurrent Стиль указанной полосы.
fAutoBreak Указывает, должен ли произойти разрыв.
