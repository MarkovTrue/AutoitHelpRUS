_WinAPI_GetFileInformationByHandleEx

Получает информацию о файле для указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetFileInformationByHandleEx ( $hFile )

Параметры

$hFile Дескриптор файла, содержащего информацию, которую требуется получить.

Возвращаемое значение

Успех: Возвращает структура $tagFILEINFO, содержащая информацию о файле.
Ошибка: Устанавливает @error ≠ 0.
