Справочник функций

_WinAPI_GetGUIThreadInfo

Получает информацию об активном окне или указанном потоке GUI.

#include <WinAPISysWin.au3>
_WinAPI_GetGUIThreadInfo ( $iThreadId )

Параметры

$iThreadId Идентификатор потока, для которого требуется получить информацию.

Возвращаемое значение

Успех: Возвращает массив, содержащий следующую информацию:
[0] - Состояние потока. Может быть одним или несколькими из значений $GUI_*.
[1] - Дескриптор активного окна в потоке.
[2] - Дескриптор окна, имеющего фокус клавиатуры.
[3] - Дескриптор окна, захватившего мышь.
[4] - Дескриптор окна, владеющего активными меню.
[5] - Дескриптор окна в цикле перемещения или изменения размера.
[6] - Дескриптор окна, отображающего курсор.
[7] - Координата x верхнего левого угла ограничивающего прямоугольника курсора в клиентских координатах.
[8] - Координата y верхнего левого угла ограничивающего прямоугольника курсора в клиентских координатах.
[9] - Ширина ограничивающего прямоугольника курсора.
[10] - Высота ограничивающего прямоугольника курсора.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Пример

#include <GUIConstantsEx.au3>
#include <WinAPISysWin.au3>
#include <WindowsStylesConstants.au3>

Local $hForm = GUICreate('', 240, 268, 10, 10, BitOR($WS_CAPTION, $WS_POPUP), $WS_EX_TOPMOST)
GUICtrlCreateLabel('Thread state:', 20, 18, 90, 14)
GUICtrlCreateLabel('Active window:', 20, 40, 90, 14)
GUICtrlCreateLabel('Keyboard focus:', 20, 62, 90, 14)
GUICtrlCreateLabel('Mouse capture:', 20, 84, 90, 14)
GUICtrlCreateLabel('Active menu:', 20, 106, 90, 14)
GUICtrlCreateLabel('Move or size loop:', 20, 128, 90, 14)
GUICtrlCreateLabel('Caret:', 20, 150, 90, 14)
GUICtrlCreateLabel('Left:', 20, 172, 90, 14)
GUICtrlCreateLabel('Top:', 20, 194, 90, 14)
GUICtrlCreateLabel('Width:', 20, 216, 90, 14)
GUICtrlCreateLabel('Height:', 20, 238, 90, 14)
Local $a_idLbl[11]
For $i = 0 To 10
    $a_idLbl[$i] = GUICtrlCreateLabel('', 114, 18 + 22 * $i, 116, 14)
Next
GUISetState(@SW_SHOWNOACTIVATE)

Local $hWnd, $aInfo, $iPID
Do
    $hWnd = WinGetHandle('[ACTIVE]')
    $aInfo = _WinAPI_GetGUIThreadInfo(_WinAPI_GetWindowThreadProcessId($hWnd, $iPID))
    If Not @error Then
        WinSetTitle($hForm, '', WinGetTitle($hWnd))
        $aInfo[0] = '0x' & Hex($aInfo[0], 8)
    Else
        WinSetTitle($hForm, '', '')
        Dim $aInfo[11]
        For $i = 0 To 10
            $aInfo[$i] = ''
        Next
        $hWnd = 0
    EndIf
    For $i = 0 To 10
        If StringCompare(GUICtrlRead($a_idLbl[$i]), $aInfo[$i]) Then
            GUICtrlSetData($a_idLbl[$i], $aInfo[$i])
        EndIf
    Next
Until GUIGetMsg() = $GUI_EVENT_CLOSE
