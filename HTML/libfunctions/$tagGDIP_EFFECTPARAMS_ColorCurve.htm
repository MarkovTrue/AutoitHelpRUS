Справочник ключевых слов

$tagGDIP_EFFECTPARAMS_ColorCurve

Структура параметров эффекта цветовой кривой.

Global Const $tagGDIP_EFFECTPARAMS_ColorCurve = "int Adjustment; int Channel; int AdjustValue"

Параметры

Adjustment Константа CurveAdjustments, определяющая применяемую регулировку ($GDIP_Adjust*).
Channel Константа CurveChannel, определяющая цветовой канал, к которому применяется регулировка ($GDIP_CurveChannel*).
AdjustValue Целое число, определяющее интенсивность регулировки.
Диапазон допустимых значений зависит от применяемой регулировки (см. константы $GDIP_Adjust*).
