Справочник функций

_WinAPI_LoadIconWithScaleDown

Загружает значок и уменьшает размер большого изображения вместо увеличения маленького.

#include <WinAPIIcons.au3>
_WinAPI_LoadIconWithScaleDown ( $hInstance, $sName, $iWidth, $iHeight )

Параметры

$hInstance Дескриптор модуля DLL или исполняемого файла (.exe), содержащего загружаемый значок.
$sName Информация о загружаемом значке. Если $hInstance не равен 0, то $sName задаёт ресурс значка
по имени или порядковому номеру, в противном случае $sName задаёт либо имя отдельного файла значка (.ico),
либо идентификатор предопределённого значка для загрузки.
$IDI_APPLICATION
$IDI_HAND
$IDI_QUESTION
$IDI_EXCLAMATION
$IDI_ASTERISK
$IDI_WINLOGO
$IDI_SHIELD
$IDI_ERROR
$IDI_INFORMATION
$IDI_WARNING
$iWidth Требуемая ширина значка в пиксельях.
$iHeight Требуемая высота значка в пиксельях.

Возвращаемое значение

Успех: Возвращает дескриптор значка.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Когда закончите использовать значок, удалите его с помощью функции _WinAPI_DestroyIcon().

Эта функция требует Windows Vista или более позднюю версию.

См. также

_WinAPI_DestroyIcon

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIIcons.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 324, 324)
GUICtrlCreateIcon('', 0, 64, 64, 196, 196)
Local $h_Icon = GUICtrlGetHandle(-1)
GUISetState(@SW_SHOW)

Local $sIco = _Extras_PathFull('Soccer.ico')
Local $hIcon = _WinAPI_LoadIconWithScaleDown(0, $sIco, 196, 196)
Local $hPrev = _SendMessage($h_Icon, $STM_SETIMAGE, 1, $hIcon)
If $hPrev Then
    _WinAPI_DestroyIcon($hPrev)
EndIf

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
