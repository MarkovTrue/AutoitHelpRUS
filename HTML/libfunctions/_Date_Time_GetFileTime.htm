Справочник функций

_Date_Time_GetFileTime

Получает дату и время создания, последнего доступа и изменения файла.

#include <Date.au3>
_Date_Time_GetFileTime ( $hFile )

Параметры

$hFile Дескриптор файла, для которого запрашиваются даты и время. Дескриптор должен быть открыт функцией CreateFile с правом доступа GENERIC_READ.

Возвращаемое значение

Возвращает массив следующего вида:
    $aArray[0] – структура $tagFILETIME с датой и временем создания файла
    $aArray[1] – структура $tagFILETIME с датой и временем последнего доступа
    $aArray[2] – структура $tagFILETIME с датой и временем последнего изменения

Примечания

Не все файловые системы поддерживают запись времени создания и последнего доступа, и не все делают это одинаково.
Например, в FAT время создания имеет точность 10 миллисекунд, время записи – 2 секунды, время доступа – 1 день (фактически только дата доступа). Поэтому _Date_Time_GetFileTime() может возвращать время, отличающееся от установленного через _Date_Time_SetFileTime().
NTFS откладывает обновление времени последнего доступа к файлу до одного часа после фактического доступа.

См. также

$tagFILETIME, _Date_Time_SetFileTime

Пример

#include "Extras\HelpFileInternals.au3"

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIError.au3>
#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hFile, $tFile, $aTime
    Local $sTempFile = @TempDir & "\Test.xyz"

    ; Создать GUI
    GUICreate("Time", 400, 300)
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Создать тестовый файл и установить времена
    $hFile = _WinAPI_CreateFile($sTempFile, 1)
    If $hFile = 0 Then _WinAPI_ShowError("Unable to create file")
    $tFile = _Date_Time_EncodeFileTime(@MON, @MDAY, @YEAR, @HOUR, @MIN, @SEC)
    _Date_Time_SetFileTime($hFile, $tFile, $tFile, $tFile)
    _WinAPI_CloseHandle($hFile)

    ; Прочитать времена файла
    $hFile = _WinAPI_CreateFile($sTempFile, 2)
    If $hFile = 0 Then _WinAPI_ShowError("Unable to open file")
    $aTime = _Date_Time_GetFileTime($hFile)
    _WinAPI_CloseHandle($hFile)

    _MemoWrite("Created ..: " & _Date_Time_FileTimeToStr($aTime[0]))
    _MemoWrite("Accessed .: " & _Date_Time_FileTimeToStr($aTime[1]))
    _MemoWrite("Modified .: " & _Date_Time_FileTimeToStr($aTime[2]))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    FileDelete($sTempFile)
EndFunc   ;==>Example
