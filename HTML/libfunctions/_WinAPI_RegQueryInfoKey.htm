_WinAPI_RegQueryInfoKey

Получает информацию об указанном ключе реестра.

#include <WinAPIReg.au3>
_WinAPI_RegQueryInfoKey ( $hKey )

Параметры

$hKey Дескриптор открытого раздела реестра. Раздел должен быть открыт с правом доступа $KEY_QUERY_VALUE.
Дескриптор возвращается функциями _WinAPI_RegCreateKey() и _WinAPI_RegOpenKey().
Также допустимы предопределённые ключи:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_PERFORMANCE_DATA
$HKEY_USERS

Возвращаемое значение

Успех: Возвращает массив, содержащий следующую информацию:
[0] – количество подключей, содержащихся в указанном ключе.
[1] – размер подключа ключа с самым длинным именем в символах, без учёта завершающего нулевого символа.
[2] – количество значений, связанных с ключом.
[3] – размер самого длинного имени значения ключа в символах. Размер не включает завершающий нулевой символ.
[4] – размер самого длинного компонента данных среди значений ключа в байтах.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

См. также

_WinAPI_RegCreateKey, _WinAPI_RegOpenKey

Дополнительно

Искать RegQueryInfoKey в библиотеке MSDN

Пример

#include <Date.au3>
#include <Debug.au3>
#include <WinAPIError.au3>
#include <WinAPIReg.au3>

Example()

Func Example()
    Local $hKey = _WinAPI_RegOpenKey($HKEY_CLASSES_ROOT, 'CLSID', $KEY_READ)
    If @error Then
        _DebugSetup(Default, True)
        _DebugReport("! RegOpenKey @error =" & @error & @CRLF & @TAB & _WinAPI_GetErrorMessage(@extended))
        Exit
    EndIf
    Local $iCount = _WinAPI_RegQueryInfoKey($hKey)
    Local $aKey[$iCount[0]][2], $tLocalFILETIME, $tFileWriteTime
    For $i = 0 To UBound($aKey) - 1
        $aKey[$i][0] = _WinAPI_RegEnumKey($hKey, $i)
        $tFileWriteTime = @extended
        If $tFileWriteTime Then
            $tLocalFILETIME = _Date_Time_FileTimeToLocalFileTime($tFileWriteTime)
            $aKey[$i][1] = _Date_Time_FileTimeToStr($tLocalFILETIME, 1)
        EndIf
    Next

    _WinAPI_RegCloseKey($hKey)

    _DebugArrayDisplay($aKey, '_WinAPI_RegEnumKey')

EndFunc   ;==>Example
