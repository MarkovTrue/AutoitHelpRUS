Справочник функций

_WinAPI_CreateNumberFormatInfo

создаёт структуру $tagNUMBERFMT с указанной информацией о формате чисел.

#include <WinAPILocale.au3>
_WinAPI_CreateNumberFormatInfo ( $iNumDigits, $iLeadingZero, $iGrouping, $sDecimalSep, $sThousandSep, $iNegativeOrder )

Параметры

$iNumDigits Количество / Размер.
$iLeadingZero Значение.
$iGrouping Координаты.
$sDecimalSep Строка.
$sThousandSep Строка.
$iNegativeOrder Количество / Размер.

Возвращаемое значение

Возвращает структуру $tagNUMBERFMT с данными форматирования чисел.

Примечания

Обычно структура, возвращённая этой функцией, передаётся в _WinAPI_GetNumberFormat().

См. также

_WinAPI_GetNumberFormat

Дополнительно

Искать NUMBERFMT в библиотеке MSDN

Пример

#include <WinAPILocale.au3>

Local $sNumber = '123456789'

ConsoleWrite(_WinAPI_GetNumberFormat(0, $sNumber) & @CRLF)
ConsoleWrite(_WinAPI_GetNumberFormat(0, $sNumber, _WinAPI_CreateNumberFormatInfo(0, 1, 3, '', ',', 1)) & @CRLF)
