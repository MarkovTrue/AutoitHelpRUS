Справочник функций

_FileListToArray

Возвращает список файлов и/или папок в указанной папке с поддержкой масок.(Аналог Dir /B с поддержкой шаблонов.).

#include <File.au3>
_FileListToArray ( $sFilePath [, $sFilter = "*" [, $iFlag = $FLTA_FILESFOLDERS [, $bReturnPath = False]]] )

Параметры

$sFilePath Папка, для которой строится список файлов.
$sFilter [необязательный] Маска фильтрации, по умолчанию *. (Поддерживаются символы * и ? – см. раздел Примечания.)
$iFlag [необязательный] Определяет, что включить в результат: файлы, папки или и то и другое
$FLTA_FILESFOLDERS (0) = (по умолчанию) возвращать файлы и папки
$FLTA_FILES (1) = только файлы
$FLTA_FOLDERS (2) = только папки
Константы определены в FileConstants.au3.
$bReturnPath [необязательный] Если True, возвращается полный путь к файлу/папке; иначе – путь относительно $sFilePath. По умолчанию False.

Возвращаемое значение

Успех: Возвращает одномерный массив:
$aArray[0] = количество найденных файлов/папок
$aArray[1] = 1-й файл/папка
$aArray[2] = 2-й файл/папка
$aArray[3] = 3-й файл/папка
$aArray[n] = n-й файл/папка
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – папка не найдена или путь недопустим
2 – недопустимый параметр $sFilter
3 – недопустимый параметр $iFlag
4 – файлы не найдены

Примечания

Функция использует средства Windows для поиска подпапок по заданной маске. Для простых масок это работает корректно, однако сложные маски с несколькими символами подстановки могут давать непредвиденные результаты (например, *test*.* совпадёт и с testX, и с testX.X). В таком случае лучше воспользоваться функцией _FileListToArrayRec(), которая сравнивает подпапки с фильтром через RegEx – хотя и работает заметно медленнее.

Подробнее о символах подстановки см. FileFindFirstFile().

Пример

Пример 1

#include <Array.au3>
#include <File.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получить список всех файлов и папок на рабочем столе с параметрами по умолчанию.
    Local $aFileList = _FileListToArray(@DesktopDir, "*")
    If @error = 1 Then
        MsgBox($MB_SYSTEMMODAL, "", "Path was invalid.")
        Exit
    EndIf
    If @error = 4 Then
        MsgBox($MB_SYSTEMMODAL, "", "No file(s) were found.")
        Exit
    EndIf
    ; Показать результаты, возвращённые _FileListToArray.
    _ArrayDisplay($aFileList, "$aFileList")
EndFunc   ;==>Example

Пример 2

#include <Array.au3>
#include <File.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получить список всех файлов и папок на рабочем столе с полными путями.
    Local $aFileList = _FileListToArray(@DesktopDir, Default, Default, True)
    If @error = 1 Then
        MsgBox($MB_SYSTEMMODAL, "", "Path was invalid.")
        Exit
    EndIf
    If @error = 4 Then
        MsgBox($MB_SYSTEMMODAL, "", "No file(s) were found.")
        Exit
    EndIf
    ; Показать результаты, возвращённые _FileListToArray.
    _ArrayDisplay($aFileList, "$aFileList")
EndFunc   ;==>Example
