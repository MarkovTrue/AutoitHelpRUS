_GUICtrlListBox_FindInText

Ищет элемент, содержащий указанный текст в любом месте его текста.

#include <GuiListBox.au3>
_GUICtrlListBox_FindInText ( $hWnd, $sText [, $iStart = -1 [, $bWrapOK = True]] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$sText Текст для сопоставления.
$iStart [необязательный] Индекс элемента, с которого начинается поиск (нумерация с 0), либо -1 – с начала.
Сам указанный элемент в поиске не участвует.
$bWrapOK [необязательный] Если True, при отсутствии совпадения поиск продолжается с первого элемента

Возвращаемое значение

Успех: Возвращает индекс элемента (нумерация с 0).
Ошибка: Возвращает -1.

Примечания

Поиск нечувствителен к регистру.

См. также

_GUICtrlListBox_FindString, _GUICtrlListBox_SelectString

Пример

#include <GUIConstantsEx.au3>
#include <GuiListBox.au3>

Example()

Func Example()
    Local $iIndex, $idListBox

    ; Создать GUI
    GUICreate("List Box Find In Text", 400, 296)
    $idListBox = GUICtrlCreateList("", 2, 2, 396, 296)

    GUISetState(@SW_SHOW)

    ; Добавить строки
    _GUICtrlListBox_BeginUpdate($idListBox)
    For $iI = 1 To 9
        _GUICtrlListBox_AddString($idListBox, StringFormat("%03d : Random string", Random(1, 100, 1)))
    Next
    _GUICtrlListBox_InsertString($idListBox, "eXaCt tExT", 3)
    _GUICtrlListBox_EndUpdate($idListBox)

    ; Найти элемент
    $iIndex = _GUICtrlListBox_FindInText($idListBox, "exa")
    _GUICtrlListBox_SetCurSel($idListBox, $iIndex)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
