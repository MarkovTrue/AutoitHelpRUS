Справочник ключевых слов

$tagGDIPENCODERPARAM

Структура параметра кодировщика GDI+.

Global Const $tagGDIPENCODERPARAM = "struct;byte GUID[16];ulong NumberOfValues;ulong Type;ptr Values;endstruct"

Параметры

GUID Определяет категорию параметра (константы GDI_EPG).
NumberOfValues Количество значений в массиве, на который указывает член Value.
Type Определяет тип данных параметров (константы GDI_EPT).
Values Указатель на массив значений. Каждое значение имеет тип, указанный членом Type.
