_WinAPI_GetHandleInformation

Получает определённые свойства дескриптора объекта.

#include <WinAPISys.au3>
_WinAPI_GetHandleInformation ( $hObject )

Параметры

$hObject Дескриптор объекта, информацию о котором требуется получить.

Возвращаемое значение

Успех: Возвращает значение, определяющее свойства дескриптора объекта ($HANDLE_FLAG_*).
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
