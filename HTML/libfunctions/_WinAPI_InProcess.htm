_WinAPI_InProcess

Определяет, принадлежит ли окно текущему процессу.

#include <WinAPISysWin.au3>
_WinAPI_InProcess ( $hWnd, ByRef $hLastWnd )

Параметры

$hWnd Дескриптор окна для проверки.
$hLastWnd Последнее проверенное окно. Если $hWnd = $hLastWnd, функция сразу вернёт значение True. В противном случае
будет вызвана _WinAPI_InProcess(). Если $hWnd принадлежит процессу, $hLastWnd будет установлена в $hWnd при возврате.

Возвращаемое значение

True: Возвращает дескриптор окна принадлежит текущему процессу.
False: Дескриптор окна не принадлежит текущему процессу.

Примечания

Это одна из ключевых функций для технологии отображения памяти элемента управления. Она проверяет идентификатор процесса
окна, чтобы определить, принадлежит ли оно текущему процессу, что означает, что к нему можно получить доступ без отображения памяти элемента управления.
