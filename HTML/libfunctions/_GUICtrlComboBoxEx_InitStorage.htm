_GUICtrlComboBoxEx_InitStorage

Выделяет память для хранения элементов ListBox.

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_InitStorage ( $hWnd, $iNum, $iBytes )

Параметры

$hWnd Дескриптор элемента управления.
$iNum Количество элементов для добавления.
$iBytes Размер памяти в байтах.

Возвращаемое значение

Успех: Возвращает общее число элементов, под которые предварительно выделена память.
Ошибка: $CB_ERRSPACE.

Примечания

Помогает ускорить инициализацию ComboBox с большим количеством элементов (более 100).

Значения параметров $iNum и $iBytes можно задавать приблизительно.
Если указать с запасом, выделится дополнительная память; если меньше, чем нужно, обычное выделение памяти отработает для тех элементов, под которые предварительно выделенной памяти не хватило.

См. также

_GUICtrlComboBoxEx_AddDir, _GUICtrlComboBoxEx_AddString, _GUICtrlComboBoxEx_InsertString

Пример

#include <GuiComboBoxEx.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI, $hCombo

    ; Создать GUI
    $hGUI = GUICreate("ComboBoxEx Init Storage", 400, 300)
    $hCombo = _GUICtrlComboBoxEx_Create($hGUI, "", 2, 2, 396, 296, $CBS_SIMPLE)
    GUISetState(@SW_SHOW)

    MsgBox($MB_SYSTEMMODAL, "Information", "Init Storage Pre-Allocated Memory For: " & _GUICtrlComboBoxEx_InitStorage($hCombo, 150, 300) & " Items")
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)

    For $x = 0 To 149
        _GUICtrlComboBoxEx_AddString($hCombo, StringFormat("%03d : Random string", Random(1, 100, 1)))
    Next
    _GUICtrlComboBoxEx_EndUpdate($hCombo)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
