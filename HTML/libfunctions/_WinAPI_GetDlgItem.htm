_WinAPI_GetDlgItem

Получает дескриптор указанного элемента управления в указанном диалоговом окне.

#include <WinAPISysWin.au3>
_WinAPI_GetDlgItem ( $hWnd, $iItemID )

Параметры

$hWnd Дескриптор диалогового окна.
$iItemID Идентификатор элемента управления, который нужно получить.

Возвращаемое значение

Успех: Возвращает дескриптор окна указанного элемента управления.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

GetDlgItem можно применять к любой паре "родительское-дочернее окно", не только к диалогам.
Пока параметр $hWnd задаёт родительское окно, а у дочернего окна есть уникальный идентификатор, GetDlgItem возвращает корректный дескриптор дочернего окна.

Дополнительно

Искать GetDlgItem в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPISysWin.au3>

Example()

Func Example()
    Local $hWnd, $idButton
    $hWnd = GUICreate("test")
    $idButton = GUICtrlCreateButton("button", 0, 0)
    MsgBox($MB_SYSTEMMODAL, "Handle", "Get Dialog Item: " & _WinAPI_GetDlgItem($hWnd, $idButton))
EndFunc   ;==>Example
