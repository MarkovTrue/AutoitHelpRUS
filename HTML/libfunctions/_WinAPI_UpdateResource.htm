Справочник функций

_WinAPI_UpdateResource

Добавляет, удаляет или заменяет ресурс в переносимом исполняемом файле (PE).

#include <WinAPIRes.au3>
_WinAPI_UpdateResource ( $hUpdate, $sType, $sName, $iLanguage, $pData, $iSize )

Параметры

$hUpdate Дескриптор модуля, возвращённый функцией _WinAPI_BeginUpdateResource(), ссылающейся на файл для обновления.
$sType Тип ресурса для обновления. Кроме того, вместо указателя этот параметр может быть
значением целого числа, представляющим предопределённый тип ресурса. Если первый символ строки
является знаком фунта (#), то оставшиеся символы представляют десятичное число, которое указывает идентификатор целого числа
типа ресурса. Например, строка "#258" представляет идентификатор 258.
Также вы можете использовать предопределённые типы ресурсов ($RT_*).
$sName Имя ресурса для обновления. Этот параметр может быть строком или целочисленным значением.
$iLanguage Идентификатор языка ресурса.
$pData Данные ресурса для вставки в файл, указанный параметром $hUpdate. Если ресурс –
one из предопределённых типов, данные должны быть действительными и правильно выровнены. Обратите внимание, что это сырой
dвоичные данные, а не данные, предоставляемые _WinAPI_LoadIcon(), _WinAPI_LoadString() или другими функциями загрузки, связанными с ресурсами.
Все данные, содержащие строки или текст, должны быть в формате Unicode. Если $pData
равен 0 и $iSize равен 0, указанный ресурс удаляется из файла, обозначенного $hUpdate.
$iSize Размер в байтах данных ресурса в $pData.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Рекомендуется, чтобы файл ресурса не загружался перед вызовом этой функции. Однако, если этот файл
уже загружен, это не вызовет ошибку.
Приложение может использовать _WinAPI_UpdateResource() многократно для внесения изменений в данные ресурса. Каждый вызов
_WinAPI_UpdateResource() вносит в список внутренних добавлений, удалений и замен, но
не записывает данные в файл. Приложение должно использовать функцию _WinAPI_EndUpdateResource() для
записи накопленных изменений.

Если $pData равен 0 и $iSize равен 0, указанный ресурс удаляется из файла, обозначенного $hUpdate.

См. также

_WinAPI_BeginUpdateResource, _WinAPI_EndUpdateResource, _WinAPI_LoadIcon, _WinAPI_LoadString

Дополнительно

Искать UpdateResource в библиотеке MSDN

Пример

#include "Extras\HelpFileInternals.au3"

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>
#include <WinAPIRes.au3>

Global Const $g_sExe = @TempDir & '\MyProg.exe'

Global Const $tagICONRESDIR = 'byte Width;byte Height;byte ColorCount;byte Reserved;ushort Planes;ushort BitCount;dword BytesInRes;ushort IconId;'
Global Const $tagNEWHEADER = 'ushort Reserved;ushort ResType;ushort ResCount;' ; & $tagICONRESDIR[ResCount]

; Выберите значок для обновления ресурса
Local $sIcon = FileOpenDialog('Select File', @ScriptDir & '\Extras', 'Icon Files (*.ico)', BitOR($FD_FILEMUSTEXIST, $FD_PATHMUSTEXIST), 'Script.ico')
If Not $sIcon Then
    Exit
EndIf

; Создать простой исполняемый файл (MyProg.exe), в который будет добавлена иконка
Local $sExe = _Extras_PathFull('MyProg.exe')
If Not FileCopy($sExe, $g_sExe, $FC_OVERWRITE) Then
    If MsgBox(($MB_OKCANCEL + $MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to copy MyProg.exe or file already exist in the current directory.') <> 1 Then
        Exit
    EndIf
    FileCopy($sExe, $g_sExe, $FC_OVERWRITE)
EndIf

Local $iError = 1
Do
    ; Начать обновление ресурсов
    Local $hUpdate = _WinAPI_BeginUpdateResource($g_sExe)
    If @error Then
        ExitLoop
    EndIf
    ; Прочитать .ico файл как сырые бинарные данные в структуру
    Local $tIcon = DllStructCreate('ushort Reserved;ushort Type;ushort Count;byte[' & (FileGetSize($sIcon) - 6) & ']')
    Local $hFile = _WinAPI_CreateFile($sIcon, 2, 2)
    If Not $hFile Then
        ExitLoop
    EndIf
    Local $iBytes = 0
    _WinAPI_ReadFile($hFile, $tIcon, DllStructGetSize($tIcon), $iBytes)
    _WinAPI_CloseHandle($hFile)
    If Not $iBytes Then
        ExitLoop
    EndIf
    ; Добавить все иконки из .ico в ресурсы RT_ICON с номерами 400, 401, ... и заполнить структуру group icons
    Local $iCount = DllStructGetData($tIcon, 'Count')
    Local $tDir = DllStructCreate($tagNEWHEADER & 'byte[' & (14 * $iCount) & ']')
    Local $pDir = DllStructGetPtr($tDir)
    DllStructSetData($tDir, 'Reserved', 0)
    DllStructSetData($tDir, 'ResType', 1)
    DllStructSetData($tDir, 'ResCount', $iCount)
    Local $tInfo, $iSize, $tData, $iID = 400
    Local $pIcon = DllStructGetPtr($tIcon)
    For $i = 1 To $iCount
        $tInfo = DllStructCreate('byte Width;byte Heigth;byte Colors;byte Reserved;ushort Planes;ushort BPP;dword Size;dword Offset', $pIcon + 6 + 16 * ($i - 1))
        $iSize = DllStructGetData($tInfo, 'Size')
        If Not _WinAPI_UpdateResource($hUpdate, $RT_ICON, $iID, 0, $pIcon + DllStructGetData($tInfo, 'Offset'), $iSize) Then
            ExitLoop 2
        EndIf
        $tData = DllStructCreate($tagICONRESDIR, $pDir + 6 + 14 * ($i - 1))
        DllStructSetData($tData, 'Width', DllStructGetData($tInfo, 'Width'))
        DllStructSetData($tData, 'Height', DllStructGetData($tInfo, 'Heigth'))
        DllStructSetData($tData, 'ColorCount', DllStructGetData($tInfo, 'Colors'))
        DllStructSetData($tData, 'Reserved', 0)
        DllStructSetData($tData, 'Planes', DllStructGetData($tInfo, 'Planes'))
        DllStructSetData($tData, 'BitCount', DllStructGetData($tInfo, 'BPP'))
        DllStructSetData($tData, 'BytesInRes', $iSize)
        DllStructSetData($tData, 'IconId', $iID)
        $iID += 1
    Next
    ; Добавить ресурс RT_GROUP_ICON с именем "MAINICON"
    If Not _WinAPI_UpdateResource($hUpdate, $RT_GROUP_ICON, 'MAINICON', 0, $pDir, DllStructGetSize($tDir)) Then
        ExitLoop
    EndIf
    $iError = 0
Until 1

; Сохранить или отменить изменения ресурсов исполняемого файла
If Not _WinAPI_EndUpdateResource($hUpdate, $iError) Then
    $iError = 1
EndIf

; Показать сообщение при возникновении ошибки
If $iError Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to change resources.')
EndIf
