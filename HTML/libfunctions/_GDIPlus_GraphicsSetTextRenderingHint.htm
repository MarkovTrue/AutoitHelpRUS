_GDIPlus_GraphicsSetTextRenderingHint

Устанавливает параметр отрисовки текста объекта Graphics.

#include <GDIPlus.au3>
_GDIPlus_GraphicsSetTextRenderingHint ( $hGraphics, $iTextRenderingHint )

Параметры

$hGraphics Указатель на объект Graphics.
$iTextRenderingHint Режим отрисовки текста:
$GDIP_TEXTRENDERINGHINTSYSTEMDEFAULT (0) – символ отрисовывается с использованием системного режима сглаживания шрифтов.
$GDIP_TEXTRENDERINGHINTSINGLEBITPERPIXELGRIDFIT (1) – символ отрисовывается по растровому изображению глифа с хинтингом для улучшения отображения штрихов и кривых.
$GDIP_TEXTRENDERINGHINTSINGLEBITPERPIXEL (2) – символ отрисовывается по растровому изображению глифа без хинтинга. Более высокая скорость за счёт качества.
$GDIP_TEXTRENDERINGHINTANTIALIASGRIDFIT (3) – символ отрисовывается по сглаженному растровому изображению глифа с хинтингом. Высокое качество за счёт более высокой стоимости.
$GDIP_TEXTRENDERINGHINTANTIALIAS (4) – символ отрисовывается по сглаженному растровому изображению глифа без хинтинга. Разница в ширине штрихов может быть заметна.
$GDIP_TEXTRENDERINGHINTCLEARTYPEGRIDFIT (5) – символ отрисовывается с применением ClearType и хинтинга. Этот режим нельзя использовать совместно с $GDIP_COMPOSITINGMODESOURCECOPY.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Дополнительно

Искать GdipSetTextRenderingHint в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI = GUICreate("GDI+ test", 800, 400)
    GUISetState(@SW_SHOW)

    If @OSBuild < 6000 Then MsgBox($MB_SYSTEMMODAL, "", "Antialiasing is automatically turned on for your operating system - no visible differences!")

    _GDIPlus_Startup()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsDrawString($hGraphics, "AutoIt rulez!", 0, 0, "Impact", 110)
    _GDIPlus_GraphicsSetTextRenderingHint($hGraphics, $GDIP_TEXTRENDERINGHINTANTIALIASGRIDFIT)
    _GDIPlus_GraphicsDrawString($hGraphics, "AutoIt rulez!", 0, 200, "Impact", 110)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    ;освободить ресурсы
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
