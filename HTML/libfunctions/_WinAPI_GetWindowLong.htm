Справочник функций

_WinAPI_GetWindowLong

Получает сведения об указанном окне.

#include <WinAPISysWin.au3>
_WinAPI_GetWindowLong ( $hWnd, $iIndex )

Параметры

$hWnd Дескриптор окна.
$iIndex Указывает смещение (начиная с нуля) значения, которое нужно получить.
Допустимые значения находятся в диапазоне от нуля до количества байтов дополнительной памяти окна минус четыре;
например, если вы указали 12 или более байтов дополнительной памяти, значение 8 будет индексом третьего 32-битного целого числа.
Для получения любого другого значения укажите одно из следующих значений:
$GWL_EXSTYLE – получает расширенные стили окна
$GWL_STYLE – получает стили окна
$GWL_WNDPROC – получает адрес процедуры окна
$GWL_HINSTANCE – получает дескриптор экземпляра приложения
$GWL_HWNDPARENT – получает дескриптор родительского окна, если оно есть
$GWL_ID – получает идентификатор окна
$GWL_USERDATA – получает 32-битное значение, связанное с окном.

Возвращаемое значение

Успех: Возвращает запрошенное значение.
Ошибка: Устанавливает @error ≠ 0, вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Приведённые выше константы требуют #include <Constants.au3>
