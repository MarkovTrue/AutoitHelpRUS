_WinAPI_ArrayToStruct

Преобразует массив строк в структуру.

#include <WinAPIMisc.au3>
_WinAPI_ArrayToStruct ( Const ByRef $aData [, $iStart = 0 [, $iEnd = -1]] )

Параметры

$aData Массив для преобразования.
$iStart [необязательный] Индекс массива, с которого начать преобразование.
$iEnd [необязательный] Индекс массива, где остановить преобразование.

Возвращаемое значение

Успех: Возвращает структура типа "string1;{0};string2;{0}; ... ;stringN;{0};{0}".
Ошибка: Устанавливает @error ≠ 0.

Примечания

Эта функция работает только с Unicode строками.
