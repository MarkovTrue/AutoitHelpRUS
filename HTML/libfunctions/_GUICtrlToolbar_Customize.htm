Справочник функций

_GUICtrlToolbar_Customize

Отображает диалоговое окно настройки панели инструментов.

#include <GuiToolbar.au3>
_GUICtrlToolbar_Customize ( $hWnd )

Параметры

$hWnd Дескриптор элемента управления.

Возвращаемое значение

Отсутствует.

Примечания

Чтобы появилось диалоговое окно настройки панели инструментов, панель должна обрабатывать уведомления $TBN_QUERYINSERT и $TBN_QUERYDELETE.
Если панель эти уведомления не обрабатывает, сообщение $TB_CUSTOMIZE не действует.
