Справочник ключевых слов

$tagBLENDFUNCTION

Управляет смешением путём задания функций смешения для исходного и целевого точечных рисунков.

Global Const $tagBLENDFUNCTION = "byte Op;byte Flags;byte Alpha;byte Format"

Параметры

Op Определяет операцию смешивания источника:.
Flags Должно быть нулём.
Alpha Определяет значение прозрачности альфа-канала, используемое для всего растрового изображения источника. Это значение комбинируется с любыми значениями альфа-канала для каждого пикселя в растровом изображении источника. Если установлено на 0, предполагается, что ваше изображение прозрачно. Установите 255 (непрозрачно), когда вы хотите использовать только значения альфа-канала для каждого пикселя.
Format Этот член управляет способом интерпретации исходного и целевого растровых изображений:
AC_SRC_ALPHA – этот флаг устанавливается, когда растровое изображение имеет альфа-канал (то есть альфа-канал для каждого пикселя). Обратите внимание, что API используют предварительно умноженный альфа-канал, что означает, что значения красного, зелёного и синего каналов в растровом изображении должны быть предварительно умножены на значение альфа-канала.

Примечания

Растровое изображение источника, используемое с $tagBLENDFUNCTION, должно быть 32 bpp
