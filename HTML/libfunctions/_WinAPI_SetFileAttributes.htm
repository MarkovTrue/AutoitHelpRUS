_WinAPI_SetFileAttributes

Устанавливает атрибуты для файла или каталога.

#include <WinAPIFiles.au3>
_WinAPI_SetFileAttributes ( $sFilePath, $iAttributes )

Параметры

$sFilePath Имя файла или каталога, атрибуты которого требуется установить.
$iAttributes Атрибуты файла для установки в файл. Этот параметр может быть одним или несколькими из следующих значений.
$FILE_ATTRIBUTE_READONLY
$FILE_ATTRIBUTE_HIDDEN
$FILE_ATTRIBUTE_SYSTEM
$FILE_ATTRIBUTE_ARCHIVE
$FILE_ATTRIBUTE_NORMAL
$FILE_ATTRIBUTE_TEMPORARY
$FILE_ATTRIBUTE_OFFLINE
$FILE_ATTRIBUTE_NOT_CONTENT_INDEXED

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Не все атрибуты поддерживаются этой функцией. Подробнее см. библиотеку MSDN.
