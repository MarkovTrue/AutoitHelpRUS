_WinAPI_ShowOwnedPopups

Показывает или скрывает все всплывающие окна, принадлежащие указанному окну.

#include <WinAPISysWin.au3>
_WinAPI_ShowOwnedPopups ( $hWnd, $bShow )

Параметры

$hWnd Дескриптор окна, владеющего всплывающими окнами, которые должны быть показаны или скрыты.
$bShow Определяет, должны ли всплывающие окна быть показаны или скрыты, допустимые значения:
True – все скрытые всплывающие окна показаны.
False – все видимые всплывающие окна скрыты.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция показывает только окна, скрытые предыдущим вызовом _WinAPI_ShowOwnedPopups().
