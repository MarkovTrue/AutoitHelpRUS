Справочник функций

_StringExplode

Разбивает строку на подстроки по указанному разделителю, как функция PHP Explode v5.

#include <String.au3>
_StringExplode ( $sString, $sDelimiter [, $iLimit = 0] )

Параметры

$sString Строка для разбиения.
$sDelimiter Разделитель для разбиения (разбиение выполняется по всей строке, а не по отдельным символам).
$iLimit [необязательный] Максимальное количество возвращаемых элементов
=0 : (по умолчанию) Разбивать по каждому экземпляру разделителя
>0 : Разбивать до достижения лимита, последний элемент будет содержать оставшуюся часть строки
<0 : Разбивать по каждому экземпляру, удаляя лимит элементов с конца массива

Возвращаемое значение

Возвращает массив содержащий разбитые строки.

Примечания

Используйте отрицательные значения лимита для удаления первых возможных элементов.

Пример

#include <Array.au3>
#include <String.au3>

Local $sText = "one##two##three##four##five##six##seven##eight"
Local $aArray1 = _StringExplode($sText, "##", 0)
_ArrayDisplay($aArray1, "StringExplode 0")

; ОТОБРАЖЕНИЕ
; [0] = один
; [1] = два
; [2] = три
; [3] = четыре
; [4] = пять
; [5] = шесть
; [6] = семь
; [7] = восемь

Local $aArray2 = _StringExplode($sText, "##", 4)
_ArrayDisplay($aArray2, "StringExplode 4")

; ОТОБРАЖЕНИЕ
; [0] = один
; [1] = два
; [2] = три
; [3] = четыре
; [4] = пять##шесть##семь##восемь

Local $aArray3 = _StringExplode($sText, "##", -3)
_ArrayDisplay($aArray3, "StringExplode -3")

; ОТОБРАЖЕНИЕ
; [0] = один
; [1] = два
; [2] = три
; [3] = четыре
; [4] = пять
