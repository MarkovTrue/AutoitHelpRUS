_WinAPI_CopyIcon

Копирует указанный значок из другого модуля.

#include <WinAPIIcons.au3>
_WinAPI_CopyIcon ( $hIcon )

Параметры

$hIcon Дескриптор значка, который должен быть скопирован.

Возвращаемое значение

Успех: Возвращает дескриптор дублированного значка.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция _WinAPI_CopyIcon() позволяет приложению или DLL получить собственный дескриптор значка, принадлежащего другому модулю.
Если другой модуль освобождается, приложение по-прежнему сможет использовать значок.
Перед закрытием вызовите функцию _WinAPI_DestroyIcon() для освобождения системных ресурсов, связанных со значком.
