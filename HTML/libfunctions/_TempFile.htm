Справочник функций

_TempFile

Генерирует имя для временного файла. Гарантировано, что такой файл не существует.

#include <File.au3>
_TempFile ( [$sDirectoryName = @TempDir [, $sFilePrefix = "~" [, $sFileExtension = ".tmp" [, $iRandomLength = 7]]]] )

Параметры

$sDirectoryName [необязательный] Имя каталога для файла, по умолчанию используется каталог %TEMP% пользователя
$sFilePrefix [необязательный] Префикс имени файла, по умолчанию "~"
$sFileExtension [необязательный] Расширение файла, по умолчанию ".tmp"
$iRandomLength [необязательный] Количество символов для создания уникального имени, по умолчанию 7

Возвращаемое значение

Возвращает имя файла для временного файла, который не существует

См. также

_WinAPI_GetTempFileName

Пример

#include <File.au3>
#include <MsgBoxConstants.au3>

; Генерировать уникальное имя файла в @TempDir
Local $sTempFile_1 = _TempFile()

; Генерировать уникальное имя файла в каталоге @HomeDrive, начинающееся с префикса prefix_
Local $sTempFile_2 = _TempFile(@HomeDrive & "\", "prefix_", ".txt", Default) ; Использовать ключевое слово Default для применения параметров по умолчанию

MsgBox($MB_SYSTEMMODAL, "", "Names suitable for temporary file usage: " & @CRLF & @CRLF & _
        "File 1: " & $sTempFile_1 & @CRLF & _
        "File 2: " & $sTempFile_2)
