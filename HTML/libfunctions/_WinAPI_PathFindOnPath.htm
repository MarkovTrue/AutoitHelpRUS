Справочник функций

_WinAPI_PathFindOnPath

Ищет файл в путях системы по умолчанию.

#include <WinAPIShPath.au3>
_WinAPI_PathFindOnPath ( $sFilePath [, $aExtraPaths = "" [, $sPathDelimiter = @LF]] )

Параметры

$sFilePath Имя файла для поиска.
$aExtraPaths [необязательный] Дополнительные пути для проверки перед остальными.
$sPathDelimiter [необязательный] Разделитель, используемый для разделения $aExtraPaths если это непустая строка (StringSplit() с флагом $STR_NOCOUNT (2)).

Возвращаемое значение

Успех: Возвращает полный путь найденного файла.
Ошибка: устанавливает флаг @error в ненулевое значение, имя файла без изменений.

Примечания

$aExtraPaths может содержать список путей для проверки перед системными значениями по умолчанию.
Это может быть массив или строка. Если первое, он не должен иметь счётчик в своём первом элементе.
Если последнее, оно будет разделено, используя $sPathDelimiter в качестве разделителя, который по умолчанию равен @LF.

Дополнительно

Искать PathFindOnPath в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIShPath.au3>

MsgBox($MB_SYSTEMMODAL, "PathFindOnPath Example", _
        StringFormat("Full path of notepad.exe:\n%s\n\n" & _
        "Find ntuser.dat in profile folder, using custom paths:\n%s", _
        _WinAPI_PathFindOnPath("notepad.exe"), _WinAPI_PathFindOnPath("ntuser.dat", @UserProfileDir) _
        ))
