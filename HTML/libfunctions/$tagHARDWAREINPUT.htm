Справочник ключевых слов

$tagHARDWAREINPUT

Содержит сведения о смоделированном сообщении, сгенерированном устройством ввода (кроме клавиатуры и мыши).

Global Const $tagHARDWAREINPUT = "dword type;struct;dword uMsg;word wParamL;word wParamH;endstruct"

Параметры

type $INPUT_HARDWARE – информация о моделируемом событии аппаратного обеспечения.
uMsg Сообщение, созданное аппаратным обеспечением ввода.
wParamL Слово младшего порядка параметра lParam для uMsg.
wParamH Слово старшего порядка параметра lParam для uMsg.
