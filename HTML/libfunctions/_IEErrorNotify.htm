Справочник функций

_IEErrorNotify

Управляет автоматическим выводом предупреждений и ошибок IE.au3 в консоль.

#include <IE.au3>
_IEErrorNotify ( [$vNotify = Default] )

Параметры

$vNotify [необязательный] задаёт, включать или отключать уведомления
-1 = (по умолчанию) вернуть текущую настройку
True = Включить
False = Отключить.

Возвращаемое значение

Возвращает текущую настройку уведомлений, если $vNotify = -1 (по умолчанию); иначе возвращает 1.

Примечания

По умолчанию IE.au3 пишет диагностические сведения, предупреждения и ошибки в консоль.
Эти сообщения удобно видеть, запуская скрипты из редактора SciTE.

Эта функция позволяет включать или выключать такое поведение.

Пример

Пример 1

; Проверить текущее состояние _IEErrorNotify, выключить если включено и наоборот

#include <IE.au3>
#include <MsgBoxConstants.au3>

If _IEErrorNotify() Then
    MsgBox($MB_SYSTEMMODAL, "_IEErrorNotify Status", "Notification is ON, turning it OFF")
    _IEErrorNotify(False)
Else
    MsgBox($MB_SYSTEMMODAL, "_IEErrorNotify Status", "Notification is OFF, turning it ON")
    _IEErrorNotify(True)
EndIf

Func _IEErrorNotify_ON_OFF()
    If _IEErrorNotify() Then
        MsgBox($MB_SYSTEMMODAL, "_IEErrorNotify Status", "Notification is ON, turning it OFF")
        ConsoleWrite('> Turning _IEErrorNotify OFF' & @CRLF)
        _IEErrorNotify(False)
    Else
        MsgBox($MB_SYSTEMMODAL, "_IEErrorNotify Status", "Notification is OFF, turning it ON")
        ConsoleWrite('> Turning _IEErrorNotify ON' & @CRLF)
        _IEErrorNotify(True)
    EndIf
EndFunc   ;==>_IEErrorNotify_ON_OFF
