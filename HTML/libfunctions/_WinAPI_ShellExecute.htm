Справочник функций

_WinAPI_ShellExecute

Выполняет операцию над указанным файлом.

#include <WinAPIShellEx.au3>
_WinAPI_ShellExecute ( $sFilePath [, $sArgs = '' [, $sDir = '' [, $sVerb = '' [, $iShow = 1 [, $hParent = 0]]]]] )

Параметры

$sFilePath Строка, которая определяет файл или объект, для которого нужно выполнить указанную операцию. Обратите внимание, что не все
операции поддерживаются для всех объектов. Например, не все типы документов поддерживают операцию "print".
$sArgs [необязательный] Строка, определяющая параметры, которые должны быть переданы приложению.
$sDir [необязательный] Строка, определяющая рабочий каталог для действия.
$sVerb [необязательный] Строка, называемая операцией, которая определяет действие, которое должно быть выполнено. Набор доступных операций
зависит от конкретного файла или папки. Как правило, действия, доступные из контекстного меню объекта
являются доступными операциями. Следующие операции часто используются:

"edit"
"explore"
"find"
"open"
"edit"
"print"
$iShow [необязательный] Флаги, определяющие, как приложение должно отображаться при открытии ($SW_*).
$hParent [необязательный] Дескриптор окна владельца, используемый для отображения пользовательского интерфейса или сообщений об ошибках.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: False, флаг @extended может содержать значение ошибки, указывающее на
причину сбоя. Это может быть одно из следующих значений (исключая (-1)).
Операционная система не хватает памяти или ресурсов (0)
ERROR_FILE_NOT_FOUND (2)
ERROR_PATH_NOT_FOUND (3)
ERROR_BAD_FORMAT (11)
SE_ERR_*

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>
#include <WinAPIShellEx.au3>

Local $sFile = InputBox('Run', 'Type the name of a program, folder, document, or Internet resource to open it', '', '', 368, 152)

If $sFile Then
    _WinAPI_ShellExecute($sFile, '', '', 'open')
    If @error Then
        Local $iError = @error
        Local $iExtended = @extended
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error = ' & $iError, 'Unable to open "' & $sFile & '".' & @CRLF & @CRLF & _WinAPI_GetErrorMessage($iExtended))
    EndIf
EndIf
