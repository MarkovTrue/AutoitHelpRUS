Справочник функций

_GUICtrlHeader_OrderToIndex

Получает значение индекса элемента на основе его порядка.

#include <GuiHeader.au3>
_GUICtrlHeader_OrderToIndex ( $hWnd, $iOrder )

Параметры

$hWnd Дескриптор элемента управления.
$iOrder Порядок, в котором элемент появляется в элементе управления Header, слева направо.

Возвращаемое значение

Успех: Возвращает индекс элемента.
Ошибка: $iOrder.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiHeader.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Header OrderToIndex (v" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $hHeader = _GUICtrlHeader_Create($hGUI)
    _MemoCreate(2, 52, 444, 220)

;~ _GUICtrlHeader_SetUnicodeFormat($hHeader, True)

    GUISetState(@SW_SHOW)

    ; Добавить столбцы
    _GUICtrlHeader_AddItem($hHeader, "Column 0", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 1", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 2", 100)
    _GUICtrlHeader_AddItem($hHeader, "Column 3", 100)

    ; Показать индекс для порядка 2
    _MemoWrite("Order 2 index: " & _GUICtrlHeader_OrderToIndex($hHeader, 2))

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
