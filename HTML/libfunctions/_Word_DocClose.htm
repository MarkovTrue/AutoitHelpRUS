Справочник функций

_Word_DocClose

Закрывает указанный документ Microsoft Word.

#include <Word.au3>
_Word_DocClose ( $oDoc [, $iSaveChanges = $WdDoNotSaveChanges [, $iOriginalFormat = $WdOriginalDocumentFormat]] )

Параметры

$oDoc Объект документа Word.
$iSaveChanges [необязательный] Указывает действие сохранения документа (по умолчанию = $WdDoNotSaveChanges). Может быть одной из констант WdSaveOptions:     $WdDoNotSaveChanges, $WdPromptToSaveChanges или $WdSaveChanges
$iOriginalFormat [необязательный] Указывает формат сохранения документа. Может быть одной из констант WdOriginalFormat:     $WdOriginalDocumentFormat, $WdPromptUser или $WdWordDocument (по умолчанию = $WdOriginalDocumentFormat)

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $oDoc не является объектом 2 – Ошибка, возвращённая при закрытии указанного документа. @extended задан код ошибки COM

См. также

_Word_DocAdd, _Word_DocAttach, _Word_DocOpen

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocClose Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ только для чтения
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocClose Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Закрыть документ Microsoft Word
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocClose Example", "Press 'OK' to close document '.\Extras\Test.doc'.")
_Word_DocClose($oDoc)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocClose Example", _
        "Error closing document '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocClose Example", "The document has been closed successfully.")
