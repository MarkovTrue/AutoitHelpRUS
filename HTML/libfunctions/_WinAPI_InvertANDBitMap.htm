Справочник функций

_WinAPI_InvertANDBitmap

Инвертирует указанное AND-растровое изображение маски путём выполнения логической операции НЕ.

#include <WinAPIGdi.au3>
_WinAPI_InvertANDBitmap ( $hBitmap [, $bDelete = False] )

Параметры

$hBitmap Дескриптор исходного растрового изображения, которое должно быть инвертировано.
$bDelete [необязательный] Определяет, следует ли удалять исходное растровое изображение после успешного завершения функции, допустимые значения:
True – растровое изображение будет удалено при успешном завершении функции.
False – не удалять, вы должны удалить растровое изображение, когда оно больше не требуется (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор инвертированного растрового изображения.
Ошибка: Возвращает 0.

Примечания

_WinAPI_InvertANDBitmap() создаёт инвертированное независимое от устройства изображение с 1 битом на пиксель из растрового изображения AND-маски с 1 битом на пиксель.
Если исходное растровое изображение не является растровым изображением с 1 битом на пиксель, функция не выполняется.

Когда вы закончите использование растрового изображения, удалите его с помощью функции _WinAPI_DeleteObject().

См. также

_WinAPI_DeleteObject

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIHObj.au3>
#include <WinAPIIcons.au3>
#include <WinAPIShellEx.au3>

; Извлекает значок и создаёт растровое изображение AND-маски
Local $sIco = _Extras_PathFull('Script.ico')
Local $hIcon = _WinAPI_ShellExtractIcon($sIco, 0, 128, 128)
Local $aInfo = _WinAPI_GetIconInfo($hIcon)
_WinAPI_DeleteObject($aInfo[5])
_WinAPI_DestroyIcon($hIcon)

; Создать инвертированный bitmask bitmap
$aInfo[5] = _WinAPI_InvertANDBitmap($aInfo[4])

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 256, 128)
Local $aidPic[2], $ahPic[2]
$aidPic[0] = GUICtrlCreatePic('', 0, 0, 128, 128)
$aidPic[1] = GUICtrlCreatePic('', 128, 0, 128, 128)
For $i = 0 To 1
    $ahPic[$i] = GUICtrlGetHandle($aidPic[$i])
Next

; Задать оба bitmap для элементов управления
For $i = 0 To 1
    _SendMessage($ahPic[$i], $STM_SETIMAGE, 0, $aInfo[$i + 4])
    Local $hObj = _SendMessage($ahPic[$i], $STM_GETIMAGE)
    If $hObj <> $aInfo[$i + 4] Then
        _WinAPI_DeleteObject($aInfo[$i + 4])
    EndIf
Next

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
