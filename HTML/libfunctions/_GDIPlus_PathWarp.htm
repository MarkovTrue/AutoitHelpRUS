Справочник функций

_GDIPlus_PathWarp

Применяет деформирующее преобразование к пути, одновременно сглаживая его (преобразуя кривые в последовательность прямых линий).

#include <GDIPlus.au3>
_GDIPlus_PathWarp ( $hPath, $hMatrix, $aPoints, $nX, $nY, $nWidth, $nHeight [, $iWarpMode = 0 [, $fFlatness = 0.25]] )

Параметры

$hPath Указатель на объект GraphicsPath.
$hMatrix Указатель на объект Matrix.
$aPoints Массив вершин параллелограмма, которые совместно с параметрами прямоугольника задают режим деформации:
[0][0] – количество точек; допустимые значения: 3 или 4
[1][0] – X-координата точки 1
[1][1] – Y-координата точки 1
[2][0] – X-координата точки 2
[2][1] – Y-координата точки 2
[n][0] – X-координата точки n
[n][1] – Y-координата точки n
$nX X-координата верхнего левого угла прямоугольника, преобразуемого в параллелограмм, заданный $aPoints.
$nY Y-координата верхнего левого угла прямоугольника, преобразуемого в параллелограмм, заданный $aPoints.
$nWidth Ширина прямоугольника, преобразуемого в параллелограмм, заданный $aPoints.
$nHeight Высота прямоугольника, преобразуемого в параллелограмм, заданный $aPoints.
$iWarpMode [необязательный] Вид применяемой деформации:
0 – перспективная деформация
1 – билинейная деформация
$fFlatness [необязательный] Вещественное число, влияющее на количество отрезков при аппроксимации исходного пути.
Малые значения дают больше отрезков, большие – меньше.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $iW, $iH, $hGUI, $hGraphic, $hBrush, $hPath, $hFormat, $hFamily, $tLayout

    ; Создать GUI
    $iW = 600
    $iH = 300
    $hGUI = GUICreate("GDI+", $iW, $iH)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;Создать графический объект из дескриптора окна
    _GDIPlus_GraphicsSetSmoothingMode($hGraphic, $GDIP_SMOOTHINGMODE_HIGHQUALITY) ;Задаёт качество отрисовки графики (сглаживание)
    _GDIPlus_GraphicsClear($hGraphic, 0xFF000000)

    $hBrush = _GDIPlus_BrushCreateSolid(0xFFFFFF00)

    $hPath = _GDIPlus_PathCreate() ;Создать новый объект path

    $hFormat = _GDIPlus_StringFormatCreate()
    _GDIPlus_StringFormatSetAlign($hFormat, 1) ;выравнивание по центру

    $hFamily = _GDIPlus_FontFamilyCreate("Arial Black") ;Создать объект font family
    $tLayout = _GDIPlus_RectFCreate(0, 0, $iW, $iH) ;создать ограничивающий прямоугольник строки
    _GDIPlus_PathAddString($hPath, "AutoIt rulez!" & @LF & "and so does" & @LF & "STAR WARS ;)", $tLayout, $hFamily, 0, 64, $hFormat) ;Добавить контур строки в path

    Local $aPoints[5][2]
    $aPoints[0][0] = 4
    $aPoints[1][0] = $iW * 0.3
    $aPoints[1][1] = $iH * 0.3
    $aPoints[2][0] = $iW * 0.7
    $aPoints[2][1] = $iH * 0.3
    $aPoints[3][0] = 0
    $aPoints[3][1] = $iH
    $aPoints[4][0] = $iW
    $aPoints[4][1] = $iH

    _GDIPlus_PathWarp($hPath, 0, $aPoints, 0, 0, $iW, $iH) ;деформировать путь

    _GDIPlus_GraphicsFillPath($hGraphic, $hPath, $hBrush) ;Нарисовать path в дескрипторе графики (GUI)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_FontFamilyDispose($hFamily)
    _GDIPlus_StringFormatDispose($hFormat)
    _GDIPlus_PathDispose($hPath)
    _GDIPlus_BrushDispose($hBrush)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
