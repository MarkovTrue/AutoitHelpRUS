_WinAPI_MulDiv

Умножает два 32-битных значения, а затем делит результат 64-битный на третье 32-битное значение.

#include <WinAPIMisc.au3>
_WinAPI_MulDiv ( $iNumber, $iNumerator, $iDenominator )

Параметры

$iNumber Задаёт множимое.
$iNumerator Задаёт множитель.
$iDenominator Задаёт число, на которое должен быть разделён результат умножения.

Возвращаемое значение

Успех: Возвращает результат умножения и деления.
Ошибка: (-1), если произошло переполнение или $iDenominator был 0

Примечания

Если результат это положительное полуцелое число (заканчивается на .5), оно округляется вверх.
Если результат это отрицательное полуцелое число, оно округляется вниз.
