_WinAPI_GetObject

Получает информацию об указанном графическом объекте.

#include <WinAPIHObj.au3>
_WinAPI_GetObject ( $hObject, $iSize, $pObject )

Параметры

$hObject Определяет логическое перо, кисть, шрифт, растр, область или палитру.
$iSize Задаёт число байтов для записи в буфер.
$pObject Указатель на буфер, который получает информацию.
Ниже показан тип информации, которую буфер получает для каждого типа графического объекта, который вы можете указать:
HBITMAP – BITMAP или DIBSECTION
HPALETTE – число записей в логической палитре
HPEN – EXTLOGPEN или LOGPEN
HBRUSH – LOGBRUSH
HFONT – LOGFONT

Если $pObject равен 0, возвращаемое значение функции – число байтов, необходимых для сохранения информации, которую она записывает в буфер для указанного графического объекта.

Возвращаемое значение

Успех: Если $pObject – действительный указатель, возвращаемое значение – число байтов, записанных в буфер.
Если функция выполнена успешно и $pObject равен 0, возвращаемое значение – число байтов, необходимых для сохранения информации, которую функция записала бы в буфер.
Ошибка: Возвращает 0.
