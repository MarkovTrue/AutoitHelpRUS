_GDIPlus_ImageGetWidth

Возвращает ширину изображения.

#include <GDIPlus.au3>
_GDIPlus_ImageGetWidth ( $hImage )

Параметры

$hImage Дескриптор объекта Image.

Возвращаемое значение

Успех: Возвращает ширина изображения в пикселях.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_ImageGetHeight

Дополнительно

Искать GdipGetImageWidth в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <ScreenCapture.au3>
#include <WinAPIHObj.au3>

Example()

Func Example()
    Local $hBitmap, $hClone, $hImage, $iX, $iY

    ; Инициализировать библиотеку GDI+
    _GDIPlus_Startup()

    ; Захватить 32-битный bitmap
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; Создать клон 24-битного bitmap
    $iX = _GDIPlus_ImageGetWidth($hImage)
    $iY = _GDIPlus_ImageGetHeight($hImage)
    $hClone = _GDIPlus_BitmapCloneArea($hImage, 0, 0, $iX, $iY, $GDIP_PXF24RGB)

    ; Сохранить bitmap в файл
    _GDIPlus_ImageSaveToFile($hClone, @MyDocumentsDir & "\GDIPlus_Image.bmp")

    ; Освободить ресурсы
    _GDIPlus_ImageDispose($hClone)
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Завершить работу с библиотекой GDI+
    _GDIPlus_Shutdown()

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.bmp")
EndFunc   ;==>Example
