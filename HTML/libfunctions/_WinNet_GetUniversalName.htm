_WinNet_GetUniversalName

Преобразует путь, основанный на диске, в универсальный формат.

#include <WinNet.au3>
_WinNet_GetUniversalName ( $sLocalPath )

Параметры

$sLocalPath Путь, основанный на диске, для сетевого ресурса.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:     $aPath[0] – Строка имени UNC, которая идентифицирует сетевой ресурс     $aPath[1] – Имя сетевого подключения     $aPath[2] – Оставшаяся строка пути.
Ошибка: Устанавливает @error ≠ 0.
