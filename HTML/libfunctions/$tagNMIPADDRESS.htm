Справочник ключевых слов

$tagNMIPADDRESS

Содержит сведения для уведомляющего сообщения $IPN_FIELDCHANGED.

Global Const $tagNMIPADDRESS = $tagNMHDR & ";int Field;int Value"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Field Номер поля, которое было изменено (с отсчётом от нуля).
Value Новое значение поля, указанного в члене Field.
При обработке уведомления $IPN_FIELDCHANGED этот член может быть установлен на любое значение в диапазоне поля, и элемент управления поместит это новое значение в поле.
