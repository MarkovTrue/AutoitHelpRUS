Справочник функций

_GUICtrlHeader_InsertItem

Вставляет новый элемент в элемент управления Header.

#include <GuiHeader.au3>
_GUICtrlHeader_InsertItem ( $hWnd, $iIndex, $sText [, $iWidth = 50 [, $iAlign = 0 [, $iImage = -1 [, $bOnRight = False]]]] )

Параметры

$hWnd Дескриптор элемента управления.
$iIndex Индекс элемента, после которого должен быть вставлен новый элемент.
Новый элемент вставляется в конец элемента управления, если индекс больше или равен количеству элементов в элементе управления.
Если индекс равен нулю, новый элемент вставляется в начало элемента управления.
$sText Текст элемента. См. раздел Примечания.
$iWidth [необязательный] Ширина элемента
$iAlign [необязательный] Выравнивание текста:
0 – текст выравнен по левому краю
1 – текст выравнен по правому краю
2 – текст выравнен по центру
$iImage [необязательный] Индекс изображения в списке изображений (нумерация с 0)
$bOnRight [необязательный] Если True – изображение столбца отображается справа от текста

Возвращаемое значение

Успех: Возвращает индекс нового элемента.
Ошибка: Возвращает -1.

Примечания

Если нужен обратный вызов уведомления, вы должны указать $sText = -1 (LPSTR_TEXTCALLBACK).

См. также

_GUICtrlHeader_AddItem, _GUICtrlHeader_DeleteItem

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiHeader.au3>
#include <GuiImageList.au3>
#include <WinAPIGdi.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Header Insert Item (v" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $hHeader = _GUICtrlHeader_Create($hGUI)
    _MemoCreate(2, 52, 444, 220)

;~ _GUICtrlHeader_SetUnicodeFormat($hHeader, True)

    GUISetState(@SW_SHOW)

    ; Создать image list с изображениями
    Local $hImage = _GUIImageList_Create(11, 11)
    _GUIImageList_Add($hImage, _WinAPI_CreateSolidBitmap($hGUI, 0xFF0000, 11, 11))
    _GUIImageList_Add($hImage, _WinAPI_CreateSolidBitmap($hGUI, 0x00FF00, 11, 11))
    _GUIImageList_Add($hImage, _WinAPI_CreateSolidBitmap($hGUI, 0x0000FF, 11, 11))
    _GUICtrlHeader_SetImageList($hHeader, $hImage)

    ; Добавить столбцы
    _GUICtrlHeader_AddItem($hHeader, "Column 0", 90, 0, 0)
    _GUICtrlHeader_AddItem($hHeader, "Column 1", 90, 0, 1)
    _GUICtrlHeader_AddItem($hHeader, "Column 2", 90, 0, 2)
    _GUICtrlHeader_AddItem($hHeader, "Column 3", 90)

    ; Вставить новый столбец
    _GUICtrlHeader_InsertItem($hHeader, 1, "Column X", 90, 2, 0)

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example

Пример 2

; == Пример 2: заголовок вставки элемента во внешний процесс

#include "Extras\HelpFileInternals.au3"

#include <GuiHeader.au3>

Example()

Func Example()
    Local $sFromTo
    Local $hWin = _MemoRunAU3OutProcess($sFromTo, False) ; OK при запуске в другом режиме

    Local $hHeader = _MemoCreateOutProcess($hWin, "SysHeader32", 0, $sFromTo)

    _GUICtrlHeader_InsertItem($hHeader, 2, "<<<Column Y", 100, 2, 2)
    _MemoWrite("Column #2 changed to <<<Column Y")

    _MemoMsgBoxStatus("", -1, $hWin) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
