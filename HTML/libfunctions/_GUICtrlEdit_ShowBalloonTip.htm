_GUICtrlEdit_ShowBalloonTip

Показывает всплывающую подсказку (balloon tip), привязанную к Edit.

#include <GuiEdit.au3>
_GUICtrlEdit_ShowBalloonTip ( $hWnd, $sTitle, $sText [, $iIcon = $TTI_NONE] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$sTitle строка для заголовка подсказки (Unicode)
$sText строка для текста подсказки (Unicode)
$iIcon [необязательный] Значок может быть одним из:
$TTI_ERROR – значок ошибки
$TTI_INFO – информационный значок
$TTI_NONE – без значка (по умолчанию)
$TTI_WARNING – значок предупреждения
Для Windows Vista и выше:
$TTI_ERROR_LARGE – значок ошибки (большой)
$TTI_INFO_LARGE – информационный значок (большой)
$TTI_WARNING_LARGE – значок предупреждения (большой).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_GUICtrlEdit_HideBalloonTip

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <ToolTipConstants.au3>

Example()

Func Example()
    Local $idEdit, $sTitle = "ShowBalloonTip", $sText = "Displays a balloon tip associated with an edit control"

    ; Создать GUI
    GUICreate("Edit ShowBalloonTip", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Установить текст
    _GUICtrlEdit_SetText($idEdit, "This is a test" & @CRLF & "Another Line" & @CRLF & "Append to the end?")

    _GUICtrlEdit_ShowBalloonTip($idEdit, $sTitle, $sText, $TTI_INFO)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
