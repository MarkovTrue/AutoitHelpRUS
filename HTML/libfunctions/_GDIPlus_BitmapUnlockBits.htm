_GDIPlus_BitmapUnlockBits

Разблокирует часть растрового изображения, заблокированную _GDIPlus_BitmapLockBits().

#include <GDIPlus.au3>
_GDIPlus_BitmapUnlockBits ( $hBitmap, $tBitmapData )

Параметры

$hBitmap Дескриптор объекта Bitmap.
$tBitmapData Структура $tagGDIPBITMAPDATA, ранее переданная в _GDIPlus_BitmapLockBits().

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

После завершения работы с заблокированной областью вызовите _GDIPlus_BitmapUnlockBits() для её освобождения.

См. также

_WinAPI_DeleteObject

Дополнительно

Искать GdipBitmapUnlockBits в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Поддержка работы в X64
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ;получить каталог установки AutoIt
    Local $sRegPath = "HKLM\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt"

    Local $sFile = RegRead($sRegPath, "InstallDir") & "\Examples\GUI\logo4.gif"
    If Not FileExists($sFile) Then
        MsgBox(($MB_SYSTEMMODAL + $MB_ICONHAND), "", $sFile & " not found!", 30)
        Return False
    EndIf

    _GDIPlus_Startup()
    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile) ;создать объект изображения на основе файла
    If @error Then
        _GDIPlus_Shutdown()
        MsgBox(($MB_SYSTEMMODAL + $MB_ICONHAND), "", "An error has occured - unable to load image!", 30)
        Return False
    EndIf

    Local $iW = _GDIPlus_ImageGetWidth($hImage), $iH = _GDIPlus_ImageGetHeight($hImage) ; Получить ширину и высоту изображения.
    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0($iW, $iH)
    Local $hContext = _GDIPlus_ImageGetGraphicsContext($hBitmap)
    _GDIPlus_GraphicsDrawImageRect($hContext, $hImage, 0, 0, $iW, $iH)

    Local $tBitmapData = _GDIPlus_BitmapLockBits($hBitmap, 0, 0, $iW, $iH, BitOR($GDIP_ILMWRITE, $GDIP_ILMREAD), $GDIP_PXF32ARGB) ; Заблокировать часть изображения для чтения и записи.
    Local $iScan0 = DllStructGetData($tBitmapData, "Scan0") ; Получить scan0 (пиксельные данные) из заблокированного изображения.
    Local $iSearchPixel = Int(0xFF000080), $iReplaceColor = 0xFF000000 ; Формат цвета 0xAARRGGBB.
    Local $tPixel = DllStructCreate("int[" & $iW * $iH & "];", $iScan0)
    Local $iPixel, $iRowOffset

    For $iY = 0 To $iH - 1
        $iRowOffset = $iY * $iW + 1
        For $iX = 0 To $iW - 1 ; Перебрать каждый пиксель в каждой строке и столбце.
            $iPixel = DllStructGetData($tPixel, 1, $iRowOffset + $iX) ; Получить цвет пикселя.
            If $iPixel = $iSearchPixel Then DllStructSetData($tPixel, 1, $iReplaceColor, $iRowOffset + $iX) ; Сравнить и заменить цвет пикселя (синий на чёрный).
        Next
    Next
    _GDIPlus_BitmapUnlockBits($hBitmap, $tBitmapData) ; Разблокировать часть изображения, заблокированную _GDIPlus_BitmapLockBits.

    ; Для сохранения изменённого изображения используйте _GDIPlus_ImageSaveToFile().

    ShellExecute($sFile) ; Отобразить исходное изображение для сравнения с изменённым.

    ; Отобразить изменённое изображение.
    Local $hGUI = GUICreate("_GDIPlus_BitmapLockBits Demo", $iW, $iH)
    GUISetState(@SW_SHOW)

    Local $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать объект Graphics из дескриптора окна
    _GDIPlus_GraphicsDrawImageRect($hGraphic, $hBitmap, 0, 0, $iW, $iH) ; Скопировать изменённое изображение в дескриптор Graphics.

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;освободить ресурсы
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hContext)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
