_WinAPI_UrlIs

Проверяет, является ли URL определённым типом.

#include <WinAPIShPath.au3>
_WinAPI_UrlIs ( $sUrl [, $iType = $URLIS_URL] )

Параметры

$sUrl URL.
$iType [необязательный] Тип URL для проверки. Это может быть одно из следующих значений.
$URLIS_APPLIABLE
$URLIS_DIRECTORY
$URLIS_FILEURL
$URLIS_HASQUERY
$URLIS_NOHISTORY
$URLIS_OPAQUE
$URLIS_URL (по умолчанию).

Возвращаемое значение

Успех: Возвращает True – URL имеет указанный тип.
Ошибка: Возвращает False.
