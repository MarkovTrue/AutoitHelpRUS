Справочник функций

_WinAPI_RegDisableReflectionKey

Отключает отражение реестра для указанного ключа.

#include <WinAPIReg.au3>
_WinAPI_RegDisableReflectionKey ( $hKey )

Параметры

$hKey Дескриптор открытого раздела реестра.
Возвращается функциями _WinAPI_RegCreateKey() и _WinAPI_RegOpenKey();
не может указывать на раздел удалённого компьютера.
Если раздел не находится в списке отражений, функция успешно завершится, но эффекта не окажет.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

В WOW64 32-битные приложения просматривают дерево реестра, которое отделено от дерева реестра, просматриваемого 64-битными приложениями.
Отражение реестра копирует определённые ключи и значения реестра между двумя видами.

Для восстановления отражения реестра для отключённого ключа используйте функцию _WinAPI_RegEnableReflectionKey().

Предупреждение: отражение реестра было удалено начиная с Windows 7 и Windows Server 2008 R2.
