_WinAPI_HashData

Вычисляет хеш блока памяти.

#include <WinAPIConv.au3>
_WinAPI_HashData ( $pMemory, $iSize [, $iLength = 32] )

Параметры

$pMemory Указатель на блок памяти, содержащий данные для хеширования.
$iSize Размер блока памяти в байтах.
$iLength [необязательный] Длина хеша в байтах. Не должна превышать 256, иначе функция не выполняется. По умолчанию – 32.

Возвращаемое значение

Успех: Возвращает хеш-данные в двоичном виде.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Дополнительно

Искать HashData в библиотеке MSDN

Пример

#include <WinAPIConv.au3>

Local $bData = Binary('0x00112233445566778899AABBCCDDEEFF00112233445566778899AABBCCDDEEFF00112233445566778899AABBCCDDEEFF00112233445566778899AABBCCDDEEFF')
Local $iSize = BinaryLen($bData)
Local $tData = DllStructCreate('byte[' & $iSize & ']')
DllStructSetData($tData, 1, $bData)

ConsoleWrite(_WinAPI_HashData($tData, $iSize) & @CRLF)
