Справочник функций

_WinAPI_SetBitmapDimensionEx

Назначает предпочтительные размеры совместимому растру.

#include <WinAPIGdi.au3>
_WinAPI_SetBitmapDimensionEx ( $hBitmap, $iWidth, $iHeight )

Параметры

$hBitmap Дескриптор растра. Этот растр не может быть DIB-section растром.
$iWidth Ширина растра в единицах 0,1 миллиметра.
$iHeight Высота растра в единицах 0,1 миллиметра.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Растр не может быть DIB-section, что является растром, созданным функциями _WinAPI_CreateDIB() или _WinAPI_CreateDIBSection().
Если растр является DIB-section, функция _WinAPI_SetBitmapDimensionEx() завершится неудачей.

Вызовите _WinAPI_GetExtended() для получения структуры $tagSIZE, содержащей предыдущие размеры растра.

См. также

_WinAPI_CreateDIB, _WinAPI_CreateDIBSection, _WinAPI_GetExtended
