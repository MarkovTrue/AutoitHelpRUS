Справочник функций

_SendMessage

Оболочка для часто используемого вызова DLL.

#include <SendMessage.au3>
_SendMessage ( $hWnd, $iMsg [, $wParam = 0 [, $lParam = 0 [, $iReturn = 0 [, $wParamType = "wparam" [, $lParamType = "lparam" [, $sReturnType = "lresult"]]]]]] )

Параметры

$hWnd Дескриптор окна/элемента управления.
$iMsg Сообщение для отправки элементу управления (число).
$wParam [необязательный] Дополнительные сведения, специфичные для сообщения
$lParam [необязательный] Дополнительные сведения, специфичные для сообщения
$iReturn [необязательный] Что возвращать:
0 – Возвращаемое значение вызова DLL
1 – $ihWnd
2 – $iMsg
3 – $wParam
4 – $lParam
<0 или > 4 – массив как в DllCall()
$wParamType [необязательный] См. DllCall в разделе "См. также"
$lParamType [необязательный] См. DllCall в разделе "См. также"
$sReturnType [необязательный] См. DllCall в разделе "См. также"

Возвращаемое значение

Успех: Возвращает выбранное пользователем значение из результата DllCall().
Ошибка: Устанавливает @error ≠ 0.

Примечания

Когда сообщение блокируется UIPI, последняя ошибка, полученная с помощью _WinAPI_GetLastError(), устанавливается в 5 (доступ запрещён).

См. также

DllCall, _SendMessageA, _WinAPI_GetLastError

Пример

#include <MsgBoxConstants.au3>
#include <SendMessage.au3>
#include <WindowsNotifsConstants.au3>

Example()

Func Example()
    Local Const $iOff = 2, $iOn = -1

    Opt("WinTitleMatchMode", 4)
    Local $hWnd = WinGetHandle('classname=Progman')
    _ToggleMonitor($hWnd, $iOff)
    Sleep(3000)
    _ToggleMonitor($hWnd, $iOn)
EndFunc   ;==>Example

Func _ToggleMonitor($hWnd, $iOnOff)
    Local Const $WM_SYSCOMMAND = 274
    Local Const $SC_MONITORPOWER = 61808
    _SendMessage($hWnd, $WM_SYSCOMMAND, $SC_MONITORPOWER, $iOnOff)
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "_ToggleMonitor", "_SendMessage Error: " & @error)
        Exit
    EndIf
EndFunc   ;==>_ToggleMonitor
