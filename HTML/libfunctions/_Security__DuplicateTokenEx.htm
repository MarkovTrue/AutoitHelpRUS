Справочник функций

_Security__DuplicateTokenEx

Создаёт новый токен доступа, который дублирует существующий токен.

#include <Security.au3>
_Security__DuplicateTokenEx ( $hExistingToken, $iDesiredAccess, $iImpersonationLevel, $iTokenType )

Параметры

$hExistingToken Дескриптор токена доступа, открытого с доступом TOKEN_DUPLICATE.
$iDesiredAccess Запрашиваемые права доступа для нового токена.
$iImpersonationLevel Уровень имперсонации нового токена.
$iTokenType Тип нового токена.

Возвращаемое значение

Успех: Возвращает дескриптор, получающий новый токен.
Ошибка: Возвращает 0.

См. также

_Security__OpenProcessToken, _Security__OpenThreadToken, _Security__OpenThreadTokenEx

Дополнительно

Искать DuplicateTokenEx в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <ProcessConstants.au3>
#include <Security.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

Local $hProcess = _WinAPI_OpenProcess($PROCESS_ALL_ACCESS, 0, ProcessExists("explorer.exe"))
; В случае успеха
If $hProcess Then
    ; Токен...
    Local $hTokOriginal = _Security__OpenProcessToken($hProcess, $TOKEN_ALL_ACCESS)
    ; Дескриптор процесса больше не нужен – закрыть
    _WinAPI_CloseHandle($hProcess)
    ; В случае успеха
    If $hTokOriginal Then
        ; Создать дубликат исходного токена
        Local $hTokDuplicate = _Security__DuplicateTokenEx($hTokOriginal, $TOKEN_ALL_ACCESS, $SECURITYIMPERSONATION, $TOKENPRIMARY)
        ; Закрыть исходный токен
        _WinAPI_CloseHandle($hTokOriginal)
        ; Создаётся основной токен (!)
        ; ... Сделайте что-то с этим токеном здесь ...

        MsgBox($MB_SYSTEMMODAL, "DuplicateTokenEx", "$hTokDuplicate = " & $hTokDuplicate)

        ; Закрыть этот токен по завершении
        _WinAPI_CloseHandle($hTokDuplicate)
    EndIf
EndIf
