Справочник функций

_Excel_Print

Печатает книгу, рабочий лист, диаграмму или диапазон.

#include <Excel.au3>
_Excel_Print ( $oExcel, $vObject [, $iCopies = Default [, $sPrinter = Default [, $bPreview = Default [, $iFrom = Default [, $iTo = Default [, $bPrintToFile = Default [, $bCollate = Default [, $sPrToFileName = ""]]]]]]]] )

Параметры

$oExcel Объект приложения Excel.
$vObject Объект книги, листа, диаграммы или диапазона для печати. Диапазон можно задать в формате A1.
$iCopies [необязательный] Количество копий для печати (по умолчанию Default = 1).
$sPrinter [необязательный] Имя используемого принтера. По умолчанию – активный принтер (Default).
$bPreview [необязательный] True – открыть предварительный просмотр перед печатью (по умолчанию Default = False).
$iFrom [необязательный] Номер страницы, с которой начинается печать (по умолчанию Default – первая страница).
$iTo [необязательный] Номер страницы, на которой завершается печать (по умолчанию Default – последняя страница).
$bPrintToFile [необязательный] True – печать в файл. См. параметр $sPrToFileName (по умолчанию Default = False).
$bCollate [необязательный] True – разобрать копии по экземплярам (по умолчанию Default = False).
$sPrToFileName [необязательный] Если $bPrintToFile = True, задаёт имя файла для печати.

Возвращаемое значение

Успех: Возвращает объект напечатанного диапазона.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oExcel не является объектом или объектом приложения
2 – $vObject не является объектом или недопустимым диапазоном A1; @error содержит код COM-ошибки
3 – ошибка печати объекта.
@extended: Содержит код COM-ошибки.

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Print Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel4.xls"), True)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Print Example", "Error opening workbook '" & _Extras_PathFull("_Excel4.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Печатаем диапазон A1:B3 активного листа на принтере по умолчанию.
_Excel_Print($oExcel, "A1:B3")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Print Example 1", "Error printing cells." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_Print Example 1", "Range successfully printed.")
