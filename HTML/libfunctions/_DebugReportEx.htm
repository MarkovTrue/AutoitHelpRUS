Справочник функций

_DebugReportEx

Записывает в сеанс отладки форматированное сообщение.

#include <Debug.au3>
_DebugReportEx ( $sData [, $bLastError = False [, $bExit = False]] )

Параметры

$sData Данные для записи в журнал (например, "dllname|functionname").
$bLastError [необязательный] True – добавить к выводимым данным результат _WinAPI_GetLastErrorMessage().
$bExit [необязательный] True – скрипт должен быть завершён

Возвращаемое значение

Успех: Возвращает 1 (см. раздел Примечания).
Ошибка: Возвращает 0.

Примечания

В зависимости от значения @error сообщение форматируется следующим образом:
0 - сообщение "Bad return from FUNCTION in DLL".
1 - сообщение "unable to open a DLL".
3 - сообщение "unable to find FUNCTION in DLL".

Если $sData не содержит "|" или @error не равен указанным выше значениям, сообщение не будет отформатировано.
Если _DebugSetup() не был вызван, функция всегда возвращает управление.
@error вызывающего кода сохраняется.
@extended может быть установлен в значение Windows API _WinAPI_GetLastError() если $bLastError = True.
Иначе он сохраняется.

Каждый вызов _DebugReportEx() выводит сообщение на новой строке.

См. также

_DebugSetup

Пример

#include <Debug.au3>

_DebugSetup()

_DebugOut("message1")

SomeUDF("anyfunction")
If @error Then _DebugReportEx("user32|anyfunction", True) ; с добавленным последним сообщением об ошибке

Local $iRet = SomeUDF("CloseClipboard")
If @error Or $iRet = 0 Then _DebugReportEx("user32|CloseClipboard")

_DebugOut("message2")

$iRet = SomeUDF("CloseClipboard")
If @error Or $iRet = 0 Then _DebugReportEx("user32|CloseClipboard", False, True) ; Скрипт будет завершён.

_DebugReport("message3") ; Не будет записано в журнал.

Func SomeUDF($sFunc)
    Local $aResult = DllCall("user32.dll", "int", $sFunc)
    If @error Then Return SetError(@error, @extended, 0)
    Return $aResult[0]
EndFunc   ;==>SomeUDF
