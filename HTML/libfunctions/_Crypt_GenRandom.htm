Справочник функций

_Crypt_GenRandom

Заполняет буфер криптографически случайными данными.

#include <Crypt.au3>
_Crypt_GenRandom ( $pBuffer, $iSize )

Параметры

$pBuffer Указатель на буфер для заполнения случайными данными.
$iSize Размер буфера, на который указывает $pBuffer.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.

Примечания

При желании приложение может предварительно заполнить буфер данными, которые будут использованы как дополнительный источник случайности.

Дополнительно

Искать CryptGenRandom в библиотеке MSDN

Пример

#include <Crypt.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Генерируем 16 байт случайных данных.
    Local $tBuff = DllStructCreate("byte[16]")
    _Crypt_GenRandom($tBuff, DllStructGetSize($tBuff))
    MsgBox($MB_SYSTEMMODAL, "Random data:", DllStructGetData($tBuff, 1))
EndFunc   ;==>Example
