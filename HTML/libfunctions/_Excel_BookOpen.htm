Справочник функций

_Excel_BookOpen

Открывает существующую рабочую книгу.

#include <Excel.au3>
_Excel_BookOpen ( $oExcel, $sFilePath [, $bReadOnly = False [, $bVisible = True [, $sPassword = Default [, $sWritePassword = Default [, $bUpdateLinks = Default]]]]] )

Параметры

$oExcel Объект приложения Excel.
$sFilePath Путь и имя открываемого файла.
$bReadOnly [необязательный] True открывает книгу только для чтения (по умолчанию False).
$bVisible [необязательный] True делает окно книги видимым (по умолчанию True).
$sPassword [необязательный] Пароль, защищающий книгу от чтения, если он есть (по умолчанию отсутствует).
$sWritePassword [необязательный] Пароль, защищающий книгу от записи, если он есть (по умолчанию отсутствует).
$bUpdateLinks [необязательный] Задаёт способ обновления внешних ссылок в файле (по умолчанию Default)
Допустимые значения:
Default: Excel спрашивает пользователя, как обновлять ссылки.
0: Excel не обновляет ссылки.
3: Excel обновляет все ссылки.

Возвращаемое значение

Успех: Возвращает объект рабочей книги. @extended устанавливается в 1, если $bReadOnly = False, но доступ для чтения и записи получить не удалось. Подробности см. раздел Примечания.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oExcel не является объектом или не является объектом приложения
2 – указанный $sFilePath не существует
3 – не удалось открыть $sFilePath. @extended содержит COM-код ошибки, возвращённый методом Open
4 – Excel не вернул объект книги, возможно из-за того, что Excel не может открыть одновременно две книги с одинаковым именем.

Примечания

Если задано $bReadOnly = False, но документ нельзя открыть для чтения и записи, @extended устанавливается в 1.
Книга открыта только для чтения, потому что её уже открыл другой пользователь или процесс, либо файл помечен как доступный только для чтения в файловой системе.
Если вы измените книгу, для сохранения в другое место или под другим именем используйте _Excel_BookSaveAs().

Если при открытии книги задано $bVisible = False, перед сохранением обязательно сделайте её снова видимой.
Чтобы снова показать книгу, используйте $oExcel.Windows($oWorkbook.Name).Visible = True.
Иначе книга не отобразится при ручном открытии в Excel.
Чаще всего этот параметр не нужен. Предпочтительнее задавать $bVisible = False в _Excel_Open().

Если $bUpdateLinks равен Default, пользователь получит запрос только при $bDisplayAlerts = True в функции _Excel_Open().
Если $bDisplayAlerts = False, Excel выбирает ответ по умолчанию.

Пример

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpen Example 1", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Открываем книгу.
Local $sWorkbook = _Extras_PathFull("_Excel1.xls")
Local $oWorkbook = _Excel_BookOpen($oExcel, $sWorkbook)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpen Example 1", "Error opening '" & $sWorkbook & "'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpen Example", "Workbook '" & $sWorkbook & "' has been opened successfully." & @CRLF & @CRLF & "Creation Date: " & $oWorkbook.BuiltinDocumentProperties("Creation Date").Value)
