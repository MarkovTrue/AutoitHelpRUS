Справочник функций

_Excel_BookOpen

Открывает указанную книгу.

#include <Excel.au3>
_Excel_BookOpen ( $oExcel, $sFilePath [, $bReadOnly = False [, $bVisible = True [, $sPassword = Default [, $sWritePassword = Default [, $bUpdateLinks = Default]]]]] )

Параметры

$oExcel Объект приложения Excel.
$sFilePath Полный путь к открываемой книге.
$bReadOnly [необязательный] True – открыть книгу в режиме только для чтения (по умолчанию False).
$bVisible [необязательный] True – отобразить книгу (по умолчанию True).
$sPassword [необязательный] Пароль для открытия защищённой книги.
$sWritePassword [необязательный] Пароль для записи в защищённую книгу.
$bUpdateLinks [необязательный] Задаёт режим обновления ссылок в файле. Default – используется параметр Excel по умолчанию. 0 – не обновлять ссылки. 3 – обновить ссылки.

Возвращаемое значение

Успех: Возвращает объект книги. @extended устанавливается в 1, если книга открыта только для чтения, хотя для $bReadOnly было задано False.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oExcel не является объектом
2 – $sFilePath не является допустимым путём
3 – $bReadOnly не является булевым значением
4 – Excel вернул ошибку.
@extended: Содержит код COM-ошибки.

Примечания

Если $bReadOnly задан как False, а книга открыта только для чтения (например, другим пользователем), @extended устанавливается в 1.
Если $bVisible задан как False, перед сохранением установите его в True, чтобы избежать запросов Excel.
При значении Default для $bUpdateLinks поведение зависит от параметра $bDisplayAlerts функции _Excel_Open().

Пример

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpen Example 1", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Открываем книгу.
Local $sWorkbook = _Extras_PathFull("_Excel1.xls")
Local $oWorkbook = _Excel_BookOpen($oExcel, $sWorkbook)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpen Example 1", "Error opening '" & $sWorkbook & "'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookOpen Example", "Workbook '" & $sWorkbook & "' has been opened successfully." & @CRLF & @CRLF & "Creation Date: " & $oWorkbook.BuiltinDocumentProperties("Creation Date").Value)
