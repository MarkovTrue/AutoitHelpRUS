Справочник функций

_WinAPI_GetCursor

Получает дескриптор текущего курсора.

#include <WinAPIRes.au3>
_WinAPI_GetCursor ( )

Возвращаемое значение

Успех: Возвращает дескриптор текущего курсора. Если курсора нет, возвращаемое значение равно 0.
Ошибка: Возвращает 0.
