_WinAPI_GetFileType

Получает тип файла указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetFileType ( $hFile )

Параметры

$hFile Дескриптор файла.

Возвращаемое значение

Успех: Возвращает тип файла ($FILE_TYPE_*).
Ошибка: (-1) и устанавливает @error ≠ 0. Флаг @extended может содержать код системной ошибки.

Примечания

Константы $FILE_TYPE_* требуют #include <APIFilesConstants.au3>
