Справочник функций

_WinAPI_AddFontResourceEx

Добавляет ресурс шрифта из указанного файла в таблицу системных шрифтов.

#include <WinAPIGdi.au3>
_WinAPI_AddFontResourceEx ( $sFont [, $iFlag = 0 [, $bNotify = False]] )

Параметры

$sFont Строка, содержащая допустимое имя файла шрифта. Этот параметр может указывать любой из следующих файлов:
.fon – файл ресурса шрифта.
.fnt – файл растрового шрифта в необработанном виде.
.ttf – файл TrueType в необработанном виде.
.ttc – Восточноазиатские версии Windows: коллекция шрифтов TrueType.
.fot – файл ресурса TrueType.
.otf – шрифт PostScript OpenType.
.mmm – файл ресурса шрифта Type1 с несколькими мастерами. Должен использоваться с файлами .pfm и .pfb.
.pfb – файл бит шрифта Type 1. Используется с файлом .pfm.
.pfm – файл метрик шрифта Type 1. Используется с файлом .pfb.

Чтобы добавить шрифт, информация о котором поступает из нескольких файлов ресурсов, они должны быть разделены символом "|".
Например, abcxxxxx.pfm|abcxxxxx.pfb.
$iFlag [необязательный] Характеристики шрифта, который будет добавлен в систему. Этот параметр может быть одним из следующих значений.
$FR_PRIVATE
$FR_NOT_ENUM
$bNotify [необязательный] Определяет, отправлять ли сообщение WM_FONTCHANGE, допустимые значения:
True – отправить сообщение WM_FONTCHANGE всем окнам верхнего уровня после изменения пула ресурсов шрифтов.
False – не отправлять (по умолчанию).

Возвращаемое значение

Успех: Возвращает значение, которое определяет количество добавленных шрифтов.
Ошибка: Возвращает 0.

Примечания

Эта функция устанавливает шрифт только для текущего сеанса. При перезагрузке системы шрифт не будет присутствовать.
Чтобы шрифт был установлен даже после перезагрузки системы, шрифт должен быть указан в реестре.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>

Local $sPathFull = _Extras_PathFull('SF Square Head Bold.ttf')
_WinAPI_AddFontResourceEx($sPathFull, $FR_PRIVATE)

GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 100)
GUICtrlCreateLabel('Simple Text', 10, 25, 380, 50, $SS_CENTER)
GUICtrlSetFont(-1, 38, -1, -1, 'SF Square Head Bold')
GUICtrlSetColor(-1, 0xF06000)
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
