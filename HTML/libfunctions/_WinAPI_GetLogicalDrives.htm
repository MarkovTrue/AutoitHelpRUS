_WinAPI_GetLogicalDrives

Получает битовую маску, представляющую доступные диски.

#include <WinAPIFiles.au3>
_WinAPI_GetLogicalDrives ( )

Возвращаемое значение

Успех: Возвращает значение – битовая маска, представляющая доступные диски. Позиция бита 0 (наименьший значащий бит)
это диск A, позиция бита 1 – диск B, позиция бита 2 – диск C и так далее.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
