_WinAPI_PathCanonicalize

Удаляет элементы пути файла согласно специальным строкам, вставленным в этот путь.

#include <WinAPIShPath.au3>
_WinAPI_PathCanonicalize ( $sFilePath )

Параметры

$sFilePath Путь для канонизации.

Возвращаемое значение

Успех: Возвращает канонизированный путь.
Ошибка: Исходный параметр $sFilePath, вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке.

Примечания

Эта функция позволяет пользователю указать, что удалить из пути, вставив специальные последовательности символов
в путь. Последовательность ".." указывает удалить сегмент пути из текущей позиции до предыдущего сегмента пути.
Последовательность "." указывает пропустить следующий сегмент пути к следующему сегменту пути. Корневой
сегмент пути не может быть удалён.

Если последовательностей ".." больше, чем сегментов пути, функция возвращает только корень, "\".

Дополнительно

Искать PathCanonicalize в библиотеке MSDN

Пример

#include <WinAPIShPath.au3>

Local $aPath[5] = ['A:\Dir1\.\Dir2\..\Dir3', 'A:\Dir1\..\Dir2\.\Dir3', 'A:\Dir1\Dir2\.\Dir3\..\Dir4', 'A:\Dir1\.\Dir2\.\Dir3\..\Dir4\..', 'A:\..']

For $i = 0 To 4
    ConsoleWrite($aPath[$i] & ' => ' & _WinAPI_PathCanonicalize($aPath[$i]) & @CRLF)
Next
