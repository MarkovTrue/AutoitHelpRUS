Справочник функций

_WinAPI_CommDlgExtendedError

Возвращает строку ошибки стандартного диалога. Эта строка указывает на самую последнюю ошибку, возникшую при выполнении одной из функций стандартного диалога.

#include <WinAPIDlg.au3>
_WinAPI_CommDlgExtendedError ( )

Возвращаемое значение

Успех: Возвращает строка ошибки.
Ошибка: Шестнадцатеричная строка расширенной ошибки.

Примечания

Может возвращать общие строки ошибок для любых функций стандартного диалога.

См. также

_WinAPI_GetOpenFileName, _WinAPI_GetSaveFileName
