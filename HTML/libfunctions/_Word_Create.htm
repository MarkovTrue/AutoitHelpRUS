Справочник функций

_Word_Create

Подключается к существующему экземпляру Microsoft Word или создаёт новый объект приложения Microsoft Word.

#include <Word.au3>
_Word_Create ( [$bVisible = True [, $bForceNew = False]] )

Параметры

$bVisible [необязательный] True указывает, что окно будет видно (по умолчанию)
$bForceNew [необязательный] True принуждает создать новый экземпляр Microsoft Word, даже если есть уже работающий экземпляр (по умолчанию = False)

Возвращаемое значение

Успех: Возвращает переменная, указывающая на объект Word.Application. Задаёт @extended: 0 – Microsoft Word уже работал 1 – Microsoft Word не работал или $bForceNew было установлено в True. Microsoft Word был запущен этой функцией.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – Ошибка, возвращённая функцией ObjCreate(). @extended задан код ошибки COM.

Пример

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать новый объект Word
_Word_Create()
If @error <> 0 Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_Create Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
If @extended = 1 Then
    MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_Create Example", "MS Word was not running when _Word_Create was called.")
Else
    MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_Create Example", "MS Word was already running when _Word_Create was called.")
EndIf
