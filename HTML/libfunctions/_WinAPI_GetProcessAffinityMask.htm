_WinAPI_GetProcessAffinityMask

Получает маски сродства для процесса и системы.

#include <WinAPIProc.au3>
_WinAPI_GetProcessAffinityMask ( $hProcess )

Параметры

$hProcess Открытый дескриптор процесса, маска сродства которого требуется получить.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
$aMask[0] – True при успехе, иначе False
$aMask[1] – маска сродства процесса
$aMask[2] – маска сродства системы
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Маска сродства – битовая маска, в которой каждый бит представляет процессор, на котором потокам процесса
разрешено выполняться. Например, если вы передаёте маску 0x05, процессорам 1 и 3 разрешено работать.
