_WinAPI_GetThemeBackgroundRegion

Вычисляет область для обычного или полупрозрачного фона, ограниченного указанным прямоугольником.

#include <WinAPITheme.au3>
_WinAPI_GetThemeBackgroundRegion ( $hTheme, $iPartID, $iStateID, $hDC, $tRECT )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит область.
$iStateID Состояние части.
$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT, которая содержит указанный прямоугольник, используемый для вычисления области.

Возвращаемое значение

Успех: Возвращает дескриптор вычисленной области.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Дескриптор области, возвращённый этой функцией, должен быть освобождён, когда он больше не нужен,
используя _WinAPI_DeleteObject().
