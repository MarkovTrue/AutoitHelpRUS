_WinAPI_EnumHardLinks

Перечисляет все жёсткие ссылки на указанный файл.

#include <WinAPIFiles.au3>
_WinAPI_EnumHardLinks ( $sFilePath )

Параметры

$sFilePath Путь к файлу, ссылки на который перечисляются.

Возвращаемое значение

Успех: Возвращает массив полных путей к ссылкам. Нулевой элемент содержит количество ссылок.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки NTSTATUS.

Примечания

Функция _WinAPI_CreateHardLink() поддерживается только в файловой системе NTFS.

Требуется Windows Vista или новее.

Дополнительно

Искать ZwQueryInformationFile в библиотеке MSDN

Пример

#include <Array.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>
#include <WinAPIShPath.au3>

Local $sFile = @DesktopDir & '\' & StringRegExpReplace(_WinAPI_PathFindFileName(@ScriptName), '\A_+', '@')

; Создать жёсткую ссылку на текущий файл с префиксом "@" на рабочем столе
If Not _WinAPI_CreateHardLink($sFile, @ScriptFullPath) Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Unable to create hard link.')
    Exit
EndIf

; Перечислить все жёсткие ссылки на файл
Local $aData = _WinAPI_EnumHardLinks($sFile)

_ArrayDisplay($aData, '_WinAPI_EnumHardLinks')

FileDelete($sFile)
