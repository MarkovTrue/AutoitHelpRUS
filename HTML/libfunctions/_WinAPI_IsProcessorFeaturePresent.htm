Справочник функций

_WinAPI_IsProcessorFeaturePresent

Определяет, поддерживается ли указанная функция процессора текущим компьютером.

#include <WinAPISys.au3>
_WinAPI_IsProcessorFeaturePresent ( $iFeature )

Параметры

$iFeature Функция процессора для проверки. Этот параметр может быть одной из констант $PF_*, определённых в APISysConstants.au3.

Возвращаемое значение

True: Возвращает функция поддерживается.
False: функция не поддерживается.
