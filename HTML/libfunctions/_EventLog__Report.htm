Справочник функций

_EventLog__Report

Записывает запись в конец указанного журнала событий.

#include <EventLog.au3>
_EventLog__Report ( $hEventLog, $iType, $iCategory, $iEventID, $sUserName, $sDesc, $aData )

Параметры

$hEventLog Дескриптор журнала событий.
$iType Тип события. Возможные значения:
0 – успех
1 – ошибка
2 – предупреждение
4 – информация
8 – аудит успеха
16 – аудит отказа
$iCategory Категория события. Это специфичная для источника информация; категория может принимать любое значение.
$iEventID Идентификатор события. Указывает запись в файле сообщений, связанном с источником событий.
$sUserName Имя пользователя для события. Может быть пустым, если имя не требуется.
$sDesc Описание события.
$aData Массив данных следующего формата:
[0] – количество байт в массиве
[1] – байт 1
[2] – байт 2
[n] – байт n

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция используется для записи события в журнал.
Запись добавляется в конец журнала, настроенного для источника, идентифицированного параметром $hEventLog.
Перед сохранением записи функция добавляет время, длину записи и смещения.

См. также

_EventLog__Close, _EventLog__Open

Пример

#include <EventLog.au3>

Example()

Func Example()
    Local $hEventLog, $aData[4] = [3, 1, 2, 3]

    $hEventLog = _EventLog__Open("", "Application")
    _EventLog__Report($hEventLog, 4, 0, 2, "Administrator", "AutoIt3 generated event", $aData)
    _EventLog__Close($hEventLog)
EndFunc   ;==>Example
