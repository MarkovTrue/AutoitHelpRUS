_WinAPI_GetRegisteredRawInputDevices

Получает информацию об устройствах сырого ввода для текущего приложения.

#include <WinAPISys.au3>
_WinAPI_GetRegisteredRawInputDevices ( $pBuffer, $iLength )

Параметры

$pBuffer Указатель на буфер для получения массива структур $tagRAWINPUTDEVICE для приложения.
Если этот параметр равен 0, функция возвращает требуемый размер буфера в байтах.
$iLength Размер буфера в байтах.

Возвращаемое значение

Успех: Возвращает количество структур $tagRAWINPUTDEVICE, записанных в буфер.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0, флаг @extended содержит код последней ошибки.
