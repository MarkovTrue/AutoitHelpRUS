Справочник ключевых слов

$tagTVITEMEX

Задаёт или получает атрибуты элемента дерева (TreeView).

Global Const $tagTVITEMEX = "struct; uint Mask;struct; handle hItem;uint State;uint StateMask;ptr Text;int TextMax;int Image;int SelectedImage;int Children;lparam Param; endstruct;" & _
"int Integral;uint uStateEx;hwnd hwnd;int iExpandedImage;int iReserved; endstruct"

Параметры

Mask Флаги, указывающие, какие из остальных полей структуры содержат верные данные:
$TVIF_CHILDREN - поле Children действительно
$TVIF_DI_SETITEM - элемент сохранит переданные сведения и не будет запрашивать их снова
$TVIF_EXPANDEDIMAGE - поле iExpandedImage действительно (IE 6.00 и Windows Vista)
$TVIF_HANDLE - поле hItem действительно
$TVIF_IMAGE - поле Image действительно
$TVIF_INTEGRAL - поле Integral действительно
$TVIF_PARAM - поле Param действительно
$TVIF_SELECTEDIMAGE - поле SelectedImage действительно
$TVIF_STATE - поля State и StateMask действительны
$TVIF_STATEEX - поле uStateEx действительно (IE 6.00 и Windows Vista)
$TVIF_TEXT - поля Text и TextMax действительны.
hItem Элемент, к которому относится эта структура.
State Набор битовых флагов и индексов списка изображений, указывающих состояние элемента, при установке состояния элемента.
Поле StateMask указывает, какие биты этого поля действительны. При получении состояния элемента это поле возвращает текущее состояние для битов, указанных в поле StateMask.
Биты с 0 по 7 этого поля содержат флаги состояния элемента.
Биты с 8 по 11 этого поля задают индекс наложенного изображения (начиная с 1).
StateMask Биты поля State, которые действительны. При получении состояния элемента установите биты поля StateMask, чтобы указать, какие биты вернуть в поле State.
При установке состояния элемента установите биты поля StateMask, чтобы указать, какие биты поля State вы хотите задать.
Text Указатель на строку с завершающим нулём, содержащую текст элемента.
TextMax Размер буфера, на который указывает поле Text, в символах.
Image Индекс в списке изображений значка, используемого, когда элемент не выбран.
SelectedImage Индекс в списке изображений значка, используемого, когда элемент выбран.
Children Флаг, указывающий, есть ли у элемента дочерние элементы. Это поле может быть одним из следующих значений:
0 - у элемента нет дочерних элементов
1 - у элемента есть один или несколько дочерних элементов
Param Значение, связываемое с элементом.
Integral Высота элемента.
uStateEx Одно или несколько (как битовая комбинация) из следующих расширенных состояний.
Значение и смысл:
TVIS_EX_DISABLED Windows Vista или новее. Создаёт элемент управления, отрисованный серым, с которым пользователь не может взаимодействовать.
TVIS_EX_FLAT создаёт плоский элемент: он виртуальный и не виден в дереве; вместо него в иерархии дерева занимают место его дочерние элементы.
TVIS_EX_HWND создаёт для элемента отдельный HWND.
hwnd Не используется; должно быть NULL.
iExpandedImage Индекс изображения в списке изображений элемента управления, отображаемого, когда элемент развёрнут.
iReserved Зарезервированное поле. Не использовать.
