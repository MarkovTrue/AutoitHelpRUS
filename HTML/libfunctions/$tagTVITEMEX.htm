Справочник ключевых слов

$tagTVITEMEX

Задаёт или принимает атрибуты элемента TreeView.

Global Const $tagTVITEMEX = "struct; uint Mask;struct; handle hItem;uint State;uint StateMask;ptr Text;int TextMax;int Image;int SelectedImage;int Children;lparam Param; endstruct;" & _
"int Integral;uint uStateEx;hwnd hwnd;int iExpandedImage;int iReserved; endstruct"

Параметры

Mask Флаги, указывающие, какие другие элементы структуры содержат действительные данные:
$TVIF_CHILDREN – элемент Children действителен
$TVIF_DI_SETITEM – сохранит предоставленную информацию и не будет запрашивать её снова
$TVIF_EXPANDEDIMAGE – элемент iExpandedImage действителен (IE версия 6.00 и Windows Vista)
$TVIF_HANDLE – элемент hItem действителен
$TVIF_IMAGE – элемент Image действителен
$TVIF_INTEGRAL – элемент Integral действителен
$TVIF_PARAM – элемент Param действителен
$TVIF_SELECTEDIMAGE – элемент SelectedImage действителен
$TVIF_STATE – элементы State и StateMask действительны
$TVIF_STATEEX – элемент uStateEx действителен (IE версия 6.00 и Windows Vista)
$TVIF_TEXT – элементы Text и TextMax действительны.
hItem Элемент, к которому относится эта структура.
State Набор битовых флагов и индексов списка изображений, указывающих состояние элемента. При установке состояния элемента.
Элемент StateMask указывает биты этого элемента, которые действительны. При получении состояния элемента этот элемент возвращает текущее состояние для битов, указанных в элементе StateMask.
Биты 0–7 этого элемента содержат флаги состояния элемента.
Биты 8–11 этого элемента определяют индекс наложенного изображения с отсчётом от единицы.
StateMask Биты элемента state, которые действительны. Если вы получаете состояние элемента, установите биты элемента stateMask, чтобы указать биты, которые будут возвращены в элементе state.
Если вы устанавливаете состояние элемента, установите биты элемента stateMask, чтобы указать биты элемента state, которые вы хотите установить.
Text Указатель на строку с нулём в конце, содержащую текст элемента.
TextMax Размер буфера, на который указывает элемент Text, в символах.
Image Индекс в списке изображений значка, используемого, когда элемент находится в невыбранном состоянии.
SelectedImage Индекс в списке изображений значка, используемого, когда элемент находится в выбранном состоянии.
Children Флаг, указывающий, есть ли у элемента связанные дочерние элементы. Этот элемент может быть одним из следующих значений:
0 – у элемента нет дочерних элементов
1 – у элемента есть один или несколько дочерних элементов
Param Значение, связываемое с элементом.
Integral Высота элемента.
uStateEx Один или несколько (как побитовая комбинация) из следующих расширенных состояний.
Значение Смысл:
TVIS_EX_DISABLED Windows Vista или более поздней версии. Создаёт элемент управления, который отрисовывается серым цветом, с которым пользователь не может взаимодействовать.
TVIS_EX_FLAT Создаёт плоский элемент – элемент является виртуальным и не видим в дереве; вместо этого его дочерние элементы занимают его место в иерархии дерева.
TVIS_EX_HWND Создаёт отдельный HWND для элемента.
hwnd Не используется; должно быть NULL.
iExpandedImage Индекс изображения в списке изображений элемента управления для отображения, когда элемент находится в развёрнутом состоянии.
iReserved Зарезервированный элемент. Не используется.
