Справочник функций

_GUICtrlEdit_Scroll

Прокручивает текст Edit по вертикали в указанном направлении.

#include <GuiEdit.au3>
_GUICtrlEdit_Scroll ( $hWnd, $iDirection )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iDirection Может принимать одно из значений:
$SB_LINEDOWN – прокрутка на строку вниз
$SB_LINEUP – прокрутка на строку вверх
$SB_PAGEDOWN – прокрутка на страницу вниз
$SB_PAGEUP – прокрутка на страницу вверх
$SB_SCROLLCARET – прокрутка к курсору.

Возвращаемое значение

Успех: Старшее слово возвращаемого значения равно 1.
Младшее слово – количество строк, на которое прокручен текст.
Ошибка: Возвращает 0.

Примечания

$SB_xxxxx требуют #include <ScrollBarsConstants.au3>

См. также

_GUICtrlEdit_LineScroll

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <MsgBoxConstants.au3>
#include <ScrollBarsConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $idEdit, $sText, $iLen

    ; Создать GUI
    GUICreate("Edit Scroll", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268, BitOR($ES_WANTRETURN, $WS_VSCROLL))
    GUISetState(@SW_SHOW)

    ; Установить текст
    For $i = 1 To 50
        $sText &= $i & @CRLF
    Next
    _GUICtrlEdit_SetText($idEdit, $sText)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Line Down")
    _GUICtrlEdit_Scroll($idEdit, $SB_LINEDOWN)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Line Up")
    _GUICtrlEdit_Scroll($idEdit, $SB_LINEUP)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Page Down")
    _GUICtrlEdit_Scroll($idEdit, $SB_PAGEDOWN)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Page Up")
    _GUICtrlEdit_Scroll($idEdit, $SB_PAGEUP)

    $iLen = _GUICtrlEdit_GetTextLen($idEdit)
    _GUICtrlEdit_SetSel($idEdit, $iLen, $iLen)

    MsgBox($MB_SYSTEMMODAL, "Information", "Scroll Caret")
    _GUICtrlEdit_Scroll($idEdit, $SB_SCROLLCARET)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
