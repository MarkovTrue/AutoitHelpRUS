Справочник функций

_WinAPI_CreateEllipticRgn

создаёт эллиптическую область.

#include <WinAPIGdi.au3>
_WinAPI_CreateEllipticRgn ( $tRECT )

Параметры

$tRECT Структура $tagRECT с координатами левого верхнего и правого нижнего углов прямоугольника, описывающего эллипс, в логических единицах.

Возвращаемое значение

Успех: Возвращает дескриптор региона.
Ошибка: Возвращает 0.

Примечания

Ограничивающий прямоугольник задаёт размер, форму и ориентацию региона: длинные стороны прямоугольника определяют длину большой оси эллипса, короткие – малой; центр прямоугольника – точку пересечения осей.

Когда объект HRGN больше не нужен, вызовите функцию _WinAPI_DeleteObject() для его удаления.
