Справочник ключевых слов

$tagNMHDDISPINFO

Содержит сведения для обработки уведомляющих сообщений $HDN_GETDISPINFO.

Global Const $tagNMHDDISPINFO = "hwnd WndFrom;int IDFrom;int Code;int Item;int Mask;ptr Text;int TextMax;int Image;int lParam"

Параметры

WndFrom Дескриптор окна элемента управления, отправляющего сообщение.
IDFrom Идентификатор элемента управления, отправляющего сообщение.
Code Код уведомления.
Item Индекс элемента в элементе управления заголовком с отсчётом от нуля.
Mask Набор битовых флагов, определяющих, какие элементы структуры должны быть заполнены владельцем элемента управления. Это значение может быть комбинацией следующих значений:
$HDI_TEXT – поле Text должно быть заполнено
$HDI_IMAGE – поле Image должно быть заполнено
$HDI_LPARAM – поле lParam должно быть заполнено
$HDI_DI_SETITEM – возвращаемое значение. Указывает, что элемент управления должен сохранить информацию об элементе и не запрашивать её снова.
Text Указатель на строку с нулём в конце, содержащую текст, который будет отображаться для элемента заголовка.
TextMax Размер буфера, на который указывает Text.
Image Индекс изображения в списке изображений с отсчётом от нуля. Указанное изображение будет отображаться вместе с элементом заголовка, но оно не займёт место растрового изображения элемента. Если iImage установлен в $I_IMAGECALLBACK, элемент управления запрашивает информацию об изображении для этого элемента с помощью сообщения уведомления $HDN_GETDISPINFO.
lParam Определённое приложением значение, связываемое с элементом.
