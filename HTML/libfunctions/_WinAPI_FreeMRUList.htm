_WinAPI_FreeMRUList

Освобождает дескриптор списка недавних элементов (MRU) и записывает кэшированные данные в реестр.

#include <WinAPIReg.au3>
_WinAPI_FreeMRUList ( $hMRU )

Параметры

$hMRU Дескриптор освобождаемого списка MRU.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_WinAPI_AddMRUString, _WinAPI_CreateMRUList
