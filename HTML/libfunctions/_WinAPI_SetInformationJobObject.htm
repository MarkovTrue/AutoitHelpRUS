_WinAPI_SetInformationJobObject

Задаёт ограничения для объекта задания.

#include <WinAPIProc.au3>
_WinAPI_SetInformationJobObject ( $hJob, $iJobObjectInfoClass, $tJobObjectInfo )

Параметры

$hJob Дескриптор задания, для которого задаются ограничения. Дескриптор должен иметь право доступа
$JOB_OBJECT_SET_ATTRIBUTES.
$iJobObjectInfoClass Класс информации для задаваемых ограничений. Этот параметр указывает тип
структуры $tJobObjectInfo, допустимые значения:
2 - $tagJOBOBJECT_BASIC_LIMIT_INFORMATION
4 - $tagJOBOBJECT_BASIC_UI_RESTRICTIONS
5 - $tagJOBOBJECT_SECURITY_LIMIT_INFORMATION
6 - $tagJOBOBJECT_END_OF_JOB_TIME_INFORMATION
7 - $tagJOBOBJECT_ASSOCIATE_COMPLETION_PORT
9 - $tagJOBOBJECT_EXTENDED_LIMIT_INFORMATION
11 - $tagJOBOBJECT_GROUP_INFORMATION
$tJobObjectInfo Структура $tagJOBOBJECT_*, которая задаёт информацию об ограничениях и состоянии задания.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
