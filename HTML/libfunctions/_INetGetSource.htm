Справочник функций

_INetGetSource

Получает исходный код страницы по URL без создания временного файла.

#include <Inet.au3>
_INetGetSource ( $sURL [, $bString = True] )

Параметры

$sURL (URL сайта.) например 'http://www.autoitscript.com'.
$bString [необязательный] Если True, данные возвращаются в виде строки, в противном случае в двоичном формате.

Возвращаемое значение

Успех: Прочитанная строка, @extended содержит количество возвращённых байт.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.

См. также

_IEDocReadHTML

Пример

#include <Inet.au3>

ConsoleWrite(_INetGetSource('http://www.autoitscript.com'))
