Справочник функций

_WinAPI_RegCreateKey

Создаёт указанный ключ реестра.

#include <WinAPIReg.au3>
_WinAPI_RegCreateKey ( $vKey [, $sSubKey = '' [, $iAccess = $KEY_ALL_ACCESS [, $iOptions = 0 [, $tSecurity = 0]]]] )

Параметры

$vKey Имя открываемого ключа (см. раздел Примечания).
или
Дескриптор открытого ключа реестра. Если ключ уже существует, функция его открывает.
Вызывающий процесс должен иметь доступ $KEY_CREATE_SUB_KEY к ключу.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey(),
или может быть одним из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sSubKey [необязательный] Имя подключа, который эта функция открывает или создаёт.
Указанный подключ должен быть подключом ключа, определённого параметром $vKey; он может быть углублен до 32 уровней в дереве реестра.
Если пустая строка (по умолчанию), функция возвращает новый дескриптор ключа, указанного в $vKey.
$iAccess [необязательный] Маска, определяющая права доступа к ключу.
Этот параметр может быть одним или несколькими из следующих значений:
$KEY_ALL_ACCESS
$KEY_CREATE_LINK
$KEY_CREATE_SUB_KEY
$KEY_ENUMERATE_SUB_KEYS
$KEY_EXECUTE
$KEY_NOTIFY
$KEY_QUERY_VALUE
$KEY_READ
$KEY_SET_VALUE
$KEY_WOW64_32KEY для создания в 32-битной области ключей
$KEY_WOW64_64KEY для создания в 64-битной области ключей
$KEY_WRITE
$iOptions [необязательный] Этот параметр может быть одним из следующих значений:
$REG_OPTION_BACKUP_RESTORE
$REG_OPTION_CREATE_LINK
$REG_OPTION_NON_VOLATILE (по умолчанию)
$REG_OPTION_VOLATILE
$tSecurity [необязательный] Структура $tagSECURITY_ATTRIBUTES, определяющая, может ли возвращённый дескриптор наследоваться дочерними процессами.
Если этот параметр равен 0 (по умолчанию), дескриптор не может быть унаследован.

Возвращаемое значение

Успех: Возвращает дескриптор открытого или созданного ключа, флаг @extended будет содержать одно из следующих значений расположения:
0 (False) – ключ существовал и был просто открыт без изменений.
1 (True) – ключ не существовал и был создан.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

$vKey может быть строкой, содержащей запрошенный подключ, в этом случае $sSubKey должна быть пуста

Приложение не может создать ключ, который является прямым потомком $HKEY_USERS или $HKEY_LOCAL_MACHINE.
Приложение может создавать подключи на нижних уровнях деревьев $HKEY_USERS или $HKEY_LOCAL_MACHINE.

Если ключ не является одним из предопределённых ключей реестра ($HKEY_*), вы должны вызвать функцию _WinAPI_RegCloseKey() после завершения использования дескриптора.

Пример

Пример 1

#RequireAdmin

#include <Debug.au3>
#include <WinAPIReg.au3>

_DebugSetup(Default, True)

Example()

Func Example()
    Local $sKey = 'SOFTWARE\Temp'

    ; должен запускаться с #RequireAdmin
    Local $sKeyroot = "HKEY_LOCAL_MACHINE"
;~ Local $sKeyroot = "HKEY_CURRENT_USER"

    _DebugReport("- Key Creation" & @TAB & $sKeyroot & "/" & $sKey & @CRLF & @CRLF)

    Local $hKey = _WinAPI_RegCreateKey($sKeyroot, $sKey)
    If $hKey Then
        _DebugReport("- Key successfully Created" & @CRLF)
        Local $iDeleteKey = _WinAPI_RegDeleteKey($hKey)
        If $iDeleteKey Then
            _DebugReport("- Key successfully Deleted" & @CRLF)
        Else
            _DebugReport("! Key cannot be deleted @error = " & @error & @CRLF)
        EndIf
        _WinAPI_RegCloseKey($hKey)
        $iDeleteKey = _WinAPI_RegDeleteKey($hKey)
        If $iDeleteKey Then
            _DebugReport("! Key Close unsuccessful" & @CRLF)
        Else
            _DebugReport("- Key already deleted" & @CRLF)
        EndIf
    Else
        _DebugReport("! Key cannot be created @error = " & @error & ' Extended code: ' & @extended & ' (0x' & Hex(@extended) & ')' & @CRLF)
    EndIf

EndFunc   ;==>Example
