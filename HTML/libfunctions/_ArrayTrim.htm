Справочник функций

_ArrayTrim

Удаляет заданное количество символов из всех элементов одномерного или двумерного массива.

#include <Array.au3>
_ArrayTrim ( ByRef $aArray, $iTrimNum [, $iDirection = 0 [, $iStart = 0 [, $iEnd = 0 [, $iSubItem = 0]]]] )

Параметры

$aArray Изменяемый массив.
$iTrimNum Количество удаляемых символов.
$iDirection [необязательный] Сторона обрезки:
0 – слева
1 – справа
$iStart [необязательный] Индекс элемента, с которого начинается обрезка.
$iEnd [необязательный] Индекс элемента, на котором заканчивается обрезка.
$iSubItem [необязательный] Столбец для обрезки.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $aArray не является массивом
2 – $aArray не является одномерным или двумерным массивом
3 – $iStart или $iEnd выходят за границы массива
4 – $iStart больше $iEnd
5 – $iSubItem выходит за границы массива

Пример

#include <Array.au3>

Local $aArray[5] = ["ab", "bc", "cd", "de", "ef"]

_ArrayDisplay($aArray, "1D array")
_ArrayTrim($aArray, 1, 1, 1, 3)
_ArrayDisplay($aArray, "Trimmed right 1 char from items 1-3")

Local $aArray[4][4], $iChr
For $i = 0 To 3
    For $j = 0 To 3
        $iChr = Random(97, 121, 1)
        $aArray[$i][$j] = Chr($iChr) & Chr($iChr + 1)
    Next
Next
_ArrayDisplay($aArray, "2D array")
_ArrayTrim($aArray, 1, 0, 1, 2, 3)
_ArrayDisplay($aArray, "Trimmed left 1 char from col 2-3 in row 1-2")
