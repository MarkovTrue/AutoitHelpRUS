Справочник функций

_SQLite_QueryReset

Сбрасывает запрос, основанный на _SQLite_Query().

#include <SQLite.au3>
_SQLite_QueryReset ( $hQuery )

Параметры

$hQuery Дескриптор запроса, генерируемый _SQLite_Query().

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове SQLite API "sqlite3_reset"
2 – вызов запрещён безопасным режимом

Примечания

Сброс незавершённого запроса позволяет повторно отсканировать полученный набор строк.

См. также

_SQLite_Query, _SQLite_QueryFinalize

Пример

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $hQuery, $aRow, $iSwitch
_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
_SQLite_Open()
_SQLite_Exec(-1, "CREATE TABLE tblTest (a,b,c);")
_SQLite_Exec(-1, "INSERT INTO tblTest VALUES ('1','1','1');" & _ ; Строка 1
        "INSERT INTO tblTest VALUES ('2','2','2');" & _ ; Строка 2
        "INSERT INTO tblTest VALUES ('3','3','3');") ; Строка 3
_SQLite_Query(-1, "SELECT RowID,* FROM tblTest;", $hQuery)
While _SQLite_FetchData($hQuery, $aRow) = $SQLITE_OK
    $iSwitch = MsgBox(($MB_YESNO + $MB_SYSTEMMODAL), "Row: " & $aRow[0], $aRow[1] & "," & $aRow[2] & "," & $aRow[3] & @CRLF & _
            "Continue Looping?")
    If $iSwitch = 6 Then ; Да
        If $aRow[0] = 3 Then _SQLite_QueryReset($hQuery)
    Else ; No
        _SQLite_QueryFinalize($hQuery)
        ExitLoop
    EndIf
WEnd
_SQLite_Close()
_SQLite_Shutdown()
