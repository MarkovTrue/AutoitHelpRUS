_GDIPlus_BitmapCreateFromScan0

Создаёт объект Bitmap на основе массива байтов с информацией о размере и формате.

#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromScan0 ( $iWidth, $iHeight [, $iPixelFormat = $GDIP_PXF32ARGB [, $iStride = 0 [, $pScan0 = 0]]] )

Параметры

$iWidth Ширина растрового изображения в пикселях.
$iHeight Высота растрового изображения в пикселях.
$iPixelFormat [необязательный] Формат данных пикселей.
$GDIP_PXF01INDEXED = 1 бит/пикс, индексированный
$GDIP_PXF04INDEXED = 4 бит/пикс, индексированный
$GDIP_PXF08INDEXED = 8 бит/пикс, индексированный
$GDIP_PXF16GRAYSCALE = 16 бит/пикс, оттенки серого
$GDIP_PXF16RGB555 = 16 бит/пикс; по 5 бит на каждый компонент RGB
$GDIP_PXF16RGB565 = 16 бит/пикс; 5 бит на красный, 6 на зелёный и 5 на синий
$GDIP_PXF16ARGB1555 = 16 бит/пикс; 1 бит на альфа и по 5 бит на каждый компонент RGB
$GDIP_PXF24RGB = 24 бит/пикс; по 8 бит на каждый компонент RGB
$GDIP_PXF32RGB = 32 бит/пикс; по 8 бит на каждый компонент RGB. Без альфа-компонента.
$GDIP_PXF32ARGB = 32 бит/пикс; по 8 бит на каждый компонент RGB и альфа
$GDIP_PXF32PARGB = 32 бит/пикс; по 8 бит на каждый компонент RGB и альфа, с предумножением.
$iStride [необязательный] Целое число, задающее смещение в байтах между началом одной строки развёртки и следующей.
$pScan0 [необязательный] Указатель на массив байтов с данными пикселей.

Возвращаемое значение

Успех: Возвращает дескриптор нового объекта Bitmap.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

Закончив работу с объектом, вызовите _GDIPlus_BitmapDispose() для освобождения его ресурсов.

См. также

_GDIPlus_ImageDispose

Дополнительно

Искать GdipCreateBitmapFromScan0 в библиотеке MSDN

Пример

Пример 1

#include <GDIPlus.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = 460, $iH = 100
    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0($iW, $iH) ;создать пустой bitmap
    Local $hBmpCtxt = _GDIPlus_ImageGetGraphicsContext($hBitmap) ;Получить графический контекст изображения.
    _GDIPlus_GraphicsSetSmoothingMode($hBmpCtxt, $GDIP_SMOOTHINGMODE_HIGHQUALITY)
    _GDIPlus_GraphicsClear($hBmpCtxt, 0xFFFFFFFF) ;Залить изображение белым цветом.
    _GDIPlus_GraphicsDrawString($hBmpCtxt, "AutoIt rulez!", 0, 0, "Comic Sans MS", 52) ;Нарисовать текст на изображении.
    Local $sFile = @TempDir & "\Test.jpg"
    _GDIPlus_ImageSaveToFile($hBitmap, $sFile) ;Сохранить изображение на диск.
    ;освободить ресурсы GDI+
    _GDIPlus_GraphicsDispose($hBmpCtxt)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_Shutdown()
    ShellExecute($sFile) ;Открыть изображение приложением по умолчанию.
EndFunc   ;==>Example

Пример 2

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    _GDIPlus_Startup()
    Local Const $iW = 120, $iH = 80

    Local $hGui = GUICreate("", $iW, $iH)
    GUISetState()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)

    Local $tPixel = DllStructCreate("uint[" & $iW * $iH & "];")
    Local $iOffset
    For $y = 0 To $iH - 1
        $iOffset = $y * $iW
        For $x = 0 To $iW - 1
            DllStructSetData($tPixel, 1, BitOR(0xFF000000, BitShift(Random(0, 255, 1), -16), BitShift(Random(0, 255, 1), -8), Random(0, 255, 1)), $iOffset + $x + 1)
        Next
    Next

    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0($iW, $iH, $GDIP_PXF32ARGB, $iW, $tPixel)
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap, 0, 0)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $GUI_EVENT_RESTORE
                ; Перерисовать Bitmap.
                _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap, 0, 0)
        EndSwitch
    WEnd

    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
