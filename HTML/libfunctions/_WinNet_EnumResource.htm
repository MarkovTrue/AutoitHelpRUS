Справочник функций

_WinNet_EnumResource

Продолжает перечисление сетевых ресурсов.

#include <WinNet.au3>
_WinNet_EnumResource ( $hEnum, ByRef $iCount, $pBuffer, ByRef $iBufSize )

Параметры

$hEnum Дескриптор, который идентифицирует экземпляр перечисления. Дескриптор возвращается функцией _WinNet_OpenEnum().
$iCount Количество запрошенных записей. Если запрошенное число равно –1, функция возвращает столько записей, сколько возможно. Если функция успешна, при возврате переменная содержит количество фактически прочитанных записей.
$pBuffer Указатель на буфер, который получает результаты перечисления. Результаты возвращаются как массив структур $tagNETRESOURCE. Буфер должен быть достаточно большим, чтобы содержать структуры плюс строки, на которые указывают их элементы. Буфер действителен до следующего вызова с использованием дескриптора, указанного в $hEnum. Порядок структур $tagNETRESOURCE в массиве непредсказуем.
$iBufSize Размер буфера в байтах. Если буфер слишком мал, чтобы получить даже одну запись, этот параметр получает требуемый размер буфера.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
