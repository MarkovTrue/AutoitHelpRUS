Справочник функций

_WinAPI_SetWinEventHook

Устанавливает функцию перехватчика событий для диапазона событий.

#include <WinAPISys.au3>
_WinAPI_SetWinEventHook ( $iEventMin, $iEventMax, $pEventProc [, $iPID = 0 [, $iThreadId = 0 [, $iFlags = $WINEVENT_OUTOFCONTEXT]]] )

Параметры

$iEventMin Самое низкое значение события в диапазоне событий ($EVENT_*), обрабатываемых функцией перехватчика.
$iEventMax Самое высокое значение события в диапазоне событий ($EVENT_*), обрабатываемых функцией перехватчика.
$pEventProc Адрес определённой приложением функции перехватчика, которую система вызывает в ответ на события, генерируемые доступным объектом.
$iPID [необязательный] Идентификатор процесса, из которого функция перехватчика получает события. Если этот параметр равен 0 (по умолчанию), функция перехватчика связана со всеми существующими процессами на текущем рабочем столе.
$iThreadId [необязательный] Идентификатор потока, из которого функция перехватчика получает события. Если этот параметр равен 0 (по умолчанию), функция перехватчика связана со всеми существующими потоками на текущем рабочем столе.
$iFlags [необязательный] Флаги, которые указывают место расположения функции перехватчика и пропускаемых событий. Следующие флаги действительны:     $WINEVENT_INCONTEXT     $WINEVENT_OUTOFCONTEXT (по умолчанию)     $WINEVENT_SKIPOWNPROCESS     $WINEVENT_SKIPOWNTHREAD Следующие отдельные флаги или комбинации флагов действительны:     $WINEVENT_INCONTEXT     $WINEVENT_OUTOFCONTEXT     $WINEVENT_INCONTEXT | $WINEVENT_SKIPOWNPROCESS     $WINEVENT_INCONTEXT | $WINEVENT_SKIPOWNTHREAD     $WINEVENT_OUTOFCONTEXT | $WINEVENT_SKIPOWNPROCESS     $WINEVENT_OUTOFCONTEXT | $WINEVENT_SKIPOWNTHREAD

Возвращаемое значение

Успех: Возвращает значение, которое идентифицирует этот экземпляр перехватчика событий.
Ошибка: Возвращает 0.

Примечания

Клиенты могут вызывать _WinAPI_SetWinEventHook() несколько раз, если они хотят зарегистрировать дополнительные функции перехватчика или прослушивать дополнительные события.

Пример

#include <GuiMenu.au3>
#include <SendMessage.au3>
#include <WinAPIGdi.au3>
#include <WinAPIMisc.au3>
#include <WinAPIProc.au3>
#include <WinAPISys.au3>
#include <WindowsNotifsConstants.au3>

Local $hEventProc = DllCallbackRegister('_EventProc', 'none', 'ptr;dword;hwnd;long;long;dword;dword')
Global $g_tRECT, $g_iIndex, $g_hMenu = 0

OnAutoItExitRegister('OnAutoItExit')

Local $hEventHook = _WinAPI_SetWinEventHook($EVENT_SYSTEM_MENUPOPUPSTART, $EVENT_SYSTEM_MENUPOPUPEND, DllCallbackGetPtr($hEventProc))

Run(@SystemDir & '\notepad.exe')

While 1
    Sleep(1000)
WEnd

Func OnAutoItExit()
    _WinAPI_UnhookWinEvent($hEventHook)
    DllCallbackFree($hEventProc)
EndFunc   ;==>OnAutoItExit

Func _EventProc($hEventHook, $iEvent, $hWnd, $iObjectID, $iChildID, $iThreadId, $iEventTime)
    #forceref $hEventHook, $iObjectID, $iChildID, $iThreadId, $iEventTime

    Switch $iEvent
        Case $EVENT_SYSTEM_MENUPOPUPSTART
            ; Добавить "View - Calculator"
            $g_hMenu = _SendMessage($hWnd, $MN_GETHMENU)
            If (_GUICtrlMenu_IsMenu($g_hMenu)) And (StringInStr(_GUICtrlMenu_GetItemText($g_hMenu, 0), 'Status Bar')) And (StringInStr(_WinAPI_GetWindowFileName($hWnd), 'notepad.exe')) Then
                $g_iIndex = _GUICtrlMenu_GetItemCount($g_hMenu)
                _GUICtrlMenu_InsertMenuItem($g_hMenu, $g_iIndex, 'Calculator' & @TAB & ':-)')
                $g_tRECT = _GUICtrlMenu_GetItemRectEx($hWnd, $g_hMenu, $g_iIndex)
            Else
                $g_hMenu = 0
            EndIf
        Case $EVENT_SYSTEM_MENUPOPUPEND
            If $g_hMenu Then
                _GUICtrlMenu_DeleteMenu($g_hMenu, $g_iIndex)
                Local $tPOINT = _WinAPI_GetMousePos()
                If _WinAPI_PtInRect($g_tRECT, $tPOINT) Then
                    Run(@SystemDir & '\calc.exe')
                EndIf
                $g_hMenu = 0
            EndIf
    EndSwitch
EndFunc   ;==>_EventProc
