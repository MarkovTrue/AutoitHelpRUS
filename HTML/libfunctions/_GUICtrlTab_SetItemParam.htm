_GUICtrlTab_SetItemParam

Устанавливает значение.

#include <GuiTab.au3>
_GUICtrlTab_SetItemParam ( $hWnd, $iIndex, $iParam )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iIndex Индекс элемента (нумерация с 0).
$iParam Данные, определяемые приложением.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

См. также

_GUICtrlTab_GetItemParam

Пример

Example Created with UDF

; == Пример создан с помощью UDF

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiTab.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Tab Get/Set Item Param (v" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $hTab = _GUICtrlTab_Create($hGUI, 2, 2, 446, 266)
    _MemoMsgBoxStatus() ; Создание статус-бара

    GUISetState(@SW_SHOW)

    ; Добавить вкладки
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 0")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 2")

    ; Получить/установить параметр вкладки 0
    _GUICtrlTab_SetItemParam($hTab, 0, 1234)
    _MemoMsgBox($MB_SYSTEMMODAL, "Information", "Tab 0 parameter: " & _GUICtrlTab_GetItemParam($hTab, 0))

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
