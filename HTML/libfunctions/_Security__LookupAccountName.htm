Справочник функций

_Security__LookupAccountName

Получает идентификатор безопасности (SID) для учётной записи и имя домена.

#include <Security.au3>
_Security__LookupAccountName ( $sAccount [, $sSystem = ""] )

Параметры

$sAccount Имя учётной записи.
Используйте полностью квалифицированную строку в формате domain_name\user_name, чтобы функция нашла учётную запись в нужном домене.
$sSystem [необязательный] Имя системы.
Это может быть имя удалённого компьютера.
Если строка пуста, трансляция имени учётной записи начинается на локальной системе.
Если имя не может быть разрешено на локальной системе, функция попытается разрешить имя с помощью контроллеров домена, доверенных локальной системе.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
$aAcct[0] – строка SID
$aAcct[1] – имя домена
$aAcct[2] – тип SID
Ошибка: Устанавливает @error ≠ 0.

Пример

#include <Security.au3>

Local $aArrayOfData = _Security__LookupAccountName(@UserName)

; Вывести возвращённые данные, если ошибки нет
If IsArray($aArrayOfData) Then
    ConsoleWrite("SID String = " & $aArrayOfData[0] & @CRLF)
    ConsoleWrite("Domain name = " & $aArrayOfData[1] & @CRLF)
    ConsoleWrite("SID type = " & _Security__SidTypeStr($aArrayOfData[2]) & @CRLF)
EndIf
