Справочник функций

_WinAPI_EnumResourceNames

Перечисляет ресурсы указанного типа внутри двоичного модуля.

#include <WinAPIRes.au3>
_WinAPI_EnumResourceNames ( $hModule, $sType )

Параметры

$hModule Дескриптор модуля, в котором выполняется поиск.
Также параметр может задавать имя загружаемого модуля – полный или относительный путь.
Если параметр равен 0 или пустой строке, используется модуль, создавший текущий процесс.
$sType Тип ресурса, для которого перечисляются имена.
Может быть строкой или целым значением, представляющим предопределённый тип ресурса.

Возвращаемое значение

Успех: Возвращает массив имён ресурсов указанного типа.
Ошибка: Устанавливает @error ≠ 0.

Дополнительно

Искать EnumResourceNames в библиотеке MSDN

Пример

#include <Array.au3>
#include <WinAPIRes.au3>

Local $aData = _WinAPI_EnumResourceNames(@SystemDir & '\shell32.dll', $RT_DIALOG)

_ArrayDisplay($aData, '_WinAPI_EnumResourceNames')
