Справочник функций

_WinAPI_GetDllDirectory

Получает специфичную для приложения часть пути поиска, используемую для поиска DLL для приложения.

#include <WinAPISys.au3>
_WinAPI_GetDllDirectory ( )

Возвращаемое значение

Успех: Возвращает путь поиска.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Дополнительно

Искать GetDllDirectory в библиотеке MSDN

Пример

#include <WinAPIRes.au3>
#include <WinAPISys.au3>

If Not _WinAPI_SetDllDirectory(@ScriptDir & '\Extras') Then
    Exit
EndIf

Local $hModule = _WinAPI_LoadLibraryEx('Resources.dll', $LOAD_LIBRARY_AS_DATAFILE)

ConsoleWrite('Path: ' & _WinAPI_GetDllDirectory() & @CRLF)
ConsoleWrite('Module: ' & $hModule & @CRLF)

_WinAPI_FreeLibrary($hModule)
