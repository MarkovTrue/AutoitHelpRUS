_WinAPI_UpdateLayeredWindowIndirect

Обновляет позицию, размер, форму, содержание и прозрачность многоуровневого окна.

#include <WinAPISysWin.au3>
_WinAPI_UpdateLayeredWindowIndirect ( $hWnd, $tULWINFO )

Параметры

$hWnd Дескриптор многоуровневого окна. Многоуровневое окно создаётся путём указания $WS_EX_LAYERED при его создании.
$tULWINFO Структура $tagUPDATELAYEREDWINDOWINFO, которая содержит информацию для окна.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Для лучшей производительности рисования многоуровневым окном и любыми базовыми окнами многоуровневое окно должно быть
как можно меньше.

Эта функция требует Windows Vista или более позднюю версию.
