Справочник функций

_DebugReportData

Записывает содержимое данных в сеанс отладки.

#include <Debug.au3>
_DebugReportData ( Const ByRef $vVar [, $sTitle = ""DebugReportData"." [, $sArrayRange = "" [, $iFlags = 0 [, $vUser_Separator = Default [, $sHeader = Default]]]]] )

Параметры

$vVar Переменная для записи в журнал.
$sTitle [необязательный] Заголовок отчёта. По умолчанию "DebugReportData".
$sArrayRange [необязательный] Диапазон отображаемых строк или столбцов. По умолчанию ("") отображается весь массив (подробности ниже).
$iFlags [необязательный] Параметры UDF. Складывайте нужные значения:
$ARRAYDISPLAY_COLALIGNLEFT (0) = (по умолчанию) выравнивание текста столбцов по левому краю
$ARRAYDISPLAY_TRANSPOSE (1) = транспонировать массив (только для двумерного массива)
$ARRAYDISPLAY_COLALIGNRIGHT (2) = выравнивание текста столбцов по правому краю
$ARRAYDISPLAY_NOROW (64) = столбец "Row" не отображается.
$vUser_Separator [необязательный] Задаёт способ отображения столбцов при копировании данных в буфер обмена.
Символ = разделитель между столбцами.
Число = фиксированная ширина столбца – более длинные значения обрезаются.
По умолчанию = текущий символ-разделитель (обычно "|").
$sHeader [необязательный] Имена столбцов в заголовке (строка имён, разделённых текущим символом-разделителем, обычно "|"). По умолчанию см. раздел Примечания.

Возвращаемое значение

Отсутствует. @error и @extended вызывающего кода сохраняются.

Примечания

Если _DebugSetup() не была вызвана, функция всегда возвращает управление.

Если функции передана переменная, не являющаяся массивом, она работает как _DebugReportVar().
Если передана строка и $sTitle = Default, она выводится как есть.
Это удобно, чтобы заменить отладочную строку ConsoleWrite() в SciTE на вывод в отчёт.

Если передан массив, вывод форматируется по столбцам, что нагляднее, чем у _DebugReportVar().
Все необязательные параметры, начиная с $sArrayRange, работают так же, как в _DebugArrayDisplay().
$ARRAYDISPLAY_COLALIGNCENTER не реализован.

Синтаксис параметра $sArrayRange:

"7" Строки 0-7, все столбцы
"7:" Строки 7-конец, все столбцы
"|7" Все строки, столбцы 0-7
"|7:" Все строки, столбцы 7-конец
"7|7" Строки 0-7, столбцы 0-7
"5:7" Строки 5-7, все столбцы
"|5:7" Все строки, столбцы 5-7
"7|5:7" Строки 0-7, столбцы 5-7
"5:7|7" Строки 5-7, столбцы 0-7
"5:7|5:7" Строки 5-7, столбцы 5-7

Для одномерных массивов значения столбцов игнорируются.

Имена из $sHeader (разделённые текущим символом-разделителем) применяются к стольким столбцам, сколько задано имён. Если пользовательские имена не заданы или их не хватает, подставляется заголовок по умолчанию "Row|Col0" для одномерного массива или "Row|Col0|Col1|...|Col n" для двумерного. При отображении массива транспонированным заголовок игнорируется.
Переменная $ARRAYDISPLAY_ROWPREFIX задаёт префикс нумерации строк (по умолчанию: #).

Элемент массива, содержащий другой массив или Map, отображается как {Array} или {Map}.

При использовании $ARRAYDISPLAY_TRANSPOSE или $sArrayRange создаётся копия $aArray, поэтому расходуется дополнительная память.

См. также

_DebugReportVar, _DebugArrayDisplay

Пример

#include <Debug.au3>

_DebugSetup("_DebugReportData examples", True)

Local $aArray2D[5][5]
For $r = 0 To UBound($aArray2D, 1) - 1
    For $c = 0 To UBound($aArray2D, 2) - 1
        $aArray2D[$r][$c] = $r & "," & $c
    Next
Next

_DebugReportData($aArray2D, "Array2D")
_DebugReportData($aArray2D, "Array2D NO ROW no header", "", $ARRAYDISPLAY_NOROW)
_DebugReportData($aArray2D, "Array2D no header Transposed", "", $ARRAYDISPLAY_TRANSPOSE)

_DebugReportData($aArray2D, "Array2D header col0", "", 0, Default, "zero")
_DebugReportData($aArray2D, "Array2D header col0 Transposed", "", $ARRAYDISPLAY_TRANSPOSE, Default, "zero")
_DebugReportData($aArray2D, "Array2D header col0 col1", "", 0, Default, "zero|one")
_DebugReportData($aArray2D, "Array2D header col0 col1 (|1)", "|1", 0, Default, "zero|one")

_DebugReportData($aArray2D, "Array2D header col0 col1 (|1:2)", "|1:2", 0, Default, "zero|one")
_DebugReportData($aArray2D, "Array2D header col0 col1 (|1:2)", "|1:2 Transposed", $ARRAYDISPLAY_TRANSPOSE, Default, "zero|one")
_DebugReportData($aArray2D, "Array2D NO ROW header col0 col1 (|1:2)", "|1:2", $ARRAYDISPLAY_NOROW, Default, "zero|one")
_DebugReportData($aArray2D, "Array2D NO ROW header col0 col1 (|1:2) Transposed", "|1:2", $ARRAYDISPLAY_NOROW + $ARRAYDISPLAY_TRANSPOSE, Default, "zero|one")
_DebugReportData($aArray2D, "Array2D header col0 col1 (2:3|1:2)", "2:3|1:2", 0, Default, "zero|one")

_DebugReportData($aArray2D, "Array2D header col0 col1 (2:3|1:2) Transposed", "2:3|1:2", $ARRAYDISPLAY_TRANSPOSE, Default, "zero|one")
_DebugReportData($aArray2D, "Array2D NO ROW header col0 col1 (2:3|1:2)", "2:3|1:2", $ARRAYDISPLAY_NOROW, Default, "zero|one")

Local $aArray[7] = [1, 1.1, "string", Binary(0x010203), Ptr(-1), False, Default]
_DebugReportData($aArray, "Array")

Local $aArray3D[5][2][3]
For $r = 0 To UBound($aArray3D, 1) - 1
    For $c = 0 To UBound($aArray3D, 2) - 1
        For $k = 0 To UBound($aArray3D, 3) - 1
            $aArray3D[$r][$c][$k] = $r & "," & $c & "," & $k
        Next
    Next
Next

_DebugReportData($aArray3D, "$Array3D")

Local $aArray4D[5][2][2][10]
_DebugReportData($aArray4D, "$Array4D")

Local $iInt = -1
_DebugReportData($iInt, "$iInt")

Local $iInt64 = 2 ^ 63
_DebugReportData($iInt64, "$iInt64")

Local $bBool = True
_DebugReportData($bBool, "$bBool")

Local $fFloat = 1.1
_DebugReportData($fFloat, "$fFloat")

Local $vKeyword = Default
_DebugReportData($vKeyword, "$vKeyword")

Local $sString = "stringstring"
_DebugReportData($sString, "$sString")

Local $dBinary = Binary("0x0102030405060708")
_DebugReportData($dBinary, "$dbinary")

$dBinary = Binary("abcdefghij")
_DebugReportData($dBinary, "$dbinary")

Local $pPtr = Ptr(0)
_DebugReportData($pPtr, "$pPtr")

Local $hWnd = WinActive("", "")
_DebugReportData($hWnd, "$hWnd")

Local $tDllstruct = DllStructCreate("int; ptr")
_DebugReportData($tDllstruct, "$tDllstruct")

Local $oObj = ObjCreate("shell.application")
_DebugReportData($oObj, "$oObj")

_DebugReport(@CRLF & "=== ConsoleWrite() can be replaced by _DebugReportData() when using Debug lines ===" & @CRLF)
; _DebugReportData() можно использовать в Debug Line вместо ConsoleWrite(); _DebugSetup() нужно добавить заранее
Local $iRet = 1
;~ ConsoleWrite('@@ Debug(' & @ScriptLineNumber & ') : $iRet = ' & $iRet & @CRLF & '>Error code: ' & @error & ' Extended code: ' & @extended & ' (0x' & Hex(@extended) & ')' & @CRLF) ;### Debug Console
_DebugReportData('@@ Debug(' & @ScriptLineNumber & ') : $iRet = ' & $iRet & @CRLF & '>Error code: ' & @error & ' Extended code: ' & @extended & ' (0x' & Hex(@extended) & ')' & @CRLF) ;### Отладочная консоль

; Для одномерных или двумерных массивов лучше использовать _DebugReportData() напрямую – вывод будет отформатирован по столбцам.
ReDim $aArray2D[5][2]
For $r = 0 To UBound($aArray2D, 1) - 1
    For $c = 0 To UBound($aArray2D, 2) - 1
        $aArray2D[$r][$c] = $r & "," & $c
    Next
Next

_DebugReportData($aArray2D, "$Array2D display by columns")
_DebugReportVar("$Array2D by _DebugReportVar()", $aArray2D)
