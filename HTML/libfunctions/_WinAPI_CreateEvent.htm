Справочник функций

_WinAPI_CreateEvent

создаёт или открывает именованный или безымянный объект события.

#include <WinAPIProc.au3>
_WinAPI_CreateEvent ( [$tAttributes = 0 [, $bManualReset = True [, $bInitialState = True [, $sName = ""]]]] )

Параметры

$tAttributes Дескриптор.
$bManualReset Необязательный параметр.
$bInitialState Необязательный параметр.
$sName Дескриптор.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: 0, @extended устанавливается в значение _WinAPI_GetLastError()

Примечания

    Если объект-событие с таким именем уже существовал до вызова, функция возвращает 0, а @extended получает значение ERROR_ALREADY_EXISTS.
