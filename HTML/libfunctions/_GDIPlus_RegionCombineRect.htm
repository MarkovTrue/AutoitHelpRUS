_GDIPlus_RegionCombineRect

Обновляет регион в соответствии с результатом операции над собой и внутренней областью указанного прямоугольника.

#include <GDIPlus.au3>
_GDIPlus_RegionCombineRect ( $hRegion, $nX, $nY, $nWidth, $nHeight [, $iCombineMode = 2] )

Параметры

$hRegion Указатель на объект Region.
$nX Координата X верхнего левого угла прямоугольника.
$nY Координата Y верхнего левого угла прямоугольника.
$nWidth Ширина прямоугольника.
$nHeight Высота прямоугольника.
$iCombineMode [необязательный] Режим объединения, задающий способ сочетания региона с прямоугольником:
0 – существующий регион заменяется новым
1 – существующий регион заменяется пересечением себя и нового региона
2 – существующий регион заменяется объединением себя и нового региона
3 – существующий регион заменяется результатом XOR двух регионов.
Точка входит в XOR двух регионов, если она принадлежит одному из них, но не обоим сразу
4 – существующий регион заменяется той его частью, которая находится вне нового региона
5 – существующий регион заменяется той частью нового региона, которая находится вне существующего

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_RegionCombinePath, _GDIPlus_RegionCombineRegion

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <WinAPIGdi.au3>
#include <WinAPIHObj.au3>

Example()

Func Example()
    Local $hGUI, $hPath, $hFamily, $tLayout, $aSize, $hRegion, $hRGN

    _GDIPlus_Startup()

    $hGUI = GUICreate("GDI+", 620, 100)
    GUISetBkColor(0x80) ;Установить синий цвет фона окна
    GUISetState(@SW_SHOW)

    $hPath = _GDIPlus_PathCreate() ;Создать новый объект path

    $hFamily = _GDIPlus_FontFamilyCreate("Arial Black") ;Создать объект font family
    $tLayout = _GDIPlus_RectFCreate() ;Создать ограничивающий прямоугольник строки X=0, Y=0
    _GDIPlus_PathAddString($hPath, "AutoIt rulez!", $tLayout, $hFamily, 0, 85, 0) ;Добавить контур строки в path

    $aSize = WinGetPos($hGUI) ;Получить размер окна

    $hRegion = _GDIPlus_RegionCreateFromRect(0, 0, 110, $aSize[3]) ;Создать регион из прямоугольника
    _GDIPlus_RegionCombineRect($hRegion, $aSize[2] - 110, 0, 110, $aSize[3], 2) ;Объединить rectangle_left И rectangle_right
    _GDIPlus_RegionCombinePath($hRegion, $hPath, 3) ;Объединить rectangles XOR string

    $hRGN = _GDIPlus_RegionGetHRgn($hRegion, 0) ;Создать GDI-регион из GDI+ региона
    _WinAPI_SetWindowRgn($hGUI, $hRGN) ;Установить регион окна

    ;Цикл до выхода пользователя
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;Освободить ресурсы
    _GDIPlus_FontFamilyDispose($hFamily)
    _WinAPI_DeleteObject($hRGN)
    _GDIPlus_RegionDispose($hRegion)
    _GDIPlus_PathDispose($hPath)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
