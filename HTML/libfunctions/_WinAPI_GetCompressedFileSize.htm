_WinAPI_GetCompressedFileSize

Получает фактическое количество байт дискового хранилища, используемых для хранения указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetCompressedFileSize ( $sFilePath )

Параметры

$sFilePath Имя файла.

Возвращаемое значение

Успех: Возвращает фактическое количество байт дискового хранилища.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0. @extended содержит системную ошибку.

Примечания

Если файл находится не на томе, поддерживающем сжатие или разреженные файлы, или если файл не сжат или не является разреженным, полученное значение представляет собой фактический размер файла.
