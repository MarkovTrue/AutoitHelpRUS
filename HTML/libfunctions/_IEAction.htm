Справочник функций

_IEAction

Выполнить набор простых действий в браузере.

#include <IE.au3>
_IEAction ( ByRef $oObject, $sAction )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$sAction Выбор действия (см. раздел Примечания).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 2 ($_IEStatus_COMError) – ошибка COM при обращении к объекту
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – недопустимое значение
@extended: Содержит номер недопустимого параметра.

Примечания

Действие Описание
"back" Навигация на один элемент назад в истории.
"blur" Убирает фокус с элемента, но не передаёт его следующему в порядке табуляции.
"click" Имитирует щелчок на элементе.
"copy" Копирует текущее выделение в буфер обмена.
"cut" Копирует текущее выделение в буфер обмена и удаляет его.
"delete" Удаляет текущее выделение.
"disable" Отключает элемент.
"enable" Включает элемент.
"focus" Передаёт фокус элементу.
"forward" Навигация на один элемент вперёд в истории.
"home" Переход на домашнюю или стартовую страницу.
"invisible" Скрывает объект.
"paste" Вставляет содержимое буфера обмена вместо текущего выделения.
"print" Открывает диалог печати для печати текущей страницы.
"printdefault" Печать на принтер по умолчанию (без диалога)
"quit" Закрывает объект.
"refresh" Обновляет текущий документ.
"saveas" Открывает диалог сохранения для сохранения текущей веб-страницы.
"scrollintoview" Прокручивает документ так, чтобы элемент был видимым.
"search" Переход на текущую страницу поиска.
"selectall" Выбирает весь документ.
"stop" Отменяет любые ожидающие навигацию или загрузку, останавливает динамические элементы страницы (звуки, анимацию).
"unselect" Снимает текущее выделение.
"visible" Делает объект видимым.

См. также

_IELoadWait, _IEPropertySet

Пример

Пример 1

; Открыть браузер с примером "form", получить ссылку
; на кнопку отправки по имени и "щёлкнуть" её. Этот метод
; отправки форм полезен, потому что многие формы используют JavaScript
; код и события "onClick" на кнопке отправки, что делает _IEFormSubmit()
; работающей не так, как ожидается

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oSubmit = _IEGetObjByName($oIE, "submitExample")
_IEAction($oSubmit, "click")
_IELoadWait($oIE)

Пример 2

; Тo же, что Пример 1, но вместо щелчка передай фокус элементу
; и используй ControlSend для отправки Enter. Используй этот метод, когда
; браузерный скрипт, связанный с действием щелчка, не позволяет вернуть
; управление автоматически в твой код.

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oSubmit = _IEGetObjByName($oIE, "submitExample")
Local $hWnd = _IEPropertyGet($oIE, "hwnd")
_IEAction($oSubmit, "focus")
ControlSend($hWnd, "", "[CLASS:Internet Explorer_Server; INSTANCE:1]", "{Enter}")

; Дождаться окна Alert, затем щёлкнуть OK
WinWait("", "ExampleFormSubmitted")
Sleep(2000)

ControlClick("", "ExampleFormSubmitted", "[CLASS:Button; TEXT:OK; Instance:1;]")
Sleep(2000)

_IEQuit($oIE)
