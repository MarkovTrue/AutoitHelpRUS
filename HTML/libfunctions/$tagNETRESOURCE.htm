Справочник ключевых слов

$tagNETRESOURCE

Структура сетевого ресурса.

Global Const $tagNETRESOURCE = "int Scope;int Type;int DisplayType;int Usage;ptr LocalName;ptr RemoteName;ptr Comment;ptr Provider"

Параметры

Scope Область перечисления:
$RESOURCE_CONNECTED – перечислить текущие подключённые ресурсы
$RESOURCE_GLOBALNET – перечислить все ресурсы в сети
$RESOURCE_REMEMBERED – перечислить запомненные подключения.
Type Набор битовых флагов, определяющих тип ресурса:
$RESOURCETYPE_ANY – все ресурсы
$RESOURCETYPE_DISK – дисковые ресурсы
$RESOURCETYPE_PRINT – ресурсы печати.
DisplayType Параметры отображения объекта сети в пользовательском интерфейсе просмотра сети:
$RESOURCEDISPLAYTYPE_DOMAIN – объект должен отображаться как домен
$RESOURCEDISPLAYTYPE_SERVER – объект должен отображаться как сервер
$RESOURCEDISPLAYTYPE_SHARE – объект должен отображаться как общая папка
$RESOURCEDISPLAYTYPE_GENERIC – способ отображения объекта не имеет значения.
Usage Набор битовых флагов, описывающих способ использования ресурса. Заметьте, что этот элемент может быть указан только если элемент Scope равен $RESOURCE_GLOBALNET. Этот элемент может быть одним из следующих значений:
$RESOURCEUSAGE_CONNECTABLE – ресурс является подключаемым ресурсом; имя, на которое указывает RemoteName, может быть передано функции _WinNet_AddConnection() для установления сетевого подключения.
$RESOURCEUSAGE_CONTAINER – ресурс является ресурсом-контейнером; имя, на которое указывает RemoteName, может быть передано функции _WinNet_OpenEnum() для перечисления ресурсов в контейнере.
LocalName Если элемент Scope равен $RESOURCE_CONNECTED или $RESOURCE_REMEMBERED, этот элемент является указателем на строку символов с нулём в конце, которая задаёт имя локального устройства.
Этот элемент равен 0, если подключение не использует устройство.
RemoteName Если запись является сетевым ресурсом, этот элемент является указателем на строку с нулём в конце, которая задаёт имя удалённой сети. Если запись является текущим или постоянным подключением, RemoteName указывает на имя сети, связанное с именем, на которое указывает элемент LocalName.
Comment Указатель на строку с нулём в конце, содержащую комментарий, предоставленный поставщиком сетевых услуг.
Provider Указатель на строку с нулём в конце, содержащую имя поставщика, владеющего ресурсом. Этот элемент может быть NULL, если имя поставщика неизвестно. Чтобы получить имя поставщика, вы можете вызвать функцию _WinNet_GetProviderName().
