Справочник ключевых слов

$tagNMLVFINDITEM

Содержит сведения, нужные владельцу для поиска элементов, запрошенных виртуальным элементом ListView.

Global Const $tagNMLVFINDITEM = $tagNMHDR & ";int Start;" & $tagLVFINDINFO

Параметры

$tagNMHDR Содержит сведения о сообщении-уведомлении.
Start Индекс элемента, с которого начнётся поиск.
$tagLVFINDINFO Flags - тип поиска. Это поле может содержать одно или несколько из следующих значений:
$LVFI_PARAM - ищет совпадение между полем Param этой структуры и полем Param элемента.
    Если задан $LVFI_PARAM, все остальные флаги игнорируются.
$LVFI_PARTIAL - проверяет, начинается ли текст элемента со строки, на которую указывает поле Text.
    Это значение подразумевает использование $LVFI_STRING.
$LVFI_STRING - поиск по тексту элемента.
    Если не заданы дополнительные значения, текст найденного элемента должен точно совпадать со строкой, на которую указывает поле Text.
$LVFI_WRAP - продолжает поиск с начала, если совпадение не найдено
LVFI_NEARESTXY - находит элемент, ближайший к позиции, заданной в полях X и Y, в направлении, заданном полем Direction.
    Этот флаг поддерживается только в режимах крупных и мелких значков.
Text - адрес строки для сравнения с текстом элемента. Действителен, если в поле Flags установлен $LVFI_STRING или $LVFI_PARTIAL.
Param - значение для сравнения с полем Param структуры $LVITEM элемента. Действителен только если в поле flags установлен $LVFI_PARAM.
X - начальная позиция поиска по X. Действителен только если в поле Flags установлен $LVFI_NEARESTXY.
Y - начальная позиция поиска по Y. Действителен только если в поле Flags установлен $LVFI_NEARESTXY.
Direction - код виртуальной клавиши, задающий направление поиска. Поддерживаются следующие коды:
VK_LEFT
VK_RIGHT
VK_UP
VK_DOWN
VK_HOME
VK_END
VK_PRIOR
VK_NEXT
Это поле действительно только если в поле flags установлен $LVFI_NEARESTXY.

Примечания

Это уведомление даёт приложению (или получателю уведомления) возможность настроить пошаговый поиск.
Например, если искомые элементы числовые, приложение может выполнять числовой поиск вместо строкового.
Приложение записывает в поле Param результат поиска или другое определяемое приложением значение, чтобы завершить поиск неудачей и указать элементу управления, как действовать дальше
