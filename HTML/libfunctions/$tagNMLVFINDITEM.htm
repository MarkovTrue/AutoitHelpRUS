Справочник ключевых слов

$tagNMLVFINDITEM

Содержит сведения, необходимые владельцу для поиска элементов виртуального ListView.

Global Const $tagNMLVFINDITEM = $tagNMHDR & ";int Start;" & $tagLVFINDINFO

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Start Индекс элемента, с которого начнётся поиск.
$tagLVFINDINFO Флаги – тип поиска для выполнения. Этот член может быть установлен на одно или несколько из следующих значений:
$LVFI_PARAM – ищет совпадение между членом Param этой структуры и членом Param элемента.
Если указан $LVFI_PARAM, все остальные флаги игнорируются.
$LVFI_PARTIAL – проверяет, начинается ли текст элемента со строки, на которую указывает член Text.
Это значение подразумевает использование $LVFI_STRING.
$LVFI_STRING – ищет на основе текста элемента.
Если не указаны дополнительные значения, текст элемента найденного элемента должен точно совпадать со строкой, на которую указывает член Text.
$LVFI_WRAP – продолжает поиск с начала, если совпадение не найдено
LVFI_NEARESTXY – находит элемент, ближайший к позиции, указанной в членах X и Y, в направлении, указанном членом Direction.
Этот флаг поддерживается только режимами больших и малых значков.
Text – адрес строки для сравнения с текстом элемента. Действителен, если в члене Flags установлен $LVFI_STRING или $LVFI_PARTIAL.
Param – значение для сравнения с членом Param структуры $LVITEM элемента. Действителен только, если в члене flags установлен $LVFI_PARAM.
X – начальная позиция поиска X. Действителен только, если в члене Flags установлен $LVFI_NEARESTXY.
Y – начальная позиция поиска Y. Действителен только, если в члене Flags установлен $LVFI_NEARESTXY.
Direction – виртуальный код клавиши, определяющий направление поиска. Поддерживаются следующие коды:
VK_LEFT
VK_RIGHT
VK_UP
VK_DOWN
VK_HOME
VK_END
VK_PRIOR
VK_NEXT
Этот член действителен только, если в члене flags установлен $LVFI_NEARESTXY.

Примечания

Это уведомление даёт приложению (или получателю уведомления) возможность настроить добавочный поиск.
Например, если элементы поиска являются числовыми, приложение может выполнить численный поиск вместо поиска по строкам.
Приложение устанавливает член Param на результат поиска или на другое определяемое приложением значение для провала поиска и указания элементу управления способ продолжения
