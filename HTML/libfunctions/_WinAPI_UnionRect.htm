_WinAPI_UnionRect

Создаёт объединение двух прямоугольников.

#include <WinAPIGdi.au3>
_WinAPI_UnionRect ( $tRECT1, $tRECT2 )

Параметры

$tRECT1 Структура $tagRECT, которая содержит первый исходный прямоугольник.
$tRECT2 Структура $tagRECT, которая содержит второй исходный прямоугольник.

Возвращаемое значение

Успех: Возвращает структура $tagRECT, которая содержит объединение прямоугольников $tRECT1 и $tRECT2.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Объединение – это наименьший прямоугольник, который содержит оба исходных прямоугольника. Система игнорирует
размеры пустого прямоугольника, то есть прямоугольника, в котором все координаты установлены на ноль, так что
он не имеет высоты или ширины.
