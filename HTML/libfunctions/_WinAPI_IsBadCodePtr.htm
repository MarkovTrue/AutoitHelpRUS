_WinAPI_IsBadCodePtr

Определяет, имеет ли вызывающий процесс доступ для чтения к памяти по указанному адресу.

#include <WinAPIMem.au3>
_WinAPI_IsBadCodePtr ( $pAddress )

Параметры

$pAddress Указатель на адрес памяти.

Возвращаемое значение

True: Возвращает вызывающий процесс не имеет доступа для чтения к указанной памяти.
False: вызывающий процесс имеет доступ для чтения, вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке.
