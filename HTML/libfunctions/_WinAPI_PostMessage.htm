Справочник функций

_WinAPI_PostMessage

Помещает сообщение в очередь сообщений и затем возвращает управление.

#include <WinAPISysWin.au3>
_WinAPI_PostMessage ( $hWnd, $iMsg, $wParam, $lParam )

Параметры

$hWnd Идентифицирует окно, процедура окна которого получит сообщение.
Если этот параметр равен 0xFFFF (HWND_BROADCAST), сообщение отправляется всем окнам верхнего уровня в системе, включая отключённые или невидимые окна, перекрывающиеся окна и всплывающие окна; но сообщение не отправляется дочерним окнам.
$iMsg Задаёт сообщение для отправки.
$wParam Первый параметр сообщения.
$lParam Второй параметр сообщения.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
