Справочник функций

_WinNet_GetNetworkInformation

Возвращает расширённую информацию о конкретном сетевом поставщике.

#include <WinNet.au3>
_WinNet_GetNetworkInformation ( $sName )

Параметры

$sName Сетевой поставщик, для которого требуется информация.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
$aInfo[0] – Номер версии программного обеспечения сетевого поставщика
$aInfo[1] – Текущее состояние программного обеспечения сетевого поставщика:
0 – Сеть работает
1 – Сеть недоступна
2 – Сеть в настоящее время не может обслуживать запросы
$aInfo[2] – Дескриптор экземпляра сетевого поставщика
$aInfo[3] – Старший разряд типа сети, уникального для работающей сети
$aInfo[4] – Набор битовых флагов, указывающих допустимые номера принтеров для переадресации локальных устройств принтера, с битом младшего порядка, соответствующим LPT1.
$aInfo[5] – Набор битовых флагов, указывающих допустимые локальные дисковые устройства, которые можно использовать для переадресации дисков, с битом младшего порядка, соответствующим A:.
Ошибка: Устанавливает @error ≠ 0.
