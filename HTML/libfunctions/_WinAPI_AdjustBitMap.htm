Справочник функций

_WinAPI_AdjustBitmap

Создаёт новое зависящее от устройства растровое изображение (DDB) из исходного растрового изображения с новыми размерами и коррекцией цвета.

#include <WinAPIGdi.au3>
_WinAPI_AdjustBitmap ( $hBitmap, $iWidth, $iHeight [, $iMode = $COLORONCOLOR [, $tAdjustment = 0]] )

Параметры

$hBitmap Дескриптор исходного растрового изображения.
$iWidth Ширина нового растрового изображения в пикселях.
Если этот параметр имеет значение (-1), ширина будет такой же, как в исходном растровом изображении.
$iHeight Высота нового растрового изображения в пикселях.
Если этот параметр имеет значение (-1), высота будет такой же, как в исходном растровом изображении.
$iMode [необязательный] Режим растяжения.
Этот параметр может быть одним из следующих значений:
$BLACKONWHITE
$COLORONCOLOR (по умолчанию)
$HALFTONE
$WHITEONBLACK
$STRETCH_ANDSCANS
$STRETCH_DELETESCANS
$STRETCH_HALFTONE
$STRETCH_ORSCANS
$tAdjustment [необязательный] Структура $tagCOLORADJUSTMENT, содержащая значения коррекции цвета.
Эта коррекция цвета используется только если установлен режим растяжения $HALFTONE ($STRETCH_HALFTONE).

Возвращаемое значение

Успех: Возвращает дескриптор вновь созданного растрового изображения (DDB).
Ошибка: Возвращает 0.

Примечания

Функция _WinAPI_AdjustBitmap() создаёт зависящие от устройства растровые изображения, совместимые с текущим экраном приложения.

Эта функция не поддерживает растровые изображения с альфа-каналом, вы можете использовать функцию _WinAPI_AlphaBlend() для работы с этими растровыми изображениями.

Когда вы закончите использовать растровое изображение, уничтожьте его с помощью функции _WinAPI_DeleteObject(). Функция не уничтожает исходное растровое изображение, вы должны уничтожить его самостоятельно.

См. также

_WinAPI_AlphaBlend, _WinAPI_DeleteObject

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIHObj.au3>
#include <WinAPIRes.au3>

; Загружает и изменяет размер (x2) изображения
Local $sBmp = _Extras_PathFull('AutoIt.bmp')
Local $hBitmap = _WinAPI_LoadImage(0, $sBmp, $IMAGE_BITMAP, 0, 0, $LR_LOADFROMFILE)
Local $tSIZE = _WinAPI_GetBitmapDimension($hBitmap)
Local $W = 2 * DllStructGetData($tSIZE, 'X')
Local $H = 2 * DllStructGetData($tSIZE, 'Y')
Local $hResize = _WinAPI_AdjustBitmap($hBitmap, $W, $H)
_WinAPI_DeleteObject($hBitmap)

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), $W, $H)
Local $idPic = GUICtrlCreatePic('', 0, 0, $W, $H)
Local $hPic = GUICtrlGetHandle($idPic)

; Установить bitmap для элемента управления
_SendMessage($hPic, $STM_SETIMAGE, 0, $hResize)
Local $hObj = _SendMessage($hPic, $STM_GETIMAGE)
If $hObj <> $hResize Then
    _WinAPI_DeleteObject($hResize)
EndIf

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
