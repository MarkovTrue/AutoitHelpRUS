_WinAPI_IsLoadKBLayout

Определяет, загружен ли указанный языковой стандарт ввода в систему.

#include <WinAPISys.au3>
_WinAPI_IsLoadKBLayout ( $iLanguage )

Параметры

$iLanguage Идентификатор языкового стандарта ввода для проверки.

Возвращаемое значение

True: Возвращает языковой стандарт ввода загружен.
False: Языковой стандарт ввода не загружен.
