_WinAPI_SetROP2

Получает режим смешивания переднего плана указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_SetROP2 ( $hDC, $iMode )

Параметры

$hDC Дескриптор контекста устройства.
$iMode Режим смешивания. Этот параметр может быть одним из следующих значений.
$R2_BLACK
$R2_COPYPEN
$R2_LAST
$R2_MASKNOTPEN
$R2_MASKPEN
$R2_MASKPENNOT
$R2_MERGENOTPEN
$R2_MERGEPEN
$R2_MERGEPENNOT
$R2_NOP
$R2_NOT
$R2_NOTCOPYPEN
$R2_NOTMASKPEN
$R2_NOTMERGEPEN
$R2_NOTXORPEN
$R2_WHITE
$R2_XORPEN

Возвращаемое значение

Успех: Возвращает значение, которое указывает предыдущий режим смешивания.
Ошибка: Возвращает 0.

Примечания

Режимы смешивания определяют, как GDI объединяет цвета источника и назначения при рисовании текущей ручкой. Режимы смешивания
являются кодами двоичных растровых операций, представляющими все возможные логические функции двух переменных, использующих двоичные
операции AND, OR и XOR (исключающее ИЛИ) и унарную операцию NOT. Режим смешивания предназначен только для растровых устройств;
он недоступен для векторных устройств.
