Справочник функций

_Excel_BookNew

Создаёт новую книгу.

#include <Excel.au3>
_Excel_BookNew ( $oExcel [, $iSheets = Default] )

Параметры

$oExcel Объект приложения Excel.
$iSheets [необязательный] Количество листов в новой книге (по умолчанию параметр Excel SheetsInNewWorkbook). Максимум 255.

Возвращаемое значение

Успех: Возвращает объект книги.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oExcel не является объектом
2 – $iSheets не является целым числом
3 – $iSheets вне допустимого диапазона (1-255)
4 – Excel вернул ошибку.
@extended: Содержит код COM-ошибки.

Примечания

Количество листов ограничено 255 независимо от параметра SheetsInNewWorkbook в настройках Excel.

См. также

_Excel_BookAttach, _Excel_BookClose, _Excel_BookOpen, _Excel_BookOpenText

Пример

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookNew Example 1", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Создаём новую книгу с тремя листами.
_Excel_BookNew($oExcel, 2)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookNew Example 1", "Error creating new workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_BookNew Example", "Workbook has been created successfully with only 2 worksheets.")
