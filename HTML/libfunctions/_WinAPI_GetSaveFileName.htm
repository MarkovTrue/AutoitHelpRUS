_WinAPI_GetSaveFileName

Создаёт диалоговое окно сохранения, которое позволяет пользователю указать диск, каталог и имя файла для сохранения.

#include <WinAPIDlg.au3>
_WinAPI_GetSaveFileName ( [$sTitle = "" [, $sFilter = "All files (*.*)" [, $sInitalDir = "." [, $sDefaultFile = "" [, $sDefaultExt = "" [, $iFilterIndex = 1 [, $iFlags = 0 [, $iFlagsEx = 0 [, $hWndOwner = 0]]]]]]]]] )

Параметры

$sTitle [необязательный] строка для размещения в заголовке диалогового окна
$sFilter [необязательный] пары строк фильтра (например "Text Files (*.txt)|All Files (*.*)")
Первая строка в каждой паре – это отображаемая строка, описывающая фильтр (например "Text Files")
Вторая строка задаёт шаблон фильтра (например "*.TXT")
Для указания нескольких шаблонов фильтра для одной отображаемой строки используйте точку с запятой для разделения шаблонов (например "*.TXT;*.DOC;*.BAK")
Строка шаблона может быть комбинацией допустимых символов имени файла и подстановочного символа звёздочки (*)
Не включайте пробелы в строку шаблона.
$sInitalDir [необязательный] строка, которая может задать начальный каталог
$sDefaultFile [необязательный] имя файла для инициализации элемента управления редактирования имени файла
$sDefaultExt [необязательный] строка, содержащая расширение по умолчанию
$iFilterIndex [необязательный] задаёт индекс текущего выбранного фильтра в элементе управления типов файлов
$iFlags [необязательный] см. флаги в сведениях $tagOPENFILENAME
$iFlagsEx [необязательный] см. FlagEx в сведениях $tagOPENFILENAME
$hWndOwner [необязательный] дескриптор окна, которому принадлежит диалоговое окно. Этот параметр может быть любым допустимым дескриптором окна или 0, если диалоговое окно не имеет владельца

Возвращаемое значение

Успех: Возвращает массив в следующем формате:
[0] – содержит количество строк (2)
[1] – содержит выбранный путь
[2] – содержит выбранный файл
Ошибка: устанавливает флаг @error в ненулевое значение, вызовите _WinAPI_CommDlgExtendedError() для получения расширенной информации об ошибке.

Примечания

Раскрывающееся окно диалога "Сохранить как тип" позволяет выбрать пользователю несколько фильтров, установленных в $sFilter – первый из них устанавливается по умолчанию.

Параметр $sDefaultExt имеет важное значение.
– Если этот параметр оставить пустым, то расширение файла НЕ добавляется к возвращённому имени файла, независимо от параметра в раскрывающемся окне диалога "Сохранить как тип", однако если пользователь добавляет расширение к имени файла в поле ввода диалога "Имя файла", то это расширение добавляется к результату.
– Но если этот параметр задан как строка, то расширение, отображаемое в раскрывающемся окне диалога "Сохранить как тип", автоматически добавляется.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <WinAPIConstants.au3>
#include <WinAPIDlg.au3>
#include <WindowsStylesConstants.au3>

_Example_Defaults()
_Example_ExplorerStyle()
_Example_OldStyle()
_Example_ExplorerStyle_NoPlaceBar()

Func _Example_Defaults()
    Local $hGui, $idBtn_Dialog, $aFile, $sError

    ; Создать GUI
    $hGui = GUICreate("GetSaveFileName use defaults", 400, 296)
    _MemoCreate(2, 32, 396, 226, $WS_HSCROLL)
    $idBtn_Dialog = GUICtrlCreateButton("Save Dialog", 155, 270, 90, 20)
    GUISetState(@SW_SHOW)

    While 1
        Switch GUIGetMsg()
            Case $idBtn_Dialog
                $aFile = _WinAPI_GetSaveFileName() ; использовать значения по умолчанию
                If @error Then
                    Local $iError = @error
                    Local $iExtended = @extended
                    $sError = _WinAPI_CommDlgExtendedError()
                    _MemoWrite("CommDlgExtendedError (" & $iError & "/" & $iExtended & "): " & $sError)
                Else
                    For $x = 1 To $aFile[0]
                        _MemoWrite($aFile[$x])
                    Next
                EndIf
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
    GUIDelete($hGui)
EndFunc   ;==>_Example_Defaults
