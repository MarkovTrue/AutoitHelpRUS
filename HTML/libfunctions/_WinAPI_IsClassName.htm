Справочник функций

_WinAPI_IsClassName

Оболочка для проверки имени класса элемента управления.

#include <WinAPISysWin.au3>
_WinAPI_IsClassName ( $hWnd, $sClassName )

Параметры

$hWnd Дескриптор элемента управления.
$sClassName Имя класса для проверки.

Возвращаемое значение

True: Возвращает $sClassName совпадает с именем класса, полученным из $hWnd.
False: $sClassName не совпадает с именем класса, полученным из $hWnd

Примечания

Используется для проверки того, что в функцию передан правильный $hWnd
