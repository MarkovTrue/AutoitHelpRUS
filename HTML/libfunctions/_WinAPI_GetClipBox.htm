_WinAPI_GetClipBox

Получает размеры ограничивающего прямоугольника видимой области.

#include <WinAPIGdi.au3>
_WinAPI_GetClipBox ( $hDC, ByRef $tRECT )

Параметры

$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT, создаваемая этой функцией и содержащая размеры прямоугольника в логических единицах.

Возвращаемое значение

Успех: Сложность области. Может принимать одно из значений:
$COMPLEXREGION
$NULLREGION
$SIMPLEREGION
Ошибка: Возвращает 0.
