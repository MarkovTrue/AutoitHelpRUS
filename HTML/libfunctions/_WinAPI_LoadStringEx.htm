Справочник функций

_WinAPI_LoadStringEx

Загружает ресурс строки для указанного языка из указанного модуля.

#include <WinAPIRes.au3>
_WinAPI_LoadStringEx ( $hModule, $iID [, $iLanguage = $LOCALE_USER_DEFAULT] )

Параметры

$hModule Дескриптор экземпляра модуля, исполняемый файл которого содержит ресурс строки.
Кроме того, этот параметр может задавать имя модуля для загрузки, он должен быть полным или относительным путём.
Если этот параметр равен 0 или пустой строке, это эквивалентно передаче дескриптора модуля,
использовавшегося при создании текущего процесса.
$iID Идентификатор загружаемой строки.
$iLanguage [необязательный] Идентификатор языка интересующего ресурса строки. Для извлечения строки на языке пользователя по умолчанию
установите этот параметр в $LOCALE_USER_DEFAULT (значение по умолчанию).

Возвращаемое значение

Успех: Возвращает строка.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.

Пример

#include "Extras\HelpFileInternals.au3"

#include <WinAPILocale.au3>
#include <WinAPIRes.au3>

Local $sDll = _Extras_PathFull('Resources.dll')
Local $hInstance = _WinAPI_LoadLibraryEx($sDll, $LOAD_LIBRARY_AS_DATAFILE)

; Почему ресурс для строки с ID = 6000 имеет номер 376? 6000 / 16 + 1 = 376
Local $aData = _WinAPI_EnumResourceLanguages($hInstance, $RT_STRING, 376)
If IsArray($aData) Then
    For $i = 1 To $aData[0]
        ConsoleWrite(StringFormat('%-10s - %s', _WinAPI_GetLocaleInfo($aData[$i], $LOCALE_SENGLANGUAGE), _WinAPI_LoadStringEx($hInstance, 6000, $aData[$i])) & @CRLF)
    Next
EndIf
