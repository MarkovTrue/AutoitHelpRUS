Справочник функций

_WinAPI_CommDlgExtendedErrorEx

Возвращает код ошибки стандартного диалога.

#include <WinAPIDlg.au3>
_WinAPI_CommDlgExtendedErrorEx ( )

Возвращаемое значение

Возвращает общие коды ошибок для любых функций стандартного диалога ($CDERR_*, $PDERR_*, $CFERR_*, $FNERR_* и $FRERR_*).

Примечания

Если функция стандартного диалога вернула False, потому что пользователь закрыл или отменил диалог, возвращаемое значение равно нулю.
