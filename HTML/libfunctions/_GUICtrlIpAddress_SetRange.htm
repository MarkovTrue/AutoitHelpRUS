Справочник функций

_GUICtrlIpAddress_SetRange

Задаёт допустимый диапазон значений для указанного поля элемента ввода IP-адреса.

#include <GuiIPAddress.au3>
_GUICtrlIpAddress_SetRange ( $hWnd, $iIndex [, $iLowRange = 0 [, $iHighRange = 255]] )

Параметры

$hWnd Дескриптор элемента управления.
$iIndex Индекс октета, для которого задаётся диапазон (отсчёт от 0).
$iLowRange [необязательный] Нижний предел диапазона
$iHighRange [необязательный] Верхний предел диапазона

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Пример

#include <GUIConstantsEx.au3>
#include <GuiIPAddress.au3>

Example()

Func Example()
    Local $hGui, $hIPAddress

    $hGui = GUICreate("IP Address Control Set Range Example", 300, 150)
    $hIPAddress = _GUICtrlIpAddress_Create($hGui, 10, 10)
    GUISetState(@SW_SHOW)

    _GUICtrlIpAddress_Set($hIPAddress, "24.168.2.128")

    ; установить диапазон на 1-е поле
    _GUICtrlIpAddress_SetRange($hIPAddress, 0, 198, 200)

    ; Ждать закрытия GUI пользователем
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
