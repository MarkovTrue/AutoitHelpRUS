_WinAPI_FlushViewOfFile

Записывает на диск диапазон байтов в отображённой проекции файла.

#include <WinAPIFiles.au3>
_WinAPI_FlushViewOfFile ( $pAddress [, $iBytes = 0] )

Параметры

$pAddress Указатель на базовый адрес диапазона байтов, который нужно сбросить в дисковое представление отображённого файла.
$iBytes [необязательный] Количество сбрасываемых байтов.
При 0 (по умолчанию) сбрасывается диапазон от базового адреса до конца отображения.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
