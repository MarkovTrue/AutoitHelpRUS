Справочник ключевых слов

$tagNMLVDISPINFO

Содержит сведения об уведомляющих сообщениях $LVN_GETDISPINFO и $LVN_SETDISPINFO.

Global Const $tagNMLVDISPINFO = $tagNMHDR & ";int Mask;int Item;int SubItem;int State;int StateMask;" & _
"ptr Text;int TextMax;int Image;int Param;int Indent;int GroupID;int Columns;ptr pColumns"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Mask Набор флагов, определяющих, какие члены этой структуры содержат данные для установки или какие члены запрашиваются. Этот член может иметь один или более следующих флагов:
$LVIF_COLUMNS – член Columns действителен
$LVIF_DI_SETITEM – операционная система должна сохранить запрошенную информацию элемента списка
$LVIF_GROUPID – член GroupID действителен
$LVIF_IMAGE – член Image действителен
$LVIF_INDENT – член Indent действителен
$LVIF_NORECOMPUTE – элемент управления не будет генерировать LVN_GETDISPINFO для получения текстовой информации
$LVIF_PARAM – член Param действителен
$LVIF_STATE – член State действителен
$LVIF_TEXT – член Text действителен.
Item 0-базовый индекс элемента, к которому относится эта структура
SubItem 1-базовый индекс подэлемента, к которому относится эта структура
State Указывает состояние элемента, изображение состояния и изображение наложения.
StateMask Значение, определяющее, какие биты члена State будут получены или изменены.
Text Указатель на строку, содержащую текст элемента.
TextMax Количество байтов в буфере, на который указывает Text, включая завершитель строки.
Image Индекс значка элемента в списке изображений элемента управления.
Param Значение, специфичное для элемента.
Indent Количество ширин изображения для выделения элемента.
GroupID Идентификатор группы представления плиток, которая получает элемент.
Columns Количество столбцов представления плиток для отображения этого элемента.
pColumns Указатель на массив индексов столбцов.
