Справочник функций

_IEFrameGetObjByName

Возвращает ссылку на Frame или iFrame по имени.

#include <IE.au3>
_IEFrameGetObjByName ( ByRef $oObject, $sName )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sName Имя Frame, который нужно найти.

Возвращаемое значение

Успех: Возвращает объектную переменную, ссылающуюся на объект Window внутри фрейма; @extended – количество фреймов.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
7 ($_IEStatus_NoMatch) – нет совпадения
@extended: Содержит номер недопустимого параметра.

Примечания

Объект, возвращаемый этой функцией, – это объект окна, дающий доступ к содержащемуся в нём документу.
Если нужен доступ к самому тегу frame или iframe и его свойствам, используйте _IETagNameGetCollection().

См. также

_IEFrameGetCollection

Пример

; Открыть пример с iFrame, получить ссылку на iFrame
; с именем "iFrameTwo" и заменить её HTML-содержимое

#include <IE.au3>

Local $oIE = _IE_Example("iframe")
Local $oFrame = _IEFrameGetObjByName($oIE, "iFrameTwo")
_IEBodyWriteHTML($oFrame, "Hello <b><font color=red>iFrame!</font></b>")
_IELoadWait($oFrame)
