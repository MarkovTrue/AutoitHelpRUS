Справочник функций

_WinAPI_CommandLineToArgv

Разбирает строку командной строки и возвращает массив аргументов командной строки.

#include <WinAPIShPath.au3>
_WinAPI_CommandLineToArgv ( $sCmd )

Параметры

$sCmd Строка, которая содержит полную командную строку. Если этот параметр – пустая строка, функция возвращает пустой массив (нулевой элемент равен 0).

Возвращаемое значение

Успех: Возвращает массив аргументов командной строки. Нулевой элемент массива содержит количество аргументов.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
