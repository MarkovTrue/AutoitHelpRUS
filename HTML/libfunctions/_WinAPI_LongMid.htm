_WinAPI_LongMid

Извлекает количество битов из значения DWORD (32-битное).

#include <WinAPIConv.au3>
_WinAPI_LongMid ( $iValue, $iStart, $iCount )

Параметры

$iValue 32-битное значение.
$iStart Позиция бита для начала. (0 – первый бит).
$iCount Количество битов для извлечения.

Возвращаемое значение

Возвращает значение, состоящее из указанных битов.

Примечания

Эта функция не проверяет параметры на корректные значения. Если значения некорректны, функция может вернуть
неожиданные результаты.

Пример

#include <WinAPIConv.au3>

Local $iValue = 0x00FA0000

ConsoleWrite('0x' & Hex(_WinAPI_LongMid($iValue, 16, 4)) & @CRLF)
ConsoleWrite('0x' & Hex(_WinAPI_LongMid($iValue, 20, 4)) & @CRLF)
