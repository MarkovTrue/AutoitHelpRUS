_WinAPI_WindowFromDC

Получает дескриптор окна, связанного с указанным контекстом устройства отображения (DC).

#include <WinAPIGdi.au3>
_WinAPI_WindowFromDC ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает дескриптор окна, связанного с указанным DC. Если нет окна, связанного с указанным
DC, возвращаемое значение равно 0.
Ошибка: Возвращает 0.
