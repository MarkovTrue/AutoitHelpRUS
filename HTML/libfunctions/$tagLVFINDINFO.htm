Справочник ключевых слов

$tagLVFINDINFO

Содержит сведения, используемые при поиске элемента ListView.

Global Const $tagLVFINDINFO = "struct; uint Flags;ptr Text;lparam Param;" & $tagPOINT & ";uint Direction; endstruct"

Параметры

Flags Тип выполняемого поиска. Может быть установлен на одно или несколько из следующих значений:
$LVFI_PARAM – ищет совпадение между членом Param этой структуры и членом Param элемента. Если указан $LVFI_PARAM, все остальные флаги игнорируются.
$LVFI_PARTIAL – проверяет, начинается ли текст элемента со строки, на которую указывает член Text. Это значение подразумевает использование $LVFI_STRING.
$LVFI_STRING – поиск по тексту элемента. Если не указаны дополнительные значения, текст совпадающего элемента должен точно совпадать со строкой, на которую указывает член Text.
$LVFI_SUBSTRING – эквивалент $LVFI_PARTIAL Windows Vista и позже.
$LVFI_WRAP – продолжает поиск с начала, если совпадение не найдено
$LVFI_NEARESTXY – находит элемент, ближайший к положению, указанному в членах X и Y, в направлении, указанном членом Direction. Этот флаг поддерживается только в режимах больших и малых значков.
Text Адрес строки для сравнения с текстом элемента. Действительна, если в члене Flags установлен $LVFI_STRING или $LVFI_PARTIAL.
Param Значение для сравнения с членом Param структуры $LVITEM элемента. Действительна только если в члене flags установлен $LVFI_PARAM.
$tagPOINT Начальная позиция поиска X,Y. Действительна только если в члене Flags установлен $LVFI_NEARESTXY.
Direction Код виртуальной клавиши, указывающий направление поиска. Поддерживаются следующие коды:
VK_LEFT
VK_RIGHT
VK_UP
VK_DOWN
VK_HOME
VK_END
VK_PRIOR
VK_NEXT
Этот член действителен только если в члене flags установлен $LVFI_NEARESTXY.
