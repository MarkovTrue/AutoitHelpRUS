Справочник функций

_FileListToArrayRec

Возвращает список файлов и\или папок по указанному пути с необязательной рекурсией до заданного уровня и сортировкой результата, с возможностью отбора по маске.(File globbing).

#include <File.au3>
_FileListToArrayRec ( $sFilePath [, $sMask = "*" [, $iReturn = $FLTAR_FILESFOLDERS [, $iRecur = $FLTAR_NORECUR [, $iSort = $FLTAR_NOSORT [, $iReturnPath = $FLTAR_RELPATH]]]]] )

Параметры

$sFilePath Начальный путь для построения списка.
Если путь оканчивается на \, папки возвращаются с завершающим \
Если длина пути больше 260 символов, добавьте перед путём префикс "\\?\" – на возвращаемые пути это не влияет.
$sMask [необязательный] Фильтр результата. Несколько фильтров разделяются ";"
Знаком "|" разделяются 3 набора фильтров: "Include|Exclude|Exclude_Folders"
Include = файлы/папки для включения (по умолчанию "*" [все])
Exclude = файлы/папки для исключения (по умолчанию "" [нет])
Exclude_Folders = используется только при $iRecur = 1 и $iReturn <> 2 для исключения заданных папок (по умолчанию "" [нет]).
$iReturn [необязательный] Определяет, возвращать ли файлы, папки или и то и другое, и какие атрибуты пропускать
$FLTAR_FILESFOLDERS (0) – (по умолчанию) возвращать файлы и папки
$FLTAR_FILES (1) – только файлы
$FLTAR_FOLDERS (2) – только папки
Прибавьте к $iReturn одно или несколько следующих значений, чтобы пропустить файлы/папки с этим атрибутом
+ $FLTAR_NOHIDDEN (4) – скрытые файлы и папки
+ $FLTAR_NOSYSTEM (8) – системные файлы и папки
+ $FLTAR_NOLINK (16) – папки-ссылки/соединения (junction).
$iRecur [необязательный] Определяет, искать ли рекурсивно в подпапках и до какого уровня
$FLTAR_NORECUR (0) – не искать в подпапках (по умолчанию)
$FLTAR_RECUR (1) – искать во всех подпапках (без ограничения глубины)
Отрицательное целое – искать в подпапках до указанной глубины.
$iSort [необязательный] Сортировать результат по алфавиту и глубине
$FLTAR_NOSORT (0) – без сортировки (по умолчанию)
$FLTAR_SORT (1) – сортировать
$FLTAR_FASTSORT (2) – сортировать быстрым алгоритмом (предполагает, что файлы в папке уже возвращаются отсортированными – требует NTFS и не гарантируется).
$iReturnPath [необязательный] Определяет отображаемый путь в результатах
$FLTAR_NOPATH (0) – только имя файла/папки
$FLTAR_RELPATH (1) – относительно начального пути (по умолчанию)
$FLTAR_FULLPATH (2) – полный путь.

Возвращаемое значение

Успех: Возвращает одномерный массив следующего вида:
$aArray[0] = количество найденных файлов/папок
$aArray[1] = 1-й файл/папка
$aArray[2] = 2-й файл/папка
...
$aArray[n] = n-й файл\папка
Ошибка: Возвращает пустую строку и устанавливает флаг @error в 1, @extended устанавливается так:
1 – путь не найден или недопустим
2 – недопустимый параметр Include
3 – недопустимый параметр Exclude
4 – недопустимый параметр Exclude_Folders
5 – недопустимый параметр $iReturn
6 – недопустимый параметр $iRecur
7 – недопустимый параметр $iSort
8 – недопустимый параметр $iReturnPath
9 – файлы/папки не найдены

Примечания

Если все возвращаемые файлы и/или папки находятся в одной папке и задаются одним простым фильтром, возможно, лучше подойдёт функция _FileListToArray(), поскольку она заметно быстрее.

Логика Include/Exclude/Exclude_Folders:
    Без рекурсии:
        Include/Exclude: файлы и/или папки
        Exclude_Folders: игнорируется
    С рекурсией:
        Include/Exclude:
            $iReturn = $FLTAR_FILESFOLDERS/$FLTAR_FILES – только файлы
            $iReturn = $FLTAR_FOLDERS – только папки
        Exclude_Folders:
            $iReturn = $FLTAR_FILESFOLDERS/$FLTAR_FILES – только папки
            $iReturn = $FLTAR_FOLDERS – игнорируется

Сортировка результата заметно увеличит время, за которое UDF вернёт массив.

Несмотря на название, эта UDF итеративная, а не рекурсивная. Константы определены в FileConstants.au3

Пример

#include <Array.au3>
#include <File.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $sAutoItDir = StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", Default, -1))
    If StringRight($sAutoItDir, 5) = "beta\" Then
        $sAutoItDir = StringTrimRight($sAutoItDir, 5)
    EndIf
    ConsoleWrite($sAutoItDir & @CRLF)

    ; Отсортированный список всех файлов и папок установки AutoIt.
    Local $aArray = _FileListToArrayRec($sAutoItDir, "*", $FLTAR_FILESFOLDERS, $FLTAR_RECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "Sorted tree")

    ; То же самое, но без папки "Include".
    $aArray = _FileListToArrayRec($sAutoItDir, "*||include", $FLTAR_FILESFOLDERS, $FLTAR_RECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "No 'Include' folder")

    ; Отсортированный список всех файлов в папке \AutoIt3, кроме .exe.
    $aArray = _FileListToArrayRec($sAutoItDir, "*|*.exe", $FLTAR_FILES, $FLTAR_NORECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "Non .EXE files")

    ; А вот те самые .exe-файлы, которые были исключены в прошлый раз.
    $aArray = _FileListToArrayRec($sAutoItDir, "*.exe", $FLTAR_FILES)
    _ArrayDisplay($aArray, ".EXE Files")

    ; Поиск по всему дереву папок: только .exe-файлы и папки, исключая папки, начинающиеся с I (Icons и Include).
    $aArray = _FileListToArrayRec($sAutoItDir, "*.exe||i*", $FLTAR_FILESFOLDERS, $FLTAR_RECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "Recur with filter")

    ; Поиск файлов иконок, исключая папку "Icons".
    $aArray = _FileListToArrayRec($sAutoItDir, "*.ico||ic*", $FLTAR_FILES, $FLTAR_RECUR, $FLTAR_SORT)
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "Ooops!", "No ico files found")
    Else
        _ArrayDisplay($aArray, "Icon files not in 'Icons' folder")
    EndIf

    ; Показывает, что фильтр применяется к файлам И папкам в нерекурсивном режиме.
    $aArray = _FileListToArrayRec($sAutoItDir, "*.exe", $FLTAR_FILESFOLDERS, $FLTAR_NORECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "Non-recur with filter")

    ; Фильтр применяется и к папкам при рекурсивном поиске папок.
    $aArray = _FileListToArrayRec($sAutoItDir, "Icons", $FLTAR_FOLDERS, $FLTAR_RECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "Folder recur with filter")

    ; Параметр exclude_folder игнорируется при поиске папок – "Icons" будет исключена, а "Include" останется.
    $aArray = _FileListToArrayRec($sAutoItDir, "*|ic*|i*", $FLTAR_FOLDERS, $FLTAR_RECUR, $FLTAR_SORT)
    _ArrayDisplay($aArray, "'Icons' out - 'Include' in")

    ; Корень C:\Windows: скрытые и системные папки включены.
    $aArray = _FileListToArrayRec("C:\Windows\", "*", $FLTAR_FOLDERS)
    _ArrayDisplay($aArray, "Show hidden folders")

    ; Корень C:\Windows: скрытые и системные папки исключены.
    $aArray = _FileListToArrayRec("C:\Windows\", "*", $FLTAR_FOLDERS + $FLTAR_NOHIDDEN + $FLTAR_NOSYSTEM)
    _ArrayDisplay($aArray, "Hide hidden folders")
EndFunc   ;==>Example
