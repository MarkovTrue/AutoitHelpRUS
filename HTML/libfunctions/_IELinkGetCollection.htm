Справочник функций

_IELinkGetCollection

Возвращает объект-коллекцию со всеми ссылками документа или одну ссылку по индексу.

#include <IE.au3>
_IELinkGetCollection ( ByRef $oObject [, $iIndex = -1] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$iIndex [необязательный] Что возвращать – коллекцию или элемент по индексу:
0 или положительное число – элемент с этим индексом
-1 – (по умолчанию) коллекцию

Возвращаемое значение

Успех: Возвращает объект-коллекцию всех ссылок документа, @extended = количество ссылок.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадение не найдено
@extended: Содержит номер недопустимого параметра.

Примечания

Не все элементы, выглядящие как ссылки, ими являются.
Частая практика – привязка JavaScript-событий onClick к другим DOM-элементам для имитации поведения ссылок.
Для активации таких элементов используйте "click" через _IEAction().

Пример

; Открытие браузера с базовым примером, получение коллекции ссылок,
; пройти в цикле по элементам и отобразить связанные ссылки URL

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("basic")
Local $oLinks = _IELinkGetCollection($oIE)
Local $iNumLinks = @extended

Local $sTxt = $iNumLinks & " links found" & @CRLF & @CRLF
For $oLink In $oLinks
    $sTxt &= $oLink.href & @CRLF
Next
MsgBox($MB_SYSTEMMODAL, "Link Info", $sTxt)
