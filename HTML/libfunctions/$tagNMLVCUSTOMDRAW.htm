Справочник ключевых слов

$tagNMLVCUSTOMDRAW

Содержит сведения, относящиеся к уведомлению NM_CUSTOMDRAW (list view), отправляемому элементом управления ListView.

Global Const $tagNMLVCUSTOMDRAW = "struct;" & $tagNMHDR & ";dword dwDrawStage;handle hdc;" & $tagRECT & _
";dword_ptr dwItemSpec;uint uItemState;lparam lItemlParam;endstruct" & _
";dword clrText;dword clrTextBk;int iSubItem;dword dwItemType;dword clrFace;int iIconEffect;" & _
"int iIconPhase;int iPartId;int iStateId;struct;long TextLeft;long TextTop;long TextRight;long TextBottom;endstruct;uint uAlign"

Параметры

$tagNMHDR Содержит сведения о сообщении-уведомлении.
dwDrawStage Текущая стадия отрисовки. Одно из следующих значений:
Глобальные значения:
$CDDS_POSTERASE - после завершения цикла стирания.
$CDDS_POSTPAINT - после завершения цикла отрисовки.
$CDDS_PREERASE - перед началом цикла стирания.
$CDDS_PREPAINT - перед началом цикла отрисовки.
Значения для элемента:
$CDDS_ITEM - указывает, что поля dwItemSpec, uItemState и lItemlParam действительны.
$CDDS_ITEMPOSTERASE - после стирания элемента.
$CDDS_ITEMPOSTPAINT - после отрисовки элемента.
$CDDS_ITEMPREERASE - перед стиранием элемента.
$CDDS_ITEMPREPAINT - перед отрисовкой элемента.
$CDDS_SUBITEM - флаг, сочетаемый с $CDDS_ITEMPREPAINT или $CDDS_ITEMPOSTPAINT, если рисуется подэлемент.
Устанавливается только если из $CDDS_PREPAINT возвращён $CDRF_NOTIFYITEMDRAW.
hdc Дескриптор контекста устройства элемента управления.
Используйте этот дескриптор контекста устройства (HDC) для вызова любых функций интерфейса графических устройств (GDI) Microsoft Windows.
$tagRECT Координаты Left, Top левого верхнего угла прямоугольника отрисовываемой области.
Это поле инициализируется только уведомлением $CDDS_ITEMPREPAINT. Координаты Right, Bottom правого нижнего угла прямоугольника отрисовываемой области.
Это поле инициализируется только уведомлением $CDDS_ITEMPREPAINT.
dwItemSpec Номер элемента.
uItemState Текущее состояние элемента. Это значение является комбинацией следующих флагов:
$CDIS_CHECKED - элемент отмечен.
$CDIS_DEFAULT - элемент в состоянии по умолчанию.
$CDIS_DISABLED - элемент недоступен.
$CDIS_FOCUS - элемент в фокусе.
$CDIS_GRAYED - элемент отображён серым.
$CDIS_HOT - элемент сейчас под указателем ("горячий").
$CDIS_INDETERMINATE - элемент в неопределённом состоянии.
$CDIS_MARKED - элемент помечен. Смысл определяется реализацией.
$CDIS_SELECTED - элемент выбран.
$CDIS_SHOWKEYBOARDCUES - Comctl32 версии 6.0. Элемент является подсказкой клавиатуры.
$CDIS_NEARHOT - элемент входит в состав элемента управления, который сейчас под указателем мыши ("горячий"), но сам элемент не "горячий".
    Смысл определяется реализацией.
$CDIS_OTHERSIDEHOT - элемент входит в состав split-кнопки, которая сейчас под указателем мыши ("горячая"), но сам элемент не "горячий".
    Смысл определяется реализацией.
$CDIS_DROPHILITED - элемент сейчас является целью операции перетаскивания.
lItemlParam Определяемые приложением данные элемента.
clrText Значение COLORREF с цветом, которым будет отображаться текст (передний план) в элементе управления ListView.
clrTextBk Значение COLORREF с цветом, которым будет отображаться фон текста в элементе управления ListView.
iSubItem Индекс отрисовываемого подэлемента. Если рисуется основной элемент, это поле равно нулю.
dwItemType Версия 6.0. DWORD с типом отрисовываемого элемента. Это поле может быть одним из следующих значений:
$LVCDI_ITEM - рисуется элемент.
$LVCDI_GROUP - рисуется группа.
clrFace Версия 6.0. Значение COLORREF с цветом, которым будет отображаться поверхность элемента.
iIconEffect Версия 6.0. Значение типа int, задающее эффект для значка, например Glow, Shadow или Pulse.
iIconPhase Версия 6.0. Значение типа int, задающее фазу значка.
iPartId Версия 6.0. Значение типа int, задающее ID отрисовываемой части элемента.
iStateId Версия 6.0. Значение типа int, задающее ID отрисовываемого состояния элемента.
TextLeft Координата X левого верхнего угла прямоугольника, в котором будет нарисован текст.
TextTop Координата Y левого верхнего угла прямоугольника, в котором будет нарисован текст.
TextRight Координата X правого нижнего угла прямоугольника, в котором будет нарисован текст.
TextBottom Координата Y правого нижнего угла прямоугольника, в котором будет нарисован текст.
uAlign Версия 6.0. UINT, задающий выравнивание группы. Это поле может быть одним из следующих значений:
$LVGA_HEADER_CENTER - выровнять группу по центру.
$LVGA_HEADER_LEFT - выровнять группу по левому краю.
$LVGA_HEADER_RIGHT - выровнять группу по правому краю.

Примечания

Константы $LVxxx_ требуют #include <ListViewConstants.au3>, константы $CDxx_ требуют #include
