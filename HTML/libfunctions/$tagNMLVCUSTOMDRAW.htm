Справочник ключевых слов

$tagNMLVCUSTOMDRAW

Содержит сведения об уведомляющем сообщении NM_CUSTOMDRAW, отправляемом элементом управления ListView.

Global Const $tagNMLVCUSTOMDRAW = "struct;" & $tagNMHDR & ";dword dwDrawStage;handle hdc;" & $tagRECT & _
";dword_ptr dwItemSpec;uint uItemState;lparam lItemlParam;endstruct" & _
";dword clrText;dword clrTextBk;int iSubItem;dword dwItemType;dword clrFace;int iIconEffect;" & _
"int iIconPhase;int iPartId;int iStateId;struct;long TextLeft;long TextTop;long TextRight;long TextBottom;endstruct;uint uAlign"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
dwDrawStage Текущий этап прорисовки. Это одно из следующих значений:
Глобальные значения:
$CDDS_POSTERASE – после завершения цикла стирания.
$CDDS_POSTPAINT – после завершения цикла рисования.
$CDDS_PREERASE – перед началом цикла стирания.
$CDDS_PREPAINT – перед началом цикла рисования.
Значения, специфичные для элемента:
$CDDS_ITEM – указывает на то, что члены dwItemSpec, uItemState и lItemlParam действительны.
$CDDS_ITEMPOSTERASE – после стирания элемента.
$CDDS_ITEMPOSTPAINT – после прорисовки элемента.
$CDDS_ITEMPREERASE – перед стиранием элемента.
$CDDS_ITEMPREPAINT – перед прорисовкой элемента.
$CDDS_SUBITEM – флаг, комбинируемый с $CDDS_ITEMPREPAINT или $CDDS_ITEMPOSTPAINT при прорисовке подэлемента.
Устанавливается только если из $CDDS_PREPAINT возвращён $CDRF_NOTIFYITEMDRAW.
hdc Дескриптор контекста устройства элемента управления.
Используйте этот дескриптор контекста устройства (HDC) для выполнения любых функций интерфейса Microsoft Windows Graphics Device Interface (GDI).
$tagRECT Координата «слева, сверху» левого верхнего угла прямоугольника область прорисовки.
Этот член инициализируется только уведомлением $CDDS_ITEMPREPAINT Координата «справа, снизу» правого нижнего угла прямоугольника области прорисовки.
Этот член инициализируется только уведомлением $CDDS_ITEMPREPAINT.
dwItemSpec Номер элемента.
uItemState Текущее состояние элемента. Это значение является комбинацией следующих флагов:
$CDIS_CHECKED – элемент отмечен.
$CDIS_DEFAULT – элемент находится в состоянии по умолчанию.
$CDIS_DISABLED – элемент отключен.
$CDIS_FOCUS – элемент имеет фокус.
$CDIS_GRAYED – элемент затенён.
$CDIS_HOT – элемент сейчас находится под указателем (горячий).
$CDIS_INDETERMINATE – элемент находится в неопределённом состоянии.
$CDIS_MARKED – элемент отмечен. Значение определяется реализацией.
$CDIS_SELECTED – элемент выбран.
$CDIS_SHOWKEYBOARDCUES – версия 6.0 Comctl32. Элемент является подсказкой клавиатуры.
$CDIS_NEARHOT – элемент является частью элемента управления, находящегося под указателем мыши (горячий), но сам элемент не горячий.
Значение определяется реализацией.
$CDIS_OTHERSIDEHOT – элемент является частью кнопки разделения, находящейся под указателем мыши (горячая), но сам элемент не горячий.
Значение определяется реализацией.
$CDIS_DROPHILITED – элемент является текущей целью операции перетаскивания.
lItemlParam Определяемые приложением данные элемента.
clrText Значение COLORREF, представляющее цвет текста на переднем плане в элементе управления представлением списка.
clrTextBk Значение COLORREF, представляющее цвет фона текста в элементе управления представлением списка.
iSubItem Индекс подэлемента, который прорисовывается. Если прорисовывается основной элемент, это значение будет равно нулю.
dwItemType Версия 6.0. DWORD, содержащее тип прорисовываемого элемента. Этот член может быть одним из следующих значений:
$LVCDI_ITEM – элемент подлежит прорисовке.
$LVCDI_GROUP – группа подлежит прорисовке.
clrFace Версия 6.0. Значение COLORREF, представляющее цвет фасада элемента.
iIconEffect Версия 6.0. Значение типа int, определяющее эффект, применяемый к значку (например, свечение, тень или импульс).
iIconPhase Версия 6.0. Значение типа int, определяющее фазу значка.
iPartId Версия 6.0. Значение типа int, определяющее идентификатор части элемента для прорисовки.
iStateId Версия 6.0. Значение типа int, определяющее идентификатор состояния прорисовываемого элемента.
TextLeft Координата X левого верхнего угла прямоугольника, в котором будет прорисован текст.
TextTop Координата Y левого верхнего угла прямоугольника, в котором будет прорисован текст.
TextRight Координата X правого нижнего угла прямоугольника, в котором будет прорисован текст.
TextBottom Координата Y правого нижнего угла прямоугольника, в котором будет прорисован текст.
uAlign Версия 6.0. UINT, определяющее способ выравнивания группы. Этот член может быть одним из следующих значений:
$LVGA_HEADER_CENTER – выровнять группу по центру.
$LVGA_HEADER_LEFT – выровнять группу слева.
$LVGA_HEADER_RIGHT – выровнять группу справа.

Примечания

Константы $LVxxx_ требуют #include , константы $CDxx_ требуют #include
