_WinAPI_OpenMutex

Открывает существующий именованный объект мьютекса.

#include <WinAPIProc.au3>
_WinAPI_OpenMutex ( $sMutex [, $iAccess = $MUTEX_ALL_ACCESS [, $bInherit = False]] )

Параметры

$sMutex Имя мьютекса для открытия. Сравнение имён чувствительно к регистру.
$iAccess [необязательный] Доступ к объекту мьютекса. Функция завершается с ошибкой, если дескриптор безопасности указанного объекта
не позволяет требуемый доступ для вызывающего процесса.
Этот параметр может быть одним из следующих значений:
$MUTEX_ALL_ACCESS (по умолчанию)
$MUTEX_MODIFY_STATE
$bInherit [необязательный] Наследуется ли дескриптор дочерними процессами:
True – процессы, созданные этим процессом, наследуют дескриптор.
False – процессы не наследуют дескриптор (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор объекта мьютекса.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция успешно выполняется только если какой-либо процесс уже создал мьютекс с использованием функции _WinAPI_CreateMutex().
Вызывающий процесс может использовать возвращённый дескриптор в любой функции, которая требует дескриптор
объекта мьютекса.
