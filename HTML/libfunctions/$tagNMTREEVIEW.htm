Справочник ключевых слов

$tagNMTREEVIEW

Содержит сведения об уведомляющем сообщении TreeView.

Global Const $tagNMTREEVIEW = $tagNMHDR & ";uint Action;" & _
"struct; uint OldMask;handle OldhItem;uint OldState;uint OldStateMask;" & _
"ptr OldText;int OldTextMax;int OldImage;int OldSelectedImage;int OldChildren;lparam OldParam; endstruct;" & _
"struct; uint NewMask;handle NewhItem;uint NewState;uint NewStateMask;" & _
"ptr NewText;int NewTextMax;int NewImage;int NewSelectedImage;int NewChildren;lparam NewParam; endstruct;" & _
"struct; long PointX;long PointY; endstruct"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Action Флаг действия, специфичный для уведомления.
OldMask Флаги, указывающие, какие члены структуры содержат корректные данные.
OldhItem Элемент, на который ссылается эта структура.
OldState Набор битовых флагов и индексов списка изображений, указывающих состояние элемента.
OldStateMask Биты члена state, которые действительны.
OldText Указатель на строку с нулём в конце, содержащую текст элемента.
OldTextMax Размер буфера, на который указывает член Text, в символах.
OldImage Индекс в списке изображений значка для использования, когда элемент не выбран.
OldSelectedImage Индекс в списке изображений значка для использования, когда элемент выбран.
OldChildren Флаг, указывающий, есть ли у элемента дочерние элементы.
OldParam Значение, связываемое с элементом.
NewMask Флаги, указывающие, какие члены структуры содержат корректные данные.
NewhItem Элемент, на который ссылается эта структура.
NewState Набор битовых флагов и индексов списка изображений, указывающих состояние элемента.
NewStateMask Биты члена state, которые действительны.
NewText Указатель на строку с нулём в конце, содержащую текст элемента.
NewTextMax Размер буфера, на который указывает член Text, в символах.
NewImage Индекс в списке изображений значка для использования, когда элемент не выбран.
NewSelectedImage Индекс в списке изображений значка для использования, когда элемент выбран.
NewChildren Флаг, указывающий, есть ли у элемента дочерние элементы.
NewParam Значение, связываемое с элементом.
PointX Координата X мыши в момент возникновения события.
PointY Координата Y мыши в момент возникновения события.
