Справочник функций

_Excel_RangeCopyPaste

Вырезает или копирует одну или несколько ячеек, строк или столбцов в диапазон либо в/из буфера обмена.

#include <Excel.au3>
_Excel_RangeCopyPaste ( $oWorksheet, $vSourceRange [, $vTargetRange = Default [, $bCut = False [, $iPaste = Default [, $iOperation = Default [, $bSkipBlanks = False [, $bTranspose = False]]]]]] )

Параметры

$oWorksheet Объект исходного рабочего листа.
$vSourceRange Исходный диапазон для копирования/вырезания. Может быть объектом Range или диапазоном A1.
При значении Default диапазон копируется из буфера обмена.
$vTargetRange [необязательный] Целевой диапазон для вставки. Может быть объектом Range или диапазоном A1.
При значении Default диапазон копируется в буфер обмена (по умолчанию Default).
$bCut [необязательный] True – исходный диапазон вырезается, а не копируется (по умолчанию False).
Игнорируется, если $vSourceRange = Default.
$iPaste [необязательный] Часть диапазона для вставки из буфера (формулы, форматы и т.д.). Константа XlPasteType
(по умолчанию Default).
$iOperation [необязательный] Операция при вставке (сложение, деление, умножение и т.д.). Константа XlPasteSpecialOperation
(по умолчанию Default).
$bSkipBlanks [необязательный] True – пустые ячейки из буфера обмена не вставляются в целевой диапазон (по умолчанию False).
$bTranspose [необязательный] True – транспонировать строки и столбцы при вставке (по умолчанию False).

Возвращаемое значение

Успех: Возвращает объект целевого диапазона, если $vTargetRange <> Default, иначе 1.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorksheet не является объектом или объектом листа
2 – $vSourceRange недопустим.
@extended: Содержит код COM-ошибки
3 – $vTargetRange недопустим.
@extended: Содержит код COM-ошибки
4 – ошибка при вставке ячеек.
@extended: Содержит код COM-ошибки
5 – ошибка при вырезании ячеек.
@extended: Содержит код COM-ошибки
6 – ошибка при копировании ячеек.
@extended: Содержит код COM-ошибки
7 – $vSourceRange и $vTargetRange не могут одновременно быть равны Default

Примечания

$vSourceRange и $vTargetRange не могут одновременно быть равны Default.
Если $vSourceRange = Default:
* диапазон копируется из буфера обмена методом PasteSpecial
* $bCut игнорируется
* $iPaste, $iOperation, $bSkipBlanks и $bTranspose применяются
Если $vSourceRange и $vTargetRange заданы явно, параметры $iPaste, $iOperation, $bSkipBlanks и $bTranspose игнорируются.

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть рабочую книгу 1
Local $oWorkbook1 = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"), True)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Копируем диапазон из 3 строк и 2 столбцов на активном листе.
; Передаём исходный диапазон как объект.
Local $oRange = $oWorkbook1.ActiveSheet.Range("I2:J4")
_Excel_RangeCopyPaste($oWorkbook1.ActiveSheet, $oRange, "G7")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 1", "Error copying cells." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 1", "Range 'I2:J4' successfully copied to 'G7'.")

Local $oWorkbook2 = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel3.xls"), True)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example", "Error opening workbook '" & _Extras_PathFull("_Excel3.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть рабочую книгу 1
Local $oWorkbook1 = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Копируем 2 строки (1 и 2) с листа 2 в буфер обмена.
_Excel_RangeCopyPaste($oWorkbook1.Worksheets(2), "1:2")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 3", "Error copying rows." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 3", "Rows 1+2 successfully copied from worksheet 2 to the clipboard.")

; Вставляем диапазон из буфера обмена (из примера 3) на активный лист.
; Вставляются только значения без форматирования.
_Excel_RangeCopyPaste($oWorkbook1.Activesheet, Default, "1:1", Default, $xlPasteValues)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 4", "Error copying rows." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 4", "2 Rows successfully pasted from the clipboard to row 1.")

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть рабочую книгу 1

    _Excel_Close($oExcel)
    Exit
EndIf

; Вставляем формат ячейки в другие ячейки.

If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example5", "Error pasting cells." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeCopy Example 5", "Format of cell 'A1' successfully pasted to 'B1:E16'.")
