Справочник функций

_ArrayMax

Возвращает наибольшее значение из одномерного 1D или двумерного 2D массива.

#include <Array.au3>
_ArrayMax ( Const ByRef $aArray [, $iCompNumeric = 0 [, $iStart = -1 [, $iEnd = -1 [, $iSubItem = 0]]]] )

Параметры

$aArray Массив для поиска.
$iCompNumeric [необязательный] Метод сравнения:
0 – буквенно-цифровое сравнение
1 – числовое сравнение
$iStart [необязательный] Индекс массива для начала поиска
$iEnd [необязательный] Индекс массива для окончания поиска
$iSubItem [необязательный] Столбец массива для поиска

Возвращаемое значение

Успех: Возвращает максимальное значение в массиве.
Ошибка: Устанавливает @error ≠ 0 (коды ошибок см. в описании _ArrayMaxIndex()).

См. также

_ArrayMaxIndex, _ArrayMin, _ArrayMinIndex, _ArrayUnique

Пример

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray = StringSplit("4,2,06,8,12,5", ",")

MsgBox($MB_SYSTEMMODAL, 'Max String value', _ArrayMax($aArray, 0, 1))
MsgBox($MB_SYSTEMMODAL, 'Max Numeric value', _ArrayMax($aArray, 1, 1))

Local $aArray[4][4]
For $i = 0 To 3
    For $j = 0 To 3
        $aArray[$i][$j] = Random(0, 99, 1)
    Next
Next
_ArrayDisplay($aArray, "2D Array")
MsgBox($MB_SYSTEMMODAL, 'Max Numeric value in column 2', _ArrayMax($aArray, 1, 0, -1, 2))
