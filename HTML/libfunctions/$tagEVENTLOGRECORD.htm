Справочник ключевых слов

$tagEVENTLOGRECORD

Содержит сведения о записи в журнале событий.

Global Const $tagEVENTLOGRECORD = "int Length;int Reserved;int RecordNumber;int TimeGenerated;int TimeWritten;int EventID;" & _
"short EventType;short NumStrings;short EventCategory;short ReservedFlags;int ClosingRecordNumber;int StringOffset;" & _
"int UserSidLength;int UserSidOffset;int DataLength;int DataOffset"

Параметры

Length Размер этой записи события в байтах.
Reserved Зарезервировано.
RecordNumber Номер записи.
TimeGenerated Время, когда эта запись была создана.
TimeWritten Время, когда эта запись была получена сервисом для записи в журнал.
EventID Идентификатор события.
EventType Тип события.
NumStrings Количество строк, присутствующих в журнале.
EventCategory Категория для этого события.
ReservedFlags Зарезервировано.
ClosingRecordNumber Зарезервировано.
StringOffset Смещение строк описания в этой записи журнала событий.
UserSidLength Размер члена UserSid в байтах.
UserSidOffset Смещение идентификатора безопасности (SID) в этой записи журнала событий.
DataLength Размер специфичных для события данных (в позиции, указанной DataOffset) в байтах.
DataOffset Смещение специфичной для события информации в этой записи журнала событий в байтах.
