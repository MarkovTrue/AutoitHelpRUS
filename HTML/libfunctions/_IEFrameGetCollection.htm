Справочник функций

_IEFrameGetCollection

Возвращает объект-коллекцию фреймов FrameSet или iFrame обычной страницы либо отдельный Frame/iFrame по индексу.

#include <IE.au3>
_IEFrameGetCollection ( ByRef $oObject [, $iIndex = -1] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$iIndex [необязательный] Что возвращать – коллекцию или элемент по индексу:
0 или положительное число – элемент с этим индексом
-1 – (по умолчанию) коллекцию

Возвращаемое значение

Успех: Возвращает объектную переменную с коллекцией фреймов; @extended – количество фреймов.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадение не найдено
@extended: Содержит номер недопустимого параметра.

Примечания

Хотя MSDN документирует возвращаемое значение этой функции как объект-коллекцию, по нему нельзя пройти циклом For...In...Next, как по обычной коллекции.
Вместо этого нужно перебирать коллекцию по индексу – см. пример.

См. также

_IEFrameGetObjByName, _IEIsFrameSet

Пример

; Открыть пример с frameset, получить коллекцию фреймов
; и пройти по ним в цикле, отобразив их исходные URL

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("frameset")
Local $oFrames = _IEFrameGetCollection($oIE)
Local $iNumFrames = @extended
Local $sTxt = $iNumFrames & " frames found" & @CRLF & @CRLF
Local $oFrame = 0
For $i = 0 To ($iNumFrames - 1)
    $oFrame = _IEFrameGetCollection($oIE, $i)
    $sTxt &= _IEPropertyGet($oFrame, "innertext") & @CRLF
Next
MsgBox($MB_SYSTEMMODAL, "Frames Info", $sTxt)

_IEQuit($oIE)
