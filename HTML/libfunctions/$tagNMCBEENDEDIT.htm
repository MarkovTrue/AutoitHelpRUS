Справочник ключевых слов

$tagNMCBEENDEDIT

Содержит сведения о завершении операции редактирования в элементе управления ComboBoxEx.

Global Const $tagNMCBEENDEDIT = $tagNMHDR & ";bool fChanged;int NewSelection;wchar szText[260];int Why"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
fChanged Указывает, изменилось ли содержимое поля редактирования элемента управления.
NewSelection Индекс с отсчётом от нуля элемента, который будет выбран после завершения операции редактирования
Это значение может быть $CB_ERR, если ни один элемент не будет выбран.
szText Строка с нулём в конце, которая содержит текст из поля редактирования элемента управления.
Why Действие, которое создало сообщение уведомления $CBEN_ENDEDIT
Это значение может быть одним из следующих:
$CBENF_DROPDOWN – пользователь активировал выпадающий список
$CBENF_ESCAPE – пользователь нажал ESC
$CBENF_KILLFOCUS – поле редактирования потеряло фокус клавиатуры
$CBENF_RETURN – пользователь завершил операцию редактирования нажатием ENTER.
