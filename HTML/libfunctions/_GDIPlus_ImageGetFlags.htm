_GDIPlus_ImageGetFlags

Возвращает перечисление атрибутов данных пикселей изображения.

#include <GDIPlus.au3>
_GDIPlus_ImageGetFlags ( $hImage )

Параметры

$hImage Дескриптор объекта Image.

Возвращаемое значение

Успех: Возвращает массив следующего вида:
$aArray[0] – целочисленный набор флагов атрибутов данных пикселей
$aArray[1] – строка атрибутов данных пикселей, разделённых знаком "|"
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).
@error: 10 – не удалось получить флаги
11 – недопустимый дескриптор изображения
12 – нет данных пикселей

Примечания

Применяйте BitAND() к возвращённому набору флагов с константами $GDIP_IMAGEFLAGS_*.

См. также

_GDIPlus_ImageGetPixelFormat

Дополнительно

Искать GdipGetImageFlags в библиотеке MSDN

Пример

#include "Extras\HelpFileInternals.au3"

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPIHObj.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hBitmap, $hImage, $aRet

    ; Создать GUI
    GUICreate("GDI+", 600, 400)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Инициализировать библиотеку GDI+
    _GDIPlus_Startup()

    ; Захватить 32-битный bitmap
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; вывести набор атрибутов пиксельных данных для захвата экрана
    $aRet = _GDIPlus_ImageGetFlags($hImage)
    $aRet[1] = @CRLF & StringReplace($aRet[1], "|", @CRLF)
    _MemoWrite("Image pixel data attributes of screen capture: " & $aRet[1]) ;
    _MemoWrite() ;
    _MemoWrite("Image flag set of pixel data attributes: " & $aRet[0]) ;
    _MemoWrite() ;

    ; Сохранить снимок экрана в файл
    _GDIPlus_ImageSaveToFile($hImage, @MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Освободить ресурсы
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Загрузить снимок экрана из файла
    $hImage = _GDIPlus_ImageLoadFromFile(@MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; вывести набор атрибутов пиксельных данных для сохранённого файла
    $aRet = _GDIPlus_ImageGetFlags($hImage)
    $aRet[1] = @CRLF & StringReplace($aRet[1], "|", @CRLF)
    _MemoWrite("Image pixel data attributes of saved file: " & $aRet[1]) ;
    _MemoWrite() ;
    _MemoWrite("Image flag set of pixel data attributes: " & $aRet[0]) ;

    ; Освободить ресурсы
    _GDIPlus_ImageDispose($hImage)

    ; Завершить работу с библиотекой GDI+
    _GDIPlus_Shutdown()

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
