Справочник функций

_SetTime

Устанавливает текущее время системы.

#include <Date.au3>
_SetTime ( $iHour, $iMinute [, $iSecond = 0 [, $iMSeconds = 0]] )

Параметры

$iHour Часы. Значения: 0–23.
$iMinute Минуты. Значения: 0–59.
$iSecond [необязательный] Секунды. Значения: 0–59
$iMSeconds [необязательный] Миллисекунды. Значения: 0–999

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – Ошибка.
@extended: _WinAPI_GetLastError()
Код ошибки: MSDN: ms681381.aspx.

Примечания

Если необязательные параметры ($iSecond и iMSeconds) не определены, функция не изменит текущее значение.

_SetDate

Дополнительно

Искать GetLastError в библиотеке MSDN

Пример

#include <Date.au3>

_SetTime(20, 15) ; Установить время на 20:15
_SetTime(20, 15, 30) ; Установить время на 20:15:30
