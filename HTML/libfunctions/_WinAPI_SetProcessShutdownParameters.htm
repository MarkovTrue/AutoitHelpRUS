Справочник функций

_WinAPI_SetProcessShutdownParameters

Задаёт порядок завершения для процесса относительно других процессов в системе.

#include <WinAPISys.au3>
_WinAPI_SetProcessShutdownParameters ( $iLevel [, $bDialog = False] )

Параметры

$iLevel Приоритет завершения. Система завершает процессы от высоких значений $iLevel к низким. Наибольший
и наименьший приоритеты завершения зарезервированы для компонентов системы.
Этот параметр должен быть в следующем диапазоне значений.
0x0000-0x00FF - Последний диапазон завершения, зарезервированный системой.
0x0100-0x01FF - Последний диапазон завершения, зарезервированный приложением.
0x0200-0x02FF - "Промежуточный" диапазон завершения, зарезервированный приложением.
0x0300-0x03FF - Первый диапазон завершения, зарезервированный приложением.
0x0400-0x04FF - Первый диапазон завершения, зарезервированный системой.

Все процессы начинают работу на уровне завершения 0x0280.
$bDialog [необязательный] Указывает, отображать ли диалоговое окно повтора для пользователя, допустимые значения:
True - Отображать диалоговое окно повтора, если процесс завершается дольше указанного времени ожидания.
False - Прямое завершение процесса (По умолчанию).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Пример

#include <ButtonConstants.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPISys.au3>
#include <WindowsNotifsConstants.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

Global $g_hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 200, 200)
Local $idButton = GUICtrlCreateButton('', 73, 62, 54, 54, $BS_ICON)
GUICtrlSetImage(-1, @SystemDir & '\shell32.dll', 45)
GUICtrlSetTip(-1, 'Log off ' & @UserName)
Local $idCheck = GUICtrlCreateCheckbox('Block Windows shutdown', 10, 173, 144, 21)
GUIRegisterMsg($WM_QUERYENDSESSION, 'WM_QUERYENDSESSION')
GUISetState(@SW_SHOW)

; Установить максимальный приоритет завершения для текущего процесса, чтобы предотвратить закрытие других процессов.
_WinAPI_SetProcessShutdownParameters(0x03FF)

While 1
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE
            ExitLoop
        Case $idButton
            Shutdown(0)
        Case $idCheck
            If GUICtrlRead($idCheck) = $GUI_CHECKED Then
                _WinAPI_ShutdownBlockReasonCreate($g_hForm, 'This application is blocking system shutdown because the saving critical data is in progress.')
            Else
                _WinAPI_ShutdownBlockReasonDestroy($g_hForm)
            EndIf
    EndSwitch
WEnd

Func WM_QUERYENDSESSION($hWnd, $iMsg, $wParam, $lParam)
    #forceref $iMsg, $wParam, $lParam

    Switch $hWnd
        Case $g_hForm
            If _WinAPI_ShutdownBlockReasonQuery($g_hForm) Then
                Return 0
            EndIf
    EndSwitch
    Return $GUI_RUNDEFMSG
EndFunc   ;==>WM_QUERYENDSESSION
