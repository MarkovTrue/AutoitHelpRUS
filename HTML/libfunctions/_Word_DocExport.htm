Справочник функций

_Word_DocExport

Экспортирует документ или диапазон как PDF или XPS.

#include <Word.au3>
_Word_DocExport ( $oDoc, $sFileName [, $iFormat = $WdExportFormatPDF [, $iRange = $WdExportAllDocument [, $iFrom = Default [, $iTo = Default [, $bOpenAfterExport = False [, $bIncludeProperties = True [, $iCreateBookmarks = Default [, $bUseISO19005 = False]]]]]]]] )

Параметры

$oDoc Объект документа Word.
$sFileName Путь/имя экспортируемого файла.
$iFormat [необязательный] Формат для использования при экспорте. Может быть любым из перечисления WdExportFormat (по умолчанию = $WdExportFormatPDF)
$iRange [необязательный] Указывает, что экспортировать. Может быть любым из перечисления WdExportRange (по умолчанию = $WdExportAllDocument)
$iFrom [необязательный] Номер страницы, с которой начать публикацию (по умолчанию = начать с начала)
$iTo [необязательный] Номер страницы, на которой закончить публикацию (по умолчанию = закончить на последней странице)
$bOpenAfterExport [необязательный] True отображает файл в средстве просмотра после его публикации (по умолчанию = False)
$bIncludeProperties [необязательный] True указывает, что свойства документа должны быть включены (по умолчанию = True)
$iCreateBookmarks [необязательный] Указывает, экспортировать ли закладки и тип экспортируемых закладок. Может быть любым из перечисления WdExportCreateBookmarks (по умолчанию)
$bUseISO19005 [необязательный] Указывает, должен ли экспорт быть создан как PDF/A (автономный PDF) (по умолчанию = False)

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $oDoc не является объектом 2 – $sFileName пуста 3 – Ошибка, возвращённая при экспорте объекта. @extended задан код ошибки COM

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Экспортировать полный документ со значениями по умолчанию
Local $sFileName = @TempDir & "\Test1.pdf"
_Word_DocExport($oDoc, $sFileName)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "Error exporting the document." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
ShellExecute($sFileName)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "The whole document has successfully been exported to: " & $sFileName)

Пример 2

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Экспортировать диапазон как PDF/A и отобразить экспортированный файл
; Установить диапазон на строки 2 и 3
Local $oRange = _Word_DocRangeSet($oDoc, -1, $wdParagraph, 1, $wdParagraph, 2)
Local $sFileName = @TempDir & "\Test2.pdf"
_Word_DocExport($oRange, $sFileName, Default, $wdExportFromTo, Default, Default, True, Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", "Error exporting the document." & _
        @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocExport Example", _
        "The specified range has successfully been exported to: " & $sFileName)
