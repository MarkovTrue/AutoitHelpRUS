_ScreenCapture_SetTIFColorDepth

Задаёт глубину цвета для снимков экрана в формате TIFF.

#include <ScreenCapture.au3>
_ScreenCapture_SetTIFColorDepth ( $iDepth )

Параметры

$iDepth Глубина цвета изображения:
0 – глубина цвета по умолчанию
24 – 24 бита
32 – 32 бита
48 – 48 бит

Возвращаемое значение

Отсутствует.

Примечания

Если значение не задано, по умолчанию используется 24 бита.
