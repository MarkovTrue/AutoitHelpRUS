_WinAPI_DwmInvalidateIconicBitmaps

Сообщает, что все ранее переданные значковые битмапы окна – и миниатюры, и Peek-представления – должны быть обновлены.

#include <WinAPIGdi.au3>
_WinAPI_DwmInvalidateIconicBitmaps ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Этот вызов заставляет Desktop Window Manager (DWM) сбросить текущие битмапы окна и запросить новые, когда они снова понадобятся. _WinAPI_DwmInvalidateIconicBitmaps() следует вызывать только когда битмапы значков и миниатюр устарели.
