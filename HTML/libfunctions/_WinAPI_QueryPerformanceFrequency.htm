_WinAPI_QueryPerformanceFrequency

Получает частоту счётчика производительности высокого разрешения.

#include <WinAPISys.au3>
_WinAPI_QueryPerformanceFrequency ( )

Возвращаемое значение

Успех: Возвращает текущая частота счётчика производительности в отсчётах в секунду.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
