_WinAPI_SfcIsKeyProtected

Определяет, защищён ли указанный ключ реестра.

#include <WinAPIReg.au3>
_WinAPI_SfcIsKeyProtected ( $hKey [, $sSubKey = Default [, $iFlag = 0]] )

Параметры

$hKey Дескриптор корневого ключа реестра, он должен быть одним из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sSubKey [необязательный] Имя ключа для проверки. Этот ключ должен быть подключом ключа, определённого параметром $hKey.
Если этот параметр не указан (по умолчанию), функция проверяет только защищён ли корневой ключ реестра.
$iFlag [необязательный] Флаг, определяющий альтернативное представление реестра, которое должны использовать приложения, работающие на 64-разрядной Windows.
Этот флаг игнорируется на платформе x86. Он может быть одним из следующих значений:
$KEY_WOW64_32KEY
$KEY_WOW64_64KEY

Возвращаемое значение

True: Возвращает ключ защищён.
False: ключ не защищён.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
