_WinAPI_GetArcDirection

Получает текущее направление дуги для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_GetArcDirection ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает текущее направление дуги ($AD_*).
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
