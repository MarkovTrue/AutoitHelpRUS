Справочник функций

_FTP_Connect

Подключается к FTP-серверу.

#include <FTPEx.au3>
_FTP_Connect ( $hInternetSession, $sServerName, $sUsername, $sPassword [, $iPassive = 0 [, $iServerPort = 0 [, $iService = $INTERNET_SERVICE_FTP [, $iFlags = $INTERNET_FLAG_DEFAULT [, $fuContext = 0]]]]] )

Параметры

$hInternetSession Дескриптор, возвращённый _FTP_Open().
$sServerName Имя или IP-адрес сервера.
$sUsername Имя пользователя.
$sPassword Пароль.
$iPassive [необязательный] Пассивный режим. По умолчанию 0. См. раздел Примечания.
$iServerPort [необязательный] Порт сервера (0 означает порт по умолчанию – 21).
$iService [необязательный] Одно из следующих значений констант:
$INTERNET_SERVICE_FTP (1) – служба FTP (значение по умолчанию)
$INTERNET_SERVICE_GOPHER (2) – служба Gopher. Доступна только в Windows XP, Windows Server 2003 R2 и более ранних.
$INTERNET_SERVICE_HTTP (3) – служба HTTP.
$iFlags [необязательный] Специальные флаги.
$INTERNET_FLAG_DEFAULT (0)
$INTERNET_FLAG_PASSIVE (0x08000000)
$INTERNET_FLAG_TRANSFER_ASCII (1)
$INTERNET_FLAG_TRANSFER_BINARY (2)
$fuContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает дескриптор подключённого сеанса.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Возвращаемый дескриптор нужно закрыть через _FTP_Close().
Если $iPassive равен 1, специальный флаг $INTERNET_FLAG_PASSIVE автоматически объединяется с пользовательским параметром $iFlag.

Пример

#include <FTPEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
;~ Local $sServer = 'ftp.cs.brown.edu' ; Brown Computer Science
    Local $sServer = 'speedtest.tele2.net' ; Сервис Tele2 Speedtest
    Local $sUsername = ''
    Local $sPass = ''

    Local $hOpen = _FTP_Open('MyFTP Control')
    Local $hConn = _FTP_Connect($hOpen, $sServer, $sUsername, $sPass)
    If @error Then
        MsgBox($MB_SYSTEMMODAL, '_FTP_Connect', 'ERROR=' & @error)
    Else
        Local $iErr = @error, $sFTP_Message
        _FTP_GetLastResponseInfo($iErr, $sFTP_Message)
        ConsoleWrite('$iErr=' & $iErr & ' $sFTP_Message:' & @CRLF & $sFTP_Message & @CRLF)
        ; выполнить нужные действия...
    EndIf
    _FTP_Close($hConn)
    _FTP_Close($hOpen)
EndFunc   ;==>_Example
