Справочник ключевых слов

$tagOVERLAPPED

Содержит сведения для асинхронного (перекрывающегося) ввода-вывода.

Global Const $tagOVERLAPPED = "ulong_ptr Internal;ulong_ptr InternalHigh;struct; dword Offset;dword OffsetHigh; endstruct;handle hEvent"

Параметры

Internal Зарезервировано для использования операционной системой. Этот член, который указывает на зависящее от системы состояние, действителен, когда функция GetOverlappedResult возвращает результат без установки расширенной информации об ошибке на ERROR_IO_PENDING.
InternalHigh Зарезервировано для использования операционной системой. Этот член, который указывает на длину переданных данных, действителен, когда функция GetOverlappedResult возвращает True.
Offset Позиция файла, с которой начинается передача. Позиция файла – это смещение в байтах от начала файла.
Вызывающий процесс должен установить этот член перед вызовом функции ReadFile или WriteFile.
Этот член используется только, когда устройство является файлом. В противном случае этот член должен быть нулём.
OffsetHigh Старшее слово позиции файла, с которой начинается передача.
Этот член используется только, когда устройство является файлом. В противном случае этот член должен быть нулём.
hEvent Дескриптор события, который будет установлен в сигнальное состояние по завершении операции.
Вызывающий процесс должен установить этот член либо на нуль, либо на допустимый дескриптор события перед вызовом любых перекрывающихся функций.
