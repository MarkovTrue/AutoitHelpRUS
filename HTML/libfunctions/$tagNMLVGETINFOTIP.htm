Справочник ключевых слов

$tagNMLVGETINFOTIP

Содержит и принимает сведения об элементе ListView для отображения подсказки.

Global Const $tagNMLVGETINFOTIP = $tagNMHDR & ";int Flags;ptr Text;int TextMax;int Item;int SubItem;int lParam"

Параметры

$tagNMHDR Содержит сведения об уведомляющем сообщении.
Flags Либо нуль, либо $LVGIT_UNFOLDED.
Text Адрес буфера строки, который получает дополнительную текстовую информацию.
Если Flags равен нулю, этот член будет содержать существующий текст элемента.
В этом случае вы должны добавить любой дополнительный текст в конец этой строки.
TextMax Размер в символах буфера, на который указывает Text.
Хотя вы никогда не должны предполагать, что этот буфер будет определённого размера, значение $INFOTIPSIZE можно использовать для целей проектирования.
Item Индекс элемента с отсчётом от нуля, к которому относится эта структура.
SubItem Индекс подэлемента с отсчётом от единицы, к которому относится эта структура.
Если этот член равен нулю, структура относится к элементу, а не к подэлементу.
Этот член в настоящее время не используется и всегда будет равен нулю.
lParam Определяемое приложением значение, связанное с элементом
Этот член в настоящее время не используется и всегда будет равен нулю.
