Справочник функций

_SQLite_GetTable

Возвращает одномерный массив, содержащий имена столбцов и данные выполненного запроса.

#include <SQLite.au3>
_SQLite_GetTable ( $hDB, $sSQL, ByRef $aResult, ByRef $iRows, ByRef $iColumns [, $iCharSize = -1] )

Параметры

$hDB Открытая база данных; -1 – использовать последнюю открытую базу данных.
$sSQL Выполняемый SQL-запрос.
$aResult Выходной параметр – результат.
$iRows Возвращает количество строк данных.
$iColumns Выходной параметр – количество столбцов.
$iCharSize [необязательный] Указывает максимальный размер поля данных

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – вызов запрещён безопасным режимом
2 – ошибка, возвращённая _SQLite_Query() в @extended
3 – ошибка, возвращённая _SQLite_FetchNames() в @extended
4 – ошибка, возвращённая _SQLite_FetchData() в @extended

Примечания

Количество значений, вставляемых в $aResult, будет (($iRows) + 1) * ($iColumns); это число вставляется в $aResult[0].
NULL будет возвращён как число 0.
Если вам не нужен результат (или результата не будет), рассмотрите использование _SQLite_Exec().

См. также

_SQLite_Exec, _SQLite_GetTable2D, _SQLite_Query

Пример

#include <Array.au3>
#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $aResult, $iRows, $iColumns, $iRval

_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; Открыть :memory: базу данных
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "Can't Load Database!")
    Exit -1
EndIf

; Пример таблицы
; Имя | Возраст
; -----------------------
; Alice | 43
; Bob | 28
; Cindy | 21

If Not _SQLite_Exec(-1, "CREATE TEMP TABLE persons (Name, Age);") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Alice','43');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Bob','28');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())
If Not _SQLite_Exec(-1, "INSERT INTO persons VALUES ('Cindy','21');") = $SQLITE_OK Then _
        MsgBox($MB_SYSTEMMODAL, "SQLite Error", _SQLite_ErrMsg())

; Запрос
$iRval = _SQLite_GetTable(-1, "SELECT * FROM persons;", $aResult, $iRows, $iColumns)
If $iRval = $SQLITE_OK Then
    ; $aResult выглядит так:
    ; [0] = 8
    ; [1] = Имя
    ; [2] = Возраст
    ; [3] = Alice
    ; [4] = 43
    ; [5] = Bob
    ; [6] = 28
    ; [7] = Cindy
    ; [8] = 21
    _ArrayDisplay($aResult, "Query Result")
Else
    MsgBox($MB_SYSTEMMODAL, "SQLite Error: " & $iRval, _SQLite_ErrMsg())
EndIf

_SQLite_Close()
_SQLite_Shutdown()
