_WinAPI_EnumWindows

Перечисляет все окна.

#include <WinAPISysWin.au3>
_WinAPI_EnumWindows ( [$bVisible = True [, $hWnd = Default]] )

Параметры

$bVisible [необязательный] Флаг отбора окон:
True – возвращает только видимые окна
False – возвращает все окна.
$hWnd [необязательный] Дескриптор стартового окна (по умолчанию – Desktop).

Возвращаемое значение

Возвращает массив в следующем формате:
    [0][0] – число строк в массиве (n)
    [1][0] – дескриптор окна
    [1][1] – имя класса окна
    [n][0] – дескриптор окна
    [n][1] – имя класса окна

См. также

_WinAPI_EnumWindowsPopup, _WinAPI_EnumWindowsTop

Пример

#include <Array.au3>
#include <WinAPISysWin.au3>

Example()

Func Example()
    Local $aWindows = _WinAPI_EnumWindows()
    Local $aResult[$aWindows[0][0]][5]
    For $i = 1 To $aWindows[0][0]
        $aResult[$i - 1][0] = $aWindows[$i][0]
        $aResult[$i - 1][1] = $aWindows[$i][1]
        $aResult[$i - 1][2] = WinGetTitle($aWindows[$i][0])
        $aResult[$i - 1][3] = WinGetText($aWindows[$i][0])
        $aResult[$i - 1][4] = WinGetProcess($aWindows[$i][0])
    Next
    _ArrayDisplay($aResult, "_WinAPI_EnumWindows", Default, Default, Default, "#|Handle|Class|Title|Text|Process")
EndFunc   ;==>Example
