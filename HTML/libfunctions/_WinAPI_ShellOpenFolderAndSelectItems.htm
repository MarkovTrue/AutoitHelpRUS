Справочник функций

_WinAPI_ShellOpenFolderAndSelectItems

Открывает окно проводника Windows с выбранными указанными элементами в определённой папке.

#include <WinAPIShellEx.au3>
_WinAPI_ShellOpenFolderAndSelectItems ( $sFilePath [, $aNames = 0 [, $iStart = 0 [, $iEnd = -1 [, $iFlags = 0]]]] )

Параметры

$sFilePath Полный квалифицированный путь, который указывает папку или файл. Если используется параметр $aNames, $sFilePath должен быть указан только до папки, в противном случае функция не выполнится. Этот параметр может быть пустой строкой.
$aNames [необязательный] Массив имён папок или файлов для выделения. Это должны быть просто имена в указанной папке без её пути.
$iStart [необязательный] Индекс массива, с которого начинать выделение.
$iEnd [необязательный] Индекс массива, на котором остановить выделение.
$iFlags [необязательный] Дополнительные флаги. Этот параметр может быть одним или несколькими из следующих значений.
Windows Vista и более поздние версии $OFASI_EDIT $OFASI_OPENDESKTOP.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Пример

#include <File.au3>
#include <WinAPIShellEx.au3>

; Поддержка работы в X64
Local $sWow64 = ""
If @AutoItX64 Then $sWow64 = "\Wow6432Node"

Local $sPath = RegRead('HKLM\SOFTWARE' & $sWow64 & '\AutoIt v3\AutoIt', 'InstallDir')

Local $aList = _FileListToArray($sPath, '*.exe', 1)

If IsArray($aList) Then
    _WinAPI_ShellOpenFolderAndSelectItems($sPath, $aList, 1)
EndIf
