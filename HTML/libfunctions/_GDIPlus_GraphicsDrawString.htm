Справочник функций

_GDIPlus_GraphicsDrawString

Рисует строку текста.

#include <GDIPlus.au3>
_GDIPlus_GraphicsDrawString ( $hGraphics, $sString, $nX, $nY [, $sFont = Default [, $fSize = Default [, $iFormat = Default [, $iARGB = Default]]]] )

Параметры

$hGraphics Дескриптор объекта Graphics.
$sString Рисуемая строка.
$nX Координата X, в которой рисуется строка.
$nY Координата Y, в которой рисуется строка.
$sFont [необязательный] Название шрифта для рисования. По умолчанию "Arial".
$fSize [необязательный] Размер шрифта для рисования. По умолчанию 10.
$iFormat [необязательный] Флаги формата. Допускается комбинация. По умолчанию 0:
0x0001 – направление чтения справа налево
0x0002 – строки текста рисуются вертикально
0x0004 – части символов могут выступать за прямоугольник компоновки
0x0020 – управляющие символы Unicode отображаются символом-заменителем
0x0400 – для неподдерживаемых символов используется альтернативный шрифт
0x0800 – пробел в конце строки включается в измерение строки
0x1000 – перенос текста на следующую строку отключён
0x2000 – в прямоугольник компоновки помещаются только целые строки
0x4000 – символы, выступающие за прямоугольник, отображаются
$iARGB [необязательный] Цвет шрифта в формате ARGB. По умолчанию 0xFF000000 (чёрный).
Если альфа-канал равен 0, он автоматически устанавливается в 0xFF во избежание пустого отображения.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_GraphicsDrawStringEx

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hGUI, $hGraphic

    ; Создать GUI
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState(@SW_SHOW)

    ; Нарисовать строку
    _GDIPlus_Startup()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI)
    _GDIPlus_GraphicsDrawString($hGraphic, "Hello world", 140, 110)
    _GDIPlus_GraphicsDrawString($hGraphic, "Hello colored world", 140, 210, Default, Default, Default, 0xFF007F)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
