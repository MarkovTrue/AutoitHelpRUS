Справочник функций

_WinAPI_ShellUserAuthenticationDlg

Создаёт и отображает настраиваемое диалоговое окно, которое принимает информацию учётных данных от пользователя.

#include <WinAPIDlg.au3>
_WinAPI_ShellUserAuthenticationDlg ( $sCaption, $sMessage, $sUser, $sPassword, $sTarget [, $iFlags = 0 [, $iError = 0 [, $bSave = False [, $hBitmap = 0 [, $hParent = 0]]]]] )

Параметры

$sCaption Заголовок диалогового окна.
$sMessage Краткое сообщение для отображения в диалоговом окне.
$sUser Имя пользователя для заполнения полей учётных данных в диалоговом окне. Для пользователей домена строка должна быть в следующем формате (если домен не указан, строка назначения используется как домен):
DomainName\UserName.
$sPassword Начальный пароль.
$sTarget Имя целевого ресурса, обычно имя сервера. Этот параметр используется для идентификации информации о целевом ресурсе при сохранении и извлечении учётных данных.
$iFlags [необязательный] Флаги, которые определяют поведение этой функции. Это может быть побитовая комбинация нуля или нескольких из следующих значений. $CREDUI_FLAGS_ALWAYS_SHOW_UI $CREDUI_FLAGS_COMPLETE_USERNAME $CREDUI_FLAGS_DO_NOT_PERSIST $CREDUI_FLAGS_EXCLUDE_CERTIFICATES $CREDUI_FLAGS_EXPECT_CONFIRMATION $CREDUI_FLAGS_GENERIC_CREDENTIALS $CREDUI_FLAGS_INCORRECT_PASSWORD $CREDUI_FLAGS_KEEP_USERNAME $CREDUI_FLAGS_PASSWORD_ONLY_OK $CREDUI_FLAGS_PERSIST $CREDUI_FLAGS_REQUEST_ADMINISTRATOR $CREDUI_FLAGS_REQUIRE_CERTIFICATE $CREDUI_FLAGS_REQUIRE_SMARTCARD $CREDUI_FLAGS_SERVER_CREDENTIAL $CREDUI_FLAGS_SHOW_SAVE_CHECK_BOX $CREDUI_FLAGS_USERNAME_TARGET_CREDENTIALS $CREDUI_FLAGS_VALIDATE_USERNAME
$iError [необязательный] Код ошибки системы, который указывает, почему необходимо диалоговое окно учётных данных.
$bSave [необязательный] Указывает, установлен ли флажок "Сохранить" в диалоговом окне (имеет смысл только если используется флаг $CREDUI_FLAGS_SHOW_SAVE_CHECK_BOX), допустимые значения:     True – отмечено.     False – не отмечено (по умолчанию).
$hBitmap [необязательный] Дескриптор растрового изображения для отображения в диалоговом окне. Если этот параметр равен 0, используется растровое изображение по умолчанию. Размер растрового изображения ограничен 320x60 пиксели.
$hParent [необязательный] Диалоговое окно является модальным по отношению к родительскому окну. Если этот параметр равен 0, рабочий стол является родительским окном диалогового окна.

Возвращаемое значение

Успех: Возвращает массив, содержащий следующую информацию: [0] – имя пользователя, включая имя домена (если необходимо). [1] – пароль. [2] – состояние флажка "Сохранить".
Ошибка: Устанавливает флаг @error в ненулевое значение. Если функция отменена пользователем, флаг @extended будет содержать код ошибки системы ERROR_CANCELLED (1223). Любое другое значение указывает, что функция не смогла загрузиться.

Примечания

Учётные данные хранятся в диспетчере учётных данных на основе имени целевого ресурса. Каждое имя целевого ресурса хранится максимально общим образом без конфликтов с учётными данными, уже хранящимися в диспетчере учётных данных. Поскольку учётные данные хранятся по имени целевого ресурса, конкретный пользователь может иметь только одни учётные данные на целевой ресурс в диспетчере учётных данных.

Пример

#include "Extras\HelpFileInternals.au3"

#include <Crypt.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIDlg.au3>
#include <WinAPIRes.au3>

Local $sBmp = _Extras_PathFull('Authentication.bmp')
Local $hBitmap = _WinAPI_LoadImage(0, $sBmp, $IMAGE_BITMAP, 0, 0, $LR_LOADFROMFILE)
Local $aData[3] = ['', '', 0]

Local $sUser = 'AutoIt'
Local $sPassword = '123'

Local $sPasswordToCryptPassword = '1234'
Local $dPaswordEncrypted = StringEncrypt($sPassword, $sPasswordToCryptPassword)

While 1
    $aData = _WinAPI_ShellUserAuthenticationDlg('Authentication', 'To continue, type a login and password, and then click OK.', $aData[0], $aData[1], 'MyScript', BitOR($CREDUI_FLAGS_ALWAYS_SHOW_UI, $CREDUI_FLAGS_EXPECT_CONFIRMATION, $CREDUI_FLAGS_GENERIC_CREDENTIALS, $CREDUI_FLAGS_SHOW_SAVE_CHECK_BOX), 0, $aData[2], $hBitmap)
    If @error Then Exit

    If StringCompare($aData[0], 'AutoIt') Or StringCompare($aData[1], StringDecrypt($dPaswordEncrypted, $sPasswordToCryptPassword)) Then
        If $aData[2] Then
            _WinAPI_ConfirmCredentials('MyScript', False)
        EndIf
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'You have typed an incorrect login or password, it should be "' & $sUser & '" and "' & $sPassword & '".')
    Else
        If $aData[2] Then
            _WinAPI_ConfirmCredentials('MyScript', True)
        EndIf
        ExitLoop
    EndIf
WEnd

Func StringEncrypt($sData, $sPassword)
    _Crypt_Startup() ; Запустить библиотеку Crypt.
    Local $sReturn = _Crypt_EncryptData($sData, $sPassword, $CALG_RC4)
    _Crypt_Shutdown() ; Завершить работу библиотеки Crypt.
    Return $sReturn
EndFunc   ;==>StringEncrypt

Func StringDecrypt($sData, $dPassword)
    _Crypt_Startup() ; Запустить библиотеку Crypt.
    Local $sReturn = BinaryToString(_Crypt_DecryptData($sData, $dPassword, $CALG_RC4))
    _Crypt_Shutdown() ; Завершить работу библиотеки Crypt.
    Return $sReturn
EndFunc   ;==>StringDecrypt
