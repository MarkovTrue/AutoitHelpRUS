_GDIPlus_HatchBrushCreate

Создаёт объект HatchBrush на основе стиля штриховки, цвета переднего плана и цвета фона.

#include <GDIPlus.au3>
_GDIPlus_HatchBrushCreate ( [$iHatchStyle = 0 [, $iARGBForeground = 0xFFFFFFFF [, $iARGBBackground = 0xFFFFFFFF]]] )

Параметры

$iHatchStyle [необязательный] Узор линий штриховки, см. раздел Примечания. По умолчанию $GDIP_HATCHSTYLE_HORIZONTAL (0).
$iARGBForeground [необязательный] Компоненты Alpha, Red, Green и Blue цвета линий штриховки.
$iARGBBackground [необязательный] Компоненты Alpha, Red, Green и Blue цвета фона штриховки.

Возвращаемое значение

Успех: Возвращает указатель на новый объект HatchBrush.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

Константы узоров объявлены в GDIPlusConstants.au3 и начинаются с $GDIP_HATCHSTYLE_*.
По завершении работы с объектом HatchBrush вызовите _GDIPlus_BrushDispose() для освобождения ресурсов.

См. также

_GDIPlus_BrushDispose

Дополнительно

Искать GdipCreateHatchBrush в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hBrush, $iX, $iY

    ; Создать GUI
    Local $hGUI = GUICreate("GDI+", 810, 610)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    Local $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ; Создать графический объект из дескриптора окна

    For $i = 0 To 52
        $iX = 10 + Mod($i, 10) * 80
        $iY = 30 + Floor($i / 10) * 100
        _GDIPlus_GraphicsDrawString($hGraphic, $i, $iX, $iY - 16)

        $hBrush = _GDIPlus_HatchBrushCreate($i, 0xFF00FF00, 0xFF0000FF)
        _GDIPlus_GraphicsFillRect($hGraphic, $iX, $iY, 70, 70, $hBrush)

        _GDIPlus_BrushDispose($hBrush)
    Next

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Освободить ресурсы
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
EndFunc   ;==>Example
