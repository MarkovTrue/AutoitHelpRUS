_GDIPlus_BitmapSetPixel

Устанавливает цвет указанного пикселя растрового изображения.

#include <GDIPlus.au3>
_GDIPlus_BitmapSetPixel ( $hBitmap, $iX, $iY, $iARGB )

Параметры

$hBitmap Указатель на объект Bitmap.
$iX Координата X пикселя.
$iY Координата Y пикселя.
$iARGB Новый цвет пикселя.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

См. также

_GDIPlus_BitmapGetPixel

Дополнительно

Искать GdipBitmapSetPixel в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <ScreenCapture.au3>
#include <WinAPIHObj.au3>

Example()

Func Example()
    _GDIPlus_Startup() ;инициализировать GDI+
    Local Const $iWidth = 150, $iHeight = 150
    Local $iColor = 0
    Local $hHBmp = _ScreenCapture_Capture("", 0, @DesktopHeight - $iHeight, $iWidth, @DesktopHeight) ;создать GDI bitmap захватом области рабочего стола
    Local $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP($hHBmp) ;преобразовать GDI bitmap в GDI+
    _WinAPI_DeleteObject($hHBmp) ;освободить ресурс GDI bitmap – больше не нужен
    Local $iR, $iG, $iB, $iGrey
    For $iY = 0 To $iHeight - 1
        For $iX = 0 To $iWidth - 1
            $iColor = _GDIPlus_BitmapGetPixel($hBitmap, $iX, $iY) ; Получить текущий цвет пикселя.
            $iR = BitShift(BitAND($iColor, 0x00FF0000), 16) ; Извлечь красный цветовой канал.
            $iG = BitShift(BitAND($iColor, 0x0000FF00), 8) ; Извлечь зелёный цветовой канал.
            $iB = BitAND($iColor, 0x000000FF) ;; Извлечь синий цветовой канал.
            $iGrey = Hex(Int(($iR + $iG + $iB) / 3), 2) ; Преобразовать пиксели в формат среднего оттенка серого.
            _GDIPlus_BitmapSetPixel($hBitmap, $iX, $iY, "0xFF" & $iGrey & $iGrey & $iGrey) ; Установить пиксель в оттенке серого.
        Next
    Next
    Local $hGUI = GUICreate("GDI+ Example (" & @ScriptName & ")", $iWidth, $iHeight) ;создать тестовый GUI
    GUISetState(@SW_SHOW)

    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать графический объект из дескриптора окна
    _GDIPlus_GraphicsDrawImage($hGraphics, $hBitmap, 0, 0) ; Скопировать инвертированное изображение в графический объект (GUI).

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;освободить ресурсы GDI+
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
