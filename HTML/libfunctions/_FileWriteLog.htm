Справочник функций

_FileWriteLog

Записывает текущую дату, время и указанный текст в файл журнала.

#include <File.au3>
_FileWriteLog ( $sLogPath, $sLogMsg [, $iFlag = -1] )

Параметры

$sLogPath Путь к файлу журнала или дескриптор файла, возвращённый функцией FileOpen().
$sLogMsg Сообщение для записи в файл журнала.
$iFlag [необязательный] Определяет, куда записывать $sLogMsg: в конец или в начало файла.
Если $iFlag = -1 (по умолчанию), сообщение добавляется в конец файла.
В противном случае – в начало файла.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – ошибка при открытии указанного файла
2 – невозможно записать в файл
@extended: 1 при попытке записать в начало файла через дескриптор

Пример

Пример 1

; Открыть файл журнала для записи.

#include <File.au3>

Local $hFile = FileOpen(@TempDir & "\Example.log", 1)

_FileWriteLog($hFile, "Text 1") ; Записать в журнал, передав дескриптор, возвращённый FileOpen.
FileClose($hFile) ; Закрыть дескриптор, чтобы освободить файл.

Пример 2

; Записать в файл журнала с передачей пути к файлу.

#include <File.au3>

_FileWriteLog(@TempDir & "\Example.log", "Text 2")
