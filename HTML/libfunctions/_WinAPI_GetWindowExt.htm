_WinAPI_GetWindowExt

Получает x-размер и y-размер окна для указанного контекста устройства.

#include <WinAPIGdi.au3>
_WinAPI_GetWindowExt ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает структура $tagSIZE, содержащая x- и y-размеры в единицах координат страницы, то есть логических единицах.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
