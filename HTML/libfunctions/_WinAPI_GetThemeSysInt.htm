Справочник функций

_WinAPI_GetThemeSysInt

Получает значение системного INT.

#include <WinAPITheme.au3>
_WinAPI_GetThemeSysInt ( $hTheme, $iIntID )

Параметры

$hTheme Дескриптор данных темы.
$iIntID Значение, которое задаёт желаемый системный INT. Может быть следующим значением.
$TMT_MINCOLORDEPTH

Возвращаемое значение

Успех: Возвращает значение системного целого числа.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
