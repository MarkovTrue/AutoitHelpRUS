Справочник функций

_GDIPlus_MatrixCreate2

Создаёт и инициализирует объект Matrix на основе шести чисел, задающих аффинное преобразование.

#include <GDIPlus.au3>
_GDIPlus_MatrixCreate2 ( [$nM11 = 1 [, $nM12 = 1 [, $nM21 = 1 [, $nM22 = 1 [, $nDX = 0 [, $nDY = 0]]]]]] )

Параметры

$nM11 [необязательный] Вещественное число, задающее элемент первой строки, первого столбца (горизонтальное масштабирование). По умолчанию 1.
$nM12 [необязательный] Вещественное число, задающее элемент первой строки, второго столбца (вертикальное масштабирование). По умолчанию 1.
$nM21 [необязательный] Вещественное число, задающее элемент второй строки, первого столбца (горизонтальный сдвиг). По умолчанию 1.
$nM22 [необязательный] Вещественное число, задающее элемент второй строки, второго столбца (вертикальный сдвиг). По умолчанию 1.
$nDX [необязательный] Вещественное число, задающее элемент третьей строки, первого столбца (горизонтальный перенос). По умолчанию 0.
$nDY [необязательный] Вещественное число, задающее элемент третьей строки, второго столбца (вертикальный перенос). По умолчанию 0.

Возвращаемое значение

Успех: Возвращает указатель на новый объект Matrix.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

См. "Transformations" -> https://msdn.microsoft.com/en-us/library/windows/desktop/ms533810(v=vs.85).aspx или "Matrix Representation of Transformations" -> https://msdn.microsoft.com/en-us/library/windows/desktop/ms536397(v=vs.85).aspx.

См. также

_GDIPlus_MatrixCreate, _GDIPlus_MatrixDispose

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>

Example_1()
Example_2()

Func Example_1()
    Local $hGui = GUICreate("GDI+", 800, 400)
    GUISetBkColor(0x282828)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    Local $hMatrix_S = _GDIPlus_MatrixCreate2(2, 0, 0, 1, 0, 0) ; удвоение по горизонтали
    Local $hMatrix_T = _GDIPlus_MatrixCreate2(1, 0, 0, 1, 50, 0) ; горизонтальный перенос на 50 единиц
    Local $hBrush_Line = _GDIPlus_LineBrushCreateFromRect(_GDIPlus_RectFCreate(0, 0, 200, 100), 0xFFFF0000, 0xFF0000FF, 1)
    _GDIPlus_GraphicsFillRect($hGraphics, 0, 0, 800, 100, $hBrush_Line) ; залить большую область градиентной кистью (без преобразования).
    _GDIPlus_LineBrushSetTransform($hBrush_Line, $hMatrix_S) ; применить преобразование масштабирования.
    _GDIPlus_GraphicsFillRect($hGraphics, 0, 150, 800, 100, $hBrush_Line) ; залить большую область масштабированной градиентной кистью.
    _GDIPlus_LineBrushMultiplyTransform($hBrush_Line, $hMatrix_T) ; создать составное преобразование: сначала масштабирование, затем перенос.
    _GDIPlus_GraphicsFillRect($hGraphics, 0, 300, 800, 100, $hBrush_Line) ; залить большую область масштабированной и перенесённой градиентной кистью.

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_MatrixDispose($hMatrix_S)
    _GDIPlus_MatrixDispose($hMatrix_T)
    _GDIPlus_BrushDispose($hBrush_Line)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
    GUIDelete($hGui)
EndFunc   ;==>Example_1

Func Example_2()
    Local $hGui = GUICreate("GDI+", 800, 400)
    GUISetBkColor(0x282828)
    GUISetState(@SW_SHOW)

    _GDIPlus_Startup()
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0(800, 400)
    Local $hContext = _GDIPlus_ImageGetGraphicsContext($hBitmap)

    Local $hBrush_Line = _GDIPlus_LineBrushCreateFromRect(_GDIPlus_RectFCreate(0, 0, 200, 100), 0xFFFF0000, 0xFF0000FF)

    Local $hMatrix_S, $i = 0
    Do
        $hMatrix_S = _GDIPlus_MatrixCreate2(2.5, 0, Cos($i / 200) * 10, 1, 0, Sin($i / 50) * 75)
        _GDIPlus_LineBrushSetTransform($hBrush_Line, $hMatrix_S)

        _GDIPlus_GraphicsFillRect($hContext, 0, 150, 800, 100, $hBrush_Line)

        _GDIPlus_GraphicsDrawImageRect($hGraphics, $hBitmap, 0, 0, 800, 400)
        _GDIPlus_MatrixDispose($hMatrix_S)

        $i += 1
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_MatrixDispose($hMatrix_S)
    _GDIPlus_BrushDispose($hBrush_Line)
    _GDIPlus_GraphicsDispose($hContext)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_Shutdown()
    GUIDelete($hGui)
EndFunc   ;==>Example_2
