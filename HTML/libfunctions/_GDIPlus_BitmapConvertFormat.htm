_GDIPlus_BitmapConvertFormat

Преобразует растровое изображение в указанный формат пикселей.

#include <GDIPlus.au3>
_GDIPlus_BitmapConvertFormat ( $hBitmap, $iPixelFormat, $iDitherType, $iPaletteType, $tPalette [, $fAlphaThresholdPercent = 0.0] )

Параметры

$hBitmap Дескриптор объекта Bitmap, к которому применяется эффект.
$iPixelFormat Константа формата пикселей, задающая новый формат ($GDIP_PXF*).
$iDitherType Константа DitherType, задающая алгоритм дизеринга ($GDIP_DitherType*).
$iPaletteType Константа PaletteType, задающая стандартную палитру для дизеринга ($GDIP_PaletteType*).
$tPalette Структура, задающая палитру, индексы которой хранятся в данных пикселей преобразованного изображения.
$fAlphaThresholdPercent [необязательный] Вещественное число от 0.0 до 100.0, задающее, какие пиксели исходного изображения станут прозрачным цветом в преобразованном изображении.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).
@error: -1 – GDIPlus.dll не поддерживает эту функцию.
10 – Недопустимые параметры.

Примечания

Исходные данные пикселей в изображении заменяются новыми.

См. также

_GDIPlus_PaletteInitialize

Дополнительно

Искать GdipBitmapConvertFormat в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

_Example()

Func _Example()
    If Not _GDIPlus_Startup() Or @extended < 6 Then
        MsgBox($MB_SYSTEMMODAL, "ERROR", "GDIPlus.dll v1.1 not available")
        Return
    EndIf

    Local $sFile = FileOpenDialog("Select an image", "", "Images (*.bmp;*.png;*.jpg;*.gif;*.tif)")
    If @error Or Not FileExists($sFile) Then Return

    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile)

    Local $iWidth = 600
    Local $iHeight = _GDIPlus_ImageGetHeight($hImage) * 600 / _GDIPlus_ImageGetWidth($hImage)

    Local $hGui = GUICreate("GDI+ v1.1 (" & @ScriptName & ")", $iWidth, $iHeight)
    Local $hGraphics = _GDIPlus_GraphicsCreateFromHWND($hGui)
    GUISetState(@SW_SHOW)

    ; Преобразовать в 16-цветное изображение (4 бит/пикс) с дизерингом FixedHalftone27.
    Local $tPalette = _GDIPlus_PaletteInitialize(16, $GDIP_PaletteTypeFixedHalftone27, 16, False, $hImage)
    _GDIPlus_BitmapConvertFormat($hImage, $GDIP_PXF04INDEXED, $GDIP_DitherTypeDualSpiral8x8, $GDIP_PaletteTypeFixedHalftone27, $tPalette)

    _GDIPlus_GraphicsDrawImageRect($hGraphics, $hImage, 0, 0, $iWidth, $iHeight)

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hGraphics)
    _GDIPlus_Shutdown()
EndFunc   ;==>_Example
