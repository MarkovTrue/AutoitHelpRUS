_WinAPI_UnloadKeyboardLayout

Выгружает идентификатор локали ввода.

#include <WinAPISys.au3>
_WinAPI_UnloadKeyboardLayout ( $hLocale )

Параметры

$hLocale Идентификатор локали ввода для выгрузки.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

_WinAPI_UnloadKeyboardLayout() не может выгрузить системный идентификатор локали ввода по умолчанию, если это единственная
загруженная раскладка клавиатуры. Вы должны сначала загрузить другой идентификатор локали ввода перед выгрузкой идентификатора локали ввода по умолчанию.
