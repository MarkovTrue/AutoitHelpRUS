Справочник функций

_WinAPI_SetTimer

Создаёт таймер с указанным значением времени ожидания.

#include <WinAPISysWin.au3>
_WinAPI_SetTimer ( $hWnd, $iTimerID, $iElapse, $pTimerFunc )

Параметры

$hWnd Дескриптор окна.
$iTimerID Идентификатор таймера. Если параметр $hWnd равен 0 и $iTimerID не совпадает с существующим
таймером, он игнорируется и генерируется новый идентификатор таймера. Если параметр $hWnd не равен 0 и
окно, указанное $hWnd, уже имеет таймер со значением $iTimerID, существующий
таймер заменяется новым таймером. Когда _WinAPI_SetTimer() заменяет таймер, таймер сбрасывается.
Поэтому сообщение будет отправлено после истечения текущего значения времени ожидания, но предыдущее
установленное значение времени ожидания игнорируется. Если вызов не предназначен для замены существующего таймера,
$iTimerID должен быть 0, если $hWnd равен 0.
$iElapse Значение времени ожидания в миллисекундах.
$pTimerFunc Адрес функции обратного вызова для оповещения при истечении значения времени ожидания. Если этот
параметр равен 0, система публикует сообщение WM_TIMER в очередь приложения.
(См. MSDN для получения дополнительной информации).

Возвращаемое значение

Успех: Возвращает идентификатор таймера. Приложение может передать это значение функции _WinAPI_KillTimer() для
уничтожения таймера.
Ошибка: Возвращает 0.

Примечания

Идентификатор таймера, $iTimerID, специфичен для связанного окна. Другое окно может иметь собственный таймер
с тем же идентификатором, как таймер, принадлежащий другому окну. Таймеры различны.

Пример

Пример 1

#include <Misc.au3>
#include <WinAPISysWin.au3>

Opt('TrayAutoPause', 0)

Example()

Func Example()
    Local $hTimerProc = DllCallbackRegister('_TimerProc', 'none', 'hwnd;uint;uint_ptr;dword')

    Local $iTimerID = _WinAPI_SetTimer(0, 0, 1000, DllCallbackGetPtr($hTimerProc))

    ; закрыть скрипт нажатием {ESC}
    Do
        Sleep(100)
    Until _IsPressed('1B')

    _WinAPI_KillTimer(0, $iTimerID)
    DllCallbackFree($hTimerProc)

EndFunc   ;==>Example

Func _TimerProc($hWnd, $iMsg, $iTimerID, $iTime)
    #forceref $hWnd, $iMsg, $iTimerID, $iTime
    Local Static $iCount = 1

    ToolTip("$iCount = " & $iCount, Default, Default, "Press {ESC} to close the script")
    $iCount += 1
EndFunc   ;==>_TimerProc

Пример 2

; == Example 2 - $pTimerFunc = 0 to use a $WM_TIMER callback

#include <GUIConstantsEx.au3>
#include <Misc.au3>
#include <WinAPISysWin.au3>
#include <WindowsNotifsConstants.au3>

Opt('TrayAutoPause', 0)

Example()

Func Example()
    GUIRegisterMsg($WM_TIMER, "WM_TIMER")

    Local $hGUI = GUICreate("_WinAPI_SetTimer[2] - v(" & @AutoItVersion & ")")
    Local $iTimerID = 999
    $iTimerID = _WinAPI_SetTimer($hGUI, $iTimerID, 1000, 0)

    ; закрыть скрипт по нажатию {ESC}
    Do
        Sleep(100)
    Until _IsPressed('1B')

    _WinAPI_KillTimer($hGUI, $iTimerID)

    GUIRegisterMsg($WM_TIMER, "")
EndFunc   ;==>Example

Func WM_TIMER($hWnd, $iMsg, $iwParam, $ilParam)
    #forceref $hWnd, $iMsg, $iwParam, $ilParam
    Local Static $iCount = 1

    ToolTip("$iCount = " & $iCount, Default, Default, "Press {ESC} to close the script")
    $iCount += 1

    Return $GUI_RUNDEFMSG
EndFunc   ;==>WM_TIMER
