Справочник функций

_WinAPI_SetSystemCursor

Позволяет приложению настраивать системные курсоры.

#include <WinAPIRes.au3>
_WinAPI_SetSystemCursor ( $hCursor, $iID [, $bCopy = False] )

Параметры

$hCursor Дескриптор курсора.
$iID Этот параметр указывает системный курсор для замены содержимым $hCursor,
и может быть одним из следующих значений.
$OCR_NORMAL
$OCR_IBEAM
$OCR_WAIT
$OCR_CROSS
$OCR_UP
$OCR_SIZE
$OCR_ICON
$OCR_SIZENWSE
$OCR_SIZENESW
$OCR_SIZEWE
$OCR_SIZENS
$OCR_SIZEALL
$OCR_ICOCUR
$OCR_NO
$OCR_HAND
$OCR_APPSTARTING
$OCR_HELP
$bCopy [необязательный] Указывает, следует ли дублировать курсор, допустимые значения:
True - Курсор дублируется.
False - Курсор не дублируется (По умолчанию).

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция заменяет содержимое системного курсора, указанного ID, содержимым курсора, обработанным
$hCursor. Система уничтожает $hCursor путём вызова функции _WinAPI_DestroyCursor(). Поэтому $hCursor не может
быть курсором, загруженным с использованием функции _WinAPI_LoadCursor(). Для указания курсора, загруженного из ресурса, скопируйте
курсор с помощью функции _WinAPI_CopyCursor(), а затем передайте копию в _WinAPI_SetSystemCursor().

См. также

_WinAPI_CopyCursor, _WinAPI_DestroyCursor, _WinAPI_LoadCursor

Дополнительно

Искать SetSystemCursor в библиотеке MSDN

Пример

#include "Extras\HelpFileInternals.au3"

#include <WinAPIRes.au3>

Local $hPrev = _WinAPI_CopyCursor(_WinAPI_LoadCursor(0, $OCR_NORMAL))

Local $sCur = _Extras_PathFull('Lens.cur')
_WinAPI_SetSystemCursor(_WinAPI_LoadCursorFromFile($sCur), $OCR_NORMAL)
Sleep(5000)
_WinAPI_SetSystemCursor($hPrev, $OCR_NORMAL)
