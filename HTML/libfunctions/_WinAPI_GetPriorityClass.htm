_WinAPI_GetPriorityClass

Получает класс приоритета для указанного процесса.

#include <WinAPIProc.au3>
_WinAPI_GetPriorityClass ( [$iPID = 0] )

Параметры

$iPID [необязательный] PID процесса. По умолчанию (0) – текущий процесс.

Возвращаемое значение

Успех: Возвращает класс приоритета указанного процесса ($*_PRIORITY_CLASS).
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Константы $*_PRIORITY_CLASS требуют #include <APIShellExConstants.au3>
