_WinAPI_GetErrorMessage

Получает текстовое сообщение об ошибке для указанного кода системной ошибки.

#include <WinAPIError.au3>
_WinAPI_GetErrorMessage ( $iCode [, $iLanguage = 0] )

Параметры

$iCode Код системной ошибки, для которого требуется получить сообщение.
$iLanguage [необязательный] Идентификатор языка.

Возвращаемое значение

Успех: Возвращает запрашиваемое сообщение.
Ошибка: Возвращает пустую строку.

Примечания

При возврате значения @error и @extended сохраняются.
