Справочник функций

_Word_DocFindReplace

Запускает указанную операцию поиска и замены.

#include <Word.au3>
_Word_DocFindReplace ( $oDoc [, $sFindText = "" [, $sReplaceWith = "" [, $iReplace = $WdReplaceAll [, $vSearchRange = 0 [, $bMatchCase = False [, $bMatchWholeWord = False [, $bMatchWildcards = False [, $bMatchSoundsLike = False [, $bMatchAllWordForms = False [, $bForward = True [, $iWrap = $WdFindContinue [, $bFormat = False]]]]]]]]]]]] )

Параметры

$oDoc Объект документа Word.
$sFindText [необязательный] Текст для поиска. Используйте пустую строку ("") для поиска только форматирования.
Вы можете искать специальные символы, указав соответствующие коды символов.
Например, "^p" соответствует отметке абзаца и "^t" соответствует символу табуляции (по умолчанию = "").
$sReplaceWith [необязательный] Текст замены. Чтобы удалить найденный текст, используйте пустую строку ("").
Вы указываете специальные символы и дополнительные критерии поиска так же, как для аргумента Find (по умолчанию = "").
$iReplace [необязательный] Сколько замен должно быть произведено: одна, все или ни одна.
Может быть константа WdReplace (по умолчанию = $WdReplaceAll).
$vSearchRange [необязательный] Указывает выделение или диапазон для поиска. Может быть:
-1 – Указывает текущее выделение
0 – Указывает весь документ (по умолчанию)
Любой объект диапазона Microsoft Word.
$bMatchCase [необязательный] Если True, поиск чувствителен к регистру (по умолчанию = False)
$bMatchWholeWord [необязательный] Если True, искать только целые слова (по умолчанию = False)
$bMatchWildcards [необязательный] Если True, текст поиска имеет специальные операторы поиска (по умолчанию = False)
$bMatchSoundsLike [необязательный] Если True, найти слова, которые звучат похоже на текст поиска (по умолчанию = False)
$bMatchAllWordForms [необязательный] Если True, найти все формы текста поиска
(например, "sit" находит "sitting" и "sat") (по умолчанию = False)
$bForward [необязательный] True для поиска вперёд (к концу документа) (по умолчанию = True)
$iWrap [необязательный] True переносит, когда достигается нижняя или верхняя часть документа, выделения или диапазона. Может быть одной из констант WdFindWrap (по умолчанию = $WdFindContinue)
$bFormat [необязательный] True для поиска форматирования в дополнение к или вместо текста поиска (по умолчанию = False)

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $oDoc не является объектом 2 – $vSearchRange не является объектом 3 – При выполнении операции замены произошла ошибка. @extended задан код ошибки COM

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ только для чтения
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Заменить "test document" на "test document with replaced text".
_Word_DocFindReplace($oDoc, "test document", "test document with replaced text")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", _
        "Error replacing text in the document." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", "Text successfully replaced.")

Пример 2

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ только для чтения
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Заменить управляющий символ абзаца на абзац + текст + абзац.
; Только изменить первое вхождение.
_Word_DocFindReplace($oDoc, "^p", "^pInserted Line^p", $wdReplaceOne)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", _
        "Error replacing text in the document." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocFindReplace Example", "Paragraph control character successfully replaced.")
