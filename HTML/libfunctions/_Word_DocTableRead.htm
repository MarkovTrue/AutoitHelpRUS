Справочник функций

_Word_DocTableRead

Читает таблицу в Microsoft Word и возвращает её содержимое как двумерный массив.

#include <Word.au3>
_Word_DocTableRead ( $oDoc, $vTable [, $iIndexBase = Default [, $sDelimiter = Default]] )

Параметры

$oDoc Объект документа Word.
$vTable Объект таблицы или индекс таблицы в коллекции таблиц.
$iIndexBase [необязательный] Может быть 0 или 1 и указывает, начинается ли массив со строки 0 или 1 (по умолчанию = 1)
$sDelimiter [необязательный] Указывает символ, используемый для разделения текста на ячейки (по умолчанию = @TAB)

Возвращаемое значение

Успех: Возвращает двумерный массив с содержимым таблицы Microsoft Word.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@error: 1 – $oDoc не является объектом
2 – Ошибка при доступе к указанному индексу таблицы в коллекции таблиц. @extended задан код ошибки COM
3 – Ошибка при доступе к указанной таблице. @extended задан код ошибки COM
4 – Ошибка при преобразовании текста в таблицу. @extended задан код ошибки COM

Примечания

Вложенные таблицы игнорируются.
Если $iIndexBase установлен в 1, то строка 0 массива содержит количество строк и столбцов.

_Word_DocTableWrite

Пример

#include "Extras\HelpFileInternals.au3"

#include <Array.au3>
#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableRead Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableRead Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Создать таблицу из массива
Local $asArray[3][3] = [[1, 2, 3], ["a", "b", "c"], ["x", "y", "z"]]
Local $oRange = _Word_DocRangeSet($oDoc, -2)
Local $oTable = _Word_DocTableWrite($oRange, $asArray, 0)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableRead Example", _
        "Error creating the table." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Прочитать первую таблицу в документе и вернуть содержимое
Local $asResult = _Word_DocTableRead($oDoc, 1, 1)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableRead Example", _
        "Error reading the table to an array." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
_ArrayDisplay($asResult, "Word UDF: _Word_DocTableRead Example")
