Справочник ключевых слов

$tagGDIPENCODERPARAMS

Структура набора параметров кодировщика GDI+.

Global Const $tagGDIPENCODERPARAMS = "uint Count;" & $tagGDIPENCODERPARAM

Параметры

Count Количество структур $tagGDIPENCODERPARAM в массиве.
$tagGDIPENCODERPARAM Первая структура $tagGDIPENCODERPARAM массива.

Примечания

Используйте такой код, чтобы запись $i была меньше $ibEntries в $tagGDIPENCODERPARAMS (Count)

; Просто для вычисления размера записи, чтобы она была ОК под x86 или x64
Local $tParam = DllStructCreate($tagGDIPENCODERPARAM)
Local $iParamSize = DllStructGetSize($tParam)

; выделить структуру $tagGDIPENCODERPARAMS с $ibEntries
Local $ibEntries = 3;
Local $tParams = DllStructCreate($tagGDIPENCODERPARAMS & "byte[$iEntrySize * ($ibEntries - 1))
DLLStructSetData($tParams, Count, $ibEntries)

; Переделить структуру записи, чтобы указать на нужную запись $i (0-базовую)
Local $i = 1;
Local $tParamI = DllStructCreate($tagGDIPENCODERPARAM, DllStructGetPtr($tParams, "GUID") + ($i * $iParamSize))

; мы можем получить доступ к выбранной записи $tagGDIPENCODERPARAM, просто используя структуру $tParam, которая была переделана

; если мы используем только одну запись, код значительно упрощается
Local $tParams = DllStructCreate($tagGDIPENCODERPARAMS)
DLLStructSetData($tParams, Count, 1)

; записьможно получить напрямую через структуру $tParams
