_GUICtrlEdit_GetFirstVisibleLine

Возвращает индекс (с нуля) верхней видимой строки в многострочном Edit.

#include <GuiEdit.au3>
_GUICtrlEdit_GetFirstVisibleLine ( $hWnd )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.

Возвращаемое значение

Возвращает индекс (с нуля) верхней видимой строки.

Примечания

Количество и длина строк в поле редактирования зависят от ширины элемента и текущей настройки переноса по словам.

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $idEdit

    ; Создать GUI
    GUICreate("Edit Get First Visible Line", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    For $x = 0 To 20
        _GUICtrlEdit_AppendText($idEdit, StringFormat("[%02d] Append to the end?", $x) & @CRLF)
    Next

    MsgBox($MB_SYSTEMMODAL, "Information", "First Visible Line: " & _GUICtrlEdit_GetFirstVisibleLine($idEdit))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
