Справочник функций

_StringBetween

Находит строки между двумя разделителями.

#include <String.au3>
_StringBetween ( $sString, $sStart, $sEnd [, $iMode = $STR_ENDISSTART [, $bCase = False]] )

Параметры

$sString Строка для поиска.
$sStart Начало искомой строки. Пустая строка означает начало сдвига поиска.
$sEnd Конец искомой строки. Пустая строка означает поиск от $sStart до конца строки.
$iMode [необязательный] Режим поиска, если $sStart = $sEnd
$STR_ENDISSTART (0) строка $sEnd в конце совпадения начинает следующее возможное совпадение (по умолчанию)
$STR_ENDNOTSTART (1) дальнейший экземпляр $sStart начинает следующее совпадение.
$bCase [необязательный] False (по умолчанию) = без учёта регистра. True = с учётом регистра.

Возвращаемое значение

Успех: Возвращает массив с индексом, начинающимся с 0 – элемент [0] содержит первую найденную строку.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – Строки не найдены.

Пример

#include <Array.au3>
#include <String.au3>

Example()

Func Example()
    ; Создать массив всех значений между "[" и "]".
    Local $aArray = _StringBetween("[18][20][3][5][500][60]", "[", "]")

    ; Показать результаты через _ArrayDisplay.
    _ArrayDisplay($aArray, "Default Search")

    ; Создать массив всех значений между "|" и "|" – обратите внимание, что $sStart и $sEnd идентичны
    Local $sString = "-----|Both Modes|$STR_ENDISSTART only|Both Modes|-----"

    ; Режим $STR_ENDISSTART – по умолчанию – $sEnd является следующим $sStart
    $aArray = _StringBetween($sString, "|", "|")
    ; Показать результаты через _ArrayDisplay.
    _ArrayDisplay($aArray, "$STR_ENDISSTART")

    ; Режим $STR_ENDNOTSTART – $sEnd не является следующим $sStart
    $aArray = _StringBetween($sString, "|", "|", $STR_ENDNOTSTART)
    ; Показать результаты через _ArrayDisplay.
    _ArrayDisplay($aArray, "$STR_ENDNOTSTART")
EndFunc   ;==>Example
