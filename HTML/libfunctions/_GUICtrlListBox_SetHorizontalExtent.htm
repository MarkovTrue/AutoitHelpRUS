Справочник функций

_GUICtrlListBox_SetHorizontalExtent

Устанавливает ширину в пикселях, на которую список может прокручиваться по горизонтали.

#include <GuiListBox.au3>
_GUICtrlListBox_SetHorizontalExtent ( $hWnd, $iWidth )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iWidth Число пикселей, на которое можно прокручивать список по горизонтали.

Возвращаемое значение

Отсутствует.

Примечания

Чтобы реакция на _GUICtrlListBox_SetHorizontalExtent() сработала, список должен быть создан со стилем $WS_HSCROLL.

См. также

_GUICtrlListBox_GetHorizontalExtent

Пример

#include <GUIConstantsEx.au3>
#include <GuiListBox.au3>
#include <MsgBoxConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $idListBox

    ; Создать GUI
    GUICreate("List Box Get/Set Horizontal Extent (v" & @AutoItVersion & ")", 400, 296)
    $idListBox = GUICtrlCreateList("", 2, 2, 396, 296, BitOR($WS_BORDER, $WS_VSCROLL, $LBS_NOTIFY, $LBS_DISABLENOSCROLL, $WS_HSCROLL))
    GUISetState(@SW_SHOW)

    ; Добавить длинную строку
    _GUICtrlListBox_AddString($idListBox, "AutoIt v3 is a freeware BASIC-like scripting language designed for automating the Windows GUI.")

    ; Показать текущий горизонтальный размер
    MsgBox($MB_SYSTEMMODAL, "Information", "Horizontal Extent: " & _GUICtrlListBox_GetHorizontalExtent($idListBox))

    _GUICtrlListBox_SetHorizontalExtent($idListBox, 500)

    ; Показать текущий горизонтальный размер
    MsgBox($MB_SYSTEMMODAL, "Information", "Horizontal Extent: " & _GUICtrlListBox_GetHorizontalExtent($idListBox))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
