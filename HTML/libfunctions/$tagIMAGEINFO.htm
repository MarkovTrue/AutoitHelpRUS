Справочник ключевых слов

$tagIMAGEINFO

Содержит сведения об изображении в списке изображений.

Global Const $tagIMAGEINFO = "hwnd hBitmap;hwnd hMask;int Unused1;int Unused2;int Left;int Top;int Right;int Bottom"

Параметры

hBitmap Дескриптор растрового изображения, содержащего изображение.
hMask Дескриптор монохромного растрового изображения, содержащего маску для изображения.
Unused1 Не используется.
Unused2 Не используется.
Left Левая сторона прямоугольника изображения.
Top Верхняя часть прямоугольника изображения.
Right Правая сторона прямоугольника изображения.
Bottom Нижняя часть прямоугольника изображения.
