Справочник функций

_WinAPI_SetWindowPos

Изменяет размер, позицию и Z-порядок дочернего, всплывающего или верхнеуровневого окна.

#include <WinAPISysWin.au3>
_WinAPI_SetWindowPos ( $hWnd, $hAfter, $iX, $iY, $iCX, $iCY, $iFlags )

Параметры

$hWnd Дескриптор окна.
$hAfter Дескриптор окна, перед которым позиционируемое окно встанет в Z-порядке. Параметр должен быть
дескриптором окна или одним из значений:
$HWND_BOTTOM – помещает окно в самый низ Z-порядка
$HWND_NOTOPMOST – помещает окно ниже всех topmost-окон
$HWND_TOP – помещает окно в самый верх Z-порядка
$HWND_TOPMOST – помещает окно выше всех не-topmost окон.
$iX Новая X-координата левого края окна.
$iY Новая Y-координата верхнего края окна.
$iCX Новая ширина окна, в пикселях.
$iCY Новая высота окна, в пикселях.
$iFlags Флаги изменения размера и позиции окна:
$SWP_DRAWFRAME – рисует рамку вокруг окна
$SWP_FRAMECHANGED – отправляет окну $WM_NCCALCSIZE, даже если размер не изменился
$SWP_HIDEWINDOW – скрывает окно
$SWP_NOACTIVATE – не активирует окно
$SWP_NOCOPYBITS – отбрасывает всё содержимое клиентской области
$SWP_NOMOVE – сохраняет текущую позицию
$SWP_NOOWNERZORDER – не меняет позицию окна-владельца в Z-порядке
$SWP_NOREDRAW – не перерисовывает изменения
$SWP_NOREPOSITION – то же, что $SWP_NOOWNERZORDER
$SWP_NOSENDCHANGING – не даёт окну получить $WM_WINDOWPOSCHANGING
$SWP_NOSIZE – сохраняет текущий размер
$SWP_NOZORDER – сохраняет текущий Z-порядок
$SWP_SHOWWINDOW – показывает окно.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Перечисленные константы требуют #include <AutoItConstants.au3>

Дополнительно

Искать SetWindowPos в библиотеке MSDN

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPISysWin.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    ; Записать в локальную переменную дескриптор окна Notepad
    Local $hWnd = WinGetHandle("[CLASS:Notepad]")

    ; Если окно не существует, показать сообщение и вернуть False.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Notepad window not found.")
        Return False
    EndIf

    ; Записать в локальную переменную стиль окна Notepad.
    Local $iStyle = _WinAPI_GetWindowLong($hWnd, $GWL_STYLE)

    ; Убрать из стиля окна MAXIMIZEBOX, MINIMIZEBOX и SIZEBOX.
    $iStyle = BitXOR($iStyle, $WS_MAXIMIZEBOX, $WS_MINIMIZEBOX, $WS_SIZEBOX)

    ; Задать стиль окна.
    _WinAPI_SetWindowLong($hWnd, $GWL_STYLE, $iStyle)

    ; Применить стиль
    _WinAPI_SetWindowPos($hWnd, $HWND_TOP, 0, 0, 0, 0, BitOR($SWP_FRAMECHANGED, $SWP_NOMOVE, $SWP_NOSIZE))
EndFunc   ;==>Example
