Справочник функций

_Security__OpenThreadToken

Открывает токен доступа, связанный с потоком.

#include <Security.au3>
_Security__OpenThreadToken ( $iAccess [, $hThread = 0 [, $bOpenAsSelf = False]] )

Параметры

$iAccess Маска доступа, определяющая запрашиваемые типы доступа к токену доступа.
Эти запрашиваемые типы доступа согласуются с дискреционным списком управления доступом (DACL) токена для определения того, какой доступ разрешён или запрещён.
$hThread [необязательный] Дескриптор потока, чей токен доступа открывается
$bOpenAsSelf [необязательный] Указывает, должна ли проверка доступа выполняться в контексте безопасности потока, вызывающего функцию OpenThreadToken, или в контексте безопасности процесса для вызывающего потока.
Если этот параметр False – проверка доступа выполняется с использованием контекста безопасности вызывающего потока.
Если поток имперсонирует клиента, этот контекст безопасности может быть контекстом процесса клиента.
Если этот параметр True – проверка доступа выполняется с использованием контекста безопасности процесса для вызывающего потока.

Возвращаемое значение

Успех: Возвращает дескриптор вновь открытого токена доступа.
Ошибка: Возвращает 0.

Примечания

OpenThreadToken не удастся выполнить, если не происходит имперсонация.

См. также

_Security__OpenProcessToken, _Security__OpenThreadTokenEx

Пример

#include <Security.au3>
#include <WinAPIError.au3>
#include <WinAPIHObj.au3>

Local $hToken = _Security__OpenThreadToken($TOKEN_ADJUST_PRIVILEGES)
If $hToken Then
    ConsoleWrite("$hToken is " & $hToken & @CRLF)
    ; $hToken – токен этого потока с доступом $TOKEN_ADJUST_PRIVILEGES

    ; ... Остаток работы с токеном здесь...

    _WinAPI_CloseHandle($hToken)
Else
    ConsoleWrite(_WinAPI_GetLastErrorMessage())
    ; Прочитать примечания к _Security__OpenThreadToken
EndIf
