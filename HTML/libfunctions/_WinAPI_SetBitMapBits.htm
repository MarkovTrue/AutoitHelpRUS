_WinAPI_SetBitmapBits

Устанавливает биты данных о цвете растра на указанные значения.

#include <WinAPIGdi.au3>
_WinAPI_SetBitmapBits ( $hBitmap, $iSize, $pBits )

Параметры

$hBitmap Дескриптор растра для установки. Это должен быть совместимый растр (DDB).
$iSize Количество байтов, на которые указывает параметр $pBits.
$pBits Указатель на массив байтов, содержащих данные о цвете указанного растра.

Возвращаемое значение

Успех: Возвращает количество байтов, использованных при установке битов растра.
Ошибка: Возвращает 0.
