Справочник функций

_Excel_RangeLinkAddRemove

Добавляет гиперссылку в указанный диапазон или удаляет её.

#include <Excel.au3>
_Excel_RangeLinkAddRemove ( $oWorkbook, $vWorksheet, $vRange, $sAddress [, $sSubAddress = Default [, $sScreenTip = Default [, $sTextToDisplay = Default]]] )

Параметры

$oWorkbook Объект рабочей книги Excel.
$vWorksheet Имя, индекс или объект листа. При значении Default используется активный лист.
$vRange Объект Range или диапазон A1, для которого задаётся гиперссылка.
$sAddress Адрес гиперссылки: адрес электронной почты, интернет-адрес или имя файла.
"" – удалить существующую гиперссылку.
$sSubAddress [необязательный] Имя объекта внутри целевого файла: закладка, именованный диапазон или номер слайда
(по умолчанию Default = нет).
$sScreenTip [необязательный] Текст всплывающей подсказки, отображаемой при наведении курсора на гиперссылку
(по умолчанию Default – используется значение $sAddress).
$sTextToDisplay [необязательный] Текст, отображаемый в ячейке вместо адреса ссылки (по умолчанию Default = нет).

Возвращаемое значение

Успех: Возвращает объект гиперссылки при добавлении ссылки или 1 при её удалении.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом или объектом книги
2 – имя или индекс $vWorksheet недопустимы либо $vWorksheet не является объектом листа; @extended содержит код COM-ошибки
3 – $vRange недопустим.
@extended: Содержит код COM-ошибки
4 – ошибка при добавлении/удалении гиперссылки.
@extended: Содержит код COM-ошибки.

Примечания

По всей видимости, Excel записывает $sTextToDisplay только в левую верхнюю ячейку диапазона (если она пустая). Остальные ячейки диапазона остаются без изменений.

Пример

Пример 1

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"))
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Добавляем ссылку на ячейки A1:C1.
_Excel_RangeLinkAddRemove($oWorkbook, $oWorkbook.Activesheet, "A1:C1", "http://www.autoitscript.com", Default, "AutoIt Homepage")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example 1", "Error setting hyperlink." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example 1", "Links set to cells 'A1:C3'.")

; Удаляем ссылки из ячеек A1:C1.
_Excel_RangeLinkAddRemove($oWorkbook, $oWorkbook.Activesheet, "A1:C1", "")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example 2", "Error setting hyperlink." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example 2", "Links removed from cells 'A1:C3'.")

Пример 2

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
Local $oWorkbook = _Excel_BookOpen($oExcel, _Extras_PathFull("_Excel1.xls"), True)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example", "Error opening workbook '" & _Extras_PathFull("_Excel1.xls'.") & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Вставляем лист-оглавление со ссылками на все остальные листы.
; Корректно обрабатывает имена листов с пробелами.
Local $oSheet = _Excel_SheetAdd($oWorkbook, 1, True, 1, "Index")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example 3", "Error adding sheet." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
$oSheet.Range("A1").Value = "Index"
Local $iRow = 2
For $iSheet = 2 To $oWorkbook.Sheets.Count
    $oSheet.Cells($iRow, 1).Value = $iRow - 1
    $oSheet.Cells($iRow, 2).Value = $oWorkbook.Worksheets($iRow).Name
    _Excel_RangeLinkAddRemove($oWorkbook, $oSheet, $oSheet.Cells($iRow, 2), $oWorkbook.Fullname, "'" & $oSheet.Cells($iRow, 2).Value & "'!A1")
    $iRow = $iRow + 1
Next
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_RangeLinkAddRemove Example 3", "Index Sheet inserted as sheet 1.")
