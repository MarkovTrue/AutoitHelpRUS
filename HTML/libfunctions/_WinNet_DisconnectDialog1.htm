Справочник функций

_WinNet_DisconnectDialog1

Запускает диалоговое окно общего просмотра для отключения от сетевых ресурсов.

#include <WinNet.au3>
_WinNet_DisconnectDialog1 ( $hWnd, $sLocalName [, $sRemoteName = "" [, $iFlags = 1]] )

Параметры

$hWnd Дескриптор окна-владельца диалогового окна.
$sLocalName Имя локального устройства, например "F:" или "LPT1".
$sRemoteName [необязательный] Имя сетевого ресурса для отключения. Этот параметр может быть 0, если указано $sLocalName.
Когда указано $sLocalName, отключается подключение к сетевому ресурсу, переадресованному из $sLocalName.
$iFlags [необязательный] Флаги, описывающие подключение. Может быть комбинацией:
1 – Если это значение установлено, указанное подключение больше не является постоянным.
Этот флаг действует только если $sLocalName указывает локальное устройство.
2 – Если это значение НЕ установлено, система применяет принуждение при отключении.

Возвращаемое значение

Успех: 1. Если пользователь отменяет диалоговое окно, возвращаемое значение равно –1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
