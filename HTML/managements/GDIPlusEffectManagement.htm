Эффекты

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_EffectCreate Создаёт объект Effect типа, указанного параметром guid
_GDIPlus_EffectCreateBlur Создаёт объект эффекта класса Blur (размытие)
_GDIPlus_EffectCreateBrightnessContrast Создаёт объект эффекта класса BrightnessContrast (яркость и контраст)
_GDIPlus_EffectCreateColorBalance Создаёт объект эффекта класса ColorBalance (баланс цветов)
_GDIPlus_EffectCreateColorCurve Создаёт объект эффекта класса ColorCurve (кривые цвета)
_GDIPlus_EffectCreateColorLUT Создаёт объект эффекта класса ColorLUT (таблица цветовых соответствий)
_GDIPlus_EffectCreateColorMatrix Создаёт объект эффекта класса ColorMatrixEffect (цветовая матрица)
_GDIPlus_EffectCreateHueSaturationLightness Создаёт объект эффекта класса HueSaturationLightness (тон, насыщенность, светлота)
_GDIPlus_EffectCreateLevels Создаёт объект эффекта класса Levels (уровни)
_GDIPlus_EffectCreateRedEyeCorrection Создаёт объект эффекта класса RedEyeCorrection (коррекция эффекта красных глаз)
_GDIPlus_EffectCreateSharpen Создаёт объект эффекта класса Sharpen (резкость)
_GDIPlus_EffectCreateTint Создаёт объект эффекта класса Tint (тонирование)
_GDIPlus_EffectDispose Удаляет указанный объект Effect
_GDIPlus_EffectGetParameters Возвращает параметры указанного объекта Effect
_GDIPlus_EffectSetParameters Задаёт параметры указанного объекта Effect
