Работа с ArrowCap

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_ArrowCapCreate Создаёт настраиваемый наконечник-стрелку с заданными высотой и шириной
_GDIPlus_ArrowCapDispose Освобождает объект настраиваемого наконечника-стрелки
_GDIPlus_ArrowCapGetFillState Определяет, заливается ли наконечник-стрелка
_GDIPlus_ArrowCapGetHeight Возвращает высоту наконечника-стрелки
_GDIPlus_ArrowCapGetMiddleInset Возвращает значение среднего отступа наконечника-стрелки
_GDIPlus_ArrowCapGetWidth Возвращает ширину наконечника-стрелки
_GDIPlus_ArrowCapSetFillState Устанавливает, заливается ли наконечник-стрелка
_GDIPlus_ArrowCapSetHeight Устанавливает высоту наконечника-стрелки
_GDIPlus_ArrowCapSetMiddleInset Устанавливает значение среднего отступа наконечника-стрелки.
_GDIPlus_ArrowCapSetWidth Устанавливает ширину наконечника-стрелки
