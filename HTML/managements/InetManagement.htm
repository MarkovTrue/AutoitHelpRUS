Работа с интернетом

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Inet.au3>.

 

Функция Описание
_GetIP Получает публичный IP-адрес сети или компьютера
_INetExplorerCapable Преобразует строку в формат, совместимый с Internet Explorer
_INetGetSource Получает исходный код страницы по URL без создания временного файла
_INetMail Открывает почтовый клиент по умолчанию с указанным адресом, темой и телом письма
_INetSmtpMail Отправляет электронное письмо без использования внешней почтовой программы
_TCPIpToName Преобразует IP-адрес в имя хоста
