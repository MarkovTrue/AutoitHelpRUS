Работа с кучей

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIMem.au3>.

 

Функция Описание
_WinAPI_CreateBuffer выделяет блок памяти из внутреннего кучи библиотеки.
_WinAPI_CreateBufferFromStruct выделяет блок памяти из внутреннего кучи библиотеки и инициализирует его данными структуры.
_WinAPI_CreateString копирует указанную строку в новый выделенный блок памяти и возвращает его указатель.
_WinAPI_EqualMemory Сравнивает два блока памяти на совпадение указанного числа байт.
_WinAPI_FillMemory Заполняет блок памяти заданным значением.
_WinAPI_FreeMemory Освобождает блок памяти во внутренней куче библиотеки.
_WinAPI_GetMemorySize Получает размер блока памяти, выделенного из внутренней кучи библиотеки
_WinAPI_GlobalMemoryStatus Получает сведения о текущей доступной памяти
_WinAPI_IsBadCodePtr Определяет, имеет ли вызывающий процесс доступ для чтения к памяти по указанному адресу
_WinAPI_IsBadReadPtr Проверяет, имеет ли вызывающий процесс доступ для чтения к указанному диапазону памяти
_WinAPI_IsBadStringPtr Проверяет, имеет ли вызывающий процесс доступ для чтения к указанному диапазону памяти
_WinAPI_IsBadWritePtr Проверяет, имеет ли вызывающий процесс доступ для записи к указанному диапазону памяти
_WinAPI_IsMemory Определяет, указывает ли указанный указатель на блок памяти во внутренней куче библиотеки
_WinAPI_LocalFree Освобождает указанный объект локальной памяти и аннулирует его дескриптор
_WinAPI_MoveMemory Перемещает блок памяти из одного местоположения в другое
_WinAPI_ReadProcessMemory Читает память в указанном процессе
_WinAPI_WriteProcessMemory Записывает память в указанный процесс
_WinAPI_ZeroMemory Заполняет блок памяти нулями
