Темы оформления

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPITheme.au3>.

 

Функция Описание
_WinAPI_BeginBufferedPaint Начинает операцию буферизированной раскраски
_WinAPI_BufferedPaintClear Очищает указанный прямоугольник в буфере до ARGB = {0,0,0,0}
_WinAPI_BufferedPaintInit Инициализирует буферизированную раскраску для текущего потока
_WinAPI_BufferedPaintSetAlpha Устанавливает альфа на указанное значение в заданном прямоугольнике
_WinAPI_BufferedPaintUnInit Закрывает буферизированную раскраску для текущего потока
_WinAPI_CloseThemeData Закрывает дескриптор данных темы
_WinAPI_DrawThemeBackground Рисует границу и заливку, заданные визуальным стилем для указанной части элемента управления
_WinAPI_DrawThemeEdge Рисует одно или несколько рёбер прямоугольника, заданных визуальным стилем
_WinAPI_DrawThemeIcon Рисует изображение из списка изображений, применяя эффект значка, заданный визуальным стилем
_WinAPI_DrawThemeParentBackground Рисует часть родительского элемента управления, перекрытую частично прозрачным или альфа-смешанным дочерним элементом
_WinAPI_DrawThemeText Рисует текст с цветом и шрифтом, заданными визуальным стилем
_WinAPI_DrawThemeTextEx Рисует текст с цветом и шрифтом, заданными визуальным стилем
_WinAPI_EndBufferedPaint Завершает буферизованную операцию отрисовки и освобождает связанный с ней дескриптор
_WinAPI_GetBufferedPaintBits Получает указатель на растровое изображение буфера, если буфер представляет собой независимое от устройства растровое изображение (DIB).
_WinAPI_GetBufferedPaintDC Получает контекст устройства отрисовки (DC).
_WinAPI_GetBufferedPaintTargetDC Получает целевой контекст устройства (DC).
_WinAPI_GetBufferedPaintTargetRect Получает целевой прямоугольник.
_WinAPI_GetCurrentThemeName Получает имя текущей визуальной темы, цветовой схемы и размера.
_WinAPI_GetThemeAppProperties Получает флаги свойств, которые управляют применением визуальных стилей в текущем приложении
_WinAPI_GetThemeBackgroundContentRect Получает размер области содержимого для фона, определённого визуальным стилем
_WinAPI_GetThemeBackgroundExtent Вычисляет размер и расположение фона, определённого визуальным стилем, для заданной области содержимого
_WinAPI_GetThemeBackgroundRegion Вычисляет область для обычного или полупрозрачного фона, ограниченного указанным прямоугольником
_WinAPI_GetThemeBitmap Получает растровое изображение, связанное с конкретной темой, частью, состоянием и свойством
_WinAPI_GetThemeBool Получает значение свойства BOOL из раздела SysMetrics данных темы
_WinAPI_GetThemeColor Получает значение свойства цвета
_WinAPI_GetThemeDocumentationProperty Получает значение свойства темы из раздела документации указанного файла темы
_WinAPI_GetThemeEnumValue Получает значение свойства перечисляемого типа
_WinAPI_GetThemeFilename Получает значение свойства имени файла
_WinAPI_GetThemeFont Получает значение свойства шрифта
_WinAPI_GetThemeInt Получает значение свойства INT
_WinAPI_GetThemeMargins Получает значение свойства полей
_WinAPI_GetThemeMetric Получает значение свойства метрики
_WinAPI_GetThemePartSize Вычисляет исходный размер части, определённой визуальным стилем
_WinAPI_GetThemePosition Получает значение свойства позиции
_WinAPI_GetThemePropertyOrigin Получает расположение определения свойства темы для свойства
_WinAPI_GetThemeRect Получает значение свойства RECT
_WinAPI_GetThemeString Получает значение свойства строки
_WinAPI_GetThemeSysBool Получает логическое значение системной метрики
_WinAPI_GetThemeSysColor Получает значение системного цвета
_WinAPI_GetThemeSysColorBrush Получает системную кисть цвета
_WinAPI_GetThemeSysFont Получает информацию о шрифте системного шрифта
_WinAPI_GetThemeSysInt Получает значение системного INT
_WinAPI_GetThemeSysSize Получает значение системной метрики размера из данных темы
_WinAPI_GetThemeSysString Получает значение системной строки
_WinAPI_GetThemeTextExtent Вычисляет размер и расположение указанного текста при отображении в шрифте визуального стиля
_WinAPI_GetThemeTextMetrics Получает информацию о шрифте, указанном визуальным стилем для конкретной части
_WinAPI_GetThemeTransitionDuration Получает длительность для указанного перехода
_WinAPI_GetWindowTheme Получает дескриптор темы для окна с применёнными визуальными стилями
_WinAPI_IsThemeActive Проверяет, активен ли визуальный стиль для текущего приложения
_WinAPI_IsThemeBackgroundPartiallyTransparent Получает информацию о том, имеет ли фон, заданный визуальным стилем, прозрачные части или части со смешиванием альфа
_WinAPI_IsThemePartDefined Получает информацию о том, задала ли визуальный стиль параметры для указанной части
_WinAPI_OpenThemeData Открывает данные темы для окна и его связанного класса
_WinAPI_SetThemeAppProperties Задаёт флаги, определяющие способ применения визуальных стилей в вызывающем приложении
_WinAPI_SetWindowTheme Указывает окну использовать визуальный стиль, отличный от штатного для его класса.
