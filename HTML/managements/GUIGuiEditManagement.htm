Работа с GuiEdit

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiEdit.au3>.

 

Функция Описание
_GUICtrlEdit_AppendText Добавляет текст
_GUICtrlEdit_BeginUpdate Блокирует обновление элемента управления до вызова функции _GUICtrlEdit_EndUpdate()
_GUICtrlEdit_CanUndo Проверяет, есть ли действия в очереди отмены Edit.
_GUICtrlEdit_CharFromPos Получает информацию о символе, ближайшем к указанной точке клиентской области
_GUICtrlEdit_Create Создаёт элемент управления Edit
_GUICtrlEdit_Destroy Удаляет элемент управления Edit
_GUICtrlEdit_EmptyUndoBuffer Сбрасывает флаг отмены Edit и очищает очередь отмены.
_GUICtrlEdit_EndUpdate Включает перерисовку экрана, которая была отключена функцией _GUICtrlEdit_BeginUpdate()
_GUICtrlEdit_Find Инициирует диалоговое окно поиска
_GUICtrlEdit_FmtLines Включает или выключает добавление символов мягкого переноса в текст, возвращаемый из многострочного Edit.
_GUICtrlEdit_GetCueBanner Возвращает текст подсказки (cue banner), отображаемый в Edit.
_GUICtrlEdit_GetFirstVisibleLine Возвращает индекс (с нуля) верхней видимой строки в многострочном Edit.
_GUICtrlEdit_GetLimitText Возвращает текущее ограничение длины текста для Edit.
_GUICtrlEdit_GetLine Получает строку текста из элемента управления Edit
_GUICtrlEdit_GetLineCount Возвращает количество строк в Edit.
_GUICtrlEdit_GetMargins Получает ширину левого и правого отступов
_GUICtrlEdit_GetModify Возвращает состояние флага изменения Edit.
_GUICtrlEdit_GetPasswordChar Возвращает символ пароля, который Edit показывает вместо вводимых символов.
_GUICtrlEdit_GetRECT Возвращает прямоугольник форматирования Edit (массив из 4 координат).
_GUICtrlEdit_GetRECTEx Возвращает прямоугольник форматирования Edit как структуру $tagRECT.
_GUICtrlEdit_GetSel Получает начальную и конечную позицию символов текущего выделения
_GUICtrlEdit_GetText Получает текст из элемента управления Edit
_GUICtrlEdit_GetTextLen Получает длину всего текста из элемента управления Edit
_GUICtrlEdit_HideBalloonTip Скрывает всплывающую подсказку (balloon tip), связанную с элементом редактирования
_GUICtrlEdit_InsertText Вставляет текст
_GUICtrlEdit_LineFromChar Возвращает индекс строки, содержащей указанный индекс символа.
_GUICtrlEdit_LineIndex Получает индекс символа первого символа указанной строки
_GUICtrlEdit_LineLength Получает длину строки в символах
_GUICtrlEdit_LineScroll Прокручивает текст в многострочном Edit.
_GUICtrlEdit_PosFromChar Возвращает координаты в клиентской области для указанного символа в Edit.
_GUICtrlEdit_ReplaceSel Заменяет текущее выделение в Edit на указанный текст.
_GUICtrlEdit_Scroll Прокручивает текст Edit по вертикали в указанном направлении.
_GUICtrlEdit_SetCueBanner Задаёт текст подсказки (cue banner), отображаемый в Edit.
_GUICtrlEdit_SetLimitText Задаёт ограничение длины текста, который пользователь может ввести в Edit.
_GUICtrlEdit_SetMargins Задаёт ширину левого и правого внутренних полей Edit.
_GUICtrlEdit_SetModify Устанавливает или сбрасывает флаг изменения Edit.
_GUICtrlEdit_SetPadding Задаёт внутренние отступы (padding) области редактирования Edit.
_GUICtrlEdit_SetPasswordChar Задаёт или сбрасывает символ пароля для Edit.
_GUICtrlEdit_SetReadOnly Устанавливает или удалить стиль только для чтения ($ES_READONLY)
_GUICtrlEdit_SetRECT Задаёт прямоугольник форматирования многострочного поля ввода
_GUICtrlEdit_SetRECTEx Задаёт прямоугольник форматирования многострочного поля ввода
_GUICtrlEdit_SetRECTNP Задаёт прямоугольник форматирования многострочного поля ввода
_GUICtrlEdit_SetRectNPEx Задаёт прямоугольник форматирования многострочного поля ввода
_GUICtrlEdit_SetSel Выделяет диапазон символов в Edit.
_GUICtrlEdit_SetTabStops Задаёт позиции табуляции в Edit.
_GUICtrlEdit_SetText Устанавливает текст
_GUICtrlEdit_ShowBalloonTip Показывает всплывающую подсказку (balloon tip), привязанную к Edit.
_GUICtrlEdit_Undo Отменяет последнюю операцию редактирования в Edit.
