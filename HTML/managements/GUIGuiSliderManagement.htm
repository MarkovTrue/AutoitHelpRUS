Работа с GuiSlider

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiSlider.au3>.

 

Функция Описание
_GUICtrlSlider_ClearSel Очищает текущий выбранный диапазон.
_GUICtrlSlider_ClearTics Удаляет текущие метки деления ползунка.
_GUICtrlSlider_Create Создаёт элемент управления Slider.
_GUICtrlSlider_Destroy Удаляет элемент управления.
_GUICtrlSlider_GetBuddy Получает дескриптор окна-партнёра.
_GUICtrlSlider_GetChannelRect Получает позицию.
_GUICtrlSlider_GetChannelRectEx Получает позицию.
_GUICtrlSlider_GetLineSize Получает позицию.
_GUICtrlSlider_GetLogicalTics Получает позицию.
_GUICtrlSlider_GetNumTics Получает метку деления.
_GUICtrlSlider_GetPageSize Получает позицию.
_GUICtrlSlider_GetPos Получает позицию.
_GUICtrlSlider_GetRange Получает позицию.
_GUICtrlSlider_GetRangeMax Получает позицию.
_GUICtrlSlider_GetRangeMin Получает позицию.
_GUICtrlSlider_GetSel Получает позицию.
_GUICtrlSlider_GetSelEnd Получает позицию.
_GUICtrlSlider_GetSelStart Получает позицию.
_GUICtrlSlider_GetThumbLength Получает значение.
_GUICtrlSlider_GetThumbRect Получает позицию.
_GUICtrlSlider_GetThumbRectEx Получает позицию.
_GUICtrlSlider_GetTic Получает позицию.
_GUICtrlSlider_GetTicPos Получает позицию.
_GUICtrlSlider_GetToolTips Получает значение.
_GUICtrlSlider_GetUnicodeFormat Возвращает флаг формата символов Юникода для элемента управления.
_GUICtrlSlider_SetBuddy Назначает окно связанным (buddy) для элемента-ползунка.
_GUICtrlSlider_SetLineSize Задаёт количество логических позиций, на которое перемещается ползунок.
_GUICtrlSlider_SetPageSize Задаёт количество логических позиций, на которое перемещается ползунок страницей.
_GUICtrlSlider_SetPos Устанавливает позицию.
_GUICtrlSlider_SetRange Устанавливает позицию.
_GUICtrlSlider_SetRangeMax Устанавливает позицию.
_GUICtrlSlider_SetRangeMin Устанавливает позицию.
_GUICtrlSlider_SetSel Устанавливает позицию.
_GUICtrlSlider_SetSelEnd Устанавливает позицию.
_GUICtrlSlider_SetSelStart Устанавливает позицию.
_GUICtrlSlider_SetThumbLength Устанавливает значение.
_GUICtrlSlider_SetTic Устанавливает позицию.
_GUICtrlSlider_SetTicFreq Устанавливает значение.
_GUICtrlSlider_SetTipSide Задаёт положение элемента управления ToolTip.
_GUICtrlSlider_SetToolTips Привязывает к ползунку элемент управления ToolTip.
_GUICtrlSlider_SetUnicodeFormat Устанавливает флаг формата символов Юникода для элемента управления.
