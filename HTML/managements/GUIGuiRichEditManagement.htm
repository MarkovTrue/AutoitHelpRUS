Работа с GuiRichEdit

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiRichEdit.au3>.

 

Функция Описание
_GUICtrlRichEdit_AppendText Добавляет текст в конец области клиента
_GUICtrlRichEdit_AutoDetectURL Включает или отключает автоматическое обнаружение URL
_GUICtrlRichEdit_CanPaste Можно ли вставить содержимое буфера обмена в элемент управления?
_GUICtrlRichEdit_CanPasteSpecial Можно ли вставить содержимое буфера обмена в элемент управления в обоих форматах?
_GUICtrlRichEdit_CanRedo Можно ли повторить отменённое действие?
_GUICtrlRichEdit_CanUndo Можно ли отменить действие?
_GUICtrlRichEdit_ChangeFontSize Увеличивает или уменьшает размер шрифта выделенного текста
_GUICtrlRichEdit_Copy Копирует текст в буфер обмена
_GUICtrlRichEdit_Create Создаёт элемент управления RichEdit.
_GUICtrlRichEdit_Cut Вырезает текст в буфер обмена
_GUICtrlRichEdit_Deselect Отменяет выделение текста
_GUICtrlRichEdit_Destroy Удаляет элемент управления RichEdit
_GUICtrlRichEdit_EmptyUndoBuffer Сбрасывает флаг изменения элемента управления
_GUICtrlRichEdit_FindText Ищет текст, начиная с точки вставки или опорной точки выделения.
_GUICtrlRichEdit_FindTextInRange Ищет текст в диапазоне позиций между символами.
_GUICtrlRichEdit_GetBkColor Получает цвет фона элемента управления
_GUICtrlRichEdit_GetCharAttributes Возвращает атрибуты выделенного текста
_GUICtrlRichEdit_GetCharBkColor Получает цвет фона выделенного текста или, если текст не выделен, цвет фона символа справа от точки вставки
_GUICtrlRichEdit_GetCharColor Получает цвет выделенного текста или, если текст не выделен, цвет символа справа от точки вставки
_GUICtrlRichEdit_GetCharPosFromXY Получает позицию между символами, ближайшую к указанной точке в области клиента
_GUICtrlRichEdit_GetCharPosOfNextWord Получает позицию между символами перед следующим словом
_GUICtrlRichEdit_GetCharPosOfPreviousWord Получает позицию между символами перед предыдущим словом
_GUICtrlRichEdit_GetCharWordBreakInfo Получает позицию между символами перед предыдущим словом
_GUICtrlRichEdit_GetFirstCharPosOnLine Получает позицию между символами перед первым символом строки
_GUICtrlRichEdit_GetFont Получает атрибуты шрифта выделения или, если выделения нет, в точке вставки
_GUICtrlRichEdit_GetLineCount Получает количество строк в многострочном элементе управления
_GUICtrlRichEdit_GetLineLength Получает длину строки
_GUICtrlRichEdit_GetLineNumberFromCharPos Получает номер строки, на которой находится позиция между символами
_GUICtrlRichEdit_GetNextRedo Получает имя или ID типа следующего возможного действия повтора
_GUICtrlRichEdit_GetNextUndo Получает имя или ID типа следующего возможного действия отмены
_GUICtrlRichEdit_GetNumberOfFirstVisibleLine Получает номер первой видимой строки в элементе управления
_GUICtrlRichEdit_GetParaAlignment Получает выравнивание выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaAttributes Получает атрибуты (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaBorder Получает параметры границ (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaIndents Получает параметры отступа границ (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaNumbering Получает стиль нумерации (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaShading Получает параметры затенения (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaSpacing Получает параметры расстояния (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetParaTabStops Получает табуляции (первого) выделенного абзаца или текущего абзаца
_GUICtrlRichEdit_GetPasswordChar Получает символ пароля, который элемент управления RichEdit отображает, когда пользователь вводит текст
_GUICtrlRichEdit_GetRECT Получает прямоугольник форматирования элемента управления
_GUICtrlRichEdit_GetScrollPos Получает позицию прокрутки элемента управления
_GUICtrlRichEdit_GetSel Получает нижнюю и верхнюю позиции между символами выделения
_GUICtrlRichEdit_GetSelAA Получает опорную и активную позиции между символами выделения в этом порядке
_GUICtrlRichEdit_GetSelText Получает текущий выделенный текст
_GUICtrlRichEdit_GetSpaceUnit Получает единицу измерения горизонтального и вертикального расстояния, используемую в параметрах различных функций _GUICtrlRichEdit
_GUICtrlRichEdit_GetText Получает весь текст в элементе управления.
_GUICtrlRichEdit_GetTextInLine Получает строку текста.
_GUICtrlRichEdit_GetTextInRange Получает текст из одной позиции между символами в другую
_GUICtrlRichEdit_GetTextLength Получает длину всего текста в элементе управления.
_GUICtrlRichEdit_GetVersion Получает версию Rich Edit
_GUICtrlRichEdit_GetXYFromCharPos Получает координаты XY позиции между символами
_GUICtrlRichEdit_GetZoom Получает уровень масштабирования элемента управления
_GUICtrlRichEdit_GotoCharPos Перемещает точку вставки в позицию между символами
_GUICtrlRichEdit_HideSelection Скрывает (или показывает) выделение.
_GUICtrlRichEdit_InsertText Вставляет текст в точку вставки или опорную точку выделения
_GUICtrlRichEdit_IsModified Получает состояние флага изменения элемента управления RichEdit
_GUICtrlRichEdit_IsTextSelected Выделен ли текст?
_GUICtrlRichEdit_Paste Вставляет RTF или RTF с объектами из буфера обмена
_GUICtrlRichEdit_PasteSpecial Вставляет RTF или RTF и объекты из буфера обмена
_GUICtrlRichEdit_PauseRedraw Приостанавливает перерисовку элемента управления
_GUICtrlRichEdit_Redo Повторяет последнее отменённое действие
_GUICtrlRichEdit_ReplaceText Заменяет выделенный текст.
_GUICtrlRichEdit_ResumeRedraw Возобновляет перерисовку элемента управления
_GUICtrlRichEdit_ScrollLineOrPage Прокручивает текст вверх или вниз на строку или страницу.
_GUICtrlRichEdit_ScrollLines Прокручивает текст вверх или вниз на количество строк
_GUICtrlRichEdit_ScrollToCaret Прокручивает для отображения строки, на которой находится каретка (точка вставки)
_GUICtrlRichEdit_SetBkColor Устанавливает цвет фона элемента управления
_GUICtrlRichEdit_SetCharAttributes Включает или отключает атрибут для выделенного текста или, если текст не выделен, для текста, вставленного в точку вставки.
_GUICtrlRichEdit_SetCharBkColor Устанавливает цвет фона выделенного текста или, если текст не выделен, цвет фона текста, вставленного в точку вставки
_GUICtrlRichEdit_SetCharColor Устанавливает цвет выделенного текста или, если текст не выделен, цвет текста, вставленного в точку вставки
_GUICtrlRichEdit_SetEventMask Указывает, какие сообщения уведомлений отправляются в родительское окно
_GUICtrlRichEdit_SetFont Устанавливает атрибуты шрифта выделенного текста или, если текст не выделен, атрибуты текста в позиции вставки.
_GUICtrlRichEdit_SetLimitOnText Изменяет количество символов, которые можно вводить, вставлять или вводить как формат Rich Text
_GUICtrlRichEdit_SetModified Устанавливает или очищает флаг изменения.
_GUICtrlRichEdit_SetParaAlignment Устанавливает выравнивание абзаца(ев) в текущем выделении или, если выделения нет, абзацев, вставленных в точку вставки
_GUICtrlRichEdit_SetParaAttributes Устанавливает атрибуты абзаца(ев) в текущем выделении или, если выделения нет, абзацев, вставленных в точку вставки
_GUICtrlRichEdit_SetParaBorder Устанавливает границу абзаца(ев) в текущем выделении или, если выделения нет, абзацев, вставленных в точку вставки.
_GUICtrlRichEdit_SetParaIndents Устанавливает отступы абзаца(ев) в текущем выделении или, если выделения нет, абзацев, вставленных в точку вставки.
_GUICtrlRichEdit_SetParaNumbering Устанавливает нумерацию абзаца(ев) в текущем выделении или, если выделения нет, абзаца(ев), вставленного в точку вставки.
_GUICtrlRichEdit_SetParaShading Устанавливает затенение абзаца(ев) в текущем выделении или, если выделения нет, абзацев, вставленных в точку вставки.
_GUICtrlRichEdit_SetParaSpacing Устанавливает расстояние между абзацами для абзацев с выделенным текстом или, если текст не выделен, для текста, вставленного в точку вставки
_GUICtrlRichEdit_SetParaTabStops Устанавливает табуляции абзацев с выделенным текстом или, если текст не выделен, для текста, вставленного в точку вставки.
_GUICtrlRichEdit_SetPasswordChar Устанавливает символы для отображения вместо введённых или показывает введённые символы.
_GUICtrlRichEdit_SetReadOnly Устанавливает или удаляет состояние только для чтения.
_GUICtrlRichEdit_SetRECT Устанавливает прямоугольник форматирования элемента управления RichEdit.
_GUICtrlRichEdit_SetScrollPos Прокручивает отображение так, чтобы ($ix,$iY) находилась в верхнем левом углу элемента управления
_GUICtrlRichEdit_SetSel Устанавливает нижнюю и верхнюю позицию символов выделения.
_GUICtrlRichEdit_SetSpaceUnit Устанавливает единицу измерения горизонтального и вертикального расстояния, используемую в параметрах различных функций _GUICtrlRichEdit
_GUICtrlRichEdit_SetTabStops Устанавливает табуляции для элемента управления.
_GUICtrlRichEdit_SetText Устанавливает текст элемента управления
_GUICtrlRichEdit_SetUndoLimit Устанавливает максимальное количество действий, которые можно сохранить в очереди отмены
_GUICtrlRichEdit_SetZoom Устанавливает уровень масштабирования элемента управления
_GUICtrlRichEdit_StreamFromFile Устанавливает текст в элементе управления из файла.
_GUICtrlRichEdit_StreamFromVar Устанавливает текст в элементе управления из переменной
_GUICtrlRichEdit_StreamToFile Записывает содержимое элемента управления в файл.
_GUICtrlRichEdit_StreamToVar Записывает содержимое элемента управления в переменную.
_GUICtrlRichEdit_Undo Отменяет последнюю операцию элемента управления в очереди отмены
