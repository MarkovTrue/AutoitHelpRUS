Работа со шрифтами

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_FontCreate Создаёт объект Font
_GDIPlus_FontDispose Освобождает объект Font
_GDIPlus_FontGetHeight Возвращает межстрочный интервал шрифта в единицах указанного объекта Graphics
