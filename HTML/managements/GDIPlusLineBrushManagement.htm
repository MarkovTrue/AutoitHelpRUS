Справочник пользовательских функций: LineBrush

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_LineBrushCreate Создаёт объект LinearGradientBrush по набору граничных точек и граничных цветов
_GDIPlus_LineBrushCreateFromRect Создаёт объект LinearGradientBrush по прямоугольнику и граничным цветам
_GDIPlus_LineBrushCreateFromRectWithAngle Создаёт объект LinearGradientBrush по прямоугольнику, граничным цветам и углу направления
_GDIPlus_LineBrushGetColors Возвращает начальный и конечный цвета кисти линейного градиента
_GDIPlus_LineBrushGetRect Возвращает прямоугольник, определяющий границы кисти линейного градиента
_GDIPlus_LineBrushMultiplyTransform Обновляет матрицу преобразования кисти произведением на другую матрицу
_GDIPlus_LineBrushResetTransform Сбрасывает матрицу преобразования кисти линейного градиента до единичной
_GDIPlus_LineBrushSetBlend Задаёт коэффициенты смешивания и позиции для кисти линейного градиента
_GDIPlus_LineBrushSetColors Задаёт начальный и конечный цвета кисти линейного градиента
_GDIPlus_LineBrushSetGammaCorrection Включает или отключает гамма-коррекцию для кисти линейного градиента
_GDIPlus_LineBrushSetLinearBlend Задаёт форму смешивания кисти линейного градиента в виде треугольной кривой
_GDIPlus_LineBrushSetPresetBlend Задаёт интерполируемые цвета и позиции для кисти линейного градиента
_GDIPlus_LineBrushSetSigmaBlend Задаёт форму смешивания кисти линейного градиента в виде колоколообразной кривой
_GDIPlus_LineBrushSetTransform Задаёт матрицу преобразования кисти линейного градиента
