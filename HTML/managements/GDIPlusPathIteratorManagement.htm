Работа с PathIterator

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_PathIterCreate Создаёт новый объект GraphicsPathIterator и связывает его с объектом GraphicsPath
_GDIPlus_PathIterDispose Освобождает объект GraphicsPathIterator
_GDIPlus_PathIterGetSubpathCount Возвращает количество подпутей (фигур) в пути
_GDIPlus_PathIterNextMarkerPath Возвращает следующий раздел пути, ограниченный маркерами
_GDIPlus_PathIterNextSubpathPath Возвращает следующую фигуру (подпуть) из пути, связанного с итератором
_GDIPlus_PathIterRewind Перематывает итератор на начало связанного с ним пути
