Работа с элементами управления

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_GetEffectiveClientRect Вычисляет размеры прямоугольника в клиентской области, содержащего все указанные элементы управления
_WinAPI_GetMUILanguage Получает язык, используемый обычными элементами управления для конкретного процесса
_WinAPI_InitMUILanguage Позволяет приложению указать язык для использования со стандартными элементами управления, отличающийся от языка системы
