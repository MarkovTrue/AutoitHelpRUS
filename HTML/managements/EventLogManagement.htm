Журнал событий

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <EventLog.au3>.

 

Функция Описание
_EventLog__Backup Сохраняет журнал событий в резервный файл.
_EventLog__Clear Очищает журнал событий.
_EventLog__Close Закрывает дескриптор чтения журнала событий.
_EventLog__Count Возвращает количество записей в журнале событий.
_EventLog__DeregisterSource Закрывает дескриптор записи журнала событий.
_EventLog__Full Определяет, заполнен ли журнал событий.
_EventLog__Notify Разрешает приложению получать уведомления о событиях журнала.
_EventLog__Oldest Возвращает абсолютный номер старейшей записи в журнале событий.
_EventLog__Open Открывает дескриптор журнала событий.
_EventLog__OpenBackup Открывает дескриптор резервной копии журнала событий.
_EventLog__Read Читает запись из журнала событий.
_EventLog__RegisterSource Возвращает зарегистрированный дескриптор указанного журнала событий.
_EventLog__Report Записывает запись в конец указанного журнала событий.
