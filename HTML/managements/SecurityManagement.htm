Безопасность

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Security.au3>.

 

Функция Описание
_Security__AdjustTokenPrivileges Включает или отключает права доступа в указанном маркере доступа
_Security__CreateProcessWithToken Создаёт новый процесс и его основной поток, выполняющиеся в контексте безопасности указанного токена
_Security__DuplicateTokenEx Создаёт новый токен доступа, который дублирует существующий токен
_Security__GetAccountSid Получает идентификатор безопасности (SID) для учётной записи
_Security__GetLengthSid Возвращает длину действительного SID в байтах
_Security__GetTokenInformation Получает указанный тип сведений о токене доступа
_Security__ImpersonateSelf Получает токен доступа, который имперсонирует контекст безопасности вызывающего процесса
_Security__IsValidSid Проверяет корректность SID
_Security__LookupAccountName Получает идентификатор безопасности (SID) для учётной записи и имя домена
_Security__LookupAccountSid Получает имя учётной записи для SID
_Security__LookupPrivilegeValue Получает локально уникальный идентификатор (LUID) для значения привилегии в виде 64-битного целого числа
_Security__OpenProcessToken Возвращает токен доступа, связанный с процессом
_Security__OpenThreadToken Открывает токен доступа, связанный с потоком
_Security__OpenThreadTokenEx Открывает токен доступа, связанный с потоком, при необходимости имперсонируя контекст безопасности клиента
_Security__SetPrivilege Включает или отключает локальную привилегию токена
_Security__SetTokenInformation Устанавливает различные типы информации для указанного токена доступа
_Security__SidToStringSid Преобразует двоичный SID в строку
_Security__SidTypeStr Преобразует тип SID в строковую форму
_Security__StringSidToSid Преобразует строковый SID в двоичный SID
