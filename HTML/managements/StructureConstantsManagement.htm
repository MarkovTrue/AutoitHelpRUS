Константы структур

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <StructureConstants.au3>.

 

Функция Описание
$tagBITMAPINFO Определяет размеры и цветовые сведения аппаратно-независимого точечного рисунка (DIB) на основе Windows.
$tagBITMAPINFOHEADER Определяет размеры аппаратно-независимого точечного рисунка (DIB) на основе Windows.
$tagBLENDFUNCTION Управляет смешением путём задания функций смешения для исходного и целевого точечных рисунков.
$tagCOLORSCHEME Содержит сведения о рисовании кнопок на панели инструментов или rebar.
$tagCOMBOBOXEXITEM Содержит сведения об элементе в элементе управления ComboBoxEx.
$tagDTPRANGE Задаёт диапазон дат и времени во всемирном координированном времени (UTC).
$tagEVENTLOGRECORD Содержит сведения о записи в журнале событий.
$tagFILETIME Содержит количество 100-наносекундных интервалов с 1 января 1601 года.
$tagGDIP_EFFECTPARAMS_Blur Структура параметров эффекта размытия.
$tagGDIP_EFFECTPARAMS_BrightnessContrast Структура параметров эффекта яркости и контрастности.
$tagGDIP_EFFECTPARAMS_ColorBalance Структура параметров эффекта баланса цвета.
$tagGDIP_EFFECTPARAMS_ColorCurve Структура параметров эффекта цветовой кривой.
$tagGDIP_EFFECTPARAMS_ColorLUT Структура параметров эффекта цветовой таблицы поиска (LUT).
$tagGDIP_EFFECTPARAMS_HueSaturationLightness Структура параметров эффекта оттенка, насыщенности и яркости.
$tagGDIP_EFFECTPARAMS_Levels Структура параметров эффекта уровней.
$tagGDIP_EFFECTPARAMS_RedEyeCorrection Структура параметров эффекта коррекции эффекта красных глаз.
$tagGDIP_EFFECTPARAMS_Sharpen Структура параметров эффекта повышения резкости.
$tagGDIP_EFFECTPARAMS_Tint Структура параметров эффекта тонирования.
$tagGDIPBITMAPDATA Данные точечного рисунка.
$tagGDIPCOLORMATRIX Содержит матрицу 5×5 вещественных чисел. Ряд методов класса ImageAttributes корректирует цвета изображения с помощью цветовой матрицы.
$tagGDIPENCODERPARAM Структура параметра кодировщика GDI+.
$tagGDIPENCODERPARAMS Структура набора параметров кодировщика GDI+.
$tagGDIPIMAGECODECINFO Структура сведений о кодеке изображений GDI+.
$tagGDIPRECTF Структура прямоугольника GDI+ с координатами в числах с плавающей запятой.
$tagGDIPSTARTUPINPUT Входные параметры инициализации GDI+.
$tagGDIPSTARTUPOUTPUT Выходные параметры инициализации GDI+.
$tagGETIPAddress Содержит сведения обо всех 4 полях элемента управления "IP-адрес".
$tagGUID Представляет глобально уникальный идентификатор (GUID).
$tagHARDWAREINPUT Содержит сведения о смоделированном сообщении, сгенерированном устройством ввода (кроме клавиатуры и мыши).
$tagHDITEM Содержит сведения об элементе заголовочного элемента управления.
$tagIMAGEINFO Содержит сведения об изображении в списке изображений.
$tagKBDLLHOOKSTRUCT Содержит сведения о событии нажатия клавиши на низком уровне.
$tagKEYBDINPUT Содержит сведения о событии нажатия клавиши на низком уровне.
$tagLOGFONT Определяет атрибуты шрифта.
$tagLVFINDINFO Содержит сведения, используемые при поиске элемента ListView.
$tagLVHITTESTINFO Содержит сведения о результате hit-тестирования.
$tagLVITEM Задаёт или принимает атрибуты элемента ListView.
$tagMARGINS Определяет поля окон, к которым применены визуальные стили.
$tagMCHITTESTINFO Содержит сведения о hit-тестировании точек в элементе управления "Месячный календарь".
$tagMCMONTHRANGE Возвращает сведения о дате, представляющие верхнюю и нижнюю границы отображения элемента управления "Месячный календарь".
$tagMCRANGE Содержит сведения для задания минимальной и максимальной допустимых дат в элементе управления "Месячный календарь".
$tagMCSELRANGE Задаёт дату и время во всемирном координированном времени (UTC).
$tagMENUINFO Содержит сведения о меню.
$tagMENUITEMINFO Содержит сведения о пункте меню.
$tagMOUSEINPUT Содержит сведения о событии мыши на низком уровне.
$tagNETRESOURCE Структура сетевого ресурса.
$tagNMCBEDRAGBEGIN Содержит сведения, используемые с уведомляющим сообщением $CBEN_DRAGBEGIN.
$tagNMCBEENDEDIT Содержит сведения о завершении операции редактирования в элементе управления ComboBoxEx.
$tagNMCOMBOBOXEX Содержит сведения об элементе ComboBoxEx для уведомляющих сообщений.
$tagNMDATETIMECHANGE Содержит сведения об изменении в элементе управления "Выбор даты и времени" (DTP).
$tagNMDATETIMEFORMAT Содержит сведения о части строки формата, определяющей поле обратного вызова в элементе управления "Выбор даты и времени" (DTP).
$tagNMDATETIMEFORMATQUERY Содержит сведения о поле обратного вызова элемента управления.
$tagNMDATETIMEKEYDOWN Содержит сведения для обработки уведомляющего сообщения $DTN_WMKEYDOWN.
$tagNMDATETIMESTRING Содержит сведения об операции редактирования в элементе управления "Выбор даты и времени".
$tagNMDAYSTATE Содержит сведения для обработки уведомляющего сообщения $MCN_GETDAYSTATE.
$tagNMHDDISPINFO Содержит сведения для обработки уведомляющих сообщений $HDN_GETDISPINFO.
$tagNMHDFILTERBTNCLICK Задаёт или принимает атрибуты щелчка по кнопке фильтра.
$tagNMHDR Содержит сведения об уведомляющем сообщении.
$tagNMHEADER Содержит сведения об уведомляющих сообщениях элемента управления.
$tagNMIPADDRESS Содержит сведения для уведомляющего сообщения $IPN_FIELDCHANGED.
$tagNMITEMACTIVATE Отправляется элементом управления ListView при активации элемента пользователем.
$tagNMLISTVIEW Содержит сведения об уведомляющем сообщении ListView.
$tagNMLVCUSTOMDRAW Содержит сведения об уведомляющем сообщении NM_CUSTOMDRAW, отправляемом элементом управления ListView.
$tagNMLVDISPINFO Содержит сведения об уведомляющих сообщениях $LVN_GETDISPINFO и $LVN_SETDISPINFO.
$tagNMLVFINDITEM Содержит сведения, необходимые владельцу для поиска элементов виртуального ListView.
$tagNMLVGETINFOTIP Содержит и принимает сведения об элементе ListView для отображения подсказки.
$tagNMLVKEYDOWN Содержит сведения для обработки уведомляющего сообщения $LVN_KEYDOWN.
$tagNMLVSCROLL Содержит сведения об операции прокрутки.
$tagNMMOUSE Содержит сведения, используемые с уведомляющими сообщениями мыши.
$tagNMOBJECTNOTIFY Содержит сведения для уведомляющих сообщений $TBN_GETOBJECT, $TCN_GETOBJECT, $RBN_GETOBJECT и $PSN_GETOBJECT.
$tagNMRBAUTOSIZE Содержит сведения для обработки уведомляющих сообщений $RBN_AUTOSIZE.
$tagNMREBAR Содержит сведения для обработки различных уведомляющих сообщений rebar.
$tagNMREBARAUTOBREAK Содержит сведения для уведомления $RBN_AUTOBREAK.
$tagNMREBARCHEVRON Содержит сведения для обработки уведомляющего сообщения RBN_CHEVRONPUSHED.
$tagNMREBARCHILDSIZE Содержит сведения для обработки уведомляющего сообщения RBN_CHILDSIZE.
$tagNMSELCHANGE Содержит сведения для обработки уведомляющего сообщения $MCN_SELCHANGE.
$tagNMTBHOTITEM Содержит сведения для уведомления $TBN_HOTITEMCHANGE.
$tagNMTCKEYDOWN Содержит сведения о нажатии клавиши во вкладочном элементе управления для кода уведомления $TCN_KEYDOWN.
$tagNMTOOLBAR Содержит сведения для обработки уведомляющих сообщений панели инструментов.
$tagNMTREEVIEW Содержит сведения об уведомляющем сообщении TreeView.
$tagNMTVCUSTOMDRAW Содержит сведения об уведомляющем сообщении NM_CUSTOMDRAW, отправляемом элементом управления TreeView.
$tagNMTVDISPINFO Содержит и принимает сведения для отображения элемента TreeView.
$tagNMTVGETINFOTIP Содержит и принимает сведения об элементе TreeView для отображения подсказки.
$tagNMTVITEMCHANGE Содержит сведения об изменении элемента TreeView в уведомлениях $TVN_ITEMCHANGED и $TVN_ITEMCHANGING.
$tagNMTVKEYDOWN Содержит сведения о нажатии клавиши в элементе управления TreeView для кода уведомления $TVN_KEYDOWN.
$tagOPENFILENAME Содержит сведения, используемые функциями GetOpenFileName и GetSaveFileName для инициализации диалоговых окон "Открыть" и "Сохранить как".
$tagOVERLAPPED Содержит сведения для асинхронного (перекрывающегося) ввода-вывода.
$tagPOINT Определяет координаты x и y точки.
$tagPROCESS_INFORMATION Содержит сведения о только что созданном процессе и его основном потоке.
$tagREBARBANDINFO Содержит сведения, определяющие полосу в элементе управления rebar.
$tagRECT Определяет координаты верхнего левого и нижнего правого углов прямоугольника.
$tagSCROLLBARINFO Содержит сведения о полосе прокрутки.
$tagSCROLLINFO Содержит параметры полосы прокрутки для сообщения $SBM_SETSCROLLINFO или $SBM_GETSCROLLINFO.
$tagSECURITY_ATTRIBUTES Содержит дескриптор безопасности объекта и указывает, наследуем ли дескриптор, полученный через эту структуру.
$tagSIZE Хранит упорядоченную пару целых чисел, обычно ширину и высоту прямоугольника.
$tagSTARTUPINFO Задаёт оконную станцию, рабочий стол, стандартные дескрипторы и внешний вид главного окна процесса при его создании.
$tagSYSTEMTIME Задаёт дату и время во всемирном координированном времени (UTC).
$tagTBBUTTON Содержит сведения о кнопке на панели инструментов.
$tagTBBUTTONINFO Содержит или принимает сведения об отдельной кнопке панели инструментов.
$tagTEXTMETRIC Содержит основные сведения о физическом шрифте.
$tagTIME_ZONE_INFORMATION Задаёт сведения о часовом поясе.
$tagTOKEN_PRIVILEGES Содержит сведения о наборе привилегий токена доступа.
$tagTVHITTESTINFO Содержит сведения для определения положения точки относительно элемента управления TreeView.
$tagTVITEMEX Задаёт или принимает атрибуты элемента TreeView.
$tagWIN32_FIND_DATA Содержит данные, найденные при поиске файла.
$tagWINDOWPLACEMENT Содержит сведения о размещении окна на экране.
$tagWINDOWPOS Содержит сведения о размере и положении окна.
