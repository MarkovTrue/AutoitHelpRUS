Работа с SQLite

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <SQLite.au3>.

 

Функция Описание
_SQLite_Changes Возвращает количество строк базы данных, которые были изменены последней завершённой инструкцией с этим соединением
_SQLite_Close Закрывает открытую базу данных
_SQLite_Display2DResult Возвращает или выводит в консоль отформатированный вывод двумерного массива
_SQLite_Encode Кодирует строку, число или бинарные данные для использования как BLOB в инструкциях SQLite
_SQLite_ErrCode Возвращает последний код ошибки (число)
_SQLite_ErrMsg Возвращает строку, описывающую условие ошибки для последнего вызова SQLite API sqlite3_*
_SQLite_Escape Экранирует строку или число для использования как TEXT в инструкциях SQLite
_SQLite_Exec Выполняет запрос SQLite, не обрабатывает результаты
_SQLite_FastEncode Быстрое кодирование бинарных данных (исключительно) для использования в инструкциях SQLite
_SQLite_FastEscape Быстрое экранирование строки или числа для использования как TEXT в инструкциях SQLite
_SQLite_FetchData Получает 1 строку данных из запроса, основанного на _SQLite_Query()
_SQLite_FetchNames Читает имена столбцов из запроса, основанного на _SQLite_Query()
_SQLite_ForeignKeys Изменяет ограничение внешних ключей
_SQLite_GetTable Возвращает одномерный массив, содержащий имена столбцов и данные выполненного запроса
_SQLite_GetTable2D Возвращает двумерный массив, содержащий имена столбцов и данные выполненного запроса
_SQLite_GetTableData2D Возвращает двумерный массив, содержащий данные выполненного запроса
_SQLite_LastInsertRowID Возвращает ROWID последней вставки в базу данных этим соединением
_SQLite_LibVersion Возвращает номер версии библиотеки
_SQLite_Open Открывает или создаёт базу данных SQLite
_SQLite_Query Подготавливает запрос SQLite
_SQLite_QueryFinalize Завершает запрос, основанный на _SQLite_Query(). Запрос прерывается
_SQLite_QueryReset Сбрасывает запрос, основанный на _SQLite_Query()
_SQLite_QuerySingleRow Читает первую строку результата из указанного запроса
_SQLite_SafeMode Отключает или включает безопасный режим
_SQLite_SetTimeout Устанавливает тайм-аут для обработчика занятости
_SQLite_Shutdown Выгружает SQLite.dll
_SQLite_SQLiteExe Выполняет команды в sqlite3.exe
_SQLite_Startup Загружает SQLite3.dll
_SQLite_TotalChanges Возвращает количество всех изменений (включая триггеры и внешние ключи) с начала соединения
