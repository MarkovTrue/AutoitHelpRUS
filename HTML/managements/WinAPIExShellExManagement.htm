Справочник WinAPIEx: ShellEx

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIShellEx.au3>.

 

Функция Описание
_WinAPI_DefSubclassProc Вызывает следующий обработчик в цепочке подклассов окна
_WinAPI_DllGetVersion Получает информацию о версии конкретной DLL
_WinAPI_FindExecutable Возвращает имя исполняемого файла, связанного с указанным именем файла
_WinAPI_GetAllUsersProfileDirectory Получает путь к корневой папке, содержащей данные программ, общие для всех пользователей
_WinAPI_GetDefaultUserProfileDirectory Получает путь к корню профиля пользователя по умолчанию
_WinAPI_GetWindowSubclass Получает справочные данные для обратного вызова подкласса окна
_WinAPI_RemoveWindowSubclass Удаляет обратный вызов подкласса из окна
_WinAPI_SetCurrentProcessExplicitAppUserModelID Указывает уникальный определённый приложением идентификатор модели пользователя приложения (AppUserModelID), который идентифицирует текущий процесс на панели задач
_WinAPI_SetWindowSubclass Устанавливает или обновляет функцию обратного вызова subclass окна
_WinAPI_ShellAddToRecentDocs Добавляет файл в список недавно и часто используемых элементов
_WinAPI_ShellChangeNotify Уведомляет систему о событии, выполненном приложением
_WinAPI_ShellChangeNotifyDeregister Отменяет регистрацию окна клиента
_WinAPI_ShellChangeNotifyRegister Регистрирует окно для получения уведомлений из файловой системы или оболочки
_WinAPI_ShellCreateDirectory Создаёт новую папку в файловой системе
_WinAPI_ShellEmptyRecycleBin Очищает корзину на указанном диске
_WinAPI_ShellExecute Выполняет операцию над указанным файлом
_WinAPI_ShellExecuteEx Выполняет операцию над указанным файлом
_WinAPI_ShellExtractAssociatedIcon Возвращает дескриптор значка, связанного с указанным файлом
_WinAPI_ShellExtractIcon Извлекает значок с указанным размером из указанного файла
_WinAPI_ShellFileOperation Копирует, перемещает, переименовывает или удаляет объект файловой системы
_WinAPI_ShellFlushSFCache Сбрасывает кэш специальной папки
_WinAPI_ShellGetFileInfo Получает информацию об объекте в файловой системе
_WinAPI_ShellGetIconOverlayIndex Получает индекс значка наложения в системном списке изображений
_WinAPI_ShellGetKnownFolderIDList Получает путь известной папки как структуру ITEMIDLIST
_WinAPI_ShellGetKnownFolderPath Получает полный путь известной папки
_WinAPI_ShellGetLocalizedName Получает локализованное имя файла в папке оболочки
_WinAPI_ShellGetPathFromIDList Преобразует список идентификаторов элементов в путь файловой системы
_WinAPI_ShellGetSetFolderCustomSettings Задаёт или получает пользовательские параметры папки
_WinAPI_ShellGetSettings Получает параметры состояния оболочки
_WinAPI_ShellGetSpecialFolderLocation Получает указатель на структуру ITEMIDLIST (PIDL) специальной папки
_WinAPI_ShellGetSpecialFolderPath Получает путь специальной папки
_WinAPI_ShellGetStockIconInfo Получает информацию о значках оболочки, определённых системой
_WinAPI_ShellILCreateFromPath Создаёт указатель на список идентификаторов элементов (PIDL) из пути
_WinAPI_ShellNotifyIcon Отправляет сообщение в область состояния панели задач
_WinAPI_ShellNotifyIconGetRect Получает координаты ограничивающего прямоугольника значка уведомления на экране
_WinAPI_ShellObjectProperties Вызывает команду контекстного меню "Свойства" для объекта оболочки
_WinAPI_ShellOpenFolderAndSelectItems Открывает окно проводника Windows с выбранными указанными элементами в определённой папке
_WinAPI_ShellQueryRecycleBin Получает размер корзины и количество элементов в ней для указанного диска
_WinAPI_ShellQueryUserNotificationState Проверяет состояние компьютера для текущего пользователя
_WinAPI_ShellRemoveLocalizedName Удаляет локализованное имя файла в папке оболочки
_WinAPI_ShellRestricted Определяет, действует ли указанная политика администратора
_WinAPI_ShellSetKnownFolderPath Перенаправляет известную папку на новое местоположение
_WinAPI_ShellSetLocalizedName Задаёт локализованное имя файла в папке оболочки
_WinAPI_ShellSetSettings Задаёт параметры состояния оболочки
_WinAPI_ShellUpdateImage Уведомляет оболочку об изменении изображения в системном списке изображений
