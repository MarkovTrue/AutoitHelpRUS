Дескрипторы и объекты

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIHObj.au3>.

 

Функция Описание
_WinAPI_CloseHandle Закрывает открытый дескриптор объекта
_WinAPI_DeleteObject Удаляет логическое перо, кисть, шрифт, точечный рисунок, регион или палитру.
_WinAPI_DuplicateHandle Дублирует дескриптор объекта
_WinAPI_GetCurrentObject Получает дескриптор объекта указанного типа, выбранного в указанный контекст устройства.
_WinAPI_GetCurrentProcess Возвращает псевдодескриптор текущего процесса.
_WinAPI_GetObject Получает информацию об указанном графическом объекте
_WinAPI_GetObjectInfoByHandle Получает информацию об указанном объекте
_WinAPI_GetObjectNameByHandle Получает имя указанного объекта
_WinAPI_GetObjectType Получает тип указанного объекта
_WinAPI_GetStdHandle Получает дескриптор для стандартного устройства ввода, стандартного вывода или стандартной ошибки
_WinAPI_GetStockObject Получает дескриптор к одному из предопределённых стандартных перьев, кистей, шрифтов или палитр
_WinAPI_SelectObject Выбирает объект GDI в указанный контекст устройства.
_WinAPI_SetHandleInformation Задаёт определённые свойства дескриптора объекта
