Работа с GuiScrollBars

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiScrollBars.au3>.

 

Функция Описание
_GUIScrollBars_EnableScrollBar Включает или отключает полосу прокрутки.
_GUIScrollBars_GetScrollBarInfoEx Возвращает сведения об указанной полосе прокрутки
_GUIScrollBars_GetScrollBarRect Возвращает координаты полосы прокрутки
_GUIScrollBars_GetScrollBarRGState Возвращает состояние компонента полосы прокрутки
_GUIScrollBars_GetScrollBarXYLineButton Возвращает высоту или ширину бегунка (ползунка) полосы прокрутки
_GUIScrollBars_GetScrollBarXYThumbBottom Возвращает позицию нижнего или правого края бегунка
_GUIScrollBars_GetScrollBarXYThumbTop Возвращает позицию верхнего или левого края бегунка
_GUIScrollBars_GetScrollInfo Получает информацию о прокрутке полосы прокрутки.
_GUIScrollBars_GetScrollInfoEx Получает положение ползунка полосы прокрутки.
_GUIScrollBars_GetScrollInfoMax Возвращает максимальную позицию прокрутки
_GUIScrollBars_GetScrollInfoMin Возвращает минимальную позицию прокрутки
_GUIScrollBars_GetScrollInfoPage Возвращает размер страницы
_GUIScrollBars_GetScrollInfoPos Возвращает позицию бегунка прокрутки
_GUIScrollBars_GetScrollInfoTrackPos Возвращает текущую позицию бегунка, который пользователь перетаскивает
_GUIScrollBars_GetScrollPos Получает положение полосы прокрутки.
_GUIScrollBars_GetScrollRange Получает диапазон горизонтальной полосы прокрутки.
_GUIScrollBars_Init Инициализирует полосу прокрутки.
_GUIScrollBars_ScrollWindow Прокручивает содержимое клиентской области указанного окна
_GUIScrollBars_SetScrollInfo Устанавливает информацию о прокрутке полосы прокрутки.
_GUIScrollBars_SetScrollInfoMax Задаёт максимальную позицию прокрутки
_GUIScrollBars_SetScrollInfoMin Задаёт минимальную позицию прокрутки
_GUIScrollBars_SetScrollInfoPage Задаёт размер страницы
_GUIScrollBars_SetScrollInfoPos Задаёт позицию бегунка указанной полосы прокрутки
_GUIScrollBars_SetScrollRange Устанавливает диапазон горизонтальной полосы прокрутки.
_GUIScrollBars_ShowScrollBar Показывает или скрывает полосу прокрутки.
