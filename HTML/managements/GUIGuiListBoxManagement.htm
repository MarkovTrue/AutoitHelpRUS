Работа с GuiListBox

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiListBox.au3>.

 

Функция Описание
_GUICtrlListBox_AddFile Добавляет указанный файл, содержащий список каталога
_GUICtrlListBox_AddString Добавляет строку
_GUICtrlListBox_BeginUpdate Запрещает обновление элемента управления до вызова EndUpdate
_GUICtrlListBox_ClickItem Щёлкнуть по элементу
_GUICtrlListBox_Create Создаёт элемент управления ListBox
_GUICtrlListBox_DeleteString Удаляет строку
_GUICtrlListBox_Destroy Удаляет элемент управления
_GUICtrlListBox_Dir Добавляет имена каталогов и файлов
_GUICtrlListBox_EndUpdate Включает обновление экрана, отключённое функцией BeginUpdate
_GUICtrlListBox_FindInText Ищет элемент, содержащий указанный текст в любом месте его текста
_GUICtrlListBox_FindString Ищет строку
_GUICtrlListBox_GetAnchorIndex Получает индекс элемента якоря
_GUICtrlListBox_GetCaretIndex Возвращает индекс элемента, имеющего прямоугольник фокуса
_GUICtrlListBox_GetCount Получает количество элементов
_GUICtrlListBox_GetCurSel Возвращает индекс текущего выбранного элемента
_GUICtrlListBox_GetHorizontalExtent Получает прокручиваемую ширину из ListBox
_GUICtrlListBox_GetItemData Получает значение, определённое приложением, связанное с элементом
_GUICtrlListBox_GetItemHeight Получает высоту элементов
_GUICtrlListBox_GetItemRect Получает прямоугольник, ограничивающий элемент
_GUICtrlListBox_GetItemRectEx Получает прямоугольник, ограничивающий элемент
_GUICtrlListBox_GetListBoxInfo Получает количество элементов на столбец в указанном ListBox
_GUICtrlListBox_GetLocale Возвращает текущую локаль
_GUICtrlListBox_GetLocaleCountry Получает текущий код страны
_GUICtrlListBox_GetLocaleLang Получает текущий идентификатор языка
_GUICtrlListBox_GetLocalePrimLang Извлекает основной идентификатор языка из идентификатора локали
_GUICtrlListBox_GetLocaleSubLang Извлекает идентификатор подъязыка из идентификатора локали
_GUICtrlListBox_GetSel Получает состояние выделения элемента
_GUICtrlListBox_GetSelCount Получает общее количество выделённых элементов
_GUICtrlListBox_GetSelItems Заполняет буфер массивом выделённых элементов
_GUICtrlListBox_GetSelItemsText Получает текст выделённых элементов
_GUICtrlListBox_GetText Возвращает элемент (строку) по указанному индексу
_GUICtrlListBox_GetTextLen Получает длину строки в списке
_GUICtrlListBox_GetTopIndex индекс
_GUICtrlListBox_InitStorage Резервирует память для хранения элементов
_GUICtrlListBox_InsertString Вставляет строку в список
_GUICtrlListBox_ItemFromPoint Получает индекс (отсчёт от 0) элемента, ближайшего к указанной точке
_GUICtrlListBox_ReplaceString Заменяет текст элемента
_GUICtrlListBox_ResetContent Удаляет все элементы из списка
_GUICtrlListBox_SelectString Ищет и выделяет элемент, начинающийся с указанной строки
_GUICtrlListBox_SelItemRange Выбирает один или несколько последовательных элементов в списке с множественным выбором
_GUICtrlListBox_SelItemRangeEx Выбирает один или несколько последовательных элементов в списке с множественным выбором
_GUICtrlListBox_SetAnchorIndex элемент
_GUICtrlListBox_SetCaretIndex элемент
_GUICtrlListBox_SetColumnWidth Задаёт ширину всех столбцов в пикселях
_GUICtrlListBox_SetCurSel Выделяет строку и при необходимости прокручивает список так, чтобы она была видна
_GUICtrlListBox_SetHorizontalExtent Устанавливает ширину в пикселях, на которую список может прокручиваться по горизонтали
_GUICtrlListBox_SetItemData Задаёт значение, связанное с указанным элементом
_GUICtrlListBox_SetItemHeight Устанавливает высоту элементов в пикселях.
_GUICtrlListBox_SetLocale Задаёт текущий языковой стандарт
_GUICtrlListBox_SetSel Выделяет строку (или строки) в списке с множественным выбором
_GUICtrlListBox_SetTabStops Устанавливает позиции табуляции.
_GUICtrlListBox_SetTopIndex Гарантирует, что указанный элемент списка виден
_GUICtrlListBox_Sort Повторно сортирует список, если у него задан стиль $LBS_SORT
_GUICtrlListBox_SwapString Меняет местами тексты двух элементов с указанными индексами
_GUICtrlListBox_UpdateHScroll Обновляет горизонтальную полосу прокрутки по самой длинной строке
