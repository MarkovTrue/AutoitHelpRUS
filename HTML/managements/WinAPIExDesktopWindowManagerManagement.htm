Desktop Window Manager

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_DwmDefWindowProc Стандартная оконная процедура для проверки попадания (hit testing) Диспетчера окон рабочего стола (DWM) в неклиентской области
_WinAPI_DwmEnableBlurBehindWindow Включает эффект размытия для указанного окна
_WinAPI_DwmEnableComposition Включает или отключает композицию Диспетчера окон рабочего стола (DWM)
_WinAPI_DwmExtendFrameIntoClientArea Продлевает рамку окна за клиентскую область
_WinAPI_DwmGetColorizationColor Получает текущий цвет, используемый для стеклянной композиции Диспетчера окон рабочего стола (DWM)
_WinAPI_DwmGetColorizationParameters Получает параметры окраски, используемые Диспетчером окон рабочего стола (DWM)
_WinAPI_DwmGetWindowAttribute Получает текущее значение указанного атрибута, применённого к окну
_WinAPI_DwmInvalidateIconicBitmaps Сообщает, что все ранее переданные значковые битмапы окна – и миниатюры, и Peek-представления – должны быть обновлены
_WinAPI_DwmIsCompositionEnabled Определяет, включена ли композиция Диспетчера окон рабочего стола (DWM)
_WinAPI_DwmQueryThumbnailSourceSize Возвращает исходный размер миниатюры Диспетчера окон рабочего стола (DWM)
_WinAPI_DwmRegisterThumbnail Создаёт DWM-связь миниатюры между целевым и исходным окнами
_WinAPI_DwmSetColorizationParameters Задаёт параметры окраски для Диспетчера окон рабочего стола (DWM)
_WinAPI_DwmSetIconicLivePreviewBitmap Задаёт статический значковый битмап для отображения live-preview (Peek-предпросмотра) окна или вкладки
_WinAPI_DwmSetIconicThumbnail Задаёт статический значковый битмап окна или вкладки для использования в качестве миниатюры
_WinAPI_DwmSetWindowAttribute Задаёт значение указанных атрибутов отрисовки неклиентской области, применяемых к окну
_WinAPI_DwmUnregisterThumbnail Удаляет связь миниатюры Desktop Window Manager (DWM)
_WinAPI_DwmUpdateThumbnailProperties Задаёт свойства миниатюры Диспетчера окон рабочего стола (DWM)
