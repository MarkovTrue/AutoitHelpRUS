Справочник пользовательских функций: Paint & Draw

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_BeginPaint Подготавливает указанное окно для раскраски
_WinAPI_DrawAnimatedRects Анимирует заголовок окна, чтобы показать раскрытие значка либо сворачивание/разворачивание окна
_WinAPI_DrawShadowText Рисует форматированный текст с тенью в указанном прямоугольнике
_WinAPI_EndPaint Обозначает конец отрисовки в указанном окне
_WinAPI_GetBoundsRect Получает текущий накопленный ограничивающий прямоугольник для указанного контекста устройства.
_WinAPI_GetROP2 Получает режим смешивания переднего плана для указанного контекста устройства
_WinAPI_GetUpdateRect Получает координаты прямоугольника, который полностью охватывает прямоугольник обновления указанного окна
_WinAPI_GetUpdateRgn Получает область обновления окна, копируя её в указанную область
_WinAPI_GetWindowRgnBox Получает размеры самого плотного ограничивающего прямоугольника для области окна
_WinAPI_InvalidateRgn Добавляет область в область обновления указанного окна
_WinAPI_LockWindowUpdate Отключает или включает рисование в указанном окне
_WinAPI_PaintDesktop Заполняет область отсечения в указанном контексте устройства узором рабочего стола или обоями
_WinAPI_SetBoundsRect Управляет накоплением информации об ограничивающем прямоугольнике для указанного контекста устройства
_WinAPI_SetROP2 Получает режим смешивания переднего плана указанного контекста устройства
_WinAPI_ValidateRect Удаляет прямоугольник из текущей области обновления указанного окна
_WinAPI_ValidateRgn Удаляет область из текущей области обновления указанного окна
_WinAPI_WindowFromDC Получает дескриптор окна, связанного с указанным контекстом устройства отображения (DC)
