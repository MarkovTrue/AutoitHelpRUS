Справочник пользовательских функций: Graphics

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_GraphicsClear Очищает объект Graphics заданным цветом
_GDIPlus_GraphicsCreateFromHDC Создаёт объект Graphics из контекста устройства (DC)
_GDIPlus_GraphicsCreateFromHWND Создаёт объект Graphics из дескриптора окна
_GDIPlus_GraphicsDispose Освобождает объект Graphics
_GDIPlus_GraphicsDrawArc Рисует дугу
_GDIPlus_GraphicsDrawBezier Рисует кривую Безье
_GDIPlus_GraphicsDrawClosedCurve Рисует замкнутый кардинальный сплайн
_GDIPlus_GraphicsDrawClosedCurve2 Рисует замкнутый кардинальный сплайн
_GDIPlus_GraphicsDrawCurve Рисует кардинальный сплайн
_GDIPlus_GraphicsDrawCurve2 Рисует кардинальный сплайн
_GDIPlus_GraphicsDrawEllipse Рисует эллипс
_GDIPlus_GraphicsDrawImage Рисует объект Image
_GDIPlus_GraphicsDrawImagePointsRect Рисует изображение
_GDIPlus_GraphicsDrawImageRect Рисует изображение в указанном месте
_GDIPlus_GraphicsDrawImageRectRect Рисует объект Image
_GDIPlus_GraphicsDrawLine Рисует линию
_GDIPlus_GraphicsDrawPath Рисует последовательность линий и кривых, заданных объектом GraphicsPath
_GDIPlus_GraphicsDrawPie Рисует сектор
_GDIPlus_GraphicsDrawPolygon Рисует многоугольник
_GDIPlus_GraphicsDrawRect Рисует прямоугольник
_GDIPlus_GraphicsFillClosedCurve Заливает замкнутый кардинальный сплайн
_GDIPlus_GraphicsFillClosedCurve2 Создаёт замкнутый кардинальный сплайн из массива точек и заливает его внутреннюю область кистью
_GDIPlus_GraphicsFillEllipse Заливает эллипс
_GDIPlus_GraphicsFillPath Заливает внутреннюю область контура кистью
_GDIPlus_GraphicsFillPie Заливает сектор
_GDIPlus_GraphicsFillPolygon Заливает многоугольник
_GDIPlus_GraphicsFillRect Заливает прямоугольник
_GDIPlus_GraphicsFillRegion Заливает указанную область кистью
_GDIPlus_GraphicsGetCompositingMode Возвращает режим компоновки, установленный для объекта Graphics
_GDIPlus_GraphicsGetCompositingQuality Возвращает качество компоновки, установленное для объекта Graphics
_GDIPlus_GraphicsGetDC Возвращает дескриптор контекста устройства объекта Graphics
_GDIPlus_GraphicsGetInterpolationMode Возвращает режим интерполяции, установленный для объекта Graphics
_GDIPlus_GraphicsGetSmoothingMode Возвращает качество рендеринга объекта Graphics
_GDIPlus_GraphicsGetTransform Возвращает матрицу мирового преобразования объекта Graphics
_GDIPlus_GraphicsReleaseDC Освобождает контекст устройства объекта Graphics
_GDIPlus_GraphicsResetClip Устанавливает область отсечения объекта Graphics в бесконечную область
_GDIPlus_GraphicsResetTransform Устанавливает матрицу мирового преобразования объекта Graphics в единичную матрицу
_GDIPlus_GraphicsRestore Восстанавливает состояние объекта Graphics, сохранённое предыдущим вызовом метода _GDIPlus_GraphicsSave()
_GDIPlus_GraphicsRotateTransform Обновляет матрицу мирового преобразования объекта Graphics произведением её самой и матрицы поворота
_GDIPlus_GraphicsSave Сохраняет текущее состояние (преобразования, область отсечения и параметры качества) объекта Graphics
_GDIPlus_GraphicsScaleTransform Обновляет матрицу мирового преобразования объекта Graphics произведением её самой и матрицы масштабирования
_GDIPlus_GraphicsSetClipPath Обновляет область отсечения объекта Graphics, объединяя её с областью, заданной графическим контуром
_GDIPlus_GraphicsSetClipRect Обновляет область отсечения объекта Graphics, объединяя её с прямоугольником
_GDIPlus_GraphicsSetClipRegion Обновляет область отсечения объекта Graphics, объединяя её с областью, заданной объектом Region
_GDIPlus_GraphicsSetCompositingMode Устанавливает режим компоновки объекта Graphics
_GDIPlus_GraphicsSetCompositingQuality Устанавливает качество компоновки объекта Graphics
_GDIPlus_GraphicsSetInterpolationMode Устанавливает режим интерполяции объекта Graphics
_GDIPlus_GraphicsSetPixelOffsetMode Устанавливает режим смещения пикселей объекта Graphics
_GDIPlus_GraphicsSetSmoothingMode Устанавливает качество рендеринга объекта Graphics
_GDIPlus_GraphicsSetTextRenderingHint Устанавливает параметр отрисовки текста объекта Graphics
_GDIPlus_GraphicsSetTransform Устанавливает матрицу мирового преобразования объекта Graphics
_GDIPlus_GraphicsTransformPoints Преобразует массив точек из одного пространства координат в другое
_GDIPlus_GraphicsTranslateTransform Обновляет матрицу мирового преобразования объекта Graphics произведением её самой и матрицы переноса
