Информация о системе

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_EnumPageFiles Возвращает информацию о каждом установленном файле подкачки в системе.
_WinAPI_ExpandEnvironmentStrings Раскрывает строки переменных окружения и заменяет их фактическими значениями.
_WinAPI_GetCurrentHwProfile Получает информацию о текущем аппаратном профиле локального компьютера.
_WinAPI_GetHandleInformation Получает определённые свойства дескриптора объекта
_WinAPI_GetPerformanceInfo Получает информацию о производительности
_WinAPI_GetPhysicallyInstalledSystemMemory Получает объём оперативной памяти, физически установленной на компьютере
_WinAPI_GetProcessShutdownParameters Получает параметры завершения для текущего вызываемого процесса
_WinAPI_GetStartupInfo Получает содержимое структуры STARTUPINFO, которая была указана при создании вызывающего процесса
_WinAPI_GetSystemDEPPolicy Получает параметр политики предотвращения выполнения данных (DEP) системы
_WinAPI_GetSystemInfo Получает информацию о текущей системе
_WinAPI_GetSystemMetrics Получает указанную системную метрику или параметр конфигурации системы
_WinAPI_GetSystemTimes Получает информацию о синхронизации системы
_WinAPI_GetSystemWow64Directory Получает путь системного каталога, используемого WOW64
_WinAPI_GetTickCount Получает количество миллисекунд, прошедших с момента запуска системы
_WinAPI_GetTickCount64 Получает количество миллисекунд, прошедших с момента запуска системы
_WinAPI_GetVersion Получает версию текущей операционной системы
_WinAPI_GetVersionEx Получает сведения о текущей операционной системе
_WinAPI_GetWorkArea Получает размер рабочей области на основном мониторе
_WinAPI_IsProcessorFeaturePresent Определяет, поддерживается ли указанная функция процессора текущим компьютером
_WinAPI_QueryDiskUsage Получает текущий процент использования дискового ввода-вывода
_WinAPI_QueryPerformanceCounter Получает текущее значение счётчика производительности высокого разрешения
_WinAPI_QueryPerformanceFrequency Получает частоту счётчика производительности высокого разрешения
_WinAPI_QueryProcessCycleTime Получает сумму времени цикла всех потоков указанного процесса
_WinAPI_QueryProcessorUsage Получает текущий процент использования процессора
_WinAPI_SetProcessShutdownParameters Задаёт порядок завершения для процесса относительно других процессов в системе
_WinAPI_SystemParametersInfo Получает или задаёт значение одного из параметров, применяемых ко всей системе
_WinAPI_WaitSystemIdle Ожидание, пока процессор и дисковый ввод-вывод не станут неактивными
