Справочник GDIPlus: CustomLineCap

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_CustomLineCapClone Создаёт копию объекта CustomLineCap
_GDIPlus_CustomLineCapCreate Создаёт объект CustomLineCap
_GDIPlus_CustomLineCapDispose Освобождает объект пользовательского наконечника линии
_GDIPlus_CustomLineCapGetStrokeCaps Возвращает стили наконечников для начала и конца линии
_GDIPlus_CustomLineCapSetStrokeCaps Устанавливает стили наконечников для начала и конца линии
