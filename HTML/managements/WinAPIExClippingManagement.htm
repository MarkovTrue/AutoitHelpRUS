Области отсечения

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_ExcludeClipRect Создаёт новую область отсечения, равную существующей области минус указанный прямоугольник
_WinAPI_ExtSelectClipRgn Объединяет указанную область с текущей областью отсечения
_WinAPI_GetClipBox Получает размеры ограничивающего прямоугольника видимой области.
_WinAPI_GetClipRgn Получает дескриптор, идентифицирующий текущую определённую приложением область отсечения.
_WinAPI_IntersectClipRect Создаёт новую область отсечения из пересечения текущей области отсечения и указанного прямоугольника
_WinAPI_OffsetClipRgn Перемещает область отсечения контекста устройства на указанные смещения
_WinAPI_PtVisible Определяет, находится ли указанная точка в пределах области отсечения
_WinAPI_RectVisible Определяет, находится ли какая-либо часть указанного прямоугольника в области отсечения
_WinAPI_SelectClipPath Выбирает текущий путь как область отсечения, объединяя новую область с любой существующей областью отсечения
_WinAPI_SelectClipRgn Выбирает область как текущую область отсечения для указанного контекста устройства
