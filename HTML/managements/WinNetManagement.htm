Работа с сетью Windows

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinNet.au3>.

 

Функция Описание
_WinNet_AddConnection Подключает локальное устройство к сетевому ресурсу
_WinNet_AddConnection2 Подключает локальное устройство к сетевому ресурсу
_WinNet_AddConnection3 Подключает локальное устройство к сетевому ресурсу
_WinNet_CancelConnection Отменяет существующее сетевое подключение
_WinNet_CancelConnection2 Отменяет существующее сетевое подключение
_WinNet_CloseEnum Завершает перечисление сетевых ресурсов, начатое вызовом _WinNet_OpenEnum()
_WinNet_ConnectionDialog Запускает диалоговое окно общего просмотра для подключения к сетевым ресурсам
_WinNet_ConnectionDialog1 Запускает диалоговое окно общего просмотра для подключения к сетевым ресурсам
_WinNet_DisconnectDialog Запускает диалоговое окно общего просмотра для отключения от сетевых ресурсов
_WinNet_DisconnectDialog1 Запускает диалоговое окно общего просмотра для отключения от сетевых ресурсов
_WinNet_EnumResource Продолжает перечисление сетевых ресурсов
_WinNet_GetConnection Получает имя сетевого ресурса, связанного с локальным устройством
_WinNet_GetConnectionPerformance Возвращает информацию о производительности ресурса сетевого подключения
_WinNet_GetLastError Получает самую свежую расширенную ошибку
_WinNet_GetNetworkInformation Возвращает расширённую информацию о конкретном сетевом поставщике
_WinNet_GetProviderName Получает имя поставщика для определённого типа сети
_WinNet_GetResourceInformation Идентифицирует сетевого поставщика, который владеет ресурсом
_WinNet_GetResourceParent Возвращает родителя сетевого ресурса в иерархии просмотра сети
_WinNet_GetUniversalName Преобразует путь, основанный на диске, в универсальный формат
_WinNet_GetUser Получает имя пользователя по умолчанию или имя пользователя, используемое для подключения
_WinNet_OpenEnum Начинает перечисление сетевых ресурсов или существующих подключений
_WinNet_RestoreConnection Восстанавливает подключение к сетевому ресурсу
_WinNet_UseConnection Подключает локальное устройство к сетевому ресурсу
