Работа с GuiButton

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiButton.au3>.

 

Функция Описание
_GUICtrlButton_Click Щёлкает на кнопке.
_GUICtrlButton_Create Создаёт элемент управления Button.
_GUICtrlButton_Destroy Уничтожает элемент управления Button.
_GUICtrlButton_Enable Включает или отключает элемент управления Button.
_GUICtrlButton_GetCheck Получает состояние флажка элемента управления Button.
_GUICtrlButton_GetFocus Получает фокус от элемента управления Button.
_GUICtrlButton_GetIdealSize Получает идеальный размер элемента управления Button.
_GUICtrlButton_GetImage Получает изображение элемента управления Button.
_GUICtrlButton_GetImageList Получает список изображений элемента управления Button.
_GUICtrlButton_GetNote Получает примечание элемента управления Button.
_GUICtrlButton_GetNoteLength Получает длину примечания элемента управления Button.
_GUICtrlButton_GetSplitInfo Получает информацию о разделённой кнопке.
_GUICtrlButton_GetState Получает состояние элемента управления Button.
_GUICtrlButton_GetText Получает текст элемента управления Button.
_GUICtrlButton_GetTextMargin Получает отступ текста элемента управления Button.
_GUICtrlButton_SetCheck Устанавливает состояние флажка элемента управления Button.
_GUICtrlButton_SetDontClick Предотвращает щелчок на кнопке.
_GUICtrlButton_SetFocus Устанавливает фокус на элемент управления Button.
_GUICtrlButton_SetImage Устанавливает изображение элемента управления Button.
_GUICtrlButton_SetImageList Устанавливает список изображений элемента управления Button.
_GUICtrlButton_SetNote Устанавливает примечание элемента управления Button.
_GUICtrlButton_SetShield Устанавливает для указанной кнопки или кнопки-команды состояние "требуется повышение прав" – с отображением значка щита
_GUICtrlButton_SetSize Устанавливает размер элемента управления Button.
_GUICtrlButton_SetSplitInfo Устанавливает информацию о разделённой кнопке.
_GUICtrlButton_SetState Устанавливает состояние элемента управления Button.
_GUICtrlButton_SetStyle Устанавливает стиль кнопки.
_GUICtrlButton_SetText Устанавливает текст элемента управления Button.
_GUICtrlButton_SetTextMargin Устанавливает отступ текста элемента управления Button.
_GUICtrlButton_Show Показывает элемент управления Button.
