Работа с FTP (FTPEx)

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <FTPEx.au3>.

 

Функция Описание
_FTP_Close Закрывает сессию _FTP_Open() или _FTP_Connect()
_FTP_Command Отправляет команду FTP-серверу
_FTP_Connect Устанавливает соединение с FTP-сервером
_FTP_DecodeInternetStatus Декодирует полученный статус интернет-соединения
_FTP_DirCreate Создаёт папку на FTP-сервере
_FTP_DirDelete Удаляет папку на FTP-сервере
_FTP_DirGetCurrent Возвращает текущую папку на FTP-сервере
_FTP_DirPutContents Загружает папку на FTP-сервер с возможностью рекурсивного обхода подпапок.
_FTP_DirSetCurrent Устанавливает текущую папку на FTP-сервере
_FTP_FileClose Закрывает дескриптор, возвращённый функцией _FTP_FileOpen()
_FTP_FileDelete Удаляет файл с FTP-сервера
_FTP_FileGet Загружает файл с FTP-сервера
_FTP_FileGetSize Возвращает размер файла на FTP-сервере
_FTP_FileOpen Инициирует доступ к удалённому файлу на FTP-сервере для чтения или записи
_FTP_FilePut Загружает файл на FTP-сервер
_FTP_FileRead Читает данные из дескриптора, открытого функцией _FTP_FileOpen()
_FTP_FileRename Переименовывает файл на FTP-сервере
_FTP_FileTimeLoHiToStr Преобразует LOWORD и HIWORD времени файла в строку
_FTP_FindFileClose Освобождает дескриптор поиска файлов
_FTP_FindFileFirst Ищет первый файл на FTP-сервере
_FTP_FindFileNext Ищет следующий файл на FTP-сервере
_FTP_GetLastResponseInfo Возвращает последнее описание ошибки или ответ сервера для вызывающего потока
_FTP_ListToArray Возвращает имена файлов, папок или всего содержимого текущей удалённой папки
_FTP_ListToArray2D Возвращает имена и размеры файлов текущей удалённой папки
_FTP_ListToArrayEx Возвращает имена, размеры, атрибуты и временные метки файлов/папок текущей удалённой папки
_FTP_Open Открывает FTP-сессию
_FTP_ProgressDownload Загружает файл в двоичном режиме, отображая окно прогресса или вызывая пользовательскую функцию
_FTP_ProgressUpload Отправляет файл в двоичном режиме, отображая окно прогресса или вызывая пользовательскую функцию
_FTP_SetStatusCallback Регистрирует функцию обратного вызова, которую функции WinINet вызывают по ходу выполнения операции
