Справочник пользовательских функций: Text

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_GraphicsDrawString Рисует строку текста
_GDIPlus_GraphicsDrawStringEx Рисует строку текста
_GDIPlus_GraphicsMeasureCharacterRanges Возвращает набор объектов Region, каждый из которых ограничивает диапазон позиций символов в строке
_GDIPlus_GraphicsMeasureString Измеряет размер строки текста
