Справочник пользовательских функций: Internationalization

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPILocale.au3>.

 

Функция Описание
_WinAPI_CompareString Сравнивает две строки символов для указанной локали
_WinAPI_CreateNumberFormatInfo создаёт структуру $tagNUMBERFMT с указанной информацией о формате чисел
_WinAPI_EnumSystemGeoID Перечисляет географические идентификаторы расположения (GEOID), доступные в операционной системе
_WinAPI_EnumSystemLocales Перечисляет локали, установленные или поддерживаемые операционной системой
_WinAPI_EnumUILanguages Перечисляет языки пользовательского интерфейса, доступные в операционной системе
_WinAPI_GetDateFormat Форматирует дату как строку даты для локали, указанной идентификатором локали.
_WinAPI_GetDurationFormat Форматирует длительность как строку времени по правилам указанной локали.
_WinAPI_GetGeoInfo Получает информацию об указанном географическом местоположении
_WinAPI_GetLocaleInfo Получает информацию о локали, определённой идентификатором
_WinAPI_GetNumberFormat Форматирует строку числа в строку числа, настроенную для локали, определённой идентификатором
_WinAPI_GetSystemDefaultLangID Возвращает идентификатор языка для системной локали
_WinAPI_GetSystemDefaultLCID Возвращает идентификатор локали (LCID) для системной локали
_WinAPI_GetSystemDefaultUILanguage Получает идентификатор языка пользовательского интерфейса операционной системы по умолчанию
_WinAPI_GetThreadLocale Получает идентификатор локали текущей локали для вызывающего потока
_WinAPI_GetThreadUILanguage Получает идентификатор языка первого языка пользовательского интерфейса для текущего потока
_WinAPI_GetTimeFormat Форматирует время как строку времени для локали, указанной идентификатором
_WinAPI_GetUserDefaultLangID Возвращает идентификатор языка для текущей локали пользователя
_WinAPI_GetUserDefaultLCID Возвращает идентификатор локали (LCID) пользователя по умолчанию
_WinAPI_GetUserDefaultUILanguage Возвращает идентификатор языка для язык UI пользователя текущего пользователя
_WinAPI_GetUserGeoID Получает сведения о географическом расположении пользователя
_WinAPI_IsValidLocale Определяет, установлен ли указанный языковой стандарт или поддерживается операционной системой
_WinAPI_SetLocaleInfo Задаёт информацию в пользовательской части переопределения текущей локали
_WinAPI_SetThreadLocale Задаёт текущий языковой стандарт вызывающего потока
_WinAPI_SetThreadUILanguage Задаёт язык пользовательского интерфейса для текущего потока
_WinAPI_SetUserGeoID Задаёт идентификатор географического расположения для пользователя
