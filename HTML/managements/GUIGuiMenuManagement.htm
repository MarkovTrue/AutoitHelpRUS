Работа с GuiMenu

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiMenu.au3>.

 

Функция Описание
_GUICtrlMenu_AddMenuItem Добавляет новый пункт меню в конец меню
_GUICtrlMenu_AppendMenu Добавляет новый пункт в конец строки меню, выпадающего меню, подменю или контекстного меню
_GUICtrlMenu_CalculatePopupWindowPosition Вычисляет подходящую позицию для всплывающего окна
_GUICtrlMenu_CheckMenuItem Устанавливает или снимает флажок у указанного пункта меню
_GUICtrlMenu_CheckRadioItem Отмечает указанный пункт меню как радиокнопку
_GUICtrlMenu_CreateMenu Создаёт меню
_GUICtrlMenu_CreatePopup Создаёт выпадающее меню, подменю или контекстное меню
_GUICtrlMenu_DeleteMenu Удаляет пункт из указанного меню
_GUICtrlMenu_DestroyMenu Уничтожает указанное меню и освобождает занятую им память
_GUICtrlMenu_DrawMenuBar Перерисовывает строку меню указанного окна
_GUICtrlMenu_EnableMenuItem Включает, отключает или выделяет серым указанный пункт меню
_GUICtrlMenu_FindItem Находит пункт меню по его тексту
_GUICtrlMenu_FindParent Получает окно, которому принадлежит меню
_GUICtrlMenu_GetItemBmp Получает растровое изображение, отображаемое для пункта
_GUICtrlMenu_GetItemBmpChecked Получает растровое изображение, отображаемое при выборе пункта
_GUICtrlMenu_GetItemBmpUnchecked Получает растровое изображение, отображаемое когда пункт не выбран
_GUICtrlMenu_GetItemChecked Получает состояние флажка пункта меню
_GUICtrlMenu_GetItemCount Получает количество пунктов в указанном меню
_GUICtrlMenu_GetItemData Получает значение, заданное приложением для пункта меню
_GUICtrlMenu_GetItemDefault Получает состояние пункта меню по умолчанию
_GUICtrlMenu_GetItemDisabled Получает состояние отключения пункта меню
_GUICtrlMenu_GetItemEnabled Получает состояние доступности пункта меню
_GUICtrlMenu_GetItemGrayed Получает состояние затенения пункта меню
_GUICtrlMenu_GetItemHighlighted Получает состояние выделения пункта меню
_GUICtrlMenu_GetItemID Получает идентификатор пункта меню
_GUICtrlMenu_GetItemInfo Получает информацию о пункте меню
_GUICtrlMenu_GetItemRect Получает ограничивающий прямоугольник указанного пункта меню
_GUICtrlMenu_GetItemRectEx Получает ограничивающий прямоугольник указанного пункта меню
_GUICtrlMenu_GetItemState Получает состояние пункта меню
_GUICtrlMenu_GetItemStateEx Получает флаги меню для указанного пункта
_GUICtrlMenu_GetItemSubMenu Получает подменю, активируемое указанным пунктом
_GUICtrlMenu_GetItemText Получает текст указанного пункта меню
_GUICtrlMenu_GetItemType Получает тип пункта меню
_GUICtrlMenu_GetMenu Получает дескриптор меню, назначенного указанному окну
_GUICtrlMenu_GetMenuBackground Получает кисть для рисования фона меню
_GUICtrlMenu_GetMenuBarInfo Возвращает информацию об указанной строке меню
_GUICtrlMenu_GetMenuContextHelpID Получает идентификатор контекстной справки
_GUICtrlMenu_GetMenuData Получает значение, заданное приложением для меню
_GUICtrlMenu_GetMenuDefaultItem Получает пункт по умолчанию в указанном меню
_GUICtrlMenu_GetMenuHeight Получает максимальную высоту меню
_GUICtrlMenu_GetMenuInfo Получает информацию об указанном меню
_GUICtrlMenu_GetMenuStyle Получает стиль меню
_GUICtrlMenu_GetSystemMenu Предоставляет приложению доступ к системному меню окна для его копирования и изменения
_GUICtrlMenu_InsertMenuItem Вставляет новый пункт меню в указанную позицию
_GUICtrlMenu_InsertMenuItemEx Вставляет новый пункт меню в указанную позицию
_GUICtrlMenu_IsMenu Определяет, является ли дескриптор дескриптором меню
_GUICtrlMenu_LoadMenu Загружает указанный ресурс меню из исполняемого файла
_GUICtrlMenu_MapAccelerator Сопоставляет клавишу-акселератор меню с позицией пункта в меню
_GUICtrlMenu_MenuItemFromPoint Определяет пункт меню, находящийся в указанной позиции
_GUICtrlMenu_RemoveMenu Удаляет пункт меню или отсоединяет подменю от указанного меню
_GUICtrlMenu_SetItemBitmaps Связывает указанные растровые изображения с пунктом меню
_GUICtrlMenu_SetItemBmp Устанавливает растровое изображение для пункта
_GUICtrlMenu_SetItemBmpChecked Устанавливает растровое изображение, отображаемое при выборе пункта
_GUICtrlMenu_SetItemBmpUnchecked Устанавливает растровое изображение, отображаемое когда пункт не выбран
_GUICtrlMenu_SetItemChecked Устанавливает состояние флажка пункта меню
_GUICtrlMenu_SetItemData Устанавливает значение приложения для пункта меню
_GUICtrlMenu_SetItemDefault Устанавливает состояние пункта меню по умолчанию
_GUICtrlMenu_SetItemDisabled Устанавливает состояние отключения пункта меню
_GUICtrlMenu_SetItemEnabled Устанавливает состояние доступности пункта меню
_GUICtrlMenu_SetItemGrayed Устанавливает состояние затенения пункта меню
_GUICtrlMenu_SetItemHighlighted Устанавливает состояние выделения пункта меню
_GUICtrlMenu_SetItemID Устанавливает идентификатор пункта меню
_GUICtrlMenu_SetItemInfo Изменяет информацию о пункте меню
_GUICtrlMenu_SetItemState Устанавливает состояние пункта меню
_GUICtrlMenu_SetItemSubMenu Устанавливает выпадающее меню или подменю для пункта
_GUICtrlMenu_SetItemText Устанавливает текст пункта меню
_GUICtrlMenu_SetItemType Устанавливает тип пункта меню
_GUICtrlMenu_SetMenu Назначает новое меню указанному окну
_GUICtrlMenu_SetMenuBackground Устанавливает кисть фона для меню
_GUICtrlMenu_SetMenuContextHelpID Устанавливает идентификатор контекстной справки для меню
_GUICtrlMenu_SetMenuData Устанавливает значение, заданное приложением для меню
_GUICtrlMenu_SetMenuDefaultItem Устанавливает пункт меню по умолчанию
