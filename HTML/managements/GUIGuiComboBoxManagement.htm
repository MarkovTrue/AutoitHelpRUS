Работа с GuiComboBox

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiComboBox.au3>.

 

Функция Описание
_GUICtrlComboBox_AddDir Добавляет имена каталогов и файлов
_GUICtrlComboBox_AddString Добавляет строку
_GUICtrlComboBox_AutoComplete Включает или отключает автодополнение
_GUICtrlComboBox_BeginUpdate Запрещает обновление элемента управления до вызова EndUpdate
_GUICtrlComboBox_Create Создаёт элемент управления ComboBox
_GUICtrlComboBox_DeleteString Удаляет строку
_GUICtrlComboBox_Destroy Удаляет элемент управления
_GUICtrlComboBox_EndUpdate Включает обновление экрана, отключённое функцией BeginUpdate
_GUICtrlComboBox_FindString Ищет строку
_GUICtrlComboBox_FindStringExact Ищет строку
_GUICtrlComboBox_GetComboBoxInfo Получает информацию об указанном выпадающем списке
_GUICtrlComboBox_GetCount Получает количество элементов
_GUICtrlComboBox_GetCueBanner Получает текст подсказки, отображаемый в поле ввода выпадающего списка
_GUICtrlComboBox_GetCurSel Возвращает индекс текущего выбранного элемента
_GUICtrlComboBox_GetDroppedControlRect Возвращает экранные координаты раскрытого выпадающего списка
_GUICtrlComboBox_GetDroppedControlRectEx Возвращает экранные координаты раскрытого выпадающего списка
_GUICtrlComboBox_GetDroppedState Определяет, развёрнут ли список выпадающего списка
_GUICtrlComboBox_GetDroppedWidth Получает ширину раскрывающегося списка
_GUICtrlComboBox_GetEditSel Получает начальную и конечную позиции текущего выбранного текста в поле ввода выпадающего списка
_GUICtrlComboBox_GetEditText Получает текст из поля ввода выпадающего списка
_GUICtrlComboBox_GetExtendedUI Определяет, имеет ли выпадающий список пользовательский интерфейс по умолчанию
_GUICtrlComboBox_GetHorizontalExtent Получает ширину в пикселях, на которую список выпадающего списка может прокручиваться по горизонтали
_GUICtrlComboBox_GetItemHeight Определяет высоту элементов списка или поля выбора в выпадающем списке
_GUICtrlComboBox_GetLBText Получает текст элемента из списка
_GUICtrlComboBox_GetLBTextLen Получает длину в символах строки в списке выпадающего списка
_GUICtrlComboBox_GetList Возвращает все элементы из списка ComboBox
_GUICtrlComboBox_GetListArray Возвращает все элементы из списка ComboBox
_GUICtrlComboBox_GetLocale Возвращает текущую локаль
_GUICtrlComboBox_GetLocaleCountry Получает текущий код страны
_GUICtrlComboBox_GetLocaleLang Получает текущий язык
_GUICtrlComboBox_GetLocalePrimLang Извлекает основной идентификатор языка из идентификатора локали
_GUICtrlComboBox_GetLocaleSubLang Извлекает идентификатор подъязыка из идентификатора локали
_GUICtrlComboBox_GetMinVisible Получает минимальное количество видимых элементов в раскрывающемся списке выпадающего списка
_GUICtrlComboBox_GetTopIndex Получает индекс первого видимого элемента
_GUICtrlComboBox_InitStorage Выделяет память для сохранения элементов списка
_GUICtrlComboBox_InsertString Вставляет строку
_GUICtrlComboBox_LimitText Ограничивает длину текста, который пользователь может вводить в поле ввода
_GUICtrlComboBox_ReplaceEditSel Заменяет текст, выбранный в поле ввода
_GUICtrlComboBox_ResetContent Удаляет все элементы из списка и поля ввода выпадающего списка
_GUICtrlComboBox_SelectString Ищет в списке выпадающего списка элемент, который начинается с символов в указанной строке
_GUICtrlComboBox_SetCueBanner Устанавливает текст подсказки, отображаемый для поля ввода выпадающего списка
_GUICtrlComboBox_SetCurSel Выбирает строку в списке выпадающего списка
_GUICtrlComboBox_SetDroppedWidth Устанавливает максимальную допустимую ширину в пикселях списка выпадающего списка
_GUICtrlComboBox_SetEditSel Выбирает символы в поле ввода выпадающего списка
_GUICtrlComboBox_SetEditText Устанавливает текст поля ввода выпадающего списка
_GUICtrlComboBox_SetExtendedUI Выбирает либо стандартный пользовательский интерфейс, либо расширенный пользовательский интерфейс для выпадающего списка
_GUICtrlComboBox_SetHorizontalExtent Устанавливает ширину в пикселях, на которую список может прокручиваться по горизонтали
_GUICtrlComboBox_SetItemHeight Устанавливает высоту элементов списка или поля выбора в выпадающем списке
_GUICtrlComboBox_SetMinVisible Устанавливает минимальное количество видимых элементов в раскрывающемся списке выпадающего списка
_GUICtrlComboBox_SetTopIndex Гарантирует, что определённый элемент видимый в списке выпадающего списка
_GUICtrlComboBox_ShowDropDown Показывает или скрывает список выпадающего списка
