Регионы

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_CombineRgn Объединяет два региона и сохраняет результат в третий регион
_WinAPI_CreateEllipticRgn создаёт эллиптическую область
_WinAPI_CreateNullRgn создаёт пустую область
_WinAPI_CreatePolygonRgn создаёт многоугольную область
_WinAPI_CreateRectRgn создаёт прямоугольную область
_WinAPI_CreateRectRgnIndirect создаёт прямоугольную область
_WinAPI_CreateRoundRectRgn создаёт прямоугольную область со скруглёнными углами
_WinAPI_EqualRgn Проверяет, идентичны ли две указанные области
_WinAPI_ExtCreateRegion Создаёт область из указанной области и данных преобразования
_WinAPI_FillRgn Заливает область указанной кистью
_WinAPI_FrameRgn Рисует границу вокруг указанной области указанной кистью
_WinAPI_GetPolyFillMode Получает текущий режим заливки многоугольников
_WinAPI_GetRegionData Заполняет указанный буфер данными, описывающими область
_WinAPI_GetRgnBox Получает описывающий прямоугольник указанной области
_WinAPI_GetWindowRgn Получает копию области окна
_WinAPI_InvertRgn Инвертирует цвета в указанной области
_WinAPI_OffsetRgn Перемещает область на указанные смещения
_WinAPI_PaintRgn Закрашивает указанный регион, используя кисть, которая в данный момент выбрана в контексте устройства
_WinAPI_PtInRegion Определяет, находится ли указанная точка внутри указанного региона
_WinAPI_RectInRegion Определяет, находится ли какая-либо часть указанного прямоугольника в границах области
_WinAPI_RedrawWindow Обновляет указанный прямоугольник или область в клиентской области окна
_WinAPI_SetPolyFillMode Задаёт режим заполнения полигона для функций, которые заполняют полигоны
_WinAPI_SetRectRgn Преобразует область в прямоугольную область с указанными координатами
_WinAPI_SetWindowRgn Задаёт область окна (window region).
