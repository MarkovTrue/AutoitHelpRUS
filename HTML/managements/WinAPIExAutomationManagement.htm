Автоматизация

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_CallNextHookEx Передаёт информацию о хуке следующей процедуре хука в текущей цепочке
_WinAPI_SetWindowsHookEx Устанавливает определённую приложением процедуру перехватчика в цепь перехватчиков
_WinAPI_SetWinEventHook Устанавливает функцию перехватчика событий для диапазона событий
_WinAPI_UnhookWindowsHookEx Удаляет процедуру hook, установленную в цепи hook функцией _WinAPI_SetWindowsHookEx()
_WinAPI_UnhookWinEvent Удаляет функцию события hook
