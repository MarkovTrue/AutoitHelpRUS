Справочник WinAPIEx: шрифты и текст

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_AddFontMemResourceEx Добавляет ресурс шрифта из образа в памяти в систему
_WinAPI_AddFontResourceEx Добавляет ресурс шрифта из указанного файла в таблицу системных шрифтов
_WinAPI_CreateFont создаёт логический шрифт с указанными характеристиками
_WinAPI_CreateFontEx создаёт логический шрифт с указанными характеристиками
_WinAPI_CreateFontIndirect создаёт логический шрифт с конкретными характеристиками
_WinAPI_EnumFontFamilies Перечисляет все шрифты системы с уникальными именами, удовлетворяющие заданным характеристикам
_WinAPI_GetFontMemoryResourceInfo Читает информацию о шрифте из TTF, загруженного в память
_WinAPI_GetFontName Получает уникальное имя шрифта на основе его названия гарнитуры, кодировки символов и стиля
_WinAPI_GetFontResourceInfo Получает имя шрифта из указанного файла ресурса шрифта
_WinAPI_GetGlyphOutline Получает контур или точечный рисунок символа в шрифте TrueType
_WinAPI_GetOutlineTextMetrics Получает метрики текста для шрифтов TrueType
_WinAPI_GetTabbedTextExtent Вычисляет ширину и высоту строки символов, которая может содержать один или несколько символов табуляции
_WinAPI_GetTextAlign Получает параметр выравнивания текста для указанного контекста устройства
_WinAPI_GetTextCharacterExtra Получает текущее межсимвольное расстояние для указанного контекста устройства
_WinAPI_GetTextExtentPoint32 Вычисляет ширину и высоту указанной строки текста
_WinAPI_GetTextFace Получает имя гарнитуры шрифта, который выбран в указанный контекст устройства
_WinAPI_GetTextMetrics Получает основную информацию о текущем выбранном шрифте
_WinAPI_RemoveFontMemResourceEx Удаляет шрифты, добавленные из образа памяти
_WinAPI_RemoveFontResourceEx Удаляет шрифты в указанном файле из таблицы системных шрифтов
_WinAPI_SetTextAlign Устанавливает флаги выравнивания текста для указанного контекста устройства
_WinAPI_SetTextCharacterExtra Задаёт дополнительное расстояние между символами для указанного контекста устройства
_WinAPI_SetTextJustification Указывает объём пространства, которое система должна добавить к символам разрыва в строке текста
_WinAPI_TabbedTextOut Записывает строку символов в указанное место и расширяет табуляторы до указанных позиций табуляции
_WinAPI_TextOut Записывает строку в указанное место, используя текущий выбранный шрифт, цвет фона и цвет текста
