Справочник пользовательских функций: Windows

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISysWin.au3>.

 

Функция Описание
_WinAPI_AdjustWindowRectEx Вычисляет требуемый размер прямоугольника окна на основе требуемого размера прямоугольника клиентской области
_WinAPI_AnimateWindow Позволяет вам создавать специальные эффекты при отображении или скрытии окон
_WinAPI_BeginDeferWindowPos Выделяет память для структуры позиций нескольких окон
_WinAPI_BringWindowToTop Перемещает указанное окно в начало Z-порядка
_WinAPI_BroadcastSystemMessage Отправляет сообщение указанным получателям
_WinAPI_CallWindowProc Передаёт информацию о хуке следующей процедуре хука в текущей цепочке
_WinAPI_CallWindowProcW Передаёт информацию о сообщении указанной процедуре окна
_WinAPI_CascadeWindows Расставляет указанные дочерние окна в каскад в пределах указанного родительского окна
_WinAPI_ChangeWindowMessageFilterEx Изменяет фильтр сообщений изоляции привилегий пользовательского интерфейса (UIPI) для указанного окна
_WinAPI_ChildWindowFromPointEx Определяет, какое из дочерних окон указанного родительского окна содержит указанную точку
_WinAPI_CloseWindow Минимизирует (но не закрывает) указанное окно
_WinAPI_CreateWindowEx создаёт наложенное, всплывающее или дочернее окно
_WinAPI_DeferWindowPos Обновляет структуру множественного позиционирования окон для указанного окна
_WinAPI_DefWindowProc Вызывает оконную процедуру по умолчанию для стандартной обработки сообщений
_WinAPI_DefWindowProcW Вызывает оконную процедуру по умолчанию для стандартной обработки любых оконных сообщений
_WinAPI_DeregisterShellHookWindow Снимает регистрацию указанного Shell-окна, зарегистрированного на получение Shell hook-сообщений
_WinAPI_DestroyWindow Уничтожает указанное окно
_WinAPI_DragAcceptFiles Задаёт, принимает ли окно перетащенные файлы
_WinAPI_DragFinish Освобождает память, выделенную системой для передачи имён файлов в приложение
_WinAPI_DragQueryFileEx Получает имена файлов, переданных в результате операции drag-and-drop.
_WinAPI_DragQueryPoint Получает координаты позиции мыши на экране, когда был отпущен файл в операции drag-and-drop.
_WinAPI_EndDeferWindowPos Одновременно обновляет позицию и размер одного или нескольких окон за один цикл перерисовки экрана
_WinAPI_EnumChildWindows Перечисляет дочерние окна указанного родительского окна
_WinAPI_EnumWindows Перечисляет все окна
_WinAPI_EnumWindowsPopup Перечисляет всплывающие окна
_WinAPI_EnumWindowsTop Перечисляет все окна верхнего уровня
_WinAPI_FindWindow Получает дескриптор окна верхнего уровня, имя класса и имя окна которого совпадают с указанными.
_WinAPI_FlashWindow Мигает указанным окном один раз
_WinAPI_FlashWindowEx Мигает указанным окном
_WinAPI_GetAncestor Получает дескриптор окна-предка для указанного окна
_WinAPI_GetClassInfoEx Получает информацию о классе окна.
_WinAPI_GetClassLongEx Получает указанное значение, связанное с указанным окном.
_WinAPI_GetClassName Получает имя класса, к которому принадлежит указанное окно.
_WinAPI_GetClientHeight Получает высоту клиентской области окна.
_WinAPI_GetClientRect Получает координаты клиентской области окна.
_WinAPI_GetClientWidth Получает ширину клиентской области окна.
_WinAPI_GetDesktopWindow Возвращает дескриптор окна рабочего стола Windows
_WinAPI_GetDlgItem Получает дескриптор указанного элемента управления в указанном диалоговом окне.
_WinAPI_GetFocus Получает дескриптор окна, которое имеет фокус клавиатуры
_WinAPI_GetForegroundWindow Возвращает дескриптор окна переднего плана
_WinAPI_GetGUIThreadInfo Получает информацию об активном окне или указанном потоке GUI
_WinAPI_GetLastActivePopup Определяет, какое всплывающее окно, принадлежащее указанному окну, было активно последним
_WinAPI_GetLayeredWindowAttributes Получает атрибуты слоистого окна
_WinAPI_GetMessageExtraInfo Получает дополнительную информацию о сообщении для текущего потока
_WinAPI_GetParent Получает дескриптор окна-родителя для указанного дочернего окна
_WinAPI_GetParentProcess Получает PID родительского процесса для указанного процесса
_WinAPI_GetShellWindow Получает дескриптор окна рабочего стола оболочки
_WinAPI_GetSysColor Получает текущий цвет указанного элемента отображения
_WinAPI_GetTopWindow Получает дескриптор дочернего окна в верхней части порядка Z
_WinAPI_GetWindow Получает дескриптор окна, которое имеет указанное отношение к заданному окну
_WinAPI_GetWindowDC Получает контекст устройства (DC) для всего окна
_WinAPI_GetWindowDisplayAffinity Получает текущий параметр политики отображения, из любого процесса, для заданного окна
_WinAPI_GetWindowExt Получает x-размер и y-размер окна для указанного контекста устройства
_WinAPI_GetWindowFileName Получает полностью определённый путь модуля, связанного с указанным дескриптором окна
_WinAPI_GetWindowHeight Возвращает высоту окна
_WinAPI_GetWindowInfo Получает сведения об указанном окне
_WinAPI_GetWindowLong Получает сведения об указанном окне
_WinAPI_GetWindowOrg Получает x-координату и y-координату начала окна для указанного контекста устройства
_WinAPI_GetWindowPlacement Получает размещение окна для свёрнутого, развёрнутого и обычного положений
_WinAPI_GetWindowRect Получает размеры ограничивающего прямоугольника указанного окна
_WinAPI_GetWindowText Получает текст строки заголовка указанного окна
_WinAPI_GetWindowThreadProcessId Получает идентификатор потока, который создал указанное окно
_WinAPI_GetWindowWidth Возвращает ширину окна
_WinAPI_InProcess Определяет, принадлежит ли окно текущему процессу
_WinAPI_InvalidateRect Добавляет прямоугольник в область обновления указанного окна
_WinAPI_IsChild Проверяет, является ли окно дочерним окном указанного родительского окна
_WinAPI_IsClassName Оболочка для проверки имени класса элемента управления
_WinAPI_IsHungAppWindow Определяет, не отвечает ли указанное приложение
_WinAPI_IsIconic Определяет, свёрнуто ли указанное окно (является ли оно значком)
_WinAPI_IsWindow Определяет, идентифицирует ли указанный дескриптор окна существующее окно
_WinAPI_IsWindowEnabled Определяет, включено ли указанное окно для ввода мышью и клавиатурой
_WinAPI_IsWindowUnicode Определяет, является ли указанное окно собственным окном Unicode
_WinAPI_IsWindowVisible Получает состояние видимости указанного окна
_WinAPI_IsZoomed Определяет, развёрнуто ли окно
_WinAPI_KillTimer Уничтожает указанный таймер
_WinAPI_MoveWindow Изменяет положение и размеры указанного окна
_WinAPI_OpenIcon Восстанавливает свёрнутое (иконочное) окно до его предыдущего размера и позиции и активирует окно
_WinAPI_PostMessage Помещает сообщение в очередь сообщений и затем возвращает управление
_WinAPI_RegisterClass Регистрирует класс окна
_WinAPI_RegisterClassEx Регистрирует класс окна
_WinAPI_RegisterShellHookWindow Регистрирует указанное окно Shell для получения определённых сообщений для событий или уведомлений
_WinAPI_RegisterWindowMessage Определяет новое сообщение окна, которое гарантированно уникально во всей системе
_WinAPI_SendMessageTimeout Отправляет указанное сообщение одному или нескольким окнам
_WinAPI_SetClassLongEx Заменяет указанное значение в указанном окне
_WinAPI_SetFocus Устанавливает фокус клавиатуры на указанное окно
_WinAPI_SetFont Устанавливает шрифт окна
_WinAPI_SetForegroundWindow Переводит указанное окно в передний план и активирует его
_WinAPI_SetLayeredWindowAttributes Устанавливает атрибуты слоистого окна
_WinAPI_SetMessageExtraInfo Задаёт дополнительную информацию сообщения для текущего потока
_WinAPI_SetParent Изменяет родительское окно указанного дочернего окна
_WinAPI_SetSysColors Получает информацию об устройствах отображения в системе
_WinAPI_SetTimer Создаёт таймер с указанным значением времени ожидания
_WinAPI_SetWindowDisplayAffinity Сохраняет параметр affinity отображения в режиме ядра для указанного окна
_WinAPI_SetWindowLong Задаёт информацию об указанном окне
_WinAPI_SetWindowPlacement Устанавливает расположение окна для минимального, максимального и нормального положений
_WinAPI_SetWindowPos Изменяет размер, позицию и Z-порядок дочернего, всплывающего или верхнеуровневого окна
_WinAPI_SetWindowText Изменяет текст заголовка указанного окна
_WinAPI_ShowOwnedPopups Показывает или скрывает все всплывающие окна, принадлежащие указанному окну
_WinAPI_ShowWindow Задаёт состояние отображения указанного окна
_WinAPI_SwitchToThisWindow Переводит фокус на указанное окно и выводит его на передний план
_WinAPI_TileWindows Располагает указанные дочерние окна указанного родительского окна в виде мозаики
_WinAPI_UnregisterClass Отменяет регистрацию класса окна, освобождая память, необходимую для класса
_WinAPI_UpdateLayeredWindow Обновляет позицию, размер, форму, содержание и прозрачность многоуровневого окна
_WinAPI_UpdateLayeredWindowEx Обновляет прозрачность растрового изображения многоуровневого окна
_WinAPI_UpdateLayeredWindowIndirect Обновляет позицию, размер, форму, содержание и прозрачность многоуровневого окна
_WinAPI_UpdateWindow Обновляет область клиента окна, отправляя сообщение WM_PAINT в окно
_WinAPI_WindowFromPoint Получает дескриптор окна, которое содержит указанную точку
