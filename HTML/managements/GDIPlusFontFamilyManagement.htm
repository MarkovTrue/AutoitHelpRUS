Работа с FontFamily

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_FontFamilyCreate Создаёт объект FontFamily
_GDIPlus_FontFamilyCreateFromCollection Создаёт объект FontFamily на основе указанного семейства шрифтов
_GDIPlus_FontFamilyDispose Освобождает объект FontFamily
_GDIPlus_FontFamilyGetCellAscent Возвращает высоту верхнего выносного элемента (cell ascent) семейства шрифтов в дизайн-единицах для указанного начертания
_GDIPlus_FontFamilyGetCellDescent Возвращает глубину нижнего выносного элемента (cell descent) семейства шрифтов в дизайн-единицах для указанного начертания
_GDIPlus_FontFamilyGetEmHeight Возвращает размер кегля (em size) семейства шрифтов в дизайн-единицах
_GDIPlus_FontFamilyGetLineSpacing Возвращает межстрочный интервал семейства шрифтов в дизайн-единицах для указанного начертания
