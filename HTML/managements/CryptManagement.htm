Справочник пользовательских функций: шифрование

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Crypt.au3>.

 

Функция Описание
_Crypt_DecryptData Расшифровывает данные с помощью указанного ключа.
_Crypt_DecryptFile Расшифровывает файл с помощью указанного ключа и алгоритма.
_Crypt_DeriveKey Создаёт криптографический ключ на основе алгоритма и пароля.
_Crypt_DestroyKey Освобождает ресурсы, занятые ключом.
_Crypt_EncryptData Шифрует данные с помощью указанного ключа.
_Crypt_EncryptFile Шифрует файл с помощью указанного ключа и алгоритма.
_Crypt_GenRandom Заполняет буфер криптографически случайными данными.
_Crypt_HashData Вычисляет хеш данных с помощью указанного алгоритма.
_Crypt_HashFile Вычисляет хеш содержимого файла с помощью указанного алгоритма.
_Crypt_Shutdown Деинициализирует библиотеку Crypt.
_Crypt_Startup Инициализирует библиотеку Crypt.
