Справочник WinAPIEx: ввод мышью

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_GetCapture Получает дескриптор окна (если оно есть), которое захватило мышь.
_WinAPI_Mouse_Event Синтезирует движение мыши и нажатия кнопок
_WinAPI_ReleaseCapture Отпускает захват мыши из окна в текущем потоке и восстанавливает обычную обработку входных данных мыши
_WinAPI_SetCapture Устанавливает захват мыши для указанного окна, принадлежащего текущему потоку
_WinAPI_TrackMouseEvent Отправляет сообщения, когда указатель мыши покидает окно или находится над окном указанное время
