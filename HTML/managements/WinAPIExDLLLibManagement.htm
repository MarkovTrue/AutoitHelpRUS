Работа с DLL

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_GetDllDirectory Получает специфичную для приложения часть пути поиска, используемую для поиска DLL для приложения.
_WinAPI_GetModuleHandle Возвращает дескриптор модуля для указанного модуля
_WinAPI_GetModuleHandleEx Получает дескриптор модуля для указанного модуля
_WinAPI_GetProcAddress Получает адрес экспортированной функции или переменной из указанного модуля
_WinAPI_SetDllDirectory Добавляет каталог в путь поиска, используемый для поиска DLL для приложения
