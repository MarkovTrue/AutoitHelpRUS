Координаты и преобразования

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_CombineTransform Объединяет две трансформации от мировых координат к координатам страницы
_WinAPI_CreateTransform создаёт структуру $tagXFORM, определяющую преобразование из координат мирового пространства в координаты страницы
_WinAPI_DPtoLP Преобразует координаты устройства в логические координаты
_WinAPI_GetCurrentPosition Получает текущую позицию для указанного контекста устройства.
_WinAPI_GetGraphicsMode Получает текущий режим графики для указанного контекста устройства
_WinAPI_GetMapMode Получает текущий режим отображения координат
_WinAPI_GetPosFromRect Интерпретирует координаты прямоугольника как координаты смещения и положения
_WinAPI_GetWorldTransform Получает текущее преобразование мировых координат в координаты страницы
_WinAPI_LPtoDP Преобразует логические координаты в координаты устройства
_WinAPI_ModifyWorldTransform Изменяет мировое преобразование для контекста устройства, используя указанный режим
_WinAPI_OffsetPoints Перемещает точки из массива на указанные смещения
_WinAPI_OffsetWindowOrg Изменяет исходную позицию окна для контекста устройства с использованием указанных горизонтальных и вертикальных смещений
_WinAPI_RotatePoints Вращает точки из массива на указанный угол
_WinAPI_ScaleWindowExt Изменяет окно для контекста устройства, используя пропорции, образованные указанными множителями и делителями
_WinAPI_SetGraphicsMode Устанавливает графический режим для указанного контекста устройства
_WinAPI_SetMapMode Устанавливает режим сопоставления для указанного контекста устройства
_WinAPI_SetWindowExt Задаёт горизонтальный и вертикальный размеры окна для контекста устройства, используя указанные значения
_WinAPI_SetWindowOrg Определяет, какая точка окна соответствует началу координат области просмотра (0,0)
_WinAPI_SetWorldTransform Задаёт двумерное линейное преобразование между мировыми координатами и координатами страницы для указанного контекста устройства.
