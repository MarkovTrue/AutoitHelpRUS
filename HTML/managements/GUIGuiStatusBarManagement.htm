Работа с GuiStatusBar

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiStatusBar.au3>.

 

Функция Описание
_GUICtrlStatusBar_Create Создаёт элемент управления StatusBar.
_GUICtrlStatusBar_Destroy Удаляет элемент управления
_GUICtrlStatusBar_EmbedControl Встраивает элемент управления.
_GUICtrlStatusBar_GetBorders Получает ширину.
_GUICtrlStatusBar_GetBordersHorz Получает ширину.
_GUICtrlStatusBar_GetBordersRect Получает ширину.
_GUICtrlStatusBar_GetBordersVert Получает ширину.
_GUICtrlStatusBar_GetCount Получает значение.
_GUICtrlStatusBar_GetHeight Получает высоту.
_GUICtrlStatusBar_GetIcon Получает значок.
_GUICtrlStatusBar_GetParts Получает значение.
_GUICtrlStatusBar_GetRect Получает прямоугольник координат.
_GUICtrlStatusBar_GetRectEx Получает прямоугольник координат.
_GUICtrlStatusBar_GetText Получает текст.
_GUICtrlStatusBar_GetTextFlags Получает текст.
_GUICtrlStatusBar_GetTextLength Получает текст.
_GUICtrlStatusBar_GetTextLengthEx Получает значение.
_GUICtrlStatusBar_GetTipText Получает текст.
_GUICtrlStatusBar_GetUnicodeFormat Получает значение.
_GUICtrlStatusBar_GetWidth Получает ширину.
_GUICtrlStatusBar_IsSimple Проверяет, находится ли строка состояния в простом режиме
_GUICtrlStatusBar_Resize Изменяет размер.
_GUICtrlStatusBar_SetBkColor Устанавливает цвет фона.
_GUICtrlStatusBar_SetIcon Устанавливает значок.
_GUICtrlStatusBar_SetMinHeight Устанавливает диапазон.
_GUICtrlStatusBar_SetParts Устанавливает значение.
_GUICtrlStatusBar_SetSimple Задаёт, как отображается строка состояния – простой текст или все её части
_GUICtrlStatusBar_SetText Устанавливает текст.
_GUICtrlStatusBar_SetTipText Устанавливает текст.
_GUICtrlStatusBar_SetUnicodeFormat Устанавливает значение.
_GUICtrlStatusBar_ShowHide Показывает или скрывает.
