Кодеки изображений

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_Decoders Возвращает массив сведений о доступных декодерах изображений
_GDIPlus_DecodersGetCount Возвращает количество доступных декодеров изображений
_GDIPlus_DecodersGetSize Возвращает общий размер структуры, возвращаемой функцией _GDIPlus_GetImageDecoders
_GDIPlus_Encoders Возвращает массив сведений о доступных кодировщиках изображений
_GDIPlus_EncodersGetCLSID Возвращает CLSID кодировщика для указанного типа файла изображения
_GDIPlus_EncodersGetCount Возвращает количество доступных кодировщиков изображений
_GDIPlus_EncodersGetParamList Возвращает список параметров указанного кодировщика изображений
_GDIPlus_EncodersGetParamListSize Возвращает размер списка параметров указанного кодировщика изображений
_GDIPlus_EncodersGetSize Возвращает общий размер структуры, возвращаемой функцией _GDIPlus_GetImageEncoders
_GDIPlus_ParamAdd Добавляет значение в список параметров кодировщика
_GDIPlus_ParamInit Выделяет пустой список параметров кодировщика
_GDIPlus_ParamSize Возвращает размер параметра кодировщика
