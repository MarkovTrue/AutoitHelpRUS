Работа с FontPrivate

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_FontPrivateAddFont Добавляет файл шрифта в закрытую коллекцию шрифтов
_GDIPlus_FontPrivateAddMemoryFont Добавляет шрифт из системной памяти в коллекцию шрифтов GDI+
_GDIPlus_FontPrivateCollectionDispose Удаляет указанный объект PrivateFontCollection
_GDIPlus_FontPrivateCreateCollection Создаёт объект PrivateFontCollection
