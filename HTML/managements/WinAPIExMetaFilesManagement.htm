Справочник пользовательских функций: MetaFiles

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_CloseEnhMetaFile Закрывает расширенный контекст устройства метафайла и возвращает дескриптор, который идентифицирует метафайл расширенного формата
_WinAPI_CopyEnhMetaFile Копирует содержимое метафайла расширенного формата в указанный файл
_WinAPI_CreateEnhMetaFile создаёт контекст устройства для метафайла расширенного формата
_WinAPI_DeleteEnhMetaFile Удаляет улучшенный метафайл или дескриптор улучшенного метафайла
_WinAPI_GdiComment Копирует комментарий из буфера в указанный расширенный метафайл
_WinAPI_GetEnhMetaFile Создаёт дескриптор, идентифицирующий улучшенный метафайл (EMF), сохранённый в указанном файле
_WinAPI_GetEnhMetaFileBits Получает содержимое указанного улучшенного метафайла (EMF)
_WinAPI_GetEnhMetaFileDescription Получает необязательное текстовое описание из улучшенного метафайла (EMF)
_WinAPI_GetEnhMetaFileDimension Получает размер указанного улучшенного метафайла (EMF)
_WinAPI_GetEnhMetaFileHeader Получает запись, содержащую заголовок указанного улучшенного метафайла (EMF)
_WinAPI_PlayEnhMetaFile Отображает изображение, хранящееся в указанном расширенном метафайле
_WinAPI_SetEnhMetaFileBits Создаёт метафайл в расширенном формате на основе памяти из указанных данных
