Ввод с клавиатуры

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_ActivateKeyboardLayout Устанавливает идентификатор локали ввода для вызывающего потока или текущего процесса
_WinAPI_EnableWindow Включает или отключает ввод с мыши и клавиатуры для указанного окна или элемента управления
_WinAPI_GetActiveWindow Получает дескриптор активного окна, присоединённого к очереди сообщений вызывающего процесса
_WinAPI_GetAsyncKeyState Определяет, нажата ли клавиша в момент вызова функции
_WinAPI_GetIdleTime Получает время, прошедшее с последнего ввода
_WinAPI_GetKeyboardLayout Получает активный идентификатор локали ввода для указанного окна
_WinAPI_GetKeyboardLayoutList Получает все идентификаторы локали ввода, соответствующие текущему набору локалей ввода в системе
_WinAPI_GetKeyboardLayoutLocale Получает строковое имя идентификатора локали ввода для указанного окна
_WinAPI_GetKeyboardState Копирует состояние 256 виртуальных клавиш в указанный буфер
_WinAPI_GetKeyboardType Получает информацию о текущей клавиатуре
_WinAPI_GetKeyNameText Получает строку, представляющую имя клавиши
_WinAPI_GetKeyState Получает состояние указанной виртуальной клавиши
_WinAPI_IsLoadKBLayout Определяет, загружен ли указанный языковой стандарт ввода в систему
_WinAPI_Keybd_Event Синтезирует нажатие клавиши
_WinAPI_LoadKeyboardLayout Загружает новый идентификатор входной локали в систему
_WinAPI_MapVirtualKey Преобразует код виртуальной клавиши в код сканирования или значение символа, или преобразует код сканирования в код виртуальной клавиши
_WinAPI_RegisterHotKey Определяет системную горячую клавишу
_WinAPI_SendInput Синтезирует нажатия клавиш, движения мыши и щелчки кнопок
_WinAPI_SetActiveWindow Активирует указанное окно
_WinAPI_SetKeyboardLayout Задаёт идентификатор входной локали для указанного окна
_WinAPI_SetKeyboardState Копирует массив из 256 байтов состояний клавиш клавиатуры в таблицу состояния ввода с клавиатуры вызывающего процесса
_WinAPI_UnloadKeyboardLayout Выгружает идентификатор локали ввода
_WinAPI_UnregisterHotKey Освобождает горячую клавишу, ранее зарегистрированную вызывающим потоком
