Работа с реестром

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIReg.au3>.

 

Функция Описание
_WinAPI_AddMRUString Добавляет строку в верхнюю часть списка недавно использованных (MRU) элементов
_WinAPI_AssocGetPerceivedType Получает предполагаемый тип файла на основе его расширения
_WinAPI_AssocQueryString Ищет и получает связанную с ассоциацией файла или протокола строку из реестра
_WinAPI_CreateMRUList создаёт новый список недавно используемых элементов (MRU)
_WinAPI_DllInstall Регистрирует OLE-элементы управления: файлы DLL или ActiveX Controls (OCX)
_WinAPI_DllUninstall Отменяет регистрацию OLE-элементов управления: файлов DLL или ActiveX Controls (OCX)
_WinAPI_EnumMRUList Перечисляет содержимое списка недавних элементов (MRU)
_WinAPI_FreeMRUList Освобождает дескриптор списка недавних элементов (MRU) и записывает кэшированные данные в реестр
_WinAPI_GetRegKeyNameByHandle Получает имя указанного ключа реестра
_WinAPI_RegCloseKey Закрывает дескриптор указанного ключа реестра
_WinAPI_RegConnectRegistry Устанавливает подключение к предопределённому ключу реестра на другом компьютере
_WinAPI_RegCopyTree Рекурсивно копирует подключи и значения исходного подключа в ключ назначения
_WinAPI_RegCopyTreeEx Копирует указанный ключ реестра со всеми его значениями и подключами в указанный ключ назначения
_WinAPI_RegCreateKey Создаёт указанный ключ реестра
_WinAPI_RegDeleteEmptyKey Удаляет пустой ключ
_WinAPI_RegDeleteKey Удаляет подключ и его значения
_WinAPI_RegDeleteKeyValue Удаляет указанное значение из указанного ключа реестра и подключа
_WinAPI_RegDeleteTree Удаляет подключ и всех его потомков
_WinAPI_RegDeleteTreeEx Удаляет подключи и значения указанного ключа рекурсивно
_WinAPI_RegDeleteValue Удаляет именованное значение из указанного ключа реестра
_WinAPI_RegDisableReflectionKey Отключает отражение реестра для указанного ключа
_WinAPI_RegDuplicateHKey Дублирует дескриптор ключа реестра
_WinAPI_RegEnableReflectionKey Восстанавливает отражение реестра для указанного отключённого ключа
_WinAPI_RegEnumKey Перечисляет подключи указанного открытого ключа реестра
_WinAPI_RegEnumValue Перечисляет значения указанного открытого ключа реестра
_WinAPI_RegFlushKey Записывает все атрибуты указанного открытого ключа реестра в реестр
_WinAPI_RegLoadMUIString Загружает указанную строку из указанного ключа и подключа
_WinAPI_RegNotifyChangeKeyValue Уведомляет вызывающего об изменениях атрибутов или содержимого указанного ключа реестра
_WinAPI_RegOpenKey Открывает указанный ключ реестра
_WinAPI_RegQueryInfoKey Получает информацию об указанном ключе реестра
_WinAPI_RegQueryLastWriteTime Получает информацию о времени последней записи в указанный ключ реестра
_WinAPI_RegQueryMultipleValues Получает тип и данные для списка имён значений, связанных с открытым ключом реестра
_WinAPI_RegQueryReflectionKey Определяет, отключена или включена отражение для указанного ключа
_WinAPI_RegQueryValue Получает тип и данные для указанного имени значения, связанного с открытым ключом реестра
_WinAPI_RegRestoreKey Читает информацию реестра из указанного файла и копирует её над указанным ключом
_WinAPI_RegSaveKey Сохраняет указанный ключ, все его подключи и значения в новый файл в стандартном формате
_WinAPI_RegSetValue Задаёт данные и тип указанного значения под ключом реестра
_WinAPI_SfcIsKeyProtected Определяет, защищён ли указанный ключ реестра
