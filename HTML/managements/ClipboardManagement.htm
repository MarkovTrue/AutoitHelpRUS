Работа с буфером обмена

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Clipboard.au3>.

 

Функция Описание
_ClipBoard_ChangeChain Удаляет указанное окно из цепочки наблюдателей буфера обмена.
_ClipBoard_Close Закрывает буфер обмена.
_ClipBoard_CountFormats Возвращает количество различных форматов данных, доступных в буфере обмена.
_ClipBoard_Empty Очищает буфер обмена и освобождает дескрипторы данных в нём.
_ClipBoard_EnumFormats Перечисляет форматы данных, доступные в буфере обмена.
_ClipBoard_FormatStr Возвращает строковое представление стандартного формата буфера обмена.
_ClipBoard_GetData Получает данные из буфера обмена в указанном формате.
_ClipBoard_GetDataEx Получает данные из буфера обмена в указанном формате.
_ClipBoard_GetFormatName Возвращает имя указанного зарегистрированного формата буфера обмена.
_ClipBoard_GetOpenWindow Возвращает дескриптор окна, которое в данный момент открыло буфер обмена.
_ClipBoard_GetOwner Возвращает дескриптор окна, являющегося текущим владельцем буфера обмена.
_ClipBoard_GetPriorityFormat Возвращает первый доступный формат буфера обмена из указанного списка.
_ClipBoard_GetSequenceNumber Возвращает порядковый номер буфера обмена для текущей оконной станции.
_ClipBoard_GetViewer Возвращает дескриптор первого окна в цепочке наблюдателей буфера обмена.
_ClipBoard_IsFormatAvailable Проверяет, содержит ли буфер обмена данные в указанном формате.
_ClipBoard_Open Открывает буфер обмена и запрещает другим приложениям его изменять.
_ClipBoard_RegisterFormat Регистрирует новый формат буфера обмена.
_ClipBoard_SetData Помещает данные в буфер обмена в указанном формате.
_ClipBoard_SetDataEx Помещает данные в буфер обмена в указанном формате.
_ClipBoard_SetViewer Добавляет указанное окно в цепочку наблюдателей буфера обмена.
