Справочник WinAPIEx: доступ к данным и хранилище

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIFiles.au3>.

 

Функция Описание
_WinAPI_BackupRead Создаёт резервную копию файла или папки, включая информацию безопасности
_WinAPI_BackupReadAbort Завершает использование _WinAPI_BackupRead() для дескриптора
_WinAPI_BackupSeek Перемещается вперёд в потоке данных, изначально доступном через функцию _WinAPI_BackupRead() или _WinAPI_BackupWrite()
_WinAPI_BackupWrite Восстанавливает файл или папку, которые были скопированы с использованием _WinAPI_BackupRead()
_WinAPI_BackupWriteAbort Завершает использование _WinAPI_BackupWrite() для дескриптора
_WinAPI_DefineDosDevice Определяет, переопределяет или удаляет имена устройств MS-DOS.
_WinAPI_GetDriveType Определяет тип диска: съёмный, локальный, CD-ROM, RAM-диск или сетевой.
_WinAPI_GetLogicalDrives Получает битовую маску, представляющую доступные диски
_WinAPI_GetPEType Получает тип машины для указанного переносимого исполняемого файла (PE)
_WinAPI_QueryDosDevice Получает текущее отображение для определённого имени устройства MS-DOS
_WinAPI_Wow64EnableWow64FsRedirection Включает или отключает перенаправление файловой системы для вызывающего потока
