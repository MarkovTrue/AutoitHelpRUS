Работа с StringFormat

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_StringFormatCreate Создаёт объект формата строки
_GDIPlus_StringFormatDispose Освобождает объект формата строки
_GDIPlus_StringFormatGetMeasurableCharacterRangeCount Возвращает количество измеримых диапазонов символов, установленных для объекта StringFormat
_GDIPlus_StringFormatSetAlign Устанавливает горизонтальное выравнивание текста в объекте формата строки
_GDIPlus_StringFormatSetLineAlign Устанавливает вертикальное выравнивание строк в объекте StringFormat относительно начала прямоугольника разметки
_GDIPlus_StringFormatSetMeasurableCharacterRanges Задаёт набор диапазонов символов для объекта StringFormat, которые можно измерить в строке
