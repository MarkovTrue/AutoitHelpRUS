Работа с ресурсами

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIRes.au3>.

 

Функция Описание
_WinAPI_BeginUpdateResource Получает дескриптор, который может быть использован для добавления, удаления или замены ресурсов в бинарном модуле
_WinAPI_EndUpdateResource Применяет или отбрасывает изменения ресурсов модуля.
_WinAPI_EnumResourceLanguages Перечисляет языкозависимые ресурсы указанного типа и имени, связанные с двоичным модулем.
_WinAPI_EnumResourceNames Перечисляет ресурсы указанного типа внутри двоичного модуля.
_WinAPI_EnumResourceTypes Перечисляет типы ресурсов внутри двоичного модуля.
_WinAPI_FindResource Определяет расположение ресурса указанного типа и имени в указанном модуле.
_WinAPI_FindResourceEx Определяет расположение ресурса указанного типа, имени и языка в указанном модуле.
_WinAPI_FreeLibrary Уменьшает счётчик ссылок загруженного модуля динамической библиотеки (DLL).
_WinAPI_FreeResource Уменьшает (на единицу) счётчик ссылок загруженного ресурса.
_WinAPI_GetFileVersionInfo Получает информацию о версии для указанного файла
_WinAPI_LoadBitmap Загружает указанный ресурс растрового изображения из исполняемого файла модуля
_WinAPI_LoadImage Загружает значок, курсор или растровое изображение
_WinAPI_LoadIndirectString Извлекает строку из указанного ресурса по непрямой строке
_WinAPI_LoadLibrary Загружает указанный исполняемый модуль в адресное пространство вызывающего процесса
_WinAPI_LoadLibraryEx Загружает указанный исполняемый модуль в адресное пространство вызывающего процесса
_WinAPI_LoadResource Загружает указанный ресурс в глобальную память
_WinAPI_LoadString загружает ресурс строки из исполняемого файла, связанного с указанным модулем
_WinAPI_LoadStringEx Загружает ресурс строки для указанного языка из указанного модуля
_WinAPI_LockResource Блокирует указанный ресурс в памяти
_WinAPI_SizeOfResource Возвращает размер в байтах указанного ресурса
_WinAPI_UpdateResource Добавляет, удаляет или заменяет ресурс в переносимом исполняемом файле (PE)
_WinAPI_VerQueryRoot Получает фиксированную информацию о версии из указанного ресурса информации о версии
_WinAPI_VerQueryValue Получает нефиксированную (строковую) информацию о версии из указанного ресурса информации о версии
_WinAPI_VerQueryValueEx Получает текстовую информацию из ресурса информации о версии указанного двоичного модуля
