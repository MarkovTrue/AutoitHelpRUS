Справочник пользовательских функций: Conversion

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIConv.au3>.

 

Функция Описание
_WinAPI_CharToOem Преобразует строку в набор символов, определённый OEM
_WinAPI_ClientToScreen Преобразует координаты клиента указанной точки в координаты экрана
_WinAPI_DWordToFloat Преобразует значение типа DWORD в значение типа FLOAT
_WinAPI_DWordToInt Преобразует значение типа DWORD в значение типа INT
_WinAPI_FloatToDWord Преобразует значение типа FLOAT в значение типа DWORD
_WinAPI_FloatToInt Возвращает 4-байтовое float-значение как целое
_WinAPI_GetXYFromPoint Возвращает значения X/Y из структуры $tagPOINT
_WinAPI_GUIDFromString Преобразует GUID из строки в бинарную форму
_WinAPI_GUIDFromStringEx Преобразует GUID из строки в бинарную форму
_WinAPI_HashData Вычисляет хеш блока памяти
_WinAPI_HashString Вычисляет хеш строки
_WinAPI_HiByte Возвращает старший байт 16-разрядного (2 байта) значения
_WinAPI_HiDWord Возвращает старшее двойное слово 64-разрядного (8 байт) значения
_WinAPI_HiWord Возвращает старшее слово 32-битного значения.
_WinAPI_IntToDWord Преобразует значение типа INT в значение типа DWORD
_WinAPI_IntToFloat Возвращает 4-байтовое целое число в виде значения с плавающей точкой
_WinAPI_LoByte Возвращает младший байт 16-разрядного (2 байта) значения
_WinAPI_LoDWord Возвращает младшее двойное слово 64-разрядного (8 байт) значения
_WinAPI_LongMid Извлекает количество битов из значения DWORD (32-битное)
_WinAPI_LoWord Возвращает младшее слово 32-битного значения
_WinAPI_MAKELANGID Строит идентификатор языка из первичного идентификатора языка и идентификатора подъязыка
_WinAPI_MAKELCID Строит идентификатор локали из идентификатора языка и идентификатора сортировки
_WinAPI_MakeLong Возвращает значение long int из двух значений int
_WinAPI_MakeQWord Возвращает значение QWORD из двух значений int
_WinAPI_MakeWord Возвращает значение WORD (16-битное) из двух значений BYTE (8-битные)
_WinAPI_MultiByteToWideChar Отображает строку символов на строку расширенных символов (Unicode)
_WinAPI_MultiByteToWideCharEx Отображает строку символов на строку расширенных символов (Unicode)
_WinAPI_OemToChar Преобразует строку из набора символов OEM в строку ANSI
_WinAPI_PointFromRect Возвращает верхние/левые координаты $tagRECT в виде структуры $tagPOINT
_WinAPI_PrimaryLangId Извлекает основной идентификатор языка из идентификатора локали
_WinAPI_ScreenToClient Преобразует экранные координаты указанной точки на экране в клиентские координаты
_WinAPI_ShortToWord Преобразует значение типа SHORT в значение типа WORD
_WinAPI_StrFormatByteSize Преобразует числовое значение в строку, представляющую число, выраженное как значение размера в байтах, килобайтах, мегабайтах или гигабайтах
_WinAPI_StrFormatByteSizeEx Преобразует числовое значение в строку, представляющую число, выраженное как разделённые группы цифр слева от десятичной точки
_WinAPI_StrFormatKBSize Преобразует числовое значение в строку, представляющую число, выраженное как значение размера в килобайтах
_WinAPI_StrFromTimeInterval Преобразует интервал времени в строку
_WinAPI_StringFromGUID Преобразует двоичный GUID в строковую форму
_WinAPI_SubLangId Извлекает идентификатор подъязыка из идентификатора локали
_WinAPI_SwapDWord Преобразует ULONG из формата little-endian в big-endian и наоборот
_WinAPI_SwapQWord Преобразует ULONGLONG из формата little-endian в big-endian и наоборот
_WinAPI_SwapWord Преобразует USHORT из формата little-endian в big-endian и наоборот
_WinAPI_WideCharToMultiByte Преобразует строку UTF-16 (широкий символ) в многобайтовую строку
_WinAPI_WordToShort Преобразует значение типа WORD в значение типа SHORT
