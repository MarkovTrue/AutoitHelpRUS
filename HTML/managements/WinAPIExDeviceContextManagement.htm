Контекст устройства

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdiDC.au3>.

 

Функция Описание
_WinAPI_CreateCompatibleDC создаёт контекст устройства памяти, совместимый с указанным устройством.
_WinAPI_DeleteDC Удаляет указанный контекст устройства.
_WinAPI_DrawEdge Рисует одну или несколько граней прямоугольника.
_WinAPI_DrawFrameControl Рисует элемент рамки указанного типа и стиля.
_WinAPI_DrawIcon Рисует значок или курсор в указанном контексте устройства.
_WinAPI_DrawIconEx Рисует значок или курсор в указанном контексте устройства.
_WinAPI_DrawText Рисует форматированный текст в указанном прямоугольнике
_WinAPI_EnumDisplayDevices Получает информацию об устройствах отображения в системе.
_WinAPI_FillRect Заливает прямоугольник указанной кистью.
_WinAPI_FrameRect Рисует границу вокруг указанного прямоугольника указанной кистью.
_WinAPI_GetBkColor Получает текущий цвет фона для указанного контекста устройства.
_WinAPI_GetBkMode Возвращает текущий режим микширования фона для указанного контекста устройства.
_WinAPI_GetDC Получает дескриптор контекста устройства отображения для клиентской области окна.
_WinAPI_GetDCEx Получает дескриптор контекста устройства для клиентской области окна, позволяя приложению рисовать с использованием соответствующих границ визуального стиля.
_WinAPI_GetDeviceCaps Получает специфичную для устройства информацию об указанном устройстве.
_WinAPI_GetTextColor Получает текущий цвет текста для указанного контекста устройства
_WinAPI_PrintWindow Копирует видимое окно в указанный контекст устройства
_WinAPI_ReleaseDC Освобождает контекст устройства
_WinAPI_RestoreDC Восстанавливает контекст устройства (DC) в указанное состояние
_WinAPI_SaveDC Сохраняет текущее состояние указанного контекста устройства (DC) путём копирования данных, описывающих выбранные объекты и графические режимы, в стек контекста
_WinAPI_SetBkColor Устанавливает текущий цвет фона на указанное значение цвета
_WinAPI_SetBkMode Устанавливает режим смешивания фона указанного контекста устройства
_WinAPI_SetTextColor Задаёт текущий цвет текста на указанное значение цвета
_WinAPI_TwipsPerPixelX Возвращает ширину пикселя в твипах
_WinAPI_TwipsPerPixelY Возвращает высоту пикселя в твипах
