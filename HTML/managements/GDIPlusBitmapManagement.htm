Работа с Bitmap

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_BitmapApplyEffect Изменяет растровое изображение, применяя к нему указанный эффект
_GDIPlus_BitmapApplyEffectEx Изменяет растровое изображение, применяя к нему указанный эффект
_GDIPlus_BitmapCloneArea Создаёт копию объекта Bitmap по заданным координатам и формату
_GDIPlus_BitmapConvertFormat Преобразует растровое изображение в указанный формат пикселей
_GDIPlus_BitmapCreateApplyEffect Создаёт новое растровое изображение, применяя указанный эффект к существующему
_GDIPlus_BitmapCreateApplyEffectEx Создаёт новое растровое изображение, применяя указанный эффект к существующему
_GDIPlus_BitmapCreateDIBFromBitmap Создаёт секцию DIB
_GDIPlus_BitmapCreateFromFile Создаёт объект Bitmap из файла
_GDIPlus_BitmapCreateFromGraphics Создаёт объект Bitmap на основе объекта Graphics, ширины и высоты
_GDIPlus_BitmapCreateFromHBITMAP Создаёт объект Bitmap из дескриптора битовой карты
_GDIPlus_BitmapCreateFromHICON Создаёт объект Bitmap на основе значка
_GDIPlus_BitmapCreateFromHICON32 Создаёт объект Bitmap на основе 32-битного значка
_GDIPlus_BitmapCreateFromMemory Загружает изображение (JPG, BMP, PNG и др.) из бинарной строки и преобразует его в Bitmap (GDI+) или HBITMAP (GDI)
_GDIPlus_BitmapCreateFromResource Создаёт объект Bitmap из ресурса исполняемого файла
_GDIPlus_BitmapCreateFromScan0 Создаёт объект Bitmap на основе массива байтов с информацией о размере и формате
_GDIPlus_BitmapCreateFromStream Создаёт объект Bitmap на основе COM-интерфейса IStream
_GDIPlus_BitmapCreateHBITMAPFromBitmap Создаёт дескриптор HBITMAP из объекта Bitmap
_GDIPlus_BitmapDispose Освобождает объект Bitmap
_GDIPlus_BitmapGetHistogram Возвращает один или несколько гистограмм для указанных цветовых каналов растрового изображения
_GDIPlus_BitmapGetHistogramEx Возвращает гистограммы цветовых каналов растрового изображения: Red, Green, Blue, Alpha и Grey
_GDIPlus_BitmapGetHistogramSize Возвращает количество элементов для выделения памяти функцией _GDIPlus_BitmapGetHistogramEx()
_GDIPlus_BitmapGetPixel Возвращает цвет указанного пикселя растрового изображения
_GDIPlus_BitmapLockBits Блокирует часть растрового изображения для чтения или записи
_GDIPlus_BitmapSetPixel Устанавливает цвет указанного пикселя растрового изображения
_GDIPlus_BitmapSetResolution Устанавливает разрешение объекта Bitmap
_GDIPlus_BitmapUnlockBits Разблокирует часть растрового изображения, заблокированную _GDIPlus_BitmapLockBits()
_GDIPlus_HICONCreateFromBitmap Создаёт дескриптор значка из объекта Bitmap
_GDIPlus_PaletteInitialize Инициализирует стандартную, оптимальную или пользовательскую цветовую палитру
