Справочник WinAPIEx: мышь

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIMisc.au3>.

 

Функция Описание
_WinAPI_GetMousePos Возвращает текущее положение мыши
_WinAPI_GetMousePosX Возвращает текущее положение мыши по оси X
_WinAPI_GetMousePosY Возвращает текущее положение мыши по оси Y
