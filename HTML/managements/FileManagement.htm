Работа с файлами

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <File.au3>.

 

Функция Описание
_FileCountLines Возвращает количество строк в указанном файле
_FileCreate Создаёт указанный файл или обнуляет его содержимое
_FileListToArray Возвращает список файлов и/или папок в указанной папке с поддержкой масок.(Аналог Dir /B с поддержкой шаблонов.)
_FileListToArrayRec Возвращает список файлов и/или папок по заданному пути с опциональным рекурсивным обходом до указанной глубины, сортировкой результата и поддержкой масок.
_FilePrint Печатает текстовый файл
_FileReadToArray Читает указанный файл в одномерный или двумерный массив
_FileWriteFromArray Записывает массив в указанный файл
_FileWriteLog Записывает текущую дату, время и указанный текст в файл журнала
_FileWriteToLine Записывает текст в указанную строку файла
_PathFull Создаёт полный путь на основе относительного пути. Возвращает созданный абсолютный путь
_PathGetRelative Возвращает относительный путь к каталогу
_PathMake Собирает путь из компонентов диска, каталога, имени файла и расширения
_PathSplit Разделяет путь на компоненты: диск, каталог, имя файла и расширение. Пустая строка устанавливается, если компонент отсутствует
_ReplaceStringInFile Заменяет подстроки в файле
_TempFile Генерирует имя для временного файла. Гарантировано, что такой файл не существует
