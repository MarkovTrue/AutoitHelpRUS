Справочник пользовательских функций: массивы

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Array.au3>.

 

Функция Описание
_Array1DToHistogram Преобразует одномерный 1D массив в гистограмму
_Array2DCreate Собирает двумерный 2D массив из двух одномерных 1D или двумерных 2D массивов
_ArrayAdd Добавляет указанное значение в конец существующего одномерного 1D или двумерного 2D массива
_ArrayBinarySearch Ищет значение в одномерном 1D или двумерном 2D массиве алгоритмом бинарного поиска
_ArrayColDelete Удаляет указанный столбец из двумерного 2D массива
_ArrayColInsert Вставляет указанный столбец в одномерный 1D или двумерный 2D массив
_ArrayCombinations Возвращает массив сочетаний элементов из выбранного одномерного 1D массива
_ArrayConcatenate Объединяет два массива – одномерных 1D или двумерных 2D с одинаковым числом столбцов
_ArrayDelete Удаляет указанный элемент или элементы из одномерного 1D или двумерного 2D массива
_ArrayDisplay Отображает одномерный 1D или двумерный 2D массив в виде ListView
_ArrayExtract Извлекает массив из указанных элементов одномерного 1D или двумерного 2D массива
_ArrayFindAll Находит индексы всех вхождений искомого значения в одномерном 1D или двумерном 2D массиве через _ArraySearch()
_ArrayFromString Восстанавливает массив из строки, созданной _ArrayToString() или _SQLite_Display2DResult()
_ArrayInsert Вставляет новое значение в указанную позицию одномерного 1D или двумерного 2D массива
_ArrayMax Возвращает наибольшее значение из одномерного 1D или двумерного 2D массива
_ArrayMaxIndex Возвращает индекс элемента с наибольшим значением в одномерном 1D или двумерном 2D массиве
_ArrayMin Возвращает наименьшее значение из одномерного 1D или двумерного 2D массива
_ArrayMinIndex Возвращает индекс элемента с наименьшим значением в одномерном 1D или двумерном 2D массиве
_ArrayPermute Возвращает массив всех перестановок элементов одномерного 1D массива
_ArrayPop Возвращает последний элемент массива и одновременно удаляет его из массива
_ArrayPush Добавляет новые значения без увеличения размера массива: вставляет новое значение в конец и удаляет первый элемент – или наоборот
_ArrayReverse Обращает порядок элементов в одномерном 1D массиве
_ArraySearch Ищет элемент в одномерном 1D или двумерном 2D массиве. Аналог _ArrayBinarySearch(), но не требует предварительной сортировки массива
_ArrayShuffle Перемешивает выбранные строки одномерного 1D или двумерного 2D массива; для 2D массива можно ограничиться одним столбцом
_ArraySort Сортирует одномерный или двумерный массив по заданному индексу с использованием алгоритмов двойного опорного элемента / быстрой сортировки / сортировки вставками.
_ArraySwap Меняет местами элементы одномерного массива или полные/частичные строки/столбцы двумерного массива.
_ArrayToClip Помещает содержимое одномерного или двумерного массива в буфер обмена, разделяя элементы заданным разделителем.
_ArrayToString Объединяет элементы одномерного или двумерного массива в одну строку, разделяя их заданными разделителями.
_ArrayTranspose Транспонирует одномерный или двумерный массив (меняет строки и столбцы местами).
_ArrayTrim Удаляет заданное количество символов из всех элементов одномерного или двумерного массива.
_ArrayUnique Возвращает элементы столбца одномерного или двумерного массива без дубликатов.
