Работа с массивами

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Array.au3>.

 

Функция Описание
_Array1DToHistogram Преобразует одномерный массив в гистограмму.
_Array2DCreate Создаёт двумерный массив из двух одномерных или двумерных массивов.
_ArrayAdd Добавляет указанное значение в конец существующего одномерного или двумерного массива.
_ArrayBinarySearch Выполняет поиск в одномерном или двумерном массиве методом бинарного поиска.
_ArrayColDelete Удаляет указанный столбец из двумерного массива.
_ArrayColInsert Вставляет указанный столбец в одномерный или двумерный массив.
_ArrayCombinations Возвращает массив всевозможных сочетаний элементов заданного одномерного массива.
_ArrayConcatenate Объединяет два массива: одномерных или двумерных с одинаковым числом столбцов.
_ArrayDelete Удаляет указанный элемент или элементы из одномерного или двумерного массива.
_ArrayDisplay Отображает одномерный или двумерный массив в виде ListView.
_ArrayExtract Извлекает массив из указанных элементов одномерного или двумерного массива.
_ArrayFindAll Находит индексы всех вхождений искомого значения в одномерном или двумерном массиве с помощью _ArraySearch().
_ArrayFromString Восстанавливает массив из строки, созданной функцией _ArrayToString() или _SQLite_Display2DResult().
_ArrayInsert Вставляет новое значение в указанную позицию одномерного или двумерного массива.
_ArrayMax Возвращает наибольшее значение из одномерного 1D или двумерного 2D массива.
_ArrayMaxIndex Возвращает индекс элемента с наибольшим значением в одномерном или двумерном массиве.
_ArrayMin Возвращает наименьшее значение из одномерного 1D или двумерного 2D массива.
_ArrayMinIndex Возвращает индекс элемента с наименьшим значением в одномерном или двумерном массиве.
_ArrayPermute Возвращает массив всевозможных перестановок элементов одномерного массива.
_ArrayPop Возвращает последний элемент массива, одновременно удаляя его из массива.
_ArrayPush Добавляет новые значения без увеличения размера массива путём вставки нового значения в конец и удаления первого элемента, или наоборот.
_ArrayReverse Обращает порядок элементов в одномерном массиве.
_ArraySearch Находит элемент в одномерном или двумерном массиве. Аналогична _ArrayBinarySearch(), но не требует предварительной сортировки массива.
_ArrayShuffle Перемешивает выбранные строки одномерного 1D или двумерного 2D массива; для 2D массива можно ограничиться одним столбцом.
_ArraySort Сортирует одномерный или двумерный массив по заданному индексу с использованием алгоритмов двойного опорного элемента / быстрой сортировки / сортировки вставками.
_ArraySwap Меняет местами элементы одномерного массива или полные/частичные строки/столбцы двумерного массива.
_ArrayToClip Помещает содержимое одномерного или двумерного массива в буфер обмена, разделяя элементы заданным разделителем.
_ArrayToString Объединяет элементы одномерного или двумерного массива в одну строку, разделяя их заданными разделителями.
_ArrayTranspose Транспонирует одномерный или двумерный массив (меняет строки и столбцы местами).
_ArrayTrim Удаляет заданное количество символов из всех элементов одномерного или двумерного массива.
_ArrayUnique Возвращает элементы столбца одномерного или двумерного массива без дубликатов.
