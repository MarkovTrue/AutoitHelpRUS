Таймеры

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Timers.au3>.

 

Функция Описание
_Timer_Diff Возвращает разницу во времени между предыдущим вызовом _Timer_Init() и текущим моментом
_Timer_GetIdleTime Возвращает количество тиков с момента последней активности пользователя (клавиатура или мышь)
_Timer_GetTimerID Возвращает идентификатор таймера из $wParam
_Timer_Init Возвращает метку времени (в миллисекундах)
_Timer_KillAllTimers Уничтожает все таймеры
_Timer_KillTimer Уничтожает указанный таймер
_Timer_SetTimer Создаёт таймер с указанным временем срабатывания
