Работа со структурами

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIMisc.au3>.

 

Функция Описание
_WinAPI_ArrayToStruct Преобразует массив строк в структуру
_WinAPI_CopyStruct Создаёт дубликат указанной структуры
_WinAPI_CreateMargins создаёт структуру $tagMARGINS с указанными границами удержания слева, справа, сверху и снизу
_WinAPI_CreatePoint создаёт структуру $tagPOINT с координатами x и y указанной точки
_WinAPI_CreateRect создаёт структуру $tagRECT с координатами указанного прямоугольника
_WinAPI_CreateRectEx создаёт структуру $tagRECT с координатами указанного прямоугольника
_WinAPI_CreateSize создаёт структуру $tagSIZE с шириной и высотой указанного прямоугольника
_WinAPI_StructToArray Преобразует структуру в массив строк
_WinAPI_UnionStruct Создаёт объединение двух структур
