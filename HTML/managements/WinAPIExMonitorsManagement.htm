Справочник пользовательских функций: Monitors

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_EnumDisplayMonitors Перечисляет мониторы (включая невидимые псевдомониторы, связанные с драйверами зеркалирования)
_WinAPI_GetMonitorInfo Получает информацию о мониторе отображения
_WinAPI_MonitorFromPoint Получает дескриптор монитора отображения, содержащего указанную точку
_WinAPI_MonitorFromRect Получает дескриптор монитора отображения, который имеет наибольшую область пересечения с указанным прямоугольником
_WinAPI_MonitorFromWindow Получает дескриптор монитора отображения, который имеет наибольшую область пересечения с указанным окном
