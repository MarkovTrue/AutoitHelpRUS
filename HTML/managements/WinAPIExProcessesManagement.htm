Справочник пользовательских функций: Processes

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIProc.au3>.

 

Функция Описание
_WinAPI_AdjustTokenPrivileges Включает или отключает привилегии в указанном токене доступа
_WinAPI_AssignProcessToJobObject Назначает процесс существующему объекту задания
_WinAPI_AttachConsole Подключает вызывающий процесс к консоли указанного процесса
_WinAPI_AttachThreadInput Подключает механизм обработки ввода одного потока к механизму другого потока
_WinAPI_CreateJobObject создаёт или открывает объект задания
_WinAPI_CreateProcess создаёт новый процесс и его основной поток
_WinAPI_CreateProcessWithToken создаёт новый процесс и его основной поток в контексте безопасности указанного токена
_WinAPI_DuplicateTokenEx Создаёт новый первичный или олицетворяющий токен доступа, дублирующий существующий токен
_WinAPI_EmptyWorkingSet Освобождает максимально возможное число страниц из рабочего набора указанного процесса
_WinAPI_EnumChildProcess Перечисляет дочерние процессы указанного процесса
_WinAPI_EnumDeviceDrivers Возвращает адрес загрузки каждого драйвера устройства в системе
_WinAPI_EnumProcessHandles Перечисляет дескрипторы указанного процесса
_WinAPI_EnumProcessModules Возвращает дескриптор и имя каждого модуля указанного процесса
_WinAPI_EnumProcessThreads Перечисляет потоки указанного процесса
_WinAPI_EnumProcessWindows Перечисляет окна указанного процесса
_WinAPI_FatalAppExit Отображает диалоговое окно сообщения и завершает приложение
_WinAPI_GetCurrentProcessExplicitAppUserModelID Возвращает явный AppUserModelID, заданный приложением для текущего процесса.
_WinAPI_GetCurrentProcessID Возвращает идентификатор процесса вызывающего процесса.
_WinAPI_GetCurrentThread Получает псевдодескриптор вызывающего потока
_WinAPI_GetCurrentThreadId Возвращает идентификатор потока вызывающего потока.
_WinAPI_GetDeviceDriverBaseName Получает базовое имя указанного драйвера устройства
_WinAPI_GetDeviceDriverFileName Получает доступный путь к указанному драйверу устройства
_WinAPI_GetExitCodeProcess Получает статус завершения указанного процесса
_WinAPI_GetGuiResources Получает количество дескрипторов объектов графического пользовательского интерфейса (GUI), используемых указанным процессом
_WinAPI_GetModuleFileNameEx Получает полный путь к файлу, содержащему указанный модуль
_WinAPI_GetModuleInformation Получает информацию об указанном модуле
_WinAPI_GetPriorityClass Получает класс приоритета для указанного процесса
_WinAPI_GetProcessAffinityMask Получает маски сродства для процесса и системы
_WinAPI_GetProcessCommandLine Получает строку командной строки для указанного процесса
_WinAPI_GetProcessFileName Получает полный путь исполняемого файла для указанного процесса
_WinAPI_GetProcessHandleCount Получает число открытых дескрипторов, принадлежащих указанному процессу
_WinAPI_GetProcessID Получает идентификатор процесса указанного процесса
_WinAPI_GetProcessIoCounters Получает информацию о всех операциях ввода-вывода, выполняемых указанным процессом
_WinAPI_GetProcessMemoryInfo Получает информацию об использовании памяти указанным процессом
_WinAPI_GetProcessName Получает имя указанного процесса
_WinAPI_GetProcessTimes Получает информацию о времени для указанного процесса
_WinAPI_GetProcessUser Получает имя пользователя и домена для указанного процесса
_WinAPI_GetProcessWorkingDirectory Возвращает текущий рабочий каталог указанного процесса
_WinAPI_GetThreadDesktop Получает дескриптор рабочего стола, назначенного указанному потоку
_WinAPI_GetThreadErrorMode Получает режим ошибок для вызывающего потока
_WinAPI_IsElevated Определяет, запущен ли текущий процесс с повышенными привилегиями
_WinAPI_IsProcessInJob Определяет, работает ли процесс в указанной задаче
_WinAPI_IsWow64Process Определяет, работает ли указанный процесс под WOW64
_WinAPI_OpenJobObject Открывает существующий объект задания
_WinAPI_OpenProcess Возвращает дескриптор существующего объекта процесса
_WinAPI_OpenProcessToken Открывает маркер доступа, связанный с процессом
_WinAPI_QueryInformationJobObject Получает информацию об ограничениях и состоянии задания из объекта задания
_WinAPI_SetInformationJobObject Задаёт ограничения для объекта задания
_WinAPI_SetPriorityClass Задаёт класс приоритета для указанного процесса
_WinAPI_SetProcessAffinityMask Задаёт маску сродства процессора для потоков указанного процесса
_WinAPI_SetThreadDesktop Назначает указанный рабочий стол вызывающему потоку
_WinAPI_SetThreadErrorMode Управляет тем, будет ли система обрабатывать указанные типы серьёзных ошибок или это будет делать вызывающий поток
_WinAPI_SetThreadExecutionState Предотвращает переход системы в режим сна или отключение дисплея во время работы текущего приложения
_WinAPI_TerminateJobObject Завершает все процессы, в настоящее время связанные с заданием
_WinAPI_TerminateProcess Завершает указанный процесс и все его потоки
_WinAPI_UserHandleGrantAccess Предоставляет или запрещает доступ к дескриптору объекта User для задания, имеющего ограничение пользовательского интерфейса
_WinAPI_WriteConsole Записывает строку символов в буфер экрана консоли
