Справочник пользовательских функций: DevicesIoCtl

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIFiles.au3>.

 

Функция Описание
_WinAPI_CreateObjectID создаёт или получает идентификатор объекта для указанного файла или каталога
_WinAPI_DeleteObjectID Удаляет идентификатор объекта у указанного файла или каталога
_WinAPI_DeviceIoControl Отправляет управляющий код напрямую драйверу указанного устройства
_WinAPI_EjectMedia Извлекает носитель из устройства
_WinAPI_GetCDType Получает тип носителя, загруженного в указанное CD-ROM-устройство.
_WinAPI_GetDriveBusType Получает тип шины для указанного диска.
_WinAPI_GetDriveGeometryEx Получает расширенную информацию о геометрии диска
_WinAPI_GetDriveNumber Получает тип устройства, номер устройства и номер раздела для указанного диска
_WinAPI_GetObjectID Получает идентификатор объекта для указанного файла или каталога
_WinAPI_GetOverlappedResult Получает результаты перекрытой операции
_WinAPI_IOCTL Создаёт уникальный системный код управления вводом-выводом (IOCTL)
_WinAPI_IsDoorOpen Проверяет, открыт ли лоток CD (DVD)
_WinAPI_IsWritable Определяет, доступен ли диск для записи
_WinAPI_LoadMedia Загружает мультимедиа в устройство
