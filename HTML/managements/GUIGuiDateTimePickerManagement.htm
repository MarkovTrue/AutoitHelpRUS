Работа с GuiDateTimePicker

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiDateTimePicker.au3>.

 

Функция Описание
_GUICtrlDTP_Create Создаёт выбор даты и времени (DTP)
_GUICtrlDTP_Destroy Удаляет элемент управления.
_GUICtrlDTP_GetMCColor Получает указанный цвет
_GUICtrlDTP_GetMCFont Получает дескриптор шрифта месячного календаря
_GUICtrlDTP_GetMonthCal Получает дескриптор элемента управления дочерний месячный календарь
_GUICtrlDTP_GetRange Получает текущие минимальное и максимальное допустимые системные времена
_GUICtrlDTP_GetRangeEx Получает текущие минимальное и максимальное допустимые системные времена
_GUICtrlDTP_GetSystemTime Получает текущую дату и время
_GUICtrlDTP_GetSystemTimeEx Получает текущую дату и время
_GUICtrlDTP_SetFormat Задаёт отображение по заданной строке формата.
_GUICtrlDTP_SetMCColor Устанавливает цвет для заданной части месячного календаря
_GUICtrlDTP_SetMCFont Задаёт шрифт календаря на месяц.
_GUICtrlDTP_SetRange Задаёт текущие минимально и максимально допустимое системное время.
_GUICtrlDTP_SetRangeEx Задаёт текущие минимально и максимально допустимое системное время.
_GUICtrlDTP_SetSystemTime Задаёт текущую выбранную дату и время.
_GUICtrlDTP_SetSystemTimeEx Задаёт текущую выбранную дату и время.
