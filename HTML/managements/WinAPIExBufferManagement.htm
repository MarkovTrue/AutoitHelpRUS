Работа с буферами

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_CompressBuffer Сжимает буфер с указанным форматом сжатия и типом механизма
_WinAPI_ComputeCrc32 Вычисляет контрольную сумму CRC32 блока памяти
_WinAPI_DecompressBuffer Распаковывает целиком сжатый буфер.
