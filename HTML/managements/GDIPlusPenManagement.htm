Справочник GDIPlus: перья

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_PenCreate Создаёт объект Pen
_GDIPlus_PenCreate2 Создаёт объект Pen на основе атрибутов кисти
_GDIPlus_PenDispose Освобождает объект Pen
_GDIPlus_PenGetAlignment Возвращает выравнивание пера
_GDIPlus_PenGetColor Возвращает цвет пера
_GDIPlus_PenGetCustomEndCap Возвращает пользовательский конечный колпачок пера
_GDIPlus_PenGetDashCap Возвращает стиль колпачка штриха пера
_GDIPlus_PenGetDashStyle Возвращает стиль штриха пера
_GDIPlus_PenGetEndCap Возвращает конечный колпачок пера
_GDIPlus_PenGetMiterLimit Возвращает предел длины среза, установленный для объекта Pen
_GDIPlus_PenGetWidth Возвращает ширину пера
_GDIPlus_PenResetTransform Сбрасывает матрицу мирового преобразования объекта Pen до единичной матрицы
_GDIPlus_PenRotateTransform Обновляет матрицу мирового преобразования объекта Pen, умножая её на матрицу вращения
_GDIPlus_PenScaleTransform Обновляет матрицу мирового преобразования объекта Pen, умножая её на матрицу масштабирования
_GDIPlus_PenSetAlignment Задаёт выравнивание пера
_GDIPlus_PenSetColor Задаёт цвет пера
_GDIPlus_PenSetCompound Задаёт составной массив для объекта Pen
_GDIPlus_PenSetCustomEndCap Задаёт пользовательский наконечник конца линии для пера
_GDIPlus_PenSetDashCap Задаёт стиль наконечников штрихов пера
_GDIPlus_PenSetDashStyle Задаёт стиль штриховки пера
_GDIPlus_PenSetEndCap Задаёт наконечник конца линии пера
_GDIPlus_PenSetLineCap Задаёт стили наконечников начала, конца и штрихов линии, рисуемой пером
_GDIPlus_PenSetLineJoin Задаёт стиль соединения линий для объекта Pen
_GDIPlus_PenSetMiterLimit Задаёт предел длины среза для объекта Pen
_GDIPlus_PenSetStartCap Задаёт наконечник начала линии для объекта Pen
_GDIPlus_PenSetTransform Задаёт матрицу мирового преобразования объекта Pen
_GDIPlus_PenSetWidth Задаёт ширину пера
