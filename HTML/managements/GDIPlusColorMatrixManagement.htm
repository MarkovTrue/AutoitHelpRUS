Работа с ColorMatrix

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_ColorMatrixCreate Создаёт и инициализирует единичную матрицу цветов
_GDIPlus_ColorMatrixCreateGrayScale Создаёт и инициализирует матрицу цветов для обесцвечивания (в оттенки серого)
_GDIPlus_ColorMatrixCreateNegative Создаёт и инициализирует матрицу цветов для инверсии (негатив)
_GDIPlus_ColorMatrixCreateSaturation Создаёт и инициализирует матрицу цветов для управления насыщенностью
_GDIPlus_ColorMatrixCreateScale Создаёт и инициализирует матрицу цветов для масштабирования компонентов
_GDIPlus_ColorMatrixCreateTranslate Создаёт и инициализирует матрицу цветов для сдвига компонентов
