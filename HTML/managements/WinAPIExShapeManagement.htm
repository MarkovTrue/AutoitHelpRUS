Фигуры

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_Ellipse Рисует эллипс.
_WinAPI_InvertRect Инвертирует прямоугольник в окне путём выполнения логической операции НЕ над значениями цветов каждого пикселя
_WinAPI_Polygon Рисует многоугольник, состоящий из двух или более вершин, соединённых прямыми линиями
_WinAPI_Rectangle Рисует прямоугольник
_WinAPI_RoundRect Рисует прямоугольник с закруглёнными углами
