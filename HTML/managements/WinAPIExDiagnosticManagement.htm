Справочник пользовательских функций: Diagnostic

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIDiag.au3>.

 

Функция Описание
_WinAPI_DisplayStruct Отображает данные из указанной структуры или по адресу памяти в виде списка
_WinAPI_EnumDllProc Перечисляет экспортируемые функции указанной динамической библиотеки (DLL)
_WinAPI_FatalExit Передаёт управление выполнением отладчику
_WinAPI_GetApplicationRestartSettings Получает информацию о перезапуске, зарегистрированную для указанного процесса
_WinAPI_GetErrorMode Получает режим ошибок текущего процесса
_WinAPI_IsInternetConnected Определяет, подключён ли текущий пользователь к Интернету
_WinAPI_IsNetworkAlive Определяет, подключена ли локальная система к сети, и определяет тип сетевого подключения
_WinAPI_NtStatusToDosError Преобразует указанный код ошибки NTSTATUS в эквивалентный системный код ошибки
_WinAPI_RegisterApplicationRestart Регистрирует активный экземпляр приложения для перезапуска
_WinAPI_SetErrorMode Управляет тем, будет ли система обрабатывать указанные типы серьёзных ошибок или будет обрабатывать их процесс
_WinAPI_UniqueHardwareID Генерирует уникальный идентификатор оборудования (ID) для локального компьютера
_WinAPI_UnregisterApplicationRestart Удаляет активный экземпляр приложения из списка перезагрузки
