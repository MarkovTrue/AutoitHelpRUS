Математические функции

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Math.au3>.

 

Функция Описание
_Degree Преобразует радианы в градусы.
_Div Выполняет целочисленное деление.
_Max Определяет, какое из двух чисел больше
_Min Определяет, какое из двух чисел меньше
_Radian Преобразует градусы в радианы
