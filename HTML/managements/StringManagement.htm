Работа со строками

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <String.au3>.

 

Функция Описание
_HexToString Преобразует шестнадцатеричную строку в строку.
_StringBetween Находит строки между двумя разделителями
_StringExplode Разбивает строку на подстроки по указанному разделителю, как функция PHP Explode v5
_StringInsert Вставляет строку в другую строку
_StringProper Преобразует строку в формат правильного регистра, как функция =Proper в Excel
_StringRepeat Повторяет строку указанное количество раз
_StringTitleCase Преобразует строку в верхний регистр для каждого слова
_StringToHex Преобразует строку в шестнадцатеричное представление
