Справочник пользовательских функций: ImageAttributes

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_ImageAttributesCreate Создаёт объект ImageAttributes
_GDIPlus_ImageAttributesDispose Освобождает объект ImageAttributes
_GDIPlus_ImageAttributesSetColorKeys Устанавливает или сбрасывает цветовой ключ (диапазон прозрачности) для указанной категории
_GDIPlus_ImageAttributesSetColorMatrix Устанавливает или сбрасывает матрицы коррекции цвета и оттенков серого для указанной категории
_GDIPlus_ImageAttributesSetRemapTable Устанавливает или сбрасывает таблицу перекодировки цветов для указанной категории
_GDIPlus_ImageAttributesSetThreshold Устанавливает пороговое значение (диапазон прозрачности) для указанной категории
