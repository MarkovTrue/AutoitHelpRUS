Разное

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Misc.au3>.

 

Функция Описание
_ChooseColor Открывает диалоговое окно выбора цвета.
_ChooseFont Создаёт диалоговое окно выбора шрифта, в котором пользователь задаёт атрибуты логического шрифта
_ClipPutFile Копирует файлы в буфер обмена так же, как это делает Проводник.
_FormatAutoItExitCode Форматирует @exitCode
_FormatAutoItExitMethod Форматирует @exitMethod
_IsPressed Проверяет, была ли нажата клавиша
_MouseTrap Ограничивает движение мыши указанной областью
_Singleton Обеспечивает парадигму проектирования, при которой может запускаться только один экземпляр скрипта
_VersionCompare Сравнивает две версии файла для проверки равенства
