Справочник WinAPIEx: COM

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPICom.au3>.

 

Функция Описание
_WinAPI_CLSIDFromProgID Ищет CLSID в реестре по ProgID
_WinAPI_CoInitialize Инициализирует библиотеку COM для использования вызывающим процессом
_WinAPI_CoTaskMemAlloc Выделяет блок памяти задачи
_WinAPI_CoTaskMemFree Освобождает блок памяти задачи
_WinAPI_CoTaskMemRealloc Изменяет размер ранее выделенного блока памяти задачи
_WinAPI_CoUninitialize Закрывает библиотеку COM в текущем процессе
_WinAPI_CreateGUID создаёт глобально уникальный идентификатор (GUID)
_WinAPI_CreateStreamOnHGlobal создаёт объект потока, который использует дескриптор памяти для хранения содержимого потока
_WinAPI_GetHGlobalFromStream Получает дескриптор глобальной памяти для потока
_WinAPI_ProgIDFromCLSID Получает ProgID для заданного CLSID
_WinAPI_ReleaseStream Освобождает объект потока
