Работа с SolidBrush

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_BrushCreateSolid Создаёт объект сплошной кисти
_GDIPlus_BrushGetSolidColor Возвращает цвет объекта сплошной кисти
_GDIPlus_BrushSetSolidColor Устанавливает цвет объекта сплошной кисти
