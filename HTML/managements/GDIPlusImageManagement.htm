Работа с изображениями

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GDIPlus.au3>.

 

Функция Описание
_GDIPlus_DrawImageFX Рисует фрагмент изображения с применением указанного эффекта
_GDIPlus_DrawImageFXEx Рисует фрагмент изображения с применением указанного эффекта
_GDIPlus_DrawImagePoints Рисует изображение в указанном месте
_GDIPlus_ImageClone Создаёт новый объект Image путём клонирования исходного изображения
_GDIPlus_ImageDispose Освобождает объект Image
_GDIPlus_ImageGetDimension Возвращает ширину и высоту изображения, поддерживаемого GDI+
_GDIPlus_ImageGetFlags Возвращает набор атрибутов пиксельных данных изображения
_GDIPlus_ImageGetFrameCount Возвращает количество кадров в указанном измерении объекта Image
_GDIPlus_ImageGetGraphicsContext Возвращает графический контекст изображения
_GDIPlus_ImageGetHeight Возвращает высоту изображения
_GDIPlus_ImageGetHorizontalResolution Возвращает горизонтальное разрешение изображения в DPI (пикселей на дюйм)
_GDIPlus_ImageGetPixelFormat Возвращает формат пикселей изображения: бит/пикс, альфа-каналы, RGB, оттенки серого, индексированный и др.
_GDIPlus_ImageGetPropertyIdList Возвращает список идентификаторов свойств, используемых в метаданных объекта Image
_GDIPlus_ImageGetPropertyItem Возвращает указанный элемент свойства (фрагмент метаданных) из объекта Image
_GDIPlus_ImageGetRawFormat Возвращает GUID формата файла и название формата изображения
_GDIPlus_ImageGetThumbnail Возвращает уменьшенную копию (thumbnail) из объекта Image
_GDIPlus_ImageGetType Возвращает тип изображения (растровый файл или метафайл)
_GDIPlus_ImageGetVerticalResolution Возвращает вертикальное разрешение изображения в DPI (пикселей на дюйм)
_GDIPlus_ImageGetWidth Возвращает ширину изображения
_GDIPlus_ImageLoadFromFile Создаёт объект Image на основе файла
_GDIPlus_ImageLoadFromStream Создаёт объект Image на основе потока
_GDIPlus_ImageResize Изменяет размер изображения до заданных значений
_GDIPlus_ImageRotateFlip Поворачивает и отражает изображение
_GDIPlus_ImageSaveAdd Добавляет кадр в файл или поток, указанный в предыдущем вызове _GDIPlus_ImageSaveToFileEx() или _GDIPlus_ImageSaveToStream()
_GDIPlus_ImageSaveAddImage Добавляет кадр в файл или поток, указанный в предыдущем вызове _GDIPlus_ImageSaveToFileEx() или _GDIPlus_ImageSaveToStream()
_GDIPlus_ImageSaveToFile Сохраняет изображение в файл
_GDIPlus_ImageSaveToFileEx Сохраняет изображение в файл
_GDIPlus_ImageSaveToStream Сохраняет объект Image в поток
_GDIPlus_ImageScale Масштабирует изображение на заданный коэффициент
_GDIPlus_ImageSelectActiveFrame Выбирает кадр в объекте Image, заданный измерением и индексом
