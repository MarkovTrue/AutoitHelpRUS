Справочник пользовательских функций: GuiImageList

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiImageList.au3>.

 

Функция Описание
_GUIImageList_Add Добавляет значок или растровое изображение в список изображений.
_GUIImageList_AddBitmap Добавляет битмап в список изображений
_GUIImageList_AddIcon Добавляет значок в список изображений
_GUIImageList_AddMasked Добавляет одно или несколько изображений в список изображений, формируя маску по указанному битмапу
_GUIImageList_BeginDrag Создаёт наложенное изображение.
_GUIImageList_Copy Копирует изображение из одного списка изображений в другой.
_GUIImageList_Create Создаёт список изображений.
_GUIImageList_Destroy Уничтожает список изображений.
_GUIImageList_DestroyIcon Уничтожает значок и освобождает занятую им память
_GUIImageList_DragEnter изображение
_GUIImageList_DragLeave изображение
_GUIImageList_DragMove Перетаскивает наложенное изображение.
_GUIImageList_Draw Рисует изображение из списка изображений.
_GUIImageList_DrawEx Получает дескриптор значка из списка изображений.
_GUIImageList_Duplicate Создаёт копию списка изображений.
_GUIImageList_EndDrag Завершает операцию перетаскивания наложения.
_GUIImageList_GetBkColor Получает цвет фона списка изображений.
_GUIImageList_GetIcon Получает значок или растровое изображение из списка изображений.
_GUIImageList_GetIconHeight Получает высоту изображений в списке изображений
_GUIImageList_GetIconSize Получает размеры изображений в списке изображений
_GUIImageList_GetIconSizeEx Получает размеры изображений в списке изображений
_GUIImageList_GetIconWidth Получает ширину изображений в списке изображений
_GUIImageList_GetImageCount Получает количество изображений в списке изображений.
_GUIImageList_GetImageInfoEx Получает расширенную информацию об изображении.
_GUIImageList_Remove Удаляет изображение из списка изображений.
_GUIImageList_ReplaceIcon Заменяет изображение значком или курсором
_GUIImageList_SetBkColor Устанавливает цвет фона списка изображений.
_GUIImageList_SetIconSize Задаёт размеры изображений в списке изображений и удаляет из списка все изображения
_GUIImageList_SetImageCount Изменяет размер существующего списка изображений
_GUIImageList_Swap индекс
