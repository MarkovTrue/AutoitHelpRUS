Работа с Internet Explorer

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <IE.au3>.

 

Функция Описание
_IE_Example Отображает новое окно браузера с предзагруженными документами для использования в примерах IE.au3 или ваших собственных тестах
_IE_Introduction Отображает вводную информацию об IE.au3 в новом окне браузера
_IE_VersionInfo Возвращает массив со сведениями о версии IE.au3
_IEAction Выполнить набор простых действий в браузере
_IEAttach Подключиться к указанному экземпляру Internet Explorer, где подстрока строки поиска совпадает (на основе выбранного режима)
_IEBodyReadHTML Возвращает HTML внутри тега <body> документа
_IEBodyReadText Возвращает текст внутри тега <body> документа
_IEBodyWriteHTML Заменяет HTML внутри тега <body> документа
_IECreate Создаёт окно браузера Internet Explorer
_IECreateEmbedded Создаёт объект веб-браузера, пригодный для встраивания в GUI AutoIt через GUICtrlCreateObj()
_IEDocGetObj По любому DOM-объекту возвращает ссылку на связанный с ним объект документа
_IEDocInsertHTML Вставляет HTML-текст в элемент или вокруг него
_IEDocInsertText Вставляет текст внутри элемента или вокруг него
_IEDocReadHTML Возвращает полный HTML-исходник документа
_IEDocWriteHTML Заменяет HTML всего документа
_IEErrorNotify Управляет автоматическим выводом предупреждений и ошибок IE.au3 в консоль
_IEFormElementCheckBoxSelect Задаёт значение указанного элемента формы
_IEFormElementGetCollection Возвращает объектную переменную-коллекцию всех элементов формы внутри заданной формы
_IEFormElementGetObjByName Возвращает ссылку на элемент формы по имени
_IEFormElementGetValue Возвращает значение указанного элемента формы
_IEFormElementOptionSelect Задаёт значение указанного элемента формы
_IEFormElementRadioSelect Задаёт значение указанного элемента формы
_IEFormElementSetValue Задаёт значение указанного элемента формы
_IEFormGetCollection Возвращает объект коллекции, представляющий формы в документе или отдельную форму по индексу
_IEFormGetObjByName Возвращает ссылку на объект формы по имени
_IEFormImageClick Имитирует щелчок мыши по <input type=image>. Сопоставление по подстроке alt-текста, name или src
_IEFormReset Сбрасывает указанную форму, возвращая её значения к загруженным по умолчанию
_IEFormSubmit Отправляет указанную форму
_IEFrameGetCollection Возвращает объект-коллекцию фреймов FrameSet или iFrame обычной страницы либо отдельный Frame/iFrame по индексу
_IEFrameGetObjByName Возвращает ссылку на Frame или iFrame по имени
_IEGetObjById Возвращает объектную переменную по id
_IEGetObjByName Возвращает объектную переменную по имени
_IEHeadInsertEventScript Вставляет JavaScript в Head документа
_IEImgClick Имитирует щелчок мыши по изображению. Сопоставление по подстроке alt-текста, name или src
_IEImgGetCollection Возвращает объектную переменную с коллекцией тегов IMG документа или одно изображение по индексу
_IEIsFrameSet Проверяет, содержит ли указанное окно FrameSet
_IELinkClickByIndex Имитирует клик мыши по ссылке с заданным индексом (отсчёт от 0, в порядке исходного кода)
_IELinkClickByText Имитирует щелчок мыши по ссылке, текст которой содержит указанную подстроку
_IELinkGetCollection Возвращает объект-коллекцию со всеми ссылками документа или одну ссылку по индексу
_IELoadWait Ожидает завершения загрузки страницы браузера перед возвратом
_IELoadWaitTimeout Возвращает или задаёт текущий таймаут (в миллисекундах), который _IELoadWait() будет ждать перед прерыванием
_IENavigate Переводит существующее окно браузера на указанный URL
_IEPropertyGet Возвращает выбранное свойство браузера или DOM-элемента
_IEPropertySet Устанавливает выбранное свойство браузера или DOM-элемента
_IEQuit Закрывает браузер и удаляет объектную ссылку на него
_IETableGetCollection Возвращает объектную переменную с коллекцией всех таблиц документа или одну таблицу по индексу
_IETableWriteToArray Считывает содержимое таблицы в массив
_IETagNameAllGetCollection Возвращает объект-коллекцию всех элементов документа или иерархии документа в порядке исходного кода или один элемент по индексу
_IETagNameGetCollection Возвращает объект-коллекцию всех элементов объекта с указанным TagName или один элемент по индексу
