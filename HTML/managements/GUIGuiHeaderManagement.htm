Работа с GuiHeader

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiHeader.au3>.

 

Функция Описание
_GUICtrlHeader_AddItem Добавляет новый элемент в элемент управления Header
_GUICtrlHeader_ClearFilter Очищает фильтр в элементе управления Header
_GUICtrlHeader_ClearFilterAll Очищает все фильтры в элементе управления Header
_GUICtrlHeader_Create Создаёт элемент управления Header
_GUICtrlHeader_CreateDragImage Создаёт полупрозрачную версию изображения элемента для использования как изображение перетаскивания
_GUICtrlHeader_DeleteItem Удаляет элемент из элемента управления Header
_GUICtrlHeader_Destroy Удаляет элемент управления Header
_GUICtrlHeader_EditFilter Начинает редактирование указанного фильтра
_GUICtrlHeader_GetBitmapMargin Получает ширину отступа растра
_GUICtrlHeader_GetFilterText Получает информацию о фильтре элемента
_GUICtrlHeader_GetImageList Получает дескриптор списка изображений
_GUICtrlHeader_GetItem Получает информацию о элементе заголовка
_GUICtrlHeader_GetItemAlign Получает выравнивание текста элемента
_GUICtrlHeader_GetItemBitmap Получает дескриптор растрового изображения элемента
_GUICtrlHeader_GetItemCount Получает количество элементов
_GUICtrlHeader_GetItemDisplay Возвращает информацию об отображении элемента
_GUICtrlHeader_GetItemFlags Возвращает информацию о флагах элемента
_GUICtrlHeader_GetItemFormat Возвращает формат элемента
_GUICtrlHeader_GetItemImage Получает индекс изображения в списке изображений
_GUICtrlHeader_GetItemOrder Получает порядок, в котором появляется элемент
_GUICtrlHeader_GetItemParam Получает значение параметра элемента
_GUICtrlHeader_GetItemRect Получает ограничивающий прямоугольник для заданного элемента
_GUICtrlHeader_GetItemRectEx Получает ограничивающий прямоугольник для заданного элемента
_GUICtrlHeader_GetItemText Получает текст элемента
_GUICtrlHeader_GetItemWidth Получает ширину элемента
_GUICtrlHeader_GetOrderArray Получает текущий порядок элементов слева направо в элементе управления Header
_GUICtrlHeader_GetUnicodeFormat Возвращает флаг формата символов Юникода для элемента управления.
_GUICtrlHeader_HitTest Проверяет точку, чтобы определить, какой элемент находится в указанной точке
_GUICtrlHeader_InsertItem Вставляет новый элемент в элемент управления Header
_GUICtrlHeader_Layout Получает правильный размер и позицию элемента управления
_GUICtrlHeader_OrderToIndex Получает значение индекса элемента на основе его порядка
_GUICtrlHeader_SetBitmapMargin Устанавливает ширину отступа, в пикселях, растрового изображения
_GUICtrlHeader_SetFilterChangeTimeout Устанавливает интервал ожидания изменения фильтра
_GUICtrlHeader_SetHotDivider Изменяет цвет активного разделителя
_GUICtrlHeader_SetImageList Назначает список изображений
_GUICtrlHeader_SetItem Устанавливает информацию об элементе
_GUICtrlHeader_SetItemAlign Устанавливает выравнивание текста элемента
_GUICtrlHeader_SetItemBitmap Устанавливает дескриптор растрового изображения элемента
_GUICtrlHeader_SetItemDisplay Устанавливает информацию об отображении элемента
_GUICtrlHeader_SetItemFlags Устанавливает информацию о флагах элемента
_GUICtrlHeader_SetItemFormat Устанавливает формат элемента
_GUICtrlHeader_SetItemImage Устанавливает индекс изображения в списке изображений
_GUICtrlHeader_SetItemOrder Устанавливает порядок, в котором появляется элемент
_GUICtrlHeader_SetItemParam Устанавливает значение параметра элемента
_GUICtrlHeader_SetItemText Устанавливает текст элемента
_GUICtrlHeader_SetItemWidth Устанавливает ширину элемента
_GUICtrlHeader_SetOrderArray Устанавливает текущий порядок элементов слева направо
_GUICtrlHeader_SetUnicodeFormat Устанавливает флаг формата символов Юникода для элемента управления.
