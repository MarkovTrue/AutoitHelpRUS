Справочник пользовательских функций: цвет

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Color.au3>.

 

Функция Описание
_ColorConvertHSLtoRGB Преобразует цвет из пространства HSL в RGB.
_ColorConvertRGBtoHSL Преобразует цвет из пространства RGB в HSL.
_ColorGetBlue Возвращает синюю составляющую заданного цвета
_ColorGetCOLORREF Возвращает цвет в формате COLORREF.
_ColorGetGreen Возвращает зелёную составляющую заданного цвета
_ColorGetRed Возвращает красную составляющую заданного цвета
_ColorGetRGB Возвращает массив со значениями RGB в соответствующих позициях.
_ColorSetCOLORREF Возвращает цвет в формате COLORREF.
_ColorSetRGB Возвращает цвет в формате RGB.
