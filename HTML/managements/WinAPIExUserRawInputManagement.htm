Raw Input

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_DefRawInputProc Вызывает обработчик raw input по умолчанию для обработки сообщений raw input, которые приложение не обработало
_WinAPI_EnumRawInputDevices Перечисляет устройства raw input, подключённые к системе
_WinAPI_GetRawInputBuffer Выполняет буферизованное чтение данных сырого ввода
_WinAPI_GetRawInputBufferLength Получает требуемый размер буфера для вызова функции _WinAPI_GetRawInputBuffer()
_WinAPI_GetRawInputData Получает сырой ввод с указанного устройства
_WinAPI_GetRawInputDeviceInfo Получает информацию об устройстве сырого ввода
_WinAPI_GetRegisteredRawInputDevices Получает информацию об устройствах сырого ввода для текущего приложения
_WinAPI_RegisterRawInputDevices Регистрирует устройства, которые поставляют данные необработанного ввода
