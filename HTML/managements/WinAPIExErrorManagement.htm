Справочник пользовательских функций: Error

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIError.au3>.

 

Функция Описание
_WinAPI_Beep Генерирует простые звуки на динамике
_WinAPI_FormatMessage Форматирует строку сообщения
_WinAPI_GetErrorMessage Получает текстовое сообщение об ошибке для указанного кода системной ошибки
_WinAPI_GetLastError Возвращает значение кода последней ошибки вызывающего потока
_WinAPI_GetLastErrorMessage Возвращает сообщение об ошибке потока вызова
_WinAPI_MessageBeep Воспроизводит звуковую волну
_WinAPI_MsgBox Отображает окно сообщения с более широким полем, чем оригинал
_WinAPI_SetLastError Задаёт код последней ошибки для вызывающего потока
_WinAPI_ShowError Отображает окно сообщения об ошибке с опциональным выходом
_WinAPI_ShowLastError Отображает последний код ошибки и сообщение
_WinAPI_ShowMsg Отображает окно сообщения "Информация"
