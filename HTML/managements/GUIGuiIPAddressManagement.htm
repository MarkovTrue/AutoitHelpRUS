Справочник пользовательских функций: GuiIPAddress

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiIPAddress.au3>.

 

Функция Описание
_GUICtrlIpAddress_ClearAddress Очищает содержимое элемента управления IP-адрес
_GUICtrlIpAddress_Create Создаёт элемент управления IP-адрес
_GUICtrlIpAddress_Destroy Удаляет элемент управления IP-адрес
_GUICtrlIpAddress_Get Получает адрес из элемента управления IP-адрес
_GUICtrlIpAddress_GetArray Получает адрес из элемента управления IP-адрес
_GUICtrlIpAddress_GetEx Получает адрес из элемента управления IP-адрес
_GUICtrlIpAddress_IsBlank Определяет, пусты ли все поля в элементе управления IP-адрес
_GUICtrlIpAddress_Set Задаёт адрес в элементе управления IP-адресом
_GUICtrlIpAddress_SetArray Задаёт адрес в элементе управления IP-адресом
_GUICtrlIpAddress_SetEx Задаёт адрес в элементе управления IP-адресом
_GUICtrlIpAddress_SetFocus Задаёт фокус ввода в указанное поле элемента ввода IP-адреса
_GUICtrlIpAddress_SetFont Задаёт шрифт элемента управления
_GUICtrlIpAddress_SetRange Задаёт допустимый диапазон значений для указанного поля элемента ввода IP-адреса
_GUICtrlIpAddress_ShowHide Показывает или скрывает элемент управления IP-адрес
