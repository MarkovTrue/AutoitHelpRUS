Директивы Pragma Compile

Специальная директива для управления параметрами Aut2Exe, влияющими на компиляцию скрипта в .exe-файл. Например, изменение свойств файла или значка.

Директива Параметры Описание Пример
Out Filename Задаёт имя файла скомпилированного исполняемого файла. #pragma compile(Out, AutoIt_Example.exe)
Icon Filename Задаёт значок скомпилированного исполняемого файла. #pragma compile(Icon, \..\back2.ico)
ExecLevel none, asInvoker, highestAvailable, requireAdministrator Задаёт уровень выполнения в манифесте скомпилированного исполняемого файла. #pragma compile(ExecLevel, highestavailable)
UPX true, false (default) Сжимает скомпилированный исполняемый файл с помощью UPX. #pragma compile(UPX, false)
AutoItExecuteAllowed true, false (default) Разрешает скомпилированному исполняемому файлу запускать нескомпилированные скрипты. #pragma compile(AutoItExecuteAllowed, true)
Console true, false (default) Компилирует исполняемый файл в консольном режиме. #pragma compile(Console, false)
Compression 1, 3, 5, 7, 9 Задаёт степень сжатия ресурсов FileInstall() для скомпилированного исполняемого файла. #pragma compile(Compression, 3)
Compatibility vista, win7, win8, win81, win10 Задаёт совместимость в манифесте скомпилированного исполняемого файла. #pragma compile(Compatibility, win10)
x64 true, false (default) Компилирует исполняемый файл в 64-битном режиме. #pragma compile(x64, false)
inputboxres true, false (default) Включает/исключает ресурсы InputBox() из скомпилированного исполняемого файла. #pragma compile(inputboxres, false)
Comments Строка комментария – в кавычках или без Добавляет комментарий к скомпилированному исполняемому файлу. #pragma compile(Comments, 'Some, ''Apostrophe'')
CompanyName Название компании – в кавычках или без Задаёт название компании для скомпилированного исполняемого файла. #pragma compile(CompanyName, 'AutoIt')
FileDescription Описание файла – в кавычках или без Задаёт описание файла для скомпилированного исполняемого файла. #pragma compile(FileDescription, AutoIt is a freeware BASIC-like scripting language.)
FileVersion Версия файла; последний параметр необязателен Задаёт версию файла для скомпилированного исполняемого файла. #pragma compile(FileVersion, 1.2.3.4, 11.12.13.14)
InternalName Внутреннее имя – в кавычках или без Задаёт внутреннее имя для скомпилированного исполняемого файла. #pragma compile(InternalName, "Internal Name")
LegalCopyright Строка авторских прав – в кавычках или без Задаёт строку авторских прав для скомпилированного исполняемого файла. #pragma compile(LegalCopyright, Legal stuff here)
LegalTrademarks, Строка торговой марки – в кавычках или без Задаёт строку торговой марки для скомпилированного исполняемого файла. #pragma compile(LegalTrademarks, '"Trademark something, and some text in "quotes" and stuff')
OriginalFilename Filename Задаёт исходное имя файла для скомпилированного исполняемого файла. #pragma compile(OriginalFilename, OriginalName.exe)
ProductName Название продукта – в кавычках или без Задаёт название продукта для скомпилированного исполняемого файла. #pragma compile(ProductName, No Name)
ProductVersion Версия файла; последний параметр необязателен Задаёт версию продукта для скомпилированного исполняемого файла. #pragma compile(ProductVersion, 0.1.3.7, 6.7.4.3)
