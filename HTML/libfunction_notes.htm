Примечания к пользовательским функциям

В AutoIt включён ряд файлов, каждый из которых содержит стандартные функции AutoIt3 – так называемые пользовательские функции (UDF).
UDF – это функции и скриптлеты, написанные на AutoIt3. Их можно подключить через #include в собственном скрипте и использовать без необходимости писать код с нуля или копировать его при каждом использовании.

Пример

#include <Date.au3>
#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "What is the date?", "Today's date is " & _Now())

Скрипт выведет сегодняшнюю дату в полном формате ПК, например "Sunday, December 18, 2004".

 

Многие UDF содержат необязательные параметры, которые можно пропустить. Однако если вы хотите указать необязательный параметр, все предшествующие ему параметры должны быть заданы!
Например, рассмотрим _ArraySort ( ByRef $a_Array [, $i_Descending [, $i_Base=0 [, $i_Ubound=0 [, $i_Dim=1 [, $i_SortIndex=0 ]]]]] )
Параметр, помеченный ByRef, будет обновлён функцией новым значением. Такие параметры должны быть переменными ($xyz) и не могут быть константами.

Большинство UDF сигнализируют об успехе или ошибке через возвращаемое значение; другие – через флаг @error. Некоторые используют оба механизма.
@error = 0 ;всегда означает успех
Return = различается, но обычно ненулевое при успехе – для удобства чтения кода...

Если функция может устанавливать флаг @error, проверяйте его перед использованием возвращаемого значения: при ошибке оно обычно не определено.
