Справочник функций

GUICtrlCreateButton

Создаёт элемент управления Button (кнопка) для GUI.

GUICtrlCreateButton ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

text Текст на кнопке.
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию автоподгонка по ширине текста).
height [необязательный] Высота элемента управления (по умолчанию автоподгонка по высоте текста).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : нет.
принудительные стили : $WS_TABSTOP
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

Кнопка может отображать значок или изображение с помощью стилей $BS_ICON или $BS_BITMAP. Для указания картинки используйте GUICtrlSetImage().

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_BUTTON, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <ButtonConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKSIZE

См. также

GUICoordMode (Option), GUICtrlUpdate..., GUIGetMsg

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Создать GUI с различными элементами управления.
    Local $hGUI = GUICreate("Example", 300, 200)

    ; Создаём элемент управления Button.
    Local $idButton_Notepad = GUICtrlCreateButton("Run Notepad", 120, 170, 85, 25)
    Local $idButton_Close = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    Local $iPID = 0

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idButton_Close
                ExitLoop

            Case $idButton_Notepad
                ; Запустить Notepad с развёрнутым окном.
                $iPID = Run("notepad.exe", "", @SW_SHOWMAXIMIZED)

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI и все элементы управления.
    GUIDelete($hGUI)

    ; Закрыть процесс Блокнота по PID, возвращённому Run.
    If $iPID Then ProcessClose($iPID)
EndFunc   ;==>Example
