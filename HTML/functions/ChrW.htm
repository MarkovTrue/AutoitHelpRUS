Справочник функций

ChrW

Возвращает символ, соответствующий unicode-коду.

ChrW ( UNICODEcode )

Параметры

UNICODEcode Unicode-код в диапазоне 0–65535 (например, 65 возвращает заглавную букву A).

Возвращаемое значение

Успех: Возвращает строку с представлением указанного кода.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0, если UNICODE-код больше 65535.

Примечания

Полный список значений см. в таблице ASCII-коды символов.

См. также

Asc, AscW, Chr, String

Пример

#include <MsgBoxConstants.au3>

Local $sText = ""
For $i = 256 To 512
    $sText = $sText & ChrW($i) ; Также можно использовать $sText &= ChrW($i).
Next
MsgBox($MB_SYSTEMMODAL, "Unicode chars 256 to 512", $sText) ; Показать unicode-символы в диапазоне 256–512.
