Справочник функций

DriveMapDel

Отключает сетевой диск.

DriveMapDel ( "drive" )

Параметры

drive Устройство для отключения, например "O:" или "LPT1:".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если отключение не удалось выполнить.

Примечания

Если подключению не назначена буква диска, для отключения можно использовать имя подключения, например \\server\share

См. также

DriveMapAdd, DriveMapGet

Пример

; Подключить диск X к \\myserver\stuff под текущим пользователем
DriveMapAdd("X:", "\\myserver\stuff")

; Отключить
DriveMapDel("X:")
