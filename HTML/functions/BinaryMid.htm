BinaryMid

Извлекает заданное количество байтов из двоичных данных.

BinaryMid ( binary, start [, count] )

Параметры

binary Двоичные данные для обработки.
start Позиция первого извлекаемого байта (1 = первый байт).
count [необязательный] Количество извлекаемых байтов. По умолчанию извлекается весь остаток двоичных данных.

Возвращаемое значение

Возвращает извлечённые двоичные данные.

Примечания

Если start выходит за границы данных, возвращается пустое двоичное значение. Если start допустим, а count выходит за границы, возвращается весь остаток двоичных данных.

См. также

Binary, BinaryLen

Пример

#include <MsgBoxConstants.au3>

Local $dBinary = Binary("0x10203040") ; Создать двоичные данные из строки.
Local $dExtract = BinaryMid($dBinary, 2, 2)
MsgBox($MB_SYSTEMMODAL, "", "The 2nd and 3rd bytes are: " & $dExtract)
