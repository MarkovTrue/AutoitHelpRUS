Справочник функций

WinGetProcess

Получает идентификатор процесса (PID), связанного с окном.

WinGetProcess ( "title" [, "text"] )

Параметры

title Заголовок/hWnd/класс окна, PID которого нужно получить. См. Специальное определение заголовка.
text [необязательный] Текст окна, PID которого нужно получить. По умолчанию – пустая строка. См. Специальное определение текста.

Возвращаемое значение

Успех: Возвращает идентификатор процесса (PID).
Ошибка: Возвращает -1, если окно не найдено.

См. также

ProcessList, ProcessWait, ProcessWaitClose

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Получаем PID Блокнота по дескриптору окна, возвращённому WinWait.
    Local $iPID = WinGetProcess($hWnd)

    ; Выводим PID окна.
    MsgBox($MB_SYSTEMMODAL, "", "The PID is: " & $iPID)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
