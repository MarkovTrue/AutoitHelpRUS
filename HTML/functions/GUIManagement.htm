Работа с GUI

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

See Gui Constants include files if you need to use the related controls Constants .

 

Функция Описание
GUICreate Создаёт GUI-окно.
GUICtrlGetHandle Возвращает дескриптор элемента управления, а также специальные дескрипторы элементов (Menu, ContextMenu, TreeViewItem).
GUICtrlGetState Возвращает текущее состояние элемента управления.
GUICtrlRead Считывает состояние или данные элемента управления.
GUICtrlRecvMsg Отправляет сообщение элементу управления и получает информацию через lParam.
GUICtrlSendMsg Отправляет сообщение элементу управления.
GUICtrlSendToDummy Отправляет сообщение элементу управления Dummy.
GUIDelete Удаляет окно GUI и все содержащиеся в нём элементы управления.
GUIGetCursorInfo Возвращает положение курсора мыши относительно окна GUI.
GUIGetMsg Опрашивает GUI, не произошли ли какие-либо события.
GUIGetStyle Возвращает стили окна GUI.
GUIRegisterMsg Регистрирует пользовательскую функцию для обработки известного идентификатора сообщения Windows (WM_MSG).
GUIStartGroup Определяет, что все последующие создаваемые элементы управления будут объединены в группу.
GUISwitch Переключает текущее окно, используемое функциями GUI.
