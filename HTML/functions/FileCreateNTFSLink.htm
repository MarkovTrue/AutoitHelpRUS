FileCreateNTFSLink

Создаёт жёсткую ссылку NTFS на файл или каталог.

FileCreateNTFSLink ( "source", "hardlink" [, flag = 0] )

Параметры

source Путь к источнику, на который создаётся жёсткая ссылка.
hardlink Путь создаваемой жёсткой ссылки.
flag [необязательный] флаг, определяющий поведение при совпадении имён ссылок.
Допустимо комбинировать следующие значения:
$FC_NOOVERWRITE (0) = (по умолчанию) не перезаписывать существующую ссылку
$FC_OVERWRITE (1) = перезаписывать существующую ссылку

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Каталог назначения должен существовать заранее.

Функция работает только на томах с файловой системой NTFS.

Если источник – файл, жёсткая ссылка должна располагаться на том же томе.
Если источник – каталог, допускается использование разных томов.

Для жёсткой ссылки можно применять FileDelete() и FileMove().

Для управления ссылками через Проводник можно воспользоваться расширением оболочки NTFSLink.

См. также

FileCreateShortcut

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создаём временный файл для создания ссылки.
    If Not FileWrite($sFilePath, "This is an example of using FileCreateNTFSLink.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Создаём NTFS-ссылку с .txt-файла на .log-файл на рабочем столе.
    Local $iNTFSLink = FileCreateNTFSLink($sFilePath, @TempDir & "\ExampleNTFSLink.log")

    ; Выводим сообщение о результате создания NTFS-ссылки.
    If $iNTFSLink Then
        ; Открываем каталог рабочего стола.
        ShellExecute(@TempDir)

        MsgBox($MB_SYSTEMMODAL, "", "The NTFS link was created." & @CRLF & "FileCreateNTFSLink returned: " & $iNTFSLink)
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The NTFS link wasn't created." & @CRLF & "FileCreateNTFSLink returned: " & $iNTFSLink)
    EndIf

    ; Удаляем временные файлы.
    FileDelete($sFilePath)
    FileDelete(@TempDir & "\ExampleNTFSLink.log")
EndFunc   ;==>Example
