Справочник функций

BitRotate

Выполняет операцию побитового сдвига с циклическим переносом.

BitRotate ( value [, shift = 1 [, size = "W"]] )

Параметры

value Число, над которым выполняется операция.
shift [необязательный] Число бит для циклического сдвига влево (отрицательные значения сдвигают вправо). По умолчанию 1.
size [необязательный] Символ, указывающий число бит, участвующих в циклическом сдвиге. По умолчанию 16 бит. См. ниже.
Параметр size:
"B" сдвинуть биты младшего байта (8 бит).
"W" сдвинуть биты младшего слова (16 бит).
"D" сдвинуть биты двойного слова (32 бита).

Возвращаемое значение

Успех: Возвращает значение, циклически сдвинутое на заданное число бит.
Ошибка: Устанавливает @error ≠ 0 при недопустимом size.

Примечания

Помните, что для чисел можно использовать шестнадцатеричную запись.

См. также

BitAND, BitNOT, BitOR, BitShift, BitXOR, Hex

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Примечание: "b" обозначает байт.

    ; Присвоить локальной переменной результат циклического сдвига влево для 2.
    Local $iBitRotate1 = BitRotate(2, 1) ; 2 = 0010b сдвиг влево на 1 -> 4 = 0100b

    ; Примечание: эквивалентно BitShift(2, -1)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate1)

    ; Присвоить локальной переменной результат циклического сдвига вправо для 1.
    Local $iBitRotate2 = BitRotate(1, -1) ; 1 = 0001b сдвиг вправо на 1 -> 32768 (32 бита) = 1000 0000 0000 0000b

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate2)

    ; Присвоить локальной переменной результат циклического сдвига вправо для 14.
    Local $iBitRotate3 = BitRotate(14, -2) ; 14 = 1110b сдвиг вправо на 2 -> 32771 (16 бит) = 1000 0000 0000 0011b

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate3)

    ; Присвоить локальной переменной результат циклического сдвига вправо для 14 на 32 бита.
    Local $iBitRotate4 = BitRotate(14, -2, "D")
    ; 14 = 1110b сдвиг вправо на 2 -> -2147483645 (32 бита) = 1000 0000 0000 0000 0000 0000 0000 0011b (первый бит – знаковый)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitRotate4)
EndFunc   ;==>Example
