GUISetFont

Устанавливает шрифт по умолчанию для окна GUI.

GUISetFont ( size [, weight [, attribute [, fontname [, winhandle [, quality]]]]] )

Параметры

size Размер шрифта (по умолчанию 8.5).
weight [необязательный] Жирность шрифта в диапазоне от 0 до 1000. Например, 400 – обычный, 700 – жирный. Если значение равно 0, используется жирность по умолчанию.
Для удобства определены следующие константы:
$FW_DONTCARE = 0 (жирность по умолчанию)
$FW_THIN = 100
$FW_EXTRALIGHT = 200
$FW_LIGHT = 300
$FW_NORMAL = 400
$FW_MEDIUM = 500
$FW_SEMIBOLD = 600
$FW_BOLD = 700
$FW_EXTRABOLD = 800
$FW_HEAVY = 900

Константы определены в FontConstants.au3.
attribute [необязательный] Атрибуты шрифта – сумма следующих значений:
$GUI_FONTNORMAL (0) = обычный (по умолчанию)
$GUI_FONTITALIC (2) = курсив
$GUI_FONTUNDER (4) = подчёркнутый
$GUI_FONTSTRIKE (8) = зачёркнутый

Константы определены в GUIConstantsEx.au3.
fontname [необязательный] Используемый шрифт. Если строка пустая ("") или шрифт не найден, используется шрифт GUI операционной системы по умолчанию.
winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).
quality [необязательный] Качество шрифта. Допустимые значения:
$DEFAULT_QUALITY (0) = Внешний вид шрифта не важен (по умолчанию).
$DRAFT_QUALITY (1) = Внешний вид менее важен, чем при $PROOF_QUALITY. Для растровых шрифтов GDI масштабирование включено, доступно больше размеров, но качество может быть ниже. Жирный, курсив, подчёркивание и зачёркивание синтезируются при необходимости.
$PROOF_QUALITY (2) = (по умолчанию) Качество символов важнее точного соответствия атрибутам логического шрифта. Для растровых шрифтов GDI масштабирование отключено, выбирается ближайший по размеру шрифт. Жирный, курсив, подчёркивание и зачёркивание синтезируются при необходимости.
$NONANTIALIASED_QUALITY (3) = Шрифт никогда не сглаживается.
$ANTIALIASED_QUALITY (4) = Шрифт всегда сглаживается, если поддерживает это и его размер не слишком мал или велик.
$CLEARTYPE_QUALITY (5) = Текст отображается (по возможности) с использованием метода сглаживания ClearType. Подробнее об условиях недоступности ClearType см. на странице MSDN для LOGFONT.
Если не выбрано ни $ANTIALIASED_QUALITY, ни $NONANTIALIASED_QUALITY, сглаживание применяется только при включённом параметре "Сглаживание шрифтов" в Панели управления.
Константы определены в FontConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Полный список шрифтов Windows и поддерживаемых версий ОС см. в приложении.

Размер может содержать десятичную дробь, например 8.5.

Для некоторых элементов управления, таких как Label, размер по умолчанию может быть 8.5 вместо 9 в зависимости от значений темы Windows.

Для параметра Quality обратитесь к MSDN – некоторые установки Windows XP требуют CLEARTYPE_QUALITY=5.

Пример

#include <FontConstants.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI default font") ; создаст диалоговое окно, отображаемое по центру

    Local $sFont = "Comic Sans MS"
    GUISetFont(9, $FW_NORMAL, $GUI_FONTUNDER, $sFont)  ; отобразит подчёркнутые символы
    GUICtrlCreateLabel("underlined label", 10, 20)

    GUISetFont(9, $FW_NORMAL, $GUI_FONTITALIC, $sFont)  ; отобразит подчёркнутые символы
    GUICtrlCreateLabel("italic label", 10, 40)

    GUISetFont(9, $FW_NORMAL, $GUI_FONTSTRIKE, $sFont)  ; отобразит подчёркнутые символы
    GUICtrlCreateLabel("strike label", 10, 60)

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
