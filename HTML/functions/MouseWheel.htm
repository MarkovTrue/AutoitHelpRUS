Справочник функций

MouseWheel

Прокручивает колёсико мыши вверх или вниз.

MouseWheel ( "direction" [, clicks = 1] )

Параметры

direction Направление прокрутки:
$MOUSE_WHEEL_UP ("up")
$MOUSE_WHEEL_DOWN ("down")

Константы определены в "AuoItConstants.au3".
clicks [необязательный] Количество щелчков колёсика. По умолчанию 1.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если направление не распознано.

См. также

MouseClick, MouseClickDrag, MouseCoordMode (Option), MouseGetPos, MouseMove

Пример

#include <AutoItConstants.au3>

; Прокрутить колёсико мыши вверх десять раз.

MouseWheel($MOUSE_WHEEL_UP, 10)
