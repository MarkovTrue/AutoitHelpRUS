Работа с окружением

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
ClipGet Получает текст из буфера обмена.
ClipPut Записывает текст в буфер обмена.
EnvGet Возвращает значение переменной окружения.
EnvSet Задаёт значение переменной окружения.
EnvUpdate Обновляет переменные окружения ОС.
MemGetStats Возвращает сведения об использовании памяти.
