Справочник функций

Floor

Возвращает число, округлённое вниз до ближайшего целого.

Floor ( expression )

Параметры

expression Любое допустимое числовое выражение.

Возвращаемое значение

Возвращает округлённое число.

См. также

Ceiling, Int, Number, Round

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной массив с числами.
    Local $aNumber[8] = [4.8, 4.5, 4.3, 4, -4.8, -4.5, -4.3, -4]

    ; Присвоить локальной переменной строку для хранения результатов.
    Local $sResults = ""

    ; Перебираем элементы массива: вычисляем Floor и форматируем результат.
    For $i = 0 To 7
        $sResults &= "Floor(" & $aNumber[$i] & ") = " & Floor($aNumber[$i]) & @CRLF & ($i = 3 ? @CRLF : "")
    Next

    ; Показать результаты.
    MsgBox($MB_SYSTEMMODAL, "", $sResults)
EndFunc   ;==>Example
