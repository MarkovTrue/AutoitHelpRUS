Справочник функций

GUICtrlCreateGroup

Создаёт элемент управления Group (группа) для GUI.

GUICtrlCreateGroup ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

text Текст элемента управления.
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию – ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию – ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : нет.
принудительные стили : $WS_GROUP, $BS_GROUPBOX.
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Элемент управления Group – это тонкая рамка вокруг элементов управления (обычно переключателей), объединяющая их визуально. Одновременно в группе может быть выбран только один переключатель.
Если нужно несколько групп без видимой рамки, используйте GUIStartGroup() для группировки переключателей.

Для использования указанных выше значений необходимо включить в скрипт #include <ButtonConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKAUTO – автоматическое изменение размера и позиции.

См. также

GUICoordMode (Option), GUIStartGroup

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI group") ; создаст диалоговое окно, отображаемое по центру

    GUICtrlCreateGroup("Group 1", 190, 60, 90, 140)
    GUICtrlCreateRadio("Radio 1", 210, 90, 50, 20)
    GUICtrlCreateRadio("Radio 2", 210, 110, 60, 50)
    GUICtrlCreateGroup("", -99, -99, 1, 1) ; закрываем группу

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
