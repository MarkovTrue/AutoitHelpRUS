Справочник функций

StderrRead

Читает данные из потока STDERR ранее запущенного дочернего процесса.

StderrRead ( process_id [, peek = False [, binary = False]] )

Параметры

process_id Идентификатор дочернего процесса (PID), возвращённый предыдущим вызовом Run.
peek [необязательный] Если True, прочитанные символы не удаляются из потока.
binary [необязательный] Если True, данные читаются в двоичном формате вместо текстового (по умолчанию – текст).

Возвращаемое значение

Успех: Возвращает прочитанные данные; @extended содержит количество прочитанных байт.
Ошибка: Устанавливает @error ≠ 0 при достижении EOF, если STDERR не был перенаправлен или при иной ошибке.

Примечания

StderrRead() читает данные из стандартного потока вывода консольного дочернего процесса. При вызове Run() для этого процесса параметр STD I/O должен включать значение $STDERR_CHILD (4) (см. описание функции Run()).
StderrRead() не блокирует выполнение и возвращает управление немедленно. Для получения всех данных необходимо вызывать функцию в цикле.
Просмотр (peek) не удаляет данные из буфера, однако возвращает доступные данные в обычном режиме.
По умолчанию данные возвращаются в текстовом формате; при указании двоичного режима – в двоичном.

См. также

Run, RunAs, StdinWrite, StdioClose, StdoutRead

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iPID = Run(@ComSpec & " /c DIR Example.au3", @SystemDir, @SW_HIDE, BitOR($STDERR_CHILD, $STDOUT_CHILD))
    Local $sOutput = ""
    While 1
        $sOutput &= StdoutRead($iPID)
        If @error Then ; Выйти из цикла при закрытии процесса или ошибке StdoutRead.
            ExitLoop
        EndIf
        MsgBox($MB_SYSTEMMODAL, "Stdout Read:", $sOutput)
    WEnd

    $sOutput = ''
    While 1
        $sOutput &= StderrRead($iPID)
        If @error Then ; Выйти из цикла, если процесс завершился или StderrRead вернул ошибку.
            ExitLoop
        EndIf
        MsgBox($MB_SYSTEMMODAL, "Stderr Read:", $sOutput)
    WEnd
EndFunc   ;==>Example
