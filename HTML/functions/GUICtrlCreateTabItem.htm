Справочник функций

GUICtrlCreateTabItem

Создаёт элемент управления TabItem (вкладку) внутри существующего элемента управления Tab в GUI.

GUICtrlCreateTabItem ( "text" )

Параметры

text Текст элемента управления.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Чтобы выбрать конкретную вкладку для отображения при открытии GUI, используйте GUICtrlSetState(-1, $GUI_SHOW), как показано в примере.

Важно закрыть структуру вкладок, создав последний TabItem с пустым текстом: GUICtrlCreateTabItem("").

Элементы управления конкретной вкладки должны создаваться после соответствующего TabItem и до создания следующего TabItem или закрытия структуры вкладок. Чтобы добавить новый элемент управления на существующую вкладку, используйте GUISwitch($hWin, $tabitem) для выбора нужного TabItem, а затем создайте нужный элемент. Не забудьте снова закрыть структуру с помощью GUICtrlCreateTabItem("").

Окрасить элемент управления TabItem нельзя (слишком много кода).

GUICtrlRead() возвращает индекс выбранной вкладки, или controlID в расширенном режиме.

Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

См. также

GUICtrlCreateTab, GUICtrlRead, GUICtrlSetState, GUIEventOptions (Option), GUIGetMsg, GUISwitch

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI Tab", 250, 150) ; создаст диалоговое окно, отображаемое по центру

    GUISetBkColor(0x00E0FFFF)
    GUISetFont(9, 300)

    Local $idTab = GUICtrlCreateTab(10, 10, 200, 100)

    GUICtrlCreateTabItem("tab0")
    GUICtrlCreateLabel("label0", 30, 80, 50, 20)
    GUICtrlCreateButton("OK0", 20, 50, 50, 20)
    GUICtrlCreateInput("default", 80, 50, 70, 20)

    GUICtrlCreateTabItem("tab----1")
    GUICtrlCreateLabel("label1", 30, 80, 50, 20)
    GUICtrlCreateCombo("", 20, 50, 60, 120)
    GUICtrlSetData(-1, "Trids|CyberSlug|Larry|Jon|Tylo|guinness", "Jon") ; по умолчанию Jon
    GUICtrlCreateButton("OK1", 80, 50, 50, 20)

    GUICtrlCreateTabItem("tab2")
    GUICtrlSetState(-1, $GUI_SHOW) ; отображается первым
    GUICtrlCreateLabel("label2", 30, 80, 50, 20)
    GUICtrlCreateButton("OK2", 140, 50, 50)

    GUICtrlCreateTabItem("") ; конец определения вкладок

    GUICtrlCreateLabel("Click on tab and see the title", 20, 130, 250, 20)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Цикл до выхода пользователя.
    While 1
        $idMsg = GUIGetMsg()

        If $idMsg = $GUI_EVENT_CLOSE Then ExitLoop
        If $idMsg = $idTab Then
            ; отобразить нажатую вкладку
            WinSetTitle("My GUI Tab", "", "My GUI Tab" & GUICtrlRead($idTab))
        EndIf
    WEnd
EndFunc   ;==>Example
