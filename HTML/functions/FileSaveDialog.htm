FileSaveDialog

Открывает диалог сохранения файла.

FileSaveDialog ( "title", "init dir", "filter" [, options = 0 [, "default name" [, hwnd]]] )

Параметры

title Текст заголовка диалогового окна.
init dir Начальный каталог, выбранный в дереве файлов диалога.
filter Фильтр типов файлов: одиночный, например "All (*.*)" или "Text files (*.txt)", либо несколько групп, разделённых символом "|", например "All (*.*)|Text files (*.txt)" (см. раздел Примечания).
options [необязательный] параметры диалога; для одновременного использования нескольких параметров объедините значения через BitOR.
$FD_PATHMUSTEXIST (2) = путь должен существовать (если пользователь вводит путь, заканчивающийся обратным слешем)
$FD_PROMPTOVERWRITE (16) = запрашивать подтверждение перезаписи файла

Константы определены в FileConstants.au3.
default name [необязательный] Имя файла, предлагаемое пользователю для сохранения. По умолчанию пустая строка ("").
hwnd [необязательный] Дескриптор окна, используемого в качестве родительского для диалога.

Возвращаемое значение

Успех: Возвращает полный путь выбранного файла. При множественном выборе возвращает строку вида "Каталог|файл1|файл2|...".
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – не удалось выбрать файл.
2 – неверный фильтр файлов.

Примечания

Несколько типов файлов внутри одной группы фильтра, разделяются точкой с запятой как показано в примере.
Несколько групп фильтров разделяются символом "|".

Если указан параметр default name, параметр options также должен быть задан. Если параметры не требуются, передайте 0.

Специальные папки Windows (например "Мои документы") можно задать в качестве корневой, используя соответствующий CLSID из Приложения.

При успешном завершении функции обновляется значение @WorkingDir.

См. также

FileOpenDialog, FileSelectFolder

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

Example()

Func Example()
    ; Создаём локальную константу с сообщением для отображения в FileSaveDialog.
    Local Const $sMessage = "Choose a filename."

    ; Открываем диалог сохранения файла.
    Local $sFileSaveDialog = FileSaveDialog($sMessage, "::{450D8FBA-AD25-11D0-98A8-0800361B1103}", "Scripts (*.au3)", $FD_PATHMUSTEXIST)
    If @error Then
        ; Показать сообщение об ошибке.
        MsgBox($MB_SYSTEMMODAL, "", "No file was saved.")
    Else
        ; Извлекаем имя файла из пути, например Example.au3.
        Local $sFileName = StringTrimLeft($sFileSaveDialog, StringInStr($sFileSaveDialog, "\", $STR_NOCASESENSEBASIC, -1))

        ; Проверяем, имеет ли файл расширение .au3.
        Local $iExtension = StringInStr($sFileName, ".", $STR_NOCASESENSEBASIC)

        ; Если точка найдена – проверяем, равно ли расширение .au3.
        If $iExtension Then
            ; Если расширение не равно .au3 – добавляем его в конец пути.
            If Not (StringTrimLeft($sFileName, $iExtension - 1) = ".au3") Then $sFileSaveDialog &= ".au3"
        Else
            ; Если точка не найдена – добавляем расширение в конец имени файла.
            $sFileSaveDialog &= ".au3"
        EndIf

        ; Выводим имя сохранённого файла.
        MsgBox($MB_SYSTEMMODAL, "", "You saved the following file:" & @CRLF & $sFileSaveDialog)
    EndIf
EndFunc   ;==>Example
