TCPSend

Отправляет данные через подключённый сокет.

TCPSend ( mainsocket, data )

Параметры

mainsocket Идентификатор подключённого сокета (SocketID), возвращённый функцией TCPConnect().
data Двоичные данные или строка, отправляемые в подключённый сокет.

Возвращаемое значение

Успех: Возвращает количество байтов, отправленных в подключённый сокет.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: Возвращает значение Windows API WSAGetLastError (см. MSDN).

Примечания

Для передачи строк Unicode их необходимо кодировать/декодировать с помощью StringToBinary()/BinaryToString().
