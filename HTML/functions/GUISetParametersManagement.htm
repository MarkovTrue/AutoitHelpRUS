Справочник функций: GUI set parameters

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

See Gui Constants include files if you need to use the related controls Constants .

 

Функция Описание
GUISetAccelerators Задаёт таблицу горячих клавиш (акселераторов) для окна GUI.
GUISetBkColor Устанавливает цвет фона окна GUI.
GUISetCoord Задаёт абсолютные координаты для следующего элемента управления.
GUISetCursor Устанавливает значок курсора мыши для окна GUI.
GUISetFont Устанавливает шрифт по умолчанию для окна GUI.
GUISetHelp Задаёт исполняемый файл, запускаемый при нажатии F1.
GUISetIcon Устанавливает значок для окна GUI.
GUISetOnEvent Определяет пользовательскую функцию, вызываемую при нажатии системной кнопки.
GUISetState Изменяет состояние окна GUI.
GUISetStyle Изменяет стили окна GUI.
