Справочник функций

Eval

Возвращает значение переменной, имя которой задано строкой.

Eval ( string )

Параметры

string строка, содержащая имя переменной.

Возвращаемое значение

Успех: Возвращает значение указанной переменной.
Ошибка: Возвращает "" (пустую строку) и устанавливает @error ≠ 0.

Примечания

Если возникла необходимость использовать Eval() для чтения переменной, то в большинстве случаев для создания/записи переменной следует применять Assign(), а для проверки её существования – IsDeclared().

См. также

Assign, Execute, IsDeclared

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить строковой переменной sString данные.
    Assign("sString", "This is a string which is declared using the function Assign")

    ; Найти значение строковой переменной sString и присвоить переменной $sEvalString.
    Local $sEvalString = Eval("sString")

    ; Показать значение $sEvalString. Оно должно совпадать с $sString.
    MsgBox($MB_SYSTEMMODAL, "", $sEvalString)
EndFunc   ;==>Example
