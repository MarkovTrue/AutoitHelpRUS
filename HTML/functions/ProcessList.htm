Справочник функций

ProcessList

Возвращает массив со списком запущенных процессов (имена и PID).

ProcessList ( ["name"] )

Параметры

name [необязательный] Если задано имя, возвращаются только процессы с этим именем.

Возвращаемое значение

Успех: Возвращает массив имён процессов и PID (см. раздел Примечания).
Ошибка: Устанавливает @error = 1, если массив не удалось построить.

Примечания

Возвращаемый массив двумерный и устроен следующим образом:
    $aArray[0][0] = Количество процессов
    $aArray[1][0] = Имя 1-го процесса
    $aArray[1][1] = PID 1-го процесса
    $aArray[2][0] = Имя 2-го процесса
    $aArray[2][1] = PID 2-го процесса
    ...
    $aArray[n][0] = Имя n-го процесса
    $aArray[n][1] = PID n-го процесса

Список может быть пустым, если $aArray[0][0] = 0. @error в этом случае не устанавливается.

См. также

ProcessClose, ProcessExists, ProcessGetStats, ProcessSetPriority, ProcessWait, ProcessWaitClose, WinGetProcess

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Отобразить список процессов Блокнота, возвращённых ProcessList.
    Local $aProcessList = ProcessList("notepad.exe")
    For $i = 1 To $aProcessList[0][0]
        MsgBox($MB_SYSTEMMODAL, "", $aProcessList[$i][0] & @CRLF & "PID: " & $aProcessList[$i][1])
    Next

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
