Справочник функций

TraySetPauseIcon

Загружает/устанавливает указанный значок паузы для трея.

TraySetPauseIcon ( [filename [, iconID]] )

Параметры

filename [необязательный] Имя файла значка, отображаемого как значок паузы в трее.
iconID [необязательный] Идентификатор значка, если файл содержит несколько значков.

Возвращаемое значение

Отсутствует.

Примечания

Для сброса значка паузы к стандартному (красный крест) вызовите функцию без параметров: TraySetPauseIcon().

Положительное число ссылается на значок по строковому эквивалентному имени.
Отрицательное число задаёт 1-базированный индекс. Некоторые DLL допускают извлечение значка только с отрицательными числами.

См. также

TraySetIcon, TraySetState

Пример

#NoTrayIcon

#include <TrayConstants.au3> ; Требуется для константы $TRAY_ICONSTATE_SHOW.

Example()

Func Example()
    TraySetPauseIcon("shell32.dll", 12) ; Установить значок паузы. Он будет мигать при выборе меню трея во время паузы скрипта.
    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    While 1
        Sleep(100) ; Цикл ожидания.
    WEnd
EndFunc   ;==>Example
