ShellExecute

Запускает внешнюю программу через API ShellExecute.

ShellExecute ( "filename" [, "parameters" [, "workingdir" [, "verb" [, showflag]]]] )

Параметры

filename Имя запускаемого файла (EXE, .txt, .lnk и т.д.).
parameters [необязательный] Параметры командной строки. Пустая строка ("") – без параметров.
workingdir [необязательный] Рабочий каталог. Пустая строка ("") – текущий рабочий каталог.
verb [необязательный] Используемый "глагол"; часто используемые глаголы:
$SHEX_OPEN ("open") = открыть указанный файл (EXE, документ или папку)
$SHEX_EDIT ("edit") = открыть документ в редакторе для правки; если файл не является документом, функция завершится ошибкой
$SHEX_PRINT ("print") = распечатать указанный документ; если файл не является документом, функция завершится ошибкой
$SHEX_PROPERTIES ("properties") = отобразить свойства файла или папки
Поведение при отсутствии глагола описано в разделе "Примечания".

Константы определены в AutoItConstants.au3.
showflag [необязательный] Режим отображения окна запускаемой программы:
@SW_HIDE = скрытое окно
@SW_MINIMIZE = свёрнутое окно
@SW_MAXIMIZE = развёрнутое окно.

Возвращаемое значение

Успех: Возвращает PID запущенного процесса или –1, если PID недоступен.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

После запуска программы выполнение скрипта продолжается. Чтобы дождаться завершения дочернего процесса, используйте ShellExecuteWait().

Если глагол не указан, используется глагол по умолчанию, заданный в реестре. Если в реестре глагол по умолчанию не задан, используется "open". Если "open" отсутствует – первый глагол, перечисленный в реестре.

См. также

Run, RunAs, RunAsWait, RunWait, ShellExecuteWait

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получить следующий текстовый файл из папки include в каталоге установки AutoIt.
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\include\_ReadMe_.txt"

    ; Открыть файл readme (.txt) редактором, используемым по умолчанию для текстовых файлов в Windows.
    Local $iPID = ShellExecute($sFile)

    MsgBox($MB_SYSTEMMODAL, "", "PID: " & $iPID)
EndFunc   ;==>Example
