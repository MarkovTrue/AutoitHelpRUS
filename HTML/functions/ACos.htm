ACos

Вычисляет арккосинус числа.

ACos ( expression )

Параметры

expression Любое значение в диапазоне от -1 до 1 включительно.

Возвращаемое значение

Возвращает тригонометрический арккосинус числа. Результат в радианах.

Примечания

ACos(x) математически определён только для чисел из промежутка [-1; 1], поэтому ACos() для других значений x обычно возвращает -1.#IND.

См. также

ASin, ATan, Cos, Sin, Tan

Пример

Пример 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной арккосинус 0.5.
    Local $fArcCos1 = ACos(0.5)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fArcCos1 & " rad.")

    ; Присвоить локальной константе приближённое значение числа Пи.
    Local Const $PI = 3.141592653589793

    ; Присвоить локальной переменной формулу перевода радиан в градусы (равно одному радиану в градусах).
    Local $fRadToDeg = 180 / $PI

    ; Присвоить локальной переменной число в градусах.
    Local $fArcCos2 = $fRadToDeg * ACos(-1)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fArcCos2 & " deg.")
EndFunc   ;==>Example
