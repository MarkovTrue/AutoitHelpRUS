Справочник функций

Cos

Вычисляет косинус числа.

Cos ( expression )

Параметры

expression Значение в радианах.

Возвращаемое значение

Возвращает тригонометрический косинус числа.
Возвращаемое значение находится в диапазоне от -1 до 1.

Примечания

1° = pi / 180 радиан.

См. также

ACos, ASin, ATan, Sin, Tan

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной константе приближённое значение числа Пи.
    Local Const $PI = 3.141592653589793

    ; Присвоить локальной переменной косинус PI / 4 (равен Sqrt(2) / 2).
    Local $fCos1 = Cos($PI / 4)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fCos1)

    ; Присвоить локальной переменной формулу перевода радиан в градусы (равно одному градусу в радианах).
    Local $fDegToRad = $PI / 180

    ; Присвоить локальной переменной косинус 90 градусов (предварительно переведённых в радианы).
    Local $fCos2 = Cos(90 * $fDegToRad)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fCos2)
EndFunc   ;==>Example
