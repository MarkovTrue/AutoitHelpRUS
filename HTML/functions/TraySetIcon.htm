Справочник функций

TraySetIcon

Загружает/устанавливает указанный значок трея.

TraySetIcon ( [filename [, iconID]] )

Параметры

filename [необязательный] Имя файла значка, отображаемого в трее.
iconID [необязательный] Идентификатор значка, если файл содержит несколько значков.

Возвращаемое значение

Отсутствует.

Примечания

Для сброса значка к стандартному вызовите функцию без параметров: TraySetIcon().

Положительное число ссылается на значок по строковому эквивалентному имени.
Отрицательное число задаёт 1-базированный индекс. Некоторые DLL допускают извлечение значка только с отрицательными числами.

Для параметра filename можно указать "blank", "info", "question", "stop" или "warning", чтобы выбрать соответствующий стандартный значок.

См. также

TraySetPauseIcon, TraySetState

Пример

#NoTrayIcon

#include <TrayConstants.au3> ; Требуется для константы $TRAY_ICONSTATE_SHOW.

Opt("TrayMenuMode", 3) ; Стандартные пункты меню трея не будут показаны, и пункты не помечаются при выборе. Это опции 1 и 2 для TrayMenuMode.

Example()

Func Example()
    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    Local $hTimer = TimerInit() ; Запустить таймер и сохранить дескриптор в переменную.
    Local $fDiff = 0, $iIndex = 0

    While 1
        $fDiff = TimerDiff($hTimer) ; Найти разницу во времени с момента предыдущего вызова TimerInit.
        If $fDiff > 1000 Then ; Если разница превышает 1 секунду, изменить значок меню трея.

            $iIndex = -Random(0, 100, 1) ; Использовать отрицательное число для порядковой нумерации.
            TraySetToolTip("Currently using the icon shell32.dll, " & $iIndex & ".") ; Установить всплывающую подсказку меню трея с информацией об индексе значка.
            TraySetIcon("shell32.dll", $iIndex) ; Установить значок меню трея из shell32.dll, используя случайный индекс.
            $hTimer = TimerInit() ; Сбросить таймер.

        EndIf

        Switch TrayGetMsg()
            Case $idExit ; Выйти из цикла.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
