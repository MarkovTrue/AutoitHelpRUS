Справочник функций

Assign

Присваивает переменной значение по её имени.

Assign ( "varname", "data" [, flag = 0] )

Параметры

varname Имя переменной, которой нужно присвоить значение. Не может быть элементом массива; должно содержать только буквы, цифры и символы подчёркивания (по обычным правилам именования переменных).
data Данные, которые нужно присвоить переменной.
flag [необязательный] Управляет способом присваивания (значения складываются):
$ASSIGN_CREATE (0) = (по умолчанию) создать переменную при необходимости
$ASSIGN_FORCELOCAL (1) = принудительно создать в локальной области видимости
$ASSIGN_FORCEGLOBAL (2) = принудительно создать в глобальной области видимости
$ASSIGN_EXISTFAIL (4) = ошибка, если переменной ещё не существует

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если не удалось создать или присвоить переменную.

Примечания

Если Assign() применяется для создания или записи переменной, то для её чтения в большинстве случаев используйте Eval(), а для проверки существования IsDeclared().

См. также

Eval, Execute, IsDeclared

Пример

#include <MsgBoxConstants.au3>

; Присвоить строковой переменной sString данные.
Assign("sString", "This is a string which is declared using the function Assign")

; Найти значение строковой переменной sString и присвоить переменной $sEvalString.
Local $sEvalString = Eval("sString")

; Показать значение $sEvalString. Оно должно совпадать с $sString.
MsgBox($MB_SYSTEMMODAL, "", $sEvalString)
