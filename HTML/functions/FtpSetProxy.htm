Справочник функций

FtpSetProxy

Задаёт интернет-прокси для доступа по FTP.

FtpSetProxy ( mode = 0 [, "proxy:port" [, "username" [, "password"]]] )

Параметры

mode Режим работы прокси:
$PROXY_IE (0) = (по умолчанию) использовать текущие настройки прокси Internet Explorer.
$PROXY_NONE (1) = не использовать прокси (прямой доступ)
$PROXY_SPECIFIED (2) = использовать указанный прокси

Константы определены в AutoItConstants.au3.
proxy:port [необязательный] Адрес и порт прокси. См. раздел Примечания.
username [необязательный] Имя пользователя для прокси, если требуется.
password [необязательный] Пароль для прокси, если требуется.

Возвращаемое значение

Отсутствует.

Примечания

Поддерживаются только прокси-серверы CERN.

Прокси-сервер обычно состоит из веб-адреса и номера порта. Большинство провайдеров используют порт 8080. Если ваш прокси "www.proxy.com", а порт "8080", задайте прокси так:
FtpSetProxy(2, "www.proxy.com:8080")

Большинству прокси-серверов имя пользователя и пароль не нужны.

В режиме 0 можно указать имя пользователя и пароль, и они будут использоваться с прокси, настроенным в Internet Explorer.

См. также

HttpSetProxy, InetGet, InetGetSize, InetRead

Пример

; Без прокси
FtpSetProxy(1)

; Использовать настройки IE для прокси
FtpSetProxy(0)

; Использовать прокси "www-cache.myisp.net" на порту 8080
FtpSetProxy(2, "www-cache.myisp.net:8080")
