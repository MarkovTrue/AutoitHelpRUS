Справочник функций

MouseGetCursor

Возвращает числовой идентификатор текущего курсора мыши.

MouseGetCursor ( )

Возвращаемое значение

Возвращает числовой идентификатор курсора:
    $MCID_UNKNOWN (-1) @error может быть установлен, если дескриптор курсора не найден
    $MCID_HAND (0)
    $MCID_APPSTARTING (1)
    $MCID_ARROW (2)
    $MCID_CROSS (3)
    $MCID_HELP (4)
    $MCID_IBEAM (5)
    $MCID_ICON (6) Устарел для приложений версии 4.0 и выше
    $MCID_NO (7)
    $MCID_SIZE (8) Устарел для приложений версии 4.0 и выше
    $MCID_SIZEALL (9)
    $MCID_SIZENESW (10)
    $MCID_SIZENS (11)
    $MCID_SIZENWSE (12)
    MCID_SIZEWE (13)
    $MCID_UPARROW (14)
    $MCID_WAIT (15)
    $MCID_NONE (16)

    Константы определены в "AutoItConstants.au3".

См. также

MouseGetPos

Пример

#include <MsgBoxConstants.au3>

Sleep(1000) ; Дать время курсору изменить состояние.

; Создать массив возможных состояний курсора с помощью StringSplit.
Local $aArray = StringSplit("Hand|AppStarting|Arrow|Cross|Help|IBeam|Icon (obsolete)|No|" & _
        "Size (obsolete)|SizeAll|SizeNESW|SizeNS|SizeNWSE|SizeWE|UpArrow|Wait|Empty", "|", 2) ; Параметр flag = 2, так как нам не нужно общее количество элементов массива.
#cs
Возвращаемый массив содержит следующие значения:
$aArray[0] = "Hand"
$aArray[1] = "AppStarting"
$aArray[2] = "Arrow"
...
$aArray[16] = "Empty"
#ce

Local $iCursor = MouseGetCursor()
MsgBox($MB_SYSTEMMODAL, "CursorID = " & $iCursor, "Which means " & $aArray[$iCursor] & ".") ; Использовать значение CursorID как индекс массива.
