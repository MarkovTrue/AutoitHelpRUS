Справочник функций

GUICtrlCreatePic

Создаёт элемент управления Picture (изображение) для GUI.

GUICtrlCreatePic ( filename, left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

filename Имя файла загружаемого изображения. Поддерживаемые форматы: BMP, JPG, GIF (кроме анимированных).
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию – ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию – ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $SS_NOTIFY
принудительные стили : $SS_BITMAP | $SS_CENTERIMAGE
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0, если изображение не удалось создать.

Примечания

Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

Чтобы обновить изображение после отображения диалогового окна, используйте GUICtrlSetImage().

Чтобы задать размер элемента по размеру файла, установите ширину и высоту равными 0.

Для создания прозрачного изображения создайте GUI с расширенным стилем WS_EX_LAYERED. Верхний левый пиксель используется в качестве цвета прозрачности. При создании нескольких изображений цвет прозрачности определяется последним. См. пример 2.

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_PIC, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <StaticConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKSIZE.

Если изображение используется в качестве фона, другие элементы управления будут перекрывать его, поэтому важно отключить элемент управления Picture: GUICtrlSetState(-1, $GUI_DISABLE).

Расширенный стиль $GUI_WS_EX_PARENTDRAG позволяет перетаскивать родительское окно для окон без заголовка (без стиля $WS_CAPTION в GUICreate()).

Фон всегда устанавливается прозрачным. GUICtrlSetBkColor() не оказывает влияния на элемент управления Picture.

См. также

GUICoordMode (Option), GUICtrlSetImage, GUICtrlUpdate..., GUIGetMsg

Пример

Пример 1

#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    GUICreate("My GUI picture", 350, 300, -1, -1, $WS_SIZEBOX + $WS_SYSMENU) ; создаст диалоговое окно, отображаемое по центру

    GUISetBkColor(0xE0FFFF)
    Local $idPic = GUICtrlCreatePic("..\GUI\mslogo.jpg", 50, 50, 200, 50)

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    ; изменить размер элемента управления
    GUICtrlSetPos($idPic, 50, 50, 200, 100)

    Local $idMsg
    ; Цикл до выхода пользователя.
    While 1
        $idMsg = GUIGetMsg()

        If $idMsg = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Пример 2

#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Global $g_hGui, $g_aGuiPos, $g_hPic, $g_aPicPos

Example()

Func Example()
    $g_hGui = GUICreate("test transparentpic", 200, 100)
    $g_hPic = GUICreate("", 68, 71, 10, 20, $WS_POPUp, BitOR($WS_EX_LAYERED, $WS_EX_MDICHILD), $g_hGui)
    GUICtrlCreatePic("..\GUI\merlin.gif", 0, 0, 0, 0)

    GUISetState(@SW_SHOW, $g_hPic)
    GUISetState(@SW_SHOW, $g_hGui)

    HotKeySet("{ESC}", "Main")
    HotKeySet("{Left}", "Left")
    HotKeySet("{Right}", "Right")
    HotKeySet("{Down}", "Down")
    HotKeySet("{Up}", "Up")
    $g_aPicPos = WinGetPos($g_hPic)
    $g_aGuiPos = WinGetPos($g_hGui)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    HotKeySet("{ESC}")
    HotKeySet("{Left}")
    HotKeySet("{Right}")
    HotKeySet("{Down}")
    HotKeySet("{Up}")
EndFunc   ;==>Example

Func Main()
    $g_aGuiPos = WinGetPos($g_hGui)
    WinMove($g_hGui, "", $g_aGuiPos[0] + 10, $g_aGuiPos[1] + 10)
EndFunc   ;==>Main

Func Left()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0] - 10, $g_aPicPos[1])
EndFunc   ;==>Left

Func Right()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0] + 10, $g_aPicPos[1])
EndFunc   ;==>Right

Func Down()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0], $g_aPicPos[1] + 10)
EndFunc   ;==>Down

Func Up()
    $g_aPicPos = WinGetPos($g_hPic)
    WinMove($g_hPic, "", $g_aPicPos[0], $g_aPicPos[1] - 10)
EndFunc   ;==>Up
