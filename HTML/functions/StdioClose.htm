Справочник функций

StdioClose

Освобождает все ресурсы, связанные с STDIO-перенаправлением ранее запущенного процесса.

StdioClose ( process_id )

Параметры

process_id Идентификатор дочернего процесса (PID), возвращённый предыдущим вызовом Run.

Возвращаемое значение

Успех: Возвращает не 0.
Ошибка: Возвращает 0, если для процесса не было настроено перенаправление STDIO или ресурсы уже закрыты.

Примечания

Функция закрывает все дескрипторы и освобождает ресурсы, связанные с STDIO. После вызова чтение данных STDIO из процесса становится невозможным; любые данные, ожидающие обработки, будут потеряны.

См. также

Run, RunAs, StderrRead, StdinWrite, StdoutRead

Пример

#include <Array.au3> ; Нужно только для _ArrayDisplay.
#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

; Рекурсивно показать список файлов каталога.
Example()

Func Example()
    Local $sFilePath = @ScriptDir ; Искать в каталоге текущего скрипта.
    Local $sFilter = "*.*" ; Искать все файлы текущего каталога. Список валидных wildcards – по 'Wildcards' в справке.
    Local $sOutput = "" ; Сохранить вывод StdoutRead в переменную.

    ; Если путь не является каталогом, выйти из функции Example.
    If Not StringInStr(FileGetAttrib($sFilePath), "D") Then
        Return SetError(1, 0, 0)
    EndIf

    ; Убрать хвостовые обратные слэши и добавить один в конце.
    $sFilePath = StringRegExpReplace($sFilePath, "[\\/]+\z", "") & "\"

    #cs
Параметры командной строки DIR:
/B – простой вывод.
/A-D – поиск всех файлов, кроме папок.
/S – поиск в подпапках.
#ce
    Local $iPID = Run(@ComSpec & ' /C DIR "' & $sFilePath & $sFilter & '" /B /A-D /S', $sFilePath, @SW_HIDE, $STDOUT_CHILD)

    ; Закрыть поток Stdio процесса с PID, возвращённым Run. Это вызовет появление диалога, так как чтение stdout станет невозможным.
    StdioClose($iPID)

    While 1
        $sOutput &= StdoutRead($iPID) ; Прочитать поток Stdout по PID, возвращённому Run.
        If @error Then ; Выйти из цикла при закрытии процесса или ошибке StdoutRead.
            ExitLoop
        EndIf
    WEnd

    ; Разбить вывод StdoutRead на массив через StringSplit. Символы возврата каретки (@CRLF) убираются, разделитель – @CRLF (перевод строки).
    Local $aArray = StringSplit(StringTrimRight(StringStripCR($sOutput), StringLen(@LF)), @LF)
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "It appears there was an error trying to find all the files in the current script directory.")
    Else
        ; Показать результаты.
        _ArrayDisplay($aArray)
    EndIf
EndFunc   ;==>Example
