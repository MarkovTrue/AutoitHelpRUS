Справочник функций

StdinWrite

Записывает данные в поток STDIN ранее запущенного дочернего процесса.

StdinWrite ( process_id [, data] )

Параметры

process_id Идентификатор дочернего процесса (PID), возвращённый предыдущим вызовом Run.
data [необязательный] Данные для записи – текст или двоичные данные.

Возвращаемое значение

Успех: Возвращает количество записанных символов.
Ошибка: Устанавливает @error ≠ 0, если STDIN не был перенаправлен или при иной ошибке.

Примечания

StdinWrite() записывает данные в стандартный поток ввода консольного дочернего процесса. При вызове Run() для этого процесса параметр STD I/O должен включать значение $STDIN_CHILD (1) (см. описание функции Run()).

Второй необязательный параметр – строка, которую StdinWrite() записывает в поток. Если функция вызвана без второго аргумента, поток закрывается и дальнейшая запись становится невозможной.

Поток представляет собой буфер FIFO ограниченного размера. Если в момент вызова функции (кроме случая закрытия потока) в буфере нет места, StdinWrite() блокируется до тех пор, пока дочерний процесс не закроет поток или не прочитает достаточно данных. В это время процесс AutoIt приостанавливается: горячие клавиши, сообщения GUI и другие события не обрабатываются.

Перед записью символы преобразуются в ANSI.

Двоичные данные записываются как есть без преобразования в строку. Для вывода шестнадцатеричного представления двоичных данных используйте явное приведение через String().

См. также

Run, RunAs, StderrRead, StdioClose, StdoutRead

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iPID = Run("sort.exe", @SystemDir, @SW_HIDE, $STDIN_CHILD + $STDOUT_CHILD)

    ; Записать строки для сортировки в поток Stdin дочернего процесса sort.exe.
    StdinWrite($iPID, "Banana" & @CRLF & "Elephant" & @CRLF & "Apple" & @CRLF & "Deer" & @CRLF & "Car" & @CRLF)

    ; Вызов StdinWrite без второго параметра закрывает поток.
    StdinWrite($iPID)

    Local $sOutput = "" ; Сохранить вывод StdoutRead в переменную.

    While 1
        $sOutput &= StdoutRead($iPID) ; Прочитать поток Stdout по PID, возвращённому Run.
        If @error Then ; Выйти из цикла при закрытии процесса или ошибке StdoutRead.
            ExitLoop
        EndIf
    WEnd

    MsgBox($MB_SYSTEMMODAL, "", "The sorted string is: " & @CRLF & $sOutput)
EndFunc   ;==>Example
