Справочник функций

StdinWrite

Записывает заданное число символов в поток STDIN ранее запущенного дочернего процесса.

StdinWrite ( process_id [, data] )

Параметры

process_id Идентификатор дочернего процесса (PID), возвращённый предыдущим вызовом Run.
data [необязательный] Данные, которые нужно вывести. Это могут быть текст или двоичные данные.

Возвращаемое значение

Успех: Возвращает число записанных символов.
Ошибка: Устанавливает @error ≠ 0, если STDIN не был перенаправлен для процесса или произошла другая ошибка.

Примечания

StdinWrite() пишет в стандартный поток ввода консоли дочернего процесса, который консольные приложения обычно используют для чтения ввода от пользователя, то есть с клавиатуры. При вызове Run() для дочернего процесса, в который нужно писать, параметр STD I/O должен включать значение $STDIN_CHILD (1), чтобы эта функция работала правильно (см. функцию Run()).

Необязательный второй параметр задаёт строку, которую нужно записать в поток через StdinWrite(). Если вызвать функцию без второго аргумента, StdinWrite() закрывает поток и делает его недоступным для дальнейшей записи.

Поток представляет собой буфер ограниченного произвольного размера, работающий по принципу очереди; если в момент вызова функции (кроме случая вызова для закрытия потока) в потоке нет места для новых символов, StdinWrite() блокирует выполнение (приостанавливает) и не возвращает управление, пока дочерний процесс не закроет поток или не прочитает из него достаточно символов, чтобы запись завершилась. Это означает, что процесс AutoIt будет остановлен, и не будет обработки горячих клавиш, сообщений GUI и т.п., пока дочерний процесс не прочитает из потока STDIN.

Символы преобразуются в ANSI перед записью.

Двоичные данные записываются как есть. Они не преобразуются в строку. Чтобы вывести шестнадцатеричное представление двоичных данных, используйте функцию String(), чтобы явно привести данные к строке.

См. также

Run, RunAs, StderrRead, StdioClose, StdoutRead

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iPID = Run("sort.exe", @SystemDir, @SW_HIDE, $STDIN_CHILD + $STDOUT_CHILD)

    ; Записать строки для сортировки в поток Stdin дочернего процесса sort.exe.
    StdinWrite($iPID, "Banana" & @CRLF & "Elephant" & @CRLF & "Apple" & @CRLF & "Deer" & @CRLF & "Car" & @CRLF)

    ; Вызов StdinWrite без второго параметра закрывает поток.
    StdinWrite($iPID)

    Local $sOutput = "" ; Сохранить вывод StdoutRead в переменную.

    While 1
        $sOutput &= StdoutRead($iPID) ; Прочитать поток Stdout по PID, возвращённому Run.
        If @error Then ; Выйти из цикла при закрытии процесса или ошибке StdoutRead.
            ExitLoop
        EndIf
    WEnd

    MsgBox($MB_SYSTEMMODAL, "", "The sorted string is: " & @CRLF & $sOutput)
EndFunc   ;==>Example
