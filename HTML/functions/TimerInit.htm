Справочник функций

TimerInit

Возвращает дескриптор, который передаётся в TimerDiff() для вычисления разницы в миллисекундах.

TimerInit ( )

Возвращаемое значение

Возвращает дескриптор, который передаётся в TimerDiff() для вычисления разницы в миллисекундах.

Примечания

Возвращаемое значение TimerInit() следует рассматривать как непрозрачный дескриптор и использовать исключительно для передачи в TimerDiff(). Любое другое использование является потенциальной ошибкой.

См. также

TimerDiff

Пример

#include <MsgBoxConstants.au3>

Local $hTimer = TimerInit() ; Запустить таймер и сохранить дескриптор в переменную.
Sleep(3000) ; Подождать 3 секунды.
Local $fDiff = TimerDiff($hTimer) ; Найти разницу во времени с момента предыдущего вызова TimerInit. Переменная с дескриптором TimerInit передаётся в TimerDiff как параметр handle.
MsgBox($MB_SYSTEMMODAL, "Time Difference", $fDiff)
