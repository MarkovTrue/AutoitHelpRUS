StringReplace

Заменяет подстроки в строке.

StringReplace ( "string", "searchstring/start", "replacestring" [, occurrence = 0 [, casesense = 0]] )

Параметры

string Оцениваемая строка.
searchstring/start Искомая подстрока или позиция символа, с которой начинается замена.
replacestring Строка замены.
occurrence [необязательный] Количество замен искомой строки. Отрицательное значение означает замену справа налево.
0 = заменяются все вхождения (по умолчанию)
casesense [необязательный] Флаг, определяющий чувствительность к регистру.
$STR_NOCASESENSE (0) = без учёта регистра, используется локаль пользователя (по умолчанию)
$STR_CASESENSE (1) = с учётом регистра
$STR_NOCASESENSEBASIC (2) = без учёта регистра, упрощённое/быстрое сравнение

Константы определены в StringConstants.au3.

Возвращаемое значение

Возвращает новую строку; количество выполненных замен сохраняется в макросе @extended.

Примечания

По умолчанию или при положительном значении occurrence поиск и замена выполняются слева направо. Например, StringReplace("aaa", "aa", "bb") возвращает "bba".

При использовании метода start параметры occurrence и casesense игнорируются. Функция заменяет символы в string, начиная с указанной позиции, символами из replacestring – заменяется столько символов, сколько содержится в replacestring. Если в string недостаточно символов для вставки всей replacestring, возвращается пустая строка, а @error устанавливается в 1.

При замене без учёта регистра ($STR_NOCASESENSE) в длинной строке время выполнения можно сократить, используя StringRegExpReplace() с ведущей опцией (?i). Обратите внимание: в этом случае метасимволы RegEx \ . ^ $ | [ ( { * + ? # нужно экранировать предшествующим символом \. Различия в производительности см. в примере 2.

См. также

StringAddCR, StringLeft, StringLen, StringLower, StringMid, StringRight, StringStripWS, StringTrimLeft, StringTrimRight, StringUpper, StringRegExpReplace

Пример

#include <MsgBoxConstants.au3>

; Заменить пробел (' ') символом - (минус).
Local $sString = StringReplace("This is a sentence with whitespace.", " ", "-")
Local $iReplacements = @extended
MsgBox($MB_SYSTEMMODAL, "", $iReplacements & " replacements were made and the new string is:" & @CRLF & @CRLF & $sString)
