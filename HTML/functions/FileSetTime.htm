Справочник функций

FileSetTime

Устанавливает временну́ю метку одного или нескольких файлов.

FileSetTime ( "file pattern", "time" [, type = 0 [, recurse = 0]] )

Параметры

file pattern Путь к файлу (файлам), например C:\*.au3, C:\Dir. (Допустимы подстановочные знаки * и ? – см. раздел Примечания).
time Новое время в формате "YYYYMMDDHHMMSS" (год, месяц, день, часы (24-ч формат), секунды). Если строка пустая (""), используется текущее время.
type [необязательный] Тип временно́й метки для изменения:
$FT_MODIFIED (0) = дата изменения (по умолчанию)
$FT_CREATED (1) = дата создания
$FT_ACCESSED (2) = дата последнего доступа

Константы определены в FileConstants.au3.
recurse [необязательный]
$FT_NONRECURSIVE (0) – без рекурсии (по умолчанию).
$FT_RECURSIVE (1) – рекурсивный обход подкаталогов.

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0 в случае ошибки при изменении временно́й метки.

Примечания

Подстановочные знаки описаны в разделе FileFindFirstFile().

Использование даты ранее 1980-01-01 не даст результата.
Попытка изменить временну́ю метку файлов только для чтения вернёт ошибку.

См. также

FileGetAttrib, FileGetTime, FileSetAttrib

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, "This is an example of using FileSetTime.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Установить дату изменения файла на 1 ноября 2003 года и использовать текущее время.
    Local $iFileSetTime = FileSetTime($sFilePath, "20031101", $FT_MODIFIED)

    ; Показать дату изменения файла в формате YYYYMMDDHHMMSS.
    If $iFileSetTime Then
        MsgBox($MB_SYSTEMMODAL, "", "Timestamp:" & @CRLF & FileGetTime($sFilePath, $FT_MODIFIED, 1))
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst setting the timestamp of the file.")
    EndIf

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
