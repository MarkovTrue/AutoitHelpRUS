Справочник функций

GUIDelete

Удаляет окно GUI и все содержащиеся в нём элементы управления.

GUIDelete ( [winhandle] )

Параметры

winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

См. также

GUICreate, GUISwitch

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI") ; создаст диалоговое окно, отображаемое по центру

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete() ; ; вернёт 1
EndFunc   ;==>Example
