Справочник функций

RunWait

Запускает внешнюю программу и приостанавливает выполнение скрипта до её завершения.

RunWait ( "program" [, "workingdir" [, show_flag [, opt_flag]]] )

Параметры

program Полный путь к запускаемой программе (EXE, BAT, COM или PIF; см. раздел Примечания).
workingdir [необязательный] Рабочий каталог. Пустая строка ("") означает текущий рабочий каталог. Это не путь к программе.
show_flag [необязательный] Флаг отображения окна запускаемой программы:
@SW_HIDE – скрытое окно (или ключевое слово Default)
@SW_MINIMIZE – свёрнутое окно
@SW_MAXIMIZE – развёрнутое окно.
opt_flag [необязательный] Управляет взаимодействием родительского и дочернего процессов.
0x10000 ($RUN_CREATE_NEW_CONSOLE) = дочерний консольный процесс создаётся в собственном окне, а не в окне родителя; полезен только для консольных приложений.

Возвращаемое значение

Успех: Возвращает код завершения запущенной программы.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Пути, содержащие пробелы, необходимо заключать в кавычки.

Для выполнения DOS-команд используйте: RunWait(@ComSpec & " /c " & "commandName")    ; не забудьте пробел перед "/c"

После запуска программы выполнение скрипта приостанавливается до её завершения. Чтобы продолжить выполнение немедленно, используйте Run().

Некоторые программы могут завершиться мгновенно, хотя на самом деле продолжают работу, запустив другой процесс. В таких случаях используйте ProcessWaitClose().

См. также

ProcessWait, ProcessWaitClose, Run, RunAs, RunAsWait, ShellExecute, ShellExecuteWait

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот и ждать его закрытия.
    Local $iReturn = RunWait("notepad.exe")

    ; Показать код возврата процесса Блокнота.
    MsgBox($MB_SYSTEMMODAL, "", "The return code from Notepad was: " & $iReturn)
EndFunc   ;==>Example
