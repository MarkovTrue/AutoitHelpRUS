Справочник функций

Round

Возвращает число, округлённое до заданного количества десятичных знаков.

Round ( expression [, decimalplaces] )

Параметры

expression Любое допустимое числовое выражение.
decimalplaces [необязательный] Количество знаков после запятой, до которых выполняется округление. Если параметр опущен, возвращается целое число.

Возвращаемое значение

Возвращает округлённое число.

Примечания

Параметр decimalplaces может быть отрицательным, тогда округление выполняется до единиц, десятков, сотен и т.д. Отображается не более 15 значащих цифр; decimalplaces никогда не дополняет число нулями справа.

См. также

Ceiling, Floor, Int, Number, Random

Пример

Пример 1

#include <MsgBoxConstants.au3>

Local $iRound1 = Round(-1.582, 1) ; Возвращает -1.6.
Local $iRound2 = Round(3.1415, 9) ; Значение не изменяется.
Local $iRound3 = Round(123.5, -1) ; Возвращает 120.

MsgBox($MB_SYSTEMMODAL, "", "The following values were rounded: " & @CRLF & _
        $iRound1 & @CRLF & $iRound2 & @CRLF & $iRound3)

Пример 2

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Объявить локальную переменную-массив для округлённых чисел.
    Local $aRound[5]

    $aRound[0] = Round(-1.582, 1) ; Возвращает -1.6.
    $aRound[1] = Round(3.1415, 9) ; Возвращает 3.1415.
    $aRound[2] = Round(123.5, -1) ; Возвращает 120.
    $aRound[3] = Round(123.5) ; Возвращает 124.
    $aRound[4] = Round(50) ; Возвращает 50.

    ; Показать результаты.
    For $i = 0 To UBound($aRound) - 1
        MsgBox($MB_SYSTEMMODAL, "", "Round" & $i & ": " & $aRound[$i] & @CRLF)
    Next
EndFunc   ;==>Example
