Справочник функций

String

Возвращает строковое представление выражения.

String ( expression )

Параметры

expression Выражение, которое нужно преобразовать в строку.

Возвращаемое значение

Возвращает строку.

Примечания

Максимальная длина строки – 2147483647 символов (однако ни одна строка скрипта AutoIt не может превышать 4095 символов).

См. также

BinaryToString, Chr, ChrW, HWnd, Int, IsString, Number, Ptr, StringToBinary

Пример

#include <MsgBoxConstants.au3>

Local $sString = String(10) ; Выражение, переданное в String, вернёт строковое представление; здесь 10 – это число.
MsgBox($MB_SYSTEMMODAL, "", "$sString contains the value: " & $sString & " and IsString returns: " & IsString($sString))
