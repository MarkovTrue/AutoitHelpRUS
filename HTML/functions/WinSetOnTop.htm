WinSetOnTop

Изменяет атрибут окна "Поверх всех" (Always On Top).

WinSetOnTop ( "title", "text", flag )

Параметры

title Заголовок / дескриптор / класс окна, которое нужно изменить. См. Заголовки и текст окон (расширенно).
text Текст окна, которое нужно изменить. См. Заголовки и текст окон (основы).
flag Определяет, нужно ли установить у окна флаг "TOPMOST".
$WINDOWS_NOONTOP (0) = снять флаг размещения поверх всех
$WINDOWS_ONTOP (1) = установить флаг размещения поверх всех

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если окно не найдено.

Примечания

Сторонние программы, добавляющие в контекстное меню пункт "Always On Top", могут не обновить этот пункт при изменении статуса TOPMOST, вызванном AutoIt.

См. также

WinSetState

Пример

#include <AutoItConstants.au3>

Example()

Func Example()
    ; Получаем дескриптор активного окна.
    Local $hWnd = WinGetHandle("[ACTIVE]")

    ; Устанавливаем активное окно поверх всех окон по дескриптору, возвращённому WinGetHandle.
    WinSetOnTop($hWnd, "", $WINDOWS_ONTOP)

    ; Ждём 2 секунды для отображения изменения.
    Sleep(2000)

    ; Убираем расположение активного окна поверх всех окон.
    WinSetOnTop($hWnd, "", $WINDOWS_NOONTOP)
EndFunc   ;==>Example
