Справочник функций

StringTrimRight

Удаляет заданное количество символов с правого края строки.

StringTrimRight ( "string", count )

Параметры

string Оцениваемая строка.
count Количество удаляемых символов.

Возвращаемое значение

Возвращает строку, из которой удалены count символов справа.

Примечания

Если count выходит за допустимые пределы, возвращается пустая строка.
StringTrimRight($str, $n) функционально эквивалентна StringLeft($str, StringLen($str) - $n)

См. также

StringCompare, StringInStr, StringLeft, StringLen, StringLower, StringMid, StringReplace, StringRight, StringSplit, StringTrimLeft, StringUpper

Пример

#include <MsgBoxConstants.au3>

Local $sString = StringTrimRight("This is a sentence with whitespace.", 5) ; Удалить 5 символов с правого края строки.
MsgBox($MB_SYSTEMMODAL, "", $sString)
