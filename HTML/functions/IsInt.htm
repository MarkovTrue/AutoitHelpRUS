Справочник функций

IsInt

Проверяет, не имеет ли значение переменной или выражения дробной части.

IsInt ( variable )

Параметры

variable Проверяемая переменная или выражение.

Возвращаемое значение

Успех: Возвращает 1 – дробная часть отсутствует.
Ошибка: Возвращает 0 – есть дробная часть.

Примечания

Функция вернёт 1, если значение является числом с плавающей точкой без дробной части (например, 1.000).
Подробное описание типов данных см. в разделе типы данных языка.

См. также

IsArray, IsBinary, IsBool, IsFloat, IsHWnd, IsMap, IsNumber, IsPtr, IsString, StringIsInt, VarGetType

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $bIsInt1 = IsInt(-12345) ; Возвращает 1
    Local $bIsInt2 = IsInt(3.0000) ; Возвращает 1
    Local $bIsInt3 = IsInt(7.5 - 4.5) ; Возвращает 1, т.к. вычисляется как целое число 3
    Local $bIsInt4 = IsInt(4.5) ; Возвращает 0, т.к. значение является числом, но не целым.
    Local $bIsInt5 = IsInt("5432") ; Возвращает 0, так как значение строковое.

    MsgBox($MB_SYSTEMMODAL, "", "IsInt: " & @CRLF & _
            $bIsInt1 & @CRLF & $bIsInt2 & @CRLF & $bIsInt3 & @CRLF & $bIsInt4 & @CRLF & $bIsInt5)
EndFunc   ;==>Example
