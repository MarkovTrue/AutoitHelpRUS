Справочник функций

FileRead

Читает указанное количество символов из ранее открытого файла.

FileRead ( "filehandle/filename" [, count] )

Параметры

filehandle/filename Дескриптор файла, возвращённый предыдущим вызовом FileOpen(). Альтернативно можно передать имя файла строкой в качестве первого параметра.
count [необязательный] Количество символов для чтения. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает прочитанные двоичные данные или строку. @extended содержит количество возвращённых байт или символов.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – файл не открыт в режиме чтения или иная ошибка
-1 – достигнут конец файла.

Примечания

Отрицательное или не заданное значение count приводит к чтению всего файла начиная с текущей позиции.

Если передано имя файла, а не дескриптор, файл будет открыт и закрыт непосредственно во время вызова функции – для обработки больших файлов это значительно медленнее, чем использование дескриптора.
Примечание: не смешивайте дескрипторы и имена файлов – не открывайте файл через FileOpen(), а затем передавайте имя файла в эту функцию. В своих процедурах используйте либо дескрипторы, либо имена файлов, но не оба подхода одновременно.

Поддерживается чтение как в формате ANSI, так и UTF-16/UTF-8 – подробнее см. FileOpen().
Если преобразование UTF-8 в строку невозможно, предполагается формат ANSI. Это может произойти при чтении очень большого числа символов.

Файл можно читать как двоичные данные (байты), открыв его через FileOpen() с двоичным флагом – в этом случае count задаётся в байтах, а не символах. Слишком большое значение count может привести к остановке AutoIt из-за ошибки выделения памяти.

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, "This is an example of using FileRead.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Открыть файл на чтение и сохранить дескриптор в переменную.
    Local $hFileOpen = FileOpen($sFilePath, $FO_READ)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred when reading the file.")
        Return False
    EndIf

    ; Прочитать содержимое файла через дескриптор, возвращённый FileOpen.
    Local $sFileRead = FileRead($hFileOpen)

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Показать содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & $sFileRead)

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
