SendKeepActive

Пытается удерживать указанное окно активным во время выполнения Send().

SendKeepActive ( "title" [, "text"] )

Параметры

title Заголовок, hWnd или класс активируемого окна. См. специальные определения заголовка. Пустая строка отключает функцию.
text [необязательный] Текст окна. По умолчанию пустая строка. См. специальные определения текста.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если окно не найдено.

Примечания

Вызов SendKeepActive("") пытается восстанавливать активное окно между каждым эмулированным нажатием клавиш из Send().

См. также

Send

Пример

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Удерживать окно Блокнота активным при использовании Send.
    SendKeepActive("[CLASS:Notepad]")

    ; Эмулировать ввод строки текста. При попытке переключиться на другое окно окно Блокнота будет
    ; снова выведено на передний план.
    For $i = 1 To 10
        Sleep(500)
        Send("notepad - ")
    Next

    ; Отключить удержание окна Блокнота активным при использовании Send.
    SendKeepActive("")

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)

    ; Сейчас появится экран с просьбой сохранить изменения, classname этого окна
    ; "#32770" и симуляция "TAB" для перехода на вторую кнопку, где симулируется "ENTER" чтобы не "сохранять файл"
    WinWaitActive("[CLASS:#32770]")
    Sleep(500)
    Send("{TAB}{ENTER}")
EndFunc   ;==>Example
