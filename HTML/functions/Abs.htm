Справочник функций

Abs

Вычисляет абсолютное значение (модуль) числа.

Abs ( expression )

Параметры

expression Любое допустимое числовое выражение.

Возвращаемое значение

Возвращает абсолютное значение выражения expression.

Примечания

Если выражение не является числом – возвращается 0.

Пример

#include <MsgBoxConstants.au3>

; Присвоить локальной переменной абсолютное значение.
Local $fAbs = Abs(-123.45)

MsgBox($MB_SYSTEMMODAL, "", $fAbs)
