TCPAccept

Принимает входящее соединение на сокете.

TCPAccept ( mainsocket )

Параметры

mainsocket Идентификатор основного сокета (SocketID), возвращённый функцией TCPListen().

Возвращаемое значение

Успех: Возвращает идентификатор подключённого сокета.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0.
@error: -2 – нет соединения.
Код возврата Windows API WSAGetLastError (см. MSDN).

См. также

TCPCloseSocket, TCPListen, TCPRecv, TCPStartup, TCPTimeout (Option)

Пример

Пример 1

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>

; Я сервер, запускайте меня первым! (Затем запускайте пример TCPConnect).

Example()

Func Example()
    TCPStartup() ; Запустить TCP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальным переменным loopback-адрес IP и порт.
    Local $sIPAddress = "127.0.0.1" ; Этот IP-адрес работает только для тестирования на локальном компьютере.
    Local $iPort = 65432 ; Порт, используемый для подключения.

    ; Привязать к указанному IP-адресу и порту с максимум 100 ожидающих соединений.
    ;(Подробнее см. пример этой функции).
    Local $iListenSocket = TCPListen($sIPAddress, $iPort, 100)
    Local $iError = 0

    ; Если произошла ошибка, показать код ошибки и вернуть False.
    If @error Then
        ; Возможно, кто-то уже слушает на этом IP и порту (скрипт уже запущен?).
        $iError = @error
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Could not listen, Error code: " & $iError)
        Return False
    EndIf

    ; Объявить локальные переменные для сокетов прослушивания и клиентского.
    Local $iSocket = 0
    Do ; Ждать подключения (без лимита).
        ; Принимать входящие подключения (сокет закрывается по завершении; один сокет на клиента).
        $iSocket = TCPAccept($iListenSocket)

        ; Если произошла ошибка, показать код ошибки и вернуть False.
        If @error Then
            $iError = @error
            MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Could not accept the incoming connection, Error code: " & $iError & @CRLF & _WinAPI_GetErrorMessage($iError))
            Return False
        EndIf
    Until $iSocket <> -1 ;если не равно -1 – клиент подключён.

    ; Закрыть слушающий сокет, чтобы разрешить дальнейшие привязки.
    TCPCloseSocket($iListenSocket)

    MsgBox($MB_SYSTEMMODAL, "", "Client Connected.")

    ; Закрыть сокет.
    TCPCloseSocket($iSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit

Пример 2

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>

; Примечание: полезные комментарии – в Примере 1; здесь демонстрируется только функция SocketToIP.

; Я сервер, запускайте меня первым! (Затем запускайте пример TCPConnect).

Example()

Func Example()
    Local $sMsgBoxTitle = "AutoItVersion = " & @AutoItVersion

    TCPStartup()

    OnAutoItExitRegister("OnAutoItExit")

    Local $sIPAddress = "127.0.0.1"
    Local $iPort = 65432

    Local $iListenSocket = TCPListen($sIPAddress, $iPort, 100)

    If @error Then
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), $sMsgBoxTitle, "Could not listen, Error code: " & @error & @CRLF & @CRLF & _WinAPI_GetErrorMessage(@error))
        Return False
    EndIf

    Local $iSocket = 0
    Do
        $iSocket = TCPAccept($iListenSocket)

        If @error Then
            MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), $sMsgBoxTitle, "Could not accept the incoming connection, Error code: " & @error & @CRLF & @CRLF & _WinAPI_GetErrorMessage(@error))
            Return False
        EndIf
    Until $iSocket <> -1

    TCPCloseSocket($iListenSocket)

    ; Получить IP-адрес, связанный с принятым сокетом, и присвоить его локальной переменной.
    Local $sClientIPAddress = SocketToIP($iSocket)

    ; Примечание: эта функция НЕ работает с сокетом прослушивания; её можно использовать с сокетом, возвращённым TCPConnect.

    ; Отобразить сообщение об успешном соединении с IP-адресом клиента.
    MsgBox($MB_SYSTEMMODAL, $sMsgBoxTitle, "Client Connected, IP Address: " & $sClientIPAddress, 3)

    TCPCloseSocket($iSocket)
EndFunc   ;==>Example

Func SocketToIP($iSocket)
    Local $tSockAddr = 0, $aRet = 0
    $tSockAddr = DllStructCreate("short;ushort;uint;char[8]")
    $aRet = DllCall("Ws2_32.dll", "int", "getpeername", "int", $iSocket, "struct*", $tSockAddr, "int*", DllStructGetSize($tSockAddr))
    If Not @error And $aRet[0] = 0 Then
        $aRet = DllCall("Ws2_32.dll", "str", "inet_ntoa", "int", DllStructGetData($tSockAddr, 3))
        If Not @error Then Return $aRet[0]
    EndIf
    Return 0
EndFunc   ;==>SocketToIP

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit
