Справочник функций

Call

Вызывает пользовательскую или встроенную функцию, имя которой передано в первом параметре.

Call ( "function" [, param_1 [, param_2 [, param_N]]] )

Параметры

function Имя вызываемой функции в виде литеральной строки.
param Аргументы, передаваемые вызываемой функции – см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает значение вызванной функции. В @error и @extended могут оказаться значения, если вызванная функция их устанавливала.
Ошибка: Устанавливает значение @error = 0xDEAD и @extended в 0xBEEF, если функция не существует или передано недопустимое число параметров.

Примечания

Аргументы можно передавать по одному или в виде массива. Этот массив должен быть единственным параметром, его элемент [0] должен иметь значение "CallArgArray", а элементы [1 – N] содержать отдельные аргументы функции. Пример использования см. в примере.

Обратите внимание: флаг @error может установить как Call(), так и сама вызванная функция. Если @error устанавливает Call(), его значение будет 0xDEAD, а @extended – 0xBEEF. Проверку отсутствия функции см. в примере.

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Вызывает функцию без аргументов.
    Call("Test1")

    ; Вызывает функцию с одним параметром и передаёт ей аргумент.
    Call("Test2", "Message from Call()!")

    ; Демонстрация использования специального массива-аргумента.
    Local $aArgs[4]
    $aArgs[0] = "CallArgArray" ; Это обязательно, иначе Call() не распознает массив как содержащий аргументы
    $aArgs[1] = "This is a string" ; Первый параметр – строка
    $aArgs[2] = 47 ; Второй параметр – число
    Local $aArray[2]
    $aArray[0] = "Array Element 0"
    $aArray[1] = "Array Element 1"
    $aArgs[3] = $aArray ; Третий параметр – массив

    ; Специальный массив собран, теперь вызываем функцию
    Call("Test3", $aArgs)

    ; Проверка вызова несуществующей функции. Показан правильный способ проверки
    ; путём сравнения @error и @extended с документированными значениями сбоя.
    Local Const $sFunction = "DoesNotExist"
    Call($sFunction)
    If @error = 0xDEAD And @extended = 0xBEEF Then MsgBox($MB_SYSTEMMODAL, "", "Function does not exist.")
EndFunc   ;==>Example

Func Test1()
    MsgBox($MB_SYSTEMMODAL, "", "Hello")
EndFunc   ;==>Test1

Func Test2($sMsg)
    MsgBox($MB_SYSTEMMODAL, "", $sMsg)
EndFunc   ;==>Test2

Func Test3($sString, $nNumber, $aArray)
    MsgBox($MB_SYSTEMMODAL, "", "The string is: " & @CRLF & $sString)
    MsgBox($MB_SYSTEMMODAL, "", "The number is: " & @CRLF & $nNumber)
    For $i = 0 To UBound($aArray) - 1
        MsgBox($MB_SYSTEMMODAL, "", "Array[" & $i & "] contains:" & @CRLF & $aArray[$i])
    Next
EndFunc   ;==>Test3
