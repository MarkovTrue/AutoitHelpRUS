Справочник функций

GUICtrlRecvMsg

Отправляет сообщение элементу управления и получает информацию через lParam.

GUICtrlRecvMsg ( controlID , msg [, wParam [, lParamType]] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
msg Тип отправляемого сообщения в соответствии с документацией по элементам управления Windows.
wParam [необязательный] Целочисленный первый параметр для отправки элементу управления.
lParamType [необязательный] Определяет тип возвращаемого lParam: 0 (по умолчанию) – wParam и lParam, 1 – строка lParam, 2 – структура RECT в lParam.

Возвращаемое значение

Успех: Возвращает значение, возвращённое API SendMessage Windows.
Ошибка: Возвращает 0.

Примечания

Функция позволяет отправлять специальные сообщения Windows напрямую элементу управления через API SendMessage. Используется для задействования возможностей элементов управления, недоступных через стандартные GUICtrlRead() и GUICtrlUpdate...().

Если параметры wParam и lParamType не указаны, возвращается массив из двух элементов (LPwParam, LPlParam).

RECT возвращается в виде массива из 4 элементов (Left, Top, Right, Bottom).

См. также

GUICtrlSendMsg, GUICtrlUpdate...

Пример

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI с элементом управления Edit.
    Local $hGUI = GUICreate("Example")
    Local $idEdit = GUICtrlCreateEdit("Line 0" & @CRLF, 0, 0, 400, 350)
    Local $idButton_Ok = GUICtrlCreateButton("OK", 310, 370, 85, 25)

    ; Установить данные элемента управления Edit.
    For $i = 1 To 25
        GUICtrlSetData($idEdit, "Line " & $i & @CRLF, 1)
    Next

    ; Установить фокус на edit-элемент.
    GUICtrlSetState($idEdit, $GUI_FOCUS)

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Инициализировать переменную $aCtrlRecvMsg для хранения значения, возвращённого GUICtrlRecvMsg.
    Local $aCtrlRecvMsg = 0

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idButton_Ok
                ; Отправить сообщение EM_GETSEL для получения текущего выделения в Edit.
                $aCtrlRecvMsg = GUICtrlRecvMsg($idEdit, $EM_GETSEL)

                ; Установить фокус на edit-элемент.
                GUICtrlSetState($idEdit, $GUI_FOCUS)

                ; Если GUICtrlRecvMsg вернул 0, произошла ошибка; иначе отобразить содержимое массива.
                If $aCtrlRecvMsg = 0 Then
                    MsgBox($MB_SYSTEMMODAL, "", "An error occurred. The value returned was - " & $aCtrlRecvMsg)
                Else
                    MsgBox($MB_SYSTEMMODAL, "", "Start: " & $aCtrlRecvMsg[0] & " End: " & $aCtrlRecvMsg[1])
                EndIf

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI и все элементы управления.
    GUIDelete($hGUI)
EndFunc   ;==>Example
