FileChangeDir

Изменяет текущий рабочий каталог.

FileChangeDir ( "path" )

Параметры

path Путь, который должен стать текущим рабочим каталогом.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0, если рабочий каталог не удалось изменить.

Примечания

При успешном завершении значение @WorkingDir обновляется.

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Выводим текущий рабочий каталог.
    MsgBox($MB_SYSTEMMODAL, "", "The current working directory: " & @CRLF & @WorkingDir)

    ; Устанавливаем рабочий каталог @WorkingDir равным @WindowsDir. Обычно это C:\Windows.
    FileChangeDir(@WindowsDir)

    ; Выводим рабочий каталог после смены на каталог Windows.
    MsgBox($MB_SYSTEMMODAL, "", "The current working directory: " & @CRLF & @WorkingDir)
EndFunc   ;==>Example
