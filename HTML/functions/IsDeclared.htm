Справочник функций

IsDeclared

Проверяет, была ли объявлена переменная.

IsDeclared ( expression )

Параметры

expression Строка с именем проверяемой переменной.

Возвращаемое значение

Успех: Возвращает $DECLARED_GLOBAL (1) для глобальной переменной или переменной, объявленной вне функций.
Special: Возвращает $DECLARED_LOCAL (-1) для локальной переменной.
Ошибка: Возвращает $DECLARED_UNKNOWN (0), если переменная не найдена.

Константы определены в AutoItConstants.au3

Примечания

Если требуется проверить существование переменной через IsDeclared(), то в большинстве случаев для создания/записи следует использовать Assign(), а для чтения – Eval().

См. также

Assign, Eval

Пример

#include <MsgBoxConstants.au3>

; Проверить, объявлена ли переменная $vVar. Если нет – отобразить сообщение об ошибке.
If Not IsDeclared("vVar") Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable $vVar is not declared.")

    Local $vVar = 0 ; Инициализировать переменную $vVar данными.
    If IsDeclared("vVar") Then ; Проверить, объявлена ли переменная $vVar.
        MsgBox($MB_SYSTEMMODAL, "", "The variable $vVar is declared.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The variable $vVar is not declared.")
    EndIf
EndIf
