Справочник функций

FileCopy

Копирует один или несколько файлов.

FileCopy ( "source", "dest" [, flag = 0] )

Параметры

source Исходный путь копируемого файла (или файлов). Допускаются символы подстановки * и ? – см. раздел Примечания.
dest Путь назначения для скопированного файла (или файлов).
flag [необязательный] флаг, определяющий поведение при совпадении имён файлов.
Допустимо комбинировать следующие значения:
$FC_NOOVERWRITE (0) = (по умолчанию) не перезаписывать существующие файлы
$FC_OVERWRITE (1) = перезаписывать существующие файлы
$FC_CREATEPATH (8) = создать структуру каталогов назначения, если она отсутствует (см. раздел Примечания).

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Об использовании символов подстановки см. описание FileFindFirstFile().

Каталог назначения должен существовать заранее – за исключением случая, когда передан флаг $FC_CREATEPATH (8).
Например, комбинация $FC_OVERWRITE (1) + $FC_CREATEPATH (8) перезаписывает файл назначения и автоматически создаёт всю структуру каталогов, если она отсутствует.

Некоторые атрибуты файла могут препятствовать перезаписи; в таком случае воспользуйтесь FileSetAttrib(), чтобы изменить атрибуты файла.

См. также

DirCopy, DirCreate, FileDelete, FileMove

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создаём временный файл для копирования.
    If Not FileWrite($sFilePath, "This is an example of using FileCopy.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Копируем au3-файлы из временного каталога в новую папку Au3Files.
    FileCopy(@TempDir & "\*.au3", @TempDir & "\Au3Files\", $FC_OVERWRITE + $FC_CREATEPATH)

    ; Показать временный каталог.
    ShellExecute(@TempDir)
EndFunc   ;==>Example
