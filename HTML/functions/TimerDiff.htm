Справочник функций

TimerDiff

Возвращает разницу во времени с момента предыдущего вызова TimerInit().

TimerDiff ( handle )

Параметры

handle Дескриптор, возвращённый предыдущим вызовом TimerInit().

Возвращаемое значение

Возвращает разницу во времени (в миллисекундах) с момента предыдущего вызова TimerInit().

Примечания

На некоторых процессорах функция может возвращать некорректный результат.

См. также

TimerInit

Пример

#include <MsgBoxConstants.au3>

Local $hTimer = TimerInit() ; Запустить таймер и сохранить дескриптор в переменную.
Sleep(3000) ; Подождать 3 секунды.
Local $fDiff = TimerDiff($hTimer) ; Найти разницу во времени с момента предыдущего вызова TimerInit. Переменная с дескриптором TimerInit передаётся в TimerDiff как параметр handle.
MsgBox($MB_SYSTEMMODAL, "Time Difference", $fDiff)
