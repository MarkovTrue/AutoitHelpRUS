Справочник функций

DriveGetDrive

Возвращает массив с перечнем дисков.

DriveGetDrive ( "type" )

Параметры

type Тип искомых дисков:
$DT_ALL ("ALL")
$DT_CDROM ("CDROM")
$DT_REMOVABLE ("REMOVABLE")
$DT_FIXED ("FIXED")
$DT_NETWORK ("NETWORK")
$DT_RAMDISK ("RAMDISK")
$DT_UNKNOWN ("UNKNOWN")

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает массив строк (буква диска с двоеточием) найденных дисков. Нулевой элемент массива содержит количество дисков.
Ошибка: Устанавливает значение @error = 1.

Примечания

@error устанавливается в 1, если параметр не является ни $-константой, ни равнозначной строкой, либо если на компьютере нет дисков запрошенного типа.
Пример: на компьютере без привода CD/DVD вызов DriveGetDrive("CDROM") устанавливает @error в 1.

См. также

DriveGetFileSystem, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Local $aArray = DriveGetDrive($DT_ALL)
If @error Then
    ; Произошла ошибка при получении списка дисков.
    MsgBox($MB_SYSTEMMODAL, "", "It appears an error occurred.")
Else
    For $i = 1 To $aArray[0]
        ; Отобразить все найденные диски и привести букву диска к верхнему регистру.
        MsgBox($MB_SYSTEMMODAL, "", "Drive " & $i & "/" & $aArray[0] & ":" & @CRLF & StringUpper($aArray[$i]))
    Next
EndIf
