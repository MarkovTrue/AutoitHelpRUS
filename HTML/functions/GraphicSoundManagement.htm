Графика и звук

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
Beep Воспроизводит звуковой сигнал для пользователя.
PixelChecksum Вычисляет контрольную сумму для области пикселей.
PixelGetColor Возвращает цвет пикселя по координатам x, y.
PixelSearch Ищет пиксель заданного цвета в прямоугольной области.
SoundPlay Воспроизводит звуковой файл.
SoundSetWaveVolume Устанавливает громкость системного звукового канала Wave в процентах.
