Справочник функций

MsgBox

Отображает простое диалоговое окно сообщения с необязательным таймаутом.

MsgBox ( flag, "title", "text" [, timeout = 0 [, hwnd]] )

Параметры

flag Флаг определяет тип окна сообщения и набор кнопок. См. раздел Примечания.
title Заголовок окна сообщения.
text Текст окна сообщения.
timeout [необязательный] Таймаут в секундах. По истечении таймаута окно закрывается автоматически. По умолчанию 0 – таймаут не задан.
hwnd [необязательный] Дескриптор окна, используемого в качестве родительского для диалога.

Возвращаемое значение

Успех: Возвращает идентификатор нажатой кнопки.
Ошибка: Возвращает $IDTIMEOUT (-1), если истёк таймаут.


Нажатая кнопка Возвращаемое значение
OK $IDOK (1)
CANCEL $IDCANCEL (2)
ABORT $IDABORT (3)
RETRY $IDRETRY (4)
IGNORE $IDIGNORE (5)
YES $IDYES (6)
NO $IDNO (7)
TRY AGAIN $IDTRYAGAIN (10)
CONTINUE $IDCONTINUE (11)

Константы определены в MsgBoxConstants.au3.

Примечания

Параметр flag может быть комбинацией следующих значений:

Имя константы Десятичный флаг Кнопки Шестнадцатеричный флаг
$MB_OK 0 OK 0x0
$MB_OKCANCEL 1 OK и Отмена 0x1
$MB_ABORTRETRYIGNORE 2 Прервать, Повторить, Пропустить 0x2
$MB_YESNOCANCEL 3 Да, Нет и Отмена 0x3
$MB_YESNO 4 Да и Нет 0x4
$MB_RETRYCANCEL 5 Повторить и Отмена 0x5
$MB_CANCELTRYCONTINUE 6 Отмена, Повторить, Продолжить 0x6
$MB_HELP 16384 Добавляет кнопку "Справка". При нажатии или F1 система посылает владельцу сообщение WM_HELP. 0x4000
Имя константы Десятичный флаг Значок Шестнадцатеричный флаг
$MB_ICONNONE 0 (Без значка) 0x0
$MB_ICONERROR 16 Значок "Стоп" 0x10
$MB_ICONQUESTION 32 Значок вопроса 0x20
$MB_ICONWARNING 48 Значок восклицательного знака 0x30
$MB_ICONINFORMATION 64 Значок информации (буква "i" в кружке) 0x40
Имя константы Десятичный флаг Кнопка по умолчанию Шестнадцатеричный флаг
$MB_DEFBUTTON1 0 Первая кнопка 0x0
$MB_DEFBUTTON2 256 Вторая кнопка 0x100
$MB_DEFBUTTON3 512 Третья кнопка 0x200
$MB_DEFBUTTON4 768 Четвёртая кнопка 0x300
Имя константы Десятичный флаг Модальность Шестнадцатеричный флаг
$MB_APPLMODAL 0 Уровень приложения 0x0
$MB_SYSTEMMODAL 4096 Системная модальность (со значком) 0x1000
$MB_TASKMODAL 8192 Задачная модальность 0x2000
Имя константы Десятичный флаг Прочее Шестнадцатеричный флаг
0 (ничего особого) 0x0
$MB_DEFAULT_DESKTOP_ONLY 131072 MsgBox() отображается на рабочем столе интерактивной оконной станции. 0x20000
$MB_RIGHT 524288 Заголовок и текст выровнены по правому краю. 0x80000
$MB_RTLREADING 1048576 Текст и заголовок отображаются справа налево (для иврита и арабского). 0x100000
$MB_SETFOREGROUND 65536 Окно сообщения выводится на передний план. 0x10000
$MB_TOPMOST 262144 MsgBox() – поверх всех окон. 0x40000
$MB_SERVICE_NOTIFICATION 2097152 Функция отображает окно на текущем активном рабочем столе, даже если пользователь не вошёл в систему. 0x200000

Например, для системно-модального окна с кнопками "Да"/"Нет" значение флага: 4096+4 (или 4100), в шестнадцатеричном виде: 0x1000+0x4 (или 0x1004).

Окно сообщения центрируется на экране и масштабируется по тексту. Однако при флаге SYSTEMMODAL (4096) заголовок может быть обрезан.

Если title равен ключевому слову Default, используется @ScriptName.

Примечание: при вызове из службы с флагом $MB_SERVICE_NOTIFICATION параметр timeout не поддерживается. Окно не принадлежит вызывающему процессу.

Константы определены в MsgBoxConstants.au3.

См. также

InputBox, SplashTextOn, ToolTip, TrayTip

Пример

Пример 1

#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "Title", "This message box will timeout after 10 seconds or select the OK button.", 10)

Пример 2

#include <MsgBoxConstants.au3>

Local $iTimeout = 10

; Отобразить окно сообщения с переменной в тексте.
MsgBox($MB_SYSTEMMODAL, "Title", "This message box will timeout after " & $iTimeout & " seconds or select the OK button.", $iTimeout)
