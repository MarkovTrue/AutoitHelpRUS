Справочник функций

GUICtrlSendToDummy

Отправляет сообщение элементу управления Dummy.

GUICtrlSendToDummy ( controlID [, state] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreateDummy().
state [необязательный] Значение, которое впоследствии можно получить через GUICtrlRead().

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

При вызове функции генерируется уведомление, которое можно обработать в цикле сообщений или через GUICtrlSetOnEvent() (как если бы элемент управления был "нажат").

Заметьте: функция не воздействует на Dummy-элемент, если GUI, в котором он был создан, скрыт – по замыслу ни один элемент управления скрытого GUI не может быть задействован.

Если "состояние", отправляемое Dummy-элементу, является числовым, оно ограничено знаковым 32-битным (4 байта) целым числом – передаются только младшие 4 байта больших значений.

См. также

GUICtrlCreateDummy, GUICtrlRead, GUICtrlSetOnEvent

Пример

#include <GUIConstantsEx.au3>

Global $g_idDmy_User, $g_iState = 0

Example()

Func Example()
    Opt("GUIOnEventMode", 1) ; Установить параметр GUIOnEventMode.

    GUICreate("GUISendToDummy", 220, 200, 100, 200)
    GUISetBkColor(0x00E0FFFF) ; Изменить цвет фона GUI.
    GUISetOnEvent($GUI_EVENT_CLOSE, "OnExit") ; Задать событие для вызова функции 'OnExit'.

    $g_idDmy_User = GUICtrlCreateDummy()
    GUICtrlSetOnEvent(-1, "OnDummy") ; Задать событие для вызова функции 'OnExit' при выборе этого элемента управления.

    GUICtrlCreateButton("Click", 70, 170, 85, 25)
    GUICtrlSetOnEvent(-1, "OnClick") ; Задать событие для вызова функции 'OnClick' при выборе этого элемента управления.

    GUICtrlSendToDummy($g_idDmy_User, 1) ; Установить состояние для проверки при нажатии

    ; Отобразить GUI.
    GUISetState(@SW_SHOW)
    ; Цикл до выхода пользователя.
    While 1
        Sleep(100)
    WEnd
EndFunc   ;==>Example

Func OnClick()
    Return GUICtrlSendToDummy($g_idDmy_User) ; Отправить Dummy-элементу сообщение о нажатии кнопки закрытия, что вызовет функцию 'OnExit'.
EndFunc   ;==>OnClick

Func OnDummy()
    If GUICtrlRead($g_idDmy_User) Then
        GUISetBkColor(0x000000FF) ; Изменить цвет фона GUI в зависимости от состояния Dummy
    Else
        Exit
    EndIf
EndFunc   ;==>OnDummy

Func OnExit()
    Exit ; Завершить скрипт.
EndFunc   ;==>OnExit
