Справочник функций

WinSetState

Показывает, скрывает, сворачивает, разворачивает или восстанавливает окно.

WinSetState ( "title", "text", flag )

Параметры

title Заголовок / дескриптор / класс окна, состояние которого нужно изменить. См. Заголовки и текст окон (расширенно).
text Текст окна, состояние которого нужно изменить. См. Заголовки и текст окон (основы).
flag Флаг показа окна:
@SW_HIDE = скрыть окно
@SW_SHOW = показать ранее скрытое окно
@SW_MINIMIZE = свернуть окно
@SW_MAXIMIZE = развернуть окно
@SW_RESTORE = отменить сворачивание или разворачивание окна
@SW_DISABLE = сделать окно недоступным
@SW_ENABLE = сделать окно доступным.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если окно не найдено.

Примечания

Если критериям соответствует несколько окон, используется то, которое было активным последним.
@SW_MINIMIZE и @SW_MAXIMIZE работают даже с модальными диалоговыми окнами.

См. также

ControlHide, WinActivate, WinClose, WinFlash, WinGetState, WinKill, WinMinimizeAll, WinMinimizeAllUndo, WinMove, WinSetOnTop

Пример

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Скрываем окно Блокнота.
    WinSetState($hWnd, "", @SW_HIDE)

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Показываем окно Блокнота.
    WinSetState($hWnd, "", @SW_SHOW)

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
