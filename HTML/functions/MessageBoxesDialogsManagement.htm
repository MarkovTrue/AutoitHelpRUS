Message Boxes and Dialogs functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
InputBox Отображает диалоговое окно для ввода строки пользователем.
MsgBox Отображает простое диалоговое окно сообщения с необязательным таймаутом.
ProgressOff Закрывает окно хода выполнения.
ProgressOn Создаёт настраиваемое окно с индикатором выполнения.
ProgressSet Устанавливает позицию и/или текст ранее созданного окна с индикатором выполнения.
SplashImageOn Создаёт настраиваемое всплывающее окно с изображением.
SplashOff Закрывает окно SplashText или SplashImage.
SplashTextOn Создаёт настраиваемое всплывающее текстовое окно.
ToolTip Создаёт всплывающую подсказку в любом месте экрана.
