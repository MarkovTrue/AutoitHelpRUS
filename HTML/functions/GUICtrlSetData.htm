Справочник функций

GUICtrlSetData

Изменяет данные элемента управления.

GUICtrlSetData ( controlID, data [, default] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
data Combo, List, ListView, ListViewItem: список элементов, разделённых символом Opt("GUIDataSeparatorChar",...).
Progress: значение в процентах.
Slider: числовое значение.
Button, Checkbox, Combo, Edit, Group, Input, Label, Menu, MenuItem, Radio, TabItem, TreeViewItem: заменяет текст.
Date: дата или время в зависимости от стиля элемента управления и региональных настроек.
Dummy: числовое значение.
default [необязательный]
Combo, List: значение по умолчанию.
Edit, Input: если не пустая строка (""), вставляется в текущую позицию курсора.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.
-1 при недопустимых данных.

Примечания

Для Combo и List:
Если значение data совпадает с уже существующим элементом, он устанавливается по умолчанию.
Если data начинается с GUIDataSeparatorChar или является пустой строкой "", предыдущий список уничтожается. Завершающий GUIDataSeparatorChar игнорируется.

Для ListView и ListViewItem:
Для обновления конкретного столбца другие столбцы опускаются, например "||update" обновляет 3-й столбец.
Если "update" пустое, столбец/подэлемент будет удалён. Например, "|" удалит второй столбец/подэлемент, "" – первый.

Для Monthcal:
Формат даты в data: "yyyy/mm/dd".

Для Date:
Дата и время задаются в формате региональных настроек. GUICtrlRead() использует тот же формат по умолчанию.

См. также

GUICtrlCreate..., GUICtrlRead, GUICtrlUpdate..., GUIDataSeparatorChar (Option)

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI") ; создаст диалоговое окно, отображаемое по центру

    GUICtrlCreateCombo("", 10, 10)

    GUICtrlSetData(-1, "item1|item2|item3", "item3")

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно с элементом управления Combo в фокусе

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
