Справочник функций

WinWaitNotActive

Приостанавливает выполнение скрипта до деактивации указанного окна.

WinWaitNotActive ( "title" [, "text" [, timeout = 0]] )

Параметры

title Заголовок/hWnd/класс проверяемого окна. См. особенности задания заголовка.
text [необязательный] Текст проверяемого окна. По умолчанию – пустая строка. См. особенности задания текста.
timeout [необязательный] Время ожидания в секундах, если окно по-прежнему активно. По умолчанию – 0 (без ограничения).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если истекло время ожидания.

Примечания

Опрос окна выполняется примерно каждые 250 миллисекунд.

См. также

WinActive, WinExists, WinWait, WinWaitActive, WinWaitClose, WinWaitDelay (Option)

Пример

; Ждём, пока окно "[CLASS:Notepad]" не станет неактивным.
WinWaitNotActive("[CLASS:Notepad]")

; Ждём не более 5 секунд, пока "[CLASS:Notepad]" не станет неактивным.
WinWaitNotActive("[CLASS:Notepad]", "", 5)
