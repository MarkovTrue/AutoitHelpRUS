Справочник функций

Log

Вычисляет натуральный логарифм числа.

Log ( expression )

Параметры

expression Любое положительное число.

Возвращаемое значение

Успех: Возвращает натуральный логарифм параметра.
Ошибка: Как правило возвращает -1.#IND для неположительных параметров.

Примечания

@error всегда равен 0.

См. также

Exp

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной натуральный логарифм числа 1000.
    Local $fLog1 = Log(1000)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fLog1)

    ; Присвоить локальной переменной десятичный логарифм числа 1000.
    Local $fLog2 = Log10(1000)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fLog2)
EndFunc   ;==>Example

; Пользовательская функция для десятичного логарифма
Func Log10($fNb)
    Return Log($fNb) / Log(10) ; 10 – основание
EndFunc   ;==>Log10
