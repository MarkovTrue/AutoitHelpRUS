Справочник функций

FileSetEnd

Устанавливает конец файла на текущей позиции.

FileSetEnd ( "filehandle" )

Параметры

filehandle Дескриптор файла, возвращённый предыдущим вызовом FileOpen().

Возвращаемое значение

Успех: Возвращает True, если операция выполнена успешно.
Ошибка: Возвращает False.

См. также

FileFlush, FileGetPos, FileOpen, FileRead, FileReadLine, FileSetPos, FileWrite, FileWriteLine

Пример

Пример 1

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Открыть файл на запись (с перезаписью) и сохранить дескриптор в переменную.
    Local $hFileOpen = FileOpen($sFilePath, $FO_OVERWRITE)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Записать данные в файл через дескриптор, возвращённый FileOpen.
    FileWriteLine($hFileOpen, "Line 1")
    FileWriteLine($hFileOpen, "Line 2")
    FileWriteLine($hFileOpen, "Line 3")

    ; Сбросить файл на диск.
    FileFlush($hFileOpen)

    ; Теперь перевести позицию в начало.
    FileSetPos($hFileOpen, 0, $FILE_BEGIN)

    ; Проверить позицию в файле и попытаться прочитать содержимое с текущей позиции.
    MsgBox($MB_SYSTEMMODAL, "", "Position: " & FileGetPos($hFileOpen) & @CRLF & "Data: " & @CRLF & FileRead($hFileOpen))

    ; Записать данные в файл через дескриптор, возвращённый FileOpen.
    FileWriteLine($hFileOpen, "Line 4")
    FileWriteLine($hFileOpen, "Line 5")
    FileWriteLine($hFileOpen, "Line 6")

    ; Теперь перевести позицию в начало.
    FileSetPos($hFileOpen, 0, $FILE_BEGIN)

    ; Проверить позицию в файле и попытаться прочитать содержимое с текущей позиции.
    MsgBox($MB_SYSTEMMODAL, "", "Position: " & FileGetPos($hFileOpen) & @CRLF & "Data: " & @CRLF & FileRead($hFileOpen))

    ; Теперь перевести позицию в начало.
    FileSetPos($hFileOpen, 0, $FILE_BEGIN)

    ; Устанавливаем конец файла на текущей позиции.
    FileSetEnd($hFileOpen)

    ; Проверяем позицию и пытаемся прочитать данные с текущей позиции. Данные уничтожены.
    MsgBox($MB_SYSTEMMODAL, "", "Position: " & FileGetPos($hFileOpen) & @CRLF & "Data: " & @CRLF & FileRead($hFileOpen))

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example

Пример 2

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Открыть файл на запись (с перезаписью) и сохранить дескриптор в переменную.
    Local $hFileOpen = FileOpen($sFilePath, $FO_OVERWRITE)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Записываем данные через дескриптор FileOpen и устанавливаем конец файла.
    FileWrite($hFileOpen, "ABCDEF")
    FileSetEnd($hFileOpen)

    ; Показать размер и содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Size: " & FileGetSize($sFilePath) & @CRLF & "Data: " & @CRLF & FileRead($sFilePath))

    ; Расширяем файл до 9 байт без записи дополнительных данных.
    FileSetPos($hFileOpen, 9, $FILE_BEGIN)
    FileSetEnd($hFileOpen)

    ; Показать размер и содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Size: " & FileGetSize($sFilePath) & @CRLF & "Data: " & @CRLF & FileRead($sFilePath))

    ; Переходим на позицию после ABCDEF (6 байт) и дописываем данные – файл расширится до 12 байт.
    FileSetPos($hFileOpen, 6, $FILE_BEGIN)
    FileWrite($hFileOpen, "HIJKLM")
    FileSetEnd($hFileOpen) ; Установить конец файла.

    ; Показать размер и содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Size: " & FileGetSize($sFilePath) & @CRLF & "Data: " & @CRLF & FileRead($sFilePath))

    ; Усекаем файл до 9 байт.
    FileSetPos($hFileOpen, 9, $FILE_BEGIN)
    FileSetEnd($hFileOpen) ; Установить конец файла.

    ; Показать размер и содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Size: " & FileGetSize($sFilePath) & @CRLF & "Data: " & @CRLF & FileRead($sFilePath))

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
