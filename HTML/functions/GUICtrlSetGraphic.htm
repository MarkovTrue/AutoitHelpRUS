Справочник функций

GUICtrlSetGraphic

Изменяет данные графического элемента управления.

GUICtrlSetGraphic ( controlID, type [, par1 [, ... par6]] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreateGraphic().
type Тип рисования: dot (точка), line (линия), bezier (кривая Безье), rect (прямоугольник), ellipse (эллипс), pie (сектор).
par1...par6 См. таблицу типов графики ниже.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.
-1 при недопустимых данных.

Примечания

Координаты точки (x,y) указываются относительно координат GUICtrlCreateGraphic(). Точка может выходить за пределы графического элемента управления, но должна находиться в пределах окна GUI.

Таблица типов графики

Тип Параметры Результат
$GUI_GR_COLOR Color [,BkColor] Задаёт цвет следующих рисунков. Если BkColor равен $GUI_GR_NOBKCOLOR, фигура не заливается (по умолчанию). Для Color цвет линии по умолчанию черный.
$GUI_GR_MOVE x,y Перемещает текущую позицию без рисования.
$GUI_GR_DOT x,y Рисует точку (наименьший квадрат вокруг точки), следующий рисунок начнётся с предыдущей позиции.
$GUI_GR_PIXEL x,y Рисует пиксель, следующий рисунок начнётся с предыдущей позиции.
$GUI_GR_LINE x,y Рисует линию.
$GUI_GR_BEZIER x,y,x1,y1,x2,y2 Рисует кривую Безье с 2 управляющими точками.
$GUI_GR_RECT x,y,w,h Рисует прямоугольник. При w=h получается квадрат.
$GUI_GR_ELLIPSE x,y,w,h Рисует эллипс. При w=h получается окружность.
$GUI_GR_PIE x,y,r,sa,wa Рисует сектор: radius=r, startangle=sa, sweepangle=wa. Углы в градусах.
$GUI_GR_CLOSE   Закрывает текущий рисунок. Добавляется к $GUI_GR_LINE или $GUI_GR_BEZIER для замыкания. При одиночном использовании игнорируется.
$GUI_GR_REFRESH   Принудительно обновляет отображение после динамического изменения графики.
$GUI_GR_HINT   Отображает управляющие и конечные точки кривой Безье/линии.
$GUI_GR_PENSIZE n Задаёт размер пера для следующих рисунков. Должен быть указан до $GUI_GR_COLOR, чтобы вступить в силу.
$GUI_GR_NOBKCOLOR   Фиктивный BkColor, запрещающий заливку замкнутых фигур. Только контур.

В силу конструктивных ограничений фигуры RECT, ELLIPSE и PIE рисуются первыми. Например, LINE всегда будет нарисована поверх RECT. Если порядок рисования важен для внешнего вида, рекомендуется использовать несколько графических элементов управления вместо одного.

Если между несколькими вызовами GUICtrlSetGraphic() используется Sleep() или другие функции, изменяющие состояние Windows, для перерисовки необходимо вызвать GUICtrlSetGraphic( ,$GUI_GR_REFRESH).

Пример

#include <GUIConstantsEx.au3>

Global Const $g_MAXGr = 6
Global $g_aidGraphics[$g_MAXGr + 1] ; записи 0 и $g_MAXGr не используются для возможности GUICtrlDelete result
Global $g_idBtn_Del, $g_hChild

Example()

Func Example()
    Local $idMsg, $iInc, $i

    GUICreate("My Main", -1, -1, 100, 100)
    Local $idBtn_Del1 = GUICtrlCreateButton("ReCreate", 50, 200, 50)
    GUISetState(@SW_SHOW)
    CreateChild()

    $i = 1
    $iInc = 1
    ;$i=5 ; uncomment to delete starting from last define Graphic control
    ;$iInc=-1

    Do
        $idMsg = GUIGetMsg()
        If $idMsg = $idBtn_Del1 Then $i = Create($iInc)

        If $idMsg = $g_idBtn_Del Then
            GUICtrlDelete($g_aidGraphics[$i])
            $i = $i + $iInc
            If $i < 0 Or $i > $g_MAXGr Then Exit
        EndIf
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

Func Create($iInc)
    GUIDelete($g_hChild)
    CreateChild()
    If $iInc = -1 Then Return 5
    Return 1
EndFunc   ;==>Create

Func CreateChild()
    $g_hChild = GUICreate("My Draw")
    $g_idBtn_Del = GUICtrlCreateButton("Delete", 50, 165, 50)

    $g_aidGraphics[1] = GUICtrlCreateGraphic(20, 50, 100, 100)
    GUICtrlSetBkColor(-1, 0xffffff)
    GUICtrlSetColor(-1, 0)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xff0000, 0xff0000)
    GUICtrlSetGraphic(-1, $GUI_GR_PIE, 50, 50, 40, 30, 270)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0x00ff00, 0xffffff)
    GUICtrlSetGraphic(-1, $GUI_GR_PIE, 58, 50, 40, -60, 90)

    GUICtrlSetGraphic(-1, $GUI_GR_ELLIPSE, 100, 100, 50, 80)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0x00ff00, 0xc0c0ff)
    GUICtrlSetGraphic(-1, $GUI_GR_RECT, 350, 200, 50, 80)
    GUICtrlCreateLabel("label", 65, 100, 30)
    GUICtrlSetColor(-1, 0xff)

    $g_aidGraphics[2] = GUICtrlCreateGraphic(220, 10, 100, 100)

    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0, 0xff)
    GUICtrlSetGraphic(-1, $GUI_GR_PIE, 50, 50, 40, 30, 270)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0x00ff00, 0xffffff)
    GUICtrlSetGraphic(-1, $GUI_GR_PIE, 58, 50, 40, -60, 90)

    $g_aidGraphics[3] = GUICtrlCreateGraphic(220, 110, 100, 100)
    GUICtrlSetBkColor(-1, 0xf08080)
    GUICtrlSetColor(-1, 0xff)
    GUICtrlSetGraphic(-1, $GUI_GR_HINT, 1)

    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xff00)
    GUICtrlSetGraphic(-1, $GUI_GR_RECT, 50, 50, 80, 80)

    $g_aidGraphics[4] = GUICtrlCreateGraphic(20, 200, 80, 80)
    GUICtrlSetBkColor(-1, 0xffffff)
    GUICtrlSetGraphic(-1, $GUI_GR_HINT, 1)

    GUICtrlSetGraphic(-1, $GUI_GR_MOVE, 10, 10)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xff)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 30, 40)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xff00)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 70, 70)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xff0000)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 10, 50)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xffff00)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 10, 10)

    $g_aidGraphics[5] = GUICtrlCreateGraphic(150, 10, 50, 50)
    GUICtrlSetBkColor(-1, 0xa0ffa0)
    GUICtrlSetGraphic(-1, $GUI_GR_MOVE, 20, 20) ; начальная точка
    ; лучше сначала рисовать линии, потом точки
    ; чтобы не менять цвет при каждой операции рисования
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0x0000ff)
    GUICtrlSetGraphic(-1, $GUI_GR_DOT, 30, 30)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 20, 40)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0xff0000)
    GUICtrlSetGraphic(-1, $GUI_GR_DOT, 25, 25)
    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 40, 40)
    GUICtrlSetGraphic(-1, $GUI_GR_DOT, 30, 40)

    $g_aidGraphics[6] = GUICtrlCreateGraphic(110, 260, 230, 130)
    GUICtrlSetColor(-1, 0) ; для отображения чёрной рамки
    GUICtrlSetBkColor(-1, 0xc0c0ff)
    GUICtrlSetGraphic(-1, $GUI_GR_HINT, 3) ; для отображения управляющих линий и конечных точек

    GUICtrlSetGraphic(-1, $GUI_GR_COLOR, 0, 0xff) ; залить синим
    GUICtrlSetGraphic(-1, $GUI_GR_MOVE, 120, 20) ; начальная точка
    GUICtrlSetGraphic(-1, $GUI_GR_BEZIER, 120, 100, 200, 20, 200, 100)
    GUICtrlSetGraphic(-1, $GUI_GR_BEZIER + $GUI_GR_CLOSE, 100, 40, 40, 100, 40, 20)
    GUICtrlSetGraphic(-1, $GUI_GR_LINE, 60, 30) ; начальная точка

    GUISetState(@SW_SHOW)
EndFunc   ;==>CreateChild
