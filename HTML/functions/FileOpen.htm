Справочник функций

FileOpen

Открывает файл для чтения или записи.

FileOpen ( "filename" [, mode = 0] )

Параметры

filename Имя открываемого файла.
mode [необязательный] Режим открытия файла.
Допустимо комбинировать следующие значения:
$FO_READ (0) = режим чтения (по умолчанию)
$FO_APPEND (1) = режим записи (добавление в конец файла)
$FO_OVERWRITE (2) = режим записи (очистка существующего содержимого)
$FO_CREATEPATH (8) = создать структуру каталогов, если она отсутствует (см. раздел Примечания).
$FO_BINARY (16) = принудительный двоичный режим (см. раздел Примечания).
$FO_UNICODE или $FO_UTF16_LE (32) = режим чтения/записи Unicode UTF-16 Little Endian.
$FO_UTF16_BE (64) = режим чтения/записи Unicode UTF-16 Big Endian.
$FO_UTF8 (128) = режим чтения/записи Unicode UTF-8 (с BOM).
$FO_UTF8_NOBOM (256) = режим чтения/записи Unicode UTF-8 (без BOM).
$FO_ANSI (512) = режим чтения/записи ANSI.
$FO_UTF16_LE_NOBOM (1024) = режим чтения/записи Unicode UTF-16 Little Endian (без BOM).
$FO_UTF16_BE_NOBOM (2048) = режим чтения/записи Unicode UTF-16 Big Endian (без BOM).
$FO_FULLFILE_DETECT (16384) = при открытии для чтения без BOM для определения кодировки (UTF-8 или UTF-16) используется весь файл. Без этого флага из соображений производительности проверяется только начало файла (до 64 КБ).
Путь к папке должен существовать заранее (кроме режима $FO_CREATEPATH – см. раздел Примечания).

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает дескриптор файла для последующих файловых функций.
Ошибка: Возвращает -1 при ошибке.

Примечания

Дескриптор файла необходимо закрыть функцией FileClose().

Открытие файла может завершиться неудачей из-за прав доступа или атрибутов файла.

По умолчанию при записи текста используется UTF-8 (без BOM) – для изменения этого поведения задействуйте флаги режима Unicode. При чтении без явного флага Unicode содержимое файла анализируется и делается предположение о кодировке: UTF-8, UTF-16 или ANSI. Если открываемый файл содержит BOM, он будет учтён независимо от переданных флагов Unicode.

Открытие файла в режиме записи создаёт файл, если он не существует. Каталоги не создаются, если не использован соответствующий флаг.

При одновременном чтении и записи через один дескриптор для обновления текущей позиции в файле необходимо использовать FileSetPos().

Файл можно читать как двоичные данные (байты), открыв его через FileOpen() с двоичным флагом.

Подробное описание см. в разделе "Поддержка Unicode".

См. также

Пример

Пример 1

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, "This is an example of using FileOpen.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Открыть файл на чтение и сохранить дескриптор в переменную.
    Local $hFileOpen = FileOpen($sFilePath, $FO_READ)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred when reading the file.")
        Return False
    EndIf

    ; Прочитать содержимое файла через дескриптор, возвращённый FileOpen.
    Local $sFileRead = FileRead($hFileOpen)

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Показать содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & $sFileRead)

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example

Пример 2

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Открыть файл для чтения/записи.
    Local $hFileOpen = FileOpen($sFilePath, $FO_READ + $FO_OVERWRITE)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred when reading/writing the file.")
        Return False
    EndIf

    ; Записать данные.
    FileWrite($hFileOpen, "This is some data to show that the handle was open with read/write access." & @CRLF)

    ; Получить текущую позицию в файле.
    Local $iFilePos = FileGetPos($hFileOpen)

    ; Теперь перевести позицию в начало.
    FileSetPos($hFileOpen, 0, $FILE_BEGIN)

    ; Показать содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", FileRead($hFileOpen))

    ; Теперь восстановим предыдущую позицию.
    FileSetPos($hFileOpen, 0, $iFilePos)

    ; Записать дополнительные данные.
    FileWrite($hFileOpen, "This is some additional data.")

    ; Восстановить предыдущую позицию.
    FileSetPos($hFileOpen, 0, $FILE_BEGIN)

    ; Показать содержимое файла.
    MsgBox($MB_SYSTEMMODAL, "", FileRead($hFileOpen))

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Удалить временный файл.
    FileDelete($sFilePath)

    Return True
EndFunc   ;==>Example
