Справочник функций: Map

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
MapAppend Добавляет элемент в переменную типа Map.
MapExists Определяет, существует ли ключ в Map.
MapKeys Возвращает массив ключей Map.
MapRemove Удаляет ключ и связанное с ним значение из Map.
