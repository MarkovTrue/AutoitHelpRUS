Справочник функций

SoundSetWaveVolume

Устанавливает громкость системного звукового канала Wave в процентах.

SoundSetWaveVolume ( percent )

Параметры

percent Процент громкости в диапазоне от 0 до 100.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0; @error устанавливается в 1, если значение percent недопустимо.

Примечания

Функция управляет громкостью канала Wave, а не основной громкостью. Значение 0 не переводит устройство в режим отключения звука.

В Windows Vista системная громкость Wave отсутствует. Функция изменяет громкость Wave только для данного скрипта и не влияет на другие программы.

См. также

SoundPlay

Пример

; Установить громкость 50%.

SoundSetWaveVolume(50)
