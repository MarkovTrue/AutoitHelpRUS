GUISetIcon

Устанавливает значок для окна GUI.

GUISetIcon ( iconfile [, iconID [, winhandle]] )

Параметры

iconfile Файл, из которого берётся значок для отображения в строке заголовка.
iconID [необязательный] Идентификатор значка в файле. По умолчанию -1.
winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Положительное число ссылается на строковое имя значка.
Отрицательное число задаёт поведение "индекс начиная с 1".

Из некоторых DLL значки можно извлечь только с отрицательными числами.

См. также

GUICreate

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Поддержка работы в X64
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ; Получить следующий ico-файл. Он находится в папке include в каталоге установки AutoIt.
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\icons\au3.ico"

    GUICreate("My GUI new icon") ; создаст диалоговое окно, отображаемое по центру

    GUISetIcon($sFile) ; изменит значок

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
