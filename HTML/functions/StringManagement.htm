Справочник функций: String

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
StringAddCR Добавляет символ возврата каретки (Chr(13)) перед каждым символом перевода строки (Chr(10)) в строке.
StringCompare Сравнивает две строки с учётом заданных параметров.
StringFormat Возвращает форматированную строку (аналогично функции sprintf() в языке C).
StringFromASCIIArray Преобразует массив ASCII-кодов в строку.
StringInStr Проверяет, содержит ли строка заданную подстроку.
StringIsAlNum Проверяет, содержит ли строка только буквенно-цифровые символы.
StringIsAlpha Проверяет, содержит ли строка только буквенные символы.
StringIsASCII Проверяет, содержит ли строка только символы ASCII в диапазоне 0x00–0x7F (0–127).
StringIsDigit Проверяет, содержит ли строка только цифровые символы (0–9).
StringIsFloat Проверяет, является ли строка числом с плавающей точкой.
StringIsInt Проверяет, является ли строка целым числом.
StringIsLower Проверяет, содержит ли строка только строчные символы.
StringIsSpace Проверяет, содержит ли строка только пробельные символы.
StringIsUpper Проверяет, содержит ли строка только прописные символы.
StringIsXDigit Проверяет, содержит ли строка только шестнадцатеричные символы (0–9, A–F).
StringLeft Возвращает заданное количество символов с левого края строки.
StringLen Возвращает количество символов в строке.
StringLower Преобразует строку в нижний регистр.
StringMid Извлекает заданное количество символов из строки.
StringRegExp Проверяет, соответствует ли строка заданному шаблону регулярного выражения.
StringRegExpReplace Заменяет текст в строке на основе регулярных выражений.
StringReplace Заменяет подстроки в строке.
StringReverse Переворачивает содержимое указанной строки.
StringRight Возвращает заданное количество символов с правого края строки.
StringSplit Разбивает строку на подстроки по заданным разделителям.
StringStripCR Удаляет все символы возврата каретки ( Chr(13) ) из строки.
StringStripWS Удаляет пробельные символы в строке.
StringToASCIIArray Преобразует строку в массив, содержащий ASCII-код каждого символа.
StringTrimLeft Удаляет заданное количество символов с левого края строки.
StringTrimRight Удаляет заданное количество символов с правого края строки.
StringUpper Преобразует строку в верхний регистр.
