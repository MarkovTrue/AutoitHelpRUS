Справочник функций

WinGetCaretPos

Возвращает координаты курсора (каретки) в окне переднего плана.

WinGetCaretPos ( )

Возвращаемое значение

Успех: Возвращает массив из 2 элементов со следующей информацией:
$aArray[0] = координата X
$aArray[1] = координата Y
Ошибка: Устанавливает @error ≠ 0.

Примечания

WinGetCaretPos() может возвращать неточные значения для приложений MDI (Multiple Document Interface), если используется абсолютный режим CaretCoordMode. Обходной путь – в примере. Примечание: некоторые приложения всегда возвращают статические координаты независимо от положения каретки!

См. также

CaretCoordMode (Option)

Пример

Local $aCaretPos = WinGetCaretPos()
If Not @error Then
    ToolTip("First Method", $aCaretPos[0], $aCaretPos[1])
EndIf
Sleep(2000)

$aCaretPos = _WinGetCaretPos()
If Not @error Then
    ToolTip("Second Method", $aCaretPos[0], $aCaretPos[1])
EndIf
Sleep(2000)

; Более надёжный способ получить координаты каретки в MDI-редакторах.
Func _WinGetCaretPos()
    Local $iXAdjust = 5
    Local $iYAdjust = 40

    Local $iOpt = Opt("CaretCoordMode", 0) ; Устанавливаем "CaretCoordMode" в относительный режим и сохраняем предыдущее значение.
    Local $aGetCaretPos = WinGetCaretPos() ; Получаем относительные координаты каретки.
    Local $aGetPos = WinGetPos("[ACTIVE]") ; Получить позицию, высоту и ширину активного окна.
    Local $sControl = ControlGetFocus("[ACTIVE]") ; Получаем имя элемента управления, имеющего фокус клавиатуры.
    Local $aControlPos = ControlGetPos("[ACTIVE]", "", $sControl) ; Получаем позицию и размер элемента управления.
    $iOpt = Opt("CaretCoordMode", $iOpt) ; Восстанавливаем "CaretCoordMode" в предыдущее значение.

    Local $aReturn[2] = [0, 0] ; Создаём массив для хранения координат x, y.
    If IsArray($aGetCaretPos) And IsArray($aGetPos) And IsArray($aControlPos) Then
        $aReturn[0] = $aGetCaretPos[0] + $aGetPos[0] + $aControlPos[0] + $iXAdjust
        $aReturn[1] = $aGetCaretPos[1] + $aGetPos[1] + $aControlPos[1] + $iYAdjust
        Return $aReturn ; Возвращаем массив.
    Else
        Return SetError(1, 0, $aReturn) ; Возвращаем массив и устанавливаем значение @error = 1.
    EndIf
EndFunc   ;==>_WinGetCaretPos
