Справочник функций

FileGetVersion

Возвращает сведения о версии, хранящиеся в файле.

FileGetVersion ( "filename" [, "stringname"] )

Параметры

filename Имя файла.
stringname [необязательный] имя строкового поля, извлекаемого из блока версии файла:
$FV_COMMENTS ("Comments") – комментарии
$FV_COMPANYNAME ("CompanyName") – название компании
$FV_FILEDESCRIPTION ("FileDescription") – описание файла
$FV_FILEVERSION ("FileVersion") – версия файла
$FV_INTERNALNAME ("InternalName") – внутреннее имя
$FV_LEGALCOPYRIGHT ("LegalCopyright") – авторские права
$FV_LEGALTRADEMARKS ("LegalTrademarks") – торговые марки
$FV_ORIGINALFILENAME ("OriginalFilename") – исходное имя файла
$FV_PRODUCTNAME ("ProductName") – название продукта
$FV_PRODUCTVERSION ("ProductVersion") – версия продукта
$FV_PRIVATEBUILD ("PrivateBuild") – частная сборка
$FV_SPECIALBUILD ("SpecialBuild") – специальная сборка

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает номер версии в формате "#.#.#.#" – либо содержимое указанного строкового поля.
Ошибка: "0.0.0.0" при отсутствии информации о версии (или иной ошибке), либо "" если строковое поле не заполнено – флаг @error устанавливается в 1.

Примечания

Специальное значение "DefaultLangCodepage" позволяет получить язык и кодовую страницу по умолчанию.

Язык и кодовую страницу можно использовать при необходимости для уточнения параметра stringname, например: "080904b0\Comments" (см. MSDN StringFileInfo в описании функции VerQueryValue).

Для получения номера версии скомпилированного скрипта AutoIt рекомендуется использовать макрос @AutoItVersion.

См. также

FileGetSize, FileGetTime

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получить версию исполняемого файла AutoIt.
    Local $sFileVersion = FileGetVersion(@AutoItExe)

    ; Показать версию файла. Должна совпадать с @AutoItVersion.
    MsgBox($MB_SYSTEMMODAL, "", $sFileVersion)
EndFunc   ;==>Example
