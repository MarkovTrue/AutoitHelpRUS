Обновление элементов GUI

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

See Gui Constants include files if you need to use the related controls Constants .

 

Функция Описание
GUICtrlRegisterListViewSort Регистрирует пользовательскую функцию в качестве обратного вызова для внутренней сортировки ListView.
GUICtrlSetBkColor Устанавливает цвет фона элемента управления.
GUICtrlSetColor Устанавливает цвет текста элемента управления.
GUICtrlSetCursor Устанавливает значок курсора мыши для конкретного элемента управления.
GUICtrlSetData Изменяет данные элемента управления.
GUICtrlSetDefBkColor Устанавливает цвет фона по умолчанию для всех элементов управления окна GUI.
GUICtrlSetDefColor Устанавливает цвет текста по умолчанию для всех элементов управления окна GUI.
GUICtrlSetFont Устанавливает шрифт для элемента управления.
GUICtrlSetGraphic Изменяет данные графического элемента управления.
GUICtrlSetImage Устанавливает растровое изображение или значок для элемента управления.
GUICtrlSetLimit Ограничивает количество символов или пикселей для элемента управления.
GUICtrlSetOnEvent Определяет пользовательскую функцию, вызываемую при нажатии на элемент управления.
GUICtrlSetPos Изменяет положение элемента управления в окне GUI.
GUICtrlSetResizing Определяет метод изменения размеров элемента управления.
GUICtrlSetState Изменяет состояние элемента управления.
GUICtrlSetStyle Изменяет стиль элемента управления.
GUICtrlSetTip Устанавливает текст всплывающей подсказки для элемента управления.
