Справочник функций

FileOpenDialog

Открывает диалог выбора файла.

FileOpenDialog ( "title", "init dir", "filter" [, options = 0 [, "default name" [, hwnd]]] )

Параметры

title Текст заголовка диалогового окна.
init dir Начальный каталог, выбранный в дереве файлов диалога.
filter Фильтр типов файлов: одиночный, например "All (*.*)" или "Text files (*.txt)", либо несколько групп фильтров, разделённых символом "|", например "All (*.*)|Text files (*.txt)" (см. раздел Примечания).
options [необязательный] параметры диалога; для одновременного использования нескольких параметров объедините нужные значения через BitOR.
$FD_FILEMUSTEXIST (1) = файл должен существовать (если пользователь вводит имя вручную)
$FD_PATHMUSTEXIST (2) = путь должен существовать (если пользователь вводит путь, заканчивающийся обратным слешем)
$FD_MULTISELECT (4) = разрешить множественный выбор
$FD_PROMPTCREATENEW (8) = предложить создать новый файл (если не существует)

Константы определены в FileConstants.au3.
default name [необязательный] Предлагаемое имя файла. По умолчанию – пустая строка ("").
hwnd [необязательный] Дескриптор окна, используемого в качестве родительского для диалога.

Возвращаемое значение

Успех: Возвращает полный путь выбранного файла (или файлов). При множественном выборе – строку вида "Каталог|файл1|файл2|...".
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – не удалось выбрать файл.
2 – неверный фильтр файлов.

Примечания

Несколько типов файлов внутри одной группы фильтра разделяются точкой с запятой – как показано в примере.
Несколько групп фильтров разделяются символом "|".

Если указан параметр default name, параметр options также должен быть задан. Если параметры не нужны, передайте 0.

Специальные папки Windows (например, "Мои документы") можно задать в качестве корневой, используя соответствующий CLSID из Приложения.

При успешном завершении значение @WorkingDir обновляется.

См. также

FileSaveDialog, FileSelectFolder, StringSplit

Пример

Пример 1

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать локальную константу с текстом для отображения в FileOpenDialog.
    Local Const $sMessage = "Hold down Ctrl or Shift to choose multiple files."

    ; Открываем диалог выбора одного или нескольких файлов.
    Local $sFileOpenDialog = FileOpenDialog($sMessage, @WindowsDir & "\", "Images (*.jpg;*.bmp)|Videos (*.avi;*.mpg)", BitOR($FD_FILEMUSTEXIST, $FD_MULTISELECT))
    If @error Then
        ; Показать сообщение об ошибке.
        MsgBox($MB_SYSTEMMODAL, "", "No file(s) were selected.")

        ; Вернуть рабочий каталог (@WorkingDir) в папку скрипта, так как FileOpenDialog устанавливает его в последнюю открытую папку.
        FileChangeDir(@ScriptDir)
    Else
        ; Вернуть рабочий каталог (@WorkingDir) в папку скрипта, так как FileOpenDialog устанавливает его в последнюю открытую папку.
        FileChangeDir(@ScriptDir)

        ; Заменить все "|" на @CRLF в строке, возвращённой FileOpenDialog.
        $sFileOpenDialog = StringReplace($sFileOpenDialog, "|", @CRLF)

        ; Выводим список выбранных файлов.
        MsgBox($MB_SYSTEMMODAL, "", "You chose the following files:" & @CRLF & $sFileOpenDialog)
    EndIf
EndFunc   ;==>Example

Пример 2

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать локальную константу с текстом для отображения в FileOpenDialog.
    Local Const $sMessage = "Select a single file of any type."

    ; Показать диалог открытия файла.
    Local $sFileOpenDialog = FileOpenDialog($sMessage, @WindowsDir & "\", "All (*.*)", $FD_FILEMUSTEXIST)
    If @error Then
        ; Показать сообщение об ошибке.
        MsgBox($MB_SYSTEMMODAL, "", "No file was selected.")

        ; Вернуть рабочий каталог (@WorkingDir) в папку скрипта, так как FileOpenDialog устанавливает его в последнюю открытую папку.
        FileChangeDir(@ScriptDir)
    Else
        ; Вернуть рабочий каталог (@WorkingDir) в папку скрипта, так как FileOpenDialog устанавливает его в последнюю открытую папку.
        FileChangeDir(@ScriptDir)

        ; Заменить все "|" на @CRLF в строке, возвращённой FileOpenDialog.
        $sFileOpenDialog = StringReplace($sFileOpenDialog, "|", @CRLF)

        ; Выводим выбранный файл.
        MsgBox($MB_SYSTEMMODAL, "", "You chose the following file:" & @CRLF & $sFileOpenDialog)
    EndIf
EndFunc   ;==>Example
