WinMenuSelectItem

Выбирает пункт меню в окне.

WinMenuSelectItem ( "title", "text", "item" [, "item" [, "item" [, "item" [, "item" [, "item" [, "item"]]]]]] )

Параметры

title Заголовок / дескриптор / класс окна, в котором нужно выбрать пункт. См. Заголовки и текст окон (расширенно).
text Текст окна, в котором нужно выбрать пункт. См. Заголовки и текст окон (основы).
item Текст пункта меню.
item [необязательный] Текст пункта подменю
item [необязательный] Текст пункта подменю
item [необязательный] Текст пункта подменю
item [необязательный] Текст пункта подменю
item [необязательный] Текст пункта подменю
item [необязательный] Текст пункта подменю

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если меню не найдено.

Примечания

Учтите, что подчёркнутые пункты меню фактически содержат символ & для обозначения подчёркивания. Так, пункт меню File на самом деле требует текста "&File", а Convert – текста "Con&vert". Доступ к пунктам меню возможен на глубину до шести уровней, окно при этом может быть неактивным, свёрнутым и/или даже скрытым.

WinMenuSelectItem() работает только со стандартными меню. К сожалению, многие современные меню на деле написаны вручную или являются панелями инструментов, "притворяющимися" меню. Это верно для большинства приложений Microsoft.

См. также

ControlCommand, Send

Пример

; Выбираем "Файл", затем "Параметры страницы" в Блокноте.

WinMenuSelectItem("[CLASS:Notepad]", "", "&File", "Page Set&up...")
