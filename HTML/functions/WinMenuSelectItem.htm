Справочник функций

WinMenuSelectItem

Вызывает пункт меню окна.

WinMenuSelectItem ( "title", "text", "item" [, "item" [, "item" [, "item" [, "item" [, "item" [, "item"]]]]]] )

Параметры

title Заголовок/hWnd/класс окна, пункт меню которого нужно выбрать. См. Специальное определение заголовка.
text Текст окна, пункт меню которого нужно выбрать. См. Специальное определение текста.
item Text of Menu Item.
item [необязательный] Text of SubMenu item
item [необязательный] Text of SubMenu item
item [необязательный] Text of SubMenu item
item [необязательный] Text of SubMenu item
item [необязательный] Text of SubMenu item
item [необязательный] Text of SubMenu item

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если меню не найдено.

Примечания

Обратите внимание, что пункты меню с подчёркнутой буквой содержат символ & для обозначения подчёркивания. Например, пункт Файл требует текста "&Файл", а Кон&вертировать – "Кон&вертировать". Доступ к пунктам возможен на глубину до шести уровней; при этом окно может быть неактивным, свёрнутым и/или скрытым.

WinMenuSelectItem() работает только со стандартными меню. К сожалению, многие современные меню реализованы как пользовательские элементы управления или панели инструментов, имитирующие меню. Это характерно для большинства приложений Microsoft.

См. также

ControlCommand, Send

Пример

; Выбираем "Файл", затем "Параметры страницы" в Блокноте.

WinMenuSelectItem("[CLASS:Notepad]", "", "&File", "Page Set&up...")
