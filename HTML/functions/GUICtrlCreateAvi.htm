Справочник функций

GUICtrlCreateAvi

Создаёт элемент управления AVI-видео для GUI.

GUICtrlCreateAvi ( filename, subfileid, left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

filename Имя видеофайла. Поддерживаются только файлы .avi.
subfileid Идентификатор подфайла. Если файл содержит только одно видео, используйте 0.
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию – ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию – ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $ACS_TRANSPARENT
$ACS_TRANSPARENT используется всегда, если не задан $ACS_NONTRANSPARENT.
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для получения значения элемента управления см. GUICtrlRead().
Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

Чтобы запустить видео сразу после создания элемента, используйте стиль $ACS_AUTOPLAY.
Запустить и остановить анимацию можно, задав состояние 1 или 0 через GUICtrlSetState(). См. пример.

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_AVI, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <AVIConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKSIZE.

См. также

GUICoordMode (option), GUICtrlUpdate..., GUIGetMsg

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Создать GUI с различными элементами управления.
    Local $hGUI = GUICreate("Example", 300, 200)

    ; Создаём элемент управления анимацией.
    Local $idAvi_Animation = GUICtrlCreateAvi(@SystemDir & "\shell32.dll", 165, 15, 0, 300)
    Local $idBtn_Start = GUICtrlCreateButton("Start", 60, 150, 85, 25)
    Local $idBtn_Stop = GUICtrlCreateButton("Stop", 160, 150, 85, 25)

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idBtn_Start ; Запускаем анимацию.
                GUICtrlSetState($idAvi_Animation, $GUI_AVISTART)

            Case $idBtn_Stop ; Останавливаем анимацию.
                GUICtrlSetState($idAvi_Animation, $GUI_AVISTOP)

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI и все элементы управления.
    GUIDelete($hGUI)
EndFunc   ;==>Example
