Справочник функций

PixelSearch

Ищет пиксель заданного цвета в прямоугольной области.

PixelSearch ( left, top, right, bottom, color [, shade-variation = 0 [, step = 1 [, hwnd]]] )

Параметры

left Левая координата прямоугольника.
top Верхняя координата прямоугольника.
right Правая координата прямоугольника.
bottom Нижняя координата прямоугольника.
color Цвет искомого пикселя (десятичный или шестнадцатеричный).
shade-variation [необязательный] Число от 0 до 255 – допустимое отклонение компонент R, G, B. По умолчанию 0 (точное совпадение).
step [необязательный] Шаг перебора пикселей: значение больше 1 пропускает пиксели для ускорения. Значение 2 проверяет каждый второй пиксель. По умолчанию 1. Не рекомендуется использовать шаг больше 1.
hwnd [необязательный] Дескриптор окна. По умолчанию используется рабочий стол. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает двухэлементный массив координат найденного пикселя: Array[0] = x, Array[1] = y.
Ошибка: Устанавливает значение @error = 1, если цвет не найден.

Примечания

Направление поиска определяется так:
    Слева направо – left < right
    Справа налево – right < left
    Сверху вниз – top < bottom
    Снизу вверх – bottom < top
Смена направления поиска полезна, если искомый цвет чаще встречается в определённом квадранте области – поиск быстрее заканчивается при правильном начальном направлении.

Учтите: типичный экран 1024×768 содержит 786 432 пикселя. Несмотря на оптимизацию PixelSearch(), сужение области поиска ускоряет результат.

Если PixelCoordMode установлен в режим SCREEN (1) по умолчанию, параметр hwnd игнорируется. Он учитывается только при режиме WINDOW (0) или CLIENT (2).

См. также

PixelChecksum, PixelCoordMode (Option), PixelGetColor

bottom>Пример

#include <MsgBoxConstants.au3>

; Найти чисто красный пиксель в области (0,0)–(20,300).
Local $aCoord = PixelSearch(0, 0, 20, 300, 0xFF0000)
If Not @error Then
    MsgBox($MB_SYSTEMMODAL, "", "X and Y are: " & $aCoord[0] & "," & $aCoord[1])
EndIf

; Найти чисто красный пиксель или красный с отклонением до 10 оттенков.
$aCoord = PixelSearch(0, 0, 20, 300, 0xFF0000, 10)
If Not @error Then
    MsgBox($MB_SYSTEMMODAL, "", "X and Y are: " & $aCoord[0] & "," & $aCoord[1])
EndIf
