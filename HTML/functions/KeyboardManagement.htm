Работа с клавиатурой

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
HotKeySet Устанавливает горячую клавишу, которая вызывает пользовательскую функцию.
Send Отправляет эмулированные нажатия клавиш в активное окно.
SendKeepActive Пытается удерживать указанное окно активным во время выполнения Send().
