Справочник функций

GUICtrlCreateListViewItem

Создаёт элемент (строку) ListView.

GUICtrlCreateListViewItem ( "text", listviewID )

Параметры

text Текст подэлементов, разделённый символом Opt("GUIDataSeparatorChar").
listviewID controlID элемента управления ListView, в который добавляется элемент.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Эта функция создаёт отдельные элементы ListView, которые можно выбирать. Элементы работают как обычные элементы управления и могут обновляться через GUICtrlSetData().
Элементы удаляются, как любые другие элементы управления, через GUICtrlDelete().
Элементы ListView можно перетаскивать в элемент управления Edit или Input с состоянием $GUI_DROPACCEPTED.
См. GUICtrlCreateListView() о расширении столбцов.

Специальный флаг $GUI_BKCOLOR_LV_ALTERNATE позволяет задать чередующийся фон строк ListView.
Нечётные строки получат цвет, заданный через GUICtrlSetBkColor() для элемента управления ListView.
Чётные строки получат цвет, заданный через GUICtrlSetBkColor() для элемента управления ListViewItem.

Указанные выше константы определены в #include <GUIConstantsEx.au3>

См. также

GUICtrlCreateListView, GUICtrlDelete, GUICtrlRead, GUICtrlSetData, GUICtrlSetState, GUIDataSeparatorChar (Option), GUIGetMsg

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    GUICreate("listview items", 220, 250, 100, 200, -1, $WS_EX_ACCEPTFILES)
    GUISetBkColor(0x00E0FFFF) ; изменит цвет фона

    Local $idListview = GUICtrlCreateListView("col1 |col2|col3 ", 10, 10, 200, 150) ;,$LVS_SORTDESCENDING)
    Local $idButton = GUICtrlCreateButton("Value?", 75, 170, 70, 20)
    Local $idLVi_Item1 = GUICtrlCreateListViewItem("item2|col22|col23", $idListview)
    Local $idLVi_Item2 = GUICtrlCreateListViewItem("item1|col12|col13", $idListview)
    Local $idLVi_Item3 = GUICtrlCreateListViewItem("item3|col32|col33", $idListview)
    GUICtrlCreateInput("", 20, 200, 150)
    GUICtrlSetState(-1, $GUI_DROPACCEPTED) ; разрешить перетаскивание
    GUISetState(@SW_SHOW)
    GUICtrlSetData($idLVi_Item2, "ITEM1")
    GUICtrlSetData($idLVi_Item3, "||COL33")
    GUICtrlDelete($idLVi_Item1)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idButton
                MsgBox($MB_SYSTEMMODAL, "listview item", GUICtrlRead(GUICtrlRead($idListview)), 2)

            Case $idListview
                MsgBox($MB_SYSTEMMODAL, "listview", "clicked=" & GUICtrlGetState($idListview), 2)

        EndSwitch
    WEnd
EndFunc   ;==>Example
