Справочник функций

Int

Возвращает целочисленное представление выражения.

Int ( expression [, flag = 0] )

Параметры

expression Выражение для преобразования в целое число.
flag [необязательный] Определяет поведение функции. Возможные значения:
$NUMBER_AUTO (0) = результат автоматически выбранного типа целого числа (по умолчанию). См. раздел Примечания.
$NUMBER_32BIT (1) = строка интерпретируется как 32-битное целое число
$NUMBER_64BIT (2) = строка интерпретируется как 64-битное целое число

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Возвращает целое число.

Примечания

По умолчанию, если результат укладывается в диапазон 32-битного целого, возвращается 32-битное целое, иначе 64-битное. Оба со знаком.
Дробная часть отбрасывается, поэтому Int(1.999999) возвращает 1.
Int(0/0), если интересно, возвращает -9223372036854775807.
Функция вносит небольшие поправки в числа с плавающей точкой, учитывая их неточную природу. Например, выражение 0.7 + 0.2 + 0.1 даёт число с плавающей точкой, не вполне равное 1.0. Int() исправляет эту аномалию, однако в крайне редких случаях Int() может вернуть неожиданное значение (вероятность этого ниже, чем если бы Int() вообще не выполняла коррекцию).

См. также

Binary, Ceiling, Floor, HWnd, Mod, Number, Ptr, Round, String

Пример

#include <MsgBoxConstants.au3>

; Присвоить локальной переменной целочисленное представление числа 10.793.
Local $iInt = Int(10.793)

; Показать результат.
MsgBox($MB_SYSTEMMODAL, "", $iInt)
