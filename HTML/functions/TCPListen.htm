Справочник функций

TCPListen

Создаёт сокет, ожидающий входящих подключений.

TCPListen ( IPAddr, port [, MaxPendingConnection] )

Параметры

IPAddr IPv4-адрес в точечной записи, например "192.162.1.1".
port Порт, на котором будет установлено соединение через сокет.
MaxPendingConnection [необязательный] Максимальная длина очереди ожидающих соединений. По умолчанию устанавливается максимально разумное значение.

Возвращаемое значение

Успех: Возвращает идентификатор основного сокета.
Ошибка: Возвращает -1 или 0 и устанавливает @error ≠ 0.
@error: 1 – неверный IPAddr.
2 – неверный порт.
Код возврата Windows API WSAGetLastError (см. MSDN).

См. также

TCPAccept, TCPCloseSocket, TCPConnect, TCPSend, TCPShutdown, TCPStartup, TCPTimeout (Option)

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>

Example()

Func Example()
    Local $sMsgBoxTitle = "AutoItVersion = " & @AutoItVersion

    TCPStartup() ; Запустить TCP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальным переменным loopback-адрес IP и порт.
    Local $sIPAddress = "127.0.0.1" ; Этот IP-адрес работает только для тестирования на локальном компьютере.
    Local $iPort = 65432 ; Порт, используемый для подключения.

    ; Присвоить локальной переменной listening-сокет, привязать к указанным IP и порту, максимум 100 ожидающих подключений.
    Local $iListenSocket = TCPListen($sIPAddress, $iPort, 100)

    ; Примечание: слушать можно только на приватных IP, как здесь;
    ; или в диапазоне 192–223 (обычно 192.168.X.X; для тестирования на локальном IP используйте @IPAddress1 [потребуется другой компьютер]).
    ; Идентификатор сокета прослушивания используется только в функции TCPAccept.

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        ; Возможно, на этом IP-адресе и порту уже ведётся прослушивание (скрипт уже запущен?).
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), $sMsgBoxTitle, "Could not listen, Error code: " & @error & @CRLF & @CRLF & _WinAPI_GetErrorMessage(@error))
        Return False
    Else
        MsgBox($MB_SYSTEMMODAL, $sMsgBoxTitle, "Listen successful.", 3)
    EndIf

    ; Закрыть слушающий сокет, чтобы разрешить дальнейшие привязки.
    ; Пока не закрыто, другие программы не могут привязаться к тому же IP-адресу и порту.
    TCPCloseSocket($iListenSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit
