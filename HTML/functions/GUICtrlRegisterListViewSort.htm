Справочник функций

GUICtrlRegisterListViewSort

Регистрирует пользовательскую функцию в качестве обратного вызова для внутренней сортировки ListView.

GUICtrlRegisterListViewSort ( controlID, "function" )

Параметры

controlID controlID элемента управления ListView, для которого регистрируется пользовательская функция.
function Имя пользовательской функции, вызываемой при срабатывании обратного вызова сортировки.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

!!! Чтобы пользовательская функция работала, необходимо объявить её с не более чем 4 параметрами, иначе функция не будет вызвана !!!
Например:

Func MySortFunction($nListViewID, $LParam1, $LParam2, $nColumn)
...
EndFunc


Или:

Func MySortFunction($nListViewID, $LParam1, $LParam2)
...
EndFunc



При вызове пользовательской функции 4 параметра имеют следующие значения:

Позиция Параметр Описание
1 controlID controlID элемента управления ListView, для которого используется функция обратного вызова.
2 lParam1 Значение lParam первого элемента (по умолчанию – controlID элемента).
3 lParam2 Значение lParam второго элемента (по умолчанию – controlID элемента).
4 column Столбец, по которому был выполнен щелчок для сортировки (нумерация начинается с 0).

Для управления поведением обратного вызова сортировки функция должна возвращать следующие значения:

Возвращаемое значение Описание
-1 Первый элемент должен стоять перед вторым.
0 Без изменений.
1 Первый элемент должен стоять после второго.

См. также примеры сортировки с самостоятельно созданными элементами ListView.

См. также

GUICtrlCreateListView

Пример

Пример 1

; сортировка 3 столбцов по-разному

#include <GUIConstantsEx.au3>
#include <ListViewConstants.au3>

Global $g_iCurCol = -1
Global $g_iSortDir = 1
Global $g_bSet = False
Global $g_iCol = -1

Example()

Func Example()
    GUICreate("Test", 300, 200)

    Local $idLv = GUICtrlCreateListView("Column1|Col2|Col3", 10, 10, 280, 180)
    GUICtrlRegisterListViewSort(-1, "LVSort") ; Зарегистрировать функцию "LVSort" как обратный вызов сортировки

    GUICtrlCreateListViewItem("ABC|666|10.05.2004", $idLv)
    GUICtrlSetImage(-1, "shell32.dll", 7)
    GUICtrlCreateListViewItem("DEF|444|11.05.2005", $idLv)
    GUICtrlSetImage(-1, "shell32.dll", 12)
    GUICtrlCreateListViewItem("CDE|444|12.05.2004", $idLv)
    GUICtrlSetImage(-1, "shell32.dll", 3)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Цикл до выхода пользователя.
    While 1
        $idMsg = GUIGetMsg()
        Switch $idMsg
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idLv
                $g_bSet = False
                $g_iCurCol = $g_iCol
                GUICtrlSendMsg($idLv, $LVM_SETSELECTEDCOLUMN, GUICtrlGetState($idLv), 0)
        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example

; Наша функция обратного вызова сортировки
Func LVSort($hWnd, $nItem1, $nItem2, $iColumn)
    Local $sVal1, $sVal2, $iResult

    ; Переключить направление сортировки
    If $iColumn = $g_iCurCol Then
        If Not $g_bSet Then
            $g_iSortDir = $g_iSortDir * -1
            $g_bSet = True
        EndIf
    Else
        $g_iSortDir = 1
    EndIf
    $g_iCol = $iColumn

    $sVal1 = GetSubItemText($hWnd, $nItem1, $iColumn)
    $sVal2 = GetSubItemText($hWnd, $nItem2, $iColumn)

    ; Если это 3-й столбец (нумерация с 0), сравнивать даты
    If $iColumn = 2 Then
        $sVal1 = StringRight($sVal1, 4) & StringMid($sVal1, 4, 2) & StringLeft($sVal1, 2)
        $sVal2 = StringRight($sVal2, 4) & StringMid($sVal2, 4, 2) & StringLeft($sVal2, 2)
    EndIf

    $iResult = 0 ; Без изменения позиций item1 и item2

    If $sVal1 < $sVal2 Then
        $iResult = -1 ; Поместить item2 перед item1
    ElseIf $sVal1 > $sVal2 Then
        $iResult = 1 ; Поместить item2 после item1
    EndIf

    $iResult = $iResult * $g_iSortDir

    Return $iResult
EndFunc   ;==>LVSort

; Получить текст элемента listview в заданном столбце
Func GetSubItemText($idCtrl, $idItem, $iColumn)
    Local $tLvfi = DllStructCreate("uint;ptr;int;int[2];int")

    DllStructSetData($tLvfi, 1, $LVFI_PARAM)
    DllStructSetData($tLvfi, 3, $idItem)

    Local $tBuffer = DllStructCreate("char[260]")

    Local $nIndex = GUICtrlSendMsg($idCtrl, $LVM_FINDITEM, -1, DllStructGetPtr($tLvfi)) ;

    Local $tLvi = DllStructCreate("uint;int;int;uint;uint;ptr;int;int;int;int")

    DllStructSetData($tLvi, 1, $LVIF_TEXT)
    DllStructSetData($tLvi, 2, $nIndex)
    DllStructSetData($tLvi, 3, $iColumn)
    DllStructSetData($tLvi, 6, DllStructGetPtr($tBuffer))
    DllStructSetData($tLvi, 7, 260)

    GUICtrlSendMsg($idCtrl, $LVM_GETITEMA, 0, DllStructGetPtr($tLvi)) ;

    Local $sItemText = DllStructGetData($tBuffer, 1)

    Return $sItemText
EndFunc   ;==>GetSubItemText

; Создать и вставить элементы напрямую в ListView
Func MyGUICtrlCreateListViewItem($sText, $idCtrl, $iIndex)
    Local $tLvitem = DllStructCreate("uint;int;int;uint;uint;ptr;int;int;int;int;")
    Local $tText = DllStructCreate("char[260]")
    Local $aText = StringSplit($sText, "|")

    If $iIndex = -1 Then $iIndex = GUICtrlSendMsg($idCtrl, $LVM_GETITEMCOUNT, 0, 0)

    DllStructSetData($tText, 1, $aText[1]) ; Сохранить текст элемента в структуре

    DllStructSetData($tLvitem, 1, BitOR($LVIF_TEXT, $LVIF_PARAM))
    DllStructSetData($tLvitem, 2, $iIndex)
    DllStructSetData($tLvitem, 6, DllStructGetPtr($tText))
    ; Установить lParam структуры равным индексу строки – уникален в пределах ListView
    DllStructSetData($tLvitem, 9, $iIndex)

    $iIndex = GUICtrlSendMsg($idCtrl, $LVM_INSERTITEMA, 0, DllStructGetPtr($tLvitem))

    If $iIndex > -1 Then
        ; Теперь вставить оставшийся текст столбцов
        For $i = 2 To $aText[0]
            DllStructSetData($tText, 1, $aText[$i])
            DllStructSetData($tLvitem, 3, $i - 1) ; Сохранить индекс подэлемента

            GUICtrlSendMsg($idCtrl, $LVM_SETITEMTEXTA, $iIndex, DllStructGetPtr($tLvitem))
        Next
    EndIf

    ; Изменить ширину столбца по размеру текста элемента
    For $i = 0 To 2
        GUICtrlSendMsg($idCtrl, $LVM_SETCOLUMNWIDTH, $i, -1)
        GUICtrlSendMsg($idCtrl, $LVM_SETCOLUMNWIDTH, $i, -2)
    Next
EndFunc   ;==>MyGUICtrlCreateListViewItem
