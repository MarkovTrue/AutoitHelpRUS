Справочник функций

FileFindFirstFile

Создаёт дескриптор поиска, заданный путём и маской файлов.

FileFindFirstFile ( "filename" )

Параметры

filename Путь и имя файла. Допускаются символы подстановки * и ? – см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает дескриптор поиска для последующих вызовов FileFindNextFile.
Ошибка: Возвращает -1, если ничего не найдено. Флаг @error устанавливается в 1 только в том случае, если папка пуста.

Примечания

Строка поиска не чувствительна к регистру.
Символы подстановки: в общем случае * означает ноль или более символов, а ? означает ноль или один символ. Если строка поиска состоит только из символов подстановки (или равна "*.*"), смотрите пример ниже для понимания возвращаемого значения.

В части имени файла и в части расширения допускается использовать только по одному символу подстановки, например a*.b?.
При использовании трёхсимвольного расширения будет найдено любое расширение, начинающееся с этих трёх символов, например: "*.log" совпадёт с "test.log_1" (это не описано и в документации Microsoft).

По окончании поиска с помощью функций FileFind... необходимо вызвать FileClose() для освобождения дескриптора поиска.

Имена каталогов возвращаются в соответствии с заданными символами подстановки.

В силу особенностей используемого Windows API (FindFirstFile) данная функция при поиске совпадений проверяет как длинные, так и короткие имена файлов.
Если вы получаете неожиданные результаты, убедитесь, что совпадение не происходит по короткому имени файла.

См. также

FileClose, FileFindNextFile

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной search-дескриптор всех файлов текущей папки.
    Local $hSearch = FileFindFirstFile("*.*")

    ; Проверить успех поиска, при неудаче показать сообщение и вернуть False.
    If $hSearch = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "Error: No files/directories matched the search pattern.")
        Return False
    EndIf

    ; Присвоить локальной переменной пустую строку для накопления найденных имён файлов.
    Local $sFileName = "", $iResult = 0

    While 1
        $sFileName = FileFindNextFile($hSearch)
        ; Если файлов, соответствующих критерию поиска, больше нет.
        If @error Then ExitLoop

        ; Показать имя файла.
        $iResult = MsgBox(($MB_OKCANCEL + $MB_SYSTEMMODAL), "", "File: " & $sFileName)
        If $iResult <> $IDOK Then ExitLoop ; Если пользователь нажал кнопку отмены или закрытия.
    WEnd

    ; Закрыть search-дескриптор.
    FileClose($hSearch)
EndFunc   ;==>Example
