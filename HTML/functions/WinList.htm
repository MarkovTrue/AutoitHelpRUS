Справочник функций

WinList

Получает список окон.

WinList ( ["title" [, "text"]] )

Параметры

title [необязательный] Заголовок/hWnd/класс окон, список которых нужно получить. См. Специальное определение заголовка.
text [необязательный] Текст окон, список которых нужно получить. По умолчанию пустая строка. См. Специальное определение текста.

Возвращаемое значение

Возвращает массив заголовков и дескрипторов окон, соответствующих критериям.

Примечания

Если заголовок и текст не указаны, возвращаются все окна верхнего уровня.

Возвращаемый массив двумерный и имеет следующую структуру:
    $aArray[0][0] = количество возвращённых окон
    $aArray[1][0] = заголовок 1-го окна
    $aArray[1][1] = дескриптор 1-го окна (HWND)
    $aArray[2][0] = заголовок 2-го окна
    $aArray[2][1] = дескриптор 2-го окна (HWND)
    ...
    $aArray[n][0] = заголовок n-го окна
    $aArray[n][1] = дескриптор n-го окна (HWND)

См. также

WinGetHandle

Пример

Пример 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получаем список дескрипторов окон.
    Local $aList = WinList()

    ; Перебираем массив, выводя только видимые окна с заголовком.
    For $i = 1 To $aList[0][0]
        If $aList[$i][0] <> "" And BitAND(WinGetState($aList[$i][1]), 2) Then
            MsgBox($MB_SYSTEMMODAL, "", "Title: " & $aList[$i][0] & @CRLF & "Handle: " & $aList[$i][1])
        EndIf
    Next
EndFunc   ;==>Example

Пример 2

#include <Array.au3>

Example()

Func Example()
    ; Получаем список дескрипторов окон с помощью регулярного выражения. Выражение ищет заголовки, содержащие слово SciTE или Internet Explorer.
    Local $aWinList = WinList("[REGEXPTITLE:(?i)(.*SciTE.*|.*Internet Explorer.*)]")
    _ArrayDisplay($aWinList)
EndFunc   ;==>Example
