Справочник функций

ObjCreate

Создаёт ссылку на COM-объект по указанному имени класса.

ObjCreate ( "classname" [, "servername" [, "username" [, "password"]]] )

Параметры

classname Класс объекта в формате "appname.objecttype".
Может также быть строковым представлением CLSID.
servername [необязательный] Имя удалённого компьютера, на котором создаётся объект.
username [необязательный] Имя пользователя на удалённом компьютере.
Вводится в формате "компьютер\пользователь" или "домен\пользователь".
password [необязательный] Пароль пользователя на удалённом компьютере.

Возвращаемое значение

Успех: Возвращает объект.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Используйте ObjCreate(), если вам нужен новый экземпляр приложения.
Для подключения к существующему процессу используйте ObjGet().

Помните, что не все компьютеры имеют одинаковый набор объектов. Поэтому всегда проверяйте ошибки после вызова ObjCreate(). Используйте ObjEvent() для проверки ошибок.

Следующие требования применяются при доступе к объектам на удалённых компьютерах:


  • Пользователь, запустивший скрипт, должен иметь соответствующие права.

  • Объект на удалённом компьютере должен поддерживать DCOM (Distributed COM).

  • На удалённом компьютере должны работать службы Remote Registry Service и File and Printer sharing.



Подробнее об объектах см. в справочнике Obj/COM.

См. также

GUICtrlCreateObj, IsObj, ObjEvent, ObjGet, ObjName

Пример

Пример 1

#include <MsgBoxConstants.au3>

; Подсчёт количества открытых окон оболочки.

Local $oShell = ObjCreate("shell.application") ; Получить объект Windows Shell.
Local $oShellWindows = $oShell.Windows() ; Получить коллекцию открытых окон оболочки.

If IsObj($oShellWindows) Then
    Local $sString = "" ; Строка для вывода.

    For $oWnd In $oShellWindows ; Подсчитать все открытые окна оболочки.
        $sString &= $oWnd.LocationName & @CRLF
    Next

    MsgBox($MB_SYSTEMMODAL, "", "Shell Windows:" & @CRLF & "You have the following shell windows:" & @CRLF & @CRLF & $sString)
EndIf

Пример 2

#include <MsgBoxConstants.au3>

; Открыть MediaPlayer на УДАЛЁННОМ компьютере.

Local $oRemoteMedia = ObjCreate("MediaPlayer.MediaPlayer.1", "name-of-remote-computer")

If Not @error Then
    MsgBox($MB_SYSTEMMODAL, "Remote ObjCreate Test", "ObjCreate() of a remote Mediaplayer Object successful !")
    $oRemoteMedia.Open(@WindowsDir & "\media\tada.wav") ; Воспроизвести звук, если файл найден.
Else
    MsgBox($MB_SYSTEMMODAL, "Remote ObjCreate Test", "Failed to open remote Object. Error code: " & Hex(@error, 8))
EndIf
