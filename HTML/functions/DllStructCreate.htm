DllStructCreate

Создаёт структуру в стиле C/C++ для использования в DllCall.

DllStructCreate ( Struct [, Pointer] )

Параметры

Struct Строка, описывающая создаваемую структуру (см. раздел Примечания).
Pointer [необязательный] Если указан, структура не будет выделять память, а будет использовать предоставленный указатель.

Возвращаемое значение

Успех: Возвращает переменную для использования в вызовах DllStruct.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – переменная, переданная в DllStructCreate, не является строкой.
2 – в переданной строке обнаружен неизвестный тип данных.
3 – не удалось выделить память для структуры, или Pointer = 0.
4 – ошибка при выделении памяти для переданной строки.

Примечания

Каждый тип данных должен быть разделён точкой с запятой ';'.

Массивы создаются добавлением '[размер]' после типа данных: DllStructCreate("int;char[128]")

Имя элемента можно добавить аналогично объявлению в стиле C: DllStructCreate("int n;char buffer[128]").
Это имя можно использовать вместо номера элемента в других функциях DllStruct... Имя должно состоять из букв, цифр и символов подчёркивания.

Если набор типов данных определяется как "struct{}" в объявлении C, необходимо использовать "STRUCT; ...; ENDSTRUCT;".
Это нужно для соблюдения выравнивания внутри всей структуры. Если все типы данных находятся в определённой структуре, не требуется – применяется неявное выравнивание.

DllStructCreate("int;STRUCT;ptr;int;ENDSTRUCT;int")    ; структура занимает 32 байта в Windows 64-бит и 16 в Windows 32-бит
DllStructCreate("int;ptr;int;int")    ; структура занимает 24 байта в Windows 64-бит и 16 в Windows 32-бит

Для нестандартного выравнивания добавьте перед структурой ключевое слово align. Значение n по умолчанию равно 8. Допустимые значения: 1, 2, 4, 8 и 16. Выравнивание члена будет кратно n или кратно размеру члена – что меньше. Это эквивалентно опции #pragma pack компилятора Microsoft Visual C++.

DllStructCreate("short;int") ; структура 8 байт, "int" на смещении 4
DllStructCreate("align 2;short;int") ; структура 6 байт, "int" на смещении 2

DllStructCreate("byte;double") ; структура 16 байт, "double" на смещении 8
DllStructCreate("align 4;byte;double") ; структура 12 байт, "double" на смещении 4

Если требуется изменить выравнивание, "align" можно использовать перед первым элементом, для которого нужно изменение.
"align" или "align 8" восстанавливают выравнивание по умолчанию.

Для освобождения выделенной памяти установите возвращённую переменную в 0.

Применяются следующие правила совокупного выравнивания:
Выравнивание массива совпадает с выравниванием одного из его элементов.

Выравнивание начала структуры равно максимальному выравниванию любого из её членов.
Каждый член структуры размещается в соответствии с его выравниванием согласно таблице выше, что требует неявного внутреннего заполнения в зависимости от предыдущего члена.

Размер структуры кратен её выравниванию, что требует заполнения после последнего члена.

См. также

DllCall, DllStructGetData, DllStructGetPtr, DllStructGetSize, DllStructSetData, IsDllStruct

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    #cs Comments:
Создание следующей структуры (язык C):
struct {
int var1;
unsigned char var2;
unsigned int var3;
char var4[128];
};

Схема:
------------------------------------
\ int \ byte \ uint \ char \
\ var1 \ var2 \ var3 \ var4 \
------------------------------------
#ce Comments:

    ; Присвоить локальной константе определение структуры (внимательно читайте примечания DllStructCreate).
    Local Const $tagSTRUCT1 = "struct;int var1;byte var2;uint var3;char var4[128];endstruct"

    ; Примечание: переменная-тег объявлена как Constant – её значение не меняется при выполнении скрипта.

    ; Присвоить локальной переменной структуру.
    Local $tSTRUCT1 = DllStructCreate($tagSTRUCT1)

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error in DllStructCreate, Code: " & @error)
        Return False
    EndIf

    ; Задать данные элемента var1 (int) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var1", -1) ; Или 1 вместо "var1".

    ; Задать данные элемента var2 (byte) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, 2, 255) ; Или "var2" вместо 2.

    ; Задать данные элемента var3 (uint) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var3", -1) ; -1 (signed int) будет преобразовано в unsigned int.
    ; Или 3 вместо "var3".

    ; Задать данные элемента var4 (char) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var4", "Hello") ; Или 4 вместо "var4".

    ; Примечание: этот элемент может содержать до 128 символов.

    ; Изменить данные элемента var4 (char) в $tSTRUCT1 по индексу 1 массива char (индексы с 1).
    DllStructSetData($tSTRUCT1, "var4", Asc("h"), 1)

    ; Показать результаты.
    MsgBox($MB_SYSTEMMODAL, "", "Struct Size: " & DllStructGetSize($tSTRUCT1) & @CRLF & _
            "Struct pointer: " & DllStructGetPtr($tSTRUCT1) & @CRLF & _
            "Data:" & @CRLF & _
            DllStructGetData($tSTRUCT1, 1) & @CRLF & _ ; Или "var1" вместо 1.
            DllStructGetData($tSTRUCT1, "var2") & @CRLF & _ ; Или 2 вместо "var2".
            DllStructGetData($tSTRUCT1, 3) & @CRLF & _ ; Или "var3" вместо 3.
            DllStructGetData($tSTRUCT1, 4)) ; Или "var4" вместо 4.

    ; Освободить ресурсы, занятые структурой.
    $tSTRUCT1 = 0
EndFunc   ;==>Example
