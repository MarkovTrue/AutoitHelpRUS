DllStructCreate

Создаёт структуру в стиле C/C++ для использования в DllCall.

DllStructCreate ( Struct [, Pointer] )

Параметры

Struct Строка с описанием создаваемой структуры (см. раздел Примечания).
Pointer [необязательный] Если задан, структура не выделяет память, а использует переданный указатель.

Возвращаемое значение

Успех: Возвращает переменную для использования в вызовах DllStruct.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – переданная в DllStructCreate переменная не является строкой.
2 – в переданной строке встречается неизвестный тип данных.
3 – не удалось выделить память под структуру, или Pointer = 0.
4 – ошибка выделения памяти под переданную строку.

BOOL знаковое 32-битное (4 байта) целое

ALIGN граница в n байт, по которой выравнивается тип.

Примечания

Типы данных разделяются точкой с запятой ';'.

Массив задаётся добавлением '[размер]' после типа данных: DllStructCreate("int;char[128]")

Элементу можно дать имя, как в объявлении на языке C: DllStructCreate("int n;char buffer[128]").
Это имя можно использовать вместо номера элемента в других функциях DllStruct... Имя должно состоять из букв, цифр и знака подчёркивания.

Если набор типов оформлен как "struct{}" в объявлении C, нужно использовать "STRUCT; ...; ENDSTRUCT;".
Это нужно, чтобы соблюсти выравнивание внутри всей создаваемой структуры. Если же все типы данных входят в определяемую структуру, этого не требуется – выравнивание выполняется неявно.

DllStructCreate("int;STRUCT;ptr;int;ENDSTRUCT;int")    ; структура занимает 32 байта в 64-битной Windows и 16 в 32-битной
DllStructCreate("int;ptr;int;int")    ; структура занимает 24 байта в 64-битной Windows и 16 в 32-битной

Для другого выравнивания добавьте перед структурой ключевое слово align. Значение n по умолчанию равно 8. Допустимы значения 1, 2, 4, 8 и 16. Член выравнивается по границе, кратной n или размеру самого члена, смотря что меньше. Это соответствует опции #pragma pack компилятора Microsoft Visual C++.

DllStructCreate("short;int") ; структура занимает 8 байт, "int" расположен по смещению 4
DllStructCreate("align 2;short;int") ; структура занимает 6 байт, "int" расположен по смещению 2

DllStructCreate("byte;double") ; структура занимает 16 байт, "double" расположен по смещению 8
DllStructCreate("align 4;byte;double") ; структура занимает 12 байт, "double" расположен по смещению 4

Если выравнивание нужно изменить, "align" можно поставить перед первым элементом, для которого нужно изменение.
"align" или "align 8" возвращают выравнивание по умолчанию.

Чтобы освободить выделенную память, просто присвойте возвращённой переменной 0.

Действуют следующие правила выравнивания совокупных типов:
Выравнивание массива совпадает с выравниванием одного из его элементов.

Выравнивание начала структуры равно наибольшему выравниванию среди её членов.
Каждый член структуры размещается со своим выравниванием согласно таблице выше, что требует неявного внутреннего заполнения в зависимости от предыдущего члена.

Размер структуры кратен её выравниванию, поэтому после последнего члена тоже может добавляться заполнение.

См. также

DllCall, DllStructGetData, DllStructGetPtr, DllStructGetSize, DllStructSetData, IsDllStruct

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    #cs Примечания:
Создать следующую структуру (на C):
struct {
int var1;
unsigned char var2;
unsigned int var3;
char var4[128];
};

Схема:
------------------------------------
\ int \ byte \ uint \ char \
\ var1 \ var2 \ var3 \ var4 \
------------------------------------
#ce Примечания:

    ; Присвоить локальной константе определение структуры (внимательно читайте примечания DllStructCreate).
    Local Const $tagSTRUCT1 = "struct;int var1;byte var2;uint var3;char var4[128];endstruct"

    ; Примечание: переменная-тег объявлена как Constant – её значение не меняется при выполнении скрипта.

    ; Присвоить локальной переменной структуру.
    Local $tSTRUCT1 = DllStructCreate($tagSTRUCT1)

    ; Если произошла ошибка, показать код ошибки и вернуть False.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error in DllStructCreate, Code: " & @error)
        Return False
    EndIf

    ; Задать данные элемента var1 (int) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var1", -1) ; Или 1 вместо "var1".

    ; Задать данные элемента var2 (byte) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, 2, 255) ; Или "var2" вместо 2.

    ; Задать данные элемента var3 (uint) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var3", -1) ; -1 (Signed int) будет преобразовано в unsigned int.
    ; Или 3 вместо "var3".

    ; Задать данные элемента var4 (char) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var4", "Hello") ; Или 4 вместо "var4".

    ; Примечание: этот элемент может содержать до 128 символов.

    ; Изменить данные элемента var4 (char) в $tSTRUCT1 по индексу 1 массива char (индексы с 1).
    DllStructSetData($tSTRUCT1, "var4", Asc("h"), 1)

    ; Показать результаты.
    MsgBox($MB_SYSTEMMODAL, "", "Struct Size: " & DllStructGetSize($tSTRUCT1) & @CRLF & _
            "Struct pointer: " & DllStructGetPtr($tSTRUCT1) & @CRLF & _
            "Data:" & @CRLF & _
            DllStructGetData($tSTRUCT1, 1) & @CRLF & _ ; Или "var1" вместо 1.
            DllStructGetData($tSTRUCT1, "var2") & @CRLF & _ ; Или 2 вместо "var2".
            DllStructGetData($tSTRUCT1, 3) & @CRLF & _ ; Или "var3" вместо 3.
            DllStructGetData($tSTRUCT1, 4)) ; Или "var4" вместо 4.

    ; Освободить ресурсы, занятые структурой.
    $tSTRUCT1 = 0
EndFunc   ;==>Example
