Справочник функций

StringRight

Возвращает заданное количество символов с правого края строки.

StringRight ( "string", count )

Параметры

string Оцениваемая строка.
count Количество возвращаемых символов.

Возвращаемое значение

Возвращает строку, содержащую count крайних правых символов исходной строки.

Примечания

Если count превышает длину строки, возвращается вся строка.
Если count отрицательный, возвращается пустая строка.

См. также

StringCompare, StringInStr, StringLeft, StringLen, StringLower, StringMid, StringReplace, StringSplit, StringTrimLeft, StringTrimRight, StringUpper

Пример

#include <MsgBoxConstants.au3>

Local $sString = StringRight("This is a sentence with whitespace.", 5) ; Получить 5 символов с правого края строки.
MsgBox($MB_SYSTEMMODAL, "", "The 5 rightmost characters are: " & $sString)
