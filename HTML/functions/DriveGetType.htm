DriveGetType

Возвращает тип диска.

DriveGetType ( "path" [, operation = 1] )

Параметры

path Путь к диску, с которого получается информация.
operation [необязательный] Операция получения типа диска.
   $DT_DRIVETYPE (1) = тип диска (по умолчанию)
   $DT_SSDSTATUS (2) = статус SSD диска
   $DT_BUSTYPE (3) = тип шины диска

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает значение согласно примечаниям.
Ошибка: Устанавливает значение @error = 1, если указан неверный путь или операция завершилась неудачей. Возвращает "".

Примечания

Если параметр режима равен $DT_DRIVETYPE (1), возвращает тип диска. Путь должен быть строковым именем тома, например "C:\". Возможные значения:
"Unknown", "Removable", "Fixed", "Network", "CDROM", "RAMDisk"

Если параметр режима равен $DT_SSDSTATUS (2), возвращает статус SSD диска. Путь может быть строковым именем тома или целочисленным индексом диска. Возможные значения:
"SSD", "" (пусто)

Если параметр режима равен $DT_BUSTYPE (3), возвращает тип шины диска. Путь может быть строковым именем тома или целочисленным индексом диска. Возможные значения:
"Unknown", "SCSI", "ATAPI", "ATA", "1394", "SSA", "Fibre", "USB", "RAID", "iSCSI", "SAS", "SATA", "SD", "MMC", "Virtual", "FileBackedVirtual"

См. также

Пример

Пример 1

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

; Проверить тип диска для C:\
Local $sInfo = DriveGetType("C:\")
MsgBox($MB_SYSTEMMODAL, "", "Drive Type: " & $sInfo)

; Проверить статус SSD для C:\
$sInfo = DriveGetType("C:\", $DT_SSDSTATUS)
MsgBox($MB_SYSTEMMODAL, "", "Drive SSD: " & $sInfo)

; Проверить статус SSD для диска 0
$sInfo = DriveGetType(0, $DT_SSDSTATUS)
MsgBox($MB_SYSTEMMODAL, "", "Drive SSD: " & $sInfo)

; Проверить тип шины для диска 0
$sInfo = DriveGetType(0, $DT_BUSTYPE)
MsgBox($MB_SYSTEMMODAL, "", "Drive Bus: " & $sInfo)

Пример 2

; == Пример 2, с номером диска и номером раздела

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Local $aDrive = DriveGetDrive("ALL")
Local $sOutput = "Nb Drives = " & $aDrive[0] & @CRLF & @CRLF, $sType, $sSsdStatus, $sBusType, $nDrive = 0

For $i = 1 To $aDrive[0]
    $sBusType = DriveGetType($aDrive[$i], $DT_BUSTYPE)
    $sSsdStatus = DriveGetType($aDrive[$i], $DT_SSDSTATUS)
    $sType = DriveGetType($aDrive[$i] & "\")
    $sOutput &= "Drive " & $aDrive[$i] & @TAB & "Disk #" & _WinAPI_GetDriveNumber($aDrive[$i])[1] & @TAB & _
            "Partition #" & _WinAPI_GetDriveNumber($aDrive[$i])[2] & @TAB & "SSD Status = " & $sSsdStatus & @TAB & "BusType = " & $sBusType & @CRLF
Next

MsgBox($MB_SYSTEMMODAL, "DriveGetType (v" & @AutoItVersion & ")", $sOutput)
