GUISetAccelerators

Задаёт таблицу горячих клавиш (акселераторов) для окна GUI.

GUISetAccelerators ( accelerators [, winhandle] )

Параметры

accelerators Двумерный массив, содержащий таблицу акселераторов (см. раздел Примечания).
winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Акселераторы похожи на HotKey, но имеют 2 важных отличия:

    - 1. Они активны только тогда, когда указанное в функции окно GUI активно, то есть, в отличие от HotKey, акселераторы не влияют на другие запущенные приложения.

    - 2. Они не могут напрямую вызвать функцию – они активируют связанный элемент управления, который затем вызывает функцию через GUIGetMsg() или GUICtrlSetOnEvent(). Если подходящего элемента управления в GUI нет, можно создать Dummy-элемент через GUICtrlCreateDummy.

Массив, передаваемый в эту функцию, содержит горячие клавиши и controlID связанных элементов управления. Массив должен быть объявлен как Local/Global $aArray[n][2], где n – количество задаваемых акселераторов:

    $aArray[0][0] = 1-я горячая клавиша (в формате HotKeySet())
    $aArray[0][1] = controlID для 1-го акселератора, возвращённый GUICtrlCreate...
    $aArray[1][0] = 2-я горячая клавиша
    $aArray[1][1] = controlID для 2-го акселератора
    ...
    $aArray[n][0] = n-я горячая клавиша
    $aArray[n][1] = controlID для n-го акселератора


Передача в функцию параметра, не являющегося массивом, отменит все акселераторы для указанного winhandle.

См. также

GUICreate, HotKeySet

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("Custom MsgBox", 225, 80)

    GUICtrlCreateLabel("Please select a button.", 10, 10)
    Local $idButton_Yes = GUICtrlCreateButton("Yes", 10, 50, 65, 25)
    Local $idButton_No = GUICtrlCreateButton("No", 80, 50, 65, 25)
    Local $idButton_Exit = GUICtrlCreateButton("Exit", 150, 50, 65, 25)

    ; Задать акселераторы GUI для controlID кнопок: Ctrl + y и Ctrl + n
    Local $aAccelKeys[2][2] = [["^y", $idButton_Yes], ["^n", $idButton_No]]
    GUISetAccelerators($aAccelKeys)

    GUISetState(@SW_SHOW) ; Отобразить GUI.

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                MsgBox($MB_SYSTEMMODAL, "You selected", "Close")
                ExitLoop

            Case $idButton_Yes
                MsgBox($MB_SYSTEMMODAL, "You selected", "Yes") ; Выводит сообщение, если нажата кнопка или сочетание Ctrl + y.

            Case $idButton_No
                MsgBox($MB_SYSTEMMODAL, "You selected", "No") ; Выводит сообщение, если нажата кнопка или сочетание Ctrl + n.

            Case $idButton_Exit
                MsgBox($MB_SYSTEMMODAL, "You selected", "Exit")
                ExitLoop

        EndSwitch
    WEnd
    GUIDelete() ; Удалить GUI.
EndFunc   ;==>Example
