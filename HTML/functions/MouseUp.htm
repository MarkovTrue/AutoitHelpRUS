Справочник функций

MouseUp

Выполняет событие отпускания кнопки мыши в текущей позиции курсора.

MouseUp ( "button" )

Параметры

button Кнопка для отпускания:
$MOUSE_CLICK_RIGHT ("right")
$MOUSE_CLICK_MIDDLE ("middle")
$MOUSE_CLICK_MAIN ("main")
$MOUSE_CLICK_MENU ("menu")
$MOUSE_CLICK_PRIMARY ("primary")
$MOUSE_CLICK_SECONDARY ("secondary")

Константы определены в "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 – кнопка отсутствует в списке.

Примечания

См. раздел Примечания к MouseClick.
Ответственность разработчика: каждому вызову MouseDown() должен соответствовать вызов MouseUp().

См. также

MouseClick, MouseClickDrag, MouseCoordMode (Option), MouseDown, MouseGetPos, MouseMove

Пример

#include <AutoItConstants.au3>

MouseDown($MOUSE_CLICK_LEFT) ; Установить нажатое состояние левой кнопки мыши.
Sleep(100)
MouseUp($MOUSE_CLICK_LEFT) ; Установить отпущенное состояние левой кнопки мыши.
