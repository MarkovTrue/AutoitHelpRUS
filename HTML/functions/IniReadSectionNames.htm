Справочник функций

IniReadSectionNames

Читает все разделы .ini-файла стандартного формата.

IniReadSectionNames ( "filename" )

Параметры

filename Имя .ini-файла.

Возвращаемое значение

Успех: Возвращает массив всех имён разделов .ini-файла.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Стандартный .ini-файл выглядит следующим образом:
[SectionName]
Key=Value

Количество возвращённых элементов содержится в $aArray[0]. При возникновении @error массив не возвращается.

См. также

IniDelete, IniRead, IniReadSection, IniRenameSection, IniWrite, IniWriteSection

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать структуру INI-секции в виде строки.
    Local $sSection = "Title=AutoIt" & @LF & "Version=" & @AutoItVersion & @LF & "OS=" & @OSVersion

    ; Записать строку в разделы 'General', 'Version' и 'Other'.
    IniWriteSection($sFilePath, "General", $sSection)
    IniWriteSection($sFilePath, "Version", $sSection)
    IniWriteSection($sFilePath, "Other", $sSection)

    ; Прочитать имена секций INI. Возвращается одномерный массив.
    Local $aArray = IniReadSectionNames($sFilePath)

    ; Проверить, произошла ли ошибка.
    If Not @error Then
        ; Пройти по массиву, показывая имена секций.
        For $i = 1 To $aArray[0]
            MsgBox($MB_SYSTEMMODAL, "", "Section: " & $aArray[$i])
        Next
    EndIf

    ; Удалить INI-файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
