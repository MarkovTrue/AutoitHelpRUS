Справочник функций

ControlGetPos

Возвращает позицию и размер элемента управления относительно его окна.

ControlGetPos ( "title", "text", controlID )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.

Возвращаемое значение

Успех: Возвращает массив, содержащий размер и позицию элемента управления относительно клиентской области окна:
    $aArray[0] = X-позиция
    $aArray[1] = Y-позиция
    $aArray[2] = ширина
    $aArray[3] = высота
Ошибка: Устанавливает значение @error = 1.

Примечания

Заголовок/текст относится к родительскому окну, поэтому будьте осторожны с "", которое ссылается на активное окно, которое может не содержать элемент с указанным controlID.

См. также

ControlCommand, ControlMove

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Получить позицию (x, y) и размер (ширина, высота) поля редактирования в Блокноте. Дескриптор, возвращённый WinWait, используется в качестве параметра "title" функции ControlGetPos.
    Local $aPos = ControlGetPos($hWnd, "", ControlGetFocus($hWnd))

    ; Отобразить позицию и размер поля редактирования.
    MsgBox($MB_SYSTEMMODAL, "", "Position: " & $aPos[0] & ", " & $aPos[1] & @CRLF & "Size: " & $aPos[2] & ", " & $aPos[3])

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
