Справочник функций

IniWrite

Записывает значение в .ini-файл стандартного формата.

IniWrite ( "filename", "section", "key", "value" )

Параметры

filename Имя .ini-файла.
section Имя секции в .ini-файле.
key Имя ключа в .ini-файле.
value Значение для записи/изменения.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если файл доступен только для чтения.

Примечания

Стандартный .ini-файл выглядит следующим образом:
[SectionName]
Key=Value

Если файл не существует, он создаётся при условии, что каталог существует. Ключи и разделы добавляются в конец без какой-либо сортировки.

Чтобы заключить значение в кавычки, нужно удвоить их: первая пара кавычек отбрасывается. Например, ""This is a test"" при чтении через IniRead() вернёт "This is a test".

Пробелы в начале и конце строки обрезаются. Для сохранения пробелов строку нужно взять в кавычки. Например, "    this is a test" сохранит пробелы, но кавычки, как сказано выше, будут отброшены.

Многострочные значения не поддерживаются.

Для работы с .ini-файлом в кодировке Unicode сначала создайте файл через FileOpen(), указав в параметре mode нужный Unicode-режим.

См. также

IniDelete, IniRead, IniReadSection, IniReadSectionNames, IniRenameSection, IniWriteSection

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Записать значение 'AutoIt' в ключ 'Title' секции 'General'.
    IniWrite($sFilePath, "General", "Title", "AutoIt")

    ; Прочитать значение 'Title' в секции 'General' INI-файла.
    Local $sRead = IniRead($sFilePath, "General", "Title", "Default Value")

    ; Показать значение, возвращённое IniRead.
    MsgBox($MB_SYSTEMMODAL, "", "The value of 'Title' in the section labelled 'General' is: " & $sRead)

    ; Удалить ключ с именем 'Title'.
    IniDelete($sFilePath, "General", "Title")

    ; Прочитать значение 'Title' в секции 'General' INI-файла.
    $sRead = IniRead($sFilePath, "General", "Title", "Default Value")

    ; Показать значение, возвращённое IniRead. Так как ключа нет, значение будет 'Default Value', переданное в IniRead.
    MsgBox($MB_SYSTEMMODAL, "", "The value of 'Title' in the section labelled 'General' is: " & $sRead)

    ; Удалить INI-файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
