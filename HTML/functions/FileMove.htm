Справочник функций

FileMove

Перемещает один или несколько файлов.

FileMove ( "source", "dest" [, flag = 0] )

Параметры

source Исходный путь и имя перемещаемого файла. Допускается символ подстановки * – см. раздел Примечания.
dest Путь назначения и имя перемещённого файла. Допускается символ подстановки * – см. раздел Примечания.
flag [необязательный] флаг, определяющий поведение при совпадении имён файлов:
Допустимо комбинировать следующие значения:
$FC_NOOVERWRITE (0) = (по умолчанию) не перезаписывать существующие файлы.
$FC_OVERWRITE (1) = перезаписывать существующие файлы.
$FC_CREATEPATH (8) = создать структуру каталогов назначения, если она отсутствует (см. раздел Примечания).

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0, если source не удалось переместить, или если dest уже существует и flag=0.

Примечания

Об использовании символов подстановки см. описание FileFindFirstFile().

Если исходный и целевой пути находятся на разных томах, вместо перемещения выполняется копирование с последующим удалением источника.

В AutoIt отсутствует функция "FileRename", однако переименовать файл можно с помощью данной функции, передав в параметры source и dest полные пути со старым и новым именем файла соответственно.

Каталог назначения должен существовать заранее – за исключением случая, когда передан флаг $FC_CREATEPATH (8).
Например, комбинация $FC_OVERWRITE (1) + $FC_CREATEPATH (8) перезаписывает файл назначения и автоматически создаёт всю структуру каталогов, если она отсутствует.

Некоторые атрибуты файла могут препятствовать перезаписи; в таком случае воспользуйтесь FileSetAttrib(), чтобы изменить атрибуты файла.

См. также

DirMove, FileCopy, FileDelete, FileRecycle

Пример

Пример 1

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создаём временный файл для перемещения.
    If Not FileWrite($sFilePath, "This is an example of using FileMove.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Перемещаем au3-файлы из временного каталога в новую папку Au3Files.
    FileMove(@TempDir & "\*.au3", @TempDir & "\Au3Files\", $FC_OVERWRITE + $FC_CREATEPATH)

    ; Показать временный каталог.
    ShellExecute(@TempDir)
EndFunc   ;==>Example

Пример 2

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создаём локальные константы с путями переименовываемых файлов.
    Local Const $sSource = _WinAPI_GetTempFileName(@TempDir), _
            $sDestination = _WinAPI_GetTempFileName(@TempDir)

    ; Создаём временный файл для переименования.
    If Not FileWrite($sSource, "This is an example of using FileMove.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Переименовываем файл с помощью FileMove; если файл с новым именем существует – перезаписываем.
    FileMove($sSource, $sDestination, $FC_OVERWRITE)

    ; Выводим результат – файл назначения переименован.
    MsgBox($MB_SYSTEMMODAL, "", "Does the source file exist?: " & FileExists($sSource) & @CRLF & _ ; FileExists должна вернуть 0.
            "Does destination file exist?: " & FileExists($sDestination) & @CRLF) ; FileExists должна вернуть 1.

    ; Удаляем временные файлы. FileDelete проверяет существование файла.
    FileDelete($sSource)
    FileDelete($sDestination)
EndFunc   ;==>Example
