Справочник функций

DirGetSize

Возвращает размер указанного каталога в байтах.

DirGetSize ( "path" [, flag = 0] )

Параметры

path Путь к каталогу, размер которого нужно получить, например "C:\Windows".
flag [необязательный] Флаг, определяющий поведение и результат функции; может быть комбинацией следующих значений:
$DIR_DEFAULT (0) = (по умолчанию)
$DIR_EXTENDED (1) = расширенный режим включён – возвращает массив с дополнительной информацией (см. раздел Примечания).
$DIR_NORECURSE (2) = не учитывать размер файлов в подкаталогах (рекурсивный режим выключен)

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает размер, больший или равный нулю.
Ошибка: Возвращает -1 и устанавливает @error ≠ 0, если путь не существует.

Примечания

Если скрипт приостановлен, эта функция тоже приостанавливается и продолжит работу только после возобновления скрипта!

В расширенном режиме функция возвращает одномерный массив со следующими элементами:
$aArray[0] = размер
$aArray[1] = количество файлов
$aArray[2] = количество каталогов

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iSize = DirGetSize(@HomeDrive)
    MsgBox($MB_SYSTEMMODAL, "", "Size(MegaBytes): " & Round($iSize / 1024 / 1024))

    $iSize = DirGetSize(@WindowsDir, $DIR_NORECURSE)
    MsgBox($MB_SYSTEMMODAL, "", "Size(MegaBytes): " & Round($iSize / 1024 / 1024))

    Local $hTimer = TimerInit()

    Local $aSize = DirGetSize("\\10.0.0.1\h$", $DIR_EXTENDED) ; расширенный режим
    If Not @error Then
        Local $iDiff = Round(TimerDiff($hTimer) / 1000) ; время в секундах
        MsgBox($MB_SYSTEMMODAL, "", "Size(Bytes): " & $aSize[0] & @CRLF _
                 & "Files: " & $aSize[1] & @CRLF & "Dirs: " & $aSize[2] & @CRLF _
                 & "TimeDiff(Sec): " & $iDiff)
    EndIf
EndFunc   ;==>Example
