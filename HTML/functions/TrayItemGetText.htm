Справочник функций

TrayItemGetText

Возвращает текст элемента меню трея.

TrayItemGetText ( controlID )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией TrayCreateItem или TrayCreateMenu.

Возвращаемое значение

Успех: Возвращает текст элемента меню трея.
Ошибка: Возвращает 0.

См. также

TrayItemGetHandle, TrayItemSetText

Пример

#NoTrayIcon

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>
#include <TrayConstants.au3> ; Требуется для константы $TRAY_ICONSTATE_SHOW.

Opt("TrayMenuMode", 3) ; Стандартные пункты меню трея не будут показаны, и пункты не помечаются при выборе. Это опции 1 и 2 для TrayMenuMode.

Example()

Func Example()
    Local $idGetText = TrayCreateItem("Get 'About' Text")
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Показать окно сообщения с версией AutoIt и путём установки исполняемого файла AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", $STR_NOCASESENSEBASIC, -1) - 1)) ; Найти каталог полного пути.

            Case $idGetText
                ; Показать диалог с текстом элемента 'About'.
                MsgBox($MB_SYSTEMMODAL, "", "The text of the 'About' item is: " & TrayItemGetText($idAbout))

            Case $idExit ; Выйти из цикла.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
