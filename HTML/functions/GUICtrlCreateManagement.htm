Справочник функций: GUI Control creation

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

See Gui Constants include files if you need to use the related controls Constants .

 

Функция Описание
GUICtrlCreateAvi Создаёт элемент управления AVI-видео для GUI.
GUICtrlCreateButton Создаёт элемент управления Button (кнопка) для GUI.
GUICtrlCreateCheckbox Создаёт элемент управления Checkbox (флажок) для GUI.
GUICtrlCreateCombo Создаёт элемент управления ComboBox (выпадающий список) для GUI.
GUICtrlCreateContextMenu Создаёт контекстное меню для элемента управления или всего GUI-окна.
GUICtrlCreateDate Создаёт элемент управления Date (дата) для GUI.
GUICtrlCreateDummy Создаёт фиктивный (Dummy) элемент управления для GUI.
GUICtrlCreateEdit Создаёт элемент управления Edit (поле ввода) для GUI.
GUICtrlCreateGraphic Создаёт графический элемент управления (Graphic) для GUI.
GUICtrlCreateGroup Создаёт элемент управления Group (группа) для GUI.
GUICtrlCreateIcon Создаёт элемент управления Icon (значок) для GUI.
GUICtrlCreateInput Создаёт элемент управления Input (однострочное поле ввода) для GUI.
GUICtrlCreateLabel Создаёт статический элемент управления Label (подпись) для GUI.
GUICtrlCreateList Создаёт элемент управления List (список) для GUI.
GUICtrlCreateListView Создаёт элемент управления ListView для GUI.
GUICtrlCreateListViewItem Создаёт элемент (строку) ListView.
GUICtrlCreateMenu Создаёт элемент управления Menu (меню) для GUI.
GUICtrlCreateMenuItem Создаёт элемент управления MenuItem (пункт меню) для GUI.
GUICtrlCreateMonthCal Создаёт элемент управления MonthCal (месячный календарь) для GUI.
GUICtrlCreateObj Создаёт элемент управления ActiveX в GUI.
GUICtrlCreatePic Создаёт элемент управления Picture (изображение) для GUI.
GUICtrlCreateProgress Создаёт элемент управления Progress (индикатор выполнения) для GUI.
GUICtrlCreateRadio Создаёт элемент управления Radio (переключатель) для GUI.
GUICtrlCreateSlider Создаёт элемент управления Slider (ползунок) для GUI.
GUICtrlCreateTab Создаёт элемент управления Tab (вкладки) для GUI.
GUICtrlCreateTabItem Создаёт элемент управления TabItem (вкладку) внутри существующего элемента управления Tab в GUI.
GUICtrlCreateTreeView Создаёт элемент управления TreeView (дерево) для GUI.
GUICtrlCreateTreeViewItem Создаёт элемент управления TreeViewItem (узел дерева) для GUI.
GUICtrlCreateUpdown Создаёт элемент управления UpDown (счётчик) для GUI.
GUICtrlDelete Удаляет элемент управления.
