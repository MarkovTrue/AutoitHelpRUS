Справочник функций

FileDelete

Удаляет один или несколько файлов.

FileDelete ( "filename" )

Параметры

Filename Путь к удаляемому файлу (или файлам). Допускаются символы подстановки * и ? – см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0, если файлы не удалось удалить или они не существуют.

Примечания

Об использовании символов подстановки см. описание FileFindFirstFile().

Примечание: если в качестве пути функции FileDelete() передан каталог, файлы в нём будут удалены так же, как при использовании маски *.*.

Некоторые атрибуты файла могут препятствовать удалению; в таком случае воспользуйтесь FileSetAttrib(), чтобы изменить атрибуты файла.

См. также

DirRemove, FileCopy, FileMove, FileRecycle, FileRecycleEmpty

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, "This is an example of using FileDelete.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Показать содержимое файла, передавая FileRead путь к файлу вместо дескриптора от FileOpen.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & FileRead($sFilePath))

    ; Удалить временный файл.
    Local $iDelete = FileDelete($sFilePath)

    ; Выводим сообщение о результате удаления файла.
    If $iDelete Then
        MsgBox($MB_SYSTEMMODAL, "", "The file was successfuly deleted.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst deleting the file.")
    EndIf
EndFunc   ;==>Example
