Справочник функций: Process

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
DllCall Динамически вызывает функцию из DLL.
DllCallAddress Динамически вызывает функцию по указанному адресу памяти.
DllCallbackFree Освобождает ранее созданный дескриптор, созданный с помощью DllCallbackRegister.
DllCallbackGetPtr Возвращает указатель на функцию обратного вызова, который можно передать в Win32 API.
DllCallbackRegister Создаёт пользовательскую функцию обратного вызова DLL.
DllClose Закрывает ранее открытую DLL.
DllOpen Открывает DLL-файл для использования в DllCall.
DllStructCreate Создаёт структуру в стиле C/C++ для использования в DllCall.
DllStructGetData Возвращает данные элемента структуры.
DllStructGetPtr Возвращает указатель на структуру или на элемент структуры.
DllStructGetSize Возвращает размер структуры в байтах.
DllStructSetData Устанавливает данные элемента структуры.
ProcessClose Завершает указанный процесс.
ProcessExists Проверяет, выполняется ли указанный процесс.
ProcessGetStats Возвращает массив с данными о памяти или операциях ввода-вывода выполняющегося процесса.
ProcessList Возвращает массив с именами и PID выполняющихся процессов.
ProcessSetPriority Изменяет приоритет процесса.
ProcessWait Приостанавливает выполнение скрипта до появления указанного процесса.
ProcessWaitClose Приостанавливает выполнение скрипта до завершения указанного процесса.
Run Запускает внешнюю программу.
RunAs Запускает внешнюю программу от имени другого пользователя.
RunAsWait Запускает внешнюю программу от имени другого пользователя и приостанавливает выполнение скрипта до завершения программы.
RunWait Запускает внешнюю программу и приостанавливает выполнение скрипта до её завершения.
ShellExecute Запускает внешнюю программу через API ShellExecute.
ShellExecuteWait Запускает внешнюю программу через API ShellExecute и приостанавливает выполнение скрипта до её завершения.
Shutdown Выполняет завершение работы системы.
StderrRead Читает данные из потока STDERR ранее запущенного дочернего процесса.
StdinWrite Записывает данные в поток STDIN ранее запущенного дочернего процесса.
StdioClose Освобождает все ресурсы, связанные с STDIO-перенаправлением ранее запущенного процесса.
StdoutRead Читает данные из потока STDOUT ранее запущенного дочернего процесса.
