Справочник функций

GUISetState

Изменяет состояние окна GUI.

GUISetState ( [flag [, winhandle]] )

Параметры

flag [необязательный] @SW_SHOW = Показывает ранее скрытое окно (по умолчанию)
@SW_HIDE = Скрыть окно
@SW_MINIMIZE = Свернуть окно
@SW_MAXIMIZE = Развернуть окно
@SW_RESTORE = Отменить свёртывание окна
@SW_DISABLE = Отключить окно
@SW_ENABLE = Включить окно
@SW_LOCK = Заблокировать окно (запретить перерисовку).
@SW_UNLOCK = Разблокировать окно (разрешить перерисовку).
@SW_SHOWDEFAULT – Устанавливает состояние отображения на основе флага SW_ из структуры STARTUPINFO
@SW_SHOWMAXIMIZED – Активирует окно и отображает его развёрнутым
@SW_SHOWMINIMIZED – Активирует окно и отображает его свёрнутым
@SW_SHOWMINNOACTIVE – Отображает окно свёрнутым без активации
@SW_SHOWNA – Отображает окно в текущем состоянии
@SW_SHOWNOACTIVATE – Отображает окно в последнем размере и положении без активации
@SW_SHOWNORMAL – Активирует и отображает окно.
winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

При создании окна изначально скрыты, поэтому для их отображения необходимо вызвать эту функцию (@SW_SHOW).

Одновременно может быть заблокировано только одно окно. Повторный вызов @SW_LOCK заблокирует новое окно.
@SW_UNLOCK игнорирует winhandle и разблокирует любое заблокированное окно.

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI") ; начало определения

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
