Справочник функций

StringIsDigit

Проверяет, содержит ли строка только цифровые символы (0-9).

StringIsDigit ( "string" )

Параметры

string Проверяемая строка.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если строка содержит нецифровые символы.

Примечания

Как и другие функции StringIs...(), проверка прекращается при обнаружении Chr(0) внутри строки.
Для ускорения обработки внутренне используется функция языка C.

См. примеры.

См. также

StringIsAlNum, StringIsAlpha, StringIsASCII, StringIsInt, StringIsLower, StringIsSpace, StringIsUpper, StringIsXDigit

Пример

#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "", "Is the string 42 a digit: " & StringIsDigit("42") & @CRLF & _ ; Возвращает 1, так как строка содержит только цифры (0-9).
        "Is the string 00 a digit: " & StringIsDigit("00") & @CRLF & _ ; Возвращает 1, так как строка содержит только цифры (0-9).
        "Is the string 1.0 a digit: " & StringIsDigit("1.0") & @CRLF & _ ; Возвращает 0 из-за десятичной точки.
        "Is the number 1.0 a digit: " & StringIsDigit(1.0) & @CRLF & _ ; Возвращает 1 из-за преобразования числа в строку.
        "Is the string 1+2 a digit: " & StringIsDigit("1+2") & @CRLF) ; Возвращает 0, так как строка содержит символ "+" (плюс).
