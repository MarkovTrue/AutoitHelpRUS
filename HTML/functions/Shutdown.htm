Справочник функций

Shutdown

Выполняет завершение работы системы.

Shutdown ( code )

Параметры

code Комбинация кодов завершения работы. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает значение @error = значение Windows API GetLastError().

Примечания

Код завершения работы является комбинацией следующих значений:
    $SD_LOGOFF (0) – выход из системы
    $SD_SHUTDOWN (1) – завершение работы
    $SD_REBOOT (2) – перезагрузка
    $SD_FORCE (4) – принудительно
    $SD_POWERDOWN (8) – выключение питания
    $SD_FORCEHUNG (16) – принудительно при зависании
    $SD_STANDBY (32) – ждущий режим
    $SD_HIBERNATE (64) – спящий режим

Константы определены в AutoItConstants.au3.

Нужные значения следует объединять через BitOR(). Например, для завершения работы с выключением питания: BitOR($SD_SHUTDOWN, $SD_POWERDOWN).

Standby и Hibernate игнорируются, если установлены другие коды.
