Справочник функций

GUICtrlCreateEdit

Создаёт элемент управления Edit (поле ввода) для GUI.

GUICtrlCreateEdit ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

text Текст элемента управления.
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $ES_WANTRETURN, $WS_VSCROLL, $WS_HSCROLL, $ES_AUTOVSCROLL, $ES_AUTOHSCROLL
принудительные стили : $ES_MULTILINE; $WS_TABSTOP – только если не задан $ES_READONLY
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для получения значения элемента управления см. GUICtrlRead().
Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_EDIT, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <EditConstants.au3>.

Чтобы разрешить перетаскивание имён файлов на этот элемент управления, добавьте расширенный стиль WS_EX_ACCEPTFILES в GUICreate() и установите состояние $GUI_DROPACCEPTED.
Несколько выбранных файлов будут вставлены на отдельных строках.

Изменение размера по умолчанию: $GUI_DOCKAUTO – автоматическое изменение размера и позиции.

Создание элемента управления RichEdit слишком сложно для базового набора, поэтому он не включён как стандартный элемент.
Используйте _GUICtrlRichEdit_Create().

См. также

GUICoordMode (Option), GUICtrlRead, GUICtrlSetData, GUICtrlSetLimit, GUICtrlSetState, GUIGetMsg

Пример

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    GUICreate("My GUI edit") ; создаст диалоговое окно, отображаемое по центру

    Local $idEdt_Myedit = GUICtrlCreateEdit("First line" & @CRLF, 176, 32, 121, 97, $ES_AUTOVSCROLL + $WS_VSCROLL)

    GUISetState(@SW_SHOW)

    Send("{END}")

    ; будет добавлено в конец – не забудьте 3-й параметр
    GUICtrlSetData($idEdt_Myedit, "Second line", 1)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
    GUIDelete()
EndFunc   ;==>Example
