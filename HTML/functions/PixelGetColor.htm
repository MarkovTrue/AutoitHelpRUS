Справочник функций

PixelGetColor

Возвращает цвет пикселя по координатам x, y.

PixelGetColor ( x , y [, hwnd] )

Параметры

x Координата x пикселя.
y Координата y пикселя.
hwnd [необязательный] Дескриптор окна. По умолчанию используется рабочий стол. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает десятичное значение цвета пикселя.
Ошибка: Возвращает -1, если координаты недопустимы.

Примечания

Если PixelCoordMode установлен в режим SCREEN (1) по умолчанию, параметр hwnd игнорируется. Он учитывается только при режиме WINDOW (0) или CLIENT (2).

См. также

MouseGetPos, PixelChecksum, PixelCoordMode (Option), PixelSearch

Пример

#include <MsgBoxConstants.au3>

Local $iColor = PixelGetColor(10, 100)

MsgBox($MB_SYSTEMMODAL, "", "The decimal color is: " & $iColor)
MsgBox($MB_SYSTEMMODAL, "", "The hex color is: " & Hex($iColor, 6))
