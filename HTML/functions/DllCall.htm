Справочник функций

DllCall

Динамически вызывает функцию из DLL.

DllCall ( "dll", "return type", "function" [, type1, param1 [, type n, param n]] )

Параметры

dll Имя файла DLL для использования, например "user32.dll". Также можно передать дескриптор, полученный от DllOpen (см. раздел Примечания).
return type Тип возвращаемого значения функции (см. ниже).
function Имя функции DLL для вызова, например "MessageBox", или её порядковый номер, например 62.
type1 [необязательный] Тип параметра (см. раздел Примечания).
param1 [необязательный] Значение параметра (см. раздел Примечания).
type n [необязательный] Тип n-го параметра (см. раздел Примечания).
param n [необязательный] Значение n-го параметра (см. раздел Примечания).

Чтобы использовать вложенные структуры внутри структуры, переопределите вложенную структуру. Например, структура с двумя структурами POINT ("long;long") объявляется как "long;long;long;long". Первые два значения long соответствуют первой структуре POINT, вторые два – второй структуре POINT.

Другие типы Windows API см. на MSDN.

Возвращаемое значение

Успех: Возвращает массив. См. раздел Примечания.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – невозможно использовать DLL-файл,
2 – неизвестный тип возвращаемого значения,
3 - function не найдена в DLL-файле,
4 – неверное количество параметров,
5 – неверный параметр.

Примечания

Если указано имя DLL-файла, то DLL автоматически загружается и закрывается по завершении вызова. Для ручного управления загрузкой/выгрузкой DLL используйте DllOpen() и DllClose(), передавая дескриптор вместо имени файла.

По умолчанию AutoIt использует соглашение о вызовах 'stdcall'. Для использования метода 'cdecl' добавьте ':cdecl' после типа возвращаемого значения.

DllCall("SQLite.dll", "int:cdecl", "sqlite3_open", "str", $sDatabase_Filename , "long*", 0)


По умолчанию AutoIt пытается использовать ANSI-версию имени функции, то есть при передаче "MessageBox" сначала вызывается MessageBoxA. Для вызова Unicode-версии используйте MessageBoxW.

Если вызов функции завершается ошибкой, @error ≠ 0.
В противном случае возвращается массив, содержащий возвращаемое значение функции и копии всех параметров (включая параметры, изменённые функцией при передаче по ссылке).

$return[0] = возвращаемое значение функции
$return[1] = param1
$return[2] = param2
...
$return[n] = paramn

Если выходной параметр типа STR или WSTR определён как пустая строка Null, будет возвращена строка длиной не более 65536 символов.

Пример

Пример 1

; Прямой вызов MessageBox API.
DllCall("user32.dll", "int", "MessageBox", _
        "hwnd", 0, _ ; Дескриптор родительского окна
        "str", "Some text", _ ; Текст диалогового окна
        "str", "Some title", _ ; Заголовок диалогового окна
        "int", 0) ; Флаги диалогового окна.

Пример 2

#include <MsgBoxConstants.au3>

; Вызов функции, изменяющей параметры,
Local $iPID = Run("notepad")

Local $hWnd = WinWait("[CLASS:Notepad]", "", 2000)

If $hWnd = 0 Then
    ; Время ожидания истекло.
    Exit MsgBox($MB_SYSTEMMODAL, Default, "Unable to start notepad!")
EndIf

Local $aResult = DllCall("user32.dll", "int", "GetWindowText", "hwnd", $hWnd, "str", "", "int", 32768)
MsgBox($MB_SYSTEMMODAL, Default, "Number of characters returned: " & $aResult[0])
MsgBox($MB_SYSTEMMODAL, Default, "Text (returned in parameter 2): '" & $aResult[2] & "'")

WinClose($hWnd)

Пример 3

#include <MsgBoxConstants.au3>

; Отобразить диалог Windows PickIconDlg.
Local $sFileName = @SystemDir & '\shell32.dll'

; Создать структуру для хранения индекса иконки
Local $tIconIndex = DllStructCreate("int")
Local $tString = DllStructCreate("wchar[260]")
Local $iStructsize = DllStructGetSize($tString) / 2
DllStructSetData($tString, 1, $sFileName)

; Запустить PickIconDlg – '62' является порядковым номером этой функции
DllCall("shell32.dll", "none", 62, _
        "hwnd", 0, _
        "struct*", $tString, _
        "int", $iStructsize, _
        "struct*", $tIconIndex)

$sFileName = DllStructGetData($tString, 1)
Local $iIconIndex = DllStructGetData($tIconIndex, 1)

; Отобразить новое имя файла и индекс иконки
MsgBox($MB_SYSTEMMODAL, "Info", "Last selected file: " & $sFileName & @CRLF & "Icon-Index: " & $iIconIndex)
