Справочник функций

ControlClick

Отправляет команду клика мыши указанному элементу управления.

ControlClick ( "title", "text", controlID [, button = "left" [, clicks = 1 [, x [, y]]]] )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.
button [необязательный] Кнопка для клика: "left" (по умолчанию), "right", "middle", "main", "menu", "primary", "secondary".
clicks [необязательный] Количество кликов мышью. По умолчанию 1.
x [необязательный] X координата клика внутри элемента управления. По умолчанию центр.
y [необязательный] Y координата клика внутри элемента управления. По умолчанию центр.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Некоторые элементы управления не реагируют на клик, пока их окно не активно. Используйте WinActivate(), чтобы вывести окно элемента на передний план перед вызовом ControlClick().
Указание 2 в количестве кликов отправит элементу сообщение о двойном клике, это можно использовать даже для запуска программ из explorer-элемента!

Если пользователь поменял местами левую и правую кнопки мыши в панели управления, поведение кнопок изменится. "Left" и "Right" всегда нажимают именно эти кнопки, независимо от их подмены. "Primary" или "Main" основной клик независимо от подмены. "Secondary" или "Menu" обычно вызывают контекстное меню, независимо от подмены.

Кнопка Обычный режим Подмена
"" Левая Левая
"left" Левая Левая
"middle" Средняя Средняя
"right" Правая Правая
"primary" Левая Правая
"main" Левая Правая
"secondary" Правая Левая
"menu" Правая Левая

Пример

#include "Extras\HelpFileInternals.au3"

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Отправить клик мыши в edit-элемент Блокнота, используя дескриптор, возвращённый WinWait.
    ControlClick($hWnd, "", ControlGetFocus($hWnd))

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
