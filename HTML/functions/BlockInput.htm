Справочник функций

BlockInput

Отключает/включает мышь и клавиатуру.

BlockInput ( flag )

Параметры

flag $BI_DISABLE (1) = отключить пользовательский ввод
$BI_ENABLE (0) = включить пользовательский ввод

Константы определены в файле "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0. Ввод уже включён или не используется #RequireAdmin.

Примечания

В таблице ниже показано, как поведение BlockInput() зависит от версии Windows. Однако нажатие Ctrl+Alt+Del на любой платформе повторно включит ввод – это особенность Windows API.

Операционная система Поведение "BlockInput()"
Windows XP Пользовательский ввод блокируется, AutoIt может имитировать ввод мышью и клавиатурой.
Windows Vista и выше Пользовательский ввод блокируется только при использовании #RequireAdmin, AutoIt может имитировать ввод мышью и клавиатурой.

BlockInput() влияет только на пользовательский ввод. Ввод от функций вроде Send(), ControlSend() или MouseMove() продолжает работать.

См. также

Send, ControlSend, MouseMove

Пример

#RequireAdmin

#include "Extras\HelpFileInternals.au3"

#include <AutoItConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Получить элемент управления, имеющий фокус клавиатуры в Блокноте.
    Local $sEditControl = ControlGetFocus($hWnd)

    ; Отключить пользовательский ввод с мыши и клавиатуры.
    BlockInput($BI_DISABLE)

    ; набор с клавиатуры не даёт эффекта

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Отправить клавишу 'F5' в поле ввода Блокнота для отображения даты и времени.
    ; Дескриптор, возвращённый WinWait, используется в параметре "title" функции ControlSend.
    ; ControlSend отработает даже при активном BlockInput()
    ControlSend($hWnd, "", $sEditControl, "{F5}")

    ; Включить пользовательский ввод с мыши и клавиатуры.
    BlockInput($BI_ENABLE)

    ; набор с клавиатуры работает

    ; Ждать 2 секунды для проверки, что ввод включён
    Sleep(4000)

    _NotepadForceClose($hWnd)
EndFunc   ;==>Example
