Справочник функций

TraySetToolTip

Устанавливает (или сбрасывает) текст всплывающей подсказки для значка трея.

TraySetToolTip ( [text] )

Параметры

text [необязательный] Новый текст всплывающей подсказки. Длина ограничена – см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Длина текста подсказки ограничена 128 символами.

Для сброса подсказки к стандартному тексту вызовите функцию без параметров: TraySetToolTip().

Текст подсказки изменяется только если значок трея видим!

См. также

TrayTip

Пример

#NoTrayIcon

#include <TrayConstants.au3>

Example()

Func Example()
    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.
    TraySetToolTip("An example of a tray menu tooltip.") ; Значок меню трея должен быть показан перед установкой текста.

    While 1
        Sleep(100) ; Цикл ожидания.
    WEnd
EndFunc   ;==>Example
