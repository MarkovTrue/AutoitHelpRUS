Справочник функций

SetExtended

Вручную устанавливает значение макроса @extended.

SetExtended ( code [, return value] )

Параметры

code Целочисленное значение, устанавливаемое в макрос @extended.
return value [необязательный] (по умолчанию 1) Значение, возвращаемое функцией.

Возвращаемое значение

Возвращает значение параметра return value.

Примечания

При входе в пользовательскую функцию макрос @extended сбрасывается в 0. Если SetExtended() не вызывается, @extended остаётся 0 по завершении функции. Поэтому для передачи значения @extended вызывающему коду его необходимо явно установить. Если вы проверяете @extended внутри цикла While-WEnd, может потребоваться сохранить его значение в переменной.
Параметр return value необязателен и позволяет использовать синтаксис Return SetError(...) для одновременного задания возвращаемого значения и кода ошибки. Если значение не указано, возвращается 1, так как SetError() выполняется успешно.

@extended допускает значения от –2147483648 до 2147483647.

См. также

SetError, Return

Пример

#include <MsgBoxConstants.au3>

SetExtended(10)
MsgBox($MB_SYSTEMMODAL, "", "Value of @extended is: " & @extended)
