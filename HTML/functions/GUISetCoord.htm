GUISetCoord

Задаёт абсолютные координаты для следующего элемента управления.

GUISetCoord ( left, top [, width [, height [, winhandle]]] )

Параметры

left Левая граница элемента управления.
top Верхняя граница элемента управления.
width [необязательный] Ширина элемента управления (по умолчанию ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию ранее использованная высота).
winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Используется исключительно в режиме Opt("GUICoordMode", 2). Позволяет установить текущую позицию в точную точку и от неё создавать элементы управления по строкам (x_offset, -1) или по столбцам (-1, y_offset).

См. также

GUICtrlCreate...

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    Opt("GUICoordMode", 2) ; режим относительно ячейки

    GUICreate("My GUI Set Coord", 200, 100)
    GUICtrlCreateCheckbox("Check #1", 20, 10, 75)
    GUICtrlCreateCheckbox("Notify #2", 10, -1) ; следующая ячейка в строке

    GUISetCoord(20, 60)

    GUICtrlCreateButton("OK #3", -1, -1)
    GUICtrlCreateButton("Cancel #4", 10, -1)
    GUICtrlSetState(-1, $GUI_FOCUS)

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example
