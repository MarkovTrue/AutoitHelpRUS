Справочник функций

FileFindNextFile

Возвращает следующее имя файла, соответствующее дескриптору поиска.

FileFindNextFile ( search [, flag = 0])

Параметры

search Дескриптор поиска, возвращённый функцией FileFindFirstFile().
flag [необязательный] флаг, определяющий содержимое @extended.
0 = (по умолчанию) @extended возвращает 1 или 0 в зависимости от того, является ли найденный элемент каталогом.
1 = @extended возвращает строку в том же формате, что и FileGetAttrib().

Возвращаемое значение

Успех: Возвращает имя файла согласно предыдущему вызову FileFindFirstFile(); @extended устанавливается в соответствии с флагом.
Ошибка: устанавливает значение @error = 1, если файлы или каталоги, соответствующие критерию поиска, закончились.

Примечания

Перед использованием необходимо вызвать FileFindFirstFile() для инициализации поиска и получения дескриптора. Каждый последующий вызов FileFindNextFile() возвращает следующий найденный файл согласно дескриптору из FileFindFirstFile(). При @error = 1 файлов, соответствующих исходному дескриптору, больше нет.

По окончании поиска с помощью функций FileFind... необходимо вызвать FileClose() для освобождения дескриптора поиска.

В силу особенностей используемого Windows API (FindFirstFile) данная функция при поиске совпадений проверяет как длинные, так и короткие имена файлов. Если вы получаете неожиданные результаты, убедитесь, что совпадение не происходит по короткому имени файла.

См. также

FileClose, FileFindFirstFile

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной search-дескриптор всех файлов текущей папки.
    Local $hSearch = FileFindFirstFile("*.*")

    ; Проверить успех поиска – при неудаче показать сообщение и вернуть False.
    If $hSearch = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "Error: No files/directories matched the search pattern.")
        Return False
    EndIf

    ; Присвоить локальной переменной пустую строку для накопления найденных имён файлов.
    Local $sFileName = "", $iResult = 0

    While 1
        $sFileName = FileFindNextFile($hSearch)
        ; Если файлов, соответствующих критерию поиска, больше нет.
        If @error Then ExitLoop

        ; Показать имя файла.
        $iResult = MsgBox(($MB_OKCANCEL + $MB_SYSTEMMODAL), "", "File: " & $sFileName)
        If $iResult <> $IDOK Then ExitLoop ; Если пользователь нажал кнопку отмены или закрытия.
    WEnd

    ; Закрыть search-дескриптор.
    FileClose($hSearch)
EndFunc   ;==>Example
