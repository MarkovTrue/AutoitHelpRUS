GUICtrlCreateDate

Создаёт элемент управления Date (дата) для GUI.

GUICtrlCreateDate ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

text Предвыбранная дата (всегда в формате "yyyy/mm/dd").
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию – ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию – ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $DTS_LONGDATEFORMAT
принудительный стиль : $WS_TABSTOP
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для получения значения элемента управления см. GUICtrlRead().
Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_DATE, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <DateTimeConstants.au3>.

Для форматирования даты/времени см. пример 3 – там показано, как использовать GUICtrlSendMsg() с $DTM_SETFORMAT.

Изменение размера по умолчанию: $GUI_DOCKHEIGHT.

См. также

GUICoordMode (Option), GUICtrlRead, GUICtrlSetState, GUIGetMsg

Пример

Пример 1

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("My GUI get date", 200, 200, 800, 200)
    Local $idDate = GUICtrlCreateDate("1953/04/25", 10, 10, 185, 20)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    MsgBox($MB_SYSTEMMODAL, "Date", GUICtrlRead($idDate))
    GUIDelete()
EndFunc   ;==>Example

Пример 3

#include <DateTimeConstants.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("My GUI get date", 200, 200, 800, 200)
    Local $idDate = GUICtrlCreateDate("1953/04/25", 10, 10, 185, 20)

    ; выбор конкретного формата по умолчанию
    Local $sStyle = "yyyy/MM/dd HH:mm:ss"
    GUICtrlSendMsg($idDate, $DTM_SETFORMATW, 0, $sStyle)

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While GUIGetMsg() <> $GUI_EVENT_CLOSE
    WEnd

    MsgBox($MB_SYSTEMMODAL, "Time", GUICtrlRead($idDate))
EndFunc   ;==>Example
