Справочник функций

Asc

Возвращает ASCII-код символа.

Asc ( "char" )

Параметры

char Символ, для которого требуется получить ASCII-код. Если передана строка, возвращается код первого символа.

Возвращаемое значение

Возвращает ASCII-код указанного символа.

Примечания

Полный список значений см. в таблице Коды ASCII символов.

Asc("0") == 48
Asc("9") == 57
Asc("A") == 65
Asc("Z") == 90
Asc("a") == 97
Asc("z") == 122 и т. д.

См. также

AscW, Chr, ChrW

Пример

#include <MsgBoxConstants.au3>

Local $iCode = Asc("A")
MsgBox($MB_SYSTEMMODAL, "", "ASCII code for A: " & $iCode)
