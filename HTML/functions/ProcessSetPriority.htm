ProcessSetPriority

Изменяет приоритет процесса.

ProcessSetPriority ( "process", priority )

Параметры

process Имя или PID проверяемого процесса.
priority Флаг, определяющий устанавливаемый приоритет:
$PROCESS_LOW (0) = Низкий
$PROCESS_BELOWNORMAL (1) = Ниже нормального
$PROCESS_NORMAL (2) = Нормальный
$PROCESS_ABOVENORMAL (3) = Выше нормального
$PROCESS_HIGH (4) = Высокий
$PROCESS_REALTIME (5) = Реального времени (применять с осторожностью, может дестабилизировать систему)

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1.
2 – при попытке использовать неподдерживаемый класс приоритета.

См. также

ProcessList

Пример

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Установить приоритет процесса Блокнота в "Низкий".
    ProcessSetPriority("notepad.exe", 0)

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
