Справочник функций

IsMap

Проверяет, является ли переменная типом Map.

IsMap ( variable )

Параметры

variable Проверяемая переменная или выражение.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если параметр не является переменной типа Map.

Примечания

Удобно использовать для проверки параметров пользовательских функций на принадлежность к типу Map.

Подробное описание типов данных см. в разделе типы данных языка.

См. также

IsArray, IsBinary, IsBool, IsFloat, IsHWnd, IsInt, IsNumber, IsPtr, IsString, MapExists, VarGetType

Пример

#include <MsgBoxConstants.au3>

Local $mMap[] ; Объявить Map.
If IsMap($mMap) Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable is a map")
Else
    MsgBox($MB_SYSTEMMODAL, "", "The variable is not a map")
EndIf
