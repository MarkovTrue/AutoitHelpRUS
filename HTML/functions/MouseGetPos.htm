Справочник функций

MouseGetPos

Возвращает текущую позицию курсора мыши.

MouseGetPos ( [dimension] )

Параметры

dimension [необязательный] Аргумент, определяющий формат возвращаемого значения. См. раздел Возвращаемое значение.

Возвращаемое значение

Dimension Описание.
Не указан Возвращает двухэлементный массив с координатами курсора: $aArray[0] = X (горизонталь), $aArray[1] = Y (вертикаль).
0 Возвращает координату X в виде целого числа.
1 Возвращает координату Y в виде целого числа.

Если dimension не является числом, @error устанавливается в 1.

Примечания

О настройке относительного/абсолютного позиционирования см. MouseCoordMode. При относительном позиционировании значения могут быть отрицательными.

См. также

MouseClick, MouseClickDrag, MouseCoordMode (Option), MouseDown, MouseGetCursor, MouseMove, MouseUp, MouseWheel, PixelGetColor

Пример

#include <MsgBoxConstants.au3>

Local $aPos = MouseGetPos()
MsgBox($MB_SYSTEMMODAL, "Mouse x, y:", $aPos[0] & ", " & $aPos[1])
