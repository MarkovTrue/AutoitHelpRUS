Справочник функций

TrayItemSetText

Устанавливает текст элемента меню трея.

TrayItemSetText ( controlID, text )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией TrayCreateItem или TrayCreateMenu.
text Новый текст элемента меню трея.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Для изменения текста стандартных элементов меню (Script Paused/Exit) используйте константы $TRAY_ITEM_EXIT и $TRAY_ITEM_PAUSE в качестве controlID.

Вышеуказанные константы определены в #include <TrayConstants.au3>.

См. также

TrayCreateItem, TrayCreateMenu, TrayItemGetText

Пример

#NoTrayIcon

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>
#include <TrayConstants.au3> ; Требуется для констант $TRAY_ICONSTATE_SHOW, $TRAY_ITEM_EXIT и $TRAY_ITEM_PAUSE.

Opt("TrayAutoPause", 0) ; Скрипт не будет приостанавливаться при выборе значка трея.
Opt("TrayMenuMode", 2) ; Элементы не отмечаются при выборе.

Example()

Func Example()
    Local $idRandom = TrayCreateItem("Random:") ; Выбрать этот элемент для изменения текста случайным числом.
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idAbout = TrayCreateItem("About")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    TrayItemSetText($TRAY_ITEM_EXIT, "Exit Program") ; Установить текст стандартного элемента 'Exit'.
    TrayItemSetText($TRAY_ITEM_PAUSE, "Pause Program") ; Установить текст стандартного элемента 'Pause'.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Показать окно сообщения с версией AutoIt и путём установки исполняемого файла AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", $STR_NOCASESENSEBASIC, -1) - 1)) ; Найти каталог полного пути.

            Case $idRandom
                ; Установить текст элемента 'Random' случайным целым числом.
                TrayItemSetText($idRandom, "Random: " & Int(Random(1, 10, 1)))

        EndSwitch
    WEnd
EndFunc   ;==>Example
