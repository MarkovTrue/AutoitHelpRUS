Справочник функций

ClipGet

Получает текст из буфера обмена.

ClipGet ( )

Возвращаемое значение

Успех: Возвращает строку с текстом из буфера обмена.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 - если буфер обмена пуст
2 - если содержит нетекстовую запись.
3 или 4 - если нет доступа к буферу обмена.

Примечания

Если в буфере обмена находится несколько выделенных файлов/каталогов, их имена возвращаются в виде текста, разделённого @LF.

См. также

ClipPut

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получить данные из буфера обмена.
    Local $sData = ClipGet()

    ; Показать данные, возвращённые ClipGet.
    MsgBox($MB_SYSTEMMODAL, "", "The following data is stored in the clipboard: " & @CRLF & $sData)

    ; Добавить новые данные в буфер обмена.
    ClipPut("A new string added to the clipboard.")

    ; Получить данные из буфера обмена.
    $sData = ClipGet()

    ; Показать данные, возвращённые ClipGet.
    MsgBox($MB_SYSTEMMODAL, "", "The following data is now stored in the clipboard: " & @CRLF & $sData)
EndFunc   ;==>Example
