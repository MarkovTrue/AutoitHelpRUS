Справочник функций

BitNOT

Выполняет побитовую операцию NOT.

BitNOT ( value )

Параметры

value Число, над которым выполняется операция.

Возвращаемое значение

Возвращает значение побитового отрицания указанного числа.
Битовые операции выполняются над 32-битными целыми числами.

Примечания

Напомним: для чисел можно использовать шестнадцатеричную запись.
В записи BitNOT() функционально эквивалентен прибавлению 1 и смене знака.
Также напомним: NOT меняет бит 0 на 1 и наоборот.

См. также

BitAND, BitOR, BitRotate, BitShift, BitXOR, Hex

Пример

#include <MsgBoxConstants.au3>

; Присвоить локальной переменной результат побитового NOT для 5.
Local $iBitNOT1 = BitNOT(5)

#cs Примечания:
Примечание: "b" – это символ байта.

Результат равен -6, так как для 32-битных чисел
+5 == 0000 0000 0000 0000 0000 0000 0000 0101b
-6 == 1111 1111 1111 1111 1111 1111 1111 1010b
и первый бит является знаковым
#ce Примечания:

; Показать результат.
MsgBox($MB_SYSTEMMODAL, "", $iBitNOT1)
