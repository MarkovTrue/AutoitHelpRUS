Справочник функций

GUICtrlGetState

Возвращает текущее состояние элемента управления.

GUICtrlGetState ( controlID )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.

Возвращаемое значение

Успех: Возвращает состояние. Список значений см. в GUICtrlSetState().
Ошибка: Возвращает -1, если элемент управления не определён.

Примечания

В отличие от GUICtrlRead(), данная функция возвращает ТОЛЬКО состояние элемента управления: enabled/disabled/hidden/show/dropaccepted.

Исключения:
Для элементов управления ListView возвращает номер нажатого столбца.

См. также

GUICtrlRead, GUICtrlSetState

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("My GUI (GetControlState)")
    Local $idCheckbox = GUICtrlCreateCheckbox("checkbox", 10, 10)
    GUICtrlSetState(-1, 1) ; отмечен

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    MsgBox($MB_SYSTEMMODAL, "state", StringFormat("GUICtrlRead=%d\nGUICtrlGetState=%d", GUICtrlRead($idCheckbox), GUICtrlGetState($idCheckbox)))
EndFunc   ;==>Example
