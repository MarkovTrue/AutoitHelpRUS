Справочник функций

ProcessWait

Приостанавливает выполнение скрипта до появления указанного процесса.

ProcessWait ( "process" [, timeout = 0] )

Параметры

process Имя ожидаемого процесса.
timeout [необязательный] Время ожидания в секундах. По умолчанию ожидание неограниченно.

Возвращаемое значение

Успех: Возвращает PID процесса.
Ошибка: Возвращает 0, если истёк таймаут ожидания.

Примечания

Имена процессов указываются без пути, например "notepad.exe" или "winword.exe".

Процесс опрашивается приблизительно каждые 250 миллисекунд.

Эта функция – единственная среди процессных, не принимающая PID. Поскольку PID назначаются случайно, ожидание конкретного PID не имеет смысла.

См. также

ProcessClose, ProcessExists, ProcessList, ProcessWaitClose, RunAsWait, RunWait, WinGetProcess, WinWait, WinWaitActive

Пример

; Ожидать появления процесса Блокнота.

ProcessWait("notepad.exe")
