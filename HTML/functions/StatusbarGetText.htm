Справочник функций

StatusbarGetText

Считывает текст из стандартного элемента управления строкой состояния.

StatusbarGetText ( "title" [, "text" [, part = 1]] )

Параметры

title Заголовок, hWnd или класс проверяемого окна.
text [необязательный] Текст проверяемого окна. По умолчанию – пустая строка.
part [необязательный] Номер секции строки состояния для чтения (по умолчанию 1). Секция 1 – первая и обычно содержит информационные сообщения вида "Готово", "Загрузка..." и т.п.

Возвращаемое значение

Успех: Возвращает считанный текст.
Ошибка: Возвращает "" (пустую строку); @error устанавливается в 1, если текст считать не удалось.

Примечания

Функция пытается считать данные из первой стандартной строки состояния окна (общий элемент управления Microsoft: msctls_statusbar32). Некоторые программы используют собственные строки состояния или специальные версии общего элемента MS, которые StatusbarGetText() не поддерживает. Например, для TextPad функция не работает, однако первую секцию его строки состояния можно прочитать через ControlGetText("TextPad", "", "HSStatusBar1").
StatusbarGetText() работает со свёрнутыми и скрытыми окнами.

См. также

ControlCommand, ControlGetText

Пример

#include <MsgBoxConstants.au3>

AutoItSetOption("WinTitleMatchMode", 2)

Local $sText = StatusbarGetText("Internet Explorer")
MsgBox($MB_SYSTEMMODAL, "", "Internet Explorer's status bar says: " & $sText)
