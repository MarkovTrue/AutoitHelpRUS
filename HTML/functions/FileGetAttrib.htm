Справочник функций

FileGetAttrib

Возвращает строку кодов, представляющую атрибуты файла.

FileGetAttrib ( "filename" )

Параметры

filename Путь к проверяемому файлу или каталогу.

Возвращаемое значение

Успех: Возвращает строку кодов, представляющую атрибуты файла.
Ошибка: Возвращает "" (пустую строку) и устанавливает значение @error = 1.

Примечания

Возвращаемая строка может содержать комбинацию следующих букв "RASHNDOCTXJ":
    "R" = READONLY (только чтение)
    "A" = ARCHIVE (архивный)
    "S" = SYSTEM (системный)
    "H" = HIDDEN (скрытый)
    "N" = NORMAL (обычный)
    "D" = DIRECTORY (каталог)
    "O" = OFFLINE (автономный)
    "C" = COMPRESSED (сжатый средствами NTFS, не ZIP)
    "T" = TEMPORARY (временный)
    "X" = EFS ENCRYPTION (зашифрован EFS)
    "J" = JOIN FOLDER (точка соединения, создана через FileCreateNTFSLink())

См. также

FileExists, FileGetSize, FileGetTime, FileSetAttrib, FileSetTime, FileCreateNTFSLink

Пример

Пример 1

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

Example()

Func Example()
    ; Создаём локальную константу для хранения атрибутов текущего скрипта.
    Local Const $sAttribute = FileGetAttrib(@ScriptFullPath)

    ; Выводим строку, возвращённую FileGetAttrib.
    MsgBox($MB_SYSTEMMODAL, "", "The attribute string: " & @CRLF & $sAttribute)

    ; Выводим строку, возвращённую AttributeToString.
    MsgBox($MB_SYSTEMMODAL, "", "The attribute string with easier to understand values: " & @CRLF & _
            AttributeToString($sAttribute))
EndFunc   ;==>Example

Func AttributeToString($sAttribute)
    ; Создаём одномерный массив букв атрибутов, разбив строку по запятой (,).
    Local $aInput = StringSplit("R,A,S,H,N,D,O,C,T,J", ",")

    ; Создаём одномерный массив понятных имён атрибутов, разбив строку по запятой (,).
    Local $aOutput = StringSplit("Read-only /, Archive /, System /, Hidden /" & _
            ", Normal /, Directory /, Offline /, Compressed /, Temporary /, JoinFolder /", ",")

    ; Перебираем массив букв атрибутов и заменяем их на понятные названия (например, A → Archive, S → System).
    For $i = 1 To $aInput[0]
        $sAttribute = StringReplace($sAttribute, $aInput[$i], $aOutput[$i], 0, $STR_CASESENSE)
    Next

    ; Удаляем пробел и завершающий слеш.
    $sAttribute = StringTrimRight($sAttribute, 2)

    ; Возвращаем строку атрибутов.
    Return $sAttribute
EndFunc   ;==>AttributeToString

Пример 2

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Задаём путь для проверки – является ли он файлом.
    Local $sFilePath = @ScriptFullPath

    If IsFile($sFilePath) Then
        MsgBox($MB_SYSTEMMODAL, "", "The filepath is a file.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The filepath is not a file.")
    EndIf
EndFunc   ;==>Example

; Проверяем, является ли путь файлом. Существование файла не проверяется.
Func IsFile($sFilePath)
    Return StringInStr(FileGetAttrib($sFilePath), "D") = 0
EndFunc   ;==>IsFile

#include <MsgBoxConstants.au3>

Func Example()
    ; Задаём путь для проверки – является ли он каталогом.
    Local $sFilePath = @ScriptDir

    If IsDir($sFilePath) Then
        MsgBox($MB_SYSTEMMODAL, "", "The filepath is a directory/folder.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The filepath is not a directory/folder.")
    EndIf
EndFunc   ;==>Example

; Проверяем, является ли путь каталогом. Существование каталога не проверяется.
Func IsDir($sFilePath)
    Return StringInStr(FileGetAttrib($sFilePath), "D") > 0
EndFunc   ;==>IsDir

Пример 4

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создаём временный каталог.
    Local $sDirPath = @TempDir & "\TempDir"
    DirRemove($sDirPath)
    DirCreate($sDirPath)

    ; Создаём NTFS-ссылку на каталог документов.
    Local $sNTFSLink = $sDirPath & "\ExampleNTFSLinkToFolder"
    Local $iNTFSLink = FileCreateNTFSLink(@MyDocumentsDir, $sNTFSLink)

    ; Открываем Проводник для проверки созданной NTFS-ссылки.
    Local $hWin = 0
    If $iNTFSLink Then
        ShellExecute($sDirPath)
        $hWin = WinWaitActive("[CLASS:CabinetWClass]", "") ; Ждём активации окна Проводника.
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The NTFS link wasn't created." & @CRLF & "FileCreateNTFSLink returned: " & $iNTFSLink)
    EndIf

    If IsNTFSLink($sNTFSLink) Then
        MsgBox($MB_SYSTEMMODAL, "", "The dirpath is a NTFSLink to a folder." & @CRLF & @CRLF & _
                "You can click on it.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The dirpath is not a NTFSLink to a folder.")
    EndIf

    ; Удаляем временные каталоги.
    DirRemove($sNTFSLink)
    DirRemove($sDirPath)

    ; Закрываем окна Проводника.
    If $hWin Then WinKill($hWin, "")

EndFunc   ;==>Example

; Проверяем, является ли путь NTFS-ссылкой на папку.
Func IsNTFSLink($sDirPath)
    Return StringInStr(FileGetAttrib($sDirPath), "J") > 0
EndFunc   ;==>IsNTFSLink
