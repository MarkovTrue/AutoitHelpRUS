Справочник функций

StringToBinary

Преобразует строку в двоичные данные.

StringToBinary ( expression [, flag = 1] )

Параметры

expression Выражение, преобразуемое в двоичные данные.
flag [необязательный] Определяет кодировку двоичного представления строки:
$SB_ANSI (1) = ANSI (по умолчанию)
$SB_UTF16LE (2) = UTF-16 Little Endian
$SB_UTF16BE (3) = UTF-16 Big Endian
$SB_UTF8 (4) = UTF-8

Константы определены в StringConstants.au3.

Возвращаемое значение

Возвращает значение типа Binary.

Примечания

Подробное описание см. в разделе "Поддержка Unicode".

См. также

Binary, BinaryToString, IsBinary, String

Пример

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

Example()

Func Example()
    ; Определить строку, которая будет преобразована позже.
    ; ПРИМЕЧАНИЕ: эта строка может отображаться как ?? в справке и в некоторых редакторах.
    ; Этот пример сохранён в UTF-8 с BOM. Должен корректно отображаться в редакторах
    ; которые поддерживают смену кодовых страниц на основе BOM.
    Local Const $sString = "Hello - 你好"

    ; Временные переменные для хранения результатов преобразования. $dBinary будет содержать
    ; исходная строка в бинарном виде, а $sConverted будет содержать результат
    ; после преобразования обратно в исходный формат.
    Local $dBinary = Binary(""), $sConverted = ""

    ; Преобразовать исходную UTF-8 строку в ANSI-совместимую бинарную строку.
    $dBinary = StringToBinary($sString)

    ; Преобразовать ANSI-совместимые двоичные данные обратно в строку.
    $sConverted = BinaryToString($dBinary)

    ; Показать результаты. Обратите внимание, что последние два символа будут отображены
    ; как ??, поскольку их нельзя представить в ANSI.
    DisplayResults($sString, $dBinary, $sConverted, "ANSI")

    ; Преобразовать исходную UTF-8 строку в UTF16-LE бинарную строку.
    $dBinary = StringToBinary($sString, $SB_UTF16LE)

    ; Преобразовать UTF16-LE бинарную строку обратно в строку.
    $sConverted = BinaryToString($dBinary, $SB_UTF16LE)

    ; Показать результаты.
    DisplayResults($sString, $dBinary, $sConverted, "UTF16-LE")

    ; Преобразовать исходную UTF-8 строку в UTF16-BE бинарную строку.
    $dBinary = StringToBinary($sString, $SB_UTF16BE)

    ; Преобразовать UTF16-BE бинарную строку обратно в строку.
    $sConverted = BinaryToString($dBinary, $SB_UTF16BE)

    ; Показать результаты.
    DisplayResults($sString, $dBinary, $sConverted, "UTF16-BE")

    ; Преобразовать исходную UTF-8 строку в UTF-8 бинарную строку.
    $dBinary = StringToBinary($sString, $SB_UTF8)

    ; Преобразовать UTF8 бинарную строку обратно в строку.
    $sConverted = BinaryToString($dBinary, $SB_UTF8)

    ; Показать результаты.
    DisplayResults($sString, $dBinary, $sConverted, "UTF8")
EndFunc   ;==>Example

; Вспомогательная функция, форматирующая сообщение для показа. Принимает следующие параметры:
; $sOriginal – исходная строка до преобразований.
; $dBinary – исходная строка после преобразования в бинарную.
; $sConverted – строка после преобразования в бинарную и обратно в строку.
; $sConversionType – человекочитаемое имя типа кодировки, использованной для преобразования.
Func DisplayResults($sOriginal, $dBinary, $sConverted, $sConversionType)
    MsgBox($MB_SYSTEMMODAL, "", "Original:" & @CRLF & $sOriginal & @CRLF & @CRLF & "Binary:" & @CRLF & $dBinary & @CRLF & @CRLF & $sConversionType & ":" & @CRLF & $sConverted)
EndFunc   ;==>DisplayResults
