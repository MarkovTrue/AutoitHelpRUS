UDPBind

Создаёт сокет, привязанный к входящему соединению.

UDPBind ( IPAddr, port )

Параметры

IPAddr IPv4-адрес в точечной записи, например "192.162.1.1".
port Порт, к которому будет привязан создаваемый сокет.

Возвращаемое значение

Успех: Возвращает массив: $aArray[1] содержит реальный сокет, $aArray[2] – указанный IP-адрес, $aArray[3] – порт. Эта информация необходима для последующих вызовов UDPRecv(), которым передаётся данная структура/массив сокета.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – неверный IPAddr.
2 – неверный порт.
Код возврата Windows API WSAGetLastError (см. MSDN).

См. также

UDPCloseSocket, UDPOpen, UDPRecv, UDPSend

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    UDPStartup() ; Запустить UDP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальным переменным loopback-адрес IP и порт.
    Local $sIPAddress = "127.0.0.1" ; Этот IP-адрес работает только для тестирования на локальном компьютере.
    Local $iPort = 65532 ; Порт, используемый для подключения.

    ; Присвоить локальной переменной сокет и привязать к указанным IP-адресу и порту.
    Local $aSocket = UDPBind($sIPAddress, $iPort)

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        ; Возможно, кто-то уже привязан к этому IP-адресу и порту (скрипт уже запущен?).
        Local $iError = @error
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Could not bind, Error code: " & $iError)
        Return False
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Bind successful.")
    EndIf

    ; Закрыть слушающий сокет, чтобы разрешить дальнейшие привязки.
    ; Пока не закрыто, другие программы не могут привязаться к тому же IP-адресу и порту.
    UDPCloseSocket($aSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    UDPShutdown() ; Закрыть UDP-сервис.
EndFunc   ;==>OnAutoItExit
