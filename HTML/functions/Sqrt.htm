Справочник функций

Sqrt

Вычисляет квадратный корень из числа.

Sqrt ( expression )

Параметры

expression Любое неотрицательное выражение, из которого извлекается квадратный корень.

Возвращаемое значение

Успех: Возвращает квадратный корень.
Ошибка: Возвращает -1.#IND, если параметр отрицательный.

Примечания

Для вычисления корня n-й степени используйте оператор возведения в степень: x ^ (1/n)
MsgBox(4096, "Кубический корень из 27 равен", 27 ^ (1/3))

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной квадратный корень из 2.
    Local $fSqrt1 = Sqrt(2)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fSqrt1)

    ; Присвоить локальной переменной квадратный корень из 9.
    Local $fSqrt2 = Sqrt(9)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fSqrt2)
EndFunc   ;==>Example
