Справочник функций

StringSplit

Разбивает строку на подстроки по заданным разделителям.

StringSplit ( "string", "delimiters" [, flag = 0] )

Параметры

string Оцениваемая строка.
delimiters Один или несколько символов-разделителей (с учётом регистра).
flag [необязательный] Изменяет режим разбиения; для нескольких режимов значения флагов складываются:
$STR_CHRSPLIT (0) = каждый символ строки разделителей является точкой разбиения (по умолчанию)
$STR_ENTIRESPLIT (1) = для разбиения требуется совпадение всей строки разделителей
$STR_NOCOUNT (2) = не помещать счётчик в первый элемент – массив фактически становится 0-базированным (для получения размера массива используйте UBound()).

Константы определены в StringConstants.au3.

Возвращаемое значение

Возвращает массив: по умолчанию первый элемент ($aArray[0]) содержит количество возвращённых строк, остальные элементы ($aArray[1], $aArray[2] и т.д.) содержат разделённые подстроки. Если параметр flag равен $STR_NOCOUNT (2), счётчик в первом элементе не возвращается.

Если разделители не найдены, @error устанавливается в 1:
    Если flag не равен $STR_NOCOUNT, $aArray[0] устанавливается в 1, а полная строка возвращается в $aArray[1].
    Если flag равен $STR_NOCOUNT, полная строка возвращается в $aArray[0].

Примечания

Если в качестве разделителя задана пустая строка "", каждый символ возвращается как отдельный элемент.

Если строка разделителей содержит несколько символов, поведение функции зависит от флага. При $STR_CHRSPLIT строка разбивается по каждому вхождению каждого символа разделителя; при $STR_ENTIRESPLIT – только при совпадении всей строки разделителей. См. второй пример ниже.

Обратите внимание: макрос @CRLF фактически является двухсимвольной строкой, поэтому без флага $STR_ENTIRESPLIT могут появиться лишние пустые строки, так как разбиение происходит и по @CR, и по @LF.

StringSplit() очень удобна как альтернатива StringInStr() и как средство заполнения массива.

См. также

StringInStr, StringLeft, StringLen, StringLower, StringMid, StringRight, StringTrimLeft, StringTrimRight, StringUpper

Пример

Пример 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aDays = StringSplit("Mon,Tues,Wed,Thur,Fri,Sat,Sun", ",") ; Разбить строку с названиями дней по разделителю "," со значением флага по умолчанию.
    #cs
Возвращаемый массив будет содержать следующие значения:
$aDays[1] = "Пн"
$aDays[2] = "Вт"
$aDays[3] = "Ср"
...
$aDays[7] = "Вс"
#ce

    For $i = 1 To $aDays[0] ; Перебрать массив, возвращённый StringSplit, и показать каждое значение.
        MsgBox($MB_SYSTEMMODAL, "", "$aDays[" & $i & "] - " & $aDays[$i])
    Next
EndFunc   ;==>Example

Пример 2

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

Example()

Func Example()
    Local $sText = "This\nline\ncontains\nC-style breaks." ; Определить переменную с текстовой строкой.
    Local $aArray = StringSplit($sText, '\n', $STR_ENTIRESPLIT) ; Передать переменную в StringSplit с разделителем "\n".
    ; Параметр flag задан как $STR_ENTIRESPLIT (1), иначе разбиение произойдёт по \ или n.
    #cs
Возвращаемый массив будет содержать следующие значения:
$aArray[1] = "Это"
$aArray[2] = "строка"
...
$aArray[4] = "переносы в стиле C."
#ce

    For $i = 1 To $aArray[0] ; Перебрать массив, возвращённый StringSplit, и показать каждое значение.
        MsgBox($MB_SYSTEMMODAL, "", "$aArray[" & $i & "] - " & $aArray[$i])
    Next
EndFunc   ;==>Example

Пример 3

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

Example()

Func Example()
    Local $sText = "This\nline\ncontains\nC-style breaks." ; Определить переменную с текстовой строкой.

    ; Передать переменную в StringSplit с разделителем "\n".
    ; Параметр flag задан как $STR_ENTIRESPLIT (1), иначе разбиение произойдёт по \ или n.
    MsgBox($MB_SYSTEMMODAL, "", StringSplit($sText, '\n', $STR_ENTIRESPLIT)[2]) ; Прямой доступ к индексу массива через обращение к выражению.
    #cs
Внутренний временный массив используется для возврата строки, которая может содержать одно из следующих значений:
$aArray[1] = "Это"
$aArray[2] = "строка"
...
$aArray[4] = "переносы в стиле C."
#ce
EndFunc   ;==>Example
