DriveGetSerial

Возвращает серийный номер диска.

DriveGetSerial ( "path" )

Параметры

path Путь к диску, с которого получается информация.

Возвращаемое значение

Успех: Возвращает серийный номер диска в виде строки.
Ошибка: Устанавливает значение @error = 1.

Примечания

Возвращаемое значение – не аппаратный серийный номер с этикетки диска, а идентификатор тома Windows (Volume ID).

См. также

DriveGetDrive, DriveGetFileSystem, DriveGetLabel, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

Пример

#include <MsgBoxConstants.au3>

Local $sSerial = DriveGetSerial(@HomeDrive & "\") ; Определить серийный номер домашнего диска, как правило, это диск C:\.
MsgBox($MB_SYSTEMMODAL, "", "Serial Number: " & $sSerial)
