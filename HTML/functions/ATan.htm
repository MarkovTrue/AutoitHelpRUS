Справочник функций

ATan

Вычисляет арктангенс числа.

ATan ( expression )

Параметры

expression Любое допустимое числовое выражение.

Возвращаемое значение

Возвращает тригонометрический арктангенс числа. Результат в радианах.

Примечания

Результат выражения 4 * ATan(1) равен числу Pi.

См. также

ACos, ASin, Cos, Sin, Tan

Пример

Пример 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной арктангенс 0.5.
    Local $fArcTan1 = ATan(0.5)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fArcTan1 & " rad.")

    ; Присвоить локальной константе приближённое значение числа Пи.
    Local Const $PI = 3.141592653589793

    ; Присвоить локальной переменной формулу перевода радиан в градусы (равно одному радиану в градусах).
    Local $fRadToDeg = 180 / $PI

    ; Присвоить локальной переменной число в градусах.
    Local $fArcTan2 = $fRadToDeg * ATan(1)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fArcTan2 & " deg.")
EndFunc   ;==>Example
