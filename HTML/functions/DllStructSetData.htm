Справочник функций

DllStructSetData

Устанавливает данные элемента структуры.

DllStructSetData ( Struct, Element, value [, index] )

Параметры

Struct Структура, возвращаемая функцией DllStructCreate().
Element Элемент структуры, к которому нужно обратиться: номер, начиная с 1, или имя элемента, определённое в DllStructCreate().
value Новое значение для помещения в элемент структуры.
index [необязательный] Для элементов-массивов задаёт индекс (начиная с 1) для установки значения. Если параметр опущен или равен ключевому слову Default, будет установлено столько значения, сколько возможно, начиная с индекса 1 (удобно для быстрой установки строк). Не используется для элементов, не являющихся массивами.

Возвращаемое значение

Успех: Возвращает значение, считанное обратно из структуры.
Ошибка: Возвращает 0.
@error: 1 – структура не соответствует структуре, возвращенной DllStructCreate().
2 – элемент выходит за пределы диапазона или неизвестен.
3 – индекс выходит за пределы диапазона.
4 – неизвестный тип данных элемента
5 – индекс <=0.

Примечания

Если элемент имеет тип char[n], wchar[n] или byte[n], данные могут быть строкой; byte[n] также принимает двоичные данные. В остальных случаях данные должны быть либо типом AutoIt, совместимым с типом элемента, либо числом, которое будет преобразовано в нужный тип.
Нативные строки AutoIt неявно преобразуются в ANSI перед копированием в элемент char[n] или byte[n]. При ожидании кодировки UTF-8 (например, в DllCall) необходимо самостоятельно конвертировать Unicode-строки.

См. также

DllStructCreate, DllStructGetData, DllStructGetSize

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    #cs Примечания:
Создать следующую структуру (на C):
struct {
int var1;
unsigned char var2;
unsigned int var3;
char var4[128];
};

Схема:
------------------------------------
\ int \ byte \ uint \ char \
\ var1 \ var2 \ var3 \ var4 \
------------------------------------
#ce Примечания:

    ; Присвоить локальной константе определение структуры (внимательно читайте примечания DllStructCreate).
    Local Const $tagSTRUCT1 = "struct;int var1;byte var2;uint var3;char var4[128];endstruct"

    ; Примечание: переменная-тег объявлена как Constant – её значение не меняется при выполнении скрипта.

    ; Присвоить локальной переменной структуру.
    Local $tSTRUCT1 = DllStructCreate($tagSTRUCT1)

    ; Если произошла ошибка, показать код ошибки и вернуть False.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error in DllStructCreate, Code: " & @error)
        Return False
    EndIf

    ; Задать данные элемента var1 (int) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var1", -1) ; Или 1 вместо "var1".

    ; Задать данные элемента var2 (byte) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, 2, 255) ; Или "var2" вместо 2.

    ; Задать данные элемента var3 (uint) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var3", -1) ; -1 (signed int) будет преобразовано в unsigned int.
    ; Или 3 вместо "var3".

    ; Задать данные элемента var4 (char) в $tSTRUCT1.
    DllStructSetData($tSTRUCT1, "var4", "Hello") ; Или 4 вместо "var4".

    ; Примечание: этот элемент может содержать до 128 символов.

    ; Изменить данные элемента var4 (char) в $tSTRUCT1 по индексу 1 массива char (индексы с 1).
    DllStructSetData($tSTRUCT1, "var4", Asc("h"), 1)

    ; Показать результаты.
    MsgBox($MB_SYSTEMMODAL, "", "Struct Size: " & DllStructGetSize($tSTRUCT1) & @CRLF & _
            "Struct pointer: " & DllStructGetPtr($tSTRUCT1) & @CRLF & _
            "Data:" & @CRLF & _
            DllStructGetData($tSTRUCT1, 1) & @CRLF & _ ; Или "var1" вместо 1.
            DllStructGetData($tSTRUCT1, "var2") & @CRLF & _ ; Или 2 вместо "var2".
            DllStructGetData($tSTRUCT1, 3) & @CRLF & _ ; Или "var3" вместо 3.
            DllStructGetData($tSTRUCT1, 4)) ; Или "var4" вместо 4.

    ; Освободить ресурсы, занятые структурой.
    $tSTRUCT1 = 0
EndFunc   ;==>Example
