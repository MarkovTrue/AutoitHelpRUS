Справочник функций: Timer and Delay

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
Sleep Приостанавливает выполнение скрипта.
TimerDiff Возвращает разницу во времени с момента предыдущего вызова TimerInit().
TimerInit Возвращает дескриптор, который передаётся в TimerDiff() для вычисления разницы в миллисекундах.
