UDPRecv

Получает данные из открытого сокета.

UDPRecv ( socketarray, maxlen [, flag = 0] )

Параметры

socketarray Сокет/массив, возвращённый функцией UDPBind().
maxlen Максимальное количество принимаемых символов.
flag [необязательный]
UDP_DATA_DEFAULT (0) – автоопределение двоичных/строковых данных.
UDP_DATA_BINARY (1) – возвращать двоичные данные.
$UDP_DATA_ARRAY (2) – возвращать массив: [0] данные, [1] IP-адрес источника, [2] порт источника.
Для обоих режимов используйте значение 3.

Константы определены в "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает двоичные данные или строку, отправленные открытым сокетом, либо массив при flag = 2 или 3.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@error: -1 – недействительный сокет.
-2 – нет соединения.
-3 или -4 – недействительный socketarray.
Код возврата Windows API WSAGetLastError (см. MSDN).

Примечания

Для обратной совместимости функция по умолчанию пытается вернуть строку. Если среди принятых данных есть нулевые байты (0x00), возвращается тип Binary.
Чтобы функция всегда возвращала двоичные данные (наиболее надёжный вариант), используйте параметр flag = 1.

См. также

BinaryLen, BinaryMid, UDPBind, UDPOpen, UDPSend

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

; Сначала нажмите "1. Server"
; Затем запустите второй экземпляр скрипта, выбрав "2. Клиент"

Example()

Func Example()
    UDPStartup() ; Запустить UDP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальным переменным loopback-адрес IP и порт.
    Local $sIPAddress = "127.0.0.1" ; Этот IP-адрес работает только для тестирования на локальном компьютере.
    Local $iPort = 65532 ; Порт, используемый для подключения.

    #Region GUI
    Local $sTitle = "UDP Start"
    Local $hGUI = GUICreate($sTitle, 250, 70)

    Local $idBtn_Server = GUICtrlCreateButton("1. Server", 65, 10, 130, 22)

    Local $idBtn_Client = GUICtrlCreateButton("2. Client", 65, 40, 130, 22)

    GUISetState(@SW_SHOW, $hGUI)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtn_Server
                WinSetTitle($sTitle, "", "UDP Server started")
                GUICtrlSetState($idBtn_Client, $GUI_HIDE)
                GUICtrlSetState($idBtn_Server, $GUI_DISABLE)
                If Not MyUDP_Server($sIPAddress, $iPort) Then ExitLoop
            Case $idBtn_Client
                WinSetTitle($sTitle, "", "UDP Client started")
                GUICtrlSetState($idBtn_Server, $GUI_HIDE)
                GUICtrlSetState($idBtn_Client, $GUI_DISABLE)
                If Not MyUDP_Client($sIPAddress, $iPort) Then ExitLoop
        EndSwitch

        Sleep(10)
    WEnd

    #EndRegion GUI
EndFunc   ;==>Example

Func MyUDP_Server($sIPAddress, $iPort)
    ; Присвоить локальной переменной сокет и привязать к указанным IP-адресу и порту.
    Local $iSocket = UDPBind($sIPAddress, $iPort)

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        ; Возможно, кто-то уже привязан к этому IP-адресу и порту (скрипт уже запущен?).
        Local $iError = @error
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Server:" & @CRLF & "Could not bind, Error code: " & $iError)
        Return False
    EndIf

    ; Присвоить локальной переменной значение для хранения полученных данных.
    Local $sReceived = ""

    Do
        ; Ожидаем строку "toto" или "tata" (см. пример UDPSend): длина 4 байта.
        $sReceived = UDPRecv($iSocket, 4)
    Until $sReceived <> ""

    ; Примечание: если не знаете заранее длину данных,
    ; используйте, например, 2048 для maxlen и вызывайте функцию, пока она не вернёт пустоту или ошибку.

    ; Показать полученную строку.
    MsgBox($MB_SYSTEMMODAL, "", "Server:" & @CRLF & "Received: " & $sReceived)

    ; Закрыть сокет.
    UDPCloseSocket($iSocket)
EndFunc   ;==>MyUDP_Server

Func MyUDP_Client($sIPAddress, $iPort)
    ; Присвоить локальной переменной сокет и подключиться к слушающему сокету по указанным IP и порту.
    Local $iSocket = UDPOpen($sIPAddress, $iPort)
    Local $iError = 0

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        ; Сервер, вероятно, офлайн, или порт на сервере не открыт.
        $iError = @error
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Client:" & @CRLF & "Could not connect, Error code: " & $iError)
        Return False
    EndIf

    ; Отправить строку "toto", преобразованную в бинарную, на сервер.
    UDPSend($iSocket, StringToBinary("toto"))

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        $iError = @error
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Client:" & @CRLF & "Could not send the data, Error code: " & $iError)
        Return False
    EndIf

    ; Закрыть сокет.
    UDPCloseSocket($iSocket)
EndFunc   ;==>MyUDP_Client

Func OnAutoItExit()
    UDPShutdown() ; Закрыть UDP-сервис.
EndFunc   ;==>OnAutoItExit
