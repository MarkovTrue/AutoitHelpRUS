Справочник функций

TrayItemSetState

Устанавливает состояние элемента меню трея.

TrayItemSetState ( controlID, state )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией TrayCreateItem или TrayCreateMenu.
state См. таблицу состояний ниже.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Таблица состояний

Состояние Значение Комментарий
No Change 0
$TRAY_CHECKED 1 Элемент меню будет отмечен
$TRAY_UNCHECKED 4 Отметка с элемента меню будет снята
$TRAY_ENABLE 64 Элемент меню будет активирован
$TRAY_DISABLE 128 Элемент меню будет заблокирован (серый)
$TRAY_FOCUS 256 Элемент меню будет выделен
$TRAY_DEFAULT 512 Элемент меню станет элементом по умолчанию

Значения состояний можно складывать: например, $TRAY_CHECKED + $TRAY_DEFAULT устанавливает элемент как отмеченный и выбранный по умолчанию.

Чтобы сбросить состояние $TRAY_DEFAULT для элемента, вызовите функцию с любым другим состоянием, например $TRAY_ENABLE.

При установке $TRAY_DEFAULT двойной щелчок основной кнопкой по значку трея по умолчанию выполняет этот элемент. Чтобы отключить это поведение, используйте Opt("TrayMenuMode", 4).

Вышеуказанные константы определены в #include <TrayConstants.au3>.

См. также

TrayCreateItem, TrayCreateMenu, TrayItemGetState, TraySetState

Пример

#NoTrayIcon

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>
#include <TrayConstants.au3> ; Нужно для констант $TRAY_CHECKED и $TRAY_ICONSTATE_SHOW.

Opt("TrayMenuMode", 3) ; Стандартные пункты меню трея не будут показаны, и пункты не помечаются при выборе. Это опции 1 и 2 для TrayMenuMode.

Example()

Func Example()
    Local $idSetState = TrayCreateItem("Set 'About' State")
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Показать окно сообщения с версией AutoIt и путём установки исполняемого файла AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", $STR_NOCASESENSEBASIC, -1) - 1)) ; Найти каталог полного пути.

            Case $idSetState
                ; Установить состояние элемента 'About' в "отмечен".
                TrayItemSetState($idAbout, $TRAY_CHECKED)

            Case $idExit ; Выйти из цикла.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
