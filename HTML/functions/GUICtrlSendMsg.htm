Справочник функций

GUICtrlSendMsg

Отправляет сообщение элементу управления.

GUICtrlSendMsg ( controlID, msg , wParam, lParam )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
msg Тип отправляемого сообщения в соответствии с документацией по элементам управления Windows.
wParam Первый параметр для отправки элементу управления.
lParam Второй параметр для отправки элементу управления.

Возвращаемое значение

Успех: Возвращает значение, возвращённое API SendMessage Windows.
Ошибка: Возвращает 0.

Примечания

Функция позволяет отправлять специальные сообщения Windows напрямую элементу управления через API SendMessage. Используется для задействования возможностей элементов управления, недоступных через стандартные GUICtrlRead() и GUICtrlUpdate...().

Параметры wParam и lParam могут быть целым числом или строкой.

GUICtrlSendMsg() следует использовать для сообщений, не требующих специального возвращаемого значения. Для более сложных сообщений, из которых необходимо получить дополнительные данные, используйте GUICtrlRecvMsg().

См. также

GUICtrlCreate..., GUICtrlRead, GUICtrlRecvMsg, GUICtrlUpdate..., GUIGetMsg

Пример

#include <GUIConstantsEx.au3>
#include <ProgressConstants.au3>

Example()

Func Example()
    GUICreate("Marquee Progress Bar", 290, 90, -1, -1) ; Пример запуска/остановки бегущей строки индикатора выполнения.
    Local $idProgress = GUICtrlCreateProgress(10, 10, 270, 20, $PBS_MARQUEE)
    Local $idBtn_Start = GUICtrlCreateButton("&Start", 10, 60, 70, 25)
    Local $idBtn_Stop = GUICtrlCreateButton("S&top", 85, 60, 70, 25)

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idBtn_Start
                GUICtrlSendMsg($idProgress, $PBM_SETMARQUEE, 1, 50) ; Отправить сообщение $PBM_SETMARQUEE с wParam=1 для запуска бегущей строки.

            Case $idBtn_Stop
                GUICtrlSendMsg($idProgress, $PBM_SETMARQUEE, 0, 50) ; Отправить сообщение $PBM_SETMARQUEE с wParam=0 для остановки бегущей строки.

        EndSwitch
    WEnd
EndFunc   ;==>Example
