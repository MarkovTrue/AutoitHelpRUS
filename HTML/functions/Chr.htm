Справочник функций

Chr

Возвращает символ, соответствующий ASCII-коду.

Chr ( ASCIIcode )

Параметры

ASCIIcode ASCII-код в диапазоне 0-255 (например, 65 возвращает заглавную букву A).

Возвращаемое значение

Успех: Возвращает строку с ASCII-представлением указанного кода.
Ошибка: Возвращает пустую строку и устанавливает значение @error = 1, если ASCIIcode больше 255.

Примечания

Полный список значений см. в таблице ASCII-коды символов.

Chr(48) == "0"
Chr(57) == "9"
Chr(65) == "A"
Chr(90) == "Z"
Chr(97) == "a"
Chr(122) == "z" и т. д.

См. также

Asc, AscW, ChrW, String

Пример

#include <MsgBoxConstants.au3>

Local $sText = ""
For $i = 65 To 90
    $sText = $sText & Chr($i) ; Также можно использовать $sText &= Chr($i).
Next
MsgBox($MB_SYSTEMMODAL, "Uppercase alphabet", $sText) ; Показать символы в диапазоне 65–90.
