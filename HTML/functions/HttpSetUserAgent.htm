Справочник функций

HttpSetUserAgent

Задаёт строку HTTP user-agent, отправляемую со всеми Inet-запросами.

HttpSetUserAgent ( "user agent" )

Параметры

user agent Строка, устанавливаемая в качестве user-agent. По умолчанию используется "AutoIt". Передача пустой строки сбрасывает user-agent к значению по умолчанию.

Возвращаемое значение

Возвращает предыдущий user-agent.

См. также

InetGet, InetGetSize, InetRead

Пример

HttpSetUserAgent("MyUserAgent")
