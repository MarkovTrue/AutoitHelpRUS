Справочник функций: Network

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
FtpSetProxy Задаёт интернет-прокси для FTP-доступа.
HttpSetProxy Задаёт интернет-прокси для HTTP-доступа.
HttpSetUserAgent Задаёт строку HTTP user-agent, отправляемую со всеми Inet-запросами.
InetClose Закрывает дескриптор, возвращённый функцией InetGet().
InetGet Загружает файл из интернета по протоколу HTTP, HTTPS или FTP.
InetGetInfo Возвращает подробные сведения для дескриптора, полученного от InetGet().
InetGetSize Возвращает размер (в байтах) файла, расположенного в интернете.
InetRead Загружает файл из интернета по протоколу HTTP, HTTPS или FTP.
Ping Отправляет ping на указанный хост и возвращает время прохождения пакета.
TCPAccept Принимает входящее соединение на сокете.
TCPCloseSocket Закрывает TCP-сокет.
TCPConnect Создаёт сокет, подключённый к существующему серверу.
TCPListen Создаёт сокет, ожидающий входящих подключений.
TCPNameToIP Преобразует доменное имя в IP-адрес.
TCPRecv Получает данные из подключённого сокета.
TCPSend Отправляет данные через подключённый сокет.
TCPShutdown, UDPShutdown Stops TCP/UDP services.
TCPStartup, UDPStartup Starts TCP or UDP services.
UDPBind Создаёт сокет, привязанный к входящему соединению.
UDPCloseSocket Закрывает UDP-сокет.
UDPOpen Открывает сокет, подключённый к существующему серверу.
UDPRecv Получает данные из открытого сокета.
UDPSend Отправляет данные через открытый сокет.
