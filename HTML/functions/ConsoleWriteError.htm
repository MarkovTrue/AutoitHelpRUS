Справочник функций

ConsoleWriteError

Записывает данные в поток STDERR. Некоторые текстовые редакторы и сторонние программы могут читать этот поток.

ConsoleWriteError ( "data" )

Параметры

data Данные для вывода. Могут быть текстовыми или двоичными.

Возвращаемое значение

Возвращает объём записанных данных. При записи двоичных данных – число байтов, при записи текста – число символов.

Примечания

Назначение этой функции – запись в поток STDERR. Многие популярные текстовые редакторы умеют читать этот поток. У скриптов, скомпилированных как консольные приложения, также есть поток STDERR.

Вывод не поступает в DOS-консоль, если только скрипт не скомпилирован как консольное приложение.

Символы перед записью преобразуются в ANSI.

Двоичные данные записываются как есть и не преобразуются в строку. Чтобы вывести шестнадцатеричное представление двоичных данных, используйте String() для явного приведения к строке.

@error и @extended не изменяются при возврате – сохраняются их значения до вызова. Полезно при отладке через вывод SciTE.

См. также

ConsoleRead, ConsoleWrite

Пример

Local $sString = "An error occurred."
ConsoleWriteError($sString & @CRLF) ; Запуск в редакторе, перехватывающем консольный вывод (например, SciTE), покажет значение $sString.
