Справочник функций

IsAdmin

Проверяет, обладает ли текущий пользователь полными правами администратора.

IsAdmin ( )

Возвращаемое значение

Успех: Возвращает 1, если текущий пользователь имеет права администратора.
Ошибка: Возвращает 0, если пользователь не имеет прав администратора.

Примечания

В Windows Vista возвращает 1 только при запуске с полным токеном администратора (то есть если применена директива #RequireAdmin или скрипт уже повышен через UAC).

Пример

#include <MsgBoxConstants.au3>

If IsAdmin() Then
    MsgBox($MB_SYSTEMMODAL, "", "IsAdmin" & @CRLF & "Admin rights are detected.")
EndIf
