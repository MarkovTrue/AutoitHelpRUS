Справочник функций

WinGetState

Получает состояние заданного окна.

WinGetState ( "title" [, "text"] )

Параметры

title Заголовок / дескриптор / класс окна, состояние которого нужно получить. См. Заголовки и текст окон (расширенно).
text [необязательный] Текст окна, состояние которого нужно получить. См. Заголовки и текст окон (основы).

Возвращаемое значение

Успех: Возвращает значение, обозначающее состояние окна. Значения складываются, поэтому используйте BitAND(), чтобы проверить интересующую часть:
$WIN_STATE_EXISTS (1) = окно существует
$WIN_STATE_VISIBLE (2) = окно видимо
$WIN_STATE_ENABLED (4) = окно доступно
$WIN_STATE_ACTIVE (8) = окно активно
$WIN_STATE_MINIMIZED (16) = окно свёрнуто
$WIN_STATE_MAXIMIZED (32) = окно развёрнуто.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0, если окно не найдено.

Константы определены в AutoItConstants.au3.

См. также

BitAND, WinGetPos, WinSetState

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Получаем состояние окна Блокнота по дескриптору, возвращённому WinWait.
    Local $iState = WinGetState($hWnd)

    ; Проверяем, свёрнуто ли окно Блокнота, и выводим соответствующее сообщение.
    If BitAND($iState, $WIN_STATE_MINIMIZED) Then
        MsgBox($MB_SYSTEMMODAL, "", "Notepad is minimized and the state returned by WinGetState was - " & $iState)
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Notepad isn't minimized and the state returned by WinGetState was - " & $iState)
    EndIf

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
