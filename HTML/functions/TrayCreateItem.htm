TrayCreateItem

Создаёт элемент меню в трее.

TrayCreateItem ( "text" [, menuID = -1 [, menuentry = -1 [, menuradioitem = 0]]] )

Параметры

text Текст элемента управления.
menuID [необязательный] Позволяет создать подменю в указанном меню. Значение -1 означает добавление после последнего созданного элемента (по умолчанию).
menuentry [необязательный] Позволяет задать порядковый номер создаваемого элемента. Нумерация начинается с 0. Значение -1 означает добавление после последнего созданного элемента (по умолчанию).
menuradioitem [необязательный]
$TRAY_ITEM_NORMAL (0) = (по умолчанию) создать обычный элемент меню.
$TRAY_ITEM_RADIO (1) = создать элемент-переключатель.

Константы определены в TrayConstants.au3.

Возвращаемое значение

Успех: Возвращает идентификатор (controlID) нового элемента меню трея.
Ошибка: Возвращает 0.

Примечания

Если параметр text – пустая строка ( "" ), создаётся разделительная линия.

По умолчанию обычный отмеченный элемент меню (не переключатель) при нажатии автоматически снимает отметку!
Чтобы отключить это поведение, используйте Opt("TrayMenuMode", 2).

Элементы-переключатели автоматически объединяются в группы, разделённые разделителем или обычным (не radio) элементом.
По умолчанию при нажатии на элемент-переключатель он отмечается, а все остальные в группе снимают отметку!
Чтобы отключить это поведение, используйте Opt("TrayMenuMode", 8).

См. также

TrayGetMsg, TrayItemDelete, TrayItemSetOnEvent, TrayItemSetState, TrayItemSetText

Пример

#NoTrayIcon

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>
#include <TrayConstants.au3> ; Нужно для констант $TRAY_CHECKED и $TRAY_ICONSTATE_SHOW.

Opt("TrayMenuMode", 3) ; Стандартные пункты меню трея не будут показаны, и пункты не помечаются при выборе. Это опции 1 и 2 для TrayMenuMode.

Example()

Func Example()
    ; Создать элемент трея с параметром radio item.
    TrayCreateItem("Radio 1", -1, -1, $TRAY_ITEM_RADIO)
    TrayItemSetState(-1, $TRAY_CHECKED)
    TrayCreateItem("Radio 2", -1, -1, $TRAY_ITEM_RADIO)
    TrayCreateItem("Radio 3", -1, -1, $TRAY_ITEM_RADIO)

    TrayCreateItem("") ; Создать разделительную линию.

    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Показать окно сообщения с версией AutoIt и путём установки исполняемого файла AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", $STR_NOCASESENSEBASIC, -1) - 1)) ; Найти каталог полного пути.

            Case $idExit ; Выйти из цикла.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
