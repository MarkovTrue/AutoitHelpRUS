AutoItWinSetTitle

Изменяет заголовок окна AutoIt.

AutoItWinSetTitle ( "newtitle" )

Параметры

newtitle Новый заголовок, который будет присвоен окну.

Возвращаемое значение

Отсутствует.

Примечания

Окно AutoIt обычно скрыто. Изменение заголовка нужно для того, чтобы другие программы (или другие скрипты AutoIt) могли взаимодействовать с AutoIt.

См. также

AutoItWinGetTitle, WinSetTitle

Пример

Пример 1

; Проверить, запущен ли скрипт уже
; Примечание: рекомендуемый подход – использовать _Singleton из Misc.au3

#include <MsgBoxConstants.au3>

Local $sMyAutoItTitle = "ThisIsSomeUniqueStringThatOtherWindowsWontHave"

If WinExists($sMyAutoItTitle) Then
    ; Скрипт уже запущен

    MsgBox($MB_SYSTEMMODAL, Default, "The script is already running." & @CRLF & @CRLF & _
            "It's PID is: " & WinGetProcess($sMyAutoItTitle))
Else
    ; Это первый экземпляр скрипта.

    ; Задать заголовок окна AutoIt
    AutoItWinSetTitle($sMyAutoItTitle)

    ; Снова запустить этот скрипт.
    If @Compiled Then
        Run('"' & @ScriptFullPath & '"')
    Else
        Run('"' & @AutoItExe & '" "' & @ScriptFullPath & '"')
    EndIf

    MsgBox($MB_SYSTEMMODAL, Default, "This is the first instance." & @CRLF & @CRLF & _
            "My PID is: " & @AutoItPID)
EndIf

Пример 2

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Задать заголовок скрытого окна AutoIt.
    AutoItWinSetTitle("My AutoIt Window")

    ; Показать скрытое окно AutoIt.
    AutoItWinShow()

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Показать скрытое окно AutoIt. Возвращает дескриптор окна.
Func AutoItWinShow()
    Local $hWnd = WinGetHandle(AutoItWinGetTitle()) ; Получить дескриптор скрытого окна AutoIt по его заголовку.
    WinMove($hWnd, "", (@DesktopWidth / 2) - 250, (@DesktopHeight / 2) - 250, 500, 500) ; Переместить и изменить размер скрытого окна AutoIt для удобного просмотра.
    WinSetState($hWnd, "", @SW_SHOW) ; Показать скрытое окно AutoIt – обычно оно скрыто, но в этом примере отображается.
    Return $hWnd
EndFunc   ;==>AutoItWinShow
