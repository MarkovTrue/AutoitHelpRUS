Справочник функций

AdlibRegister

Регистрирует Adlib-функцию.

AdlibRegister ( "function" [, time = 250] )

Параметры

function Имя регистрируемой adlib-функции.
time [необязательный] Как часто вызывать функцию, в миллисекундах. По умолчанию 250 мс.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Каждые 250 мс (или time мс) вызывается указанная function. Первый вызов произойдёт через заданный интервал, а не сразу после регистрации.
Обычно такие функции используют для отслеживания непредвиденных ошибок. Например, adlib может пригодиться в скрипте, где непредсказуемо всплывает окно ошибки.
Adlib-функцию стоит держать простой: она вызывается часто, и на это время основной скрипт приостановлен. Параметр time тоже подбирайте аккуратно, чтобы не нагружать процессор.
Функцию с параметрами зарегистрировать нельзя.

Можно зарегистрировать несколько Adlib-функций. Повторная регистрация существующей функции обновит её интервал.

См. также

AdlibUnRegister

Пример

#include <MsgBoxConstants.au3>

If ProcessExists("SciTE.exe") = 0 Then
    MsgBox($MB_SYSTEMMODAL, "", "You will need SciTE.exe to be running for ConsoleWrite to display.")
EndIf

Example()

Func Example()
    ; Зарегистрировать функцию MyAdLibFunc() для вызова каждые 250 мс (по умолчанию).
    AdlibRegister("MyAdLibFunc")

    ; Sleep не останавливает выполнение AdLib-функций.
    Sleep(1000)

    ; AdLib-функции не выполняются во время блокирующих функций (MsgBox, InputBox, WinWait, WinWaitClose и т. д.).
    MsgBox($MB_SYSTEMMODAL, "", "No console message(s) will be shown whilst the messagebox is displayed.")

    ; AdLib-функция MyAdLibFunc() снова начнёт вызываться.
    Sleep(2000)

    ; Отменить регистрацию функции MyAdLibFunc(), чтобы она больше не вызывалась каждые 250 мс.
    AdlibUnRegister("MyAdLibFunc")
EndFunc   ;==>Example

Func MyAdLibFunc()
    ; Объявить статическую переменную для хранения числа вызовов функции.
    Local Static $iCount = 0
    $iCount += 1

    ConsoleWrite("MyAdLibFunc called " & $iCount & " time(s)" & @CRLF)
EndFunc   ;==>MyAdLibFunc
