Справочник функций

GUICtrlCreateCombo

Создаёт элемент управления ComboBox (выпадающий список) для GUI.

GUICtrlCreateCombo ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

text Текст, отображаемый в элементе управления ComboBox.
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию – ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию – ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $CBS_DROPDOWN, $CBS_AUTOHSCROLL, $WS_VSCROLL
принудительный стиль : $WS_TABSTOP
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для получения значения элемента управления см. GUICtrlRead().
Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

В Windows XP/2003 система автоматически корректирует размер открытого списка. В других версиях Windows можно задать этот размер параметром height, если значение по умолчанию недостаточно велико для отображения хотя бы одной строки.

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_COMBO, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <ComboConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKHEIGHT.

См. также

GUICoordMode (Option), GUICtrlSetData, GUICtrlUpdate..., GUIGetMsg

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI с различными элементами управления.
    Local $hGUI = GUICreate("Example", 300, 200)

    ; Создаём элемент управления ComboBox.
    Local $idComboBox = GUICtrlCreateCombo("Item 1", 10, 10, 185, 20)
    Local $idButton_Close = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Добавляем дополнительные элементы в ComboBox.
    GUICtrlSetData($idComboBox, "Item 2|Item 3", "Item 2")

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    Local $sComboRead = ""

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idButton_Close
                ExitLoop

            Case $idComboBox
                $sComboRead = GUICtrlRead($idComboBox)
                MsgBox($MB_SYSTEMMODAL, "", "The combobox is currently displaying: " & $sComboRead, 0, $hGUI)

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI и все элементы управления.
    GUIDelete($hGUI)
EndFunc   ;==>Example
