GUICtrlSetTip

Устанавливает текст всплывающей подсказки для элемента управления.

GUICtrlSetTip ( controlID, tiptext [, "title" [, icon [, options]]] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
tiptext Текст подсказки, отображаемый при наведении курсора на элемент управления.
title [необязательный] Заголовок всплывающей подсказки.
icon [необязательный] Стандартный значок рядом с заголовком (требует наличия заголовка):
$TIP_NOICON (0) = Без значка
$TIP_INFOICON (1) = Информационный значок
$TIP_WARNINGICON (2) = Значок предупреждения
$TIP_ERRORICON (3) = Значок ошибки

Константы определены в "AutoItConstants.au3".
options [необязательный] Параметры отображения подсказки (можно суммировать):
$TIP_BALLOON (1) = Отображать в виде выноски.
$TIP_CENTER (2) = Выравнивать подсказку по горизонтальному центру элемента управления.

Константы определены в "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Текст подсказки отображается в прямоугольной области всплывающей подсказки.
Для пропуска необязательного параметра с сохранением значения по умолчанию используйте ключевое слово Default.
Для создания многострочных подсказок можно использовать @CR или @LF.
Для отображения значка необходимо указать непустой заголовок. Значок размещается в той же строке, что и заголовок, поэтому заголовок обязателен.

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI control tip") ; создаст диалоговое окно, отображаемое по центру

    GUICtrlCreateLabel("my label", 10, 20)
    GUICtrlSetTip(-1, "tip of my label")

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
