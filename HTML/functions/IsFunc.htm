Справочник функций

IsFunc

Проверяет, является ли переменная или выражение функцией.

IsFunc ( expression )

Параметры

expression Проверяемая переменная или выражение.

Возвращаемое значение

Успех: Возвращает 1, если переменная является пользовательской функцией; 2, если встроенной.
Ошибка: Возвращает 0, если выражение не является функцией.

Примечания

Подробное описание см. Типы данных.

См. также

Binary, BinaryToString, IsArray, IsBool, IsFloat, IsHWnd, IsInt, IsMap, IsNumber, IsString, StringToBinary, VarGetType

Пример

Пример 1

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать объект первого класса MsgBox и присвоить локальной переменной.
    Local $hMsgBox = MsgBox

    ; Показать MsgBox, используя ранее присвоенную переменную.
    $hMsgBox($MB_SYSTEMMODAL, "", "This is a sentence with whitespace.")

    ; Отобразить сообщение о том, что $hMsgBox является встроенной функцией.
    $hMsgBox($MB_SYSTEMMODAL, "", "Check if $hMsgBox is a function or not." & @CRLF & _
            @CRLF & _
            "This will return 2 as the function is native: " & IsFunc($hMsgBox))
EndFunc   ;==>Example

Пример 2

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать объект первого класса MsgBox и присвоить локальной переменной.
    Local $hMsgBox = MsgBox

    ; Показать MsgBox, используя ранее присвоенную переменную.
    $hMsgBox($MB_SYSTEMMODAL, "", "This is a sentence with whitespace.")

    ; Переприсвоить переменной $hMsgBox нашу пользовательскую функцию окна сообщения.
    $hMsgBox = MyMsgBox

    ; Отобразить сообщение о том, что $hMsgBox является функцией.
    $hMsgBox($MB_SYSTEMMODAL, "", "Check if $hMsgBox is a function or not." & @CRLF & _
            @CRLF & _
            "This will return 1 as the function is a user function: " & IsFunc($hMsgBox))
EndFunc   ;==>Example

; Пользовательская функция окна сообщения.
Func MyMsgBox($iFlag, $sTitle, $sText = "")
    Return MsgBox($iFlag, $sTitle, $sText)
EndFunc   ;==>MyMsgBox
