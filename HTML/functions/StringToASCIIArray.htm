Справочник функций

StringToASCIIArray

Преобразует строку в массив, содержащий ASCII-код каждого символа.

StringToASCIIArray ( "string",[start = 0 [, end [, encoding = 0]]] )

Параметры

"string" Строка, преобразуемая в массив ASCII-кодов.
start [необязательный] Начальная позиция обработки (отсчёт с 0 и по умолчанию 0).
end [необязательный] Конечная позиция обработки (отсчёт с 0, по умолчанию StringLen("string")).
encoding [необязательный] Возвращаемый массив будет содержать значения в указанной кодировке:
$SE_UTF16 (0) = UTF-16 (по умолчанию)
$SE_ANSI (1) = ANSI
$SE_UTF8 (2) = UTF-8

Константы определены в StringConstants.au3.

Возвращаемое значение

Успех: Возвращает массив, в котором каждый элемент это код символа Unicode на соответствующей позиции.
Ошибка: Возвращает пустую строку.

Примечания

Строка может содержать встроенные символы Chr(0). Они и все данные после них войдут в возвращаемый массив. Обработка останавливается только при достижении конца строки или указанной пользователем конечной позиции.

Чтобы преобразовать двоичные данные в массив с помощью этой функции, их необходимо сначала перевести в строку функцией BinaryToString().

См. также

BinaryToString, StringFromASCIIArray

Пример

#include <Array.au3> ; Нужно только для _ArrayDisplay().
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Преобразовать строку в массив ASCII.
    Local $aArray = StringToASCIIArray("This is a sentence with whitespace.")

    ; Показать массив, чтобы убедиться, что он содержит ASCII-значения каждого символа строки.
    _ArrayDisplay($aArray)

    ; Преобразовать массив в строку.
    Local $sString = StringFromASCIIArray($aArray)

    ; Показать строку, чтобы убедиться, что она совпадает с исходной до преобразования в массив.
    MsgBox($MB_SYSTEMMODAL, "", $sString)
EndFunc   ;==>Example
