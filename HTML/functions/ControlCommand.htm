Справочник функций

ControlCommand

Отправляет команду элементу управления.

ControlCommand ( "title", "text", controlID, "command" [, "option"] )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.
command Команда, отправляемая элементу управления.
option [необязательный] Дополнительный параметр, необходимый для некоторых команд.

Возвращаемое значение

Возвращаемое значение зависит от команды (см. таблицу ниже). В случае ошибки (например, задана недопустимая команда или окно/элемент управления) устанавливается @error=1.

Команда, параметр Возвращаемое значение
"IsVisible", "" Возвращает 1, если элемент управления видим; иначе 0.
"IsEnabled", "" Возвращает 1, если элемент управления активен; иначе 0.
"ShowDropDown", "" Открывает выпадающий список ComboBox.
"HideDropDown", "" Скрывает выпадающий список ComboBox.
"AddString", 'string' Добавляет строку в конец списка ListBox или ComboBox.
"DelString", occurrence Удаляет строку по её порядковому номеру в ListBox или ComboBox.
"FindString", 'string' Возвращает порядковый номер строки, точно совпадающей со string, в ListBox или ComboBox.
"GetCount", "" Возвращает количество элементов в ListBox или ComboBox.
"SetCurrentSelection", occurrence Выделяет элемент с указанным порядковым номером в ListBox или ComboBox.
"SelectString", 'string' Выделяет элемент, совпадающий со string, в ListBox или ComboBox.
"IsChecked", "" Возвращает 1, если кнопка (Button) отмечена; иначе 0.
"Check", "" Отмечает радиокнопку или флажок (Button).
"UnCheck", "" Снимает отметку с радиокнопки или флажка (Button).
"GetCurrentLine", "" Возвращает номер строки, в которой находится каретка, для Edit.
"GetCurrentCol", "" Возвращает номер колонки, в которой находится каретка, для Edit.
"GetCurrentSelection", "" Возвращает имя выделенного элемента в ListBox или ComboBox.
"GetLineCount", "" Возвращает количество строк в Edit.
"GetLine", line# Возвращает текст строки с указанным номером в Edit.
"GetSelected", "" Возвращает выделенный текст Edit.
"EditPaste", 'string' Вставляет строку string в позицию каретки Edit.
"CurrentTab", "" Возвращает номер текущей активной вкладки SysTabControl32.
"TabRight", "" Переходит к следующей вкладке справа в SysTabControl32.
"TabLeft", "" Переходит к следующей вкладке слева в SysTabControl32.
"SendCommandID", Command ID Имитирует сообщение WM_COMMAND. Обычно используется для элементов управления ToolbarWindow32 – идентификатор команды (Command ID) можно получить на вкладке ToolBar утилиты Au3Info.

Примечания

Некоторые элементы управления не поддаются автоматизации, пока их окно не активно. Используйте WinActivate(), чтобы вывести окно элемента на передний план перед вызовом ControlCommand().

Некоторые команды, работающие в обычных Combo и ListBox, не работают с элементами "ComboLBox".

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Отправить edit-элементу Блокнота команду для получения количества строк. Дескриптор, возвращённый WinWait, используется в параметре "title" функции ControlCommand.
    Local $iCount = ControlCommand($hWnd, "", ControlGetFocus($hWnd), "GetLineCount", "")

    ; Показать количество строк.
    MsgBox($MB_SYSTEMMODAL, "", "The number of lines in Notepad are: " & $iCount)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
