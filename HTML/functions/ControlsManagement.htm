Элементы управления

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
ControlClick Отправляет команду клика мыши указанному элементу управления.
ControlCommand Отправляет команду элементу управления.
ControlDisable Отключает или "обесцвечивает" элемент управления.
ControlEnable Включает "обесцвеченный" элемент управления.
ControlFocus Устанавливает фокус ввода на указанный элемент управления окна.
ControlGetFocus Возвращает ControlRef# элемента управления, который имеет фокус клавиатуры в указанном окне.
ControlGetHandle Возвращает внутренний дескриптор элемента управления.
ControlGetPos Возвращает позицию и размер элемента управления относительно его окна.
ControlGetText Возвращает текст из элемента управления.
ControlHide Скрывает элемент управления.
ControlListView Отправляет команду элементу управления ListView32.
ControlMove Перемещает элемент управления в пределах окна.
ControlSend Отправляет строку символов элементу управления.
ControlSetText Устанавливает текст элемента управления.
ControlShow Отображает скрытый элемент управления.
ControlTreeView Отправляет команду элементу управления TreeView32.
StatusbarGetText Считывает текст из стандартного элемента управления строкой состояния.
