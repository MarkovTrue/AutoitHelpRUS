Справочник функций

GUICtrlSetStyle

Изменяет стиль элемента управления.

GUICtrlSetStyle ( controlID, style [, exStyle] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
style Задаёт стиль элемента управления. См. приложение по стилям элементов управления GUI.
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Некоторые стили нельзя изменять динамически. Обратитесь к документации MSDN. Пример такого стиля: $CBS_UPPERCASE для Combo.

См. также

GUICtrlCreate...

Пример

#include <GUIConstantsEx.au3>
#include <StaticConstants.au3>

Example()

Func Example()
    GUICreate("My GUI style") ; создаст диалоговое окно, отображаемое по центру

    GUICtrlCreateLabel("my label which will split on several lines", 10, 20, 100, 100)
    GUICtrlSetStyle(-1, $SS_RIGHT)

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
