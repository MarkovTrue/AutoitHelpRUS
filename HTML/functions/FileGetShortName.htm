FileGetShortName

Возвращает короткий путь и имя файла в формате 8.3 для переданного пути.

FileGetShortName ( "filename" [, flag = 0] )

Параметры

filename полный путь и имя файла для преобразования.
flag [необязательный]
$FN_FULLPATH (0) – по умолчанию
$FN_RELATIVEPATH (1) – файл может содержать относительный путь, например "..\file.txt"

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает короткий путь и имя файла в формате 8.3 для переданного пути.
Ошибка: значение переданного параметра и устанавливает значение @error = 1.

Примечания

Файл должен существовать, так как невозможно определить точный номер ~i, если несколько файлов имеют одинаковые первые 8 символов в имени.

См. также

FileGetLongName

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создаём локальную константу с путём, для которого нужно получить короткое имя.
    Local Const $sFilePath = @ProgramFilesDir

    ; Выводим короткий путь к каталогу Program Files.
    MsgBox($MB_SYSTEMMODAL, "", FileGetShortName($sFilePath))
EndFunc   ;==>Example
