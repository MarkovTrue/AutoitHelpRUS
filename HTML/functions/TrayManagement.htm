Работа с треем

Ниже приведён полный список функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

 

Функция Описание
TrayCreateItem Создаёт элемент меню в трее.
TrayCreateMenu Создаёт элемент меню (подменю) в меню трея.
TrayGetMsg Опрашивает трей на наличие произошедших событий.
TrayItemDelete Удаляет элемент управления (меню/пункт) из меню трея.
TrayItemGetHandle Возвращает дескриптор элемента меню трея.
TrayItemGetState Возвращает текущее состояние элемента управления.
TrayItemGetText Возвращает текст элемента меню трея.
TrayItemSetOnEvent Задаёт пользовательскую функцию, вызываемую при щелчке по элементу трея.
TrayItemSetState Устанавливает состояние элемента меню трея.
TrayItemSetText Устанавливает текст элемента меню трея.
TraySetClick Устанавливает режим кликов по значку трея – какие нажатия мыши отображают меню трея.
TraySetIcon Загружает/устанавливает указанный значок трея.
TraySetOnEvent Задаёт пользовательскую функцию, вызываемую при наступлении специального события трея.
TraySetPauseIcon Загружает/устанавливает указанный значок паузы для трея.
TraySetState Устанавливает состояние значка трея.
TraySetToolTip Устанавливает (или сбрасывает) текст всплывающей подсказки для значка трея.
TrayTip Отображает всплывающую подсказку в виде воздушного шарика от значка AutoIt.
