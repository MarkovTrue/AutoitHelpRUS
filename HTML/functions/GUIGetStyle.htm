GUIGetStyle

Возвращает стили окна GUI.

GUIGetStyle ( [winhandle] )

Параметры

winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает массив из двух элементов с информацией о стилях:
$aArray[0] = Style (стиль)
$aArray[1] = ExStyle (расширенный стиль)
Ошибка: Устанавливает @error ≠ 0.

Примечания

Обратите внимание: стиль меняется после вызова GUISetState().

См. также

GUICreate, GUISetStyle

Пример

#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()

    Local $hGUI = GUICreate("Gui Style", 260, 100)
    Local $idBtn_Style = GUICtrlCreateButton("Set Style", 45, 50, 150, 20)

    Local $aGUIStyle = GUIGetStyle($hGUI) ; учтите, что стиль меняется после открытия

    GUISetState(@SW_SHOW)

    Local $bNewStyle = False, $idMsg
    While 1
        $idMsg = GUIGetMsg()
        Switch $idMsg
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idBtn_Style
                If Not $bNewStyle Then
                    GUISetStyle(BitOR($WS_POPUPWINDOW, $WS_THICKFRAME), BitOR($WS_EX_CLIENTEDGE, $WS_EX_TOOLWINDOW))
                    GUICtrlSetData($idBtn_Style, 'Undo Style')
                    $bNewStyle = True
                Else
                    GUISetStyle($aGUIStyle[0], $aGUIStyle[1])
                    GUICtrlSetData($idBtn_Style, 'Set Style')
                    $bNewStyle = False
                EndIf
            Case Else
        EndSwitch
    WEnd

    GUIDelete($hGUI)
EndFunc   ;==>Example
