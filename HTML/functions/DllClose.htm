Справочник функций

DllClose

Закрывает ранее открытую DLL.

DllClose ( dllhandle )

Параметры

dllhandle Дескриптор DLL, возвращённый предыдущим вызовом DllOpen().

Возвращаемое значение

Отсутствует.

Примечания

При завершении AutoIt автоматически закрывает все открытые им DLL, однако явный вызов DllClose() является хорошей практикой.

См. также

DllCall, DllOpen

Пример

Local $hDLL = DllOpen("user32.dll")
DllCall($hDLL, "int", "MessageBox", "hwnd", 0, "str", "Some text", "str", "Some title", "int", 0)
DllClose($hDLL)
