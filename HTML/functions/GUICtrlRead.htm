Справочник функций

GUICtrlRead

Считывает состояние или данные элемента управления.

GUICtrlRead ( controlID [, advanced = 0] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
advanced [необязательный] Возвращает расширенную информацию об элементе управления.
$GUI_READ_DEFAULT (0) = (по умолчанию) Возвращает состояние или данные элемента управления.
$GUI_READ_EXTENDED (1) = Возвращает расширенную информацию об элементе управления (см. раздел Примечания).

Константы определены в GUIConstantsEx.au3.

Возвращаемое значение

Успех: Возвращает значение, зависящее от типа элемента управления (см. ниже).
Ошибка: Возвращает 0.


Type Value
Checkbox, Radio The checked state ($GUI_CHECKED or $GUI_UNCHECKED).
Combo, List The value selected.
Input, Edit The text entered.
Button The display text.
Date The selected date in the format defined by the regional settings.
Progress Current percentage.
Slider Current value.
Tab The index of the selected tabitem (0-based).
Menu, MenuItem State of the menu/item. See State table .
TreeView Control identifier (controlID) of the selected TreeViewItem.
TreeViewItem State of the TreeViewItem.
ListView Control identifier (controlID) of the selected ListViewItem. 0 means no item is selected.
ListViewItem The text of the selected item/lines in the ListView.
Dummy The value set by GUICtrlSendToDummy() or GUICtrlSetData().

Примечания

В расширенном режиме возвращаемое значение содержит дополнительные данные элемента управления (см. ниже).

Примечание: не все элементы управления возвращают дополнительные данные!

Тип Дополнительное значение
Checkbox, Radio Текст элемента управления.
Menu, MenuItem Текст элемента управления.
TreeView Текст текущего выбранного элемента TreeViewItem.
TreeViewItem Текст элемента TreeViewItem.
ListViewItem Состояние ListViewItem при использовании расширенного стиля $LVS_EX_CHECKBOXES в расширенном режиме. См. таблицу состояний
Tab controlID выбранного TabItem.

Для элементов управления Checkbox и Radio возвращаются только состояния $GUI_CHECKED (1), $GUI_UNCHECKED (4) или $GUI_INDETERMINATE (2) – значение можно использовать напрямую.

Для элементов ListView могут возвращаться несколько состояний: $GUI_CHECKED и $GUI_UNCHECKED (только для ListView с расширенным стилем LVS_EX_CHECKBOXES и в расширенном режиме). Для проверки используйте, например, BitAND(GUICtrlRead($Item, 1), $GUI_CHECKED).

Для элементов TreeView могут возвращаться несколько состояний: $GUI_FOCUS, $GUI_EXPAND, $GUI_CHECKED, $GUI_UNCHECKED (только для TreeView со стилем TVS_CHECKBOXES). Для проверки используйте, например, BitAND(GUICtrlRead($Item), $GUI_CHECKED).

См. также

GUICtrlCreate..., GUICtrlGetState, GUICtrlSendMsg, GUICtrlSendToDummy, GUICtrlSetData, GUICtrlUpdate..., GUIEventOptions (Option), GUIGetMsg

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("My GUICtrlRead") ; создаст диалоговое окно, отображаемое по центру

    Local $idMenu1 = GUICtrlCreateMenu("File")

    Local $idList = GUICtrlCreateList("", 10, 10, -1, 100)
    GUICtrlSetData(-1, "item1|item2|item3", "item2")

    Local $idButton = GUICtrlCreateButton("Read", 10, 110, 50)
    GUICtrlSetState(-1, $GUI_FOCUS) ; фокус установлен на этой кнопке

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    Local $idMsg, $iMenustate, $sMenutext
    ; Цикл до выхода пользователя.
    Do
        $idMsg = GUIGetMsg()
        If $idMsg = $idButton Then
            MsgBox($MB_SYSTEMMODAL, "Selected listbox entry", GUICtrlRead($idList)) ; отобразить выбранный элемент списка
            $iMenustate = GUICtrlRead($idMenu1) ; вернуть состояние пункта меню
            $sMenutext = GUICtrlRead($idMenu1, $GUI_READ_EXTENDED) ; вернуть текст пункта меню
            MsgBox($MB_SYSTEMMODAL, "State and text of the menuitem", "state:" & $iMenustate & @CRLF & "text:" & $sMenutext)
        EndIf
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
