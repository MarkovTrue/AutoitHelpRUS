ProcessGetStats

Возвращает массив с информацией о памяти или вводе-выводе запущенного процесса.

ProcessGetStats ( ["process" [, type = 0]] )

Параметры

process [необязательный] Имя или PID процесса, о котором нужна информация. По умолчанию (-1) текущий процесс.
type [необязательный]
$PROCESS_STATS_MEMORY (0) = (по умолчанию) информация о памяти.
$PROCESS_STATS_IO (1) = информация о вводе-выводе.

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает массив данных с информацией (см. раздел Примечания).
Ошибка: Устанавливает @error ≠ 0.

Примечания

Возвращаемый массив одномерный и содержит следующую информацию:

Тип = $PROCESS_STATS_MEMORY
    $aArray[0] = занимаемый размер (WorkingSetSize)
    $aArray[1] = максимальный занимаемый размер (PeakWorkingSetSize)


Тип = $PROCESS_STATS_IO
    $aArray[0] = количество выполненных операций чтения.
    $aArray[1] = количество выполненных операций записи.
    $aArray[2] = количество операций ввода-вывода, кроме чтения и записи.
    $aArray[3] = число прочитанных байт.
    $aArray[4] = число записанных байт.
    $aArray[5] = число байт, переданных во время операций, кроме чтения и записи.

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получить данные о памяти текущего процесса.
    Local $aMemory = ProcessGetStats()

    ; Если $aMemory является массивом, показываем следующие сведения о процессе.
    If IsArray($aMemory) Then
        MsgBox($MB_SYSTEMMODAL, "", "WorkingSetSize: " & $aMemory[0] & @CRLF & _
                "PeakWorkingSetSize: " & $aMemory[1])
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred.")
    EndIf
EndFunc   ;==>Example
