Справочник функций

GUICtrlCreateListView

Создаёт элемент управления ListView для GUI.

GUICtrlCreateListView ( "text", left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

text Определения заголовков столбцов, разделённые символом Opt("GUIDataSeparatorChar").
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $LVS_SHOWSELALWAYS, $LVS_SINGLESEL
принудительный стиль : $LVS_REPORT
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей или таблицу расширенных стилей ListView.
по умолчанию (-1) : $LVS_EX_FULLROWSELECT, $WS_EX_CLIENTEDGE

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для добавления элементов в ListView используйте GUICtrlCreateListViewItem().

По умолчанию ListView отображается как в Проводнике в режиме подробностей (стиль $LVS_REPORT обязателен).
Начальную ширину столбца можно управлять, добавляя пробелы к определению заголовка. Столбец может расширяться при добавлении элементов через GUICtrlCreateListViewItem() в соответствии с размером элемента. Размер столбца может быть до 25 символов. При обновлении через GUICtrlSetData() ширина не изменяется.

Для создания ListView со стилем Icon, SmallIcon или List используйте после создания GUICtrlSetStyle() со стилями $LVS_ICON, $LVS_LIST или $LVS_SMALLICON.

Сортировка списка по клику на заголовок столбца (как в Проводнике) в настоящий момент не реализована.

Для визуального выделения всей строки используйте расширенный стиль LVS_EX_FULLROWSELECT.

Для объединения стилей с стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_LISTVIEW, новый_стиль, ...).
Для использования указанных значений необходимо #include <ListViewConstants.au3> в скрипт.

Если добавить стиль $LVS_EX_CHECKBOXES к ListView, границы исчезнут. Добавьте стиль $WS_EX_CLIENTEDGE для их отображения.


Специальный флаг $GUI_BKCOLOR_LV_ALTERNATE используется с элементом управления ListView для задания чередующегося фона строк ListviewItems.
Нечётные строки получат цвет, заданный через GUICtrlSetBkColor() элемента управления ListView.
Чётные строки получат цвет, заданный через GUICtrlSetBkColor() элемента управления ListviewItem.

См. также

GUICoordMode (Option), GUICtrlCreateListViewItem, GUICtrlRegisterListViewSort, GUICtrlSetData, GUIDataSeparatorChar (Option), GUIGetMsg

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    GUICreate("listview items", 220, 250, 100, 200, -1, $WS_EX_ACCEPTFILES)
    GUISetBkColor(0x00E0FFFF) ; изменит цвет фона

    Local $idListview = GUICtrlCreateListView("col1 |col2|col3 ", 10, 10, 200, 150) ;,$LVS_SORTDESCENDING)
    Local $idButton = GUICtrlCreateButton("Value?", 75, 170, 70, 20)
    Local $idLVi_Item1 = GUICtrlCreateListViewItem("item2|col22|col23", $idListview)
    Local $idLVi_Item2 = GUICtrlCreateListViewItem("item1|col12|col13", $idListview)
    Local $idLVi_Item3 = GUICtrlCreateListViewItem("item3|col32|col33", $idListview)
    GUICtrlCreateInput("", 20, 200, 150)
    GUICtrlSetState(-1, $GUI_DROPACCEPTED) ; разрешить перетаскивание
    GUISetState(@SW_SHOW)
    GUICtrlSetData($idLVi_Item2, "ITEM1")
    GUICtrlSetData($idLVi_Item3, "||COL33")
    GUICtrlDelete($idLVi_Item1)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idButton
                MsgBox($MB_SYSTEMMODAL, "listview item", GUICtrlRead(GUICtrlRead($idListview)), 2)

            Case $idListview
                MsgBox($MB_SYSTEMMODAL, "listview", "clicked=" & GUICtrlGetState($idListview), 2)

        EndSwitch
    WEnd
EndFunc   ;==>Example
