Справочник функций

RegRead

Читает значение из реестра.

RegRead ( "keyname", "valuename" )

Параметры

keyname Читаемый раздел реестра.
valuename Имя читаемого значения.

Возвращаемое значение

Успех: Возвращает запрошенное значение реестра; @extended устанавливается в тип значения ($REG_...). Типы определены в Constants.au3.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – невозможно открыть запрошенный раздел
2 – невозможно открыть главный раздел
3 – невозможно подключиться к удалённому реестру
-1 – невозможно открыть запрошенное значение
-2 – тип значения не поддерживается

Примечания

Раздел реестра должен начинаться с "HKEY_LOCAL_MACHINE" ("HKLM"), "HKEY_USERS" ("HKU"), "HKEY_CURRENT_USER" ("HKCU"), "HKEY_CLASSES_ROOT" ("HKCR") или "HKEY_CURRENT_CONFIG" ("HKCC").

При работе в 64-разрядной Windows для чтения значений 64-разрядной ветки реестра добавьте суффикс 64 к корневому ключу, например HKLM64.

AutoIt поддерживает разделы реестра типов REG_BINARY, REG_SZ, REG_MULTI_SZ, REG_EXPAND_SZ, REG_QWORD и REG_DWORD.

Для доступа к значению (Default) передайте "" (пустую строку) в качестве valuename.

При чтении REG_BINARY результат возвращается как двоичный тип (в предыдущих версиях – строка шестнадцатеричных символов).

При чтении REG_MULTI_SZ несколько записей разделяются символом @LF – используйте StringSplit(..., @LF) для получения каждой строки.

Для доступа к удалённому реестру используйте имя раздела вида "\\имя_компьютера\раздел". Требуются соответствующие права доступа.

См. также

RegDelete, RegWrite, StringSplit

Пример

#include <MsgBoxConstants.au3>

Local $sFilePath = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion", "ProgramFilesDir")
MsgBox($MB_SYSTEMMODAL, "", "Program files are located at: " & $sFilePath)
