BitOR

Выполняет побитовую операцию OR.

BitOR ( value1, value2 [, value n] )

Параметры

value1 Первое число.
value2 Второе число.
value n [необязательный] n-е число – всего можно задать до 255 значений.

Возвращаемое значение

Возвращает результат побитового OR двух параметров.
Битовые операции выполняются над 32-битными целыми числами.

Примечания

Напомним: для чисел можно использовать шестнадцатеричную запись.
BitOR() возвращает 0 в каждой позиции бита, где все входные аргументы имеют 0, и 1 там, где хотя бы один бит равен 1.

См. также

BitAND, BitNOT, BitRotate, BitShift, BitXOR, Hex

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Примечание: "b" – это символ байта.

    ; Присвоить локальной переменной результат побитового OR для 1 и 0.
    Local $iBitOR1 = BitOR(1, 0) ; Присвоить локальной переменной результат побитового OR для 1 и 0.

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR1)

    ; Присвоить локальной переменной результат побитового OR для 1 и 1.
    Local $iBitOR2 = BitOR(1, 1) ; Присвоить локальной переменной результат побитового OR для 1 и 0.

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR2)

    ; Присвоить локальной переменной результат побитового OR для 13 (1101b) и 7 (0111b).
    Local $iBitOR3 = BitOR(13, 7) ; Присвоить локальной переменной результат побитового OR для 1 и 1.

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR3)

    ; Присвоить локальной переменной результат побитового OR для 2 (0010b), 3 (0011b) и 6 (0110b).
    Local $iBitOR4 = BitOR(2, 3, 6) ; 0001b OR 0000b = 0001b

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitOR4)
EndFunc   ;==>Example
