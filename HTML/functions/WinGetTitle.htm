WinGetTitle

Получает полный заголовок окна.

WinGetTitle ( "title" [, "text"] )

Параметры

title Заголовок/hWnd/класс окна, заголовок которого нужно получить. См. Специальное определение заголовка.
text [необязательный] Текст окна, заголовок которого нужно получить. По умолчанию пустая строка. См. Специальное определение текста.

Возвращаемое значение

Успех: Возвращает строку, содержащую полный заголовок окна.
Ошибка: Возвращает "", если окно не найдено.

Примечания

WinGetTitle("[active]") возвращает заголовок активного окна. WinGetTitle() работает как со свёрнутыми, так и со скрытыми окнами. Если несколько окон соответствуют критериям, используется наиболее недавно активное.

См. также

AutoItWinGetTitle, WinGetText, WinSetTitle, WinTitleMatchMode (Option)

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получаем заголовок активного окна.
    Local $sText = WinGetTitle("[ACTIVE]")

    ; Выводим заголовок окна.
    MsgBox($MB_SYSTEMMODAL, "", $sText)
EndFunc   ;==>Example
