StringIsXDigit

Проверяет, содержит ли строка только шестнадцатеричные символы (0–9, A–F).

StringIsXDigit ( "string" )

Параметры

string Проверяемая строка.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если строка содержит нешестнадцатеричные символы.

Примечания

Как и другие функции StringIs...(), проверка прекращается при обнаружении Chr(0) внутри строки.
Для ускорения обработки внутренне используется функция языка C.

Допустимые символы: 0123456789abcdefABCDEF.
Обратите внимание: пробельные символы или префикс "0x" приведут к тому, что StringIsXDigit() вернёт 0.

См. также

StringIsAlNum, StringIsAlpha, StringIsASCII, StringIsDigit, StringIsLower, StringIsSpace, StringIsUpper, StringLower, StringUpper

Пример

#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "", "Is the string 42 a hexadecimal digit: " & StringIsXDigit("42") & @CRLF & _ ; Возвращает 1, так как строка содержит только шестнадцатеричные символы (0-9, A-F).
        "Is the string 00 a hexadecimal digit: " & StringIsXDigit("00") & @CRLF & _ ; Возвращает 1, так как строка содержит только шестнадцатеричные символы (0-9, A-F).
        "Is the string 1.0 a hexadecimal digit: " & StringIsXDigit("1.0") & @CRLF & _ ; Возвращает 0 из-за десятичной точки.
        "Is the number 1.0 a hexadecimal digit: " & StringIsXDigit(1.0) & @CRLF & _ ; Возвращает 1 из-за преобразования числа в строку.
        "Is the string 'A string' a hexadecimal digit: " & StringIsXDigit("A string.") & @CRLF) ; Возвращает 0, так как строка содержит пробельные символы.
