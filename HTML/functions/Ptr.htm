Справочник функций

Ptr

Преобразует выражение в вариант типа Pointer.

Ptr ( expression )

Параметры

expression Выражение, которое нужно преобразовать в тип Pointer.

Возвращаемое значение

Возвращает указательное представление выражения.

См. также

HWnd, Int, IsPtr, Number, String

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Преобразуем -1 в указатель.
    Local $pPtr = Ptr(-1)

    ; Проверяем, является ли $hPtr допустимым указателем, и выводим результат.
    If IsPtr($pPtr) Then
        MsgBox($MB_SYSTEMMODAL, "", "It's a valid Ptr")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "It's not a valid Ptr")
    EndIf
EndFunc   ;==>Example
