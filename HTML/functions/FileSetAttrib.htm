Справочник функций

FileSetAttrib

Устанавливает атрибуты одного или нескольких файлов или каталогов.

FileSetAttrib ( "file pattern", "+-RASHNOT" [, recurse = 0] )

Параметры

file pattern Путь к файлу (или файлам), например C:\*.au3, C:\Dir. Допускаются символы подстановки * и ? – см. раздел Примечания.
+-RASHNOT Устанавливаемые (+) или снимаемые (-) атрибуты. Например: "+A", "+RA-SH".
recurse [необязательный]
$FT_NONRECURSIVE (0) – без рекурсии (по умолчанию)
$FT_RECURSIVE (1) – с рекурсивным обходом подкаталогов.

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0, если возникли ошибки.

Примечания

Об использовании символов подстановки см. описание FileFindFirstFile().

Параметр file pattern не может содержать пробелы!
Функция позволяет устанавливать (+) или снимать (-) следующие атрибуты:
    "R" = READONLY (только чтение)
    "A" = ARCHIVE (архивный)
    "S" = SYSTEM (системный)
    "H" = HIDDEN (скрытый)
    "N" = NORMAL (обычный)
    "O" = OFFLINE (автономный)
    "T" = TEMPORARY (временный)

(Атрибуты сжатия и каталога через эту функцию установить нельзя.)

См. также

FileGetAttrib, FileGetTime, FileSetTime

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>

If MsgBox(($MB_YESNO + $MB_SYSTEMMODAL), "", "Note: This is an example and shouldn't be run under normal circumstances. " & @CRLF & @CRLF & "Do you wish to run the example?") = $IDYES Then
    ; Помечаем все .au3-файлы в текущем каталоге как "только чтение" и "системный".
    If Not FileSetAttrib(@ScriptDir & "\*.au3", "+RS") Then
        MsgBox($MB_SYSTEMMODAL, "Error", "Problem setting attributes.")
    EndIf

    ; Помечаем все .bmp-файлы на @HomeDrive и в подкаталогах как "архивный" (снимаем "только чтение").
    If Not FileSetAttrib(@HomeDrive & "\*.bmp", "-R+A", $FT_RECURSIVE) Then
        MsgBox($MB_SYSTEMMODAL, "Error", "Problem setting attributes.")
    EndIf
EndIf
