Справочник функций

BitShift

Выполняет операцию побитового сдвига.

BitShift ( value, shift )

Параметры

value Число, которое требуется сдвинуть.
shift Число бит для сдвига вправо (отрицательные значения – сдвиг влево).

Возвращаемое значение

Возвращает значение, сдвинутое на указанное количество битов.
Битовые операции выполняются над 32-битными целыми числами.

Примечания

Напомним: для чисел можно использовать шестнадцатеричную запись.
Сдвиг вправо эквивалентен делению на 2, сдвиг влево – умножению на 2.

См. также

BitAND, BitNOT, BitOR, BitRotate, BitXOR, Hex

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Примечание: "b" – это символ байта.

    ; Присвоить локальной переменной результат побитового сдвига влево для 2.
    Local $iBitShift1 = BitShift(2, -1) ; 2 = 0010b сдвиг влево на 1 -> 4 = 0100b

    ; Примечание: эквивалентно BitRotate(2, 1)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift1)

    ; Присвоить локальной переменной результат побитового сдвига вправо для 1.
    Local $iBitShift2 = BitShift(1, 1) ; 1 = 0001b сдвиг вправо на 1 -> 0 = 0000b

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift2)

    ; Присвоить локальной переменной результат побитового сдвига влево для 14.
    Local $iBitShift3 = BitShift(14, -2) ; 14 = 1110b сдвиг влево на 2 -> 56 = 0011 1000b

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift3)

    ; Присвоить локальной переменной результат побитовой операции
    Local $iBitShift4 = BitShift(1, -31)
    ;1 = 0001b сдвиг вправо на 31 -> -2147483648 (32 бита) = 1000 0000 0000 0000 0000 0000 0000 0000b

    ;Примечание: в записи "дополнение до двух" 32-й разряд справа является знаковым.

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $iBitShift4)
EndFunc   ;==>Example
