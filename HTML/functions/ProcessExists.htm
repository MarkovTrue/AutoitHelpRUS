ProcessExists

Проверяет, выполняется ли указанный процесс.

ProcessExists ( "process" )

Параметры

process Имя или PID проверяемого процесса.

Возвращаемое значение

Успех: Возвращает PID процесса.
Ошибка: Возвращает 0, если процесс не существует.

Примечания

Имена процессов указываются без пути, например "notepad.exe" или "winword.exe".

PID – уникальный номер, идентифицирующий процесс.

Процесс опрашивается приблизительно каждые 250 миллисекунд.

См. также

ProcessClose, ProcessList, ProcessWait, ProcessWaitClose, WinExists

Пример

#include <MsgBoxConstants.au3>

If ProcessExists("notepad.exe") Then ; Проверить, запущен ли процесс Блокнота.
    MsgBox($MB_SYSTEMMODAL, "", "Notepad is running")
Else
    MsgBox($MB_SYSTEMMODAL, "", "Notepad is not running")
EndIf
