PixelChecksum

Вычисляет контрольную сумму для области пикселей.

PixelChecksum ( left, top, right, bottom [, step = 1 [, hwnd [, mode = 0]]] )

Параметры

left Левая координата прямоугольника.
top Верхняя координата прямоугольника.
right Правая координата прямоугольника.
bottom Нижняя координата прямоугольника.
step [необязательный] Шаг перебора пикселей: значение больше 1 пропускает пиксели для ускорения. Например, значение 2 проверяет каждый второй пиксель. По умолчанию 1. Не рекомендуется использовать шаг больше 1.
hwnd [необязательный] Дескриптор окна. По умолчанию используется рабочий стол. См. раздел Примечания.
mode [необязательный] По умолчанию 0 – контрольная сумма ADLER; 1 – CRC32.

Возвращаемое значение

Успех: Возвращает контрольную сумму области.
Ошибка: Возвращает 0.

Примечания

Контрольная сумма позволяет определить, изменилось ли что-либо в области, но не показывает, что именно изменилось.

CRC32 немного медленнее ADLER, но лучше обнаруживает изменения пикселей.

Если PixelCoordMode установлен в режим SCREEN (1) по умолчанию, параметр hwnd игнорируется. Он учитывается только при режиме WINDOW (0) или CLIENT (2).

См. также

PixelCoordMode (Option), PixelGetColor, PixelSearch

bottom>Пример

#include <MsgBoxConstants.au3>

; Ожидать изменений в области от (0,0) до (50,50).

; Получить начальную контрольную сумму.
Local $iCheckSum = PixelChecksum(0, 0, 50, 50)

; Ожидать изменения области; проверка выполняется каждые 100 мс для снижения нагрузки на ЦП.
While $iCheckSum = PixelChecksum(0, 0, 50, 50)
    Sleep(100)
WEnd

MsgBox($MB_SYSTEMMODAL, "", "Something in the region has changed!")
