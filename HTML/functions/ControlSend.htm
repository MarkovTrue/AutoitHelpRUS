Справочник функций

ControlSend

Отправляет строку символов элементу управления.

ControlSend ( "title", "text", controlID, "string" [, flag = 0] )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.
string Строка символов для отправки элементу управления.
flag [необязательный] Изменяет способ обработки keys:
$SEND_DEFAULT (0) = текст содержит специальные символы, такие как + и ! для имитации нажатий SHIFT и ALT (по умолчанию).
$SEND_RAW (1) = клавиши отправляются как есть.

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если окно или элемент управления не найдены.

Примечания

ControlSend() работает похоже на Send(), но отправляет нажатия клавиш напрямую окну или элементу управления, а не только активному окну.

ControlSend() ненадёжна только с окнами командной строки: они работают иначе, чем обычные окна (похоже, проверяют физическое состояние клавиш, а не принимают сообщения о нажатиях). Для обычных окон ControlSend() заметно надёжнее обычного Send() – и да, она передаёт Shift, Ctrl, Alt и подобные.

Как сказано в справке по Send(), раскладки, которые при включённом CAPS LOCK и нажатом Shift дают другие символы, имитировать нельзя. Например, чешская раскладка. Обойти это можно через ControlSetText().

Возможно, элементу управления сначала придётся передать фокус командой ControlFocus(), особенно если controlID создан самим скриптом.

Opt("SendKeyDelay",...) меняет длительность короткой паузы между отправляемыми нажатиями.
Opt("SendKeyDownDelay",...) меняет время удержания клавиши перед отпусканием при нажатии.

См. также

ControlCommand, ControlFocus, ControlSetText, Send, SendKeyDelay (Option), SendKeyDownDelay (Option)

Пример

#include "Extras\HelpFileInternals.au3"

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Отправить строку текста в поле редактирования Блокнота. Дескриптор, возвращённый WinWait, используется в качестве параметра "title" функции ControlSend.
    ControlSend($hWnd, "", ControlGetFocus($hWnd), "This is some text")

    _NotepadForceClose($hWnd)
EndFunc   ;==>Example
