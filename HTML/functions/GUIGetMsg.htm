Справочник функций

GUIGetMsg

Опрашивает GUI, не произошли ли какие-либо события.

GUIGetMsg ( [advanced = 0] )

Параметры

advanced [необязательный] Возвращает расширенную информацию в массиве.
$GUI_EVENT_SINGLE (0) = (по умолчанию) Возвращает одно событие.
$GUI_EVENT_ARRAY (1) = Возвращает массив с событием и расширенной информацией.

Константы определены в GUIConstantsEx.au3.

Возвращаемое значение

Returns an event, or an array depending on the advanced parameter.

Возвращаемое "событие" – это ID элемента управления, отправившего сообщение, либо специальное событие (например, окно закрывается или сворачивается). Если сообщения нет, событие равно 0.

Event ID: ID элемента управления, отправившего сообщение.
$GUI_EVENT_NONE (0): Нет события.
$GUI_EVENT_CLOSE: диалоговое окно закрывается (кнопкой или через системное меню).
$GUI_EVENT_MINIMIZE: окно свёрнуто кнопкой в заголовке Windows.
$GUI_EVENT_RESTORE: окно восстановлено кликом по значку на панели задач.
$GUI_EVENT_MAXIMIZE: окно развёрнуто кнопкой в заголовке Windows.
$GUI_EVENT_MOUSEMOVE: курсор мыши переместился.
$GUI_EVENT_PRIMARYDOWN: нажата основная кнопка мыши.
$GUI_EVENT_PRIMARYUP: отпущена основная кнопка мыши.
$GUI_EVENT_SECONDARYDOWN: нажата вторичная кнопка мыши.
$GUI_EVENT_SECONDARYUP: отпущена вторичная кнопка мыши.
$GUI_EVENT_RESIZED: размер окна изменён.
$GUI_EVENT_DROPPED: Конец операции Drag&Drop. @GUI_DragId, @GUI_DragFile и @GUI_DropId используются для получения ID или файла соответствующего контрола.

Константы определены в GUIConstantsEx.au3.

When using the advanced параметр, информация возвращается в массиве с расширенными данными:

$aArray[0] = 0 or Event ID or Control ID
$aArray[1] = Дескриптор окна, из которого пришло событие
$aArray[2] = Дескриптор элемента управления, из которого пришло событие (если применимо)
$aArray[3] = Текущая позиция X курсора мыши (относительно окна GUI)
$aArray[4] = Текущая позиция Y курсора мыши (относительно окна GUI)

Если опция GUIOnEventMode установлена в 1, GUIGetMsg всегда возвращает 0, а @error is set to 1.

Если опция GUIEventOptions установлена в 1, кнопки свернуть, восстановить и развернуть не выполняют действий над окном, а только уведомляют.

Примечания

Функция автоматически освобождает процессор при необходимости, поэтому её безопасно использовать в тесных циклах, не загружая CPU на 100%.

Информацию о положении мыши и элементе управления под курсором можно получить через GUIGetCursorInfo(). При наведении курсора на элемент управления событие не генерируется, поэтому для получения ID элемента управления нужно вызывать GUIGetCursorInfo().

См. также

GUICreate, GUICtrlCreate..., GUICtrlRead, GUICtrlSendMsg, GUICtrlSetOnEvent, GUIEventOptions (Option), GUIGetCursorInfo, GUIOnEventMode (Option)

Пример

Пример 1

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

; -------------------------------------------------------------------------------------
; Пример – нажмите кнопку, чтобы увидеть значение переключателей
; Скрипт также обнаруживает изменения состояния (закрытие/свёртывание/таймауты и т.п.).
Func Example()
    Opt("GUICoordMode", 1)

    GUICreate("Radio Box Demo", 400, 280)

    ; Создать элементы управления
    Local $idButton_1 = GUICtrlCreateButton("B&utton 1", 30, 20, 120, 40)
    GUICtrlCreateGroup("Group 1", 30, 90, 165, 160)
    GUIStartGroup()
    Local $idRadio_1 = GUICtrlCreateRadio("Radio &0", 50, 120, 70, 20)
    GUICtrlCreateRadio("Radio &1", 50, 150, 60, 20)
    Local $idRadio_3 = GUICtrlCreateRadio("Radio &2", 50, 180, 60, 20)

    ; Инициализировать переменные для отслеживания событий GUI
    Local $iRadioVal1 = 0 ; Предполагаем: 0 = первый переключатель выбран, 2 = последний

    ; Показать GUI
    GUISetState(@SW_SHOW)

    Local $idMsg = 0
    ; В этом message-loop мы используем переменные для отслеживания изменений радиокнопок, другие
    ; способ – вызвать GUICtrlRead() в конце для чтения состояния каждого элемента управления
    While 1
        $idMsg = GUIGetMsg()
        Select
            Case $idMsg = $GUI_EVENT_CLOSE
                MsgBox($MB_SYSTEMMODAL, "", "Dialog was closed")
                ExitLoop
            Case $idMsg = $GUI_EVENT_MINIMIZE
                MsgBox($MB_SYSTEMMODAL, "", "Dialog minimized", 2)
            Case $idMsg = $GUI_EVENT_RESTORE
                MsgBox($MB_SYSTEMMODAL, "", "Dialog restored", 2)
            Case $idMsg = $idButton_1
                MsgBox($MB_SYSTEMMODAL, "", "Default button clicked:" & @CRLF & "Radio " & $iRadioVal1)
            Case $idMsg >= $idRadio_1 And $idMsg <= $idRadio_3
                $iRadioVal1 = $idMsg - $idRadio_1
        EndSelect
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Пример 2

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI.
    Local $hGUI1 = GUICreate("Example GUI1")
    ; Создать кнопку.
    Local $idButton1 = GUICtrlCreateButton("Button1", 10, 10, 80, 22)
    ; Показать GUI
    GUISetState(@SW_SHOW, $hGUI1)

    ; Создать GUI.
    Local $hGUI2 = GUICreate("Example GUI2", 300, 300)
    ; Создать кнопку.
    Local $idButton2 = GUICtrlCreateButton("Button2", 10, 10, 80, 22)
    ; Показать GUI
    GUISetState(@SW_SHOW, $hGUI2)

    ; Инициализировать локальную переменную.
    Local $aMsg = 0

    While 1
        ; Записать в $aMsg расширенные сообщения GUI.
        $aMsg = GUIGetMsg($GUI_EVENT_ARRAY)

        ; Переключение между GUI
        Switch $aMsg[1]
            Case $hGUI1
                ; Событие поступает из GUI1

                ; Переключение по ID события
                Switch $aMsg[0]
                    Case $GUI_EVENT_CLOSE
                        ExitLoop
                    Case $idButton1
                        MsgBox($MB_SYSTEMMODAL, "", "Button1 clicked.")
                EndSwitch
            Case $hGUI2
                ; Событие поступает из GUI2

                ; Переключение по ID события
                Switch $aMsg[0]
                    Case $GUI_EVENT_CLOSE
                        GUIDelete($hGUI2)
                    Case $idButton2
                        MsgBox($MB_SYSTEMMODAL, "", "Button2 clicked.")
                EndSwitch
        EndSwitch
    WEnd

    ; Удалить предыдущие GUI и все элементы управления.
    GUIDelete($hGUI1)
EndFunc   ;==>Example
