FileGetEncoding

Определяет кодировку текста в файле.

FileGetEncoding ( "filehandle/filename" [, mode = 1] )

Параметры

filehandle/filename Дескриптор файла, возвращённый предыдущим вызовом FileOpen(). Альтернативно можно передать имя файла строкой в качестве первого параметра.
mode [необязательный] Режим обнаружения UTF-8.
$FE_ENTIRE_UTF8 (1) = проверять весь файл на наличие последовательностей UTF-8 (по умолчанию)
$FE_PARTIALFIRST_UTF8 (2) = проверять только начало файла (так же, как это делает FileOpen() по умолчанию)

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает кодировку файла в виде значений, аналогичных функции FileOpen():
$FO_UTF16_LE (32) = UTF-16 Little Endian.
$FO_UTF16_BE (64) = UTF-16 Big Endian.
$FO_UTF8 (128) = UTF-8 (с BOM).
$FO_UTF8_NOBOM (256) = UTF-8 (без BOM).
$FO_ANSI (512) = ANSI (содержит символы со значениями > 127 и < 255).
Ошибка: Возвращает -1.

Константы определены в FileConstants.au3.

Примечания

Если передано имя файла, а не дескриптор, файл будет открыт и закрыт непосредственно во время вызова функции.
Примечание: не смешивайте дескрипторы и имена файлов – не открывайте файл через FileOpen(), а затем передавайте его имя в эту функцию. В своих процедурах используйте либо дескрипторы, либо имена файлов, но не оба подхода одновременно.

Если передан дескриптор файла, параметр mode не учитывается – применяется режим, указанный при вызове FileOpen().

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iEncoding = FileGetEncoding(@ScriptFullPath) ; Определяем кодировку текущего скрипта.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error: Could not obtain the file encoding.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "FileGetEncoding: The value returned was: " & $iEncoding) ; Для данного примера должно быть возвращено 0 или $FO_ANSI.
    EndIf
EndFunc   ;==>Example
