Справочник функций

AutoItWinGetTitle

Возвращает заголовок окна AutoIt.

AutoItWinGetTitle ( )

Возвращаемое значение

Возвращает строку, содержащую заголовок окна AutoIt.

См. также

AutoItWinSetTitle, WinGetTitle

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Задать заголовок скрытого окна AutoIt.
    AutoItWinSetTitle("My AutoIt Window")

    ; Показать скрытое окно AutoIt.
    AutoItWinShow()

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example

; Показать скрытое окно AutoIt. Возвращает дескриптор окна.
Func AutoItWinShow()
    Local $hWnd = WinGetHandle(AutoItWinGetTitle()) ; Получить дескриптор скрытого окна AutoIt по его заголовку.
    WinMove($hWnd, "", (@DesktopWidth / 2) - 250, (@DesktopHeight / 2) - 250, 500, 500) ; Переместить и изменить размер скрытого окна AutoIt для удобного просмотра.
    WinSetState($hWnd, "", @SW_SHOW) ; Показать скрытое окно AutoIt – обычно оно скрыто, но в этом примере отображается.
    Return $hWnd
EndFunc   ;==>AutoItWinShow
