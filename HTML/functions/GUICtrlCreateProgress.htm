Справочник функций

GUICtrlCreateProgress

Создаёт элемент управления Progress (индикатор выполнения) для GUI.

GUICtrlCreateProgress ( left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию – ранее использованная ширина).
height [необязательный] Высота элемента управления (по умолчанию – ранее использованная высота).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для получения значения элемента управления см. GUICtrlRead().
Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().

Для обновления позиции индикатора используйте GUICtrlSetData().

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_PROGRESS, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <ProgressConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKAUTO – автоматическое изменение размера и позиции.

См. также

GUICoordMode (Option), GUICtrlSetData, GUICtrlUpdate..., GUIGetMsg

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <ProgressConstants.au3>

Example()

Func Example()
    GUICreate("My GUI Progressbar", 220, 100, 100, 200)
    Local $idProgress_bar1 = GUICtrlCreateProgress(10, 10, 200, 20)
    GUICtrlSetColor(-1, 32250) ; не работает со стилем Windows XP
    Local $idProgress_bar2 = GUICtrlCreateProgress(10, 40, 200, 20, $PBS_SMOOTH)
    Local $idButton = GUICtrlCreateButton("Start", 75, 70, 70, 20)
    GUISetState(@SW_SHOW)

    Local $iWait = 20 ; ожидание 20 мс до следующего шага прогресса
    Local $iSavPos = 0 ; сохранение позиции индикатора

    Local $idMsg, $idM
    ; Цикл до выхода пользователя.
    Do
        $idMsg = GUIGetMsg()
        If $idMsg = $idButton Then
            GUICtrlSetData($idButton, "Stop")
            For $i = $iSavPos To 100
                If GUICtrlRead($idProgress_bar1) = 50 Then MsgBox($MB_SYSTEMMODAL, "Info", "The half is done...", 1)

                $idM = GUIGetMsg()
                If $idM = $GUI_EVENT_CLOSE Then ExitLoop

                If $idM = $idButton Then
                    GUICtrlSetData($idButton, "Next")
                    $iSavPos = $i ; сохранить текущую позицию индикатора в $iSavPos
                    ExitLoop
                Else
                    $iSavPos = 0
                    GUICtrlSetData($idProgress_bar1, $i)
                    GUICtrlSetData($idProgress_bar2, (100 - $i))
                    Sleep($iWait)
                EndIf
            Next
            If $i > 100 Then
                GUICtrlSetData($idButton, "Start")
            EndIf
        EndIf
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
