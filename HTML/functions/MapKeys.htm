Справочник функций

MapKeys

Возвращает массив ключей Map.

MapKeys ( map )

Параметры

map Существующий объект Map.

Возвращаемое значение

Успех: Возвращает одномерный массив ключей.
Ошибка: Возвращает массив с нулевым базовым индексом и устанавливает @error ≠ 0.

Примечания

Возвращённый массив можно использовать для перебора значений Map.

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Объявить Map и присвоить пары ключ-значение.
    Local $mMap[]
    $mMap[1] = "Integer One" ; Целое число как ключ.
    $mMap["2"] = "String Two" ; Строковое значение, представляющее целое число, в качестве ключа. Это строка, а не целое число.
    MapAppend($mMap, "Integer Two") ; Добавить значение с использованием следующего доступного целого, в данном случае 2.

    ; Получить ключи map. Возвращается одномерный массив с индексами от 0.
    Local $aMapKeys = MapKeys($mMap)
    For $vKey In $aMapKeys ; Можно также использовать цикл For.
        MsgBox($MB_SYSTEMMODAL, "", "Key: " & $vKey & @CRLF & _ ; Ключ.
                "Value: " & $mMap[$vKey] & @CRLF & _ ; Использовать элемент массива MapKeys() для отображения значения ключа.
                "Variable Type: " & VarGetType($vKey) & @CRLF) ; Показать тип переменной ключа (целое число или строка).
    Next
EndFunc   ;==>Example
