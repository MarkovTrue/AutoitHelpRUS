Справочник функций

InputBox

Отображает диалоговое окно для ввода строки пользователем.

InputBox ( "title", "prompt" [, "default" [, "password char" [, width = -1 [, height = -1 [, left = Default [, top = Default [, timeout = 0 [, hwnd]]]]]]]] )

Параметры

title Заголовок окна ввода.
prompt Сообщение для пользователя с описанием ожидаемого ввода.
default [необязательный] Начальное значение в поле ввода.
password char [необязательный] Символ, которым заменяются все вводимые символы при отображении. Чтобы показывать реальные символы, передайте пустую строку ("") (по умолчанию) или пробел в качестве первого символа. При передаче многосимвольной строки для маскировки используется только первый символ. Второй и последующие символы имеют особый смысл – см. раздел Примечания.
width [необязательный] Ширина окна.
height [необязательный] Высота окна.
left [необязательный] Положение левой стороны окна ввода. По умолчанию окно центрируется.
top [необязательный] Положение верхней стороны окна ввода. По умолчанию окно центрируется.
timeout [необязательный] Количество секунд ожидания до автоматической отмены InputBox().
hwnd [необязательный] Дескриптор окна, используемого в качестве родительского для диалога.

Возвращаемое значение

Успех: Возвращает введённую строку.
Ошибка: Возвращает "" (пустую строку) и устанавливает @error ≠ 0.
@error: 1 – нажата кнопка "Отмена".
2 – истекло время ожидания.
3 – не удалось открыть InputBox (обычно из-за неверных аргументов).
4 = InputBox не может быть отображён ни на одном мониторе.
5 – недопустимые параметры: ширина без высоты или left без top.

Примечания

InputBox() поддерживает изменение размера пользователем, но имеет минимальный размер около 190 × 115 пикселей. Размер по умолчанию – около 250 × 190 пикселей.

Возвращаемая строка не может превышать 254 символа; если ввод содержит символы возврата каретки или перевода строки, результат обрезается на первом из них.

Второй и последующие символы поля пароля можно использовать для ограничения ввода. Используйте пробел в качестве первого символа, чтобы вводимые символы отображались. Символ M после первого символа означает, что ввод обязателен: при пустом поле нажатие кнопки OK не закроет окно и не вернёт строку.

Если скрипт использует InputBox, необходимо добавить директиву pragma: "#pragma compile(inputboxres, true)"

В конце поля пароля можно также указать максимальную длину строки – укажите нужное число как последний символ (или символы).

MsgBox

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Размещает окно ввода в левом верхнем углу, отображая символы по мере
    ; их ввода.
    Local $sAnswer = InputBox("Question", "Where were you born?", "Planet Earth", "", _
             -1, -1, 0, 0)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $sAnswer)

    ; Запрашивает у пользователя пароль. Не забудьте его проверить!
    Local $sPasswd = InputBox("Security Check", "Enter your password.", "", "*")

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $sPasswd)

    ; Запрашивает у пользователя ответ из 1 или 2 символов. M в поле пароля
    ; означает, что пустая строка не принимается, а 2 указывает, что
    ; ответ не может быть длиннее 2 символов.
    Local $sValue = InputBox("Testing", "Enter the 1 or 2 character code.", "", " M2")

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $sValue)
EndFunc   ;==>Example
