Ping

Отправляет ping на указанный хост и возвращает время прохождения пакета.

Ping ( "address/hostname" [, timeout = 4000] )

Параметры

address/hostname Адрес или имя хоста, например "www.autoitscript.com" или "87.106.244.38".
timeout [необязательный] Время ожидания ответа в миллисекундах (по умолчанию 4000).

Возвращаемое значение

Успех: Возвращает время прохождения пакета в миллисекундах (больше 0).
Ошибка: Возвращает 0, если хост недоступен или возникли другие сетевые ошибки; @error ≠ 0.
@error: 1 – Хост отключён
2 – Хост недостижим
3 – Недопустимый адрес назначения
4 – Прочие ошибки

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Отправить ping на сайт AutoIt с таймаутом 250 мс.
    Local $iPing = Ping("autoitscript.com", 250)

    If $iPing Then ; Если возвращено значение больше 0 – вывести сообщение.
        MsgBox($MB_SYSTEMMODAL, "", "The roundtrip-time took: " & $iPing & "ms.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred with @error value of: " & @error)
    EndIf
EndFunc   ;==>Example
