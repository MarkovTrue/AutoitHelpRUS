Справочник функций

Mod

Вычисляет остаток от деления.

Mod ( value1, value2 )

Параметры

value1 Делимое.
value2 Делитель.

Возвращаемое значение

Успех: Возвращает остаток от деления value1 на value2.
Ошибка: Возвращает -1.#IND, если делитель равен нулю.

Примечания

Функция гарантирует, что Mod(делимое, делитель) = делимое - Int(делимое / делитель) * делитель.

Функция не гарантирует точного представления делимого или делителя, особенно для чисел с плавающей точкой.

Если переданы целые числа, выполняется целочисленная операция модуля. В противном случае используется операция с плавающей точкой, что, согласно предыдущему замечанию, может дать неожиданный результат.

См. также

Int

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной переменной число 18.
    Local $iNb = 18

    ; Если остаток от деления $iNb на 2 равен 0, то $iNb – чётное.
    If Mod($iNb, 2) = 0 Then
        MsgBox($MB_SYSTEMMODAL, "", $iNb & " is an even number.")
    Else
        MsgBox($MB_SYSTEMMODAL, "", $iNb & " is an odd number.")
    EndIf

    ; Присвоить локальной переменной остаток от деления 4 на 7.
    Local $iMod1 = Mod(4, 7) ; 4%7 = 4, т.к. делитель > делимого

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "$iMod1", $iMod1)

    ; Присвоить локальной переменной остаток от деления 1 на 3/4.
    Local $iMod2 = Mod(1, 3 / 4) ; 1%(3/4) = 0.25, т.к. делитель является числом с плавающей точкой

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "$iMod2", $iMod2)
EndFunc   ;==>Example
