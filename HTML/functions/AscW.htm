Справочник функций

AscW

Возвращает Unicode-код символа.

AscW ( "char" )

Параметры

char Символ, для которого нужен код. Если передана строка, возвращается код её первого символа.

Возвращаемое значение

Возвращает Unicode-код указанного символа.

Примечания

Полный список значений см. в таблице Коды ASCII символов.
Подробное описание см. в разделе Поддержка Unicode.

См. также

Asc, Chr, ChrW

Пример

#include <MsgBoxConstants.au3>

Local $iCode = AscW("ß")
MsgBox($MB_SYSTEMMODAL, "", "Unicode code for ß: U+" & Hex($iCode, 6))
