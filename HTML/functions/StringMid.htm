Справочник функций

StringMid

Извлекает заданное количество символов из строки.

StringMid ( "string", start [, count = -1] )

Параметры

string Оцениваемая строка.
start Начальная позиция символа (1 = первый символ).
count [необязательный] Количество извлекаемых символов. По умолчанию весь остаток строки.

Возвращаемое значение

Возвращает извлечённую подстроку.

Примечания

Если start выходит за пределы строки, возвращается пустая строка. Если start допустим, но count выходит за пределы, возвращается весь остаток строки.

См. также

StringCompare, StringInStr, StringLeft, StringLen, StringLower, StringReplace, StringRight, StringSplit, StringTrimLeft, StringTrimRight, StringUpper

Пример

#include <MsgBoxConstants.au3>

; Получить 5 символов начиная с 10-й позиции строки.
Local $sString = StringMid("This is a sentence with whitespace.", 10, 5)
MsgBox($MB_SYSTEMMODAL, "", $sString)
