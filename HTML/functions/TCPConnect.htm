Справочник функций

TCPConnect

Создаёт сокет, подключённый к существующему серверу.

TCPConnect ( IPAddr, port )

Параметры

IPAddr IPv4-адрес в точечной записи, например "192.162.1.1".
port Порт, на котором будет установлено соединение через сокет.

Возвращаемое значение

Успех: Возвращает идентификатор основного сокета.
Ошибка: Возвращает -1 или 0 и устанавливает @error ≠ 0.
@error: -2 – нет соединения.
1 – неверный IPAddr.
2 – неверный порт.
10060 – истекло время ожидания соединения.
Попытка соединения не удалась: удалённая сторона не ответила за отведённое время,
либо установленное соединение разорвалось из-за отсутствия ответа от хоста.
См. пример с таймаутом соединения.
Код возврата Windows API WSAGetLastError (см. MSDN).

Примечания

Используется клиентом для взаимодействия с сервером.

См. также

TCPListen, TCPRecv, TCPSend, TCPStartup, TCPTimeout (Option)

Пример

Пример 1

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>

; Я клиент, запусти меня после сервера! (Сначала запусти пример TCPAccept).

Example()

Func Example()
    TCPStartup() ; Запустить TCP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальным переменным loopback-адрес IP и порт.
    Local $sIPAddress = "127.0.0.1" ; Этот IP-адрес работает только для тестирования на локальном компьютере.
    Local $iPort = 65432 ; Порт, используемый для подключения.

    ; Присвоить локальной переменной сокет и подключиться к слушающему сокету по указанным IP и порту.
    Local $iSocket = TCPConnect($sIPAddress, $iPort)

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        ; Сервер, вероятно, офлайн, или порт на сервере не открыт.
        Local $iError = @error
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), "", "Could not connect, Error code: " & $iError & @CRLF & _WinAPI_GetErrorMessage($iError))
        Return False
    Else
        MsgBox($MB_SYSTEMMODAL, "", "Connection successful")
    EndIf

    ; Закрыть сокет.
    TCPCloseSocket($iSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit

TCPConnect with timeout

; == TCPConnect с таймаутом

#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>

; Я клиент, запускайте меня после сервера, не более чем через 10 секунд! (Скрипт сервера – это пример TCPAccept).

Example()

Func Example()
    Local $sMsgBoxTitle = "AutoItVersion = " & @AutoItVersion

    TCPStartup() ; Запустить TCP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальным переменным loopback-адрес IP и порт.
    Local $sIPAddress = "127.0.0.1" ; Этот IP-адрес работает только для тестирования на локальном компьютере.
    Local $iPort = 65432 ; Порт, используемый для подключения.

    Opt("TCPTimeout", 1000)
    Local $nMaxTimeout = 10 ; скрипт прервётся, если сервер недоступен в течение 10 секунд

    Local $iSocket, $hTimer = TimerInit()

    While 1
        ; Присвоить локальной переменной сокет и подключиться к слушающему сокету по указанным IP и порту.
        $iSocket = TCPConnect($sIPAddress, $iPort)

        ; Если произошла ошибка – показать код ошибки и вернуть False.
        If @error = 10060 Then
            ; Таймаут – попробовать снова
            $nMaxTimeout -= 1
            If $nMaxTimeout <= 0 Then
                MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), $sMsgBoxTitle, "Could not connect, after " & Int(TimerDiff($hTimer) / 1000) & " sec")
                Return False
            EndIf
            ContinueLoop
        ElseIf @error Then
            ; Сервер, вероятно, офлайн, или порт на сервере не открыт.
            MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), $sMsgBoxTitle, "Could not connect, Error code: " & @error & @CRLF & _WinAPI_GetErrorMessage(@error))
            Return False
        Else
            MsgBox($MB_SYSTEMMODAL, $sMsgBoxTitle, "Connection successful after " & Int(TimerDiff($hTimer) / 1000) & " sec")
            ExitLoop
        EndIf

    WEnd

    ; Закрыть сокет.
    TCPCloseSocket($iSocket)
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit
