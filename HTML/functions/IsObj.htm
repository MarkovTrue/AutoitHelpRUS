Справочник функций

IsObj

Проверяет, является ли переменная или выражение объектом.

IsObj ( variable )

Параметры

variable Проверяемая переменная или выражение.

Возвращаемое значение

Успех: Возвращает 1, если выражение является объектной переменной.
Ошибка: Возвращает 0, если выражение не является объектной переменной.

Примечания

Подробное описание см. Типы данных.

См. также

GUICtrlCreateObj, ObjCreate, ObjEvent, ObjGet, ObjName, VarGetType

Пример

#include <MsgBoxConstants.au3>

Local $oShell = ObjCreate("shell.application")
If IsObj($oShell) Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable is an object")
Else
    MsgBox($MB_SYSTEMMODAL, "", "The variable is not an object")
EndIf
