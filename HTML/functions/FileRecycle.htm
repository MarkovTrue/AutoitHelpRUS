FileRecycle

Отправляет файл или каталог в корзину.

FileRecycle ( "source" )

Параметры

source Путь к файлу (или файлам) или каталогу, отправляемому в корзину. Допускаются символы подстановки * и ? – см. раздел Примечания.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0 (как правило, означает, что файл занят или не существует).

Примечания

Об использовании символов подстановки см. описание FileFindFirstFile().

Для отправки каталога в корзину укажите путь без завершающего обратного слеша.

См. также

DirRemove, FileDelete, FileMove, FileRecycleEmpty

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, "This is an example of using FileRecycle.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Показать содержимое файла, передавая FileRead путь к файлу вместо дескриптора от FileOpen.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & FileRead($sFilePath))

    ; Удаляем временный файл, отправив его в корзину.
    Local $iRecycle = FileRecycle($sFilePath)

    ; Выводим результат операции; при успехе открываем корзину.
    If $iRecycle Then
        MsgBox($MB_SYSTEMMODAL, "", "The file was successfuly deleted.")

        ; Открыть корзину по указанному CLSID.
        ShellExecute("::{645FF040-5081-101B-9F08-00AA002F954E}")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst deleting the file.")
    EndIf
EndFunc   ;==>Example
