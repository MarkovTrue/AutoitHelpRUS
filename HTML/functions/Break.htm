Справочник функций

Break

Включает или отключает возможность пользователя завершать скрипт из меню иконки в трее.

Break ( mode )

Параметры

mode Задаёт режим прерывания скрипта:
$BREAK_ENABLE (1) = прерывание разрешено (пользователь может выйти) (по умолчанию)
$BREAK_DISABLE (0) = прерывание запрещено (пользователь не может выйти)

Константы определены в файле "AutoItConstants.au3".

Возвращаемое значение

Отсутствует.

Примечания

Отключайте прерывание только при наличии веской причины.
В рабочем режиме AutoIt создаёт иконку в трее, и клик правой кнопкой по ней позволяет пользователю приостановить или завершить скрипт. Если Break() отключён (0), пользователь не сможет завершить скрипт этим способом.

Пример

Break(0) ;Отключить прерывание
