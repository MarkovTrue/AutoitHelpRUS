Справочник функций

SetError

Вручную устанавливает значение макроса @error (и, при необходимости, @extended и возвращаемое значение).

SetError ( code [, extended = 0 [, return value]] )

Параметры

code Целочисленное значение, устанавливаемое в макрос @error.
extended [необязательный] (по умолчанию 0) Целочисленное значение для макроса @extended. Устанавливает тот же макрос, что и SetExtended().
return value [необязательный] (по умолчанию 1) Значение, возвращаемое функцией.

Возвращаемое значение

Возвращает значение параметра return value.
@extended устанавливается в значение, переданное при вызове функции.

Примечания

При входе в пользовательскую функцию макрос @error сбрасывается в 0. Если SetError() не вызывается, @error остаётся 0 по завершении функции. Это означает, что для передачи кода ошибки вызывающему коду его необходимо явно установить. Кроме того, если вы проверяете @error внутри цикла While-WEnd, может потребоваться сохранить его значение в переменной.
Параметр extended необязателен и позволяет установить @error и @extended одновременно. Если нужно установить только @extended, рекомендуется использовать SetExtended().
Параметр return value необязателен и позволяет использовать синтаксис Return SetError(...) для одновременного задания возвращаемого значения и кода ошибки. Если значение не указано, возвращается 1, так как SetError() выполняется успешно.

@error допускает значения от -2147483648 до 2147483647.

См. также

SetExtended, Return

Пример

Пример 1

#include <MsgBoxConstants.au3>

Local $fResult = myDiv(5, 0)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Error", "Division by Zero")
Else
    MsgBox($MB_SYSTEMMODAL, "Result", $fResult)
EndIf
Exit

Func myDiv($iDividend, $iDivisor)
    If $iDividend = 0 And $iDivisor = 0 Then
        SetError(2) ; Неопределённая форма 0/0.
    ElseIf $iDivisor = 0 Then
        SetError(1) ; Деление на ноль.
    EndIf
    Return $iDividend / $iDivisor
EndFunc   ;==>myDiv

Пример 2

#include <MsgBoxConstants.au3>

Local $sReturn = Example()

; Вывести возвращаемое значение, @error и @extended.
MsgBox($MB_SYSTEMMODAL, "", "Return value = " & $sReturn & @CRLF & _
        "Value of @error is: " & @error & @CRLF & _
        "Value of @extended is: " & @extended)

Func Example()
    Return SetError(3, 10, "Some return value") ; Установить @error в 3, @extended в 10 и вернуть строку "Some return value.".
EndFunc   ;==>Example
