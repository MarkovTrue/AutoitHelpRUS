Справочник функций

Send

Отправляет имитированные нажатия клавиш в активное окно.

Send ( "keys" [, flag = 0] )

Параметры

keys Последовательность отправляемых клавиш.
flag [необязательный] Изменяет способ обработки keys:
$SEND_DEFAULT (0) = текст содержит специальные символы, такие как + и ! для имитации нажатий SHIFT и ALT (по умолчанию).
$SEND_RAW (1) = клавиши отправляются как есть.

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Отсутствует.

Примечания

Советы по использованию Send() см. в Приложении. AutoIt поддерживает отправку всех ASCII и расширенных ASCII-символов (0-255). Для отправки UNICODE-символов используйте параметр "ASC" и код нужного символа (см. {ASC} в конце таблицы ниже).

Синтаксис команды Send() схож с синтаксисом ScriptIt и команды Visual Basic "SendKeys". Символы отправляются как написаны, за исключением следующих:

'!'
Отправляет нажатие ALT. Например, Send("This is text!a") отправит текст "This is text", а затем нажмёт "ALT+a".

Примечание: некоторые программы различают регистр букв при комбинациях с ALT, то есть "!A" означает ALT+SHIFT+A, а "!a" – ALT+a. В случае сомнений используйте строчные буквы!

'+'
Отправляет нажатие SHIFT. Например, Send("Hell+o") отправит текст "HellO", а Send("!+a") отправит "ALT+SHIFT+a".

'^'
Отправляет нажатие CTRL. Например, Send("^!a") отправит "CTRL+ALT+a".

Примечание: некоторые программы различают регистр букв при комбинациях с CTRL, то есть "^A" означает CTRL+SHIFT+A, а "^a" – CTRL+a. В случае сомнений используйте строчные буквы!

'#'
Отправляет нажатие клавиши Windows. Например, Send("#r") отправит Win+r, открывая диалог Run().


С помощью SendCapslockMode можно отключить CAPS LOCK перед началом операции Send() и восстановить его по завершении.
Однако если пользователь удерживает Shift в момент вызова Send(), текст может быть отправлен в верхнем регистре.
Обойти это можно, отправив Send("{SHIFTDOWN}{SHIFTUP}") перед другими операциями Send().

Некоторые клавиатуры, например чешская, при включённом CAPS LOCK или нажатом Shift отправляют другие символы. В реализации AutoIt символ, набранный при CAPS LOCK, отправляется как Shift-символ, поэтому такой вариант работать не будет.

Для отправки ASCII-символа A (аналогично нажатию ALT+065 на цифровой клавиатуре):

Send("{ASC 065}")


(При двузначных ASCII-кодах добавляйте ведущий ноль, иначе используется устаревшая кодовая страница 437.)

Для отправки UNICODE-символов укажите код символа (десятичный или шестнадцатеричный). Например, следующие команды отправляют китайский иероглиф:

Send("{ASC 2709}") или Send("{ASC 0xA95}")


Одну клавишу можно отправлять несколько раз:

Send("{DEL 4}") ; нажать DEL 4 раза
Send("{S 30}") ; отправить 30 символов 'S'
Send("+{TAB 4}") ; нажать SHIFT+TAB 4 раза


При нулевом счётчике клавиша не отправляется, что удобно для программного управления:

Send("{M " & $iCount & "}") ; нажать M столько раз, сколько задано в $iCount


Удержание клавиши:

Send("{a down}") ; удержать клавишу A
Send("{a up}") ; отпустить клавишу A


Управление состоянием CapsLock, NumLock и ScrollLock:

Send("{NumLock on}") ; включить NumLock
Send("{CapsLock off}") ; выключить CapsLock
Send("{ScrollLock toggle}") ; переключить ScrollLock


Если нужно использовать переменную для счётчика:

$n = 4
Send("+{TAB " & $n & "}")


Для отправки ASCII-символа A четыре раза:

$x = Chr(65)
Send("{" & $x & " 4}")


На большинстве ноутбуков есть специальная клавиша Fn. Её эмуляция невозможна.

Примечание: при флаге 1 вся описанная "специальная" обработка отключается. Это удобно, когда нужно отправить текст из переменной ровно так, как он записан.


Пример: откройте "Свойства папки" (Панель управления) и опробуйте следующее:

Send("{TAB}") перейти к следующему элементу управления (кнопка, флажок и т.д.)
Send("+{TAB}") перейти к предыдущему элементу управления.
Send("^{TAB}") перейти на следующую вкладку диалогового окна
Send("^+{TAB}") перейти на предыдущую вкладку.
Send("{SPACE}") переключить флажок или нажать кнопку.
Send("{+}") обычно устанавливает флажок (если это "настоящий" флажок).
Send("{-}") обычно снимает флажок.
Send("{NumPadMult}") рекурсивно разворачивает папки в SysTreeView32.

Для доступа к пунктам меню используйте комбинации с Alt. Откройте Блокнот и попробуйте:
Send("!f"), отправляет Alt+f, клавишу доступа к меню "Файл" Блокнота. Попробуйте другие буквы!

Send("{DOWN}") перемещение вниз по меню.
Send("{UP}") перемещение вверх по меню.
Send("{LEFT}") переход к предыдущему меню или раскрытие подменю.
Send("{RIGHT}") переход к следующему меню или свёртывание подменю.


При запуске скрипта на удалённом компьютере через psexec (www.sysinternals.com) или beyondexec (www.beyondlogic.org) для отправки нажатий программе, запущенной через Run(), следует использовать ControlSend() или другие ControlXXX-функции для прямого взаимодействия с элементами управления. Send() даже с Opt("SendAttachMode", 1) работать не будет.
Использование режима -s при подключении может помочь получить расширенные права на удалённом компьютере.

Opt("SendKeyDelay",...) задаёт длительность паузы между нажатиями клавиш.
Opt("SendKeyDownDelay",...) задаёт время удержания клавиши перед отпусканием.
Установите оба параметра в 0, чтобы убрать все задержки. Это может быть необходимо, например, при блокировке системы ("#l"): без задержек клавиша WIN не зависнет в нажатом состоянии.

См. также

BlockInput, ControlSend, HotKeySet, SendAttachMode (Option), SendKeepActive, SendKeyDelay (Option), SendKeyDownDelay (Option), WinMenuSelectItem

Пример

Example()

Func Example()
    ; Эмулировать нажатие Win+R для открытия диалога "Выполнить".
    Send("#r")

    ; Ждать появления диалога "Выполнить" не более 10 секунд.
    WinWait("Run", "", 10)

    ; Эмулировать ввод notepad.exe и нажатие клавиши Enter.
    Send("notepad.exe{Enter}")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Эмулировать ввод строки и нажатие F5 для вставки даты и времени в поле редактирования Блокнота.
    Send("Today's time/date is {F5}")

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)

    ; Сейчас появится экран с просьбой сохранить изменения, classname этого окна
    ; "#32770" и симуляция "TAB" для перехода на вторую кнопку, где симулируется "ENTER" чтобы не "сохранять файл"
    WinWaitActive("[CLASS:#32770]")
    Sleep(500)
    Send("{TAB}{ENTER}")
EndFunc   ;==>Example
