Справочник функций

GUICtrlSetDefBkColor

Устанавливает цвет фона по умолчанию для всех элементов управления окна GUI.

GUICtrlSetDefBkColor ( defbkcolor [, winhandle] )

Параметры

defbkcolor Цвет фона по умолчанию для всех элементов управления.
winhandle [необязательный] Дескриптор окна, возвращаемый функцией GUICreate() (по умолчанию используется ранее применённое окно).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

См. также

GUICreate, GUICtrlSetBkColor, GUICtrlSetDefColor

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("test GUISetTextColor", 100, 100) ; создаст диалоговое окно, отображаемое по центру

    GUICtrlSetDefBkColor(0xFF0000) ; изменит цвет текста для всех определённых элементов управления

    GUICtrlCreateLabel("label", 10, 5)

    GUICtrlCreateRadio("radio", 10, 25, 50)
    GUICtrlSetBkColor(-1, 0x0000FF) ; изменит цвет текста для указанного элемента управления

    GUICtrlCreateButton("button", 10, 55)

    GUISetState(@SW_SHOW) ; отобразит пустое диалоговое окно

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
