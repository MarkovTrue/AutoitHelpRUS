Справочник функций

SplashTextOn

Создаёт настраиваемое всплывающее текстовое окно.

SplashTextOn ( "title", "text" [, w = 500 [, h = 400 [, x pos [, y pos [, opt = 0 [, "fontname" [, fontsz = 12 [, fontwt]]]]]]]] )

Параметры

title Заголовок всплывающего окна.
text Текст всплывающего окна.
w [необязательный] Ширина окна в пикселях (по умолчанию 500).
h [необязательный] Высота окна в пикселях (по умолчанию 400).
x pos [необязательный] Позиция окна от левого края экрана в пикселях (по умолчанию – по центру).
y pos [необязательный] Позиция окна от верхнего края экрана в пикселях (по умолчанию – по центру).
opt [необязательный] Суммируйте нужные значения. По умолчанию – "текст по центру / всегда поверх / с заголовком":
$DLG_CENTERONTOP (0) = текст по центру / всегда поверх / с заголовком (по умолчанию)
$DLG_NOTITLE (1) = окно с тонкой рамкой без заголовка
$DLG_NOTONTOP (2) = без атрибута "всегда поверх"
$DLG_TEXTLEFT (4) = текст по левому краю
$DLG_TEXTRIGHT (8) = текст по правому краю
$DLG_MOVEABLE (16) = окно можно перемещать
$DLG_TEXTVCENTER (32) = текст по вертикальному центру

Константы определены в AutoItConstants.au3.
fontname [необязательный] Имя используемого шрифта. Если задана пустая строка или шрифт не найден, используется шрифт GUI по умолчанию.
fontsz [необязательный] Размер шрифта (по умолчанию 12; стандартные размеры: 6 8 9 10 11 12 14 16 18 20 22 24 26 28 36 48 72).
fontwt [необязательный] Насыщенность шрифта в диапазоне от 0 до 1000 (400 – обычный, 700 – жирный). При значении 0 используется насыщенность по умолчанию.
Для удобства определены следующие константы:
$FW_DONTCARE (0) – насыщенность по умолчанию
$FW_THIN (100)
$FW_EXTRALIGHT (200)
$FW_LIGHT (300)
$FW_NORMAL (400)
$FW_MEDIUM (500)
$FW_SEMIBOLD (600)
$FW_BOLD (700)
$FW_EXTRABOLD (800)
$FW_HEAVY (900)
Константы определены в FontConstants.au3.

Возвращаемое значение

Возвращает дескриптор всплывающего окна, который можно использовать в ControlSetText().

Примечания

Для пропуска необязательного параметра с сохранением значения по умолчанию передайте:
    "" для строковых параметров
    –1 для числовых параметров

Одновременно допускается только одно окно SplashImage/Text. Для смены изображения или текста достаточно снова вызвать SplashImageOn() или SplashTextOn() с новыми данными.

Для обновления текста без мерцания рекомендуется использовать ControlSetText().
Если текст выровнен по центру и многострочный, ControlSetText() не изменит количество строк, заданное при вызове SplashTextOn().

Окно с opt=1 нельзя перемещать и активировать щелчком мыши.

Стандартные шрифты: Arial, Comic Sans MS, Courier New, Lucida Console, Microsoft Sans Serif, System, Tahoma, Times New Roman, WingDings.
Полный список шрифтов Windows см. в Приложении.

Для вывода нескольких строк используйте @LF.

См. также

ControlSetText, MsgBox, SplashImageOn, SplashOff, ToolTip

Пример

#include <AutoItConstants.au3>

SplashTextOn("Title", "Message goes here.", -1, -1, -1, -1, $DLG_TEXTLEFT, "", 24)
Sleep(3000)
SplashOff()

; ; С МЕРЦАНИЕМ
Local $sMessage = ""
SplashTextOn("TitleFoo", $sMessage, -1, -1, -1, -1, $DLG_TEXTLEFT, "")
For $i = 1 To 20
    $sMessage = $sMessage & $i & @CRLF
    SplashTextOn("TitleFoo", $sMessage, -1, -1, -1, -1, $DLG_TEXTLEFT, "")
    Sleep(100)
Next

; ; БЕЗ МЕРЦАНИЯ
$sMessage = ""
SplashTextOn("TitleFoo", $sMessage, -1, -1, -1, -1, $DLG_TEXTLEFT, "")
For $i = 1 To 20
    $sMessage = $sMessage & $i & @CRLF
    ControlSetText("TitleFoo", "", "Static1", $sMessage)
    Sleep(100)
Next
