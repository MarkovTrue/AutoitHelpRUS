Registry functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
RegDelete Удаляет раздел или значение из реестра.
RegEnumKey Читает имя подраздела по его порядковому номеру.
RegEnumVal Читает имя значения реестра по его порядковому номеру.
RegRead Читает значение из реестра.
RegWrite Создаёт раздел или значение в реестре.
