Справочник функций

TCPStartup, UDPStartup

Запускает службы TCP или UDP.

TCPStartup ( )
UDPStartup ( )

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: Код возврата Windows API WSAStartup (см. MSDN).

Примечания

Необходимо вызвать TCPShutdown(), чтобы избежать утечки памяти.
На каждый успешный вызов TCPStartup() скрипт должен вызвать TCPShutdown().
UDPStartup() является псевдонимом TCPStartup().

См. также

TCPAccept, TCPCloseSocket, TCPConnect, TCPListen, TCPNameToIP, TCPRecv, TCPSend, TCPShutdown, UDPCloseSocket

Пример

Start the TCP service.

; == Запустить TCP-сервис.

Example()

Func Example()
    TCPStartup() ; Запустить TCP-сервис.

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit
