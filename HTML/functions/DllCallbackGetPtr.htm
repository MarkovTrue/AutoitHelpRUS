Справочник функций

DllCallbackGetPtr

Возвращает указатель на функцию обратного вызова, который можно передать в Win32 API.

DllCallbackGetPtr ( handle )

Параметры

handle Дескриптор DllCallback, возвращённый функцией DllCallbackRegister().

Возвращаемое значение

Успех: Возвращает указатель на функцию обратного вызова.
Ошибка: Возвращает 0.

Примечания

Используйте DllCallbackGetPtr() для передачи адреса функции обратного вызова в Win32 API при использовании DllCall().

См. также

DllCall, DllCallbackFree, DllCallbackRegister

Пример

#include <MsgBoxConstants.au3>

; Создать функцию обратного вызова.
Local $hHandle = DllCallbackRegister("_EnumWindowsProc", "int", "hwnd;lparam")

; Вызвать EnumWindows.
DllCall("user32.dll", "int", "EnumWindows", "ptr", DllCallbackGetPtr($hHandle), "lparam", 10)

; Удалить функцию обратного вызова.
DllCallbackFree($hHandle)

; Процедура обратного вызова
Func _EnumWindowsProc($hWnd, $lParam)
    ; Если заголовок пуст или окно невидимо – продолжить перечисление.
    If WinGetTitle($hWnd) = "" Or BitAND(WinGetState($hWnd), 2) = 0 Then Return 1

    Local $iRes = MsgBox(($MB_OKCANCEL + $MB_SYSTEMMODAL), _
            WinGetTitle($hWnd), "$hWnd=" & $hWnd & @CRLF & _
            "$lParam=" & $lParam & @CRLF & _
            "$hWnd(type)=" & VarGetType($hWnd))

    If $iRes <> $IDOK Then Return 0 ; Нажата кнопка Cancel/Close – вернуть 0 для остановки перечисления.

    Return 1 ; Вернуть 1 для продолжения перечисления.
EndFunc   ;==>_EnumWindowsProc
