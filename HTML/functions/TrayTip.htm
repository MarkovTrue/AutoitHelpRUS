Справочник функций

TrayTip

Отображает всплывающую подсказку в виде воздушного шарика от значка AutoIt.

TrayTip ( "title", "text", timeout [, option = 0] )

Параметры

title Текст, отображаемый жирным шрифтом в заголовке подсказки. (максимум 63 символа).
text Сообщение, отображаемое во всплывающей подсказке. (максимум 255 символов).
timeout Приблизительное время отображения подсказки (в секундах). (Windows имеет ограничение 10–30 секунд и не всегда точно соблюдает указанное значение.).
option [необязательный]
$TIP_ICONNONE (0) = без значка (по умолчанию)
$TIP_ICONASTERISK (1) = значок информации
$TIP_ICONEXCLAMATION (2) = значок предупреждения
$TIP_ICONHAND (3) = значок ошибки
$TIP_NOSOUND (16) = отключить звук

Константы определены в TrayConstants.au3.

Возвращаемое значение

Отсутствует.

Примечания

Подсказка закрывается при исчезновении значка AutoIt в трее. Поэтому TrayTip() может не отображаться, если установлен параметр Opt("TrayIconHide", 1) или пользователь отключил всплывающие подсказки через реестр!

Windows XP обычно воспроизводит звук при отображении подсказки. Звук можно отключить, добавив $TIP_NOSOUND к параметру option или задав пустой заголовок.

Чтобы скрыть отображаемую подсказку, вызовите функцию с пустой строкой текста (и любым заголовком).

См. также

MsgBox, ToolTip, TrayIconHide (Option), TraySetToolTip

Пример

#include <MsgBoxConstants.au3>
#include <TrayConstants.au3>

TrayTip("I'm a title", "I'm the message", 0, $TIP_ICONASTERISK)
MsgBox($MB_SYSTEMMODAL, "", "Press OK to reset the tip.")
TrayTip("clears any tray tip", "", 0)
MsgBox($MB_SYSTEMMODAL, "", "Press OK to see another tip.")
TrayTip("", "A different tray tip.", 5)
Sleep(5000)
