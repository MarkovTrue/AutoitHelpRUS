WinGetText

Получает текст из окна.

WinGetText ( "title" [, "text"] )

Параметры

title Заголовок / дескриптор / класс окна, текст которого нужно получить. См. Заголовки и текст окон (расширенно).
text [необязательный] Текст окна, из которого нужно получить текст. См. Заголовки и текст окон (основы).

Возвращаемое значение

Успех: Возвращает строку с прочитанным текстом окна.
Ошибка: Возвращает "", если окно не найдено.

Примечания

Можно получить до 64 КБ текста окна. WinGetText() работает со свёрнутыми окнами, но со скрытыми окнами работает только если задано Opt("WinDetectHiddenText", 1)
Если критериям WinGetText() соответствует несколько окон, возвращается информация для окна, которое было активным последним.
Чтобы получить текст активного окна, используйте WinGetText("[active]").

См. также

ControlGetText, WinGetClassList, WinGetTitle

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получаем текст активного окна.
    Local $sText = WinGetText("[ACTIVE]")

    ; Выводим текст окна.
    MsgBox($MB_SYSTEMMODAL, "", $sText)
EndFunc   ;==>Example
