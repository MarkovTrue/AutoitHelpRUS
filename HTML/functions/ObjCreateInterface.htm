ObjCreateInterface

Создаёт ссылку на объект по имени класса/указателю объекта, идентификатору интерфейса и строке описания.

ObjCreateInterface ( "CLSID" , "IID" [, "interface_description",[flag = True]] )

Параметры

CLSID Идентификатор класса или указатель объекта. Идентификатор класса может быть задан как ProgID или строковое представление CLSID.
IID Строковое представление идентификатора интерфейса.
interface_description [необязательный] Строка описания vtable объекта. Используйте ключевое слово Default для доступа к IDispatch в dual-интерфейсах.
flag [необязательный] По умолчанию True – интерфейс объекта наследуется от IUnknown.

Возвращаемое значение

Успех: Возвращает объект.
Ошибка: Устанавливает @error ≠ 0.

Примечания

ObjCreateInterface() создаёт объекты с методами, перечисленными в строке описания интерфейса.
Для COM-объектов (на базе IUnknown) первые три метода – всегда QueryInterface, AddRef и Release. Их не нужно указывать в строке описания.

Методы в строке описания должны быть перечислены в порядке vtable интерфейса. Некорректное описание может привести к нестабильности или аварийному завершению AutoIt.

Формат строки описания:
$sTagInterface = "MethodName1 RetType(ParamType1;ParamType2;...); MethodName2 RetType(..."

По умолчанию AutoIt использует соглашение вызова 'stdcall'. Для 'cdecl' добавьте ':cdecl' после типа возврата.

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Объявить CLSID, IID и описание интерфейса для ITaskbarList.
    ; Методы IUnknown описывать не нужно.
    Local Const $sCLSID_TaskbarList = "{56FDF344-FD6D-11D0-958A-006097C9A090}"
    Local Const $sIID_ITaskbarList = "{56FDF342-FD6D-11D0-958A-006097C9A090}"
    Local Const $sTagITaskbarList = "HrInit hresult(); AddTab hresult(hwnd); DeleteTab hresult(hwnd); ActivateTab hresult(hwnd); SetActiveAlt hresult(hwnd);"

    ; Создать объект.
    Local $oTaskbarList = ObjCreateInterface($sCLSID_TaskbarList, $sIID_ITaskbarList, $sTagITaskbarList)

    ; Инициализировать объект ITaskbarList.
    $oTaskbarList.HrInit()

    ; Запустить Блокнот.
    Run("notepad.exe")

    ; Ожидать появления окна Блокнота и получить его дескриптор.
    Local $hNotepad = WinWait("[CLASS:Notepad]")

    ; Сообщить пользователю, что нужно найти.
    MsgBox($MB_SYSTEMMODAL, "", "Look in the Taskbar and you should see an entry for Notepad." & @CRLF & @CRLF & "Press OK to continue.")

    ; Удалить запись Блокнота с панели задач.
    $oTaskbarList.DeleteTab($hNotepad)

    ; Попросить пользователя проверить снова.
    MsgBox($MB_SYSTEMMODAL, "", "Look in the Taskbar. There should no longer be a Notepad entry but Notepad is still running." & @CRLF & @CRLF & "Press OK to continue.")

    ; Закрыть Блокнот.
    WinClose($hNotepad)
EndFunc   ;==>Example
