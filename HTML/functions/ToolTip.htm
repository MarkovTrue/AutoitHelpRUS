ToolTip

Создаёт всплывающую подсказку в любом месте экрана.

ToolTip ( "text" [, x [, y [, "title" [, icon = 0 [, options]]]]] )

Параметры

text Текст подсказки. Пустая строка убирает показанную подсказку.
x [необязательный] Координата x подсказки.
y [необязательный] Координата y подсказки.
title [необязательный] Заголовок подсказки.
icon [необязательный] Стандартный значок рядом с заголовком (нужен заголовок):
$TIP_NOICON (0) = без значка
$TIP_INFOICON (1) = информационный значок
$TIP_WARNINGICON (2) = значок предупреждения
$TIP_ERRORICON (3) = значок ошибки

Константы определены в AutoItConstants.au3.
options [необязательный] Параметры отображения подсказки (можно складывать):
$TIP_BALLOON (1) = показывать в виде выноски
$TIP_CENTER (2) = центрировать подсказку по координатам x,y, а не использовать их как верхний левый угол.
$TIP_FORCEVISIBLE (4) = всегда держать подсказку видимой, при необходимости вписывая её в границы монитора. При нескольких мониторах подсказка притягивается к ближайшему монитору.

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если длина заголовка больше 99.

Примечания

Чтобы пропустить необязательный параметр, сохранив его значение по умолчанию, используйте ключевое слово Default.

Если координаты x и y опущены, подсказка размещается рядом с курсором мыши.
Подсказка остаётся на экране, пока скрипт не завершится или не будет вызвана ToolTip("").
Для многострочных подсказок можно использовать @CR или @LF.
Чтобы показать значок, задайте непустой заголовок: значок располагается в одной строке с заголовком, поэтому заголовок обязателен.
С флагом центрирования центр подсказки будет в указанных координатах.
С флагом центрирования и выноской текст центрируется на выноске, а сама она указывает на заданные координаты.

См. также

MsgBox, SplashTextOn, TrayTip

Пример

; Создаём всплывающую подсказку в верхнем левом углу экрана.

ToolTip("This is a tooltip", 0, 0)
Sleep(2000) ; Пауза, чтобы подсказка успела отобразиться.
