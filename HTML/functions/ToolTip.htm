Справочник функций

ToolTip

Создаёт всплывающую подсказку в любом месте экрана.

ToolTip ( "text" [, x [, y [, "title" [, icon = 0 [, options]]]]] )

Параметры

text Текст всплывающей подсказки. Пустая строка скрывает отображаемую подсказку.
x [необязательный] Координата X всплывающей подсказки.
y [необязательный] Координата Y всплывающей подсказки.
title [необязательный] Заголовок всплывающей подсказки.
icon [необязательный] Предустановленный значок рядом с заголовком (требуется заголовок):
$TIP_NOICON (0) – без значка
$TIP_INFOICON (1) – информационный значок
$TIP_WARNINGICON (2) – значок предупреждения
$TIP_ERRORICON (3) – значок ошибки

Константы определены в "AutoItConstants.au3".
options [необязательный] Параметры отображения подсказки (можно суммировать):
$TIP_BALLOON (1) – отображать как выноску (Balloon Tip)
$TIP_CENTER (2) – центрировать подсказку по заданным координатам x,y, а не располагать верхний левый угол в этой точке
$TIP_FORCEVISIBLE (4) – всегда держать подсказку в пределах монитора; при нескольких мониторах подсказка "прилипает" к ближайшему

Константы определены в "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если длина заголовка превышает 99 символов.

Примечания

Для пропуска необязательного параметра с сохранением его значения по умолчанию используйте ключевое слово Default.

Если координаты x и y не указаны, подсказка размещается рядом с курсором мыши.
Подсказка отображается до завершения скрипта или вызова ToolTip("").
Для многострочных подсказок можно использовать @CR или @LF.
Для отображения значка необходимо указать непустой заголовок: значок располагается в той же строке, что и заголовок.
При использовании флага центрирования центр подсказки будет находиться в указанных координатах.
При использовании флага центрирования с выноской (Balloon Tip) текст центрируется внутри выноски, которая указывает на заданные координаты.

См. также

MsgBox, SplashTextOn, TrayTip

Пример

; Создаём всплывающую подсказку в верхнем левом углу экрана.

ToolTip("This is a tooltip", 0, 0)
Sleep(2000) ; Пауза, чтобы подсказка успела отобразиться.
