IsPtr

Проверяет, является ли базовый тип переменной указателем.

IsPtr ( variable )

Параметры

variable Проверяемая переменная или выражение.

Возвращаемое значение

Успех: Возвращает 1, если выражение является типом указателя.
Ошибка: Возвращает 0, если выражение не является типом указателя.

Примечания

Подробное описание см. Типы данных.

См. также

IsArray, IsBool, IsFloat, IsHWnd, IsInt, IsMap, IsNumber, IsString, Ptr

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Проверить, является ли $hWnd допустимым указателем, и отобразить результат.
    If IsPtr($hWnd) Then
        MsgBox($MB_SYSTEMMODAL, "", "It's a valid Ptr")
    Else
        MsgBox($MB_SYSTEMMODAL, "", "It's not a valid Ptr")
    EndIf

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
