Справочник функций

WinActive

Проверяет, существует ли указанное окно и является ли оно активным в данный момент.

WinActive ( "title" [, "text"] )

Параметры

title Заголовок/hWnd/класс проверяемого окна. См. особенности задания заголовка.
text [необязательный] Текст проверяемого окна. По умолчанию – пустая строка. См. особенности задания текста.

Возвращаемое значение

Успех: Возвращает дескриптор окна, если оно активно.
Ошибка: Возвращает 0 в остальных случаях.

См. также

WinExists, WinTitleMatchMode (Option), WinWait, WinWaitActive, WinWaitClose, WinWaitNotActive

Пример

#include <MsgBoxConstants.au3>

If WinActive("[CLASS:Notepad]") Then ; Проверяем, активен ли Блокнот в данный момент.
    MsgBox($MB_SYSTEMMODAL, "", "WinActive" & @CRLF & "Notepad is active.")
Else
    MsgBox($MB_SYSTEMMODAL, "", "WinActive" & @CRLF & "Notepad is not active.")
EndIf
