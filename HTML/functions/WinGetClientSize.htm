Справочник функций

WinGetClientSize

Получает размер клиентской области заданного окна.

WinGetClientSize ( "title" [, "text"] )

Параметры

title Заголовок / дескриптор / класс окна, размер которого нужно получить. См. Заголовки и текст окон (расширенно).
text [необязательный] Текст окна, размер которого нужно получить. См. Заголовки и текст окон (основы).

Возвращаемое значение

Успех: Возвращает массив из 2 элементов со следующими данными:
$aArray[0] = Ширина клиентской области окна
$aArray[1] = Высота клиентской области окна
Ошибка: Устанавливает @error ≠ 0, если окно не найдено.

Примечания

Если окно свёрнуто, возвращаемые ширина и высота равны нулю. Однако WinGetClientSize() корректно работает с (несвёрнутыми) скрытыми окнами. Если задан заголовок окна "Program Manager", функция вернёт размер рабочего стола. Если критериям соответствует несколько окон, используется то, которое было активным последним.

См. также

WinGetPos, WinMove

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Получаем клиентскую область окна Блокнота по дескриптору, возвращённому WinWait.
    Local $aClientSize = WinGetClientSize($hWnd)

    ; Выводим высоту и ширину клиентской области.
    MsgBox($MB_SYSTEMMODAL, "", "Width: " & $aClientSize[0] & @CRLF & "Height: " & $aClientSize[1])

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
