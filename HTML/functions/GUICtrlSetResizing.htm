Справочник функций

GUICtrlSetResizing

Определяет метод изменения размеров элемента управления.

GUICtrlSetResizing ( controlID, resizing )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
resizing Допустимые значения см. в таблице значений докинга ниже (при необходимости суммируйте несколько значений).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

При изменении размеров окна GUI входящие в него элементы управления реагируют – эта функция определяет, как именно. Чтобы окно GUI допускало изменение размеров, оно должно быть создано со стилями $WS_SIZEBOX и $WS_SYSMENU. См. GUICreate().

    Таблица значений докинга

Константа Значение Фиксируется
$GUI_DOCKAUTO 1 Изменение размера и положения пропорционально новому размеру окна
$GUI_DOCKLEFT 2 Левая сторона
$GUI_DOCKRIGHT 4 Правая сторона
$GUI_DOCKHCENTER 8 Положение не смещается относительно горизонтального центра
$GUI_DOCKTOP 32 Верхняя сторона
$GUI_DOCKBOTTOM 64 Нижняя сторона
$GUI_DOCKVCENTER 128 Положение не смещается относительно вертикального центра
$GUI_DOCKWIDTH 256 Ширина не изменяется
$GUI_DOCKHEIGHT 512 Высота не изменяется
Составные значения
$GUI_DOCKSIZE 768 (256+512) Размер не изменяется
$GUI_DOCKMENUBAR 544 (512+32) Элемент остаётся вверху окна без изменения высоты
$GUI_DOCKSTATEBAR 576 (512+64) Элемент остаётся внизу окна без изменения высоты
$GUI_DOCKALL 802 (2+32+256+512) Элемент не перемещается при изменении размеров
$GUI_DOCKBORDERS 102 (2+4+32+64) Элемент растягивается вместе с окном

Поведение изменения размеров по умолчанию зависит от типа элемента управления – см. документацию конкретного элемента.
Значение по умолчанию для любого элемента можно задать через GUIResizeMode (Option).

Автоматическое событие изменения размеров можно отключить, установив GUIEventOptions (Option) в 1.

См. также

GUICtrlCreate..., GUIEventOptions (Option), GUIResizeMode (Option)

Пример

#include <EditConstants.au3>
#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Opt("GUICoordMode", 2)
    GUICreate("My InputBox", 190, 114, -1, -1, $WS_SIZEBOX + $WS_SYSMENU) ; начало определения

    GUISetFont(8, -1, "Arial")

    GUICtrlCreateLabel("Prompt", 8, 7) ; добавить подсказку
    GUICtrlSetResizing(-1, $GUI_DOCKLEFT + $GUI_DOCKTOP)

    Local $idInp_Button = GUICtrlCreateInput("Default", -1, 3, 175, 20, $ES_PASSWORD) ; добавить поле ввода
    GUICtrlSetState($idInp_Button, $GUI_FOCUS)
    GUICtrlSetResizing($idInp_Button, $GUI_DOCKBOTTOM + $GUI_DOCKHEIGHT)

    Local $idButton_OK = GUICtrlCreateButton("OK", -1, 3, 75, 24) ; добавить кнопку закрытия GUI
    GUICtrlSetResizing($idButton_OK, $GUI_DOCKBOTTOM + $GUI_DOCKSIZE + $GUI_DOCKHCENTER)

    Local $idButton_Cancel = GUICtrlCreateButton("Cancel", 25, -1) ; добавить кнопку закрытия GUI
    GUICtrlSetResizing($idButton_Cancel, $GUI_DOCKBOTTOM + $GUI_DOCKSIZE + $GUI_DOCKHCENTER)

    GUISetState(@SW_SHOW) ; отобразить GUI

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
