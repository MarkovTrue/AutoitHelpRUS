Справочник функций

TrayGetMsg

Опрашивает трей на наличие произошедших событий.

TrayGetMsg ( )

Возвращаемое значение

Returns an event.

Возвращаемое "событие" – это ID элемента управления, отправившего сообщение, либо специальное событие (например, щелчок мыши по значку в трее). Если сообщения нет, событие равно 0.

Event IDs
$TRAY_EVENT_NONE (0): Нет события.
Control ID: ID элемента, отправившего сообщение.
$TRAY_EVENT_PRIMARYDOWN: нажата основная кнопка мыши
$TRAY_EVENT_PRIMARYUP: отпущена основная кнопка мыши
$TRAY_EVENT_SECONDARYDOWN: нажата вторичная кнопка мыши
$TRAY_EVENT_SECONDARYUP: отпущена вторичная кнопка мыши
$TRAY_EVENT_MOUSEOVER: указатель мыши проходит над значком в трее
$TRAY_EVENT_PRIMARYDOUBLE: двойное нажатие основной кнопки мыши
$TRAY_EVENT_SECONDARYDOUBLE: двойное нажатие вторичной кнопки мыши

Константы определены в TrayConstants.au3

Примечания

Функция автоматически снижает нагрузку на процессор при необходимости, поэтому её можно безопасно использовать в плотных циклах без монополизации CPU.

См. также

TrayCreateItem, TrayCreateMenu, TrayItemSetOnEvent

Пример

#NoTrayIcon

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>
#include <TrayConstants.au3> ; Требуется для константы $TRAY_ICONSTATE_SHOW.

Opt("TrayMenuMode", 3) ; Стандартные пункты меню трея не будут показаны, и пункты не помечаются при выборе. Это опции 1 и 2 для TrayMenuMode.

Example()

Func Example()
    Local $idAbout = TrayCreateItem("About")
    TrayCreateItem("") ; Создать разделительную линию.

    Local $idExit = TrayCreateItem("Exit")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    While 1
        Switch TrayGetMsg()
            Case $idAbout ; Показать окно сообщения с версией AutoIt и путём установки исполняемого файла AutoIt.
                MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                        "Version: " & @AutoItVersion & @CRLF & _
                        "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", $STR_NOCASESENSEBASIC, -1) - 1)) ; Найти каталог полного пути.

            Case $idExit ; Выйти из цикла.
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>Example
