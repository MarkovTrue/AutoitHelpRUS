Справочник функций

FileExists

Проверяет существование файла или каталога.

FileExists ( "path" )

Параметры

Path Путь к проверяемому каталогу или файлу.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если путь или файл не существует.

Примечания

FileExists() возвращает 0, если указан флоппи-дисковод без вставленного диска.

Если скрипт работает в 32-разрядном режиме (@AutoItX64 = 0) под 64-разрядной Windows (@OSArch = "X64") и требуется обратиться к @WindowsDir\System32, файлы x64 найдены не будут. В этом случае используйте @WindowsDir\Sysnative.

См. также

DriveStatus, FileGetAttrib

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    Local $iFileExists = FileExists($sFilePath)

    ; Выводим сообщение о том, существует ли файл.
    If $iFileExists Then
        MsgBox($MB_SYSTEMMODAL, "", "The file exists." & @CRLF & "FileExist returned: " & $iFileExists)
    Else
        MsgBox($MB_SYSTEMMODAL, "", "The file doesn't exist." & @CRLF & "FileExist returned: " & $iFileExists)
    EndIf

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
