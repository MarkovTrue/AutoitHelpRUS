StringFromASCIIArray

Преобразует массив ASCII-кодов в строку.

StringFromASCIIArray ( array,[start = 0 [, end = -1 [, encoding = 0]]] )

Параметры

array Массив ASCII-кодов для преобразования в символы.
start [необязательный] Начальный индекс (с нуля) для обработки (по умолчанию 0).
end [необязательный] Конечный индекс (с нуля) для обработки (по умолчанию UBound($aArray)). Символ с этим индексом в результат НЕ включается.
encoding [необязательный] Кодировка значений в массиве:
$SE_UTF16 (0) = UTF-16 (по умолчанию)
$SE_ANSI (1) = ANSI
$SE_UTF8 (2) = UTF-8

Константы определены в StringConstants.au3.

Возвращаемое значение

Успех: Возвращает строку, составленную из символов, соответствующих переданным ASCII-кодам.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@error: 1 – входной параметр не является массивом.
2 – недопустимый начальный индекс.

Примечания

Возвращаемая строка может содержать встроенные символы Chr(0), оставаясь при этом строковым типом. Большинство строковых функций останавливаются на первом Chr(0); если необходим доступ ко всему содержимому, используйте StringToBinary() для преобразования в двоичную строку с сохранением всех данных.

При создании массива вручную (в отличие от массива, возвращённого StringToASCIIArray()) коды должны быть в кодировке Unicode.

Пример

#include <Array.au3> ; Нужно только для _ArrayDisplay().
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Преобразовать строку в массив ASCII.
    Local $aArray = StringToASCIIArray("This is a sentence with whitespace.")

    ; Показать массив, чтобы убедиться, что он содержит ASCII-значения каждого символа строки.
    _ArrayDisplay($aArray)

    ; Преобразовать массив в строку.
    Local $sString = StringFromASCIIArray($aArray)

    ; Показать строку, чтобы убедиться, что она совпадает с исходной до преобразования в массив.
    MsgBox($MB_SYSTEMMODAL, "", $sString)
EndFunc   ;==>Example
