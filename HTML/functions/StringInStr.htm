Справочник функций

StringInStr

Проверяет, содержит ли строка заданную подстроку.

StringInStr ( "string", "substring" [, casesense = 0 [, occurrence = 1 [, start = 1 [, count]]]] )

Параметры

string Оцениваемая строка.
substring Искомая подстрока.
casesense [необязательный] Флаг чувствительности к регистру.
$STR_NOCASESENSE (0) = без учёта регистра, с использованием локали пользователя (по умолчанию)
$STR_CASESENSE (1) = с учётом регистра
$STR_NOCASESENSEBASIC (2) = без учёта регистра, ускоренное базовое сравнение

Константы определены в StringConstants.au3.
occurrence [необязательный] Порядковый номер вхождения подстроки. Отрицательное значение – поиск справа налево. По умолчанию 1 (первое вхождение).
start [необязательный] Начальная позиция поиска.
count [необязательный] Количество символов для поиска. Ограничивает область поиска частью строки. См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает позицию подстроки.
Ошибка: Возвращает 0, если подстрока не найдена.
@error: 1 – недопустимый параметр start или occurrence.

Примечания

Позиция первого символа – 1.

Параметр count не должен быть меньше длины искомой подстроки. Совместно с параметром start он ограничивает поиск частью строки. Следующие два вызова эквивалентны:
StringInStr("the string to search", string, 0, 1, 1, 11)
StringInStr(StringMid("the string to search", 1, 11), string)

См. также

StringCompare, StringLeft, StringLen, StringLower, StringMid, StringRegExp, StringRight, StringSplit, StringTrimLeft, StringTrimRight, StringUpper

Пример

#include <MsgBoxConstants.au3>

; Найти позицию первого вхождения строки "white" в предложении.
Local $iPosition = StringInStr("This is a sentence with whitespace.", "white")
MsgBox($MB_SYSTEMMODAL, "", "The search string 'white' first appears at position: " & $iPosition)
