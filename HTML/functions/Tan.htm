Справочник функций

Tan

Вычисляет тангенс числа.

Tan ( expression )

Параметры

expression Значение в радианах.

Возвращаемое значение

Возвращает тригонометрический тангенс числа.

Примечания

1° = Pi / 180 радиан.
Обратите внимание, что Tan() "бесконечен" для значений ... -3Pi/2, -Pi/2, Pi/2, 3Pi/2, 5Pi/2, ... и для таких кратных Pi/2 возвращает приблизительно 1.63317787283838e+016.

См. также

ACos, ASin, ATan, Cos, Sin

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной константе приближённое значение числа Пи.
    Local Const $PI = 3.141592653589793

    ; Присваиваем локальной переменной тангенс PI / 4.
    Local $fTan1 = Tan($PI / 4)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fTan1)

    ; Присвоить локальной переменной формулу перевода радиан в градусы (равно одному градусу в радианах).
    Local $iDegToRad = $PI / 180

    ; Присваиваем локальной переменной тангенс 90 градусов (предварительно переведённых в радианы).
    Local $fTan2 = Tan(90 * $iDegToRad)

    ; Показать результат.
    MsgBox($MB_SYSTEMMODAL, "", $fTan2)
EndFunc   ;==>Example
