Справочник функций

ConsoleRead

Читает из потока STDIN процесса AutoIt-скрипта.

ConsoleRead ( [peek = False [, binary = False]] )

Параметры

peek [необязательный] Если true, данные будут прочитаны, но не удалены из буфера.
binary [необязательный] Если true, данные возвращаются в двоичном виде. Иначе – в текстовом.

Возвращаемое значение

Успех: Возвращает прочитанные данные; @extended содержит количество прочитанных байт.
Ошибка: Устанавливает @error ≠ 0 при достижении EOF, если STDIN не подключён к процессу или при другой ошибке.

Примечания

ConsoleRead() читает из стандартного потока ввода консоли процесса AutoIt-скрипта. Обычно этот поток используется консольными приложениями для чтения ввода от родительского процесса.
ConsoleRead() не блокирует выполнение и возвращает управление сразу. Чтобы получить все данные, функцию необходимо вызывать в цикле.
Режим peek не удаляет данные из буфера, но возвращает доступные данные как обычно.
По умолчанию данные возвращаются в текстовом формате. При использовании бинарной опции данные возвращаются в двоичном формате.

См. также

ConsoleWrite, ConsoleWriteError, Run

Пример

#include <MsgBoxConstants.au3>

#cs
Скомпилировать этот скрипт в "ConsoleRead.exe".
Открыть командную строку в каталоге, где лежит ConsoleRead.exe.
Введите следующее в командной строке:
echo Hello! | ConsoleRead.exe

При выполнении в консольном окне эта команда выводит текст "Hello!",
но вместо его отображения символ | перенаправляет его в поток STDIN
процесса ConsoleRead.exe.
#ce

Example()

Func Example()
    If Not @Compiled Then
        MsgBox($MB_SYSTEMMODAL, "", "This script must be compiled in order to run the example.")
        Exit
    EndIf

    Local $sOutput
    While True
        $sOutput &= ConsoleRead()
        If @error Then ExitLoop
        Sleep(25)
    WEnd
    MsgBox($MB_SYSTEMMODAL, "", "Received: " & @CRLF & @CRLF & $sOutput)
EndFunc   ;==>Example
