Справочник функций

TraySetOnEvent

Задаёт пользовательскую функцию, вызываемую при наступлении специального события трея.

TraySetOnEvent ( specialID, "function" )

Параметры

specialID См. таблицу специальных идентификаторов ниже.
function Имя вызываемой пользовательской функции.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.
@error: 1 – функция не определена.

Примечания

OnEvent-функции вызываются только при установленном параметре Opt("TrayOnEventMode", 1) – в этом режиме TrayGetMsg() не используется вовсе.

Таблица специальных идентификаторов

Специальный ID Значение Комментарий
$TRAY_EVENT_SHOWICON -3 Значок трея будет показан.
$TRAY_EVENT_HIDEICON -4 Значок трея будет скрыт.
$TRAY_EVENT_FLASHICON -5 Пользователь включил мигание значка трея.
$TRAY_EVENT_NOFLASHICON -6 Пользователь отключил мигание значка трея.
$TRAY_EVENT_PRIMARYDOWN -7 Основная кнопка мыши нажата на значке трея.
$TRAY_EVENT_PRIMARYUP -8 Основная кнопка мыши отпущена на значке трея.
$TRAY_EVENT_SECONDARYDOWN -9 Дополнительная кнопка мыши нажата на значке трея.
$TRAY_EVENT_SECONDARYUP -10 Дополнительная кнопка мыши отпущена на значке трея.
$TRAY_EVENT_MOUSEOVER -11 Указатель мыши перемещается над значком трея.
$TRAY_EVENT_PRIMARYDOUBLE -13 Двойной щелчок основной кнопкой мыши по значку трея.
$TRAY_EVENT_SECONDARYDOUBLE -14 Двойной щелчок дополнительной кнопкой мыши по значку трея.

Если функция – пустая строка "", ранее назначенная функция отключается.

Вышеуказанные константы определены в #include <TrayConstants.au3>.

См. также

TrayItemSetOnEvent, TrayOnEventMode (Option), TraySetClick

Пример

#NoTrayIcon

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>
#include <TrayConstants.au3> ; Требуется для констант $TRAY_EVENT_PRIMARYDOUBLE, $TRAY_EVENT_SECONDARYUP и $TRAY_ICONSTATE_SHOW.

Opt("TrayMenuMode", 3) ; Стандартные пункты меню трея не будут показаны, и пункты не помечаются при выборе. Это опции 1 и 2 для TrayMenuMode.
Opt("TrayOnEventMode", 1) ; Включить TrayOnEventMode.

Example()

Func Example()
    TrayCreateItem("About")

    TrayCreateItem("") ; Создать разделительную линию.

    TrayCreateItem("Exit")
    TrayItemSetOnEvent(-1, "ExitScript")

    TraySetOnEvent($TRAY_EVENT_PRIMARYDOUBLE, "TrayEvent")
    TraySetOnEvent($TRAY_EVENT_SECONDARYUP, "TrayEvent")

    TraySetState($TRAY_ICONSTATE_SHOW) ; Показать меню в трее.

    While 1
        Sleep(100) ; Цикл ожидания.
    WEnd
EndFunc   ;==>Example

Func TrayEvent()
    Switch @TRAY_ID ; Проверить идентификатор последнего элемента трея.
        Case $TRAY_EVENT_PRIMARYDOUBLE
            ; Показать окно сообщения с версией AutoIt и путём установки исполняемого файла AutoIt.
            MsgBox($MB_SYSTEMMODAL, "", "AutoIt tray menu example." & @CRLF & @CRLF & _
                    "Version: " & @AutoItVersion & @CRLF & _
                    "Install Path: " & StringLeft(@AutoItExe, StringInStr(@AutoItExe, "\", $STR_NOCASESENSEBASIC, -1) - 1)) ; Найти каталог полного пути.

        Case $TRAY_EVENT_SECONDARYUP
            MsgBox($MB_SYSTEMMODAL, "", "The secondary mouse button was released on the tray icon.")

    EndSwitch
EndFunc   ;==>TrayEvent

Func ExitScript()
    Exit
EndFunc   ;==>ExitScript
