Справочник функций

DriveGetLabel

Возвращает метку тома диска, если она назначена.

DriveGetLabel ( "path" )

Параметры

path Путь к диску, с которого получается информация.

Возвращаемое значение

Успех: Возвращает метку тома диска в виде строки.
Ошибка: Устанавливает значение @error = 1.

См. также

DriveGetDrive, DriveGetFileSystem, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

Пример

#include <MsgBoxConstants.au3>

Local $sLabel = DriveGetLabel(@HomeDrive & "\") ; Определить метку тома домашнего диска, как правило, это диск C:\.
MsgBox($MB_SYSTEMMODAL, "", "Volume Label: " & $sLabel)
