Справочник функций

DllCallAddress

Динамически вызывает функцию по указанному адресу памяти.

DllCallAddress ( "return type", address [, type1, param1 [, type n, param n]] )

Параметры

return type Тип возвращаемого значения функции (см. ниже).
address Адрес функции. Если это значение неверно, скрипт аварийно завершится!
type1 [необязательный] Тип параметра (см. раздел Примечания).
param1 [необязательный] Значение параметра (см. раздел Примечания).
type n [необязательный] Тип n-го параметра (см. раздел Примечания).
param n [необязательный] Значение n-го параметра (см. раздел Примечания).

Допустимые типы:

Тип Описание
NONE нет значения (допустимо только для возвращаемого типа, аналог void в C)
BYTE беззнаковое 8-битное целое
BOOLEAN беззнаковое 8-битное целое
SHORT 16-битное целое
USHORT беззнаковое 16-битное целое
WORD беззнаковое 16-битное целое
INT 32-битное целое
LONG 32-битное целое
BOOL 32-битное целое
UINT беззнаковое 32-битное целое
ULONG беззнаковое 32-битное целое
DWORD беззнаковое 32-битное целое
INT64 64-битное целое
UINT64 беззнаковое 64-битное целое
PTR обычный указатель (void *)
HWND дескриптор окна (указатель)
HANDLE дескриптор (указатель)
FLOAT число с плавающей точкой одинарной точности
DOUBLE число с плавающей точкой двойной точности
INT_PTR, LONG_PTR, LRESULT, LPARAM целое, достаточно большое для хранения указателя в x86 и x64 версиях AutoIt.
UINT_PTR, ULONG_PTR, DWORD_PTR, WPARAM беззнаковое целое, достаточно большое для хранения указателя в x86 и x64 версиях AutoIt.
STR строка ANSI (выделяется минимум 65536 символов).
WSTR строка широких символов UNICODE (выделяется минимум 65536 символов).

Соответствие типов Windows API типам AutoIt:

Тип Windows API Тип AutoIt
LPCSTR/LPSTR STR
LPCWSTR/LPWSTR WSTR
LPVOID PTR
LPxyz xyz*
HINSTANCE HANDLE
HRESULT LONG
LONGLONG/LARGE_INTEGER INT64

SIZE_T ULONG_PTR
Чтобы использовать вложенные структуры внутри структуры, переопределите вложенную структуру. Например, структура с двумя структурами POINT ("long;long") объявляется как "long;long;long;long". Первые два значения long соответствуют первой структуре POINT, вторые два – второй структуре POINT.

Другие типы Windows API см. на MSDN.

Возвращаемое значение

Успех: Возвращает массив. См. раздел Примечания.
Ошибка: Устанавливает @error ≠ 0.
@error: 2 – неизвестный тип возвращаемого значения,
4 – неверное количество параметров,
5 – неверный параметр

Примечания

По умолчанию AutoIt использует соглашение о вызовах 'stdcall'. Для использования метода 'cdecl' добавьте ':cdecl' после типа возвращаемого значения.

Если вызов функции завершается ошибкой, @error ≠ 0.
В противном случае возвращается массив, содержащий возвращаемое значение функции и копии всех параметров (включая параметры, изменённые функцией при передаче по ссылке).
$return[0] = возвращаемое значение функции
$return[1] = param1
$return[2] = param2
...
$return[n] = paramn

Особое замечание: это продвинутая функция. Неправильное использование может привести к аварийному завершению AutoIt. Перед использованием убедитесь, что DllCall() не предоставляет нужную функциональность.

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPISysWin.au3>
#include <WindowsNotifsConstants.au3>

Example()

Func Example()
    ; Включить режим GUI-событий.
    Opt("GUIOnEventMode", 1)

    ; Создать простой GUI.
    Local $hWnd = GUICreate("DllCallAddress Example")

    ; Зарегистрировать обработчик события закрытия.
    GUISetOnEvent($GUI_EVENT_CLOSE, "OnClose")

    ; Отобразить GUI.
    GUISetState(@SW_SHOWNORMAL, $hWnd)

    ; Получить указатель на WindowProc() окна.
    Local $pWndProc = _WinAPI_GetWindowLong($hWnd, $GWL_WNDPROC)

    ; Сообщить пользователю о предстоящем действии.
    MsgBox($MB_SYSTEMMODAL, "DllCallAddress Example Msg", "When you press OK the test window will close.")

    ; Явно сгенерировать событие WM_CLOSE и передать его напрямую в WindowProc().
    ; В реальном приложении так делать не следует (лучше использовать _SendMessage()), но
    ; это демонстрирует использование функции.
    DllCallAddress("LRESULT", $pWndProc, "HWND", $hWnd, "UINT", $WM_CLOSE, "WPARAM", 0, "LPARAM", 0)
EndFunc   ;==>Example

Func OnClose()
    GUIDelete(@GUI_WinHandle)
    MsgBox($MB_SYSTEMMODAL, "DllCallAddress Example Msg", "Close event received, the test window should now be closed.")
EndFunc   ;==>OnClose
