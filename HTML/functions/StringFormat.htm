Справочник функций

StringFormat

Возвращает форматированную строку (аналогично функции sprintf() в языке C).

StringFormat ( "format control" [, var1 [, ... var32]] )

Параметры

format control Строка формата и флаги (см. раздел Примечания).
var1...var32 До 32 переменных, выводимых в соответствии со строкой формата.

Возвращаемое значение

Возвращает строку, сформатированную согласно параметру строки формата.

Примечания

Во избежание переполнения буфера каждая переменная ограничена 65535 символами.
В строке формата можно использовать управляющие символы: \n (@LF), \r (@CR), \t (@TAB). Для вывода "\" используйте \\, для "%" – %%.

Формат переменной: %[флаги] [ширина] [.точность] тип

При недопустимой спецификации формата поведение не определено. Некорректные входные данные могут в особых случаях привести к необработанному исключению и завершению программы.

    Спецификация ширины

Второе необязательное поле – спецификация ширины. Это неотрицательное целое число, задающее минимальное количество выводимых символов. Если результат короче, слева (или справа при флаге –) добавляются пробелы. Если ширина начинается с 0, вместо пробелов добавляются нули (для чисел с выравниванием влево не имеет смысла).

Спецификация ширины никогда не вызывает усечения значения. Если результат длиннее ширины или ширина не задана, выводятся все символы (с учётом точности).

    Спецификация типа

Тип Тип переменной Формат вывода
d, i Integer Десятичное целое со знаком.
o Integer Восьмеричное целое без знака.
u Integer Десятичное целое без знака.
x Integer Шестнадцатеричное целое без знака, строчные буквы "abcdef".
X Integer Шестнадцатеричное целое без знака, прописные буквы "ABCDEF".
e Float Значение со знаком в форме [ - ]d.dddd e [знак]ddd, где d – одна десятичная цифра, dddd – одна или более цифр, ddd – ровно три цифры, знак – "+" или "–".
E Float Аналогично e, но экспонента вводится буквой E.
f Float Значение со знаком в форме [ - ]dddd.dddd. Количество цифр до точки зависит от величины числа, после – от заданной точности.
g Float Значение со знаком в формате f или e – в зависимости от того, который компактнее. Формат e используется, если показатель степени меньше –4 или не меньше precision. Конечные нули отбрасываются; десятичная точка выводится только при наличии цифр после неё.
G Float Аналогично g, но экспонента вводится буквой E.
s String Строка.

    Спецификация флагов

Флаг Значение По умолчанию
- Выравнивание результата по левому краю поля. По правому краю.
+ Перед значением всегда ставится знак (+ или –), если тип со знаком. Знак ставится только для отрицательных значений (–).
0 Если ширина начинается с 0, дополняется нулями. При наличии флага – флаг 0 игнорируется. Для целых форматов (i, u, x, X, o, d) также игнорируется. Без дополнения.
Пробел Перед положительным значением со знаком ставится пробел; если указаны оба флага – пробел и +, пробел игнорируется. Без пробела.
# С форматами o, x, X: ненулевые значения предваряются префиксом 0, 0x или 0X соответственно. Без пробела.
# С форматами e, E, f: десятичная точка выводится всегда. Точка выводится только при наличии цифр после неё.
# С форматами g, G: десятичная точка выводится всегда, конечные нули не отбрасываются.
Игнорируется с d, i, u, s.		 Точка выводится только при наличии цифр после неё. Конечные нули отбрасываются.

    Спецификация точности

Третье необязательное поле – спецификация точности: неотрицательное десятичное число, предваряемое точкой (.), задающее количество выводимых символов, десятичных знаков или значащих цифр (см. таблицу). В отличие от ширины, точность может вызвать усечение или округление числа с плавающей точкой. Если precision равно 0 и преобразуемое значение равно 0, вывод пуст:

StringFormat( "%.0d", 0 ); /* Символы не выводятся */

    Влияние точности на типы

Тип Значение По умолчанию
d, i, u, o, x, X Минимальное количество выводимых цифр. При нехватке цифр слева добавляются нули. При превышении precision значение не усекается. Точность по умолчанию – 1.
e, E Количество цифр после десятичной точки. Последняя цифра округляется. По умолчанию 6; при precision=0 или одиночной точке без числа десятичная точка не выводится.
f Количество цифр после десятичной точки. При наличии точки перед ней стоит хотя бы одна цифра. Значение округляется. По умолчанию 6; при precision=0 или одиночной точке без числа десятичная точка не выводится.
g, G Максимальное количество значащих цифр. 6 значащих цифр; конечные нули отбрасываются.
s Максимальное количество выводимых символов; лишние символы не выводятся. Вывод до нулевого символа.

Пример

Example()

Func Example()
    Local $iInt_Unsigned = 43951789
    Local $iInt_Negative = -43951789

    ConsoleWrite(@CRLF & "Numeric Formats" & @CRLF)

    PrintFormat($iInt_Unsigned, "%d", "standard positive integer with no sign", 1) ; 43951789
    PrintFormat($iInt_Negative, "%d", "standard negative integer with sign", 1) ; -43951789
    PrintFormat($iInt_Unsigned, "%i", "standard integer", 1) ; 43951789
    PrintFormat($iInt_Unsigned, "%09i", "9 digits with leading zero", 1) ; 043951789
    PrintFormat($iInt_Unsigned, "%e", "scientific notation") ; 4.395179e+007
    PrintFormat($iInt_Unsigned, "%u", "unsigned integer with positive integer", 1) ; 43951789
    PrintFormat($iInt_Negative, "%u", "unsigned integer with negative integer", 1) ; 4251015507
    PrintFormat($iInt_Unsigned, "%f", "floating point") ; 43951789.000000
    PrintFormat($iInt_Unsigned, "%.2f", "floating point with 2 digits after decimal point", 1) ; 43951789.00
    PrintFormat($iInt_Unsigned, "%o", "octal", 1) ; 247523255
    PrintFormat($iInt_Unsigned, "%s", "string", 1) ; 43951789
    PrintFormat($iInt_Unsigned, "%x", "hexadecimal (lower-case)", 1) ; 29ea6ad
    PrintFormat($iInt_Unsigned, "%X", "hexadecimal (upper-case)", 1) ; 29EA6AD
    PrintFormat($iInt_Unsigned, "%+d", "sign specifier on a positive integer", 1) ; +43951789
    PrintFormat($iInt_Negative, "%+d", "sign specifier on a negative integer", 1) ; -43951789

    Local $sString = "string"
    Local $sString_Long = "longstring"

    ConsoleWrite(@CRLF & "String Formats - [ ] used to show beginning/end of string" & @CRLF)

    PrintFormat($sString, "[%s]", "standard string", 1) ; [string]
    PrintFormat($sString, "[%10s]", "10 chars right justified with added spaces") ; [ string]
    PrintFormat($sString, "[%-10s]", "10 chars left justified with added spaces") ; [string ]
    PrintFormat($sString_Long, "[%10.8s]", "right justified but precision 8 so truncated") ; [ longer s]
    PrintFormat($sString_Long, "[%-10.8s]", "left justifed but precision 8 so truncated") ; [longer s ]
    PrintFormat($sString, "[%010s]", "10 chars with leading zero") ; [0000string]

    ConsoleWrite(@CRLF & "Date Format - each % uses a new parameter" & @CRLF)
    ConsoleWrite('"%02i\%02i\%04i" 0n (1, 9, 2013) => ' & StringFormat("%02i\%02i\%04i", 1, 9, 2013) & @CRLF)

    ConsoleWrite(@CRLF & "Just string to format without Vars" & @CRLF)
    ConsoleWrite('"Some \texample text\n" => ' & StringFormat('Some \texample text\n'))
EndFunc   ;==>Example

Func PrintFormat($vVar, $sFormat, $sExplan, $iTab = 0)
    ConsoleWrite('"' & $sFormat & '" on ' & $vVar & @TAB & ' => ' & StringFormat($sFormat, $vVar))
    If $iTab Then ConsoleWrite(@TAB)
    ConsoleWrite(@TAB & " ; " & $sExplan & @CRLF)
EndFunc   ;==>PrintFormat
