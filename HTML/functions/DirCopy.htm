Справочник функций

DirCopy

Копирует каталог со всеми подкаталогами и файлами (аналог xcopy).

DirCopy ( "source dir", "dest dir" [, flag = 0] )

Параметры

source dir Путь к исходному каталогу (без завершающего обратного слеша), например "C:\Path1".
dest dir Путь к каталогу назначения (без завершающего обратного слеша), например "C:\Path_Copy".
flag [необязательный] Флаг, определяющий, перезаписывать ли существующие файлы:
$FC_NOOVERWRITE (0) = (по умолчанию) не перезаписывать существующие файлы
$FC_OVERWRITE (1) = перезаписывать существующие файлы

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если при копировании каталога произошла ошибка.

Примечания

Если структура каталога назначения не существует, она будет создана (если возможно).

См. также

DirMove, DirRemove, FileCopy, FileMove

Пример

#include <FileConstants.au3>

DirCopy(@MyDocumentsDir, @TempDir & "\Backups\MyDocs", $FC_OVERWRITE)
