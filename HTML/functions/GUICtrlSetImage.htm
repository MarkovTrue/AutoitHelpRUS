Справочник функций

GUICtrlSetImage

Устанавливает растровое изображение или значок для элемента управления.

GUICtrlSetImage ( controlID, filename [, iconname [, icontype]] )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
filename Имя файла с изображением, которое будет отображаться на элементе управления.
iconname [необязательный] Имя значка, если файл содержит несколько значков. При отрицательном числе используется порядковое имя. Иначе -1.
icontype [необязательный] Выбор размера значка: 0 = маленький, 1 = обычный (по умолчанию).
Для TreeViewItem: 2 = выбранный элемент, 4 = невыбранный элемент.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Для получения допустимого имени значка в файле используйте Resource Hacker.

При использовании на элементе Button изображение отображается на кнопке. Изображения можно также задавать для Checkbox при условии использования стиля $BS_PUSHLIKE. В обоих случаях необходим стиль $BS_ICON или $BS_BITMAP для указания типа изображения. В файле с несколькими разрешениями значка будет использовано первое из них. Например, если первое разрешение 128×128, а элемент управления 64×64, изображение будет обрезано.

!!! При первом использовании этой команды на TreeViewItem все остальные элементы автоматически получат тот же значок/изображение !!!
Если применить GUICtrlSetImage() к TreeView или ListView, все его элементы сменят значок/изображение.

Положительное число ссылается на строковое имя значка.
Отрицательное число задаёт поведение "индекс начиная с 1". Из некоторых DLL значки можно извлечь только с отрицательными числами.

См. также

GUICtrlCreateButton, GUICtrlCreateCheckbox, GUICtrlCreateIcon, GUICtrlCreatePic

Пример

#include <ButtonConstants.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate("My GUI") ; создаст диалоговое окно, отображаемое по центру

    GUICtrlCreateButton("my picture button", 10, 20, 40, 40, $BS_ICON)
    GUICtrlSetImage(-1, "shell32.dll", 22)

    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd
EndFunc   ;==>Example
