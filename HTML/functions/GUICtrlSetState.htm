Справочник функций

GUICtrlSetState

Изменяет состояние элемента управления.

GUICtrlSetState ( controlID, state )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
state См. таблицу состояний ниже.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

    Таблица состояний

Состояние Описание
$GUI_CHECKED (1) Radio, Checkbox, MenuItem или ListViewItem будет отмечен.
$GUI_INDETERMINATE (2) Checkbox с атрибутом tristate будет затенён.
$GUI_UNCHECKED (4) Radio, Checkbox, MenuItem или ListViewItem будет снят.
$GUI_DROPACCEPTED (8) Элемент управления будет принимать операцию перетаскивания: из файла или из другого элемента управления. См. раздел Примечания.
$GUI_SHOW (16) Элемент управления будет видим. Для TabItem выбирает первую вкладку для отображения.
$GUI_HIDE (32) Элемент управления будет скрыт.
$GUI_ENABLE (64) Элемент управления будет активен.
$GUI_DISABLE (128) Элемент управления будет затенён (недоступен).
$GUI_FOCUS (256) Элемент управления получит фокус ввода/выделения.
$GUI_DEFBUTTON (512) Элемент управления будет назначен кнопкой по умолчанию в окне. См. раздел Примечания о TreeViewItem.
$GUI_EXPAND (1024) TreeViewItem раскроет дочерние элементы.
$GUI_ONTOP (2048) Элемент управления получит атрибут "поверх" для окна (z-порядок).
$GUI_NODROPACCEPTED (4096) Элемент управления не будет принимать операцию перетаскивания.
$GUI_NOFOCUS (8192) Элемент управления ListView потеряет фокус.
$GUI_AVISTART (0) Элемент управления AVI начнёт воспроизведение.
$GUI_AVISTOP (1) Элемент управления AVI остановит воспроизведение.
$GUI_AVICLOSE (2) Элемент управления AVI остановит воспроизведение и освободит ресурсы.

Значения состояний можно суммировать: например, $GUI_DISABLE (128) + $GUI_HIDE (32) делает элемент недоступным и скрытым.

Если элемент управления AVI нужно скрыть через $GUI_HIDE (32), предварительно его следует закрыть через $GUI_AVICLOSE (1).

Состояние элемента управления "contextmenu" изменить невозможно.
Состояние "listviewitem" можно изменить, если связанный "listview" создан с расширенным стилем $LVS_EX_CHECKBOXES. $GUI_FOCUS (256) и $GUI_NOFOCUS (8192) применимы к конкретному listviewitem при условии, что стиль listview позволяет его отображение: $LVS_SHOWSELALWAYS.
Состояние элементов "menu" и "menuitem" скрыть невозможно.

! Важно для $GUI_EXPAND: это состояние применяется только к TreeViewItem. Чтобы использовать это действие, у элемента должен существовать хотя бы один дочерний Sub-TreeViewItem !
Для выбора другого элемента в TreeView используйте $GUI_FOCUS (256) – родительский TreeView получает фокус окна, а указанный элемент помечается как выбранный.
Для назначения элемента TreeView элементом по умолчанию (жирное начертание) используйте $GUI_DEFBUTTON (512) – для отмены передайте любое другое значение, например 0. Это состояние не возвращается функцией GUICtrlGetState().

Если для видимого элемента управления установлено $GUI_DROPACCEPTED (8), перетаскивание будет учтено. Для Edit/Input будет задано имя файла.
Для других элементов при получении события $GUI_EVENT_DROPPED @GUI_DragId вернёт controlID, с которого началось перетаскивание (-1 при перетаскивании файла; @GUI_DragFile содержит имя файла), а @GUI_DropId – controlID элемента назначения.
Процесс перетаскивания запускается только при перетаскивании ListViewItem. @GUI_DragId будет содержать controlID ListView.

Пример

#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    ; Создать GUI с различными элементами управления.
    Local $hGUI = GUICreate("Example", 420, 200, -1, -1, -1, $WS_EX_ACCEPTFILES)

    ; Создать метку и разрешить для неё приём перетаскивания.
    Local $idLabel = GUICtrlCreateLabel("Drop a file on this label.", 10, 10, 400, 40, $WS_BORDER)
    GUICtrlSetState($idLabel, $GUI_DROPACCEPTED)

    ; Создать поле ввода и разрешить для него приём перетаскивания.
    Local $idInput = GUICtrlCreateInput("", 10, 60, 400, 22)
    GUICtrlSetState($idInput, $GUI_DROPACCEPTED)

    Local $idButton_OK = GUICtrlCreateButton("OK", 310, 170, 85, 25)

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idButton_OK
                ExitLoop

            Case $GUI_EVENT_DROPPED
                ; Если @GUI_DropId равен $idLabel, установить текст метки из имени перетащенного файла.
                If @GUI_DropId = $idLabel Then GUICtrlSetData($idLabel, @GUI_DragFile)

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI и все элементы управления.
    GUIDelete($hGUI)
EndFunc   ;==>Example
