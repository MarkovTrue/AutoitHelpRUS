InetGet

Загружает файл из интернета по протоколу HTTP, HTTPS или FTP.

InetGet ( "URL", "filename" [, options = 0 [, background = 0]] )

Параметры

URL URL-адрес загружаемого файла. См. раздел Примечания.
filename Локальное имя файла для сохранения загрузки.
options [необязательный]$INET_LOCALCACHE (0) = Взять файл из локального кэша, если доступен (по умолчанию).
$INET_FORCERELOAD (1) = Принудительно перезагрузить с удалённого сайта.
$INET_IGNORESSL (2) = Игнорировать все ошибки SSL (для HTTPS-соединений).
$INET_ASCIITRANSFER (4) = Использовать ASCII при передаче файлов по FTP (нельзя совмещать с флагом $INET_BINARYTRANSFER (8)).
$INET_BINARYTRANSFER (8) = Использовать BINARY при передаче файлов по FTP (нельзя совмещать с флагом $INET_ASCIITRANSFER (4)). Это режим по умолчанию, если флаги не заданы.
$INET_FORCEBYPASS (16) = Пропустить принудительный перевод соединения в онлайн (см. раздел Примечания).

Константы определены в InetConstants.au3.
background [необязательный]
$INET_DOWNLOADWAIT (0) = Ожидать завершения загрузки перед продолжением (по умолчанию).
$INET_DOWNLOADBACKGROUND (1) = Вернуть управление немедленно и загружать в фоне (см. раздел Примечания).
Константы определены в InetConstants.au3.

Возвращаемое значение

Успех: Возвращаемое значение зависит от режима загрузки:
Фоновый: возвращает дескриптор.
Ожидание: возвращает количество загруженных байт.
Ошибка: Фоновый: возвращает дескриптор.
Ожидание: возвращает 0 и устанавливает @error ≠ 0.

Примечания

Используйте возвращённый дескриптор с InetGetInfo(), чтобы проверить наличие ошибок загрузки.
Возвращённый дескриптор необходимо закрыть с помощью InetClose().

Параметр URL должен иметь вид "http://www.somesite.com/path/file.html" – как адрес в адресной строке браузера.

Для подключения с именем пользователя и паролем добавьте их перед именем сервера в формате "username:password@", например:
"http://myuser:mypassword@www.somesite.com"


Примечания о параметре "background"

По умолчанию функция ожидает завершения загрузки перед возвратом. Если параметр background равен $INET_DOWNLOADBACKGROUND (1), функция возвращает управление немедленно, а загрузка продолжается в фоне. Для проверки состояния загрузки используйте InetGetInfo(), передав ей дескриптор, возвращённый InetGet().

Поддерживается несколько одновременных загрузок в фоновом режиме.

Для отмены загрузки вызовите InetClose(), передав дескриптор, возвращённый InetGet().

По умолчанию AutoIt принудительно устанавливает соединение перед загрузкой. Для коммутируемых пользователей это вызовет запрос на выход в сеть или набор номера (в зависимости от настроек системы). Флаг $INET_FORCEBYPASS (16) отключает это поведение. Отключение удобно для постоянных подключений (широкополосных, локальных). Кроме того, оно необходимо для обхода некоторых проблем в Windows Vista и Windows 7.

Пример

Пример 1

#include <InetConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

; InetGet загружает файл в фоновом режиме.
; Скрипт AutoIt ожидает завершения загрузки в цикле.

Example()

Func Example()
    ; Сохранить загруженный файл во временную папку.
    Local $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Загрузить файл в фоне с опцией 'принудительно перечитать с удалённого сайта.'
    Local $hDownload = InetGet("http://www.autoitscript.com/autoit3/files/beta/update.dat", $sFilePath, $INET_FORCERELOAD, $INET_DOWNLOADBACKGROUND)

    ; Ждать завершения загрузки, отслеживая возврат True вторым индексом InetGetInfo.
    Do
        Sleep(250)
    Until InetGetInfo($hDownload, $INET_DOWNLOADCOMPLETE)

    ; Получить общее число принятых байт и размер файла.
    Local $iBytesSize = InetGetInfo($hDownload, $INET_DOWNLOADREAD)
    Local $iFileSize = FileGetSize($sFilePath)

    ; Закрыть дескриптор, возвращённый InetGet.
    InetClose($hDownload)

    ; Показать сведения об общем числе прочитанных байт и размере файла.
    MsgBox($MB_SYSTEMMODAL, "", "The total download size: " & $iBytesSize & @CRLF & _
            "The total filesize: " & $iFileSize)

    ; Удалить файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example

Пример 2

#include <InetConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

; Загрузить файл в фоновом режиме.
; Ожидать завершения загрузки.

Example()

Func Example()
    ; Сохранить загруженный файл во временную папку.
    Local $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Загрузить файл с ожиданием завершения. Флаг $INET_FORCERELOAD принудительно перезагружает файл с удалённого сайта.
    Local $iBytesSize = InetGet("http://www.autoitscript.com/autoit3/files/beta/update.dat", $sFilePath, $INET_FORCERELOAD)

    ; Получить размер файла.
    Local $iFileSize = FileGetSize($sFilePath)

    ; Показать сведения об общем числе прочитанных байт и размере файла.
    MsgBox($MB_SYSTEMMODAL, "", "The total download size: " & $iBytesSize & @CRLF & _
            "The total filesize: " & $iFileSize)

    ; Удалить загруженный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
