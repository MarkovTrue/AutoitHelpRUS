TCPNameToIP

Преобразует доменное имя в IP-адрес.

TCPNameToIP ( name )

Параметры

name Строка, содержащая доменное имя.

Возвращаемое значение

Успех: Возвращает строку, содержащую IP-адрес, соответствующий указанному имени.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@error: Возвращает значение Windows API WSAGetLastError (см. MSDN).

Примечания

Перед вызовом необходимо выполнить TCPStartup().

См. также

TCPStartup

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить TCP-сервис.
    TCPStartup()

    ; Зарегистрировать OnAutoItExit для вызова при завершении скрипта.
    OnAutoItExitRegister("OnAutoItExit")

    ; Присвоить локальной переменной IP-адрес сайта www.autoitscript.com.
    Local $sIPAddress = TCPNameToIP("www.autoitscript.com")

    ; Если произошла ошибка, показать код ошибки и вернуть False.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error code: " & @error)
        Return False
    Else
        ; Отобразить IP-адрес.
        MsgBox($MB_SYSTEMMODAL, "", "Domain name to IP : " & $sIPAddress)
    EndIf
EndFunc   ;==>Example

Func OnAutoItExit()
    TCPShutdown() ; Закрыть TCP-сервис.
EndFunc   ;==>OnAutoItExit
