Справочник функций

ControlFocus

Устанавливает фокус ввода на указанный элемент управления окна.

ControlFocus ( "title", "text", controlID )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

См. также

ControlCommand, ControlGetFocus, ControlSend

Пример

#include "Extras\HelpFileInternals.au3"

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Установить фокус ввода на edit-элемент Блокнота, используя дескриптор, возвращённый WinWait.
    ControlFocus($hWnd, "", ControlGetFocus($hWnd))

    ; Пауза 2 секунды.
    Sleep(2000)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
