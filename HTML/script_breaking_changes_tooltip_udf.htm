ToolTip UDF – критические изменения

ToolTip UDF полностью переписан: устранены "магические числа", упрощена реализация всплывающих подсказок, стандартизированы параметры и документация, исправлены ошибки.
Это означает, что скрипты, написанные для AutoIt до версии 3.3.8.1, не будут работать с более поздними версиями и потребуют изменений.
В таблице ниже перечислены различия между старой и новой версией.

Изменённые и удалённые функции/параметры
Старая функция/параметр UDF Примечания
_GUIToolTip_Activate() Параметр $fActivate удалён: функция теперь только активирует подсказку, деактивация вынесена в отдельную функцию.
_GUIToolTip_AddTool() Параметры $iFlags изменены: теперь используются константы ToolTip вместо "магических чисел"; значение по умолчанию изменено на BitOr($TTF_SUBCLASS, $TTF_IDISHWND).
_GUIToolTip_BitsToTTF() Функциональность полностью изменена: так как "магических чисел" для флагов инструмента больше нет, функция теперь преобразует значение флагов в строку с разделителями-запятыми, содержащую названия используемых констант.
_GUIToolTip_Create() Новая константа по умолчанию для создания элемента управления ToolTip.
   $_TT_ghTTDefaultStyle Значение равно BitOr($TTS_ALWAYSTIP, $TTS_NOPREFIX).
_GUIToolTip_Destroy() Задокументированы возвращаемые значения @error.
_GUIToolTip_EnumTools() Возвращаемые значения изменены в соответствии с изменениями в _GUIToolTip_Add и _GUIToolTip_BitsToTTF(): элемент [0] возвращаемого массива теперь содержит строку из _GUIToolTip_BitsToTTF() вместо прежних "магических чисел".
_GUIToolTip_GetBubbleHeight() Значение по умолчанию для $iFlags изменено на BitOR($TTF_IDISHWND, $TTF_SUBCLASS).
_GUIToolTip_GetBubbleSize() Значение по умолчанию для $iFlags изменено на BitOR($TTF_IDISHWND, $TTF_SUBCLASS).
_GUIToolTip_GetBubbleWidth() Значение по умолчанию для $iFlags изменено на BitOR($TTF_IDISHWND, $TTF_SUBCLASS).
_GUIToolTip_GetCurrentTools Возвращаемые значения изменены в соответствии с изменениями в _GUIToolTip_Add и _GUIToolTip_BitsToTTF(): элемент [0] возвращаемого массива теперь содержит строку из _GUIToolTip_BitsToTTF() вместо прежних "магических чисел".
_GUIToolTip_GetDelayTime() Функция теперь корректно использует константы ToolTip для получения времени задержки. Прежние значения были 0–2; правильные значения констант длительности – 1–3.
_GUIToolTip_GetTitleBitMap() Исправлена давняя ошибка, из-за которой функция возвращала неверные данные.
_GUIToolTip_GetTitleText() Исправлена давняя ошибка, из-за которой функция возвращала неверные данные.
_GUIToolTip_GetToolInfo() Возвращаемые значения изменены в соответствии с изменениями в _GUIToolTip_Add и _GUIToolTip_BitsToTTF(): элемент [0] возвращаемого массива теперь содержит строку из _GUIToolTip_BitsToTTF() вместо прежних "магических чисел".
_GUIToolTip_HitTest() Возвращаемые значения изменены в соответствии с изменениями в _GUIToolTip_Add и _GUIToolTip_BitsToTTF(): элемент [0] возвращаемого массива теперь содержит строку из _GUIToolTip_BitsToTTF() вместо прежних "магических чисел".
_GUIToolTip_SetToolInfo() Параметры $iFlags изменены: теперь используются константы ToolTip вместо "магических чисел"; значение по умолчанию изменено на BitOr($TTF_SUBCLASS, $TTF_IDISHWND).
_GUIToolTip_ToolToArray() Элемент [0] возвращаемого массива теперь корректно возвращает значение флага на основе констант ToolTip вместо прежних "магических чисел".
_GUIToolTip_TTFToBits Удалена.


Новые функции/параметры
Новая функция/параметр UDF Примечания
_GUIToolTip_Deactivate() Деактивирует всплывающую подсказку; заменяет двойную функциональность, которую прежде выполняла _GUIToolTip_Activate().
