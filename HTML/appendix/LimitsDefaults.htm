Ограничения AutoIt3 и значения по умолчанию


Ограничение AutoIt3 Значение Описание
MAX_BINARYSIZE 2 147 483 647 Максимальный размер двоичных данных в байтах.
MAX_ENVSIZE 32767 Максимальный размер переменной окружения ENV.
MAX_LINESIZE 4095 Максимальный размер строки скрипта.
MAX_STRINGLEN 2 147 483 647 Максимальная длина строки.
WINTEXTBUFFER 32767 WM_GETTEXT иногда сбоит при значении больше 32767.
MAXCALLRECURSE 3900 для x64 и 1900 для x86 Максимальное число рекурсивных вызовов функции Call().
MAXEXECUTERECURSE 5100 Максимальное число рекурсивных вызовов функции Execute().
GUI_MAXCONTROLS 65532 Максимальное число контролов в GUI-окне.
GUI_MAXPOINTS 256 Максимальное число точек на странице графической информации.
COM_MAXEVENT 64 Максимальное число COM-событий, которые можно буферизировать.
VAR_SUBSCRIPT_ELEMENTS 16 777 216 Максимальное число элементов массива.
VAR_SUBSCRIPT_MAX 64 Максимальное число индексов (измерений) массива.
MAIN_TIMER_DELAY 750 мс Скрытие, мигание и отрисовка значка в трее проверяются каждые 750 мс.
CMDLINEPARAM_MAXLEN 4096 Каждый параметр командной строки может содержать столько символов.
TRAY_MAXITEMS 505 Максимальное число пунктов в меню трея.
TRAY_MAXEVENT 32 Максимальное число событий, которые можно буферизировать.
TRAY_TOOLTIPWIDTH 64 Максимальное число символов в подсказке (tooltip) трея.

 

Параметр по умолчанию Значение Описание
ADLIB delay 250 мс Задержка по умолчанию между срабатываниями ADLIB. Задаётся в AdlibRegister().
    Следующие значения по умолчанию можно изменить через Opt().
MouseClickDelay 10 мс Время между щелчками мыши.
MouseClickDownDelay 10 мс Время удержания нажатия.
MouseClickDragDelay 250 мс Задержка в начале и в конце операции перетаскивания.
TCPTimeout 100 мс Время, после которого прерывается TCP-функция.
