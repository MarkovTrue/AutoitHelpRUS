Инклуд-файлы констант AutoIt

Ниже приведён полный список подключаемых файлов с константами AutoIt.

    Использование #include <Constants.au3> подключает все из них, кроме констант Font, GUI, Inet и Windows.

Файл Относится к
#include <AutoItConstants.au3> Связано с AutoIt
#include <ColorConstants.au3> Цвета, $CLR_*
#include <Constants.au3> Функции AutoIt (включает AutoIt, Color, Dir, File, MsgBox, Process, String, Tray)
#include <DirConstants.au3> Каталоги, $DDL_*
#include <FileConstants.au3> Файлы, $F*_*
#include <FontConstants.au3> Шрифты
#include <GUIConstantsEx.au3> GUI
#include <InetConstants.au3> Интернет, $INET_*
#include <MsgBoxConstants.au3> MsgBox, $MB_*
#include <ProcessConstants.au3> Процессы, $PROCESS_*
#include <StringConstants.au3> Строки, $STR_*
#include <WindowsNotifsConstants.au3> Уведомления Windows, $WM_*
#include <WindowsStylesConstants.au3> Стили окон Windows, $WS_*
#include <WindowsSysColorConstants.au3> Системные цвета Windows, $COLOR_*
