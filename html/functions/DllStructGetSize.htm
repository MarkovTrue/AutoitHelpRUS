Справочник функций

DllStructGetSize

Возвращает размер структуры в байтах.

DllStructGetSize ( Struct )

Параметры

Struct Структура, возвращаемая функцией DllStructCreate().

Возвращаемое значение

Успех: Возвращает размер структуры в байтах.
Ошибка: Возвращает 0.
@error: 1 – структура не является корректной структурой, возвращённой DllStructCreate().

Примечания

Используется, когда данные в структуре должны содержать её размер.

См. также

DllStructCreate, DllStructSetData

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Присвоить локальной константной переменной определение структуры.
    Local Const $tagSTRUCT1 = "struct;int var1;byte var2;uint var3;char var4[128];endstruct"

    ; Присвоить локальной переменной структуру.
    Local $tSTRUCT1 = DllStructCreate($tagSTRUCT1)

    ; Если произошла ошибка – показать код ошибки и вернуть False.
    If @error Then
        MsgBox($MB_SYSTEMMODAL, "", "Error in DllStructCreate, Code: " & @error)
        Return False
    EndIf

    #cs Comments:
– выравнивание данных по умолчанию составляет 4 байта, поэтому каждый элемент занимает не менее 4 байт.
– минимальный размер можно получить, установив выравнивание 1 в начале определения структуры: "align 1;struct; ..."

– подробная информация о размере структуры:
var1: int – 4 байта
var2: byte – 1 байт + 3 байта выравнивания = 4 байта
var3: uint – 4 байта
var4: char[128] – 1 × 128 байт
#ce Comments:

    MsgBox($MB_SYSTEMMODAL, "", "Struct Size: " & DllStructGetSize($tSTRUCT1))
EndFunc   ;==>Example
