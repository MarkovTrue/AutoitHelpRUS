Справочник функций

InetRead

Загружает файл из интернета по протоколу HTTP, HTTPS или FTP.

InetRead ( "URL" [, options = 0] )

Параметры

URL URL-адрес загружаемого файла. См. раздел Примечания.
options [необязательный]
$INET_LOCALCACHE (0) = Взять файл из локального кэша, если доступен (по умолчанию).
$INET_FORCERELOAD (1) = Принудительно перезагрузить с удалённого сайта.
$INET_IGNORESSL (2) = Игнорировать все ошибки SSL (для HTTPS-соединений).
$INET_ASCIITRANSFER (4) = Использовать ASCII при передаче файлов по FTP (нельзя совмещать с флагом $INET_BINARYTRANSFER (8)).
$INET_BINARYTRANSFER (8) = Использовать BINARY при передаче файлов по FTP (нельзя совмещать с флагом $INET_ASCIITRANSFER (4)). Режим по умолчанию, если флаги не заданы.
$INET_FORCEBYPASS (16) = Пропустить принудительный перевод соединения в онлайн (см. раздел Примечания).

Константы определены в InetConstants.au3.

Возвращаемое значение

Успех: Возвращает двоичную строку; @extended содержит количество загруженных байт.
Ошибка: Возвращает "" (пустую строку) и устанавливает @error ≠ 0.

Примечания

Параметр URL должен иметь вид "http://www.somesite.com/path/file.html" – как адрес в адресной строке браузера.

Для подключения с именем пользователя и паролем добавьте их перед именем сервера в формате "username:password@", например:
"http://myuser:mypassword@www.somesite.com"

Возвращаемые данные имеют двоичный формат. Для преобразования в строку используйте BinaryToString().

По умолчанию AutoIt принудительно устанавливает соединение перед загрузкой. Для коммутируемых пользователей это вызовет запрос на выход в сеть или набор номера (в зависимости от настроек системы). Флаг $INET_FORCEBYPASS (16) отключает это поведение. Отключение удобно для постоянных подключений (широкополосных, локальных). Кроме того, оно необходимо для обхода некоторых проблем в Windows Vista и Windows 7.

См. также

FtpSetProxy, HttpSetProxy, HttpSetUserAgent, InetGet, InetGetSize

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Прочитать файл без сохранения в папку. Выбрана опция "взять файл из локального кэша".
    Local $dData = InetRead("http://www.autoitscript.com/autoit3/files/beta/update.dat")

    ; Количество прочитанных байт возвращается через макрос @extended.
    Local $iBytesRead = @extended

    ; Преобразовать ANSI-совместимые двоичные данные обратно в строку.
    Local $sData = BinaryToString($dData)

    ; Показать результаты.
    MsgBox($MB_SYSTEMMODAL, "", "The number of bytes read: " & $iBytesRead & @CRLF & @CRLF & $sData)
EndFunc   ;==>Example
