Справочник функций

GUICtrlCreateUpdown

Создаёт элемент управления UpDown (счётчик) для GUI.

GUICtrlCreateUpdown ( inputcontrolID [, style = -1] )

Параметры

inputcontrolID controlID элемента управления Input, к которому привязывается UpDown, или -1 для последнего созданного элемента управления.
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $GUI_SS_DEFAULT_UPDOWN.
принудительные стили : $UDS_SETBUDDYINT и $UDS_ALIGNRIGHT, если выравнивание не задано.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для использования указанных выше значений необходимо включить в скрипт #include <UpDownConstants.au3>.

Минимальное и максимальное значения задаются через GUICtrlSetLimit().
По умолчанию Windows увеличивает значение при нажатии верхней кнопки со стрелкой.

Высота по умолчанию соответствует высоте связанного элемента управления Input.

См. также

GUICtrlCreateInput, GUICtrlSetData, GUICtrlSetLimit

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    GUICreate("My GUI UpDown", -1, -1, -1, -1, $WS_SIZEBOX)

    Local $idInput = GUICtrlCreateInput("2", 10, 10, 50, 20)
    GUICtrlCreateUpdown($idInput)

    ; Попытка изменить размер элемента управления Input
    GUICtrlSetPos($idInput, 10, 10, 100, 40)

    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Цикл до выхода пользователя.
    While 1
        $idMsg = GUIGetMsg()

        Switch $idMsg
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    MsgBox($MB_SYSTEMMODAL, "Updown", GUICtrlRead($idInput))
EndFunc   ;==>Example
