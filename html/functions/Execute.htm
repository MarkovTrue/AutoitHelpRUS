Справочник функций

Execute

Вычисляет выражение.

Execute ( string )

Параметры

string строка, содержащая вычисляемое выражение.

Возвращаемое значение

Успех: Возвращает результат вычисленного выражения.
Ошибка: Возвращает "" (пустую строку) и устанавливает @error ≠ 0.

См. также

Assign, Call, Eval

Пример

Local $a = 1
Local $v = Execute("$a+1") ; $v установлен в 2
