File, Directory and Disk functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
ConsoleRead Читает из потока STDIN процесса AutoIt-скрипта.
ConsoleWrite Записывает данные в поток STDOUT. Некоторые текстовые редакторы и сторонние программы могут читать этот поток.
ConsoleWriteError Записывает данные в поток STDERR. Некоторые текстовые редакторы и сторонние программы могут читать этот поток.
DirCopy Копирует каталог со всеми подкаталогами и файлами (аналог xcopy).
DirCreate Создаёт каталог/папку.
DirGetSize Возвращает размер указанного каталога в байтах.
DirMove Перемещает каталог со всеми подкаталогами и файлами.
DirRemove Удаляет каталог/папку.
DriveGetDrive Возвращает массив с перечнем дисков.
DriveGetFileSystem Возвращает тип файловой системы диска.
DriveGetLabel Возвращает метку тома диска, если она назначена.
DriveGetSerial Возвращает серийный номер диска.
DriveGetType Возвращает тип диска.
DriveMapAdd Подключает сетевой диск.
DriveMapDel Отключает сетевой диск.
DriveMapGet Возвращает сведения о подключённом сетевом диске.
DriveSetLabel Устанавливает метку тома диска.
DriveSpaceFree Возвращает объём свободного дискового пространства по указанному пути в мегабайтах.
DriveSpaceTotal Возвращает общий объём диска по указанному пути в мегабайтах.
DriveStatus Возвращает статус диска в виде строки.
FileChangeDir Изменяет текущий рабочий каталог.
FileClose Закрывает ранее открытый файл.
FileCopy Копирует один или несколько файлов.
FileCreateNTFSLink Создаёт жёсткую ссылку NTFS на файл или каталог.
FileCreateShortcut Создаёт ярлык (.lnk) для файла.
FileDelete Удаляет один или несколько файлов.
FileExists Проверяет существование файла или каталога.
FileFindFirstFile Создаёт дескриптор поиска, заданный путём и маской файлов.
FileFindNextFile Возвращает следующее имя файла, соответствующее дескриптору поиска.
FileFlush Сбрасывает буфер файла на диск.
FileGetAttrib Возвращает строку кодов, представляющую атрибуты файла.
FileGetEncoding Определяет кодировку текста в файле.
FileGetLongName Возвращает длинный путь и имя файла для переданного пути.
FileGetPos Возвращает текущую позицию в файле.
FileGetShortcut Возвращает сведения о ярлыке.
FileGetShortName Возвращает короткий путь и имя файла в формате 8.3 для переданного пути.
FileGetSize Возвращает размер файла в байтах.
FileGetTime Возвращает сведения о дате и времени файла.
FileGetVersion Возвращает сведения о версии, хранящиеся в файле.
FileInstall Включает файл в скомпилированный скрипт и извлекает его при выполнении.
FileMove Перемещает один или несколько файлов.
FileOpen Открывает файл для чтения или записи.
FileOpenDialog Открывает диалог выбора файла.
FileRead Читает указанное количество символов из ранее открытого файла.
FileReadLine Читает строку текста из ранее открытого текстового файла.
FileReadToArray Читает указанный файл в массив.
FileRecycle Отправляет файл или каталог в корзину.
FileRecycleEmpty Очищает корзину.
FileSaveDialog Открывает диалог сохранения файла.
FileSelectFolder Открывает диалог выбора папки.
FileSetAttrib Устанавливает атрибуты одного или нескольких файлов или каталогов.
FileSetEnd Устанавливает конец файла на текущей позиции.
FileSetPos Устанавливает текущую позицию в файле.
FileSetTime Устанавливает временну́ю метку одного или нескольких файлов.
FileWrite Записывает текст/данные в конец ранее открытого файла.
FileWriteLine Добавляет строку текста в конец ранее открытого текстового файла.
IniDelete Удаляет значение из .ini-файла стандартного формата.
IniRead Читает значение из .ini-файла стандартного формата.
IniReadSection Читает все пары ключ/значение из раздела .ini-файла стандартного формата.
IniReadSectionNames Читает все разделы .ini-файла стандартного формата.
IniRenameSection Переименовывает раздел в .ini-файле стандартного формата.
IniWrite Записывает значение в .ini-файл стандартного формата.
IniWriteSection Записывает раздел в .ini-файл стандартного формата.
