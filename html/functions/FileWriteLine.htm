Справочник функций

FileWriteLine

Добавляет строку текста в конец ранее открытого текстового файла.

FileWriteLine ( "filehandle/filename", "line" )

Параметры

filehandle/filename Дескриптор файла, возвращённый предыдущим вызовом FileOpen(). В качестве первого параметра можно также передать имя файла в виде строки.
line Строка текста для записи в текстовый файл. Если строка НЕ заканчивается символами @CR или @LF, автоматически добавляется перевод строки DOS (@CRLF).

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: 0 если файл не открыт в режиме записи, файл доступен только для чтения или запись по иным причинам невозможна.

Примечания

Текстовый файл должен быть открыт в режиме записи, иначе команда FileWriteLine() завершится с ошибкой.

Если вместо дескриптора указано имя файла, файл будет открыт и закрыт в ходе выполнения функции. При разборе больших текстовых файлов это значительно медленнее, чем работа через дескриптор. Однако, если файла не существует, он будет создан.

Примечание: не смешивайте дескрипторы и имена файлов – не вызывайте FileOpen() для файла, а затем не используйте в этой функции имя файла. Используйте в своих процедурах либо дескрипторы, либо имена файлов, но не то и другое одновременно.

При записи текста AutoIt по умолчанию использует UTF-8 (без BOM). Для записи в другой кодировке файл должен быть открыт через FileOpen() с соответствующими флагами.

Записываемый текст не может содержать символы Chr(0), так как вывод обрезается. Для записи таких символов необходимо использовать FileWrite() с файлом, открытым в двоичном режиме.

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для записи данных.
    If Not FileWrite($sFilePath, "Start of the FileWriteLine example, line 1. " & @CRLF) Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Открыть файл для записи (в конец файла) и сохранить дескриптор в переменной.
    Local $hFileOpen = FileOpen($sFilePath, $FO_APPEND)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Записать данные в файл через дескриптор, возвращённый FileOpen.
    FileWriteLine($hFileOpen, "Line 2")
    FileWriteLine($hFileOpen, "This is line 3 as a new line was appended to the last FileWriteLine call." & @CRLF)
    FileWriteLine($hFileOpen, "Line 4" & @CRLF)
    FileWriteLine($hFileOpen, "Line 5")

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Показать содержимое файла, передавая FileRead путь к файлу вместо дескриптора от FileOpen.
    MsgBox($MB_SYSTEMMODAL, "", "Contents of the file:" & @CRLF & FileRead($sFilePath))

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
