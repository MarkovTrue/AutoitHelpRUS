Справочник функций

GUICtrlCreateDummy

Создаёт фиктивный (Dummy) элемент управления для GUI.

GUICtrlCreateDummy ( )

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Этот элемент управления может получать сообщения через вызов GUICtrlSendToDummy(). Элемент "оповещает" в стандартном режиме, а значение, переданное через GUICtrlSendToDummy, можно прочитать с помощью GUICtrlRead().

См. также

GUICtrlRead, GUICtrlSendToDummy, GUICtrlSetData, GUICtrlSetOnEvent

Пример

#include <GUIConstantsEx.au3>

Example()

Func Example()

    GUICreate("GUICtrlCreateDummy", 250, 200, 100, 200)
    GUISetBkColor(0x00E0FFFF) ; изменит цвет фона

    Local $idDmy_User = GUICtrlCreateDummy()
    Local $idButton_Event = GUICtrlCreateButton("event", 75, 170, 70, 20)
    Local $idButton_Cancel = GUICtrlCreateButton("Cancel", 150, 170, 70, 20)
    GUISetState(@SW_SHOW)

    Local $idMsg
    ; Цикл до выхода пользователя.
    Do
        $idMsg = GUIGetMsg()

        Select
            Case $idMsg = $idButton_Event
                GUICtrlSendToDummy($idDmy_User)
            Case $idMsg = $idButton_Cancel
                GUICtrlSendToDummy($idDmy_User)
            Case $idMsg = $idDmy_User
                ; специальное действие перед закрытием
                ; ...
                Exit
        EndSelect
    Until $idMsg = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
