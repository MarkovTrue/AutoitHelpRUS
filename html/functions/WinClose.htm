WinClose

Закрывает окно.

WinClose ( "title" [, "text"] )

Параметры

title Заголовок/hWnd/класс закрываемого окна. См. Специальное определение заголовка.
text [необязательный] Текст закрываемого окна. По умолчанию – пустая строка. См. Специальное определение текста.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если окно не найдено.

Примечания

Функция отправляет окну сообщение о закрытии; результат зависит от окна (оно может запросить сохранение данных и т.д.). Чтобы принудительно закрыть окно, используйте WinKill(). Если несколько окон соответствуют критериям, закрывается то из них, которое было активно последним.

См. также

ProcessClose, WinActivate, WinExists, WinKill, WinMove, WinSetState, WinTitleMatchMode (Option)

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать 2 секунды отображения окна Блокнота.
    Sleep(2000)

    ; Закрыть окно Блокнота по classname Notepad.
    If WinClose("[CLASS:Notepad]", "") Then
        MsgBox($MB_SYSTEMMODAL, "", "Window closed")
    Else
        MsgBox($MB_SYSTEMMODAL + $MB_ICONERROR, "Error", "Window not Found")
    EndIf
EndFunc   ;==>Example
