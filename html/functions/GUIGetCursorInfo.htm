Справочник функций

GUIGetCursorInfo

Возвращает положение курсора мыши относительно окна GUI.

GUIGetCursorInfo ( [winhandle] )

Параметры

winhandle [необязательный] Дескриптор окна. Если не указан, используется "текущее" окно.

Возвращаемое значение

Успех: Возвращает массив из пяти элементов с информацией о курсоре мыши:
$aArray[0] = координата X (горизонтальная)
$aArray[1] = координата Y (вертикальная)
$aArray[2] = основная кнопка (1 – нажата, 0 – не нажата)
$aArray[3] = дополнительная кнопка (1 – нажата, 0 – не нажата)
$aArray[4] = идентификатор элемента управления, над которым находится курсор (или 0)
Ошибка: Устанавливает @error ≠ 0.

Примечания

Координаты задаются относительно окна GUI (клиентские координаты).

Если указан параметр winhandle, указанное окно становится новым "текущим" окном.

Положение курсора мыши определяется только для окна, созданного через GUICreate(). Без winhandle работает при условии, что окно GUI активно.

ControlID для ListViewItem или TreeViewItem никогда не возвращается – возвращается только controlID родительского ListView или TreeView.

См. также

GUICreate, GUIGetMsg

Пример

#include <GUIConstantsEx.au3>

Global $g_idLbl_X = 0, $g_idLbl_Y = 0

Example()

Func Example()
    HotKeySet("{ESC}", "GetPos")

    GUICreate("Press Esc to Get Pos", 400, 400)
    $g_idLbl_X = GUICtrlCreateLabel("0", 10, 10, 50)
    $g_idLbl_Y = GUICtrlCreateLabel("0", 10, 30, 50)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Func GetPos()
    Local $a = GUIGetCursorInfo()
    GUICtrlSetData($g_idLbl_X, $a[0])
    GUICtrlSetData($g_idLbl_Y, $a[1])
EndFunc   ;==>GetPos
