Справочник функций

HotKeySet

Устанавливает горячую клавишу, которая вызывает пользовательскую функцию.

HotKeySet ( "key" [, "function"] )

Параметры

key Сочетание клавиш, назначаемое как горячая клавиша. Формат тот же, что у Send(). См. раздел Примечания.
function [необязательный] Имя функции, которая будет вызвана при нажатии горячей клавиши. Если параметр не указан, ранее назначенная горячая клавиша сбрасывается.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Для назначения горячих клавиш лучше использовать строчные буквы (например, "b", а не "B") – в некоторых раскладках клавиатуры заглавные и строчные символы могут быть привязаны к разным клавишам.

На клавиатурах со 102 клавишами, например на венгерской, для перехвата клавиши "í" используйте "{OEM_102}".

Если два AutoIt-скрипта назначают одинаковые горячие клавиши, лучше не запускать их одновременно: второй скрипт не сможет перехватить клавишу, пока первый не завершится или не отменит её регистрацию. Если скрипты используют GUI, попробуйте GUISetAccelerators – эти клавиши активны только когда родительское окно в фокусе.

Нажатие горячей клавиши обычно прерывает выполняющуюся AutoIt-функцию или инструкцию и запускает пользовательскую функцию, пока та не завершится или не будет прервана. Исключения:
1) Если текущая функция "блокирующая", нажатия клавиш буферизуются и выполняются сразу после её завершения. Примеры блокирующих функций: MsgBox() и FileSelectFolder(). Посмотрите поведение Shift-Alt-d в примере.
2) Если скрипт приостановлен щелчком по иконке AutoIt в трее, нажатия горячих клавиш в этом состоянии игнорируются.

Следующие горячие клавиши назначить нельзя:

ASC nnnn Не работает так же, как в Send(). При недопустимой записи "{...}" берётся первый символ "A" – так же, как это делает Send().
Ctrl+Alt+Delete Зарезервировано Windows.
F12 Тоже зарезервировано Windows, согласно Windows API.
Клавиша Enter на числовой клавиатуре Используйте {Enter} – она перехватывает обе клавиши Enter.
Win+B,D,E,F,L,M,R,U; и Win+Shift+M Это встроенные сочетания Windows. Примечание: Win+B и Win+L могут быть зарезервированы только начиная с Windows XP.
Alt, Ctrl, Shift, Win Это сами модификаторы!
Прочее Любые глобальные горячие клавиши, назначенные сторонними программами. Любые комбинации из двух и более "основных" клавиш, например "{F1}{F2}". Любые клавиши вида "{LALT}" или "{ALTDOWN}".

Когда вы назначаете горячую клавишу, AutoIt перехватывает её нажатие и не передаёт активному приложению. Исключение – клавиши-переключатели (NumLock, CapsLock, ScrollLock): они всё равно переключают своё состояние!
Чтобы отправить через Send() комбинацию, которая сама вызовет событие HotKeySet(), используйте ControlSend() или заранее отмените регистрацию HotKeySet() – иначе возможен бесконечный цикл.

; перехватить и передать нажатие далее
HotKeySet("{Esc}", "captureEsc")
Func captureEsc()
    ; ... здесь можно что-то сделать
    HotKeySet("{Esc}")
    Send("{Esc}")
    HotKeySet("{Esc}", "captureEsc")
EndFunc



Вызываемой функции нельзя передать параметры – они будут проигнорированы.

Внутри функции можно использовать макрос @HotKeyPressed, чтобы одной функцией обрабатывать несколько клавиш.

Пример

Пример 1

#include <MsgBoxConstants.au3>

; Нажмите Esc для завершения скрипта, Pause/Break чтобы "поставить на паузу"

Global $g_bPaused = False

HotKeySet("{PAUSE}", "TogglePause")
HotKeySet("{ESC}", "Terminate")
HotKeySet("+!d", "ShowMessage") ; Shift-Alt-d

While 1
    Sleep(100)
WEnd

Func TogglePause()
    $g_bPaused = Not $g_bPaused
    While $g_bPaused
        Sleep(100)
        ToolTip('Script is "Paused"', 0, 0)
    WEnd
    ToolTip("")
EndFunc   ;==>TogglePause

Func Terminate()
    Exit
EndFunc   ;==>Terminate

Func ShowMessage()
    MsgBox($MB_SYSTEMMODAL, "", "This is a message.")
EndFunc   ;==>ShowMessage

Пример 2

; == Пример 2, использование @HotKeyPressed

#include <MsgBoxConstants.au3>

; Нажмите Esc для завершения скрипта, Pause/Break чтобы "поставить на паузу"

Global $g_bPaused = False

HotKeySet("{PAUSE}", "HotKeyPressed")
HotKeySet("{ESC}", "HotKeyPressed")
HotKeySet("+!d", "HotKeyPressed") ; Shift-Alt-d

While 1
    Sleep(100)
WEnd

Func HotKeyPressed()
    Switch @HotKeyPressed ; Последняя нажатая горячая клавиша.
        Case "{PAUSE}" ; Строка – горячая клавиша {PAUSE}.
            $g_bPaused = Not $g_bPaused
            While $g_bPaused
                Sleep(100)
                ToolTip('Script is "Paused"', 0, 0)
            WEnd
            ToolTip("")

        Case "{ESC}" ; Строка – горячая клавиша {ESC}.
            Exit

        Case "+!d" ; Строка – горячая клавиша Shift-Alt-d.
            MsgBox($MB_SYSTEMMODAL, "", "This is a message.")

    EndSwitch
EndFunc   ;==>HotKeyPressed
