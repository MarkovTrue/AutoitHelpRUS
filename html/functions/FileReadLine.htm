Справочник функций

FileReadLine

Читает строку текста из ранее открытого текстового файла.

FileReadLine ( "filehandle/filename" [, line = 1] )

Параметры

filehandle/filename Дескриптор файла, возвращённый предыдущим вызовом FileOpen(). Альтернативно можно передать имя файла строкой в качестве первого параметра.
line [необязательный] Номер читаемой строки. Первая строка файла имеет номер 1 (не 0); последняя строка – -1.

Возвращаемое значение

Успех: Возвращает прочитанную строку текста.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – файл не открыт в режиме чтения или иная ошибка
-1 – достигнут конец файла.

Примечания

Возвращает текст прочитанной строки; символы перевода строки (@CR или @LF) в конце строки удаляются автоматически.

Если передан дескриптор файла без указания номера строки, читается "следующая" строка – для только что открытого файла это первая строка. При чтении в цикле рекомендуется не использовать параметр line и позволить счётчику увеличиваться автоматически: явное указание номера заставляет AutoIt каждый раз перечитывать файл с начала, что значительно замедляет работу с большими файлами.

Если передано имя файла, при каждом вызове функции файл открывается и закрывается, что существенно снижает производительность, особенно при чтении нескольких строк. Кроме того, поскольку файл закрывается после каждого вызова, автоматического перехода к "следующей" строке нет, и при чтении в цикле необходимо явно задавать параметр line. Это дополнительно замедляет работу, поскольку AutoIt каждый раз перечитывает файл заново.

Не смешивайте дескрипторы и имена файлов: не открывайте файл через FileOpen(), а затем передавайте его имя в эту функцию. В своих процедурах используйте либо дескрипторы (рекомендуется), либо имена файлов, но не оба подхода одновременно.

Поддерживается чтение как в формате ANSI, так и UTF-16/UTF-8 – подробнее см. FileOpen().

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать временный файл для чтения данных.
    If Not FileWrite($sFilePath, "This is an example of using FileReadLine.") Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred whilst writing the temporary file.")
        Return False
    EndIf

    ; Открыть файл на чтение и сохранить дескриптор в переменную.
    Local $hFileOpen = FileOpen($sFilePath, $FO_READ)
    If $hFileOpen = -1 Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred when reading the file.")
        Return False
    EndIf

    ; Читаем первую строку файла, используя дескриптор, возвращённый FileOpen.
    Local $sFileRead = FileReadLine($hFileOpen, 1)

    ; Закрыть дескриптор, возвращённый FileOpen.
    FileClose($hFileOpen)

    ; Выводим первую строку файла.
    MsgBox($MB_SYSTEMMODAL, "", "First line of the file:" & @CRLF & $sFileRead)

    ; Удалить временный файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
