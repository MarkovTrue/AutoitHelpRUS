Справочник функций

SplashImageOn

Создаёт настраиваемое всплывающее окно с изображением.

SplashImageOn ( "title", "file" [, width [, height [, x pos [, y pos [, opt]]]]] )

Параметры

title Заголовок всплывающего окна.
file Полный путь и имя файла изображения (BMP, GIF или JPG).
width [необязательный] Ширина окна в пикселях (по умолчанию 500).
height [необязательный] Высота окна в пикселях (по умолчанию 400).
x pos [необязательный] Позиция окна от левого края экрана в пикселях (по умолчанию – по центру).
y pos [необязательный] Позиция окна от верхнего края экрана в пикселях (по умолчанию – по центру).
opt [необязательный] По умолчанию – "всегда поверх с заголовком". Суммируйте нужные значения:
$DLG_NOTITLE (1) = окно без рамки и заголовка
$DLG_NOTONTOP (2) = без атрибута "всегда поверх"
$DLG_MOVEABLE (16) = окно можно перемещать

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Отсутствует.

Примечания

Для пропуска необязательного параметра с сохранением значения по умолчанию передайте –1.

Изображение масштабируется до указанных ширины и высоты; поддерживаются форматы BMP, GIF и JPEG. Значки (ICO) не отображаются, хотя окно всё равно появляется. PNG-файлы и несуществующие пути вызывают ошибку загрузки.

Изображения не встраиваются в исполняемый файл, если не использовать FileInstall().

Одновременно допускается только одно окно SplashImage/Text. Для смены изображения или текста достаточно вызвать SplashImageOn() или SplashTextOn() с новыми данными.

Окно с opt=1 нельзя перемещать и активировать щелчком мыши.

См. также

SplashOff, SplashTextOn

Пример

Local $sDestination = "..\GUI\mslogo.jpg"

SplashImageOn("Splash Screen", $sDestination, 250, 50)
Sleep(3000)
SplashOff()
