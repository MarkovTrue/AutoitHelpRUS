Справочник функций

DirMove

Перемещает каталог со всеми подкаталогами и файлами.

DirMove ( "source dir", "dest dir" [, flag = 0] )

Параметры

source dir Путь к исходному каталогу (без завершающего обратного слеша), например "C:\Path1".
dest dir Путь к каталогу назначения (без завершающего обратного слеша), например "C:\Path_Copy".
flag [необязательный] Флаг, определяющий, перезаписывать ли существующие файлы:
$FC_NOOVERWRITE (0) = (по умолчанию) не перезаписывать существующие файлы
$FC_OVERWRITE (1) = перезаписывать существующие файлы

Константы определены в FileConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если при перемещении каталога произошла ошибка.

Примечания

Если источник и назначение находятся на разных томах или используются UNC-пути, вместо перемещения выполняется копирование с последующим удалением.

Если каталог назначения уже существует и указан флаг перезаписи, исходный каталог будет перемещён внутрь каталога назначения.

В AutoIt нет функции "DirRename" – для переименования папки используйте эту функцию, указав "Full_Path\Old_Name" и "Full_Path\New_Name" в качестве параметров source dir и dest dir.

См. также

DirRemove, FileMove

Пример

DirMove(@MyDocumentsDir, @TempDir & "\Backups\MyDocs")
