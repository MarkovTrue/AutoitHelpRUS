DirCreate

Создаёт каталог/папку.

DirCreate ( "path" )

Параметры

path Путь к создаваемому каталогу.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если при создании каталога произошла ошибка.

Примечания

Функция также создаст все родительские каталоги, указанные в path, если они ещё не существуют.

См. также

DirRemove, FileCopy

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать локальную константу с каталогом.
    Local Const $sFilePath = @TempDir & "\DirCreateFolder"

    ; Если каталог существует – не продолжать.
    If FileExists($sFilePath) Then
        MsgBox($MB_SYSTEMMODAL, "", "An error occurred. The directory already exists.")
        Return False
    EndIf

    ; Открыть временный каталог.
    ShellExecute(@TempDir)

    ; Создать каталог.
    DirCreate($sFilePath)

    ; Показать сообщение о создании каталога.
    MsgBox($MB_SYSTEMMODAL, "", "The directory has been created.")

    ; Удалить каталог и все подкаталоги.
    DirRemove($sFilePath, $DIR_REMOVE)

    ; Показать сообщение об удалении каталога.
    MsgBox($MB_SYSTEMMODAL, "", "The sub folder: Folder2 has been deleted.")
EndFunc   ;==>Example
