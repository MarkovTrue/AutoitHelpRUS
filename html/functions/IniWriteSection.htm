Справочник функций

IniWriteSection

Записывает раздел в .ini-файл стандартного формата.

IniWriteSection ( "filename", "section", "data" [, index = 1] )

Параметры

filename Имя .ini-файла.
section Имя секции в .ini-файле.
data Данные для записи. Можно передать строку или массив. Если передаётся строка, каждая пара ключ=значение должна разделяться @LF. Если передаётся массив, он должен быть двумерным, а вторая размерность – 2 элемента.
index [необязательный] Если в качестве данных передаётся массив, задаёт начальный индекс для записи. По умолчанию равен 1, что позволяет сразу использовать результат IniReadSection(). Для вручную созданных массивов может потребоваться другое значение в зависимости от способа создания массива. При передаче строки параметр игнорируется.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0. Функция устанавливает значение @error = 1, если формат данных недопустим.

Примечания

Стандартный .ini-файл выглядит следующим образом:
[SectionName]
Key=Value

Если файл не существует, он создаётся. Несуществующие каталоги не создаются. Ключи и разделы добавляются в конец без какой-либо сортировки.
Если записываемый раздел уже существует, его содержимое перезаписывается.

Для работы с .ini-файлом в кодировке Unicode сначала создайте файл через FileOpen(), указав в параметре mode значение "Unicode UTF16 Little Endian".

См. также

IniDelete, IniRead, IniReadSection, IniReadSectionNames, IniRenameSection, IniWrite

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Создать локальную константу с путём к файлу для чтения/записи.
    Local Const $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Создать структуру раздела .ini в виде массива. Нулевой элемент – количество элементов, в данном случае 3.
    Local $aSection[4][2] = [[3, ""], ["Title", "AutoIt"], ["Version", @AutoItVersion], ["OS", @OSVersion]]

    ; Записать массив в раздел с именем 'General'.
    IniWriteSection($sFilePath, "General", $aSection)

    ; Прочитать секцию INI 'General'. Возвращается двумерный массив.
    Local $aArray = IniReadSection($sFilePath, "General")

    ; Проверить, произошла ли ошибка.
    If Not @error Then
        ; Пройти по массиву, показывая ключи и их значения.
        For $i = 1 To $aArray[0][0]
            MsgBox($MB_SYSTEMMODAL, "", "Key: " & $aArray[$i][0] & @CRLF & "Value: " & $aArray[$i][1])
        Next
    EndIf

    ; Удалить INI-файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
