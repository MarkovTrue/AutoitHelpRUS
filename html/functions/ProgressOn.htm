Справочник функций

ProgressOn

Создаёт настраиваемое окно с индикатором выполнения.

ProgressOn ( "title", "maintext" [, "subtext" [, x pos [, y pos [, opt]]]] )

Параметры

title Заголовок окна.
maintext Основной текст – жирный, расположен в верхней части окна.
subtext [необязательный] Вспомогательный текст – обычный шрифт, расположен ниже. По умолчанию пусто.
x pos [необязательный] Положение окна по горизонтали в пикселях. По умолчанию по центру.
y pos [необязательный] Положение окна по вертикали в пикселях. По умолчанию по центру.
opt [необязательный] По умолчанию "поверх всех/с заголовком/нельзя двигать".
Суммируйте нужные значения:
$DLG_NOTITLE (1) = без рамки и заголовка
$DLG_NOTONTOP (2) = без атрибута "поверх всех"
$DLG_MOVEABLE (16) = окно можно перемещать

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Отсутствует.

Примечания

Чтобы пропустить необязательный параметр, сохранив его значение по умолчанию, передайте:
    "" для строковых параметров
    -1 для числовых параметров
Если ProgressOn() вызывается, когда окно уже открыто, оно перерисовывается или перемещается. (Для одновременного отображения нескольких окон требуется несколько скриптов.)

Если вы хотите использовать основной текст из 2 строк через ProgressSet(), при создании окна также необходимо задать 2 строки.
Более 2 строк приведёт к искажению отображения.

См. также

ProgressOff, ProgressSet

Пример

#include <AutoItConstants.au3>

Example()

Func Example()
    ; Показать окно индикатора выполнения (maintext в 2 строки).
    ProgressOn("Progress Meter", "Increments every second" & @CRLF & "...", "0%", -1, -1, BitOR($DLG_NOTONTOP, $DLG_MOVEABLE))

    ; Обновлять значение индикатора выполнения каждую секунду.
    For $i = 10 To 100 Step 10
        Sleep(1000)
        ProgressSet($i, $i & "%")
    Next

    ; Установить "subtext" и "maintext" окна индикатора выполнения.
    ProgressSet(100, "Done", "Complete")
    Sleep(5000)

    ; Закрыть окно индикатора выполнения.
    ProgressOff()
EndFunc   ;==>Example
