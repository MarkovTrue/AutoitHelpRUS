Справочник функций

InetGetInfo

Возвращает подробные сведения для дескриптора, полученного от InetGet().

InetGetInfo ( [handle [, index = -1]] )

Параметры

handle [необязательный] Дескриптор, возвращённый функцией InetGet().
index [необязательный] Индекс запрашиваемых данных. При значении -1 возвращается массив, содержащий все перечисленные ниже данные.
$INET_DOWNLOADREAD (0) – количество прочитанных байт на текущий момент (обновляется в процессе загрузки).
$INET_DOWNLOADSIZE (1) – размер загрузки в байтах (может отсутствовать).
$INET_DOWNLOADCOMPLETE (2) – True, если загрузка завершена; False, если загрузка ещё продолжается.
$INET_DOWNLOADSUCCESS (3) – True, если загрузка прошла успешно. Если False, следующий элемент будет ненулевым.
$INET_DOWNLOADERROR (4) – код ошибки загрузки. Значение произвольное; для определения ошибки достаточно проверить, что оно ненулевое.
$INET_DOWNLOADEXTENDED (5) – расширенное значение загрузки. Значение произвольное и предназначено главным образом для разработчиков AutoIt.

Константы определены в InetConstants.au3.

Возвращаемое значение

Успех: Возвращает запрошенные данные.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.

Примечания

Если функция вызвана без аргументов, возвращается общее количество активных загрузок.

Функцию можно вызывать в цикле для получения количества загруженных байт или ожидания завершения загрузки.

InetGet

Пример

#include <InetConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Example()

Func Example()
    ; Сохранить загруженный файл во временную папку.
    Local $sFilePath = _WinAPI_GetTempFileName(@TempDir)

    ; Загрузить файл в фоне с опцией 'принудительно перечитать с удалённого сайта.'
    Local $hDownload = InetGet("http://www.autoitscript.com/autoit3/files/beta/update.dat", $sFilePath, $INET_FORCERELOAD, $INET_DOWNLOADBACKGROUND)

    ; Ждать завершения загрузки, отслеживая возврат True вторым индексом InetGetInfo.
    Do
        Sleep(250)
    Until InetGetInfo($hDownload, $INET_DOWNLOADCOMPLETE)

    ; Получить сведения о загружаемом файле.
    Local $aData = InetGetInfo($hDownload)
    If @error Then
        FileDelete($sFilePath)
        Return False ; Если произошла ошибка – выйти из функции и удалить файл.
    EndIf

    ; Закрыть дескриптор, возвращённый InetGet.
    InetClose($hDownload)

    ; Отобразить сведения о загруженном файле.
    MsgBox($MB_SYSTEMMODAL, "", "Bytes read: " & $aData[$INET_DOWNLOADREAD] & @CRLF & _
            "Size: " & $aData[$INET_DOWNLOADSIZE] & @CRLF & _
            "Complete: " & $aData[$INET_DOWNLOADCOMPLETE] & @CRLF & _
            "successful: " & $aData[$INET_DOWNLOADSUCCESS] & @CRLF & _
            "@error: " & $aData[$INET_DOWNLOADERROR] & @CRLF & _
            "@extended: " & $aData[$INET_DOWNLOADEXTENDED] & @CRLF)

    ; Удалить файл.
    FileDelete($sFilePath)
EndFunc   ;==>Example
