Справочник функций

StringStripWS

Удаляет пробельные символы в строке.

StringStripWS ( "string", flag )

Параметры

string Строка, из которой удаляются пробельные символы.
flag Определяет тип удаления пробельных символов:
$STR_STRIPLEADING (1) = удалить начальные пробелы
$STR_STRIPTRAILING (2) = удалить конечные пробелы
$STR_STRIPSPACES (4) = удалить двойные (и более) пробелы/табуляции между словами
$STR_STRIPALL (8) = удалить все пробелы и табуляции

Константы определены в StringConstants.au3.

Возвращаемое значение

Возвращает новую строку с удалёнными пробельными символами.

Примечания

Пробельными считаются символы Chr(9) – Chr(13), Chr(0) и Chr(32).

Для удаления одиночных пробелов используйте StringReplace().

См. также

StringIsSpace, StringReplace, StringStripCR

Пример

#include <MsgBoxConstants.au3>
#include <StringConstants.au3>

; Удалить начальные и конечные пробелы из строки.
Local $sString = StringStripWS(" This is a sentence with whitespace. ", $STR_STRIPLEADING + $STR_STRIPTRAILING + $STR_STRIPSPACES)
MsgBox($MB_SYSTEMMODAL, "", $sString)
