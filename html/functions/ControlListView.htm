Справочник функций

ControlListView

Отправляет команду элементу управления ListView32.

ControlListView ( "title", "text", controlID, "command" [, option1 [, option2]] )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.
command Команда, передаваемая элементу управления (см. ниже).
option1 [необязательный] Дополнительный параметр, необходимый для некоторых команд.
option2 [необязательный] Дополнительный параметр, необходимый для некоторых команд.

Возвращаемое значение

Возвращаемое значение зависит от команды, как показано в таблице ниже. В случае ошибки (например, неверная команда или окно/элемент управления не найдены) @error устанавливается в 1.

Команда, Параметр1, Параметр2 Операция
"DeSelect", От [, До] Снимает выделение с одного или нескольких элементов.
"FindItem", "строка поиска" [, SubItem] Возвращает индекс элемента с указанной строкой. Возвращает -1, если строка не найдена.
"GetItemCount" Возвращает количество элементов списка.
"GetSelected" [, параметр] Возвращает строку с индексами выбранных элементов. Если параметр=0 (по умолчанию), возвращается только первый выбранный элемент. Если параметр=1, возвращаются все выбранные элементы, разделённые |, например: "0|3|4|10". Если ни один элемент не выбран, возвращается пустая строка "".
"GetSelectedCount" Возвращает количество выбранных элементов.
"GetSubItemCount" Возвращает количество подэлементов.
"GetText", Элемент, Подэлемент Возвращает текст указанного элемента/подэлемента.
"IsSelected", Элемент Возвращает 1, если элемент выбран, иначе возвращает 0.
"Select", От [, До] Выбирает один или несколько элементов.
"SelectAll" Выбирает все элементы.
"SelectClear" Снимает выделение со всех элементов.
"SelectInvert" Инвертирует текущее выделение.
"ViewChange", "вид" Изменяет текущий вид. Допустимые виды: "list", "details", "smallicons", "largeicons".


В режиме "Details" элемента управления ListView32 понятие "item" соответствует "строке", а "subitem" – "столбцу".

Примечания

Некоторые команды могут завершиться ошибкой при использовании 32-разрядного процесса AutoIt для чтения из 64-разрядного процесса. Аналогично команды могут не работать при использовании 64-разрядного процесса AutoIt для чтения из 32-разрядного.

Пример

#include <MsgBoxConstants.au3>

ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "SelectAll")
ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "Deselect", 2, 5)
MsgBox($MB_SYSTEMMODAL, "", ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "GetText", 9, 0))
MsgBox($MB_SYSTEMMODAL, "", ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "FindItem", "14 KB", 1))
MsgBox($MB_SYSTEMMODAL, "", ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "GetSelected", 1))
