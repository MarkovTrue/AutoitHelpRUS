Справочник функций

EnvGet

Возвращает значение переменной окружения.

EnvGet ( "envvariable" )

Параметры

envvariable Имя переменной окружения, например "TEMP" или "PATH".

Возвращаемое значение

Возвращает значение запрошенной переменной (или пустую строку, если переменная не существует).

См. также

EnvSet, EnvUpdate

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получение значения переменной окружения %APPDATA%.
    ; При присваивании и получении имя переменной окружения указывается без знаков процента (%).
    Local $sEnvVar = EnvGet("APPDATA")

    ; Отображение значения переменной окружения %APPDATA%.
    MsgBox($MB_SYSTEMMODAL, "", "The environment variable %APPDATA% has the value of: " & @CRLF & @CRLF & $sEnvVar) ; Возвращает то же значение, что и макрос @AppDataDir.
EndFunc   ;==>Example
