Справочник функций

EnvUpdate

Обновляет переменные окружения ОС.

EnvUpdate ( )

Возвращаемое значение

Успех: Нет.
Ошибка: Устанавливает значение @error = 1.

Примечания

Эффект аналогичен выходу из системы и повторному входу. Например, изменения переменной окружения %path% могут не вступить в силу, пока не будет вызвана EnvUpdate() (либо пока вы не выйдете из системы или не перезагрузите компьютер).

См. также

EnvGet, EnvSet

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Получаем системную переменную окружения %PATH%.
    Local $sEnvVar = EnvGet("PATH")

    ; Присваиваем системной переменной %PATH% её текущее значение, дополненное каталогом скрипта.
    ; При записи в переменную окружения знаки процента (%) указывать не нужно.
    EnvSet("PATH", $sEnvVar & ";" & @ScriptDir)

    ; Обновляем переменные окружения ОС, чтобы изменения вступили в силу.
    EnvUpdate()

    ; Считываем переменную окружения, которой только что присвоили значение.
    $sEnvVar = EnvGet("PATH")

    ; Выводим значение переменной окружения %PATH%.
    MsgBox($MB_SYSTEMMODAL, "", "The environment variable %PATH% has the value of: " & $sEnvVar)
EndFunc   ;==>Example
