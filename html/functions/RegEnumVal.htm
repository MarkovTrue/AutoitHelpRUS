Справочник функций

RegEnumVal

Читает имя значения реестра по его порядковому номеру.

RegEnumVal ( "keyname", instance )

Параметры

keyname Читаемый раздел реестра.
instance Порядковый номер значения (нумерация с 1).

Возвращаемое значение

Успех: Возвращает имя запрошенного значения реестра; @extended устанавливается в тип значения.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – невозможно открыть запрошенный раздел
2 – невозможно открыть главный раздел
3 – невозможно подключиться к удалённому реестру
-1 – невозможно получить имя значения (номер экземпляра вне диапазона)

Примечания

Раздел реестра должен начинаться с "HKEY_LOCAL_MACHINE" ("HKLM"), "HKEY_USERS" ("HKU"), "HKEY_CURRENT_USER" ("HKCU"), "HKEY_CLASSES_ROOT" ("HKCR") или "HKEY_CURRENT_CONFIG" ("HKCC").

При работе в 64-разрядной Windows для перебора значений 64-разрядной ветки реестра добавьте суффикс 64 к корневому ключу, например HKLM64.

См. также

RegDelete, RegEnumKey, RegWrite

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $i, $sVar, $sWow64 = "", $sEnumVal = "Under AutoIt3 key." & @CRLF & @CRLF
    Local $aRegValueType[12] = ["REG_NONE", "REG_SZ", "REG_EXPAND_SZ", "REG_BINARY", _
            "REG_DWORD_LITTLE_ENDIAN", "REG_DWORD_BIG_ENDIAN", "REG_LINK", _
            "REG_MULTI_SZ", "REG_RESOURCE_LIST", "REG_FULL_RESOURCE_DESCRIPTOR", _
            "REG_RESOURCE_REQUIREMENTS_LIST", "REG_QWORD_LITTLE_ENDIAN"]

    ; Поддержка работы в X64
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    For $i = 1 To 100
        $sVar = RegEnumVal("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", $i)
        If @error <> 0 Then ExitLoop
        $sEnumVal &= "# " & $i & ":" & StringLeft($aRegValueType[@extended] & " ", 31) & $sVar & @CRLF
    Next

    MsgBox($MB_SYSTEMMODAL, "RegEnumVal example", $sEnumVal)

EndFunc   ;==>Example
