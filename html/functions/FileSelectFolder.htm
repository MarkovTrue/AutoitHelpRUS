FileSelectFolder

Открывает диалог выбора папки.

FileSelectFolder ( "dialog text", "root dir" [, flag = 0 [, "initial dir" [, hwnd]]] )

Параметры

dialog text Текст приветствия в диалоговом окне.
root dir Корневой каталог дерева файлов диалога – используется для ограничения выбора пользователя. Пустая строка "" означает Рабочий стол – см. раздел Примечания.
flag [необязательный]
$FSF_CREATEBUTTON (1) = показывать кнопку "Создать папку" (только XP)
$FSF_NEWDIALOG (2) = использовать новый стиль диалога (только XP)
$FSF_EDITCONTROL (4) = показывать поле ввода пути (только XP)

Константы определены в FileConstants.au3.
initial dir [необязательный] Полный путь к папке, которая будет выделена при открытии диалога (если она существует в корневой папке). По умолчанию – пустая строка ("").
hwnd [необязательный] Дескриптор окна, используемого в качестве родительского для диалога.

Возвращаемое значение

Успех: Возвращает полный путь к выбранной папке.
Ошибка: "" (пустую строку) и устанавливает значение @error = 1, если пользователь отменил выбор или закрыл окно.

Примечания

Указание несуществующего корневого каталога также приводит к использованию папки "Рабочий стол". Обратите внимание, что Desktop является корневой папкой всего компьютера и не совпадает с @DesktopDir – каталогом рабочего стола текущего пользователя.
Специальные папки Windows (например, "Мои документы") можно задать в качестве корневой, используя соответствующий CLSID из Приложения.
UNC-пути не поддерживаются. Если пользователи могут выбирать файлы по UNC-пути, его необходимо предварительно подключить как диск.

При наличии возможности используется новый стиль диалога Vista и выше, независимо от флага Use New Dialog Style. В стиле Vista поле ввода пути и кнопка создания папки всегда отображаются (флаги игнорируются), а параметр dialog text задаёт заголовок окна.

FileOpenDialog, FileSaveDialog

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создаём локальную константу с сообщением для отображения в FileSelectFolder.
    Local Const $sMessage = "Select a folder"

    ; Показать диалог открытия файла.
    Local $sFileSelectFolder = FileSelectFolder($sMessage, "")
    If @error Then
        ; Показать сообщение об ошибке.
        MsgBox($MB_SYSTEMMODAL, "", "No folder was selected.")
    Else
        ; Выводим выбранную папку.
        MsgBox($MB_SYSTEMMODAL, "", "You chose the following folder:" & @CRLF & $sFileSelectFolder)
    EndIf
EndFunc   ;==>Example
