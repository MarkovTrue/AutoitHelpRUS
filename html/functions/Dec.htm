Справочник функций

Dec

Возвращает числовое представление шестнадцатеричной строки.

Dec ( "hex" [, flag = 0] )

Параметры

hex Шестнадцатеричная строка для преобразования.
flag [необязательный] Определяет поведение функции.
Может принимать одно из следующих значений:
   $NUMBER_AUTO (0) = строка интерпретируется как целое число (по умолчанию). См. раздел Примечания.
   $NUMBER_32BIT (1) = строка интерпретируется как 32-разрядное целое
   $NUMBER_64BIT (2) = строка интерпретируется как 64-разрядное целое
   $NUMBER_DOUBLE (3) = строка интерпретируется как число double

Константы определены в "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает требуемые данные.
Failure: Возвращает 0 и устанавливает значение @error = 1, если передана недопустимая шестнадцатеричная строка или произошло переполнение.

Примечания

По умолчанию входная строка рассматривается как целое число. В этом случае, если результат находится в пределах диапазона 32-разрядного целого, он возвращается как 32-разрядное целое, иначе – как 64-разрядное; оба со знаком.

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

_Example()
Func _Example()
    Local $iDec = Dec("FFF", $NUMBER_AUTO)
    MsgBox($MB_SYSTEMMODAL, "", $iDec) ; Отображает число 4095.
EndFunc   ;==>_Example
