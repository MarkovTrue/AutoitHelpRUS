Справочник функций

WinMove

Перемещает и/или изменяет размер окна.

WinMove ( "title", "text", x, y [, width [, height [, speed]]] )

Параметры

title Заголовок/hWnd/класс перемещаемого/изменяемого окна. См. Специальное определение заголовка.
text Текст перемещаемого/изменяемого окна. См. Специальное определение текста.
x Координата X, в которую нужно переместить окно.
y Координата Y, в которую нужно переместить окно.
width [необязательный] Новая ширина окна.
height [необязательный] Новая высота окна.
speed [необязательный] Скорость перемещения окна от 1 (быстрее всего) до 100 (медленнее всего). Если не задано, перемещение происходит мгновенно.

Возвращаемое значение

Успех: Возвращает дескриптор окна.
Ошибка: Возвращает 0, если окно не найдено.

Примечания

WinMove() не действует на свёрнутые окна, но работает со скрытыми.
Если ширина и высота малы (или отрицательны), окно не будет меньше 112 × 27 пикселей. При больших значениях окно не превысит приблизительно [12+@DesktopWidth] × [12+@DesktopHeight] пикселей.
Для координат x и y допускаются отрицательные значения: окно можно вынести за пределы экрана; если программа запоминает последнее положение окна, при следующем запуске оно появится в углу экрана (полностью видимым).
Если несколько окон соответствуют критериям, используется наиболее недавно активное.

Если x и y равны ключевому слову Default, перемещения не происходит – выполняется только изменение размера.

При использовании параметра speed перемещение выполняется уже с новым размером.

См. также

WinActivate, WinClose, WinGetClientSize, WinGetPos, WinSetState

Пример

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Получаем позицию, высоту и ширину окна Блокнота, чтобы вернуть его на место после перемещения.
    Local $aPos = WinGetPos($hWnd)

    ; Перемещаем Блокнот в позицию x=0, y=0 и задаём ширину и высоту 200×200.
    WinMove($hWnd, "", 0, 0, 200, 200)

    ; Ждём 2 секунды, чтобы отобразить новую позицию окна Блокнота.
    Sleep(2000)

    ; Возвращаем окно Блокнота в исходную позицию, используя массив, возвращённый WinGetPos.
    WinMove($hWnd, "", $aPos[0], $aPos[1], $aPos[2], $aPos[3])

    ; Ждём 2 секунды, чтобы отобразить исходную позицию окна Блокнота.
    Sleep(2000)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
