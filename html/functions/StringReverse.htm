Справочник функций

StringReverse

Переворачивает содержимое указанной строки.

StringReverse ( "string" [, flag = 0] )

Параметры

string Переворачиваемая строка.
flag [необязательный] Изменяет способ переворота строки.
$STR_UTF16 (0) = переворот в полном режиме UTF-16 (по умолчанию).
$STR_UCS2 (1) = значительно более быстрый метод – использовать только для текста UCS-2.

Константы определены в "StringConstants.au3".

Возвращаемое значение

Возвращает перевёрнутую строку.

См. также

StringCompare, StringInStr, StringLen, StringLower, StringMid, StringReplace, StringRight, StringSplit, StringTrimLeft, StringTrimRight, StringUpper

Пример

#include <MsgBoxConstants.au3>

; Перевернуть строку esreveR.
MsgBox($MB_SYSTEMMODAL, "", StringReverse("esreveR"))
