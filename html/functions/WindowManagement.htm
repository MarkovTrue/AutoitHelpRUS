Window functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
WinActivate Активирует окно (передаёт ему фокус).
WinActive Проверяет, существует ли указанное окно и является ли оно активным в данный момент.
WinClose Закрывает окно.
WinExists Проверяет, существует ли указанное окно.
WinFlash Мигает окном на панели задач.
WinGetCaretPos Возвращает координаты курсора (каретки) в окне переднего плана.
WinGetClassList Получает список классов окна.
WinGetClientSize Получает размер клиентской области указанного окна.
WinGetHandle Получает внутренний дескриптор окна.
WinGetPos Получает позицию и размер указанного окна.
WinGetProcess Получает идентификатор процесса (PID), связанного с окном.
WinGetState Получает состояние указанного окна.
WinGetText Получает текст из окна.
WinGetTitle Получает полный заголовок окна.
WinKill Принудительно закрывает окно.
WinList Получает список окон.
WinMenuSelectItem Вызывает пункт меню окна.
WinMinimizeAll Сворачивает все окна.
WinMinimizeAllUndo Отменяет предыдущий вызов WinMinimizeAll.
WinMove Перемещает и/или изменяет размер окна.
WinSetOnTop Изменяет атрибут окна "Поверх всех".
WinSetState Показывает, скрывает, сворачивает, разворачивает или восстанавливает окно.
WinSetTitle Изменяет заголовок окна.
WinSetTrans Устанавливает прозрачность окна.
WinWait Приостанавливает выполнение скрипта до появления указанного окна.
WinWaitActive Приостанавливает выполнение скрипта до активации указанного окна.
WinWaitClose Приостанавливает выполнение скрипта до исчезновения указанного окна.
WinWaitNotActive Приостанавливает выполнение скрипта до деактивации указанного окна.
