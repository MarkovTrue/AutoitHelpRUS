Справочник функций

GUICtrlCreateIcon

Создаёт элемент управления Icon (значок) для GUI.

GUICtrlCreateIcon ( filename, iconName, left, top [, width [, height [, style = -1 [, exStyle = -1]]]] )

Параметры

filename Имя файла загружаемого значка.
iconName Имя значка, если файл содержит несколько значков. Может быть порядковым именем при отрицательном значении. В противном случае -1.
left Левая граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
top Верхняя граница элемента управления. Если указано -1, значение будет вычислено согласно GUICoordMode.
width [необязательный] Ширина элемента управления (по умолчанию 32).
height [необязательный] Высота элемента управления (по умолчанию 32).
style [необязательный] Задаёт стиль элемента управления. См. приложение "Стили элементов управления GUI".
по умолчанию (-1) : $SS_NOTIFY
принудительные стили : $WS_TABSTOP, $SS_ICON
exStyle [необязательный] Задаёт расширенный стиль элемента управления. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает идентификатор нового элемента управления (controlID).
Ошибка: Возвращает 0.

Примечания

Для установки или изменения параметров элемента управления см. функции GUICtrlUpdate...().
Если filename – пустая строка, элемент управления создаётся, но не реагирует на наведение курсора.

Для обновления значка после отображения диалогового окна используйте GUICtrlSetImage().

iconName может ссылаться на номер группы значка. Для нахождения нужного значения воспользуйтесь Resource Hacker.

Для объединения стилей со стилем по умолчанию используйте BitOR($GUI_SS_DEFAULT_ICON, новый_стиль, ...).
Для использования указанных выше значений необходимо включить в скрипт #include <StaticConstants.au3>.

Изменение размера по умолчанию: $GUI_DOCKSIZE.

Положительное число ссылается на значок по строковому имени.
Отрицательное число задаёт индексную адресацию (начиная с 1). Некоторые DLL позволяют извлекать значки только с отрицательными числами.

См. также

GUICoordMode (Option), GUICtrlSetImage, GUICtrlUpdate..., GUIGetMsg

Пример

Пример 1

#include <GUIConstantsEx.au3>

Example()

Func Example()
    GUICreate(" My GUI Icons", 250, 250)

    GUICtrlCreateIcon("shell32.dll", 10, 20, 20)
    GUICtrlCreateIcon(@ScriptDir & '\Extras\horse.ani', -1, 20, 40, 32, 32)
    GUICtrlCreateIcon("shell32.dll", 7, 20, 75, 32, 32)
    GUISetState(@SW_SHOW)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd

    GUIDelete()
EndFunc   ;==>Example

Пример 2

Example()

Func Example()
    Local $iOldOpt = Opt("GUICoordMode", 1)

    GUICreate("My GUI icon Race", 350, 74, -1, -1)
    GUICtrlCreateLabel("", 331, 0, 1, 74, 5)
    Local $idIcon_1 = GUICtrlCreateIcon(@ScriptDir & '\Extras\dinosaur.ani', -1, 0, 0, 32, 32)
    Local $idIcon_2 = GUICtrlCreateIcon(@ScriptDir & '\Extras\horse.ani', -1, 0, 40, 32, 32)

    GUISetState(@SW_SHOW)

    Local $a = 0
    Local $b = 0
    While ($a < 300) And ($b < 300)
        $a = $a + Int(Random(0, 1) + 0.5)
        $b = $b + Int(Random(0, 1) + 0.5)
        GUICtrlSetPos($idIcon_1, $a, 0)
        GUICtrlSetPos($idIcon_2, $b, 40)
        Sleep(10)
    WEnd
    Sleep(3000)
    Opt("GUICoordMode", $iOldOpt)
EndFunc   ;==>Example
