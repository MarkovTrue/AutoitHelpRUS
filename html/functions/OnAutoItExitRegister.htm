Справочник функций

OnAutoItExitRegister

Регистрирует функцию, вызываемую при завершении AutoIt.

OnAutoItExitRegister ( "function" )

Параметры

function Имя вызываемой пользовательской функции.

Возвращаемое значение

Успех: Возвращает 1. @extended может быть установлен, если функция уже зарегистрирована.
Ошибка: Возвращает 0.

Примечания

Код завершения можно получить через @exitCode.
Способ завершения – через @exitMethod.

$EXITCLOSE_NORMAL (0) Нормальное закрытие.
$EXITCLOSE_BYEXIT (1) Закрытие функцией Exit.
$EXITCLOSE_BYCLICK (2) Закрытие по нажатию "Выход" в системном трее.
$EXITCLOSE_BYLOGOFF (3) Закрытие при выходе пользователя из системы.
$EXITCLOSE_BYSHUTDOWN (4) Закрытие при завершении работы Windows.

@exitMethod можно отформатировать с помощью _FormatAutoItExitMethod().

Если используется Opt("SetExitCode", 1), @exitCode устанавливается согласно сообщению о фатальной ошибке – см. коды завершения AutoIt3.

Если подключён "AutoItExitCodes.au3", можно вызвать _FormatAutoItExitCode() для получения строкового представления @ExitCode.

Константы определены в "AutoItConstants.au3".

См. также

Exit, OnAutoItExitUnRegister, OnAutoItStartRegister, SetExitCode (Option), _FormatAutoItExitCode, _FormatAutoItExitMethod

Пример

#include <MsgBoxConstants.au3>

OnAutoItExitRegister("MyTestFunc")
OnAutoItExitRegister("MyTestFunc2")

Sleep(1000)

Func MyTestFunc()
    MsgBox($MB_SYSTEMMODAL, "Exit Results 1", 'Exit Message from MyTestFunc()')
EndFunc   ;==>MyTestFunc

Func MyTestFunc2()
    MsgBox($MB_SYSTEMMODAL, "Exit Results 2", 'Exit Message from MyTestFunc()')
EndFunc   ;==>MyTestFunc2
