Variables and Conversions functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
Asc Возвращает ASCII-код символа.
AscW Возвращает Unicode-код символа.
Assign Присваивает переменной значение по её имени.
Binary Возвращает двоичное представление выражения.
BinaryLen Возвращает количество байтов в двоичных данных.
BinaryMid Извлекает заданное количество байтов из двоичных данных.
BinaryToString Преобразует двоичные данные в строку.
Chr Возвращает символ, соответствующий ASCII-коду.
ChrW Возвращает символ, соответствующий unicode-коду.
Dec Возвращает числовое представление шестнадцатеричной строки.
Eval Возвращает значение переменной, имя которой задано строкой.
FuncName Возвращает имя функции, хранящейся в переменной.
Hex Возвращает строковое представление целого числа или двоичного значения в шестнадцатеричном формате.
HWnd Преобразует выражение в дескриптор HWND.
Int Возвращает целочисленное представление выражения.
IsAdmin Проверяет, обладает ли текущий пользователь полными правами администратора.
IsArray Проверяет, является ли переменная массивом.
IsBinary Проверяет, является ли переменная или выражение двоичным типом.
IsBool Проверяет, является ли базовый тип переменной булевым.
IsDeclared Проверяет, была ли объявлена переменная.
IsDllStruct Проверяет, является ли переменная типом DllStruct.
IsFloat Проверяет, имеет ли значение переменной или выражения дробную часть.
IsFunc Проверяет, является ли переменная или выражение функцией.
IsHWnd Проверяет, является ли базовый тип переменной указателем и дескриптором окна.
IsInt Проверяет, не имеет ли значение переменной или выражения дробной части.
IsKeyword Проверяет, является ли переменная ключевым словом (например, Default).
IsMap Проверяет, является ли переменная типом Map.
IsNumber Проверяет, является ли базовый тип переменной числовым.
IsObj Проверяет, является ли переменная или выражение объектом.
IsPtr Проверяет, является ли базовый тип переменной указателем.
IsString Проверяет, является ли переменная строкой.
Number Возвращает числовое представление выражения.
Ptr Преобразует выражение в вариант типа Pointer.
String Возвращает строковое представление выражения.
StringToBinary Преобразует строку в двоичные данные.
UBound Возвращает размер измерений массива или количество ключей в map.
