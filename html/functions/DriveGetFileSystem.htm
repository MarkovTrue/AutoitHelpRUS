Справочник функций

DriveGetFileSystem

Возвращает тип файловой системы диска.

DriveGetFileSystem ( "path" )

Параметры

path Путь к диску, с которого получается информация.

Возвращаемое значение

Успех: Возвращает тип файловой системы диска в виде строки; см. таблицу ниже.
Ошибка: Устанавливает значение @error = 1.


Возвращаемое значение Описание
$DT_UNDEFINED (1) (numeric) Диск не содержит носителя (CD, Floppy, Zip) или носитель не отформатирован (RAW).
$DT_FAT ("FAT") Типичная файловая система для дисков меньше ~500 МБ: дискеты, RAM-диски, USB-флешки и т.п.
$DT_FAT32 ("FAT32") Типичная файловая система для жёстких дисков Windows 9x/Me.
$DT_EXFAT ("exFAT") Типичная файловая система для дисков с файлами больше 4 ГБ: RAM-диски, USB-флешки и т.п.
$DT_NTFS ("NTFS") Типичная файловая система для жёстких дисков Windows.
$DT_UNWFS ("NWFS") Типичная файловая система для файловых серверов Novell Netware.
$DT_CDFS ("CDFS") Обычно указывает на CD (или ISO-образ, смонтированный как виртуальный CD-привод).
$DT_UDF ("UDF") Typically indicates a DVD.

Константы определены в "AutoItConstants.au3".

Примечания

Список возможных возвращаемых значений может быть неполным.

См. также

DriveGetDrive, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

Пример

#include <MsgBoxConstants.au3>

Local $sFileSystem = DriveGetFileSystem(@HomeDrive & "\") ; Определить тип файловой системы домашнего диска, как правило, это диск C:\.
MsgBox($MB_SYSTEMMODAL, "", "File System Type: " & $sFileSystem)
