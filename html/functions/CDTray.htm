CDTray

Открывает или закрывает лоток CD.

CDTray ( "drive", "status" )

Параметры

drive Буква диска CD, которым требуется управлять, в формате D:, E: и т. д.
status Указывает, должен ли лоток CD быть открыт или закрыт:
$CDTRAY_OPEN ("open") – открыть
$CDTRAY_CLOSED ("closed") – закрыть (не подходит для ноутбучных лотков)

Константы определены в AutoItConstants.au3.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0, если диск заблокирован программой записи CD или буква не относится к CD-приводу.

Примечания

CDTray() корректно работает с виртуальными CD-приводами (например, DAEMON Tools).
CDTray() не работает на нелокальных и сетевых CD-приводах; CDTray() должен запускаться на компьютере, привод которого требуется открыть/закрыть.
CDTray("X:", "close") возвращает 1 для ноутбучных лотков, которые можно закрыть только вручную.

См. также

DriveGetType, DriveStatus

Пример

#include <AutoItConstants.au3>
#include <MsgBoxConstants.au3>

; Получить список всех CD-приводов
Local $aCDRom = DriveGetDrive($DT_CDROM)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "", "No CDRom found.")
    Exit
EndIf

; Открыть лоток CD первого привода
CDTray($aCDRom[1], $CDTRAY_OPEN)
Sleep(2000)
CDTray($aCDRom[1], $CDTRAY_CLOSED) ; не работает для ноутбучных лотков, которые можно закрыть только вручную.
