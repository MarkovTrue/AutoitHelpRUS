Obj/COM functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
ObjCreate Создаёт ссылку на COM-объект по указанному имени класса.
ObjCreateInterface Создаёт ссылку на объект по имени класса/указателю объекта, идентификатору интерфейса и строке описания.
ObjEvent Обрабатывает входящие события от указанного объекта.
ObjGet Возвращает ссылку на COM-объект из существующего процесса или файла.
ObjName Возвращает имя или описание интерфейса объекта.
