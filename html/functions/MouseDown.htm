MouseDown

Выполняет событие нажатия кнопки мыши в текущей позиции курсора.

MouseDown ( "button" )

Параметры

button Кнопка для нажатия:
$MOUSE_CLICK_RIGHT ("right")
$MOUSE_CLICK_MIDDLE ("middle")
$MOUSE_CLICK_MAIN ("main")
$MOUSE_CLICK_MENU ("menu")
$MOUSE_CLICK_PRIMARY ("primary")
$MOUSE_CLICK_SECONDARY ("secondary")

Константы определены в "AutoItConstants.au3".

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 – кнопка отсутствует в списке.

Примечания

См. раздел Примечания к MouseClick.
Ответственность разработчика: каждому вызову MouseDown() должен соответствовать вызов MouseUp().

См. также

MouseClick, MouseClickDownDelay (Option), MouseClickDrag, MouseCoordMode (Option), MouseGetPos, MouseMove, MouseUp

Пример

#include <AutoItConstants.au3>

MouseDown($MOUSE_CLICK_LEFT) ; Установить состояние левой кнопки мыши – нажата.
Sleep(100)
MouseUp($MOUSE_CLICK_LEFT) ; Установить состояние левой кнопки мыши – отпущена.
