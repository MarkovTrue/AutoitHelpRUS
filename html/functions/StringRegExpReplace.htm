Справочник функций

StringRegExpReplace

Заменяет текст в строке на основе регулярных выражений.

StringRegExpReplace ( "test", "pattern", "replace" [, count = 0] )

Параметры

test Проверяемая строка.
pattern Регулярное выражение для сравнения. Описание символов шаблона см. в StringRegExp.
replace Текст, которым заменяется совпадение. Для вставки текста захваченных групп можно использовать \0 – \9 (или $0 – $9) в качестве обратных ссылок. (См. раздел Примечания).
count [необязательный] Количество замен в строке. По умолчанию равно 0. Значение 0 означает глобальную замену.

Возвращаемое значение

Успех: Возвращает изменённую строку. @extended содержит количество выполненных замен.
Ошибка: Устанавливает @error ≠ 0.
@error: 2 – Недопустимый шаблон. @extended = смещение ошибки в шаблоне.

Примечания

Чтобы отделить обратную ссылку от следующих за ней цифр, заключите её номер в фигурные скобки, например: "${1}5".
Если в строке замены необходим символ "\", его нужно удвоить. Это следствие механизма обратных ссылок.
Форматы замены "\" и "$" – единственные поддерживаемые форматы обратных ссылок.

См. также руководство по регулярным выражениям, в котором можно запустить скрипт для тестирования шаблонов.

См. также

StringRegExp, StringReplace

Пример

#include <MsgBoxConstants.au3>

Test1()
Test2()
Test3()

; Этот пример демонстрирует базовую замену: гласные буквы aeiou
; заменяются символом @.
Func Test1()
    Local $sInput = "Where have all the flowers gone, long time passing?"
    Local $sOutput = StringRegExpReplace($sInput, "[aeiou]", "@")
    Display($sInput, $sOutput)
EndFunc   ;==>Test1

; Следующий пример демонстрирует использование обратных ссылок для изменения формата даты
; из MM/DD/YYYY в DD.MM.YYYY.
Func Test2()
    Local $sInput = 'some text1 12/31/2009 01:02:03 some text2' & @CRLF & _
            'some text3 02/28/2009 11:22:33 some text4'
    Local $sOutput = StringRegExpReplace($sInput, '(\d{2})/(\d{2})/(\d{4})', ' $2.$1.$3 ')
    Display($sInput, $sOutput)
EndFunc   ;==>Test2

; Следующий пример демонстрирует необходимость удвоения обратной косой черты.
Func Test3()
    Local $sInput = '%CommonProgramFiles%\Microsoft Shared\'
    Local $sOutput = StringRegExpReplace($sInput, '%([^%]*?)%', 'C:\\WINDOWS\\Some Other Folder$')
    Display($sInput, $sOutput)
EndFunc   ;==>Test3

Func Display($sInput, $sOutput)
    ; Форматирование вывода.
    Local $sMsg = StringFormat("Input:\t%s\n\nOutput:\t%s", $sInput, $sOutput)
    MsgBox($MB_SYSTEMMODAL, "Results", $sMsg)
EndFunc   ;==>Display
