Справочник функций

Exp

Вычисляет число e в указанной степени.

Exp ( expression )

Параметры

expression Любое допустимое числовое выражение.

Возвращаемое значение

Возвращает e, возведённое в степень expression.

Примечания

e – основание натурального логарифма, приближённо равное 2.71828182845905

См. также

Log

Пример

#include <MsgBoxConstants.au3>

; Присваиваем локальной переменной число Эйлера в степени 5.
Local $fExp1 = Exp(5)

; Показать результат.
MsgBox($MB_SYSTEMMODAL, "", $fExp1)
