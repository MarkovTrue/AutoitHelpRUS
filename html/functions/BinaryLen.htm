Справочник функций

BinaryLen

Возвращает количество байтов в двоичных данных.

BinaryLen ( binary )

Параметры

binary Двоичные данные для измерения.

Возвращаемое значение

Возвращает длину двоичных данных в байтах.

См. также

Binary, BinaryMid

Пример

#include <MsgBoxConstants.au3>

Local $dBinary = Binary("0x10203040") ; Создать двоичные данные из строки.
MsgBox($MB_SYSTEMMODAL, "", "The length of the binary is: " & BinaryLen($dBinary))
