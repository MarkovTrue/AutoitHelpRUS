Справочник функций

GUICtrlSetBkColor

Устанавливает цвет фона элемента управления.

GUICtrlSetBkColor ( controlID, backgroundcolor )

Параметры

controlID Идентификатор элемента управления (controlID), возвращённый функцией GUICtrlCreate...(), или -1 для последнего созданного элемента.
backgroundcolor Используемый цвет в формате RGB.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

В настоящее время поддерживается окраска следующих элементов управления: Button, Label, Checkbox, Group, Radio, Edit, Input, List, Listview, ListviewItem, Treeview, TreeviewItem, Graphic, Progress, Slider и Combo.

Элементы управления Progress не могут быть окрашены при использовании стиля "Windows XP".
Элементы управления Checkbox, Radio, Group, Progress, Combo не могут быть окрашены при использовании стиля "Windows XP/Vista".

Элементы управления Button всегда отображаются в "классическом стиле Windows". Им нельзя задать стиль $BS_ICON.

Специальный флаг $GUI_BKCOLOR_TRANSPARENT применим к Label, Group, Radio, Checkbox для задания прозрачного фона.

Специальный флаг $GUI_BKCOLOR_LV_ALTERNATE позволяет задать чередующийся фон строк ListView.
Нечётные строки получат цвет, заданный через GUICtrlSetBkColor() для элемента управления ListView.
Чётные строки получат цвет, заданный через GUICtrlSetBkColor() для элемента управления ListviewItem.

См. также

GUICtrlCreate..., GUICtrlSetColor, GUICtrlSetDefBkColor

Пример

Пример 1

#include <ColorConstants.au3>
#include <GUIConstantsEx.au3>
#include <WindowsSysColorConstants.au3>

Example()

Func Example()
    ; Создать GUI с различными элементами управления.
    Local $hGUI = GUICreate("Example", 300, 200)

    ; Создать элемент label.
    Local $idLabel = GUICtrlCreateLabel("A string of text", 10, 10, 185, 17)
    Local $idButton_Close = GUICtrlCreateButton("Close", 210, 170, 85, 25)

    ; Установить цвет фона элемента управления Label.
    GUICtrlSetBkColor($idLabel, $COLOR_RED)

    ; Отобразить GUI.
    GUISetState(@SW_SHOW, $hGUI)

    ; Цикл до выхода пользователя.
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE, $idButton_Close
                ExitLoop

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI и все элементы управления.
    GUIDelete($hGUI)
EndFunc   ;==>Example

Пример 2

#include <ColorConstants.au3>
#include <GUIConstantsEx.au3>
#include <WindowsSysColorConstants.au3>

Example()

Func Example()
    ; Создать GUI с ListView.
    Local $hGUI = GUICreate("Colored ListView Items", 250, 170, 100, 200, -1)
    Local $idListview = GUICtrlCreateListView("col1|col2|col3", 10, 10, 230, 150)

    ; Чередование цвета фона ListView и цвета фона элемента ListView.
    GUICtrlSetBkColor(-1, $GUI_BKCOLOR_LV_ALTERNATE)

    ; Установить цвет фона для ListView.
    ; Нечётные элементы ListView будут показаны с цветом фона ListView,
    ; чётные – с цветом фона элемента ListView.
    GUICtrlSetBkColor(-1, $COLOR_AQUA)

    ; Создать элементы ListView и задать цвет фона для каждого.
    GUICtrlCreateListViewItem("item1|col12|col13", $idListview)

    ; Следующую строку можно опустить – цвет фона берётся из listview.
    GUICtrlSetBkColor(-1, $COLOR_GREEN)
    GUICtrlCreateListViewItem("item2|col22|col23", $idListview)
    GUICtrlSetBkColor(-1, $COLOR_GREEN)
    GUICtrlCreateListViewItem("item3|col32|col33", $idListview)

    ; Следующую строку можно опустить – цвет фона берётся из listview.
    GUICtrlSetBkColor(-1, $COLOR_GREEN)

    ; Изменить цвет одного элемента ListView.
    GUICtrlCreateListViewItem("Now|change|color", $idListview)
    GUICtrlSetBkColor(-1, $COLOR_LIME)
    GUICtrlCreateListViewItem("item5|col52|col53", $idListview)

    ; Следующую строку можно опустить – цвет фона берётся из listview.
    GUICtrlSetBkColor(-1, $COLOR_GREEN)
    GUICtrlCreateListViewItem("item6|col62|col63", $idListview)
    GUICtrlSetBkColor(-1, $COLOR_GREEN)
    GUISetState(@SW_SHOW, $hGUI)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    ; Удалить предыдущий GUI со всеми элементами управления
    GUIDelete($hGUI)
EndFunc   ;==>Example
