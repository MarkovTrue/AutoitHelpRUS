Справочник функций

ControlGetText

Возвращает текст из элемента управления.

ControlGetText ( "title", "text", controlID )

Параметры

title Заголовок / дескриптор / класс окна, с которым нужно работать. См. Заголовки и текст окон (расширенно).
text Текст окна, с которым нужно работать. См. Заголовки и текст окон (основы).
controlID Элемент управления, с которым нужно работать. См. Элементы управления.

Возвращаемое значение

Успех: Возвращает текст из элемента управления.
Ошибка: Возвращает "" (пустую строку) и устанавливает значение @error = 1.

См. также

ControlCommand, ControlSetText, StatusbarGetText, WinGetText

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Запустить Блокнот
    Run("notepad.exe")

    ; Ждать до 10 секунд появления окна Блокнота.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Установить текст edit-элемента Notepad. Дескриптор от WinWait используется как параметр "title" функции ControlSetText.
    ControlSetText($hWnd, "", ControlGetFocus($hWnd), "This is some text")

    ; Получить текст edit-элемента Notepad. Дескриптор от WinWait используется как параметр "title" функции ControlGetText.
    Local $sText = ControlGetText($hWnd, "", ControlGetFocus($hWnd))

    ; Показать текст edit-элемента.
    MsgBox($MB_SYSTEMMODAL, "", "The text in " & ControlGetFocus($hWnd) & " is: " & $sText)

    ; Закрыть окно Блокнота по дескриптору, возвращённому WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
