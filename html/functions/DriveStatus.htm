Справочник функций

DriveStatus

Возвращает статус диска в виде строки.

DriveStatus ( "path" )

Параметры

path Путь к диску, с которого получается информация.

Возвращаемое значение

Значение Описание
$DS_UNKNOWN ("UNKNOWN") Диск, возможно, не отформатирован (RAW).
$DS_READY ("READY") Типично для жёстких дисков и дисков со сменными носителями.
$DS_NOTREADY ("NOTREADY") Типично для дисководов и оптических приводов без носителя.
$DS_INVALID ("INVALID") Может означать, что буква диска отсутствует или подключённый сетевой диск недоступен.

Константы определены в "AutoItConstants.au3".

Примечания

Список возможных возвращаемых значений может быть неполным.
DriveStatus() может работать и при передаче полного пути к существующему каталогу. Однако путь к файлу не сработает.

См. также

CDTray, DriveGetDrive, DriveGetFileSystem, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, FileExists

Пример

#include <MsgBoxConstants.au3>

Local $sStatus = DriveStatus(@HomeDrive & "\") ; Получение статуса диска, где находится пользовательский профиль (обычно C:\).
MsgBox($MB_SYSTEMMODAL, "", "Status: " & $sStatus)
