Справочник функций

Hex

Возвращает строковое представление целого числа или двоичного значения в шестнадцатеричном формате.

Hex ( expression [, length] )

Параметры

expression Выражение для преобразования.
length [необязательный] Количество возвращаемых символов для целого числа.
Если length слишком мало, символы усекаются слева.
Для двоичных данных параметр игнорируется.

Возвращаемое значение

Успех: Возвращает строку из length символов (не более 16), дополненную нулями слева при необходимости (для целых чисел). Для двоичных данных возвращает преобразованное значение.
Special: Возвращает "" (пустую строку), если length меньше 1.

Примечания

Если второй параметр (length) не указан или передано ключевое слово Default, размер вывода определяется автоматически: 8 символов для чисел, умещающихся в 8 знаков, и 16 символов для остальных.
Для указателя по умолчанию используется размер указателя (8 символов для x86, 16 для x64).
Для 64-разрядных целых чисел по умолчанию выводится 16 символов.
Числа, переданные не в виде целых (с разделителем дробной части или экспонентой), обрабатываются как double.

Пример

#include <MsgBoxConstants.au3>

; Присвоить локальной переменной шестнадцатеричное представление числа 1033.
Local $sHex1 = Hex(1033, 4)

; Показать результат.
MsgBox($MB_SYSTEMMODAL, "", $sHex1)
