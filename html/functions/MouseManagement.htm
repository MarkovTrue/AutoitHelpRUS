Mouse functions Reference

Below is a complete list of the functions available in AutoIt. Click on a function name for a detailed description.

 

Функция Описание
MouseClick Выполняет операцию нажатия кнопки мыши.
MouseClickDrag Выполняет операцию нажатия кнопки мыши и перетаскивания.
MouseDown Выполняет событие нажатия кнопки мыши в текущей позиции курсора.
MouseGetCursor Возвращает числовой идентификатор текущего курсора мыши.
MouseGetPos Возвращает текущую позицию курсора мыши.
MouseMove Перемещает курсор мыши.
MouseUp Выполняет событие отпускания кнопки мыши в текущей позиции курсора.
MouseWheel Прокручивает колёсико мыши вверх или вниз.
