Справочник ключевых слов

Ниже приведён полный список ключевых слов, доступных в AutoIt. Щёлкните по имени, чтобы открыть подробное описание.

 

Ключевое слово Описание
False / True Логические значения для использования в логических выражениях.
#comments-start Указывает, что целый раздел скрипта должен быть закомментирован.
ContinueCase Прерывает выполнение текущего блока Case и переходит к следующему блоку Case в структуре Select или Switch.
ContinueLoop Продолжает цикл While/Do/For.
Default Значение ключевого слова для использования при вызове функции.
Dim / Global / Local / Const Объявляет переменную, константу или создаёт массив.
Do...Until Цикл с условием.
Enum Перечисляет константы.
Exit Завершает скрипт.
ExitLoop Завершает цикл While/Do/For.
For...To...Step...Next Цикл с условием.
For...In...Next Перечисляет элементы коллекции Object или массива.
Func...Return...EndFunc Определяет пользовательскую функцию, принимающую ноль или более аргументов и опционально возвращающую результат.
If...Then Условное выполнение одного оператора.
If...ElseIf...Else...EndIf Условное выполнение операторов.
#include-once Указывает, что текущий файл должен быть подключён только один раз.
#include Подключает файл к текущему скрипту.
#NoTrayIcon Указывает, что значок AutoIt в трее не отображается при запуске скрипта.
Null Значение ключевого слова для использования при вызове функции.
#OnAutoItStartRegister Регистрирует функцию, вызываемую при запуске AutoIt.
#pragma Специальная директива для управления аспектами компиляции скрипта.
ReDim Изменяет размер существующего массива.
#RequireAdmin Указывает, что текущий скрипт требует полных прав администратора для запуска.
Select...Case...EndSelect Условное выполнение операторов.
Static Объявляет статическую переменную или создаёт статический массив.
Switch...Case...EndSwitch Условное выполнение операторов.
Volatile Квалификатор функции.
While...WEnd Цикл с условием.
With...EndWith Используется для сокращения длинных обращений к переменным с точечным доступом.
