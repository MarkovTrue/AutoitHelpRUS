Справочник функций

_Date_Time_EncodeSystemTime

Кодирует и возвращает структуру $tagSYSTEMTIME.

#include <Date.au3>
_Date_Time_EncodeSystemTime ( $iMonth, $iDay, $iYear [, $iHour = 0 [, $iMinute = 0 [, $iSecond = 0 [, $iMSeconds = 0]]]] )

Параметры

$iMonth Месяц.
$iDay День.
$iYear Год.
$iHour [необязательный] Час.
$iMinute [необязательный] Минуты.
$iSecond [необязательный] Секунды.
$iMSeconds [необязательный] Миллисекунды.

Возвращаемое значение

Возвращает структуру $tagSYSTEMTIME.

См. также

$tagSYSTEMTIME, _Date_Time_EncodeFileTime

Пример

#include "Extras\HelpFileInternals.au3"

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $tSystem

    ; Создать GUI
    GUICreate("Time", 400, 300)
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Кодируем системное время.
    $tSystem = _Date_Time_EncodeSystemTime(@MON, @MDAY, @YEAR, @HOUR, @MIN, @SEC)
    _MemoWrite("Current date/time .: " & _Date_Time_SystemTimeToDateTimeStr($tSystem))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
