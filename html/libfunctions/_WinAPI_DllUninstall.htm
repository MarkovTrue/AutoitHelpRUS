Справочник функций

_WinAPI_DllUninstall

Отменяет регистрацию OLE-элементов управления: файлов DLL или ActiveX Controls (OCX).

#include <WinAPIReg.au3>
_WinAPI_DllUninstall ( $sFilePath )

Параметры

$sFilePath Путь к DLL-файлу, для которого нужно отменить регистрацию.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
