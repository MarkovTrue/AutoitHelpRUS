_WinAPI_GetRawInputBuffer

Выполняет буферизованное чтение данных сырого ввода.

#include <WinAPISys.au3>
_WinAPI_GetRawInputBuffer ( $pBuffer, $iLength )

Параметры

$pBuffer Указатель на буфер для получения массива структур $tagRAWINPUT, содержащих данные сырого ввода.
$iLength Размер буфера в байтах.

Возвращаемое значение

Успех: Возвращает количество структур $tagRAWINPUT, записанных в буфер.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Чтобы получить минимальный требуемый размер буфера для вызова этой функции, используйте _WinAPI_GetRawInputBufferLength().

Для обеспечения правильной работы функции _WinAPI_GetRawInputBuffer() на WOW64
вы должны выровнять структуру $tagRAWINPUT по 8 байт.
