Справочник функций

_Security__SidTypeStr

Преобразует тип SID в строковую форму.

#include <Security.au3>
_Security__SidTypeStr ( $iType )

Параметры

$iType Тип SID.

Возвращаемое значение

Возвращает тип SID в строковой форме.

Пример

#include <Security.au3>

ConsoleWrite("SID type for $SIDTYPEDOMAIN is: " & _Security__SidTypeStr($SIDTYPEDOMAIN) & @CRLF)
