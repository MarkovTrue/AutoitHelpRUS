_WinAPI_GetSysColor

Получает текущий цвет указанного элемента отображения.

#include <WinAPISysWin.au3>
_WinAPI_GetSysColor ( $iIndex )

Параметры

$iIndex Элемент отображения, цвет которого должен быть получен. Может быть одним из следующих:
$COLOR_3DDKSHADOW – тёмная тень для трёхмерных элементов отображения.
$COLOR_3DFACE – цвет поверхности для трёхмерных элементов отображения и фонов диалоговых окон.
$COLOR_3DHIGHLIGHT – цвет подсветки для трёхмерных элементов отображения (для краёв, обращённых к источнику света).
$COLOR_3DHILIGHT – цвет подсветки для трёхмерных элементов отображения (для краёв, обращённых к источнику света).
$COLOR_3DLIGHT – светлый цвет для трёхмерных элементов отображения (для краёв, обращённых к источнику света).
$COLOR_3DSHADOW – цвет тени для трёхмерных элементов отображения (для краёв, обращённых в сторону от источника света).
$COLOR_ACTIVEBORDER – рамка активного окна.
$COLOR_ACTIVECAPTION – заголовок активного окна.
Задаёт цвет левой части градиента цвета в заголовке активного окна, если эффект градиента включен.
$COLOR_APPWORKSPACE – цвет фона приложений интерфейса множественных документов (MDI).
$COLOR_BACKGROUND – рабочий стол.
$COLOR_BTNFACE – цвет поверхности для трёхмерных элементов отображения и фонов диалоговых окон.
$COLOR_BTNHIGHLIGHT – цвет подсветки для трёхмерных элементов отображения (для краёв, обращённых к источнику света).
$COLOR_BTNHILIGHT – цвет подсветки для трёхмерных элементов отображения (для краёв, обращённых к источнику света).
$COLOR_BTNSHADOW – цвет тени для трёхмерных элементов отображения (для краёв, обращённых в сторону от источника света).
$COLOR_BTNTEXT – текст на кнопках.
$COLOR_CAPTIONTEXT – текст в заголовке, поле размера и поле стрелок полосы прокрутки.
$COLOR_DESKTOP – рабочий стол.
$COLOR_GRADIENTACTIVECAPTION – цвет правой части градиента цвета в заголовке активного окна.
$COLOR_ACTIVECAPTION задаёт цвет левой части.
Используйте SPI_GETGRADIENTCAPTIONS с функцией SystemParametersInfo, чтобы определить, включен ли эффект градиента.
$COLOR_GRADIENTINACTIVECAPTION – цвет правой части градиента цвета в заголовке неактивного окна.
$COLOR_INACTIVECAPTION задаёт цвет левой части.
$COLOR_GRAYTEXT – серый (отключённый) текст. Этот цвет устанавливается на 0, если текущий драйвер отображения не поддерживает сплошной серый цвет.
$COLOR_HIGHLIGHT – элемент(ы) выбранные в элементе управления.
$COLOR_HIGHLIGHTTEXT – текст выбранного элемента(ов) в элементе управления.
$COLOR_HOTLIGHT – цвет для гиперссылки или элемента с отслеживанием.
$COLOR_INACTIVEBORDER – рамка неактивного окна.
$COLOR_INACTIVECAPTION – заголовок неактивного окна.
Задаёт цвет левой части градиента цвета в заголовке неактивного окна, если эффект градиента включен.
$COLOR_INACTIVECAPTIONTEXT – цвет текста в неактивном заголовке.
$COLOR_INFOBK – цвет фона для элементов управления подсказки.
$COLOR_INFOTEXT – цвет текста для элементов управления подсказки.
$COLOR_MENU – фон меню.
$COLOR_MENUHILIGHT – цвет, используемый для выделения пунктов меню, когда меню выглядит как плоское меню.
Выделенный пункт меню обводится $COLOR_HIGHLIGHT.
$COLOR_MENUBAR – цвет фона для панели меню, когда меню выглядит как плоские меню.
Однако $COLOR_MENU продолжает задавать цвет фона всплывающего меню.
$COLOR_MENUTEXT – текст в меню.
$COLOR_SCROLLBAR – серая область полосы прокрутки.
$COLOR_WINDOW – фон окна.
$COLOR_WINDOWFRAME – рамка окна.
$COLOR_WINDOWTEXT – текст в окнах.

Возвращаемое значение

Возвращает значение цвета RGB (красный, зелёный, синий) указанного элемента

Примечания

Вышеуказанные константы требуют #include <WindowsSysColorConstants.au3>

Если параметр $iIndex выходит за пределы диапазона, возвращаемое значение равно нулю. Поскольку ноль также является допустимым значением RGB, вы не можете использовать GetSysColor для определения, поддерживается ли системный цвет текущей платформой. Вместо этого используйте функцию GetSysColorBrush, которая возвращает NULL, если цвет не поддерживается.

_WinAPI_SetSysColors

Искать GetSysColor в библиотеке MSDN

Пример

#include <GUIConstantsEx.au3>
#include <WinAPISysWin.au3>
#include <WindowsSysColorConstants.au3>

Example()

Func Example()
    Local $aElements[2] = [$COLOR_ACTIVECAPTION, $COLOR_GRADIENTACTIVECAPTION]
    ; Красный и жёлтый
    Local $aColors[2] = [255, 65535], $aSaveColors[2]

    GUICreate("My GUI", 300, 200)

    $aSaveColors[0] = _WinAPI_GetSysColor($COLOR_ACTIVECAPTION)
    $aSaveColors[1] = _WinAPI_GetSysColor($COLOR_GRADIENTACTIVECAPTION)

    _WinAPI_SetSysColors($aElements, $aColors)

    GUISetState(@SW_SHOW)

    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

        EndSwitch
    WEnd

    GUIDelete()

    _WinAPI_SetSysColors($aElements, $aSaveColors)

    Exit
EndFunc   ;==>Example
