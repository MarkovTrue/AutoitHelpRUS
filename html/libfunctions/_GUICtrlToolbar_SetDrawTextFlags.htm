Справочник функций

_GUICtrlToolbar_SetDrawTextFlags

Задаёт флаги отрисовки текста для панели инструментов.

#include <GuiToolbar.au3>
_GUICtrlToolbar_SetDrawTextFlags ( $hWnd, $iMask, $iDTFlags )

Параметры

$hWnd Дескриптор элемента управления.
$iMask Маска.
$iDTFlags Флаги рисования текста.

Возвращаемое значение

Возвращает предыдущие флаги рисования текста.

Примечания

Параметр $iMask позволяет указать, какие флаги учитываются при отрисовке текста, даже если они отключены.
Например, чтобы запретить использование $DT_CENTER при отрисовке текста, добавьте $DT_CENTER в $iMask и не задавайте этот флаг в $iDTFlags.
Это помешает элементу передавать $DT_CENTER в API DrawText.
