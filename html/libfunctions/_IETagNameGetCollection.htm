Справочник функций

_IETagNameGetCollection

Возвращает объект-коллекцию всех элементов объекта с указанным TagName или один элемент по индексу.

#include <IE.au3>
_IETagNameGetCollection ( ByRef $oObject, $sTagName [, $iIndex = -1] )

Параметры

$oObject Объектная переменная экземпляра InternetExplorer.Application.
$sTagName Имя тега для возвращаемой коллекции (например, IMG, TR и т. п.).
$iIndex [необязательный] Что возвращать – коллекцию или элемент по индексу:
0 или положительное число – элемент с этим индексом
-1 – (по умолчанию) коллекцию

Возвращаемое значение

Успех: Возвращает объектную переменную с коллекцией указанных тегов; @extended – количество таких тегов.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадений не найдено
@extended: Содержит номер недопустимого параметра.

Примечания

DOM иерархичен, поэтому если переданный объект – объект документа, возвращаются указанные элементы документа.
Если переданный объект – объект внутри документа (например, объект TABLE), возвращаются только указанные элементы внутри этого объекта.

Пример

; Открытие браузера с примером формы, получение коллекции
; всех тегов INPUT и отобразить имя формы и тип каждого

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("form")
Local $oInputs = _IETagNameGetCollection($oIE, "input")
Local $sTxt = ""
For $oInput In $oInputs
    $sTxt &= $oInput.type & @CRLF
Next
MsgBox($MB_SYSTEMMODAL, "Form Input Type", "Form: " & $oInput.form.name & @CRLF & @CRLF & " Types :" & @CRLF & $sTxt)

_IEQuit($oIE)
