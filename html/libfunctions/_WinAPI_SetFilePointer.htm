Справочник функций

_WinAPI_SetFilePointer

Перемещает указатель файла указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_SetFilePointer ( $hFile, $iPos [, $iMethod = 0] )

Параметры

$hFile Дескриптор файла для обработки.
$iPos Количество байтов для перемещения указателя файла. Максимальное значение – 2^32.
Положительное значение перемещает указатель файла вперёд в файле, отрицательное значение перемещает указатель назад.
$iMethod [необязательный] Начальная точка для перемещения указателя файла.
Может быть одним из предопределённых значений:
$FILE_BEGIN (0) = (по умолчанию) Начальная точка – ноль (0) или начало файла
$FILE_CURRENT (1) = Начальная точка – текущее значение указателя файла.
$FILE_END (2) = Начальная точка – текущая позиция конца файла.

Возвращаемое значение

Успех: Возвращает новый указатель файла.
Ошибка: (-1) INVALID_SET_FILE_POINTER, вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке.

Примечания

Эта функция также может быть использована для запроса текущей позиции указателя файла путём указания метода перемещения FILE_CURRENT и расстояния нулевого.
Эта функция хранит указатель файла в значении LONG. Для работы с указателями файлов, которые больше одного значения LONG, необходимо использовать функцию SetFilePointerEx.
Указатель файла – это позиция в файле для чтения/записи из/в _WinAPI_ReadFile()/_WinAPI_WriteFile()

См. также

_WinAPI_CloseHandle, _WinAPI_CreateFile, _WinAPI_FlushFileBuffers, _WinAPI_GetFileSizeEx, _WinAPI_ReadFile, _WinAPI_SetEndOfFile, _WinAPI_WriteFile

Пример

#include <WinAPIFiles.au3>
#include <WinAPIHObj.au3>

Local $sTempFile, $hFile, $sText, $nBytes, $tBuffer, $iSize

; 1) создать файл и записать в него данные
$sTempFile = @TempDir & '\test.txt'
$sText = 'abcdefghijklmnopqrstuvwxyz'
$tBuffer = DllStructCreate("byte[" & StringLen($sText) & "]")
DllStructSetData($tBuffer, 1, $sText)
$hFile = _WinAPI_CreateFile($sTempFile, 1)
_WinAPI_WriteFile($hFile, $tBuffer, StringLen($sText), $nBytes)
$iSize = _WinAPI_GetFileSizeEx($hFile)
_WinAPI_CloseHandle($hFile)
ConsoleWrite('1):' & $iSize & ' ' & FileRead($sTempFile) & @CRLF)

; 2) прочитать 6 байт начиная с позиции 3
$tBuffer = DllStructCreate("byte[6]")
$hFile = _WinAPI_CreateFile($sTempFile, 2, 2)
$iSize = _WinAPI_GetFileSizeEx($hFile)
_WinAPI_SetFilePointer($hFile, 3)
_WinAPI_ReadFile($hFile, $tBuffer, 6, $nBytes)
_WinAPI_CloseHandle($hFile)
$sText = BinaryToString(DllStructGetData($tBuffer, 1))
ConsoleWrite('2):' & $iSize & ' ' & $sText & @CRLF)

; 3) записать ранее прочитанные 6 байт обратно с позиции 3, но в верхнем регистре
DllStructSetData($tBuffer, 1, StringUpper($sText))
$hFile = _WinAPI_CreateFile($sTempFile, 2, 4)
$iSize = _WinAPI_GetFileSizeEx($hFile)
_WinAPI_SetFilePointer($hFile, 3)
_WinAPI_WriteFile($hFile, $tBuffer, 6, $nBytes)
_WinAPI_CloseHandle($hFile)
$tBuffer = 0
ConsoleWrite('3):' & $iSize & ' ' & FileRead($sTempFile) & @CRLF)

; 4) обрезать размер файла до 12 байт
$hFile = _WinAPI_CreateFile($sTempFile, 2, 4)
_WinAPI_SetFilePointer($hFile, 12)
_WinAPI_SetEndOfFile($hFile)
$iSize = _WinAPI_GetFileSizeEx($hFile)
_WinAPI_CloseHandle($hFile)
ConsoleWrite('4):' & $iSize & ' ' & FileRead($sTempFile) & @CRLF)

FileDelete($sTempFile)
