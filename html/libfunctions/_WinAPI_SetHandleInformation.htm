_WinAPI_SetHandleInformation

Задаёт определённые свойства дескриптора объекта.

#include <WinAPIHObj.au3>
_WinAPI_SetHandleInformation ( $hObject, $iMask, $iFlags )

Параметры

$hObject Дескриптор объекта.
$iMask Битовые флаги, которые должны быть изменены.
$iFlags Свойства дескриптора объекта.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
