Справочник функций

_WinAPI_ShellSetKnownFolderPath

Перенаправляет известную папку на новое местоположение.

#include <WinAPIShellEx.au3>
_WinAPI_ShellSetKnownFolderPath ( $sGUID, $sFilePath [, $iFlags = 0 [, $hToken = 0]] )

Параметры

$sGUID Идентификатор GUID ($FOLDERID_*), который идентифицирует известную папку.
$sFilePath Новый путь папки.
$iFlags [необязательный] Этот параметр может быть 0 или следующим значением. $KF_FLAG_DONT_UNEXPAND
$hToken [необязательный] Маркер доступа, который представляет конкретного пользователя. Если этот параметр равен 0, функция запрашивает известную папку текущего пользователя. Присвоение параметру $hToken значения (-1) указывает пользователя по умолчанию. Обратите внимание, что доступ к папкам пользователя по умолчанию требует привилегий администратора.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для вызова этой функции на общедоступных известных папках вызывающая сторона должна иметь привилегии администратора. Для пользовательских известных папок вызывающей стороне требуются только привилегии пользователя.
Для вызова этой функции для папки типа $KF_CATEGORY_COMMON вызывающее приложение должно работать с повышенными привилегиями.
Эта функция не может быть вызвана для папок типов $KF_CATEGORY_FIXED и $KF_CATEGORY_VIRTUAL.
Эта функция требует Windows Vista или более поздней версии.
