_WinAPI_GetThreadErrorMode

Получает режим ошибок для вызывающего потока.

#include <WinAPIProc.au3>
_WinAPI_GetThreadErrorMode ( )

Возвращаемое значение

Возвращает режим ошибок процесса ($SEM_*).

Примечания

Эта функция требует Windows 7 или более позднюю версию.
