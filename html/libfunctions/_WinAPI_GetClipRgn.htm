_WinAPI_GetClipRgn

Получает дескриптор, идентифицирующий текущую определённую приложением область отсечения.

#include <WinAPIGdi.au3>
_WinAPI_GetClipRgn ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает дескриптор копии текущей области отсечения.
Ошибка: Возвращает 0.

Примечания

Определённая приложением область отсечения – это область отсечения, идентифицированная функцией _WinAPI_SelectClipRgn().
Это не область отсечения, созданная при вызове приложением _WinAPI_BeginPaint().

См. также

_WinAPI_BeginPaint, _WinAPI_SelectClipRgn
