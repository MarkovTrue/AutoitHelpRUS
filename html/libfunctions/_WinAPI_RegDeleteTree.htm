Справочник функций

_WinAPI_RegDeleteTree

Удаляет подключ и всех его потомков.

#include <WinAPIReg.au3>
_WinAPI_RegDeleteTree ( $hKey [, $sSubKey = ''] )

Параметры

$hKey Дескриптор открытого ключа реестра или один из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_PERFORMANCE_DATA
$HKEY_USERS
$sSubKey [необязательный] Имя удаляемого ключа.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.
