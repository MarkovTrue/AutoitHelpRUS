_WinAPI_ShellGetKnownFolderIDList

Получает путь известной папки как структуру ITEMIDLIST.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetKnownFolderIDList ( $sGUID [, $iFlags = 0 [, $hToken = 0]] )

Параметры

$sGUID GUID ($FOLDERID_*), который определяет стандартные папки, зарегистрированные в системе.
$iFlags [необязательный] Флаги, определяющие специальные параметры получения. Этот параметр может быть одним или несколькими из следующих значений.
$KF_FLAG_CREATE
$KF_FLAG_DONT_VERIFY
$KF_FLAG_DONT_UNEXPAND
$KF_FLAG_NO_ALIAS
$KF_FLAG_INIT
$KF_FLAG_DEFAULT_PATH
$KF_FLAG_NOT_PARENT_RELATIVE
$KF_FLAG_SIMPLE_IDLIST

Windows 7 или более позднюю версию
$KF_FLAG_ALIAS_ONLY
$hToken [необязательный] Маркер доступа, представляющий конкретного пользователя. Если этот параметр равен 0, функция запрашивает
известную папку для текущего пользователя. Назначение параметру $hToken значения (-1) указывает на
пользователя по умолчанию. Обратите внимание, доступ к папкам пользователя по умолчанию требует привилегий администратора.

Возвращаемое значение

Успех: Возвращает PIDL папки.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Чтобы освободить возвращённый PIDL, вызовите функцию _WinAPI_CoTaskMemFree().

Эта функция требует Windows Vista или более позднюю версию.

См. также
