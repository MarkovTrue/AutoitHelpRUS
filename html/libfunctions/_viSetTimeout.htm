Справочник функций

_viSetTimeout

Устанавливает тайм-аут VISA в МИЛЛИСЕКУНДАХ.

#include <Visa.au3>
_viSetTimeout ( $hSession, $iTimeoutMS )

Параметры

$hSession Дескриптор VISA (строка) или дескриптор сессии VISA (целое число).
Подробности – в разделе "Примечания" _viExecCommand().
$iTimeoutMS Тайм-аут В МИЛЛИСЕКУНДАХ для операций VISA (главным образом для запросов GPIB)
Если установить на 0, тайм-ауты ОТКЛЮЧЕНЫ.
Если установить на "INF", операции VISA НИКОГДА не превысят тайм-аут.
Будьте осторожны с этим, так как это может легко зависнуть вашей программе, если ваш прибор не ответит на один из ваших запросов.
В зависимости от типа шины (GPIB, TCP и т. д.) тайм-аут может быть установлен не на точное значение, которое вы запросили. Вместо этого будет использован ближайший действительный тайм-аут больше, чем запрошенный.

Возвращаемое значение

Успех: Возвращает 0.
Ошибка: -1 и устанавливает флаг @error в ненулевое значение, если DLL-библиотеку VISA не удаётся открыть
или ненулевое значение, представляющее код ошибки VISA (см. руководство программиста VISA).

Примечания

В большинстве случаев нет необходимости вызывать эту функцию, так как _viExecCommand() имеет третий НЕОБЯЗАТЕЛЬНЫЙ параметр $iTimeoutMS, который также может использоваться для установки тайм-аута VISA.

Как и для всех функций VISA, библиотеки VISA должны быть установлены (вы можете проверить, находится ли visa32.dll в {WINDOWS}\system32), а также должна быть установлена карта GPIB (такая как NI PCI-GPIB от National Instruments или Agilent 82350B PCI High-Performance GPIB)

* Подробное описание наиболее распространённых ДЕСКРИПТОРОВ VISA см. в примечаниях функции _viExecCommand()

Пример

; - Предполагает, что прибор установлен на адрес GPIB 3.
; Если прибор находится по другому адресу, измените "GPIB::3::0" на
; соответствующий дескриптор. То же самое для вызова _viOpen
; Показывает, как использовать функцию _viExecCommand с тайм-аутом или как
; вместо этого вызвать _viSetTimeout.

#include <MsgBoxConstants.au3>
#include <Visa.au3>

Local $h_Session = 0

; Запросить ID прибора по адресу GPIB 3
MsgBox($MB_SYSTEMMODAL, "Step 1", "Simple GPIB query with explicit TIMEOUT set")
Local $s_Answer = _viExecCommand("GPIB::3::0", "*IDN?", 10000) ; таймаут 10 секунд
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

; Это то же самое, что сначала использовать функцию _viSetTimeout:
MsgBox($MB_SYSTEMMODAL, "Step 2", "_vOpen + timeout using _viSetTimeout + GPIB query")
Local $h_Instr = _viOpen(3)
_viSetTimeout($h_Instr, 10000) ; 10000 мс = 10 секунд
$s_Answer = _viExecCommand($h_Instr, "*IDN?") ; Сейчас не нужно задавать таймаут
MsgBox($MB_SYSTEMMODAL, "GPIB QUERY result", $s_Answer) ; Показать ответ

MsgBox($MB_SYSTEMMODAL, "Step 3", "Close the Instrument connection using _viClose")
_viClose($h_Instr) ; Закрыть соединение с прибором
