Справочник функций

_WinAPI_GetTickCount64

Получает количество миллисекунд, прошедших с момента запуска системы.

#include <WinAPISys.au3>
_WinAPI_GetTickCount64 ( )

Возвращаемое значение

Возвращает число миллисекунд, прошедших с момента запуска системы.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
