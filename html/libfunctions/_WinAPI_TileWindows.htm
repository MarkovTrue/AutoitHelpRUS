Справочник функций

_WinAPI_TileWindows

Располагает указанные дочерние окна указанного родительского окна в виде мозаики.

#include <WinAPISysWin.au3>
_WinAPI_TileWindows ( $aWnds [, $tRECT = 0 [, $hParent = 0 [, $iFlags = $MDITILE_VERTICAL [, $iStart = 0 [, $iEnd = -1]]]]] )

Параметры

$aWnds Массив дескрипторов дочерних окон для расположения. Если указанное дочернее окно является окном верхнего уровня
со стилем $WS_EX_TOPMOST или $WS_EX_TOOLWINDOW, дочернее окно не расположено. Если этот параметр
равен 0, расположены все дочерние окна указанного родительского окна (или окна рабочего стола).
$tRECT [необязательный] Структура $tagRECT, которая задаёт прямоугольную область в координатах клиента, в пределах которой располагаются окна.
Если этот параметр равен 0 (по умолчанию), используется клиентская область родительского окна.
$hParent [необязательный] Дескриптор родительского окна. Если этот параметр равен 0 (по умолчанию), предполагается окно рабочего стола.
$iFlags [необязательный] Флаг каскада. Этот параметр может быть одним или несколькими из следующих значений.
$MDITILE_HORIZONTAL
$MDITILE_VERTICAL (по умолчанию).
$iStart [необязательный] Индекс массива, с которого начинать расположение.
$iEnd [необязательный] Индекс массива, на котором остановить расположение.

Возвращаемое значение

Успех: Возвращает значение – количество расположенных окон.
Ошибка: Возвращает 0.

Пример

#include <GUIConstantsEx.au3>
#include <WinAPIMisc.au3>
#include <WinAPISysWin.au3>
#include <WindowsStylesConstants.au3>

_Example()

Func _Example()
    Local $aWnds[4]

    For $i = 0 To UBound($aWnds) - 1
        $aWnds[$i] = GUICreate('#' & ($i + 1), 400, 400, -1, -1, BitOR($WS_CAPTION, $WS_POPUP, $WS_SIZEBOX, $WS_SYSMENU))
        GUISetState(@SW_SHOW)
    Next

    _WinAPI_TileWindows($aWnds, _WinAPI_CreateRectEx(20, 20, 600, 600))

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>_Example
