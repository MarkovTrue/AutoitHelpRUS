_WinAPI_LocalFree

Освобождает указанный объект локальной памяти и аннулирует его дескриптор.

#include <WinAPIMem.au3>
_WinAPI_LocalFree ( $hMemory )

Параметры

$hMemory Дескриптор объекта локальной памяти.
Этот дескриптор возвращается функцией LocalAlloc или LocalReAlloc.
Опасно освобождать память, выделенную с помощью GlobalAlloc.

Возвращаемое значение

Успех: Возвращает Null.
Ошибка: $hMemory, вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке
