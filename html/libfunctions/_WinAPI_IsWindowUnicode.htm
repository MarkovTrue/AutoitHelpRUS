_WinAPI_IsWindowUnicode

Определяет, является ли указанное окно собственным окном Unicode.

#include <WinAPISysWin.au3>
_WinAPI_IsWindowUnicode ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

True: Возвращает окно является собственным окном Unicode.
False: окно является окном ANSI.
