_WinAPI_GetModuleFileNameEx

Получает полный путь к файлу, содержащему указанный модуль.

#include <WinAPIProc.au3>
_WinAPI_GetModuleFileNameEx ( $hProcess [, $hModule = 0] )

Параметры

$hProcess Дескриптор процесса, содержащего модуль. Дескриптор должен иметь право доступа $PROCESS_QUERY_INFORMATION или
$PROCESS_QUERY_LIMITED_INFORMATION и право доступа $PROCESS_VM_READ.
$hModule [необязательный] Дескриптор модуля. Если этот параметр равен 0 (по умолчанию), функция получает путь исполняемого
файла процесса.

Возвращаемое значение

Успех: Возвращает полный путь к модулю.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Эта функция не получает путь для модулей, загруженных с флагом $LOAD_LIBRARY_AS_DATAFILE.
Для получения дополнительной информации см. _WinAPI_LoadLibraryEx().
