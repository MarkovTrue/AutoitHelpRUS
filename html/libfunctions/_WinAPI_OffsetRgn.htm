_WinAPI_OffsetRgn

Перемещает область на указанные смещения.

#include <WinAPIGdi.au3>
_WinAPI_OffsetRgn ( $hRgn, $iXOffset, $iYOffset )

Параметры

$hRgn Дескриптор области для перемещения.
$iXOffset Количество логических единиц для перемещения по горизонтали.
$iYOffset Количество логических единиц для перемещения по вертикали.

Возвращаемое значение

Успех: Возвращает значение, которое определяет сложность новой области; может быть одним из следующих значений.
$COMPLEXREGION
$NULLREGION
$SIMPLEREGION
Ошибка: Возвращает 0.
