Справочник функций

_IELinkClickByText

Имитирует щелчок мыши по ссылке, текст которой содержит указанную подстроку.

#include <IE.au3>
_IELinkClickByText ( ByRef $oObject, $sLinkText [, $iIndex = 0 [, $iWait = 1]] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sLinkText Текст ссылки, отображаемый на странице, по которой нужно кликнуть.
$iIndex [необязательный] Если текст ссылки встречается больше одного раза, задайте нужный экземпляр по индексу (с нуля)
$iWait [необязательный] Ждать ли загрузку страницы перед возвратом:
0 – вернуться сразу, не ожидая загрузки
1 – (по умолчанию) дождаться полной загрузки и вернуться

Возвращаемое значение

Успех: Нет.
Ошибка: Возвращает 0 или -1 и устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) – общая ошибка
2 ($_IEStatus_COMError) – ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
6 ($_IEStatus_LoadWaitTimeout) – истекло ожидание загрузки
7 ($_IEStatus_NoMatch) – нет совпадения
8 ($_IEStatus_AccessIsDenied) – доступ запрещён
9 ($_IEStatus_ClientDisconnected) – клиент отключён
@extended: Содержит номер недопустимого параметра.

Примечания

Не все элементы, выглядящие как ссылки, ими являются.
Частая практика – привязка JavaScript-событий onClick к другим DOM-элементам для имитации поведения ссылок.
Для активации таких элементов используйте "click" через _IEAction().

См. также

_IELinkClickByIndex, _IELoadWait

Пример

Пример 1

; Открытие браузера с базовым примером, щелчок по ссылке с текстом "user forum"

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IELinkClickByText($oIE, "user forum")

Пример 2

; Открытие браузера на домашней странице AutoIt, цикл по ссылкам
; на странице и кликнуть по ссылке с текстом "overview"
; используя частичное совпадение строки.

#include <IE.au3>

Local $oIE = _IECreate("http://www.autoitscript.com")

Local $sMyString = "overview"
Local $oLinks = _IELinkGetCollection($oIE)
For $oLink In $oLinks
    Local $sLinkText = _IEPropertyGet($oLink, "innerText")
    If StringInStr($sLinkText, $sMyString) Then
        _IEAction($oLink, "click")
        ExitLoop
    EndIf
Next
