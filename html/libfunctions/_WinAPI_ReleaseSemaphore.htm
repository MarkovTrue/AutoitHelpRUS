_WinAPI_ReleaseSemaphore

Увеличивает счётчик указанного объекта семафора на указанную величину.

#include <WinAPIProc.au3>
_WinAPI_ReleaseSemaphore ( $hSemaphore [, $iIncrease = 1] )

Параметры

$hSemaphore Дескриптор объекта семафора. Функция _WinAPI_CreateSemaphore() или _WinAPI_OpenSemaphore()
возвращает этот дескриптор.
$iIncrease [необязательный] Величина, на которую должен быть увеличен текущий счётчик объекта семафора. Значение должно быть больше
нуля. Если указанная величина приведёт к превышению счётчика семафора над максимальным счётчиком, который
был задан при создании семафора, счётчик не изменяется и функция возвращает 0. По умолчанию – 1.

Возвращаемое значение

Успех: Возвращает предыдущий счётчик семафора.
Ошибка: Задаёт флаг @error в ненулевое значение; вызовите _WinAPI_GetLastError() для получения расширенной информации об ошибке.

Примечания

Состояние объекта семафора имеет сигнальное значение, когда его счётчик больше нуля, и несигнальное, когда его счётчик
равен нулю. Процесс, который вызывает функцию _WinAPI_CreateSemaphore(), задаёт начальный
счётчик семафора. Каждый раз, когда ожидающий процесс освобождается из-за сигнального состояния семафора, счётчик
семафора уменьшается на один.

Пример

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>

If Not @Compiled Then
    MsgBox($MB_SYSTEMMODAL, '', 'To run this script, you must first compile it and then run the (.exe) file.')
    Exit
EndIf

Local $hSemaphore = _WinAPI_CreateSemaphore('MySemaphore', 2, 2)

_WinAPI_WaitForSingleObject($hSemaphore)
_MyGUI()
_WinAPI_ReleaseSemaphore($hSemaphore)
_WinAPI_CloseHandle($hSemaphore)

Func _MyGUI()
    GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'))
    GUISetState(@SW_SHOW)
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
        EndSwitch
    WEnd
EndFunc   ;==>_MyGUI
