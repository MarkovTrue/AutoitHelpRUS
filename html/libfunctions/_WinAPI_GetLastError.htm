Справочник функций

_WinAPI_GetLastError

Возвращает значение кода последней ошибки вызывающего потока.

#include <WinAPIError.au3>
_WinAPI_GetLastError ( )

Возвращаемое значение

Возвращает код последней ошибки

Примечания

При возврате значения @error и @extended сохраняются.
