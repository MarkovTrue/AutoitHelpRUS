_WinAPI_OpenJobObject

Открывает существующий объект задания.

#include <WinAPIProc.au3>
_WinAPI_OpenJobObject ( $sName [, $iAccess = $JOB_OBJECT_ALL_ACCESS [, $bInherit = False]] )

Параметры

$sName Имя задания для открытия. Сравнение имён чувствительно к регистру.
$iAccess [необязательный] Доступ к объекту задания. Этот параметр может быть одним или несколькими из следующих значений:
$JOB_OBJECT_ALL_ACCESS
$JOB_OBJECT_ASSIGN_PROCESS
$JOB_OBJECT_QUERY
$JOB_OBJECT_SET_ATTRIBUTES
$JOB_OBJECT_SET_SECURITY_ATTRIBUTES
$JOB_OBJECT_TERMINATE
$bInherit [необязательный] Наследуется ли дескриптор дочерними процессами:
True – процессы, созданные этим процессом, наследуют дескриптор.
False – процессы не наследуют дескриптор (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор объекта задания.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
