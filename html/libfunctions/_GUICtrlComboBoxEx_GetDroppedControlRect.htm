Справочник функций

_GUICtrlComboBoxEx_GetDroppedControlRect

Возвращает экранные координаты выпадающего списка в раскрытом состоянии.

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetDroppedControlRect ( $hWnd )

Параметры

$hWnd Дескриптор элемента управления.

Возвращаемое значение

Возвращает массив в следующем формате:
    [0] – X-координата верхнего левого угла прямоугольника
    [1] – Y-координата верхнего левого угла прямоугольника
    [2] – X-координата нижнего правого угла прямоугольника
    [3] – Y-координата нижнего правого угла прямоугольника

См. также

_GUICtrlComboBoxEx_GetDroppedControlRectEx

Пример

#include <DirConstants.au3>
#include <GuiComboBoxEx.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $hGUI, $aRect, $hCombo

    ; Создать GUI
    $hGUI = GUICreate("ComboBoxEx Get Dropped Control Rect", 400, 300)
    $hCombo = _GUICtrlComboBoxEx_Create($hGUI, "", 2, 2, 394, 100)
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, "", $DDL_DRIVES, False)
    _GUICtrlComboBoxEx_AddDir($hCombo, "", $DDL_DRIVES)
    _GUICtrlComboBoxEx_BeginUpdate($hCombo)
    _GUICtrlComboBoxEx_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBoxEx_EndUpdate($hCombo)
    _GUICtrlComboBoxEx_EndUpdate($hCombo)

    ; Получить прямоугольник выпадающего элемента
    $aRect = _GUICtrlComboBoxEx_GetDroppedControlRect($hCombo)
    MsgBox($MB_SYSTEMMODAL, "Information", "Dropped Control Rect: " & StringFormat("[%d][%d][%d][%d]", $aRect[0], $aRect[1], $aRect[2], $aRect[3]))

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
