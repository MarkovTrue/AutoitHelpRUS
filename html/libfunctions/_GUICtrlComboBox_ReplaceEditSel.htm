_GUICtrlComboBox_ReplaceEditSel

Заменяет текст, выбранный в поле ввода.

#include <GuiComboBox.au3>
_GUICtrlComboBox_ReplaceEditSel ( $hWnd, $sText )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$sText Текст замены.

Возвращаемое значение

Отсутствует.

Примечания

Если сообщение отправляется элементу ComboBox со стилем $CBS_DROPDOWN или $CBS_DROPDOWNLIST, функция завершится с ошибкой.

См. также

_GUICtrlComboBox_SetEditSel, _GUICtrlComboBox_SetEditText

Пример

#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $idCombo

    ; Создать GUI
    GUICreate("ComboBox Replace Edit Sel", 400, 296)
    $idCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    GUISetState(@SW_SHOW)

    ; Установить текст Edit
    _GUICtrlComboBox_SetEditText($idCombo, "Old Edit Text")

    ; Добавить файлы
    _GUICtrlComboBox_BeginUpdate($idCombo)
    _GUICtrlComboBox_AddDir($idCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($idCombo)

    Sleep(500)

    ; Установить выделенный текст в поле ввода
    _GUICtrlComboBox_SetEditSel($idCombo, 0, -1)

    Sleep(500)

    ; Заменить текст в поле ввода
    _GUICtrlComboBox_ReplaceEditSel($idCombo, "New Edit Text")

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
