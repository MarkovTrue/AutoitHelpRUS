_WinAPI_GetFullPathName

Получает полный путь и имя файла указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetFullPathName ( $sFilePath )

Параметры

$sFilePath Имя файла.

Возвращаемое значение

Успех: Возвращает диск и путь.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция _WinAPI_GetFullPathName() объединяет имя текущего диска и каталога с указанным именем файла для определения полного пути и имени файла указанного файла.
Эта функция не проверяет, являются ли полученный путь и имя файла допустимыми или видят ли существующий файл на связанном томе.

Пример

#include <WinAPIFiles.au3>

Local Const $sPath1 = 'Test.txt'
Local Const $sPath2 = 'Test\Test.txt'
Local Const $sPath3 = 'C:\Test\Test.txt'

ConsoleWrite($sPath1 & ' => ' & _WinAPI_GetFullPathName($sPath1) & @CRLF)
ConsoleWrite($sPath2 & ' => ' & _WinAPI_GetFullPathName($sPath2) & @CRLF)
ConsoleWrite($sPath3 & ' => ' & _WinAPI_GetFullPathName($sPath3) & @CRLF)
