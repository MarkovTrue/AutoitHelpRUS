Справочник функций

_WinAPI_GetGlyphOutline

Получает контур или точечный рисунок символа в шрифте TrueType.

#include <WinAPIGdi.au3>
_WinAPI_GetGlyphOutline ( $hDC, $sChar, $iFormat, ByRef $pBuffer [, $tMAT2 = 0] )

Параметры

$hDC Дескриптор контекста устройства, к которому применён шрифт.
$sChar Символ, для которого требуется вернуть данные.
$iFormat Формат данных, которые получает функция.
Этот параметр может быть одним из следующих значений:
$GGO_BEZIER
$GGO_BITMAP
$GGO_GLYPH_INDEX
$GGO_GRAY2_BITMAP
$GGO_GRAY4_BITMAP
$GGO_GRAY8_BITMAP
$GGO_METRICS
$GGO_NATIVE
$GGO_UNHINTED
$pBuffer Возвращает указатель на блок памяти (буфер), получающий данные контура или точечного рисунка.
Вы можете установить этот параметр в 0 перед вызовом функции, и она выделит необходимый блок памяти самостоятельно.
В противном случае это должен быть действительный указатель на память, возвращённый функцией _WinAPI_CreateBuffer(), или полученный предыдущим вызовом этой функции.
Если указан $GGO_METRICS, этот параметр игнорируется, и функция возвращает только информацию о глифе (см. ниже).
$tMAT2 [необязательный] Структура $tagMAT2, задающая матрицу преобразования для символа.
Если этот параметр имеет значение 0 (по умолчанию), преобразование не будет использоваться (это единичная матрица).

Возвращаемое значение

Успех: Возвращает структура $tagGLYPHMETRICS, содержащая информацию о размещении и ориентации глифа.
Флаг @extended возвращает количество скопированных в буфер байтов (если используется).
Ошибка: Устанавливает @error ≠ 0.

Примечания

Обратите внимание, что для значений $GGO_GRAY... функция получает точечный рисунок глифа, содержащий n^2+1 (n в квадрате плюс один) уровней серого.

Точечный рисунок глифа, возвращаемый функцией _WinAPI_GetGlyphOutline() при указании $GGO_BITMAP, – это выравниваемый по DWORD, упорядоченный по строкам монохромный точечный рисунок.
При указании $GGO_GRAY2_BITMAP возвращаемый точечный рисунок – это выравниваемый по DWORD, упорядоченный по строкам массив байтов, значения которых варьируются от 0 до 4.
При указании $GGO_GRAY4_BITMAP возвращаемый точечный рисунок – это выравниваемый по DWORD, упорядоченный по строкам массив байтов, значения которых варьируются от 0 до 16.
При указании $GGO_GRAY8_BITMAP возвращаемый точечный рисунок – это выравниваемый по DWORD, упорядоченный по строкам массив байтов, значения которых варьируются от 0 до 64.

Когда буфер, выделенный этой функцией, больше не нужен, следует вызвать функцию _WinAPI_FreeMemory() (не используйте другие подпрограммы работы с памятью) для освобождения занимаемой памяти.

См. также

_WinAPI_CreateBuffer, _WinAPI_FreeMemory

Пример

#include <FontConstants.au3>
#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPIMem.au3>

; Создаёт логический шрифт ("Times") и получает биты точечного рисунка для произвольного символа
Local $hDC = _WinAPI_CreateCompatibleDC(0)
Local $hFont = _WinAPI_CreateFont(512, 0, 0, 0, 400, False, False, False, $DEFAULT_CHARSET, $OUT_DEFAULT_PRECIS, $CLIP_DEFAULT_PRECIS, $DEFAULT_QUALITY, 0, 'Times')
Local $hSv = _WinAPI_SelectObject($hDC, $hFont)
Local $pData = 0
Local $tGM = _WinAPI_GetGlyphOutline($hDC, ChrW(Random(65, 90, 1)), $GGO_BITMAP, $pData)
Local $W = DllStructGetData($tGM, 'BlackBoxX')
Local $H = DllStructGetData($tGM, 'BlackBoxY')
_WinAPI_SelectObject($hDC, $hSv)
_WinAPI_DeleteObject($hFont)

; Создаёт точечный рисунок 1 бит/пикс
Local $hBmp = _WinAPI_CreateBitmap(32 * Ceiling($W / 32), $H, 1, 1, $pData)

; Обрезает точечный рисунок до нужного размера и раскрашивает его
Local $aColorTable[2] = [0xFFFFFF, 0xC00000]
Local $hDib = _WinAPI_CreateDIB($W, $H, 1, _WinAPI_CreateDIBColorTable($aColorTable), 2)
$hSv = _WinAPI_SelectObject($hDC, $hDib)
_WinAPI_DrawBitmap($hDC, 0, 0, $hBmp, $MERGEPAINT)
_WinAPI_SelectObject($hDC, $hSv)
_WinAPI_DeleteObject($hBmp)

; Освобождает ненужные ресурсы
_WinAPI_FreeMemory($pData)
_WinAPI_DeleteDC($hDC)

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), $W, $H)
Local $idPic = GUICtrlCreatePic('', 0, 0, $W, $H)
Local $hPic = GUICtrlGetHandle($idPic)

; Установить bitmap для элемента управления
_SendMessage($hPic, $STM_SETIMAGE, 0, $hDib)
$hBmp = _SendMessage($hPic, $STM_GETIMAGE)
If $hBmp <> $hDib Then
    _WinAPI_DeleteObject($hDib)
EndIf

; Показать GUI
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
