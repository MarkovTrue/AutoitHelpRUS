Справочник функций

_WinAPI_GetWindowRgnBox

Получает размеры самого плотного ограничивающего прямоугольника для области окна.

#include <WinAPIGdi.au3>
_WinAPI_GetWindowRgnBox ( $hWnd, ByRef $tRECT )

Параметры

$hWnd Дескриптор окна.
$tRECT Возвращает структуру $tagRECT, которая создаётся этой функцией и содержит размеры прямоугольника в единицах устройства
относительно верхнего левого угла окна.

Возвращаемое значение

Успех: Сложность области. Может принимать одно из значений:
$COMPLEXREGION
$NULLREGION
$SIMPLEREGION
Ошибка: Возвращает 0.
