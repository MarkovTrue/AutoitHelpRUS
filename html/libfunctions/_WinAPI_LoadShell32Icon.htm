Справочник функций

_WinAPI_LoadShell32Icon

Извлекает значок из файла shell32.dll.

#include <WinAPIIcons.au3>
_WinAPI_LoadShell32Icon ( $iIconID )

Параметры

$iIconID Идентификатор значка для извлечения.

Возвращаемое значение

Успех: Возвращает дескриптор указанного значка.
Ошибка: Возвращает 0.

Примечания

Когда закончите работать со значком, вызовите _WinAPI_DestroyIcon() для освобождения дескриптора значка

См. также

_WinAPI_DestroyIcon
