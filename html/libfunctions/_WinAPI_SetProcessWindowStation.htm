Справочник функций

_WinAPI_SetProcessWindowStation

Назначает указанную станцию окна вызывающему процессу.

#include <WinAPISys.au3>
_WinAPI_SetProcessWindowStation ( $hStation )

Параметры

$hStation Дескриптор станции окна. Эта станция окна должна быть связана с текущим сеансом.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
