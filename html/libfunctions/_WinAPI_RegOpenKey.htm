Справочник функций

_WinAPI_RegOpenKey

Открывает указанный ключ реестра.

#include <WinAPIReg.au3>
_WinAPI_RegOpenKey ( $vKey [, $sSubKey = '' [, $iAccess = $KEY_ALL_ACCESS]] )

Параметры

$vKey Имя открываемого ключа (см. раздел Примечания).
или
Дескриптор открытого ключа реестра. Этот дескриптор возвращается функцией _WinAPI_RegCreateKey() или _WinAPI_RegOpenKey()
, или может быть одним из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sSubKey [необязательный] Имя открываемого подключа реестра (см. раздел Примечания).
$iAccess [необязательный] Маска, определяющая требуемые права доступа к ключу.
Функция завершается с ошибкой, если дескриптор безопасности ключа не разрешает запрошенный доступ для вызывающего процесса.
Этот параметр может быть одной или несколькими константами $KEY_*. По умолчанию $KEY_ALL_ACCESS.
Этот параметр может также иметь следующие значения для открытия в любом режиме работы AutoIt:
$KEY_WOW64_32KEY для 32-битного ключа
$KEY_WOW64_64KEY для 64-битного ключа
По умолчанию ключ соответствует значению @AutoItX64, если не переопределено $KEY_WOW64_32KEY.

Возвращаемое значение

Успех: Возвращает дескриптор открытого ключа.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

$vKey может быть строкой, содержащей запрошенный подключ, в этом случае $sSubKey должна быть пуста

В отличие от функции _WinAPI_RegCreateKey(),
функция _WinAPI_RegOpenKey() не создаёт указанный ключ, если ключ не существует в реестре.

Если ключ не является одним из предопределённых ключей реестра ($HKEY_*), вы должны вызвать функцию _WinAPI_RegCloseKey() после завершения использования дескриптора.

Имена ключей не чувствительны к регистру.

Параметр $sSubKey может быть пустой строкой.
Если $sSubKey – пустая строка и $vKey равен $HKEY_CLASSES_ROOT, возвращается тот же дескриптор $vKey, переданный в функцию.
В противном случае возвращается новый дескриптор ключа, указанного в $vKey.

Параметр $sSubKey может быть NULL только если $vKey является одним из предопределённых ключей.
Если $sSubKey равен NULL и $vKey равен $HKEY_CLASSES_ROOT, возвращается новый дескриптор ключа, указанного в $vKey.
В противном случае возвращается тот же дескриптор $vKey, переданный в функцию.

См. также

_WinAPI_RegCloseKey, _WinAPI_RegCreateKey

Пример

Пример 1

#include <Date.au3>
#include <Debug.au3>
#include <WinAPIError.au3>
#include <WinAPILocale.au3>
#include <WinAPIReg.au3>

_DebugSetup(Default, True)

Example()

Func Example()
    Local $hKey = _WinAPI_RegOpenKey($HKEY_LOCAL_MACHINE, 'SOFTWARE\AutoIt v3\AutoIt', $KEY_QUERY_VALUE)
    If @error Then
        _DebugReport("! RegOpenKey @error =" & @error & @CRLF & @TAB & _WinAPI_GetErrorMessage(@extended) & @CRLF)
        Exit
    EndIf

    Local $tFT = _WinAPI_RegQueryLastWriteTime($hKey)
    $tFT = _Date_Time_FileTimeToLocalFileTime($tFT)
    Local $tST = _Date_Time_FileTimeToSystemTime($tFT)
    _WinAPI_RegCloseKey($hKey)

    _DebugReport('! Last modified at: ' & _WinAPI_GetDateFormat(0, $tST) & ' ' & _WinAPI_GetTimeFormat(0, $tST) & @CRLF)

EndFunc   ;==>Example
