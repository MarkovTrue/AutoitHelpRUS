Справочник функций

_Word_DocTableWrite

Записывает одномерный или двумерный массив в таблицу Microsoft Word.

#include <Word.au3>
_Word_DocTableWrite ( $oRange, ByRef $aArray [, $iIndexBase = Default [, $sDelimiter = Default]] )

Параметры

$oRange Объект диапазона в Microsoft Word, где должна быть вставлена таблица.
$aArray одномерный или двумерный массив для преобразования в таблицу.
Может быть отсчитан с нуля или с единицы.
$iIndexBase [необязательный] Может быть 0 или 1 и указывает, начинается ли массив со строки 0 или 1 (по умолчанию = 1)
$sDelimiter [необязательный] Указывает символ, используемый для разделения текста на ячейки (по умолчанию = @TAB)

Возвращаемое значение

Успех: Возвращает объект таблицы Microsoft Word.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $oRange не является объектом
2 – $aArray не является массивом или имеет более 2 измерений
3 – Ошибка при доступе к указанному объекту диапазона. @extended задан код ошибки COM
4 – Ошибка при преобразовании текста в таблицу. @extended задан код ошибки COM

Примечания

Массив не может содержать @CR, @CRLF или @LF ни в одной ячейке.
Если массив содержит символы @TAB, необходимо установить параметр $sDelimiter в другой символ.

См. также

_Word_DocTableRead

Пример

#include "Extras\HelpFileInternals.au3"

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Создать объект приложения
Local $oWord = _Word_Create()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableWrite Example", _
        "Error creating the Word application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Открыть тестовый документ
Local $oDoc = _Word_DocOpen($oWord, _Extras_PathFull("Test.doc"), Default, Default, True)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableWrite Example", _
        "Error opening '.\Extras\Test.doc'." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Записать содержимое отсчитанного с нуля двумерного массива в таблицу Microsoft Word
Local $asArray[3][3] = [[1, 2, 3], ["a", "b", "c"], ["x", "y", "z"]]
Local $oRange = _Word_DocRangeSet($oDoc, -2)
_Word_DocTableWrite($oRange, $asArray, 0)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableWrite Example", _
        "Error creating the table." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocTableWrite Example", _
        "Table successfully added to the end of the document.")
