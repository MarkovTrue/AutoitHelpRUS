Справочник функций

_WinAPI_CreateRect

создаёт структуру $tagRECT с координатами указанного прямоугольника.

#include <WinAPIMisc.au3>
_WinAPI_CreateRect ( $iLeft, $iTop, $iRight, $iBottom )

Параметры

$iLeft Координата X левого верхнего угла прямоугольника.
$iTop Координата Y левого верхнего угла прямоугольника.
$iRight X-координата правого нижнего угла прямоугольника.
$iBottom Y-координата правого нижнего угла прямоугольника.

Возвращаемое значение

Возвращает структуру $tagRECT, содержащую указанный прямоугольник.
