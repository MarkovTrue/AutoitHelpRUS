_GUICtrlComboBox_GetEditSel

Получает начальную и конечную позиции текущего выбранного текста в поле ввода выпадающего списка.

#include <GuiComboBox.au3>
_GUICtrlComboBox_GetEditSel ( $hWnd )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.

Возвращаемое значение

Успех: Возвращает массив в формате [начало, конец] позиций выделения.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Если сообщение отправляется элементу ComboBox со стилем $CBS_DROPDOWN или $CBS_DROPDOWNLIST, функция завершится с ошибкой.

См. также

_GUICtrlComboBox_SetEditSel, _GUICtrlComboBoxEx_SetEditSel

Пример

#include "Extras\HelpFileInternals.au3"

#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("ComboBox Get/Set Edit Sel (v" & @AutoItVersion & ")", 400, 296)
    Local $idCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    _MemoCreate(2, 32, 396, 266, 0)
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBox_BeginUpdate($idCombo)
    _GUICtrlComboBox_AddDir($idCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($idCombo)

    ; Выбор элемента
    _GUICtrlComboBox_SetCurSel($idCombo, 2)

    ; Задать выделение Edit
    _GUICtrlComboBox_SetEditSel($idCombo, 0, 4)

    ; Получить выделение в edit
    Local $aSel = _GUICtrlComboBox_GetEditSel($idCombo)
    _MemoWrite(StringFormat("Edit Sel: %d - %d", $aSel[0], $aSel[1]))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
