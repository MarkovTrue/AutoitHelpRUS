Справочник функций

_GUICtrlEdit_LineScroll

Прокручивает текст в многострочном Edit.

#include <GuiEdit.au3>
_GUICtrlEdit_LineScroll ( $hWnd, $iHoriz, $iVert )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iHoriz Количество символов для прокрутки по горизонтали.
$iVert Количество строк для прокрутки по вертикали.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Поле редактирования не прокручивается по вертикали ниже последней строки текста.

Если сумма текущей строки и значения параметра $iVert превышает общее число строк, значение корректируется так, чтобы последняя строка оказалась наверху окна редактирования.

_GUICtrlEdit_LineScroll() прокручивает текст по вертикали или горизонтали.
_GUICtrlEdit_LineScroll() может прокручивать по горизонтали и за последний символ строки.

См. также

_GUICtrlEdit_Scroll

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <GuiStatusBar.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hStatusBar, $idEdit, $hGUI, $sText

    ; Создать GUI
    $hGUI = GUICreate("Edit Line Scroll", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268, BitOR($ES_WANTRETURN, $WS_VSCROLL))
    $hStatusBar = _GUICtrlStatusBar_Create($hGUI)
    GUISetState(@SW_SHOW)

    ; Установить текст
    For $i = 1 To 50
        $sText &= $i & @CRLF
    Next
    _GUICtrlEdit_SetText($idEdit, $sText)

    ; прокрутка
    _GUICtrlStatusBar_SetText($hStatusBar, "Scrolled: " & _GUICtrlEdit_LineScroll($idEdit, 0, _GUICtrlEdit_GetLineCount($idEdit)))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
