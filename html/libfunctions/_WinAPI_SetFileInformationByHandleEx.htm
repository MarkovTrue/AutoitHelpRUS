Справочник функций

_WinAPI_SetFileInformationByHandleEx

Устанавливает информацию о файле для указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_SetFileInformationByHandleEx ( $hFile, $tFILEINFO )

Параметры

$hFile Дескриптор файла, для которого требуется изменить информацию. Этот дескриптор должен иметь соответствующие
разрешения для запрашиваемого изменения.
$tFILEINFO Структура $tagFILEINFO, содержащая информацию для изменения.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки NTSTATUS.
