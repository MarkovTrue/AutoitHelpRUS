Справочник функций

_WinAPI_MsgBox

Отображает окно сообщения с более широким полем, чем оригинал.

#include <WinAPIError.au3>
_WinAPI_MsgBox ( $iFlags, $sTitle, $sText )

Параметры

$iFlags Флаги для использования при создании окна.
$sTitle Заголовок окна.
$sText Текст окна.

Возвращаемое значение

Отсутствует.

Примечания

Эта функция создаёт (на мой взгляд) более красиво выглядящее окно сообщения. Она также гарантирует, что BlockInput отключено
для того, чтобы пользователь мог перемещать мышь.

См. также

_WinAPI_ShowError, _WinAPI_ShowMsg
