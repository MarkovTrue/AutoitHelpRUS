_WinAPI_EqualMemory

Сравнивает два блока памяти на совпадение указанного числа байт.

#include <WinAPIMem.au3>
_WinAPI_EqualMemory ( $pSource1, $pSource2, $iLength )

Параметры

$pSource1 Указатель на блок памяти для сравнения.
$pSource2 Указатель на блок памяти, который сравнивается с блоком памяти, на который указывает $pSource1.
$iLength Задаёт количество байт для сравнения.

Возвращаемое значение

Успех: 1 – блоки памяти равны.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Дополнительно

Искать RtlCompareMemory в библиотеке MSDN

Пример

#include <WinAPIMem.au3>

Local $tStruct1 = DllStructCreate('byte[8]')
Local $tStruct2 = DllStructCreate('byte[8]')

_WinAPI_FillMemory($tStruct1, 8, 0x1D)
_WinAPI_FillMemory($tStruct2, 8, 0x1D)

ConsoleWrite('Two structures are equivalent: ' & _WinAPI_EqualMemory($tStruct1, $tStruct2, 8) & @CRLF)
