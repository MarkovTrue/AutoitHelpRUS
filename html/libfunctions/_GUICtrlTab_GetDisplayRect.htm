_GUICtrlTab_GetDisplayRect

Получает прямоугольник координат.

#include <GuiTab.au3>
_GUICtrlTab_GetDisplayRect ( $hWnd )

Параметры

$hWnd Дескриптор элемента управления.

Возвращаемое значение

Возвращает массив в следующем формате:
    [0] = X-координата верхнего левого угла прямоугольника
    [1] = Y-координата верхнего левого угла прямоугольника
    [2] = X-координата нижнего правого угла прямоугольника
    [3] = Y-координата нижнего правого угла прямоугольника

Примечания

Используйте эту функцию, чтобы получить координаты (в пикселях) клиентской области, доступной для размещения дочерних элементов.

См. также

_GUICtrlTab_GetDisplayRectEx

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <GuiTab.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI
    Local $hGUI = GUICreate("Tab Control Get Display Rect (v" & @AutoItVersion & ")", 450, 300, 100, 100)
    Local $idTab = GUICtrlCreateTab(2, 2, 446, 266)
    _MemoMsgBoxStatus() ; Создание статус-бара

    GUISetState(@SW_SHOW)

    ; Добавить вкладки
    _GUICtrlTab_InsertItem($idTab, 0, "Tab 0")
    _GUICtrlTab_InsertItem($idTab, 1, "Tab 1")
    _GUICtrlTab_InsertItem($idTab, 2, "Tab 2")

    ; Получить прямоугольник отображения
    Local $aRect = _GUICtrlTab_GetDisplayRect(GUICtrlGetHandle($idTab))
    Local $sRect = StringFormat("[%d, %d, %d, %d]", $aRect[0], $aRect[1], $aRect[2], $aRect[3])
    _MemoMsgBox($MB_SYSTEMMODAL, "Information", "Display rectangle: " & $sRect)

    _MemoMsgBoxStatus("", -1, $hGUI) ; больше действий нет, ждём закрытия GUI

EndFunc   ;==>Example
