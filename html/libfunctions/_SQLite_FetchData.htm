Справочник функций

_SQLite_FetchData

Получает 1 строку данных из запроса, основанного на _SQLite_Query().

#include <SQLite.au3>
_SQLite_FetchData ( $hQuery, ByRef $aRow [, $bBinary = False [, $bDoNotFinalize = False [, $iColumns = 0 [, $bAutoItTypeConversion = False]]]] )

Параметры

$hQuery Дескриптор запроса, возвращённый _SQLite_Query().
$aRow Одномерный массив, содержащий строку данных.
$bBinary [необязательный] Флаг двоичного режима ($aRow будет массивом двоичных строк)
$bDoNotFinalize [необязательный] Флаг можно установить в True, если нужно оставить запрос незавершённым для дальнейшего использования.
(Тогда ответственность вызывающей стороны – вызвать _SQLite_QueryFinalize() перед закрытием базы данных.).
$iColumns [необязательный] Количество возвращаемых столбцов (по умолчанию = все)
$bAutoItTypeConversion [необязательный] Принудительное преобразование из строки в переменную типа AutoIt (по умолчанию = False). См. раздел Примечания.

Возвращаемое значение

Успех: Возвращает $SQLITE_OK.
Ошибка: Возвращает значение, которое можно сравнить с константами $SQLITE_*.
@error: -1 – SQLite сообщил об ошибке (проверьте возвращаемое значение)
1 – ошибка при вызове SQLite API "sqlite3_step"
2 – ошибка при вызове SQLite API "sqlite3_data_count"
3 – ошибка при вызове SQLite API "sqlite3_column_text16"
4 – ошибка при вызове SQLite API "sqlite3_column_type"
5 – ошибка при вызове SQLite API "sqlite3_column_bytes"
6 – ошибка при вызове SQLite API "sqlite3_column_blob"
7 – вызов запрещён безопасным режимом
8 – ошибка при вызове SQLite API "sqlite3_column_int64"
9 – ошибка при вызове SQLite API "sqlite3_column_double"
13 – ошибка при вызове SQLite API "sqlite3_column_text16"
15 – ошибка при вызове SQLite API "sqlite3_column_bytes"
16 – ошибка при вызове SQLite API "sqlite3_column_blob"

Примечания

$bAutoItTypeConversion полезна, когда весь скрипт не может использовать параметр _SQLite_Startup().
По умолчанию возвращается строка или бинарная переменная AutoIt.

См. также

_SQLite_Query, _SQLite_QueryFinalize, _SQlite_Startup

Пример

#include <SQLite.au3>
#include <SQLite.dll.au3>

Local $hQuery, $aRow, $aNames
_SQLite_Startup()
If @error Then
    MsgBox($MB_SYSTEMMODAL, "SQLite Error", "SQLite3.dll Can't be Loaded!")
    Exit -1
EndIf
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open() ; открыть :memory: базу данных
_SQLite_Exec(-1, "CREATE TABLE aTest (A,B,C);")
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('c','2','World');")
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('b','3',' ');")
_SQLite_Exec(-1, "INSERT INTO aTest(a,b,c) VALUES ('a','1','Hello');")
_SQLite_Query(-1, "SELECT ROWID,* FROM aTest ORDER BY a;", $hQuery)
_SQLite_FetchNames($hQuery, $aNames)
ConsoleWrite(StringFormat(" %-10s %-10s %-10s %-10s ", $aNames[0], $aNames[1], $aNames[2], $aNames[3]) & @CRLF)
While _SQLite_FetchData($hQuery, $aRow, False, False) = $SQLITE_OK ; Прочитать следующую строку
    ConsoleWrite(StringFormat(" %-10s %-10s %-10s %-10s ", $aRow[0], $aRow[1], $aRow[2], $aRow[3]) & @CRLF)
WEnd
_SQLite_QueryFinalize($hQuery)
_SQLite_Exec(-1, "DROP TABLE aTest;")
_SQLite_Close()
_SQLite_Shutdown()

; Вывод:
; rowid A B C
; 3 a 1 Hello
; 2 b 3
; 1 c 2 World
