_WinAPI_FatalExit

Передаёт управление выполнением отладчику.

#include <WinAPIDiag.au3>
_WinAPI_FatalExit ( $iCode )

Параметры

$iCode Первое слово предложения ошибка code associated с exit.

Возвращаемое значение

Отсутствует.

Примечания

@error возвращается, если произошла ошибка.
