Справочник функций

_WinAPI_OpenDesktop

Открывает указанный объект рабочего стола.

#include <WinAPISys.au3>
_WinAPI_OpenDesktop ( $sName [, $iAccess = 0 [, $iFlags = 0 [, $bInherit = False]]] )

Параметры

$sName Имя рабочего стола для открытия. Имена рабочих столов нечувствительны к регистру. Этот рабочий стол должен принадлежать
текущей станции окна.
$iAccess [необязательный] Доступ к рабочему столу. Этот параметр может быть одним или несколькими из следующих значений:
$DESKTOP_ALL_ACCESS
$DESKTOP_CREATEMENU
$DESKTOP_CREATEWINDOW
$DESKTOP_ENUMERATE
$DESKTOP_HOOKCONTROL
$DESKTOP_JOURNALPLAYBACK
$DESKTOP_JOURNALRECORD
$DESKTOP_READOBJECTS
$DESKTOP_SWITCHDESKTOP
$DESKTOP_WRITEOBJECTS
$iFlags [необязательный] Дополнительные флаги. Может быть нулем или следующим значением:
$DF_ALLOWOTHERACCOUNTHOOK
$bInherit [необязательный] Наследуется ли дескриптор дочерними процессами:
True – процессы, созданные этим процессом, наследуют дескриптор.
False – процессы не наследуют дескриптор (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор открытого рабочего стола.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Когда вы закончите работать с рабочим столом, вызовите функцию _WinAPI_CloseDesktop() для его закрытия.
