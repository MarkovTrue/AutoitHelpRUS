_WinAPI_DeviceIoControl

Отправляет управляющий код напрямую драйверу указанного устройства.

#include <WinAPIFiles.au3>
_WinAPI_DeviceIoControl ( $hDevice, $iControlCode [, $pInBuffer = 0 [, $iInBufferSize = 0 [, $pOutBuffer = 0 [, $iOutBufferSize = 0]]]] )

Параметры

$hDevice Дескриптор устройства, на котором выполняется операция.
Обычно это том, каталог, файл или поток. Дескриптор устройства получают через _WinAPI_CreateFileEx().
Имя устройства имеет вид:
\\.\DeviceName.
$iControlCode Управляющий код операции.
Значение задаёт конкретную операцию и тип устройства, для которого она выполняется.
$pInBuffer [необязательный] Указатель на входной буфер с данными, необходимыми для выполнения операции.
$iInBufferSize [необязательный] Размер входного буфера в байтах. По умолчанию 0.
$pOutBuffer [необязательный] Указатель на выходной буфер, в который операция запишет возвращаемые данные.
$iOutBufferSize [необязательный] Размер выходного буфера в байтах. По умолчанию 0.

Возвращаемое значение

Успех: Возвращает True; @extended содержит размер данных, записанных в выходной буфер, в байтах.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
