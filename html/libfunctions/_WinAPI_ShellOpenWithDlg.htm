_WinAPI_ShellOpenWithDlg

Отображает диалоговое окно "Открыть с помощью".

#include <WinAPIDlg.au3>
_WinAPI_ShellOpenWithDlg ( $sFilePath [, $iFlags = 0 [, $hParent = 0]] )

Параметры

$sFilePath Имя файла.
$iFlags [необязательный] Характеристики диалогового окна "Открыть с помощью". Этот параметр может быть одним или несколькими из следующих значений. $OAIF_ALLOW_REGISTRATION $OAIF_REGISTER_EXT $OAIF_EXEC $OAIF_FORCE_REGISTRATION $OAIF_HIDE_REGISTRATION $OAIF_URL_PROTOCOL
$hParent [необязательный] Дескриптор родительского окна.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

Дополнительно

Искать SHOpenWithDialog в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIDlg.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

_WinAPI_ShellOpenWithDlg(@ScriptFullPath, $OAIF_EXEC)
