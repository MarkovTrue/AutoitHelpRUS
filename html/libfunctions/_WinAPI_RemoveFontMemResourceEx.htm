_WinAPI_RemoveFontMemResourceEx

Удаляет шрифты, добавленные из образа памяти.

#include <WinAPIGdi.au3>
_WinAPI_RemoveFontMemResourceEx ( $hFont )

Параметры

$hFont Дескриптор ресурса шрифта. Этот дескриптор возвращается функцией _WinAPI_AddFontMemResourceEx().

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
