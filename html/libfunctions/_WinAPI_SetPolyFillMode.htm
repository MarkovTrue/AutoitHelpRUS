_WinAPI_SetPolyFillMode

Задаёт режим заполнения полигона для функций, которые заполняют полигоны.

#include <WinAPIGdi.au3>
_WinAPI_SetPolyFillMode ( $hDC [, $iMode = $ALTERNATE] )

Параметры

$hDC Дескриптор контекста устройства.
$iMode [необязательный] Новый режим заполнения. Этот параметр может быть одним из следующих значений.
$ALTERNATE (По умолчанию)
$WINDING

Возвращаемое значение

Успех: Возвращает предыдущий режим заполнения.
Ошибка: Возвращает 0.
