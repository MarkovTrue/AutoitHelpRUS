Справочник функций

_WinAPI_WriteConsole

Записывает строку символов в буфер экрана консоли.

#include <WinAPIProc.au3>
_WinAPI_WriteConsole ( $hConsole, $sText )

Параметры

$hConsole Дескриптор буфера экрана консоли.
$sText Текст для записи в буфер экрана консоли.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
