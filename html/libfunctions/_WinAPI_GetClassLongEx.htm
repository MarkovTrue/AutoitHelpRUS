_WinAPI_GetClassLongEx

Получает указанное значение, связанное с указанным окном.

#include <WinAPISysWin.au3>
_WinAPI_GetClassLongEx ( $hWnd, $iIndex )

Параметры

$hWnd Дескриптор окна.
$iIndex Значение для получения. Этот параметр может быть одним из следующих значений.
$GCL_CBCLSEXTRA
$GCL_CBWNDEXTRA
$GCL_HBRBACKGROUND
$GCL_HCURSOR
$GCL_HICON
$GCL_HICONSM
$GCL_HMODULE
$GCL_MENUNAME
$GCL_STYLE
$GCL_WNDPROC

Возвращаемое значение

Успех: Возвращает запрошенное значение.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
