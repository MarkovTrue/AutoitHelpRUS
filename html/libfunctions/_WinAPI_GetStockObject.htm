Справочник функций

_WinAPI_GetStockObject

Получает дескриптор к одному из предопределённых стандартных перьев, кистей, шрифтов или палитр.

#include <WinAPIHObj.au3>
_WinAPI_GetStockObject ( $iObject )

Параметры

$iObject Задаёт тип стандартного объекта. Этот параметр может быть одним из следующих значений:
$BLACK_BRUSH – чёрная кисть
$DKGRAY_BRUSH – тёмно-серая кисть
$GRAY_BRUSH – серая кисть
$HOLLOW_BRUSH – полая кисть (эквивалентна NULL_BRUSH)
$LTGRAY_BRUSH – светло-серая кисть
$NULL_BRUSH – нулевая кисть (эквивалентна HOLLOW_BRUSH)
$WHITE_BRUSH – белая кисть
$BLACK_PEN – чёрное перо
$NULL_PEN – нулевое перо
$WHITE_PEN – белое перо
$ANSI_FIXED_FONT – системный шрифт с фиксированным шагом Windows (моноширинный)
$ANSI_VAR_FONT – системный шрифт с переменным шагом Windows (пропорциональный)
$DEVICE_DEFAULT_FONT – зависимый от устройства шрифт
$DEFAULT_GUI_FONT – шрифт по умолчанию для объектов пользовательского интерфейса
$OEM_FIXED_FONT – зависимый от OEM шрифт с фиксированным шагом (моноширинный)
$SYSTEM_FONT – системный шрифт
$SYSTEM_FIXED_FONT – системный шрифт с фиксированным шагом (моноширинный), используемый в версиях Windows старше 3.0
$DEFAULT_PALETTE – палитра по умолчанию. Эта палитра состоит из статических цветов системной палитры.

Возвращаемое значение

Успех: Возвращает запрошенный логический объект.
Ошибка: Возвращает 0.
