Справочник функций

_GDIPlus_ParamAdd

Добавляет значение в список параметров кодировщика.

#include <GDIPlus.au3>
_GDIPlus_ParamAdd ( ByRef $tParams, $sGUID, $iNbOfValues, $iType, $pValues )

Параметры

$tParams Структура $tagGDIPENCODERPARAMS, возвращённая функцией _GDIPlus_ParamInit().
$sGUID GUID параметра кодировщика. Допустимые значения:
$GDIP_EPGCHROMINANCETABLE – таблица цветности
$GDIP_EPGCOLORDEPTH – глубина цвета
$GDIP_EPGCOMPRESSION – сжатие
$GDIP_EPGLUMINANCETABLE – таблица яркости
$GDIP_EPGQUALITY – качество
$GDIP_EPGRENDERMETHOD – метод отрисовки
$GDIP_EPGSAVEFLAG – флаг сохранения
$GDIP_EPGSCANMETHOD – режим сканирования
$GDIP_EPGTRANSFORMATION – преобразование
$GDIP_EPGVERSION – версия программы.
$iNbOfValues Количество элементов в массиве $pValues.
$iType Тип значения параметра кодировщика. Допустимые значения:
$GDIP_EPTBYTE – 8-битное целое без знака
$GDIP_EPTASCII – строка с нулевым завершением
$GDIP_EPTSHORT – 16-битное целое без знака
$GDIP_EPTLONG – 32-битное целое без знака
$GDIP_EPTRATIONAL – два long (числитель, знаменатель)
$GDIP_EPTLONGRANGE – два long (минимум, максимум)
$GDIP_EPTUNDEFINED – массив байт произвольного типа
$GDIP_EPTRATIONALRANGE – два long (минимум, максимум).
$pValues Указатель на массив значений. Тип каждого значения определяется параметром $iType.

Возвращаемое значение

Отсутствует.

См. также

$tagGDIPENCODERPARAMS, _GDIPlus_ParamInit

Пример

#include <GDIPlus.au3>
#include <ScreenCapture.au3>

Example()

Func Example()
    Local $hImage, $sCLSID, $tData, $tParams

    ; Захватить экран
    _ScreenCapture_Capture(@MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Инициализировать библиотеку GDI+
    _GDIPlus_Startup()

    ; Загрузить изображение
    $hImage = _GDIPlus_ImageLoadFromFile(@MyDocumentsDir & "\GDIPlus_Image.jpg")

    ; Получить CLSID JPEG-кодера
    $sCLSID = _GDIPlus_EncodersGetCLSID("JPG")

    ; Установить параметры поворота на 90 градусов
    $tData = DllStructCreate("int Data")
    DllStructSetData($tData, "Data", $GDIP_EVTTRANSFORMROTATE90)
    $tParams = _GDIPlus_ParamInit(1)
    _GDIPlus_ParamAdd($tParams, $GDIP_EPGTRANSFORMATION, 1, $GDIP_EPTLONG, DllStructGetPtr($tData, "Data"))

    ; Сохранить изображение с поворотом
    _GDIPlus_ImageSaveToFileEx($hImage, @MyDocumentsDir & "\GDIPlus_Image2.jpg", $sCLSID, $tParams)

    ; Завершить работу с библиотекой GDI+
    _GDIPlus_Shutdown()

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image2.jpg")
EndFunc   ;==>Example
