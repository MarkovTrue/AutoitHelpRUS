Справочник функций

_WinAPI_LookupIconIdFromDirectoryEx

Ищет в данных значков или курсоров значок или курсор, который лучше всего подходит для текущего устройства отображения.

#include <WinAPIIcons.au3>
_WinAPI_LookupIconIdFromDirectoryEx ( $pData [, $bIcon = True [, $iXDesiredPixels = 0 [, $iYDesiredPixels = 0 [, $iFlags = 0]]]] )

Параметры

$pData Данные каталога значков или курсоров. Так как эта функция не проверяет данные ресурса,
она вызывает ошибку защиты (GP) или возвращает неопределённое значение, если presbits не указывает на корректные данные ресурса.
$bIcon [необязательный] Указывает, ищется ли значок или курсор, допустимые значения:
True – функция ищет значок (по умолчанию).
False – функция ищет курсор.
$iXDesiredPixels [необязательный] Требуемая ширина значка или курсора в пиксельях.
Если этот параметр равен нулю (по умолчанию), функция использует значение системной метрики.
$iYDesiredPixels [необязательный] Требуемая высота значка или курсора в пиксельях.
Если этот параметр равен нулю (по умолчанию), функция использует значение системной метрики.
$iFlags [необязательный] Этот параметр может быть одним или несколькими из следующих значений.
$LR_DEFAULTCOLOR (по умолчанию)
$LR_MONOCHROME

Возвращаемое значение

Успех: Возвращает целочисленный идентификатор ресурса для значка или курсора, который лучше всего подходит для текущего устройства отображения.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Каталог значков загружается из файла ресурса с типом ресурса $RT_GROUP_ICON (или $RT_GROUP_CURSOR),
и целочисленное имя ресурса для конкретного загружаемого значка.
_WinAPI_LookupIconIdFromDirectoryEx() возвращает целочисленный идентификатор, являющийся именем ресурса значка, который лучше всего подходит для текущего устройства отображения.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <StaticConstants.au3>
#include <WinAPIIcons.au3>
#include <WinAPIRes.au3>

; Загрузить Resources.dll в память
Local $sDll = _Extras_PathFull('Resources.dll')
Local $hInstance = _WinAPI_LoadLibraryEx($sDll, $LOAD_LIBRARY_AS_DATAFILE)

; Загрузить ресурс RT_GROUP_ICON из библиотеки Resources.dll
Local $hResource = _WinAPI_FindResource($hInstance, $RT_GROUP_ICON, 1)
Local $hData = _WinAPI_LoadResource($hInstance, $hResource)
Local $pData = _WinAPI_LockResource($hData)

; Найти целочисленное имя ресурса для иконки, лучше всего подходящей под размер (48x48)
Local $iIcon = _WinAPI_LookupIconIdFromDirectoryEx($pData, 1, 48, 48)

; Загрузить ресурс RT_ICON из библиотеки Resources.dll
$hResource = _WinAPI_FindResource($hInstance, $RT_ICON, $iIcon)
Local $iSize = _WinAPI_SizeOfResource($hInstance, $hResource)
$hData = _WinAPI_LoadResource($hInstance, $hResource)
$pData = _WinAPI_LockResource($hData)

; Создать иконку из ресурса
Local $hIcon = _WinAPI_CreateIconFromResourceEx($pData, $iSize)

; Выгрузить Resources.dll из памяти
_WinAPI_FreeLibrary($hInstance)

; Создать GUI
GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 128, 128)
GUICtrlCreateIcon('', 0, 40, 40, 48, 48)
GUICtrlSendMsg(-1, $STM_SETIMAGE, 1, $hIcon)
GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = $GUI_EVENT_CLOSE
