_WinAPI_PathIsRoot

Анализирует путь для определения, является ли он корнем каталога.

#include <WinAPIShPath.au3>
_WinAPI_PathIsRoot ( $sFilePath )

Параметры

$sFilePath Путь для проверки.

Возвращаемое значение

True: Возвращает путь является корнем.
False: путь не является корнем.

Примечания

Возвращает True для путей, таких как "\", "X:\" или "\\server\share". Пути, такие как "..\path2" или "\\server\" возвращают False.
