_WinAPI_RegCopyTree

Рекурсивно копирует подключи и значения исходного подключа в ключ назначения.

#include <WinAPIReg.au3>
_WinAPI_RegCopyTree ( $hSrcKey, $sSrcSubKey, $hDestKey )

Параметры

$hSrcKey Дескриптор исходного ключа или один из предопределённых ключей реестра ($HKEY_*).
$sSrcSubKey Подключ, подключи и значения которого должны быть скопированы.
$hDestKey Дескриптор ключа назначения.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Эта функция не дублирует атрибуты безопасности копируемых ключей и значений.
Вместо этого все атрибуты безопасности в ключе назначения – это атрибуты по умолчанию.
