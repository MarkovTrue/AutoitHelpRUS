Справочник функций

_WinAPI_LoadBitmap

Загружает указанный ресурс растрового изображения из исполняемого файла модуля.

#include <WinAPIRes.au3>
_WinAPI_LoadBitmap ( $hInstance, $sBitmap )

Параметры

$hInstance Дескриптор экземпляра модуля, исполняемый файл которого содержит загружаемое растровое изображение.
$sBitmap Имя загружаемого ресурса растрового изображения. Кроме того, может состоять из идентификатора ресурса
в младшем слове и 0 в старшем слове.

Возвращаемое значение

Успех: Возвращает дескриптор указанного растрового изображения.
Ошибка: Возвращает 0.
