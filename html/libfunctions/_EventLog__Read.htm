Справочник функций

_EventLog__Read

Читает запись из журнала событий.

#include <EventLog.au3>
_EventLog__Read ( $hEventLog [, $bRead = True [, $bForward = True [, $iOffset = 0]]] )

Параметры

$hEventLog Дескриптор журнала событий.
$bRead [необязательный] True – чтение выполняется последовательно, начиная с позиции после последнего вызова функции с данным дескриптором.
False – чтение начинается с записи, заданной параметром $iOffset.
$bForward [необязательный] True – журнал читается в хронологическом порядке. False – в обратном хронологическом порядке.
$iOffset [необязательный] Номер записи события, с которой начинается чтение.
Игнорируется, если $bRead равен True.

Возвращаемое значение

Успех: Возвращает массив следующей структуры:
$aArray[0] – True
$aArray[1] – номер записи
$aArray[2] – дата отправки записи
$aArray[3] – время отправки записи
$aArray[4] – дата получения записи для записи в журнал
$aArray[5] – время получения записи для записи в журнал
$aArray[6] – идентификатор события
$aArray[7] – тип события. Возможные значения:
1 – ошибка
2 – предупреждение
4 – информация
8 – аудит успеха
16 – аудит отказа
$aArray[8] – строка типа события
$aArray[9] – категория события
$aArray[10] – источник события
$aArray[11] – имя компьютера
$aArray[12] – имя пользователя
$aArray[13] – описание события
$aArray[14] – массив данных события
Ошибка: Возвращает массив, в котором $Array[0] = False.

Примечания

При успешном выполнении позиция чтения в журнале событий смещается на количество прочитанных записей.
Несмотря на то что можно читать несколько записей, функция возвращает по одной записи за вызов.

См. также

_EventLog__Close, _EventLog__Open

Пример

#include "Extras\HelpFileInternals.au3"

#include <EventLog.au3>
#include <FontConstants.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    Local $hEventLog, $aEvent

    ; Создать GUI
    GUICreate("EventLog", 600, 300)
    _MemoCreate(2, 2, 596, 294, 0)
    GUISetState(@SW_SHOW)

    ; Читаем самую последнюю запись журнала событий.
    $hEventLog = _EventLog__Open("", "Application")
    $aEvent = _EventLog__Read($hEventLog, True, False) ; Читаем последнее событие.
    ; $hEventLog = _EventLog__Open("", "System")
    ; $aEvent = _EventLog__Read($hEventLog)
    ; $aEvent = _EventLog__Read($hEventLog, True, False)
    _MemoWrite("Result ............: " & $aEvent[0])
    _MemoWrite("Record number .....: " & $aEvent[1])
    _MemoWrite("Submitted .........: " & $aEvent[2] & " " & $aEvent[3])
    _MemoWrite("Generated .........: " & $aEvent[4] & " " & $aEvent[5])
    _MemoWrite("Event ID ..........: " & $aEvent[6])
    _MemoWrite("Type ..............: " & $aEvent[8])
    _MemoWrite("Category ..........: " & $aEvent[9])
    _MemoWrite("Source ............: " & $aEvent[10])
    _MemoWrite("Computer ..........: " & $aEvent[11])
    _MemoWrite("Username ..........: " & $aEvent[12])
    _MemoWrite("Description .......: " & $aEvent[13])
    _EventLog__Close($hEventLog)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
