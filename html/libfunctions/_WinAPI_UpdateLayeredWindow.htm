_WinAPI_UpdateLayeredWindow

Обновляет позицию, размер, форму, содержание и прозрачность многоуровневого окна.

#include <WinAPISysWin.au3>
_WinAPI_UpdateLayeredWindow ( $hWnd, $hDestDC, $tPTDest, $tSize, $hSrcDC, $tPTSrce, $iRGB, $tBlend, $iFlags )

Параметры

$hWnd Дескриптор многоуровневого окна. Многоуровневое окно создаётся путём указания $WS_EX_LAYERED при создании окна.
$hDestDC Дескриптор контекста устройства для экрана.
$tPTDest Структура $tagPOINT или указатель на неё, которая указывает новую позицию экрана многоуровневого окна.
Если текущая позиция не меняется, это может быть ноль.
$tSize Структура $tagSIZE или указатель на неё, которая указывает новый размер многоуровневого окна.
Если размер окна не меняется, это может быть 0.
$hSrcDC Дескриптор контекста устройства для поверхности, которая определяет многоуровневое окно.
Этот дескриптор можно получить, вызвав функцию _WinAPI_CreateCompatibleDC().
$tPTSrce Структура $tagPOINT или указатель на неё, которая указывает расположение слоя в контексте устройства.
$iRGB Ключ цвета, используемый при композиции многоуровневого окна.
$tBlend Структура $tagBLENDFUNCTION или указатель на неё, которая указывает значение прозрачности при композиции многоуровневого окна.
$iFlags Этот параметр может быть одним из следующих значений.
$ULW_ALPHA – используйте $tblend как функцию смешивания
$ULW_COLORKEY – используйте $iRGB как цвет прозрачности
$ULW_OPAQUE – нарисовать непрозрачное многоуровневое окно.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

См. также

$tagBLENDFUNCTION, $tagPOINT, $tagSIZE
