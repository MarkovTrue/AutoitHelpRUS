Справочник функций

_WinAPI_SetEnhMetaFileBits

Создаёт метафайл в расширенном формате на основе памяти из указанных данных.

#include <WinAPIGdi.au3>
_WinAPI_SetEnhMetaFileBits ( $pData, $iLength )

Параметры

$pData Указатель на буфер, содержащий данные расширенного метафайла. Чтобы получить данные метафайла, вызовите
функцию _WinAPI_GetEnhMetaFileBits().
$iLength Размер буфера в байтах.

Возвращаемое значение

Успех: Возвращает дескриптор метафайла в расширенном формате на основе памяти.
Ошибка: Возвращает 0.

Примечания

Когда приложению больше не нужен дескриптор расширенного метафайла, его следует удалить вызовом
_WinAPI_DeleteEnhMetaFile().

См. также

_WinAPI_DeleteEnhMetaFile, _WinAPI_GetEnhMetaFileBits
