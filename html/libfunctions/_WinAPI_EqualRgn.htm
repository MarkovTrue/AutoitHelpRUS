_WinAPI_EqualRgn

Проверяет, идентичны ли две указанные области.

#include <WinAPIGdi.au3>
_WinAPI_EqualRgn ( $hRgn1, $hRgn2 )

Параметры

$hRgn1 Дескриптор региона.
$hRgn2 Дескриптор региона.

Возвращаемое значение

Успех: Возвращает True – области равны.
Ошибка: Возвращает False.
