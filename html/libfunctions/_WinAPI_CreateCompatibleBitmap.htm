_WinAPI_CreateCompatibleBitmap

создаёт растровое изображение, совместимое с указанным контекстом устройства.

#include <WinAPIGdi.au3>
_WinAPI_CreateCompatibleBitmap ( $hDC, $iWidth, $iHeight )

Параметры

$hDC идентифицирует контекст устройства
$iWidth Растровое изображение width в pixels.
$iHeight Растровое изображение height в pixels.

Возвращаемое значение

Успех: Возвращает дескриптор на созданное растровое изображение.
Ошибка: Возвращает 0.

Примечания

Когда битмап больше не нужен, удалите его вызовом _WinAPI_DeleteObject().

См. также

_WinAPI_CreateSolidBitmap, _WinAPI_DeleteObject
