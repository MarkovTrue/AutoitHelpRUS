Справочник функций

_IEFormElementCheckBoxSelect

Задаёт значение указанного элемента формы.

#include <IE.au3>
_IEFormElementCheckBoxSelect ( ByRef $oObject, $sString [, $sName = "" [, $iSelect = 1 [, $sMode = "byValue" [, $iFireEvent = 1]]]] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application или Form.
$sString Значение для сравнения с элементом – обработка зависит от $sMode.
$sName [необязательный] Имя или Id флажка(ов)
$iSelect [необязательный] задаёт, должен ли элемент быть отмечен или снят
-1 = Вернуть состояние отмеченности
0 = Снять отметку с элемента
1 = (по умолчанию) Отметить элемент
$sMode [необязательный] задаёт режим поиска
"byValue" = (по умолчанию) значение флажка, который требуется выбрать
"byIndex" = индекс флажка (с нуля), который требуется выбрать
$iFireEvent [необязательный] Запускать ли события OnChange и OnClick после изменения значения:
0 – не запускать события OnChange/OnClick после установки значения
1 – (по умолчанию) запускать события OnChange и OnClick после установки значения

Возвращаемое значение

Успех: Возвращает текущее состояние флажка, если $iSelect = -1; иначе возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) = неверный тип данных
4 ($_IEStatus_InvalidObjectType) = неверный тип объекта
5 ($_IEStatus_InvalidValue) = неверное значение
7 ($_IEStatus_NoMatch) = нет совпадения
@extended: Содержит номер недопустимого параметра.

Примечания

Параметр $iFireEvent имеет значение только если с элементом формы связано событие onChange.

$sName для этой функции необязателен.
Если он опущен, функция работает с коллекцией всех элементов <input type=checkbox> в форме.
Если указан, функция работает с коллекцией элементов <input type=checkbox> с этим именем.

См. также

_IEFormElementGetValue, _IEFormElementOptionSelect, _IEFormElementRadioSelect, _IEFormElementSetValue

Пример

Пример 1

; Открыть браузер с примером формы, получить ссылку на форму, выбрать и
; снять выделение с флажков по значению. Так как $s_Name не задан, действие
; над коллекцией всех элементов <input type=checkbox> в форме
; Примечание: вероятно, потребуется прокрутить страницу, чтобы увидеть изменения

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
For $i = 1 To 3
    _IEFormElementCheckBoxSelect($oForm, "gameBasketball", "", 1, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameFootball", "", 1, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameTennis", "", 1, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameBaseball", "", 1, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameBasketball", "", 0, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameFootball", "", 0, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameTennis", "", 0, "byValue")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, "gameBaseball", "", 0, "byValue")
    Sleep(1000)
Next

_IEQuit($oIE)

Пример 2

; Открыть браузер с примером формы, получить ссылку на форму, выбрать и
; снять выделение с флажков по индексу. Так как $s_Name не задан, действие
; над коллекцией всех элементов <input type=checkbox> в форме

#include <IE.au3>

Local $oIE = _IE_Example("form")
Local $oForm = _IEFormGetObjByName($oIE, "ExampleForm")
For $i = 1 To 3
    _IEFormElementCheckBoxSelect($oForm, 3, "", 1, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 2, "", 1, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 1, "", 1, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 0, "", 1, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 3, "", 0, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 2, "", 0, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 1, "", 0, "byIndex")
    Sleep(1000)
    _IEFormElementCheckBoxSelect($oForm, 0, "", 0, "byIndex")
    Sleep(1000)
Next

_IEQuit($oIE)
