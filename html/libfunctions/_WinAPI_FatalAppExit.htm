Справочник функций

_WinAPI_FatalAppExit

Отображает диалоговое окно сообщения и завершает приложение.

#include <WinAPIProc.au3>
_WinAPI_FatalAppExit ( $sMessage )

Параметры

$sMessage Задаёт строку, которая отображается в окне сообщения.

Возвращаемое значение

Отсутствует.

Примечания

FatalAppExit вызывается только когда приложение не может завершиться другим способом.
FatalAppExit может не всегда освобождать память приложения или закрывать его файлы и может вызвать общий сбой системы.
При неожиданной ошибке приложение должно завершаться, освобождая всю память и выходя из основного цикла сообщений.
