Справочник функций

_WinAPI_SetKeyboardState

Копирует массив из 256 байтов состояний клавиш клавиатуры в таблицу состояния ввода с клавиатуры вызывающего процесса.

#include <WinAPISys.au3>
_WinAPI_SetKeyboardState ( $tState )

Параметры

$tState Структура "byte[256]", которая содержит состояния клавиш клавиатуры.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция изменяет состояние ввода вызывающего процесса, а не глобальное состояние ввода системы.
Вы не можете использовать эту функцию для установки индикаторных огней NUM LOCK, CAPS LOCK или SCROLL LOCK на
клавиатуре.
