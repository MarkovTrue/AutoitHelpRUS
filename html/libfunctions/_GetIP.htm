Справочник функций

_GetIP

Получает публичный IP-адрес сети или компьютера.

#include <Inet.au3>
_GetIP ( )

Возвращаемое значение

Успех: Возвращает публичный IP-адрес. (См. раздел Примечания.).
Ошибка: Возвращает -1 и устанавливает @error в ненулевое значение.

Примечания

Возвращает только публичный IP-адрес компьютера или сети (то есть публичный IP на WAN-интерфейсе вашего NAT-маршрутизатора). Адрес определяется обращением к внешнему сервису обнаружения IP. Используются сайты https://www.ipify.org, http://checkip.dyndns.org, http://www.myexternalip.com/raw и http://www.bot.whatismyipaddress.com.

Чтобы не перегружать эти сервисы, между вызовами выдерживается 5-минутный таймер. Если _GetIP() вызвана повторно в течение этих 5 минут, возвращается прежний IP, а @extended получает значение 1.

Пример

#include <Inet.au3>
#include <MsgBoxConstants.au3>

Local $sPublicIP = _GetIP()
MsgBox($MB_SYSTEMMODAL, "", "Your external IP address is: " & $sPublicIP)
