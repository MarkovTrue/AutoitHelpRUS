Справочник функций

_WinAPI_EncryptFile

Шифрует файл или каталог.

#include <WinAPIFiles.au3>
_WinAPI_EncryptFile ( $sFilePath )

Параметры

$sFilePath Имя шифруемого файла или каталога.
Если $sFilePath указывает на файл "только для чтения", функция завершается ошибкой ERROR_FILE_READ_ONLY (6009).
Если $sFilePath указывает на каталог, содержащий файл "только для чтения", функция завершается успешно, но каталог не шифруется.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция _WinAPI_EncryptFile() требует монопольного доступа к файлу и завершится ошибкой, если файл используется другим процессом.

Если файл уже зашифрован, функция возвращает ненулевое значение, означающее успех.
Если файл сжат, функция распакует его перед шифрованием.

Для расшифровки используйте _WinAPI_DecryptFile().

Пример

#include <FileConstants.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIFiles.au3>

Local $sFile = FileOpenDialog('Select File', @ScriptDir, 'All Files (*.*)', BitOR($FD_FILEMUSTEXIST, $FD_PATHMUSTEXIST))
If @error Then Exit

Switch _WinAPI_FileEncryptionStatus($sFile)
    Case $FILE_ENCRYPTABLE
        If _WinAPI_EncryptFile($sFile) Then
            MsgBox(($MB_ICONINFORMATION + $MB_SYSTEMMODAL), 'Encryption File', 'The file encrypted is successfully.')
        Else
            MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Encryption File', 'Unable to encrypt file.')
        EndIf
    Case $FILE_IS_ENCRYPTED
        If MsgBox(($MB_YESNO + $MB_ICONQUESTION + $MB_SYSTEMMODAL), 'Encryption File', 'The file is already encrypted.' & @CRLF & @CRLF & 'Decrypt?') = 6 Then
            If _WinAPI_DecryptFile($sFile) Then
                MsgBox(($MB_ICONINFORMATION + $MB_SYSTEMMODAL), 'Encryption File', 'The file decrypted is successfully.')
            Else
                MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Encryption File', 'Unable to decrypt file.')
            EndIf
        EndIf
    Case Else
        MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Encryption File', 'Unable to perform operation.')
EndSwitch
