_WinAPI_GetThemeSysBool

Получает логическое значение системной метрики.

#include <WinAPITheme.au3>
_WinAPI_GetThemeSysBool ( $hTheme, $iBoolID )

Параметры

$hTheme Дескриптор данных темы.
$iBoolID Значение, которое задаёт желаемую логическую системную метрику. Может быть следующим значением.
$TMT_FLATMENUS

Возвращаемое значение

Возвращает значение желаемой системной метрики.

Примечания

Если дескриптор данных темы не равен 0, функция возвращает желаемый BOOL из раздела SysMetrics визуального стиля,
в противном случае функция возвращает значение указанного логического значения системы.
