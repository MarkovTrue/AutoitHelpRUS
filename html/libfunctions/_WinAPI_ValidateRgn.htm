_WinAPI_ValidateRgn

Удаляет область из текущей области обновления указанного окна.

#include <WinAPIGdi.au3>
_WinAPI_ValidateRgn ( $hWnd [, $hRgn = 0] )

Параметры

$hWnd Дескриптор окна.
$hRgn [необязательный] Дескриптор области, которая определяет область, подлежащую удалению из области обновления. Если этот параметр равен 0 (по умолчанию),
вся клиентская область удаляется.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
