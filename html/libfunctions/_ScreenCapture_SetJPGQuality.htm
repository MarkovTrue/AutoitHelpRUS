Справочник функций

_ScreenCapture_SetJPGQuality

Задаёт уровень качества для снимков экрана в формате JPEG.

#include <ScreenCapture.au3>
_ScreenCapture_SetJPGQuality ( $iQuality )

Параметры

$iQuality Уровень качества изображения. Должен быть в диапазоне от 0 до 100.

Возвращаемое значение

Отсутствует.

Примечания

Если значение не задано, по умолчанию используется качество 100.
