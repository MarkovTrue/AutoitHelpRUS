_GUICtrlComboBoxEx_SetEditSel

Выбирает символы в поле ввода выпадающего списка.

#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetEditSel ( $hWnd, $iStart, $iStop )

Параметры

$hWnd Дескриптор элемента управления.
$iStart Начальная позиция выделения.
$iStop Конечная позиция.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Ложь, если сообщение отправлено выпадающему списку со стилем $CBS_DROPDOWN или $CBS_DROPDOWNLIST.

Примечания

Позиции отсчитываются от 0. Первый символ поля редактирования имеет позицию 0.
Если $iStop = –1, выделяется весь текст от стартовой позиции до последнего символа поля редактирования.
Если $iStart = –1, выделение снимается.

См. также

_GUICtrlComboBox_GetEditSel, _GUICtrlComboBoxEx_GetEditSel

Пример

#include "Extras\HelpFileInternals.au3"

#include <GuiComboBox.au3>
#include <GUIConstantsEx.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("ComboBox Get/Set Edit Sel (v" & @AutoItVersion & ")", 400, 296)
    Local $idCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    _MemoCreate(2, 32, 396, 266, 0)
    GUISetState(@SW_SHOW)

    ; Добавить файлы
    _GUICtrlComboBox_BeginUpdate($idCombo)
    _GUICtrlComboBox_AddDir($idCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($idCombo)

    ; Выбор элемента
    _GUICtrlComboBox_SetCurSel($idCombo, 2)

    ; Задать выделение Edit
    _GUICtrlComboBox_SetEditSel($idCombo, 0, 4)

    ; Получить выделение в edit
    Local $aSel = _GUICtrlComboBox_GetEditSel($idCombo)
    _MemoWrite(StringFormat("Edit Sel: %d - %d", $aSel[0], $aSel[1]))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
