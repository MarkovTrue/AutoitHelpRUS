Справочник функций

_INetExplorerCapable

Преобразует строку в формат, совместимый с Internet Explorer.

#include <Inet.au3>
_INetExplorerCapable ( $sIEString )

Параметры

$sIEString Строка для преобразования.

Возвращаемое значение

Успех: Возвращает преобразованная строка.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.

Примечания

Преобразует символы, несовместимые с IE, в '%' и их шестнадцатеричное значение.

Пример

#include <Inet.au3>
#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "", _INetExplorerCapable(InputBox("", "Enter a string to convert into browser readable text")))
