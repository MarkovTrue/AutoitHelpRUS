Справочник функций

_Date_Time_GetTimeZoneInformation

Получает текущие настройки часового пояса.

#include <Date.au3>
_Date_Time_GetTimeZoneInformation ( )

Возвращаемое значение

Успех: Возвращает массив следующего вида:
$aArray[0] – состояние летнего времени:
-1 – ошибка
0 – летнее время в текущем часовом поясе не используется
1 – действует стандартное время
2 – действует летнее время
$aArray[1] – текущее смещение для перевода местного времени;
смещение – разница в минутах между UTC и местным временем;
формула: UTC = местное время + смещение
$aArray[2] – описание стандартного времени
$aArray[3] – структура $tagSYSTEMTIME с датой и местным временем перехода с летнего на стандартное
$aArray[4] – смещение, используемое для перевода местного времени в период стандартного времени;
прибавляется к Bias; в большинстве часовых поясов равно нулю
$aArray[5] – описание летнего времени
$aArray[6] – структура $tagSYSTEMTIME с датой и местным временем перехода со стандартного на летнее
$aArray[7] – смещение для перевода местного времени в период летнего времени;
прибавляется к Bias; в большинстве часовых поясов равно –60.
Ошибка: Устанавливает @error ≠ 0.

Примечания

В структуре $tagSYSTEMTIME поля wHour и wMinute задают время перехода, wDayOfWeek – день недели, wDay – номер вхождения этого дня недели в месяц (от 1 до 5, где 5 означает последнее вхождение, если данный день встречается менее 5 раз).

Пример: 02:00 первого воскресенья апреля: wHour = 2, wMonth = 4, wDayOfWeek = 0, wDay = 1. 02:00 последнего четверга октября: wHour = 2, wMonth = 10, wDayOfWeek = 4, wDay = 5.

См. также

$tagSYSTEMTIME, _Date_Time_SetTimeZoneInformation

Пример

;~ #RequireAdmin
; Windows API "SetTimeZonePrivilege" требует привилегии "SeTimeZonePrivilege", поэтому необходимо указать #RequireAdmin.

#include "Extras\HelpFileInternals.au3"

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <WinAPIError.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("Date_Time Get/Set Time ZoneInformation (v" & @AutoItVersion & ")", 460, 460)
    _MemoCreate(2, 2, 456, 456, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Показать сведения о текущем часовом поясе
    Local $aOld = _Date_Time_GetTimeZoneInformation()
    ShowTimeZoneInformation($aOld, "Current")

    ; Обновить сведения о часовом поясе (только имена)
    If Not _Date_Time_SetTimeZoneInformation($aOld[1], "A3L CST", $aOld[3], $aOld[4], "A3L CDT", $aOld[6], $aOld[7]) Then
        MsgBox($MB_SYSTEMMODAL, "Error", "System timezone cannot be SET @error=" & @error & @CRLF & @CRLF & _WinAPI_GetErrorMessage(@extended))
        Exit
    EndIf

    ; Отображаем новые данные часового пояса.
    Local $aNew = _Date_Time_GetTimeZoneInformation()
    ShowTimeZoneInformation($aNew, "New")

    ; Сбросить исходные сведения о часовом поясе
    _Date_Time_SetTimeZoneInformation($aOld[1], $aOld[2], $aOld[3], $aOld[4], $aOld[5], $aOld[6], $aOld[7])

    ; Показать сведения о текущем часовом поясе
    $aOld = _Date_Time_GetTimeZoneInformation()
    ShowTimeZoneInformation($aOld, "Reset")

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Показать сведения о часовом поясе
Func ShowTimeZoneInformation(ByRef $aInfo, $sComment)
    _MemoWrite("******************* " & $sComment & " *******************")
    Local $sResult = "TIME_ZONE_ID_INVALID"
    Switch $aInfo[0]
        Case 0
            $sResult = "TIME_ZONE_ID_UNKNOWN"
        Case 1
            $sResult = "TIME_ZONE_ID_STANDARD"
        Case 2
            $sResult = "TIME_ZONE_ID_DAYLIGHT"
    EndSwitch
    _MemoWrite("Result ............: " & $sResult)
    _MemoWrite("Current bias ......: " & $aInfo[1])
    _MemoWrite("Standard name .....: " & $aInfo[2])
    _MemoWrite("Standard date/time : " & _Date_Time_SystemTimeToDateTimeStr($aInfo[3]))
    _MemoWrite("Standard bias......: " & $aInfo[4])
    _MemoWrite("Daylight name .....: " & $aInfo[5])
    _MemoWrite("Daylight date/time : " & _Date_Time_SystemTimeToDateTimeStr($aInfo[6]))
    _MemoWrite("Daylight bias......: " & $aInfo[7])
EndFunc   ;==>ShowTimeZoneInformation
