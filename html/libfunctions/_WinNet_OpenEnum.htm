Справочник функций

_WinNet_OpenEnum

Начинает перечисление сетевых ресурсов или существующих подключений.

#include <WinNet.au3>
_WinNet_OpenEnum ( $iScope, $iType, $iUsage, $tResource, ByRef $hEnum )

Параметры

$iScope Область перечисления:     0 – Перечислить все текущие подключённые ресурсы     1 – Перечислить все ресурсы в сети     2 – Перечислить все запомненные (постоянные) подключения     3 – Перечислить только ресурсы в контексте сети вызывающей стороны.
$iType Типы ресурсов:     0 – Все ресурсы     1 – Дисковые ресурсы     2 – Принтерные ресурсы.
$iUsage Типы использования ресурсов:     0 – Все ресурсы     1 – Все подключаемые ресурсы     2 – Все контейнерные ресурсы     4 – Принудительно отказывает функции, если пользователь не аутентифицирован.
$tResource структура $tagNETRESOURCE, которая указывает контейнер для перечисления или указатель на неё. Если $iScope не равен 1, это должно быть 0. Если 0, предполагается корень сети.
$hEnum При возврате дескриптор, который можно использовать при вызовах _WinNet_EnumResource().

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
