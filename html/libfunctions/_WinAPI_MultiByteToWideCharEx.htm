_WinAPI_MultiByteToWideCharEx

Отображает строку символов на строку расширенных символов (Unicode).

#include <WinAPIConv.au3>
_WinAPI_MultiByteToWideCharEx ( $sText, $pText [, $iCodePage = $CP_ACP [, $iFlags = 0]] )

Параметры

$sText Текст для преобразования.
$pText Указатель на байтовую структуру, в которой будет сохранена преобразованная строка.
$iCodePage [необязательный] Задаёт кодовую страницу, используемую для выполнения преобразования:
$CP_ACP (0) – текущая кодовая страница Windows ANSI системы
$CP_OEM (1) – текущая кодовая страница OEM системы
$CP_MACCP (2) – текущая кодовая страница Macintosh системы.
$iFlags [необязательный] Флаги, указывающие на перевод в предварительно составленные или составные широкие символы:
$MB_PRECOMPOSED – всегда использовать предварительно составленные символы
$MB_COMPOSITE – всегда использовать составные символы
$MB_USEGLYPHCHARS – использовать глифовые символы вместо управляющих символов
$MB_ERR_INVALID_CHARS – завершить работу, если встречен недопустимый входной символ.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Байтовая структура должна быть по крайней мере в два раза больше длины $sText.
