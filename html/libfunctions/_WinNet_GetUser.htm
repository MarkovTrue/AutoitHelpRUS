Справочник функций

_WinNet_GetUser

Получает имя пользователя по умолчанию или имя пользователя, используемое для подключения.

#include <WinNet.au3>
_WinNet_GetUser ( $sName )

Параметры

$sName Либо имя локального устройства, переадресованного к сетевому ресурсу, либо удалённое имя сетевого ресурса, к которому подключение было создано без переадресации локального устройства. Если пусто, система возвращает имя текущего пользователя процесса.

Возвращаемое значение

Успех: Возвращает имя пользователя.
Ошибка: Устанавливает @error ≠ 0.
