Справочник функций

_WinAPI_ShellNotifyIconGetRect

Получает координаты ограничивающего прямоугольника значка уведомления на экране.

#include <WinAPIShellEx.au3>
_WinAPI_ShellNotifyIconGetRect ( $hWnd, $iID [, $tGUID = 0] )

Параметры

$hWnd Дескриптор родительского окна, используемого функцией обратного вызова уведомления. Дополнительные сведения см. в члене "hWnd" структуры $tagNOTIFYICONDATA.
$iID Определённый приложением идентификатор значка уведомления. Несколько значков могут быть связаны с одним $hWnd, каждый со своим $iID.
$tGUID [необязательный] Структура $tagGUID, которая идентифицирует значок.

Возвращаемое значение

Успех: Возвращает структура $tagRECT, содержащая координаты значка.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Значок может быть идентифицирован через $hWnd плюс $iID или $tGUID отдельно (рекомендуется).
Эта функция требует Windows 7 или более поздней версии.

Дополнительно

Искать Shell_NotifyIconGetRect в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIShellEx.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.1 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows 7 or later.')
    Exit
EndIf

Local $tRECT = _WinAPI_ShellNotifyIconGetRect(WinGetHandle(AutoItWinGetTitle()), 1)

Local $aPos
If Not @error Then
    $aPos = _WinAPI_GetPosFromRect($tRECT)
    MouseMove($aPos[0] + 12, $aPos[1] + 12)
    MouseClick('left')
    While 1
        Sleep(1000)
    WEnd
EndIf
