Справочник функций

_WinAPI_CreateFileEx

создаёт или открывает файл или устройство ввода-вывода.

#include <WinAPIFiles.au3>
_WinAPI_CreateFileEx ( $sFilePath, $iCreation [, $iAccess = 0 [, $iShare = 0 [, $iFlagsAndAttributes = $SECURITY_ANONYMOUS [, $tSecurity = 0 [, $hTemplate = 0]]]]] )

Параметры

$sFilePath Имя.
$iCreation Координаты.
$iAccess Необязательный параметр.
$iShare Файловый дескриптор.
$iFlagsAndAttributes Координаты.
$tSecurity Файловый дескриптор.
$hTemplate Файловый дескриптор.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: Ошибка.

Примечания

Когда приложение закончило работу с дескриптором объекта, возвращённым этой функцией, закройте его вызовом _WinAPI_CloseHandle().
Это не только освобождает системные ресурсы, но и влияет на параметры совместного доступа к файлу, удаление файлов, помеченных как FILE_FLAG_DELETE_ON_CLOSE, и тому подобное.
