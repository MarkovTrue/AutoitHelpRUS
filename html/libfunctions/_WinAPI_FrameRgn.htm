_WinAPI_FrameRgn

Рисует границу вокруг указанной области указанной кистью.

#include <WinAPIGdi.au3>
_WinAPI_FrameRgn ( $hDC, $hRgn, $hBrush, $iWidth, $iHeight )

Параметры

$hDC Дескриптор контекста устройства.
$hRgn Дескриптор области, вокруг которой рисуется граница.
Координаты области интерпретируются как логические единицы.
$hBrush Дескриптор кисти, используемой для отрисовки границы.
$iWidth Ширина вертикальных штрихов кисти в логических единицах.
$iHeight Высота горизонтальных штрихов кисти в логических единицах.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
