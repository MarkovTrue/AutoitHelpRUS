_WinAPI_SetLastError

Задаёт код последней ошибки для вызывающего потока.

#include <WinAPIError.au3>
_WinAPI_SetLastError ( $iErrorCode )

Параметры

$iErrorCode Код последней ошибки для потока.

Возвращаемое значение

Отсутствует.

Примечания

Код последней ошибки хранится в локальном хранилище потока, чтобы несколько потоков не перезаписывали значения друг друга.
