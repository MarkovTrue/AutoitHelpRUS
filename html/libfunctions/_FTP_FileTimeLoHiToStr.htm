Справочник функций

_FTP_FileTimeLoHiToStr

Преобразует LOWORD и HIWORD времени файла в строку.

#include <FTPEx.au3>
_FTP_FileTimeLoHiToStr ( $iLoDWORD, $iHiDWORD [, $iFmt = 0] )

Параметры

$iLoDWORD Младшее слово (LOWORD) времени файла.
$iHiDWORD Старшее слово (HIWORD) времени файла.
$iFmt [необязательный]
0 – возвращает mm/dd/yyyy hh:mm:ss (по умолчанию)
1 – возвращает yyyy/mm/dd hh:mm:ss

Возвращаемое значение

Успех: Возвращает дату и время в формате, заданном параметром $iFmt.
Ошибка: Возвращает пустую строку.
