_WinAPI_MAKELCID

Строит идентификатор локали из идентификатора языка и идентификатора сортировки.

#include <WinAPIConv.au3>
_WinAPI_MAKELCID ( $iLngID, $iSortID )

Параметры

$iLngID Идентификатор языка.
$iSortID Идентификатор сортировки.

Возвращаемое значение

Возвращает идентификатор локали

Примечания

Константы для $iSortID определены в WinAPILangConstants.au3.

См. также

_WinAPI_MAKELANGID
