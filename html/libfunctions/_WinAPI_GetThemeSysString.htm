_WinAPI_GetThemeSysString

Получает значение системной строки.

#include <WinAPITheme.au3>
_WinAPI_GetThemeSysString ( $hTheme, $iStringID )

Параметры

$hTheme Дескриптор данных темы.
$iStringID Значение, которое задаёт системную строку. Может быть одним из следующих значений.
$TMT_CSSNAME
$TMT_XMLNAME

Возвращаемое значение

Успех: Возвращает значение строки.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Если дескриптор данных темы не равен 0, функция возвращает желаемую строку из раздела SysMetrics визуального стиля,
в противном случае функция возвращает значение глобальной системной метрики.
