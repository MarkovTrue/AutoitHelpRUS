Справочник функций

_WinAPI_CreateNullRgn

создаёт пустую область.

#include <WinAPIGdi.au3>
_WinAPI_CreateNullRgn ( )

Возвращаемое значение

Успех: Возвращает дескриптор региона.
Ошибка: Возвращает 0.

Примечания

Когда HRGN-объект больше не нужен, удалите его вызовом _WinAPI_DeleteObject().

_WinAPI_DeleteObject
