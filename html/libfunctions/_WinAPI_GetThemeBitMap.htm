_WinAPI_GetThemeBitmap

Получает растровое изображение, связанное с конкретной темой, частью, состоянием и свойством.

#include <WinAPITheme.au3>
_WinAPI_GetThemeBitmap ( $hTheme, $iPartID, $iStateID, $iPropID [, $iFlag = $GBF_DIRECT] )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит растровое изображение.
$iStateID Состояние части.
$iPropID Свойство для получения. Установите этот параметр на 0, чтобы автоматически выбрать первое доступное растровое изображение
для этой части и состояния, или используйте одно из следующих значений.
$TMT_DIBDATA
$TMT_GLYPHDIBDATA
$TMT_HBITMAP
$iFlag [необязательный] этот параметр может быть одним из следующих значений.
$GBF_DIRECT (по умолчанию)
$GBF_COPY
$GBF_VALIDBITS

Возвращаемое значение

Успех: Возвращает дескриптор запрошенного растрового изображения.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Если $iFlag установлен на $GBF_COPY, освободите растровое изображение, возвращённое этой функцией, когда оно больше не нужно, вызвав
_WinAPI_DeleteObject().

Эта функция требует Windows Vista или более позднюю версию.

См. также

_WinAPI_DeleteObject
