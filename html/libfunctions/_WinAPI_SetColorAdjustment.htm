_WinAPI_SetColorAdjustment

Устанавливает коррекцию цвета для контекста устройства (DC).

#include <WinAPIGdi.au3>
_WinAPI_SetColorAdjustment ( $hDC, $tAdjustment )

Параметры

$hDC Дескриптор контекста устройства.
$tAdjustment Структура $tagCOLORADJUSTMENT, содержащая значения коррекции цвета.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Коррекция цвета используется для корректировки входящего цвета исходного растра при вызовах функций _WinAPI_StretchBlt()
и _WinAPI_StretchDIBits() когда установлен режим $HALFTONE ($STRETCH_HALFTONE).
