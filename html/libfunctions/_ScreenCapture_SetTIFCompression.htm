Справочник функций

_ScreenCapture_SetTIFCompression

Задаёт способ сжатия для снимков экрана в формате TIFF.

#include <ScreenCapture.au3>
_ScreenCapture_SetTIFCompression ( $iCompress )

Параметры

$iCompress Тип сжатия изображения:
0 – сжатие по умолчанию
1 – без сжатия
2 – сжатие CCITT3
3 – сжатие CCITT4
4 – сжатие LZW
5 – сжатие RLE

Возвращаемое значение

Отсутствует.

Примечания

Если значение не задано, по умолчанию используется LZW-сжатие.
