Справочник функций

_WinAPI_GetProfilesDirectory

Возвращает путь к корневому каталогу, в котором хранятся пользовательские профили.

#include <WinAPIFiles.au3>
_WinAPI_GetProfilesDirectory ( )

Возвращаемое значение

Успех: Путь к каталогу.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
