Справочник функций

_MemGlobalLock

Блокирует объект глобальной памяти и возвращает указатель на первый байт блока памяти объекта.

#include <Memory.au3>
_MemGlobalLock ( $hMemory )

Параметры

$hMemory Дескриптор объекта глобальной памяти.

Возвращаемое значение

Успех: Возвращает указатель на первый байт блока памяти.
Ошибка: Возвращает 0.

См. также

_MemGlobalAlloc, _MemGlobalUnlock
