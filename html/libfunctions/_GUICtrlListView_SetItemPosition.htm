_GUICtrlListView_SetItemPosition

Перемещает элемент в указанную позицию.

#include <GuiListView.au3>
_GUICtrlListView_SetItemPosition ( $hWnd, $iIndex, $iCX, $iCY )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iIndex Индекс элемента (нумерация с 0).
$iCX Новая X-координата левого верхнего угла элемента в координатах вида.
$iCY Новая Y-координата левого верхнего угла элемента в координатах вида.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

В представлении значков или мелких значков возвращает общее количество элементов.

См. также

_GUICtrlListView_Arrange, _GUICtrlListView_GetItemPosition, _GUICtrlListView_SetItemPosition32

Пример

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    GUICreate("ListView Get Item Position (v" & @AutoItVersion & ")", 400, 300)
    Local $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Установить формат ANSI
;~ _GUICtrlListView_SetUnicodeFormat($idListview, False)

    ; Добавить столбцы
    _GUICtrlListView_AddColumn($idListview, "Items", 100)

    ; Добавить элементы
    _GUICtrlListView_AddItem($idListview, "Item 0")
    _GUICtrlListView_AddItem($idListview, "Item 1")
    _GUICtrlListView_AddItem($idListview, "Item 2")

    ; Получить позицию элемента 1
    Local $aPos = _GUICtrlListView_GetItemPosition($idListview, 1)
    MsgBox($MB_SYSTEMMODAL, "Information", StringFormat("Item 1 Position : [%d, %d]", $aPos[0], $aPos[1]))

    ; Установить вид – значки
    _GUICtrlListView_SetView($idListview, 3)
    MsgBox($MB_SYSTEMMODAL, "Information", "Moving item 1")
    _GUICtrlListView_SetItemPosition($idListview, 1, 100, 100)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
