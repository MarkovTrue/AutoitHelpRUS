_WinAPI_UrlCompare

Выполняет сравнение двух строк URL с учётом регистра.

#include <WinAPIShPath.au3>
_WinAPI_UrlCompare ( $sUrl1, $sUrl2 [, $bIgnoreSlash = False] )

Параметры

$sUrl1 Первый URL.
$sUrl2 Второй URL.
$bIgnoreSlash [необязательный] Указывает, следует ли игнорировать замыкающий символ '/' в одном или обоих URL, допустимые значения:
True – функция игнорирует замыкающие символы.
False – функция учитывает замыкающие символы (по умолчанию).

Возвращаемое значение

Возвращает 0 – URL равны.
    >0 – первый URL больше второго URL.
    <0 – первый URL меньше второго URL.

Примечания

Для лучших результатов вы должны сначала привести URL к канонической форме с помощью _WinAPI_UrlCanonicalize(). Затем сравните
канонические URL с помощью _WinAPI_UrlCompare().

Дополнительно

Искать UrlCompare в библиотеке MSDN

Пример

#include <WinAPIShPath.au3>

Local $sUrl1 = 'http://xyz/abc/'
Local $sUrl2 = 'http://xyz/abc'

ConsoleWrite('URLs comparison result: ' & _WinAPI_UrlCompare($sUrl1, $sUrl2) & @CRLF)
