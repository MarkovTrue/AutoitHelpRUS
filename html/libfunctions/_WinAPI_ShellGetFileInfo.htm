_WinAPI_ShellGetFileInfo

Получает информацию об объекте в файловой системе.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetFileInfo ( $sFilePath, $iFlags, $iAttributes, ByRef $tSHFILEINFO )

Параметры

$sFilePath Строка, содержащая абсолютный или относительный путь и имя файла. Эта строка может использовать
короткие (форма 8.3) или длинные имена файлов.

Если параметр $iFlags включает флаг $SHGFI_PIDL, этот параметр должен быть адресом
структуры ITEMIDLIST (PIDL), содержащей список идентификаторов элементов, которые однозначно определяют
файл в пространстве имён оболочки. Указатель на список идентификаторов элементов (PIDL) должен быть полностью
квалифицированным PIDL. Относительные PIDL не допускаются.

Если параметр $iFlags включает флаг $SHGFI_USEFILEATTRIBUTES, этот параметр не обязан
быть действительным именем файла. Функция продолжит работу так, как если бы файл существовал с указанным именем и
с атрибутами файла, переданными в параметре $iAttributes. Это позволяет получить информацию
о типе файла, передав просто расширение для $sFilePath и передав $FILE_ATTRIBUTE_NORMAL
в $iAttributes.
$iFlags Флаги, определяющие информацию о файле, которую нужно получить. Этот параметр может быть комбинацией
следующих значений.
$SHGFI_ATTR_SPECIFIED
$SHGFI_ATTRIBUTES
$SHGFI_DISPLAYNAME
$SHGFI_EXETYPE
$SHGFI_ICON
$SHGFI_ICONLOCATION
$SHGFI_LARGEICON
$SHGFI_LINKOVERLAY
$SHGFI_OPENICON
$SHGFI_OVERLAYINDEX
$SHGFI_PIDL
$SHGFI_SELECTED
$SHGFI_SHELLICONSIZE
$SHGFI_SMALLICON
$SHGFI_SYSICONINDEX
$SHGFI_TYPENAME
$SHGFI_USEFILEATTRIBUTES
$iAttributes Комбинация одного или нескольких флагов атрибутов файла ($FILE_ATTRIBUTE_*).
$tSHFILEINFO Структура $tagSHFILEINFO для получения информации о файле. Эта структура должна быть создана перед вызовом функции.

Возвращаемое значение

Успех: Если $iFlags содержит флаг $SHGFI_EXETYPE, возвращает тип исполняемого файла. Если $iFlags содержит
флаг $SHGFI_SYSICONINDEX , возвращает дескриптор системного списка изображений.
Ошибка: Возвращает 0.

Примечания

Если эта функция возвращает дескриптор значка в элементе "hIcon" структуры $tagSHFILEINFO, вы несёте ответственность
за его освобождение с помощью _WinAPI_DestroyIcon() когда он больше не нужен.
