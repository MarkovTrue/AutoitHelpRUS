Справочник функций

_WinAPI_GetTickCount

Получает количество миллисекунд, прошедших с момента запуска системы.

#include <WinAPISys.au3>
_WinAPI_GetTickCount ( )

Возвращаемое значение

Возвращает число миллисекунд, прошедших с момента запуска системы.

Примечания

Время будет переноситься на ноль, если система работает непрерывно в течение 49,7 дней. Вы должны проверить условие
переполнения при сравнении времён. Чтобы получить время, прошедшее с момента запуска компьютера, получите счётчик
System Up Time в данных производительности в ключе реестра HKEY_PERFORMANCE_DATA.

Дополнительно

Искать GetTickCount в библиотеке MSDN

Пример

#include <WinAPIConv.au3>
#include <WinAPISys.au3>

ConsoleWrite(_WinAPI_StrFromTimeInterval(_WinAPI_GetTickCount()) & @CRLF)
