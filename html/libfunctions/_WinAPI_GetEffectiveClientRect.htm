Справочник функций

_WinAPI_GetEffectiveClientRect

Вычисляет размеры прямоугольника в клиентской области, содержащего все указанные элементы управления.

#include <WinAPISys.au3>
_WinAPI_GetEffectiveClientRect ( $hWnd, $aCtrl [, $iStart = 0 [, $iEnd = -1]] )

Параметры

$hWnd Дескриптор окна, клиентскую область которого требуется проверить.
$aCtrl Массив, содержащий дескрипторы или идентификаторы элементов управления, которые должны быть включены в расчёт клиентской области. Также может быть единственный дескриптор или идентификатор элемента управления.
$iStart [необязательный] Индекс элемента массива, содержащего первый элемент управления.
$iEnd [необязательный] Индекс элемента массива, содержащего последний элемент управления.

Возвращаемое значение

Успех: Возвращает структура $tagRECT, содержащая прямоугольник эффективной клиентской области.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Если элемент управления в массиве видим или будет видим при появлении его родителя, его прямоугольник вычитается из эффективного прямоугольника клиентской области.

Если все указанные элементы управления отсутствуют, функция возвращает клиентскую область родительского окна.

Пример

#include <GUIConstantsEx.au3>
#include <GUIStatusBar.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPISys.au3>

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 600, 400)
Local $idMenu = GUICtrlCreateMenu("&File")
Local $idMni_Exit = GUICtrlCreateMenuItem("E&xit", $idMenu)
Local $aCtrl[2]
$aCtrl[0] = _GUICtrlStatusBar_Create($hForm)
Local $idLV = GUICtrlCreateListView('', 0, 0, 600, 200, -1, 0)
$aCtrl[0] = $idLV

; Вычисляет эффективную клиентскую область (исключая элементы управления Menu, ListView и StatusBar)
Local $tRECT = _WinAPI_GetEffectiveClientRect($hForm, $aCtrl)
Local $aPos = _WinAPI_GetPosFromRect($tRECT)
GUICtrlCreateLabel($aPos[2] & 'x' & $aPos[3], $aPos[0], $aPos[1], $aPos[2], $aPos[3], BitOR($SS_CENTER, $SS_CENTERIMAGE))
GUICtrlSetFont(-1, 25, 400, 0, 'Tahoma')
GUICtrlSetBkColor(-1, 0xFFD0D0)

; Показать GUI
GUISetState(@SW_SHOW)

While 1
    Switch GUIGetMsg()
        Case $GUI_EVENT_CLOSE, $idMni_Exit
            ExitLoop
    EndSwitch
WEnd
