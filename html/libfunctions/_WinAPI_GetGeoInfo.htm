_WinAPI_GetGeoInfo

Получает информацию об указанном географическом местоположении.

#include <WinAPILocale.au3>
_WinAPI_GetGeoInfo ( $iGEOID, $iType [, $iLanguage = 0] )

Параметры

$iGEOID Идентификатор географического местоположения (GEOID), для которого требуется получить информацию.
$iType Тип информации для получения. Может быть одним из следующих значений.
$GEO_NATION
$GEO_LATITUDE
$GEO_LONGITUDE
$GEO_ISO2
$GEO_ISO3
$GEO_LCID
$GEO_FRIENDLYNAME
$GEO_OFFICIALNAME
$GEO_TIMEZONES
$GEO_OFFICIALLANGUAGES

Windows Vista или более поздняя версия
$GEO_RFC1766

Windows 8 или более поздняя версия
$GEO_ISO_UN_NUMBER
$GEO_PARENT
$iLanguage [необязательный] Идентификатор языка, используемый с географическим местоположением.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Дополнительно

Искать GetGeoInfo в библиотеке MSDN

Пример

Пример 1

#include <WinAPILocale.au3>

Local $iID = _WinAPI_GetUserGeoID()

ConsoleWrite('ID: 0x' & Hex($iID) & @CRLF)
ConsoleWrite('Latitude: ' & _WinAPI_GetGeoInfo($iID, $GEO_LATITUDE) & @CRLF)
ConsoleWrite('Longitude: ' & _WinAPI_GetGeoInfo($iID, $GEO_LONGITUDE) & @CRLF)
ConsoleWrite('Name: ' & _WinAPI_GetGeoInfo($iID, $GEO_FRIENDLYNAME) & @CRLF)
ConsoleWrite('ISO code: ' & _WinAPI_GetGeoInfo($iID, $GEO_ISO3) & @CRLF)

Пример 2

#include <Array.au3>
#include <WinAPILocale.au3>

Local $aData = _WinAPI_EnumSystemGeoID()

If Not @error Then
    Local $aData2[UBound($aData)][$GEO_PARENT]
    For $iData_idx = 0 To $aData[0]
        For $iData2_idx = 1 To $GEO_PARENT
            $aData2[$iData_idx][$iData2_idx - 1] = _WinAPI_GetGeoInfo($aData[$iData_idx], $iData2_idx)
        Next
    Next
    _ArrayDisplay($aData2, '_WinAPI_GetGeoInfo')
EndIf
