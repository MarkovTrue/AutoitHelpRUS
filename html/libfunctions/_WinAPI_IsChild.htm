_WinAPI_IsChild

Проверяет, является ли окно дочерним окном указанного родительского окна.

#include <WinAPISysWin.au3>
_WinAPI_IsChild ( $hWnd, $hWndParent )

Параметры

$hWnd Дескриптор окна.
$hWndParent Дескриптор родительского окна.

Возвращаемое значение

True: Возвращает окно является дочерним окном указанного родительского окна.
False: окно не является дочерним окном указанного родительского окна.
