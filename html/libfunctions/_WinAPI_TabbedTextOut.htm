Справочник функций

_WinAPI_TabbedTextOut

Записывает строку символов в указанное место и расширяет табуляторы до указанных позиций табуляции.

#include <WinAPIGdi.au3>
_WinAPI_TabbedTextOut ( $hDC, $iX, $iY, $sText [, $aTab = 0 [, $iStart = 0 [, $iEnd = -1 [, $iOrigin = 0]]]] )

Параметры

$hDC Дескриптор контекста устройства.
$iX Координата X начальной точки строки в логических единицах.
$iY Координата Y начальной точки строки в логических единицах.
$sText Рисуемая строка символов.
$aTab [необязательный] Массив, содержащий позиции табуляции в логических единицах. Позиции табуляции должны быть отсортированы в порядке возрастания;
наименьшее значение x должно быть первым элементом в массиве. Также это может быть целое число, которое является
одной позицией табуляции. В этом случае позиции табуляции разделены расстоянием, указанным этим значением.
Если этот параметр равен 0, табуляторы расширяются до восьмикратной средней ширины символа.
$iStart [необязательный] Индекс элемента массива, который содержит первую позицию табуляции.
$iEnd [необязательный] Индекс элемента массива, который содержит последнюю позицию табуляции.
$iOrigin [необязательный] Координата X начальной позиции, из которой расширяются табуляторы, в логических единицах. Это позволяет
приложению вызывать функцию несколько раз для одной строки. Если приложение вызывает функцию
больше одного раза с начальной позицией, установленной на одно и то же значение каждый раз, функция расширяет
все табуляторы относительно позиции, указанной этим параметром.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0.

Примечания

Строка записывается текущим выбранным шрифтом, цветом фона и цветом текста.

По умолчанию текущая позиция не используется и не обновляется этой функцией. Если приложению требуется обновить
текущую позицию при вызове функции, приложение может вызвать _WinAPI_SetTextAlign() с флагом $TA_UPDATECP
установленным. Затем система игнорирует параметры X и Y при последующих вызовах функции _WinAPI_TabbedTextOut(),
используя вместо этого текущую позицию.

Вызовите _WinAPI_GetExtended() для получения структуры $tagSIZE, содержащей размеры строки.

См. также

_WinAPI_GetExtended

Пример

#include <FontConstants.au3>
#include <SendMessage.au3>
#include <StaticConstants.au3>
#include <WinAPIGdi.au3>
#include <WinAPIGdiDC.au3>
#include <WinAPIHObj.au3>
#include <WinAPISysWin.au3>
#include <WindowsSysColorConstants.au3>

; Создать GUI
Local $hForm = GUICreate('Test ' & StringReplace(@ScriptName, '.au3', '()'), 400, 400)
Local $idPic = GUICtrlCreatePic('', 0, 0, 400, 400)
Local $hPic = GUICtrlGetHandle($idPic)

; Создать bitmap
Local $hDev = _WinAPI_GetDC($hPic)
Local $hDC = _WinAPI_CreateCompatibleDC($hDev)
Local $hSource = _WinAPI_CreateCompatibleBitmapEx($hDev, 400, 400, _WinAPI_SwitchColor(_WinAPI_GetSysColor($COLOR_3DFACE)))
Local $hSv = _WinAPI_SelectObject($hDC, $hSource)

; Нарисовать объекты
Local $hOldBrush = _WinAPI_SelectObject($hDC, _WinAPI_GetStockObject($DC_BRUSH))
Local $hOldPen = _WinAPI_SelectObject($hDC, _WinAPI_GetStockObject($DC_PEN))

_WinAPI_SetTextColor($hDC, 0xCD0091)
_WinAPI_SetBkMode($hDC, $TRANSPARENT)
Local $hFont = _WinAPI_CreateFont(12, 0, 0, 0, $FW_NORMAL, 0, 0, 0, $DEFAULT_CHARSET, $OUT_DEFAULT_PRECIS, $CLIP_DEFAULT_PRECIS, $ANTIALIASED_QUALITY, $DEFAULT_PITCH, 'Arial Black')
Local $hObj = _WinAPI_SelectObject($hDC, $hFont)
_WinAPI_TextOut($hDC, 10, 15, 'Simple Text')
Local $sTabbedTextOut = 'Tabbed ' & @TAB & 'Text ' & @TAB & 'Out'
_WinAPI_TextOut($hDC, 10, 35, $sTabbedTextOut)
_WinAPI_TabbedTextOut($hDC, 10, 65, $sTabbedTextOut)
_WinAPI_SelectObject($hDC, $hObj)
_WinAPI_DeleteObject($hFont)

; ; слияние растра
Local $hBitmap = _WinAPI_CreateCompatibleBitmap($hDev, 400, 400)
Local $hBrush = _WinAPI_SelectObject($hDC, $hOldBrush)
_WinAPI_DeleteObject($hBrush)
Local $hPen = _WinAPI_SelectObject($hDC, $hOldPen)
_WinAPI_DeleteObject($hPen)
_WinAPI_SelectObject($hDC, $hBitmap)
_WinAPI_DrawBitmap($hDC, 0, 0, $hSource, $MERGECOPY)
_WinAPI_ReleaseDC($hPic, $hDev)
_WinAPI_SelectObject($hDC, $hSv)
_WinAPI_DeleteObject($hSource)
_WinAPI_DeleteDC($hDC)

; Установить bitmap для элемента управления
_SendMessage($hPic, $STM_SETIMAGE, 0, $hBitmap)
$hObj = _SendMessage($hPic, $STM_GETIMAGE)
If $hObj <> $hBitmap Then
    _WinAPI_DeleteObject($hBitmap)
EndIf

GUISetState(@SW_SHOW)

Do
Until GUIGetMsg() = -3
