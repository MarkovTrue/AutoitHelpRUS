_WinAPI_GetTopWindow

Получает дескриптор дочернего окна в верхней части порядка Z.

#include <WinAPISysWin.au3>
_WinAPI_GetTopWindow ( $hWnd )

Параметры

$hWnd Дескриптор родительского окна, дочерние окна которого должны быть проверены. Если этот параметр равен 0, функция
возвращает дескриптор окна в верхней части порядка Z.

Возвращаемое значение

Успех: Возвращает дескриптор дочернего окна в верхней части порядка Z. Если указанное окно не имеет дочерних окон,
возвращаемое значение равно 0.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
