Справочник функций

_IEBodyReadHTML

Возвращает HTML внутри тега <body> документа.

#include <IE.au3>
_IEBodyReadHTML ( ByRef $oObject )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.

Возвращаемое значение

Успех: Возвращает HTML, содержащийся в <body> документа.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных.
@extended: Содержит номер недопустимого параметра.

Примечания

Эта функция использует свойство .innerHTML тела документа (Body).
Свойство .innerHTML доступно для почти всех DOM-элементов через _IEPropertyGet().

См. также

_IEBodyReadText, _IEBodyWriteHTML, _IEDocInsertHTML, _IEDocInsertText, _IEDocReadHTML, _IEPropertyGet

Пример

; Открытие браузера с базовым примером, чтение HTML body,
; Добавить новый HTML к исходному и записать обратно в браузер

#include <IE.au3>

Local $oIE = _IE_Example("basic")
Local $sHTML = _IEBodyReadHTML($oIE)
$sHTML = $sHTML & "<p><font color=red size=+5>Big RED text!</font>"
_IEBodyWriteHTML($oIE, $sHTML)
