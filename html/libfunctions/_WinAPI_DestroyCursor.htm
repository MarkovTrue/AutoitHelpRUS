_WinAPI_DestroyCursor

Уничтожает курсор и освобождает занятую им память.

#include <WinAPIRes.au3>
_WinAPI_DestroyCursor ( $hCursor )

Параметры

$hCursor Дескриптор уничтожаемого курсора. Курсор не должен использоваться.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция уничтожает не разделяемый (nonshared) курсор. Не используйте её для разделяемых курсоров.
Разделяемый курсор остаётся действительным, пока в памяти находится модуль, из которого он был загружен.
