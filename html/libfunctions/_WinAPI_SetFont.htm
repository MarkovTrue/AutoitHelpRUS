Справочник функций

_WinAPI_SetFont

Устанавливает шрифт окна.

#include <WinAPISysWin.au3>
_WinAPI_SetFont ( $hWnd, $hFont [, $bRedraw = True] )

Параметры

$hWnd Дескриптор окна.
$hFont Дескриптор шрифта.
$bRedraw [необязательный] True для перерисовки элемента управления

Возвращаемое значение

Отсутствует.
