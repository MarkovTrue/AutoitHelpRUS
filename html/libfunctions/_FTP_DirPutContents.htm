Справочник функций

_FTP_DirPutContents

Загружает папку на FTP-сервер, при необходимости – рекурсивно.

#include <FTPEx.au3>
_FTP_DirPutContents ( $hFTPSession, $sLocalFolder, $sRemoteFolder, $bRecursivePut [, $iContext = 0] )

Параметры

$hFTPSession возвращается функцией _FTP_Connect().
$sLocalFolder Локальная папка, например "c:\temp".
$sRemoteFolder Удалённая папка, например '/website/home'.
$bRecursivePut Рекурсивный обход подпапок. 0 – без рекурсии, 1 – с рекурсией.
$iContext [необязательный] Переменная, содержащая определённое приложением значение, которое связывает этот поиск с данными приложения.
Используется только если приложение уже вызывало _FTP_SetStatusCallback() для установки функции обратного вызова статуса.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

См. также

_FTP_Connect, _FTP_SetStatusCallback
