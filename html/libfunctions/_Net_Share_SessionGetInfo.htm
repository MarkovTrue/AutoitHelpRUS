Справочник функций

_Net_Share_SessionGetInfo

Получает информацию о сессии, установленной между сервером и рабочей станцией.

#include <NetShare.au3>
_Net_Share_SessionGetInfo ( $sServer, $sClientName, $sUserName )

Параметры

$sServer Строка с DNS- или NetBIOS-именем удалённого сервера, на котором выполняется функция.
Если параметр пустой, используется локальный компьютер.
$sClientName Имя сессии компьютера, для которой должна быть возвращена информация.
Этот параметр не может быть пуст.
$sUserName Строка, указывающая имя пользователя, информация о сессии которого должна быть возвращена.
Этот параметр не может быть пуст.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
[0] – имя компьютера, установившего сессию
[1] – имя пользователя, установившего сессию
[2] – количество файлов, устройств и каналов, открытых во время сессии
[3] – количество секунд, в течение которых сессия была активна
[4] – количество секунд, в течение которых сессия была неактивна
[5] – указывает, как пользователь установил сессию:
1 – пользователь установил сессию с использованием гостевой учётной записи
2 – пользователь установил сессию без использования шифрования пароля
[6] – указывает тип клиента, установившего сессию
Ошибка: Устанавливает @error ≠ 0.

Примечания

Эту функцию могут выполнять только члены локальных групп "Администраторы" или "Операторы серверов".

См. также

_Net_Share_FileGetInfo, _Net_Share_ShareGetInfo

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <NetShare.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $sServer, $aInfo

    ; Создать GUI
    GUICreate("NetShare", 400, 300)

    ; Создать элемент memo
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Получить сведения о сервере и общем ресурсе
    $sServer = InputBox("NetWork Demo", "Enter Server Name:", "\\MyServer", "", 200, 130)
    If @error Then Exit

    ; Получить информацию о сессии
    $aInfo = _Net_Share_SessionGetInfo($sServer, @ComputerName, "Administrator")
    _MemoWrite("Error ..........: " & @error)
    _MemoWrite("Computer name ..: " & $aInfo[0])
    _MemoWrite("User name ......: " & $aInfo[1])
    _MemoWrite("Resources open .: " & $aInfo[2])
    _MemoWrite("Seconds active .: " & $aInfo[3])
    _MemoWrite("Seconds idle ...: " & $aInfo[4])
    _MemoWrite("Connection type : " & $aInfo[5])
    _MemoWrite("Client type ....: " & $aInfo[6])

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
