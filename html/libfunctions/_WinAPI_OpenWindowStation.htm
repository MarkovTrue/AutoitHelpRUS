Справочник функций

_WinAPI_OpenWindowStation

Открывает указанную станцию окна.

#include <WinAPISys.au3>
_WinAPI_OpenWindowStation ( $sName [, $iAccess = 0 [, $bInherit = False]] )

Параметры

$sName Имя станции окна для открытия. Имена станций окна нечувствительны к регистру. Эта станция окна
должна принадлежать текущему сеансу.
$iAccess [необязательный] Доступ к станции окна. Этот параметр может быть одним или несколькими из следующих значений:
$WINSTA_ALL_ACCESS
$WINSTA_ACCESSCLIPBOARD
$WINSTA_ACCESSGLOBALATOMS
$WINSTA_CREATEDESKTOP
$WINSTA_ENUMDESKTOPS
$WINSTA_ENUMERATE
$WINSTA_EXITWINDOWS
$WINSTA_READATTRIBUTES
$WINSTA_READSCREEN
$WINSTA_WRITEATTRIBUTES
$bInherit [необязательный] Наследуется ли дескриптор дочерними процессами:
True – процессы, созданные этим процессом, наследуют дескриптор.
False – процессы не наследуют дескриптор (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор указанной станции окна.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

После того как вы закончите с дескриптором, вы должны вызвать _WinAPI_CloseWindowStation() для освобождения дескриптора.
