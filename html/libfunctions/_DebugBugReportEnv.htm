Справочник функций

_DebugBugReportEnv

Возвращает строку с информацией, необходимой для отправки отчёта об ошибке.

#include <Debug.au3>
_DebugBugReportEnv ( )

Возвращаемое значение

Возвращает строку со всеми данными, необходимыми для отправки отчёта.

Примечания

Необходимо использовать при отправке отчёта об ошибке.

Пример

#include <Debug.au3>
#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "Bug Report Environment", _DebugBugReportEnv())
