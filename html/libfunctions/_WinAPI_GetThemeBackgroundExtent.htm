_WinAPI_GetThemeBackgroundExtent

Вычисляет размер и расположение фона, определённого визуальным стилем, для заданной области содержимого.

#include <WinAPITheme.au3>
_WinAPI_GetThemeBackgroundExtent ( $hTheme, $iPartID, $iStateID, $hDC, $tRECT )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит область содержимого.
$iStateID Состояние части.
$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT, которая содержит прямоугольник фона содержимого.

Возвращаемое значение

Успех: Возвращает структура $tagRECT, которая содержит прямоугольник фона.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
