Справочник функций

_WinNet_CancelConnection2

Отменяет существующее сетевое подключение.

#include <WinNet.au3>
_WinNet_CancelConnection2 ( $sName [, $bUpdate = True [, $bForce = True]] )

Параметры

$sName Имя переадресованного локального устройства или удалённого сетевого ресурса для отключения.
Когда этот параметр указывает переадресованное локальное устройство, функция отменяет только указанную переадресацию устройства.
Если параметр указывает удалённый сетевой ресурс, отменяются только подключения к удалённым сетям без устройств.
$bUpdate [необязательный] Если True, профиль пользователя обновляется информацией о том, что подключение больше не является постоянным.
$bForce [необязательный] Указывает, должно ли отключение происходить, если в подключении есть открытые файлы или задачи.
Если этот параметр равен False, функция завершается ошибкой при наличии открытых файлов или задач.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
