Справочник функций

_WinAPI_EnumUILanguages

Перечисляет языки пользовательского интерфейса, доступные в операционной системе.

#include <WinAPILocale.au3>
_WinAPI_EnumUILanguages ( [$iFlag = 0] )

Параметры

$iFlag [необязательный] Флаг, определяющий формат языка. Должен быть 0 или одно из значений (для Windows Vista и новее):
$MUI_LANGUAGE_ID
$MUI_LANGUAGE_NAME

Возвращаемое значение

Успех: Возвращает массив идентификаторов или имён языков интерфейса.
Ошибка: Устанавливает @error ≠ 0.

Дополнительно

Искать EnumUILanguages в библиотеке MSDN

Пример

#include <Array.au3>
#include <WinAPILocale.au3>

Local $aData = _WinAPI_EnumUILanguages()

_ArrayDisplay($aData, '_WinAPI_EnumUILanguages')
