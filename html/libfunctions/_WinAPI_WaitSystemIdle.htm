_WinAPI_WaitSystemIdle

Ожидание, пока процессор и дисковый ввод-вывод не станут неактивными.

#include <WinAPISys.au3>
_WinAPI_WaitSystemIdle ( [$bToolTip = False [, $nMaxLoop = Default [, $fSystemPercent = 1.0 [, $fIOPercent = 1.0 [, $iRefresh = 2]]]]] )

Параметры

$bToolTip [необязательный] для отображения подсказки использования процессора или диска (по умолчанию не отображается)
$nMaxLoop [необязательный] максимальное количество оценок (по умолчанию нет проверки)
$fSystemPercent [необязательный] подсчёт остановится, если текущее использование процессора ниже.
$fIOPercent [необязательный] подсчёт остановится, если текущее использование диска ниже.
$iRefresh [необязательный] время между оценками (по умолчанию = 2 сек).

Возвращаемое значение

Успех: Возвращает количество раз, когда использование процессора было выше $fSystemPercent ($fSystemPercent по умолчанию 1.0).
@extended – количество раз, когда использование диска было выше $fIOPercent ($fIOPercent по умолчанию 1.0).
Ошибка: Задаёт флаг @error, если количество оценок превышено.

Примечания

Во время циклов пользовательские входные данные отключены: BlockInput($BI_DISABLE).
Если нет странного поведения, оценка процента использования диска не будет эффективной.

См. также

_WinAPI_QueryProcessorUsage, _WinAPI_QueryDiskUsage

Пример

#pragma compile(AutoItExecuteAllowed, True)

#include <MsgBoxConstants.au3>
#include <WinAPISys.au3>

; создать скрипт, нагружающий CPU
Local $sTestAu3 = @TempDir & "\TestSystemIdle.au3"
FileDelete($sTestAu3)
FileWriteLine($sTestAu3, 'For $i = 1 To 20000000')
FileWriteLine($sTestAu3, '$i = $i')
FileWriteLine($sTestAu3, 'Next')

;~ Local $sTestAu3 = @TempDir & "\TestIOIdle.au3"
;~ FileDelete($sTestAu3)
;~ FileWriteLine($sTestAu3, 'For $i = 1 To 100000')
;~ FileWriteLine($sTestAu3, 'FileRead(@ScriptFullPath)')
;~ FileWriteLine($sTestAu3, 'Next')

If @Compiled Then
    Run(@AutoItExe & ' /AutoIt3ExecuteScript "' & $sTestAu3 & '"')
Else
    Run(@AutoItExe & " " & $sTestAu3)
EndIf

Local $hTimer = TimerInit()
Local $nSleep = _WinAPI_WaitSystemIdle(True)
Local $iExtended = @extended
Local $sElapse = Round(TimerDiff($hTimer) / 1000, 2)

FileDelete($sTestAu3)

MsgBox($MB_TOPMOST, 'Test duration', $sElapse & " sec (" & $nSleep & "/" & $iExtended & ')')
