_WinAPI_PathAddExtension

Добавляет расширение имени файла к строке пути.

#include <WinAPIShPath.au3>
_WinAPI_PathAddExtension ( $sFilePath [, $sExt = ''] )

Параметры

$sFilePath Путь, к которому будет добавлено расширение имени файла. Если уже присутствует расширение имени файла,
расширение не будет добавлено.
$sExt [необязательный] Расширение имени файла. Если этот параметр является пустой строкой, будет добавлено расширение ".exe".

Возвращаемое значение

Возвращает результирующий путь, @extended будет установлено в 1 если расширение было добавлено, или 0 в противном случае.

Примечания

Если уже присутствует расширение имени файла, расширение не будет добавлено.
Если $sFilePath является пустой строкой, результат будет только расширением имени файла.
Если $sExt является пустой строкой, будет добавлено расширение ".exe".

Пример

#include <WinAPIShPath.au3>

Local $aPath[2] = ['C:\Documents\Test.txt', 'C:\Documents\Test']

For $i = 0 To 1
    ConsoleWrite($aPath[$i] & ' => ' & _WinAPI_PathAddExtension($aPath[$i], '.doc') & @CRLF)
