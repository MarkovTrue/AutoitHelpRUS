_WinAPI_ReleaseStream

Освобождает объект потока.

#include <WinAPICom.au3>
_WinAPI_ReleaseStream ( $pStream )

Параметры

$pStream Указатель на объект потока, ранее созданный вызовом функции _WinAPI_CreateStreamOnHGlobal().

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

См. также
