Справочник функций

_GUICtrlListView_Scroll

Прокручивает содержимое ListView.

#include <GuiListView.au3>
_GUICtrlListView_Scroll ( $hWnd, $iDX, $iDY )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iDX Целое число, задающее величину горизонтальной прокрутки в пикселях.
Если ListView в режиме list-view, значение задаёт количество прокручиваемых столбцов.
$iDY Целое число, задающее величину вертикальной прокрутки в пикселях.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Когда ListView в режиме report, прокрутка возможна только по вертикали и кратна целым строкам.
Поэтому параметр $iDY округляется до ближайшего кратного высоте строки числа пикселей.
Например, если высота строки 16 пикселей, и для $iDY передано 8, список прокрутится на 16 пикселей (1 строка).
Если передано 7, прокрутка составит 0 пикселей (0 строк).

Пример

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>

Example()

Func Example()
    Local $idListview

    GUICreate("ListView Scroll", 400, 300)
    $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Добавить столбец
    _GUICtrlListView_AddColumn($idListview, "Items", 100)

    ; Добавить элементы
    _GUICtrlListView_BeginUpdate($idListview)
    For $iI = 1 To 100
        _GUICtrlListView_AddItem($idListview, "Item " & $iI)
    Next
    _GUICtrlListView_EndUpdate($idListview)

    Прокручивает
    _GUICtrlListView_Scroll($idListview, 0, 500)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
