Справочник функций

_WinAPI_GetAllUsersProfileDirectory

Получает путь к корневой папке, содержащей данные программ, общие для всех пользователей.

#include <WinAPIShellEx.au3>
_WinAPI_GetAllUsersProfileDirectory ( )

Возвращаемое значение

Успех: Путь к каталогу.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
