Справочник функций

_WinAPI_ExpandEnvironmentStrings

Раскрывает строки переменных окружения и заменяет их фактическими значениями.

#include <WinAPISys.au3>
_WinAPI_ExpandEnvironmentStrings ( $sString )

Параметры

$sString Строка для подстановки переменных окружения.

Возвращаемое значение

Успех: Возвращает Converted string.
Ошибка: Устанавливает @error в ненулевое значение; для получения расширенной информации об ошибке вызовите _WinAPI_GetLastError().

Дополнительно

Искать ExpandEnvironmentStrings в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPISys.au3>

MsgBox($MB_SYSTEMMODAL, "Environment string", "%windir% = " & _WinAPI_ExpandEnvironmentStrings("%windir%"))
