Справочник функций

_WinAPI_RegisterWindowMessage

Определяет новое сообщение окна, которое гарантированно уникально во всей системе.

#include <WinAPISysWin.au3>
_WinAPI_RegisterWindowMessage ( $sMessage )

Параметры

$sMessage Строка, которая указывает сообщение для регистрации.

Возвращаемое значение

Успех: Возвращает идентификатор сообщения в диапазоне 0xC000 по 0xFFFF.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция RegisterWindowMessage используется для регистрации сообщений для взаимодействия между двумя взаимодействующими
приложениями. Если два разных приложения регистрируют одну и ту же строку сообщения, приложения возвращают одно и то же
значение сообщения. Сообщение остаётся зарегистрированным до конца сеанса.

Дополнительно

Искать RegisterWindowMessage в библиотеке MSDN

Пример

#include <APISysConstants.au3>
#include <WinAPISysWin.au3>

Opt('TrayAutoPause', 0)

OnAutoItExitRegister(OnAutoItExit)

Global $g_hForm = GUICreate('')
GUIRegisterMsg(_WinAPI_RegisterWindowMessage('SHELLHOOK'), WM_SHELLHOOK)
_WinAPI_RegisterShellHookWindow($g_hForm)

ToolTip("Type ESC to exit the script")
HotKeySet("{ESC}", "_Exit")

While 1
    Sleep(100)
WEnd

Func WM_SHELLHOOK($hWnd, $iMsg, $wParam, $lParam)
    #forceref $iMsg

    Local $sTitle = WinGetTitle($lParam)
    Switch $hWnd
        Case $g_hForm
            Switch $wParam
                Case $HSHELL_REDRAW
                    ConsoleWrite('Redrawn: ' & $sTitle & @CRLF)
                Case $HSHELL_WINDOWACTIVATED, $HSHELL_RUDEAPPACTIVATED
                    ConsoleWrite('Activated: ' & $sTitle & @CRLF)
            EndSwitch
    EndSwitch
EndFunc   ;==>WM_SHELLHOOK

Func OnAutoItExit()
    _WinAPI_DeregisterShellHookWindow($g_hForm)
EndFunc   ;==>OnAutoItExit

Func _Exit()
    Exit 1
EndFunc   ;==>_Exit
