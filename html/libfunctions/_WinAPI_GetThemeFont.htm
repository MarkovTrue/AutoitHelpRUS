_WinAPI_GetThemeFont

Получает значение свойства шрифта.

#include <WinAPITheme.au3>
_WinAPI_GetThemeFont ( $hTheme, $iPartID, $iStateID, $iPropID [, $hDC = 0] )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит свойство шрифта.
$iStateID Состояние части.
$iPropID Получаемое свойство ($TMT_*).
$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает структура $tagLOGFONT, которая содержит значение свойства шрифта.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
