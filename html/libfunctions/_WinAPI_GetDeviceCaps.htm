_WinAPI_GetDeviceCaps

Получает специфичную для устройства информацию об указанном устройстве.

#include <WinAPIGdiDC.au3>
_WinAPI_GetDeviceCaps ( $hDC, $iIndex )

Параметры

$hDC Идентифицирует контекст устройства.
$iIndex Определяет элемент для возврата.

Возвращаемое значение

Возвращает значение требуемого элемента.
