_WinAPI_CoTaskMemRealloc

Изменяет размер ранее выделенного блока памяти задачи.

#include <WinAPICom.au3>
_WinAPI_CoTaskMemRealloc ( $pMemory, $iSize )

Параметры

$pMemory Указатель на блок памяти, который должен быть перевыделен.
$iSize Размер перевыделяемого блока памяти в байтах.

Возвращаемое значение

Успех: Возвращает перевыделенный блок памяти.
Ошибка: Возвращает 0.

Примечания

Если $pMemory равен 0, функция выделяет новый блок памяти тем же способом, что и функция _WinAPI_CoTaskMemAlloc().
Если $pMemory не равен 0, это должен быть указатель, возвращённый предыдущим вызовом _WinAPI_CoTaskMemAlloc().

Содержимое блока остаётся неизменным вплоть до меньшего из новых и старых размеров, хотя новый блок может находиться в другом месте.
Если $pMemory не равен 0 и $iSize равен 0, то память, на которую указывает $pMemory, освобождается.
