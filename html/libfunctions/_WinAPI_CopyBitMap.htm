_WinAPI_CopyBitmap

Создаёт дубликат указанного растра с разделом независимого от устройства растра (DIB).

#include <WinAPIGdi.au3>
_WinAPI_CopyBitmap ( $hBitmap )

Параметры

$hBitmap Дескриптор растра, который должен быть дублирован.

Возвращаемое значение

Успех: Возвращает дескриптор созданного нового растра.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

По завершении работы с растром удалите его, используя функцию _WinAPI_DeleteObject().

См. также

_WinAPI_DeleteObject
