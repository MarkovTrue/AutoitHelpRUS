_WinAPI_CoInitialize

Инициализирует библиотеку COM для использования вызывающим процессом.

#include <WinAPICom.au3>
_WinAPI_CoInitialize ( [$iFlags = $COINIT_MULTITHREADED] )

Параметры

$iFlags [необязательный] Этот параметр может быть одним или несколькими из следующих значений.
$COINIT_APARTMENTTHREADED
$COINIT_DISABLE_OLE1DDE
$COINIT_MULTITHREADED (по умолчанию)
$COINIT_SPEED_OVER_MEMORY

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
