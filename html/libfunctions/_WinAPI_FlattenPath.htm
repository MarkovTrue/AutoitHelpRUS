_WinAPI_FlattenPath

Преобразует все кривые в пути, выбранном в текущий контекст устройства, в последовательность отрезков.

#include <WinAPIGdi.au3>
_WinAPI_FlattenPath ( $hDC )

Параметры

$hDC Дескриптор device context, содержащего валидный путь.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
