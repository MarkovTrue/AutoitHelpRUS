_WinAPI_GetThemeTextExtent

Вычисляет размер и расположение указанного текста при отображении в шрифте визуального стиля.

#include <WinAPITheme.au3>
_WinAPI_GetThemeTextExtent ( $hTheme, $iPartID, $iStateID, $hDC, $sText, $tRECT, $iFlags )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, в которой будет отображаться текст.
$iStateID Состояние части.
$hDC Дескриптор контекста устройства.
$sText Строка с отрисовываемым текстом.
$tRECT Структура $tagRECT, которая содержит прямоугольник, используемый для управления расположением текста. Этот параметр может быть установлен на 0.
$iFlags Флаги форматирования строки ($DT_* определены в "APIGdiConstants.au3).

Возвращаемое значение

Успех: Возвращает структура $tagRECT, которая содержит в логических координатах прямоугольник, необходимый для размещения отображаемого текста.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
