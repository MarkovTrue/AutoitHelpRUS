_WinAPI_GetColorAdjustment

Получает коррекцию цвета для указанного контекста устройства (DC).

#include <WinAPIGdi.au3>
_WinAPI_GetColorAdjustment ( $hDC )

Параметры

$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает структуру $tagCOLORADJUSTMENT, содержащую значения коррекции цвета.
Ошибка: Устанавливает @error ≠ 0.
