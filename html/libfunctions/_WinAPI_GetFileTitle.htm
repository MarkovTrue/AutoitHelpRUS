Справочник функций

_WinAPI_GetFileTitle

Получает имя указанного файла.

#include <WinAPIFiles.au3>
_WinAPI_GetFileTitle ( $sFilePath )

Параметры

$sFilePath Имя и место расположения файла.

Возвращаемое значение

Успех: Имя файла.
Ошибка: Возвращает пустую строку.

Примечания

Функция _WinAPI_GetFileTitle() должна вызваться только с допустимыми именами файлов; использование недопустимого имени файла имеет неопределённый результат.

Функция _WinAPI_GetFileTitle() возвращает строку, которую система использует для отображения имени файла пользователю.
Имя отображения включает расширение только в том случае, если это предпочтение пользователя для отображения имён файлов.
Это означает, что возвращённая строка может не точно идентифицировать файл, если она используется в вызовах функций файловой системы.
