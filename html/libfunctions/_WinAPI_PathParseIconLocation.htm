_WinAPI_PathParseIconLocation

Разбирает строку расположения файла, которая содержит расположение файла и индекс значка.

#include <WinAPIShPath.au3>
_WinAPI_PathParseIconLocation ( $sFilePath )

Параметры

$sFilePath Путь, который содержит строку расположения файла. Он должен быть в форме "путь,индекс_значка".

Возвращаемое значение

Возвращает массив, содержащий следующую информацию:
    [0] – путь файла, который содержит значок.
    [1] – индекс значка.

Примечания

Эта функция полезна для получения значения DefaultIcon из реестра и отделения индекса значка от пути.

Дополнительно

Искать PathParseIconLocation в библиотеке MSDN

Пример

#include <WinAPIReg.au3>
#include <WinAPIShPath.au3>

Local $sData = _WinAPI_AssocQueryString('.txt', $ASSOCSTR_DEFAULTICON)
Local $aIcon = _WinAPI_PathParseIconLocation($sData)

If IsArray($aIcon) Then
    ConsoleWrite('DefaultIcon: ' & $sData & @CRLF)
    ConsoleWrite('Icon: ' & $aIcon[0] & @CRLF)
    ConsoleWrite('Index: ' & $aIcon[1] & @CRLF)
EndIf
