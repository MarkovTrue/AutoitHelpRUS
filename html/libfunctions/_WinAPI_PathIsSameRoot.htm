_WinAPI_PathIsSameRoot

Сравнивает два пути, чтобы определить, имеют ли они общий корневой компонент.

#include <WinAPIShPath.au3>
_WinAPI_PathIsSameRoot ( $sPath1, $sPath2 )

Параметры

$sPath1 Первый путь для сравнения.
$sPath2 Второй путь для сравнения.

Возвращаемое значение

True: Возвращает оба пути имеют одинаковый корневой компонент.
False: пути не имеют одинакового корневого компонента.
