Справочник функций

_GUICtrlListView_InsertColumn

Вставляет новый столбец в элемент управления.

#include <GuiListView.au3>
_GUICtrlListView_InsertColumn ( $hWnd, $iIndex, $sText [, $iWidth = 50 [, $iAlign = -1 [, $iImage = -1 [, $bOnRight = False]]]] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iIndex Индекс нового столбца (нумерация с 0).
$sText Текст заголовка столбца.
$iWidth [необязательный] Ширина столбца в пикселях
$iAlign [необязательный] Выравнивание заголовка столбца и текста подэлементов в столбце:
0 – по левому краю
1 – по правому краю
2 – по центру
$iImage [необязательный] Индекс изображения в списке изображений (нумерация с 0)
$bOnRight [необязательный] Если True – изображение столбца отображается справа от текста

Возвращаемое значение

Успех: индекс нового столбца.
Ошибка: Возвращает -1.

Примечания

Выравнивание самого левого столбца всегда по левому краю; изменить его нельзя.

См. также

_GUICtrlListView_DeleteColumn

Пример

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>

Example()

Func Example()
    GUICreate("ListView Insert Column (v" & @AutoItVersion & ")", 400, 300)
    Local $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Установить формат ANSI
;~ _GUICtrlListView_SetUnicodeFormat($idListview, False)

    ; Вставить столбцы
    _GUICtrlListView_InsertColumn($idListview, 0, "Column 1", 100)
    _GUICtrlListView_InsertColumn($idListview, 1, "Column 2", 100)
    _GUICtrlListView_InsertColumn($idListview, 2, "Column 3", 100)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
