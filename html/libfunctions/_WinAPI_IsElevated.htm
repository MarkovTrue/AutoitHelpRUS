Справочник функций

_WinAPI_IsElevated

Определяет, запущен ли текущий процесс с повышенными привилегиями.

#include <WinAPIProc.au3>
_WinAPI_IsElevated ( )

Возвращаемое значение

Успех: Возвращает True – текущий процесс запущен с повышенными привилегиями.
False – в противном случае.
Кроме того, функция всегда устанавливает флаг @extended в одно из следующих значений (любое другое значение
указывает на неожиданную ошибку).
0 – указывает, что либо управление учётными записями пользователей (UAC) отключено, либо процесс запущен обычным
пользователем (не членом группы администраторов).
Следующие два значения могут быть возвращены только если оба UAC включены и пользователь является членом
группы администраторов (то есть у пользователя есть "разделённый" токен).
1 – указывает, что процесс запущен с повышенными привилегиями.
2 – указывает, что процесс не запущен с повышенными привилегиями.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.
