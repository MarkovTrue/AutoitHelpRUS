Справочник функций

_FTP_FileRead

Читает данные из дескриптора, открытого функцией _FTP_FileOpen().

#include <FTPEx.au3>
_FTP_FileRead ( $hFTPFile, $iNumberOfBytesToRead )

Параметры

$hFTPFile Дескриптор FTP-файла, возвращённый функцией _FTP_FileOpen().
$iNumberOfBytesToRead Количество байт для чтения.

Возвращаемое значение

Успех: Возвращает прочитанные двоичные данные или строку.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: -1 – конец файла
≠ 0 – прочие ошибки.
