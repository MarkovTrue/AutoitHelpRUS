_WinAPI_DeferWindowPos

Обновляет структуру множественного позиционирования окон для указанного окна.

#include <WinAPISysWin.au3>
_WinAPI_DeferWindowPos ( $hInfo, $hWnd, $hAfter, $iX, $iY, $iWidth, $iHeight, $iFlags )

Параметры

$hInfo Дескриптор структуры, содержащей информацию о размере и положении одного или нескольких окон. Структура возвращается функцией _WinAPI_BeginDeferWindowPos() или последним вызовом _WinAPI_DeferWindowPos().
$hWnd Дескриптор окна.
$hAfter Дескриптор окна, предшествующего размещаемому в Z-порядке.
Параметр должен быть либо дескриптором окна, либо одним из следующих значений. Параметр игнорируется, если в $iFlags установлен $SWP_NOZORDER.
$HWND_BOTTOM
$HWND_NOTOPMOST
$HWND_TOP
$HWND_TOPMOST
$iX X-координата левого верхнего угла окна.
$iY Y-координата левого верхнего угла окна.
$iWidth Задаёт новую ширину окна в пикселях.
$iHeight Задаёт новую высоту окна в пикселях.
$iFlags Комбинация значений, влияющих на размер и положение окна:
$SWP_DRAWFRAME
$SWP_FRAMECHANGED
$SWP_HIDEWINDOW
$SWP_NOACTIVATE
$SWP_NOCOPYBITS
$SWP_NOMOVE
$SWP_NOOWNERZORDER
$SWP_NOREDRAW
$SWP_NOREPOSITION
$SWP_NOSENDCHANGING
$SWP_NOSIZE
$SWP_NOZORDER
$SWP_SHOWWINDOW

Возвращаемое значение

Успех: Возвращаемое значение указывает на обновлённую структуру "несколько окон".
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
