Справочник функций

_Net_Share_ShareGetInfo

Получает информацию о конкретном общем ресурсе на сервере.

#include <NetShare.au3>
_Net_Share_ShareGetInfo ( $sServer, $sShare )

Параметры

$sServer Строка с DNS- или NetBIOS-именем удалённого сервера, на котором выполняется функция.
Если параметр пустой, используется локальный компьютер.
$sShare Имя общего ресурса, для которого возвращать информацию.

Возвращаемое значение

Успех: Возвращает массив со следующим форматом:
[0] – имя общего ресурса
[1] – тип общего ресурса. Может быть комбинацией:
$STYPE_DISKTREE – очередь печати
$STYPE_PRINTQ – дисковый диск
$STYPE_DEVICE – устройство связи
$STYPE_IPC – IPC
$STYPE_SPECIAL – специальный общий ресурс, зарезервированный для IPC$ или удалённого администрирования сервера
$STYPE_TEMPORARY – временный общий ресурс
[2] – содержит необязательный комментарий об общем ресурсе
[3] – указывает разрешения общего ресурса:
1 – разрешение на чтение данных из ресурса и выполнение
2 – разрешение на запись данных в ресурс
4 – разрешение на создание экземпляра ресурса
8 – разрешение на выполнение ресурса
16 – разрешение на удаление ресурса
32 – разрешение на изменение атрибутов ресурса
64 – разрешение на изменение разрешений, назначенных ресурсу
[4] – указывает максимальное количество подключений, которое может иметь ресурс
[5] – указывает количество текущих подключений к ресурсу
[6] – указывает локальный путь для общего ресурса
[7] – указывает пароль общего ресурса
Ошибка: Устанавливает @error ≠ 0.

Примечания

Только члены локальной группы администраторов, операторов серверов или опытных пользователей, а также члены группы операторов печати или серверов могут успешно выполнить эту функцию.

Пример

#include "Extras\HelpFileInternals.au3"

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <NetShare.au3>
#include <WindowsStylesConstants.au3>

#RequireAdmin ; необходимо для _Net_Share_ShareAdd()

Example()

Func Example()
    Local $aInfo
    Local Const $sShareName = "AutoIt Share"
    Local Const $sResourcePath = "C:\"

    ; Создать GUI
    GUICreate("NetShare", 400, 300)

    ; Создать элемент memo
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    Local $bShareAdded = False
    ; Проверить существование общего ресурса
    If _Net_Share_ShareCheck(@ComputerName, "") = -1 Then
        ; Создать общий ресурс на локальном компьютере
        $bShareAdded = _Net_Share_ShareAdd(@ComputerName, $sShareName, 0, $sResourcePath, "AutoIt Share Comment")
        If @error Then
            MsgBox($MB_SYSTEMMODAL, "Information", "Share add error : " & @error)
        Else
            _MemoWrite("Share added" & @CRLF)
        EndIf
    Else
        _MemoWrite("Share exists" & @CRLF)
    EndIf

    If $bShareAdded Then
        ; Показать сведения о добавленном общем ресурсе
        $aInfo = _Net_Share_ShareGetInfo(@ComputerName, $sShareName)
        _MemoWrite("Share name ..............: " & $aInfo[0])
        _MemoWrite("Share type...............: " & _Net_Share_ResourceStr($aInfo[1]))
        _MemoWrite("Comment .................: " & $aInfo[2])
        _MemoWrite("Permissions .............: " & _Net_Share_PermStr($aInfo[3]))
        _MemoWrite("Maximum connections .....: " & $aInfo[4])
        _MemoWrite("Current connections .....: " & $aInfo[5])
        _MemoWrite("Local path ..............: " & $aInfo[6])
        _MemoWrite("Password ................: " & $aInfo[7])

        ; Удалить общий ресурс
        _Net_Share_ShareDel(@ComputerName, $sShareName)
        If @error Then MsgBox($MB_SYSTEMMODAL, "Information", "Share delete error : " & @error)
        _MemoWrite(@CRLF & "Share deleted")

        ; Цикл до выхода пользователя.
        Do
        Until GUIGetMsg() = $GUI_EVENT_CLOSE
    EndIf

EndFunc   ;==>Example
