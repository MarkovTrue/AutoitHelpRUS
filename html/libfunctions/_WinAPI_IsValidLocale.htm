_WinAPI_IsValidLocale

Определяет, установлен ли указанный языковой стандарт или поддерживается операционной системой.

#include <WinAPILocale.au3>
_WinAPI_IsValidLocale ( $iLCID [, $iFlag = 0] )

Параметры

$iLCID Идентификатор локали (LCID) – задаёт локаль либо одно из предопределённых значений:
$LOCALE_INVARIANT
$LOCALE_SYSTEM_DEFAULT
$LOCALE_USER_DEFAULT

Windows Vista и выше
$LOCALE_CUSTOM_DEFAULT
$LOCALE_CUSTOM_UI_DEFAULT
$LOCALE_CUSTOM_UNSPECIFIED
$iFlag [необязательный] Флаг, определяющий тест достоверности для применения к идентификатору языкового стандарта.
Этот параметр может иметь одно из следующих значений.
$LCID_INSTALLED
$LCID_SUPPORTED

Возвращаемое значение

True: Возвращает идентификатор языкового стандарта проходит указанный тест достоверности.
False: идентификатор языкового стандарта является недействительным.
