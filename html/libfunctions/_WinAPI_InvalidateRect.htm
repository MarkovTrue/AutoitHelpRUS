_WinAPI_InvalidateRect

Добавляет прямоугольник в область обновления указанного окна.

#include <WinAPISysWin.au3>
_WinAPI_InvalidateRect ( $hWnd [, $tRECT = 0 [, $bErase = True]] )

Параметры

$hWnd Дескриптор окна или идентификатор элемента управления GUI.
$tRECT [необязательный] Структура $tagRECT, содержащая клиентские координаты добавляемого в область обновления
прямоугольника. Если этот параметр равен 0, вся клиентская область добавляется в область обновления.
$bErase [необязательный] Определяет, следует ли удалять фон в области обновления при обработке области обновления.
Если этот параметр равен True, фон удаляется при вызове функции BeginPaint.
Если этот параметр равен False, фон остаётся неизменным.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
