_WinAPI_GetPhysicallyInstalledSystemMemory

Получает объём оперативной памяти, физически установленной на компьютере.

#include <WinAPISys.au3>
_WinAPI_GetPhysicallyInstalledSystemMemory ( )

Возвращаемое значение

Успех: Возвращает объём физически установленной памяти в килобайтах.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Объём памяти, доступный операционной системе, может быть меньше объёма памяти, физически установленной
на компьютере, поскольку BIOS и некоторые драйверы могут зарезервировать память в виде областей ввода-вывода для устройств с отображением памяти,
делая память недоступной для операционной системы и приложений.

Эта функция требует Windows Vista SP1 или позже.
