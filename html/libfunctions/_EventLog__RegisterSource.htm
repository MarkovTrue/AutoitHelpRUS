Справочник функций

_EventLog__RegisterSource

Возвращает зарегистрированный дескриптор указанного журнала событий.

#include <EventLog.au3>
_EventLog__RegisterSource ( $sServerName, $sSourceName )

Параметры

$sServerName UNC-имя сервера, на котором будет открыт журнал событий. Если строка пустая, операция выполняется на локальном компьютере.
$sSourceName Имя источника событий, дескриптор которого требуется получить. Имя источника должно быть подразделом раздела журнала в ключе реестра Eventlog.

Возвращаемое значение

Успех: Возвращает дескриптор журнала событий.
Ошибка: Возвращает 0.

Примечания

Если имя источника не найдено, служба журнала событий использует журнал Application и не создаёт новый источник.
События по-прежнему записываются для этого источника, однако файлы сообщений и категорий для поиска описаний идентификаторов событий не заданы.
Для закрытия дескриптора журнала используйте _EventLog__DeregisterSource().
