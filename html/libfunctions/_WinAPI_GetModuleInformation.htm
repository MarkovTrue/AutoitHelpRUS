_WinAPI_GetModuleInformation

Получает информацию об указанном модуле.

#include <WinAPIProc.au3>
_WinAPI_GetModuleInformation ( $hProcess [, $hModule = 0] )

Параметры

$hProcess Дескриптор процесса, содержащего модуль. Дескриптор должен иметь право доступа $PROCESS_QUERY_INFORMATION или
$PROCESS_QUERY_LIMITED_INFORMATION и право доступа $PROCESS_VM_READ.
$hModule [необязательный] Дескриптор модуля. Если этот параметр равен 0, функция получает информацию только об
исполняемом файле (члены "SizeOfImage" и "EntryPoint" структуры $tagMODULEINFO).

Возвращаемое значение

Успех: Возвращает структура $tagMODULEINFO, содержащая информацию об модуле.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Эта функция не получает информацию для модулей, загруженных с флагом $LOAD_LIBRARY_AS_DATAFILE.
Для получения дополнительной информации см. _WinAPI_LoadLibraryEx().
