Справочник функций

_WinAPI_FreeResource

Уменьшает (на единицу) счётчик ссылок загруженного ресурса.

#include <WinAPIRes.au3>
_WinAPI_FreeResource ( $hData )

Параметры

$hData Дескриптор ресурса, возвращённый _WinAPI_LoadResource().

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Счётчик ссылок ресурса увеличивается на единицу при каждом вызове _WinAPI_LoadResource() для этого ресурса.
Функция устарела и поддерживается только для совместимости с 16-битной Microsoft Windows.
Для 32-битных приложений Windows освобождать ресурсы, загруженные через _WinAPI_LoadResource(), не нужно.
