_WinAPI_InvertRect

Инвертирует прямоугольник в окне путём выполнения логической операции НЕ над значениями цветов каждого пикселя.

#include <WinAPIGdi.au3>
_WinAPI_InvertRect ( $hDC, $tRECT )

Параметры

$hDC Дескриптор контекста устройства.
$tRECT Структура $tagRECT, содержащая логические координаты инвертируемого прямоугольника.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

На монохромных экранах _WinAPI_InvertRect() делает белые пиксели чёрными, а чёрные пиксели белыми. На цветных экранах
инвертирование зависит от того, как генерируются цвета для экрана. Вызов _WinAPI_InvertRect() два раза для одного
прямоугольника восстанавливает дисплей в его предыдущие цвета.
