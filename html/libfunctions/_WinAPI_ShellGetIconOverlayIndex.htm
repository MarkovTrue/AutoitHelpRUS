Справочник функций

_WinAPI_ShellGetIconOverlayIndex

Получает индекс значка наложения в системном списке изображений.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetIconOverlayIndex ( $sIcon, $iIndex )

Параметры

$sIcon Полностью определённый путь файла, содержащего значок.
$iIndex Индекс значка. Чтобы запросить стандартный значок наложения, установите путь на пустую строку,
и индекс на одно из следующих значений.
$IDO_SHGIOI_LINK
$IDO_SHGIOI_SHARE
$IDO_SHGIOI_SLOWFILE

Windows 7 или более позднюю версию
$IDO_SHGIOI_DEFAULT

Возвращаемое значение

Успех: Возвращает индекс значка наложения.
Ошибка: (-1).

Примечания

Для использования этой функции при запуске приложения нужно вызвать _WinAPI_FileIconInit().

См. также

_WinAPI_FileIconInit
