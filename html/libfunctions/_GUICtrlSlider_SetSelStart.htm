Справочник функций

_GUICtrlSlider_SetSelStart

Устанавливает позицию.

#include <GuiSlider.au3>
_GUICtrlSlider_SetSelStart ( $hWnd, $iMinimum )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iMinimum Начальная логическая позиция диапазона выделения.

Возвращаемое значение

Отсутствует.

См. также

_GUICtrlSlider_ClearSel, _GUICtrlSlider_GetSel, _GUICtrlSlider_GetSelStart, _GUICtrlSlider_SetSel, _GUICtrlSlider_SetSelEnd

Пример

#include <GUIConstantsEx.au3>
#include <GuiSlider.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Создать GUI
    GUICreate("Slider Get/Set Sel Start (v" & @AutoItVersion & ")", 400, 296)
    Local $idSlider = GUICtrlCreateSlider(2, 2, 396, 20, BitOR($TBS_TOOLTIPS, $TBS_AUTOTICKS, $TBS_ENABLESELRANGE))
    GUISetState(@SW_SHOW)

    ; Установить начало выделения
    _GUICtrlSlider_SetSelStart($idSlider, 10)

    ; Получить начало выделения
    MsgBox($MB_SYSTEMMODAL, "Information", "Sel Start: " & _GUICtrlSlider_GetSelStart($idSlider))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
