_WinAPI_IsWindowEnabled

Определяет, включено ли указанное окно для ввода мышью и клавиатурой.

#include <WinAPISys.au3>
_WinAPI_IsWindowEnabled ( $hWnd )

Параметры

$hWnd Дескриптор окна.

Возвращаемое значение

True: Возвращает окно включено.
False: окно не включено.

Примечания

Дочернее окно получает ввод только если оно одновременно включено и видимо.
