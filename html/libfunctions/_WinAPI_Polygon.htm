_WinAPI_Polygon

Рисует многоугольник, состоящий из двух или более вершин, соединённых прямыми линиями.

#include <WinAPIGdi.au3>
_WinAPI_Polygon ( $hDC, Const ByRef $aPoint [, $iStart = 0 [, $iEnd = -1]] )

Параметры

$hDC Дескриптор контекста устройства.
$aPoint Двумерный массив ([x1, y1], [x2, y2], ... [xN, yN]), который содержит вершины многоугольника в логических
единицах. Многоугольник закрывается автоматически путём рисования линии от последней вершины к первой.
$iStart [необязательный] Индекс массива для начала создания.
$iEnd [необязательный] Индекс массива для остановки создания.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Многоугольник обводится с помощью текущего пера и заполняется с использованием текущей кисти и режима заполнения многоугольника.
