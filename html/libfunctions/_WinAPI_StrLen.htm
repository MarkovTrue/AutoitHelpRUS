_WinAPI_StrLen

Возвращает длину указанной строки.

#include <WinAPIMisc.au3>
_WinAPI_StrLen ( $pString [, $bUnicode = True] )

Параметры

$pString Указатель на строку, завершённую нулевым символом.
$bUnicode [необязательный] Указывает, является ли строка Unicode или кодом ASCII символа, допустимые значения:
True - Unicode (По умолчанию).
False - ASCII.

Возвращаемое значение

Успех: Возвращает длина строки в значениях TCHAR (не включая завершающий нулевой символ).
Ошибка: Возвращает 0.
