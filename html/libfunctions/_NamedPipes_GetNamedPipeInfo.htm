_NamedPipes_GetNamedPipeInfo

Получает информацию об указанном именованном канале.

#include <NamedPipes.au3>
_NamedPipes_GetNamedPipeInfo ( $hNamedPipe )

Параметры

$hNamedPipe Дескриптор экземпляра именованного канала. Дескриптор должен иметь доступ GENERIC_READ к именованному каналу.

Возвращаемое значение

Успех: Возвращает массив в следующем формате:
$aInfo[0] – True, если дескриптор ссылается на конец сервера, иначе конец клиента
$aInfo[1] – True для канала сообщений, иначе канала байтов
$aInfo[2] – размер буфера для исходящих данных в байтах
$aInfo[3] – размер буфера для входящих данных в байтах
$aInfo[4] – максимальное количество экземпляров канала, которые можно создать
Ошибка: Устанавливает @error ≠ 0.
