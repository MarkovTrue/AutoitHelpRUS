Справочник функций

_WinNet_CancelConnection

Отменяет существующее сетевое подключение.

#include <WinNet.au3>
_WinNet_CancelConnection ( $sName [, $bForce = True] )

Параметры

$sName Имя переадресованного локального устройства или удалённого сетевого ресурса для отключения. Когда этот параметр указывает переадресованное локальное устройство, функция отменяет только указанную переадресацию устройства. Если параметр указывает удалённый сетевой ресурс, отменяются только подключения к удалённым сетям без устройств.
$bForce [необязательный] Указывает, должно ли отключение происходить, если в подключении есть открытые файлы или задачи. Если этот параметр равен False, функция завершается ошибкой при наличии открытых файлов или задач.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция предоставляется для совместимости с 16-разрядными версиями Windows. Другие приложения на базе Windows должны вызывать функцию _WinNet_CancelConnection2().
