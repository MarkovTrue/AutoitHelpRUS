_WinAPI_GetTimeFormat

Форматирует время как строку времени для локали, указанной идентификатором.

#include <WinAPILocale.au3>
_WinAPI_GetTimeFormat ( [$iLCID = 0 [, $tSYSTEMTIME = 0 [, $iFlags = 0 [, $sFormat = '']]]] )

Параметры

$iLCID Идентификатор локали (LCID) – задаёт локаль либо одно из предопределённых значений:
$LOCALE_INVARIANT
$LOCALE_SYSTEM_DEFAULT
$LOCALE_USER_DEFAULT

Windows Vista и выше
$LOCALE_CUSTOM_DEFAULT
$LOCALE_CUSTOM_UI_DEFAULT
$LOCALE_CUSTOM_UNSPECIFIED
$tSYSTEMTIME Структура $tagSYSTEMTIME.
$iFlags [необязательный] флаги, которые задают параметры формата времени.
Этот параметр может быть одним или несколькими из следующих значений.
$TIME_FORCE24HOURFORMAT
$TIME_NOMINUTESORSECONDS
$TIME_NOSECONDS
$TIME_NOTIMEMARKER
$sFormat [необязательный] строка, которая используется для формирования времени. Например "hh:mm:ss tt". Если этот параметр
пропущен или пустая строка (по умолчанию), функция возвращает строку в соответствии с форматом времени
для указанной локали.

Возвращаемое значение

Успех: Возвращает отформатированная строка времени.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Пример

#include <WinAPILocale.au3>

ConsoleWrite(_WinAPI_GetTimeFormat() & @CRLF)
ConsoleWrite(_WinAPI_GetTimeFormat(0, 0, BitOR($TIME_FORCE24HOURFORMAT, $TIME_NOSECONDS)) & @CRLF)
