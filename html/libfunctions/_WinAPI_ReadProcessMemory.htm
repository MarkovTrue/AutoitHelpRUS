_WinAPI_ReadProcessMemory

Читает память в указанном процессе.

#include <WinAPIMem.au3>
_WinAPI_ReadProcessMemory ( $hProcess, $pBaseAddress, $pBuffer, $iSize, ByRef $iRead )

Параметры

$hProcess Идентифицирует открытый дескриптор процесса, из памяти которого осуществляется чтение.
$pBaseAddress На базовый адрес в указанном процессе для чтения.
$pBuffer На буфер, который получает содержимое из адресного пространства.
$iSize Требуемое количество байтов для чтения из указанного процесса.
$iRead Фактическое количество байтов, переданное в указанный буфер.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
