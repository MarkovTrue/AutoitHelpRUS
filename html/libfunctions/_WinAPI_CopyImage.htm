Справочник функций

_WinAPI_CopyImage

Создаёт новое изображение (значок, курсор или растр) и копирует атрибуты указанного изображения в новое.

#include <WinAPIGdi.au3>
_WinAPI_CopyImage ( $hImage [, $iType = 0 [, $iXDesiredPixels = 0 [, $iYDesiredPixels = 0 [, $iFlags = $LR_DEFAULTCOLOR]]]] )

Параметры

$hImage Дескриптор изображения, которое должно быть скопировано.
$iType [необязательный] Задаёт тип копируемого изображения. Этот параметр может быть одним из следующих значений:
$IMAGE_BITMAP (по умолчанию)
$IMAGE_CURSOR
$IMAGE_ICON
$iXDesiredPixels [необязательный] Задаёт желаемую ширину изображения в пиксельях. Если это ноль (по умолчанию), то возвращаемое изображение будет иметь ту же ширину, что и исходное $hImage.
$iYDesiredPixels [необязательный] Задаёт желаемую высоту изображения в пиксельях. Если это ноль (по умолчанию), то возвращаемое изображение будет иметь ту же высоту, что и исходное $hImage.
$iFlags [необязательный] Этот параметр может быть одним или несколькими из следующих значений.
$LR_DEFAULTCOLOR (по умолчанию)
$LR_COPYDELETEORG
$LR_COPYFROMRESOURCE
$LR_COPYRETURNORG
$LR_CREATEDIBSECTION
$LR_DEFAULTSIZE
$LR_MONOCHROME

Возвращаемое значение

Успех: Возвращает дескриптор вновь созданного изображения.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

По завершении работы с ресурсом вы можете освободить связанную с ним память, вызвав функцию _WinAPI_DeleteObject() соответственно для растров, курсоров или значков.
Система автоматически удаляет ресурс при завершении процесса, однако вызов соответствующей функции экономит память и уменьшает размер рабочего набора процесса.
