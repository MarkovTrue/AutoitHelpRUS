_WinAPI_ReleaseCapture

Отпускает захват мыши из окна в текущем потоке и восстанавливает обычную обработку входных данных мыши.

#include <WinAPISys.au3>
_WinAPI_ReleaseCapture ( )

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
