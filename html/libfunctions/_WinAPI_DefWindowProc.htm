_WinAPI_DefWindowProc

Вызывает оконную процедуру по умолчанию для стандартной обработки сообщений.

#include <WinAPISysWin.au3>
_WinAPI_DefWindowProc ( $hWnd, $iMsg, $wParam, $lParam )

Параметры

$hWnd Дескриптор окна.
$iMsg Задаёт сообщение.
$wParam Задаёт дополнительные данные сообщения.
$lParam Задаёт дополнительные данные сообщения.

Возвращаемое значение

Возвращает результат обработки сообщения; зависит от самого сообщения.
