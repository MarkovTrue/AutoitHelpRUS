_WinAPI_ExtractIconEx

Создаёт массив дескрипторов больших или маленьких значков, извлечённых из файла.

#include <WinAPIIcons.au3>
_WinAPI_ExtractIconEx ( $sFilePath, $iIndex, $paLarge, $paSmall, $iIcons )

Параметры

$sFilePath Имя исполняемого файла, DLL или файла значков, из которого извлекаются значки.
$iIndex Индекс первого извлекаемого значка (нумерация с 0).
$paLarge Указатель на массив дескрипторов значков, который принимает дескрипторы больших значков, извлечённых из файла.
При 0 большие значки не извлекаются.
$paSmall Указатель на массив дескрипторов значков, который принимает дескрипторы маленьких значков, извлечённых из файла.
При 0 маленькие значки не извлекаются.
$iIcons Количество извлекаемых из файла значков.

Возвращаемое значение

Возвращает количество значков в указанном файле, если $iIndex = -1, $paLarge = 0 и $paSmall = 0.
Иначе – количество успешно извлечённых значков.

Дополнительно

Искать ExtractIconEx в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIIcons.au3>

MsgBox($MB_SYSTEMMODAL, "ExtractIconEx", "# of Icons in file shell32.dll: " & _WinAPI_ExtractIconEx("shell32.dll", -1, 0, 0, 0))
