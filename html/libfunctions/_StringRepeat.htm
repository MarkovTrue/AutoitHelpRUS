Справочник функций

_StringRepeat

Повторяет строку указанное количество раз.

#include <String.au3>
_StringRepeat ( $sString, $iRepeatCount )

Параметры

$sString Строка для повтора.
$iRepeatCount Количество повторений строки.

Возвращаемое значение

Успех: Возвращает строка с указанным количеством повторений.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@error: 1 – Один из параметров недопустим.

Пример

#include <MsgBoxConstants.au3>
#include <String.au3>

; Повторить строку "+-" сорок раз.
MsgBox($MB_SYSTEMMODAL, '', _StringRepeat("+-", 40))
