Справочник функций

_WinAPI_GetThemeMetric

Получает значение свойства метрики.

#include <WinAPITheme.au3>
_WinAPI_GetThemeMetric ( $hTheme, $iPartID, $iStateID, $iPropID [, $hDC = 0] )

Параметры

$hTheme Дескриптор данных темы для окна.
$iPartID Часть, которая содержит свойство метрики.
$iStateID Состояние части.
$iPropID Свойство для получения. Может быть одним из следующих значений.
$TMT_ALPHALEVEL
$TMT_ALPHATHRESHOLD
$TMT_BORDERSIZE
$TMT_GLYPHINDEX
$TMT_GRADIENTRATIO1
$TMT_GRADIENTRATIO2
$TMT_GRADIENTRATIO3
$TMT_GRADIENTRATIO4
$TMT_GRADIENTRATIO5
$TMT_HEIGHT
$TMT_IMAGECOUNT
$TMT_MINDPI1
$TMT_MINDPI2
$TMT_MINDPI3
$TMT_MINDPI4
$TMT_MINDPI5
$TMT_PROGRESSCHUNKSIZE
$TMT_PROGRESSSPACESIZE
$TMT_ROUNDCORNERWIDTH
$TMT_ROUNDCORNERHEIGHT
$TMT_SATURATION
$TMT_TEXTBORDERSIZE
$TMT_TRUESIZESTRETCHMARK
$TMT_WIDTH
$hDC Дескриптор контекста устройства.

Возвращаемое значение

Успех: Возвращает значение свойства метрики.
Ошибка: Устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
