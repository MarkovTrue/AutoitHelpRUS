_TicksToTime

Преобразует указанное количество тиков в часы, минуты и секунды.

#include <Date.au3>
_TicksToTime ( $iTicks, ByRef $iHours, ByRef $iMins, ByRef $iSecs )

Параметры

$iTicks Количество тиков.
$iHours Переменная для хранения часов.
$iMins Переменная для хранения минут.
$iSecs Переменная для хранения секунд.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $iTicks не является целым числом.

См. также

_TimeToTicks

Пример

; *** Демонстрация окна таймера

#include <Date.au3>
#include <GUIConstantsEx.au3>

Global $g_hTimer, $g_iSecs, $g_iMins, $g_iHour, $g_sTime

Example()

Func Example()
    ; Создать GUI
    GUICreate("Timer", 120, 50)
    GUICtrlCreateLabel("00:00:00", 10, 10)
    GUISetState(@SW_SHOW)
    ; Запустить таймер
    $g_hTimer = TimerInit()
    AdlibRegister("Timer", 50)
    While 1
        ; FileWriteLine("debug.log",@MIN & ":" & @SEC & " ==> before")
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                Exit
        EndSwitch
        ; FileWriteLine("debug.log",@MIN & ":" & @SEC & " ==> after")
    WEnd
EndFunc   ;==>Example

Func Timer()
    _TicksToTime(Int(TimerDiff($g_hTimer)), $g_iHour, $g_iMins, $g_iSecs)
    Local $sTime = $g_sTime ; сохранить текущее время для проверки и избежания мерцания
    $g_sTime = StringFormat("%02i:%02i:%02i", $g_iHour, $g_iMins, $g_iSecs)
    If $sTime <> $g_sTime Then ControlSetText("Timer", "", "Static1", $g_sTime)
EndFunc   ;==>Timer
