Справочник функций

_WinAPI_PathCreateFromUrl

Преобразует URL-адрес файла в путь MS-DOS компании Microsoft.

#include <WinAPIShPath.au3>
_WinAPI_PathCreateFromUrl ( $sUrl )

Параметры

$sUrl URL.

Возвращаемое значение

Успех: Возвращает путь MS-DOS.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Дополнительно

Искать PathCreateFromUrl в библиотеке MSDN

Пример

#include <WinAPIShPath.au3>

Local $sPath = @ScriptFullPath

$sPath = _WinAPI_UrlCreateFromPath($sPath)
ConsoleWrite($sPath & @CRLF)

$sPath = _WinAPI_PathCreateFromUrl($sPath)
ConsoleWrite($sPath & @CRLF)
