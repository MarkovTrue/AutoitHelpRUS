_WinAPI_GetModuleHandleEx

Получает дескриптор модуля для указанного модуля.

#include <WinAPISys.au3>
_WinAPI_GetModuleHandleEx ( $sModule [, $iFlags = $GET_MODULE_HANDLE_EX_FLAG_DEFAULT] )

Параметры

$sModule Имя загруженного модуля (файл .dll или .exe) или указатель на адрес в модуле
(если указан флаг $GET_MODULE_HANDLE_EX_FLAG_FROM_ADDRESS). Если этот параметр равен 0 или пустой
строке, функция возвращает дескриптор файла, используемого для создания вызывающего процесса (файл .exe).
$iFlags [необязательный] Этот параметр может быть одним или более из следующих значений.
$GET_MODULE_HANDLE_EX_FLAG_FROM_ADDRESS
$GET_MODULE_HANDLE_EX_FLAG_PIN
$GET_MODULE_HANDLE_EX_FLAG_UNCHANGED_REFCOUNT
$GET_MODULE_HANDLE_EX_FLAG_DEFAULT (по умолчанию).

Возвращаемое значение

Успех: Возвращает дескриптор указанного модуля.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
