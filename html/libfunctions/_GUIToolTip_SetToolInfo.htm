Справочник функций

_GUIToolTip_SetToolInfo

Устанавливает информацию об инструменте в элементе управления ToolTip.

#include <GuiToolTip.au3>
_GUIToolTip_SetToolInfo ( $hTool, $hWnd, $iID [, $sText = "" [, $iFlags = Default [, $iLeft = 0 [, $iTop = 0 [, $iRight = 0 [, $iBottom = 0 [, $hInst = 0 [, $iParam = 0]]]]]]]] )

Параметры

$hTool Дескриптор окна, содержащего инструмент, или 0.
$hWnd Дескриптор окна.
$iID Идентификатор инструмента или дескриптор окна элемента, к которому он привязан.
$sText [необязательный] текст для элемента управления ToolTip
$iFlags [необязательный] флаги управления отображением подсказки:
$TTF_IDISHWND – $iID содержит дескриптор окна или элемента управления вместо ID инструмента [по умолчанию]
$TTF_CENTERTIP – центрирует подсказку под элементом управления
$TTF_RTLREADING – текст отображается в противоположном направлении (справа налево)
$TTF_SUBCLASS – элемент управления выполняет подклассирование окна инструмента [по умолчанию]
$TTF_TRACK – позиционирует окно подсказки рядом с инструментом
$TTF_ABSOLUTE – позиционирует окно с координатами из TTM_TRACKPOSITION (см. раздел Примечания)
$TTF_TRANSPARENT – передаёт сообщения мыши окну-родителю
$TTF_PARSELINKS – обрабатывает ссылки в тексте подсказки как ссылки
По умолчанию: BitOr($TTF_SUBCLASS, $TTF_IDISHWND).
$iLeft [необязательный] Координата X левого верхнего угла прямоугольника
$iTop [необязательный] Координата Y левого верхнего угла прямоугольника
$iRight [необязательный] координата X нижнего правого угла прямоугольника
$iBottom [необязательный] Координата Y правого нижнего угла прямоугольника
$hInst [необязательный] дескриптор экземпляра, связанного с инструментом
$iParam [необязательный] Определяемое приложением значение

Возвращаемое значение

Возвращает см. раздел Примечания.

Примечания

Если элемент управления находится в другом процессе и оба процесса работают в разных режимах AutoIt (@AutoItVersion), @error будет установлена на 6.

Некоторые внутренние свойства инструмента устанавливаются при создании инструмента и не пересчитываются при отправке сообщения $TTM_SETTOOLINFO.
Если вы просто присваиваете значения структуре TOOLINFO и передаёте её элементу управления ToolTip с сообщением $TTM_SETTOOLINFO, эти свойства могут быть потеряны.
Вместо этого ваше приложение должно сначала запросить текущую структуру TOOLINFO инструмента, отправив сообщение $TTM_GETTOOLINFO.
Затем измените члены этой структуры по необходимости и передайте её обратно элементу управления ToolTip с $TTM_SETTOOLINFO.
При вызове $TTM_SETTOOLINFO строка, на которую указывает член Text структуры TOOLINFO, не должна превышать 80 символов.
