_WinAPI_SetProcessAffinityMask

Задаёт маску сродства процессора для потоков указанного процесса.

#include <WinAPIProc.au3>
_WinAPI_SetProcessAffinityMask ( $hProcess, $iMask )

Параметры

$hProcess Дескриптор процесса, маску сродства которого задаёт функция.
$iMask Маска сродства.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Маска сродства – это битовая маска, в которой каждый бит представляет процессор, на котором потокам процесса
разрешено работать. Например, если вы передаёте маску 0x05, процессорам 1 и 3 разрешено работать.
