_WinAPI_CopyCursor

Создаёт дубликат указанного курсора.

#include <WinAPIRes.au3>
_WinAPI_CopyCursor ( $hCursor )

Параметры

$hCursor Дескриптор курсора, который должен быть дублирован.

Возвращаемое значение

Успех: Возвращает дескриптор созданного нового курсора.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

По завершении работы с курсором удалите его, используя функцию _WinAPI_DestroyCursor().
