_WinAPI_PathIsSystemFolder

Определяет, содержит ли существующая папка атрибуты, которые делают её системной папкой.

#include <WinAPIShPath.au3>
_WinAPI_PathIsSystemFolder ( $sFilePath )

Параметры

$sFilePath Имя существующей папки для проверки атрибутов системной папки.

Возвращаемое значение

True: Возвращает путь является системной папкой.
False: путь не является системной папкой.
