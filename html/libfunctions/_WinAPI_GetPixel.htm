_WinAPI_GetPixel

Получает значение цвета пикселя в указанных координатах.

#include <WinAPIGdi.au3>
_WinAPI_GetPixel ( $hDC, $iX, $iY )

Параметры

$hDC Дескриптор контекста устройства.
$iX Координата X в логических единицах пикселя, подлежащего проверке.
$iY Координата Y в логических единицах пикселя, подлежащего проверке.

Возвращаемое значение

Успех: Возвращает цвет пикселя в формате RGB.
Ошибка: (-1).
