_WinAPI_SetUserObjectInformation

Задаёт сведения об указанном объекте станции окна или рабочего стола.

#include <WinAPISys.au3>
_WinAPI_SetUserObjectInformation ( $hObject, $iIndex, $tData )

Параметры

$hObject Дескриптор объекта станции окна или рабочего стола.
$iIndex Сведения об объекте для установки. Этот параметр может быть одним из следующих значений. $UOI_FLAGS.
$tData Данные сведений об объекте. Тип данных зависит от типа сведений. Дополнительные сведения см. в функции _WinAPI_GetUserObjectInformation().

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
