Справочник функций

_WinAPI_ShowLastError

Отображает последний код ошибки и сообщение.

#include <WinAPIError.au3>
_WinAPI_ShowLastError ( [$sText = '' [, $bAbort = False [, $iLanguage = 0]]] )

Параметры

$sText [необязательный] Текст пользователя, который должен отображаться с сообщением.
$bAbort [необязательный] Указывает, выходить ли из скрипта после отображения сообщения об ошибке, допустимые значения:
True - Выйти из скрипта после отображения сообщения, если оно указывает на ошибку.
False - Всегда нормально возвращаться (По умолчанию).
$iLanguage [необязательный] Идентификатор языка для сообщения.

Возвращаемое значение

Отсутствует.

Примечания

При возврате значения @error и @extended сохраняются.
