_NamedPipes_WaitNamedPipe

Ожидает, пока экземпляр именованного канала станет доступным.

#include <NamedPipes.au3>
_NamedPipes_WaitNamedPipe ( $sPipeName [, $iTimeOut = 0] )

Параметры

$sPipeName Имя именованного канала. Строка должна включать имя компьютера, на котором выполняется процесс сервера.
Для имени сервера может быть использована точка, если канал локальный.
$iTimeout [необязательный] Количество миллисекунд, в течение которого функция будет ждать доступности именованного канала.
Вы также можете использовать одно из следующих значений:
-1 – функция не возвращает значение до тех пор, пока экземпляр именованного канала не станет доступным
0 – интервал тайм-аута является значением по умолчанию, указанным процессом сервера

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Если не существует экземпляров указанного именованного канала, функция _NamedPipes_WaitNamedPipe() возвращает значение немедленно.
