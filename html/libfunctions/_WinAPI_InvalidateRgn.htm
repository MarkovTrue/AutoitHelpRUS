_WinAPI_InvalidateRgn

Добавляет область в область обновления указанного окна.

#include <WinAPIGdi.au3>
_WinAPI_InvalidateRgn ( $hWnd [, $hRgn = 0 [, $bErase = True]] )

Параметры

$hWnd Дескриптор окна.
$hRgn [необязательный] Дескриптор области, добавляемой в область обновления. Предполагается, что область имеет клиентские координаты.
Если этот параметр равен 0, вся клиентская область добавляется в область обновления.
$bErase [необязательный] Определяет, следует ли удалять фон в области обновления при обработке области обновления, допустимые значения:
True – фон удаляется (по умолчанию).
False – фон остаётся неизменным.

Возвращаемое значение

Всегда возвращает True.
