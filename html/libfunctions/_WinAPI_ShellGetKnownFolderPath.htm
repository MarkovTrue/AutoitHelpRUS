Справочник функций

_WinAPI_ShellGetKnownFolderPath

Получает полный путь известной папки.

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetKnownFolderPath ( $sGUID [, $iFlags = 0 [, $hToken = 0]] )

Параметры

$sGUID GUID ($FOLDERID_*), который определяет стандартные папки, зарегистрированные в системе.
$iFlags [необязательный] Флаги, определяющие специальные параметры получения. Этот параметр может быть одним или несколькими из следующих значений.
$KF_FLAG_CREATE
$KF_FLAG_DONT_VERIFY
$KF_FLAG_DONT_UNEXPAND
$KF_FLAG_NO_ALIAS
$KF_FLAG_INIT
$KF_FLAG_DEFAULT_PATH
$KF_FLAG_NOT_PARENT_RELATIVE
$KF_FLAG_SIMPLE_IDLIST

Windows 7 или более позднюю версию
$KF_FLAG_ALIAS_ONLY
$hToken [необязательный] Маркер доступа, представляющий конкретного пользователя. Если этот параметр равен 0, функция запрашивает
известную папку для текущего пользователя. Назначение параметру $hToken значения (-1) указывает на
пользователя по умолчанию. Обратите внимание, доступ к папкам пользователя по умолчанию требует привилегий администратора.

Возвращаемое значение

Успех: Возвращает путь известной папки.
Ошибка: Возвращает пустую строку и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.

Примечания

Для этой функции требуется Windows Vista или более поздняя версия.

Дополнительно

Искать SHGetKnownFolderPath в библиотеке MSDN

Пример

#include <MsgBoxConstants.au3>
#include <WinAPIShellEx.au3>
#include <WinAPISys.au3>

If Number(_WinAPI_GetVersion()) < 6.0 Then
    MsgBox(($MB_ICONERROR + $MB_SYSTEMMODAL), 'Error', 'Require Windows Vista or later.')
    Exit
EndIf

ConsoleWrite(_WinAPI_ShellGetKnownFolderPath($FOLDERID_SendTo, 0, -1) & @CRLF)
ConsoleWrite(_WinAPI_ShellGetKnownFolderPath($FOLDERID_SendTo) & @CRLF)

Local Const $sFOLDERID_wrong = '{8983036C-27C0-404B-8F08-102D10DCFD75}'
ConsoleWrite('"' & _WinAPI_ShellGetKnownFolderPath($sFOLDERID_wrong) & '" @error = ' & @error & ' @extended = 0x' & Hex(@extended) & @CRLF)
