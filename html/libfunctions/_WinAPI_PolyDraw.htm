_WinAPI_PolyDraw

Рисует набор отрезков линий и кривых Безье.

#include <WinAPIGdi.au3>
_WinAPI_PolyDraw ( $hDC, Const ByRef $aPoint [, $iStart = 0 [, $iEnd = -1]] )

Параметры

$hDC Дескриптор контекста устройства.
$aPoint Двумерный массив ([x1, y1, type1], [x2, y2, type2], ... [xN, yN, typeN]), который содержит конечные точки для
каждого отрезка линии и конечные точки и контрольные точки для каждой кривой Безье в логических единицах. Кроме того,
массив содержит параметр, который указывает, как используется каждая точка. Третий параметр массива
может быть одной из следующих констант.
$PT_BEZIERTO
$PT_LINETO
$PT_MOVETO

$PT_BEZIERTO или $PT_LINETO тип может быть объединён со следующей константой, которая означает, что соответствующая точка
является последней точкой фигуры и фигура закрыта.

$PT_CLOSEFIGURE
$iStart [необязательный] Индекс массива для начала рисования.
$iEnd [необязательный] Индекс массива для остановки рисования.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Эта функция может использоваться вместо последовательных вызовов _WinAPI_MoveToEx(), _WinAPI_LineTo(), и _WinAPI_PolyBezierTo()
для рисования несвязанных фигур. Линии и кривые рисуются с использованием текущего пера, и фигуры не заполняются.
Если существует активный путь, запущенный вызовом _WinAPI_BeginPath(), _WinAPI_PolyDraw() добавляет к пути.

Эта функция обновляет текущую позицию.

См. также

_WinAPI_BeginPath, _WinAPI_LineTo, _WinAPI_MoveToEx, _WinAPI_PolyBezierTo
