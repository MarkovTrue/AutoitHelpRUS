_WinAPI_PathIsLFNFileSpec

Определяет, находится ли имя файла в длинном формате.

#include <WinAPIShPath.au3>
_WinAPI_PathIsLFNFileSpec ( $sFilePath )

Параметры

$sFilePath Имя файла для проверки.

Возвращаемое значение

True: Возвращает файл превышает количество символов, допустимых формате 8.3.
False: файл находится в формате 8.3.
