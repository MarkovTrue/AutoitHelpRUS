_NamedPipes_TransactNamedPipe

Выполняет чтение и запись в именованный канал в одной сетевой операции.

#include <NamedPipes.au3>
_NamedPipes_TransactNamedPipe ( $hNamedPipe, $pInpBuf, $iInpSize, $pOutBuf, $iOutSize [, $tOverlapped = 0] )

Параметры

$hNamedPipe Дескриптор именованного канала.
$pInpBuf Указатель на буфер, содержащий данные для записи в канал.
$iInpSize Размер буфера записи в байтах.
$pOutBuf Указатель на буфер, который получает данные, прочитанные из канала.
$iOutSize Размер буфера чтения в байтах.
$tOverlapped [необязательный] структура $tagOVERLAPPED.
Эта структура требуется, если $hNamedPipe был открыт с $PIPE_FLAG_OVERLAPPED.
Если $hNamedPipe был открыт с $PIPE_FLAG_OVERLAPPED, $tOverlapped не должен быть 0.
Если $hNamedPipe был открыт с $PIPE_FLAG_OVERLAPPED и $tOverlapped не равен 0, функция _NamedPipes_TransactNamedPipe() выполняется как операция с перекрытием.
Структура $tagOVERLAPPED должна содержать объект события с ручным сбросом.
Если операция не может быть завершена немедленно, функция _NamedPipes_TransactNamedPipe() возвращает False, и _WinAPI_GetLastError() возвращает ERROR_IO_PENDING.

Возвращаемое значение

Успех: Возвращает количество байтов, прочитанных из канала.
Ошибка: Устанавливает @error ≠ 0.

Примечания

TransactNamedPipe не удаётся выполнить, если сервер не создал канал как канал типа сообщений или если дескриптор канала не находится в режиме чтения сообщений.
