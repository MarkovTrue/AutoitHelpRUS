Справочник функций

_WinAPI_OpenProcessToken

Открывает маркер доступа, связанный с процессом.

#include <WinAPIProc.au3>
_WinAPI_OpenProcessToken ( $iAccess [, $hProcess = 0] )

Параметры

$iAccess Маска доступа, которая определяет запрашиваемые типы доступа к маркеру доступа.
Этот параметр может быть одним или несколькими из следующих значений:
$TOKEN_ALL_ACCESS
$TOKEN_ADJUST_DEFAULT
$TOKEN_ADJUST_GROUPS
$TOKEN_ADJUST_PRIVILEGES
$TOKEN_ADJUST_SESSIONID
$TOKEN_ASSIGN_PRIMARY
$TOKEN_DUPLICATE
$TOKEN_EXECUTE
$TOKEN_IMPERSONATE
$TOKEN_QUERY
$TOKEN_QUERY_SOURCE
$TOKEN_READ
$TOKEN_WRITE
$hProcess [необязательный] Дескриптор процесса, чей маркер доступа открывается.
Процесс должен иметь разрешение доступа $PROCESS_QUERY_INFORMATION.
Если этот параметр равен 0 (по умолчанию), будет использоваться текущий процесс.

Возвращаемое значение

Успех: Возвращает дескриптор, идентифицирующий вновь открытый маркер доступа.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Закройте дескриптор маркера доступа, возвращённый этой функцией, вызвав _WinAPI_CloseHandle().

См. также

_WinAPI_CloseHandle

Дополнительно

Искать OpenProcessToken в библиотеке MSDN

Пример

#include <Debug.au3>
#include <SecurityConstants.au3>
#include <WinAPIError.au3>
#include <WinAPIHObj.au3>
#include <WinAPIProc.au3>
#include <WinAPIReg.au3>

#RequireAdmin

_DebugSetup(Default, True)

Example()

Func Example()
    Local $aPrivileges[2] = [$SE_BACKUP_NAME, $SE_RESTORE_NAME]

    ; Включить привилегии "SeBackupPrivilege" и "SeRestorePrivilege" для сохранения и восстановления куста реестра
    Local $hToken = _WinAPI_OpenProcessToken(BitOR($TOKEN_ADJUST_PRIVILEGES, $TOKEN_QUERY))
    Local $aAdjust
    _WinAPI_AdjustTokenPrivileges($hToken, $aPrivileges, $SE_PRIVILEGE_ENABLED, $aAdjust)
    If @error Or @extended Then ; определить, успешно ли функция изменила все указанные привилегии
        _DebugReport('Error' & @TAB & 'You do not have the required privileges.' & @CRLF)
        Exit
    EndIf

    ; Сохранить "HKEY_CURRENT_USER\Software\AutoIt v3" в reg.dat
    Local $hKey = _WinAPI_RegOpenKey($HKEY_CURRENT_USER, 'Software\AutoIt v3', $KEY_READ)
    If _WinAPI_RegSaveKey($hKey, @TempDir & '\reg.dat', 1) Then
        _DebugReport('- "HKEY_CURRENT_USER\Software\AutoIt v3" has been saved to reg.dat.' & @CRLF)
    Else
        _DebugReport("! RegSaveKey @error =" & @error & @TAB & _WinAPI_GetErrorMessage(@extended) & @CRLF)
    EndIf
    _WinAPI_RegCloseKey($hKey)

    ; Восстановить "HKEY_CURRENT_USER\Software\AutoIt v3" в "HKEY_CURRENT_USER\Software\AutoIt v3 (Duplicate)"
    $hKey = _WinAPI_RegCreateKey($HKEY_CURRENT_USER, 'Software\AutoIt v3 (Duplicate)', $KEY_WRITE)
    If _WinAPI_RegRestoreKey($hKey, @TempDir & '\reg.dat') Then
        _DebugReport('- "HKEY_CURRENT_USER\Software\AutoIt v3" has been restored to "HKEY_CURRENT_USER\Software\AutoIt v3 (Duplicate)".' & @CRLF)
    Else
        _DebugReport("! RegRestoreKey @error =" & @error & @TAB & _WinAPI_GetErrorMessage(@extended) & @CRLF)
    EndIf
    _WinAPI_RegCloseKey($hKey)

    ; Восстановить привилегии "SeBackupPrivilege" и "SeRestorePrivilege" в значения по умолчанию
    _WinAPI_AdjustTokenPrivileges($hToken, $aAdjust, 0, $aAdjust)
    _WinAPI_CloseHandle($hToken)

    FileDelete(@TempDir & '\reg.dat')

EndFunc   ;==>Example
