_WinAPI_BackupSeek

Перемещается вперёд в потоке данных, изначально доступном через функцию _WinAPI_BackupRead() или _WinAPI_BackupWrite().

#include <WinAPIFiles.au3>
_WinAPI_BackupSeek ( $hFile, $iSeek, ByRef $iBytes, ByRef $pContext )

Параметры

$hFile Дескриптор файла или папки.
$iSeek Количество байтов для перемещения.
$iBytes Количество байтов, на которое функция фактически переместилась.
$pContext Указатель на внутреннюю структуру данных. Эта структура должна быть той же структурой, которая была инициализирована функцией _WinAPI_BackupRead().
Приложение не должно изменять содержимое этой структуры.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Приложения используют _WinAPI_BackupSeek() для пропуска частей потока данных, которые вызывают ошибки.
Эта функция не перемещается через заголовки потока.
