Справочник функций

_WinAPI_SetErrorMode

Управляет тем, будет ли система обрабатывать указанные типы серьёзных ошибок или будет обрабатывать их процесс.

#include <WinAPIDiag.au3>
_WinAPI_SetErrorMode ( $iMode )

Параметры

$iMode Режим ошибки процесса. Этот параметр может быть одним или несколькими из следующих значений.
$SEM_FAILCRITICALERRORS
$SEM_NOALIGNMENTFAULTEXCEPT
$SEM_NOGPFAULTERRORBOX
$SEM_NOOPENFILEERRORBOX

Возвращаемое значение

Возвращает предыдущее состояние флагов режима ошибок ($SEM_*).
