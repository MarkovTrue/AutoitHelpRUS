Справочник функций

_GUICtrlMenu_EnableMenuItem

Включает, отключает или выделяет серым указанный пункт меню.

#include <GuiMenu.au3>
_GUICtrlMenu_EnableMenuItem ( $hMenu, $iItem [, $iState = 0 [, $bByPos = True]] )

Параметры

$hMenu Дескриптор меню.
$iItem Идентификатор или позиция пункта меню.
$iState [необязательный] Состояние пункта меню:
0 – включён ($MF_ENABLED)
1 – серый ($MF_GRAYED)
2 – отключён ($MF_DISABLED)
$bByPos [необязательный] Флаг идентификатора меню:
True – $iItem задаёт позицию элемента (начиная с нуля)
False – $iItem задаёт идентификатор элемента меню.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Дополнительно

Искать EnableMenuItem в библиотеке MSDN

Пример

#include <GuiMenu.au3>

Example()

Func Example()
    Local $hWnd, $hMain

    Run("notepad.exe")
    WinWaitActive("[CLASS:Notepad]")
    $hWnd = WinGetHandle("[CLASS:Notepad]")
    $hMain = _GUICtrlMenu_GetMenu($hWnd)

    ; Отключить/Выделить серым пункт "Справка"
    _GUICtrlMenu_EnableMenuItem($hMain, 4, 3)
EndFunc   ;==>Example
