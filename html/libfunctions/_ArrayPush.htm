Справочник функций

_ArrayPush

Добавляет новые значения без увеличения размера массива: вставляет новое значение в конец и удаляет первый элемент – или наоборот.

#include <Array.au3>
_ArrayPush ( ByRef $aArray, $vValue [, $iDirection = 0] )

Параметры

$aArray Изменяемый массив.
$vValue Добавляемое значение или значения (можно передать массивом).
$iDirection [необязательный] Направление сдвига существующих элементов:
0 = сдвиг влево, добавление в конец (по умолчанию)
1 = сдвиг вправо, добавление в начало

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 1 – $aArray не массив
2 – $vValue – массив, размер которого превышает $aArray
3 – $aArray не 1D массив

Примечания

Функция используется для непрерывного обновления данных в массиве без его роста.
Она сохраняет все значения внутри массива (по принципу истории), вытесняя первый или последний элемент в зависимости от выбранного направления.
Аналог команды push в ассемблере.

См. также

_ArrayAdd, _ArrayConcatenate, _ArrayDelete, _ArrayInsert, _ArrayPop

Пример

#include <Array.au3>

Local $avArrayTarget[9] = [1, 2, 3, 4, 5, 6, 7, 8, 9]
Local $avArraySource[2] = [100, 200]

_ArrayDisplay($avArrayTarget, "$avArrayTarget BEFORE _ArrayPush()")
_ArrayPush($avArrayTarget, $avArraySource)
_ArrayDisplay($avArrayTarget, "$avArrayTarget AFTER _ArrayPush() array to end")
_ArrayPush($avArrayTarget, $avArraySource, 1)
_ArrayDisplay($avArrayTarget, "$avArrayTarget AFTER _ArrayPush() array to beginning")
_ArrayPush($avArrayTarget, "Hello world!", 1)
_ArrayDisplay($avArrayTarget, "$avArrayTarget AFTER _ArrayPush() string to beginning")
