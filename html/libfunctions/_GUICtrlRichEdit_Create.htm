Справочник функций

_GUICtrlRichEdit_Create

Создаёт элемент управления RichEdit.

#include <GuiRichEdit.au3>
_GUICtrlRichEdit_Create ( $hWnd, $sText, $iLeft, $iTop [, $iWidth = 150 [, $iHeight = 150 [, $iStyle = -1 [, $iExStyle = -1]]]] )

Параметры

$hWnd Дескриптор родительского или владеющего окна.
$sText Текст для отображения в элементе управления.
$iLeft Горизонтальная позиция элемента управления.
$iTop Вертикальная позиция элемента управления.
$iWidth [необязательный] Ширина элемента управления
$iHeight [необязательный] Высота элемента управления
$iStyle [необязательный] Стили элемента:
$ES_AUTOHSCROLL – автоматически прокручивает текст вправо на 10 символов при наборе в конце строки.
$ES_AUTOVSCROLL – автоматически прокручивает текст на страницу вверх, когда пользователь нажимает Enter на последней строке.
$WS_HSCROLL – горизонтальная полоса прокрутки.
$WS_VSCROLL – вертикальная полоса прокрутки.
$ES_CENTER – центрирует текст.
$ES_LEFT – выравнивает текст по левому краю.
$ES_MULTILINE – многострочный элемент (по умолчанию).
$ES_NOHIDESEL – выделенный текст показывается инверсно даже без фокуса.
$ES_NUMBER – допускается ввод только цифр.
$ES_READONLY – запрещает пользователю редактировать текст.
$ES_RIGHT – выравнивает текст по правому краю.
$ES_WANTRETURN – вставляет символ перевода каретки при нажатии Enter (по умолчанию).
$ES_PASSWORD – вместо каждого вводимого символа отображает звёздочку (*).

По умолчанию: 0
Принудительно: WS_CHILD, $WS_VISIBLE, $WS_TABSTOP, кроме случая $ES_READONLY.
$iExStyle [необязательный] Расширенный стиль элемента. Соответствует стандартным константам $WS_EX_*. См. таблицу расширенных стилей.

Возвращаемое значение

Успех: Возвращает дескриптор элемента управления Rich Edit.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 105 – параметр неверного типа
106 – параметр неверного типа
107 – параметр неверного типа
108 – параметр неверного типа

_GUICtrlRichEdit_Destroy

Пример

#include <GUIConstantsEx.au3>
#include <GuiRichEdit.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hGui, $hRichEdit, $iMsg
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName, StringLen(".exe")) & ")", 320, 350, -1, -1)
    $hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 300, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    _GUICtrlRichEdit_AppendText($hRichEdit, @CRLF & "This is more text")
    GUISetState(@SW_SHOW)

    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                _GUICtrlRichEdit_Destroy($hRichEdit) ; нужно, если только скрипт не упадёт
                ; GUIDelete() ; так тоже работает
                Exit
        EndSelect
    WEnd
EndFunc   ;==>Example
