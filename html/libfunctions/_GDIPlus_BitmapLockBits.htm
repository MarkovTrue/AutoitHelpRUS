Справочник функций

_GDIPlus_BitmapLockBits

Блокирует часть растрового изображения для чтения или записи.

#include <GDIPlus.au3>
_GDIPlus_BitmapLockBits ( $hBitmap, $iLeft, $iTop, $iWidth, $iHeight [, $iFlags = $GDIP_ILMREAD [, $iFormat = $GDIP_PXF32RGB]] )

Параметры

$hBitmap Дескриптор объекта Bitmap.
$iLeft Координата X левого верхнего угла блокируемой области.
$iTop Координата Y левого верхнего угла блокируемой области.
$iWidth Ширина блокируемой области.
$iHeight Высота блокируемой области.
$iFlags [необязательный] Набор флагов, определяющих режим доступа к заблокированной части и наличие пользовательского буфера. Допускается сочетание следующих значений:
$GDIP_ILMREAD – часть изображения заблокирована для чтения
$GDIP_ILMWRITE – часть изображения заблокирована для записи
$GDIP_ILMUSERINPUTBUF – буфер выделен пользователем.
$iFormat [необязательный] Формат пиксельных данных во временном буфере. Одно из следующих значений:
$GDIP_PXF01INDEXED – 1 бит/пикс., индексированный
$GDIP_PXF04INDEXED – 4 бит/пикс., индексированный
$GDIP_PXF08INDEXED – 8 бит/пикс., индексированный
$GDIP_PXF16GRAYSCALE – 16 бит/пикс., оттенки серого
$GDIP_PXF16RGB555 – 16 бит/пикс.; по 5 бит на каждую составляющую RGB
$GDIP_PXF16RGB565 – 16 бит/пикс.; 5 бит красный, 6 бит зелёный, 5 бит синий
$GDIP_PXF16ARGB1555 – 16 бит/пикс.; 1 бит на альфа-канал и 5 бит на каждую составляющую RGB
$GDIP_PXF24RGB – 24 бит/пикс.; по 8 бит на каждую составляющую RGB
$GDIP_PXF32RGB – 32 бит/пикс.; по 8 бит на каждую составляющую RGB. Без альфа-канала.
$GDIP_PXF32ARGB – 32 бит/пикс.; по 8 бит на каждую составляющую RGB и альфа-канал
$GDIP_PXF32PARGB – 32 бит/пикс.; по 8 бит на составляющие RGB и альфа-канал, предумноженные.

Возвращаемое значение

Успех: Возвращает структуру $tagGDIPBITMAPDATA.
Ошибка: Устанавливает @error ≠ 0.

Примечания

После завершения работы с заблокированной областью вызовите _GDIPlus_BitmapUnlockBits() для её освобождения.

См. также

_GDIPlus_ImageGetPixelFormat, _WinAPI_DeleteObject

Дополнительно

Искать GdipBitmapLockBits в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Поддержка работы в X64
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"

    ;получить каталог установки AutoIt
    Local $sRegPath = "HKLM\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt"

    Local $sFile = RegRead($sRegPath, "InstallDir") & "\Examples\GUI\logo4.gif"
    If Not FileExists($sFile) Then
        MsgBox(($MB_SYSTEMMODAL + $MB_ICONHAND), "", $sFile & " not found!", 30)
        Return False
    EndIf

    _GDIPlus_Startup()
    Local $hImage = _GDIPlus_ImageLoadFromFile($sFile) ;создать объект изображения на основе файла
    If @error Then
        _GDIPlus_Shutdown()
        MsgBox(($MB_SYSTEMMODAL + $MB_ICONHAND), "", "An error has occured - unable to load image!", 30)
        Return False
    EndIf

    Local $iW = _GDIPlus_ImageGetWidth($hImage), $iH = _GDIPlus_ImageGetHeight($hImage) ; Получить ширину и высоту изображения.
    Local $hBitmap = _GDIPlus_BitmapCreateFromScan0($iW, $iH)
    Local $hContext = _GDIPlus_ImageGetGraphicsContext($hBitmap)
    _GDIPlus_GraphicsDrawImageRect($hContext, $hImage, 0, 0, $iW, $iH)

    Local $tBitmapData = _GDIPlus_BitmapLockBits($hBitmap, 0, 0, $iW, $iH, BitOR($GDIP_ILMWRITE, $GDIP_ILMREAD), $GDIP_PXF32ARGB) ; Заблокировать часть изображения для чтения и записи.
    Local $iScan0 = DllStructGetData($tBitmapData, "Scan0") ; Получить scan0 (пиксельные данные) из заблокированного изображения.
    Local $iSearchPixel = Int(0xFF000080), $iReplaceColor = 0x00000000 ; Формат цвета 0xAARRGGBB.
    Local $tPixel = DllStructCreate("int[" & $iW * $iH & "];", $iScan0)
    Local $iPixel, $iRowOffset

    For $iY = 0 To $iH - 1
        $iRowOffset = $iY * $iW + 1
        For $iX = 0 To $iW - 1 ; Перебрать каждый пиксель в каждой строке и столбце.
            $iPixel = DllStructGetData($tPixel, 1, $iRowOffset + $iX) ; Получить цвет пикселя.
            If $iPixel = $iSearchPixel Then DllStructSetData($tPixel, 1, $iReplaceColor, $iRowOffset + $iX) ; Сравнить и заменить цвет пикселя (синий на прозрачный).
        Next
    Next
    _GDIPlus_BitmapUnlockBits($hBitmap, $tBitmapData) ; Разблокировать часть изображения, заблокированную _GDIPlus_BitmapLockBits.

    ; Для сохранения изменённого изображения используйте _GDIPlus_ImageSaveToFile().

    ShellExecute($sFile) ; Отобразить исходное изображение для сравнения с изменённым.

    ; Отобразить изменённое изображение.
    Local $hGUI = GUICreate("_GDIPlus_BitmapLockBits Demo", $iW, $iH)
    GUISetState(@SW_SHOW)

    Local $hGraphic = _GDIPlus_GraphicsCreateFromHWND($hGUI) ;создать объект Graphics из дескриптора окна
    _GDIPlus_GraphicsDrawImageRect($hGraphic, $hBitmap, 0, 0, $iW, $iH) ; Скопировать изменённое изображение в дескриптор Graphics.

    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ;освободить ресурсы
    _GDIPlus_ImageDispose($hImage)
    _GDIPlus_GraphicsDispose($hContext)
    _GDIPlus_BitmapDispose($hBitmap)
    _GDIPlus_GraphicsDispose($hGraphic)
    _GDIPlus_Shutdown()
    GUIDelete($hGUI)
EndFunc   ;==>Example
