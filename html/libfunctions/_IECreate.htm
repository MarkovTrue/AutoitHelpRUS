Справочник функций

_IECreate

Создаёт окно браузера Internet Explorer.

#include <IE.au3>
_IECreate ( [$sUrl = "about:blank" [, $iTryAttach = 0 [, $iVisible = 1 [, $iWait = 1 [, $iTakeFocus = 1]]]]] )

Параметры

$sUrl [необязательный] указывает URL, на который переходить при создании
$iTryAttach [необязательный] указывает, пытаться ли подключиться к существующему окну
0 = (по умолчанию) не пытаться подключиться
1 = пытаться подключиться к существующему окну
$iVisible [необязательный] указывает, будет ли окно браузера видимым
0 = окно браузера скрыто
1 = (по умолчанию) окно браузера видимо
$iWait [необязательный] Ждать ли загрузку страницы перед возвратом:
0 – вернуться сразу, не ожидая загрузки
1 – (по умолчанию) дождаться полной загрузки и вернуться
$iTakeFocus [необязательный] указывает, нужно ли привести окно в фокус
0 = не приводить окно в фокус
1 = (по умолчанию) привести окно в фокус

Возвращаемое значение

Успех: Возвращает объектную переменную, указывающую на объект InternetExplorer.Application.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) - общая ошибка
3 ($_IEStatus_InvalidDataType) - неверный тип данных
4 ($_IEStatus_InvalidObjectType) - неверный тип объекта
6 ($_IEStatus_LoadWaitTimeout) - тайм-аут ожидания загрузки
8 ($_IEStatus_AccessIsDenied) - доступ запрещён
9 ($_IEStatus_ClientDisconnected) - клиент отключился
@extended: установлено в true (1) или false (0) в зависимости от успеха $iTryAttach

Примечания

При использовании tryAttach сообщение об ошибке No Match будет отображено в консоли из _IEAttach().
Это можно контролировать с помощью _IEErrorNotify().

При открытии локального файла HyperTextApplication (.hta) Internet Explorer откроет новый процесс браузера для него, который не будет доступен через переменную объекта, возвращаемую _IECreate().
Для управления этим новым браузером необходимо использовать _IEAttach() с опцией "embedded".

Новая безопасность в Windows Vista приводит к созданию нового окна браузера, когда браузеру даётся указание перейти на URL в другую зону безопасности.
Это происходит и при начальном создании и навигации, инициированной _IECreate().
Новое окно – это новый экземпляр браузера, и предыдущая переменная объекта браузера больше на него не указывает.
Есть несколько решений:
    1) добавить #RequireAdmin в свой код (это требуется даже если учётная запись входит в группу администраторов и будет запрашивать учётные данные при необходимости),
    2) использовать _IEAttach() для подключения к новому окну браузера
    3) добавить целевой веб-сайт в зону доверенных сайтов в IE,
    4) отключить "защищённый режим" в IE,
    или 5) отключить UAC. Необходимо проявлять осторожность при отключении функций безопасности IE при доступе к ненадёжным сайтам.

См. также

_IEAttach, _IECreateEmbedded, _IELoadWait, _IENavigate, _IEQuit

Пример

Пример 1

; Создать окно браузера и перейти на веб-сайт

#include <IE.au3>

Local $oIE = _IECreate("www.autoitscript.com")

Пример 2

; Создать новые окна браузера, указывающие на каждый из 3 разных URL
; если он ещё не существует ($f_tryAttach = 1)
; не ожидать завершения загрузки страниц ($f_wait = 0)

#include <IE.au3>

_IECreate("www.autoitscript.com", 1, 1, 0)
_IECreate("my.yahoo.com", 1, 1, 0)
_IECreate("www.google.com", 1, 1, 0)

Пример 3

; Попытаться подключиться к существующему браузеру, отображающему конкретный URL сайта
; Создать новый браузер и перейти на этот сайт, если его ещё нет

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IECreate("www.autoitscript.com", 1)
; Проверить значение возврата @extended, чтобы увидеть, была ли подключение успешным
If @extended Then
    MsgBox($MB_SYSTEMMODAL, "", "Attached to Existing Browser")
Else
    MsgBox($MB_SYSTEMMODAL, "", "Created New Browser")
EndIf

Пример 4

; Создать пустое окно браузера и заполнить его пользовательским HTML

#include <IE.au3>

Local $oIE = _IECreate()
Local $sHTML = "<h1>Hello World!</h1>"
_IEBodyWriteHTML($oIE, $sHTML)

Пример 5

; Создать окно браузера, подключённое к новому экземпляру iexplore.exe
; Это часто необходимо для получения контекста нового файла cookie сеанса
; (cookie сеанса общие для всех экземпляров браузера, использующих одинаковый iexplore.exe)

#include <IE.au3>

Opt("WinTitleMatchMode", 2)

ShellExecute("iexplore.exe", "about:blank")
WinWait("Windows Internet Explorer")

Local $oIE = _IEAttach("about:blank", "url")
_IELoadWait($oIE)
_IENavigate($oIE, "www.autoitscript.com")
