Справочник функций

_WinAPI_FindResourceEx

Определяет расположение ресурса указанного типа, имени и языка в указанном модуле.

#include <WinAPIRes.au3>
_WinAPI_FindResourceEx ( $hInstance, $sType, $sName, $iLanguage )

Параметры

$hInstance Дескриптор модуля, исполняемый файл которого содержит ресурс.
Значение 0 указывает дескриптор модуля, связанного с образом, использованным операционной системой для создания текущего процесса.
$sType Тип ресурса. Может быть строкой или целым значением.
$sName Имя ресурса. Может быть строкой или целым значением.
$iLanguage Язык ресурса.

Возвращаемое значение

Успех: Возвращает дескриптор на блок сведений ресурса.
Чтобы получить дескриптор самого ресурса, передайте этот дескриптор в _WinAPI_LoadResource().
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Если первый символ строки $sName или $sType – решётка (#), остальные символы представляют десятичное число, задающее целочисленный идентификатор имени или типа ресурса.
Например, строка "#258" представляет идентификатор 258.

Чтобы уменьшить объём памяти, требуемый для ресурса, ссылайтесь на него по целочисленному идентификатору, а не по имени.

Приложение может использовать _WinAPI_FindResource() для поиска ресурса любого типа, но эта функция нужна только если приложению потребуется доступ к двоичным данным ресурса в последующих вызовах _WinAPI_LockResource().
