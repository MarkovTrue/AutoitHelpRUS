Справочник функций

_Crypt_HashFile

Вычисляет хеш содержимого файла с помощью указанного алгоритма.

#include <Crypt.au3>
_Crypt_HashFile ( $sFilePath, $iAlgID )

Параметры

$sFilePath Путь к файлу для хеширования.
$iAlgID Идентификатор алгоритма хеширования. См. _Crypt_DeriveKey().

Возвращаемое значение

Успех: Возвращает хеш файла в двоичном виде.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – не удалось открыть файл
10+ – ошибка хеширования последнего блока
100+ – ошибка получения блока хеша
1000+ – ошибка _Crypt_Startup()

См. также

_Crypt_HashData

Пример

#include <ComboConstants.au3>
#include <Crypt.au3>
#include <GUIConstantsEx.au3>
#include <StringConstants.au3>

Example()

Func Example()
    Local $iAlgorithm = $CALG_SHA1

    Local $hGUI = GUICreate("Hash File", 500, 100)
    Local $idInput = GUICtrlCreateInput(@ScriptFullPath, 5, 5, 200, 20)
    Local $idBtn_Browse = GUICtrlCreateButton("...", 210, 5, 35, 20)
    Local $idCombo = GUICtrlCreateCombo("", 250, 5, 100, 20, $CBS_DROPDOWNLIST)
    GUICtrlSetData($idCombo, "MD2 (128bit)|MD4 (128bit)|MD5 (128bit)|SHA1 (160bit)|SHA_256 (256bit)|SHA_384 (384bit)|SHA_512 (512bit)", "SHA1 (160bit)")
    Local $idBtn_Calculate = GUICtrlCreateButton("Calculate", 400, 40, 65, 25)
    Local $idEdt_Hash = GUICtrlCreateEdit("Hash Digest", 5, 45, 350, 40)
    GUISetState(@SW_SHOW, $hGUI)

    _Crypt_Startup() ; Для оптимизации производительности запустить библиотеку Crypt.

    Local $dHash = 0, _
            $sRead = ""
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop

            Case $idBtn_Browse
                Local $sFilePath = FileOpenDialog("Open a file", "", "All files (*.*)") ; Выбираем файл для вычисления хеша.
                If @error Then
                    ContinueLoop
                EndIf
                GUICtrlSetData($idInput, $sFilePath) ; Задать поле ввода путём к файлу.
                GUICtrlSetData($idEdt_Hash, "Hash Digest") ; Сбрасываем метку хеш-дайджеста.

            Case $idCombo ; Проверить выбор в combobox и получить соответствующий алгоритм.
                Switch GUICtrlRead($idCombo) ; Прочитать выбор в combobox.
                    Case "MD2 (128bit)"
                        $iAlgorithm = $CALG_MD2

                    Case "MD4 (128bit)"
                        $iAlgorithm = $CALG_MD4

                    Case "MD5 (128bit)"
                        $iAlgorithm = $CALG_MD5

                    Case "SHA1 (160bit)"
                        $iAlgorithm = $CALG_SHA1

                    Case "SHA_256 (256bit)"
                        $iAlgorithm = $CALG_SHA_256

                    Case "SHA_384 (384bit)"
                        $iAlgorithm = $CALG_SHA_384

                    Case "SHA_512 (512bit)"
                        $iAlgorithm = $CALG_SHA_512

                EndSwitch

            Case $idBtn_Calculate
                $sRead = GUICtrlRead($idInput)
                If StringStripWS($sRead, $STR_STRIPALL) <> "" And FileExists($sRead) Then ; Проверяем наличие файла для вычисления хеш-дайджеста.
                    $dHash = _Crypt_HashFile($sRead, $iAlgorithm) ; Вычисляем хеш файла.
                    GUICtrlSetData($idEdt_Hash, $dHash) ; Отображаем хеш-дайджест в метке.
                EndIf
        EndSwitch
    WEnd

    GUIDelete($hGUI) ; Удалить предыдущий GUI и все элементы управления.
    _Crypt_Shutdown() ; Завершить работу библиотеки Crypt.
EndFunc   ;==>Example
