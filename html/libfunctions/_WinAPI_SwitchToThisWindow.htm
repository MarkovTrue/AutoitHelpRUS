_WinAPI_SwitchToThisWindow

Переводит фокус на указанное окно и выводит его на передний план.

#include <WinAPISysWin.au3>
_WinAPI_SwitchToThisWindow ( $hWnd [, $bAltTab = False] )

Параметры

$hWnd Дескриптор окна.
$bAltTab [необязательный] Указывает, переключается ли на использование последовательности клавиш Alt/Ctl+Tab, допустимые значения:
True - Окно переключается с использованием последовательности клавиш Alt/Ctl+Tab.
False - Иначе (По умолчанию).

Возвращаемое значение

Отсутствует.
