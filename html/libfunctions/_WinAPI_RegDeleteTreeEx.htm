Справочник функций

_WinAPI_RegDeleteTreeEx

Удаляет подключи и значения указанного ключа рекурсивно.

#include <WinAPIReg.au3>
_WinAPI_RegDeleteTreeEx ( $hKey [, $sSubKey = 0] )

Параметры

$hKey Дескриптор открытого ключа реестра.
Ключ должен быть открыт со следующими правами доступа $STANDARD_RIGHT_DELETE, $KEY_ENUMERATE_SUB_KEYS и $KEY_QUERY_VALUE.
Этот дескриптор возвращается функцией _WinAPI_RegCreateKey()
или _WinAPI_RegOpenKey(),
или может быть одним из следующих предопределённых ключей:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_USERS
$sSubKey [необязательный] Имя удаляемого ключа. Этот ключ должен быть подключом ключа, определённого параметром $hKey.
Если этот параметр не указан (по умолчанию), удаляются подключи и значения $hKey.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Если ключ имеет значения, он должен быть открыт с $KEY_SET_VALUE, иначе функция завершится с ошибкой.

Эта функция требует Windows Vista или позднее.
