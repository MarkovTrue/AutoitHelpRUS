_WinAPI_IsWow64Process

Определяет, работает ли указанный процесс под WOW64.

#include <WinAPIProc.au3>
_WinAPI_IsWow64Process ( [$iPID = 0] )

Параметры

$iPID [необязательный] PID процесса. По умолчанию (0) – текущий процесс.

Возвращаемое значение

Успех: Возвращает True – процесс работает под WOW64.
False – в противном случае.
Ошибка: Устанавливает @error ≠ 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
