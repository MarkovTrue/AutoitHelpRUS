_NamedPipes_DisconnectNamedPipe

Отключает конец сервера экземпляра именованного канала от клиентского процесса.

#include <NamedPipes.au3>
_NamedPipes_DisconnectNamedPipe ( $hNamedPipe )

Параметры

$hNamedPipe Дескриптор конца сервера экземпляра именованного канала.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Если конец клиента именованного канала открыт, функция _NamedPipes_DisconnectNamedPipe() принудительно закрывает этот конец канала.
Клиент получит ошибку при следующей попытке доступа к каналу.
Клиент, вытеснённый с канала, должен всё равно использовать функцию _WinAPI_CloseHandle() для закрытия своего конца канала.

См. также

_WinAPI_CloseHandle
