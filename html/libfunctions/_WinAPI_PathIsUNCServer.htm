_WinAPI_PathIsUNCServer

Определяет, является ли строка допустимым соглашением об универсальном именовании (UNC) только для пути сервера.

#include <WinAPIShPath.au3>
_WinAPI_PathIsUNCServer ( $sFilePath )

Параметры

$sFilePath Проверяемый путь.

Возвращаемое значение

True: Возвращает путь является допустимым путём UNC только для сервера (без имени общей папки).
False: путь не является допустимым путём UNC.
