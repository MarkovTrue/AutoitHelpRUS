Справочник функций

_IEFormElementGetCollection

Возвращает объектную переменную-коллекцию всех элементов формы внутри заданной формы.

#include <IE.au3>
_IEFormElementGetCollection ( ByRef $oObject [, $iIndex = -1] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application или Form.
$iIndex [необязательный] Что возвращать – коллекцию или элемент по индексу:
0 или положительное число – элемент с этим индексом
-1 – (по умолчанию) коллекцию

Возвращаемое значение

Успех: Возвращает объектную переменную с коллекцией Form Elements, @extended = количество элементов формы.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – недопустимое значение
7 ($_IEStatus_NoMatch) – совпадений не найдено
@extended: Содержит номер недопустимого параметра.

См. также

_IEFormElementGetObjByName, _IEFormElementGetValue, _IEFormElementSetValue, _IEFormSubmit

Пример

; Получить ссылку на конкретный элемент формы по индексу (с 0).
; В этом случае отправить запрос в поисковую систему Google

#include <IE.au3>

Local $oIE = _IECreate("http://www.google.com")
Local $oForm = _IEFormGetCollection($oIE, 0)
Local $oQuery = _IEFormElementGetCollection($oForm, 4)
_IEFormElementSetValue($oQuery, "AutoIt IE.au3")
_IEFormSubmit($oForm)
