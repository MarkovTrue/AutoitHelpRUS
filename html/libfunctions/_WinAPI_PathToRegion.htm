_WinAPI_PathToRegion

Создаёт регион из пути, который выбран в указанный контекст устройства.

#include <WinAPIGdi.au3>
_WinAPI_PathToRegion ( $hDC )

Параметры

$hDC Дескриптор контекста устройства, который содержит закрытый путь.

Возвращаемое значение

Успех: Возвращает дескриптор региона.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Контекст устройства, определённый параметром $hDC, должен содержать закрытый путь.

После того как _WinAPI_PathToRegion() преобразует путь в регион, система отбрасывает закрытый путь из указанного
контекста устройства.
