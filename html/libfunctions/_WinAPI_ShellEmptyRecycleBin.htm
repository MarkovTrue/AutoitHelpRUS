_WinAPI_ShellEmptyRecycleBin

Очищает корзину на указанном диске.

#include <WinAPIShellEx.au3>
_WinAPI_ShellEmptyRecycleBin ( [$sRoot = '' [, $iFlags = 0 [, $hParent = 0]]] )

Параметры

$sRoot [необязательный] Строка, содержащая путь корневого диска, на котором находится корзина. Эта строка
может быть отформатирована с помощью имён диска, папки и подпапки, например "c:\windows\system\".
Если этот параметр пустая строка (по умолчанию), все корзины на всех дисках будут очищены.
$iFlags [необязательный] Этот параметр может быть одним или несколькими из следующих значений.
$SHERB_NOCONFIRMATION
$SHERB_NOPROGRESSUI
$SHERB_NOSOUND
$SHERB_NO_UI
$hParent [необязательный] Дескриптор родительского окна любых диалоговых окон, которые могут быть отображены во время операции.

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки HRESULT.
