_WinAPI_RestartDlg

Отображает диалоговое окно, которое подсказывает пользователю перезагрузить Microsoft Windows.

#include <WinAPIDlg.au3>
_WinAPI_RestartDlg ( [$sText = '' [, $iFlags = $EWX_REBOOT [, $hParent = 0]]] )

Параметры

$sText [необязательный] Текст, который отображается в диалоговом окне, подсказывающем пользователю.
$iFlags [необязательный] Флаги, которые задают тип завершения работы.

Этот параметр должен включать одно из следующих значений.
$EWX_LOGOFF
$EWX_POWEROFF
$EWX_REBOOT (по умолчанию)
$EWX_SHUTDOWN

Этот параметр может дополнительно включать следующие значения.
$EWX_FORCE
$EWX_FORCEIFHUNG
$hParent [необязательный] Дескриптор родительского окна.

Возвращаемое значение

Возвращает идентификатор кнопки, нажатой для закрытия диалогового окна.
