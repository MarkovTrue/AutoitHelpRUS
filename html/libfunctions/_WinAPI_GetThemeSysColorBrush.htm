_WinAPI_GetThemeSysColorBrush

Получает системную кисть цвета.

#include <WinAPITheme.au3>
_WinAPI_GetThemeSysColorBrush ( $hTheme, $iColorID )

Параметры

$hTheme Дескриптор данных темы.
$iColorID Значение, которое задаёт номер желаемого системного цвета. Может быть одним из следующих значений.
$TMT_SCROLLBAR
$TMT_BACKGROUND
$TMT_ACTIVECAPTION
$TMT_INACTIVECAPTION
$TMT_WINDOW
$TMT_WINDOWFRAME
$TMT_MENUTEXT
$TMT_WINDOWTEXT
$TMT_CAPTIONTEXT
$TMT_ACTIVEBORDER
$TMT_INACTIVEBORDER
$TMT_APPWORKSPACE
$TMT_HIGHLIGHT
$TMT_HIGHLIGHTTEXT
$TMT_BTNFACE
$TMT_BTNSHADOW
$TMT_GRAYTEXT
$TMT_BTNTEXT
$TMT_INACTIVECAPTIONTEXT
$TMT_BTNHIGHLIGHT
$TMT_DKSHADOW3D
$TMT_LIGHT3D
$TMT_INFOTEXT
$TMT_INFOBK
$TMT_BUTTONALTERNATEFACE
$TMT_HOTTRACKING
$TMT_GRADIENTACTIVECAPTION
$TMT_GRADIENTINACTIVECAPTION
$TMT_MENUHILIGHT
$TMT_MENUBAR

Возвращаемое значение

Возвращает дескриптор кисти.

Примечания

Если дескриптор данных темы не равен 0, функция возвращает кисть, которая совпадает с указанным цветом из раздела SysMetrics
визуального стиля, в противном случае функция возвращает кисть, совпадающую с глобальным системным цветом.
