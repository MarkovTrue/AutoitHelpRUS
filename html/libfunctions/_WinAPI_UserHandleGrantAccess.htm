_WinAPI_UserHandleGrantAccess

Предоставляет или запрещает доступ к дескриптору объекта User для задания, имеющего ограничение пользовательского интерфейса.

#include <WinAPIProc.au3>
_WinAPI_UserHandleGrantAccess ( $hObject, $hJob, $bGrant )

Параметры

$hObject Дескриптор объекта User.
$hJob Дескриптор задания для предоставления доступа к дескриптору User.
$bGrant Следует ли предоставлять или запрещать доступ к дескриптору User, допустимые значения:
True – процессы, связанные с заданием, могут распознавать и использовать дескриптор.
False – процессы не могут использовать дескриптор.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция _WinAPI_UserHandleGrantAccess() может быть вызвана только из процесса, не связанного с заданием,
указанным параметром $hJob. Дескриптор User не должен принадлежать процессу или потоку, связанному
с заданием.
