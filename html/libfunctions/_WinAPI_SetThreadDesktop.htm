_WinAPI_SetThreadDesktop

Назначает указанный рабочий стол вызывающему потоку.

#include <WinAPIProc.au3>
_WinAPI_SetThreadDesktop ( $hDesktop )

Параметры

$hDesktop Дескриптор рабочего стола, назначаемого вызывающему потоку. Этот рабочий стол должен быть связан с
текущей оконной станцией процесса.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

Функция _WinAPI_SetThreadDesktop() завершится с ошибкой, если у вызывающего потока есть окна или хуки на его текущем
рабочем столе (если только $hDesktop не указывает на текущий рабочий стол).
