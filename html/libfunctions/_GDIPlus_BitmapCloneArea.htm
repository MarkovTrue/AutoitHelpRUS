_GDIPlus_BitmapCloneArea

Создаёт копию объекта Bitmap по заданным координатам и формату.

#include <GDIPlus.au3>
_GDIPlus_BitmapCloneArea ( $hBitmap, $nLeft, $nTop, $nWidth, $nHeight [, $iFormat = $GDIP_PXF24RGB] )

Параметры

$hBitmap Дескриптор объекта Bitmap.
$nLeft Координата X левого верхнего угла копируемой области.
$nTop Координата Y левого верхнего угла копируемой области.
$nWidth Ширина прямоугольника.
$nHeight Высота прямоугольника.
$iFormat [необязательный] Формат пикселей для нового изображения:
$GDIP_PXF01INDEXED = 1 бит/пикс, индексированный
$GDIP_PXF04INDEXED = 4 бит/пикс, индексированный
$GDIP_PXF08INDEXED = 8 бит/пикс, индексированный
$GDIP_PXF16GRAYSCALE = 16 бит/пикс, оттенки серого
$GDIP_PXF16RGB555 = 16 бит/пикс; по 5 бит на каждую составляющую RGB
$GDIP_PXF16RGB565 = 16 бит/пикс; 5 бит красный, 6 бит зелёный, 5 бит синий
$GDIP_PXF16ARGB1555 = 16 бит/пикс; 1 бит на альфа-канал и по 5 бит на каждую составляющую RGB
$GDIP_PXF24RGB = 24 бит/пикс; по 8 бит на каждую составляющую RGB (по умолчанию)
$GDIP_PXF32RGB = 32 бит/пикс; по 8 бит на каждую составляющую RGB. Без альфа-канала.
$GDIP_PXF32ARGB = 32 бит/пикс; по 8 бит на каждую составляющую RGB и альфа-канал
$GDIP_PXF32PARGB = 32 бит/пикс; по 8 бит на составляющие RGB и альфа-канал, предумноженные
$GDIP_PXF48RGB = 48 бит/пикс; по 16 бит на каждую составляющую RGB
$GDIP_PXF64ARGB = 64 бит/пикс; по 16 бит на каждую составляющую RGB и альфа-канал
$GDIP_PXF64PARGB = 64 бит/пикс; по 16 бит на составляющие RGB и альфа-канал, предумноженные.

Возвращаемое значение

Успех: Возвращает дескриптор нового объекта Bitmap.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки GPSTATUS ($GDIP_ERR* см. GPIPlusConstants.au3).

Примечания

После завершения работы с объектом вызовите _GDIPlus_BitmapDispose() для освобождения ресурсов.

См. также

_GDIPlus_BitmapDispose, _GDIPlus_ImageGetPixelFormat

Дополнительно

Искать GdipCloneBitmapAreaI в библиотеке MSDN

Пример

#include <GDIPlus.au3>
#include <ScreenCapture.au3>
#include <WinAPIHObj.au3>

Example()

Func Example()
    Local $hBitmap, $hClone, $hImage, $iX, $iY

    ; Инициализировать библиотеку GDI+
    _GDIPlus_Startup()

    ; Захватить 32-битный bitmap
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; Создать клон 24-битного bitmap
    $iX = _GDIPlus_ImageGetWidth($hImage)
    $iY = _GDIPlus_ImageGetHeight($hImage)
    $hClone = _GDIPlus_BitmapCloneArea($hImage, 0, 0, $iX, $iY, $GDIP_PXF24RGB)

    ; Сохранить bitmap в файл
    _GDIPlus_ImageSaveToFile($hClone, @MyDocumentsDir & "\GDIPlus_Image.bmp")

    ; Освободить ресурсы
    _GDIPlus_ImageDispose($hClone)
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; Завершить работу с библиотекой GDI+
    _GDIPlus_Shutdown()

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.jpg")
EndFunc   ;==>Example
