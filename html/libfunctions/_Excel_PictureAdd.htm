Справочник функций

_Excel_PictureAdd

Добавляет изображение на указанный лист книги.

#include <Excel.au3>
_Excel_PictureAdd ( $oWorkbook, $vWorksheet, $sFile, $vRangeOrLeft [, $iTop = Default [, $iWidth = Default [, $iHeight = Default [, $bKeepRatio = True]]]] )

Параметры

$oWorkbook Объект рабочей книги Excel.
$vWorksheet Имя, индекс или объект листа, на который добавляется изображение. При значении Default используется активный лист.
$sFile Полный путь к добавляемому файлу изображения.
$vRangeOrLeft Диапазон A1, объект Range или целое число, задающее положение левого верхнего угла изображения по горизонтали.
$iTop [необязательный] Если $vRangeOrLeft является целым числом, задаёт положение левого верхнего угла изображения по вертикали.
$iWidth [необязательный] Задаёт ширину изображения. Если не указана, ширина подстраивается автоматически (по умолчанию – автоматически).
$iHeight [необязательный] Задаёт высоту изображения. Если не указана, высота подстраивается автоматически (по умолчанию – автоматически).
$bKeepRatio [необязательный] Используется только при многоячеечном диапазоне в $vRangeOrLeft (по умолчанию True).
True – сохранять пропорции изображения в границах $vRangeOrLeft.
False – растянуть изображение по $vRangeOrLeft без сохранения пропорций.

Возвращаемое значение

Успех: Возвращает объект Shape, представляющий добавленное изображение.
Ошибка: Возвращает 0 и устанавливает @error.
@error: 1 – $oWorkbook не является объектом или объектом книги
2 – имя или индекс $vWorksheet недопустимы либо $vWorksheet не является объектом листа; @extended содержит код COM-ошибки
3 – $vRangeOrLeft недопустим.
@extended: Содержит код COM-ошибки
4 – ошибка при добавлении изображения.
@extended: Содержит код COM-ошибки
5 – файл $sFile не существует

Примечания

Если $vRangeOrLeft является многоячеечным диапазоном, параметры $iWidth и $iHeight игнорируются (чтобы задать размеры, не привязанные к диапазону, укажите одну ячейку в $vRangeOrLeft).

Если задан только один из параметров $iWidth или $iHeight, второй (со значением по умолчанию) масштабируется для сохранения пропорций.
Если заданы оба параметра, изображение принимает указанные размеры независимо от исходных пропорций.
Если ни $iWidth, ни $iHeight не заданы, изображение принимает исходный размер файла.

$bKeepRatio учитывается только при указании многоячеечного диапазона (подробности в описании параметров).

Пример

#include "Extras\HelpFileInternals.au3"

#include <Excel.au3>
#include <MsgBoxConstants.au3>

; Создать объект приложения и открыть пример рабочей книги
Local $oExcel = _Excel_Open()
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example", "Error creating the Excel application object." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
; Создаём новую книгу.
Local $oWorkbook = _Excel_BookNew($oExcel)
If @error Then
    MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example", "Error creating workbook." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
    _Excel_Close($oExcel)
    Exit
EndIf

; Вставляем изображение в диапазон ячеек с изменением размера. Пропорции сохраняются.
Local $sPicture = _Extras_PathFull("_Excel.jpg")
_Excel_PictureAdd($oWorkbook, Default, $sPicture, "B2:D8")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 1", "Error inserting picture." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 1", "Picture inserted/resized at 'B2:D8', aspect ratio retained.")

; Вставляем изображение без изменения размера.
_Excel_PictureAdd($oWorkbook, Default, $sPicture, "F8")
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 2", "Error inserting picture." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 2", "Picture inserted at 'F8' without resizing.")

; Вставить изображение с заданной шириной/высотой.
_Excel_PictureAdd($oWorkbook, Default, $sPicture, "A8", Default, 300, 250)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 3", "Error inserting picture." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 3", "Picture inserted at 'A8' with defined size/height, aspect ratio ignored")

; Вставить изображение с заданной шириной/высотой.
_Excel_PictureAdd($oWorkbook, Default, $sPicture, 250, 300, 300, 250)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 4", "Error inserting picture." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 4", "Picture inserted at position 250/300' with defined size/height, aspect ratio ignored")

; Вставить изображение с заданной шириной/высотой.
_Excel_PictureAdd($oWorkbook, Default, $sPicture, "F2:H9", Default, Default, Default, False)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 5", "Error inserting picture." & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Excel UDF: _Excel_PictureAdd Example 5", "Picture inserted/resized at 'F2:H9', aspect ratio ignored.")
