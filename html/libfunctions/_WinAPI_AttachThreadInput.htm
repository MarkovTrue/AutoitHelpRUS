_WinAPI_AttachThreadInput

Подключает механизм обработки ввода одного потока к механизму другого потока.

#include <WinAPIProc.au3>
_WinAPI_AttachThreadInput ( $iAttach, $iAttachTo, $bAttach )

Параметры

$iAttach Идентификатор потока, который будет подключен к другому потоку.
$iAttachTo Идентификатор потока, к которому будет подключен другой поток.
$bAttach Режим подключения:
True – потоки подключены
False – потоки отключены.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
