_GUIToolTip_SetWindowTheme

Устанавливает визуальный стиль окна подсказки.

#include <GuiToolTip.au3>
_GUIToolTip_SetWindowTheme ( $hTool, $sStyle )

Параметры

$hTool Дескриптор элемента управления ToolTip (возвращается функцией _GUIToolTip_Create).
$sStyle строка содержащая визуальный стиль подсказки

Возвращаемое значение

Отсутствует.
