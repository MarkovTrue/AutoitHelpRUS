Справочник функций

_WinAPI_GetNumberFormat

Форматирует строку числа в строку числа, настроенную для локали, определённой идентификатором.

#include <WinAPILocale.au3>
_WinAPI_GetNumberFormat ( $iLCID, $sNumber [, $tNUMBERFMT = 0] )

Параметры

$iLCID Идентификатор локали (LCID) – задаёт локаль либо одно из предопределённых значений:
$LOCALE_INVARIANT
$LOCALE_SYSTEM_DEFAULT
$LOCALE_USER_DEFAULT

Windows Vista и выше
$LOCALE_CUSTOM_DEFAULT
$LOCALE_CUSTOM_UI_DEFAULT
$LOCALE_CUSTOM_UNSPECIFIED
$sNumber Строка, содержащая строку числа для форматирования. Эта строка может содержать только следующие
символы. Все остальные символы недопустимы.

Символы от "0" до "9".
Минус в первой позиции символа, если число отрицательное.
Одну десятичную точку (точку), если число – число с плавающей точкой.
$tNUMBERFMT [необязательный] Структура $tagNUMBERFMT, содержащая информацию о форматировании чисел. Если этот параметр равен 0 (по умолчанию),
функция возвращает строку в соответствии с форматом чисел для указанной локали.
Вы можете использовать функцию _WinAPI_CreateNumberFormatInfo() для создания этой структуры.

Возвращаемое значение

Успех: Возвращает отформатированная строка числа.
Ошибка: Возвращает пустую строку; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

См. также

_WinAPI_CreateNumberFormatInfo

Дополнительно

Искать GetNumberFormat в библиотеке MSDN

Пример

#include <WinAPILocale.au3>

Local $sNumber = '123456789'

ConsoleWrite(_WinAPI_GetNumberFormat(0, $sNumber) & @CRLF)
ConsoleWrite(_WinAPI_GetNumberFormat(0, $sNumber, _WinAPI_CreateNumberFormatInfo(0, 1, 3, '', ',', 1)) & @CRLF)
