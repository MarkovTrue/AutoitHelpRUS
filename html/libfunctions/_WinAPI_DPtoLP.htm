_WinAPI_DPtoLP

Преобразует координаты устройства в логические координаты.

#include <WinAPIGdi.au3>
_WinAPI_DPtoLP ( $hDC, ByRef $tPOINT [, $iCount = 1] )

Параметры

$hDC Дескриптор контекста устройства.
$tPOINT Структура $tagPOINT или структура точек ("long x1;long y1;...long xN;long yN"), содержащая x- и y-координаты для преобразования.
$iCount [необязательный] Количество точек. По умолчанию 1.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.

Примечания

_WinAPI_DPtoLP() завершается ошибкой, если координаты устройства превышают 27 бит, либо если полученные логические координаты превышают 32 бита.
При таком переполнении результаты для всех точек не определены.
