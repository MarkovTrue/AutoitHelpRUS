_WinAPI_GetGuiResources

Получает количество дескрипторов объектов графического пользовательского интерфейса (GUI), используемых указанным процессом.

#include <WinAPIProc.au3>
_WinAPI_GetGuiResources ( [$iFlag = 0 [, $hProcess = -1]] )

Параметры

$iFlag [необязательный] 0 (по умолчанию) Возвращает количество объектов GDI.
1 Возвращает количество объектов USER.
$hProcess [необязательный] Дескриптор процесса. По умолчанию текущий процесс.

Возвращаемое значение

Успех: Возвращает количество дескрипторов объектов GUI, используемых процессом, в соответствии с $iFlag.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
