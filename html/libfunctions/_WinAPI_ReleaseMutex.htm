_WinAPI_ReleaseMutex

Отпускает владение указанным объектом мьютекса.

#include <WinAPIProc.au3>
_WinAPI_ReleaseMutex ( $hMutex )

Параметры

$hMutex Дескриптор объекта мьютекса. Функция _WinAPI_CreateMutex() или _WinAPI_OpenMutex() возвращает этот дескриптор.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Функция _WinAPI_ReleaseMutex() завершается с ошибкой, если вызывающий поток не владеет объектом мьютекса.

См. также

_WinAPI_CreateMutex, _WinAPI_OpenMutex
