Справочник функций

_WinAPI_GetClipboardSequenceNumber

Получает номер последовательности буфера обмена для текущей рабочей станции.

#include <WinAPISys.au3>
_WinAPI_GetClipboardSequenceNumber ( )

Возвращаемое значение

Успех: Возвращает номер последовательности буфера обмена.
Ошибка: Возвращает 0.

Примечания

Вызывающий процесс должен иметь доступ WINSTA_READATTRIBUTES к рабочей станции.
Эта функция требует Windows Vista или выше.
