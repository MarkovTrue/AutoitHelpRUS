Справочник функций

_IEDocReadHTML

Возвращает полный HTML-исходник документа.

#include <IE.au3>
_IEDocReadHTML ( ByRef $oObject )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.

Возвращаемое значение

Успех: Возвращает HTML, содержащийся в <HTML> документа, включая теги <html> и </html>.
Ошибка: Устанавливает @error ≠ 0.
@error: 3 ($_IEStatus_InvalidDataType) – недопустимый тип данных.
@extended: Содержит номер недопустимого параметра.

Примечания

Эта функция возвращает исходник документа после любых клиентских модификаций (например, через AutoIt или клиентский JavaScript).
Поэтому он может отличаться от того, что показывает браузер в режиме "Просмотр исходного кода" или функция _INetGetSource().

См. также

_IEBodyReadHTML, _IEDocInsertHTML, _IEDocInsertText, _IEDocWriteHTML, _INetGetSource

Пример

; Открытие браузера с базовым примером, чтение HTML документа
; (весь HTML, включая <HEAD> и скрипт) и отобразить в MsgBox

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $oIE = _IE_Example("basic")
Local $sHTML = _IEDocReadHTML($oIE)
MsgBox($MB_SYSTEMMODAL, "Document Source", $sHTML)

_IEQuit($oIE)
