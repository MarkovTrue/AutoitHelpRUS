_WinAPI_DefWindowProcW

Вызывает оконную процедуру по умолчанию для стандартной обработки любых оконных сообщений.

#include <WinAPISysWin.au3>
_WinAPI_DefWindowProcW ( $hWnd, $iMsg, $wParam, $lParam )

Параметры

$hWnd Дескриптор оконной процедуры, получившей сообщение.
$iMsg Сообщение.
$wParam Дополнительные сведения, зависящие от сообщения.
$lParam Дополнительные сведения, зависящие от сообщения.

Возвращаемое значение

Возвращает результат обработки сообщения; значение зависит от отправленного сообщения.
