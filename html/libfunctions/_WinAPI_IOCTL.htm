Справочник функций

_WinAPI_IOCTL

Создаёт уникальный системный код управления вводом-выводом (IOCTL).

#include <WinAPIFiles.au3>
_WinAPI_IOCTL ( $iDeviceType, $iFunction, $iMethod, $iAccess )

Параметры

$iDeviceType Тип устройства.
$iFunction Действие в категории устройства.
$iMethod Коды метода для способа передачи буферов для элементов управления вводом-выводом и файловой системой.
$iAccess Значение проверки доступа для любого доступа.

Возвращаемое значение

Возвращает значение IOCTL (DWORD).
