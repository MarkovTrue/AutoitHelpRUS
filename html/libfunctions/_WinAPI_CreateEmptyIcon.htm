_WinAPI_CreateEmptyIcon

создаёт полностью прозрачный значок с указанными шириной, высотой и глубиной цвета.

#include <WinAPIIcons.au3>
_WinAPI_CreateEmptyIcon ( $iWidth, $iHeight [, $iBitsPerPel = 32] )

Параметры

$iWidth Ширина иконки в пикселях.
$iHeight Высота иконки в пикселях.
$iBitsPerPel [необязательный] Количество бит/пикс в XOR-bitmask значка. По умолчанию 32.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный значок.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.

Примечания

Эта функция создаёт значок с чёрным XOR-битмапом и белым AND-битмапом маски.
У этого значка нет альфа-канала; он отображается только по AND-маске.

Когда значок больше не нужен, удалите его вызовом _WinAPI_DestroyIcon().
