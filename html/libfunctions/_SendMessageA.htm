Справочник функций

_SendMessageA

Отправляет сообщение окну/элементу управления (принудительный вызов ANSI).

#include <SendMessage.au3>
_SendMessageA ( $hWnd, $iMsg [, $wParam = 0 [, $lParam = 0 [, $iReturn = 0 [, $wParamType = "wparam" [, $lParamType = "lparam" [, $sReturnType = "lresult"]]]]]] )

Параметры

$hWnd Дескриптор окна/элемента управления.
$iMsg Сообщение для отправки элементу управления (число).
$wParam [необязательный] Дополнительные сведения, специфичные для сообщения
$lParam [необязательный] Дополнительные сведения, специфичные для сообщения
$iReturn [необязательный] Что возвращать:
0 – Возвращаемое значение вызова DLL
1 – $ihWnd
2 – $iMsg
3 – $wParam
4 – $lParam
<0 или > 4 – массив как в DllCall
$wParamType [необязательный] См. DllCall в разделе "См. также"
$lParamType [необязательный] См. DllCall в разделе "См. также"
$sReturnType [необязательный] См. DllCall в разделе "См. также"

Возвращаемое значение

Успех: Возвращает выбранное пользователем значение из результата DllCall().
Ошибка: Устанавливает @error ≠ 0.

Примечания

Когда сообщение блокируется UIPI, последняя ошибка, полученная с помощью _WinAPI_GetLastError(), устанавливается в 5 (доступ запрещён).

См. также

DllCall, _SendMessage, _WinAPI_GetLastError
