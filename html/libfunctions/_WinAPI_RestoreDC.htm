_WinAPI_RestoreDC

Восстанавливает контекст устройства (DC) в указанное состояние.

#include <WinAPIGdiDC.au3>
_WinAPI_RestoreDC ( $hDC, $iID )

Параметры

$hDC Дескриптор DC.
$iID Сохранённое состояние для восстановления. Если этот параметр положительный, $DC представляет конкретный экземпляр состояния для восстановления.
Если этот параметр отрицательный, $DC представляет экземпляр относительно текущего состояния.
Например, (-1) восстанавливает самое недавно сохранённое состояние.

Возвращаемое значение

Успех: Возвращает True.
Ошибка: Возвращает False.
