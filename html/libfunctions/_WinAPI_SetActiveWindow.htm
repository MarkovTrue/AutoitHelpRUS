Справочник функций

_WinAPI_SetActiveWindow

Активирует указанное окно.

#include <WinAPISys.au3>
_WinAPI_SetActiveWindow ( $hWnd )

Параметры

$hWnd Дескриптор окна верхнего уровня для активирования.

Возвращаемое значение

Успех: Возвращает дескриптор окна, которое было активным ранее.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.
