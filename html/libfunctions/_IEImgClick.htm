Справочник функций

_IEImgClick

Имитирует щелчок мыши по изображению. Сопоставление по подстроке alt-текста, name или src.

#include <IE.au3>
_IEImgClick ( ByRef $oObject, $sLinkText [, $sMode = "src" [, $iIndex = 0 [, $iWait = 1]]] )

Параметры

$oObject Объектная переменная для объекта InternetExplorer.Application, Window или Frame.
$sLinkText Текст для сравнения со значением атрибута, заданного через $sMode.
$sMode [необязательный] задаёт режим поиска
"src" = (по умолчанию) сопоставление по url изображения
"id" = сопоставление по id изображения (см. раздел Примечания)
"alt" = сопоставление по альтернативному тексту изображения
$iIndex [необязательный] Если текст img встречается больше одного раза, задайте нужный экземпляр по индексу (с нуля)
$iWait [необязательный] Ждать ли загрузку страницы перед возвратом:
0 – вернуться сразу, не ожидая загрузки
1 – (по умолчанию) дождаться полной загрузки и вернуться

Возвращаемое значение

Успех: Нет.
Ошибка: Возвращает 0 или -1 и устанавливает @error ≠ 0.
@error: 1 ($_IEStatus_GeneralError) – общая ошибка
2 ($_IEStatus_COMError) – ошибка COM в ссылке на объект
3 ($_IEStatus_InvalidDataType) – неверный тип данных
4 ($_IEStatus_InvalidObjectType) – неверный тип объекта
5 ($_IEStatus_InvalidValue) – неверное значение
6 ($_IEStatus_LoadWaitTimeout) – истекло ожидание загрузки
7 ($_IEStatus_NoMatch) – нет совпадения
8 ($_IEStatus_AccessIsDenied) – доступ запрещён
9 ($_IEStatus_ClientDisconnected) – клиент отключён
@extended: Содержит номер недопустимого параметра.

Примечания

Атрибут "name" в HTML5 больше не поддерживается, но всё ещё принимается и для HTML5-страниц эквивалентен "id".

См. также

_IEFormImageClick, _IEImgGetCollection, _IELoadWait

Пример

Пример 1

; Кликнуть по IMG по тексту атрибута Alt

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEImgClick($oIE, "AutoIt Homepage Image", "alt")

Пример 2

; Кликнуть по IMG по id

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEImgClick($oIE, "AutoItImage", "id") ; "name" по-прежнему эквивалентен для HTML5-страниц

Пример 3

; Щелчок по IMG по подстроке src

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEImgClick($oIE, "autoit_6_240x100.jpg", "src")

Пример 4

; Щёлкнуть на IMG по полной строке src

#include <IE.au3>

Local $oIE = _IE_Example("basic")
_IEImgClick($oIE, "http://www.autoitscript.com/images/logo_autoit_210x72.png")
