Справочник функций

_GUICtrlListView_GetItemRect

Получает ограничивающий прямоугольник для всего элемента или его части.

#include <GuiListView.au3>
_GUICtrlListView_GetItemRect ( $hWnd, $iIndex [, $iPart = 3] )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$iIndex Индекс элемента (нумерация с 0).
$iPart [необязательный] Часть элемента для получения:
$LVIR_BOUNDS – возвращает ограничивающий прямоугольник всего элемента, включая значок и метку
$LVIR_ICON – возвращает ограничивающий прямоугольник значка или малого значка
$LVIR_LABEL – возвращает ограничивающий прямоугольник текста элемента
$LVIR_SELECTBOUNDS – возвращает объединение прямоугольников $LVIR_ICON и $LVIR_LABEL, исключая столбцы в представлении отчёта.

Возвращаемое значение


Возвращает массив со следующим формат:
    [0] – координата X левого верхнего угла прямоугольника
    [1] – координата Y левого верхнего угла прямоугольника
    [2] – координата X правого нижнего угла прямоугольника
    [3] – координата Y правого нижнего угла прямоугольника

_GUICtrlListView_GetItemRectEx

Пример

#include <GUIConstantsEx.au3>
#include <GuiListView.au3>
#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $aRect, $idListview

    GUICreate("ListView Get Item Rectangle", 400, 300)
    $idListview = GUICtrlCreateListView("", 2, 2, 394, 268)
    GUISetState(@SW_SHOW)

    ; Добавить столбцы
    _GUICtrlListView_AddColumn($idListview, "Items", 100)

    ; Добавить элементы
    _GUICtrlListView_AddItem($idListview, "Item 1")
    _GUICtrlListView_AddItem($idListview, "Item 2")
    _GUICtrlListView_AddItem($idListview, "Item 3")

    ; Получить прямоугольник элемента 2
    $aRect = _GUICtrlListView_GetItemRect($idListview, 1)
    MsgBox($MB_SYSTEMMODAL, "Information", StringFormat("Item 2 Rectangle : [%d, %d, %d, %d]", $aRect[0], $aRect[1], $aRect[2], $aRect[3]))

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
