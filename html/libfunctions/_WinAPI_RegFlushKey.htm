Справочник функций

_WinAPI_RegFlushKey

Записывает все атрибуты указанного открытого ключа реестра в реестр.

#include <WinAPIReg.au3>
_WinAPI_RegFlushKey ( $hKey )

Параметры

$hKey Дескриптор открытого раздела реестра. Раздел должен быть открыт с правом доступа $KEY_QUERY_VALUE.
Дескриптор возвращается функциями _WinAPI_RegCreateKey() и _WinAPI_RegOpenKey().
Также допустимы предопределённые ключи:
$HKEY_CLASSES_ROOT
$HKEY_CURRENT_CONFIG
$HKEY_CURRENT_USER
$HKEY_LOCAL_MACHINE
$HKEY_PERFORMANCE_DATA
$HKEY_USERS

Возвращаемое значение

Успех: Возвращает 1.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать системный код ошибки.

Примечания

Функция _WinAPI_RegFlushKey() возвращается только после того, как все данные для улья, содержащего указанный ключ, были записаны в хранилище реестра на диск.
Функция _WinAPI_RegFlushKey() записывает данные для других ключей в улье, которые были изменены с момента последней ленивой промывки или запуска системы.
После того, как _WinAPI_RegFlushKey() вернётся, используйте _WinAPI_RegCloseKey() для закрытия дескриптора ключа реестра.

См. также

_WinAPI_RegCloseKey, _WinAPI_RegCreateKey, _WinAPI_RegOpenKey
