Справочник функций

_ClipBoard_ChangeChain

Удаляет указанное окно из цепочки наблюдателей буфера обмена.

#include <Clipboard.au3>
_ClipBoard_ChangeChain ( $hRemove, $hNewNext )

Параметры

$hRemove Дескриптор окна, удаляемого из цепочки.
Дескриптор должен был быть ранее передан функции _ClipBoard_SetViewer().
$hNewNext Дескриптор окна, следующего за $hRemove в цепочке наблюдателей буфера обмена.
Обычно это дескриптор, возвращённый _ClipBoard_SetViewer(), если только порядок не был изменён в ответ на сообщение $WM_CHANGECBCHAIN.

Возвращаемое значение

Отсутствует.

Примечания

Окно, идентифицированное $hNewNext, занимает место $hRemove в цепочке.
_ClipBoard_SetViewer() отправляет сообщение $WM_CHANGECBCHAIN первому окну в цепочке наблюдателей буфера обмена.

См. также

_ClipBoard_SetViewer

Дополнительно

Искать ChangeClipboardChain в библиотеке MSDN

Пример

#include "Extras\HelpFileInternals.au3"

#include <Clipboard.au3>
#include <GUIConstantsEx.au3>
#include <SendMessage.au3>
#include <WindowsNotifsConstants.au3>
#include <WindowsStylesConstants.au3>

Global $g_hNext = 0

Example()

Func Example()
    Local $hGUI

    ; Создать GUI
    $hGUI = GUICreate("Clipboard", 600, 400)
    _MemoCreate(2, 2, 596, 396, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Инициализировать просмотрщик буфера обмена
    $g_hNext = _ClipBoard_SetViewer($hGUI)

    GUIRegisterMsg($WM_CHANGECBCHAIN, "WM_CHANGECBCHAIN")
    GUIRegisterMsg($WM_DRAWCLIPBOARD, "WM_DRAWCLIPBOARD")

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; Завершить работу просмотрщика буфера обмена
    _ClipBoard_ChangeChain($hGUI, $g_hNext)
EndFunc   ;==>Example

; Обработать сообщения $WM_CHANGECBCHAIN
Func WM_CHANGECBCHAIN($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg
    ; Показать, что сообщение получено
    _MemoWrite("***** $WM_CHANGECBCHAIN *****")

    ; Если следующее окно закрывается – восстановить цепочку
    If $wParam = $g_hNext Then
        $g_hNext = $lParam
        ; Иначе передать сообщение следующему просмотрщику
    ElseIf $g_hNext <> 0 Then
        _SendMessage($g_hNext, $WM_CHANGECBCHAIN, $wParam, $lParam, 0, "hwnd", "hwnd")
    EndIf
EndFunc   ;==>WM_CHANGECBCHAIN

; Обработать сообщения $WM_DRAWCLIPBOARD
Func WM_DRAWCLIPBOARD($hWnd, $iMsg, $wParam, $lParam)
    #forceref $hWnd, $iMsg
    ; Показать любой текст в буфере обмена
    _MemoWrite(_ClipBoard_GetData())

    ; Передать сообщение следующему просмотрщику
    If $g_hNext <> 0 Then _SendMessage($g_hNext, $WM_DRAWCLIPBOARD, $wParam, $lParam)
EndFunc   ;==>WM_DRAWCLIPBOARD
