Справочник функций

_WinAPI_CreateMutex

создаёт или открывает именованный или безымянный объект мьютекса.

#include <WinAPIProc.au3>
_WinAPI_CreateMutex ( $sMutex [, $bInitial = True [, $tSecurity = 0]] )

Параметры

$sMutex Имя.
$bInitial Значение.
$tSecurity Координаты.

Возвращаемое значение

Успех: Возвращает дескриптор на созданный объект.
Ошибка: Возвращает 0; вызовите _WinAPI_GetLastError() для получения подробных сведений об ошибке.

Примечания

Если мьютекс именован, и объект уже существовал до вызова, функция возвращает дескриптор существующего объекта; _WinAPI_GetLastError() возвращает ERROR_ALREADY_EXISTS (183), параметр $bInitial игнорируется, и вызывающий поток не получает владения мьютексом.
