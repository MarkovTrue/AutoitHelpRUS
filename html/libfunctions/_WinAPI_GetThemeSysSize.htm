_WinAPI_GetThemeSysSize

Получает значение системной метрики размера из данных темы.

#include <WinAPITheme.au3>
_WinAPI_GetThemeSysSize ( $hTheme, $iSizeID )

Параметры

$hTheme Дескриптор данных темы.
$iSizeID Значение, которое задаёт желаемую системную метрику размера. Действительны следующие значения.
$SM_CXBORDER
$SM_CXVSCROLL
$SM_CXHSCROLL
$SM_CXSIZE
$SM_CYSIZE
$SM_CXSMSIZE
$SM_CYSMSIZE
$SM_CXMENUSIZE
$SM_CYMENUSIZE
$SM_CXPADDEDBORDER

Возвращаемое значение

Возвращает размер в пиксельях.

Примечания

Если $hTheme не равен 0, функция возвращает размер, сохранённый в текущем визуальном стиле (раздел SysMetrics
визуального стиля), масштабированный по текущему экрану dpi. Если $hTheme равен 0, функция возвращает глобальную системную
метрику в пиксельях, которая масштабируется по текущему dpi только если приложение помечено как чувствительное к dpi; в противном случае
возвращаемые пиксели не масштабируются.
