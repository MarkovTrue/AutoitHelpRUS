Справочник функций

_GUICtrlEdit_SetRECTNP

Задаёт прямоугольник форматирования многострочного поля ввода.

#include <GuiEdit.au3>
_GUICtrlEdit_SetRECTNP ( $hWnd, $aRect )

Параметры

$hWnd Идентификатор или дескриптор элемента управления.
$aRect массив в следующем формате:
[0] – указывает координату x верхнего левого угла прямоугольника
[1] – указывает координату y верхнего левого угла прямоугольника
[2] – указывает координату x нижнего правого угла прямоугольника
[3] – указывает координату y нижнего правого угла прямоугольника

Возвращаемое значение

Отсутствует.

Примечания

Функция _GUICtrlEdit_SetRECTNP() идентична _GUICtrlEdit_SetRECT(), но не перерисовывает окно поля редактирования.

См. также

_GUICtrlEdit_SetRectNPEx

Пример

#include <GUIConstantsEx.au3>
#include <GuiEdit.au3>
#include <GuiStatusBar.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $hStatusBar, $idEdit, $hGUI
    Local $sWow64 = ""
    If @AutoItX64 Then $sWow64 = "\Wow6432Node"
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $sWow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\include\_ReadMe_.txt"
    Local $aPartRightSide[6] = [50, 87, 87, 87, 87, -1], $aRect

    ; Создать GUI
    $hGUI = GUICreate("Edit Set RECTNP", 400, 300)
    $idEdit = GUICtrlCreateEdit("", 2, 2, 394, 268, BitOR($ES_WANTRETURN, $WS_VSCROLL))
    $hStatusBar = _GUICtrlStatusBar_Create($hGUI, $aPartRightSide)
    _GUICtrlStatusBar_SetIcon($hStatusBar, 5, 97, "shell32.dll")
    _GUICtrlStatusBar_SetText($hStatusBar, "Rect")
    GUISetState(@SW_SHOW)

    ; Получить прямоугольник
    $aRect = _GUICtrlEdit_GetRECT($idEdit)
    $aRect[0] += 10
    $aRect[1] += 10
    $aRect[2] -= 10
    $aRect[3] -= 10

    ; Добавить текст
    _GUICtrlEdit_AppendText($idEdit, FileRead($sFile))
    _GUICtrlEdit_LineScroll($idEdit, 0, _GUICtrlEdit_GetLineCount($idEdit) * -1)

    ; Установить Rect
    _GUICtrlEdit_SetRECTNP($idEdit, $aRect)

    ; Получить прямоугольник
    $aRect = _GUICtrlEdit_GetRECT($idEdit)
    _GUICtrlStatusBar_SetText($hStatusBar, "Left: " & $aRect[0], 1)
    _GUICtrlStatusBar_SetText($hStatusBar, "Topt: " & $aRect[1], 2)
    _GUICtrlStatusBar_SetText($hStatusBar, "Right: " & $aRect[2], 3)
    _GUICtrlStatusBar_SetText($hStatusBar, "Bottom: " & $aRect[3], 4)

    ; Цикл до выхода пользователя.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
