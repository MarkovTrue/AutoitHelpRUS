_WinAPI_CompressBuffer

Сжимает буфер с указанным форматом сжатия и типом механизма.

#include <WinAPISys.au3>
_WinAPI_CompressBuffer ( $pUncompressedBuffer, $iUncompressedSize, $pCompressedBuffer, $iCompressedSize [, $iFormatAndEngine = $COMPRESSION_FORMAT_LZNT1] )

Параметры

$pUncompressedBuffer Указатель на выделенный вызывающей программой буфер, который содержит данные, подлежащие сжатию.
$iUncompressedSize Размер несжатого буфера в байтах.
$pCompressedBuffer Указатель на выделенный вызывающей программой буфер, который получает сжатые данные.
$iCompressedSize Размер сжатого буфера в байтах.
$iFormatAndEngine [необязательный] Маска битов, которая задаёт формат сжатия и тип механизма.
Этот параметр должен быть установлен в допустимую побитовую комбинацию ИЛИ одного типа формата и одного типа механизма.
$COMPRESSION_FORMAT_LZNT1 (по умолчанию)
$COMPRESSION_FORMAT_XPRESS
$COMPRESSION_FORMAT_XPRESS_HUFF

$COMPRESSION_ENGINE_STANDARD (по умолчанию)
$COMPRESSION_ENGINE_MAXIMUM

Возвращаемое значение

Успех: Возвращает размер сжатых данных, сохранённых в сжатом буфере, в байтах.
Ошибка: Возвращает 0 и устанавливает @error ≠ 0.
@extended: Может содержать код ошибки NTSTATUS.

Примечания

Функция _WinAPI_CompressBuffer() принимает на входе несжатый буфер и выдаёт его сжатый эквивалент, при условии, что сжатые данные помещаются в указанный буфер назначения.

Для распаковки сжатого буфера используйте функцию _WinAPI_DecompressBuffer().

См. также

_WinAPI_DecompressBuffer

Пример

#include <WinAPIMem.au3>
#include <WinAPISys.au3>

; Создать сжатый и несжатый буферы
Local $a_pBuffer[2]
For $i = 0 To 1
    $a_pBuffer[$i] = _WinAPI_CreateBuffer(1024)
Next

; Сжать бинарные данные
Local $dData = Binary('0x00112233445566778899AABBCCDDEEFF00112233445566778899AABBCCDDEEFF00112233445566778899AABBCCDDEEFF00112233445566778899AABBCCDDEEFF')
Local $iSize = BinaryLen($dData)
DllStructSetData(DllStructCreate('byte[' & $iSize & ']', $a_pBuffer[0]), 1, $dData)
$iSize = _WinAPI_CompressBuffer($a_pBuffer[0], $iSize, $a_pBuffer[1], 1024, BitOR($COMPRESSION_FORMAT_LZNT1, $COMPRESSION_ENGINE_MAXIMUM))
If Not @error Then
    ConsoleWrite('Compressed: ' & DllStructGetData(DllStructCreate('byte[' & $iSize & ']', $a_pBuffer[1]), 1) & @CRLF)
EndIf

; Распаковать данные
$iSize = _WinAPI_DecompressBuffer($a_pBuffer[0], 1024, $a_pBuffer[1], $iSize)
If Not @error Then
    ConsoleWrite('Uncompressed: ' & DllStructGetData(DllStructCreate('byte[' & $iSize & ']', $a_pBuffer[0]), 1) & @CRLF)
EndIf

; Освободить память
For $i = 0 To 1
    _WinAPI_FreeMemory($a_pBuffer[$i])
Next
