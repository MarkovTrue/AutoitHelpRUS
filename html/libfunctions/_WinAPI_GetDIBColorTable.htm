_WinAPI_GetDIBColorTable

Получает таблицу цветов RGB из DIB-растра раздела.

#include <WinAPIGdi.au3>
_WinAPI_GetDIBColorTable ( $hBitmap )

Параметры

$hBitmap DIB-растр раздела, из которого следует получить таблицу цветов.

Возвращаемое значение

Успех: Возвращает структуру "dword[n]", содержащую информацию о цвете из таблицы цветов DIB. Флаг @extended будет содержать количество цветов в этой структуре.
Ошибка: Устанавливает @error ≠ 0.

Примечания

Функция _WinAPI_GetDIBColorTable() получает таблицу цветов для DIB-растров раздела, которые используют 1, 4 или 8 бит/пикс.
DIB-растры раздела, которые используют значение бит/пикс больше восьми, не имеют таблицы цветов, но имеют связанные маски цвета.
Вызовите функцию _WinAPI_GetObject(), чтобы получить эти маски цвета.
