Справочник функций

_ArrayCombinations

Возвращает массив сочетаний элементов из выбранного одномерного 1D массива.

#include <Array.au3>
_ArrayCombinations ( Const ByRef $aArray, $iSet [, $sDelimiter = ""] )

Параметры

$aArray Исходный массив.
$iSet Размер подмножества сочетания.
$sDelimiter [необязательный] Разделитель элементов в строке результата. По умолчанию "" (без разделителя).

Возвращаемое значение

Успех: Возвращает массив сочетаний.
$aArray[0] содержит количество возвращённых строк.
Остальные элементы ($aArray[1], $aArray[2] и т. д.) содержат сами сочетания.
Ошибка: Устанавливает @error ≠ 0.
@error: 1 – входной параметр должен быть массивом
2 – $aArray не 1D массив

Примечания

Входной массив должен быть с нулевым индексом – без счётчика в $aArray[0]. Реализация основана на алгоритме Kenneth H. Rosen.

См. также

_ArrayPermute

Пример

; Объявить 1D массив и создать массив, показывающий возможные комбинации.

#include <Array.au3>

Local $aArray[5] = [1, 2, 3, 4, 5]

For $i = 1 To UBound($aArray)
    Local $aArrayCombo = _ArrayCombinations($aArray, $i, ",")
    _ArrayDisplay($aArrayCombo, "iSet = " & $i)
Next
