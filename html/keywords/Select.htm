Select...Case...EndSelect

Условное выполнение операторов.

<strong>Select</strong>
<strong>Case</strong> <expression1>
<em>statement1</em>
...
[<strong>Case</strong> <expression2>
<em>statement2</em>
...]
[<strong>Case</strong> <expressionn>
<em>statementn</em>
...]
[<strong>Case</strong> Else
<em>statementElse</em>
...]
<strong>EndSelect</strong>

Параметры

<expression> Если выражение истинно, выполняются следующие инструкции до следующей инструкции Case или EndSelect. Если истинны более одного оператора Case, выполняется только первый.

Примечания

Инструкции Select могут быть вложены.
expression может содержать булевы операторы And, Or и Not, а также логические операторы <, <=, >, >=, =, ==, и <>, сгруппированные скобками по мере необходимости.

См. также

If...Then, If...Else...EndIf, Switch...EndSwitch, ContinueCase

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    Local $iValue = 0
    Local $sBlank = "Test"

    Select
        Case $iValue = 1
            MsgBox($MB_SYSTEMMODAL, "", "The first expression was True.")
        Case $sBlank = "Test"
            MsgBox($MB_SYSTEMMODAL, "", "The second expression was True")
        Case Else ; Если ничего не совпадает, то выполнить следующее.
            MsgBox($MB_SYSTEMMODAL, "", "No preceding case was True.")
    EndSelect
EndFunc   ;==>Пример
