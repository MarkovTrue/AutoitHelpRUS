Справочник ключевых слов

For...In...Next

Перечисляет элементы в коллекции Object или массиве.

<strong>For</strong> <Variable> <strong>In</strong> <expression>
<em>statements</em>
...
<strong>Next</strong>

Параметры

Variable Переменная, которой присваивается элемент.
expression Должно быть выражение, результирующее в Object или массив с по крайней мере одним элементом.

Примечания

Переменная будет создана автоматически с локальной областью видимости, даже если MustDeclareVars включен.
Если выражение – коллекция Object без элементов или многомерный массив, цикл будет пропущен и переменная будет содержать пустую строку.
Если выражение – ни Object ни Array, скрипт остановится с ошибкой, если только обработчик ошибок COM не был сконфигурирован.
Массивы AutoIt доступны только для чтения при использовании For...In. Хотя вы можете присвоить переменной внутри цикла For...In значение, это изменение не отразится на самом массиве. Для изменения содержимого массива во время перечисления используйте цикл For...To.

Инструкции For...In...Next могут быть вложены.

См. также

With...EndWith

Пример

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; С использованием массива
    Local $aArray[4]

    $aArray[0] = "a"
    $aArray[1] = 0
    $aArray[2] = 1.3434
    $aArray[3] = "test"

    Local $sString = ""
    For $vElement In $aArray
        $sString = $sString & $vElement & @CRLF
    Next

    MsgBox($MB_SYSTEMMODAL, "", "For..IN Arraytest:" & @CRLF & "Result is: " & @CRLF & $sString)

    ; С использованием коллекции объектов

    Local $oShell = ObjCreate("shell.application")
    Local $oShellWindows = $oShell.windows

    If IsObj($oShellWindows) Then
        $sString = ""

        For $Window In $oShellWindows
            $sString = $sString & $Window.LocationName & @CRLF
        Next

        MsgBox($MB_SYSTEMMODAL, "", "You have the following windows open:" & @CRLF & $sString)
    Else

        MsgBox($MB_SYSTEMMODAL, "", "You have no open shell windows.")
    EndIf
EndFunc   ;==>Пример
