Справочник ключевых слов

#RequireAdmin

Указывает, что текущий скрипт требует полных прав администратора для запуска.

<strong>#RequireAdmin</strong>

Примечания

Эта функция была в основном предназначена для того, чтобы позволить скриптам AutoIt правильно работать с Windows User Account Control (UAC).

Для более подробной информации см. AutoIt on Windows Vista.

Так как эта функция запускает новый повышенный процесс, возвращаемые значения из некоторых функций могут быть не захвачены оригиналом. Однако, когда скрипт запускается из SciTE4AutoIt3, возврат, такой как ConsoleWrite(), должен быть отображён. Если система не имеет механизма повышения (например, UAC отключен), любой новый процесс будет работать под тем же пользователем, что и оригинал.

См. также

IsAdmin

Пример

#include <MsgBoxConstants.au3>

#RequireAdmin
ConsoleWrite("! TEST1" & @CRLF)
ConsoleWrite("+ TEST1" & @CRLF)
ConsoleWrite("- TEST1" & @CRLF)
ConsoleWrite("> TEST1" & @CRLF)

If IsAdmin() Then MsgBox($MB_SYSTEMMODAL, "", "The script is running with admin rights.")
