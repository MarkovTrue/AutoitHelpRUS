Макросы каталогов

Ниже приведён список макросов каталогов. Обратная косая черта в конце пути не добавляется.

Полный список макросов – здесь.

 

Макрос Описание
Макросы для данных "Все пользователи".
@AppDataCommonDir Путь к общему Application Data.
@DesktopCommonDir Путь к общему Рабочему столу.
@DocumentsCommonDir Путь к общим документам.
@FavoritesCommonDir Путь к общим Избранным.
@ProgramsCommonDir Путь к общей папке Programs меню "Пуск".
@StartMenuCommonDir Путь к общей папке меню "Пуск".
@StartupCommonDir Путь к общей папке Startup.

Макросы для данных текущего пользователя.
@AppDataDir Путь к Roaming Application Data текущего пользователя.
@LocalAppDataDir Путь к Local Application Data текущего пользователя.
@DesktopDir Путь к Рабочему столу текущего пользователя.
@MyDocumentsDir Путь к каталогу "Мои документы".
@FavoritesDir Путь к Избранным текущего пользователя.
@ProgramsDir Путь к папке Programs текущего пользователя (в меню "Пуск").
@StartMenuDir Путь к меню "Пуск" текущего пользователя.
@StartupDir Папка Startup текущего пользователя.
@UserProfileDir Путь к папке профиля текущего пользователя.

Прочие макросы компьютерной системы:
@HomeDrive Буква диска, содержащего домашний каталог текущего пользователя.
@HomePath Каталог в домашнем пути текущего пользователя. Для получения полного пути используйте вместе с @HomeDrive.
@HomeShare Имя сервера и общего ресурса, содержащих домашний каталог текущего пользователя.
@LogonDNSDomain DNS-домен входа в систему.
@LogonDomain Домен входа в систему.
@LogonServer Сервер входа в систему.
@ProgramFilesDir Путь к папке Program Files.
@CommonFilesDir Путь к папке Common Files.
@WindowsDir Путь к папке Windows.
@SystemDir Путь к папке Windows System (или System32).
@TempDir Путь к папке временных файлов.
@ComSpec Значение %COMSPEC% – путь к интерпретатору команд; используется для запуска команд, например Run(@ComSpec & " /k help | more").
