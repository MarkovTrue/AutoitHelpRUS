Справочник GUI: ToolTip

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiToolTip.au3>.

 

Функция Описание
_GUIToolTip_Activate Активирует элемент управления ToolTip.
_GUIToolTip_AddTool Регистрирует инструмент (область) в элементе управления ToolTip.
_GUIToolTip_AdjustRect Вычисляет прямоугольник отображения текста на основе прямоугольника окна.
_GUIToolTip_BitsToTTF Декодирует битовые флаги в строки TTF_*.
_GUIToolTip_Create Создаёт элемент управления ToolTip.
_GUIToolTip_Deactivate Деактивирует элемент управления ToolTip.
_GUIToolTip_DelTool Удаляет инструмент из элемента управления подсказки.
_GUIToolTip_Destroy Уничтожает элемент управления ToolTip.
_GUIToolTip_EnumTools Получает информацию о текущем инструменте.
_GUIToolTip_GetBubbleHeight Возвращает высоту элемента управления.
_GUIToolTip_GetBubbleSize Возвращает ширину и высоту элемента управления ToolTip.
_GUIToolTip_GetBubbleWidth Возвращает ширину элемента управления ToolTip.
_GUIToolTip_GetCurrentTool Получает информацию о текущем инструменте.
_GUIToolTip_GetDelayTime Получает начальное, всплывающее или повторное время задержки элемента управления ToolTip.
_GUIToolTip_GetMargin Получает верхние, левые, нижние и правые поля элемента управления ToolTip.
_GUIToolTip_GetMarginEx Получает верхние, левые, нижние и правые поля элемента управления ToolTip.
_GUIToolTip_GetMaxTipWidth Получает максимальную ширину окна подсказки.
_GUIToolTip_GetText Получает текст инструмента.
_GUIToolTip_GetTipBkColor Получает цвет фона подсказки.
_GUIToolTip_GetTipTextColor Получает цвет текста элемента управления ToolTip.
_GUIToolTip_GetTitleBitMap Получает значок растрового изображения заголовка.
_GUIToolTip_GetTitleText Получает текст заголовка.
_GUIToolTip_GetToolCount Возвращает количество инструментов, поддерживаемых элементом управления ToolTip.
_GUIToolTip_GetToolInfo Получает информацию об инструменте в элементе управления ToolTip.
_GUIToolTip_HitTest Получает информацию о инструменте в заданной точке.
_GUIToolTip_NewToolRect Устанавливает новый ограничивающий прямоугольник для инструмента.
_GUIToolTip_Pop Удаляет отображаемую подсказку из поля зрения.
_GUIToolTip_PopUp Заставляет подсказку отображаться в координатах последнего сообщения мыши.
_GUIToolTip_SetDelayTime Устанавливает начальное, всплывающее и повторное время отображения элемента управления ToolTip.
_GUIToolTip_SetMargin Устанавливает верхние, левые, нижние и правые поля элемента управления ToolTip.
_GUIToolTip_SetMaxTipWidth Устанавливает максимальную ширину окна подсказки.
_GUIToolTip_SetTipBkColor Устанавливает цвет фона подсказки.
_GUIToolTip_SetTipTextColor Устанавливает цвет текста подсказки.
_GUIToolTip_SetTitle Устанавливает заголовок и значок элемента управления ToolTip.
_GUIToolTip_SetToolInfo Устанавливает информацию об инструменте в элементе управления ToolTip.
_GUIToolTip_SetWindowTheme Устанавливает визуальный стиль окна подсказки.
_GUIToolTip_ToolExists Определяет, существует ли инструмент в элементе управления ToolTip.
_GUIToolTip_ToolToArray Преобразует структуру ToolInfo в массив.
_GUIToolTip_TrackActivate Активирует или деактивирует отслеживающую подсказку.
_GUIToolTip_TrackPosition Устанавливает положение отслеживающей подсказки.
_GUIToolTip_Update Заставляет текущий инструмент перерисовываться при изменении.
_GUIToolTip_UpdateTipText Устанавливает текст подсказки для инструмента.
