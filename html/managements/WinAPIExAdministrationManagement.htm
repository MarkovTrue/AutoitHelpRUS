Справочник WinAPIEx: администрирование

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_LockWorkStation Блокирует дисплей рабочей станции
_WinAPI_ShutdownBlockReasonCreate Указывает, что система не может быть завершена и задаёт строку причины, которая будет отображена пользователю, если инициируется завершение работы системы
_WinAPI_ShutdownBlockReasonDestroy Указывает, что система может быть завершена и освобождает строку причины
_WinAPI_ShutdownBlockReasonQuery Получает строку причины, установленную функцией _WinAPI_ShutdownBlockReasonCreate()
