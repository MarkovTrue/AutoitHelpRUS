Справочник пользовательских функций: SendMessage

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <SendMessage.au3>.

 

Функция Описание
_SendMessage Оболочка для часто используемого вызова DLL
_SendMessageA Отправляет сообщение окну/элементу управления (принудительный вызов ANSI)
