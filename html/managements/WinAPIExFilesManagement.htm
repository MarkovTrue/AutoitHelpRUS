Справочник WinAPIEx: файлы

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIFiles.au3>.

 

Функция Описание
_WinAPI_CopyFileEx Копирует существующий файл в новый файл, уведомляя приложение о его ходе выполнения через функцию обратного вызова
_WinAPI_CreateDirectory создаёт новый каталог
_WinAPI_CreateDirectoryEx создаёт новый каталог с атрибутами шаблонного каталога
_WinAPI_CreateFile создаёт или открывает файл или другое устройство
_WinAPI_CreateFileEx создаёт или открывает файл или устройство ввода-вывода
_WinAPI_CreateFileMapping создаёт или открывает именованный или безымянный объект отображения файла
_WinAPI_CreateHardLink создаёт жёсткую ссылку между существующим файлом и новым файлом
_WinAPI_CreateSymbolicLink создаёт символическую ссылку
_WinAPI_DecryptFile Расшифровывает зашифрованный файл или каталог
_WinAPI_DeleteFile Удаляет существующий файл
_WinAPI_DeleteVolumeMountPoint Удаляет букву диска или подключённую папку
_WinAPI_DuplicateEncryptionInfoFile Копирует EFS-метаданные из одного файла или каталога в другой
_WinAPI_EncryptFile Шифрует файл или каталог
_WinAPI_EncryptionDisable Отключает или включает шифрование указанного каталога и файлов в нём
_WinAPI_EnumFiles Перечисляет файлы и подкаталоги указанного каталога, имена которых соответствуют шаблону
_WinAPI_EnumFileStreams Перечисляет все потоки типа ::$DATA в указанном файле или каталоге
_WinAPI_EnumHardLinks Перечисляет все жёсткие ссылки на указанный файл
_WinAPI_FileEncryptionStatus Получает статус шифрования указанного файла.
_WinAPI_FileExists Проверяет, существует ли файл по указанному пути
_WinAPI_FileInUse Проверяет, используется ли указанный файл другим приложением
_WinAPI_FindClose Закрывает дескриптор поиска файлов
_WinAPI_FindCloseChangeNotification Прекращает мониторинг через дескриптор уведомления об изменениях
_WinAPI_FindFirstChangeNotification Создаёт дескриптор уведомления об изменениях и устанавливает начальные условия фильтра
_WinAPI_FindFirstFile Ищет в каталоге файл или подкаталог с именем, соответствующим заданному шаблону
_WinAPI_FindFirstFileName Создаёт перечисление всех жёстких ссылок на указанный файл
_WinAPI_FindFirstStream Возвращает первый поток типа ::$DATA в указанном файле или каталоге
_WinAPI_FindNextChangeNotification Просит операционную систему подать сигнал на дескриптор уведомления об изменениях при следующем подходящем событии
_WinAPI_FindNextFile Продолжает поиск файлов или каталогов
_WinAPI_FindNextFileName Продолжает перечисление жёстких ссылок
_WinAPI_FindNextStream Продолжает поиск потоков данных.
_WinAPI_FlushFileBuffers Сбрасывает буферы указанного файла и записывает все буферизованные данные
_WinAPI_FlushViewOfFile Записывает на диск диапазон байтов в отображённой проекции файла
_WinAPI_GetBinaryType Определяет, является ли файл исполняемым файлом (.exe), и если да, какая подсистема запускает этот файл
_WinAPI_GetCompressedFileSize Получает фактическое количество байт дискового хранилища, используемых для хранения указанного файла.
_WinAPI_GetCompression Получает текущее состояние сжатия файла или каталога.
_WinAPI_GetCurrentDirectory Получает текущий каталог для текущего процесса.
_WinAPI_GetDiskFreeSpaceEx Получает информацию о количестве доступного пространства на томе диска.
_WinAPI_GetFileAttributes Получает атрибуты файловой системы для указанного файла или каталога
_WinAPI_GetFileID Получает 8-байтовый номер ссылки на файл файловой системы
_WinAPI_GetFileInformationByHandle Получает информацию о файле для указанного файла
_WinAPI_GetFileInformationByHandleEx Получает информацию о файле для указанного файла
_WinAPI_GetFilePointerEx Получает указатель файла указанного файла
_WinAPI_GetFileSizeEx Получает размер указанного файла
_WinAPI_GetFileSizeOnDisk Получает размер выделения файла на диске
_WinAPI_GetFileTitle Получает имя указанного файла
_WinAPI_GetFileType Получает тип файла указанного файла
_WinAPI_GetFinalPathNameByHandle Получает конечный полный путь указанного файла
_WinAPI_GetFinalPathNameByHandleEx Получает конечный полный путь указанного файла
_WinAPI_GetFullPathName Получает полный путь и имя файла указанного файла
_WinAPI_GetProfilesDirectory Возвращает путь к корневому каталогу, в котором хранятся пользовательские профили
_WinAPI_GetTempFileName Создаёт имя для временного файла
_WinAPI_GetVolumeInformation Получает сведения о файловой системе и томе, связанном с указанным корневым каталогом
_WinAPI_GetVolumeInformationByHandle Получает сведения о файловой системе и томе, связанном с указанным файлом
_WinAPI_GetVolumeNameForVolumeMountPoint Получает путь GUID тома для тома, связанного с указанной точкой монтирования тома
_WinAPI_IsPathShared Определяет, является ли путь общим
_WinAPI_LockDevice Включает или отключает механизм извлечения носителя для устройств, имеющих такую возможность блокировки
_WinAPI_LockFile Блокирует указанный файл для эксклюзивного доступа вызывающим процессом
_WinAPI_MapViewOfFile Отображает представление отображения файла в адресное пространство вызывающего процесса
_WinAPI_MoveFileEx Перемещает файл или каталог, уведомляя приложение о его прогрессе через функцию обратного вызова
_WinAPI_OpenFileById Открывает файл, соответствующий указанному идентификатору объекта
_WinAPI_OpenFileMapping Открывает именованный объект отображения файла
_WinAPI_PathIsDirectory Проверяет, является ли путь допустимым каталогом
_WinAPI_PathIsDirectoryEmpty Определяет, является ли указанный путь пустым каталогом
_WinAPI_ReadDirectoryChanges Получает информацию, описывающую изменения в указанном каталоге
_WinAPI_ReadFile Читает данные из файла
_WinAPI_RemoveDirectory Удаляет существующую пустую папку
_WinAPI_ReOpenFile Открывает заново указанный объект файловой системы с другими правами доступа, режимом совместного доступа и флагами
_WinAPI_ReplaceFile Заменяет один файл другим файлом и создаёт резервную копию исходного файла
_WinAPI_SearchPath Поиск указанного файла в указанном пути
_WinAPI_SetCompression Устанавливает состояние сжатия файла или каталога
_WinAPI_SetCurrentDirectory Изменяет текущий каталог для текущего процесса
_WinAPI_SetEndOfFile Устанавливает физический размер файла для указанного файла в текущей позиции указателя файла
_WinAPI_SetFileAttributes Устанавливает атрибуты для файла или каталога
_WinAPI_SetFileInformationByHandleEx Устанавливает информацию о файле для указанного файла
_WinAPI_SetFilePointer Перемещает указатель файла указанного файла
_WinAPI_SetFilePointerEx Перемещает указатель файла указанного файла
_WinAPI_SetFileShortName Устанавливает короткое имя для указанного файла
_WinAPI_SetFileValidData Устанавливает допустимую длину данных указанного файла
_WinAPI_SetSearchPathMode Задаёт режим для каждого процесса, который функция _WinAPI_SearchPath() использует при поиске файлов
_WinAPI_SetVolumeMountPoint Связывает том с буквой диска или каталогом на другом томе
_WinAPI_SfcIsFileProtected Определяет, защищён ли указанный файл
_WinAPI_UnlockFile Разблокирует регион в открытом файле
_WinAPI_UnmapViewOfFile Отменяет отображение представления файла из адресного пространства вызывающего процесса
_WinAPI_WriteFile Записывает данные в файл в позицию, указанную указателем файла
