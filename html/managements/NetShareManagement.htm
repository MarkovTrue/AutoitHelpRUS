Справочник пользовательских функций: сетевые ресурсы

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <NetShare.au3>.

 

Функция Описание
_Net_Share_ConnectionEnum Список всех подключений, сделанных к общему ресурсу
_Net_Share_FileClose Принудительно закрывает ресурс
_Net_Share_FileEnum Возвращает информацию об открытых файлах на сервере
_Net_Share_FileGetInfo Получает информацию о конкретном открытии ресурса сервера
_Net_Share_PermStr Возвращает строковое представление разрешений ресурса
_Net_Share_ResourceStr Возвращает строковое представление ресурса
_Net_Share_SessionDel Завершает сетевую сессию между сервером и рабочей станцией
_Net_Share_SessionEnum Предоставляет информацию о сессиях, установленных на сервере
_Net_Share_SessionGetInfo Получает информацию о сессии, установленной между сервером и рабочей станцией
_Net_Share_ShareAdd Предоставляет общий доступ к ресурсу сервера
_Net_Share_ShareCheck Проверяет, предоставляет ли сервер общий доступ к устройству
_Net_Share_ShareDel Удаляет имя общего ресурса из списка общих ресурсов сервера
_Net_Share_ShareEnum Получает информацию о каждом общем ресурсе на сервере
_Net_Share_ShareGetInfo Получает информацию о конкретном общем ресурсе на сервере
_Net_Share_ShareSetInfo Предоставляет общий доступ к ресурсу сервера
_Net_Share_StatisticsGetSvr Получает статистику операций для сервера
_Net_Share_StatisticsGetWrk Получает статистику операций для рабочей станции
