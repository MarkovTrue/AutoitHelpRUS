Справочник WinAPIEx: пути оболочки

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIShPath.au3>.

 

Функция Описание
_WinAPI_CommandLineToArgv Разбирает строку командной строки и возвращает массив аргументов командной строки
_WinAPI_IsNameInExpression Определяет, соответствует ли строка указанному шаблону
_WinAPI_ParseURL Выполняет простой разбор URL-адреса
_WinAPI_ParseUserName Извлекает доменное имя и имя учётной записи пользователя из полностью определённого имени пользователя
_WinAPI_PathAddBackslash Добавляет обратную косую черту в конец строки для создания корректного синтаксиса пути
_WinAPI_PathAddExtension Добавляет расширение имени файла к строке пути
_WinAPI_PathAppend Добавляет один путь в конец другого
_WinAPI_PathBuildRoot Создаёт корневой путь из заданного номера диска
_WinAPI_PathCanonicalize Удаляет элементы пути файла согласно специальным строкам, вставленным в этот путь
_WinAPI_PathCommonPrefix Сравнивает два пути для определения наличия общего префикса
_WinAPI_PathCompactPath Усекает путь файла, чтобы он поместился в заданную ширину пикселей путём замены компонентов пути на многоточия
_WinAPI_PathCompactPathEx Усекает путь, чтобы он поместился в определённое количество символов путём замены компонентов пути на многоточия
_WinAPI_PathCreateFromUrl Преобразует URL-адрес файла в путь MS-DOS компании Microsoft
_WinAPI_PathFindExtension Поиск расширения в пути
_WinAPI_PathFindFileName Поиск имени файла в пути
_WinAPI_PathFindNextComponent Анализирует путь и возвращает часть пути, которая следует после первой обратной косой черты
_WinAPI_PathFindOnPath Ищет файл в путях системы по умолчанию
_WinAPI_PathGetArgs Находит аргументы командной строки в заданном пути
_WinAPI_PathGetCharType Определяет тип символа в отношении пути
_WinAPI_PathGetDriveNumber Ищет букву диска в диапазоне от 'A' до 'Z' в пути и возвращает соответствующий номер диска
_WinAPI_PathIsContentType Определяет, совпадает ли зарегистрированный тип содержимого файла с указанным типом содержимого
_WinAPI_PathIsExe Определяет, является ли файл исполняемым путём проверки расширения файла
_WinAPI_PathIsFileSpec Ищет в пути любые символы-разделители пути
_WinAPI_PathIsLFNFileSpec Определяет, находится ли имя файла в длинном формате
_WinAPI_PathIsRelative Ищет путь и определяет, является ли он относительным
_WinAPI_PathIsRoot Анализирует путь для определения, является ли он корнем каталога
_WinAPI_PathIsSameRoot Сравнивает два пути, чтобы определить, имеют ли они общий корневой компонент
_WinAPI_PathIsSystemFolder Определяет, содержит ли существующая папка атрибуты, которые делают её системной папкой
_WinAPI_PathIsUNC Определяет, является ли строка допустимым соглашением об универсальном именовании (UNC) для пути сервера и общей папки
_WinAPI_PathIsUNCServer Определяет, является ли строка допустимым соглашением об универсальном именовании (UNC) только для пути сервера
_WinAPI_PathIsUNCServerShare Определяет, является ли строка допустимым путём общей папки соглашения об универсальном именовании (UNC)
_WinAPI_PathMakeSystemFolder Задаёт существующей папке необходимые атрибуты, чтобы она стала системной папкой
_WinAPI_PathMatchSpec Выполняет поиск в строке с использованием типа совпадения подстановочных символов MS-DOS
_WinAPI_PathParseIconLocation Разбирает строку расположения файла, которая содержит расположение файла и индекс значка
_WinAPI_PathRelativePathTo Создаёт относительный путь от одного файла или папки к другому
_WinAPI_PathRemoveArgs Удаляет любые аргументы из заданного пути
_WinAPI_PathRemoveBackslash Удаляет завершающую обратную косую черту из заданного пути
_WinAPI_PathRemoveExtension Удаляет расширение имени файла из пути, если оно присутствует
_WinAPI_PathRemoveFileSpec Удаляет имя конечного файла и обратную косую черту из пути, если они присутствуют
_WinAPI_PathRenameExtension Заменяет расширение имени файла новым расширением
_WinAPI_PathSearchAndQualify Форматирует путь в полностью уточненный путь
_WinAPI_PathSkipRoot Разбирает путь, игнорируя букву диска или элементы пути сервера/общей папки соглашения об универсальном именовании (UNC)
_WinAPI_PathStripPath Удаляет часть пути из полностью определённого пути и имени файла
_WinAPI_PathStripToRoot Удаляет все части пути, кроме информации о корневой папке
_WinAPI_PathUndecorate Удаляет украшения из строки пути
_WinAPI_PathUnExpandEnvStrings Заменяет имена папок в полностью определённом пути с их связанными переменными среды
_WinAPI_PathUnmakeSystemFolder Удаляет атрибуты из папки, которые делают её системной папкой
_WinAPI_PathUnquoteSpaces Удаляет кавычки из начала и конца пути
_WinAPI_PathYetAnotherMakeUniqueName Создаёт уникальное имя файла на основе существующего имени файла
_WinAPI_ShellGetImageList Получает системный список изображений для маленьких или больших значков
_WinAPI_UrlApplyScheme Определяет схему для указанной строки URL и возвращает строку с подходящим префиксом
_WinAPI_UrlCanonicalize Преобразует строку URL в каноническую форму
_WinAPI_UrlCombine Объединяет базовый и относительный URL в каноническую форму
_WinAPI_UrlCompare Выполняет сравнение двух строк URL с учётом регистра
_WinAPI_UrlCreateFromPath Преобразует путь MS-DOS в каноническую форму URL
_WinAPI_UrlFixup Пытается исправить URL, чей идентификатор протокола неправильный
_WinAPI_UrlGetPart Извлекает указанную часть из URL
_WinAPI_UrlHash Хеширует строку URL
_WinAPI_UrlIs Проверяет, является ли URL определённым типом
