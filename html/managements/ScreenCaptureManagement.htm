Справочник пользовательских функций: снимок экрана

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <ScreenCapture.au3>.

 

Функция Описание
_ScreenCapture_Capture Делает снимок участка экрана
_ScreenCapture_CaptureWnd Делает снимок указанного окна или элемента управления по controlID
_ScreenCapture_SaveImage Сохраняет изображение в файл
_ScreenCapture_SetBMPFormat Задаёт битовую глубину для снимков экрана в формате BMP
_ScreenCapture_SetJPGQuality Задаёт уровень качества для снимков экрана в формате JPEG
_ScreenCapture_SetTIFColorDepth Задаёт глубину цвета для снимков экрана в формате TIFF
_ScreenCapture_SetTIFCompression Задаёт способ сжатия для снимков экрана в формате TIFF
