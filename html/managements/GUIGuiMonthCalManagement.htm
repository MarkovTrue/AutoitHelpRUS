Справочник GUI: MonthCal

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <GuiMonthCal.au3>.

 

Функция Описание
_GUICtrlMonthCal_Create Создаёт элемент управления месячный календарь
_GUICtrlMonthCal_Destroy Удаляет элемент управления месячный календарь
_GUICtrlMonthCal_GetCalendarBorder Получает размер границы в пикселях
_GUICtrlMonthCal_GetCalendarCount Получает количество календарей, которые в данный момент отображаются в элементе управления календарь
_GUICtrlMonthCal_GetColor Получает заданный цвет элемента управления
_GUICtrlMonthCal_GetColorArray Получает цвет для указанной части элемента управления месячный календарь
_GUICtrlMonthCal_GetCurSel Получает текущую выделенную дату
_GUICtrlMonthCal_GetCurSelStr Получает текущую выделенную дату в формате строки
_GUICtrlMonthCal_GetFirstDOW Получает первый день недели
_GUICtrlMonthCal_GetFirstDOWStr Возвращает первый день недели в виде строки
_GUICtrlMonthCal_GetMaxSelCount Возвращает максимальный диапазон дат, который можно выделить в календаре
_GUICtrlMonthCal_GetMaxTodayWidth Возвращает максимальную ширину строки "today" в месячном календаре
_GUICtrlMonthCal_GetMinReqHeight Получает минимальную высоту, необходимую для отображения полного месяца
_GUICtrlMonthCal_GetMinReqRect Возвращает минимальный размер, необходимый для отображения целого месяца
_GUICtrlMonthCal_GetMinReqRectArray Возвращает минимальный размер, необходимый для отображения целого месяца в месячном календаре
_GUICtrlMonthCal_GetMinReqWidth Возвращает минимальную ширину, необходимую для отображения целого месяца
_GUICtrlMonthCal_GetMonthDelta Возвращает шаг прокрутки месячного календаря
_GUICtrlMonthCal_GetMonthRange Возвращает данные о верхней и нижней границах отображения
_GUICtrlMonthCal_GetMonthRangeMax Возвращает данные о верхней границе отображения элемента
_GUICtrlMonthCal_GetMonthRangeMaxStr Возвращает данные о верхней границе отображения элемента в виде строки
_GUICtrlMonthCal_GetMonthRangeMin Возвращает данные о нижней границе отображения элемента
_GUICtrlMonthCal_GetMonthRangeMinStr Возвращает данные о нижней границе отображения элемента в виде строки
_GUICtrlMonthCal_GetMonthRangeSpan Возвращает значение, представляющее охватываемый диапазон в месяцах
_GUICtrlMonthCal_GetRange Возвращает минимально и максимально допустимые даты
_GUICtrlMonthCal_GetRangeMax Возвращает верхнюю границу диапазона дат
_GUICtrlMonthCal_GetRangeMaxStr Возвращает верхнюю границу диапазона дат в виде строки
_GUICtrlMonthCal_GetRangeMin Возвращает нижнюю границу диапазона дат
_GUICtrlMonthCal_GetRangeMinStr Возвращает нижнюю границу диапазона дат в виде строки
_GUICtrlMonthCal_GetSelRange Возвращает верхнюю и нижнюю границы выделенного диапазона дат
_GUICtrlMonthCal_GetSelRangeMax Возвращает верхнюю границу выделенного пользователем диапазона дат
_GUICtrlMonthCal_GetSelRangeMaxStr Возвращает верхнюю границу выделенного пользователем диапазона дат в виде строки
_GUICtrlMonthCal_GetSelRangeMin Возвращает нижнюю границу выделенного пользователем диапазона дат
_GUICtrlMonthCal_GetSelRangeMinStr Возвращает нижнюю границу выделенного пользователем диапазона дат в виде строки
_GUICtrlMonthCal_GetToday Возвращает данные о дате, указанной как "today" (сегодня)
_GUICtrlMonthCal_GetTodayStr Возвращает данные о дате, указанной как "today" (сегодня), в виде строки
_GUICtrlMonthCal_GetUnicodeFormat Возвращает флаг формата символов Unicode для элемента управления
_GUICtrlMonthCal_HitTest Определяет, какая часть элемента управления месячный календарь находится в указанной точке
_GUICtrlMonthCal_SetCalendarBorder Задаёт размер границы в пикселях
_GUICtrlMonthCal_SetColor Устанавливает цвет для заданной части месячного календаря
_GUICtrlMonthCal_SetCurSel Устанавливает текущую выделенную дату
_GUICtrlMonthCal_SetDayState Задаёт состояния дней для всех видимых месяцев
_GUICtrlMonthCal_SetFirstDOW Задаёт первый день недели для месячного календаря
_GUICtrlMonthCal_SetMaxSelCount Задаёт максимальное количество дней, которые можно выделить в месячном календаре
_GUICtrlMonthCal_SetMonthDelta Задаёт шаг прокрутки месячного календаря
_GUICtrlMonthCal_SetRange Задаёт даты, представляющие верхнюю и нижнюю границы диапазона
_GUICtrlMonthCal_SetSelRange Устанавливает выделение в месячном календаре по заданному диапазону дат
_GUICtrlMonthCal_SetToday Устанавливает информацию о дате для указанной в качестве сегодняшней даты
_GUICtrlMonthCal_SetUnicodeFormat Задаёт флаг формата символов Unicode для элемента управления
