Справочник WinAPIEx: буферы обмена

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPISys.au3>.

 

Функция Описание
_WinAPI_AddClipboardFormatListener Помещает указанное окно в список подписчиков формата буфера обмена, поддерживаемый системой
_WinAPI_GetClipboardSequenceNumber Получает номер последовательности буфера обмена для текущей рабочей станции.
_WinAPI_RemoveClipboardFormatListener Удаляет данное окно из поддерживаемого системой списка слушателей формата буфера обмена
