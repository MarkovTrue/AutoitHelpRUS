Справочник пользовательских функций: отладка

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Debug.au3>.

 

Функция Описание
_Assert Отображает сообщение при невыполнении утверждения (assertion).
_DebugArrayDisplay Отображает одномерный или двумерный массив в элементе ListView для помощи при отладке.
_DebugBugReportEnv Возвращает строку с информацией, необходимой для отправки отчёта об ошибке.
_DebugCOMError Устанавливает, сбрасывает или запрашивает уровень отладки COM-ошибок.
_DebugOut Выводит данные в сеанс отладки, запущенный функцией _DebugSetup().
_DebugReport Записывает данные в сеанс отладки.
_DebugReportData Записывает содержимое данных в сеанс отладки.
_DebugReportEx Записывает в сеанс отладки форматированное сообщение.
_DebugReportVar Записывает содержимое переменной в сеанс отладки.
_DebugSetup Настраивает сеанс отладки с заданным типом вывода.
