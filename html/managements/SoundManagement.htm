Справочник пользовательских функций: звук

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <Sound.au3>.

 

Функция Описание
_SoundClose Закрывает звук, ранее открытый с помощью _SoundOpen()
_SoundLength Возвращает длительность звукового файла
_SoundOpen Открывает звуковой файл для использования с другими функциями _Sound
_SoundPause Ставит звук на паузу
_SoundPlay Воспроизводит звуковой файл
_SoundPos Возвращает текущую позицию воспроизведения звука
_SoundResume Возобновляет воспроизведение приостановленного звука
_SoundSeek Перемещает звук на указанную позицию
_SoundStatus Возвращает состояние звука
_SoundStop Останавливает воспроизводящийся звук
