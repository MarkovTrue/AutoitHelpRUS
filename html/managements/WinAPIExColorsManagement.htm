Справочник WinAPIEx: цвета

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIGdi.au3>.

 

Функция Описание
_WinAPI_ColorAdjustLuma Изменяет светлоту значения RGB
_WinAPI_ColorHLSToRGB Преобразует цвета из формата оттенок-светлота-насыщенность (HLS) в RGB
_WinAPI_ColorRGBToHLS Преобразует цвета из RGB в формат оттенок-светлота-насыщенность (HLS)
_WinAPI_CreateColorAdjustment создаёт структуру $tagCOLORADJUSTMENT с указанной коррекцией цвета
_WinAPI_GetBValue Получает значение интенсивности синей компоненты 32-битного значения RGB
_WinAPI_GetColorAdjustment Получает коррекцию цвета для указанного контекста устройства (DC).
_WinAPI_GetDeviceGammaRamp Получает таблицу гамма-коррекции с полноцветных видеоадаптеров, поддерживающих аппаратную загрузку гамма-таблицы.
_WinAPI_GetGValue Получает значение интенсивности зелёного компонента 32-битного значения RGB
_WinAPI_GetRValue Получает значение интенсивности красной составляющей 32-битного значения RGB
_WinAPI_GetUDFColorMode Получает текущий режим цветов для библиотеки UDF WinAPIEx
_WinAPI_InvertColor Инвертирует (негатив) указанный цвет
_WinAPI_RGB Создаёт значение RGB-цвета на основе компонентов красного, зелёного и синего
_WinAPI_SetColorAdjustment Устанавливает коррекцию цвета для контекста устройства (DC)
_WinAPI_SetDeviceGammaRamp Устанавливает гамму на прямых цветовых дисплеях, поддерживающих загружаемые гаммы в оборудовании
_WinAPI_SetUDFColorMode Задаёт режим цвета для библиотеки WinAPIEx
_WinAPI_SwitchColor Преобразует цвет из BGR в RGB и наоборот
