Справочник WinAPIEx: диалоговые окна

Ниже приведён полный список пользовательских функций, доступных в AutoIt. Нажмите на имя функции для просмотра подробного описания.

Для использования добавьте #include <WinAPIDlg.au3>.

 

Функция Описание
_WinAPI_BrowseForFolderDlg Отображает диалоговое окно, которое позволяет пользователю выбрать папку оболочки
_WinAPI_CommDlgExtendedError Возвращает строку ошибки стандартного диалога. Эта строка указывает на самую последнюю ошибку, возникшую при выполнении одной из функций стандартного диалога
_WinAPI_CommDlgExtendedErrorEx Возвращает код ошибки стандартного диалога
_WinAPI_ConfirmCredentials Подтверждает действительность собранного учётного данного
_WinAPI_FindTextDlg Создаёт системное немодальное диалоговое окно "Найти" для поиска текста в документе
_WinAPI_FlushFRBuffer Уничтожает внутренний буфер, используемый функциями _WinAPI_FindTextDlg() и _WinAPI_ReplaceTextDlg()
_WinAPI_FormatDriveDlg Открывает системный диалог форматирования
_WinAPI_GetConnectedDlg Запускает мастер подключения внутри вызывающего приложения для обеспечения сетевого подключения.
_WinAPI_GetDlgCtrlID Возвращает идентификатор указанного элемента управления
_WinAPI_GetFRBuffer Получает текущий размер внутреннего буфера, используемого функциями _WinAPI_FindTextDlg() и _WinAPI_ReplaceTextDlg()
_WinAPI_GetOpenFileName Создаёт диалоговое окно "Открыть", которое позволяет пользователю указать диск, каталог и имя файла или набора файлов для открытия
_WinAPI_GetSaveFileName Создаёт диалоговое окно сохранения, которое позволяет пользователю указать диск, каталог и имя файла для сохранения
_WinAPI_MessageBoxCheck Отображает окно сообщения, которое даёт пользователю возможность подавлять дальнейшие вхождения
_WinAPI_MessageBoxIndirect Создаёт, отображает и управляет окном сообщения
_WinAPI_OpenFileDlg Создаёт диалог открытия файла, который позволяет пользователю указать диск, каталог и имя файла или набора файлов для открытия
_WinAPI_PageSetupDlg Создаёт диалоговое окно параметров страницы, позволяющее пользователю задать атрибуты печатаемой страницы
_WinAPI_PickIconDlg Отображает диалоговое окно, которое позволяет пользователю выбрать значок
_WinAPI_PrintDlg Отображает диалоговое окно печати
_WinAPI_PrintDlgEx Отображает лист свойств печати, который позволяет пользователю указать свойства определённого задания печати
_WinAPI_ReplaceTextDlg Создаёт определённое системой немодальное диалоговое окно, которое позволяет пользователю задать строку для поиска и строку замены
_WinAPI_RestartDlg Отображает диалоговое окно, которое подсказывает пользователю перезагрузить Microsoft Windows
_WinAPI_SaveFileDlg Создаёт диалоговое окно, позволяющее пользователю указать диск, каталог и имя файла для сохранения
_WinAPI_SetFRBuffer Устанавливает размер внутреннего буфера, используемого функциями _WinAPI_FindTextDlg() и _WinAPI_ReplaceTextDlg()
_WinAPI_ShellAboutDlg Отображает диалог "О программе" оболочки
_WinAPI_ShellOpenWithDlg Отображает диалоговое окно "Открыть с помощью"
_WinAPI_ShellStartNetConnectionDlg Отображает диалоговое окно общего просмотра для подключения сетевого ресурса
_WinAPI_ShellUserAuthenticationDlg Создаёт и отображает настраиваемое диалоговое окно, которое принимает информацию учётных данных от пользователя
_WinAPI_ShellUserAuthenticationDlgEx Создаёт и отображает настраиваемое диалоговое окно, которое принимает информацию учётных данных от пользователя
